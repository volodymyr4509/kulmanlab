# Redo

La commande `redo` rétablit la dernière action inversée par [Undo](./undo), avançant dans la pile d'historique. Redo n'est disponible qu'après au moins un Undo et est effacé dès qu'une nouvelle action de dessin est effectuée.

## Rétablir une action

- Tapez `redo` dans le terminal, cliquez sur **Redo** dans la barre d'outils, ou appuyez sur **Ctrl+Y** (Windows/Linux) / **Cmd+Y** (macOS) / **Ctrl+Shift+Z** / **Cmd+Shift+Z**.

## Raccourci clavier

| Touche | Action |
|--------|--------|
| `Ctrl+Y` / `Cmd+Y` | Rétablir la dernière action annulée |
| `Ctrl+Shift+Z` / `Cmd+Shift+Z` | Rétablir (alternative) |
