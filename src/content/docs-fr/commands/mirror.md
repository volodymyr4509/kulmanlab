# Mirror

La commande `mirror` symétrise les entités sélectionnées par rapport à une ligne de symétrie définie par deux clics. Les originaux sont **toujours conservés** — Mirror crée de nouvelles copies symétriques.

## Symétriser des entités

**Avec pré-sélection :**
1. Sélectionnez les entités.
2. Tapez `mirror` ou cliquez sur **Mirror**.
3. **Cliquez sur le premier point** de l'axe de symétrie.
4. **Cliquez sur le deuxième point** — les copies symétriques sont placées.

**Sans pré-sélection :**
1. Tapez `mirror` ou cliquez sur **Mirror**.
2. **Sélectionnez les entités**, puis appuyez sur **Entrée** ou **Espace**.
3. **Définissez l'axe de symétrie** par deux clics.

L'axe se verrouille à des incréments de 45° lors du déplacement du curseur.

## Référence clavier

| Touche | Action |
|--------|--------|
| `0`–`9`, `.`, `-` | Saisie de coordonnée X |
| `,` | Verrouiller X et passer à Y |
| `Entrée` | Confirmer la coordonnée ou la sélection |
| `Espace` | Confirmer la sélection |
| `Échap` | Annuler |
