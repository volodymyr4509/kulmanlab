# PageManager

La commande `PageManager` ouvre le dialogue de configuration de page pour la mise en page active, vous permettant de changer le format du papier, l'orientation et l'échelle entre les unités de dessin et les millimètres.

## Ouvrir le Gestionnaire de Pages

- Tapez `PageManager` dans le terminal, **ou**
- Cliquez droit sur un onglet de mise en page en bas et choisissez **Page Manager**.

## Paramètres

| Réglage | Description |
|---------|-------------|
| **Format du papier** | Taille de feuille standard (A4, A3, A2, Letter, etc.) |
| **Orientation** | Portrait ou paysage |
| **Échelle** | Unités de dessin par mm — contrôle la correspondance entre les coordonnées du modèle et le papier |

Changer la taille du papier ou l'échelle redimensionne tous les viewports existants dans la mise en page proportionnellement pour que leurs positions restent correctes.

## Notes

- PageManager s'applique uniquement aux mises en page papier. Il n'est pas disponible dans l'espace modèle.
- Pour ajouter une nouvelle mise en page, cliquez sur le bouton **+** dans la barre d'onglets en bas.
- Pour renommer ou supprimer une mise en page, cliquez droit sur son onglet.
