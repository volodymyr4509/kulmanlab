# Rotate

La commande `rotate` fait pivoter les entités sélectionnées autour d'un point de base. L'angle de rotation peut être tapé directement ou défini par la direction du curseur depuis le point de base.

## Faire pivoter des entités

**Avec pré-sélection :**
1. Sélectionnez les entités.
2. Tapez `rotate` ou cliquez sur **Rotate**.
3. **Cliquez sur le point de base** — le centre de rotation.
4. **Tapez l'angle** et appuyez sur **Entrée**, ou cliquez pour définir l'angle par la direction du curseur.

**Sans pré-sélection :**
1. Tapez `rotate` ou cliquez sur **Rotate**.
2. **Sélectionnez les entités**, puis appuyez sur **Entrée** ou **Espace**.
3. **Cliquez sur le point de base**, puis **définissez l'angle**.

Les angles positifs sont dans le **sens antihoraire** (convention DXF). Tapez `-90` pour une rotation de 90° dans le sens horaire.

## Référence clavier

| Touche | Action |
|--------|--------|
| `0`–`9`, `.`, `-` | Saisie de l'angle |
| `Retour arrière` | Supprimer le dernier caractère |
| `Entrée` | Confirmer l'angle ou la sélection |
| `Espace` | Confirmer la sélection |
| `Échap` | Annuler |
