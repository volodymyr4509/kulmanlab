# Scale

La commande `scale` redimensionne les entités sélectionnées uniformément par un facteur saisi autour d'un point de base fixe. Le facteur est toujours saisi au clavier — il n'y a pas de mise à l'échelle par clic.

## Mettre à l'échelle des entités

**Avec pré-sélection :**
1. Sélectionnez les entités.
2. Tapez `scale` ou cliquez sur **Scale**.
3. **Cliquez sur le point de base** — le centre de la mise à l'échelle.
4. **Tapez le facteur d'échelle** et appuyez sur **Entrée**.

**Sans pré-sélection :**
1. Tapez `scale` ou cliquez sur **Scale**.
2. **Sélectionnez les entités**, puis appuyez sur **Entrée** ou **Espace**.
3. **Cliquez sur le point de base**, puis **tapez le facteur**.

Un facteur de `2` double la taille ; `0.5` la réduit de moitié. Les facteurs négatifs sont refusés.

## Référence clavier

| Touche | Action |
|--------|--------|
| `0`–`9`, `.` | Saisie du facteur d'échelle |
| `Retour arrière` | Supprimer le dernier caractère |
| `Entrée` | Confirmer le facteur ou la sélection |
| `Espace` | Confirmer la sélection |
| `Échap` | Annuler |
