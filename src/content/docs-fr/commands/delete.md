# Delete

La commande `delete` supprime les entités sélectionnées du dessin. L'opération est annulable via [Undo](./undo).

## Supprimer des entités

**Avec pré-sélection :**
1. Sélectionnez les entités.
2. Tapez `delete`, cliquez sur **Delete**, ou appuyez sur la touche **Suppr** — les entités sont supprimées immédiatement.

**Sans pré-sélection :**
1. Tapez `delete` ou cliquez sur **Delete**.
2. **Sélectionnez les entités**, puis appuyez sur **Entrée** ou **Espace**.

## Raccourci clavier global

La touche **Suppr** supprime les entités sélectionnées à tout moment, même lorsqu'aucune commande n'est active. C'est le moyen le plus rapide de supprimer une sélection.

## Annulation

La suppression peut être annulée avec [Undo](./undo) (Ctrl+Z / Cmd+Z). Jusqu'à 20 étapes d'annulation sont disponibles par fichier.
