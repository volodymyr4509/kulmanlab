# Help

Ouvre le panneau d'aide intégré avec un résumé de tous les raccourcis clavier, conseils de navigation et utilisation générale.

## Comment utiliser

Cliquez sur le bouton **Help** dans la barre d'outils ou tapez `help` dans le terminal. Le panneau d'aide s'ouvre immédiatement et la commande se termine.

## Ce que contient le panneau d'aide

### Navigation
| Raccourci | Action |
|-----------|--------|
| `Molette de défilement` | Zoomer / Dézoomer |
| `Clic central + glisser` | Déplacer la vue |
| `Double clic central` | Ajuster toutes les entités à l'écran |

### Commandes
| Raccourci | Action |
|-----------|--------|
| Taper | Commencez à taper un nom de commande — des suggestions apparaissent sur le canevas |
| `Suppr` | Supprimer toutes les entités sélectionnées |
| `Entrée` / `Espace` | Confirmer la commande ou la saisie |
| `Entrée` / `Espace` (inactif) | Répéter la dernière commande utilisée |
| `Échap` | Annuler la commande actuelle ou effacer la sélection |

### Sélection
| Raccourci | Action |
|-----------|--------|
| `Clic gauche` | Sélectionner une entité |
| `Glisser →` | Sélectionner uniquement les entités **complètement à l'intérieur** de la zone de glissement |
| `Glisser ←` | Sélectionner toute entité qui **touche** la zone de glissement |
| `Clic gauche sur poignée` | Modifier un sommet ou une extrémité directement |

### Conseils de dessin
- **Points d'accrochage** — le curseur s'accroche automatiquement aux extrémités proches des entités.
- **Angle fixe** — déplacez le curseur suffisamment loin dans une direction pour le verrouiller par incréments de 45°.
- **Longueur exacte** — pendant que l'angle est verrouillé, tapez un nombre et appuyez sur `Entrée` pour définir une distance exacte.

## Version

La version actuelle de l'application est affichée dans le coin supérieur gauche du panneau d'aide (ex. `KULMAN 2026.04.15`).
