# Text Editor

Ο επεξεργαστής κειμένου ανοίγει κατά την τοποθέτηση ή το διπλό κλικ σε επεξεργάσιμη οντότητα. Ένα μικρό **mode chip** στην κεφαλίδα — **rich** (χρώμα έμφασης) ή **simple** (αχνό) — δείχνει ποιο mode είναι ενεργό για την τρέχουσα οντότητα.

## Modes επεξεργαστή

### Rich mode

Χρησιμοποιείται από: **Text** (ετικέτες MTEXT) και σχόλια **Multileader**.

| Λειτουργία | Συμπεριφορά |
|-----------|------------|
| Bold / Italic / Strikethrough | Ανά χαρακτήρα (εφαρμόζεται σε επιλογή, ή σε ολόκληρη οντότητα χωρίς επιλογή) |
| Font και Height | Παράκαμψη ανά χαρακτήρα, ή προεπιλογή ολόκληρης οντότητας |
| `Enter` | Εισάγει κύρια αλλαγή γραμμής |
| `Shift+←/→` | Επεκτείνει ή συρρικνώνει επιλογή κειμένου |
| `Home` / `End` | Μεταφορά στην αρχή / τέλος της τρέχουσας κύριας γραμμής |
| Αναδίπλωση λέξεων | Υποστηρίζεται μέσω grips αλλαγής μεγέθους πλάτους αναφοράς |

### Simple mode

Χρησιμοποιείται από: **Dimension Linear**, **Dimension Aligned**, **Dimension Angular**, **Dimension Radius**, **Dimension Diameter**.

Ο επεξεργαστής είναι προ-συμπληρωμένος με την τρέχουσα αποδοσμένη ετικέτα διάστασης ώστε να μπορείτε να τοποθετήσετε τον κέρσορα και να επεξεργαστείτε την τιμή απευθείας.

| Λειτουργία | Συμπεριφορά |
|-----------|------------|
| Bold / Italic / Font / Height | Διαθέσιμο — εφαρμόζεται σε **ολόκληρη** την ετικέτα ταυτόχρονα |
| Μορφοποίηση ανά χαρακτήρα | Δεν υποστηρίζεται |
| `Enter` | **Επικυρώνει** την τιμή και κλείνει τον επεξεργαστή (χωρίς αλλαγή γραμμής) |
| Πολυγραμμικό | Δεν υποστηρίζεται |
| Αναδίπλωση λέξεων | Δεν υποστηρίζεται |

## Άνοιγμα επεξεργαστή

| Ενέργεια | Αποτέλεσμα |
|----------|-----------|
| Εντολή `text` → κλικ θέσης | Δημιουργεί νέα οντότητα κειμένου και ανοίγει τον επεξεργαστή (**rich**) |
| Διπλό κλικ σε υπάρχουσα οντότητα **Text** | Ανοίγει ξανά τον επεξεργαστή σε **rich** mode |
| Διπλό κλικ σε υπάρχον **Multileader** | Ανοίγει τον επεξεργαστή σε **rich** mode |
| Διπλό κλικ σε οντότητα **διάστασης** | Ανοίγει τον επεξεργαστή σε **simple** mode |
| `Escape` μέσα στον επεξεργαστή | Κλείνει τον επεξεργαστή και κρατά όλες τις αλλαγές |

## Γραμμή εργαλείων

Η γραμμή εργαλείων επιπλέει πάνω από το πλαίσιο οριοθέτησης του κειμένου και παραμένει αγκυρωμένη στην οντότητα καθώς κάνετε pan ή ζουμ.

### Έντονο · Πλάγιο · Διαγραμμισμένο

| Κουμπί | Συντόμευση | Τι κάνει |
|--------|-----------|----------|
| **B** | — | Εναλλαγή έντονης γραφής |
| *I* | — | Εναλλαγή πλάγιας γραφής |
| ~~S~~ | — | Εναλλαγή διαγραμμισμένης γραφής |

**Πώς εφαρμόζεται η εναλλαγή:**

- **Με επιλογή κειμένου** — το στυλ εφαρμόζεται ακριβώς στους επιλεγμένους χαρακτήρες μόνο.
- **Χωρίς επιλογή, κέρσορας σε υπάρχον κείμενο** — εναλλαγή στυλ σε ολόκληρη την οντότητα (όλα τα τμήματα).
- **Κενό κείμενο ή νέα οντότητα** — το στυλ αποθηκεύεται στο κενό τμήμα και εφαρμόζεται σε κάθε χαρακτήρα που πληκτρολογείτε από εκεί.

Το κουμπί εμφανίζεται επισημασμένο (ενεργό) όταν κάθε χαρακτήρας στην τρέχουσα επιλογή — ή ο χαρακτήρας αμέσως αριστερά του κέρσορα — έχει οριστεί με αυτό το στυλ.

### Γραμματοσειρά

Το αναπτυσσόμενο ομαδοποιεί τις διαθέσιμες γραμματοσειρές σε **Default** (η ενσωματωμένη γραμματοσειρά sans-serif), **User** (οι δικές σας γραμματοσειρές που έχετε μεταφορτώσει, εφόσον υπάρχουν), **Free** (ένα σύνολο ενσωματωμένων Google Fonts) και **System** (κοινές γραμματοσειρές λειτουργικού συστήματος, όπως Helvetica, Times New Roman, Georgia, Courier New, Verdana, Tahoma, Trebuchet MS, Lucida Console και Impact).

- **Με επιλογή** — παρακάμπτει τη γραμματοσειρά μόνο για τους επιλεγμένους χαρακτήρες.
- **Χωρίς επιλογή** — εφαρμόζει τη γραμματοσειρά σε ολόκληρη την οντότητα.

Το αναπτυσσόμενο αντικατοπτρίζει τη γραμματοσειρά του χαρακτήρα αριστερά του κέρσορα όταν δεν υπάρχει επιλογή.

Δεν περιορίζεστε στην ενσωματωμένη λίστα — κάντε κλικ στο κουμπί **Font Manager** στη γραμμή εργαλείων για να μεταφορτώσετε το δικό σας αρχείο `.ttf` και να το προσθέσετε στην ομάδα **User**. Δείτε το [Font Manager](../../commands/font-manager/) για λεπτομέρειες.

### Ύψος

Το πεδίο αριθμού ορίζει το **ύψος κεφαλαίου** (ύψος κεφαλαίου γράμματος) σε μονάδες σχεδίασης.

- **Με επιλογή** — παρακάμπτει το ύψος για τους επιλεγμένους χαρακτήρες, ανεξάρτητα από το βασικό ύψος οντότητας.
- **Χωρίς επιλογή** — αλλάζει το βασικό ύψος οντότητας (εφαρμόζεται σε όλους τους χαρακτήρες που δεν έχουν μεμονωμένη παράκαμψη ύψους).

Το πεδίο αντικατοπτρίζει το ύψος του χαρακτήρα αριστερά του κέρσορα. Αφήστε το κενό για χρήση της προεπιλογής οντότητας.

## Κέρσορας και πλοήγηση

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `←` / `→` | Μετακίνηση κέρσορα ένα χαρακτήρα αριστερά ή δεξιά |
| `Home` | Μεταφορά στην αρχή της τρέχουσας κύριας γραμμής |
| `End` | Μεταφορά στο τέλος της τρέχουσας κύριας γραμμής |
| `Shift` + `←` / `→` | Επέκταση ή συρρίκνωση επιλογής |
| `Backspace` | Διαγραφή χαρακτήρα αριστερά (ή επιλογής) |
| `Delete` | Διαγραφή χαρακτήρα δεξιά (ή επιλογής) |
| `Enter` | Εισαγωγή αλλαγής γραμμής |
| `Escape` | Κλείσιμο επεξεργαστή |

Το ύψος κέρσορα αντιστοιχίζεται αυτόματα στο ύψος κεφαλαίου του παρακείμενου χαρακτήρα, συμπεριλαμβανομένου του μικρότερου μεγέθους που χρησιμοποιείται για δείκτες και εκθέτες.

## Αντιγραφή, αποκοπή και επικόλληση

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `Ctrl+C` / `Cmd+C` | Αντιγραφή του επιλεγμένου κειμένου |
| `Ctrl+X` / `Cmd+X` | Αποκοπή του επιλεγμένου κειμένου |
| `Ctrl+V` / `Cmd+V` | Επικόλληση στον κέρσορα |

Η αντιγραφή και η αποκοπή απαιτούν ενεργή επιλογή κειμένου. Το επικολλημένο κείμενο είναι πάντα απλό — παίρνει τη μορφοποίηση (έντονα, πλάγια, γραμματοσειρά, ύψος) που υπάρχει ήδη στον κέρσορα, αντί να διατηρεί τη μορφοποίηση που είχε όταν αντιγράφηκε.

Στο **Rich mode**, οι αλλαγές γραμμής στο επικολλημένο κείμενο διατηρούνται. Στο **Simple mode**, οι αλλαγές γραμμής αφαιρούνται, καθώς οι ετικέτες διαστάσεων είναι μονογραμμικές.

## Αναδίπλωση λέξεων

Όταν μια οντότητα κειμένου έχει ορισμένο **πλάτος αναφοράς**, οι μακριές γραμμές αναδιπλώνονται σε όρια λέξεων για να χωρέσουν εντός αυτού του πλάτους.

Για ορισμό ή αλλαγή πλάτους αναφοράς ενώ η οντότητα είναι επιλεγμένη, σύρτε τα **grips αλλαγής μεγέθους** — τα λεπτά ορθογώνια στα αριστερά και δεξιά άκρα του διακεκομμένου πλαισίου οριοθέτησης. Το περιεχόμενο αναδιαμορφώνεται σε πραγματικό χρόνο καθώς σύρετε.

Ορισμός πλάτους αναφοράς σε μηδέν (σύρτε τα grips μαζί ή διαγράψτε την τιμή στο πάνελ ιδιοτήτων) αφαιρεί την αναδίπλωση λέξεων και επιτρέπει στις γραμμές να αναπτύσσονται ελεύθερα.

## Πολυγραμμικό κείμενο

Πατήστε `Enter` για εισαγωγή κύριας αλλαγής γραμμής. Κάθε κύρια γραμμή είναι ανεξάρτητη — `Home` και `End` πλοηγούνται εντός της τρέχουσας κύριας γραμμής μόνο.

Οι κύριες αλλαγές γραμμής και η μορφοποίηση ανά χαρακτήρα αποθηκεύονται χρησιμοποιώντας τη μορφή MTEXT και επιβιώνουν πλήρους DXF round-trip.

## Συμβατότητα DXF

Οι οντότητες κειμένου αποθηκεύονται ως **MTEXT** στα αρχεία DXF. Η έντονη και πλάγια γραφή κωδικοποιείται ως `\L`, `\K`, `\O`, και ενσωματωμένες εναλλαγές γραμματοσειράς (`\f`). Το ύψος ανά χαρακτήρα κωδικοποιείται ως `\H`. Όλη η μορφοποίηση διατηρείται κατά την εξαγωγή και είναι αναγνώσιμη από LibreCAD, FreeCAD και άλλες εφαρμογές συμβατές με DXF.
