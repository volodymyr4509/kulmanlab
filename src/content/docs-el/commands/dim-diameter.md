# Dimension Diameter

Η εντολή `dimdiameter` τοποθετεί διάσταση διαμέτρου σε τόξο ή κύκλο. Η γραμμή διάστασης εκτείνεται σε ολόκληρη τη διάμετρο — περνώντας μέσα από το κέντρο μεταξύ δύο διαμετρικά αντίθετων σημείων τόξου — και επισημαίνεται `⌀ <τιμή>`. Για σήμανση μόνο της ακτίνας από κέντρο σε ένα άκρο, χρησιμοποιήστε [Dimension Radius](../dim-radius/).

## Ανατομία διάστασης διαμέτρου

```
  ●──────────── ⌀ 10.00 ────────────●
  (far arc point)              (near arc point / text side)
```

- **Γραμμή διάστασης** — εκτείνεται σε ολόκληρη τη διάμετρο, με βέλη και στα δύο σημεία τομής τόξου.
- **Κοντινό σημείο τόξου** — το σημείο περιφέρειας στην πλευρά του κέρσορα (όπου βρίσκεται η ετικέτα κειμένου).
- **Απομακρυσμένο σημείο τόξου** — το διαμετρικά αντίθετο σημείο.
- **Ετικέτα** — `⌀` ακολουθούμενο από την τιμή διαμέτρου.

## Τοποθέτηση διάστασης διαμέτρου

1. Πληκτρολογήστε `dimdiameter` στο τερματικό ή κάντε κλικ στο κουμπί **Dimension Diameter** της γραμμής εργαλείων.
2. **Κάντε κλικ σε τόξο ή κύκλο** για επιλογή.
3. **Μετακινήστε τον κέρσορα** για περιστροφή της γραμμής διάστασης στην επιθυμητή γωνία.
4. **Κάντε κλικ** για τοποθέτηση της διάστασης.

Μόνο οντότητες **Arc** και **Circle** μπορούν να επιλεγούν.

## Αναφορά πληκτρολογίου

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `Escape` | Ακύρωση |

## Dimension Diameter vs Dimension Radius

| | Dimension Diameter | Dimension Radius |
|---|-------------------|-----------------|
| Μετράει | Ολόκληρη διάμετρο (2 × ακτίνα) | Ακτίνα (κέντρο σε άκρο) |
| Γραμμή διάστασης | Άκρο → άκρο μέσω κέντρου | Κέντρο → σημείο τόξου |
| Πρόθεμα ετικέτας | `⌀` | `R` |
| Βέλη | Δύο (και στα δύο σημεία τόξου) | Ένα (στο σημείο τόξου) |
| Καλύτερο για | Διαστάσεις πλήρους κυκλικής οπής ή άξονα | Σήμανση μίας πλευράς καμπυλωτού χαρακτηριστικού |

## Επεξεργασία ετικέτας — simple mode

**Διπλό κλικ** σε τοποθετημένη διάσταση διαμέτρου για άνοιγμα του επεξεργαστή κειμένου σε **simple** mode. Ο επεξεργαστής είναι προ-συμπληρωμένος με την τρέχουσα αποδοσμένη τιμή (π.χ. `⌀ 10.00`) ώστε να μπορείτε να τοποθετήσετε τον κέρσορα και να την επεξεργαστείτε απευθείας.

| Λειτουργία | Συμπεριφορά |
|-----------|------------|
| Bold / Italic / Font / Height | Εφαρμόζεται σε **ολόκληρη** την ετικέτα ταυτόχρονα |
| Μορφοποίηση ανά χαρακτήρα | Δεν υποστηρίζεται |
| `Enter` | Επικυρώνει την τιμή και κλείνει τον επεξεργαστή |
| Πολυγραμμικό | Δεν υποστηρίζεται |

Δείτε [Text Editor — simple mode](../../interface/text-editor/#simple-mode) για την πλήρη αναφορά.

## DXF — οντότητα DIMENSION διαμέτρου

Οι διαστάσεις διαμέτρου αποθηκεύονται ως οντότητες `DIMENSION` με γεωμετρία τύπου diameter, αποθηκεύοντας και τις δύο θέσεις σημείων τόξου και την μετρούμενη τιμή διαμέτρου (2 × ακτίνα). Όλες οι ιδιότητες ταξιδεύουν χωρίς απώλεια.
