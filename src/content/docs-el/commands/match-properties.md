# Match Properties

Η εντολή `MatchProperties` αντιγράφει **οπτικές ιδιότητες και ιδιότητες επιπέδου** από μια πηγαία οντότητα σε μία ή περισσότερες οντότητες στόχους. Μεταφέρονται μόνο ιδιότητες που είναι κοινές μεταξύ των τύπων πηγής και στόχου — η γεωμετρία δεν αλλάζει ποτέ.

## Τρόπος ενεργοποίησης

Κάντε κλικ στο κουμπί **Match Properties** της γραμμής εργαλείων (εικονίδιο ρολού βαφής) στον πίνακα Style, ή πληκτρολογήστε `MatchProperties` στο τερματικό.

## Ροή εργασίας

**Πρώτα ενεργοποίηση, μετά επιλογή πηγής:**

1. Πληκτρολογήστε `MatchProperties` ή κάντε κλικ στο κουμπί της γραμμής εργαλείων χωρίς τίποτα προεπιλεγμένο.
2. **Κάντε κλικ στην πηγαία οντότητα** — αυτή της οποίας τις ιδιότητες θέλετε να αντιγράψετε.
3. **Κάντε κλικ σε κάθε οντότητα στόχο** για εφαρμογή των ιδιοτήτων πηγής. Μπορείτε να κάνετε κλικ σε πολλές οντότητες μία-μία.
4. Για εφαρμογή σε ομάδα ταυτόχρονα, **σύρτε ένα πλαίσιο επιλογής** πάνω από τους στόχους.
5. Πατήστε **Enter** ή **Escape** για να τελειώσετε.

**Προεπιλογή πηγής, μετά ενεργοποίηση:**

1. Κάντε κλικ σε μια μεμονωμένη οντότητα για να την επιλέξετε.
2. Ενεργοποιήστε `MatchProperties`. Η επιλεγμένη οντότητα χρησιμοποιείται αυτόματα ως πηγή.
3. Κάντε κλικ σε οντότητες στόχους ή επιλέξτε με σύρσιμο, μετά **Enter** ή **Escape** για να τελειώσετε.

## Ποιες ιδιότητες αντιγράφονται

Η MatchProperties αντιγράφει ιδιότητες που ανήκουν σε κοινή βασική κλάση μεταξύ πηγής και στόχου. Τουλάχιστον, **όλοι οι τύποι οντοτήτων** μοιράζονται αυτές τις ιδιότητες:

| Ιδιότητα | Περιγραφή |
|----------|-----------|
| **Color** | Ο δείκτης χρώματος της οντότητας (περιλαμβάνει "By Layer" / "By Block") |
| **Layer** | Το επίπεδο στο οποίο ανήκει η οντότητα |

Όταν η πηγή και ο στόχος είναι ίδιου τύπου (π.χ. και οι δύο είναι διαστάσεις), αντιγράφονται επίσης επιπλέον ιδιότητες ειδικές για τον τύπο — για παράδειγμα ύψος κειμένου, μέγεθος βέλους, ρυθμίσεις γραμμής επέκτασης.

Η γεωμετρία (συντεταγμένες, ακτίνα, μήκος κ.λπ.) δεν επηρεάζεται ποτέ.

## Αναφορά πληκτρολογίου

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `Enter` / `Space` | Επιβεβαίωση επιλογής περιοχής ή ολοκλήρωση εντολής |
| `Escape` | Ολοκλήρωση εφαρμογής (αν η πηγή έχει οριστεί) ή ακύρωση |

## Λεπτομέρειες συμπεριφοράς

- Η πηγαία οντότητα δεν τροποποιείται ποτέ.
- Κάθε κλικ ή επιλογή με σύρσιμο εφαρμόζει αμέσως τις ιδιότητες πηγής — δεν υπάρχει βήμα επιβεβαίωσης.
- Η επιλογή περιοχής ακολουθεί τους τυπικούς κανόνες: σύρσιμο **δεξιά** για αυστηρή επιλογή (πλήρως εντός), σύρσιμο **αριστερά** για διασταυρούμενη επιλογή (οποιαδήποτε τομή).
- Το κλικ στην πηγαία οντότητα ως στόχος αγνοείται.
- Για οντότητες με κείμενο (**Text**, **Διαστάσεις**, **Multileader**), αντιγράφεται μόνο το ύψος κειμένου — η γραμματοσειρά, τα έντονα, τα πλάγια και άλλες ρυθμίσεις στυλ κειμένου δεν αντιστοιχίζονται.
