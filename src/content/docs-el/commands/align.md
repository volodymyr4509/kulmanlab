# Align

Η εντολή `align` επανατοποθετεί τις επιλεγμένες οντότητες χρησιμοποιώντας ένα ή δύο ζεύγη σημείων πηγής/προορισμού. Με ένα ζεύγος, συμπεριφέρεται ακριβώς όπως η [Move](../move/) (μόνο μετατόπιση). Με δύο ζεύγη, περιστρέφει επίσης την επιλογή ώστε η κατεύθυνση πηγής-προς-πηγή να ταιριάζει με την κατεύθυνση προορισμού-προς-προορισμό, και μπορεί προαιρετικά να την κλιμακώσει ώστε το μήκος του τμήματος πηγής να ταιριάζει με αυτό του τμήματος προορισμού — μετατόπιση, περιστροφή και κλιμάκωση σε μία λειτουργία.

## Δύο τρόποι εκκίνησης

**Προεπιλογή, μετά ευθυγράμμιση** — επιλέξτε πρώτα οντότητες, μετά ενεργοποιήστε:

1. Επιλέξτε μία ή περισσότερες οντότητες στον καμβά.
2. Πληκτρολογήστε `align` στο τερματικό ή κάντε κλικ στο κουμπί **Align** της γραμμής εργαλείων.
3. **Κάντε κλικ στο πρώτο σημείο πηγής (S1)**, μετά **στο πρώτο σημείο προορισμού (D1)**.
4. **Κάντε κλικ στο δεύτερο σημείο πηγής (S2)**, ή πατήστε **Enter** για να εφαρμόσετε αμέσως μια ευθυγράμμιση μόνο μετατόπισης.
5. **Κάντε κλικ στο δεύτερο σημείο προορισμού (D2)**.
6. Απαντήστε στο ερώτημα κλιμάκωσης: πατήστε **Y** για κλιμάκωση, ή **N** / **Enter** για διατήρηση του αρχικού μεγέθους.

**Ενεργοποίηση, μετά επιλογή** — ξεκινήστε εντολή χωρίς τίποτα επιλεγμένο:

1. Πληκτρολογήστε `align` ή κάντε κλικ στο κουμπί της γραμμής εργαλείων.
2. **Επιλέξτε αντικείμενα** — κλικ για εναλλαγή μεμονωμένων οντοτήτων, ή σύρσιμο για επιλογή κατά περιοχή.
3. Πατήστε **Enter** ή **Space** για επιβεβαίωση επιλογής.
4. Συνεχίστε με S1 → D1 → S2 → D2 → ερώτημα κλιμάκωσης όπως παραπάνω.

> Το τερματικό χρειάζεται μόνο όσα γράμματα αρκούν για να μην υπάρχει ασάφεια — πληκτρολογώντας `al` και πατώντας **Enter** ενεργοποιείται απευθείας η Align, καθώς καμία άλλη εντολή δεν ξεκινά με αυτά τα δύο γράμματα.

## Ανατομία μιας ευθυγράμμισης

```
  Σημεία πηγής (στις οντότητες):       Σημεία προορισμού:
  ● S1                                 ● D1
   \                                    \
    ● S2                                 ● D2

  Αποτέλεσμα: η επιλογή μετατοπίζεται ώστε το S1 να προσγειωθεί
  στο D1, μετά περιστρέφεται γύρω από το D1 ώστε η κατεύθυνση
  S1→S2 να ταιριάζει με την κατεύθυνση D1→D2 — και, αν επιλέξετε
  κλιμάκωση, αλλάζει μέγεθος ώστε το |S1S2| να γίνει |D1D2|.
```

Μια φαντομ προεπισκόπηση ακολουθεί τον κέρσορα σε κάθε βήμα: μια προεπισκόπηση μετακίνησης κατά την τοποθέτηση του D1, μετά μια περιστρεφόμενη (διακεκομμένη) προεπισκόπηση μόλις τοποθετείται το D2.

## Ευθυγράμμιση ενός σημείου (μόνο μετατόπιση)

Αφού τοποθετηθεί το D1, πατήστε **Enter** αντί να κάνετε κλικ σε δεύτερο σημείο πηγής. Η επιλογή μετατοπίζεται κατά το διάνυσμα S1→D1 — χωρίς περιστροφή ή κλιμάκωση — ταυτόσημη με μια [Move](../move/) που χρησιμοποιεί το S1 ως βασικό σημείο και το D1 ως προορισμό.

## Ευθυγράμμιση δύο σημείων (μετατόπιση + περιστροφή + προαιρετική κλιμάκωση)

Μόλις τοποθετηθούν και τα S2 και D2:

- **Γωνία περιστροφής** — η διαφορά μεταξύ της κατεύθυνσης προορισμού (`D1 → D2`) και της κατεύθυνσης πηγής (`S1 → S2`).
- **Ερώτημα κλιμάκωσης** — εμφανίζεται το `scale objects to alignment points? [Yes/No] <N>`, με προεπιλογή το **No**:
  - Πατήστε **Y** για να κλιμακώσετε επίσης την επιλογή ομοιόμορφα γύρω από το D1 ώστε η απόσταση `S1–S2` να γίνει η απόσταση `D1–D2`.
  - Πατήστε **N** ή **Enter** για διατήρηση του αρχικού μεγέθους — εφαρμόζονται μόνο μετατόπιση και περιστροφή.

Πατώντας οποιοδήποτε πλήκτρο στο ερώτημα κλιμάκωσης εφαρμόζεται αμέσως η ευθυγράμμιση — δεν υπάρχει ξεχωριστό βήμα επιβεβαίωσης μετά την επιλογή Ναι ή Όχι.

## Αναφορά πληκτρολογίου

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `Enter` / `Space` | Επιβεβαίωση επιλογής και πρόοδος στη φάση S1 |
| `Enter` (στο βήμα S2) | Παράλειψη περιστροφής — εφαρμογή ευθυγράμμισης μόνο μετατόπισης με χρήση S1 και D1 |
| `Y` | Εφαρμογή ευθυγράμμισης με κλιμάκωση |
| `N` / `Enter` (στο ερώτημα κλιμάκωσης) | Εφαρμογή ευθυγράμμισης χωρίς κλιμάκωση |
| `Escape` | Κατά την τοποθέτηση σημείων: απορρίπτει τα σημεία και επιστρέφει στη φάση επιλογής· χωρίς καμία επιλογή: ακυρώνει την εντολή |

## Επιλογή κατά τη διάρκεια της εντολής

Όταν η εντολή ξεκινά στη φάση επιλογής:

| Μέθοδος | Συμπεριφορά |
|---------|------------|
| **Κλικ** | Εναλλαγή οντότητας κάτω από τον κέρσορα μέσα/έξω από επιλογή |
| **Σύρσιμο δεξιά** (αυστηρό) | Προσθέτει οντότητες πλήρως εντός του πλαισίου |
| **Σύρσιμο αριστερά** (διασταυρούμενο) | Προσθέτει οντότητες που τέμνουν το όριο πλαισίου |
| **Enter** / **Space** | Επιβεβαιώνει επιλογή και μεταβαίνει στη φάση S1 |

## Μετά την ευθυγράμμιση

Οι ευθυγραμμισμένες οντότητες παραμένουν επιλεγμένες στη νέα τους θέση, και η εντολή τερματίζεται αυτόματα — εκτελέστε ξανά την **Align**, ή αλλάξτε σε [Move](../move/), [Rotate](../rotate/), ή [Scale](../scale/) χωρίς επανεπιλογή.

## Align έναντι Move

| | Align | Move |
|---|-------|------|
| Ζεύγη σημείων | 1 (μόνο μετατόπιση) ή 2 (μετατόπιση + περιστροφή + κλιμάκωση) | 1 (μόνο μετατόπιση) |
| Περιστροφή | Ναι, με δεύτερο ζεύγος σημείων | Όχι |
| Κλιμάκωση | Προαιρετική, με δεύτερο ζεύγος σημείων | Όχι |
| Καλύτερο για | Προσαρμογή ενός σχήματος σε άλλο χρησιμοποιώντας σημεία αναφοράς | Απλή αλλαγή θέσης |

## Υποστηριζόμενες οντότητες

Η Align λειτουργεί σε κάθε τύπο οντότητας που υποστηρίζεται από τις Move, Rotate και Scale — οι ίδιες λειτουργίες `translate`, `rotate` και `scale` που χρησιμοποιούν αυτές οι εντολές εφαρμόζονται διαδοχικά, οπότε καμία δεν αποκλείεται.
