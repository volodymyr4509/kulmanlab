# Zoom Out

Η εντολή `zoomout` διαιρεί το τρέχον επίπεδο ζουμ κατά **1.5×** (ισοδύναμο με πολλαπλασιασμό επί ~0.667) και εξέρχεται αμέσως, κεντρωμένη στο μέσο σημείο viewport. Είναι το αντίστροφο του [Zoom In](../zoom-in/).

## Μείωση ζουμ προβολής

Κάντε κλικ στο κουμπί **Zoom Out** της γραμμής εργαλείων ή πληκτρολογήστε `zoomout` στο τερματικό. Το ζουμ εφαρμόζεται αστραπιαία και η εντολή εξέρχεται — δεν απαιτείται κλικ στον καμβά.

## Πώς λειτουργεί το βήμα 1.5×

| Τρέχον ζουμ | Μετά από ένα Zoom Out |
|------------|----------------------|
| 1.50× | 1.00× |
| 2.25× | 1.50× |
| 10.00× | 6.67× |
| 0.015× | 0.01× (κατώτατο όριο) |

Το επίπεδο ζουμ εμφανίζεται πάντα στη **γωνία κάτω-δεξιά** του καμβά. Το κατώτατο όριο είναι **0.01×**· περαιτέρω βήματα δεν κάνουν τίποτα.

## Zoom Out εργαλείου vs ρόδα κύλισης

| | Κουμπί Zoom Out | Ρόδα κύλισης |
|---|----------------|-------------|
| Κέντρο ζουμ | Μέσο σημείο viewport | Θέση κέρσορα |
| Μέγεθος βήματος | 1.5× ανά κλικ | ~1.1× ανά tick |
| Απαιτεί ενεργοποίηση | Όχι | Όχι — λειτουργεί πάντα |
| Καλύτερο για | Βήματα πίσω για περισσότερο context | Ομαλό, ζουμ έξω με άγκυρα κέρσορα |

## Αναφορά πληκτρολογίου

Δεν υπάρχουν συντομεύσεις πληκτρολογίου για αυτή την εντολή. Χρησιμοποιήστε τη ρόδα κύλισης αντί — λειτουργεί οποτεδήποτε χωρίς ενεργοποίηση εντολής.
