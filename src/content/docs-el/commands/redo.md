# Redo

Η εντολή `redo` προχωρά μπροστά στο ιστορικό αναίρεσης, επανεφαρμόζοντας ενέργειες που αναιρέθηκαν από το [Undo](../undo/). Το Redo είναι διαθέσιμο μόνο όταν έχετε κάνει βήματα πίσω με Undo και δεν έχετε ακόμα κάνει νέα αλλαγή.

## Πώς να κάνετε redo

- Πληκτρολογήστε `redo` στο τερματικό, ή
- Κάντε κλικ στο κουμπί **Redo** της γραμμής εργαλείων.

Κάθε κλήση επανεφαρμόζει μία προηγουμένως αναιρεθείσα ενέργεια. Καλέστε την επανειλημμένα για να προχωρήσετε μέσα σε όλες τις διαθέσιμες εγγραφές redo.

## Συμπεριφορά στοίβας Redo

| Λεπτομέρεια | Συμπεριφορά |
|------------|------------|
| Διαθέσιμο μετά | Ένα ή περισσότερα βήματα [Undo](../undo/) |
| Διαγράφεται από | **Οποιαδήποτε νέα ενέργεια σχεδίασης** — προσθήκη, επεξεργασία ή διαγραφή οντότητας |
| Αποθήκευση | Browser, ανά αρχείο — επιβιώνει επαναφόρτωσης σελίδας (εφόσον δεν έγινε νέα ενέργεια πριν την επαναφόρτωση) |
| Μέγιστο βάθος | Έως 20 εγγραφές (ίδιο pool με Undo) |

Μόλις σχεδιαστεί, διαγραφεί ή τροποποιηθεί μια νέα οντότητα, η στοίβα redo διαγράφεται και αυτές οι εγγραφές δεν μπορούν να ανακτηθούν. Μόνο αναιρεθείσες ενέργειες που δεν έχουν αντικατασταθεί από νέα εργασία μπορούν να επαναληφθούν.

## Redo vs Undo

| | Redo | Undo |
|---|------|------|
| Κατεύθυνση | Βήματα **μπροστά** μέσα σε αναιρεθείσες εγγραφές | Βήματα **πίσω** στο ιστορικό |
| Διαθέσιμο όταν | Μετά από τουλάχιστον ένα Undo, χωρίς νέα ενέργεια | Υπάρχει τουλάχιστον μία καταγεγραμμένη ενέργεια |
| Διαγράφεται από | Οποιαδήποτε νέα ενέργεια σχεδίασης | Τίποτα |

Το κουμπί Redo της γραμμής εργαλείων είναι γκρι όταν δεν υπάρχουν εγγραφές για redo. Χρησιμοποιήστε πρώτα [Undo](../undo/) για να δημιουργήσετε εγγραφές redo.
