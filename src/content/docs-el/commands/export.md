# Export

Η εντολή **Export** κατεβάζει το τρέχον σχέδιο στο σύστημα αρχείων σας. Διατίθενται δύο μορφές: **DXF** για συμβατότητα με άλλα εργαλεία CAD και **JSON** για πλήρεις αποθηκεύσεις εντός KulmanLab CAD.

## Πώς να εξάγετε

1. Κάντε κλικ στο κουμπί **Export** της γραμμής εργαλείων (εικονίδιο λήψης) στο πάνελ File.
2. Ανοίγει το αναδυόμενο παράθυρο **Export Manager**.
3. Κάντε κλικ σε κάρτα μορφής για επιλογή της μορφής — **JSON** ή **DXF**.
4. Κάντε κλικ στο κουμπί **Export**. Το αρχείο κατεβαίνει αυτόματα στον προεπιλεγμένο φάκελο λήψεων σας.

## Επιλογή μορφής

| Μορφή | Επέκταση | Καλύτερο για | Περιορισμοί |
|-------|----------|------------|-------------|
| **JSON** *(native)* | `.json` | Αποθήκευση εργασίας για επαναφόρτωση στο KulmanLab CAD | Δεν είναι συμβατό με άλλα εργαλεία CAD |
| **DXF** | `.dxf` | Κοινή χρήση με FreeCAD, LibreCAD κ.λπ. | Διαστάσεις και leaders δεν εξάγονται |

**Πότε να χρησιμοποιείτε JSON:** κάθε φορά που θέλετε να αποθηκεύσετε πλήρες αντίγραφο της εργασίας σας. Το JSON είναι η native μορφή του KulmanLab και διατηρεί κάθε οντότητα ακριβώς — συμπεριλαμβανομένων διαστάσεων, leaders και όλων των δεδομένων επιπέδου.

**Πότε να χρησιμοποιείτε DXF:** όταν χρειάζεται να παραδώσετε το σχέδιο σε κάποιον που χρησιμοποιεί άλλη εφαρμογή CAD. Το εξαγόμενο αρχείο χρησιμοποιεί μορφή DXF AC1012 και μπορεί να ανοιχτεί στα περισσότερα εργαλεία συμβατά με DXF.

## Τι εξάγεται ανά μορφή

### Εξαγωγή JSON

Περιλαμβάνονται όλοι οι τύποι οντοτήτων:

- Lines, circles, arcs, ellipses, polylines, splines, text
- Διαστάσεις (linear, aligned, continued, radius, diameter)
- Multileaders
- Ορισμοί επιπέδων και πίνακες linetype

### Εξαγωγή DXF

Περιλαμβάνονται μόνο οντότητες γεωμετρίας:

- Lines, circles, arcs, ellipses, polylines (εξάγονται ως `LWPOLYLINE`), splines, text
- Ορισμοί επιπέδων και πίνακες linetype

**Δεν περιλαμβάνονται σε DXF:** οντότητες διαστάσεων και multileaders. Αυτά χρησιμοποιούν δομές δεδομένων ειδικές για KulmanLab που δεν μπορούν να αναπαρασταθούν πιστά σε τυπικό DXF. Αν το σχέδιό σας έχει σημάνσεις, χρησιμοποιήστε JSON ή [Print](../print/) για οπτική καταγραφή τους.

## Όνομα εξαγόμενου αρχείου

Το κατεβασμένο αρχείο ονομάζεται από το τρέχον αρχείο σχεδίου (π.χ. `myplan_May22_14:30:00.json`). Η επέκταση αλλάζει ώστε να αντιστοιχεί στην επιλεγμένη μορφή.

## Διαφορά μεταξύ Export και Print

| Χαρακτηριστικό | Export | Print |
|----------------|--------|-------|
| Έξοδος | Αρχείο πηγής διανύσματος (.dxf / .json) | Εικόνα raster (.png / .jpeg / .webp / .pdf) |
| Επεξεργάσιμο σε άλλα εργαλεία | Ναι (DXF) | Όχι |
| Διατηρεί επίπεδα & linetypes | Ναι | Όχι (αποδίδεται επίπεδο) |
| Καταγράφει διαστάσεις & leaders | Μόνο JSON | Ναι |

Χρησιμοποιήστε **Export** όταν χρειάζεστε επεξεργάσιμο αρχείο. Χρησιμοποιήστε [Print](../print/) όταν χρειάζεστε οπτικό στιγμιότυπο.
