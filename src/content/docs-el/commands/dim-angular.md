# Dimension Angular

Η εντολή `DimensionAngular` τοποθετεί **γωνιακή διάσταση** τόξου σήμανσης στο σχέδιο. Μετράει και επισημαίνει τη γωνία μεταξύ δύο γραμμών, το εύρος ενός τόξου ή έναν τομέα ενός κύκλου.

## Τρόπος ενεργοποίησης

Κάντε κλικ στο κουμπί **Dimension Angular** της γραμμής εργαλείων στο πάνελ Markup, ή πληκτρολογήστε `DimensionAngular` στο τερματικό.

## Τρεις λειτουργίες εισόδου

Το πρώτο κλικ καθορίζει ποια λειτουργία χρησιμοποιείται:

### Δύο γραμμές

1. **Κάντε κλικ στην πρώτη γραμμή.** Η θέση κέρσορα καθορίζει ποια πλευρά της γραμμής χρησιμοποιείται.
2. **Κάντε κλικ στη δεύτερη γραμμή.** Οι δύο γραμμές πρέπει να τέμνονται (η τομή υπολογίζεται αυτόματα· δεν χρειάζεται να είναι ορατή στην οθόνη).
3. **Κάντε κλικ για τοποθέτηση** του τόξου διάστασης. Μετακινήστε τον κέρσορα για επιλογή της ακτίνας και ποιος γωνιακός τομέας επισημαίνεται — η σήμανση ακολουθεί τον κέρσορα σε όποια πλευρά της κορυφής βρίσκεστε.

Παράλληλες γραμμές δεν μπορούν να σχηματίσουν γωνιακή διάσταση· η εντολή αγνοεί το δεύτερο κλικ αν οι γραμμές δεν τέμνονται.

### Τόξο

1. **Κάντε κλικ σε τόξο.** Η διάσταση δημιουργείται αμέσως από τη γωνία έναρξης του τόξου μέχρι τη γωνία λήξης, χρησιμοποιώντας το κέντρο του τόξου ως κορυφή.
2. **Κάντε κλικ για τοποθέτηση** του τόξου διάστασης στην επιθυμητή ακτίνα.

### Κύκλος

1. **Κάντε κλικ σε κύκλο.** Το πρώτο άκρο γωνίας κάνει snap στο πλησιέστερο σημείο στον κύκλο.
2. **Κάντε κλικ σε δεύτερο σημείο** στον κύκλο για ορισμό του δεύτερου άκρου γωνίας.
3. **Κάντε κλικ για τοποθέτηση** του τόξου διάστασης.

## Αναφορά πληκτρολογίου

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `Escape` | Ακύρωση και επιστροφή στην πρώτη επιλογή |

## Λεπτομέρειες συμπεριφοράς

- Το τόξο διάστασης σχεδιάζεται πάντα στην πλευρά της κορυφής όπου το τοποθετείτε — μετακινήστε τον κέρσορα πέρα από την κορυφή για αντιστροφή στη συμπληρωματική γωνία.
- Η μετρούμενη γωνία εμφανίζεται σε μοίρες και ενημερώνεται ζωντανά καθώς μετακινείτε τον κέρσορα κατά την τοποθέτηση.
- Η προκύπτουσα σήμανση είναι πλήρης οντότητα `DimensionAngular` αποθηκευμένη στο τρέχον επίπεδο. Οι ιδιότητες εμφάνισης (μέγεθος βέλους, ύψος κειμένου, μήκος γραμμής επέκτασης) μπορούν να ρυθμιστούν στο πάνελ Ιδιοτήτων.
- Οι γωνιακές διαστάσεις περιλαμβάνονται στην εξαγωγή JSON αλλά δεν υποστηρίζονται από τον εξαγωγέα DXF.

## Επεξεργασία ετικέτας — simple mode

**Διπλό κλικ** σε τοποθετημένη γωνιακή διάσταση για άνοιγμα του επεξεργαστή κειμένου σε **simple** mode. Ο επεξεργαστής είναι προ-συμπληρωμένος με την τρέχουσα αποδοσμένη τιμή ώστε να μπορείτε να τοποθετήσετε τον κέρσορα και να την επεξεργαστείτε απευθείας.

| Λειτουργία | Συμπεριφορά |
|-----------|------------|
| Bold / Italic / Font / Height | Εφαρμόζεται σε **ολόκληρη** την ετικέτα ταυτόχρονα |
| Μορφοποίηση ανά χαρακτήρα | Δεν υποστηρίζεται |
| `Enter` | Επικυρώνει την τιμή και κλείνει τον επεξεργαστή |
| Πολυγραμμικό | Δεν υποστηρίζεται |

Δείτε [Text Editor — simple mode](../interface/text-editor#simple-mode) για την πλήρη αναφορά.
