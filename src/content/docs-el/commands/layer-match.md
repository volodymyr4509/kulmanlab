# LayerMatch

Η εντολή `LayerMatch` αναθέτει εκ νέου το επίπεδο επιλεγμένων οντοτήτων ώστε να ταιριάζει με το επίπεδο μιας πηγαίας οντότητας που κλικάρατε. Είναι ο ταχύτερος τρόπος μεταφοράς ομάδας αντικειμένων στο σωστό επίπεδο χωρίς άνοιγμα του Layer Manager.

## Ροή εργασίας

**Προεπιλογή, μετά ταίριασμα**:

1. Επιλέξτε τις οντότητες των οποίων το επίπεδο θέλετε να αλλάξετε.
2. Πληκτρολογήστε `LayerMatch` ή κάντε κλικ στο κουμπί **Layer Match** της γραμμής εργαλείων (εικονίδιο βαφής).
3. **Κάντε κλικ στην πηγαία οντότητα** — αυτή της οποίας το επίπεδο θέλετε να αντιγράψετε.
4. Όλες οι επιλεγμένες οντότητες μετακινούνται στο επίπεδο της πηγαίας οντότητας αμέσως.

**Ενεργοποίηση, μετά επιλογή**:

1. Πληκτρολογήστε `LayerMatch` ή κάντε κλικ στο κουμπί γραμμής εργαλείων χωρίς τίποτα επιλεγμένο.
2. **Επιλέξτε αντικείμενα-στόχους** — κλικ για εναλλαγή μεμονωμένων οντοτήτων ή σύρσιμο για επιλογή ανά περιοχή.
3. Πατήστε **Enter** ή **Space** για επιβεβαίωση της επιλογής.
4. **Κάντε κλικ στην πηγαία οντότητα** — το επίπεδό της εφαρμόζεται σε όλους τους στόχους.

## Αναφορά πληκτρολογίου

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `Enter` / `Space` | Επιβεβαίωση επιλογής στόχων και προχώρηση στη φάση επιλογής πηγής |
| `Escape` | Επαναφορά — επιστροφή στην επιλογή στόχου ή πλήρης ακύρωση |

## Λεπτομέρειες συμπεριφοράς

- Αλλάζει μόνο η ιδιότητα `layer` — χρώμα, linetype, lineweight και γεωμετρία παραμένουν αναλλοίωτα.
- Η ίδια η πηγαία οντότητα δεν τροποποιείται.
- Η εντολή τερματίζεται αφού κλικαριστεί η πηγή.
- Κάνοντας κλικ σε κενό καμβά κατά τη φάση επιλογής πηγής δεν έχει αποτέλεσμα.
