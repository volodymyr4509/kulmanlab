# Mirror

Η εντολή `mirror` δημιουργεί αντικατοπτρισμένα αντίγραφα επιλεγμένων οντοτήτων που αντανακλώνται ως προς έναν άξονα δύο σημείων. Τα πρωτότυπα **διατηρούνται πάντα** — σε αντίθεση με [Move](./move) ή [Rotate](./rotate), η Mirror δεν τροποποιεί ποτέ υπάρχουσες οντότητες· μόνο προσθέτει νέες.

## Δύο τρόποι εκκίνησης

**Προεπιλογή, μετά αντικατοπτρισμός** — επιλέξτε πρώτα οντότητες, μετά ενεργοποιήστε:

1. Επιλέξτε μία ή περισσότερες οντότητες στον καμβά.
2. Πληκτρολογήστε `mirror` στο τερματικό ή κάντε κλικ στο κουμπί **Mirror** της γραμμής εργαλείων.
3. **Κάντε κλικ στο πρώτο σημείο** του άξονα αντικατοπτρισμού, ή πληκτρολογήστε `X,Y` και πατήστε **Enter** για ακριβή συντεταγμένη.
4. **Κάντε κλικ στο δεύτερο σημείο** — τα αντικατοπτρισμένα αντίγραφα τοποθετούνται και η εντολή εξέρχεται. Η εισαγωγή συντεταγμένης λειτουργεί και εδώ.

**Ενεργοποίηση, μετά επιλογή** — ξεκινήστε εντολή χωρίς τίποτα επιλεγμένο:

1. Πληκτρολογήστε `mirror` ή κάντε κλικ στο κουμπί της γραμμής εργαλείων.
2. **Επιλέξτε αντικείμενα** — κλικ για εναλλαγή, ή σύρσιμο για επιλογή κατά περιοχή.
3. Πατήστε **Enter** ή **Space** για επιβεβαίωση επιλογής.
4. **Κάντε κλικ στο πρώτο σημείο**, μετά **κλικ στο δεύτερο σημείο** του άξονα αντικατοπτρισμού (εισαγωγή συντεταγμένης διαθέσιμη και στα δύο βήματα).

```
  Πρωτότυπο:       Άξονας:         Αποτέλεσμα:
                   |
  [οντότητα A] →  |    →    [οντότητα A] + [αντικατοπτρισμένο A]
                   |
```

Μια ζωντανή προεπισκόπηση των αντικατοπτρισμένων αντιγράφων ακολουθεί τον κέρσορα ενώ τοποθετείτε το δεύτερο σημείο άξονα.

## Άξονας αντικατοπτρισμού

Ο άξονας είναι άπειρη γραμμή μέσα από τα δύο σημεία κλικ. Μπορεί να έχει οποιαδήποτε γωνία:

- Μετακινήστε τον κέρσορα κοντά σε **άξονα ευθυγράμμισης 45°** (0°, 45°, 90°, 135°, …) και ο άξονας κλειδώνει σε αυτή τη γωνία — χρήσιμο για καθαρούς οριζόντιους, κατακόρυφους ή διαγώνιους αντικατοπτρισμούς.
- Κάντε κλικ έξω από τη ζώνη ευθυγράμμισης για άξονα ελεύθερης γωνίας.

## Εισαγωγή συντεταγμένης

Σε οποιοδήποτε βήμα σημείου άξονα, πληκτρολογήστε ακριβή θέση αντί για κλικ:

1. Πληκτρολογήστε την τιμή X.
2. Πατήστε `,` — το τερματικό εμφανίζει `[X], [Y{cursor}]`.
3. Πληκτρολογήστε την τιμή Y.
4. Πατήστε **Enter** για επιβεβαίωση.

## Αναφορά πληκτρολογίου

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `Enter` / `Space` | Επιβεβαίωση επιλογής |
| `0`–`9`, `.`, `-` | Έναρξη εισαγωγής συντεταγμένης X |
| `,` | Κλείδωμα X και μετάβαση στην εισαγωγή Y |
| `Backspace` | Διαγραφή τελευταίου πληκτρολογημένου χαρακτήρα |
| `Enter` | Επιβεβαίωση πληκτρολογημένης συντεταγμένης |
| `Escape` | Ακύρωση και επαναφορά |

## Επιλογή κατά τη διάρκεια της εντολής

| Μέθοδος | Συμπεριφορά |
|---------|------------|
| **Κλικ** | Εναλλαγή οντότητας κάτω από τον κέρσορα |
| **Σύρσιμο δεξιά** (αυστηρό) | Προσθέτει οντότητες πλήρως εντός του πλαισίου |
| **Σύρσιμο αριστερά** (διασταυρούμενο) | Προσθέτει οντότητες που τέμνουν το πλαίσιο |
| **Enter** / **Space** | Επιβεβαίωση επιλογής |

## Τι αντικατοπτρίζεται

Κάθε τύπος οντότητας υποστηρίζεται. Η γεωμετρία αντανακλάται ως προς τον άξονα μαθηματικά:

| Οντότητα | Τι αλλάζει |
|----------|-----------|
| Line | Και τα δύο τελικά σημεία αντανακλώνται |
| Circle | Το κέντρο αντανακλάται· η ακτίνα παραμένει αμετάβλητη |
| Arc | Το κέντρο αντανακλάται· οι γωνίες αρχής και τέλους επανυπολογίζονται ως προς τον άξονα |
| Ellipse | Το κέντρο αντανακλάται· η κατεύθυνση κύριου άξονα αναστρέφεται ως προς τον άξονα |
| Polyline / Rectangle | Κάθε κορυφή αντανακλάται |
| Text | Το σημείο άγκυρας αντανακλάται· η συμβολοσειρά κειμένου **δεν** αντιστρέφεται |
| Spline | Όλες οι κορυφές ελέγχου / σημεία fit αντανακλώνται |

## Mirror vs Copy

| | Mirror | Copy |
|---|--------|------|
| Πρωτότυπα | Διατηρούνται πάντα | Διατηρούνται πάντα |
| Θέση νέας οντότητας | Αντικατοπτρισμένη ως προς άξονα | Μετατοπισμένη κατά διάνυσμα |
| Καλύτερο για | Συμμετρικά σχέδια, διμερή χαρακτηριστικά | Επανάληψη γεωμετρίας σε οποιαδήποτε κατεύθυνση |
