# Leader+

Η εντολή `leader+` προσθέτει νέο βραχίονα βέλους σε υπάρχουσα γραμμή οδήγησης (multileader). Ο νέος βραχίονας δείχνει από το υπάρχον dogleg της γραμμής οδήγησης σε νέα άκρη βέλους που κλικάρετε. Όλο το στυλ — θέση dogleg, κείμενο, τύπος βέλους και μέγεθος — κληρονομείται από την επιλεγμένη γραμμή οδήγησης.

## Προσθήκη βραχίονα

1. Πληκτρολογήστε `leader+` στο τερματικό.
2. **Κάντε κλικ σε υπάρχουσα γραμμή οδήγησης** για επιλογή της.
3. **Κάντε κλικ στη νέα άκρη βέλους**, ή πληκτρολογήστε `X,Y` και πατήστε **Enter** για ακριβή συντεταγμένη. Μια γραμμή προεπισκόπησης δείχνει από τον κέρσορα στο dogleg της γραμμής οδήγησης.

Ο βραχίονας τοποθετείται και η εντολή παραμένει ενεργή — μπορείτε αμέσως να κλικάρετε άλλη γραμμή οδήγησης για προσθήκη περισσότερων βραχιόνων. Πατήστε **Enter**, **Space** ή **Escape** για τερματισμό.

```
  Before:                        After:
  ◄── arm 1                      ◄── arm 1
       \                               \
        ●──── dogleg ──── text          ●──── dogleg ──── text
                                       /
                                  arm 2 ──►  (new tip you clicked)
```

## Εισαγωγή συντεταγμένης για την άκρη

Αντί για κλικ, μπορείτε να πληκτρολογήσετε ακριβή θέση:

1. Πληκτρολογήστε την τιμή X.
2. Πατήστε `,` — το τερματικό επιβεβαιώνει ότι το X είναι κλειδωμένο.
3. Πληκτρολογήστε την τιμή Y.
4. Πατήστε **Enter** για τοποθέτηση.

## Αναφορά πληκτρολογίου

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `0`–`9`, `.`, `-` | Αρχίστε να πληκτρολογείτε συντεταγμένη X |
| `,` | Κλείδωμα X και μετάβαση στην εισαγωγή Y |
| `Enter` | Επιβεβαίωση πληκτρολογημένης συντεταγμένης και τοποθέτηση βραχίονα |
| `Enter` / `Space` | Τερματισμός (όταν δεν υπάρχει είσοδος σε εξέλιξη) |
| `Escape` | Ακύρωση και επαναφορά |

## Σημειώσεις

- Μόνο οντότητες **Multileader** μπορούν να επιλεγούν — το κλικ σε οποιονδήποτε άλλο τύπο οντότητας δεν έχει αποτέλεσμα.
- Ο νέος βραχίονας ξεκινά από το υπάρχον dogleg· επιλέγετε μόνο την κατεύθυνση της άκρης βέλους.
- Δεν υπάρχει όριο στον αριθμό βραχιόνων που μπορεί να έχει μια γραμμή οδήγησης.
