# LayerMakeCurrent

Η εντολή `LayerMakeCurrent` ορίζει το **τρέχον επίπεδο σχεδίου** στο επίπεδο στο οποίο ανήκει η οντότητα που κλικάρατε. Νέες οντότητες θα σχεδιάζονται τότε αυτόματα σε εκείνο το επίπεδο.

## Χρήση

1. Πληκτρολογήστε `LayerMakeCurrent` στο τερματικό ή κάντε κλικ στο κουμπί **Make Current** της γραμμής εργαλείων (εικονίδιο eyedropper).
2. **Κάντε κλικ σε οποιαδήποτε οντότητα** στον καμβά.
3. Το τρέχον επίπεδο ενημερώνεται ώστε να ταιριάζει με το επίπεδο εκείνης της οντότητας. Η εντολή τερματίζεται αμέσως.

## Λεπτομέρειες συμπεριφοράς

- Αν κάνετε κλικ σε κενό καμβά (χωρίς εύρεση οντότητας), το τερματικό εμφανίζει `no object found` και η εντολή παραμένει ενεργή ώστε να μπορείτε να ξαναδοκιμάσετε.
- Αλλάζει μόνο η ρύθμιση τρέχοντος επιπέδου — δεν τροποποιείται καμία οντότητα.
- Το ενημερωμένο επίπεδο αντικατοπτρίζεται στον επιλογέα επιπέδου στη γραμμή εργαλείων.
