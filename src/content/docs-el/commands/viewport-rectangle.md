# ViewportRectangle

Η εντολή `ViewportRectangle` δημιουργεί νέο viewport στο ενεργό layout χαρτιού επιλέγοντας δύο απέναντι γωνίες. Διαθέσιμη μόνο στον χώρο layout.

## Δημιουργία viewport

1. Μεταβείτε σε layout χαρτιού χρησιμοποιώντας την καρτέλα στο κάτω μέρος της οθόνης.
2. Πληκτρολογήστε `ViewportRectangle` στο τερματικό ή κάντε κλικ στο κουμπί **Viewport Rectangle** της γραμμής εργαλείων.
3. **Κάντε κλικ στην πρώτη γωνία**, ή πληκτρολογήστε `X,Y` και πατήστε **Enter** για ακριβή συντεταγμένη.
4. **Κάντε κλικ στην απέναντι γωνία** — το viewport τοποθετείται αμέσως. Η εισαγωγή συντεταγμένης λειτουργεί και εδώ.

Το νέο viewport εμφανίζει ολόκληρο το μοντέλο στην προεπιλεγμένη κλίμακα του layout. Χρησιμοποιήστε τη ρόδα κύλισης μέσα στο viewport για ζουμ, ή σύρσιμο μέσου κουμπιού για μετακίνηση της προβολής μοντέλου.

## Εισαγωγή συντεταγμένης

Σε οποιοδήποτε βήμα γωνίας μπορείτε να πληκτρολογήσετε ακριβή συντεταγμένη:

1. Πληκτρολογήστε την τιμή X.
2. Πατήστε `,` — το τερματικό εμφανίζει `[X], [Y{cursor}]`.
3. Πληκτρολογήστε την τιμή Y.
4. Πατήστε **Enter** για τοποθέτηση σημείου.

## Αναφορά πληκτρολογίου

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `0`–`9`, `.`, `-` | Έναρξη εισαγωγής συντεταγμένης X |
| `,` | Κλείδωμα X και μετάβαση στην εισαγωγή Y |
| `Enter` | Επιβεβαίωση πληκτρολογημένης συντεταγμένης |
| `Escape` | Ακύρωση |

## Επεξεργασία viewport

Μετά την τοποθέτηση viewport, κάντε κλικ για επιλογή:

- **Σύρτε τα άκρα ή τις γωνίες** για αλλαγή μεγέθους.
- **Σύρτε το grip κέντρου** για μετακίνηση.
- Χρησιμοποιήστε τον **επιλογέα κλίμακας** στη γραμμή ελέγχου για ορισμό ακριβούς κλίμακας (π.χ. 1:50). Για εισαγωγή κλίμακας που δεν βρίσκεται στη λίστα, πληκτρολογήστε την απευθείας στο πεδίο εισαγωγής στο κάτω μέρος του αναπτυσσόμενου — αποδέχεται μορφή αναλογίας (`1:200`, `5:1`) ή απλό δεκαδικό (`0.005`), μετά πατήστε **Enter**.
- Κάντε δεξί κλικ σε viewport και χρησιμοποιήστε **Lock** για αποτροπή τυχαίων αλλαγών.

## Σημειώσεις

- Η ViewportRectangle είναι διαθέσιμη μόνο όταν είναι ενεργή καρτέλα layout χαρτιού. Εκτέλεση στον χώρο μοντέλου εμφανίζει μήνυμα σφάλματος και εξέρχεται.
- Για αντιγραφή υπάρχοντος viewport, χρησιμοποιήστε [ViewportCopy](../viewport-copy/).
