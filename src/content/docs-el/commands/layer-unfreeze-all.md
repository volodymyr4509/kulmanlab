# LayerUnfreezeAll

Η εντολή `LayerUnfreezeAll` αφαιρεί τη σημαία παγώματος από **κάθε επίπεδο** στο σχέδιο άμεσα. Δεν απαιτείται επιλογή ή επιβεβαίωση — εκτελείται και τελειώνει με ένα βήμα.

## Χρήση

Πληκτρολογήστε `LayerUnfreezeAll` στο τερματικό ή κάντε κλικ στο κουμπί **Unfreeze All** της γραμμής εργαλείων (εικονίδιο ήλιου). Όλα τα παγωμένα επίπεδα γίνονται ορατά αμέσως.

## Πότε να χρησιμοποιείτε

Χρησιμοποιείται συνήθως μετά από [LayerIsolate](../layer-isolate/) για επαναφορά όλων των επιπέδων στην κανονική ορατή τους κατάσταση.

## Λεπτομέρειες συμπεριφοράς

- Εφαρμόζεται σε όλα τα επίπεδα ανεξάρτητα από την τρέχουσα κατάστασή τους.
- Δεν επηρεάζει σημαίες κλειδώματος ή εκτύπωσης — αλλάζει μόνο η σημαία παγώματος.
- Η εντολή τερματίζεται αμέσως χωρίς προτροπές.
