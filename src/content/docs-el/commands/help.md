# Help

Ανοίγει το πάνελ βοήθειας εντός εφαρμογής με σύνοψη όλων των συντομεύσεων πληκτρολογίου, συμβουλές πλοήγησης και γενική χρήση.

## Πώς να χρησιμοποιήσετε

Κάντε κλικ στο κουμπί **Help** της γραμμής εργαλείων ή πληκτρολογήστε `help` στο τερματικό. Το πάνελ βοήθειας ανοίγει αμέσως και η εντολή τερματίζεται.

## Τι περιέχει το πάνελ βοήθειας

### Πλοήγηση
| Συντόμευση | Ενέργεια |
|------------|----------|
| `Scroll wheel` | Μεγέθυνση / σμίκρυνση |
| `Middle click + drag` | Μετακίνηση προβολής |
| `Double middle click` | Προσαρμογή όλων οντοτήτων στην οθόνη |

### Εντολές
| Συντόμευση | Ενέργεια |
|------------|----------|
| Πληκτρολόγηση | Αρχίστε να πληκτρολογείτε όνομα εντολής — προτάσεις εμφανίζονται στον καμβά |
| `Delete` | Διαγραφή όλων επιλεγμένων οντοτήτων |
| `Enter` / `Space` | Επιβεβαίωση εντολής ή εισόδου |
| `Enter` / `Space` (αδρανές) | Επανάληψη τελευταίας χρησιμοποιημένης εντολής |
| `Escape` | Ακύρωση τρέχουσας εντολής ή εκκαθάριση επιλογής |

### Επιλογή
| Συντόμευση | Ενέργεια |
|------------|----------|
| `Left click` | Επιλογή οντότητας |
| `Drag →` | Επιλογή μόνο οντοτήτων **πλήρως μέσα** στην περιοχή σύρσιμου |
| `Drag ←` | Επιλογή κάθε οντότητας που **αγγίζει** την περιοχή σύρσιμου |
| `Left click grip` | Άμεση επεξεργασία κορυφής ή τελικού σημείου |

### Συμβουλές σχεδίασης
- **Snap points** — ο κέρσορας κάνει snap στα κοντινά τελικά σημεία οντοτήτων αυτόματα.
- **Sticky angle** — μετακινήστε τον κέρσορα αρκετά μακριά σε μία κατεύθυνση για κλείδωμα σε τμήματα 45°.
- **Ακριβές μήκος** — ενώ η γωνία είναι κλειδωμένη, πληκτρολογήστε αριθμό και πατήστε `Enter` για ορισμό ακριβούς απόστασης.

## Έκδοση

Η τρέχουσα έκδοση εφαρμογής εμφανίζεται στην επάνω αριστερή γωνία του πάνελ βοήθειας (π.χ. `KULMAN 2026.04.15`).
