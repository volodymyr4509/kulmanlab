# Polyline

Η εντολή `polyline` σχεδιάζει συνδεδεμένο μονοπάτι οποιουδήποτε πλήθους ευθύγραμμων τμημάτων, αποθηκευμένα ως μία οντότητα `LWPOLYLINE`. Επειδή το σύνολο του μονοπατιού είναι ένα αντικείμενο, η επιλογή του επιλέγει κάθε τμήμα ταυτόχρονα — μεταφέρετε, περιστρέψτε ή κλιμακώστε ολόκληρο το σχήμα σε μια λειτουργία. Αυτή είναι η βασική διαφορά από αλυσιδωμένες [Lines](../line/), όπου κάθε τμήμα είναι ανεξάρτητη οντότητα.

Τα Polylines μπορούν επίσης να **κλείσουν**: η εντολή [Rectangle](../rectangle/) χρησιμοποιεί την ίδια οντότητα `LWPOLYLINE` με ενεργοποιημένη σημαία κλεισίματος.

## Σχεδίαση polyline

1. Πληκτρολογήστε `polyline` στο τερματικό ή κάντε κλικ στο κουμπί **Polyline** της γραμμής εργαλείων.
2. **Κάντε κλικ στο πρώτο σημείο**, ή πληκτρολογήστε `X,Y` και πατήστε **Enter** για ακριβή συντεταγμένη.
3. **Κάντε κλικ σε κάθε επόμενο σημείο** — κάθε κλικ προσθέτει ένα τμήμα. Η εισαγωγή συντεταγμένης λειτουργεί σε κάθε βήμα.
4. Πατήστε **Enter** ή **Space** για ολοκλήρωση (απαιτούνται τουλάχιστον 2 σημεία).

```
  ●──────●
  1ο     2ο
          \
           \  τμήμα 3 (σε εξέλιξη — κέρσορας εδώ)
            ●  ← κλικ για προσθήκη, Enter/Space για ολοκλήρωση
```

Πατώντας **Escape** οποτεδήποτε απορρίπτονται όλα τα τοποθετημένα σημεία και η εντολή εξέρχεται.

## Εισαγωγή συντεταγμένης

Αντί για κλικ, πληκτρολογήστε ακριβή θέση για οποιαδήποτε κορυφή:

1. Πληκτρολογήστε την τιμή X.
2. Πατήστε `,` — το τερματικό εμφανίζει `[X], [Y{cursor}]`.
3. Πληκτρολογήστε την τιμή Y.
4. Πατήστε **Enter** για τοποθέτηση της κορυφής.

## Κλείδωμα γωνίας και ακριβές μήκος τμήματος

Η ίδια λογική ευθυγράμμισης 45° με την εντολή [Line](../line/#κλείδωμα-γωνίας-και-εισαγωγή-ακριβούς-μήκους) εφαρμόζεται μεταξύ οποιωνδήποτε δύο διαδοχικών σημείων. Όταν κλειδωθεί σε άξονα:

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `0`–`9`, `.` | Προσθήκη ψηφίου στο μήκος τμήματος |
| `-` | Αρνητικό μήκος — αντιστρέφει κατεύθυνση κατά μήκος του άξονα (μόνο πρώτος χαρακτήρας) |
| `Backspace` | Διαγραφή τελευταίου πληκτρολογημένου χαρακτήρα |
| `Enter` | Τοποθέτηση επόμενου σημείου στην πληκτρολογημένη απόσταση |

Το τρέχον συσσωρευμένο μήκος εμφανίζεται στη γραμμή τερματικού σε πραγματικό χρόνο. Το κλικ ενώ είναι κλειδωμένο προβάλλεται στον άξονα ώστε η νέα κορυφή να βρίσκεται ακριβώς πάνω σε αυτόν.

## Αναφορά πληκτρολογίου

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `0`–`9`, `.`, `-` | Έναρξη εισαγωγής συντεταγμένης X, ή μήκος τμήματος όταν είναι κλειδωμένη η γωνία |
| `,` | Κλείδωμα X και μετάβαση στην εισαγωγή Y |
| `Backspace` | Διαγραφή τελευταίου πληκτρολογημένου χαρακτήρα |
| `Enter` | Επιβεβαίωση πληκτρολογημένης συντεταγμένης ή μήκους, ή ολοκλήρωση polyline αν δεν υπάρχει εισαγωγή και ≥ 2 σημεία υπάρχουν |
| `Space` | Ολοκλήρωση polyline (ίδιο με Enter όταν δεν υπάρχει εισαγωγή σε εξέλιξη) |
| `Escape` | Απόρριψη όλων των σημείων και έξοδος |

## Επεξεργασία grip — κορυφές και μέσα τμημάτων

Ένα επιλεγμένο polyline εμφανίζει δύο τύπους grips:

| Grip | Θέση | Τι κάνει |
|------|------|----------|
| **Κορυφή** | Σε κάθε τοποθετημένο σημείο | Σύρσιμο για επανατοποθέτηση αυτής της κορυφής· όλα τα συνδεδεμένα τμήματα ακολουθούν |
| **Μέσο τμήματος** | Κέντρο κάθε τμήματος | Σύρσιμο για μετατόπιση **και** των δύο τελικών σημείων αυτού του τμήματος μαζί, διατηρώντας το μήκος και τη γωνία του τμήματος |

Το grip μέσου τμήματος είναι μοναδικό για polylines — επιτρέπει ολίσθηση μεμονωμένου τμήματος πλάγια χωρίς αλλαγή του μήκους του. Σε [Line](../line/), το grip μέσου σημείου αντίθετα ενεργοποιεί την εντολή Move για ολόκληρη την οντότητα.

Δεν υπάρχει grip "μεταφορά ολόκληρου polyline". Για μετακίνηση ολόκληρου του μονοπατιού χρησιμοποιήστε την εντολή [Move](../move/).

## Επιλογή polylines

| Μέθοδος | Συμπεριφορά |
|---------|------------|
| **Κλικ** | Επιλέγει το polyline αν το κλικ βρίσκεται εντός hit-test απόστασης από οποιοδήποτε τμήμα |
| **Σύρσιμο δεξιά** (αυστηρό) | Όλες οι κορυφές πρέπει να βρίσκονται εντός του πλαισίου |
| **Σύρσιμο αριστερά** (διασταυρούμενο) | Οποιοδήποτε τμήμα που διασταυρώνει το όριο πλαισίου επιλέγει ολόκληρο το polyline |

Επειδή ένα polyline είναι μία οντότητα, μια διασταυρούμενη επιλογή που αγγίζει οποιοδήποτε τμήμα επιλέγει όλα τα τμήματα.

## Υποστηριζόμενες εντολές επεξεργασίας

Τα Polylines υποστηρίζουν όλους τους γενικούς μετασχηματισμούς και offset, αλλά **όχι** trim ή extend (αυτά είναι μόνο για [Line](../line/)):

| Εντολή | Τι συμβαίνει στο polyline |
|--------|--------------------------|
| [Move](../move/) | Μεταφράζει όλες τις κορυφές κατά την ίδια μετατόπιση |
| [Copy](../copy/) | Δημιουργεί πανομοιότυπο polyline σε νέα θέση |
| [Rotate](../rotate/) | Περιστρέφει όλες τις κορυφές γύρω από το επιλεγμένο βασικό σημείο |
| [Mirror](../mirror/) | Αντικατοπτρίζει όλες τις κορυφές ως προς τον άξονα αντικατοπτρισμού |
| [Scale](../scale/) | Κλιμακώνει όλες τις κορυφές ομοιόμορφα από το βασικό σημείο |
| [Offset](../offset/) | Δημιουργεί παράλληλο polyline σε σταθερή κάθετη απόσταση |
| [Delete](../delete/) | Αφαιρεί το polyline από το σχέδιο |

## Ιδιότητες

Όταν ένα polyline είναι επιλεγμένο, το πάνελ ιδιοτήτων εμφανίζει:

**Γενικά**

| Ιδιότητα | Προεπιλογή | Σημασία |
|----------|-----------|---------|
| Color | 256 (ByLayer) | Δείκτης χρώματος ACI |
| Layer | `0` | Ανάθεση επιπέδου |
| Linetype | ByLayer | Μοτίβο linetype ονόματος |
| Linetype Scale | 1 | Συντελεστής κλίμακας στο μοτίβο linetype |
| Thickness | 0 | Πάχος εξώθησης |

**Γεωμετρία**

| Ιδιότητα | Σημασία |
|----------|---------|
| Closed | Αν η τελευταία κορυφή συνδέεται πίσω στην πρώτη |
| Vertex Count | Συνολικός αριθμός κορυφών |
| Vertices | Λίστα συντεταγμένων όλων των κορυφών |

## Polyline vs Line — πότε να χρησιμοποιείτε το καθένα

| | Polyline | Line |
|---|---------|------|
| Πλήθος οντοτήτων | Ένα `LWPOLYLINE` για ολόκληρο το μονοπάτι | Ένα `LINE` ανά τμήμα |
| Κλειστό σχήμα | Ναι (σημαία κλεισίματος) | Όχι |
| Trim / Extend | Όχι | Ναι — τμήμα-τμήμα |
| Grip μέσου τμήματος | Μεταφράζει ολόκληρο το τμήμα | Ενεργοποιεί Move για την οντότητα |
| Καλύτερο για | Περιγράμματα, σχήματα που διατηρούνται ακέραια | Κατασκευαστικές γραμμές, γεωμετρία που θα κοπεί |

## DXF — οντότητα LWPOLYLINE

Τα Polylines αποθηκεύονται ως οντότητες `LWPOLYLINE` στο αρχείο DXF. Όλες οι ιδιότητες — συντεταγμένες κορυφών, σημαία κλεισίματος, χρώμα, επίπεδο, linetype, κλίμακα linetype και πάχος — ταξιδεύουν χωρίς απώλεια. Τα Rectangles που σχεδιάζονται με την εντολή [Rectangle](../rectangle/) αποθηκεύονται επίσης ως `LWPOLYLINE` (κλειστό, τέσσερις κορυφές) και είναι αδιάκριτα σε επίπεδο DXF.

Οι οντότητες `LWPOLYLINE` από οποιαδήποτε εφαρμογή συμβατή με DXF (LibreCAD, FreeCAD κ.λπ.) διαβάζονται ξανά ως πλήρως επεξεργάσιμα polylines στον επεξεργαστή.
