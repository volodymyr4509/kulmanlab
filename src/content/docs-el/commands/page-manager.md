# PageManager

Η εντολή `PageManager` ανοίγει το παράθυρο ρυθμίσεων σελίδας για το ενεργό layout, επιτρέποντάς σας να αλλάξετε τη μορφή χαρτιού, τον προσανατολισμό και την κλίμακα μεταξύ μονάδων σχεδίασης και χιλιοστών.

## Άνοιγμα του Page Manager

- Πληκτρολογήστε `PageManager` στο τερματικό, **ή**
- Κάντε δεξί κλικ σε καρτέλα layout στο κάτω μέρος και επιλέξτε **Page Manager**.

## Ρυθμίσεις

| Ρύθμιση | Περιγραφή |
|---------|-----------|
| **Μορφή χαρτιού** | Τυπικό μέγεθος φύλλου (A4, A3, A2, Letter κ.λπ.) |
| **Προσανατολισμός** | Κατακόρυφος ή οριζόντιος |
| **Κλίμακα** | Μονάδες σχεδίου ανά mm — ελέγχει πώς οι συντεταγμένες μοντέλου αντιστοιχούν στο χαρτί |

Η αλλαγή μεγέθους χαρτιού ή κλίμακας αλλάζει αναλογικά τα υπάρχοντα viewports στο layout ώστε οι θέσεις τους να παραμένουν σωστές.

## Σημειώσεις

- Η PageManager εφαρμόζεται μόνο σε layouts χαρτιού. Δεν είναι διαθέσιμη στον χώρο μοντέλου.
- Για προσθήκη νέου layout, κάντε κλικ στο κουμπί **+** στη γραμμή καρτελών στο κάτω μέρος.
- Για μετονομασία ή διαγραφή layout, κάντε δεξί κλικ στην καρτέλα του.
