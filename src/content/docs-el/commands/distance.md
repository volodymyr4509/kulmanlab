# Distance

Η εντολή `distance` μετράει την ευθύγραμμη (Ευκλείδεια) απόσταση μεταξύ δύο κλικαρισμένων σημείων και εκτυπώνει το αποτέλεσμα στο τερματικό με 4 δεκαδικά ψηφία. Είναι μία από τις δύο εντολές μέτρησης — το [Angle](../angle/) μετράει το γωνιακό άνοιγμα σε κορυφή αντί.

## Ανατομία μέτρησης απόστασης

```
  ● first point
   \
    \  preview line (live)
     \
      ● second point    →  terminal: "Distance: 12.3456"
```

- **Πρώτο σημείο** — αφετηρία της μέτρησης.
- **Δεύτερο σημείο** — τελικό σημείο· η τοποθέτησή του εκτυπώνει αμέσως το αποτέλεσμα.
- **Αποτέλεσμα** — εμφανίζεται στο τερματικό, δεν τοποθετείται στον καμβά.

## Μέτρηση απόστασης

1. Πληκτρολογήστε `distance` στο τερματικό ή κάντε κλικ στο κουμπί **Distance** της γραμμής εργαλείων.
2. **Κάντε κλικ στο πρώτο σημείο**, ή πληκτρολογήστε `X,Y` και πατήστε **Enter** για ακριβή συντεταγμένη.
3. **Κάντε κλικ στο δεύτερο σημείο** — η μετρούμενη απόσταση εμφανίζεται στο τερματικό. Η εισαγωγή συντεταγμένης λειτουργεί και εδώ.
4. **Κάντε κλικ ξανά** (προαιρετικό) για εκκίνηση νέας μέτρησης. Η εντολή παραμένει ενεργή.

Πατήστε `Escape` ανά πάσα στιγμή για επαναφορά στο βήμα 2.

## Αλυσίδα μετρήσεων

Αφού εμφανιστεί το αποτέλεσμα, κάνοντας κλικ ξεκινά αμέσως η επόμενη μέτρηση — το κλικαρισμένο σημείο γίνεται το νέο πρώτο σημείο. Αυτό σας επιτρέπει να μετράτε ακολουθία αποστάσεων χωρίς επανενεργοποίηση της εντολής.

## Distance vs Angle

| | Distance | Angle |
|---|---------|-------|
| Τι μετράει | Ευθύγραμμο μήκος | Εσωτερική γωνία σε κορυφή |
| Αριθμός κλικ | 2 | 3 |
| Μορφή αποτελέσματος | `12.3456` (μονάδες) | `45.0000°` |
| Προεπισκόπηση καμβά | Γραμμή από πρώτο σημείο σε κέρσορα | Δύο γραμμές από κορυφή σε κέρσορα |
| Καλύτερο για | Μήκος κενού ή τμήματος | Γωνιακό άνοιγμα μεταξύ δύο χαρακτηριστικών |

## Εισαγωγή συντεταγμένης

Αντί για κλικ, πληκτρολογήστε ακριβή θέση για οποιοδήποτε σημείο:

1. Πληκτρολογήστε την τιμή X.
2. Πατήστε `,` — το τερματικό εμφανίζει `[X], [Y{cursor}]`.
3. Πληκτρολογήστε την τιμή Y.
4. Πατήστε **Enter** για επιβεβαίωση.

## Αναφορά πληκτρολογίου

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `0`–`9`, `.`, `-` | Έναρξη εισαγωγής συντεταγμένης X |
| `,` | Κλείδωμα X και μετάβαση στην εισαγωγή Y |
| `Backspace` | Διαγραφή τελευταίου πληκτρολογημένου χαρακτήρα |
| `Enter` | Επιβεβαίωση πληκτρολογημένης συντεταγμένης |
| `Escape` | Ακύρωση και επαναφορά στο βήμα 2 |

## Σημειώσεις

- Τα αποτελέσματα εμφανίζονται **μόνο στο τερματικό** — τίποτα δεν προστίθεται στο σχέδιο.
- Το αποτέλεσμα εκφράζεται στις ίδιες μονάδες με τις συντεταγμένες σχεδίου (χωρίς μετατροπή μονάδων).
- Η ακρίβεια είναι πάντα 4 δεκαδικά ψηφία.
