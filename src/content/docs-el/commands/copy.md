# Copy

Η εντολή `copy` δημιουργεί μεταφρασμένα αντίγραφα επιλεγμένων οντοτήτων και τα τοποθετεί μετατοπισμένα από βασικό σημείο σε προορισμό — τα πρωτότυπα παραμένουν ακριβώς εκεί που είναι. Αυτή είναι η κύρια διαφορά από το [Move](../move/): το Copy προσθέτει νέες οντότητες στο σχέδιο· το Move μετακινεί υπάρχουσες.

## Δύο τρόποι εκκίνησης

**Προεπιλογή, μετά αντιγραφή** — επιλέξτε οντότητες πρώτα, μετά ενεργοποιήστε:

1. Επιλέξτε μία ή περισσότερες οντότητες στον καμβά.
2. Πληκτρολογήστε `copy` στο τερματικό ή κάντε κλικ στο κουμπί **Copy** της γραμμής εργαλείων.
3. **Κάντε κλικ στο βασικό σημείο**, ή πληκτρολογήστε `X,Y` και πατήστε **Enter** για ακριβή συντεταγμένη.
4. **Κάντε κλικ στον προορισμό** — τα αντίγραφα εμφανίζονται με τη μετατόπιση βασικό→προορισμός. Η εισαγωγή συντεταγμένης λειτουργεί και εδώ.

**Ενεργοποίηση, μετά επιλογή** — ξεκινήστε την εντολή χωρίς τίποτα επιλεγμένο:

1. Πληκτρολογήστε `copy` ή κάντε κλικ στο κουμπί γραμμής εργαλείων.
2. **Επιλέξτε αντικείμενα** — κλικ για εναλλαγή μεμονωμένων οντοτήτων, ή σύρσιμο για επιλογή ανά περιοχή.
3. Πατήστε **Enter** ή **Space** για επιβεβαίωση της επιλογής.
4. **Κάντε κλικ στο βασικό σημείο**, μετά **κλικ στον προορισμό** (η εισαγωγή συντεταγμένης είναι διαθέσιμη και στα δύο βήματα).

```
  Before:               After:
  [entity A]            [entity A]  ← originals untouched
  [entity B]            [entity B]
                        [copy of A] ← new entities
                        [copy of B]
```

Μια φαντoματική προεπισκόπηση των αντιγράφων ακολουθεί τον κέρσορα από το βασικό σημείο στον προορισμό.

## Εισαγωγή συντεταγμένης

Στο βήμα βασικού σημείου ή προορισμού, πληκτρολογήστε ακριβή θέση αντί να κάνετε κλικ:

1. Πληκτρολογήστε την τιμή X.
2. Πατήστε `,` — το τερματικό εμφανίζει `[X], [Y{cursor}]`.
3. Πληκτρολογήστε την τιμή Y.
4. Πατήστε **Enter** για επιβεβαίωση.

## Κλείδωμα γωνίας και ακριβής απόσταση

Αφού οριστεί το βασικό σημείο, η εντολή κάνει snap σε άξονες 45° (0°, 45°, 90°, 135°, …) όταν ο κέρσορας είναι αρκετά μακριά και κοντά στον άξονα. Ενώ είναι κλειδωμένο, πληκτρολογήστε απόσταση και πατήστε **Enter** για τοποθέτηση των αντιγράφων ακριβώς σε εκείνη τη μετατόπιση.

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `0`–`9`, `.` | Προσθέτει στην τιμή απόστασης |
| `-` | Αρνητική απόσταση — αντιστρέφει κατεύθυνση κατά μήκος του άξονα (μόνο πρώτος χαρακτήρας) |
| `Backspace` | Διαγράφει τον τελευταίο πληκτρολογημένο χαρακτήρα |
| `Enter` | Τοποθετεί αντίγραφα στην πληκτρολογημένη απόσταση |

## Αναφορά πληκτρολογίου

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `Enter` / `Space` | Επιβεβαίωση επιλογής και προχώρηση στη φάση βασικού σημείου |
| `0`–`9`, `.`, `-` | Έναρξη εισαγωγής συντεταγμένης X, ή απόσταση όταν η γωνία είναι κλειδωμένη |
| `,` | Κλείδωμα X και μετάβαση στην εισαγωγή Y |
| `Backspace` | Διαγραφή τελευταίου πληκτρολογημένου χαρακτήρα |
| `Enter` | Επιβεβαίωση συντεταγμένης ή εφαρμογή αντιγραφής στην πληκτρολογημένη απόσταση |
| `Escape` | Ακύρωση και επαναφορά |

## Επιλογή κατά τη διάρκεια της εντολής

| Μέθοδος | Συμπεριφορά |
|---------|------------|
| **Κλικ** | Εναλλάσσει την οντότητα κάτω από τον κέρσορα εντός/εκτός επιλογής |
| **Σύρσιμο δεξιά** (αυστηρό) | Προσθέτει οντότητες πλήρως μέσα στο πλαίσιο |
| **Σύρσιμο αριστερά** (διασταυρούμενο) | Προσθέτει οντότητες που τέμνουν το όριο του πλαισίου |
| **Enter** / **Space** | Επιβεβαιώνει την επιλογή |

## Μετά την αντιγραφή

Τα **πρωτότυπα παραμένουν επιλεγμένα** — τα νέα αντίγραφα προστίθενται στο σχέδιο αλλά η επιλογή διαγράφεται και η εντολή τερματίζεται. Για άμεση εργασία με τα αντίγραφα, εκτελέστε ξανά Copy στην επιλογή ή ξεκινήστε νέα εντολή.

## Copy vs Move

| | Copy | Move |
|---|------|------|
| Πρωτότυπα | Παραμένουν στη θέση τους | Αφαιρούνται από την αρχική θέση |
| Πλήθος αποτελεσμάτων | Αυξάνεται κατά τον αριθμό αντιγραμμένων οντοτήτων | Αμετάβλητο |
| Μετά την λειτουργία | Πρωτότυπα ακόμα επιλεγμένα | Μετακινημένες οντότητες επιλεγμένες σε νέα θέση |
| Καλύτερο για | Επανάληψη γεωμετρίας, συμμετρικές διατάξεις | Αναθέση γεωμετρίας |

## Υποστηριζόμενες οντότητες

Το Copy λειτουργεί σε κάθε τύπο οντότητας. Όλες οι οντότητες υλοποιούν εσωτερικά `translate(dx, dy)` οπότε καμία δεν εξαιρείται.
