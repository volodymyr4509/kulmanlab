# Scale

Η εντολή `scale` αλλάζει ομοιόμορφα το μέγεθος επιλεγμένων οντοτήτων γύρω από ένα βασικό σημείο. Όλες οι αποστάσεις από το βασικό σημείο πολλαπλασιάζονται με τον συντελεστή κλίμακας — συντελεστής `2` διπλασιάζει όλες τις διαστάσεις, `0.5` τις μισεύει. Ο συντελεστής εισάγεται πάντα πληκτρολογώντας· δεν υπάρχει κλικ-για-ορισμό-κλίμακας.

## Δύο τρόποι εκκίνησης

**Προεπιλογή, μετά κλίμακα** — επιλέξτε πρώτα οντότητες, μετά ενεργοποιήστε:

1. Επιλέξτε μία ή περισσότερες οντότητες στον καμβά.
2. Πληκτρολογήστε `scale` στο τερματικό ή κάντε κλικ στο κουμπί **Scale** της γραμμής εργαλείων.
3. **Κάντε κλικ στο βασικό σημείο** — το σταθερό σημείο που δεν μετακινείται κατά την κλίμακα. Ή πληκτρολογήστε `X,Y` και πατήστε **Enter** για ακριβή συντεταγμένη.
4. **Πληκτρολογήστε τον συντελεστή κλίμακας** και πατήστε **Enter**.

**Ενεργοποίηση, μετά επιλογή** — ξεκινήστε εντολή χωρίς τίποτα επιλεγμένο:

1. Πληκτρολογήστε `scale` ή κάντε κλικ στο κουμπί της γραμμής εργαλείων.
2. **Επιλέξτε αντικείμενα** — κλικ για εναλλαγή, ή σύρσιμο για επιλογή κατά περιοχή.
3. Πατήστε **Enter** ή **Space** για επιβεβαίωση επιλογής.
4. **Κάντε κλικ στο βασικό σημείο** (εισαγωγή συντεταγμένης διαθέσιμη), μετά πληκτρολογήστε τον συντελεστή.

```
  Βάση ●                Βάση ●
        [οντότητα]  →          [μεγαλύτερη οντότητα]
  συντελεστής = 2 → αποστάσεις από ● διπλασιάζονται
```

## Πληκτρολόγηση συντελεστή κλίμακας

Αφού τοποθετηθεί το βασικό σημείο, το τερματικό εμφανίζει `enter scale factor:` και αναμένει εισαγωγή από πληκτρολόγιο:

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `0`–`9`, `.` | Προσθήκη ψηφίου στον συντελεστή |
| `-` | Αρνητικός συντελεστής (μόνο πρώτος χαρακτήρας — αντιστρέφει μετά κλιμακώνει) |
| `Backspace` | Διαγραφή τελευταίου πληκτρολογημένου χαρακτήρα |
| `Enter` | Εφαρμογή κλίμακας στον πληκτρολογημένο συντελεστή |

Ο συντελεστής πρέπει να είναι μη-μηδενικός. Συνηθισμένες τιμές:

| Συντελεστής | Αποτέλεσμα |
|------------|-----------|
| `2` | Διπλασιάζει όλες τις διαστάσεις |
| `0.5` | Μισεύει όλες τις διαστάσεις |
| `1.5` | Αύξηση κατά 50% |
| `-1` | Αντικατοπτρισμός ως προς το βασικό σημείο (ίσο με περιστροφή 180°) |

Δεν υπάρχει ζωντανή προεπισκόπηση κατά την πληκτρολόγηση — το αποτέλεσμα κλίμακας εμφανίζεται μόνο αφού πατήσετε **Enter**.

## Αναφορά πληκτρολογίου

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `Enter` / `Space` | Επιβεβαίωση επιλογής |
| `0`–`9`, `.`, `-` | Έναρξη εισαγωγής συντεταγμένης X (φάση βασικού σημείου), ή συντελεστής κλίμακας (φάση συντελεστή) |
| `,` | Κλείδωμα X και μετάβαση στην εισαγωγή Y (φάση βασικού σημείου) |
| `Backspace` | Διαγραφή τελευταίου πληκτρολογημένου χαρακτήρα |
| `Enter` | Επιβεβαίωση συντεταγμένης ή εφαρμογή κλίμακας |
| `Escape` | Ακύρωση και επαναφορά |

## Επιλογή κατά τη διάρκεια της εντολής

| Μέθοδος | Συμπεριφορά |
|---------|------------|
| **Κλικ** | Εναλλαγή οντότητας κάτω από τον κέρσορα |
| **Σύρσιμο δεξιά** (αυστηρό) | Προσθέτει οντότητες πλήρως εντός του πλαισίου |
| **Σύρσιμο αριστερά** (διασταυρούμενο) | Προσθέτει οντότητες που τέμνουν το πλαίσιο |
| **Enter** / **Space** | Επιβεβαίωση επιλογής |

## Τι κλιμακώνεται

Όλοι οι τύποι οντοτήτων υποστηρίζονται. Η γεωμετρία κάθε οντότητας κλιμακώνεται σε σχέση με το βασικό σημείο:

| Οντότητα | Τι αλλάζει |
|----------|-----------|
| Line | Και τα δύο τελικά σημεία κλιμακώνονται από το βασικό σημείο |
| Circle | Το κέντρο κλιμακώνεται από το βασικό σημείο· η ακτίνα πολλαπλασιάζεται με τον συντελεστή |
| Arc | Το κέντρο κλιμακώνεται· η ακτίνα πολλαπλασιάζεται με τον συντελεστή· οι γωνίες παραμένουν αμετάβλητες |
| Ellipse | Το κέντρο κλιμακώνεται· και τα δύο μήκη ημιαξόνων πολλαπλασιάζονται με τον συντελεστή |
| Polyline / Rectangle | Κάθε κορυφή κλιμακώνεται από το βασικό σημείο |
| Text | Το σημείο άγκυρας κλιμακώνεται· το ύψος πολλαπλασιάζεται με τον συντελεστή |
| Spline | Όλες οι κορυφές ελέγχου / σημεία fit κλιμακώνονται |
