# Leader

Η εντολή `leader` σχεδιάζει σχόλιο multileader σε τέσσερα βήματα: ένα βέλος που αγγίζει ένα χαρακτηριστικό, μια γραμμή leader που κάμπτεται σε dogleg, μια άγκυρα κειμένου και μια πληκτρολογημένη ετικέτα. Από όλες τις εντολές σχολίων, ο Leader είναι ο μόνος που περιλαμβάνει διαδραστική φάση εισαγωγής κειμένου με προεπισκόπηση αναβοσβήνοντος κέρσορα.

## Ανατομία ενός multileader

```
  ◄── άκρο βέλους  (βήμα 2 — αγγίζει το χαρακτηριστικό)
      \
       \  γραμμή leader
        \
         ●──── dogleg (βήμα 3) ──── άγκυρα κειμένου (βήμα 4)
                                    Κείμενο ετικέτας  (βήμα 5)
```

- **Άκρο βέλους** — το αιχμηρό άκρο τοποθετημένο στο χαρακτηριστικό που σχολιάζεται.
- **Dogleg** — ο αγκώνας όπου η γραμμή leader κάμπτεται προς το κείμενο.
- **Άγκυρα κειμένου** — όπου η ετικέτα τοποθετείται. Το κείμενο στοιχίζεται αριστερά ή δεξιά αυτόματα.

## Σχεδίαση leader

1. Πληκτρολογήστε `leader` στο τερματικό ή κάντε κλικ στο κουμπί **Leader** της γραμμής εργαλείων.
2. **Κάντε κλικ στο άκρο βέλους**, ή πληκτρολογήστε `X,Y` και πατήστε **Enter** για ακριβή συντεταγμένη.
3. **Κάντε κλικ στο dogleg** — η κάμψη στο leader. Η γωνία κλειδώνει σε βήματα 45°· πληκτρολογήστε μήκος και πατήστε **Enter** για ακριβή τοποθέτηση. Ή πληκτρολογήστε `X,Y` για απόλυτη συντεταγμένη.
4. **Κάντε κλικ στη θέση κειμένου** — εκεί που αγκυρώνεται η ετικέτα. Ίδιες επιλογές: κλικ, κλείδωμα γωνίας + μήκος, ή `X,Y`.
5. **Πληκτρολογήστε την ετικέτα κειμένου** — η προεπισκόπηση καμβά ενημερώνεται ζωντανά με αναβοσβήνοντα κέρσορα. Πατήστε **Enter** για τοποθέτηση.

## Εισαγωγή συντεταγμένης (όλες οι φάσεις σημείου)

Σε οποιοδήποτε βήμα επιλογής σημείου (άκρο, dogleg, θέση κειμένου) μπορείτε να πληκτρολογήσετε ακριβή συντεταγμένη αντί για κλικ:

1. Πληκτρολογήστε την τιμή X (ψηφία, `.`, ή `-`).
2. Πατήστε `,` — το τερματικό εμφανίζει `[X], [Y{cursor}]` επιβεβαιώνοντας ότι το X είναι κλειδωμένο.
3. Πληκτρολογήστε την τιμή Y.
4. Πατήστε **Enter** για τοποθέτηση σημείου.

## Κλείδωμα γωνίας (βήματα 3 και 4)

Μετά από κάθε τοποθετημένο σημείο, η εντολή ευθυγραμμίζεται σε άξονες 45° όταν ο κέρσορας είναι αρκετά μακριά. Ενώ είναι κλειδωμένο:
- Η προεπισκόπηση ευθυγραμμίζεται στον άξονα.
- Πληκτρολογήστε μήκος και πατήστε **Enter** για τοποθέτηση του επόμενου σημείου σε αυτή την ακριβή απόσταση.

Το κλείδωμα γωνίας και η εισαγωγή συντεταγμένης είναι αμοιβαία αποκλειόμενα — μόλις πληκτρολογήσετε ψηφίο χωρίς προηγούμενο `,`, η εντολή το ερμηνεύει ως απόσταση (το κλείδωμα γωνίας πρέπει να είναι ενεργό). Για να εισαγάγετε απόλυτη συντεταγμένη, ξεκινήστε με τον αριθμό X ακολουθούμενο από κόμμα.

## Επεξεργασία ετικέτας κειμένου

Κατά την πληκτρολόγηση της ετικέτας στο βήμα 5, μπορείτε να πλοηγηθείτε και να επεξεργαστείτε το κείμενο πριν την τοποθέτηση:

| Πλήκτρο | Ενέργεια |
|---------|----------|
| Οποιοσδήποτε εκτυπώσιμος χαρακτήρας | Εισαγωγή στη θέση κέρσορα |
| `←` / `→` | Μετακίνηση κέρσορα αριστερά ή δεξιά |
| `Backspace` | Διαγραφή χαρακτήρα αριστερά του κέρσορα |
| `Delete` | Διαγραφή χαρακτήρα δεξιά του κέρσορα |
| `Enter` | Τοποθέτηση του leader |

## Αυτόματη κατεύθυνση κειμένου

Η στοίχιση κειμένου προσαρμόζεται στη θέση του κέρσορα σε σχέση με το dogleg:

| Θέση κέρσορα | Κατεύθυνση κειμένου |
|-------------|---------------------|
| **Δεξιά** του dogleg | Από αριστερά προς δεξιά από την άγκυρα κειμένου |
| **Αριστερά** του dogleg | Από δεξιά προς αριστερά (αγκυρωμένο στη δεξιά πλευρά) |

Δεν χρειάζεται χειροκίνητη ρύθμιση — μετακινήστε τον κέρσορα στην πλευρά όπου θέλετε την ετικέτα και στοιχίζεται σωστά.

## Αναφορά πληκτρολογίου

**Φάσεις σημείου (άκρο, dogleg, θέση κειμένου)**

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `0`–`9`, `.`, `-` | Έναρξη πληκτρολόγησης συντεταγμένης X (μετά `,` για κλείδωμα X και εισαγωγή Y) |
| `,` | Επιβεβαίωση X και μετάβαση στην εισαγωγή Y |
| `0`–`9`, `.`, `-` | Συσσώρευση απόστασης όταν είναι κλειδωμένη η γωνία |
| `Backspace` | Διαγραφή τελευταίου πληκτρολογημένου χαρακτήρα |
| `Enter` | Επιβεβαίωση πληκτρολογημένης συντεταγμένης ή απόστασης |

**Φάση εισαγωγής κειμένου**

| Πλήκτρο | Ενέργεια |
|---------|----------|
| Εκτυπώσιμος χαρακτήρας | Εισαγωγή στον κέρσορα |
| `←` / `→` | Μετακίνηση κέρσορα |
| `Backspace` | Διαγραφή αριστερά |
| `Delete` | Διαγραφή δεξιά |
| `Enter` | Τοποθέτηση του leader |

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `Escape` | Ακύρωση και επαναφορά στο βήμα 2 |

## Επεξεργασία υπάρχοντος leader

**Διπλό κλικ** σε τοποθετημένο multileader για επαναφορά του επεξεργαστή κειμένου σε **rich** mode. Σε rich mode μπορείτε να εφαρμόσετε έντονη, πλάγια γραφή και παρακάμψεις γραμματοσειράς ή ύψους ανά χαρακτήρα, και να εισάγετε αλλαγές γραμμής με `Enter`. Πατήστε **Escape** για επικύρωση και κλείσιμο.

Δείτε [Text Editor — rich mode](../interface/text-editor#rich-mode) για την πλήρη αναφορά.

## DXF — μόνο εισαγωγή

**Τα leaders είναι μόνο για εισαγωγή.** Οι οντότητες `MLEADER` από αρχεία DXF διαβάζονται και εμφανίζονται σωστά, αλλά τα leaders που σχεδιάζονται στον επεξεργαστή **δεν γράφονται** κατά την αποθήκευση αρχείου DXF. Χρησιμοποιήστε τα leaders για οπτικά σχόλια· αποφύγετε να βασίζεστε σε αυτά για ροές εργασίας round-trip.
