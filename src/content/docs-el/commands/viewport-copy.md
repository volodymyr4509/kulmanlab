# ViewportCopy

Η εντολή `ViewportCopy` αντιγράφει ένα viewport σε νέα θέση, διατηρώντας την κλίμακα και το κέντρο μοντέλου. Διαθέσιμη μόνο στον χώρο layout.

## Αντιγραφή viewport

1. Μεταβείτε σε καρτέλα layout χαρτιού.
2. Προαιρετικά κάντε κλικ σε viewport για προεπιλογή.
3. Πληκτρολογήστε `ViewportCopy` στο τερματικό ή κάντε κλικ στο κουμπί **Viewport Copy** της γραμμής εργαλείων.
4. Αν δεν υπήρχε προεπιλεγμένο viewport, **κάντε κλικ στο viewport** για αντιγραφή.
5. **Κάντε κλικ στο βασικό σημείο** — η αναφορά για τη μετατόπιση. Ή πληκτρολογήστε `X,Y` και πατήστε **Enter** για ακριβή συντεταγμένη.
6. **Κάντε κλικ στον προορισμό** — το viewport τοποθετείται στη μετατόπιση βάση→προορισμός. Ή χρησιμοποιήστε εισαγωγή συντεταγμένης / κλείδωμα γωνίας.

Μετά την τοποθέτηση, η εντολή παραμένει ενεργή — κάντε κλικ σε άλλο προορισμό για τοποθέτηση άλλου αντιγράφου του ίδιου viewport. Πατήστε **Enter**, **Space** ή **Escape** για να τελειώσετε.

## Εισαγωγή συντεταγμένης

Στα βήματα βασικού σημείου και προορισμού μπορείτε να πληκτρολογήσετε ακριβή συντεταγμένη αντί για κλικ:

1. Πληκτρολογήστε την τιμή X.
2. Πατήστε `,` — το τερματικό εμφανίζει `[X], [Y{cursor}]`.
3. Πληκτρολογήστε την τιμή Y.
4. Πατήστε **Enter** για επιβεβαίωση.

## Κλείδωμα γωνίας και ακριβής απόσταση

Αφού οριστεί το βασικό σημείο, η εντολή ευθυγραμμίζεται σε άξονες 45° (0°, 45°, 90°, 135°, …) όταν ο κέρσορας ευθυγραμμίζεται. Ενώ είναι κλειδωμένο:

- Η προεπισκόπηση ευθυγραμμίζεται στον άξονα.
- Πληκτρολογήστε απόσταση και πατήστε **Enter** για τοποθέτηση αντιγράφου ακριβώς σε αυτή τη μετατόπιση κατά μήκος της κλειδωμένης κατεύθυνσης.

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `0`–`9`, `.` | Προσθήκη ψηφίου στην τιμή απόστασης |
| `-` | Αρνητική απόσταση (αντιστρέφει κατεύθυνση· μόνο πρώτος χαρακτήρας) |
| `Backspace` | Διαγραφή τελευταίου πληκτρολογημένου χαρακτήρα |
| `Enter` | Τοποθέτηση αντιγράφου στην πληκτρολογημένη απόσταση |

## Αναφορά πληκτρολογίου

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `0`–`9`, `.`, `-` | Έναρξη εισαγωγής συντεταγμένης X, ή απόσταση όταν είναι κλειδωμένη η γωνία |
| `,` | Κλείδωμα X και μετάβαση στην εισαγωγή Y |
| `Enter` | Επιβεβαίωση πληκτρολογημένης συντεταγμένης ή απόστασης |
| `Enter` / `Space` | Ολοκλήρωση (όταν δεν υπάρχει εισαγωγή σε εξέλιξη) |
| `Escape` | Ακύρωση και επαναφορά |

## Σημειώσεις

- Η ViewportCopy είναι διαθέσιμη μόνο όταν είναι ενεργή καρτέλα layout χαρτιού.
- Το αντιγραμμένο viewport κληρονομεί την ίδια κλίμακα, κέντρο μοντέλου, κατάσταση κλειδώματος και διαστάσεις με το πρωτότυπο.
- Για δημιουργία νέου viewport από την αρχή, χρησιμοποιήστε [ViewportRectangle](./viewport-rectangle).
