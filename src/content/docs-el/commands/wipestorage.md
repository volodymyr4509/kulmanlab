# Wipe Storage

Η εντολή `wipestorage` διαγράφει μόνιμα **όλα τα δεδομένα αποθηκευμένα στον browser** για το KulmanLab CAD — κάθε αποθηκευμένο αρχείο, πίνακα επιπέδων και linetypes, και ιστορικό αναίρεσης. Η σελίδα φορτώνει ξανά αυτόματα μετά.

:::danger Μη αναστρέψιμο
Αυτή η ενέργεια δεν μπορεί να αναιρεθεί. Όλα τα αρχεία αποθηκευμένα στον browser διαγράφονται. Εξαγάγετε τα σχέδια που θέλετε να κρατήσετε ως αρχεία `.json` ή `.dxf` πριν εκτελέσετε αυτή την εντολή.
:::

## Πότε να τη χρησιμοποιείτε

- Η αποθήκευση browser είναι κατεστραμμένη και η εφαρμογή αποτυγχάνει να φορτώσει ή να αποθηκεύσει αρχεία.
- Θέλετε να επαναφέρετε πλήρως την εφαρμογή σε καθαρή κατάσταση.
- Αλλάζετε browser ή συσκευή και δεν χρειάζεστε πλέον το τοπικό αντίγραφο.

## Πώς να την εκτελέσετε

1. Πληκτρολογήστε `wipestorage` στο τερματικό και πατήστε **Enter**.
2. Το τερματικό ζητά: *Wipe all browser local storage? Type YES to confirm*
3. Πληκτρολογήστε `YES` (οποιοσδήποτε συνδυασμός κεφαλαίων/πεζών) και πατήστε **Enter**.

Η εφαρμογή διαγράφει τη βάση δεδομένων και φορτώνει ξανά τη σελίδα. Αν πληκτρολογήσετε κάτι διαφορετικό από `YES` και πατήσετε **Enter**, ή πατήσετε **Escape**, η εντολή ακυρώνεται και τίποτα δεν διαγράφεται.

## Τι διαγράφεται

| Δεδομένα | Διαγράφεται |
|----------|------------|
| Όλα τα αρχεία αποθηκευμένα στον browser | Ναι |
| Πίνακες επιπέδων και linetypes για κάθε αρχείο | Ναι |
| Ιστορικό Undo / Redo για κάθε αρχείο | Ναι |

Επηρεάζονται μόνο τα δεδομένα αποθηκευμένα τοπικά **σε αυτόν τον browser**. Αρχεία που έχετε ήδη εξαγάγει ως `.json` ή `.dxf` δεν επηρεάζονται.
