# New File

Η εντολή **New File** καθαρίζει τον καμβά και ξεκινά φρέσκο κενό σχέδιο. Δημιουργείται αυτόματα μοναδικό όνομα αρχείου με χρονική σήμανση.

## Πώς να δημιουργήσετε νέο αρχείο

Κάντε κλικ στο κουμπί **New File** της γραμμής εργαλείων (εικονίδιο νέας σελίδας) στον πίνακα File. Ο καμβάς καθαρίζει αμέσως — χωρίς προτροπές ή παράθυρα επιβεβαίωσης.

## Τι περιέχει το νέο αρχείο

Ένα νεοδημιουργημένο αρχείο ξεκινά με:

- **Καμία οντότητα** στον καμβά.
- **Ένα προεπιλεγμένο επίπεδο** με όνομα `0`, χρώμα λευκό και linetype `Continuous`.
- **Δημιουργημένο όνομα αρχείου** στη μορφή `kulman-Mon01_HH:MM:SS.dxf` (π.χ. `kulman-May22_14:30:00.dxf`).

Το αρχείο αποθηκεύεται αυτόματα στη μνήμη του προγράμματος περιήγησης και εμφανίζεται στα [Recent Files](../files/).

## Προειδοποίηση — η μη αποθηκευμένη εργασία χάνεται

Το κλικ στο **New File** απορρίπτει όλες τις οντότητες στον τρέχοντα καμβά χωρίς προειδοποίηση. Αν θέλετε να διατηρήσετε το τρέχον σχέδιο, πρώτα [εξαγάγετέ](../export/) το.

## Πότε να χρησιμοποιείτε New File vs Import

| Κατάσταση | Συνιστώμενη ενέργεια |
|-----------|---------------------|
| Έναρξη σχεδίου από το μηδέν | **New File** |
| Άνοιγμα υπάρχοντος αρχείου DXF ή JSON | [Import](../import/) |
| Αντιγραφή σχεδίου για εργασία σε παραλλαγή | [Εξαγωγή](../export/) του τρέχοντος αρχείου, μετά [εισαγωγή](../import/) του αντιγράφου |

## Σχετικές εντολές

- [Import](../import/) — άνοιγμα υπάρχοντος σχεδίου DXF ή JSON
- [Export](../export/) — λήψη του σχεδίου πριν την έναρξη νέου
- [Files](../files/) — επαναφορά προηγούμενου σχεδίου από τη μνήμη του προγράμματος περιήγησης
