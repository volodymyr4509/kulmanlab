# Circle

Η εντολή `circle` σχεδιάζει κύκλο ορισμένο από κέντρο και ακτίνα. Αφού κάνετε κλικ στο κέντρο, μπορείτε να ορίσετε την ακτίνα είτε κάνοντας κλικ σε δεύτερο σημείο στον καμβά είτε πληκτρολογώντας έναν ακριβή αριθμό — και οι δύο επιλογές είναι ζωντανές ταυτόχρονα.

## Σχεδίαση κύκλου

1. Πληκτρολογήστε `circle` στο τερματικό ή κάντε κλικ στο κουμπί **Circle** της γραμμής εργαλείων.
2. **Κάντε κλικ στο κέντρο**, ή πληκτρολογήστε `X,Y` και πατήστε **Enter** για ακριβή συντεταγμένη.
3. Ορίστε την ακτίνα — είτε:
   - **Κάντε κλικ σε οποιοδήποτε σημείο** στον καμβά — η απόσταση από το κέντρο γίνεται η ακτίνα, ή
   - **Πληκτρολογήστε την ακτίνα** και πατήστε **Enter** για ακριβή τιμή.

Ο κύκλος τοποθετείται αμέσως και η εντολή τερματίζεται.

```
  center ●
          \  radius line preview
           \
            ● ← click here, or type a number
```

Κατά τη φάση ακτίνας, η ζωντανή προεπισκόπηση εμφανίζει τον κύκλο στην τρέχουσα απόσταση κέρσορα και σχεδιάζει επίσης γραμμή ακτίνας από το κέντρο στο τρέχον σημείο.

## Εισαγωγή συντεταγμένης κέντρου

Αντί για κλικ, μπορείτε να πληκτρολογήσετε τη θέση κέντρου:

1. Πληκτρολογήστε την τιμή X.
2. Πατήστε `,` — το τερματικό εμφανίζει `[X], [Y{cursor}]`.
3. Πληκτρολογήστε την τιμή Y.
4. Πατήστε **Enter** για τοποθέτηση κέντρου και μετάβαση στην εισαγωγή ακτίνας.

## Πληκτρολογημένη εισαγωγή ακτίνας

Αφού τοποθετηθεί το κέντρο, η πληκτρολόγηση δημιουργεί αμέσως τιμή ακτίνας:

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `0`–`9`, `.` | Προσθέτει ψηφίο στην τιμή ακτίνας |
| `Backspace` | Διαγράφει τον τελευταίο πληκτρολογημένο χαρακτήρα |
| `Enter` | Τοποθετεί τον κύκλο στην πληκτρολογημένη ακτίνα |

Η συσσωρευμένη τιμή εμφανίζεται στην προτροπή τερματικού (π.χ. `enter radius of circle: 25`). Η προεπισκόπηση ενημερώνεται για να εμφανίσει την πληκτρολογημένη ακτίνα ενώ ο κέρσορας ελέγχει την κατεύθυνση της γραμμής ακτίνας.

## Αναφορά πληκτρολογίου

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `0`–`9`, `.`, `-` | Έναρξη εισαγωγής συντεταγμένης X (φάση κέντρου), ή ψηφίο ακτίνας (φάση ακτίνας) |
| `,` | Κλείδωμα X και μετάβαση στην εισαγωγή Y (φάση κέντρου) |
| `Backspace` | Διαγραφή τελευταίου πληκτρολογημένου χαρακτήρα |
| `Enter` | Επιβεβαίωση πληκτρολογημένης συντεταγμένης ή ακτίνας |
| `Escape` | Ακύρωση και επαναφορά |

## Επεξεργασία grip — αλλαγή μεγέθους ακτίνας

Ένας επιλεγμένος κύκλος εκθέτει πέντε grips:

| Grip | Θέση | Τι κάνει |
|------|------|----------|
| **Center** | Κέντρο | Μετακινεί ολόκληρο τον κύκλο· η ακτίνα παραμένει αμετάβλητη |
| **Left** | Αριστερότερο σημείο (κέντρο − ακτίνα) | Σύρσιμο για ορισμό νέας ακτίνας = απόσταση στο κέντρο |
| **Right** | Δεξιότερο σημείο (κέντρο + ακτίνα) | Σύρσιμο για ορισμό νέας ακτίνας = απόσταση στο κέντρο |
| **Top** | Ανώτατο σημείο | Σύρσιμο για ορισμό νέας ακτίνας = απόσταση στο κέντρο |
| **Bottom** | Κατώτατο σημείο | Σύρσιμο για ορισμό νέας ακτίνας = απόσταση στο κέντρο |

Και τα τέσσερα βασικά grips συμπεριφέρονται πανομοιότυπα — η νέα ακτίνα ισούται με την απόσταση από το κέντρο στη θέση σύρσιμου. Το κέντρο παραμένει σταθερό.

## Επιλογή κύκλων

| Μέθοδος | Συμπεριφορά |
|---------|------------|
| **Κλικ** | Επιλέγει αν το κλικ βρίσκεται κοντά στην περιφέρεια |
| **Σύρσιμο δεξιά** (αυστηρό) | Ολόκληρο το οριακό τετράγωνο (κέντρο ± ακτίνα) πρέπει να βρίσκεται μέσα στο πλαίσιο |
| **Σύρσιμο αριστερά** (διασταυρούμενο) | Οποιοδήποτε τμήμα της περιφέρειας που τέμνει ή αγγίζει το όριο του πλαισίου επιλέγει τον κύκλο |

## Circle vs Arc — πότε να χρησιμοποιείτε το καθένα

| | Circle | Arc |
|---|--------|-----|
| Εύρος | Πλήρης 360° | Μερικό — ορισμένο από γωνία έναρξης και λήξης |
| Τρόπος σχεδίασης | Κέντρο + ακτίνα | Τρία σημεία στην καμπύλη |
| Πληκτρολογημένη είσοδος | Τιμή ακτίνας | Κανένα — μόνο κλικ |
| Grip αλλαγής μεγέθους | 4 βασικά σημεία | Σημεία έναρξης και λήξης (γωνία + ακτίνα) |
| Διαστασιολόγηση | Ακτίνα: [Dim Radius](../dim-radius/) · Διάμετρος: [Dim Diameter](../dim-diameter/) | [Dim Radius](../dim-radius/) |
| Καλύτερο για | Πλήρεις οπές, βλόνια, στρογγυλά χαρακτηριστικά | Fillets, μερικές καμπύλες, καμπυλωτά μονοπάτια |

## DXF — οντότητα CIRCLE

Οι κύκλοι αποθηκεύονται ως οντότητες `CIRCLE` στο αρχείο DXF. Οι συντεταγμένες κέντρου, η ακτίνα, το χρώμα, το επίπεδο, το linetype, η κλίμακα linetype και το πάχος ταξιδεύουν χωρίς απώλεια. Οποιαδήποτε εφαρμογή συμβατή με DXF τα διαβάζει ως τυπικούς κύκλους.
