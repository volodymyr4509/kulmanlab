# LayerIsolate

Η εντολή `LayerIsolate` παγώνει κάθε επίπεδο **εκτός** από αυτά στα οποία ανήκουν τα επιλεγμένα αντικείμενα. Χρησιμοποιήστε τη για γρήγορη εστίαση σε συγκεκριμένη γεωμετρία χωρίς να αποκρύψετε ή να διαγράψετε οτιδήποτε μόνιμα — ξεπαγώστε με [LayerUnfreezeAll](../layer-unfreeze-all/) όταν τελειώσετε.

## Δύο τρόποι εκκίνησης

**Προεπιλογή, μετά απομόνωση** — επιλέξτε οντότητες πρώτα, μετά ενεργοποιήστε:

1. Επιλέξτε μία ή περισσότερες οντότητες στον καμβά.
2. Πληκτρολογήστε `LayerIsolate` στο τερματικό ή κάντε κλικ στο κουμπί **Layer Isolate** της γραμμής εργαλείων.
3. Τα επίπεδα των επιλεγμένων οντοτήτων παραμένουν ορατά· όλα τα άλλα παγώνουν αμέσως.

**Ενεργοποίηση, μετά επιλογή**:

1. Πληκτρολογήστε `LayerIsolate` ή κάντε κλικ στο κουμπί γραμμής εργαλείων.
2. **Επιλέξτε αντικείμενα** — κλικ σε μεμονωμένες οντότητες ή σύρσιμο για επιλογή ανά περιοχή.
3. Πατήστε **Enter** ή **Space** για επιβεβαίωση — εφαρμόζεται η απομόνωση.

## Αναφορά πληκτρολογίου

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `Enter` / `Space` | Επιβεβαίωση επιλογής και εφαρμογή απομόνωσης |
| `Escape` | Ακύρωση και εκκαθάριση επιλογής |

## Λεπτομέρειες συμπεριφοράς

- Όλα τα επίπεδα που **δεν** εκπροσωπούνται στην επιλογή ορίζονται ως παγωμένα.
- Τα επίπεδα που **εκπροσωπούνται** παραμένουν ξεπαγωμένα, ακόμα και αν ήταν παγωμένα πριν.
- Η επιλογή διαγράφεται μετά την εφαρμογή της απομόνωσης.
- Η εντολή τερματίζεται αυτόματα μετά την εφαρμογή.

## Αναίρεση απομόνωσης

Εκτελέστε [LayerUnfreezeAll](../layer-unfreeze-all/) για επαναφορά όλων των επιπέδων σε ορατή κατάσταση με ένα βήμα.
