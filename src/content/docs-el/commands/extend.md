# Extend

Η εντολή `extend` επεκτείνει το πλησιέστερο τελικό σημείο μιας [Line](../line/) μέχρι την πλησιέστερη τομή που θα σχημάτιζε με άλλη οντότητα στο σχέδιο. Περάστε τον δείκτη κοντά στο τελικό σημείο που θέλετε να επεκτείνετε — μια προεπισκόπηση δείχνει την επεκταμένη γραμμή — μετά κάντε κλικ για εφαρμογή.

Το Extend λειτουργεί **μόνο σε οντότητες Line**. Τα όρια προς τα οποία επεκτείνεται η γραμμή μπορούν να είναι οποιοσδήποτε άλλος τύπος οντότητας εκτός από Text, Mtext, Multileader και Spline.

## Επέκταση γραμμής

1. Πληκτρολογήστε `extend` στο τερματικό ή κάντε κλικ στο κουμπί **Extend** της γραμμής εργαλείων.
2. **Περάστε τον δείκτη κοντά σε ένα άκρο γραμμής** — η προεπισκόπηση δείχνει τη γραμμή επεκταμένη στο πλησιέστερο όριο προς εκείνη την κατεύθυνση.
3. **Κάντε κλικ** για εφαρμογή της επέκτασης.

Η εντολή παραμένει ενεργή μετά από κάθε επέκταση ώστε να μπορείτε να επεκτείνετε πολλές γραμμές διαδοχικά. Πατήστε **Escape** για έξοδο.

```
  Before:                      After:

  ──────           |           ──────────────|
  (short line)     (boundary)  (extended to boundary)
```

## Πώς επιλέγεται το τελικό σημείο

Η εντολή εξετάζει ποιο τελικό σημείο είναι πλησιέστερα στον κέρσορα:

- Κέρσορας **πλησιέστερα στο end point** → το τέλος επεκτείνεται εμπρός κατά μήκος της κατεύθυνσης γραμμής.
- Κέρσορας **πλησιέστερα στο start point** → η αρχή επεκτείνεται προς τα πίσω (στην αντίθετη κατεύθυνση).

Μια ακτίνα εκπέμπεται από το επιλεγμένο τελικό σημείο κατά μήκος της κατεύθυνσης γραμμής, και η **πλησιέστερη τομή** κατά μήκος αυτής της ακτίνας με οποιαδήποτε άλλη οντότητα (εξαιρουμένης της ίδιας της γραμμής και των αγνοούμενων τύπων) γίνεται το νέο τελικό σημείο.

Αν δεν βρεθεί τομή προς εκείνη την κατεύθυνση, δεν εμφανίζεται προεπισκόπηση και το κλικ δεν κάνει τίποτα.

## Εξαιρέσεις ορίων

Οι ακόλουθοι τύποι οντοτήτων αγνοούνται ως όρια — μια γραμμή δεν επεκτείνεται για να συναντήσει αυτούς:

- Text / Mtext
- Multileader
- Spline

Όλοι οι άλλοι τύποι (Line, Arc, Circle, Ellipse, Polyline, Dimension) χρησιμεύουν ως έγκυρα όρια.

## Αναφορά πληκτρολογίου

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `Escape` | Έξοδος από λειτουργία extend |

## Υποστηριζόμενες οντότητες

| Οντότητα | Μπορεί να επεκταθεί; |
|----------|---------------------|
| Line | Ναι |
| Arc, Circle, Ellipse | Όχι |
| Polyline / Rectangle | Όχι |
| Text, Spline, Dimension, Leader | Όχι |

## Extend vs Trim

| | Extend | Trim |
|---|--------|------|
| Τι κάνει | Επεκτείνει τελικό σημείο γραμμής σε όριο | Αφαιρεί τμήμα γραμμής |
| Έναυσμα | Hover κοντά στο τελικό σημείο για επέκταση | Hover πάνω από το τμήμα για κοπή |
| Αποτέλεσμα | Τελικό σημείο γραμμής μετακινείται προς τα έξω | Η γραμμή διαιρείται ή συντομεύεται |
| Και οι δύο | Μόνο Lines | Μόνο Lines |
