# Angle

Η εντολή `angle` μετρά την εσωτερική γωνία σε μια κορυφή που σχηματίζεται από δύο ευθύγραμμα τμήματα μέσω τριών σημείων κλικ. Το αποτέλεσμα — πάντα μεταξύ 0° και 180° — εμφανίζεται στο τερματικό με 4 δεκαδικά ψηφία. Είναι μία από τις τρεις εντολές μέτρησης — η [Distance](../distance/) μετρά το μήκος ευθείας γραμμής, και το [Area](../area/) μετρά το εμβαδόν και την περίμετρο ενός πολυγώνου.

## Ανατομία μέτρησης γωνίας

```
  ● πρώτο σημείο (άκρο πρώτης ακτίνας)
   \
    \  προεπισκόπηση πρώτης ακτίνας
     \
      ● κορυφή (βήμα 3)
     /
    /  προεπισκόπηση δεύτερης ακτίνας (προς κέρσορα)
   /
  ● τρίτο σημείο  →  τερματικό: "Angle: 45.0000°"
```

- **Πρώτο σημείο** — ένα άκρο της γωνίας (βήμα 2).
- **Κορυφή** — η γωνία στην οποία μετράται η γωνία (βήμα 3).
- **Τρίτο σημείο** — το άλλο άκρο της γωνίας (βήμα 4).

## Μέτρηση γωνίας

1. Πληκτρολογήστε `angle` στο τερματικό ή κάντε κλικ στο κουμπί **Angle** της γραμμής εργαλείων.
2. **Κάντε κλικ στο πρώτο σημείο** — ένα άκρο της γωνίας. Ή πληκτρολογήστε `X,Y` και πατήστε **Enter** για ακριβή συντεταγμένη.
3. **Κάντε κλικ στην κορυφή** — τη γωνία όπου συναντώνται οι δύο βραχίονες. Η εισαγωγή συντεταγμένης λειτουργεί και εδώ.
4. **Κάντε κλικ στο τρίτο σημείο** — το άκρο του δεύτερου βραχίονα. Η εισαγωγή συντεταγμένης λειτουργεί και εδώ. Η τοποθέτηση αυτού του σημείου εμφανίζει το αποτέλεσμα.
5. **Κάντε κλικ ξανά** (προαιρετικά) για να ξεκινήσετε νέα μέτρηση από αυτό το κλικ ως νέο πρώτο σημείο.

## Σύμβαση εσωτερικής γωνίας

Η εντολή υπολογίζει τη γωνία χρησιμοποιώντας το εσωτερικό γινόμενο των δύο ακτίνων από την κορυφή:

- **Πάντα εσωτερική**: το αποτέλεσμα είναι η μικρότερη γωνία, μεταξύ 0° και 180°.
- Η σειρά με την οποία κάνετε κλικ στα άκρα δεν επηρεάζει το αποτέλεσμα — μόνο η θέση της κορυφής έχει σημασία.
- Συγγραμμικά σημεία (και τα τρία σε μία γραμμή) επιστρέφουν 0° ή 180°.

## Αλυσίδα μετρήσεων

Αφού εμφανιστεί το αποτέλεσμα, το κλικ ξεκινά αμέσως την επόμενη μέτρηση — το σημείο κλικ γίνεται το νέο πρώτο σημείο. Η εντολή δεν εξέρχεται αυτόματα μέχρι να πατήσετε `Escape`.

## Angle vs Distance

| | Angle | Distance |
|---|-------|---------|
| Τι μετρά | Εσωτερική γωνία σε κορυφή | Μήκος ευθείας γραμμής |
| Αριθμός κλικ | 3 | 2 |
| Μορφή αποτελέσματος | `45.0000°` | `12.3456` (μονάδες) |
| Προεπισκόπηση καμβά | Δύο γραμμές από κορυφή προς τα άκρα | Γραμμή από πρώτο σημείο προς κέρσορα |
| Καλύτερο για | Γωνία ανοίγματος μεταξύ δύο χαρακτηριστικών | Μήκος κενού ή τμήματος |

## Εισαγωγή συντεταγμένης

Αντί για κλικ, πληκτρολογήστε ακριβή θέση για οποιοδήποτε από τα τρία σημεία:

1. Πληκτρολογήστε την τιμή X.
2. Πατήστε `,` — το τερματικό εμφανίζει `[X], [Y{cursor}]`.
3. Πληκτρολογήστε την τιμή Y.
4. Πατήστε **Enter** για επιβεβαίωση.

## Αναφορά πληκτρολογίου

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `0`–`9`, `.`, `-` | Έναρξη εισαγωγής συντεταγμένης X |
| `,` | Κλείδωμα X και μετάβαση στην εισαγωγή Y |
| `Backspace` | Διαγραφή τελευταίου πληκτρολογημένου χαρακτήρα |
| `Enter` | Επιβεβαίωση πληκτρολογημένης συντεταγμένης |
| `Escape` | Ακύρωση και επαναφορά στο βήμα 2 |

## Σημειώσεις

- Τα αποτελέσματα εμφανίζονται **μόνο στο τερματικό** — τίποτα δεν προστίθεται στο σχέδιο.
- Η ακρίβεια είναι πάντα 4 δεκαδικά ψηφία σε μοίρες.
