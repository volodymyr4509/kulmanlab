# Files

Η εντολή **Files** ανοίγει το πάνελ **Πρόσφατων Αρχείων** — μια λίστα όλων των σχεδίων που έχουν αποθηκευτεί στην τοπική αποθήκευση του browser σας. Χρησιμοποιήστε το για επαναφόρτωση προηγούμενου σχεδίου ή αφαίρεση αρχείων που δεν χρειάζεστε πλέον.

## Πώς να ανοίξετε το πάνελ Πρόσφατων Αρχείων

Κάντε κλικ στο κουμπί **Files** της γραμμής εργαλείων (εικονίδιο ιστορικού) στο πάνελ File στο πάνω μέρος της οθόνης. Το πάνελ ανοίγει στην αριστερή πλευρά του καμβά.

## Πάνελ Πρόσφατων Αρχείων

Κάθε καταχώριση στη λίστα εμφανίζει το όνομα αρχείου και δύο κουμπιά ενέργειας:

| Κουμπί | Ενέργεια |
|--------|----------|
| ✓ **Restore** | Φορτώνει το σχέδιο στον καμβά — αντικαθιστά το τρέχον περιεχόμενο |
| ✕ **Delete** | Αφαιρεί το αρχείο από την αποθήκευση browser μόνιμα |

Αν δεν έχουν αποθηκευτεί αρχεία ακόμα, το πάνελ εμφανίζει "No files saved".

## Πώς αποθηκεύονται τα αρχεία

Το KulmanLab αποθηκεύει σχέδια στο **IndexedDB** — μια επίμονη βάση δεδομένων εντός browser. Αυτό σημαίνει:

- Τα αρχεία επιβιώνουν από επαναφορτώσεις σελίδας και επανεκκινήσεις browser.
- Τα αρχεία αποθηκεύονται **τοπικά στη συσκευή σας** — δεν ανεβαίνουν σε κανέναν διακομιστή.
- Κάθε browser και συσκευή έχει τη δική της ανεξάρτητη αποθήκευση. Ένα σχέδιο αποθηκευμένο στο Chrome σε έναν υπολογιστή δεν είναι διαθέσιμο στο Firefox ή σε άλλη μηχανή.
- Η εκκαθάριση δεδομένων ιστότοπου του browser θα διαγράψει όλα τα αποθηκευμένα αρχεία.

## Αυτόματη φόρτωση αρχείου κατά την εκκίνηση

Όταν ανοίγετε το KulmanLab CAD, η εφαρμογή φορτώνει αυτόματα το **πιο πρόσφατα τροποποιημένο αρχείο** από την αποθήκευση. Δεν χρειάζεται να το ανοίξετε χειροκίνητα από το πάνελ Files κάθε φορά.

## Διαχείριση αποθήκευσης

Δεν υπάρχει σταθερό όριο στον αριθμό σχεδίων που μπορείτε να αποθηκεύσετε, αλλά η αποθήκευση browser είναι πεπερασμένη. Αν παρατηρήσετε προειδοποιήσεις αποθήκευσης, διαγράψτε παλαιότερα αρχεία από το πάνελ Files.

Για αφαίρεση όλων των αποθηκευμένων σχεδίων ταυτόχρονα, χρησιμοποιήστε την εντολή [WipeStorage](../wipestorage/).

## Ονόματα αρχείων και εκδόσεις

Κάθε αρχείο έχει χρονική σήμανση στο όνομά του (π.χ. `myplan_May22_14:30:00.dxf`). Αυτή η χρονική σήμανση προστίθεται αυτόματα κατά την [εισαγωγή](../import/) αρχείου, διασφαλίζοντας ότι η επανεισαγωγή του ίδιου αρχείου πηγής δημιουργεί νέα καταχώριση αντί να αντικαθιστά την προηγούμενη έκδοση.

## Σχετικές εντολές

- [Import](../import/) — φόρτωση σχεδίου από το σύστημα αρχείων σας στην αποθήκευση browser
- [Export](../export/) — λήψη σχεδίου στο σύστημα αρχείων σας
- [New File](../new-file/) — εκκίνηση κενού σχεδίου (αποθηκεύεται επίσης αυτόματα)
- [WipeStorage](../wipestorage/) — εκκαθάριση όλων των αποθηκευμένων αρχείων από την αποθήκευση browser
