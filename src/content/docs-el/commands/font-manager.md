# Font Manager

Η εντολή `FontManager` ανοίγει ένα παράθυρο διαλόγου για περιήγηση και επιλογή γραμματοσειρών, καθώς και για μεταφόρτωση των δικών σας αρχείων `.ttf` για χρήση σε οντότητες [Text](../text/) και [Multileader](../leader/).

## Άνοιγμα του Font Manager

- Πληκτρολογήστε `FontManager` στο τερματικό, **ή**
- Κάντε κλικ στο κουμπί **Font Manager** στη γραμμή εργαλείων του [επεξεργαστή κειμένου](../../interface/text-editor/).

## Ομάδες γραμματοσειρών

| Ομάδα | Περιεχόμενα |
|-------|-------------|
| **Default** | Η ενσωματωμένη γραμματοσειρά sans-serif — πάντα διαθέσιμη |
| **User** | Οι δικές σας μεταφορτωμένες γραμματοσειρές `.ttf` (εμφανίζονται μόνο αφού προσθέσετε μία) |
| **Free** | 15 ενσωματωμένες γραμματοσειρές Google Fonts (EB Garamond, Fira Code, Inter, Lato, Merriweather, Montserrat, Nunito, Open Sans, Oswald, Playfair Display, Poppins, Raleway, Roboto, Roboto Condensed, Source Code Pro) |
| **System** | Κοινές γραμματοσειρές λειτουργικού συστήματος (Courier New, Georgia, Helvetica, Impact, Lucida Console, Tahoma, Times New Roman, Trebuchet MS, Verdana) |

Κάντε κλικ σε οποιαδήποτε γραμματοσειρά της λίστας για προεπισκόπηση στα δεξιά — όνομα, δείγμα αλφαβήτου, ένα pangram και ψηφία.

## Μεταφόρτωση προσαρμοσμένης γραμματοσειράς

1. Κάντε κλικ στο **Add Font** στο υποσέλιδο του παραθύρου διαλόγου (ή πληκτρολογήστε `Font+` στο τερματικό για να ανοίξετε απευθείας τον επιλογέα αρχείων).
2. Επιλέξτε ένα αρχείο `.ttf`. Υποστηρίζονται μόνο γραμματοσειρές TrueType — τα `.otf` και `.woff`/`.woff2` δεν υποστηρίζονται.
3. Το όνομα του αρχείου (χωρίς την επέκταση) γίνεται το όνομα της γραμματοσειράς στην ομάδα **User**. Για παράδειγμα, η μεταφόρτωση του `MyFont.ttf` προσθέτει μια γραμματοσειρά με όνομα `MyFont`.

Οι μεταφορτωμένες γραμματοσειρές αποθηκεύονται μόνιμα στον browser (IndexedDB) και φορτώνονται ξανά αυτόματα την επόμενη φορά που ανοίγετε το KulmanLab CAD.

## Αφαίρεση προσαρμοσμένης γραμματοσειράς

Κάντε hover πάνω σε μια γραμματοσειρά της ομάδας **User** και κάντε κλικ στο κουμπί **×** δίπλα της. Οι ενσωματωμένες γραμματοσειρές (Default, Free, System) δεν μπορούν να αφαιρεθούν.

## Αναφορά πληκτρολογίου

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `↑` / `↓` | Μετακινεί την επιλογή πάνω ή κάτω στη λίστα γραμματοσειρών |
| `Escape` | Κλείνει το Font Manager |

## Συμβατότητα DXF

Το όνομα της γραμματοσειράς ενσωματώνεται στις εξαγόμενες οντότητες **MTEXT** ως ενσωματωμένος κωδικός μορφοποίησης, οπότε ένα DXF που περνά από το KulmanLab CAD και επιστρέφει διατηρεί την ανάθεση γραμματοσειράς του. Τα *αρχεία* προσαρμοσμένων γραμματοσειρών δεν ενσωματώνονται στο DXF — ενσωματώνεται μόνο το *όνομα* της γραμματοσειράς. Αν επανεισάγετε ένα σχέδιο που αναφέρεται σε μια προσαρμοσμένη γραμματοσειρά που δεν έχετε μεταφορτώσει σε αυτή τη συσκευή, το κείμενο αποδίδεται με την προεπιλεγμένη γραμματοσειρά μέχρι να μεταφορτώσετε μια γραμματοσειρά με το ίδιο όνομα.

## Σχετικές εντολές

- [Text](../text/) — τοποθετεί τις ετικέτες κειμένου στις οποίες εφαρμόζονται οι επιλογές γραμματοσειράς
- [Match Properties](../match-properties/) — αντιγράφει το ύψος κειμένου, αλλά όχι τη γραμματοσειρά, μεταξύ οντοτήτων
