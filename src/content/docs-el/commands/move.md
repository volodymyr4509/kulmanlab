# Move

Η εντολή `move` μεταφράζει επιλεγμένες οντότητες από βασικό σημείο σε σημείο προορισμού. Η μετατόπιση που εφαρμόζεται σε κάθε επιλεγμένη οντότητα είναι το διάνυσμα από τη βάση στον προορισμό. Μετά τη μεταφορά όλες οι οντότητες παραμένουν επιλεγμένες στη νέα τους θέση, έτοιμες για περαιτέρω επεξεργασία.

## Δύο τρόποι εκκίνησης

**Προεπιλογή, μετά μεταφορά** — επιλέξτε πρώτα οντότητες, μετά ενεργοποιήστε:

1. Επιλέξτε μία ή περισσότερες οντότητες στον καμβά.
2. Πληκτρολογήστε `move` στο τερματικό ή κάντε κλικ στο κουμπί **Move** της γραμμής εργαλείων.
3. **Κάντε κλικ στο βασικό σημείο**, ή πληκτρολογήστε `X,Y` και πατήστε **Enter** για ακριβή συντεταγμένη.
4. **Κάντε κλικ στον προορισμό** — όλες οι επιλεγμένες οντότητες μετατοπίζονται κατά το διάνυσμα βάση→προορισμός. Η εισαγωγή συντεταγμένης λειτουργεί και εδώ.

**Ενεργοποίηση, μετά επιλογή** — ξεκινήστε εντολή χωρίς τίποτα επιλεγμένο:

1. Πληκτρολογήστε `move` ή κάντε κλικ στο κουμπί της γραμμής εργαλείων.
2. **Επιλέξτε αντικείμενα** — κλικ για εναλλαγή μεμονωμένων οντοτήτων, ή σύρσιμο για επιλογή κατά περιοχή.
3. Πατήστε **Enter** ή **Space** για επιβεβαίωση επιλογής.
4. **Κάντε κλικ στο βασικό σημείο**, μετά **κλικ στον προορισμό** (εισαγωγή συντεταγμένης διαθέσιμη και στα δύο βήματα).

```
  Πριν:                    Μετά:
  ● βάση                     → ● προορισμός
  [οντότητα A]                  [οντότητα A μετακινημένη]
  [οντότητα B]                  [οντότητα B μετακινημένη]
```

Μια φαντομ προεπισκόπηση όλων των επιλεγμένων οντοτήτων ακολουθεί τον κέρσορα από το βασικό σημείο ως τον προορισμό, δείχνοντας το αποτέλεσμα πριν κάνετε κλικ.

## Εισαγωγή συντεταγμένης

Στο βήμα βασικού σημείου ή προορισμού, πληκτρολογήστε ακριβή θέση αντί για κλικ:

1. Πληκτρολογήστε την τιμή X.
2. Πατήστε `,` — το τερματικό εμφανίζει `[X], [Y{cursor}]`.
3. Πληκτρολογήστε την τιμή Y.
4. Πατήστε **Enter** για επιβεβαίωση.

## Κλείδωμα γωνίας και ακριβής απόσταση

Αφού οριστεί το βασικό σημείο, η εντολή παρακολουθεί για άξονα ευθυγράμμισης 45° (0°, 45°, 90°, 135°, …). Η κατεύθυνση **κλειδώνει** όταν ο κέρσορας είναι αρκετά μακριά από τη βάση και εντός ενός πλάτους grip από τον άξονα. Ενώ είναι κλειδωμένο:

- Η φαντομ προεπισκόπηση ευθυγραμμίζεται στον άξονα.
- Πληκτρολογήστε απόσταση και πατήστε **Enter** για μεταφορά ακριβώς τόσο μακριά κατά μήκος της κλειδωμένης κατεύθυνσης.
- Το κλικ προβάλλεται στον άξονα, ώστε ο προορισμός να βρίσκεται πάντα ακριβώς πάνω σε αυτόν.

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `0`–`9`, `.` | Προσθήκη στην τιμή απόστασης |
| `-` | Αρνητική απόσταση — αντιστρέφει κατεύθυνση κατά μήκος του άξονα (μόνο πρώτος χαρακτήρας) |
| `Backspace` | Διαγραφή τελευταίου πληκτρολογημένου χαρακτήρα |
| `Enter` | Εφαρμογή μεταφοράς στην πληκτρολογημένη απόσταση |

## Αναφορά πληκτρολογίου

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `Enter` / `Space` | Επιβεβαίωση επιλογής και πρόοδος στη φάση βασικού σημείου |
| `0`–`9`, `.`, `-` | Έναρξη εισαγωγής συντεταγμένης X, ή απόσταση όταν είναι κλειδωμένη η γωνία |
| `,` | Κλείδωμα X και μετάβαση στην εισαγωγή Y |
| `Backspace` | Διαγραφή τελευταίου πληκτρολογημένου χαρακτήρα |
| `Enter` | Επιβεβαίωση συντεταγμένης ή εφαρμογή μεταφοράς στην πληκτρολογημένη απόσταση |
| `Escape` | Ακύρωση και επαναφορά |

## Ενεργοποίηση Move από grip

Κλικ στο **grip μέσου σημείου** επιλεγμένης [Line](../line/) εκκινεί αυτόματα την Move, με το μέσο σημείο ήδη ορισμένο ως βασικό σημείο και τη φάση μεταφοράς ενεργή. Αυτός είναι ο πιο γρήγορος τρόπος για αλλαγή θέσης μιας μεμονωμένης γραμμής χωρίς το βήμα επιλογής.

## Επιλογή κατά τη διάρκεια της εντολής

Όταν η εντολή ξεκινά στη φάση επιλογής:

| Μέθοδος | Συμπεριφορά |
|---------|------------|
| **Κλικ** | Εναλλαγή οντότητας κάτω από τον κέρσορα μέσα/έξω από επιλογή |
| **Σύρσιμο δεξιά** (αυστηρό) | Προσθέτει οντότητες πλήρως εντός του πλαισίου |
| **Σύρσιμο αριστερά** (διασταυρούμενο) | Προσθέτει οντότητες που τέμνουν το όριο πλαισίου |
| **Enter** / **Space** | Επιβεβαιώνει επιλογή και μεταβαίνει στη φάση βασικού σημείου |

## Μετά τη μεταφορά

Οι μετακινημένες οντότητες παραμένουν επιλεγμένες στη νέα τους θέση. Αυτό σημαίνει ότι μπορείτε αμέσως:
- Να εκτελέσετε **Move** ξανά για μικρορύθμιση.
- Να εκτελέσετε [Copy](../copy/), [Rotate](../rotate/), ή [Scale](../scale/) χωρίς επανεπιλογή.
- Να πατήσετε **Delete** για αφαίρεση.

## Move vs Copy

| | Move | Copy |
|---|------|------|
| Αρχική θέση | Κενή — οι οντότητες δεν βρίσκονται πλέον εκεί | Διατηρείται — τα πρωτότυπα παραμένουν στη θέση τους |
| Πλήθος αποτελέσματος | Ίδιος αριθμός οντοτήτων | Ένα επιπλέον σύνολο ανά λειτουργία |
| Επιλογή μετά | Μετακινημένες οντότητες επιλεγμένες στη νέα θέση | Αντιγραμμένες οντότητες επιλεγμένες στη νέα θέση |
| Καλύτερο για | Αλλαγή θέσης γεωμετρίας | Αντιγραφή γεωμετρίας |

## Υποστηριζόμενες οντότητες

Η Move λειτουργεί σε κάθε τύπο οντότητας: Line, Polyline, Rectangle, Circle, Arc, Ellipse, Text, Spline, Dimension, Leader και όλους τους άλλους. Όλες οι οντότητες υλοποιούν `translate(dx, dy)` οπότε καμία δεν αποκλείεται.
