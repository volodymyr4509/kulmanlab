# Fillet

Η εντολή `fillet` στρογγυλοποιεί τη γωνία μεταξύ δύο οντοτήτων [Line](./line) εισάγοντας εφαπτομενικό τόξο δοσμένης ακτίνας και κόβοντας κάθε γραμμή πίσω στο σημείο όπου ξεκινά το τόξο. Το αποτέλεσμα είναι μια ομαλή, ακτινωτή γωνία που συνδέει και τις δύο γραμμές.

Το Fillet λειτουργεί **μόνο σε οντότητες Line**.

## Χρήση του fillet

1. Πληκτρολογήστε `fillet` στο τερματικό ή κάντε κλικ στο κουμπί **Fillet** της γραμμής εργαλείων.
2. **Πληκτρολογήστε την ακτίνα fillet** και πατήστε **Enter**.
3. **Κάντε κλικ στην πρώτη γραμμή** — το τμήμα που κλικάρετε καθορίζει ποια πλευρά οποιασδήποτε τομής διατηρείται.
4. **Περάστε τον δείκτη πάνω από τη δεύτερη γραμμή** — μια διακεκομμένη προεπισκόπηση τόξου δείχνει το προκύπτον fillet. Μετακινήστε τον κέρσορα στην πλευρά που θέλετε να διατηρήσετε.
5. **Κάντε κλικ** για εφαρμογή. Και οι δύο γραμμές κόβονται και το τόξο εισάγεται.

```
  Before:                     After fillet (radius r):

  ──────────────              ──────────╮
                │                        ╰────
                │
```

## Επιλογή πλευράς για τεμνόμενες γραμμές

Όταν δύο γραμμές τέμνονται, το fillet εφαρμόζεται στη γωνία που ορίζεται από τις θέσεις κλικ — το τμήμα κάθε γραμμής στην **ίδια πλευρά με τον κέρσορα** διατηρείται.

- Κάντε κλικ κοντά σε ένα άκρο της πρώτης γραμμής για να επιλέξετε εκείνο το μισό.
- Μετακινήστε τον κέρσορα στο επιθυμητό μισό της δεύτερης γραμμής — η διακεκομμένη προεπισκόπηση ενημερώνεται ζωντανά.

## Τι δημιουργεί η εντολή

- Το τελικό σημείο της πρώτης γραμμής πλησιέστερα στην τομή μετακινείται στο εφαπτόμενο σημείο **T1**.
- Το τελικό σημείο της δεύτερης γραμμής πλησιέστερα στην τομή μετακινείται στο εφαπτόμενο σημείο **T2**.
- Μια νέα οντότητα Arc εισάγεται από το **T1** στο **T2**, εφαπτόμενη και στις δύο γραμμές.

Το εισαχθέν τόξο κληρονομεί τις τρέχουσες ρυθμίσεις lineweight, χρώματος, επιπέδου και linetype.

## Αναφορά πληκτρολογίου

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `0`–`9`, `.` | Προσθέτει ψηφίο στην τιμή ακτίνας |
| `Backspace` | Διαγράφει τον τελευταίο πληκτρολογημένο χαρακτήρα |
| `Enter` | Επιβεβαίωση πληκτρολογημένης ακτίνας και μετάβαση στην επιλογή γραμμής |
| `Escape` | Ακύρωση και επαναφορά |

## Υποστηριζόμενες οντότητες

| Οντότητα | Υποστηρίζεται |
|----------|-------------|
| Line | Ναι — ως πρώτη και δεύτερη οντότητα |
| Arc, Circle, Ellipse, Polyline | Όχι |
| Text, Spline, Dimension, Leader | Όχι |

## Fillet vs Chamfer

| | Fillet | Chamfer |
|---|--------|---------|
| Τύπος γωνίας | Στρογγυλό τόξο | Ευθεία κοπή |
| Είσοδος | Μία ακτίνα | Δύο αποστάσεις (d1, d2) |
| Εισαχθείσα οντότητα | Arc | Line |
| Υποστηριζόμενες οντότητες | Μόνο Lines | Lines και Polylines |
