# Ellipse

Η εντολή `ellipse` σχεδιάζει έλλειψη με τρία κλικ: ένα κέντρο, το τελικό σημείο του πρώτου (κύριου) ημι-άξονα σε οποιαδήποτε γωνία, και το μήκος του δεύτερου (δευτερεύοντος) ημι-άξονα. Οι δύο άξονες είναι πάντα κάθετοι μεταξύ τους — η κατεύθυνση του δεύτερου άξονα προκύπτει αυτόματα από τον πρώτο.

## Σχεδίαση έλλειψης

1. Πληκτρολογήστε `ellipse` στο τερματικό ή κάντε κλικ στο κουμπί **Ellipse** της γραμμής εργαλείων.
2. **Κάντε κλικ στο κέντρο**, ή πληκτρολογήστε `X,Y` και πατήστε **Enter** για ακριβή συντεταγμένη.
3. **Κάντε κλικ στο τελικό σημείο πρώτου άξονα** — ορίζει τόσο την κατεύθυνση όσο και το μήκος του πρώτου ημι-άξονα. Η εισαγωγή συντεταγμένης λειτουργεί και εδώ.
4. **Ορίστε το μήκος δεύτερου άξονα** — μετακινήστε τον κέρσορα κάθετα στον πρώτο άξονα, μετά κάντε κλικ ή πληκτρολογήστε μήκος.

```
               ● ← first axis endpoint (step 3)
              /
  center ●  /  ← first axis (any angle)
            |
            ● ← cursor here sets second axis length (step 4)
```

Η έλλειψη τοποθετείται μετά το βήμα 4 και η εντολή τερματίζεται.

## Είσοδος άξονα — κλικ, συντεταγμένη ή πληκτρολογημένο μήκος

**Κέντρο (βήμα 2):** κλικ ή πληκτρολόγηση `X,Y` για ακριβή θέση.

**Τελικό σημείο πρώτου άξονα (βήμα 3):** κλικ, ή πληκτρολόγηση `X,Y` για ακριβή συντεταγμένη. Το κλείδωμα γωνίας κάνει επίσης snap σε τμήματα 45° — ενώ είναι κλειδωμένο, πληκτρολογήστε μήκος και πατήστε **Enter** για τοποθέτηση τελικού σημείου σε εκείνη την ακριβή απόσταση.

**Δεύτερος άξονας (βήμα 4):** το πληκτρολογημένο μήκος είναι πάντα διαθέσιμο — δεν απαιτείται κλείδωμα γωνίας. Η κατεύθυνση είναι ήδη σταθερή κάθετα στον πρώτο άξονα· η πληκτρολόγηση ορίζει μόνο το μήκος.

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `0`–`9`, `.` | Προσθέτει ψηφίο στο μήκος άξονα (φάση δεύτερου άξονα) |
| `Backspace` | Διαγράφει τον τελευταίο πληκτρολογημένο χαρακτήρα |
| `Enter` | Τοποθετεί το τελικό σημείο άξονα στο πληκτρολογημένο μήκος |

## Εισαγωγή συντεταγμένης (κέντρο και τελικό σημείο πρώτου άξονα)

1. Πληκτρολογήστε την τιμή X.
2. Πατήστε `,` — το τερματικό εμφανίζει `[X], [Y{cursor}]`.
3. Πληκτρολογήστε την τιμή Y.
4. Πατήστε **Enter** για επιβεβαίωση.

## Αναφορά πληκτρολογίου

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `0`–`9`, `.`, `-` | Έναρξη εισαγωγής συντεταγμένης X (φάσεις κέντρου/πρώτου άξονα), ή μήκος άξονα όταν η γωνία είναι κλειδωμένη |
| `,` | Κλείδωμα X και μετάβαση στην εισαγωγή Y |
| `Backspace` | Διαγραφή τελευταίου πληκτρολογημένου χαρακτήρα |
| `Enter` | Επιβεβαίωση πληκτρολογημένης συντεταγμένης ή μήκους |
| `Escape` | Ακύρωση και επαναφορά |

## Επεξεργασία grip — ανεξάρτητη αλλαγή μεγέθους αξόνων

Μια επιλεγμένη έλλειψη εκθέτει πέντε grips:

| Grip | Πλήθος | Τι κάνει |
|------|--------|----------|
| **Center** | 1 | Μετακινεί ολόκληρη την έλλειψη· και οι δύο άξονες αμετάβλητοι |
| **Τελικά σημεία κύριου άξονα** | 2 (αντίθετα άκρα του μεγαλύτερου άξονα) | Σύρσιμο για αλλαγή μήκους κύριου ημι-άξονα· το απόλυτο μέγεθος δευτερεύοντος άξονα παραμένει σταθερό |
| **Τελικά σημεία δευτερεύοντος άξονα** | 2 (αντίθετα άκρα του μικρότερου άξονα) | Σύρσιμο για αλλαγή μήκους δευτερεύοντος ημι-άξονα· κύριος άξονας αμετάβλητος |

Τα grips κύριου και δευτερεύοντος άξονα είναι ανεξάρτητα — μπορείτε να αναδιαμορφώσετε την έλλειψη χωρίς επανεκτέλεση της εντολής.

## Επιλογή ελλείψεων

| Μέθοδος | Συμπεριφορά |
|---------|------------|
| **Κλικ** | Επιλέγει αν το κλικ βρίσκεται κοντά στο περίγραμμα της έλλειψης |
| **Σύρσιμο δεξιά** (αυστηρό) | Το οριακό πλαίσιο (axis-aligned) της έλλειψης πρέπει να χωράει πλήρως μέσα στο πλαίσιο επιλογής |
| **Σύρσιμο αριστερά** (διασταυρούμενο) | Οποιοδήποτε τμήμα του περιγράμματος έλλειψης που τέμνει το πλαίσιο επιλογής την επιλέγει |

## Ιδιότητες

**Γενικά**

| Ιδιότητα | Προεπιλογή | Σημασία |
|----------|-----------|---------|
| Color | 256 (ByLayer) | Δείκτης χρώματος ACI |
| Layer | `0` | Ανάθεση επιπέδου |
| Linetype | ByLayer | Μοτίβο linetype ονόματος |
| Linetype Scale | 1 | Συντελεστής κλίμακας στο μοτίβο linetype |
| Thickness | 0 | Πάχος εξώθησης |

**Γεωμετρία**

| Ιδιότητα | Σημασία |
|----------|---------|
| Center X / Center Y | Κέντρο της έλλειψης |
| Major Axis X / Major Axis Y | Διάνυσμα από κέντρο στο τελικό σημείο κύριου άξονα (κωδικοποιεί κατεύθυνση και μήκος) |
| Axis Ratio | Αναλογία δευτερεύοντος ημι-άξονα προς κύριο ημι-άξονα (0 < αναλογία ≤ 1) |
| Start Angle / End Angle | Παραμετρικές γωνίες σε μοίρες· και οι δύο 0°/360° για πλήρη έλλειψη |

## Ellipse vs Circle — πότε να χρησιμοποιείτε το καθένα

| | Ellipse | Circle |
|---|---------|--------|
| Άξονες | Δύο ανεξάρτητοι ημι-άξονες σε οποιαδήποτε γωνία | Μία ακτίνα, συμμετρική |
| Περιστροφή | Μπορεί να τοποθετηθεί σε οποιαδήποτε γωνία | Χωρίς περιστροφή |
| Πληκτρολογημένη είσοδος | Μήκος ανά άξονα | Μόνο ακτίνα |
| Αλλαγή μεγέθους grip | Κύριος και δευτερεύων ανεξάρτητα | Και τα τέσσερα βασικά σημεία εξίσου |
| Καλύτερο για | Λοξές προβολές, ωοειδή χαρακτηριστικά, οπτικές τρύπες | Συμμετρικά στρογγυλά χαρακτηριστικά |

## DXF — οντότητα ELLIPSE

Οι ελλείψεις αποθηκεύονται ως οντότητες `ELLIPSE` στο αρχείο DXF. Η μορφή αποθηκεύει το κέντρο, το πλήρες διάνυσμα κύριου άξονα (κατεύθυνση + μήκος) και την αναλογία αξόνων. Η περιστροφή, το σχήμα και όλες οι ιδιότητες στυλ ταξιδεύουν χωρίς απώλεια. Ένας κύκλος **δεν** αποθηκεύεται ως εκφυλισμένη έλλειψη — οι δύο τύποι οντοτήτων παραμένουν διακριτοί στο μοντέλο DXF.
