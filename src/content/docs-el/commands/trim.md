# Trim

Η εντολή `trim` αφαιρεί το τμήμα μιας [Line](./line) που βρίσκεται μεταξύ δύο παρακείμενων σημείων τομής, χωρίζοντας τη γραμμή σε ένα ή δύο κοντύτερα τμήματα. Το τμήμα που θα κοπεί καθορίζεται από τη θέση του κέρσορα — τοποθετήστε τον κέρσορα πάνω από το τμήμα που θέλετε να αφαιρεθεί και κάντε κλικ για κόψιμο.

Το Trim λειτουργεί **μόνο σε οντότητες Line**. Για τόξα, κύκλους, polylines και άλλους τύπους οντοτήτων χρησιμοποιήστε [Delete](./delete) ή επεξεργασία grip.

## Κόψιμο γραμμής

1. Πληκτρολογήστε `trim` στο τερματικό ή κάντε κλικ στο κουμπί **Trim** της γραμμής εργαλείων.
2. **Τοποθετήστε τον κέρσορα πάνω από το τμήμα γραμμής** που θέλετε να αφαιρέσετε — μια κόκκινη προεπισκόπηση επισημαίνει ακριβώς το τμήμα που θα κοπεί.
3. **Κάντε κλικ** για αφαίρεση αυτού του τμήματος.

Η εντολή παραμένει ενεργή μετά από κάθε κόψιμο, ώστε να μπορείτε να συνεχίσετε να τοποθετείτε τον κέρσορα και να κάνετε κλικ για κόψιμο περισσότερων τμημάτων. Πατήστε **Escape** για έξοδο.

```
  Πριν:                          Μετά κόψιμο μεσαίου τμήματος:

  ──────●──────●──────           ──────●          ●──────
      τομή    τομή                (αριστερό)    (δεξί τμήμα)
                                  (μεσαίο τμήμα αφαιρέθηκε)
```

## Πώς καθορίζεται το τμήμα κοπής

Η εντολή προβάλλει τη θέση κέρσορα στη γραμμή κάτω από τον κέρσορα και βρίσκει όλα τα σημεία τομής που έχει η γραμμή με άλλες οντότητες. Αυτές οι παράμετροι τομής χωρίζουν τη γραμμή σε τμήματα. Το τμήμα του οποίου το διάστημα περιέχει την προβολή κέρσορα επισημαίνεται και αφαιρείται με κλικ.

- Αν ο κέρσορας είναι **πριν την πρώτη τομή**: αφαιρείται αυτό το αρχικό τμήμα της γραμμής.
- Αν ο κέρσορας είναι **μεταξύ δύο τομών**: αφαιρείται αυτό το μεσαίο τμήμα· η γραμμή χωρίζεται σε δύο.
- Αν ο κέρσορας είναι **μετά την τελευταία τομή**: αφαιρείται αυτό το τελευταίο τμήμα.
- Αν η γραμμή **δεν έχει τομές** με καμία άλλη οντότητα: δεν εμφανίζεται προεπισκόπηση και το κλικ δεν κάνει τίποτα.

## Αναφορά πληκτρολογίου

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `Escape` | Έξοδος από λειτουργία trim |

## Υποστηριζόμενες οντότητες

| Οντότητα | Μπορεί να κοπεί; |
|----------|----------------|
| Line | Ναι |
| Arc, Circle, Ellipse | Όχι |
| Polyline / Rectangle | Όχι |
| Text, Spline, Dimension, Leader | Όχι |

Οι οντότητες που χρησιμοποιούνται ως **όρια κοπής** μπορεί να είναι οποιουδήποτε τύπου — μόνο η γραμμή που κόβεται πρέπει να είναι οντότητα Line.

## Trim vs Extend

| | Trim | Extend |
|---|------|--------|
| Τι κάνει | Αφαιρεί τμήμα γραμμής | Τεντώνει τελικό σημείο γραμμής ως όριο |
| Ενεργοποίηση | Hover πάνω από τμήμα που θα κοπεί | Hover κοντά σε τελικό σημείο που θα επεκταθεί |
| Αποτέλεσμα | Η γραμμή χωρίζεται ή συντομεύεται | Το τελικό σημείο γραμμής μετακινείται στο όριο |
| Και τα δύο | Μόνο γραμμές | Μόνο γραμμές |
