# Delete

Η εντολή `delete` αφαιρεί επιλεγμένες οντότητες από το σχέδιο. Οι διαγραφές καταγράφονται στο ιστορικό [Undo](../undo/) και μπορούν να αναιρεθούν έως και 20 βήματα. Δεν υπάρχει ξεχωριστό παράθυρο "επιβεβαίωσης διαγραφής" — η επιβεβαίωση είναι ένα μόνο πάτημα πλήκτρου.

## Δύο τρόποι διαγραφής

**Προεπιλογή, μετά διαγραφή** — η ταχύτερη διαδρομή:

1. Επιλέξτε μία ή περισσότερες οντότητες στον καμβά.
2. Πληκτρολογήστε `delete` στο τερματικό, κάντε κλικ στο κουμπί **Delete** της γραμμής εργαλείων, **ή πατήστε το πλήκτρο `Delete`** απευθείας.

Οι οντότητες αφαιρούνται αμέσως — δεν απαιτείται επιπλέον βήμα επιβεβαίωσης.

**Ενεργοποίηση, μετά επιλογή**:

1. Πληκτρολογήστε `delete` ή κάντε κλικ στο κουμπί γραμμής εργαλείων (χωρίς τίποτα επιλεγμένο).
2. **Επιλέξτε αντικείμενα** — κλικ για εναλλαγή, ή σύρσιμο για επιλογή ανά περιοχή.
3. Πατήστε **Enter**, **Space** ή **Delete** για επιβεβαίωση και αφαίρεση των επιλεγμένων οντοτήτων.

## Συντόμευση πλήκτρου Delete

Το πλήκτρο `Delete` στο πληκτρολόγιο λειτουργεί ως **γενική συντόμευση** — αν υπάρχουν τρέχουσες επιλεγμένες οντότητες, το πάτημά του τις διαγράφει αμέσως, ακόμα και χωρίς άνοιγμα της εντολής Delete στο τερματικό. Αυτή είναι η ταχύτερη ροή εργασίας διαγραφής ενός βήματος:

```
Click entity → press Delete key → done
```

## Επιλογή κατά τη διάρκεια της εντολής

| Μέθοδος | Συμπεριφορά |
|---------|------------|
| **Κλικ** | Εναλλάσσει την οντότητα κάτω από τον κέρσορα εντός/εκτός επιλογής |
| **Σύρσιμο δεξιά** (αυστηρό) | Επιλέγει μόνο οντότητες πλήρως μέσα στο πλαίσιο |
| **Σύρσιμο αριστερά** (διασταυρούμενο) | Επιλέγει οντότητες που τέμνουν το όριο του πλαισίου |
| **Enter** / **Space** / **Delete** | Επιβεβαιώνει και διαγράφει επιλεγμένες οντότητες |

## Ανάκτηση διαγραμμένων οντοτήτων

Οι διαγραφές μπορούν να αναιρεθούν με την εντολή [Undo](../undo/) (πληκτρολογήστε `undo` ή χρησιμοποιήστε το κουμπί γραμμής εργαλείων). Έως **20 βήματα** μπορούν να αναιρεθούν ανά αρχείο, και το ιστορικό διατηρείται μεταξύ επαναφορτώσεων σελίδας. Αν έχετε υπερβεί 20 διαγραφές χωρίς αποθήκευση, οι παλαιότερες διαγραφές δεν μπορούν να ανακτηθούν.

## Υποστηριζόμενες οντότητες

Το Delete λειτουργεί σε κάθε τύπο οντότητας — Line, Polyline, Rectangle, Circle, Arc, Ellipse, Text, Spline, Dimension, Leader και όλες τις άλλες.
