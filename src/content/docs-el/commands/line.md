# Line

Η εντολή `line` σχεδιάζει μεμονωμένα ευθύγραμμα τμήματα αποθηκευμένα ως ξεχωριστές οντότητες `LINE` στο μοντέλο DXF. Μετά από κάθε τμήμα η εντολή παραμένει ενεργή και χρησιμοποιεί το τελικό σημείο ως νέο αρχικό σημείο, ώστε να μπορείτε να δημιουργείτε συνδεδεμένα μονοπάτια ένα τμήμα τη φορά. Σε αντίθεση με ένα [Polyline](./polyline), οι αλυσιδωμένες γραμμές παραμένουν ανεξάρτητα αντικείμενα — το καθένα μπορεί να κοπεί, να επεκταθεί ή να διαγραφεί χωρίς να επηρεάσει τις γειτονικές του.

## Σχεδίαση γραμμών

1. Πληκτρολογήστε `line` στο τερματικό ή κάντε κλικ στο κουμπί **Line** της γραμμής εργαλείων.
2. **Κάντε κλικ στο αρχικό σημείο**, ή πληκτρολογήστε `X,Y` και πατήστε **Enter** για ακριβή συντεταγμένη.
3. **Κάντε κλικ στο τελικό σημείο** — το τμήμα τοποθετείται και το τελικό σημείο γίνεται το νέο αρχικό σημείο. Η εισαγωγή συντεταγμένης λειτουργεί και εδώ.
4. Συνεχίστε να κάνετε κλικ (ή να πληκτρολογείτε) για να αλυσιδώσετε περισσότερα τμήματα.
5. Πατήστε **Enter** ή **Escape** για να σταματήσετε.

```
  ●──────────●──────────●──────────●
 start     2nd click  3rd click   Enter to finish
            (auto-becomes next start)
```

Χρειάζεστε μόνο ένα τμήμα; Πατήστε **Enter** ή **Escape** αμέσως μετά το βήμα 3.

## Εισαγωγή συντεταγμένης

Αντί για κλικ, πληκτρολογήστε ακριβή θέση για το αρχικό ή οποιοδήποτε επόμενο σημείο:

1. Πληκτρολογήστε την τιμή X (ψηφία, `.` ή `-`).
2. Πατήστε `,` — το τερματικό εμφανίζει `[X], [Y{cursor}]`.
3. Πληκτρολογήστε την τιμή Y.
4. Πατήστε **Enter** για τοποθέτηση του σημείου.

## Κλείδωμα γωνίας και εισαγωγή ακριβούς μήκους

Καθώς κινείτε τον κέρσορα μετά από τοποθέτηση σημείου, η εντολή παρακολουθεί για άξονα snap 45° (0°, 45°, 90°, 135°, …). Η γωνία **κλειδώνει** όταν:

- ο κέρσορας βρίσκεται τουλάχιστον **5 × μέγεθος grip** από την άγκυρα, **και**
- βρίσκεται εντός **1 μεγέθους grip** από κάθετη απόσταση από τον πλησιέστερο άξονα.

Όταν κλειδωθεί, η προεπισκόπηση snap στον άξονα και μπορείτε να εισάγετε ακριβές μήκος:

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `0`–`9`, `.` | Προσθέτει ψηφίο στην τιμή μήκους |
| `-` | Αρνητικό μήκος — αντιστρέφει κατεύθυνση κατά μήκος του άξονα (μόνο πρώτος χαρακτήρας) |
| `Backspace` | Διαγράφει τον τελευταίο πληκτρολογημένο χαρακτήρα |
| `Enter` | Τοποθετεί το τελικό σημείο στην πληκτρολογημένη απόσταση |

Η συσσωρευμένη τιμή εμφανίζεται ζωντανά στο τερματικό (π.χ. `click end point or enter length: 12.5`). Κάνοντας κλικ ενώ είναι κλειδωμένο, το κλικ προβάλλεται στον άξονα, ώστε το τελικό σημείο να βρίσκεται πάντα ακριβώς πάνω σε αυτόν.

Η επιστροφή κοντά στο σημείο άγκυρας απεμπλέκει το κλείδωμα.

## Αναφορά πληκτρολογίου

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `0`–`9`, `.`, `-` | Έναρξη εισαγωγής συντεταγμένης X, ή απόσταση όταν η γωνία είναι κλειδωμένη |
| `,` | Κλείδωμα X και μετάβαση στην εισαγωγή Y |
| `Backspace` | Διαγραφή τελευταίου πληκτρολογημένου χαρακτήρα |
| `Enter` | Επιβεβαίωση πληκτρολογημένης συντεταγμένης ή μήκους, ή ολοκλήρωση αλυσίδας αν δεν υπάρχει τίποτα πληκτρολογημένο |
| `Escape` | Ολοκλήρωση αλυσίδας και έξοδος |

## Επεξεργασία grip — επέκταση τελικών σημείων

Μια επιλεγμένη γραμμή εμφανίζει τρία grips:

| Grip | Θέση | Τι κάνει |
|------|------|----------|
| **Start** | Πρώτο τελικό σημείο | Σύρσιμο για αναθέση — το τέλος παραμένει σταθερό |
| **Midpoint** | Κέντρο της γραμμής | Ενεργοποιεί **Move** για ολόκληρη τη γραμμή |
| **End** | Δεύτερο τελικό σημείο | Σύρσιμο για αναθέση — η αρχή παραμένει σταθερή |

Η επέκταση ενός τελικού σημείου δεν επηρεάζει ποτέ το άλλο. Αυτό διαφέρει από την επεξεργασία grip του [Polyline](./polyline), όπου η μετακίνηση μιας κορυφής αναδιαμορφώνει ολόκληρο το μονοπάτι.

## Επιλογή γραμμών

| Μέθοδος | Συμπεριφορά |
|---------|------------|
| **Κλικ** | Επιλέγει τη γραμμή αν το κλικ βρίσκεται εντός απόστασης hit-test από το τμήμα |
| **Σύρσιμο δεξιά** (αυστηρό) | Η γραμμή επιλέγεται μόνο αν και τα δύο τελικά σημεία βρίσκονται μέσα στο πλαίσιο |
| **Σύρσιμο αριστερά** (διασταυρούμενο) | Η γραμμή επιλέγεται αν οποιοδήποτε τμήμα τέμνει το όριο του πλαισίου |

## Υποστηριζόμενες εντολές επεξεργασίας

Οι γραμμές είναι η **μόνη** οντότητα στην οποία λειτουργούν [Trim](./trim) και [Extend](./extend). Εφαρμόζονται επίσης όλες οι τυπικές εντολές μετασχηματισμού:

| Εντολή | Τι συμβαίνει σε μια γραμμή |
|--------|---------------------------|
| [Move](./move) | Μεταφράζει και τα δύο τελικά σημεία κατά την ίδια μετατόπιση |
| [Copy](./copy) | Δημιουργεί πανομοιότυπη γραμμή σε νέα θέση |
| [Rotate](./rotate) | Περιστρέφει και τα δύο τελικά σημεία γύρω από το επιλεγμένο βασικό σημείο |
| [Mirror](./mirror) | Αντικατοπτρίζει και τα δύο τελικά σημεία ως προς τον άξονα αντικατοπτρισμού |
| [Scale](./scale) | Κλιμακώνει και τα δύο τελικά σημεία ομοιόμορφα από το βασικό σημείο |
| [Offset](./offset) | Δημιουργεί παράλληλη γραμμή σε σταθερή κάθετη απόσταση |
| [Trim](./trim) | Κόβει τη γραμμή στις τομές — **μόνο γραμμές** |
| [Extend](./extend) | Επεκτείνει το πλησιέστερο τελικό σημείο μέχρι ένα όριο — **μόνο γραμμές** |
| [Delete](./delete) | Αφαιρεί τη γραμμή από το σχέδιο |

## Ιδιότητες

Όταν μια γραμμή είναι επιλεγμένη, το πάνελ ιδιοτήτων εμφανίζει κάθε πεδίο που φέρει η εγγραφή DXF `LINE`:

**Γενικά**

| Ιδιότητα | Προεπιλογή | Σημασία |
|----------|-----------|---------|
| Color | 256 (ByLayer) | Δείκτης χρώματος ACI |
| Layer | `0` | Ανάθεση επιπέδου |
| Linetype | ByLayer | Μοτίβο linetype ονόματος |
| Linetype Scale | 1 | Συντελεστής κλίμακας στο μοτίβο linetype |
| Thickness | 0 | Πάχος εξώθησης |

**Γεωμετρία**

| Ιδιότητα | Σημασία |
|----------|---------|
| Start X / Start Y | Συντεταγμένες πρώτου τελικού σημείου |
| End X / End Y | Συντεταγμένες δεύτερου τελικού σημείου |

Όλα τα πεδία είναι επεξεργάσιμα απευθείας στο πάνελ χωρίς επανεκτέλεση της εντολής.

## Line vs Polyline — πότε να χρησιμοποιείτε το καθένα

| | Line | Polyline |
|---|------|---------|
| Πλήθος αντικειμένων | Ένα `LINE` ανά τμήμα | Ένα `LWPOLYLINE` για ολόκληρο το μονοπάτι |
| Trim / Extend | Ναι — τμήμα-τμήμα | Όχι |
| Κλειστό σχήμα | Όχι | Ναι (σημαία κλεισίματος) |
| Επεξεργασία grip | Επέκταση μεμονωμένων τελικών σημείων | Μετακίνηση οποιασδήποτε κορυφής κατά μήκος του μονοπατιού |
| Καλύτερο για | Κατασκευαστικές γραμμές, μεμονωμένα τμήματα, γεωμετρία που θα κοπεί | Περιγράμματα, σχήματα που διατηρούνται ακέραια |

## DXF — οντότητα LINE

Οι γραμμές αποθηκεύονται ως οντότητες `LINE` στο αρχείο DXF. Κάθε ιδιότητα — συντεταγμένες αρχής/τέλους, χρώμα, επίπεδο, linetype, κλίμακα linetype και πάχος — ταξιδεύει χωρίς απώλεια. Όταν ανοίγετε ένα DXF που περιέχει οντότητες `LINE`, γίνονται πλήρως επεξεργάσιμα αντικείμενα `Line` στον επεξεργαστή.

Οι γραμμές που σχεδιάζονται στον επεξεργαστή γράφονται επίσης ως οντότητες `LINE` κατά την αποθήκευση, οπότε μπορούν να διαβαστούν από LibreCAD, FreeCAD και οποιαδήποτε άλλη εφαρμογή συμβατή με DXF.
