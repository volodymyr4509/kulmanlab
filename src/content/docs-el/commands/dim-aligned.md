# Dimension Aligned

Η εντολή `dimaligned` τοποθετεί διάσταση που μετράει την **πραγματική ευθύγραμμη απόσταση** μεταξύ δύο σημείων. Η γραμμή διάστασης τρέχει παράλληλα στη γραμμή που συνδέει τα δύο σημεία, οπότε μπορεί να βρίσκεται σε οποιαδήποτε γωνία. Αυτή είναι η κύρια διαφορά από το [Dimension Linear](../dim-linear/), που περιορίζεται σε οριζόντιο ή κατακόρυφο.

## Ανατομία ευθυγραμμισμένης διάστασης

```
     ●  p2
    /|
   / |  (ext line 2, perpendicular to dim line)
  /  |
 /←5.00→/
/  /
●  /  (ext line 1, perpendicular to dim line)
p1
```

- **Γραμμές επέκτασης** — κάθετες στη γραμμή διάστασης, σχεδιαζόμενες από κάθε μετρούμενο σημείο.
- **Γραμμή διάστασης** — παράλληλη στο p1→p2, μετατοπισμένη σε μία πλευρά από τη θέση κέρσορα.
- **Τιμή** — η πραγματική Ευκλείδεια απόσταση `|p1 – p2|`.

## Τοποθέτηση ευθυγραμμισμένης διάστασης

1. Πληκτρολογήστε `dimaligned` στο τερματικό ή κάντε κλικ στο κουμπί **Dimension Aligned** της γραμμής εργαλείων.
2. **Κάντε κλικ στην αφετηρία πρώτης γραμμής επέκτασης** (p1), ή πληκτρολογήστε `X,Y` και πατήστε **Enter** για ακριβή συντεταγμένη.
3. **Κάντε κλικ στην αφετηρία δεύτερης γραμμής επέκτασης** (p2). Η εισαγωγή συντεταγμένης λειτουργεί και εδώ.
4. **Μετακινήστε τον κέρσορα** σε μία πλευρά για ορισμό της κάθετης μετατόπισης της γραμμής διάστασης.
5. **Κάντε κλικ** για τοποθέτηση, ή πληκτρολογήστε απόσταση μετατόπισης και πατήστε **Enter** για ακριβή τοποθέτηση.

## Πληκτρολογημένη απόσταση μετατόπισης

Πληκτρολογήστε αριθμό κατά την τοποθέτηση για να σταθεροποιήσετε τη γραμμή διάστασης σε ακριβή κάθετη απόσταση από τη γραμμή p1→p2:

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `0`–`9`, `.` | Προσθέτει ψηφίο στη μετατόπιση |
| `Backspace` | Διαγράφει τον τελευταίο πληκτρολογημένο χαρακτήρα |
| `Enter` / `Space` | Τοποθέτηση στην πληκτρολογημένη μετατόπιση |

Η πλευρά του κέρσορα καθορίζει ποια πλευρά εμφανίζεται η γραμμή διάστασης.

## Αναφορά πληκτρολογίου

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `0`–`9`, `.`, `-` | Έναρξη εισαγωγής συντεταγμένης X (φάσεις p1/p2), ή απόσταση μετατόπισης (φάση τοποθέτησης) |
| `,` | Κλείδωμα X και μετάβαση στην εισαγωγή Y (φάσεις p1/p2) |
| `Backspace` | Διαγραφή τελευταίου πληκτρολογημένου χαρακτήρα |
| `Enter` / `Space` | Επιβεβαίωση πληκτρολογημένης συντεταγμένης ή μετατόπισης |
| `Escape` | Ακύρωση |

## Dimension Aligned vs Dimension Linear

| | Dimension Aligned | Dimension Linear |
|---|------------------|-----------------|
| Γωνία γραμμής διάστασης | Παράλληλη στο p1→p2 — οποιαδήποτε γωνία | Πάντα οριζόντια ή κατακόρυφη |
| Μετράει | Πραγματική Ευκλείδεια απόσταση | Μόνο συνιστώσα X ή Y |
| Κλείδωμα προσανατολισμού Ο/Κ | Όχι | Ναι — πλήκτρα `H` και `V` |
| Καλύτερο για | Διαγώνια χαρακτηριστικά, γωνιακές κοπές | Ορθογώνιες διατάξεις, εξαρτήματα ευθυγραμμισμένα με πλέγμα |

## Επεξεργασία ετικέτας — simple mode

**Διπλό κλικ** σε τοποθετημένη ευθυγραμμισμένη διάσταση για άνοιγμα του επεξεργαστή κειμένου σε **simple** mode. Ο επεξεργαστής είναι προ-συμπληρωμένος με την τρέχουσα αποδοσμένη τιμή ώστε να μπορείτε να τοποθετήσετε τον κέρσορα και να την επεξεργαστείτε απευθείας.

| Λειτουργία | Συμπεριφορά |
|-----------|------------|
| Bold / Italic / Font / Height | Εφαρμόζεται σε **ολόκληρη** την ετικέτα ταυτόχρονα |
| Μορφοποίηση ανά χαρακτήρα | Δεν υποστηρίζεται |
| `Enter` | Επικυρώνει την τιμή και κλείνει τον επεξεργαστή |
| Πολυγραμμικό | Δεν υποστηρίζεται |

Δείτε [Text Editor — simple mode](../../interface/text-editor/#simple-mode) για την πλήρη αναφορά.

## Αλυσίδα διαστάσεων

Για προσθήκη περισσότερων διαστάσεων που συνεχίζουν από τη δεύτερη γραμμή επέκτασης αυτής, χρησιμοποιήστε [Dimension Continue](../dim-continue/) — κλειδώνει στην ίδια γωνία μέτρησης με αυτή την ευθυγραμμισμένη διάσταση.

## DXF — οντότητα DIMENSION (τύπος aligned)

Οι ευθυγραμμισμένες διαστάσεις αποθηκεύονται ως οντότητες `DIMENSION` με `dimType = 1` (aligned). Αφετηρίες γραμμών επέκτασης, θέση γραμμής διάστασης, θέση κειμένου, μετρούμενη τιμή, στροφή, στυλ βέλους και όλες οι σημαίες εμφάνισης ταξιδεύουν χωρίς απώλεια.
