# Spline Fit

Η εντολή `splinefit` σχεδιάζει κυβική spline που διέρχεται από κάθε σημείο που κάνετε κλικ — παρεμβολική καμπύλη. Σε αντίθεση με [Spline CV](../spline-cv/), όπου η καμπύλη μόνο έλκεται προς τις κορυφές ελέγχου, εδώ η καμπύλη αναγκάζεται να αγγίξει ακριβώς κάθε πληκτρολογημένη συντεταγμένη. Εσωτερικά ο επεξεργαστής προσαρμόζει κορυφές ελέγχου για να επιτύχει αυτό, και αυτές οι CVs αποθηκεύονται μαζί με τα σημεία fit στο αρχείο DXF.

## Σχεδίαση spline μέσα από σημεία fit

1. Πληκτρολογήστε `splinefit` στο τερματικό ή κάντε κλικ στο κουμπί **Spline Fit** της γραμμής εργαλείων.
2. **Κάντε κλικ για τοποθέτηση σημείων fit** — η καμπύλη θα διέρχεται από κάθε ένα. Ή πληκτρολογήστε `X,Y` και πατήστε **Enter** για ακριβή συντεταγμένη.
3. Πατήστε **Enter** για ολοκλήρωση (απαιτούνται τουλάχιστον 2 σημεία).

```
  ●──────●──────●──────●  ← η καμπύλη διέρχεται ακριβώς από κάθε κλικ
  p1     p2     p3     p4
```

Η ζωντανή προεπισκόπηση δείχνει την τρέχουσα παρεμβολική καμπύλη καθώς μετακινείτε τον κέρσορα, συμπεριλαμβανομένου του πιθανού επόμενου σημείου στη θέση κέρσορα. Πατήστε **Escape** για απόρριψη όλων των τοποθετημένων σημείων και έξοδο.

## Εισαγωγή συντεταγμένης

Αντί για κλικ, πληκτρολογήστε ακριβή θέση για οποιοδήποτε σημείο fit:

1. Πληκτρολογήστε την τιμή X.
2. Πατήστε `,` — το τερματικό εμφανίζει `[X], [Y{cursor}]`.
3. Πληκτρολογήστε την τιμή Y.
4. Πατήστε **Enter** για τοποθέτηση σημείου fit.

## Αναφορά πληκτρολογίου

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `0`–`9`, `.`, `-` | Έναρξη εισαγωγής συντεταγμένης X |
| `,` | Κλείδωμα X και μετάβαση στην εισαγωγή Y |
| `Backspace` | Διαγραφή τελευταίου πληκτρολογημένου χαρακτήρα |
| `Enter` | Επιβεβαίωση πληκτρολογημένης συντεταγμένης, ή ολοκλήρωση spline αν δεν υπάρχει εισαγωγή σε εξέλιξη και ≥ 2 σημεία υπάρχουν |
| `Escape` | Απόρριψη όλων των σημείων και έξοδος |

## Επεξεργασία grip — αναδιαμόρφωση μέσω σημείων fit

Μια επιλεγμένη fit spline εμφανίζει ένα grip ανά σημείο fit:

| Grip | Θέση | Τι κάνει |
|------|------|----------|
| **Σημείο fit** | Σε κάθε θέση κλικ | Σύρσιμο για μετακίνηση αυτού του σημείου fit — ολόκληρη η καμπύλη επαναπαρεμβάλλεται για να διέλθει από τη νέα θέση |

Σύρσιμο ενός grip επαναπροσαρμόζει ολόκληρη την καμπύλη, όχι μόνο τα παρακείμενα τμήματα. Αυτό διαφέρει από την επεξεργασία grip polyline, όπου η μετακίνηση κορυφής αναδιαμορφώνει μόνο τα δύο παρακείμενα τμήματα.

Δεν υπάρχει grip "μεταφορά ολόκληρης spline". Για μετατόπιση ολόκληρης της spline χρησιμοποιήστε την εντολή [Move](../move/).

## Επιλογή fit splines

| Μέθοδος | Συμπεριφορά |
|---------|------------|
| **Κλικ** | Επιλέγει αν το κλικ βρίσκεται κοντά σε οποιοδήποτε σημείο της καμπύλης |
| **Σύρσιμο δεξιά** (αυστηρό) | Όλα τα δείγματα σημεία κατά μήκος της καμπύλης πρέπει να βρίσκονται εντός του πλαισίου επιλογής |
| **Σύρσιμο αριστερά** (διασταυρούμενο) | Οποιοδήποτε τμήμα καμπύλης που διασταυρώνει το όριο πλαισίου επιλογής την επιλέγει |

## Υποστηριζόμενες εντολές επεξεργασίας

| Εντολή | Τι συμβαίνει στη spline |
|--------|------------------------|
| [Move](../move/) | Μεταφράζει όλα τα σημεία fit και επανυπολογισμένες CVs κατά την ίδια μετατόπιση |
| [Copy](../copy/) | Δημιουργεί πανομοιότυπη spline σε νέα θέση |
| [Rotate](../rotate/) | Περιστρέφει όλα τα σημεία fit γύρω από το επιλεγμένο βασικό σημείο |
| [Mirror](../mirror/) | Αντικατοπτρίζει όλα τα σημεία fit ως προς τον άξονα αντικατοπτρισμού |
| [Scale](../scale/) | Κλιμακώνει όλα τα σημεία fit ομοιόμορφα από το βασικό σημείο |
| [Delete](../delete/) | Αφαιρεί τη spline |

Οι splines δεν υποστηρίζουν **Offset**, **Trim**, ή **Extend**.

## Ιδιότητες

**Γενικά**

| Ιδιότητα | Προεπιλογή | Σημασία |
|----------|-----------|---------|
| Color | 256 (ByLayer) | Δείκτης χρώματος ACI |
| Layer | `0` | Ανάθεση επιπέδου |
| Linetype | ByLayer | Μοτίβο linetype ονόματος |
| Linetype Scale | 1 | Συντελεστής κλίμακας στο μοτίβο linetype |
| Thickness | 0 | Πάχος εξώθησης |

**Γεωμετρία**

| Ιδιότητα | Σημασία |
|----------|---------|
| Degree | Βαθμός πολυωνύμου — πάντα 3 (κυβική) |
| Fit Points | Συντεταγμένες όλων των σημείων pass-through που κλικαρίστηκαν |
| Control Vertices | Εσωτερικά υπολογισμένες CVs για απόδοση καμπύλης |

## Spline Fit vs Spline CV — ποια να χρησιμοποιήσετε

| | Spline Fit | Spline CV |
|---|------------|-----------|
| Η καμπύλη διέρχεται από σημεία | Κάθε σημείο κλικ ακριβώς | Μόνο πρώτο και τελευταίο (clamped) |
| Αποτέλεσμα επεξεργασίας grip | Σημείο fit μετακινείται → ολόκληρη η καμπύλη επαναπαρεμβάλλεται | CV μετακινείται → καμπύλη έλκεται προς νέα θέση |
| Προβλεψιμότητα σχήματος | Υψηλή — η καμπύλη ακολουθεί κλικ | Χαμηλότερη — η καμπύλη υστερεί πίσω από CVs |
| Καλύτερο για | Καμπύλες που πρέπει να αγγίξουν συγκεκριμένες συντεταγμένες | Ομαλές αισθητικές καμπύλες, ελεύθερες διαδρομές |

## DXF — οντότητα SPLINE (μορφή σημείων fit)

Οι fit splines αποθηκεύονται ως οντότητες `SPLINE` στο αρχείο DXF, αποθηκεύοντας τόσο τις συντεταγμένες σημείων fit όσο και τις υπολογισμένες κορυφές ελέγχου. Το `splineFlag` ορίζεται σε `8` (fit-point spline) ώστε η εφαρμογή επαναφόρτωσης να ξέρει ποιο σύνολο σημείων να εμφανίσει ως επεξεργάσιμα grips. Όλες οι ιδιότητες — χρώμα, επίπεδο, linetype, κλίμακα linetype και πάχος — ταξιδεύουν χωρίς απώλεια. Εφαρμογές DXF που υποστηρίζουν fit-point splines (LibreCAD, FreeCAD) θα εμφανίσουν τα σημεία fit ως τα κύρια επεξεργάσιμα δεδομένα.
