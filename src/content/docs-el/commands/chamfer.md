# Chamfer

Η εντολή `chamfer` κόβει ευθεία διαγώνια γωνία μεταξύ δύο οντοτήτων [Line](../line/) ή [Polyline](../polyline/). Ορίζετε πόσο πίσω να κόψετε κατά μήκος κάθε οντότητας (d1 και d2), και η εντολή κόβει και τις δύο οντότητες στα σημεία αυτά και εισάγει μια συνδετική γραμμή μεταξύ τους.

Η χρήση ίσων αποστάσεων παράγει συμμετρική κοπή 45°· διαφορετικές αποστάσεις παράγουν ασύμμετρο bevel.

Το Chamfer λειτουργεί σε οντότητες **Line και Polyline**.

## Χρήση του chamfer

1. Πληκτρολογήστε `chamfer` στο τερματικό ή κάντε κλικ στο κουμπί **Chamfer** της γραμμής εργαλείων.
2. **Πληκτρολογήστε την πρώτη απόσταση chamfer** (d1 — απόσταση κατά μήκος της πρώτης οντότητας) και πατήστε **Enter**.
3. **Πληκτρολογήστε τη δεύτερη απόσταση chamfer** (d2 — απόσταση κατά μήκος της δεύτερης οντότητας) και πατήστε **Enter**.
4. **Κάντε κλικ στην πρώτη οντότητα** — το τμήμα που κλικάρετε καθορίζει ποια πλευρά οποιασδήποτε τομής διατηρείται.
5. **Περνήστε τον κέρσορα πάνω από τη δεύτερη οντότητα** — μια διακεκομμένη γραμμή προεπισκόπησης δείχνει την προκύπτουσα κοπή chamfer. Μετακινήστε τον κέρσορα στην πλευρά που θέλετε να διατηρήσετε.
6. **Κάντε κλικ** για εφαρμογή. Και οι δύο οντότητες κόβονται και η γραμμή chamfer εισάγεται.

```
  Before (d1=5, d2=8):        After:

  ──────────────              ──────────╲
                │                        ╲────
                │
```

## Επιλογή πλευράς

Όταν δύο γραμμές τέμνονται, το chamfer εφαρμόζεται στη γωνία που ορίζεται από τις θέσεις κλικ — το τμήμα κάθε οντότητας στην **ίδια πλευρά με τον κέρσορα** διατηρείται.

- Κάντε κλικ κοντά σε ένα άκρο της πρώτης οντότητας για να επιλέξετε εκείνο το μισό.
- Μετακινήστε τον κέρσορα στο επιθυμητό μισό της δεύτερης οντότητας — η διακεκομμένη προεπισκόπηση ενημερώνεται ζωντανά.

Για Polylines, η θέση κλικ καθορίζει ποιο **τμήμα** της polyline συμμετέχει, και η πλησιέστερη κορυφή στην πλευρά τομής είναι αυτή που κόβεται.

## Τι δημιουργεί η εντολή

- Το τελικό σημείο (ή κορυφή polyline) της πρώτης οντότητας πλησιέστερα στην τομή μετακινείται στο σημείο **T1**, που βρίσκεται d1 κατά μήκος της πρώτης οντότητας από την τομή.
- Το τελικό σημείο (ή κορυφή polyline) της δεύτερης οντότητας πλησιέστερα στην τομή μετακινείται στο σημείο **T2**, που βρίσκεται d2 κατά μήκος της δεύτερης οντότητας από την τομή.
- Μια νέα οντότητα Line εισάγεται από το **T1** στο **T2**.

Η εισαχθείσα γραμμή κληρονομεί τις τρέχουσες ρυθμίσεις lineweight, χρώματος, επιπέδου και linetype.

## Αναφορά πληκτρολογίου

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `0`–`9`, `.` | Προσθέτει ψηφίο στην τρέχουσα τιμή απόστασης |
| `Backspace` | Διαγράφει τον τελευταίο πληκτρολογημένο χαρακτήρα |
| `Enter` | Επιβεβαίωση πληκτρολογημένης απόστασης και προχώρηση |
| `Escape` | Ακύρωση και επαναφορά |

## Υποστηριζόμενες οντότητες

| Οντότητα | Υποστηρίζεται |
|----------|-------------|
| Line | Ναι |
| Polyline / Rectangle | Ναι |
| Arc, Circle, Ellipse | Όχι |
| Text, Spline, Dimension, Leader | Όχι |

## Chamfer vs Fillet

| | Chamfer | Fillet |
|---|---------|--------|
| Τύπος γωνίας | Ευθεία κοπή | Στρογγυλό τόξο |
| Είσοδος | Δύο αποστάσεις (d1, d2) | Μία ακτίνα |
| Εισαχθείσα οντότητα | Line | Arc |
| Υποστηριζόμενες οντότητες | Lines και Polylines | Μόνο Lines |
