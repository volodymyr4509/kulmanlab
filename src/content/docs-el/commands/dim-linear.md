# Dimension Linear

Η εντολή `dimlinear` τοποθετεί οριζόντια ή κατακόρυφη διάσταση μεταξύ δύο αφετηριών γραμμής επέκτασης. Η γραμμή διάστασης τρέχει πάντα τέλεια οριζόντια ή τέλεια κατακόρυφα — δεν μπορεί να τοποθετηθεί σε αυθαίρετη γωνία. Χρησιμοποιήστε [Dimension Aligned](../dim-aligned/) όταν χρειάζεστε διάσταση παράλληλη σε διαγώνια γραμμή.

## Ανατομία linear διάστασης

```
  |←————— 5.00 —————→|
  |                   |
  ●  (ext line 1)     ●  (ext line 2)
  p1                  p2
```

- **Γραμμές επέκτασης** — κατεβαίνουν από κάθε μετρούμενο σημείο κάθετα στη γραμμή διάστασης.
- **Γραμμή διάστασης** — οριζόντια (μετράει απόσταση X) ή κατακόρυφη (μετράει απόσταση Y).
- **Τιμή** — η προβαλλόμενη απόσταση κατά τον επιλεγμένο άξονα, όχι η πραγματική απόσταση μεταξύ σημείων.

## Τοποθέτηση linear διάστασης

1. Πληκτρολογήστε `dimlinear` στο τερματικό ή κάντε κλικ στο κουμπί **Dimension Linear** της γραμμής εργαλείων.
2. **Κάντε κλικ στην αφετηρία πρώτης γραμμής επέκτασης** (p1), ή πληκτρολογήστε `X,Y` και πατήστε **Enter** για ακριβή συντεταγμένη.
3. **Κάντε κλικ στην αφετηρία δεύτερης γραμμής επέκτασης** (p2). Η εισαγωγή συντεταγμένης λειτουργεί και εδώ.
4. **Μετακινήστε τον κέρσορα** για τοποθέτηση γραμμής διάστασης. Ο προσανατολισμός ανιχνεύεται αυτόματα από τη θέση κέρσορα.
5. **Κάντε κλικ** για τοποθέτηση, ή πληκτρολογήστε απόσταση μετατόπισης και πατήστε **Enter** για ακριβή τοποθέτηση.

## Αυτόματη ανίχνευση προσανατολισμού

Όταν δεν επιβάλλεται προσανατολισμός, η εντολή διαβάζει τη θέση κέρσορα σε σχέση με τα δύο μετρούμενα σημεία:

| Θέση κέρσορα | Ανιχνευόμενος προσανατολισμός | Τι μετράται |
|-------------|------------------------------|------------|
| Πάνω ή κάτω από τα σημεία | Οριζόντιο | Δ X μεταξύ p1 και p2 |
| Αριστερά ή δεξιά από τα σημεία | Κατακόρυφο | Δ Y μεταξύ p1 και p2 |

Πατήστε **H** για κλείδωμα οριζόντιου ή **V** για κλείδωμα κατακόρυφου ανά πάσα στιγμή κατά τη φάση τοποθέτησης. Μόλις κλειδωθεί, ο προσανατολισμός δεν αλλάζει καθώς μετακινείτε τον κέρσορα.

## Πληκτρολογημένη απόσταση μετατόπισης

Πληκτρολογήστε αριθμό κατά την τοποθέτηση για να σταθεροποιήσετε τη γραμμή διάστασης σε ακριβή απόσταση από τα μετρούμενα σημεία:

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `0`–`9`, `.` | Προσθέτει ψηφίο στην απόσταση μετατόπισης |
| `Backspace` | Διαγράφει τον τελευταίο πληκτρολογημένο χαρακτήρα |
| `Enter` / `Space` | Τοποθέτηση στην πληκτρολογημένη απόσταση |

Η πλευρά κέρσορα (πάνω/κάτω για οριζόντιο, αριστερά/δεξιά για κατακόρυφο) καθορίζει το πρόσημο — η γραμμή διάστασης εμφανίζεται στην πλευρά που βρίσκεται τρέχοντας ο κέρσορας.

## Αναφορά πληκτρολογίου

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `0`–`9`, `.`, `-` | Έναρξη εισαγωγής συντεταγμένης X (φάσεις p1/p2), ή απόσταση μετατόπισης (φάση τοποθέτησης) |
| `,` | Κλείδωμα X και μετάβαση στην εισαγωγή Y (φάσεις p1/p2) |
| `H` | Κλείδωμα οριζόντιου προσανατολισμού (μόνο φάση τοποθέτησης) |
| `V` | Κλείδωμα κατακόρυφου προσανατολισμού (μόνο φάση τοποθέτησης) |
| `Backspace` | Διαγραφή τελευταίου πληκτρολογημένου χαρακτήρα |
| `Enter` / `Space` | Επιβεβαίωση πληκτρολογημένης συντεταγμένης ή μετατόπισης |
| `Escape` | Ακύρωση |

## Dimension Linear vs Dimension Aligned

| | Dimension Linear | Dimension Aligned |
|---|-----------------|------------------|
| Άξονας | Πάντα Ο ή Κ | Παράλληλος στη μετρούμενη γραμμή |
| Μετράει | Μόνο συνιστώσα X ή Y | Πραγματική Ευκλείδεια απόσταση |
| Πλήκτρα Ο/Κ | Ναι — κλείδωμα προσανατολισμού | Όχι — πάντα ακολουθεί p1→p2 |
| Καλύτερο για | Ορθογώνιες διατάξεις, κατόψεις | Διαγώνια χαρακτηριστικά, γωνιακές κοπές |

## Επεξεργασία ετικέτας — simple mode

**Διπλό κλικ** σε τοποθετημένη linear διάσταση για άνοιγμα του επεξεργαστή κειμένου σε **simple** mode. Ο επεξεργαστής είναι προ-συμπληρωμένος με την τρέχουσα αποδοσμένη τιμή ώστε να μπορείτε να τοποθετήσετε τον κέρσορα και να την επεξεργαστείτε απευθείας.

| Λειτουργία | Συμπεριφορά |
|-----------|------------|
| Bold / Italic / Font / Height | Εφαρμόζεται σε **ολόκληρη** την ετικέτα ταυτόχρονα |
| Μορφοποίηση ανά χαρακτήρα | Δεν υποστηρίζεται |
| `Enter` | Επικυρώνει την τιμή και κλείνει τον επεξεργαστή |
| Πολυγραμμικό | Δεν υποστηρίζεται |

Δείτε [Text Editor — simple mode](../../interface/text-editor/#simple-mode) για την πλήρη αναφορά.

## Αλυσίδα διαστάσεων

Για προσθήκη περισσότερων διαστάσεων που συνεχίζουν από την τελευταία γραμμή επέκτασης, χρησιμοποιήστε [Dimension Continue](../dim-continue/) αμέσως μετά την τοποθέτηση αυτής.

## DXF — οντότητα DIMENSION

Οι linear διαστάσεις αποθηκεύονται ως οντότητες `DIMENSION` με `rotationDeg` ορισμένο σε `0` (οριζόντιο) ή `90` (κατακόρυφο). Αφετηρίες γραμμών επέκτασης, θέση γραμμής διάστασης, θέση κειμένου, μετρούμενη τιμή, στυλ βέλους, ύψος κειμένου και όλες οι σημαίες εμφάνισης ταξιδεύουν χωρίς απώλεια.
