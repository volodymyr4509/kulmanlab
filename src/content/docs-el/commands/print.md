# Print

Η εντολή `print` ανοίγει τον **Print Manager** — ειδικό παράθυρο εξαγωγής με ζωντανό καμβά προεπισκόπησης, επιλογέα μορφής (PNG / JPEG / WebP / PDF), εναλλαγή ασπρόμαυρου και προαιρετική περικοπή περιοχής. Τίποτα δεν αποστέλλεται σε φυσικό εκτυπωτή· η έξοδος λαμβάνεται ως αρχείο.

## Άνοιγμα του Print Manager

Κάντε κλικ στο κουμπί **Print** της γραμμής εργαλείων ή πληκτρολογήστε `print` στο τερματικό. Ο Print Manager ανοίγει αμέσως εμφανίζοντας προεπισκόπηση του τρέχοντος viewport.

## Διάταξη Print Manager

Το παράθυρο έχει δύο πάνελ:
- **Αριστερή πλαϊνή στήλη** — όλα τα controls εξαγωγής.
- **Δεξί πάνελ** — ζωντανός καμβάς προεπισκόπησης που ενημερώνεται καθώς αλλάζετε ρυθμίσεις.

### Controls πλαϊνής στήλης

| Control | Περιγραφή |
|---------|-----------|
| **Change Area** | Περικοπή σε προσαρμοσμένο ορθογώνιο στον καμβά (βλ. παρακάτω) |
| Εναλλαγή **Monochrome** | Μετατροπή όλων των έγχρωμων γραμμών σε μαύρο — ενεργό από προεπιλογή για καθαρή εκτύπωση |
| Αναπτυσσόμενο **Format** | PNG, JPEG, WebP ή PDF |
| Κουμπί **Export** | Δημιουργία και λήψη αρχείου |

## Επιλογή προσαρμοσμένης περιοχής εξαγωγής

Από προεπιλογή η προεπισκόπηση εμφανίζει ακριβώς ό,τι ήταν ορατό στον καμβά όταν ανοίξατε τον Print Manager. Για εξαγωγή συγκεκριμένης περιοχής:

1. Κάντε κλικ στο **Change Area** — ο Print Manager κρύβεται και ο καμβάς γίνεται διαδραστικός.
2. **Κάντε κλικ στην πρώτη γωνία** του ορθογωνίου εξαγωγής.
3. **Κάντε κλικ στην απέναντι γωνία** — ο Print Manager ανοίγει ξανά με την επιλεγμένη περιοχή στην προεπισκόπηση.

Πατήστε `Escape` κατά την επιλογή περιοχής για ακύρωση και επαναφορά της προηγούμενης περιοχής.

Ο καμβάς προεπισκόπησης αλλάζει μέγεθος δυναμικά ώστε να ταιριάζει με **ακριβώς την αναλογία διαστάσεων** της επιλεγμένης περιοχής, οπότε η προεπισκόπηση είναι pixel-ακριβής.

## Μορφές εξαγωγής

| Μορφή | Καλύτερο για | Σημειώσεις |
|-------|-------------|-----------|
| **PNG** | Χωρίς απώλειες, ευκρινείς γραμμές | Λευκό φόντο, χωρίς διαφάνεια |
| **JPEG** | Μικρότερο αρχείο για κοινοποίηση | Ποιότητα 95%, ελαφρά συμπίεση |
| **WebP** | Μικρότερο αρχείο για web | Ίδια ποιότητα 95%, καλύτερη συμπίεση από JPEG |
| **PDF** | Έτοιμα για εκτύπωση έγγραφα | Εικόνα ενσωματωμένη σε 150 DPI μέσα σε PDF container |

Το εξαγόμενο αρχείο ονομάζεται `kulman-<timestamp>.<ext>` και γίνεται λήψη αυτόματα.

## Ανάλυση εξαγωγής και φόντο

- Μέγιστη ανάλυση: **2000 × 2000 pixels**, κλιμακωμένη αναλογικά στην επιλεγμένη περιοχή.
- Το φόντο είναι πάντα **λευκό**.
- Τα επίπεδα που είναι σημειωμένα ως **non-plotting** εξαιρούνται από την εξαγωγή.

## Αναφορά πληκτρολογίου

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `Escape` (κατά την επιλογή περιοχής) | Ακύρωση επιλογής περιοχής, επαναφορά προηγούμενης περιοχής |
| `Escape` (στον Print Manager) | Κλείσιμο Print Manager |
