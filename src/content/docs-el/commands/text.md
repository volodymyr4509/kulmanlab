# Text

Η εντολή `text` τοποθετεί πολυγραμμική ετικέτα κειμένου. Αφού κάνετε κλικ σε θέση στον καμβά, ανοίγει αναδυόμενος επεξεργαστής σε **rich** mode — μπορείτε να πληκτρολογήσετε περιεχόμενο, να εφαρμόσετε έντονη/πλάγια/διαγραμμισμένη γραφή ανά χαρακτήρα, να αλλάξετε γραμματοσειρές και ύψη, και να εισαγάγετε αλλαγές γραμμής. Πατήστε **Escape** για αποθήκευση και κλείσιμο του επεξεργαστή.

Ανατρέξτε στη σελίδα [Text Editor](../interface/text-editor) για την πλήρη αναφορά επεξεργαστή, συμπεριλαμβανομένης σύγκρισης των **rich** και **simple** modes.

## Τοποθέτηση ετικέτας κειμένου

1. Πληκτρολογήστε `text` στο τερματικό ή κάντε κλικ στο κουμπί **Text** της γραμμής εργαλείων.
2. **Κάντε κλικ στη θέση άγκυρας** στον καμβά. Ή πληκτρολογήστε `X,Y` και πατήστε **Enter** για ακριβή συντεταγμένη.
3. Ανοίγει το **αναδυόμενο παράθυρο επεξεργαστή κειμένου** πάνω από τη νέα ετικέτα. Πληκτρολογήστε το περιεχόμενό σας.
4. Πατήστε **Escape** για αποθήκευση της ετικέτας και κλείσιμο του επεξεργαστή.

Το προεπιλεγμένο ύψος είναι **12 μονάδες σχεδίασης**.

## Επεξεργασία υπάρχουσας ετικέτας

**Διπλό κλικ** σε οποιαδήποτε ετικέτα κειμένου στον καμβά για επαναφορά του επεξεργαστή για αυτήν την ετικέτα.

## Εισαγωγή συντεταγμένης για άγκυρα

Αντί για κλικ, πληκτρολογήστε ακριβή θέση:

1. Πληκτρολογήστε την τιμή X.
2. Πατήστε `,` — το τερματικό εμφανίζει `[X], [Y{cursor}]`.
3. Πληκτρολογήστε την τιμή Y.
4. Πατήστε **Enter** για τοποθέτηση άγκυρας και άνοιγμα επεξεργαστή.

## Αναφορά πληκτρολογίου

**Φάση άγκυρας**

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `0`–`9`, `.`, `-` | Έναρξη εισαγωγής συντεταγμένης X |
| `,` | Κλείδωμα X και μετάβαση στην εισαγωγή Y |
| `Backspace` | Διαγραφή τελευταίου πληκτρολογημένου χαρακτήρα |
| `Enter` | Επιβεβαίωση πληκτρολογημένης συντεταγμένης |

**Φάση επεξεργαστή κειμένου** (βλ. [Text Editor](../interface/text-editor) για πλήρη αναφορά)

| Πλήκτρο | Ενέργεια |
|---------|----------|
| Οποιοσδήποτε εκτυπώσιμος χαρακτήρας | Εισαγωγή στον κέρσορα |
| `Backspace` / `Delete` | Διαγραφή παρακείμενου χαρακτήρα ή επιλογής |
| `Enter` | Εισαγωγή αλλαγής γραμμής |
| `←` / `→` | Μετακίνηση δρομέα |
| `Home` / `End` | Μεταφορά στην αρχή / τέλος της κύριας γραμμής |
| `Escape` | Αποθήκευση και κλείσιμο επεξεργαστή |

## Επεξεργασία grip — επανατοποθέτηση

Μια επιλεγμένη ετικέτα κειμένου εμφανίζει ένα grip στο σημείο άγκυρας:

| Grip | Θέση | Τι κάνει |
|------|------|----------|
| **Άγκυρα** | Κάτω-αριστερά του κειμένου | Σύρσιμο για επανατοποθέτηση ετικέτας |

## Επιλογή κειμένου

| Μέθοδος | Συμπεριφορά |
|---------|------------|
| **Κλικ** | Επιλέγει αν το κλικ βρίσκεται εντός του περιστρεφόμενου πλαισίου οριοθέτησης του κειμένου |
| **Σύρσιμο δεξιά** (αυστηρό) | Και οι τέσσερις γωνίες του πλαισίου οριοθέτησης πρέπει να βρίσκονται εντός της περιοχής επιλογής |
| **Σύρσιμο αριστερά** (διασταυρούμενο) | Οποιαδήποτε επικάλυψη μεταξύ πλαισίου οριοθέτησης κειμένου και περιοχής επιλογής το επιλέγει |

## Υποστηριζόμενες εντολές επεξεργασίας

| Εντολή | Τι συμβαίνει στο κείμενο |
|--------|--------------------------|
| [Move](./move) | Μετακινεί το σημείο άγκυρας |
| [Copy](./copy) | Δημιουργεί πανομοιότυπη ετικέτα σε νέα θέση |
| [Rotate](./rotate) | Περιστρέφει τη θέση άγκυρας και προσθέτει τη γωνία στην ιδιότητα Rotation Degree |
| [Mirror](./mirror) | Αντικατοπτρίζει το σημείο άγκυρας ως προς τον άξονα αντικατοπτρισμού (η συμβολοσειρά κειμένου δεν αναστρέφεται) |
| [Scale](./scale) | Κλιμακώνει τη θέση άγκυρας και πολλαπλασιάζει το ύψος με τον συντελεστή κλίμακας |
| [Delete](./delete) | Αφαιρεί την ετικέτα |

Το Text δεν υποστηρίζει **Offset**, **Trim**, ή **Extend**.

## Ιδιότητες

Όταν μια ετικέτα κειμένου είναι επιλεγμένη, το πάνελ ιδιοτήτων εμφανίζει:

**Γενικά**

| Ιδιότητα | Προεπιλογή | Σημασία |
|----------|-----------|---------|
| Color | 256 (ByLayer) | Δείκτης χρώματος ACI |
| Layer | `0` | Ανάθεση επιπέδου |

**Γεωμετρία**

| Ιδιότητα | Σημασία |
|----------|---------|
| Position X / Position Y | Συντεταγμένες σημείου άγκυρας |
| Height | Βασικό ύψος κειμένου σε μονάδες σχεδίασης (προεπιλογή: **12**) |
| Rotation Degree | Περιστροφή αντίθετα με τη φορά ρολογιού σε μοίρες |

**Ιδιότητες**

| Ιδιότητα | Σημασία |
|----------|---------|
| Content | Η συμβολοσειρά κειμένου (κωδικοί MTEXT inline διατηρούνται) |
| Attachment Point | Κωδικός στοίχισης (1 = πάνω-αριστερά … 9 = κάτω-δεξιά) |

Το Text δεν έχει ιδιότητες Linetype, Linetype Scale ή Thickness.

## DXF — οντότητα MTEXT

Οι ετικέτες κειμένου αποθηκεύονται ως οντότητες **MTEXT** στο αρχείο DXF. Η έντονη και πλάγια γραφή κωδικοποιείται χρησιμοποιώντας `\L`, `\K`, `\O`, και ενσωματωμένες εναλλαγές γραμματοσειράς (`\f`). Το ύψος ανά χαρακτήρα κωδικοποιείται ως `\H`. Όλη η μορφοποίηση επιβιώνει πλήρους DXF round-trip και είναι αναγνώσιμη από LibreCAD, FreeCAD και άλλες εφαρμογές συμβατές με DXF.
