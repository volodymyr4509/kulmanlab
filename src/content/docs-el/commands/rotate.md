# Rotate

Η εντολή `rotate` περιστρέφει επιλεγμένες οντότητες γύρω από ένα βασικό σημείο. Ορίζετε τη γωνία περιστροφής είτε πληκτρολογώντας αριθμό σε μοίρες είτε με κλικ — η γωνία υπολογίζεται από την κατεύθυνση μεταξύ του βασικού σημείου και της θέσης κλικ.

## Δύο τρόποι εκκίνησης

**Προεπιλογή, μετά περιστροφή** — επιλέξτε πρώτα οντότητες, μετά ενεργοποιήστε:

1. Επιλέξτε μία ή περισσότερες οντότητες στον καμβά.
2. Πληκτρολογήστε `rotate` στο τερματικό ή κάντε κλικ στο κουμπί **Rotate** της γραμμής εργαλείων.
3. **Κάντε κλικ στο βασικό σημείο** — το κέντρο περιστροφής. Ή πληκτρολογήστε `X,Y` και πατήστε **Enter** για ακριβή συντεταγμένη.
4. **Πληκτρολογήστε γωνία και πατήστε Enter**, ή **κάντε κλικ** για ορισμό γωνίας από την κατεύθυνση κέρσορα.

**Ενεργοποίηση, μετά επιλογή** — ξεκινήστε εντολή χωρίς τίποτα επιλεγμένο:

1. Πληκτρολογήστε `rotate` ή κάντε κλικ στο κουμπί της γραμμής εργαλείων.
2. **Επιλέξτε αντικείμενα** — κλικ για εναλλαγή, ή σύρσιμο για επιλογή κατά περιοχή.
3. Πατήστε **Enter** ή **Space** για επιβεβαίωση επιλογής.
4. **Κάντε κλικ στο βασικό σημείο** (εισαγωγή συντεταγμένης διαθέσιμη), μετά ορίστε τη γωνία.

```
  Πριν:              Μετά (περιστροφή 90° γύρω από ●):
                        ╔══╗
  ●  [οντότητα]  →   ● ║    ║
                        ╚══╝
```

Μια ζωντανή φαντομ προεπισκόπηση των περιστρεφόμενων οντοτήτων ακολουθεί τη γωνία κέρσορα αφού οριστεί το βασικό σημείο.

## Ορισμός γωνίας

**Πληκτρολογημένη γωνία** — πληκτρολογήστε αριθμό (σε μοίρες) οποτεδήποτε αφού τοποθετηθεί το βασικό σημείο. Η προεπισκόπηση ευθυγραμμίζεται στην πληκτρολογημένη γωνία ενώ συνεχίζετε να προσαρμόζετε πριν πατήσετε Enter.

**Γωνία κλικ** — αν δεν υπάρχει πληκτρολογημένη τιμή, το κλικ ορίζει τη γωνία ίση με `atan2(cursorY − baseY, cursorX − baseX)` — η κατεύθυνση από το βασικό σημείο στο κλικ, σε μοίρες.

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `0`–`9`, `.` | Προσθήκη ψηφίου στην τιμή γωνίας |
| `-` | Αρνητική γωνία (μόνο πρώτος χαρακτήρας) |
| `Backspace` | Διαγραφή τελευταίου πληκτρολογημένου χαρακτήρα |
| `Enter` | Εφαρμογή περιστροφής στην πληκτρολογημένη γωνία |

## Κατεύθυνση γωνίας

Οι γωνίες ακολουθούν τη **σύμβαση DXF**:

- **Θετικές** τιμές περιστρέφουν **αντίθετα με τη φορά των δεικτών ρολογιού** στις συντεταγμένες σχεδίου (Y-πάνω).
- Στην οθόνη, όπου ο άξονας Y είναι αναστρεμμένος (Y-κάτω), οι θετικές γωνίες εμφανίζονται **με τη φορά των δεικτών ρολογιού**.

Συνηθισμένες τιμές: `90` = τεταρτημόριο, `180` = μισή στροφή, `-90` = αντίθετο τεταρτημόριο.

## Αναφορά πληκτρολογίου

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `Enter` / `Space` | Επιβεβαίωση επιλογής |
| `0`–`9`, `.`, `-` | Έναρξη εισαγωγής συντεταγμένης X (φάση βασικού σημείου), ή τιμή γωνίας (φάση γωνίας) |
| `,` | Κλείδωμα X και μετάβαση στην εισαγωγή Y (φάση βασικού σημείου) |
| `Backspace` | Διαγραφή τελευταίου πληκτρολογημένου χαρακτήρα |
| `Enter` | Επιβεβαίωση συντεταγμένης ή εφαρμογή περιστροφής |
| `Escape` | Ακύρωση και επαναφορά |

## Επιλογή κατά τη διάρκεια της εντολής

| Μέθοδος | Συμπεριφορά |
|---------|------------|
| **Κλικ** | Εναλλαγή οντότητας κάτω από τον κέρσορα |
| **Σύρσιμο δεξιά** (αυστηρό) | Προσθέτει οντότητες πλήρως εντός του πλαισίου |
| **Σύρσιμο αριστερά** (διασταυρούμενο) | Προσθέτει οντότητες που τέμνουν το πλαίσιο |
| **Enter** / **Space** | Επιβεβαίωση επιλογής |

## Υποστηριζόμενες οντότητες

Η Rotate λειτουργεί σε κάθε τύπο οντότητας. Η γεωμετρία κάθε οντότητας περιστρέφεται γύρω από το βασικό σημείο — για παράδειγμα, ένας Circle μετακινεί το κέντρο του ενώ η ακτίνα παραμένει αμετάβλητη· ένα Arc μετακινεί το κέντρο του και μετατοπίζει τις γωνίες αρχής και τέλους κατά το ποσό περιστροφής· μια οντότητα Text μετακινεί το σημείο άγκυρας και προσθέτει τη γωνία στην ιδιότητα Rotation Degree.
