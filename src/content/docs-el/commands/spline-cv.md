# Spline CV

Η εντολή `splinecv` σχεδιάζει **κυβική B-spline** τοποθετώντας κορυφές ελέγχου (CVs). Η καμπύλη που προκύπτει έλκεται προς κάθε CV αλλά δεν διέρχεται από αυτές — εκτός στην πρώτη και τελευταία κορυφή, όπου τα **clamped knots** αγκυρώνουν ακριβώς την καμπύλη. Αυτό παρέχει διαισθητικό έλεγχο σχήματος: τραβήξτε μια κορυφή για να ωθήσετε την καμπύλη προς αυτή χωρίς να την αναγκάζετε να αγγίξει κάθε σημείο.

## Σχεδίαση spline με κορυφές ελέγχου

1. Πληκτρολογήστε `splinecv` στο τερματικό ή κάντε κλικ στο κουμπί **Spline CV** της γραμμής εργαλείων.
2. **Κάντε κλικ για τοποθέτηση κορυφών ελέγχου** — κάθε κλικ προσθέτει μια κορυφή. Ή πληκτρολογήστε `X,Y` και πατήστε **Enter** για ακριβή συντεταγμένη.
3. Πατήστε **Enter** για ολοκλήρωση (απαιτούνται τουλάχιστον 2 κορυφές).

```
  CV ●         ● CV
      \       /
       \     /    ← καμπύλη έλκεται προς CVs
        \   /         αλλά δεν διέρχεται από αυτές
  CV ●   ●   ● CV (αρχή/τέλος: η καμπύλη αγγίζει εδώ)
```

Η ζωντανή προεπισκόπηση ενημερώνεται καθώς μετακινείτε τον κέρσορα μετά από κάθε κορυφή, δείχνοντας πώς θα φαίνεται η spline με το επόμενο σημείο στη θέση κέρσορα. Πατήστε **Escape** για απόρριψη όλων των τοποθετημένων κορυφών και έξοδο.

## Εισαγωγή συντεταγμένης

Αντί για κλικ, πληκτρολογήστε ακριβή θέση για οποιαδήποτε κορυφή ελέγχου:

1. Πληκτρολογήστε την τιμή X.
2. Πατήστε `,` — το τερματικό εμφανίζει `[X], [Y{cursor}]`.
3. Πληκτρολογήστε την τιμή Y.
4. Πατήστε **Enter** για τοποθέτηση κορυφής.

## Αναφορά πληκτρολογίου

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `0`–`9`, `.`, `-` | Έναρξη εισαγωγής συντεταγμένης X |
| `,` | Κλείδωμα X και μετάβαση στην εισαγωγή Y |
| `Backspace` | Διαγραφή τελευταίου πληκτρολογημένου χαρακτήρα |
| `Enter` | Επιβεβαίωση πληκτρολογημένης συντεταγμένης, ή ολοκλήρωση spline αν δεν υπάρχει εισαγωγή σε εξέλιξη και ≥ 2 κορυφές υπάρχουν |
| `Escape` | Απόρριψη όλων των κορυφών και έξοδος |

## Επεξεργασία grip — αναδιαμόρφωση μέσω κορυφών ελέγχου

Μια επιλεγμένη CV spline εμφανίζει ένα grip ανά κορυφή ελέγχου:

| Grip | Θέση | Τι κάνει |
|------|------|----------|
| **Κορυφή ελέγχου** | Σε κάθε θέση CV | Σύρσιμο για μετακίνηση αυτής της CV — η καμπύλη αναδιαμορφώνεται προς τη νέα θέση |

Δεν υπάρχει grip "μεταφορά ολόκληρης spline". Για μετατόπιση ολόκληρης της spline χρησιμοποιήστε την εντολή [Move](../move/).

## Επιλογή CV splines

| Μέθοδος | Συμπεριφορά |
|---------|------------|
| **Κλικ** | Επιλέγει αν το κλικ βρίσκεται κοντά σε οποιοδήποτε σημείο της καμπύλης |
| **Σύρσιμο δεξιά** (αυστηρό) | Όλα τα δείγματα σημεία κατά μήκος της καμπύλης πρέπει να βρίσκονται εντός του πλαισίου επιλογής |
| **Σύρσιμο αριστερά** (διασταυρούμενο) | Οποιοδήποτε τμήμα καμπύλης που διασταυρώνει το όριο πλαισίου επιλογής την επιλέγει |

## Υποστηριζόμενες εντολές επεξεργασίας

| Εντολή | Τι συμβαίνει στη spline |
|--------|------------------------|
| [Move](../move/) | Μεταφράζει όλες τις κορυφές ελέγχου κατά την ίδια μετατόπιση |
| [Copy](../copy/) | Δημιουργεί πανομοιότυπη spline σε νέα θέση |
| [Rotate](../rotate/) | Περιστρέφει όλες τις CVs γύρω από το επιλεγμένο βασικό σημείο |
| [Mirror](../mirror/) | Αντικατοπτρίζει όλες τις CVs ως προς τον άξονα αντικατοπτρισμού |
| [Scale](../scale/) | Κλιμακώνει όλες τις CVs ομοιόμορφα από το βασικό σημείο |
| [Delete](../delete/) | Αφαιρεί τη spline |

Οι splines δεν υποστηρίζουν **Offset**, **Trim**, ή **Extend**.

## Ιδιότητες

**Γενικά**

| Ιδιότητα | Προεπιλογή | Σημασία |
|----------|-----------|---------|
| Color | 256 (ByLayer) | Δείκτης χρώματος ACI |
| Layer | `0` | Ανάθεση επιπέδου |
| Linetype | ByLayer | Μοτίβο linetype ονόματος |
| Linetype Scale | 1 | Συντελεστής κλίμακας στο μοτίβο linetype |
| Thickness | 0 | Πάχος εξώθησης |

**Γεωμετρία**

| Ιδιότητα | Σημασία |
|----------|---------|
| Degree | Βαθμός πολυωνύμου — πάντα 3 (κυβική) |
| Control Vertices | Συντεταγμένες όλων των CVs |
| Fit Points | Κενό για CV splines· συμπληρώνεται μόνο για fit-point splines |

## Spline CV vs Spline Fit — ποια να χρησιμοποιήσετε

| | Spline CV | Spline Fit |
|---|-----------|------------|
| Η καμπύλη διέρχεται από σημεία | Μόνο πρώτο και τελευταίο (clamped) | Κάθε σημείο κλικ ακριβώς |
| Έλεγχος σχήματος | Τράβηγμα CVs προς μια περιοχή | Μετακίνηση σημείων fit που η καμπύλη πρέπει να αγγίξει |
| Αποτέλεσμα επεξεργασίας grip | CV μετακινείται → καμπύλη έλκεται | Σημείο fit μετακινείται → καμπύλη επαναπαρεμβάλλεται |
| Καλύτερο για | Ομαλές αισθητικές καμπύλες, ελεύθερες διαδρομές | Καμπύλες που πρέπει να αγγίξουν συγκεκριμένες συντεταγμένες |

## DXF — οντότητα SPLINE (μορφή κορυφών ελέγχου)

Οι CV splines αποθηκεύονται ως οντότητες `SPLINE` στο αρχείο DXF, αποθηκεύοντας τον βαθμό, το διάνυσμα knot και όλες τις συντεταγμένες κορυφών ελέγχου. Όλες οι ιδιότητες — χρώμα, επίπεδο, linetype, κλίμακα linetype και πάχος — ταξιδεύουν χωρίς απώλεια. Το `splineFlag` ορίζεται σε `9` (CV spline) ώστε η μορφή να διατηρείται κατά την επαναφόρτωση. Οποιαδήποτε εφαρμογή DXF που υποστηρίζει οντότητες `SPLINE` με δεδομένα CV τις διαβάζει σωστά.
