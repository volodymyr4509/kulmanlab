# Offset

Η εντολή `offset` δημιουργεί παράλληλο αντίγραφο μιας οντότητας σε σταθερή κάθετη απόσταση. Πληκτρολογείτε την απόσταση μια φορά, μετά κάνετε κλικ σε οντότητες και επιλέγετε πλευρά — η εντολή παραμένει έτοιμη στην ίδια απόσταση ώστε να μπορείτε να κάνετε offset σε πολλά αντικείμενα σε μια συνεδρία.

Υποστηριζόμενοι τύποι οντοτήτων: **Line, Circle, Arc, Ellipse, Polyline** (περιλαμβάνονται Rectangles).

## Χρήση offset

1. Πληκτρολογήστε `offset` στο τερματικό ή κάντε κλικ στο κουμπί **Offset** της γραμμής εργαλείων.
2. **Πληκτρολογήστε την απόσταση offset** και πατήστε **Enter** ή **Space**.
3. **Κάντε κλικ σε μια οντότητα** για offset — αν η οντότητα δεν είναι υποστηριζόμενου τύπου, εμφανίζεται μήνυμα σφάλματος και μπορείτε να κάνετε κλικ σε διαφορετική οντότητα.
4. **Μετακινήστε τον κέρσορα** στην πλευρά όπου πρέπει να εμφανιστεί το αντίγραφο — μια ζωντανή προεπισκόπηση ακολουθεί.
5. **Κάντε κλικ** για τοποθέτηση του αντιγράφου offset.

Μετά από κάθε τοποθέτηση η εντολή επιστρέφει στο βήμα 3 με **την ίδια απόσταση**, έτοιμη για το επόμενο offset. Πατήστε **Escape** για επαναφορά στο βήμα εισαγωγής απόστασης.

```
  Απόσταση: 10

  ─────────────────    ← αρχική γραμμή
  ─────────────────    ← αντίγραφο offset (10 μονάδες κάτω)
```

## Συμπεριφορά offset ανά οντότητα

| Οντότητα | Πώς υπολογίζεται το offset |
|----------|---------------------------|
| **Line** | Παράλληλη γραμμή μετατοπισμένη κάθετα στην αρχική κατεύθυνση |
| **Circle** | Ομόκεντρος κύκλος· κλικ έξω → μεγαλύτερη ακτίνα, μέσα → μικρότερη ακτίνα |
| **Arc** | Ομόκεντρο τόξο σε νέα ακτίνα· διατηρείται η ίδια γωνιακή εκτεταμένη |
| **Ellipse** | Και οι δύο ημιάξονες αυξάνονται ή μειώνονται κατά την ίδια απόσταση |
| **Polyline** | Κάθε τμήμα offset ανεξάρτητα· γειτονικά offset τμήματα μιτερώνονται στις γωνίες |

Για **Circle**, **Arc**, και **Ellipse**: αν το εσωτερικό offset μειώσει οποιαδήποτε ακτίνα ή ημιάξονα σε μηδέν ή κάτω, το offset δεν εφαρμόζεται.

## Αναφορά πληκτρολογίου

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `0`–`9`, `.` | Προσθήκη ψηφίου στην τιμή απόστασης |
| `Backspace` | Διαγραφή τελευταίου πληκτρολογημένου χαρακτήρα |
| `Enter` / `Space` | Επιβεβαίωση πληκτρολογημένης απόστασης και πρόοδος στην επιλογή οντότητας |
| `Escape` | Επαναφορά στο βήμα εισαγωγής απόστασης |

## Σημείωση ροής εργασίας

Η απόσταση παραμένει ορισμένη μέχρι να πατήσετε **Escape**. Αυτό καθιστά αποδοτικό το offset πολλών οντοτήτων στην ίδια απόσταση — πληκτρολογήστε την απόσταση μια φορά, μετά κάντε κλικ και επιλέξτε πλευρά για κάθε οντότητα σειριακά.

## Offset vs Copy

| | Offset | Copy |
|---|--------|------|
| Μετατόπιση | Κάθετα στη γεωμετρία της οντότητας | Αυθαίρετο διάνυσμα (βάση → προορισμός) |
| Υποστηριζόμενες οντότητες | Line, Circle, Arc, Ellipse, Polyline | Όλοι οι τύποι οντοτήτων |
| Εισαγωγή απόστασης | Πληκτρολογείται πριν την επιλογή οντότητας | Πληκτρολογείται ή κλικ μετά την επιλογή |
| Καλύτερο για | Παράλληλες γραμμές, ομόκεντροι κύκλοι, εσωτερικές/εξωτερικές διαδρομές | Τοποθέτηση αντιγράφων σε αυθαίρετες θέσεις |
