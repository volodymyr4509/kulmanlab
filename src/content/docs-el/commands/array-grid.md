# Array Grid

Η εντολή `ArrayGrid` δημιουργεί ένα ορθογώνιο πλέγμα αντιγράφων από τις επιλεγμένες οντότητες — εισαγάγετε τον αριθμό γραμμών, τον αριθμό στηλών και το διάστημα μεταξύ τους, όλα πληκτρολογημένα στο τερματικό. Η αρχική επιλογή καταλαμβάνει το κελί γραμμή 0, στήλη 0· κάθε άλλο κελί είναι ένα μετατοπισμένο αντίγραφο.

## Δύο τρόποι εκκίνησης

**Προεπιλογή, μετά array** — επιλέξτε οντότητες πρώτα, μετά ενεργοποιήστε:

1. Επιλέξτε μία ή περισσότερες οντότητες στον καμβά.
2. Πληκτρολογήστε `arraygrid` στο τερματικό (αρκεί το `arr` — χωρίς αμφισημία) ή κάντε κλικ στο κουμπί **Array Grid** της γραμμής εργαλείων.
3. Πληκτρολογήστε τον αριθμό **γραμμών** και πατήστε **Enter**.
4. Πληκτρολογήστε τον αριθμό **στηλών** και πατήστε **Enter**.
5. Πληκτρολογήστε το **διάστημα μεταξύ γραμμών** και πατήστε **Enter**.
6. Πληκτρολογήστε το **διάστημα μεταξύ στηλών** και πατήστε **Enter** — το πλέγμα δημιουργείται αμέσως.

**Ενεργοποίηση, μετά επιλογή** — ξεκινήστε την εντολή χωρίς τίποτα επιλεγμένο:

1. Πληκτρολογήστε `arraygrid` ή κάντε κλικ στο κουμπί γραμμής εργαλείων.
2. **Επιλέξτε αντικείμενα** — κλικ για εναλλαγή μεμονωμένων οντοτήτων, ή σύρσιμο για επιλογή ανά περιοχή.
3. Πατήστε **Enter** ή **Space** για επιβεβαίωση της επιλογής.
4. Συνεχίστε με γραμμές → στήλες → διάστημα γραμμών → διάστημα στηλών όπως παραπάνω.

```
  2 γραμμές x 3 στήλες:

  [B] [B] [B]   <- γραμμή 1 (μετατοπισμένα αντίγραφα)
  [A] [A] [A]   <- γραμμή 0: αρχική επιλογή, αντίγραφα προς τα δεξιά
```

> Το τερματικό χρειάζεται μόνο όσα γράμματα αρκούν για να μην υπάρχει αμφισημία — πληκτρολογώντας `arr` και πατώντας **Enter** ενεργοποιείται απευθείας το Array Grid, καθώς κανένα άλλο όνομα εντολής δεν ξεκινά με αυτά τα τρία γράμματα (τα Arc, Area, Align και Angle αποκλίνουν όλα νωρίτερα).

## Γραμμές, στήλες και διάστημα

| Προτροπή | Δέχεται | Σημειώσεις |
|----------|---------|------------|
| Γραμμές | Θετικοί ακέραιοι (1, 2, 3…) | Μόνο ψηφία — χωρίς δεκαδική τελεία ή πρόσημο |
| Στήλες | Θετικοί ακέραιοι (1, 2, 3…) | Μόνο ψηφία — χωρίς δεκαδική τελεία ή πρόσημο |
| Διάστημα γραμμών | Δεκαδικός με πρόσημο (π.χ. `10`, `-5.5`) | Απόσταση μεταξύ γραμμών· αρνητικό αντιστρέφει την κατεύθυνση |
| Διάστημα στηλών | Δεκαδικός με πρόσημο (π.χ. `10`, `-5.5`) | Απόσταση μεταξύ στηλών· αρνητικό αντιστρέφει την κατεύθυνση |

Με 1 γραμμή και 1 στήλη, δεν δημιουργούνται αντίγραφα — η εντολή τερματίζεται χωρίς αλλαγή στο σχέδιο.

## Αναφορά πληκτρολογίου

| Πλήκτρο | Ενέργεια |
|---------|----------|
| `Enter` / `Space` | Επιβεβαιώνει την επιλογή και προχωρά στην προτροπή γραμμών |
| `0`–`9` | Εισαγωγή ψηφίων για γραμμές ή στήλες |
| `0`–`9`, `.`, `-` | Εισαγωγή ψηφίων για διάστημα γραμμών/στηλών (`-` μόνο ως πρώτος χαρακτήρας) |
| `Backspace` | Διαγράφει τον τελευταίο πληκτρολογημένο χαρακτήρα της τρέχουσας προτροπής |
| `Enter` | Επιβεβαιώνει την τρέχουσα προτροπή και προχωρά στην επόμενη |
| `Escape` | Καθαρίζει τις πληκτρολογημένες τιμές γραμμών/στηλών/διαστήματος και επιστρέφει στη φάση επιλογής |

## Επιλογή κατά τη διάρκεια της εντολής

| Μέθοδος | Συμπεριφορά |
|---------|------------|
| **Κλικ** | Εναλλάσσει την οντότητα κάτω από τον κέρσορα εντός/εκτός επιλογής |
| **Σύρσιμο δεξιά** (αυστηρό) | Προσθέτει οντότητες πλήρως μέσα στο πλαίσιο |
| **Σύρσιμο αριστερά** (διασταυρούμενο) | Προσθέτει οντότητες που τέμνουν το όριο του πλαισίου |
| **Enter** / **Space** | Επιβεβαιώνει την επιλογή και προχωρά στην προτροπή γραμμών |

## Μετά το array

Τα νέα αντίγραφα προστίθενται στο σχέδιο και η εντολή τερματίζεται — η αρχική επιλογή διαγράφεται. Εκτελέστε ξανά **Array Grid**, ή ξεκινήστε νέα εντολή.

## Array Grid εναντίον Copy

| | Array Grid | Copy |
|---|-----------|------|
| Επιλογή σημείων | Καμία — γραμμές, στήλες και διάστημα πληκτρολογούνται | Το βασικό σημείο και ο προορισμός κλικάρονται (ή πληκτρολογούνται) |
| Αντίγραφα που δημιουργούνται | Γραμμές × Στήλες − 1 | Ακριβώς 1 ανά λειτουργία αντιγραφής |
| Διάταξη | Κανονικό ορθογώνιο πλέγμα | Οπουδήποτε, σε οποιαδήποτε μετατόπιση |
| Καλύτερο για | Επανάληψη μιας μονάδας σε κανονικό μοτίβο (τρύπες, πλακίδια, στοιχεία στερέωσης) | Ένα μόνο αντίγραφο σε αυθαίρετη θέση |

## Υποστηριζόμενες οντότητες

Το Array Grid λειτουργεί σε κάθε τύπο οντότητας. Όλες οι οντότητες υλοποιούν εσωτερικά `translate(dx, dy)` — την ίδια λειτουργία που χρησιμοποιούν τα [Copy](../copy/) και [Move](../move/) — οπότε καμία δεν εξαιρείται.
