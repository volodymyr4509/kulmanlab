# KulmanLab CAD — Αναφορά Εντολών

Καλώς ήρθατε στην αναφορά εντολών του **KulmanLab CAD**. Το [KulmanLab CAD](https://kulmanlab.com) είναι ένα δωρεάν, browser-based εργαλείο CAD για σχεδίαση, επεξεργασία και εξαγωγή αρχείων DXF — χωρίς εγκατάσταση. Χρησιμοποιήστε την πλαϊνή μπάρα για να περιηγηθείτε σε όλες τις διαθέσιμες εντολές ομαδοποιημένες ανά πάνελ.

| Εντολή | Τι κάνει |
|--------|----------|

| [Polyline](./commands/polyline/) | Σχεδιάζει πολυτμηματικό ανοιχτό μονοπάτι |

| [Arc](./commands/arc/) | Σχεδιάζει τόξο μέσα από τρία σημεία |
| [Ellipse](./commands/ellipse/) | Σχεδιάζει έλλειψη με κέντρο και δύο άξονες |
| [Text](./commands/text/) | Τοποθετεί ετικέτα κειμένου στον καμβά |

## Ανάκτηση

Αν η εφαρμογή καταρρέει σε κάθε εκκίνηση (για παράδειγμα, μετά από εργασία με εξαιρετικά μεγάλες συντεταγμένες), μπορείτε να διαγράψετε όλα τα τοπικά αποθηκευμένα δεδομένα προσαρτώντας `?reset` στη URL:

```
https://kulmanlab.com/?resetKulmanLocalStorage
```

Αυτό διαγράφει τα πάντα από την τοπική βάση δεδομένων του browser και ξεκινά ένα νέο κενό σχέδιο. Η παράμετρος `?reset` αφαιρείται αυτόματα από τη URL. Χρησιμοποιήστε τη ως έσχατη λύση όταν το [WipeStorage](./commands/wipestorage/) δεν είναι προσβάσιμο επειδή η εφαρμογή δεν φορτώνει καθόλου.

## Πώς λειτουργούν οι εντολές

Κάθε εντολή ακολουθεί το ίδιο μοτίβο:

1. **Ενεργοποίηση** — κλικ στο κουμπί της γραμμής εργαλείων ή πληκτρολόγηση του ονόματος της εντολής στο τερματικό στο κάτω μέρος της οθόνης.
2. **Ακολουθήστε την προτροπή** — το τερματικό δείχνει ποια είσοδος αναμένεται στη συνέχεια.
3. **Ολοκλήρωση ή ακύρωση** — οι περισσότερες εντολές ολοκληρώνονται αυτόματα μετά την τελευταία είσοδο. Πατήστε **Escape** ανά πάσα στιγμή για ακύρωση.

## Επιλογή αντικειμένων

Πολλές εντολές επεξεργασίας (Move, Copy, Rotate, Mirror, Scale, Delete) μοιράζονται την ίδια συμπεριφορά επιλογής:

- **Κλικ** σε αντικείμενο για επιλογή ή αποεπιλογή.
- **Σύρσιμο δεξιά** (αριστερά προς δεξιά) για αυστηρή επιλογή — επιλέγονται μόνο αντικείμενα που βρίσκονται πλήρως μέσα στο πλαίσιο.
- **Σύρσιμο αριστερά** (δεξιά προς αριστερά) για διασταυρούμενη επιλογή — επιλέγεται κάθε αντικείμενο που τέμνει το πλαίσιο.
- Πατήστε **Enter** ή **Space** για επιβεβαίωση της επιλογής και προχωρήστε στο επόμενο βήμα.
