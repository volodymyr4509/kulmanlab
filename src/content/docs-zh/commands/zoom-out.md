# Zoom Out

`zoomout`（缩小）命令将当前缩放级别除以 **1.5×**（相当于乘以约 0.667），立即退出，以视口中点为中心。这是[Zoom In](../zoom-in/)的反向操作。

## 缩小

单击工具栏中的**Zoom Out**按钮，或在命令行中输入 `zoomout`。缩放立即应用，命令退出 — 无需在画布上单击。

## 1.5× 步长的工作原理

| 当前缩放 | 缩小一次后 |
|-------------|-------------------|
| 1.50× | 1.00× |
| 2.25× | 1.50× |
| 10.00× | 6.67× |
| 0.015× | 0.01×（已设底限） |

缩放级别始终显示在画布**右下角**。下限为 **0.01×**；超过此限制的步骤不会产生任何效果。

## 工具栏缩小 vs 滚轮

| | 缩小按钮 | 滚轮 |
|---|----------------|-------------|
| 缩放中心 | 视口中点 | 光标位置 |
| 步长 | 每次单击 1.5× | 每格约 1.1× |
| 需要激活 | 否 | 否 — 始终有效 |
| 最适用于 | 逐步后退以看到更多上下文 | 平滑的光标锚定缩小 |

## 键盘参考

此命令没有键盘快捷键。请改用滚轮 — 它随时可用，无需激活任何命令。
