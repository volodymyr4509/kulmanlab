# 旋转

`rotate`（旋转）命令绕基点旋转选中图元。您可以通过输入度数或单击来指定旋转角度 — 角度由基点和单击位置之间的方向计算。

## 两种启动方式

**预先选择，然后旋转** — 先选择图元，再激活：

1. 在画布上选择一个或多个图元。
2. 在命令行中输入 `rotate`，或单击工具栏中的**旋转**按钮。
3. **单击基点** — 旋转中心。或输入 `X,Y` 并按 **Enter** 键精确输入坐标。
4. **输入角度并按 Enter**，或**单击**以从光标方向设置角度。

**激活后选择** — 在未选中任何内容的情况下启动命令：

1. 输入 `rotate` 或单击工具栏按钮。
2. **选择对象** — 单击以切换，或拖动以按区域选择。
3. 按 **Enter** 或**空格**确认选择。
4. **单击基点**（可输入坐标），然后设置角度。

```
  之前：            绕 ● 旋转 90° 后：
                        ╔══╗
  ●  [图元]    →   ● ║    ║
                        ╚══╝
```

设置基点后，旋转图元的实时幽灵预览跟随光标角度。

## 设置角度

**输入角度** — 放置基点后随时输入数字（度）。调整前预览捕捉到输入的角度。

**单击角度** — 如果没有输入值，单击设置角度等于 `atan2(光标Y − 基点Y, 光标X − 基点X)` — 从基点到单击的方向，以度为单位。

| 按键 | 操作 |
|-----|--------|
| `0`–`9`、`.` | 在角度值后追加数字 |
| `-` | 负角度（仅第一个字符） |
| `Backspace` | 删除最后输入的字符 |
| `Enter` | 以输入的角度应用旋转 |

## 角度方向

角度遵循 **DXF 约定**：

- **正值**在图形坐标（Y 轴向上）中**逆时针**旋转。
- 在屏幕上，Y 轴反转（Y 轴向下），正角度显示为**顺时针**。

常用值：`90` = 四分之一圈，`180` = 半圈，`-90` = 反方向四分之一圈。

## 键盘参考

| 按键 | 操作 |
|-----|--------|
| `Enter` / 空格 | 确认选择 |
| `0`–`9`、`.`、`-` | 开始输入 X 坐标（基点阶段），或角度值（角度阶段） |
| `,` | 锁定 X 并移至 Y 输入（基点阶段） |
| `Backspace` | 删除最后输入的字符 |
| `Enter` | 确认坐标或应用旋转 |
| `Escape` | 取消并重置 |

## 命令中的选择

| 方法 | 行为 |
|--------|-----------|
| **单击** | 切换光标下方图元 |
| **向右拖动**（严格） | 添加完全在选框内的图元 |
| **向左拖动**（交叉） | 添加与选框相交的图元 |
| **Enter** / 空格 | 确认选择 |

## 支持的图元

旋转适用于每种图元类型。每个图元的几何图形绕基点旋转 — 例如，圆移动其圆心而半径不变；圆弧移动其圆心并将起止角按旋转量偏移；文字图元移动其锚点并将角度添加到其旋转度数属性。
