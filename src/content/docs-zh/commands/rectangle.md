# Rectangle

`rectangle`（矩形）命令通过两个对角单击绘制轴对齐矩形。结果存储为带四个顶点（每个角一个）的**闭合 `LWPOLYLINE`**。没有专用的矩形图元类型：创建后形状与任何其他[Polyline](../polyline/)完全相同，每个多段线编辑都适用。

## 绘制矩形

1. 在命令行中输入 `rectangle`，或单击工具栏中的**Rectangle**按钮。
2. **单击第一个角**，或输入 `X,Y` 并按 **Enter** 键精确输入坐标。
3. **单击对角** — 矩形立即放置，命令退出。此处也可以输入坐标。

```
  ● （第一次单击）────────────┐
  |                          |
  |   实时预览跟随第 2 步      |
  |   后的光标               |
  └──────────────────────────● （第二次单击）
```

两次单击可以是任意一对对角 — 左上 + 右下，或左下 + 右上，等等。顺序不重要。

## 坐标输入

在任一角步骤，可以输入精确位置：

1. 输入 X 值。
2. 按 `,` — 命令行显示 `[X], [Y{光标}]`。
3. 输入 Y 值。
4. 按 **Enter** 放置角点。

## 键盘参考

| 按键 | 操作 |
|-----|--------|
| `0`–`9`、`.`、`-` | 开始输入 X 坐标 |
| `,` | 锁定 X 并移至 Y 输入 |
| `Enter` | 确认输入的坐标 |
| `Escape` | 取消 |

各边始终水平和垂直 — 矩形命令没有角度锁定。

## 夹点编辑 — 创建后重塑

选中的矩形在每个顶点和每条边的中点处显示夹点：

| 夹点 | 位置 | 功能 |
|------|----------|--------------|
| **角** | 4 个顶点各一个 | 拖动以移动该顶点；两条相邻边随之延伸 — 对角保持固定 |
| **边中点** | 4 条边各中心一个 | 拖动以同时平移该边的两个端点，保持边的长度和角度 |

拖动角夹点会将矩形变为非矩形四边形。如果只需要不同大小的矩形，在保持边大致正交的同时拖动角，或删除后重新绘制。

## 选择矩形

因为矩形是多段线，选择方式相同：

| 方法 | 行为 |
|--------|-----------|
| **单击** | 如果单击落在四条边的任意一条上，则选中 |
| **向右拖动**（严格） | 四个顶点都必须在选框内 |
| **向左拖动**（交叉） | 与选框边界相交的任何边都将选中整个矩形 |

## 支持的编辑命令

所有多段线编辑命令均适用。修剪和延伸仅适用于[Line](../line/)，不适用于矩形：

| 命令 | 对矩形的影响 |
|---------|-------------------------------|
| [Move](../move/) | 四个顶点平移相同位移 |
| [Copy](../copy/) | 在新位置创建相同的矩形 |
| [Rotate](../rotate/) | 绕选定基点旋转四个顶点 |
| [Mirror](../mirror/) | 沿镜像轴反射四个顶点 |
| [Scale](../scale/) | 从基点统一缩放四个顶点 |
| [Offset](../offset/) | 创建固定距离的平行（内缩或外扩）矩形 |
| [Delete](../delete/) | 从图形中删除矩形 |

## 属性

选中矩形时，属性面板显示与任何多段线相同的字段：

**常规**

| 属性 | 默认值 | 含义 |
|----------|---------|---------|
| 颜色 | 256（随层） | ACI 颜色索引 |
| 图层 | `0` | 图层分配 |
| 线型 | 随层 | 命名线型样式 |
| 线型比例 | 1 | 线型样式的缩放系数 |
| 厚度 | 0 | 拉伸厚度 |

**几何**

| 属性 | 含义 |
|----------|---------|
| 闭合 | 矩形始终为 `true` |
| 顶点数 | 未修改矩形始终为 `4` |
| 顶点 | 四个角的坐标 |

## Rectangle vs Polyline vs Line

| | 矩形 | 多段线 | 直线 |
|---|-----------|---------|------|
| 绘制方式 | 2 次单击（角） | 单击每个顶点 | 单击每个端点 |
| 图元类型 | 闭合 `LWPOLYLINE` | 开放或闭合 `LWPOLYLINE` | 每段一个 `LINE` |
| 边始终正交 | 是（创建时） | 否 | 否 |
| 修剪 / 延伸 | 否 | 否 | 是 |
| 最适用于 | 框、框架、矩形区域 | 任意轮廓和路径 | 单个线段、辅助线 |

## DXF — LWPOLYLINE 实体

矩形以带四个顶点的闭合 `LWPOLYLINE` 实体保存。所有属性（顶点坐标、颜色、图层、线型、线型比例和厚度）均可完整往返转换。

DXF 中没有专用的 `RECTANGLE` 类型。重新打开文件时，形状显示为闭合四顶点多段线而非矩形。任何支持 `LWPOLYLINE` 的 DXF 查看器或编辑器（LibreCAD、FreeCAD 等）都可以正确显示它。
