# 角度

`angle`（角度）命令通过三个单击点测量由两条线段在顶点处形成的内角。结果始终在 0° 到 180° 之间，以 4 位小数显示在命令行中。这是两个测量命令之一 — [距离](../distance/)命令用于测量直线长度。

## 角度测量的结构

```
  ● 第一点（第一条射线的端点）
   \
    \  第一条射线预览
     \
      ● 顶点（第 3 步）
     /
    /  第二条射线预览（到光标）
   /
  ● 第三点  →  命令行："Angle: 45.0000°"
```

- **第一点** — 角度的一个端点（第 2 步）。
- **顶点** — 测量角度的角点（第 3 步）。
- **第三点** — 角度的另一个端点（第 4 步）。

## 测量角度

1. 在命令行中输入 `angle`，或单击工具栏中的**角度**按钮。
2. **单击第一点** — 角度一臂的端点。也可以输入 `X,Y` 并按 **Enter** 键精确输入坐标。
3. **单击顶点** — 两臂相交的角点。此处也可以输入坐标。
4. **单击第三点** — 第二臂的端点。此处也可以输入坐标。放置该点后将输出结果。
5. **再次单击**（可选）以该单击点为新的第一点开始新的测量。

## 内角约定

命令使用从顶点出发的两条射线的点积计算角度：

- **始终为内角**：结果是较小的角，在 0° 到 180° 之间。
- 单击端点的顺序不影响结果 — 只有顶点位置有关。
- 共线点（三点在同一直线上）返回 0° 或 180°。

## 链式测量

结果显示后，立即单击即可开始下一次测量 — 单击的点成为新的第一点。命令不会自动退出，直到按下 `Escape` 键。

## 角度 vs 距离

| | 角度 | 距离 |
|---|-------|---------|
| 测量内容 | 顶点处的内角 | 直线长度 |
| 单击次数 | 3 | 2 |
| 结果格式 | `45.0000°` | `12.3456`（单位） |
| 画布预览 | 从顶点到两端的两条线 | 从第一点到光标的直线 |
| 最适用于 | 两个特征之间的张角 | 间隙或线段的长度 |

## 坐标输入

可以为三点中的任意一点输入精确位置，而不必单击：

1. 输入 X 值。
2. 按 `,` — 命令行显示 `[X], [Y{光标}]`。
3. 输入 Y 值。
4. 按 **Enter** 确认。

## 键盘参考

| 按键 | 操作 |
|-----|--------|
| `0`–`9`、`.`、`-` | 开始输入 X 坐标 |
| `,` | 锁定 X 并移至 Y 输入 |
| `Backspace` | 删除最后输入的字符 |
| `Enter` | 确认输入的坐标 |
| `Escape` | 取消并重置至第 2 步 |

## 注意事项

- 结果仅显示在**命令行**中 — 不会在图形中添加任何内容。
- 精度始终为角度的 4 位小数。
