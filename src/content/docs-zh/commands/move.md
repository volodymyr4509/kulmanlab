# Move

`move`（移动）命令将选中图元从基点平移到目标点。应用于每个选中图元的位移是从基点到目标点的向量。移动后所有图元在新位置保持选中状态，可供进一步编辑。

## 两种启动方式

**预先选择，然后移动** — 先选择图元，再激活：

1. 在画布上选择一个或多个图元。
2. 在命令行中输入 `move`，或单击工具栏中的**Move**按钮。
3. **单击基点**，或输入 `X,Y` 并按 **Enter** 键精确输入坐标。
4. **单击目标点** — 所有选中图元按基点→目标点向量移动。此处也可以输入坐标。

**激活后选择** — 在未选中任何内容的情况下启动命令：

1. 输入 `move` 或单击工具栏按钮。
2. **选择对象** — 单击以切换单个图元，或拖动以按区域选择。
3. 按 **Enter** 或**空格**确认选择。
4. **单击基点**，然后**单击目标点**（两步均可输入坐标）。

```
  之前：                    之后：
  ● 基点                       → ● 目标点
  [图元 A]                      [已移动的图元 A]
  [图元 B]                      [已移动的图元 B]
```

所有选中图元的幽灵预览从基点跟随光标到目标点，在单击前显示结果。

## 坐标输入

在基点或目标点步骤，可以输入精确位置而不必单击：

1. 输入 X 值。
2. 按 `,` — 命令行显示 `[X], [Y{光标}]`。
3. 输入 Y 值。
4. 按 **Enter** 确认。

## 角度锁定和精确距离

设置基点后，命令监测 45° 捕捉轴（0°、45°、90°、135°……）。当光标距离基点足够远且在轴的一个夹点宽度内时，方向**锁定**。锁定时：

- 幽灵预览捕捉到轴。
- 输入距离并按 **Enter** 可沿锁定方向精确移动该距离。
- 单击会投影到轴上，因此目标点始终精确落在轴上。

| 按键 | 操作 |
|-----|--------|
| `0`–`9`、`.` | 在距离值后追加数字 |
| `-` | 负距离 — 沿轴线反向（仅第一个字符） |
| `Backspace` | 删除最后输入的字符 |
| `Enter` | 在输入的距离处应用移动 |

## 键盘参考

| 按键 | 操作 |
|-----|--------|
| `Enter` / 空格 | 确认选择并进入基点阶段 |
| `0`–`9`、`.`、`-` | 开始输入 X 坐标，或角度锁定时输入距离 |
| `,` | 锁定 X 并移至 Y 输入 |
| `Backspace` | 删除最后输入的字符 |
| `Enter` | 确认坐标或在输入的距离处应用移动 |
| `Escape` | 取消并重置 |

## 从夹点激活 Move

单击选中[Line](../line/)的**中点夹点**会自动启动 Move 命令，中点已设为基点，移动阶段已激活。这是无需经过选择步骤即可重新定位单条直线的最快方式。

## 命令中的选择

命令在选择阶段启动时：

| 方法 | 行为 |
|--------|-----------|
| **单击** | 切换光标下方图元的选中/取消状态 |
| **向右拖动**（严格） | 添加完全在选框内的图元 |
| **向左拖动**（交叉） | 添加与选框边界相交的图元 |
| **Enter** / 空格 | 确认选择并进入基点阶段 |

## 移动后

移动的图元在新位置保持选中状态。这意味着您可以立即：
- 再次运行**Move**以进一步微调。
- 运行[Copy](../copy/)、[Rotate](../rotate/)或[Scale](../scale/)而无需重新选择。
- 按 **Delete** 删除它们。

## Move vs Copy

| | 移动 | 复制 |
|---|------|------|
| 原始位置 | 已腾空 — 图元不再在那里 | 保留 — 原图元保持原位 |
| 结果数量 | 图元数量不变 | 每次操作增加一组 |
| 操作后选择 | 图元在新位置处于选中状态 | 复制的图元在新位置处于选中状态 |
| 最适用于 | 重新定位几何图形 | 复制几何图形 |

## 支持的图元

Move 适用于每种图元类型：直线、多段线、矩形、圆、圆弧、椭圆、文字、样条线、标注、引线及所有其他类型。所有图元都实现了 `translate(dx, dy)`，因此没有任何图元被排除。
