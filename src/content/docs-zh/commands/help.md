# Help

打开应用内帮助面板，其中包含所有键盘快捷键、导航提示和一般使用说明的摘要。

## 使用方法

单击工具栏中的**Help**按钮，或在命令行中输入 `help`。帮助面板立即打开，命令退出。

## 帮助面板的内容

### 导航
| 快捷键 | 操作 |
|----------|--------|
| 滚轮 | 放大 / 缩小 |
| 中键单击 + 拖动 | 平移视图 |
| 双击中键 | 将所有图元适应屏幕 |

### 命令
| 快捷键 | 操作 |
|----------|--------|
| 输入 | 开始输入命令名称 — 建议出现在画布上 |
| `Delete` | 删除所有选中图元 |
| `Enter` / 空格 | 确认命令或输入 |
| `Enter` / 空格（空闲状态） | 重复上一次使用的命令 |
| `Escape` | 取消当前命令或清除选择 |

### 选择
| 快捷键 | 操作 |
|----------|--------|
| 左键单击 | 选择图元 |
| 向右拖动 | 仅选择**完全在**拖动区域内的图元 |
| 向左拖动 | 选择**触及**拖动区域的任何图元 |
| 左键单击夹点 | 直接编辑顶点或端点 |

### 绘图提示
- **捕捉点** — 光标自动捕捉到附近图元的端点。
- **角度锁定** — 将光标向某一方向移动足够远以锁定到 45° 增量。
- **精确长度** — 角度锁定时，输入数字并按 `Enter` 设置精确距离。
