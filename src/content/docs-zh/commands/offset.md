# Offset

`offset`（偏移）命令以固定垂直距离创建图元的平行副本。输入一次距离，然后单击图元并选择一侧 — 命令以相同距离保持就绪，可以在一次会话中偏移多个对象。

支持的图元类型：**直线、圆、圆弧、椭圆、多段线**（包括矩形）。

## 使用偏移

1. 在命令行中输入 `offset`，或单击工具栏中的**Offset**按钮。
2. **输入偏移距离**并按 **Enter** 或**空格**。
3. **单击要偏移的图元** — 如果图元不是支持的类型，将显示错误消息，可以单击不同的图元。
4. **将光标移到**副本应出现的一侧 — 实时预览跟随。
5. **单击**以放置偏移副本。

每次放置后，命令以**相同距离**返回第 3 步，准备下一次偏移。按 **Escape** 重置回距离输入步骤。

```
  距离：10

  ─────────────────    ← 原始直线
  ─────────────────    ← 偏移副本（下方 10 个单位）
```

## 各图元的偏移行为

| 图元 | 偏移计算方式 |
|--------|---------------------------|
| **直线** | 垂直于原始方向移动的平行直线 |
| **圆** | 同心圆；在外侧单击 → 半径更大，在内侧单击 → 半径更小 |
| **圆弧** | 新半径处的同心圆弧；保留相同角度范围 |
| **椭圆** | 两条半轴均增大或减小相同距离 |
| **多段线** | 每段独立偏移；相邻偏移段在角点处斜接 |

对于**圆**、**圆弧**和**椭圆**：如果向内偏移会将任何半径或半轴减小到零或以下，则不应用偏移。

## 键盘参考

| 按键 | 操作 |
|-----|--------|
| `0`–`9`、`.` | 在距离值后追加数字 |
| `Backspace` | 删除最后输入的字符 |
| `Enter` / 空格 | 确认输入的距离并进入图元选择 |
| `Escape` | 重置到距离输入步骤 |

## 工作流说明

距离保持设置直到按 **Escape**。这使得以相同间距偏移多个图元非常高效 — 输入一次距离，然后依次单击每个图元并选择侧面。

## Offset vs Copy

| | 偏移 | 复制 |
|---|--------|------|
| 位移 | 垂直于图元几何形状 | 任意向量（基点→目标点） |
| 支持的图元 | 直线、圆、圆弧、椭圆、多段线 | 所有图元类型 |
| 距离输入 | 在选择图元前输入 | 选择后输入或单击 |
| 最适用于 | 平行线、同心圆、内缩/外扩路径 | 在任意位置放置副本 |
