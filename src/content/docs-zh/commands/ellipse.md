# 椭圆

`ellipse`（椭圆）命令通过三次单击绘制椭圆：圆心点、第一条（主）半轴的端点（可以任意角度），以及第二条（次）半轴的长度。两轴始终相互垂直 — 第二轴方向由第一轴自动推导。

## 绘制椭圆

1. 在命令行中输入 `ellipse`，或单击工具栏中的**椭圆**按钮。
2. **单击圆心点**，或输入 `X,Y` 并按 **Enter** 键精确输入坐标。
3. **单击第一轴端点** — 设置第一条半轴的方向和长度。此处也可以输入坐标。
4. **设置第二轴长度** — 将光标移至垂直于第一轴的方向，然后单击或输入长度。

```
               ● ← 第一轴端点（第 3 步）
              /
  圆心 ●  /  ← 第一轴（任意角度）
            |
            ● ← 在此处移动光标设置第二轴长度（第 4 步）
```

第 4 步后椭圆被放置，命令退出。

## 轴输入 — 单击、坐标或输入长度

**圆心（第 2 步）：** 单击或输入 `X,Y` 精确定位。

**第一轴端点（第 3 步）：** 单击，或输入 `X,Y` 精确坐标。角度锁定也会捕捉到 45° 增量 — 锁定时，输入长度并按 **Enter** 可将端点精确放置在该距离处。

**第二轴（第 4 步）：** 始终可以输入长度 — 无需角度锁定。方向已固定为垂直于第一轴；输入仅设置长度。

| 按键 | 操作 |
|-----|--------|
| `0`–`9`、`.` | 在轴长度后追加数字（第二轴阶段） |
| `Backspace` | 删除最后输入的字符 |
| `Enter` | 在输入的长度处放置轴端点 |

## 坐标输入（圆心和第一轴端点）

1. 输入 X 值。
2. 按 `,` — 命令行显示 `[X], [Y{光标}]`。
3. 输入 Y 值。
4. 按 **Enter** 确认。

## 键盘参考

| 按键 | 操作 |
|-----|--------|
| `0`–`9`、`.`、`-` | 开始输入 X 坐标（圆心/第一轴阶段），或角度锁定时输入轴长度 |
| `,` | 锁定 X 并移至 Y 输入 |
| `Backspace` | 删除最后输入的字符 |
| `Enter` | 确认输入的坐标或长度 |
| `Escape` | 取消并重置 |

## 夹点编辑 — 独立调整轴

选中的椭圆显示五个夹点：

| 夹点 | 数量 | 功能 |
|------|-------|--------------|
| **圆心** | 1 | 移动整个椭圆；两轴不变 |
| **长轴端点** | 2（较长轴的两端） | 拖动以调整长半轴长度；短轴绝对大小保持不变 |
| **短轴端点** | 2（较短轴的两端） | 拖动以调整短半轴长度；长轴不变 |

长轴和短轴夹点相互独立 — 可以在不重新运行命令的情况下调整椭圆形状。

## 选择椭圆

| 方法 | 行为 |
|--------|-----------|
| **单击** | 如果单击落在椭圆轮廓附近，则选中 |
| **向右拖动**（严格） | 椭圆的轴对齐边界框必须完全在选框内 |
| **向左拖动**（交叉） | 椭圆轮廓与选框边界相交的任何部分都将选中它 |

## 支持的编辑命令

| 命令 | 对椭圆的影响 |
|---------|-----------------------------|
| [移动](./move) | 平移圆心；两轴不变 |
| [复制](./copy) | 在新圆心创建相同的椭圆 |
| [旋转](./rotate) | 旋转圆心位置和长轴向量 |
| [镜像](./mirror) | 镜像圆心并跨镜像轴重新计算长轴方向 |
| [缩放](./scale) | 缩放圆心位置并将两个半轴长度乘以缩放系数 |
| [偏移](./offset) | 创建以固定距离向外或向内偏移的同心椭圆 |
| [删除](./delete) | 删除椭圆 |

## 属性

**常规**

| 属性 | 默认值 | 含义 |
|----------|---------|---------|
| 颜色 | 256（随层） | ACI 颜色索引 |
| 图层 | `0` | 图层分配 |
| 线型 | 随层 | 命名线型样式 |
| 线型比例 | 1 | 线型样式的缩放系数 |
| 厚度 | 0 | 拉伸厚度 |

**几何**

| 属性 | 含义 |
|----------|---------|
| 圆心 X / 圆心 Y | 椭圆的圆心 |
| 长轴 X / 长轴 Y | 从圆心到长轴端点的向量（编码方向和长度） |
| 轴比 | 短半轴与长半轴之比（0 < 比值 ≤ 1） |
| 起始角度 / 终止角度 | 参数角度（度）；完整椭圆时均为 0°/360° |

## 椭圆 vs 圆 — 选用时机

| | 椭圆 | 圆 |
|---|---------|--------|
| 轴 | 任意角度的两条独立半轴 | 一个半径，对称 |
| 旋转 | 可以任意角度放置 | 无旋转 |
| 输入类型 | 每轴长度 | 仅半径 |
| 夹点调整 | 长轴和短轴独立 | 四个象限点相同 |
| 最适用于 | 斜视图、椭圆特征、透视孔 | 对称圆形特征 |

## DXF — ELLIPSE 实体

椭圆以 `ELLIPSE` 实体保存在 DXF 文件中。格式存储圆心点、完整长轴向量（方向 + 长度）和轴比。旋转、形状和所有样式属性均可完整往返转换。圆**不**作为退化椭圆存储 — 两种图元类型在 DXF 模型中保持独立。
