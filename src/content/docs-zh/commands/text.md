# Text

`text`（文字）命令放置多行文字标签。单击画布上的位置后，弹出编辑器在**富文本**模式下打开 — 您可以输入内容、按字符应用粗体/斜体/删除线、更改字体和高度，以及插入换行符。按 **Escape** 键提交并关闭编辑器。

有关编辑器的完整参考（包括**富文本**和**简单**模式的对比），请参阅[文字编辑器](../../interface/text-editor/)页面。

## 放置文字标签

1. 在命令行中输入 `text`，或单击工具栏中的**Text**按钮。
2. 在画布上**单击锚点位置**。或输入 `X,Y` 并按 **Enter** 键精确输入坐标。
3. **文字编辑器弹出框**在新标签上方打开。输入您的内容。
4. 按 **Escape** 提交标签并关闭编辑器。

默认高度为 **12 个图形单位**。

## 编辑现有标签

**双击**画布上的任意文字标签可重新打开该标签的编辑器。

## 锚点的坐标输入

可以输入精确位置，而不必单击：

1. 输入 X 值。
2. 按 `,` — 命令行显示 `[X], [Y{光标}]`。
3. 输入 Y 值。
4. 按 **Enter** 放置锚点并打开编辑器。

## 键盘参考

**锚点阶段**

| 按键 | 操作 |
|-----|--------|
| `0`–`9`、`.`、`-` | 开始输入 X 坐标 |
| `,` | 锁定 X 并移至 Y 输入 |
| `Backspace` | 删除最后输入的字符 |
| `Enter` | 确认输入的坐标 |

**文字编辑器阶段**（完整参考请见[文字编辑器](../../interface/text-editor/)）

| 按键 | 操作 |
|-----|--------|
| 任意可打印字符 | 在光标处插入 |
| `Backspace` / `Delete` | 删除相邻字符或选中内容 |
| `Enter` | 插入换行符 |
| `←` / `→` | 移动插入符 |
| `Home` / `End` | 跳到硬换行的起始/结束位置 |
| `Escape` | 提交并关闭编辑器 |

## 夹点编辑 — 重新定位

选中的文字标签在锚点处显示一个夹点：

| 夹点 | 位置 | 功能 |
|------|----------|--------------|
| **锚点** | 文字左下角 | 拖动以重新定位标签 |

## 选择文字

| 方法 | 行为 |
|--------|-----------|
| **单击** | 如果单击落在文字的旋转边界框内，则选中 |
| **向右拖动**（严格） | 边界框的四个角必须都在选区内 |
| **向左拖动**（交叉） | 文字边界框与选区之间的任何重叠都将选中它 |

## 支持的编辑命令

| 命令 | 对文字的影响 |
|---------|--------------------------|
| [Move](../move/) | 移动锚点 |
| [Copy](../copy/) | 在新位置创建相同的标签 |
| [Rotate](../rotate/) | 旋转锚点位置并将角度添加到旋转度数 |
| [Mirror](../mirror/) | 沿镜像轴反射锚点（文字字符串不翻转） |
| [Scale](../scale/) | 缩放锚点位置并将高度乘以缩放系数 |
| [Delete](../delete/) | 删除标签 |

文字不支持**偏移**、**修剪**或**延伸**。

## 属性

选中文字标签时，属性面板显示：

**常规**

| 属性 | 默认值 | 含义 |
|----------|---------|---------|
| 颜色 | 256（随层） | ACI 颜色索引 |
| 图层 | `0` | 图层分配 |

**几何**

| 属性 | 含义 |
|----------|---------|
| 位置 X / 位置 Y | 锚点坐标 |
| 高度 | 图形单位中的基础文字高度（默认：**12**） |
| 旋转角度 | 逆时针旋转度数 |

**属性**

| 属性 | 含义 |
|----------|---------|
| 内容 | 文字字符串（保留 MTEXT 内联代码） |
| 附着点 | 对齐代码（1 = 左上 … 9 = 右下） |

文字没有线型、线型比例或厚度属性。

## DXF — MTEXT 实体

文字标签以 **MTEXT** 实体存储在 DXF 文件中。粗体和斜体使用 `\L`、`\K`、`\O` 和内联字体切换（`\f`）编码。每字符高度使用 `\H` 编码。所有格式化在完整 DXF 往返转换后仍然存在，可被 LibreCAD、FreeCAD 和其他支持 DXF 的应用程序读取。
