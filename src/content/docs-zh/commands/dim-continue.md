# 连续标注

`dimcontinue`（连续标注）命令从现有标注的**第二条延伸线**链接新标注。每个新线段沿相同的测量轴放置，并与基准的标注线偏移量相同。所有样式属性（箭头大小、文字高度、延伸线长度）均自动从基准复制。

## 链式标注的外观

```
  |←— 3.00 —→|←— 2.50 —→|←— 4.00 —→|
  |           |           |           |
  ●           ●           ●           ●
  p1        p2（基准     p3           p4
           ext2 → 新起点）
```

每个矩形是一个独立的 `DIMENSION` 实体。它们共享相同的标注线位置和测量方向。

## 开始链式标注

1. 在命令行中输入 `dimcontinue`，或单击工具栏中的**连续标注**按钮。
2. **如果刚刚放置了标注** — 命令自动将其作为基准拾取（无需单击）。
3. **如果没有最近的标注** — 单击任意现有标注将其用作基准。
4. **单击下一条延伸线原点** — 移动光标时预览显示新标注。或输入 `X,Y` 并按 **Enter** 键精确输入坐标。
5. 继续单击（或输入）以延伸链。每个放置的标注自动成为新基准。
6. 按 **Enter**、**空格**或 **Escape** 完成链式标注。

## 从基准标注继承的属性

| 属性 | 是否从基准继承 |
|----------|---------------------|
| 测量方向 / 角度 | 是 — 整个链锁定 |
| 标注线偏移量（距测量点的距离） | 是 |
| 箭头大小 | 是 |
| 文字高度 | 是 |
| 延伸线偏移量和延伸量 | 是 |
| 文字对齐 | 是 |
| 样式名称 | 是 |
| 颜色、图层 | 不继承 — 使用当前图层 |

## 测量方向锁定

链的测量方向**固定为基准标注的角度**：

- 线性基准（H）→ 所有连续测量水平距离（Δ X）。
- 线性基准（V）→ 所有连续测量垂直距离（Δ Y）。
- 任意角度的对齐基准 → 所有连续沿该角度测量。

链中途无法改变方向。请启动新的[线性标注](../dim-linear/)或[对齐标注](../dim-aligned/)以在不同方向进行标注。

## 键盘参考

| 按键 | 操作 |
|-----|--------|
| `0`–`9`、`.`、`-` | 开始输入 X 坐标 |
| `,` | 锁定 X 并移至 Y 输入 |
| `Backspace` | 删除最后输入的字符 |
| `Enter` | 确认输入的坐标，或在没有输入的情况下完成链 |
| 空格 / `Escape` | 完成链 |

## 连续标注 vs 重新开始

| | 连续标注 | 线性标注 / 对齐标注 |
|---|-------------------|--------------------------|
| 起点 | 固定在上一个基准的 ext2 | 可单击任意位置 |
| 角度 | 锁定为基准 | 自由 |
| 偏移量 | 从基准继承 | 由光标或输入设置 |
| 样式 | 从基准继承 | 当前样式 |
| 最适用于 | 沿一排的累积测量 | 第一个标注或改变方向 |

## 放置后编辑标签 — 简单模式

**双击**链中的任意标注可在**简单**模式下打开文字编辑器。每个线段独立存在，可单独编辑。

| 功能 | 行为 |
|---------|-----------|
| Bold / Italic / Font / Height | 一次性应用于**整个**标签 |
| 每字符格式 | 不支持 |
| `Enter` | 提交值并关闭编辑器 |
| 多行 | 不支持 |

请参阅[文字编辑器 — 简单模式](../interface/text-editor#simple-mode)获取完整参考。

## DXF — DIMENSION 实体

链中的每个线段都作为独立的 `DIMENSION` 实体存储在 DXF 文件中。它们在文件中没有关联 — 它们共享属性是因为由同一个基准创建，但每个都可以在放置后单独编辑。
