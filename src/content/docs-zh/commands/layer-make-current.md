# 置为当前图层

`LayerMakeCurrent`（置为当前图层）命令将**当前绘图图层**设置为单击图元所属的图层。此后新图元将自动在该图层上绘制。

## 使用方法

1. 在命令行中输入 `LayerMakeCurrent`，或单击工具栏中的**置为当前图层**按钮（吸管图标）。
2. **单击画布上的任意图元**。
3. 当前图层更新为与该图元的图层匹配。命令立即完成。

## 行为详情

- 如果单击画布空白处（未命中任何图元），命令行显示 `no object found`，命令保持激活以便再次尝试。
- 只更改当前图层设置 — 不修改任何图元。
- 更新后的图层在工具栏的图层选择器中反映。
