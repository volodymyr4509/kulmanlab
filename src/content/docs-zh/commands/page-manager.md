# PageManager

`PageManager`（页面管理器）命令打开当前布局的页面设置对话框，让您更改图纸格式、方向以及图形单位与毫米之间的比例。

## 打开 PageManager

- 在命令行中输入 `PageManager`，**或**
- 右键单击底部的布局标签，选择**页面管理器**。

## 设置

| 设置 | 说明 |
|---------|-------------|
| **图纸格式** | 标准图纸尺寸（A4、A3、A2、Letter 等） |
| **方向** | 纵向或横向 |
| **比例** | 每毫米的图形单位 — 控制模型坐标如何映射到图纸 |

更改图纸大小或比例会按比例重新缩放布局中所有现有视口，以使其位置保持正确。

## 注意事项

- PageManager 仅适用于图纸布局。在模型空间中不可用。
- 要添加新布局，单击底部标签栏中的 **+** 按钮。
- 要重命名或删除布局，右键单击其标签。
