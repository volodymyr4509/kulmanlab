# 延伸

`extend`（延伸）命令将[直线](../line/)的最近端点延伸到它与图形中另一个图元的最近交点处。将光标悬停在要延伸的端点附近 — 预览显示延伸后的直线 — 然后单击以应用。

延伸仅适用于**直线图元**。直线延伸方向上的边界可以是任何其他图元类型，文字、多行文字、多重引线和样条线除外。

## 延伸直线

1. 在命令行中输入 `extend`，或单击工具栏中的**延伸**按钮。
2. **将光标悬停在直线的一端附近** — 预览显示直线延伸到该方向最近边界的效果。
3. **单击**以应用延伸。

每次延伸后命令保持激活状态，可以继续延伸多条直线。按 **Escape** 键退出。

```
  之前：                      之后：

  ──────           |           ──────────────|
  （短直线）     （边界）  （延伸到边界）
```

## 如何选择端点

命令查看光标距离哪个端点更近：

- 光标**更靠近终点** → 终点沿直线方向向前延伸。
- 光标**更靠近起点** → 起点沿相反方向向后延伸。

从所选端点沿直线方向投射射线，沿该射线与任何其他图元（排除直线本身和被忽略的类型）的**最近交点**成为新端点。

如果在该方向上找不到交点，则不显示预览，单击不会产生任何效果。

## 边界排除

以下图元类型被忽略作为边界 — 直线不会延伸到与它们相交处：

- 文字 / 多行文字
- 多重引线
- 样条线

所有其他类型（直线、圆弧、圆、椭圆、多段线、标注）均作为有效边界。

## 键盘参考

| 按键 | 操作 |
|-----|--------|
| `Escape` | 退出延伸模式 |

## 支持的图元

| 图元 | 可以延伸？ |
|--------|----------------|
| 直线 | 是 |
| 圆弧、圆、椭圆 | 否 |
| 多段线 / 矩形 | 否 |
| 文字、样条线、标注、引线 | 否 |

## 延伸 vs 修剪

| | 延伸 | 修剪 |
|---|--------|------|
| 功能 | 将直线端点延伸到边界 | 删除直线的一段 |
| 触发方式 | 悬停在要延伸的端点附近 | 悬停在要切割的线段上 |
| 结果 | 直线端点向外移动 | 直线分割或缩短 |
| 两者 | 仅直线 | 仅直线 |
