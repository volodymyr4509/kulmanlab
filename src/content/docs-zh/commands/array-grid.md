# Array Grid

`ArrayGrid`（阵列网格）命令根据选中图元创建矩形阵列副本 — 输入行数、列数以及它们之间的间距，全部在命令行中输入。原始选择占据第 0 行第 0 列单元格；其余每个单元格都是平移后的副本。

## 两种启动方式

**预先选择，然后阵列** — 先选择图元，再激活：

1. 在画布上选择一个或多个图元。
2. 在命令行中输入 `arraygrid`（输入 `arr` 即可 — 不会产生歧义），或单击工具栏中的**Array Grid** 按钮。
3. 输入**行数**并按 **Enter**。
4. 输入**列数**并按 **Enter**。
5. 输入**行间距**并按 **Enter**。
6. 输入**列间距**并按 **Enter** — 阵列会立即创建。

**激活后选择** — 在未选中任何内容的情况下启动命令：

1. 输入 `arraygrid` 或单击工具栏按钮。
2. **选择对象** — 单击以切换单个图元，或拖动以按区域选择。
3. 按 **Enter** 或**空格**确认选择。
4. 按照上面的顺序继续：行数 → 列数 → 行间距 → 列间距。

```
  2 行 x 3 列：

  [B] [B] [B]   <- 第 1 行（平移的副本）
  [A] [A] [A]   <- 第 0 行：原始选择，副本在右侧
```

> 命令行只需要足够消除歧义的字母 — 输入 `arr` 并按 **Enter** 即可直接激活 Array Grid，因为没有其他命令名称以这三个字母开头（Arc、Area、Align 和 Angle 都在更早处出现分歧）。

## 行、列和间距

| 提示 | 接受 | 说明 |
|------|------|------|
| 行数 | 正整数（1、2、3……） | 仅限数字 — 不含小数点或符号 |
| 列数 | 正整数（1、2、3……） | 仅限数字 — 不含小数点或符号 |
| 行间距 | 带符号小数（如 `10`、`-5.5`） | 行与行之间的距离；负值反转方向 |
| 列间距 | 带符号小数（如 `10`、`-5.5`） | 列与列之间的距离；负值反转方向 |

当行数和列数均为 1 时，不会创建任何副本 — 命令退出，不会更改图形。

## 键盘参考

| 按键 | 操作 |
|-----|--------|
| `Enter` / 空格 | 确认选择并进入行数提示 |
| `0`–`9` | 为行数或列数输入数字 |
| `0`–`9`、`.`、`-` | 为行/列间距输入数字（`-` 仅可作为第一个字符） |
| `Backspace` | 删除当前提示中最后输入的字符 |
| `Enter` | 确认当前提示并进入下一个 |
| `Escape` | 清除已输入的行/列/间距值并返回选择阶段 |

## 命令中的选择

| 方法 | 行为 |
|--------|-----------|
| **单击** | 切换光标下方图元的选中/取消状态 |
| **向右拖动**（严格） | 添加完全在选框内的图元 |
| **向左拖动**（交叉） | 添加与选框边界相交的图元 |
| **Enter** / 空格 | 确认选择并进入行数提示 |

## 阵列完成后

新副本被添加到图形中，命令随即退出 — 原始选择被清除。再次运行 **Array Grid**，或启动新命令。

## Array Grid 与 Copy 对比

| | Array Grid | Copy |
|---|-----------|------|
| 点选 | 无 — 行数、列数和间距均为输入 | 需要点击（或输入）基点和目标点 |
| 创建的副本数 | 行数 × 列数 − 1 | 每次复制操作恰好 1 个 |
| 布局 | 规则的矩形网格 | 任意位置，任意偏移 |
| 最适合 | 在规则图案中重复某个单元（孔位、瓷砖、紧固件） | 在任意位置生成单个副本 |

## 支持的图元

Array Grid 适用于每种图元类型。所有图元都在内部实现了 `translate(dx, dy)`，这与 [Copy](../copy/) 和 [Move](../move/) 使用的操作相同，因此没有任何图元被排除。
