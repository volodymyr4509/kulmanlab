# Pan

`pan`（平移）命令进入持续的拖动滚动模式 — 在画布上任意位置单击并拖动以移动视图。缩放级别不变。平移模式保持激活直到按 `Escape` 键，因此可以在一次激活中多次拖动。

## 平移视图

1. 在命令行中输入 `pan`，或单击工具栏中的**Pan**按钮。
2. 在画布上任意位置**单击并拖动**以移动视图。
3. 释放并根据需要多次拖动。
4. 按 `Escape` 退出平移模式。

## 中键拖动 — 无需激活命令即可平移

最快的平移方式：**按住鼠标中键并拖动**，随时可用，即使在另一个命令激活时也可以。无需激活任何命令。释放中键后返回之前的状态。

## Pan vs 缩放控件（Zoom Controls）

| 操作 | 效果 | 需要平移模式？ |
|--------|--------|-------------------|
| 左键单击拖动（平移模式） | 移动视口 | 是 |
| 中键拖动 | 移动视口 | 否 — 始终有效 |
| 滚轮 | 向光标方向缩放 | 否 — 始终有效 |
| 双击中键 | 适应所有图元 | 否 — 始终有效 |
| [Zoom In](../zoom-in/) / [Zoom Out](../zoom-out/) | 步进缩放 1.5× | 否 |
| [Fit](../fit/) | 适应所有图元到视图 | 否 |

## 键盘参考

| 按键 | 操作 |
|-----|--------|
| `Escape` | 退出平移模式 |

## 提示

- 使用滚轮缩放到目标区域，然后平移以微调位置。
- **双击鼠标中键**立即触发[Fit](../fit/) — 这是重置丢失视图的最快方式。
- 如果您正处于命令中（例如放置直线端点），可以中键拖动进行平移而不取消命令。
