# Zoom In

`zoomin`（放大）命令将当前缩放级别乘以 **1.5×**，立即退出，以视口中点为中心。这是向屏幕中心而非光标方向滚动一格滚轮的工具栏等效操作。

## 放大

单击工具栏中的**Zoom In**按钮，或在命令行中输入 `zoomin`。缩放立即应用，命令退出 — 无需在画布上单击。

## 1.5× 步长的工作原理

| 当前缩放 | 放大一次后 |
|-------------|------------------|
| 1.00× | 1.50× |
| 1.50× | 2.25× |
| 10.00× | 15.00× |
| 6,667× | 10,000×（已封顶） |

缩放级别始终显示在画布**右下角**的 `zoom` 标签旁边。上限为 **10,000×**；超过此限制的步骤不会产生任何效果。

## 工具栏放大 vs 滚轮

| | 放大按钮 | 滚轮 |
|---|--------------|-------------|
| 缩放中心 | 视口中点 | 光标位置 |
| 步长 | 每次单击 1.5× | 每格约 1.1× |
| 需要激活 | 否 | 否 — 始终有效 |
| 最适用于 | 粗略导航 | 精确的光标目标缩放 |

## 键盘参考

此命令没有键盘快捷键。请改用滚轮 — 它随时可用，无需激活任何命令。
