# Polyline

`polyline`（多段线）命令绘制任意数量直线段连接的路径，全部存储为单个 `LWPOLYLINE` 实体。因为整个路径是一个对象，选中它就选中了每段 — 在单次操作中移动、旋转或缩放整个形状。这是与连接的[Line](../line/)的关键区别，在连接的直线中每段是独立的图元。

多段线也可以**闭合**：[Rectangle](../rectangle/)命令使用相同的 `LWPOLYLINE` 实体，并设置了闭合标志。

## 绘制多段线

1. 在命令行中输入 `polyline`，或单击工具栏中的**Polyline**按钮。
2. **单击第一点**，或输入 `X,Y` 并按 **Enter** 键精确输入坐标。
3. **单击每个后续点** — 每次单击添加一段。坐标输入在每步都有效。
4. 按 **Enter** 或**空格**完成（至少需要放置 2 个点）。

```
  ●──────●
  第 1 点    第 2 点
          \
           \  第 3 段（进行中 — 光标在此）
            ●  ← 单击添加，Enter/空格完成
```

随时按 **Escape** 可丢弃所有已放置的点并退出命令。

## 坐标输入

可以输入精确位置，而不必单击任何顶点：

1. 输入 X 值。
2. 按 `,` — 命令行显示 `[X], [Y{光标}]`。
3. 输入 Y 值。
4. 按 **Enter** 放置顶点。

## 角度锁定和精确段长度

与[Line](../line/#angle-locking-and-exact-length-input)命令相同的 45° 捕捉逻辑适用于任意两个连续点之间。锁定到轴时：

| 按键 | 操作 |
|-----|--------|
| `0`–`9`、`.` | 在段长度后追加数字 |
| `-` | 负长度 — 沿轴线反向（仅第一个字符） |
| `Backspace` | 删除最后输入的字符 |
| `Enter` | 在输入的距离处放置下一个点 |

当前累积长度实时显示在命令行提示中。锁定时单击会投影到轴上，因此新顶点精确落在轴上。

## 键盘参考

| 按键 | 操作 |
|-----|--------|
| `0`–`9`、`.`、`-` | 开始输入 X 坐标，或角度锁定时输入段长度 |
| `,` | 锁定 X 并移至 Y 输入 |
| `Backspace` | 删除最后输入的字符 |
| `Enter` | 确认输入的坐标或长度，或未输入任何内容且 ≥ 2 个点时完成多段线 |
| 空格 | 完成多段线（无输入时与 Enter 相同） |
| `Escape` | 丢弃所有点并退出 |

## 夹点编辑 — 顶点和线段中点

选中的多段线显示两种类型的夹点：

| 夹点 | 位置 | 功能 |
|------|----------|--------------|
| **顶点** | 每个放置点处 | 拖动以重新定位该顶点；所有连接段随之延伸 |
| **线段中点** | 每段中心 | 拖动以**同时**平移该段的两个端点，保持段的长度和角度不变 |

线段中点夹点是多段线特有的 — 它可以在不改变长度的情况下横向滑动单个线段。在[Line](../line/)上，中点夹点则激活整个图元的移动命令。

没有单独的"移动整条多段线"夹点。要移动整个路径，请使用[Move](../move/)命令。

## 选择多段线

| 方法 | 行为 |
|--------|-----------|
| **单击** | 如果单击落在任何线段的命中测试距离内，则选中多段线 |
| **向右拖动**（严格） | 所有顶点必须在选框内 |
| **向左拖动**（交叉） | 与选框边界相交的任何线段都将选中整条多段线 |

因为多段线是一个图元，触及任何线段的交叉选择会选中所有线段。

## 支持的编辑命令

多段线支持所有常规变换和偏移，但**不**支持修剪或延伸（这些仅适用于[Line](../line/)）：

| 命令 | 对多段线的影响 |
|---------|------------------------------|
| [Move](../move/) | 所有顶点平移相同位移 |
| [Copy](../copy/) | 在新位置创建相同的多段线 |
| [Rotate](../rotate/) | 绕选定基点旋转所有顶点 |
| [Mirror](../mirror/) | 沿镜像轴反射所有顶点 |
| [Scale](../scale/) | 从基点统一缩放所有顶点 |
| [Offset](../offset/) | 创建固定垂直距离的平行多段线 |
| [Delete](../delete/) | 从图形中删除多段线 |

## 属性

选中多段线时，属性面板显示：

**常规**

| 属性 | 默认值 | 含义 |
|----------|---------|---------|
| 颜色 | 256（随层） | ACI 颜色索引 |
| 图层 | `0` | 图层分配 |
| 线型 | 随层 | 命名线型样式 |
| 线型比例 | 1 | 线型样式的缩放系数 |
| 厚度 | 0 | 拉伸厚度 |

**几何**

| 属性 | 含义 |
|----------|---------|
| 闭合 | 最后一个顶点是否连接到第一个 |
| 顶点数 | 顶点总数 |
| 顶点 | 所有顶点的坐标列表 |

## Polyline vs Line — 选用时机

| | 多段线 | 直线 |
|---|---------|------|
| 图元数量 | 整个路径一个 `LWPOLYLINE` | 每段一个 `LINE` |
| 闭合形状 | 是（闭合标志） | 否 |
| 修剪 / 延伸 | 否 | 是 — 逐段 |
| 线段中点夹点 | 平移整个线段 | 激活图元的移动 |
| 最适用于 | 轮廓线、外形、保持整体的形状 | 辅助线、需要修剪的几何 |

## DXF — LWPOLYLINE 实体

多段线以 `LWPOLYLINE` 实体保存在 DXF 文件中。所有属性（顶点坐标、闭合标志、颜色、图层、线型、线型比例和厚度）均可完整往返转换。用[Rectangle](../rectangle/)命令绘制的矩形也以 `LWPOLYLINE`（闭合、四个顶点）保存，在 DXF 级别无法区分。

来自任何支持 DXF 应用程序（LibreCAD、FreeCAD 等）的 `LWPOLYLINE` 实体在编辑器中作为完全可编辑的多段线读回。
