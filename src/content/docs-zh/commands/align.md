# Align

`align`（对齐）命令使用一对或两对源/目标点重新定位选中图元。只用一对点时，其行为与 [Move](../move/) 完全相同（仅平移）。用两对点时，它还会旋转选区，使源到源的方向与目标到目标的方向一致，并可选择缩放，使源线段长度与目标线段长度相匹配——平移、旋转和缩放合并为一次操作。

## 两种启动方式

**预先选择，然后对齐** — 先选择图元，再激活：

1. 在画布上选择一个或多个图元。
2. 在命令行中输入 `align`，或单击工具栏中的 **Align** 按钮。
3. **单击第一个源点（S1）**，然后**单击第一个目标点（D1）**。
4. **单击第二个源点（S2）**，或按 **Enter** 立即应用仅平移的对齐。
5. **单击第二个目标点（D2）**。
6. 回答缩放提示：按 **Y** 进行缩放，或按 **N** / **Enter** 保持原始大小。

**激活后选择** — 在未选中任何内容的情况下启动命令：

1. 输入 `align` 或单击工具栏按钮。
2. **选择对象** — 单击以切换单个图元，或拖动以按区域选择。
3. 按 **Enter** 或**空格**确认选择。
4. 按上述方式继续：S1 → D1 → S2 → D2 → 缩放提示。

> 命令行只需要输入足够消除歧义的字母——输入 `al` 并按 **Enter** 即可直接激活 Align，因为没有其他命令名以这两个字母开头。

## 对齐的结构

```
  源点（在图元上）：                    目标点：
  ● S1                                 ● D1
   \                                    \
    ● S2                                 ● D2

  结果：选区被平移，使 S1 落在 D1 上，然后围绕 D1 旋转，使
  S1→S2 方向与 D1→D2 方向一致——如果你选择缩放，还会调整大小，
  使 |S1S2| 变为 |D1D2|。
```

每一步都有幽灵预览跟随光标：放置 D1 时显示移动预览，放置 D2 时显示虚线旋转预览。

## 单点对齐（仅平移）

放置 D1 后，按 **Enter** 而不是单击第二个源点。选区将按 S1→D1 向量平移——不旋转也不缩放——与使用 S1 作为基点、D1 作为目标点的 [Move](../move/) 完全相同。

## 双点对齐（平移 + 旋转 + 可选缩放）

当 S2 和 D2 都放置好后：

- **旋转角度** — 目标方向（`D1 → D2`）与源方向（`S1 → S2`）之间的差值。
- **缩放提示** — 出现 `scale objects to alignment points? [Yes/No] <N>`，默认为 **No**：
  - 按 **Y** 同时围绕 D1 等比缩放选区，使 `S1–S2` 的距离变为 `D1–D2` 的距离。
  - 按 **N** 或 **Enter** 保持原始大小——仅应用平移和旋转。

在缩放提示处按下任意键会立即应用对齐——选择 Yes 或 No 后没有单独的确认步骤。

## 键盘参考

| 按键 | 操作 |
|-----|--------|
| `Enter` / 空格 | 确认选择并进入 S1 阶段 |
| `Enter`（在 S2 步骤） | 跳过旋转——仅使用 S1 和 D1 应用平移对齐 |
| `Y` | 应用带缩放的对齐 |
| `N` / `Enter`（在缩放提示处） | 应用不带缩放的对齐 |
| `Escape` | 选取点时：放弃已放置的点并返回选择阶段；未选中任何内容时：取消命令 |

## 命令中的选择

命令在选择阶段启动时：

| 方法 | 行为 |
|--------|-----------|
| **单击** | 切换光标下方图元的选中/取消状态 |
| **向右拖动**（严格） | 添加完全在选框内的图元 |
| **向左拖动**（交叉） | 添加与选框边界相交的图元 |
| **Enter** / 空格 | 确认选择并进入 S1 阶段 |

## 对齐之后

对齐后的图元在新位置保持选中状态，命令自动结束——再次运行 **Align**，或无需重新选择即可切换到 [Move](../move/)、[Rotate](../rotate/) 或 [Scale](../scale/)。

## Align vs Move

| | Align | Move |
|---|------|------|
| 点对数量 | 1 对（仅平移）或 2 对（平移 + 旋转 + 缩放） | 1 对（仅平移） |
| 旋转 | 有第二对点时支持 | 不支持 |
| 缩放 | 有第二对点时可选 | 不支持 |
| 最适用于 | 使用参考点将一个图形对齐到另一个图形 | 简单的重新定位 |

## 支持的图元

Align 适用于 Move、Rotate 和 Scale 支持的每种图元类型——这些命令使用的相同 `translate`、`rotate` 和 `scale` 操作会依次应用，因此没有任何图元被排除。
