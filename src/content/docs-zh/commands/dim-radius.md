# 半径标注

`dimradius`（半径标注）命令在圆弧或圆上放置半径标注。标注线从圆心沿光标方向延伸到圆周上的一点，标注为 `R <值>`。如果要标注完整直径，请使用[直径标注](../dim-diameter/)。

## 半径标注的结构

```
  ● （圆心）
   \
    \  R 5.00
     \
      ●────── 文字（光标侧）
   （弧点）
```

- **标注线** — 从圆心穿过弧点朝向光标，弧点处有箭头。
- **标签** — `R` 后跟半径值。

## 放置半径标注

1. 在命令行中输入 `dimradius`，或单击工具栏中的**半径标注**按钮。
2. **单击圆弧或圆**以选中它。
3. **移动光标**以确定标注线方向 — 弧点跟随从圆心到光标的方向。
4. **单击**以放置标注。

只能选择**圆弧**和**圆**图元。单击任何其他图元类型不会产生任何效果。

## 键盘参考

| 按键 | 操作 |
|-----|--------|
| `Escape` | 取消 |

## 半径标注 vs 直径标注

| | 半径标注 | 直径标注 |
|---|-----------------|-------------------|
| 测量内容 | 半径（圆心到边缘） | 直径（边缘经圆心到边缘） |
| 标注线 | 圆心 → 弧点 | 弧点 → 弧点（经圆心） |
| 标签前缀 | `R` | `⌀` |
| 箭头 | 一个（在弧点处） | 两个（在两个弧点处） |
| 最适用于 | 标注曲线特征的一侧 | 标注完整圆形尺寸 |

## 编辑标签 — 简单模式

**双击**已放置的半径标注可在**简单**模式下打开文字编辑器。编辑器预填充当前渲染值（例如 `R 5.00`），以便您定位光标并直接编辑。

| 功能 | 行为 |
|---------|-----------|
| Bold / Italic / Font / Height | 一次性应用于**整个**标签 |
| 每字符格式 | 不支持 |
| `Enter` | 提交值并关闭编辑器 |
| 多行 | 不支持 |

请参阅[文字编辑器 — 简单模式](../../interface/text-editor/#simple-mode)获取完整参考。

## DXF — DIMENSION 半径实体

半径标注以半径类型几何的 `DIMENSION` 实体保存，存储圆心坐标、弧点位置和测量的半径值。所有属性均可完整往返转换。
