# 特性匹配

`MatchProperties`（特性匹配）命令将**视觉和图层属性**从源图元复制到一个或多个目标图元。只有源图元和目标图元类型之间共享的属性才会被传递 — 几何图形永远不会改变。

## 激活方式

在样式面板中单击工具栏中的**特性匹配**按钮（滚筒图标），或在命令行中输入 `MatchProperties`。

## 工作流程

**先激活，然后拾取源：**

1. 在未预先选中任何内容的情况下输入 `MatchProperties` 或单击工具栏按钮。
2. **单击源图元** — 要复制属性的那个图元。
3. **单击每个目标图元**以应用源属性。可以逐个单击多个图元。
4. 要一次应用于一组，**在目标上拖动选框**。
5. 按 **Enter** 或 **Escape** 完成。

**预先选中源，然后激活：**

1. 单击单个图元以选中它。
2. 激活 `MatchProperties`。所选图元自动用作源。
3. 单击目标图元或拖动选择，然后按 **Enter** 或 **Escape** 完成。

## 复制哪些属性

特性匹配复制源图元和目标图元之间共享基类的属性。至少，**所有图元类型**共享这些属性：

| 属性 | 说明 |
|----------|-------------|
| **颜色** | 图元的颜色索引（包括"随层"/"随块"） |
| **图层** | 图元所属的图层 |

当源图元和目标图元是同一图元类型（例如都是标注）时，还会复制其他特定类型的属性 — 例如文字高度、箭头大小、延伸线设置。

几何图形（坐标、半径、长度等）永远不受影响。

## 键盘参考

| 按键 | 操作 |
|-----|--------|
| `Enter` / 空格 | 确认区域选择或完成命令 |
| `Escape` | 完成应用（如果源已设置）或取消 |

## 行为详情

- 源图元本身永远不会被修改。
- 每次单击或拖动选择都会立即应用源属性 — 没有确认步骤。
- 区域选择遵循标准规则：**向右**拖动进行严格选择（完全封闭），**向左**拖动进行交叉选择（任何交叉）。
- 将源图元作为目标单击时被忽略。
- 对于包含文字的图元(**文字**、**标注**、**多重引线**),仅复制文字高度 — 字体、粗体、斜体等文字样式设置不会被匹配。
