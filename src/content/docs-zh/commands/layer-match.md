# LayerMatch

`LayerMatch`（图层匹配）命令将选中图元的图层重新指定为与您单击的源图元的图层相匹配。这是将一组对象移动到正确图层最快的方式，无需打开图层管理器。

## 工作流程

**预先选择，然后匹配**：

1. 选择要更改图层的图元。
2. 输入 `LayerMatch` 或单击工具栏中的**LayerMatch**按钮（画笔图标）。
3. **单击源对象** — 要复制图层的那个图元。
4. 所有选中的图元立即移动到源对象的图层。

**激活后选择**：

1. 在未选中任何内容的情况下输入 `LayerMatch` 或单击工具栏按钮。
2. **拾取目标对象** — 单击以切换单个图元，或拖动以按区域选择。
3. 按 **Enter** 或**空格**确认选择。
4. **单击源对象** — 其图层应用于所有目标。

## 键盘参考

| 按键 | 操作 |
|-----|--------|
| `Enter` / 空格 | 确认目标选择并进入源拾取阶段 |
| `Escape` | 重置 — 返回目标选择或完全取消 |

## 行为详情

- 只更改 `layer`（图层）属性 — 颜色、线型、线宽和几何图形不受影响。
- 源对象本身不被修改。
- 单击源后命令完成。
- 在源拾取阶段单击画布空白处不会产生任何效果。
