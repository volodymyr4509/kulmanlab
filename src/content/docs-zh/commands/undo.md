# Undo

`undo`（撤销）命令撤销对图形的最后一次更改 — 每次调用一步。每次添加、删除或编辑图元都作为单独的历史条目记录。撤销以相反顺序后退这些条目。

## 如何撤销

- 在命令行中输入 `undo`，或
- 单击工具栏中的**Undo**按钮。

每次调用撤销一个已记录的操作。反复调用以进一步后退。

## 历史行为

| 详情 | 值 |
|--------|-------|
| 每文件步数 | 最多 **20** |
| 存储 | 浏览器（IndexedDB / localStorage），按文件名 |
| 在页面重新加载后仍然存在 | 是 — 重新打开文件时历史记录被恢复 |
| 撤销后执行新操作 | 清除当前位置之前的所有重做条目 |
| 满时最旧的条目 | 被丢弃以为最新更改腾出空间 |

每次图元变化都被记录：绘制新图元、删除图元、夹点编辑端点、应用移动、旋转、缩放、镜像、修剪、延伸和偏移均创建历史条目。

## Undo vs Redo

| | 撤销 | 重做 |
|---|------|------|
| 方向 | 在历史记录中**向后**步进 | 在撤销条目中**向前**步进 |
| 可用条件 | 至少存在一个已记录的操作 | 至少执行了一次撤销且没有执行新操作 |
| 被以下清除 | 无 — 历史记录一直累积直到 20 步限制 | 任何新的绘图操作 |

使用[Redo](../redo/)重新应用已撤销的操作。当相应方向不可用时，工具栏按钮变灰。
