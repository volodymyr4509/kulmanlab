# 圆

`circle`（圆）命令绘制由圆心和半径定义的圆。单击圆心后，您可以通过在画布上单击第二个点或输入精确数值来设置半径 — 两种选项同时可用。

## 绘制圆

1. 在命令行中输入 `circle`，或单击工具栏中的**圆**按钮。
2. **单击圆心点**，或输入 `X,Y` 并按 **Enter** 键精确输入坐标。
3. 设置半径 — 可以：
   - 在画布上**单击任意点** — 从圆心到该点的距离成为半径，或
   - **输入半径**并按 **Enter** 键精确设置值。

圆立即放置，命令退出。

```
  圆心 ●
          \  半径线预览
           \
            ● ← 在此单击，或输入数字
```

在半径阶段，实时预览以当前光标距离显示圆，并绘制从圆心到当前点的半径线。

## 圆心坐标输入

可以输入圆心位置，而不必单击：

1. 输入 X 值。
2. 按 `,` — 命令行显示 `[X], [Y{光标}]`。
3. 输入 Y 值。
4. 按 **Enter** 放置圆心并进入半径输入。

## 键盘输入半径

放置圆心后，立即开始输入半径值：

| 按键 | 操作 |
|-----|--------|
| `0`–`9`、`.` | 在半径值后追加数字 |
| `Backspace` | 删除最后输入的字符 |
| `Enter` | 以输入的半径放置圆 |

命令行提示中实时显示累积的值（例如 `enter radius of circle: 25`）。预览更新以显示输入的半径，光标控制半径线标记的方向。

## 键盘参考

| 按键 | 操作 |
|-----|--------|
| `0`–`9`、`.`、`-` | 开始输入 X 坐标（圆心阶段），或输入半径数字（半径阶段） |
| `,` | 锁定 X 并移至 Y 输入（圆心阶段） |
| `Backspace` | 删除最后输入的字符 |
| `Enter` | 确认输入的坐标或半径 |
| `Escape` | 取消并重置 |

## 夹点编辑 — 调整半径

选中的圆显示五个夹点：

| 夹点 | 位置 | 功能 |
|------|----------|--------------|
| **圆心** | 圆心点 | 移动整个圆；半径不变 |
| **左** | 最左点（圆心 − 半径） | 拖动以设置新半径 = 到圆心的距离 |
| **右** | 最右点（圆心 + 半径） | 拖动以设置新半径 = 到圆心的距离 |
| **上** | 最高点 | 拖动以设置新半径 = 到圆心的距离 |
| **下** | 最低点 | 拖动以设置新半径 = 到圆心的距离 |

四个象限夹点行为相同 — 新半径等于从圆心到拖动位置的距离。圆心保持固定。

## 选择圆

| 方法 | 行为 |
|--------|-----------|
| **单击** | 如果单击落在圆周附近，则选中 |
| **向右拖动**（严格） | 整个边界正方形（圆心 ± 半径）必须在选框内 |
| **向左拖动**（交叉） | 圆周与选框边界相交或接触的任何部分都将选中该圆 |

## 支持的编辑命令

| 命令 | 对圆的影响 |
|---------|---------------------------|
| [移动](./move) | 平移圆心；半径不变 |
| [复制](./copy) | 在新圆心创建相同的圆 |
| [旋转](./rotate) | 绕基点旋转圆心；半径不变 |
| [镜像](./mirror) | 沿镜像轴反射圆心；半径不变 |
| [缩放](./scale) | 缩放圆心位置并将半径乘以缩放系数 |
| [偏移](./offset) | 创建半径更大或更小的同心圆 |
| [删除](./delete) | 删除圆 |

## 属性

**常规**

| 属性 | 默认值 | 含义 |
|----------|---------|---------|
| 颜色 | 256（随层） | ACI 颜色索引 |
| 图层 | `0` | 图层分配 |
| 线型 | 随层 | 命名线型样式 |
| 线型比例 | 1 | 线型样式的缩放系数 |
| 厚度 | 0 | 拉伸厚度 |

**几何**

| 属性 | 含义 |
|----------|---------|
| 圆心 X / 圆心 Y | 圆心点坐标 |
| 半径 | 图形单位中的圆半径 |

## 圆 vs 圆弧 — 选用时机

| | 圆 | 圆弧 |
|---|--------|-----|
| 范围 | 完整 360° | 局部 — 由起始和终止角度定义 |
| 绘制方式 | 圆心 + 半径 | 曲线上的三个点 |
| 输入类型 | 半径值 | 无 — 仅单击 |
| 调整夹点 | 4 个象限点 | 起点和终点（角度 + 半径） |
| 标注 | 半径：[半径标注](./dim-radius) · 直径：[直径标注](./dim-diameter) | [半径标注](./dim-radius) |
| 最适用于 | 完整孔、螺栓圆、圆形特征 | 圆角、局部曲线、拱形路径 |

## DXF — CIRCLE 实体

圆以 `CIRCLE` 实体保存在 DXF 文件中。圆心坐标、半径、颜色、图层、线型、线型比例和厚度均可完整往返转换。任何支持 DXF 的应用程序均可将其作为标准圆读取。
