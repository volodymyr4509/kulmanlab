# 对齐标注

`dimaligned`（对齐标注）命令放置一个测量两点之间**真实直线距离**的标注。标注线平行于连接两点的直线，因此可以是任意角度。这是与[线性标注](./dim-linear)的关键区别，后者仅限水平或垂直方向。

## 对齐标注的结构

```
     ●  p2
    /|
   / |  （延伸线 2，垂直于标注线）
  /  |
 /←5.00→/
/  /
●  /  （延伸线 1，垂直于标注线）
p1
```

- **延伸线** — 垂直于标注线，从每个测量点引出。
- **标注线** — 平行于 p1→p2，偏移到光标位置一侧。
- **标注值** — 真实欧几里得距离 `|p1 – p2|`。

## 放置对齐标注

1. 在命令行中输入 `dimaligned`，或单击工具栏中的**对齐标注**按钮。
2. **单击第一条延伸线原点**（p1），或输入 `X,Y` 并按 **Enter** 键精确输入坐标。
3. **单击第二条延伸线原点**（p2）。此处也可以输入坐标。
4. **移动光标**到一侧以设置标注线的垂直偏移量。
5. **单击**放置，或输入偏移距离并按 **Enter** 键精确放置。

## 输入偏移距离

在放置阶段输入数字可将标注线固定在距 p1→p2 线的精确垂直距离处：

| 按键 | 操作 |
|-----|--------|
| `0`–`9`、`.` | 在偏移量后追加数字 |
| `Backspace` | 删除最后输入的字符 |
| `Enter` / 空格 | 在输入的偏移量处放置 |

光标所在侧决定标注线出现在哪一侧。

## 键盘参考

| 按键 | 操作 |
|-----|--------|
| `0`–`9`、`.`、`-` | 开始输入 X 坐标（p1/p2 阶段），或输入偏移距离（放置阶段） |
| `,` | 锁定 X 并移至 Y 输入（p1/p2 阶段） |
| `Backspace` | 删除最后输入的字符 |
| `Enter` / 空格 | 确认输入的坐标或偏移量 |
| `Escape` | 取消 |

## 对齐标注 vs 线性标注

| | 对齐标注 | 线性标注 |
|---|------------------|-----------------|
| 标注线角度 | 平行于 p1→p2 — 任意角度 | 始终水平或垂直 |
| 测量内容 | 真实欧几里得距离 | 仅 X 或 Y 分量 |
| H/V 方向锁定 | 否 | 是 — `H` 和 `V` 键 |
| 最适用于 | 斜向特征、斜切 | 正交布局、网格对齐零件 |

## 编辑标签 — 简单模式

**双击**已放置的对齐标注可在**简单**模式下打开文字编辑器。编辑器预填充当前渲染值，以便您定位光标并直接编辑。

| 功能 | 行为 |
|---------|-----------|
| Bold / Italic / Font / Height | 一次性应用于**整个**标签 |
| 每字符格式 | 不支持 |
| `Enter` | 提交值并关闭编辑器 |
| 多行 | 不支持 |

请参阅[文字编辑器 — 简单模式](../interface/text-editor#simple-mode)获取完整参考。

## 链式标注

要从该标注的第二条延伸线继续添加更多标注，请使用[连续标注](./dim-continue) — 它锁定与该对齐标注相同的测量角度。

## DXF — DIMENSION 实体（对齐类型）

对齐标注以 `dimType = 1`（对齐）的 `DIMENSION` 实体保存在 DXF 文件中。延伸线原点、标注线位置、文字位置、测量值、旋转角度、箭头样式和所有显示标志均可完整往返转换。
