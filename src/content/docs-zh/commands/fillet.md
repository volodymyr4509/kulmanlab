# Fillet

`fillet`（圆角）命令通过插入给定半径的切弧并将每条直线修剪至弧的起点，使两条[Line](../line/)图元之间的角变为圆角。结果是连接两条直线的平滑圆角。

圆角仅适用于**直线图元**。

## 使用圆角

1. 在命令行中输入 `fillet`，或单击工具栏中的**Fillet**按钮。
2. **输入圆角半径**并按 **Enter** 键。
3. **单击第一条直线** — 单击位置决定在任意交点的哪一侧保留。
4. **将光标悬停在第二条直线上** — 虚线弧预览显示产生的圆角效果。将光标移至要保留的一侧。
5. **单击**以应用。两条直线均被修剪，圆弧被插入。

```
  之前：                     圆角后（半径 r）：

  ──────────────              ──────────╮
                │                        ╰────
                │
```

## 相交直线的边侧选择

当两条直线相互交叉时，圆角应用于由单击位置定义的角 — 每条直线上**与光标同侧**的部分被保留。

- 单击第一条直线的一端附近以选择该半部分。
- 将光标移到第二条直线的所需半部分 — 虚线预览实时更新。

## 命令创建的内容

- 第一条直线距交点最近的端点被移动到切点 **T1**。
- 第二条直线距交点最近的端点被移动到切点 **T2**。
- 从 **T1** 到 **T2** 插入一条新圆弧图元，与两条直线相切。

插入的圆弧继承当前的线宽、颜色、图层和线型设置。

## 键盘参考

| 按键 | 操作 |
|-----|--------|
| `0`–`9`、`.` | 在半径值后追加数字 |
| `Backspace` | 删除最后输入的字符 |
| `Enter` | 确认输入的半径并进入直线选择 |
| `Escape` | 取消并重置 |

## 支持的图元

| 图元 | 是否支持 |
|--------|-----------|
| 直线 | 是 — 作为第一和第二图元 |
| 圆弧、圆、椭圆、多段线 | 否 |
| 文字、样条线、标注、引线 | 否 |

## Fillet vs Chamfer

| | 圆角 | 倒角 |
|---|--------|---------|
| 角的类型 | 圆弧圆角 | 直线切割 |
| 输入 | 一个半径 | 两个距离（d1, d2） |
| 插入图元 | 圆弧 | 直线 |
| 支持的图元 | 仅直线 | 直线和多段线 |
