# 清除存储

`wipestorage`（清除存储）命令永久删除 KulmanLab CAD **在浏览器中存储的所有数据** — 每个已保存的文件、图层和线型表，以及撤销历史。之后页面自动重新加载。

:::danger 不可撤销
此操作无法撤销。浏览器中存储的所有文件都将被删除。在运行此命令之前，请将您想保留的所有图形导出为 `.json` 或 `.dxf` 文件。
:::

## 使用时机

- 浏览器存储损坏，应用程序无法加载或保存文件。
- 想要将应用程序完全重置为干净状态。
- 正在切换浏览器或设备，不再需要本地副本。

## 如何运行

1. 在命令行中输入 `wipestorage` 并按 **Enter** 键。
2. 命令行提示：*Wipe all browser local storage? Type YES to confirm*
3. 输入 `YES`（任意大小写）并按 **Enter** 键。

应用程序删除数据库并重新加载页面。如果输入的不是 `YES` 并按 **Enter**，或按 **Escape**，命令被取消，不删除任何内容。

## 删除的内容

| 数据 | 是否删除 |
|------|---------|
| 浏览器中保存的所有文件 | 是 |
| 每个文件的图层和线型表 | 是 |
| 每个文件的撤销/重做历史 | 是 |

只有**此浏览器**中本地存储的数据受到影响。您已导出为 `.json` 或 `.dxf` 的文件不受影响。
