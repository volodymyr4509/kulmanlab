# 多重引线

`leader`（多重引线）命令通过四个步骤绘制多重引线注释：触及特征的箭头、在折点弯曲的引线、文字锚点和输入的标签。在所有注释命令中，多重引线是唯一包含带闪烁光标预览的交互式文字输入阶段的命令。

## 多重引线的结构

```
  ◄── 箭头尖端  （第 2 步 — 触及特征）
      \
       \  引线
        \
         ●──── 折点（第 3 步）──── 文字锚点（第 4 步）
                                    标签文字  （第 5 步）
```

- **箭头尖端** — 放置在被注释特征处的尖端。
- **折点** — 引线向文字弯曲的肘部。
- **文字锚点** — 标签定位的位置。文字自动左对齐或右对齐。

## 绘制多重引线

1. 在命令行中输入 `leader`，或单击工具栏中的**多重引线**按钮。
2. **单击箭头尖端**，或输入 `X,Y` 并按 **Enter** 键精确输入坐标。
3. **单击折点** — 引线的弯折处。角度锁定到 45° 增量；输入长度并按 **Enter** 精确放置。或输入 `X,Y` 进行绝对坐标输入。
4. **单击文字位置** — 标签锚定的位置。同样适用：单击、角度锁定 + 长度，或 `X,Y`。
5. **输入标签文字** — 画布预览随带闪烁光标的文字实时更新。按 **Enter** 放置。

## 坐标输入（所有点阶段）

在任意拾取步骤（尖端、折点、文字位置），都可以输入精确坐标而不必单击：

1. 输入 X 值（数字、`.` 或 `-`）。
2. 按 `,` — 命令行显示 `[X], [Y{光标}]`，确认 X 已锁定。
3. 输入 Y 值。
4. 按 **Enter** 放置该点。

## 角度锁定（第 3 和第 4 步）

放置每个点后，当光标距离足够远时，命令会捕捉到 45° 轴。锁定时：
- 预览捕捉到轴。
- 输入长度并按 **Enter** 可在该精确距离处放置下一个点。

角度锁定和坐标输入互斥 — 一旦先输入数字而不是 `,`，命令将其解释为距离（必须激活角度锁定）。要改为输入绝对坐标，以 X 数字加逗号开头。

## 文字标签编辑

在第 5 步输入标签时，可以在放置前导航和编辑文字：

| 按键 | 操作 |
|-----|--------|
| 任意可打印字符 | 在光标位置插入 |
| `←` / `→` | 向左或向右移动光标 |
| `Backspace` | 删除光标左侧的字符 |
| `Delete` | 删除光标右侧的字符 |
| `Enter` | 放置多重引线 |

## 自动文字方向

文字对齐根据光标相对于折点的位置自动调整：

| 光标位置 | 文字方向 |
|-----------------|---------------|
| 折点**右侧** | 从文字锚点向左到右 |
| 折点**左侧** | 从右到左（锚定在右侧） |

无需手动调整 — 将光标移到标签所在的一侧，它会自动正确对齐。

## 键盘参考

**点阶段（尖端、折点、文字位置）**

| 按键 | 操作 |
|-----|--------|
| `0`–`9`、`.`、`-` | 开始输入 X 坐标（然后按 `,` 锁定 X 并输入 Y） |
| `,` | 确认 X 并移至 Y 输入 |
| `0`–`9`、`.`、`-` | 角度锁定时累积距离 |
| `Backspace` | 删除最后输入的字符 |
| `Enter` | 确认输入的坐标或距离 |

**文字输入阶段**

| 按键 | 操作 |
|-----|--------|
| 可打印字符 | 在光标处插入 |
| `←` / `→` | 移动光标 |
| `Backspace` | 向左删除 |
| `Delete` | 向右删除 |
| `Enter` | 放置多重引线 |

| 按键 | 操作 |
|-----|--------|
| `Escape` | 取消并重置至第 2 步 |

## 编辑已有引线

**双击**已放置的多重引线可在**富文本**模式下重新打开文字编辑器。在富文本模式下，您可以应用粗体、斜体及每字符的字体或高度覆盖，并用 `Enter` 插入换行符。按 **Escape** 提交并关闭。

请参阅[文字编辑器 — 富文本模式](../interface/text-editor#rich-mode)获取完整参考。

## 添加和删除引线臂

- 要从拥有两条或以上引线臂的引线中删除一条：[删除引线](../leader-remove/)

## DXF — 仅支持导入

**多重引线仅支持导入。** DXF 文件中的 `MLEADER` 图元可以正确读取和显示，但在编辑器中绘制的引线**在保存 DXF 文件时不写入**。将引线用于可视注释；避免在往返工作流中依赖它们。
