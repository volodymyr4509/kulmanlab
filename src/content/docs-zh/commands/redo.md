# 重做

`redo`（重做）命令在撤销历史中向前移动，重新应用被[撤销](./undo)命令撤销的操作。重做仅在您已用撤销命令后退且尚未进行新更改时可用。

## 如何重做

- 在命令行中输入 `redo`，或
- 单击工具栏中的**重做**按钮。

每次调用重新应用一个之前撤销的操作。反复调用以在所有可用重做条目中向前步进。

## 重做堆栈行为

| 详情 | 行为 |
|--------|-----------|
| 在以下情况后可用 | 一次或多次[撤销](./undo)步骤 |
| 被以下清除 | **任何新的绘图操作** — 添加、编辑或删除图元 |
| 存储 | 浏览器，按文件 — 在页面重新加载后仍然存在（只要重新加载前没有执行新操作） |
| 最大深度 | 最多 20 个条目（与撤销共用同一池） |

一旦绘制、删除或修改了新图元，重做堆栈就会被清除，这些条目无法恢复。只有尚未被新工作取代的撤销操作才能重做。

## 重做 vs 撤销

| | 重做 | 撤销 |
|---|------|------|
| 方向 | 在撤销条目中**向前**步进 | 在历史记录中**向后**步进 |
| 可用条件 | 至少执行了一次撤销，且没有执行新操作 | 至少存在一个已记录的操作 |
| 被以下清除 | 任何新的绘图操作 | 无 |

当没有可重做的条目时，工具栏重做按钮变灰。先使用[撤销](./undo)以创建重做条目。
