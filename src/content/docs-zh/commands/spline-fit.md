# 拟合样条线

`splinefit`（拟合样条线）命令绘制精确经过每个单击点的三次样条线 — 一条插值曲线。与 [CV 样条线](./spline-cv)不同，在 CV 样条线中曲线只是被吸引向控制顶点，这里的曲线被强制精确经过每个单击坐标。编辑器内部拟合控制顶点以实现这一点，这些 CV 与拟合点一起存储在 DXF 文件中。

## 通过拟合点绘制样条线

1. 在命令行中输入 `splinefit`，或单击工具栏中的**拟合样条线**按钮。
2. **单击以放置拟合点** — 曲线将经过每个点。或输入 `X,Y` 并按 **Enter** 键精确输入坐标。
3. 按 **Enter** 完成（至少需要 2 个点）。

```
  ●──────●──────●──────●  ← 曲线精确经过每次单击
  p1     p2     p3     p4
```

实时预览在移动光标时显示当前插值曲线，包括光标位置处的预期下一个点。按 **Escape** 可丢弃所有已放置的点并退出。

## 坐标输入

可以输入精确位置，而不必单击任何拟合点：

1. 输入 X 值。
2. 按 `,` — 命令行显示 `[X], [Y{光标}]`。
3. 输入 Y 值。
4. 按 **Enter** 放置拟合点。

## 键盘参考

| 按键 | 操作 |
|-----|--------|
| `0`–`9`、`.`、`-` | 开始输入 X 坐标 |
| `,` | 锁定 X 并移至 Y 输入 |
| `Backspace` | 删除最后输入的字符 |
| `Enter` | 确认输入的坐标，或未输入且 ≥ 2 个点时完成样条线 |
| `Escape` | 丢弃所有点并退出 |

## 夹点编辑 — 通过拟合点重塑

选中的拟合样条线每个拟合点显示一个夹点：

| 夹点 | 位置 | 功能 |
|------|----------|--------------|
| **拟合点** | 每个单击位置 | 拖动以移动该拟合点 — 整条曲线重新插值以经过新位置 |

拖动一个夹点会重新拟合整条曲线，而不仅仅是相邻段。这与多段线夹点编辑不同，在多段线中移动顶点只重塑两个相邻段。

没有"移动整条样条线"夹点。要平移整条样条线，请使用[移动](./move)命令。

## 选择拟合样条线

| 方法 | 行为 |
|--------|-----------|
| **单击** | 如果单击落在曲线上的任意点附近，则选中 |
| **向右拖动**（严格） | 沿曲线的所有采样点必须在选框内 |
| **向左拖动**（交叉） | 与选框边界相交的曲线任意部分都将选中 |

## 支持的编辑命令

| 命令 | 对样条线的影响 |
|---------|---------------------------|
| [移动](./move) | 所有拟合点和重新计算的 CV 平移相同位移 |
| [复制](./copy) | 在新位置创建相同的样条线 |
| [旋转](./rotate) | 绕选定基点旋转所有拟合点 |
| [镜像](./mirror) | 沿镜像轴反射所有拟合点 |
| [缩放](./scale) | 从基点统一缩放所有拟合点 |
| [删除](./delete) | 删除样条线 |

样条线不支持**偏移**、**修剪**或**延伸**。

## 属性

**常规**

| 属性 | 默认值 | 含义 |
|----------|---------|---------|
| 颜色 | 256（随层） | ACI 颜色索引 |
| 图层 | `0` | 图层分配 |
| 线型 | 随层 | 命名线型样式 |
| 线型比例 | 1 | 线型样式的缩放系数 |
| 厚度 | 0 | 拉伸厚度 |

**几何**

| 属性 | 含义 |
|----------|---------|
| 次数 | 多项式次数 — 始终为 3（三次） |
| 拟合点 | 所有单击穿越点的坐标 |
| 控制顶点 | 用于渲染曲线的内部计算 CV |

## 拟合样条线 vs CV 样条线 — 选用哪个

| | 拟合样条线 | CV 样条线 |
|---|------------|-----------|
| 曲线经过点 | 每个单击点精确经过 | 仅第一个和最后一个（固定端点） |
| 夹点编辑效果 | 拟合点移动 → 整条曲线重新插值 | CV 移动 → 曲线向新位置吸引 |
| 形状可预测性 | 高 — 曲线跟随单击 | 较低 — 曲线滞后于 CV |
| 最适用于 | 必须经过特定坐标的曲线 | 平滑美观曲线、自由形态路径 |

## DXF — SPLINE 实体（拟合点形式）

拟合样条线以 `SPLINE` 实体保存在 DXF 文件中，同时存储拟合点坐标和计算出的控制顶点。`splineFlag` 设置为 `8`（拟合点样条线），因此重新加载的应用程序知道哪组点显示为可编辑夹点。所有属性（颜色、图层、线型、线型比例和厚度）均可完整往返转换。支持拟合点样条线的 DXF 应用程序（LibreCAD、FreeCAD）将拟合点显示为主要可编辑数据。
