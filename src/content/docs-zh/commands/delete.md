# 删除

`delete`（删除）命令从图形中删除选中的图元。删除操作记录在[撤销](../undo/)历史中，最多可逆转 20 步。没有单独的"确认删除"对话框 — 确认只需按一个键。

## 两种删除方式

**预先选择，然后删除** — 最快捷的方式：

1. 在画布上选择一个或多个图元。
2. 在命令行中输入 `delete`，单击工具栏中的**删除**按钮，**或直接按 `Delete` 键**。

图元立即被删除 — 无需额外的确认步骤。

**激活后选择**：

1. 输入 `delete` 或单击工具栏按钮（未选中任何内容）。
2. **选择对象** — 单击以切换，或拖动以按区域选择。
3. 按 **Enter**、**空格**或 **Delete** 确认并删除选中的图元。

## Delete 键快捷方式

键盘上的 `Delete` 键充当**全局快捷键** — 如果当前有图元被选中，按下它会立即删除它们，即使没有在命令行中打开删除命令。这是最快的单步删除工作流：

```
单击图元 → 按 Delete 键 → 完成
```

## 命令中的选择

| 方法 | 行为 |
|--------|-----------|
| **单击** | 切换光标下方图元的选中/取消状态 |
| **向右拖动**（严格） | 仅选择完全在选框内的图元 |
| **向左拖动**（交叉） | 选择与选框边界相交的图元 |
| **Enter** / 空格 / **Delete** | 确认并删除选中的图元 |

## 恢复已删除的图元

删除操作可使用[撤销](../undo/)命令撤销（输入 `undo` 或使用工具栏按钮）。每个文件最多可逆转 **20 步**，历史记录在页面重新加载后仍然保留。如果在不保存的情况下执行了超过 20 次删除，则无法恢复较早的删除操作。

## 支持的图元

删除适用于每种图元类型 — 直线、多段线、矩形、圆、圆弧、椭圆、文字、样条线、标注、引线及所有其他类型。
