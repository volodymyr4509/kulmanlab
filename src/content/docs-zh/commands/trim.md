# Trim

`trim`（修剪）命令删除[Line](../line/)位于两个相邻交点之间的部分，将直线分割为一个或两个较短的线段。要切割的线段由光标位置决定 — 将光标悬停在要删除的部分上并单击以修剪。

修剪仅适用于**直线图元**。对于圆弧、圆、多段线和其他图元类型，请改用[Delete](../delete/)或夹点编辑。

## 修剪直线

1. 在命令行中输入 `trim`，或单击工具栏中的**Trim**按钮。
2. **将光标悬停在要删除的线段上** — 红色预览精确高亮显示将被切割的部分。
3. **单击**以删除该线段。

每次修剪后命令保持激活，可以继续悬停和单击以切割更多线段。按 **Escape** 退出。

```
  之前：                     修剪中间段后：

  ──────●──────●──────        ──────●          ●──────
      交点  交点       （左部分）  （右部分）
                                 （中间段已删除）
```

## 如何确定修剪线段

命令将光标位置投影到悬停直线上，并找到直线与其他图元的所有交点。这些交点参数将直线分成若干段。光标投影所在区间对应的线段被高亮显示，单击后将被删除。

- 如果光标在**第一个交点之前**：直线的前导部分被删除。
- 如果光标在**两个交点之间**：中间部分被删除；直线分成两段。
- 如果光标在**最后一个交点之后**：尾随部分被删除。
- 如果直线与其他图元**没有交点**：不显示预览，单击不会产生任何效果。

## 键盘参考

| 按键 | 操作 |
|-----|--------|
| `Escape` | 退出修剪模式 |

## 支持的图元

| 图元 | 可以修剪？ |
|--------|----------------|
| 直线 | 是 |
| 圆弧、圆、椭圆 | 否 |
| 多段线 / 矩形 | 否 |
| 文字、样条线、标注、引线 | 否 |

用作**切割边界**的图元可以是任何类型 — 只有被修剪的图元必须是直线图元。

## Trim vs Extend

| | 修剪 | 延伸 |
|---|------|--------|
| 功能 | 删除直线的一段 | 将直线端点延伸到边界 |
| 触发方式 | 悬停在要切割的线段上 | 悬停在要延伸的端点附近 |
| 结果 | 直线分割或缩短 | 直线端点移动到边界 |
| 两者 | 仅直线 | 仅直线 |
