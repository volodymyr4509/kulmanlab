# Files

**文件列表**命令打开**最近文件**面板 — 列出所有已保存到浏览器本地存储的图形。使用它可以重新打开以前的图形，或删除不再需要的文件。

## 如何打开最近文件面板

单击屏幕顶部文件面板中的**文件列表**工具栏按钮（历史图标）。面板在画布左侧打开。

## 最近文件面板

列表中的每个条目显示文件名和两个操作按钮：

| 按钮 | 操作 |
|--------|--------|
| ✓ **恢复** | 将图形加载到画布上 — 替换当前内容 |
| ✕ **删除** | 从浏览器存储中永久删除文件 |

如果尚未保存任何文件，面板显示"No files saved"。

## 文件的存储方式

KulmanLab 将图形保存到 **IndexedDB** — 一个持久化的浏览器内数据库。这意味着：

- 文件在页面重新加载和浏览器重启后仍然存在。
- 文件**仅存储在您的设备上** — 不会上传到任何服务器。
- 每个浏览器和设备都有各自独立的存储。在一台计算机上用 Chrome 保存的图形在 Firefox 或另一台计算机上不可用。
- 清除浏览器的网站数据将删除所有已保存的文件。

## 启动时自动加载文件

打开 KulmanLab CAD 时，应用程序自动加载存储中**最近修改的文件**。无需每次都从文件面板手动打开它。

## 管理存储

可以保存的图形数量没有固定限制，但浏览器存储是有限的。如果注意到存储警告，请从文件面板删除旧文件。

要一次删除所有已保存的图形，请使用[Wipe Storage](../wipestorage/)命令。

## 文件名和版本控制

每个文件的名称中都有时间戳（例如 `myplan_May22_14:30:00.dxf`）。此时间戳在您[Import](../import/)文件时自动添加，确保重新导入同一源文件会创建新条目，而不是覆盖先前的版本。

## 相关命令

- [Import](../import/) — 从文件系统将图形加载到浏览器存储
- [Export](../export/) — 将图形下载到文件系统
- [New File](../new-file/) — 开始空白图形（也自动保存）
- [Wipe Storage](../wipestorage/) — 从浏览器存储中清除所有已保存文件
