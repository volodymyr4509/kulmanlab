# Dimension Linear

`dimlinear`（线性标注）命令在两条延伸线原点之间放置水平或垂直标注。标注线始终完全水平或完全垂直 — 不能以任意角度放置。当需要平行于斜线的标注时，请使用[Dimension Aligned](../dim-aligned/)。

## 线性标注的结构

```
  |←————— 5.00 —————→|
  |                   |
  ●  （延伸线 1）     ●  （延伸线 2）
  p1                  p2
```

- **延伸线** — 从每个测量点垂直于标注线引出。
- **标注线** — 水平（测量 X 距离）或垂直（测量 Y 距离）。
- **标注值** — 沿所选轴的投影距离，而非真实的点到点距离。

## 放置线性标注

1. 在命令行中输入 `dimlinear`，或单击工具栏中的**Dimension Linear** 按钮。
2. **单击第一条延伸线原点**（p1），或输入 `X,Y` 并按 **Enter** 键精确输入坐标。
3. **单击第二条延伸线原点**（p2）。此处也可以输入坐标。
4. **移动光标**以定位标注线。方向由光标位置自动检测。
5. **单击**放置，或输入偏移距离并按 **Enter** 键精确放置。

## 自动方向检测

未强制方向时，命令根据光标相对于两个测量点的位置来检测：

| 光标位置 | 检测到的方向 | 测量内容 |
|-----------------|---------------------|-----------------|
| 点的上方或下方 | 水平 | p1 和 p2 之间的 Δ X |
| 点的左侧或右侧 | 垂直 | p1 和 p2 之间的 Δ Y |

在放置阶段随时按 **H** 锁定水平方向，按 **V** 锁定垂直方向。锁定后，移动光标不会改变方向。

## 输入偏移距离

在放置阶段输入数字可将标注线固定在距测量点的精确距离处：

| 按键 | 操作 |
|-----|--------|
| `0`–`9`、`.` | 在偏移距离后追加数字 |
| `Backspace` | 删除最后输入的字符 |
| `Enter` / 空格 | 在输入的距离处放置 |

光标所在侧（水平标注时上/下，垂直标注时左/右）决定符号 — 标注线出现在光标当前所在一侧。

## 键盘参考

| 按键 | 操作 |
|-----|--------|
| `0`–`9`、`.`、`-` | 开始输入 X 坐标（p1/p2 阶段），或输入偏移距离（放置阶段） |
| `,` | 锁定 X 并移至 Y 输入（p1/p2 阶段） |
| `H` | 锁定水平方向（仅放置阶段） |
| `V` | 锁定垂直方向（仅放置阶段） |
| `Backspace` | 删除最后输入的字符 |
| `Enter` / 空格 | 确认输入的坐标或偏移量 |
| `Escape` | 取消 |

## Dimension Linear vs Dimension Aligned

| | Dimension Linear | Dimension Aligned |
|---|-----------------|------------------|
| 轴向 | 始终 H 或 V | 平行于测量直线 |
| 测量内容 | 仅 X 或 Y 分量 | 真实欧几里得距离 |
| H/V 键 | 是 — 锁定方向 | 否 — 始终跟随 p1→p2 |
| 最适用于 | 正交布局、平面图 | 斜向特征、斜切 |

## 编辑标签 — 简单模式

**双击**已放置的线性标注可在**简单**模式下打开文字编辑器。编辑器预填充当前渲染值，以便您定位光标并直接编辑。

| 功能 | 行为 |
|---------|-----------|
| Bold / Italic / Font / Height | 一次性应用于**整个**标签 |
| 每字符格式 | 不支持 |
| `Enter` | 提交值并关闭编辑器 |
| 多行 | 不支持 |

请参阅[文字编辑器 — 简单模式](../../interface/text-editor/#simple-mode)获取完整参考。

## 链式标注

要从上一条延伸线继续添加更多标注，放置此标注后立即使用[Dimension Continue](../dim-continue/)。

## DXF — DIMENSION 实体

线性标注以 `rotationDeg` 设为 `0`（水平）或 `90`（垂直）的 `DIMENSION` 实体保存。延伸线原点、标注线位置、文字位置、测量值、箭头样式、文字高度和所有显示标志均可完整往返转换。
