# 添加引线

`leader+`（添加引线）命令为已有多重引线添加新的箭头臂。新引线臂从引线现有折点指向您单击的新箭头尖端。所有样式（折点位置、文字、箭头类型和大小）均从选中的引线继承。

## 添加引线臂

1. 在命令行中输入 `leader+`。
2. **单击已有多重引线**以选中它。
3. **单击新箭头尖端**，或输入 `X,Y` 并按 **Enter** 键精确输入坐标。预览线从光标显示到引线的折点。

引线臂放置后命令保持激活 — 可以立即单击另一个引线以添加更多臂。按 **Enter**、**空格**或 **Escape** 完成。

```
  之前：                        之后：
  ◄── 引线臂 1                  ◄── 引线臂 1
       \                               \
        ●──── 折点 ──── 文字          ●──── 折点 ──── 文字
                                       /
                                  引线臂 2 ──►  （您单击的新尖端）
```

## 尖端的坐标输入

可以输入精确位置而不必单击：

1. 输入 X 值。
2. 按 `,` — 命令行确认 X 已锁定。
3. 输入 Y 值。
4. 按 **Enter** 放置。

## 键盘参考

| 按键 | 操作 |
|-----|--------|
| `0`–`9`、`.`、`-` | 开始输入 X 坐标 |
| `,` | 锁定 X 并移至 Y 输入 |
| `Enter` | 确认输入的坐标并放置引线臂 |
| `Enter` / 空格 | 完成（无输入时） |
| `Escape` | 取消并重置 |

## 注意事项

- 只能选择**多重引线**图元 — 单击任何其他图元类型不会产生任何效果。
- 新引线臂从现有折点出发；您只需选择箭头尖端的位置。
- 一个多重引线可以拥有的引线臂数量没有限制。
