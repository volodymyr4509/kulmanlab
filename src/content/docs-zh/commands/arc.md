# 圆弧

`arc`（圆弧）命令通过您单击的三个点绘制圆弧。圆弧由经过所有三个点的唯一外接圆计算得出 — 无需直接指定圆心或半径。圆弧从第一次单击处延伸到第三次单击处，经过第二个点。

## 绘制圆弧

1. 在命令行中输入 `arc`，或单击工具栏中的**圆弧**按钮。
2. **单击第一点** — 圆弧的一个端点。也可以输入 `X,Y` 并按 **Enter** 键精确输入坐标。
3. **单击第二点** — 圆弧必须经过的一个点（控制曲率和方向）。此处也可以输入坐标。
4. **单击第三点** — 圆弧的另一个端点。圆弧被放置，命令退出。此处也可以输入坐标。

```
           ● （第 2 次单击 — 曲线上的中点）
          / \
         /   \
        ●     ●
     第 1 点       第 3 点
```

在定位第三点时，直线预览连接前两次单击。从第二次单击起，实时圆弧预览随光标移动。

> **共线点**：如果三点在同一直线上，则无法计算圆弧，不会放置任何图元。请移动第二点使其偏离直线后重试。

## 坐标输入

在三个步骤中的任意一步，可以输入精确位置而不必单击：

1. 输入 X 值。
2. 按 `,` — 命令行显示 `[X], [Y{光标}]`。
3. 输入 Y 值。
4. 按 **Enter** 放置该点。

## 键盘参考

| 按键 | 操作 |
|-----|--------|
| `0`–`9`、`.`、`-` | 开始输入 X 坐标 |
| `,` | 锁定 X 并移至 Y 输入 |
| `Backspace` | 删除最后输入的字符 |
| `Enter` | 确认输入的坐标 |
| `Escape` | 丢弃所有已放置的点并退出 |

## 夹点编辑 — 调整端点和半径

选中的圆弧显示三个夹点：

| 夹点 | 位置 | 功能 |
|------|----------|--------------|
| **圆心** | 外接圆的几何中心 | 移动整个圆弧；半径和角度不变 |
| **起点** | 圆弧上的第一个端点 | 拖动以沿外接圆移动起点 — 同时改变起始角度和半径 |
| **终点** | 圆弧上的最后一个端点 | 拖动以沿外接圆移动终点 — 同时改变终止角度和半径 |

拖动起点或终点夹点会将其重新定位至拖动位置，并根据该新位置相对于圆心重新计算角度和半径。另一个端点保持固定。

## 选择圆弧

| 方法 | 行为 |
|--------|-----------|
| **单击** | 如果单击落在圆弧曲线附近（而非弦处），则选中 |
| **向右拖动**（严格） | 沿圆弧分布的采样点必须全部在选框内 |
| **向左拖动**（交叉） | 落在选框内的任意采样点都可选中圆弧 |

## 支持的编辑命令

| 命令 | 对圆弧的影响 |
|---------|------------------------|
| [移动](../move/) | 平移圆心；半径和角度不变 |
| [复制](../copy/) | 在新位置创建相同的圆弧 |
| [旋转](../rotate/) | 旋转圆心并将起止角按旋转量偏移 |
| [镜像](../mirror/) | 镜像圆心并跨镜像轴反转起止角 |
| [缩放](../scale/) | 缩放圆心位置并将半径乘以缩放比例 |
| [偏移](../offset/) | 创建半径更大或更小的同心圆弧，角度范围不变 |
| [删除](../delete/) | 删除圆弧 |

## 属性

**常规**

| 属性 | 默认值 | 含义 |
|----------|---------|---------|
| 颜色 | 256（随层） | ACI 颜色索引 |
| 图层 | `0` | 图层分配 |
| 线型 | 随层 | 命名线型样式 |
| 线型比例 | 1 | 线型样式的缩放系数 |
| 厚度 | 0 | 拉伸厚度 |

**几何**

| 属性 | 含义 |
|----------|---------|
| 圆心 X / 圆心 Y | 外接圆的圆心 |
| 半径 | 外接圆的半径 |
| 起始角度 | 圆弧起始处的角度（从正 X 轴量起，单位：度） |
| 终止角度 | 圆弧终止处的角度 |

## 圆弧 vs 圆 — 选用时机

| | 圆弧 | 圆 |
|---|-----|--------|
| 范围 | 局部 — 从第一点到第三点 | 完整 360° |
| 输入方式 | 曲线上的三个点 | 圆心 + 半径（单击或输入） |
| 输入类型 | 每个点的 X,Y 坐标 | 半径值（圆心也接受 X,Y） |
| 放置后调整 | 拖动起点/终点夹点 | 拖动任意象限夹点 |
| 最适用于 | 圆角、圆弧路径 | 完整孔、圆形特征 |

## DXF — ARC 实体

圆弧以 `ARC` 实体保存在 DXF 文件中，存储圆心坐标、半径、起始角度和终止角度。所有属性（包括颜色、图层、线型、线型比例和厚度）均可完整往返转换。任何支持 DXF 的应用程序（LibreCAD、FreeCAD 等）均可将其作为标准圆弧读取。
