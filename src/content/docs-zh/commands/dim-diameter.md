# 直径标注

`dimdiameter`（直径标注）命令在圆弧或圆上放置直径标注。标注线跨越完整直径 — 通过圆心连接两个对径圆弧点 — 并标注为 `⌀ <值>`。如果只标注从圆心到一侧边缘的半径，请使用[半径标注](./dim-radius)。

## 直径标注的结构

```
  ●──────────── ⌀ 10.00 ────────────●
  （远弧点）              （近弧点 / 文字侧）
```

- **标注线** — 跨越完整直径，两端的圆弧交点处各有一个箭头。
- **近弧点** — 光标侧的圆周点（文字标签所在位置）。
- **远弧点** — 对径位置的点。
- **标签** — `⌀` 后跟直径值。

## 放置直径标注

1. 在命令行中输入 `dimdiameter`，或单击工具栏中的**直径标注**按钮。
2. **单击圆弧或圆**以选中它。
3. **移动光标**以将标注线旋转到所需角度。
4. **单击**以放置标注。

只能选择**圆弧**和**圆**图元。

## 键盘参考

| 按键 | 操作 |
|-----|--------|
| `Escape` | 取消 |

## 直径标注 vs 半径标注

| | 直径标注 | 半径标注 |
|---|-------------------|-----------------|
| 测量内容 | 完整直径（2 × 半径） | 半径（圆心到边缘） |
| 标注线 | 边缘 → 经圆心 → 边缘 | 圆心 → 边缘 |
| 标签前缀 | `⌀` | `R` |
| 箭头 | 两个（在两个弧点处） | 一个（在弧点处） |
| 最适用于 | 完整圆形孔或轴径标注 | 标注曲线特征的一侧 |

## 编辑标签 — 简单模式

**双击**已放置的直径标注可在**简单**模式下打开文字编辑器。编辑器预填充当前渲染值（例如 `⌀ 10.00`），以便您定位光标并直接编辑。

| 功能 | 行为 |
|---------|-----------|
| Bold / Italic / Font / Height | 一次性应用于**整个**标签 |
| 每字符格式 | 不支持 |
| `Enter` | 提交值并关闭编辑器 |
| 多行 | 不支持 |

请参阅[文字编辑器 — 简单模式](../interface/text-editor#simple-mode)获取完整参考。

## DXF — DIMENSION 直径实体

直径标注以直径类型几何的 `DIMENSION` 实体保存，存储两个弧点位置和测量的直径值（2 × 半径）。所有属性均可完整往返转换。
