# 删除引线

`leader-`（删除引线）命令从已有多重引线中删除一条箭头臂。文字标签、折点和所有其余引线臂均被保留 — 只有选定的引线臂被删除。只有一条引线臂的多重引线不能删除其引线臂。

## 删除引线臂

1. 在命令行中输入 `leader-`。
2. **单击拥有两条或以上引线臂的多重引线**。如果单击的引线只有一条引线臂，命令行显示错误并等待有效选择。
3. **将光标移到要删除的引线臂附近** — 最近的引线臂被标记高亮显示。
4. **单击**以删除该引线臂。

引线臂删除后命令保持激活 — 可以立即单击另一个引线（或同一个）以删除更多引线臂。按 **Enter**、**空格**或 **Escape** 完成。

```
  之前：                   之后：
  ◄── 引线臂 1             ◄── 引线臂 1
       \                         \
        ●──── 折点 ──── 文字     ●──── 折点 ──── 文字
       /
  引线臂 2 ──►  ← 此引线臂被删除
```

## 如何确定最近的引线臂

命令测量光标到每条引线臂线段（包括从最后一个引线臂点到折点的线段）的垂直距离。距离最小的引线臂被高亮显示，单击时将被删除。

## 键盘参考

| 按键 | 操作 |
|-----|--------|
| `Enter` / 空格 | 完成删除引线臂 |
| `Escape` | 取消并重置 |

## 注意事项

- **只有一条引线臂**的引线受到保护 — 必须先添加引线臂才能删除。
- 无论删除哪条引线臂，折点位置和文字内容始终被保留。
