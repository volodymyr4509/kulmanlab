# Area

`area`（面积）命令测量由三个或更多单击点定义的多边形的面积和周长，并在命令行中以 4 位小数打印两个结果。这是第三个测量命令，与 [Distance](../distance/)（直线长度）和 [Angle](../angle/)（顶点处的内角）并列。

## 面积测量的结构

```
  ● 第一点
   \
    \
     ● 第二点
      \
       \             （虚线）闭合边预览
        ●───────────────┐
      第三点            │  （虚线）到光标的下一条边预览
                        ✕ 光标  →  命令行："Area: 12.3456  Perimeter: 45.6789"
```

- **顶点** — 每个单击（或输入）的点都会成为多边形的一个顶点；已确认的边以实线绘制，内部以半透明高亮填充。
- **预览边** — 虚线显示从最后一个顶点到光标的待定边，以及从光标返回第一个顶点的闭合边。
- **闭合边** — 你无需再次单击第一点；按 Enter 会自动闭合多边形。

## 测量面积

1. 在命令行中输入 `area`，或单击工具栏中的 **Area** 按钮（Measure 面板的下排）。
2. **单击第一点**，或输入 `X,Y` 并按 **Enter** 键精确输入坐标。
3. 按顺序**单击每个额外的顶点**，绕图形一周。每一步都可以输入坐标。
4. 放置至少 **3 个点**后，按 **Enter**（没有待定的坐标或距离输入）以闭合多边形并计算结果。
5. 命令行打印 `Area: <值>  Perimeter: <值>`，闭合的多边形——填充、轮廓和顶点夹点——在画布上保持高亮。
6. **单击任意位置、按任意键或按 `Escape`**，可关闭结果并结束命令。

## 角度锁定与精确距离

放置第一个顶点后，向已配置的角度追踪增量（10°、15°、20°、30°、45° 或 90°，可通过工具栏下拉菜单设置）之一移动会将下一条边锁定到该方向：

- 边预览会吸附到锁定方向，并在锚点顶点处绘制角度追踪指示器。
- 输入长度并按 **Enter**，即可沿锁定方向将下一个顶点精确放置在该距离处。
- 锁定时单击（未输入长度）会将顶点放置在光标在锁定方向上的投影处。

| 按键 | 操作 |
|-----|--------|
| `0`–`9`、`.` | 添加到边长度值 |
| `-` | 负长度（仅作为第一个字符） |
| `Backspace` | 删除最后输入的字符 |
| `Enter` | 按输入的长度放置下一个顶点 |

## 闭合多边形

- 只有放置了 **3 个或更多**顶点后，Enter 才会闭合图形——数量不足时无效。
- 从最后一个顶点回到第一个顶点的边会自动添加，并计入面积和周长。
- 点可以按任意顺序放置（顺时针或逆时针）——结果相同。

## Area vs Distance vs Angle

| | Area | Distance | Angle |
|---|------|---------|-------|
| 测量内容 | 多边形的面积和周长 | 直线长度 | 顶点处的内角 |
| 单击次数 | 3 次或更多，按 Enter 闭合 | 2 | 3 |
| 结果格式 | `12.3456  Perimeter: 45.6789` | `12.3456`（单位） | `45.0000°` |
| 画布预览 | 带虚线闭合边的填充多边形 | 从第一点到光标的直线 | 从顶点到光标的两条线 |
| 结果之后 | 任意输入即可关闭结果，随后命令结束 | 单击可链式开始新测量 | 单击可链式开始新测量 |
| 最适用于 | 封闭区域、房间或面板面积 | 间隙或线段的长度 | 两个特征之间的张角 |

## 坐标输入

可以为任意顶点输入精确位置，而不必单击：

1. 输入 X 值。
2. 按 `,` — 命令行显示 `[X], [Y{光标}]`。
3. 输入 Y 值。
4. 按 **Enter** 确认。

## 键盘参考

| 按键 | 操作 |
|-----|--------|
| `0`–`9`、`.`、`-` | 开始输入 X 坐标，或在角度锁定时输入边长度 |
| `,` | 锁定 X 并移至 Y 输入 |
| `Backspace` | 删除最后输入的字符 |
| `Enter` | 确认输入的坐标或长度；当有 3 个以上顶点且无待定输入时，闭合多边形 |
| `Escape` | 选择顶点时：放弃已放置的点并从第一点重新开始；结果显示后：关闭结果并结束命令 |

## 注意事项

- 面积使用鞋带公式（shoelace formula）计算，无论单击顺序如何，结果始终报告为正值。
- 自相交的多边形（边相互交叉）仍会产生数值结果，但该值可能与视觉上封闭的区域不符——保持不交叉的单击顺序才能得到有意义的面积。
- 结果仅显示在**命令行和画布上的临时高亮**中——不会向图形添加任何永久性内容。
- 与 Distance 和 Angle 不同，Area **不会**自动链式开始新的测量——关闭结果后，需再次运行 `area` 以测量另一个多边形。
- 面积和周长的精度始终为 4 位小数，单位与图形坐标相同（无单位转换）。
