# 复制视口

`ViewportCopy`（复制视口）命令将视口复制到新位置，保留其比例和模型中心。仅在布局空间中可用。

## 复制视口

1. 切换到图纸布局标签。
2. 可选择单击视口以预先选中它。
3. 在命令行中输入 `ViewportCopy`，或单击工具栏中的**复制视口**按钮。
4. 如果未预先选中视口，**单击要复制的视口**。
5. **单击基点** — 位移的参考点。或输入 `X,Y` 并按 **Enter** 键精确输入坐标。
6. **单击目标点** — 视口放置在基点→目标点的偏移处。或使用坐标输入/角度锁定。

放置后命令保持激活 — 单击另一个目标点以再次放置同一视口的副本。按 **Enter**、**空格**或 **Escape** 完成。

## 坐标输入

在基点和目标点步骤，可以输入精确坐标而不必单击：

1. 输入 X 值。
2. 按 `,` — 命令行显示 `[X], [Y{光标}]`。
3. 输入 Y 值。
4. 按 **Enter** 确认。

## 角度锁定和精确距离

设置基点后，命令在光标对齐时捕捉到 45° 轴（0°、45°、90°、135°……）。锁定时：

- 预览捕捉到轴。
- 输入距离并按 **Enter** 可沿锁定方向精确放置副本。

| 按键 | 操作 |
|-----|--------|
| `0`–`9`、`.` | 在距离值后追加数字 |
| `-` | 负距离（反向；仅第一个字符） |
| `Backspace` | 删除最后输入的字符 |
| `Enter` | 在输入的距离处放置副本 |

## 键盘参考

| 按键 | 操作 |
|-----|--------|
| `0`–`9`、`.`、`-` | 开始输入 X 坐标，或角度锁定时输入距离 |
| `,` | 锁定 X 并移至 Y 输入 |
| `Enter` | 确认输入的坐标或距离 |
| `Enter` / 空格 | 完成（无输入时） |
| `Escape` | 取消并重置 |

## 注意事项

- 复制视口仅在图纸布局标签激活时可用。
- 复制的视口继承与原始视口相同的比例、模型中心、锁定状态和尺寸。
- 要从头创建新视口，请使用[矩形视口](./viewport-rectangle)。
