# 倒角

`chamfer`（倒角）命令在两条[直线](./line)或[多段线](./polyline)图元之间切出直线斜角。您指定沿每个图元回退的距离（d1 和 d2），命令将两个图元都修剪到这些点并在它们之间插入一条连接直线。

使用相等的距离产生对称的 45° 切割；不同距离产生不对称的斜面。

倒角适用于**直线和多段线**图元。

## 使用倒角

1. 在命令行中输入 `chamfer`，或单击工具栏中的**倒角**按钮。
2. **输入第一个倒角距离**（d1 — 沿第一个图元的距离）并按 **Enter** 键。
3. **输入第二个倒角距离**（d2 — 沿第二个图元的距离）并按 **Enter** 键。
4. **单击第一个图元** — 单击位置决定在任意交点的哪一侧保留。
5. **将光标悬停在第二个图元上** — 虚线预览显示产生的倒角切割结果。将光标移至要保留的一侧。
6. **单击**以应用。两个图元均被修剪，倒角直线被插入。

## 边侧选择

当两条直线相互交叉时，倒角应用于由单击位置定义的角 — 每个图元上**与光标同侧**的部分被保留。

- 单击第一个图元的一端附近以选择该半部分。
- 将光标移到第二个图元的所需半部分 — 虚线预览实时更新。

对于多段线，单击位置决定哪个**线段**参与，最靠近交叉侧的顶点将被修剪。

## 命令创建的内容

- 第一个图元距交点最近的端点（或多段线顶点）被移动到点 **T1**，该点位于从交点沿第一个图元距离 d1 处。
- 第二个图元距交点最近的端点（或多段线顶点）被移动到点 **T2**，该点位于从交点沿第二个图元距离 d2 处。
- 从 **T1** 到 **T2** 插入一条新直线图元。

插入的直线继承当前的线宽、颜色、图层和线型设置。

## 键盘参考

| 按键 | 操作 |
|-----|--------|
| `0`–`9`、`.` | 在当前距离值后追加数字 |
| `Backspace` | 删除最后输入的字符 |
| `Enter` | 确认输入的距离并继续 |
| `Escape` | 取消并重置 |

## 支持的图元

| 图元 | 是否支持 |
|--------|-----------|
| 直线 | 是 |
| 多段线 / 矩形 | 是 |
| 圆弧、圆、椭圆 | 否 |
| 文字、样条线、标注、引线 | 否 |

## 倒角 vs 圆角

| | 倒角 | 圆角 |
|---|---------|--------|
| 角的类型 | 直线切割 | 圆弧圆角 |
| 输入 | 两个距离（d1, d2） | 一个半径 |
| 插入图元 | 直线 | 圆弧 |
| 支持的图元 | 直线和多段线 | 仅直线 |
