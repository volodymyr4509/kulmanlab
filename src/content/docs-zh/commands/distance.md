# Distance

`distance`（距离）命令测量两个单击点之间的直线（欧几里得）距离，并在命令行中以 4 位小数打印结果。这是三个测量命令之一 — [Angle](../angle/)命令用于测量顶点处的角度开度，[Area](../area/)命令用于测量多边形的面积和周长。

## 距离测量的结构

```
  ● 第一点
   \
    \  预览线（实时）
     \
      ● 第二点    →  命令行："Distance: 12.3456"
```

- **第一点** — 测量的起点。
- **第二点** — 终点；放置后立即打印结果。
- **结果** — 显示在命令行中，不放置在画布上。

## 测量距离

1. 在命令行中输入 `distance`，或单击工具栏中的**Distance** 按钮。
2. **单击第一点**，或输入 `X,Y` 并按 **Enter** 键精确输入坐标。
3. **单击第二点** — 测量距离出现在命令行中。此处也可以输入坐标。
4. **再次单击**（可选）以开始新的测量。命令保持激活状态。

随时按 `Escape` 键可重置至第 2 步。

## 链式测量

结果显示后，立即单击即可开始下一次测量 — 单击的点成为新的第一点。这样无需重新激活命令即可测量一系列距离。

## Distance vs Angle

| | Distance | Angle |
|---|---------|-------|
| 测量内容 | 直线长度 | 顶点处的内角 |
| 单击次数 | 2 | 3 |
| 结果格式 | `12.3456`（单位） | `45.0000°` |
| 画布预览 | 从第一点到光标的直线 | 从顶点到光标的两条线 |
| 最适用于 | 间隙或线段的长度 | 两个特征之间的张角 |

## 坐标输入

可以为任意一点输入精确位置，而不必单击：

1. 输入 X 值。
2. 按 `,` — 命令行显示 `[X], [Y{光标}]`。
3. 输入 Y 值。
4. 按 **Enter** 确认。

## 键盘参考

| 按键 | 操作 |
|-----|--------|
| `0`–`9`、`.`、`-` | 开始输入 X 坐标 |
| `,` | 锁定 X 并移至 Y 输入 |
| `Backspace` | 删除最后输入的字符 |
| `Enter` | 确认输入的坐标 |
| `Escape` | 取消并重置至第 2 步 |

## 注意事项

- 结果仅显示在**命令行**中 — 不会在图形中添加任何内容。
- 结果以与图形坐标相同的单位表示（无单位转换）。
- 精度始终为 4 位小数。
