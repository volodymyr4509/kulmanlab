# Mirror

`mirror`（镜像）命令创建选中图元的镜像副本，跨两点轴反射。原图元**始终保留** — 与[Move](../move/)或[Rotate](../rotate/)不同，镜像从不修改现有图元；它只添加新图元。

## 两种启动方式

**预先选择，然后镜像** — 先选择图元，再激活：

1. 在画布上选择一个或多个图元。
2. 在命令行中输入 `mirror`，或单击工具栏中的**Mirror**按钮。
3. **单击镜像轴的第一点**，或输入 `X,Y` 并按 **Enter** 键精确输入坐标。
4. **单击第二点** — 放置镜像副本，命令退出。此处也可以输入坐标。

**激活后选择** — 在未选中任何内容的情况下启动命令：

1. 输入 `mirror` 或单击工具栏按钮。
2. **选择对象** — 单击以切换，或拖动以按区域选择。
3. 按 **Enter** 或**空格**确认选择。
4. **单击第一点**，然后**单击镜像轴的第二点**（两步均可输入坐标）。

```
  原始：          镜像轴：        结果：
                   |
  [图元 A]    →   |    →    [图元 A] + [镜像 A]
                   |
```

定位第二个轴点时，镜像副本的实时预览跟随光标。

## 镜像轴

轴是经过两个单击点的无限直线。可以是任意角度：

- 将光标移近 **45° 捕捉轴**（0°、45°、90°、135°……），轴将锁定到该角度 — 适用于干净的水平、垂直或斜向反射。
- 在捕捉区域外单击以获得自由角度轴。

## 坐标输入

在任一轴点步骤，可以输入精确位置而不必单击：

1. 输入 X 值。
2. 按 `,` — 命令行显示 `[X], [Y{光标}]`。
3. 输入 Y 值。
4. 按 **Enter** 确认。

## 键盘参考

| 按键 | 操作 |
|-----|--------|
| `Enter` / 空格 | 确认选择 |
| `0`–`9`、`.`、`-` | 开始输入 X 坐标 |
| `,` | 锁定 X 并移至 Y 输入 |
| `Backspace` | 删除最后输入的字符 |
| `Enter` | 确认输入的坐标 |
| `Escape` | 取消并重置 |

## 命令中的选择

| 方法 | 行为 |
|--------|-----------|
| **单击** | 切换光标下方图元 |
| **向右拖动**（严格） | 添加完全在选框内的图元 |
| **向左拖动**（交叉） | 添加与选框相交的图元 |
| **Enter** / 空格 | 确认选择 |

## 镜像的内容

支持每种图元类型。几何图形以数学方式跨轴反射：

| 图元 | 变化内容 |
|--------|-------------|
| 直线 | 两个端点均反射 |
| 圆 | 圆心反射；半径不变 |
| 圆弧 | 圆心反射；起止角跨轴重新计算 |
| 椭圆 | 圆心反射；长轴方向跨轴翻转 |
| 多段线 / 矩形 | 每个顶点均反射 |
| 文字 | 锚点反射；文字字符串**不**反转 |
| 样条线 | 所有控制顶点 / 拟合点均反射 |

## Mirror vs Copy

| | 镜像 | 复制 |
|---|--------|------|
| 原图元 | 始终保留 | 始终保留 |
| 新图元位置 | 跨轴反射 | 按位移向量偏移 |
| 最适用于 | 对称设计、双侧特征 | 任意方向的重复几何 |
