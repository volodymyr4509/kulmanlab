# CV 样条线

`splinecv`（CV 样条线）命令通过放置控制顶点（CV）绘制**三次 B 样条线**。生成的曲线被每个 CV 吸引但不经过它们 — 只在第一个和最后一个顶点处，**固定端点**使曲线精确锚定。这提供了直观的形状控制：拉动顶点可将曲线推向它，而不需要强制曲线触及每个点。

## 通过控制顶点绘制样条线

1. 在命令行中输入 `splinecv`，或单击工具栏中的 **CV 样条线**按钮。
2. **单击以放置控制顶点** — 每次单击添加一个顶点。或输入 `X,Y` 并按 **Enter** 键精确输入坐标。
3. 按 **Enter** 完成（至少需要 2 个顶点）。

```
  CV ●         ● CV
      \       /
       \     /    ← 曲线被吸引向 CV
        \   /         但不经过它们
  CV ●   ●   ● CV（起点/终点：曲线在此处经过）
```

放置每个顶点后，实时预览随光标更新，显示如果下一个点在光标位置处样条线的外观。按 **Escape** 可丢弃所有已放置的顶点并退出。

## 坐标输入

可以输入精确位置，而不必单击任何控制顶点：

1. 输入 X 值。
2. 按 `,` — 命令行显示 `[X], [Y{光标}]`。
3. 输入 Y 值。
4. 按 **Enter** 放置顶点。

## 键盘参考

| 按键 | 操作 |
|-----|--------|
| `0`–`9`、`.`、`-` | 开始输入 X 坐标 |
| `,` | 锁定 X 并移至 Y 输入 |
| `Backspace` | 删除最后输入的字符 |
| `Enter` | 确认输入的坐标，或未输入且 ≥ 2 个顶点时完成样条线 |
| `Escape` | 丢弃所有顶点并退出 |

## 夹点编辑 — 通过控制顶点重塑

选中的 CV 样条线每个控制顶点显示一个夹点：

| 夹点 | 位置 | 功能 |
|------|----------|--------------|
| **控制顶点** | 每个 CV 位置 | 拖动以移动该 CV — 曲线向新位置重塑 |

没有"移动整条样条线"夹点。要平移整条样条线，请使用[移动](./move)命令。

## 选择 CV 样条线

| 方法 | 行为 |
|--------|-----------|
| **单击** | 如果单击落在曲线上的任意点附近，则选中 |
| **向右拖动**（严格） | 沿曲线的所有采样点必须在选框内 |
| **向左拖动**（交叉） | 与选框边界相交的曲线任意部分都将选中 |

## 支持的编辑命令

| 命令 | 对样条线的影响 |
|---------|---------------------------|
| [移动](./move) | 所有控制顶点平移相同位移 |
| [复制](./copy) | 在新位置创建相同的样条线 |
| [旋转](./rotate) | 绕选定基点旋转所有 CV |
| [镜像](./mirror) | 沿镜像轴反射所有 CV |
| [缩放](./scale) | 从基点统一缩放所有 CV |
| [删除](./delete) | 删除样条线 |

样条线不支持**偏移**、**修剪**或**延伸**。

## 属性

**常规**

| 属性 | 默认值 | 含义 |
|----------|---------|---------|
| 颜色 | 256（随层） | ACI 颜色索引 |
| 图层 | `0` | 图层分配 |
| 线型 | 随层 | 命名线型样式 |
| 线型比例 | 1 | 线型样式的缩放系数 |
| 厚度 | 0 | 拉伸厚度 |

**几何**

| 属性 | 含义 |
|----------|---------|
| 次数 | 多项式次数 — 始终为 3（三次） |
| 控制顶点 | 所有 CV 的坐标 |
| 拟合点 | CV 样条线为空；仅拟合点样条线时填充 |

## CV 样条线 vs 拟合样条线 — 选用哪个

| | CV 样条线 | 拟合样条线 |
|---|-----------|------------|
| 曲线经过点 | 仅第一个和最后一个（固定端点） | 每个单击点精确经过 |
| 形状控制 | 将 CV 拉向某个区域 | 移动曲线必须触及的拟合点 |
| 夹点编辑效果 | CV 移动 → 曲线被吸引 | 拟合点移动 → 曲线重新插值 |
| 最适用于 | 平滑美观曲线、自由形态路径 | 必须经过特定坐标的曲线 |

## DXF — SPLINE 实体（控制顶点形式）

CV 样条线以 `SPLINE` 实体保存在 DXF 文件中，存储次数、节点向量和所有控制顶点坐标。所有属性（颜色、图层、线型、线型比例和厚度）均可完整往返转换。`splineFlag` 设置为 `9`（CV 样条线），因此重新加载时保留形式。任何支持带 CV 数据的 `SPLINE` 实体的 DXF 应用程序都可以正确读取这些。
