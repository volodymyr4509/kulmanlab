# Line

`line`（直线）命令绘制在 DXF 模型中作为独立 `LINE` 实体存储的单独直线段。每段完成后命令保持激活，并将端点重新用作新起点，因此可以一次绘制一段连接路径。与[Polyline](../polyline/)不同，连接的直线保持独立图元 — 每条都可以单独修剪、延伸或删除，而不影响其邻居。

## 绘制直线

1. 在命令行中输入 `line`，或单击工具栏中的**Line**按钮。
2. **单击起点**，或输入 `X,Y` 并按 **Enter** 键精确输入坐标。
3. **单击终点** — 线段被放置，终点成为下一个起点。此处也可以输入坐标。
4. 继续单击（或输入）以连接更多线段。
5. 按 **Enter** 或 **Escape** 停止。

```
  ●──────────●──────────●──────────●
 起点     第 2 次单击  第 3 次单击   按 Enter 完成
            （自动成为下一个起点）
```

只需一段？在第 3 步后立即按 **Enter** 或 **Escape**。

## 坐标输入

可以输入精确位置，而不必单击起点或任何后续点：

1. 输入 X 值（数字、`.` 或 `-`）。
2. 按 `,` — 命令行显示 `[X], [Y{光标}]`。
3. 输入 Y 值。
4. 按 **Enter** 放置该点。

## 角度锁定和精确长度输入

放置点后移动光标时，命令会监测 45° 捕捉轴（0°、45°、90°、135°……）。满足以下条件时角度**锁定**：

- 光标距锚点至少 **5 × 夹点大小**，**且**
- 距最近轴的垂直距离在 **1 个夹点大小**以内。

锁定时预览捕捉到轴，可以输入精确长度：

| 按键 | 操作 |
|-----|--------|
| `0`–`9`、`.` | 在长度值后追加数字 |
| `-` | 负长度 — 沿轴线反向（仅第一个字符） |
| `Backspace` | 删除最后输入的字符 |
| `Enter` | 在输入的距离处放置端点 |

命令行中实时显示累积值（例如 `click end point or enter length: 12.5`）。锁定时单击会投影到轴上，因此端点始终精确落在轴上。

移回靠近锚点处会解除锁定。

## 键盘参考

| 按键 | 操作 |
|-----|--------|
| `0`–`9`、`.`、`-` | 开始输入 X 坐标，或角度锁定时输入距离 |
| `,` | 锁定 X 并移至 Y 输入 |
| `Backspace` | 删除最后输入的字符 |
| `Enter` | 确认输入的坐标或长度，或未输入任何内容时完成链 |
| `Escape` | 完成链并退出 |

## 夹点编辑 — 拉伸端点

选中的直线显示三个夹点：

| 夹点 | 位置 | 功能 |
|------|-------|--------------|
| **起点** | 第一个端点 | 拖动重新定位 — 终点保持固定 |
| **中点** | 直线中心 | 激活整条直线的**移动** |
| **终点** | 第二个端点 | 拖动重新定位 — 起点保持固定 |

拉伸一个端点不影响另一个。这与[Polyline](../polyline/)夹点编辑不同，在多段线中移动顶点会重塑整个路径。

## 选择直线

| 方法 | 行为 |
|--------|-----------|
| **单击** | 如果单击在线段命中测试距离内，则选中直线 |
| **向右拖动**（严格） | 只有当两个端点都在选框内时才选中直线 |
| **向左拖动**（交叉） | 如果线段的任何部分与选框边界相交，则选中直线 |

## 支持的编辑命令

直线是[Trim](../trim/)和[Extend](../extend/)命令操作的**唯一**图元。所有标准变换命令也适用：

| 命令 | 对直线的影响 |
|---------|------------------------|
| [Move](../move/) | 两个端点平移相同位移 |
| [Copy](../copy/) | 在新位置创建相同的直线 |
| [Rotate](../rotate/) | 绕选定基点旋转两个端点 |
| [Mirror](../mirror/) | 沿镜像轴反射两个端点 |
| [Scale](../scale/) | 从基点统一缩放两个端点 |
| [Offset](../offset/) | 创建固定垂直距离的平行直线 |
| [Trim](../trim/) | 在交点处切割直线 — **仅直线** |
| [Extend](../extend/) | 将最近端点延伸至边界 — **仅直线** |
| [Delete](../delete/) | 从图形中删除直线 |

## 属性

选中直线时，属性面板显示 DXF `LINE` 记录包含的每个字段：

**常规**

| 属性 | 默认值 | 含义 |
|----------|---------|---------|
| 颜色 | 256（随层） | ACI 颜色索引 |
| 图层 | `0` | 图层分配 |
| 线型 | 随层 | 命名线型样式 |
| 线型比例 | 1 | 线型样式的缩放系数 |
| 厚度 | 0 | 拉伸厚度 |

**几何**

| 属性 | 含义 |
|----------|---------|
| 起点 X / 起点 Y | 第一个端点坐标 |
| 终点 X / 终点 Y | 第二个端点坐标 |

所有字段均可在面板中直接编辑，无需重新运行命令。

## Line vs Polyline — 选用时机

| | 直线 | 多段线 |
|---|------|---------|
| 图元数量 | 每段一个 `LINE` | 整个路径一个 `LWPOLYLINE` |
| 修剪 / 延伸 | 是 — 逐段 | 否 |
| 闭合形状 | 否 | 是（闭合标志） |
| 夹点编辑 | 拉伸单个端点 | 沿路径移动任意顶点 |
| 最适用于 | 辅助线、单段线、需要修剪的几何 | 轮廓线、外形、保持整体的形状 |

## DXF — LINE 实体

直线以 `LINE` 实体保存在 DXF 文件中。每个属性（起点/终点坐标、颜色、图层、线型、线型比例和厚度）均可完整往返转换。当您打开包含 `LINE` 实体的 DXF 时，它们成为编辑器中完全可编辑的 `Line` 对象。

在编辑器中绘制的直线也以 `LINE` 实体写入保存，因此可以被 LibreCAD、FreeCAD 和任何其他支持 DXF 的应用程序读取。
