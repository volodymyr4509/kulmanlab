# 缩放

`scale`（缩放）命令绕基点统一缩放选中图元。所有距基点的距离乘以缩放系数 — 系数 `2` 使所有尺寸翻倍，`0.5` 使它们减半。系数始终通过输入方式设置；没有单击设置缩放的功能。

## 两种启动方式

**预先选择，然后缩放** — 先选择图元，再激活：

1. 在画布上选择一个或多个图元。
2. 在命令行中输入 `scale`，或单击工具栏中的**缩放**按钮。
3. **单击基点** — 缩放期间不移动的固定点。或输入 `X,Y` 并按 **Enter** 键精确输入坐标。
4. **输入缩放系数**并按 **Enter** 键。

**激活后选择** — 在未选中任何内容的情况下启动命令：

1. 输入 `scale` 或单击工具栏按钮。
2. **选择对象** — 单击以切换，或拖动以按区域选择。
3. 按 **Enter** 或**空格**确认选择。
4. **单击基点**（可输入坐标），然后输入系数。

```
  基点 ●                基点 ●
        [图元]   →          [较大的图元]
  系数 = 2 → 从 ● 开始的距离翻倍
```

## 输入缩放系数

放置基点后，命令行显示 `enter scale factor:`，等待键盘输入：

| 按键 | 操作 |
|-----|--------|
| `0`–`9`、`.` | 在系数后追加数字 |
| `-` | 负系数（仅第一个字符 — 先反转后缩放） |
| `Backspace` | 删除最后输入的字符 |
| `Enter` | 以输入的系数应用缩放 |

系数必须非零。常用值：

| 系数 | 效果 |
|--------|--------|
| `2` | 所有尺寸翻倍 |
| `0.5` | 所有尺寸减半 |
| `1.5` | 增大 50% |
| `-1` | 以基点为中心镜像（等同于 180° 旋转） |

输入时没有实时预览 — 按 **Enter** 后才会显示缩放结果。

## 键盘参考

| 按键 | 操作 |
|-----|--------|
| `Enter` / 空格 | 确认选择 |
| `0`–`9`、`.`、`-` | 开始输入 X 坐标（基点阶段），或缩放系数（系数阶段） |
| `,` | 锁定 X 并移至 Y 输入（基点阶段） |
| `Backspace` | 删除最后输入的字符 |
| `Enter` | 确认坐标或应用缩放 |
| `Escape` | 取消并重置 |

## 命令中的选择

| 方法 | 行为 |
|--------|-----------|
| **单击** | 切换光标下方图元 |
| **向右拖动**（严格） | 添加完全在选框内的图元 |
| **向左拖动**（交叉） | 添加与选框相交的图元 |
| **Enter** / 空格 | 确认选择 |

## 缩放的内容

支持所有图元类型。每个图元相对于基点缩放其几何图形：

| 图元 | 变化内容 |
|--------|-------------|
| 直线 | 两个端点从基点缩放 |
| 圆 | 圆心从基点缩放；半径乘以系数 |
| 圆弧 | 圆心缩放；半径乘以系数；角度不变 |
| 椭圆 | 圆心缩放；两条半轴长度乘以系数 |
| 多段线 / 矩形 | 每个顶点从基点统一缩放 |
| 文字 | 锚点缩放；高度乘以系数 |
| 样条线 | 所有控制顶点 / 拟合点缩放 |
