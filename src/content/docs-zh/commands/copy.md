# 复制

`copy`（复制）命令创建选中图元的平移副本并将其从基点偏移到目标点放置 — 原图元保持在原位不变。这是与[移动](./move)命令的主要区别：复制会向图形中添加新图元；移动则重定位现有图元。

## 两种启动方式

**预先选择，然后复制** — 先选择图元，再激活：

1. 在画布上选择一个或多个图元。
2. 在命令行中输入 `copy`，或单击工具栏中的**复制**按钮。
3. **单击基点**，或输入 `X,Y` 并按 **Enter** 键精确输入坐标。
4. **单击目标点** — 副本出现在基点→目标点的偏移处。此处也可以输入坐标。

**激活后选择** — 在未选中任何内容的情况下启动命令：

1. 输入 `copy` 或单击工具栏按钮。
2. **选择对象** — 单击以切换单个图元，或拖动以按区域选择。
3. 按 **Enter** 或**空格**确认选择。
4. **单击基点**，然后**单击目标点**（两步均可输入坐标）。

```
  之前：               之后：
  [图元 A]            [图元 A]  ← 原图元不动
  [图元 B]            [图元 B]
                      [A 的副本] ← 新图元
                      [B 的副本]
```

副本的幽灵预览从基点跟随光标到目标点。

## 坐标输入

在基点或目标点步骤，可以输入精确位置而不必单击：

1. 输入 X 值。
2. 按 `,` — 命令行显示 `[X], [Y{光标}]`。
3. 输入 Y 值。
4. 按 **Enter** 确认。

## 角度锁定和精确距离

设置基点后，当光标距离足够远且接近某轴线时，命令会捕捉到 45° 轴（0°、45°、90°、135°……）。锁定时，输入距离并按 **Enter** 可将副本精确放置在该偏移处。

| 按键 | 操作 |
|-----|--------|
| `0`–`9`、`.` | 在距离值后追加数字 |
| `-` | 负距离 — 沿轴线反向（仅第一个字符） |
| `Backspace` | 删除最后输入的字符 |
| `Enter` | 在输入的距离处放置副本 |

## 键盘参考

| 按键 | 操作 |
|-----|--------|
| `Enter` / 空格 | 确认选择并进入基点阶段 |
| `0`–`9`、`.`、`-` | 开始输入 X 坐标，或角度锁定时输入距离 |
| `,` | 锁定 X 并移至 Y 输入 |
| `Backspace` | 删除最后输入的字符 |
| `Enter` | 确认坐标或在输入的距离处应用复制 |
| `Escape` | 取消并重置 |

## 命令中的选择

| 方法 | 行为 |
|--------|-----------|
| **单击** | 切换光标下方图元的选中/取消状态 |
| **向右拖动**（严格） | 添加完全在选框内的图元 |
| **向左拖动**（交叉） | 添加与选框边界相交的图元 |
| **Enter** / 空格 | 确认选择 |

## 复制后

**原图元保持选中** — 新副本添加到图形中，但选择被清除，命令退出。要立即处理副本，请对选择再次运行复制，或启动新命令。

## 复制 vs 移动

| | 复制 | 移动 |
|---|------|------|
| 原图元 | 保持原位 | 从原位移除 |
| 结果数量 | 增加所复制图元的数量 | 不变 |
| 操作后 | 原图元仍处于选中状态 | 移动后的图元在新位置处于选中状态 |
| 最适用于 | 重复几何图形、对称布局 | 重新定位几何图形 |

## 支持的图元

复制适用于每种图元类型。所有图元都在内部实现了 `translate(dx, dy)`，因此没有任何图元被排除。
