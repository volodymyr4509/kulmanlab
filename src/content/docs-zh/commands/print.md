# Print

`print`（打印）命令打开**打印管理器** — 带实时预览画布、格式选择器（PNG / JPEG / WebP / PDF）、单色切换和可选区域裁剪的专用导出窗口。不会向物理打印机发送任何内容；输出以文件形式下载。

## 打开打印管理器

单击工具栏中的**Print**按钮，或在命令行中输入 `print`。打印管理器立即打开，显示当前视口的预览。

## 打印管理器布局

窗口有两个面板：
- **左侧边栏** — 所有导出控件。
- **右侧面板** — 随设置更改实时更新的实时预览画布。

### 侧边栏控件

| 控件 | 说明 |
|---------|-------------|
| **更改区域** | 裁剪到画布上的自定义矩形（见下文） |
| **单色**切换 | 将所有彩色线条转换为黑色 — 默认开启以获得清晰的打印输出 |
| **格式**下拉菜单 | PNG、JPEG、WebP 或 PDF |
| **导出**按钮 | 生成并下载文件 |

## 选择自定义导出区域

默认情况下，预览显示打开打印管理器时画布上可见的确切内容。要导出特定区域：

1. 单击**更改区域** — 打印管理器隐藏，画布变为交互式。
2. **单击导出矩形的第一个角**。
3. **单击对角** — 打印管理器重新打开，预览中显示选定区域。

区域选择期间按 `Escape` 可取消并恢复之前的区域。

预览画布动态调整大小以匹配选定区域的**精确宽高比**，因此预览像素精确。

## 导出格式

| 格式 | 最适用于 | 备注 |
|--------|----------|-------|
| **PNG** | 无损、清晰线条 | 白色背景，无透明度 |
| **JPEG** | 用于共享的较小文件 | 95% 质量，轻微压缩 |
| **WebP** | 网络最小文件 | 相同 95% 质量，比 JPEG 更好的压缩 |
| **PDF** | 可打印文档 | 图像以 150 DPI 嵌入 PDF 容器 |

导出的文件命名为 `kulman-<时间戳>.<扩展名>` 并自动下载。

## 导出分辨率和背景

- 最大分辨率：**2000 × 2000 像素**，按选定区域比例缩放。
- 背景始终为**白色**。
- 标记为**不打印**的图层被排除在导出之外。

## 键盘参考

| 按键 | 操作 |
|-----|--------|
| `Escape`（区域选择期间） | 取消区域选择，恢复之前的区域 |
| `Escape`（在打印管理器中） | 关闭打印管理器 |
