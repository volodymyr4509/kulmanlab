# 角度标注

`DimensionAngular`（角度标注）命令在图形上放置**角度标注**弧形注释。它测量并标注两条直线之间的角度、圆弧的角度范围或圆的扇形角度。

## 激活方式

在标注面板中单击**角度标注**工具栏按钮，或在命令行中输入 `DimensionAngular`。

## 三种输入模式

第一次单击决定使用哪种模式：

### 两条直线

1. **单击第一条直线。** 光标位置决定使用直线的哪一侧。
2. **单击第二条直线。** 两条直线必须相交（交点自动计算；无需在屏幕上可见）。
3. **单击放置**标注弧。移动光标以选择半径和要标注的角度扇形 — 注释跟随光标到顶点的任意一侧。

平行直线无法形成角度标注；如果直线不相交，命令忽略第二次单击。

### 圆弧

1. **单击圆弧。** 标注立即从圆弧的起始角度到终止角度创建，以圆弧的圆心为顶点。
2. **单击放置**标注弧于所需半径处。

### 圆

1. **单击圆。** 第一个角度端点捕捉到圆上最近的点。
2. **在圆上单击第二个点**以定义第二个角度端点。
3. **单击放置**标注弧。

## 键盘参考

| 按键 | 操作 |
|-----|--------|
| `Escape` | 取消并返回第一次拾取 |

## 行为详情

- 标注弧始终绘制在您放置它的顶点一侧 — 将光标移过顶点可切换到补角。
- 测量的角度以度数显示，并在放置阶段移动光标时实时更新。
- 生成的注释是存储在当前图层上的完整 `DimensionAngular` 实体。其外观属性（箭头大小、文字高度、延伸线长度）可在属性面板中调整。
- 角度标注包含在 JSON 导出中，但 DXF 导出程序不支持。

## 编辑标签 — 简单模式

**双击**已放置的角度标注可在**简单**模式下打开文字编辑器。编辑器预填充当前渲染值，以便您定位光标并直接编辑。

| 功能 | 行为 |
|---------|-----------|
| Bold / Italic / Font / Height | 一次性应用于**整个**标签 |
| 每字符格式 | 不支持 |
| `Enter` | 提交值并关闭编辑器 |
| 多行 | 不支持 |

请参阅[文字编辑器 — 简单模式](../interface/text-editor#simple-mode)获取完整参考。
