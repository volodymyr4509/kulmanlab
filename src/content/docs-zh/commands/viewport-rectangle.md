# ViewportRectangle

`ViewportRectangle`（矩形视口）命令通过拾取两个对角在当前图纸布局中创建新视口。仅在布局空间中可用。

## 创建视口

1. 使用屏幕底部的标签切换到图纸布局。
2. 在命令行中输入 `ViewportRectangle`，或单击工具栏中的**ViewportRectangle**按钮。
3. **单击第一个角**，或输入 `X,Y` 并按 **Enter** 键精确输入坐标。
4. **单击对角** — 视口立即放置。此处也可以输入坐标。

新视口以布局默认比例显示完整模型。在视口内使用滚轮缩放，或中键拖动平移模型视图。

## 坐标输入

在任一角步骤，可以输入精确坐标：

1. 输入 X 值。
2. 按 `,` — 命令行显示 `[X], [Y{光标}]`。
3. 输入 Y 值。
4. 按 **Enter** 放置该点。

## 键盘参考

| 按键 | 操作 |
|-----|--------|
| `0`–`9`、`.`、`-` | 开始输入 X 坐标 |
| `,` | 锁定 X 并移至 Y 输入 |
| `Enter` | 确认输入的坐标 |
| `Escape` | 取消 |

## 编辑视口

放置视口后，单击以选中它：

- **拖动边或角**以调整大小。
- **拖动中心夹点**以移动它。
- 使用控制栏中的**比例选择器**设置精确比例（例如 1:50）。要输入列表中没有的比例，直接在下拉菜单底部的输入字段中输入 — 接受比例格式（`1:200`、`5:1`）或纯小数（`0.005`），然后按 **Enter**。
- 右键单击视口并使用**锁定**以防止意外更改。

## 注意事项

- 矩形视口仅在图纸布局标签激活时可用。在模型空间中运行会显示错误消息并退出。
- 要复制现有视口，请使用[ViewportCopy](../viewport-copy/)。
