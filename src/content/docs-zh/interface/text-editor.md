# 文字编辑器

文字编辑器在放置或双击可编辑图元时打开。标题中的小**模式标记** — **富文本**（强调色）或**简单**（淡色）— 显示当前图元的活动模式。

## 编辑器模式

### 富文本模式

适用于：**文字**（MTEXT 标签）和**多重引线**注释。

| 功能 | 行为 |
|---------|-----------|
| 粗体 / 斜体 / 删除线 | 每字符（应用于选中内容，或无选中时应用于整个图元） |
| 字体和高度 | 每字符覆盖，或整图元默认 |
| Alignment（Left / Center / Right / Justify） | **仅文字** — 多重引线不可用 |
| `Enter` | 插入硬换行 |
| `Shift+←/→` | 扩展或缩小文字选择 |
| `Home` / `End` | 跳到当前硬换行的开头/结尾 |
| 自动换行 | 通过参考宽度调整夹点支持 |

### 简单模式

适用于：**线性标注**、**对齐标注**、**角度标注**、**半径标注**、**直径标注**。

编辑器预填充标注当前渲染的标签，以便您定位光标并直接编辑值。

| 功能 | 行为 |
|---------|-----------|
| 粗体 / 斜体 / 字体 / 高度 | 可用 — 一次性应用于**整个**标签 |
| 每字符格式 | 不支持 |
| `Enter` | **提交**值并关闭编辑器（无换行） |
| 多行 | 不支持 |
| 自动换行 | 不支持 |

## 打开编辑器

| 操作 | 结果 |
|--------|--------|
| `text` 命令 → 单击位置 | 创建新文字图元并打开编辑器（**富文本**） |
| 双击现有**文字**图元 | 在**富文本**模式下重新打开编辑器 |
| 双击现有**多重引线** | 在**富文本**模式下打开编辑器 |
| 双击**标注**图元 | 在**简单**模式下打开编辑器 |
| 编辑器内按 `Escape` | 关闭编辑器并保留所有更改 |

## 工具栏

工具栏浮动在文字边界框上方，随您平移或缩放保持锚定到图元。

### 粗体 · 斜体 · 删除线

| 按钮 | 快捷键 | 功能 |
|--------|----------|--------------|
| **B** | — | 切换粗体 |
| *I* | — | 切换斜体 |
| ~~S~~ | — | 切换删除线 |

**切换如何应用：**

- **有文字选择** — 样式仅应用于所选字符。
- **无选择，光标在现有文字中** — 切换整个图元（所有段）的样式。
- **空文字或新图元** — 样式存储在空段上，并应用于从该点起输入的每个字符。

当当前选择中的每个字符 — 或光标左侧的字符 — 都设置了该样式时，按钮显示为高亮（激活状态）。

### 字体

下拉菜单将可用字体分为 **Default**（内置无衬线字体）、**User**（您自己上传的字体，如果有）、**Free**（一组内置的 Google Fonts 字体）和 **System**（常见的系统字体，例如 Helvetica、Times New Roman、Georgia、Courier New、Verdana、Tahoma、Trebuchet MS、Lucida Console 和 Impact）。

- **有选择** — 仅覆盖所选字符的字体。
- **无选择** — 应用字体到整个图元。

下拉菜单反映无选择时光标左侧字符的字体。

不仅限于内置列表 — 点击工具栏中的 **Font Manager** 按钮即可上传您自己的 `.ttf` 文件并将其添加到 **User** 分组。详见 [Font Manager](../../commands/font-manager/)。

### 高度

数字字段以图形单位设置**大写字母高度**（大写字母的高度）。

- **有选择** — 覆盖所选字符的高度，独立于图元的基础高度。
- **无选择** — 更改图元的基础高度（应用于所有没有单独高度覆盖的字符）。

字段反映光标左侧字符的高度。留空以使用图元默认值。

### 对齐

四个按钮 — **Align Left**、**Align Center**、**Align Right**、**Justify** — 设置段落对齐方式。仅适用于**文字**图元；多重引线和标注标签不显示这些按钮。

- 点击按钮会在图元现有的边界框内重新对齐每一行 — 不会移动插入点，也不会调整边界框大小。
- 再次点击已激活的按钮会清除覆盖，恢复到由图元锚点隐含的列。
- **Justify** 会拉伸词间距，使每一行都填满整行宽度。

## 光标和导航

| 按键 | 操作 |
|-----|--------|
| `←` / `→` | 向左或向右移动插入符一个字符 |
| `Home` | 跳到当前硬换行的开头 |
| `End` | 跳到当前硬换行的结尾 |
| `Shift` + `←` / `→` | 扩展或缩小选择 |
| `Backspace` | 删除左侧字符（或选中内容） |
| `Delete` | 删除右侧字符（或选中内容） |
| `Enter` | 插入换行符 |
| `Escape` | 关闭编辑器 |

光标高度自动匹配相邻字符的大写字母高度，包括用于下标和上标的较小尺寸。

## 复制、剪切与粘贴

| 按键 | 操作 |
|-----|--------|
| `Ctrl+C` / `Cmd+C` | 复制选中的文字 |
| `Ctrl+X` / `Cmd+X` | 剪切选中的文字 |
| `Ctrl+V` / `Cmd+V` | 在光标处粘贴 |

复制和剪切需要有效的文字选择。粘贴的文字始终是纯文本 — 它采用光标处已有的样式(粗体、斜体、字体、高度),而不会保留复制时的样式。

在**富文本模式**下,粘贴文字中的换行会被保留。在**简单模式**下,由于标注标签是单行的,换行会被去除。

## 自动换行

当文字图元设置了**参考宽度**时，长行在词边界处软换行以适应该宽度。

在图元选中时设置或更改参考宽度，拖动**调整大小夹点** — 虚线边界框左右边缘上的细矩形。内容在拖动时实时重新排版。

将参考宽度设置为零（将夹点拖在一起或删除属性面板中的值）可删除自动换行并让行自由增长。

## 多行文字

按 `Enter` 插入硬换行。每条硬换行是独立的 — `Home` 和 `End` 仅在当前硬换行内导航。

硬换行和每字符格式使用 MTEXT 格式存储，在完整 DXF 往返转换后仍然存在。

## DXF 兼容性

文字图元在 DXF 文件中以 **MTEXT** 格式存储。粗体和斜体使用 `\L`、`\K`、`\O` 和内联字体切换（`\f`）编码。每字符高度使用 `\H` 编码。所有格式化在导出时保留，可被 LibreCAD、FreeCAD 和其他支持 DXF 的应用程序读取。
