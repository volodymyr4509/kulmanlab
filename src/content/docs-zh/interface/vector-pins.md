# Vector Pins

**Vector Pins**（矢量固定点）是一种绘图辅助功能，无需绘制辅助线即可将新几何图形与现有点对齐。将光标悬停在捕捉点上半秒即可将其*固定*——固定点随后会投射出不可见的水平和垂直参考线，光标靠近时就会捕捉到这些线上。它相当于桌面 CAD 应用中的对象捕捉追踪功能。

该功能由控制栏中的 **Pins** 开关控制（位于 Grid、Snap 和 ANGL 旁边），**默认开启**，设置在会话之间保持不变。

## 固定一个点

1. 启动一个需要输入点的命令——[Line](../../commands/line/)、[Circle](../../commands/circle/)、[Move](../../commands/move/) 等。
2. 将光标移到现有几何图形的捕捉点上——端点、中点或圆心标记。
3. **保持光标静止 500 毫秒。** 标记变为实心的强调色**方块**——该点已被固定。
4. 重复操作可固定任意数量的点。每个固定点都会持续投射其参考线。

在命令之外也可以固定：将光标悬停在选中图元的**夹点**上，同样可以将其固定。

## 沿参考线追踪

每个固定点都会通过其精确坐标投射两条不可见的参考线——一条**水平**、一条**垂直**。移动光标时：

- 距离固定点垂直线 **12 px** 以内时，光标会捕捉到该线上：一条强调色虚线穿过固定点贯穿整个视图，**X 标记**显示捕捉位置。此时你的 X 坐标*精确*等于固定点的 X。
- 水平线同理，对应固定点的 Y 坐标。
- 靠近两个方向各一条线时——即使来自**两个不同的固定点**——光标会捕捉到它们的**交点**，并同时显示两条虚线。这样就能将点精确放置在（固定点 A 的 X，固定点 B 的 Y）。

```
                    ┆ （虚线，固定点 ■ 的垂直线）
                    ┆
   ■ 固定点 A ┄┄┄┄┄ ✕ ← 光标捕捉到交点：
                    ┆    X 来自固定点 B，Y 来自固定点 A
                    ┆
                    ■ 固定点 B
```

捕捉坐标直接取自固定点，因此对齐是精确的——没有舍入或浮点漂移。

## 捕捉优先级

常规几何捕捉——端点、中点、圆心和交点——**优先于**固定点参考线。如果光标距离点捕捉比距离参考线更近，则点捕捉获胜。固定点追踪填补几何图形之间的空白，绝不会阻碍对几何图形本身的捕捉。

## 与角度锁定结合

Vector pins 可与角度追踪（控制栏中的 **ANGL** 开关）配合使用。当命令将光标锁定到角度追踪射线上时：

- 光标保持约束在锁定方向上。
- 固定点捕捉转为瞄准**锁定射线与固定点参考线的交点**（仅限射线起点前方）。

这可以回答类似*"从上一个点出发的 45° 方向在哪里穿过那个圆心的高度？"*的问题——锁定角度，光标就会卡入交叉点。射线捕捉适用于所有带角度锁定的命令：Line、Polyline、Arc、Circle、Move、Copy、Area、Leader 和 ViewportCopy。

## 固定点的生命周期

固定点是为当前操作服务的，不是永久标记。以下情况会清除所有固定点：

| 事件 | 原因 |
|------|------|
| 触发**新命令** | 每个操作都从一组干净的参考开始 |
| 按下 **Escape** | 标准的"取消一切"行为 |
| 关闭 **Pins** 开关 | 关闭功能即清除其状态 |
| 在**模型空间和图纸空间**之间切换 | 固定点坐标只属于一个空间 |

在单个命令内，你可以固定、绘制、再固定、继续操作——固定点在 Polyline 等多点命令的每次点击后都会保留。

## 典型工作流程

绘制一条起点正好位于某个圆心正下方的直线：

1. 输入 `line`（或单击 Line 按钮）。
2. 将光标悬停在圆的**圆心标记**上半秒——它变为强调色方块。
3. 向下移动光标：靠近圆的垂直线时，光标锁定到虚线参考线上。
4. 单击——直线的起点精确位于圆的 X 坐标上。
5. 照常继续绘制；固定点对后续的点仍然可用。

## 注意事项

- 500 毫秒悬停对光标能到达的任何捕捉标记都有效——包括命令执行过程中出现的捕捉点。
- 悬停在已固定的点上不会有任何效果；没有悬停取消固定的操作。用 **Escape** 或关闭 **Pins** 来清除固定点。
- 参考线的捕捉距离与常规点捕捉相同，为 12 屏幕像素，因此在任何缩放级别下手感一致。
- 固定点以强调色方块显示，取代其常规捕捉标记。
