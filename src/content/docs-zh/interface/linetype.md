# 线型

工具栏中的**线型**选择器控制分配给每个新绘制图元的虚线样式。单击它可打开选择器下拉菜单。

## 选项

| 值 | 含义 |
|-------|---------|
| **随层** | 图元继承其图层上定义的线型。在 DXF 中显示为 `ByLayer`。 |
| **随块** | 图元继承其所属块的线型。在块外没有可见效果。 |
| **连续** | 实线 — 无虚线样式。 |
| **命名线型** | 从当前 DXF 文件加载的任何线型（例如 `DASHED`、`CENTER`、`HIDDEN`、`PHANTOM`……）。下拉菜单显示每种样式的实时预览及其定义字符串。 |

## 适用方式

所选线型应用于更改后创建的每个图元。不会对现有图元产生追溯影响。

要更改现有图元的线型，请选中它们并在属性面板中编辑**线型**字段，或使用[Match Properties](../../commands/match-properties/)从另一个图元复制。

## 线型比例

每个图元还有一个**线型比例**属性（默认值 `1`）。虚线样式乘以此系数。值为 `2` 使虚线长度加倍；`0.5` 使其减半。选中图元后在属性面板中编辑。

## 可用线型

下拉菜单仅列出当前加载的 DXF 文件中存在的线型。新创建的文件只包含 `ByLayer`、`ByBlock` 和 `Continuous`。导入 DXF 时，文件的 `$LTYPE` 表中定义的所有线型都变为可用。

如果需要列表中没有的特定线型（例如 `DASHED2`），请导入包含该线型的 DXF 文件 — 该线型将在当前会话的选择器中出现。

## DXF 兼容性

线型名称以字符串形式存储在图元记录中。`ByLayer` 和 `ByBlock` 是标准 DXF 哨兵值。所有命名线型及其虚线样式在导出时完整保留，并在 LibreCAD、FreeCAD 和其他支持 DXF 的应用程序中往返转换无损。
