# Help

Membuka panel bantuan dalam aplikasi dengan ringkasan semua pintasan papan kekunci, petua navigasi, dan penggunaan umum.

## Cara menggunakan

Klik butang bar alat **Help** atau taip `help` dalam terminal. Panel bantuan terbuka serta-merta dan arahan keluar.

## Apa yang ada dalam panel bantuan

### Navigasi
| Pintasan | Tindakan |
|----------|---------|
| `Scroll wheel` | Zum masuk / keluar |
| `Middle click + drag` | Pan pandangan |
| `Double middle click` | Muatkan semua entiti ke skrin |

### Arahan
| Pintasan | Tindakan |
|----------|---------|
| Taip | Mula menaip nama arahan — cadangan muncul pada kanvas |
| `Delete` | Padam semua entiti yang dipilih |
| `Enter` / `Space` | Sahkan arahan atau input |
| `Enter` / `Space` (melahu) | Ulangi arahan terakhir yang digunakan |
| `Escape` | Batal arahan semasa atau kosongkan pemilihan |

### Pemilihan
| Pintasan | Tindakan |
|----------|---------|
| `Left click` | Pilih entiti |
| `Drag →` | Pilih hanya entiti yang **sepenuhnya di dalam** kawasan seret |
| `Drag ←` | Pilih mana-mana entiti yang **menyentuh** kawasan seret |
| `Left click grip` | Edit bucu atau titik akhir terus |

### Petua lukisan
- **Titik snap** — kursor snap ke titik akhir entiti terdekat secara automatik.
- **Sudut melekit** — gerakkan kursor cukup jauh dalam satu arah untuk mengunci ke gandaan 45°.
- **Panjang tepat** — semasa sudut dikunci, taip nombor dan tekan `Enter` untuk menetapkan jarak tepat.
