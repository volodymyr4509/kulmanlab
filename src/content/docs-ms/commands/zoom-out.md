# Zoom Out

Arahan `zoomout` membahagi paras zum semasa dengan **1.5×** (bersamaan dengan mendarab dengan ~0.667) dan keluar serta-merta, berpusat pada titik tengah viewport. Ia adalah songsangan [Zoom In](./zoom-in).

## Zum keluar

Klik butang bar alat **Zoom Out** atau taip `zoomout` dalam terminal. Zum digunakan serta-merta dan arahan keluar — tiada klik pada kanvas diperlukan.

## Bagaimana langkah 1.5× berfungsi

| Zum semasa | Selepas satu Zoom Out |
|------------|----------------------|
| 1.50× | 1.00× |
| 2.25× | 1.50× |
| 10.00× | 6.67× |
| 0.015× | 0.01× (dihadkan) |

Paras zum sentiasa ditunjukkan di **penjuru kanan bawah** kanvas. Had bawah adalah **0.01×**; langkah selanjutnya tidak membuat apa-apa.

## Zum keluar bar alat berbanding roda tatal

| | Butang Zoom Out | Roda tatal |
|---|----------------|------------|
| Pusat zum | Titik tengah viewport | Kedudukan kursor |
| Saiz langkah | 1.5× setiap klik | ~1.1× setiap tick |
| Pengaktifan diperlukan | Tidak | Tidak — berfungsi sentiasa |
| Terbaik untuk | Melangkah kembali untuk melihat lebih banyak konteks | Zum keluar halus, berlabuh pada kursor |

## Rujukan papan kekunci

Tiada pintasan papan kekunci untuk arahan ini. Gunakan roda tatal sebaliknya — ia berfungsi pada bila-bila masa tanpa mengaktifkan mana-mana arahan.
