# PageManager

Arahan `PageManager` membuka dialog tetapan halaman untuk susun atur aktif, membolehkan anda menukar format kertas, orientasi, dan skala antara unit lukisan dan milimeter.

## Membuka Pengurus Halaman

- Taip `PageManager` dalam terminal, **atau**
- Klik kanan tab susun atur di bahagian bawah dan pilih **Page Manager**.

## Tetapan

| Tetapan | Penerangan |
|---------|------------|
| **Format kertas** | Saiz lembaran standard (A4, A3, A2, Letter, dll.) |
| **Orientasi** | Potret atau Landskap |
| **Skala** | Unit lukisan setiap mm — mengawal bagaimana koordinat model dipetakan ke kertas |

Menukar saiz kertas atau skala mengskala semula semua viewport sedia ada dalam susun atur secara berkadar supaya kedudukan mereka kekal betul.

## Nota

- PageManager hanya terpakai pada susun atur kertas. Ia tidak tersedia dalam ruang model.
- Untuk menambah susun atur baharu, klik butang **+** dalam bar tab di bahagian bawah.
- Untuk menamakan semula atau memadam susun atur, klik kanan tabnya.
