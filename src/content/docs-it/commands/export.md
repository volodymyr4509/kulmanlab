# Export

Scarica il disegno corrente come file DXF o JSON sul tuo computer.

## Come usarlo

1. Attiva il comando **Export** dalla barra degli strumenti o digita `export` nel terminale.
2. Scegli il **formato** (DXF o JSON).
3. Il file viene scaricato nella cartella dei download del browser.

## Formati

| Formato | Uso consigliato |
|---------|----------------|
| **DXF** | Condivisione con AutoCAD e altri software CAD |
| **JSON** | Backup e reimportazione in KulmanLab CAD |

## Suggerimenti

- Esporta regolarmente per conservare copie permanenti dei tuoi disegni.
- Il formato JSON è più fedele all'interno di KulmanLab CAD; usa DXF per l'interoperabilità.
