# LayerMatch

Riassegna le entità selezionate al livello di un'entità sorgente.

## Come usarlo

1. Attiva il comando **LayerMatch** dalla barra degli strumenti o digita `layermatch` nel terminale.
2. **Seleziona** le entità da riassegnare.
3. Premi **Invio** o **Spazio** per confermare la selezione.
4. Clicca sull'**entità sorgente** il cui livello vuoi applicare — le entità selezionate vengono spostate su quel livello.

## Suggerimenti

- Utile per correggere rapidamente entità finita sul livello sbagliato.
- Vedi anche [LayerMakeCurrent](./layer-make-current) per impostare il livello prima del disegno.
