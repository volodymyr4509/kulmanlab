# Move

Sposta le entità selezionate in una nuova posizione.

## Come usarlo

1. Attiva il comando **Move** dalla barra degli strumenti o digita `move` nel terminale.
2. **Seleziona** le entità da spostare (clicca o trascina per selezionare).
3. Premi **Invio** o **Spazio** per confermare la selezione.
4. Clicca per impostare il **punto base** (punto di riferimento per lo spostamento).
5. Clicca per impostare il **punto di destinazione** — le entità vengono spostate.

## Suggerimenti

- Puoi selezionare più entità prima di confermare.
- Usa **Griglia/Snap** per uno spostamento preciso.
- Per spostamenti esatti, dopo il punto base digita le coordinate di destinazione nel terminale.
