# Undo

Annulla l'ultima azione, ripristinando il disegno allo stato precedente.

## Come usarlo

- Clicca il pulsante **Undo** nella barra degli strumenti.
- Oppure premi **Ctrl+Z** (Windows/Linux) o **Cmd+Z** (Mac).
- Oppure digita `undo` nel terminale.

## Suggerimenti

- Puoi annullare più azioni consecutive premendo ripetutamente la scorciatoia.
- Usa [Redo](./redo) per ripristinare un'azione annullata per errore.
