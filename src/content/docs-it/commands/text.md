# Text

Posiziona etichette di testo sulla lavagna.

## Come usarlo

1. Attiva il comando **Text** dalla barra degli strumenti o digita `text` nel terminale.
2. Clicca per impostare il **punto di inserimento** del testo.
3. Si apre l'**editor di testo** — digita il contenuto e premi **Invio** per confermare.

## Editor di testo

L'editor di testo consente di modificare il contenuto dell'etichetta. Vedi [Text Editor](../interface/text-editor) per la documentazione completa delle opzioni disponibili.

## Suggerimenti

- Usa **Griglia/Snap** per allineare il testo ad altri elementi.
- Puoi spostare o eliminare le etichette di testo come qualsiasi altra entità.
