# Delete

Elimina le entità selezionate dal disegno.

## Come usarlo

1. Attiva il comando **Delete** dalla barra degli strumenti o digita `delete` nel terminale.
2. **Seleziona** le entità da eliminare (clicca o trascina).
3. Premi **Invio** o **Spazio** — le entità vengono eliminate.

In alternativa, seleziona prima le entità e poi premi il tasto **Canc** (Delete) sulla tastiera.

## Suggerimenti

- Usa [Undo](./undo) per ripristinare le entità eliminate per errore.
- La selezione con trascinamento verso sinistra (da destra a sinistra) seleziona anche le entità che intersecano il riquadro.
