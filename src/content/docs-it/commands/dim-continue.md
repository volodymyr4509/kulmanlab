# Dimension Continue

Concatena una nuova quota dall'estremità dell'ultima quota creata.

## Come usarlo

1. Crea prima una quota con [Dimension Linear](./dim-linear) o [Dimension Aligned](./dim-aligned).
2. Attiva il comando **Dimension Continue** dalla barra degli strumenti o digita `dimcontinue` nel terminale.
3. Clicca per impostare il **nuovo punto di estensione** — la quota viene aggiunta in catena.
4. Continua a cliccare per aggiungere altre quote in serie, oppure premi **Esc** per terminare.

## Suggerimenti

- Ideale per quotare una serie di elementi allineati in modo ordinato.
- La nuova quota parte automaticamente dall'estremità dell'ultima quota.
