# Match Properties

Copia il colore, il livello e altre proprietà da un'entità sorgente ad altre entità.

## Come usarlo

1. Attiva il comando **Match Properties** dalla barra degli strumenti o digita `matchprop` nel terminale.
2. Clicca sull'**entità sorgente** le cui proprietà vuoi copiare.
3. Clicca sulle **entità di destinazione** — le proprietà vengono applicate a ciascuna.
4. Premi **Esc** per terminare.

## Proprietà copiate

- Livello
- Colore
- Tipo di linea
- Spessore di linea

## Suggerimenti

- Utile per uniformare rapidamente proprietà di stile tra entità diverse.
