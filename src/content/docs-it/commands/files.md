# Files

Apre il pannello dei file recenti per sfogliare e ripristinare i disegni salvati nell'archivio del browser.

## Come usarlo

1. Attiva il comando **Files** dalla barra degli strumenti o digita `files` nel terminale.
2. Si apre il pannello con l'elenco dei disegni recenti.
3. Clicca su un disegno per caricarlo nella lavagna.

## Suggerimenti

- KulmanLab CAD salva automaticamente i disegni nel browser ad ogni modifica.
- I file sono conservati localmente nel browser — non vengono inviati a nessun server.
- Usa [Export](./export) per scaricare una copia permanente sul tuo computer.
- Usa [WipeStorage](./wipestorage) per cancellare tutti i disegni salvati.
