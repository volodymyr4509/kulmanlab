# Redo

Ripristina l'ultima azione annullata con Undo.

## Come usarlo

- Clicca il pulsante **Redo** nella barra degli strumenti.
- Oppure premi **Ctrl+Y** (Windows/Linux) o **Cmd+Y** (Mac).
- Oppure digita `redo` nel terminale.

## Suggerimenti

- Redo è disponibile solo dopo aver utilizzato [Undo](./undo).
- Se esegui una nuova azione dopo un Undo, la cronologia Redo viene cancellata.
