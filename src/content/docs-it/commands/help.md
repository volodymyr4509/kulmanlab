# Help

Apre il pannello di aiuto in-app con un riepilogo di tutte le scorciatoie da tastiera, suggerimenti per la navigazione e utilizzo generale.

## Come usarlo

Clicca il pulsante **Help** nella barra degli strumenti o digita `help` nel terminale. Il pannello di aiuto si apre immediatamente e il comando termina.

## Cosa c'è nel pannello di aiuto

### Navigazione
| Scorciatoia | Azione |
|-------------|--------|
| `Rotella scroll` | Zoom in / out |
| `Clic centrale + trascina` | Pan della vista |
| `Doppio clic centrale` | Adatta tutte le entità allo schermo |

### Comandi
| Scorciatoia | Azione |
|-------------|--------|
| Digita | Inizia a digitare il nome di un comando — i suggerimenti appaiono sul canvas |
| `Canc` | Elimina tutte le entità selezionate |
| `Invio` / `Spazio` | Conferma comando o input |
| `Invio` / `Spazio` (inattivo) | Ripete l'ultimo comando usato |
| `Escape` | Annulla il comando corrente o deseleziona |

### Selezione
| Scorciatoia | Azione |
|-------------|--------|
| `Clic sinistro` | Seleziona un'entità |
| `Trascina →` | Seleziona solo le entità **completamente all'interno** dell'area di trascinamento |
| `Trascina ←` | Seleziona qualsiasi entità che **tocca** l'area di trascinamento |
| `Clic sinistro grip` | Modifica direttamente un vertice o punto finale |

### Suggerimenti per il disegno
- **Punti di ancoraggio** — il cursore si aggancia automaticamente ai punti finali delle entità vicine.
- **Angolo fisso** — sposta il cursore abbastanza lontano in una direzione per bloccarti agli incrementi di 45°.
- **Lunghezza esatta** — mentre l'angolo è bloccato, digita un numero e premi `Invio` per impostare una distanza esatta.

## Versione

La versione corrente dell'app è mostrata nell'angolo in alto a sinistra del pannello di aiuto (es. `KULMAN 2026.04.15`).
