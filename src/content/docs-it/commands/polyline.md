# Polyline

Traccia un percorso aperto multi-segmento come una singola entità polilinea.

## Come usarlo

1. Attiva il comando **Polyline** dalla barra degli strumenti o digita `polyline` nel terminale.
2. Clicca per impostare il **primo punto**.
3. Clicca per aggiungere i punti successivi — ogni clic aggiunge un segmento.
4. Premi **Esc** per terminare la polilinea.

### Chiudere la polilinea

Digita `c` nel terminale o clicca **Chiudi** per collegare l'ultimo punto al primo e creare un contorno chiuso.

## Differenza rispetto a Line

- **Line** crea segmenti separati ad ogni clic; **Polyline** riunisce tutti i segmenti in un'unica entità.
- Le polilinee possono essere spostate, copiate o modificate come oggetto singolo.

## Suggerimenti

- Usa **Griglia/Snap** per tracciare con precisione.
- Premi **Esc** per terminare senza chiudere il percorso.
