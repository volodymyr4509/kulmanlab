# Scale

Scala le entità selezionate attorno a un punto base applicando un fattore di scala uniforme.

## Come usarlo

1. Attiva il comando **Scale** dalla barra degli strumenti o digita `scale` nel terminale.
2. **Seleziona** le entità da scalare.
3. Premi **Invio** o **Spazio** per confermare la selezione.
4. Clicca per impostare il **punto base** (centro di scala).
5. Clicca per impostare il **fattore di scala** o digitane uno nel terminale (es. `2` per raddoppiare, `0.5` per dimezzare).

## Suggerimenti

- Un fattore `1` non modifica nulla; `2` raddoppia le dimensioni; `0.5` dimezza.
- La scala è uniforme — le proporzioni vengono mantenute.
- Usa **Griglia/Snap** per il punto base.
