# Fit

Adatta tutte le entità del disegno nella finestra di visualizzazione con un solo clic.

## Come usarlo

- Clicca il pulsante **Fit** nella barra degli strumenti.
- Oppure premi **F** sulla tastiera.
- Oppure digita `fit` nel terminale.

## Suggerimenti

- Utile per ritrovare rapidamente il disegno dopo aver zoomato troppo o essersi allontanati con Pan.
- Equivalente al comando "Zoom Extents" di altri programmi CAD.
