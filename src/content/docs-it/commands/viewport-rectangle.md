# ViewportRectangle

Crea una finestra di visualizzazione rettangolare in un layout carta selezionando due angoli opposti.

## Come usarlo

1. Passa a un **layout carta** (non al modello).
2. Attiva il comando **ViewportRectangle** dalla barra degli strumenti o digita `viewportrectangle` nel terminale.
3. Clicca per impostare il **primo angolo** della finestra di visualizzazione.
4. Clicca per impostare l'**angolo opposto** — la finestra di visualizzazione viene creata.

## Suggerimenti

- Le finestre di visualizzazione mostrano una porzione dello spazio modello nel layout carta.
- Usa [PageManager](./page-manager) per configurare il formato carta e la scala del layout.
- Usa [ViewportCopy](./viewport-copy) per duplicare una finestra di visualizzazione esistente.
