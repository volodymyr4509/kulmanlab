# Copy

Copia le entità selezionate in una nuova posizione, mantenendo gli originali al loro posto.

## Come usarlo

1. Attiva il comando **Copy** dalla barra degli strumenti o digita `copy` nel terminale.
2. **Seleziona** le entità da copiare (clicca o trascina).
3. Premi **Invio** o **Spazio** per confermare la selezione.
4. Clicca per impostare il **punto base**.
5. Clicca per impostare il **punto di destinazione** — le copie vengono create.

## Suggerimenti

- Le entità originali rimangono nella posizione iniziale.
- Puoi piazzare più copie cliccando più volte prima di premere **Esc**.
- Usa **Griglia/Snap** per distanze precise.
