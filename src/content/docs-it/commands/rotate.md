# Rotate

Ruota le entità selezionate attorno a un punto base.

## Come usarlo

1. Attiva il comando **Rotate** dalla barra degli strumenti o digita `rotate` nel terminale.
2. **Seleziona** le entità da ruotare.
3. Premi **Invio** o **Spazio** per confermare la selezione.
4. Clicca per impostare il **punto base** di rotazione.
5. Clicca per impostare l'**angolo di rotazione** oppure digita un valore in gradi nel terminale.

## Suggerimenti

- Gli angoli positivi ruotano in senso antiorario, quelli negativi in senso orario.
- Digita un valore numerico per una rotazione precisa (es. `90` per 90°).
- Usa **Griglia/Snap** per il punto base.
