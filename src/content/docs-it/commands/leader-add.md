# Leader+

Aggiunge un braccio aggiuntivo a una linea guida multipla esistente.

## Come usarlo

1. Attiva il comando **Leader+** dalla barra degli strumenti o digita `leaderadd` nel terminale.
2. Clicca sulla **linea guida esistente** a cui aggiungere un braccio.
3. Clicca per impostare il **punto della freccia** del nuovo braccio.
4. Clicca per aggiungere i **punti del percorso** del nuovo braccio.
5. Premi **Invio** per confermare.

## Suggerimenti

- Utile per indicare più punti con una singola annotazione di testo condivisa.
- Vedi anche [Leader−](./leader-remove) per rimuovere un braccio.
