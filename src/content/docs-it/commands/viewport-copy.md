# ViewportCopy

Duplica una finestra di visualizzazione esistente in una nuova posizione nel layout carta.

## Come usarlo

1. Passa a un **layout carta** con una finestra di visualizzazione esistente.
2. Attiva il comando **ViewportCopy** dalla barra degli strumenti o digita `viewportcopy` nel terminale.
3. Clicca sulla **finestra di visualizzazione** da duplicare.
4. Clicca per impostare la **nuova posizione** — la copia viene creata.

## Suggerimenti

- La copia mostra la stessa vista del modello dell'originale.
- Puoi poi regolare la vista nella copia in modo indipendente.
