# Print

Esporta un'area del disegno come immagine o PDF tramite la finestra di dialogo di stampa del browser.

## Come usarlo

1. Attiva il comando **Print** dalla barra degli strumenti o digita `print` nel terminale.
2. Definisci l'**area di stampa** selezionando due angoli opposti (oppure il layout carta corrente viene usato automaticamente).
3. Si apre la finestra di dialogo di stampa del browser.
4. Scegli la stampante o seleziona **Salva come PDF** per esportare un PDF.

## Suggerimenti

- Per risultati migliori usa un layout carta configurato con [PageManager](./page-manager).
- In alternativa usa [Export](./export) per scaricare un file DXF.
