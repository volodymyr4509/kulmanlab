# 선종류

도구 모음의 **선종류** 칩은 그리는 모든 새 객체에 지정되는 대시 패턴을 제어합니다. 클릭하면 선택기 드롭다운이 열립니다.

## 옵션

| 값 | 의미 |
|----|------|
| **레이어별** | 객체가 해당 레이어에 정의된 선종류를 상속합니다. DXF에서는 `ByLayer`로 표시됩니다. |
| **ByBlock** | 객체가 속한 블록의 선종류를 상속합니다. 블록 외부에서는 시각적 효과가 없습니다. |
| **연속** | 실선 — 대시 패턴 없음. |
| **명명된 선종류** | 현재 DXF 파일에서 로드된 모든 선종류(예: `DASHED`, `CENTER`, `HIDDEN`, `PHANTOM` 등). 드롭다운은 각 패턴의 라이브 미리보기와 정의 문자열을 표시합니다. |

## 적용 방법

선택된 선종류는 변경 후 생성되는 모든 객체에 적용됩니다. 기존 객체에 소급 적용되지 않습니다.

기존 객체의 선종류를 변경하려면 해당 객체를 선택하고 특성 패널에서 **선종류** 필드를 편집하거나, [특성 일치](../commands/match-properties)를 사용하여 다른 객체에서 복사합니다.

## 선종류 축척

각 객체에는 **선종류 축척** 특성(기본값 `1`)이 있습니다. 대시 패턴이 이 인수로 곱해집니다. `2` 값은 대시를 두 배 길게 만들고, `0.5`는 절반 길이로 만듭니다. 객체를 선택한 후 특성 패널에서 편집합니다.

## 사용 가능한 선종류

드롭다운은 현재 로드된 DXF 파일에 있는 선종류만 나열합니다. 새로 만든 파일에는 `레이어별`, `ByBlock`, `연속`만 포함됩니다. DXF를 가져오면 파일의 `$LTYPE` 테이블에 정의된 모든 선종류를 사용할 수 있습니다.

특정 선종류(예: `DASHED2`)가 목록에 없으면 해당 선종류가 포함된 DXF 파일을 가져옵니다 — 그러면 현재 세션에 대한 선택기에 나타납니다.

## DXF 호환성

선종류 이름은 객체 레코드의 문자열로 저장됩니다. `ByLayer`와 `ByBlock`은 표준 DXF 센티넬 값입니다. 명명된 모든 선종류와 대시 패턴은 내보내기 시 정확하게 보존되며 LibreCAD, FreeCAD 및 기타 DXF 호환 애플리케이션에서 손실 없이 왕복합니다.
