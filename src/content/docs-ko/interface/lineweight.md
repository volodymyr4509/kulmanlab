# 선 가중치

도구 모음의 **선 가중치** 칩은 그리는 모든 새 객체에 지정되는 획 너비를 제어합니다. 클릭하면 선택기 드롭다운이 열립니다.

## 옵션

| 값 | 의미 |
|----|------|
| **레이어별** | 객체가 해당 레이어에 정의된 선 가중치를 상속합니다. 표시되는 실제 너비는 레이어 설정에 따라 다릅니다. |
| **기본** | 애플리케이션 기본 너비를 사용합니다 — 가는 선(1픽셀)으로 렌더링됩니다. DXF에서 레이어 설정을 재정의하지 않습니다. |
| **0.00mm – 2.11mm** | 명시적인 고정 너비. 레이어 선 가중치에 관계없이 객체가 이 값을 가집니다. |

사용 가능한 표준 DXF 선 가중치 값: 0.00, 0.05, 0.09, 0.13, 0.15, 0.18, 0.20, 0.25, 0.30, 0.35, 0.40, 0.50, 0.53, 0.60, 0.70, 0.80, 0.90, 1.00, 1.06, 1.20, 1.40, 1.58, 2.00, 2.11mm.

## 적용 방법

선택된 선 가중치는 변경 후 생성되는 모든 객체에 적용됩니다. 기존 객체에 소급 적용되지 않습니다.

기존 객체의 선 가중치를 변경하려면 해당 객체를 선택하고 특성 패널에서 **선 가중치** 필드를 편집하거나, [특성 일치](../commands/match-properties)를 사용하여 다른 객체에서 복사합니다.

## 렌더링

선 가중치는 **3.78픽셀/mm(96dpi)**로 렌더링됩니다. 0.25mm 선은 화면에서 약 1픽셀 너비이고, 1.00mm 선은 약 4픽셀입니다. 매우 얇은 값(0.00mm 및 음수)은 어떤 줌 수준에서도 보이도록 항상 최소 0.5픽셀로 렌더링됩니다.

## DXF 호환성

선 가중치 값은 DXF의 `LWPOLYLINE`, `LINE`, `CIRCLE` 및 기타 객체 레코드에서 밀리미터의 100분의 1로 정수로 저장됩니다(예: 25 = 0.25mm). **레이어별**은 `-1`로, **기본**은 `-3`으로 저장됩니다 — DXF 사양과 일치합니다. 파일은 모든 DXF 호환 애플리케이션에서 손실 없이 왕복합니다.
