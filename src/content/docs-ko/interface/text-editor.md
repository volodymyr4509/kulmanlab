# 텍스트 편집기

텍스트 편집기는 `text` 명령어로 새 텍스트 레이블을 배치하거나 기존 텍스트 객체를 더블클릭할 때마다 열립니다. 여러 줄 내용, 글자별 서식 및 자동 줄 바꿈을 지원합니다.

## 편집기 여는 방법

| 동작 | 결과 |
|------|------|
| `text` 명령어 → 위치 클릭 | 새 텍스트 객체를 만들고 편집기를 엽니다 |
| 기존 텍스트 객체 더블클릭 | 해당 객체의 편집기를 다시 엽니다 |
| 편집기 내에서 `Escape` | 편집기를 닫고 모든 변경 사항을 유지합니다 |

## 도구 모음

도구 모음은 텍스트의 경계 상자 위에 떠 있으며 이동하거나 줌해도 객체에 고정된 상태를 유지합니다.

### 굵게 · 기울임꼴 · 취소선

| 버튼 | 단축키 | 기능 |
|------|--------|------|
| **B** | — | 굵게 전환 |
| *I* | — | 기울임꼴 전환 |
| ~~S~~ | — | 취소선 전환 |

**전환 적용 방법:**

- **텍스트 선택한 경우** — 선택한 글자에만 스타일이 적용됩니다.
- **선택 없이 기존 텍스트의 커서** — 전체 객체(모든 세그먼트)에서 스타일을 전환합니다.
- **빈 텍스트 또는 새 객체** — 스타일이 빈 세그먼트에 저장되고 그 이후 입력하는 모든 글자에 적용됩니다.

현재 선택의 모든 글자 — 또는 커서 바로 왼쪽의 글자 — 가 해당 스타일로 설정된 경우 버튼이 활성(강조 표시)으로 나타납니다.

### 글꼴

드롭다운은 선별된 크로스 플랫폼 글꼴 세트(Helvetica, Times New Roman, Georgia, Courier New, Verdana, Tahoma, Trebuchet MS, Lucida Console, Impact)를 나열합니다. 상단 옵션 **Arial**이 기본값입니다.

- **선택한 경우** — 선택한 글자에만 글꼴을 재정의합니다.
- **선택 없이** — 전체 객체에 글꼴을 적용합니다.

드롭다운은 선택이 없을 때 커서 왼쪽 글자의 글꼴을 반영합니다.

### 높이

숫자 필드는 도면 단위로 **대문자 높이**(대문자 글자의 높이)를 설정합니다.

- **선택한 경우** — 선택한 글자의 높이를 재정의합니다. 객체의 기본 높이와는 독립적입니다.
- **선택 없이** — 객체의 기본 높이를 변경합니다(개별 높이 재정의가 없는 모든 글자에 적용됩니다).

필드는 커서 왼쪽 글자의 높이를 반영합니다. 비워두면 객체 기본값을 사용합니다.

## 커서 및 탐색

| 키 | 동작 |
|----|------|
| `←` / `→` | 캐럿을 한 글자 왼쪽 또는 오른쪽으로 이동 |
| `Home` | 현재 하드 줄의 시작으로 이동 |
| `End` | 현재 하드 줄의 끝으로 이동 |
| `Shift` + `←` / `→` | 선택 영역 확장 또는 축소 |
| `Backspace` | 왼쪽 글자 삭제(또는 선택 영역 삭제) |
| `Delete` | 오른쪽 글자 삭제(또는 선택 영역 삭제) |
| `Enter` | 줄 바꿈 삽입 |
| `Escape` | 편집기 닫기 |

커서 높이는 인접한 글자의 대문자 높이에 자동으로 맞춰집니다.

## 자동 줄 바꿈

텍스트 객체에 **기준 너비**가 설정된 경우 긴 줄은 해당 너비에 맞도록 단어 경계에서 부드럽게 줄 바꿈됩니다.

객체가 선택된 상태에서 기준 너비를 설정하거나 변경하려면 파선 경계 상자의 왼쪽 및 오른쪽 가장자리에 있는 **크기 조정 그립** — 얇은 직사각형을 드래그합니다. 드래그하는 동안 실시간으로 내용이 재배치됩니다.

기준 너비를 0으로 설정하면(그립을 함께 드래그하거나 특성 패널에서 값 삭제) 자동 줄 바꿈이 제거되고 줄이 자유롭게 늘어납니다.

## 여러 줄 텍스트

`Enter`를 눌러 하드 줄 바꿈을 삽입합니다. 각 하드 줄은 독립적입니다 — `Home`과 `End`는 현재 하드 줄 내에서만 탐색합니다.

하드 줄 바꿈과 글자별 서식은 MTEXT 형식을 사용하여 저장되며 전체 DXF 왕복을 유지합니다.

## DXF 호환성

텍스트 객체는 DXF 파일에서 **MTEXT**로 저장됩니다. 굵게 및 기울임꼴은 `\L`, `\K`, `\O` 및 인라인 글꼴 전환(`\f`)으로 인코딩됩니다. 글자별 높이는 `\H`로 인코딩됩니다. 모든 서식은 내보내기 시 보존되며 LibreCAD, FreeCAD 및 기타 DXF 호환 애플리케이션에서 읽을 수 있습니다.
