# Vector Pins

**Vector Pins**(벡터 핀)는 보조선을 그리지 않고도 새 도형을 기존 점에 정렬할 수 있게 하는 제도 보조 기능입니다. 스냅 점 위에 커서를 반 초 동안 올리면 그 점이 *고정*됩니다 — 핀은 보이지 않는 수평·수직 참조선을 투영하고, 커서가 가까이 가면 그 선에 스냅됩니다. 데스크톱 CAD 애플리케이션의 객체 스냅 추적에 해당하는 KulmanLab CAD 기능입니다.

이 기능은 컨트롤 바의 **Pins** 토글(Grid, Snap, ANGL 옆)로 제어합니다. **기본적으로 켜져 있으며**, 설정은 세션 간에 유지됩니다.

## 점 고정하기

1. 점 입력을 요구하는 명령어를 시작합니다 — [Line](../../commands/line/), [Circle](../../commands/circle/), [Move](../../commands/move/) 등.
2. 기존 도형의 스냅 점 — 끝점, 중간점, 중심 마커 — 위로 커서를 이동합니다.
3. **커서를 500ms 동안 정지 상태로 유지합니다.** 마커가 채워진 강조색 **사각형**으로 바뀝니다 — 점이 고정되었습니다.
4. 필요한 만큼 반복해서 고정할 수 있습니다. 각 핀은 계속 참조선을 투영합니다.

고정은 명령어 밖에서도 작동합니다: 선택된 객체의 **그립** 위에 커서를 올리면 같은 방식으로 고정됩니다.

## 참조선을 따라 추적하기

고정된 각 점은 자신의 정확한 좌표를 지나는 두 개의 보이지 않는 참조선 — **수평선** 하나와 **수직선** 하나 — 을 투영합니다. 커서를 움직이면:

- 핀의 수직선에서 **12px** 이내로 들어오면 커서가 그 선에 스냅됩니다: 핀을 지나 뷰 전체에 걸쳐 점선 강조색 선이 그려지고, **X 마커**가 스냅된 위치를 표시합니다. 이제 X 좌표는 핀의 X와 *정확히* 일치합니다.
- 핀의 Y 좌표에 대한 수평선도 마찬가지입니다.
- 두 방향의 선 하나씩 근처에 있으면 — 심지어 **서로 다른 두 핀**의 선이라도 — 커서가 그 **교차점**에 스냅되고 두 점선이 모두 표시됩니다. 이렇게 하면 점을 정확히 (핀 A의 X, 핀 B의 Y)에 배치할 수 있습니다.

```
                    ┆ (점선, 핀 ■의 수직선)
                    ┆
   ■ 핀 A ┄┄┄┄┄┄┄┄┄ ✕ ← 커서가 교차점에 스냅됨:
                    ┆    X는 핀 B에서, Y는 핀 A에서
                    ┆
                    ■ 핀 B
```

스냅된 좌표는 핀에서 직접 가져오므로 정렬이 정확합니다 — 반올림이나 부동소수점 오차가 없습니다.

## 스냅 우선순위

일반 도형 스냅 — 끝점, 중간점, 중심, 교차점 — 이 핀 참조선보다 **우선**합니다. 커서가 참조선보다 점 스냅에 더 가까우면 점 스냅이 이깁니다. 핀 추적은 도형 사이의 빈 공간을 채우는 것이며, 도형 자체로의 스냅을 절대 방해하지 않습니다.

## 각도 잠금과의 결합

Vector pins는 각도 추적(컨트롤 바의 **ANGL** 토글)과 함께 작동합니다. 명령어가 커서를 각도 추적 광선에 잠갔을 때:

- 커서는 잠긴 방향에 구속된 상태를 유지합니다.
- 핀 스냅은 **잠긴 광선과 핀 참조선의 교차점**을 대상으로 전환됩니다(광선 원점의 앞쪽만).

이는 *"마지막 점에서 45° 방향이 저 원 중심의 높이와 어디에서 교차하는가?"* 같은 질문에 답해 줍니다 — 각도를 잠그면 커서가 교차점에 딱 들어맞습니다. 광선 스냅은 각도 잠금이 있는 모든 명령어에서 작동합니다: Line, Polyline, Arc, Circle, Move, Copy, Area, Leader, ViewportCopy.

## 핀의 수명 주기

핀은 진행 중인 작업을 위한 것이지 영구 마커가 아닙니다. 다음 경우에 모든 핀이 지워집니다:

| 이벤트 | 이유 |
|--------|------|
| **새 명령어**가 실행될 때 | 각 작업은 깨끗한 참조 집합으로 시작 |
| **Escape**를 누를 때 | 표준 전체-취소 동작 |
| **Pins** 토글을 끌 때 | 기능을 끄면 상태도 제거됨 |
| **모델 공간과 종이 공간** 사이를 전환할 때 | 핀 좌표는 한 공간에만 유효 |

하나의 명령어 안에서는 고정하고, 그리고, 다시 고정하며 계속할 수 있습니다 — Polyline 같은 다중 점 명령어의 모든 클릭 후에도 핀은 유지됩니다.

## 일반적인 워크플로

원 중심 바로 아래에서 시작하는 선 그리기:

1. `line`을 입력합니다(또는 Line 버튼 클릭).
2. 원의 **중심 마커** 위에 커서를 반 초 동안 올립니다 — 강조색 사각형으로 바뀝니다.
3. 커서를 아래로 이동합니다: 원의 수직선 근처에서 커서가 점선 참조선에 고정됩니다.
4. 클릭합니다 — 선이 정확히 원의 X 좌표에서 시작됩니다.
5. 평소처럼 선을 계속 그립니다. 핀은 다음 점들을 위해 계속 사용할 수 있습니다.

## 참고

- 500ms 호버는 커서가 닿을 수 있는 모든 스냅 마커에서 작동합니다 — 명령어 도중에 나타나는 스냅 점도 포함됩니다.
- 이미 고정된 점 위에 커서를 올려도 아무 일도 일어나지 않습니다. 호버로 고정을 해제할 수는 없습니다. **Escape**를 누르거나 **Pins**를 꺼서 핀을 지우세요.
- 참조선의 스냅 거리는 일반 점 스냅과 같은 화면 12픽셀이므로, 어떤 줌 수준에서도 감각이 일관됩니다.
- 고정된 점은 일반 스냅 마커 대신 강조색 사각형으로 그려집니다.
