# 그리드 & 스냅

컨트롤 바의 두 전환 버튼을 사용하면 그리드를 표시하고 그리기 중에 커서를 격자 교점에 고정할 수 있습니다.

| 버튼 | 동작 |
|------|------|
| **그리드** | 캔버스에 시각적 점 또는 선 격자를 표시 |
| **스냅** | 더 가까운 기하학적 스냅이 없을 때 커서를 가장 가까운 격자 점에 고정 |

두 전환은 독립적입니다 — 스냅 없이 그리드를 표시하거나, 그리드 없이 스냅하거나, 둘 다 함께 사용할 수 있습니다.

## 그리드 및 스냅 활성화

컨트롤 바 도구 모음에서 **그리드** 또는 **스냅**을 클릭합니다. 활성 상태가 강조 표시됩니다. 설정은 세션 간에 유지됩니다.

**스냅**이 활성화되면 그리드 표시가 선에서 **점**으로 자동으로 전환됩니다 — 점이 커서가 스냅할 정확한 위치를 표시합니다.

## 적응형 격자 간격

격자 간격은 화면에서 격자 선이 항상 편안한 간격(~40 픽셀)이 되도록 줌에 따라 자동으로 조정됩니다. 간격은 항상 "깔끔한" 숫자 — 10의 임의 거듭제곱에서 1, 2 또는 5의 배수:

| 예시 줌 / 모델 축척 | 격자 간격 |
|--------------------|----------|
| 줌 아웃(넓은 영역) | 100, 500, 1000 … |
| 중간 줌 | 10, 20, 50 … |
| 줌 인(세밀한 작업) | 1, 2, 5 … |
| 매우 가까이 | 0.1, 0.2, 0.5 … |

즉 모든 스냅 점이 모델 공간의 둥근 좌표에 위치합니다 — 부동소수점 오프셋이 누적되지 않습니다.

## 스냅 우선순위

**끝점 및 교점 스냅은 항상 그리드보다 우선합니다.** 커서는 기하학적 스냅 후보(끝점, 중간점, 중심점 또는 교점)에 가깝지 않을 때만 격자 점에 스냅됩니다.

즉 그리드 스냅을 활성화한 상태에서 그리기를 할 때도 커서가 기존 기하학에 충분히 가까이 지나가면 정확하게 스냅할 수 있습니다. 그리드는 대체 수단이지 덮어쓰기가 아닙니다.

## 레이아웃 모드

- **모델 공간** — 점 또는 선이 보이는 전체 캔버스 영역을 채웁니다.
- **레이아웃(용지) 공간** — 점이 용지 직사각형에 잘리며 그 바깥으로 확장되지 않습니다.
- **뷰포트 내부** — 그리드는 뷰포트의 축척에서 모델 좌표계를 따르므로 뷰포트 배율에 관계없이 점이 동일한 모델 단위에 정렬됩니다.

## 일반적인 작업 흐름

1. 규칙적인 간격이 필요한 도면을 시작하기 전에 **그리드**와 **스냅**을 켭니다.
2. 격자 간격이 원하는 증분과 일치하는 수준으로 줌합니다(예: 점이 10단위 간격이 될 때까지 줌).
3. 그리기 — 커서가 격자 점에 자동으로 스냅됩니다. 기존 기하학은 가까이 있을 때 여전히 정상적으로 스냅됩니다.
4. 자유로운 커서 이동이 필요하거나 기하학에만 스냅하고 싶을 때는 **스냅**을 끕니다.
