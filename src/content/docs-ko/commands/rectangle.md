# 직사각형

`rectangle` 명령어는 두 개의 반대쪽 모서리 클릭으로 정의된 축 정렬 직사각형을 그립니다. 결과는 각 모서리에 하나씩, 네 꼭짓점이 있는 **닫힌 `LWPOLYLINE`**으로 저장됩니다. 전용 직사각형 객체 유형은 없습니다: 생성 후 도형은 다른 [폴리선](./polyline)과 동일하게 동작하며 모든 폴리선 편집을 적용할 수 있습니다.

## 직사각형 그리기

1. 터미널에 `rectangle`을 입력하거나 **직사각형** 도구 모음 버튼을 클릭합니다.
2. **첫 번째 모서리를 클릭**하거나, `X,Y`를 입력하고 **Enter**를 눌러 정확한 좌표를 입력합니다.
3. **반대쪽 모서리를 클릭** — 직사각형이 즉시 배치되고 명령어가 종료됩니다. 여기서도 좌표 입력이 작동합니다.

```
  ● (첫 번째 클릭)────────────┐
  |                          |
  |   실시간 미리보기가        |
  |   2단계 후 커서를 따름    |
  └──────────────────────────● (두 번째 클릭)
```

두 클릭은 대각선으로 반대쪽 모서리 쌍이면 됩니다 — 왼쪽 위 + 오른쪽 아래, 또는 왼쪽 아래 + 오른쪽 위 등. 순서는 중요하지 않습니다.

## 좌표 입력

모서리 단계에서 정확한 위치를 입력할 수 있습니다:

1. X 값을 입력합니다.
2. `,`를 누릅니다 — 터미널에 `[X], [Y{커서}]`가 표시됩니다.
3. Y 값을 입력합니다.
4. **Enter**를 눌러 모서리를 배치합니다.

## 키보드 참고

| 키 | 동작 |
|----|------|
| `0`–`9`, `.`, `-` | X 좌표 입력 시작 |
| `,` | X를 잠그고 Y 입력으로 이동 |
| `Enter` | 입력한 좌표 확인 |
| `Escape` | 취소 |

면은 항상 수평 및 수직입니다 — 직사각형 명령어에는 각도 잠금이 없습니다.

## 그립 편집 — 생성 후 재형성

선택한 직사각형은 모든 꼭짓점과 각 면의 중간점에 그립을 표시합니다:

| 그립 | 위치 | 기능 |
|------|------|------|
| **모서리** | 4개 꼭짓점 각각 | 드래그하여 해당 꼭짓점 이동; 두 인접 면이 따라서 늘어남 — 반대쪽 모서리는 고정 |
| **면 중간점** | 4개 면 각각의 중심 | 해당 면의 두 끝점을 함께 이동하여 면의 길이와 각도 유지 |

모서리 그립을 드래그하면 직사각형이 비직각 사각형으로 바뀝니다. 크기만 다른 직사각형이 필요하다면 면이 대략 직교 상태를 유지하면서 모서리를 드래그하거나, 삭제하고 새로 그리세요.

## 직사각형 선택

직사각형이 폴리선이므로 선택은 같은 방식으로 작동합니다:

| 방법 | 동작 |
|------|------|
| **클릭** | 클릭이 네 면 중 어느 하나에 닿으면 선택 |
| **오른쪽으로 드래그** (엄격) | 네 꼭짓점이 모두 선택 박스 안에 있어야 함 |
| **왼쪽으로 드래그** (교차) | 박스 경계와 교차하는 임의 면이 전체 직사각형을 선택 |

## 지원되는 편집 명령어

모든 폴리선 편집 명령어를 적용할 수 있습니다. 자르기와 연장은 [선](./line) 전용이며 직사각형에서는 작동하지 않습니다:

| 명령어 | 직사각형에 일어나는 일 |
|--------|----------------------|
| [이동](./move) | 네 꼭짓점 모두를 동일한 변위만큼 이동 |
| [복사](./copy) | 새 위치에 동일한 직사각형 생성 |
| [회전](./rotate) | 선택한 기준점을 중심으로 네 꼭짓점 모두 회전 |
| [대칭](./mirror) | 대칭축을 기준으로 네 꼭짓점 모두 반사 |
| [축척](./scale) | 기준점에서 네 꼭짓점 모두를 균일하게 축척 |
| [간격띄우기](./offset) | 고정된 거리만큼 평행한(안쪽 또는 바깥쪽) 직사각형 생성 |
| [삭제](./delete) | 도면에서 직사각형 제거 |

## 특성

직사각형을 선택하면 특성 패널에 모든 폴리선과 동일한 필드가 표시됩니다:

**일반**

| 특성 | 기본값 | 의미 |
|------|--------|------|
| 색상 | 256 (레이어별) | ACI 색상 인덱스 |
| 레이어 | `0` | 레이어 지정 |
| 선종류 | 레이어별 | 명명된 선종류 패턴 |
| 선종류 축척 | 1 | 선종류 패턴의 축척 인수 |
| 두께 | 0 | 돌출 두께 |

**기하학**

| 특성 | 의미 |
|------|------|
| 닫힘 | 직사각형의 경우 항상 `true` |
| 꼭짓점 수 | 수정되지 않은 직사각형의 경우 항상 `4` |
| 꼭짓점 | 네 모서리의 좌표 |

## 직사각형 vs 폴리선 vs 선

| | 직사각형 | 폴리선 | 선 |
|--|----------|--------|-----|
| 그리기 방법 | 2번 클릭(모서리) | 각 꼭짓점 클릭 | 각 끝점 클릭 |
| 객체 유형 | 닫힌 `LWPOLYLINE` | 열린 또는 닫힌 `LWPOLYLINE` | 세그먼트당 `LINE` |
| 면은 항상 직교 | 예(생성 시) | 아니오 | 아니오 |
| 자르기 / 연장 | 불가능 | 불가능 | 가능 |
| 최적 용도 | 상자, 프레임, 직각 영역 | 임의 윤곽선 및 경로 | 개별 세그먼트, 구성 선 |

## DXF — LWPOLYLINE 객체

직사각형은 네 꼭짓점이 있는 닫힌 `LWPOLYLINE` 객체로 저장됩니다. 꼭짓점 좌표, 색상, 레이어, 선종류, 선종류 축척, 두께를 포함한 모든 특성은 손실 없이 왕복 저장됩니다.

DXF에는 전용 `RECTANGLE` 유형이 없습니다. 파일을 다시 열면 도형은 직사각형이 아닌 닫힌 네 꼭짓점 폴리선으로 나타납니다. `LWPOLYLINE`을 지원하는 모든 DXF 뷰어나 편집기(LibreCAD, FreeCAD 등)에서 올바르게 표시됩니다.
