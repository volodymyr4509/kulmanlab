# Help

모든 키보드 단축키, 탐색 팁 및 일반 사용법 요약이 있는 인앱 도움말 패널을 엽니다.

## 사용 방법

**Help** 도구 모음 버튼을 클릭하거나 터미널에 `help`를 입력합니다. 도움말 패널이 즉시 열리고 명령어가 종료됩니다.

## 도움말 패널의 내용

### 탐색

| 단축키 | 동작 |
|--------|------|
| `스크롤 휠` | 확대/축소 |
| `가운데 클릭 + 드래그` | 뷰 이동(패닝) |
| `가운데 더블클릭` | 화면에 모든 객체 맞추기 |

### 명령어

| 단축키 | 동작 |
|--------|------|
| 입력 시작 | 명령어 이름 입력 시작 — 캔버스에 제안이 나타남 |
| `Delete` | 선택한 모든 객체 삭제 |
| `Enter` / `Space` | 명령어 또는 입력 확인 |
| `Enter` / `Space` (대기 중) | 마지막으로 사용한 명령어 반복 |
| `Escape` | 현재 명령어 취소 또는 선택 해제 |

### 선택

| 단축키 | 동작 |
|--------|------|
| `왼쪽 클릭` | 객체 선택 |
| `오른쪽으로 드래그` | 드래그 영역 **내부에 완전히 포함된** 객체만 선택 |
| `왼쪽으로 드래그` | 드래그 영역에 **교차하는** 모든 객체 선택 |
| `왼쪽 클릭 그립` | 꼭짓점 또는 끝점 직접 편집 |

### 그리기 팁

- **스냅 점** — 커서가 근처 객체의 끝점에 자동으로 스냅됩니다.
- **고정 각도** — 커서를 한 방향으로 충분히 이동하면 45° 간격으로 고정됩니다.
- **정확한 길이** — 각도가 고정된 상태에서 숫자를 입력하고 `Enter`를 눌러 정확한 거리를 설정합니다.
