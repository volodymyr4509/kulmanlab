# Distance

`distance` 명령어는 두 클릭한 점 사이의 직선(유클리드) 거리를 측정하고 결과를 소수점 4자리까지 터미널에 출력합니다. 세 가지 측정 명령어 중 하나입니다 — [Angle](../angle/)은 꼭짓점에서의 각도 개구를, [Area](../area/)는 다각형의 면적과 둘레를 측정합니다.

## 거리 측정의 구조

```
  ● 첫 번째 점
   \
    \  미리보기 선(실시간)
     \
      ● 두 번째 점    →  터미널: "거리: 12.3456"
```

- **첫 번째 점** — 측정의 시작점.
- **두 번째 점** — 끝점; 배치하면 즉시 결과가 출력됩니다.
- **결과** — 터미널에 표시되며 캔버스에는 추가되지 않습니다.

## 거리 측정

1. 터미널에 `distance`를 입력하거나 **Distance** 도구 모음 버튼을 클릭합니다.
2. **첫 번째 점을 클릭**하거나, `X,Y`를 입력하고 **Enter**를 눌러 정확한 좌표를 입력합니다.
3. **두 번째 점을 클릭** — 측정된 거리가 터미널에 나타납니다. 여기서도 좌표 입력이 작동합니다.
4. 선택적으로 **다시 클릭**하여 새 측정을 시작합니다. 명령어가 활성 상태로 유지됩니다.

언제든지 `Escape`를 눌러 2단계로 초기화합니다.

## 측정 연결

결과가 표시된 후 클릭하면 즉시 다음 측정이 시작됩니다 — 클릭한 점이 새 첫 번째 점이 됩니다. 명령어를 다시 활성화하지 않고도 일련의 거리를 측정할 수 있습니다.

## Distance vs Angle

| | Distance | Angle |
|--|------|------|
| 측정 내용 | 직선 길이 | 꼭짓점에서의 내부 각도 |
| 클릭 수 | 2 | 3 |
| 결과 형식 | `12.3456` (단위) | `45.0000°` |
| 캔버스 미리보기 | 첫 번째 점에서 커서까지의 선 | 꼭짓점에서 커서까지의 두 선 |
| 최적 용도 | 간격 또는 세그먼트의 길이 | 두 형상 사이의 개구 각도 |

## 좌표 입력

클릭 대신 어느 점에서든 정확한 위치를 입력합니다:

1. X 값을 입력합니다.
2. `,`를 누릅니다 — 터미널에 `[X], [Y{커서}]`가 표시됩니다.
3. Y 값을 입력합니다.
4. **Enter**를 눌러 확인합니다.

## 키보드 참고

| 키 | 동작 |
|----|------|
| `0`–`9`, `.`, `-` | X 좌표 입력 시작 |
| `,` | X를 잠그고 Y 입력으로 이동 |
| `Backspace` | 마지막으로 입력한 문자 삭제 |
| `Enter` | 입력한 좌표 확인 |
| `Escape` | 취소하고 2단계로 초기화 |

## 참고

- 결과는 **터미널에만** 표시됩니다 — 도면에는 아무것도 추가되지 않습니다.
- 결과는 도면 좌표와 동일한 단위로 표현됩니다(단위 변환 없음).
- 정밀도는 항상 소수점 4자리입니다.
