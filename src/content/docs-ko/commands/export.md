# 내보내기

**내보내기** 명령어는 현재 도면을 파일 시스템으로 다운로드합니다. 다른 CAD 도구와의 호환성을 위한 **DXF**와 KulmanLab CAD 내에서 완전한 충실도로 저장하기 위한 **JSON** 두 가지 형식을 사용할 수 있습니다.

## 내보내기 방법

1. 파일 패널의 **내보내기** 도구 모음 버튼(다운로드 아이콘)을 클릭합니다.
2. **내보내기 관리자** 팝업이 열립니다.
3. 형식 카드를 클릭하여 형식을 선택합니다 — **JSON** 또는 **DXF**.
4. **내보내기** 버튼을 클릭합니다. 파일이 자동으로 기본 다운로드 폴더로 다운로드됩니다.

## 형식 선택

| 형식 | 확장자 | 최적 용도 | 제한 사항 |
|------|--------|----------|----------|
| **JSON** *(기본)* | `.json` | KulmanLab CAD에서 다시 열기 위해 작업 저장 | 다른 CAD 도구와 호환되지 않음 |
| **DXF** | `.dxf` | AutoCAD, FreeCAD, LibreCAD 등과 공유 | 치수와 지시선은 내보내기에 포함되지 않음 |

**JSON 사용 시기:** 작업의 완전한 복사본을 저장하고 싶을 때마다. JSON은 KulmanLab의 기본 형식으로 치수, 지시선, 모든 레이어 데이터를 포함한 모든 객체를 정확하게 보존합니다.

**DXF 사용 시기:** 다른 CAD 애플리케이션을 사용하는 사람에게 도면을 전달해야 할 때. 내보낸 파일은 AC1012 DXF 형식을 사용하며 대부분의 DXF 호환 도구에서 열 수 있습니다.

## 형식별 내보내기 내용

### JSON 내보내기

모든 객체 유형이 포함됩니다:

- 선, 원, 호, 타원, 폴리선, 스플라인, 텍스트
- 치수(선형, 정렬, 연속, 반지름, 지름)
- 다중 지시선
- 레이어 정의 및 선종류 테이블

### DXF 내보내기

기하학 전용 객체가 포함됩니다:

- 선, 원, 호, 타원, 폴리선(`LWPOLYLINE`으로 내보내기), 스플라인, 텍스트
- 레이어 정의 및 선종류 테이블

**DXF에 포함되지 않는 것:** 치수 객체와 다중 지시선. 이것들은 표준 DXF에서 충실하게 표현할 수 없는 KulmanLab 특정 데이터 구조를 사용합니다. 도면에 주석이 있으면 JSON을 사용하거나 시각적으로 캡처하려면 [인쇄](../print/)를 사용하세요.

## 내보낸 파일 이름

다운로드된 파일은 현재 도면 파일의 이름을 따라 지정됩니다(예: `myplan_May22_14:30:00.json`). 확장자는 선택한 형식에 맞게 변경됩니다.

## 내보내기 vs 인쇄

| 기능 | 내보내기 | 인쇄 |
|------|---------|------|
| 출력 | 벡터 소스 파일(.dxf / .json) | 래스터 이미지(.png / .jpeg / .webp / .pdf) |
| 다른 도구에서 편집 가능 | 예(DXF) | 아니오 |
| 레이어 및 선종류 보존 | 예 | 아니오(평면 렌더링) |
| 치수 및 지시선 캡처 | JSON만 가능 | 예 |

편집 가능한 파일이 필요할 때 **내보내기**를 사용합니다. 시각적 스냅샷이 필요하면 [인쇄](../print/)를 사용합니다.
