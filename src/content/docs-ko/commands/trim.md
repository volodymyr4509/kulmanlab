# 자르기

`trim` 명령어는 두 인접 교차점 사이에 있는 [선](../line/)의 부분을 제거하여 선을 하나 또는 두 개의 짧은 세그먼트로 분할합니다. 자를 세그먼트는 커서 위치로 결정됩니다 — 제거하려는 부분 위에 커서를 올리고 클릭하여 자릅니다.

자르기는 **선 객체에서만** 작동합니다. 호, 원, 폴리선 및 기타 객체 유형에는 [삭제](../delete/) 또는 그립 편집을 사용하세요.

## 선 자르기

1. 터미널에 `trim`을 입력하거나 **자르기** 도구 모음 버튼을 클릭합니다.
2. **제거하려는 선 세그먼트 위에 커서를 올립니다** — 빨간색 미리보기가 잘릴 부분을 정확히 강조 표시합니다.
3. **클릭**하여 해당 세그먼트를 제거합니다.

각 자르기 후 명령어가 활성 상태를 유지하므로 계속 호버하고 클릭하여 더 많은 세그먼트를 자를 수 있습니다. **Escape**를 눌러 종료합니다.

```
  이전:                     중간 세그먼트 자른 후:

  ──────●──────●──────        ──────●          ●──────
      교차    교차          (왼쪽 부분)  (오른쪽 부분)
                             (중간 세그먼트 제거됨)
```

## 자르기 세그먼트가 결정되는 방법

명령어는 커서 위치를 호버된 선에 투영하고 선이 다른 객체와 갖는 모든 교차점을 찾습니다. 이러한 교차 매개변수가 선을 세그먼트로 나눕니다. 커서의 투영이 포함된 세그먼트가 강조 표시되고 클릭 시 제거됩니다.

- 커서가 **첫 번째 교차점 앞**에 있으면: 선의 앞부분이 제거됩니다.
- 커서가 **두 교차점 사이**에 있으면: 그 중간 부분이 제거됩니다; 선이 둘로 분할됩니다.
- 커서가 **마지막 교차점 뒤**에 있으면: 뒷부분이 제거됩니다.
- 선이 다른 객체와 **교차점이 없으면**: 미리보기가 표시되지 않고 클릭해도 아무 일도 일어나지 않습니다.

## 지원되는 객체

| 객체 | 자를 수 있나요? |
|------|---------------|
| 선 | 예 |
| 호, 원, 타원 | 아니오 |
| 폴리선 / 직사각형 | 아니오 |
| 텍스트, 스플라인, 치수, 지시선 | 아니오 |

**절단 경계**로 사용되는 객체는 모든 유형이 될 수 있습니다 — 자르는 대상 선만 선 객체여야 합니다.

## 자르기 vs 연장

| | 자르기 | 연장 |
|--|--------|------|
| 하는 일 | 선의 세그먼트 제거 | 선 끝점을 경계까지 늘리기 |
| 트리거 | 자를 세그먼트 위에 커서 올리기 | 연장할 끝점 근처에 커서 올리기 |
| 결과 | 선이 분할되거나 짧아짐 | 선 끝점이 경계로 이동 |
| 모두 | 선만 해당 | 선만 해당 |
