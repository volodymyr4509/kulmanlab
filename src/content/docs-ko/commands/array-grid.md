# Array Grid

`ArrayGrid` 명령어는 선택한 객체로부터 직사각형 그리드 형태의 복사본을 만듭니다 — 행 수, 열 수, 그리고 그 사이의 간격을 입력하며, 모두 터미널에 입력합니다. 원본 선택은 행 0, 열 0 셀을 차지하며, 다른 모든 셀은 이동된 복사본입니다.

## 두 가지 시작 방법

**사전 선택 후 array** — 먼저 객체를 선택한 다음 활성화:

1. 캔버스에서 하나 이상의 객체를 선택합니다.
2. 터미널에 `arraygrid`를 입력하거나(`arr`만으로도 충분합니다 — 명확함) **Array Grid** 도구 모음 버튼을 클릭합니다.
3. **행** 수를 입력하고 **Enter**를 누릅니다.
4. **열** 수를 입력하고 **Enter**를 누릅니다.
5. **행 사이 간격**을 입력하고 **Enter**를 누릅니다.
6. **열 사이 간격**을 입력하고 **Enter**를 누릅니다 — 그리드가 즉시 생성됩니다.

**활성화 후 선택** — 아무것도 선택하지 않고 명령어 시작:

1. `arraygrid`를 입력하거나 도구 모음 버튼을 클릭합니다.
2. **객체 선택** — 클릭하여 개별 객체를 토글하거나, 드래그하여 영역으로 선택합니다.
3. **Enter** 또는 **Space**를 눌러 선택을 확인합니다.
4. 위와 같이 행 → 열 → 행 간격 → 열 간격 순으로 계속합니다.

```
  2행 x 3열:

  [B] [B] [B]   <- 행 1 (이동된 복사본)
  [A] [A] [A]   <- 행 0: 원본 선택, 복사본은 오른쪽으로
```

> 터미널은 명확하게 구분될 만큼의 글자만 있으면 됩니다 — `arr`을 입력하고 **Enter**를 누르면 다른 어떤 명령어 이름도 이 세 글자로 시작하지 않으므로(Arc, Area, Align, Angle은 모두 그보다 앞에서 갈라짐) Array Grid가 바로 활성화됩니다.

## 행, 열, 간격

| 프롬프트 | 허용 값 | 참고 |
|---------|--------|------|
| 행 | 양의 정수 (1, 2, 3…) | 숫자만 가능 — 소수점이나 부호 불가 |
| 열 | 양의 정수 (1, 2, 3…) | 숫자만 가능 — 소수점이나 부호 불가 |
| 행 간격 | 부호 있는 소수 (예: `10`, `-5.5`) | 행 사이의 거리; 음수는 방향을 반전 |
| 열 간격 | 부호 있는 소수 (예: `10`, `-5.5`) | 열 사이의 거리; 음수는 방향을 반전 |

행과 열이 각각 1일 경우 복사본이 생성되지 않습니다 — 명령어는 도면을 변경하지 않고 종료됩니다.

## 키보드 참고

| 키 | 동작 |
|----|------|
| `Enter` / `Space` | 선택을 확인하고 행 프롬프트로 진행 |
| `0`–`9` | 행 또는 열에 숫자 입력 |
| `0`–`9`, `.`, `-` | 행/열 간격에 숫자 입력(`-`는 첫 번째 문자로만 가능) |
| `Backspace` | 현재 프롬프트에서 마지막으로 입력한 문자 삭제 |
| `Enter` | 현재 프롬프트를 확인하고 다음으로 진행 |
| `Escape` | 입력한 행/열/간격 값을 지우고 선택 단계로 돌아감 |

## 명령어 중 선택

| 방법 | 동작 |
|------|------|
| **클릭** | 커서 아래 객체를 선택에서 토글 |
| **오른쪽으로 드래그** (엄격) | 박스 안에 완전히 있는 객체 추가 |
| **왼쪽으로 드래그** (교차) | 박스 경계와 교차하는 객체 추가 |
| **Enter** / **Space** | 선택을 확인하고 행 프롬프트로 진행 |

## array 이후

새 복사본이 도면에 추가되고 명령어가 종료됩니다 — 원본 선택은 지워집니다. **Array Grid**를 다시 실행하거나 새 명령어를 시작하세요.

## Array Grid와 Copy 비교

| | Array Grid | Copy |
|---|-----------|------|
| 점 선택 | 없음 — 행, 열, 간격을 입력 | 기준점과 대상을 클릭(또는 입력) |
| 생성되는 복사본 | 행 × 열 − 1 | 복사 작업당 정확히 1개 |
| 배치 | 규칙적인 직사각형 그리드 | 어디든, 어떤 오프셋이든 |
| 최적 용도 | 규칙적인 패턴(구멍, 타일, 고정구)으로 단위를 반복 | 임의의 위치에 단일 복제본 |

## 지원되는 객체

Array Grid는 모든 객체 유형에서 작동합니다. 모든 객체는 내부적으로 `translate(dx, dy)`를 구현하므로 — [Copy](../copy/)와 [Move](../move/)가 사용하는 것과 동일한 연산 — 제외되는 것이 없습니다.
