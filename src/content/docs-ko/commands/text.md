# 텍스트

`text` 명령어는 여러 줄 텍스트 레이블을 배치합니다. 캔버스에서 위치를 클릭하면 팝업 편집기가 열리고 내용을 입력하고, 굵게/기울임체/취소선을 적용하고, 폰트와 높이를 변경하고, 줄 바꿈을 삽입할 수 있습니다. **Escape**를 눌러 편집기를 확정하고 닫습니다.

전체 편집기 참고는 [텍스트 편집기](../interface/text-editor) 페이지를 참조하세요.

## 텍스트 레이블 배치

1. 터미널에 `text`를 입력하거나 **텍스트** 도구 모음 버튼을 클릭합니다.
2. 캔버스에서 **앵커 위치를 클릭**합니다. 또는 `X,Y`를 입력하고 **Enter**를 눌러 정확한 좌표를 입력합니다.
3. **텍스트 편집기 팝업**이 새 레이블 위에 열립니다. 내용을 입력합니다.
4. **Escape**를 눌러 레이블을 확정하고 편집기를 닫습니다.

기본 높이는 **도면 단위 12**입니다.

## 기존 레이블 편집

캔버스의 임의 텍스트 레이블을 **더블 클릭**하면 해당 레이블에 대한 편집기가 다시 열립니다.

## 앵커에 대한 좌표 입력

클릭 대신 정확한 위치를 입력합니다:

1. X 값을 입력합니다.
2. `,`를 누릅니다 — 터미널에 `[X], [Y{커서}]`가 표시됩니다.
3. Y 값을 입력합니다.
4. **Enter**를 눌러 앵커를 배치하고 편집기를 엽니다.

## 키보드 참고

**앵커 단계**

| 키 | 동작 |
|----|------|
| `0`–`9`, `.`, `-` | X 좌표 입력 시작 |
| `,` | X를 잠그고 Y 입력으로 이동 |
| `Backspace` | 마지막으로 입력한 문자 삭제 |
| `Enter` | 입력한 좌표 확인 |

**텍스트 편집기 단계** (전체 참고는 [텍스트 편집기](../interface/text-editor) 참조)

| 키 | 동작 |
|----|------|
| 인쇄 가능한 모든 문자 | 커서에 삽입 |
| `Backspace` / `Delete` | 인접 문자 또는 선택 영역 삭제 |
| `Enter` | 줄 바꿈 삽입 |
| `←` / `→` | 캐럿 이동 |
| `Home` / `End` | 하드 줄의 시작 / 끝으로 이동 |
| `Escape` | 확정하고 편집기 닫기 |

## 그립 편집 — 위치 변경

선택한 텍스트 레이블은 앵커 점에 하나의 그립을 노출합니다:

| 그립 | 위치 | 기능 |
|------|------|------|
| **앵커** | 텍스트의 왼쪽 아래 | 드래그하여 레이블 위치 변경 |

## 텍스트 선택

| 방법 | 동작 |
|------|------|
| **클릭** | 클릭이 텍스트의 회전된 경계 박스 내에 닿으면 선택 |
| **오른쪽으로 드래그** (엄격) | 경계 박스의 네 모서리가 모두 선택 영역 내에 있어야 함 |
| **왼쪽으로 드래그** (교차) | 텍스트 경계 박스와 선택 영역 사이에 겹치는 부분이 있으면 선택 |

## 지원되는 편집 명령어

| 명령어 | 텍스트에 일어나는 일 |
|--------|---------------------|
| [이동](./move) | 앵커 점 이동 |
| [복사](./copy) | 새 위치에 동일한 레이블 생성 |
| [회전](./rotate) | 앵커 위치를 회전하고 회전 각도 속성에 각도 추가 |
| [대칭](./mirror) | 대칭축을 기준으로 앵커 점 반사(텍스트 문자열은 뒤집히지 않음) |
| [축척](./scale) | 앵커 위치를 축척하고 높이에 축척 인수 곱하기 |
| [삭제](./delete) | 레이블 제거 |

텍스트는 **간격띄우기**, **자르기**, **연장**을 지원하지 않습니다.

## 특성

텍스트 레이블을 선택하면 특성 패널에 다음이 표시됩니다:

**일반**

| 특성 | 기본값 | 의미 |
|------|--------|------|
| 색상 | 256 (레이어별) | ACI 색상 인덱스 |
| 레이어 | `0` | 레이어 지정 |

**기하학**

| 특성 | 의미 |
|------|------|
| 위치 X / 위치 Y | 앵커 점 좌표 |
| 높이 | 도면 단위로 표시된 기본 텍스트 높이(기본값: **12**) |
| 회전 각도 | 반시계 방향 회전(도 단위) |

**특성**

| 특성 | 의미 |
|------|------|
| 내용 | 텍스트 문자열(MTEXT 인라인 코드 보존) |
| 부착 점 | 정렬 코드(1 = 왼쪽 위 … 9 = 오른쪽 아래) |

텍스트에는 선종류, 선종류 축척, 두께 특성이 없습니다.

## DXF — MTEXT 객체

텍스트 레이블은 DXF 파일에 **MTEXT** 객체로 저장됩니다. 굵게 및 기울임체는 `\L`, `\K`, `\O`, 인라인 폰트 전환(`\f`)을 사용하여 인코딩됩니다. 문자별 높이는 `\H`로 인코딩됩니다. 모든 서식은 완전한 DXF 왕복을 거쳐도 유지되며 LibreCAD, FreeCAD 및 기타 DXF 호환 응용 프로그램에서 읽을 수 있습니다.
