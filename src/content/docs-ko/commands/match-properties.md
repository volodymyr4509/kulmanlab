# 특성 일치

`MatchProperties` 명령어는 원본 객체에서 하나 이상의 대상 객체로 **시각적 및 레이어 특성**을 복사합니다. 원본과 대상 객체 유형 사이에 공유된 특성만 전달됩니다 — 기하학은 절대 변경되지 않습니다.

## 활성화 방법

스타일 패널의 **특성 일치** 도구 모음 버튼(페인트 롤러 아이콘)을 클릭하거나, 터미널에 `MatchProperties`를 입력합니다.

## 워크플로우

**먼저 활성화한 다음 원본 선택:**

1. 아무것도 사전 선택하지 않고 `MatchProperties`를 입력하거나 도구 모음 버튼을 클릭합니다.
2. **원본 객체를 클릭** — 복사하려는 특성을 가진 객체.
3. **각 대상 객체를 클릭**하여 원본 특성을 적용합니다. 여러 객체를 하나씩 클릭할 수 있습니다.
4. 한 번에 그룹에 적용하려면 대상 위에 **선택 박스를 드래그**합니다.
5. **Enter** 또는 **Escape**를 눌러 완료합니다.

**원본을 사전 선택한 다음 활성화:**

1. 단일 객체를 클릭하여 선택합니다.
2. `MatchProperties`를 활성화합니다. 선택한 객체가 자동으로 원본으로 사용됩니다.
3. 대상 객체를 클릭하거나 드래그-선택한 다음, **Enter** 또는 **Escape**를 눌러 완료합니다.

## 복사되는 특성

MatchProperties는 원본과 대상 사이의 공유 기본 클래스에 속하는 특성을 복사합니다. 최소한 **모든 객체 유형**이 이러한 특성을 공유합니다:

| 특성 | 설명 |
|------|------|
| **색상** | 객체의 색상 인덱스("레이어별" / "블록별" 포함) |
| **레이어** | 객체가 속한 레이어 |

원본과 대상이 같은 객체 유형인 경우(예: 두 치수 모두), 추가적인 유형별 특성도 복사됩니다 — 예를 들어 텍스트 높이, 화살표 크기, 치수 보조선 설정.

기하학(좌표, 반지름, 길이 등)은 절대 영향을 받지 않습니다.

## 키보드 참고

| 키 | 동작 |
|----|------|
| `Enter` / `Space` | 영역 선택을 확인하거나 명령어 완료 |
| `Escape` | 적용 완료(원본이 설정된 경우) 또는 취소 |

## 동작 세부 정보

- 원본 객체 자체는 절대 수정되지 않습니다.
- 각 클릭 또는 드래그-선택은 원본 특성을 즉시 적용합니다 — 확인 단계가 없습니다.
- 영역 선택은 표준 규칙을 따릅니다: **오른쪽으로** 드래그는 엄격 선택(완전히 포함), **왼쪽으로** 드래그는 교차 선택(임의 교차).
- 원본 객체를 대상으로 클릭하면 무시됩니다.
