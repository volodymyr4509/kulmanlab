# 연장

`extend` 명령어는 [선](./line)의 가장 가까운 끝점을 도면의 다른 객체와 형성할 가장 가까운 교차점까지 늘립니다. 연장하려는 끝점 근처에 커서를 올리면 미리보기가 연장된 선을 보여줍니다 — 그런 다음 클릭하여 적용합니다.

연장은 **선 객체에서만** 작동합니다. 선이 연장 방향으로 향하는 경계는 텍스트, Mtext, 다중 지시선, 스플라인을 제외한 모든 다른 객체 유형이 될 수 있습니다.

## 선 연장

1. 터미널에 `extend`를 입력하거나 **연장** 도구 모음 버튼을 클릭합니다.
2. **선의 한쪽 끝 근처에 커서를 올립니다** — 미리보기가 그 방향에서 가장 가까운 경계까지 연장된 선을 보여줍니다.
3. **클릭**하여 연장을 적용합니다.

각 연장 후 명령어가 활성 상태를 유지하므로 여러 선을 순서대로 연장할 수 있습니다. **Escape**를 눌러 종료합니다.

```
  이전:                      이후:

  ──────           |           ──────────────|
  (짧은 선)     (경계)  (경계까지 연장됨)
```

## 끝점이 선택되는 방법

명령어는 커서가 어느 끝점에 더 가까운지 확인합니다:

- 커서가 **끝점에 더 가까우면** → 끝이 선 방향을 따라 앞으로 연장됩니다.
- 커서가 **시작점에 더 가까우면** → 시작점이 반대 방향으로(역방향으로) 연장됩니다.

선택된 끝점에서 선 방향을 따라 광선을 보내고, 다른 모든 객체(선 자체 및 무시된 유형 제외)와의 해당 광선을 따른 **가장 가까운 교차점**이 새 끝점이 됩니다.

그 방향에서 교차점이 없으면 미리보기가 나타나지 않고 클릭해도 아무 일도 일어나지 않습니다.

## 경계 제외

다음 객체 유형은 경계로 무시됩니다 — 선이 만나도록 연장되지 않습니다:

- 텍스트 / Mtext
- 다중 지시선
- 스플라인

다른 모든 유형(선, 호, 원, 타원, 폴리선, 치수)은 유효한 경계 역할을 합니다.

## 지원되는 객체

| 객체 | 연장할 수 있나요? |
|------|----------------|
| 선 | 예 |
| 호, 원, 타원 | 아니오 |
| 폴리선 / 직사각형 | 아니오 |
| 텍스트, 스플라인, 치수, 지시선 | 아니오 |

## 연장 vs 자르기

| | 연장 | 자르기 |
|--|------|--------|
| 하는 일 | 선 끝점을 경계까지 늘리기 | 선의 세그먼트 제거 |
| 트리거 | 늘릴 끝점 근처에 커서 올리기 | 자를 세그먼트 위에 커서 올리기 |
| 결과 | 선 끝점이 바깥쪽으로 이동 | 선이 분할되거나 짧아짐 |
| 모두 | 선만 해당 | 선만 해당 |
