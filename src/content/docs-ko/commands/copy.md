# Copy

`copy` 명령어는 선택한 객체의 이동된 복사본을 만들어 기준점에서 대상까지의 오프셋에 배치합니다 — 원본은 정확히 그 자리에 유지됩니다. 이것이 [Move](../move/)과의 주요 차이점입니다: 복사는 도면에 새 객체를 추가하고; 이동은 기존 객체를 재배치합니다.

## 두 가지 시작 방법

**사전 선택 후 복사** — 먼저 객체를 선택한 다음 활성화:

1. 캔버스에서 하나 이상의 객체를 선택합니다.
2. 터미널에 `copy`를 입력하거나 **Copy** 도구 모음 버튼을 클릭합니다.
3. **기준점을 클릭**하거나, `X,Y`를 입력하고 **Enter**를 눌러 정확한 좌표를 입력합니다.
4. **대상을 클릭** — 복사본이 기준→대상 오프셋에 나타납니다. 여기서도 좌표 입력이 작동합니다.

**활성화 후 선택** — 아무것도 선택하지 않고 명령어 시작:

1. `copy`를 입력하거나 도구 모음 버튼을 클릭합니다.
2. **객체 선택** — 클릭하여 개별 객체를 토글하거나, 드래그하여 영역으로 선택합니다.
3. **Enter** 또는 **Space**를 눌러 선택을 확인합니다.
4. **기준점을 클릭**한 다음 **대상을 클릭**합니다(두 단계에서 좌표 입력 가능).

```
  이전:               이후:
  [객체 A]            [객체 A]  ← 원본 그대로
  [객체 B]            [객체 B]
                      [A의 복사본] ← 새 객체
                      [B의 복사본]
```

복사본의 고스트 미리보기가 기준점에서 대상으로 커서를 따라 이동합니다.

## 좌표 입력

기준점 또는 대상 단계에서 클릭 대신 정확한 위치를 입력합니다:

1. X 값을 입력합니다.
2. `,`를 누릅니다 — 터미널에 `[X], [Y{커서}]`가 표시됩니다.
3. Y 값을 입력합니다.
4. **Enter**를 눌러 확인합니다.

## 각도 잠금 및 정확한 거리

기준점이 설정된 후 커서가 충분히 멀리 있고 축 근처에 있을 때 명령어가 45° 축(0°, 45°, 90°, 135°, …)에 스냅됩니다. 잠긴 상태에서 거리를 입력하고 **Enter**를 누르면 복사본이 정확히 그 오프셋에 배치됩니다.

| 키 | 동작 |
|----|------|
| `0`–`9`, `.` | 거리 값에 숫자 추가 |
| `-` | 음수 거리 — 축을 따라 방향 반전(첫 번째 문자만) |
| `Backspace` | 마지막으로 입력한 문자 삭제 |
| `Enter` | 입력한 거리에 복사본 배치 |

## 키보드 참고

| 키 | 동작 |
|----|------|
| `Enter` / `Space` | 선택 확인 및 기준점 단계로 진행 |
| `0`–`9`, `.`, `-` | X 좌표 입력 시작 또는 각도 잠금 시 거리 |
| `,` | X를 잠그고 Y 입력으로 이동 |
| `Backspace` | 마지막으로 입력한 문자 삭제 |
| `Enter` | 좌표 확인 또는 입력한 거리로 복사 적용 |
| `Escape` | 취소하고 초기화 |

## 명령어 중 선택

| 방법 | 동작 |
|------|------|
| **클릭** | 커서 아래 객체를 선택에서 토글 |
| **오른쪽으로 드래그** (엄격) | 박스 안에 완전히 있는 객체 추가 |
| **왼쪽으로 드래그** (교차) | 박스 경계와 교차하는 객체 추가 |
| **Enter** / **Space** | 선택 확인 |

## 복사 후

**원본은 선택된 상태로 유지됩니다** — 새 복사본은 도면에 추가되지만 선택이 지워지고 명령어가 종료됩니다. 복사본을 즉시 작업하려면 선택으로 다시 복사를 실행하거나 새 명령어를 시작하세요.

## Copy vs Move

| | Copy | Move |
|--|------|------|
| 원본 | 제자리에 유지 | 원래 위치에서 제거 |
| 결과 수 | 복사된 객체 수만큼 증가 | 변경 없음 |
| 작업 후 | 원본이 여전히 선택됨 | 새 위치에서 이동된 객체 선택 |
| 최적 용도 | 기하학 반복, 대칭 레이아웃 | 기하학 재배치 |

## 지원되는 객체

복사는 모든 객체 유형에서 작동합니다. 모든 객체는 내부적으로 `translate(dx, dy)`를 구현하므로 제외되는 것이 없습니다.
