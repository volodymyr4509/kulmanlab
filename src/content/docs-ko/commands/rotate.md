# Rotate

`rotate` 명령어는 기준점을 중심으로 선택한 객체를 회전합니다. 숫자를 입력하거나 클릭하여 회전 각도를 지정합니다 — 기준점과 클릭 위치 사이의 방향에서 각도가 계산됩니다.

## 두 가지 시작 방법

**사전 선택 후 회전** — 먼저 객체를 선택한 다음 활성화:

1. 캔버스에서 하나 이상의 객체를 선택합니다.
2. 터미널에 `rotate`를 입력하거나 **Rotate** 도구 모음 버튼을 클릭합니다.
3. **기준점을 클릭** — 회전 중심. 또는 `X,Y`를 입력하고 **Enter**를 눌러 정확한 좌표를 입력합니다.
4. **각도를 입력하고 Enter를 누르거나**, **클릭**하여 커서 방향에서 각도를 설정합니다.

**활성화 후 선택** — 아무것도 선택하지 않고 명령어 시작:

1. `rotate`를 입력하거나 도구 모음 버튼을 클릭합니다.
2. **객체 선택** — 클릭하여 토글하거나, 드래그하여 영역으로 선택합니다.
3. **Enter** 또는 **Space**를 눌러 선택을 확인합니다.
4. **기준점을 클릭**(좌표 입력 가능)한 다음 각도를 설정합니다.

```
  이전:            이후(● 중심 90° 회전):
                      ╔══╗
  ●  [객체]    →   ● ║    ║
                      ╚══╝
```

기준점이 설정된 후 회전된 객체의 실시간 고스트 미리보기가 커서 각도를 따라 이동합니다.

## 각도 설정

**입력한 각도** — 기준점이 배치된 후 언제든지 숫자(도 단위)를 입력합니다. 미리보기가 Enter를 누르기 전에 계속 조정하는 동안 입력한 각도로 스냅됩니다.

**클릭 각도** — 입력한 값이 없으면 클릭이 `atan2(커서Y − 기준Y, 커서X − 기준X)`와 같은 각도를 설정합니다 — 기준점에서 클릭까지의 방향, 도 단위.

| 키 | 동작 |
|----|------|
| `0`–`9`, `.` | 각도 값에 숫자 추가 |
| `-` | 음수 각도(첫 번째 문자만) |
| `Backspace` | 마지막으로 입력한 문자 삭제 |
| `Enter` | 입력한 각도로 회전 적용 |

## 각도 방향

각도는 **DXF 규칙**을 따릅니다:

- **양의** 값은 도면 좌표(Y 위쪽)에서 **반시계 방향**으로 회전합니다.
- Y축이 반전된(Y 아래쪽) 화면에서는 양의 각도가 **시계 방향**으로 나타납니다.

일반적인 값: `90` = 사분면 회전, `180` = 반 회전, `-90` = 반대 사분면 회전.

## 키보드 참고

| 키 | 동작 |
|----|------|
| `Enter` / `Space` | 선택 확인 |
| `0`–`9`, `.`, `-` | X 좌표 입력 시작(기준점 단계) 또는 각도 값(각도 단계) |
| `,` | X를 잠그고 Y 입력으로 이동(기준점 단계) |
| `Backspace` | 마지막으로 입력한 문자 삭제 |
| `Enter` | 좌표 확인 또는 회전 적용 |
| `Escape` | 취소하고 초기화 |

## 명령어 중 선택

| 방법 | 동작 |
|------|------|
| **클릭** | 커서 아래 객체 토글 |
| **오른쪽으로 드래그** (엄격) | 박스 안에 완전히 있는 객체 추가 |
| **왼쪽으로 드래그** (교차) | 박스와 교차하는 객체 추가 |
| **Enter** / **Space** | 선택 확인 |

## 지원되는 객체

회전은 모든 객체 유형에서 작동합니다. 각 객체의 기하학이 기준점을 중심으로 회전됩니다 — 예를 들어 원은 중심이 이동하면서 반지름은 동일하게 유지됩니다; 호는 중심이 이동하고 시작 및 끝 각도가 회전량만큼 이동합니다; 텍스트 객체는 앵커 점이 이동하고 회전 각도에 각도가 추가됩니다.
