# WipeStorage

`wipestorage` 명령어는 KulmanLab CAD의 **브라우저에 저장된 모든 데이터**를 영구적으로 삭제합니다 — 저장된 모든 파일, 레이어 및 선종류 테이블, 실행 취소 기록. 이후 페이지가 자동으로 새로 고침됩니다.

:::danger 되돌릴 수 없음
이 작업은 취소할 수 없습니다. 브라우저에 저장된 모든 파일이 삭제됩니다. 이 명령어를 실행하기 전에 보관하고 싶은 모든 도면을 `.json` 또는 `.dxf` 파일로 내보내세요.
:::

## 언제 사용할지

- 브라우저 저장소가 손상되어 앱이 파일을 로드하거나 저장하는 데 실패할 때.
- 앱을 완전히 초기화된 상태로 재설정하고 싶을 때.
- 브라우저 또는 기기를 바꾸었고 로컬 복사본이 더 이상 필요하지 않을 때.

## 실행 방법

1. 터미널에 `wipestorage`를 입력하고 **Enter**를 누릅니다.
2. 터미널이 묻습니다: *모든 브라우저 로컬 저장소를 지울까요? YES를 입력하여 확인하세요*
3. `YES`(대소문자 관계없이)를 입력하고 **Enter**를 누릅니다.

앱이 데이터베이스를 삭제하고 페이지를 새로 고침합니다. `YES` 이외의 것을 입력하고 **Enter**를 누르거나 **Escape**를 누르면 명령어가 취소되고 아무것도 삭제되지 않습니다.

## 삭제되는 내용

| 데이터 | 삭제됨 |
|--------|--------|
| 브라우저에 저장된 모든 파일 | 예 |
| 각 파일의 레이어 및 선종류 테이블 | 예 |
| 각 파일의 실행 취소/다시 실행 기록 | 예 |

**이 브라우저에 로컬로 저장된 데이터만** 영향을 받습니다. 이미 `.json` 또는 `.dxf`로 내보낸 파일은 손상되지 않습니다.

## 개별 파일 삭제

모든 저장소를 지우는 대신 단일 파일만 제거하려면 [파일](../files/) 패널에서 해당 파일 옆의 **삭제** 버튼을 사용하세요.
