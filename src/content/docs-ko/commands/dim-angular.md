# Dimension Angular

`DimensionAngular` 명령어는 도면에 **각도 치수** 호 주석을 배치합니다. 두 선 사이의 각도, 호의 범위, 또는 원의 섹터를 측정하고 레이블을 붙입니다.

## 활성화 방법

마크업 패널의 **Dimension Angular** 도구 모음 버튼을 클릭하거나, 터미널에 `DimensionAngular`를 입력합니다.

## 세 가지 입력 모드

첫 번째 클릭이 어떤 모드를 사용할지 결정합니다:

### 두 선

1. **첫 번째 선을 클릭합니다.** 커서 위치가 선의 어느 쪽을 사용할지 결정합니다.
2. **두 번째 선을 클릭합니다.** 두 선은 교차해야 합니다(교차점이 자동으로 계산됩니다; 화면에 표시될 필요가 없음).
3. **클릭하여 치수 호를 배치합니다.** 커서를 이동하여 반지름과 어떤 각도 섹터에 레이블이 붙을지 선택합니다 — 주석은 꼭짓점의 어느 쪽에 있든지 커서를 따릅니다.

평행선은 각도 치수를 형성할 수 없습니다; 선이 교차하지 않으면 명령어가 두 번째 클릭을 무시합니다.

### 호

1. **호를 클릭합니다.** 치수는 호의 시작 각도에서 끝 각도까지 호의 중심을 꼭짓점으로 사용하여 즉시 생성됩니다.
2. **클릭하여** 원하는 반지름에서 치수 호를 배치합니다.

### 원

1. **원을 클릭합니다.** 첫 번째 각도 끝점이 원의 가장 가까운 점에 스냅됩니다.
2. **원의 두 번째 점을 클릭**하여 두 번째 각도 끝점을 정의합니다.
3. **클릭하여** 치수 호를 배치합니다.

## 키보드 참고

| 키 | 동작 |
|----|------|
| `Escape` | 취소하고 첫 번째 선택으로 돌아가기 |

## 동작 세부 정보

- 치수 호는 배치한 꼭짓점 측면에 항상 그려집니다 — 보완 각도로 전환하려면 꼭짓점을 가로질러 커서를 이동합니다.
- 측정된 각도는 도 단위로 표시되며 배치 중 커서를 이동할 때 실시간으로 업데이트됩니다.
- 결과 주석은 현재 레이어에 저장된 완전한 `DimensionAngular` 객체입니다. 외형 특성(화살표 크기, 텍스트 높이, 치수 보조선 길이)은 특성 패널에서 조정할 수 있습니다.
- 각도 치수는 JSON 내보내기에 포함되지만 DXF 내보내기에서는 지원되지 않습니다.

## 레이블 편집 — 단순 모드

배치된 각도 치수를 **더블클릭**하면 **단순** 모드로 텍스트 편집기가 열립니다. 편집기에는 현재 렌더링된 값이 미리 입력되어 있어 커서를 놓고 직접 편집할 수 있습니다.

| 기능 | 동작 |
|------|------|
| Bold / Italic / Font / Height | **레이블 전체**에 한 번에 적용 |
| 글자별 서식 | 지원되지 않음 |
| `Enter` | 값을 확정하고 편집기 닫기 |
| 여러 줄 | 지원되지 않음 |

자세한 내용은 [텍스트 편집기 — 단순 모드](../../interface/text-editor/#simple-mode)를 참조하세요.
