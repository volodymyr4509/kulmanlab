# ViewportCopy

`ViewportCopy` 명령어는 뷰포트를 새 위치에 복사하며 축척과 모델 중심을 보존합니다. 레이아웃 공간에서만 사용 가능합니다.

## 뷰포트 복사

1. 용지 레이아웃 탭으로 전환합니다.
2. 선택적으로 뷰포트를 클릭하여 사전 선택합니다.
3. 터미널에 `ViewportCopy`를 입력하거나 **뷰포트 복사** 도구 모음 버튼을 클릭합니다.
4. 뷰포트가 사전 선택되지 않은 경우 복사할 **뷰포트를 클릭**합니다.
5. **기준점을 클릭** — 변위의 참조. 또는 `X,Y`를 입력하고 **Enter**를 눌러 정확한 좌표를 입력합니다.
6. **대상을 클릭** — 뷰포트가 기준→대상 오프셋에 배치됩니다. 또는 좌표 입력 / 각도 잠금을 사용합니다.

배치 후 명령어가 활성 상태로 유지됩니다 — 다른 대상을 클릭하여 같은 뷰포트의 다른 복사본을 배치합니다. **Enter**, **Space** 또는 **Escape**를 눌러 완료합니다.

## 좌표 입력

기준점 및 대상 단계에서 클릭 대신 정확한 좌표를 입력할 수 있습니다:

1. X 값을 입력합니다.
2. `,`를 누릅니다 — 터미널에 `[X], [Y{커서}]`가 표시됩니다.
3. Y 값을 입력합니다.
4. **Enter**를 눌러 확인합니다.

## 각도 잠금 및 정확한 거리

기준점이 설정된 후 명령어는 커서가 정렬될 때 45° 축(0°, 45°, 90°, 135°, …)에 스냅됩니다. 잠긴 상태에서:

- 미리보기가 축에 스냅됩니다.
- 거리를 입력하고 **Enter**를 누르면 잠긴 방향을 따라 정확히 그 오프셋에 복사본이 배치됩니다.

| 키 | 동작 |
|----|------|
| `0`–`9`, `.` | 거리 값에 숫자 추가 |
| `-` | 음수 거리(방향 반전; 첫 번째 문자만) |
| `Backspace` | 마지막으로 입력한 문자 삭제 |
| `Enter` | 입력한 거리에 복사본 배치 |

## 키보드 참고

| 키 | 동작 |
|----|------|
| `0`–`9`, `.`, `-` | X 좌표 입력 시작 또는 각도 잠금 시 거리 |
| `,` | X를 잠그고 Y 입력으로 이동 |
| `Enter` | 입력한 좌표 또는 거리 확인 |
| `Enter` / `Space` | 완료(입력이 진행 중이지 않을 때) |
| `Escape` | 취소하고 초기화 |

## 참고

- ViewportCopy는 용지 레이아웃 탭이 활성화된 경우에만 사용 가능합니다.
- 복사된 뷰포트는 원본과 동일한 축척, 모델 중심, 잠금 상태, 치수를 상속합니다.
- 처음부터 새 뷰포트를 생성하려면 [ViewportRectangle](../viewport-rectangle/)을 사용하세요.
