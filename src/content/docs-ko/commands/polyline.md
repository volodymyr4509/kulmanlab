# Polyline

`polyline` 명령어는 단일 `LWPOLYLINE` 객체로 저장된 임의 개수의 직선 세그먼트로 이루어진 연결 경로를 그립니다. 전체 경로가 하나의 객체이므로, 선택하면 모든 세그먼트가 한 번에 선택됩니다 — 단일 작업으로 전체 도형을 이동, 회전 또는 축척할 수 있습니다. 이것이 각 세그먼트가 독립적인 객체인 연결된 [Line](../line/)과의 주요 차이점입니다.

폴리선은 **닫힌** 상태가 될 수도 있습니다: [Rectangle](../rectangle/) 명령어는 닫기 플래그가 설정된 동일한 `LWPOLYLINE` 객체를 사용합니다.

## 폴리선 그리기

1. 터미널에 `polyline`을 입력하거나 **Polyline** 도구 모음 버튼을 클릭합니다.
2. **첫 번째 점을 클릭**하거나, `X,Y`를 입력하고 **Enter**를 눌러 정확한 좌표를 입력합니다.
3. **이후 각 점을 클릭** — 각 클릭이 세그먼트를 추가합니다. 모든 단계에서 좌표 입력이 작동합니다.
4. **Enter** 또는 **Space**를 눌러 완료합니다(최소 2개 점이 배치되어야 함).

```
  ●──────●
  1번째   2번째
          \
           \  세그먼트 3 (진행 중 — 커서 위치)
            ●  ← 클릭하여 추가, Enter/Space로 완료
```

언제든지 **Escape**를 누르면 배치된 모든 점이 버려지고 명령어가 종료됩니다.

## 좌표 입력

클릭 대신 임의 꼭짓점에 대한 정확한 위치를 입력합니다:

1. X 값을 입력합니다.
2. `,`를 누릅니다 — 터미널에 `[X], [Y{커서}]`가 표시됩니다.
3. Y 값을 입력합니다.
4. **Enter**를 눌러 꼭짓점을 배치합니다.

## 각도 잠금 및 정확한 세그먼트 길이

[선](../line/#각도-잠금-및-정확한-길이-입력) 명령어와 동일한 45° 스냅 로직이 연속된 두 점 사이에 적용됩니다. 축에 잠긴 상태에서:

| 키 | 동작 |
|----|------|
| `0`–`9`, `.` | 세그먼트 길이에 숫자 추가 |
| `-` | 음수 길이 — 축을 따라 방향을 반전(첫 번째 문자만) |
| `Backspace` | 마지막으로 입력한 문자 삭제 |
| `Enter` | 입력한 거리에 다음 점 배치 |

현재 누적 길이는 터미널 프롬프트에 실시간으로 표시됩니다. 잠긴 상태에서 클릭하면 축에 투영되어 새 꼭짓점이 정확히 그 위에 위치합니다.

## 키보드 참고

| 키 | 동작 |
|----|------|
| `0`–`9`, `.`, `-` | X 좌표 입력 시작 또는 각도 잠금 시 세그먼트 길이 |
| `,` | X를 잠그고 Y 입력으로 이동 |
| `Backspace` | 마지막으로 입력한 문자 삭제 |
| `Enter` | 입력한 좌표 또는 길이 확인, 또는 아무것도 입력하지 않고 점이 2개 이상 있으면 폴리선 완료 |
| `Space` | 폴리선 완료(입력이 진행 중이지 않을 때 Enter와 동일) |
| `Escape` | 모든 점을 버리고 종료 |

## 그립 편집 — 꼭짓점 및 세그먼트 중간점

선택한 폴리선은 두 가지 유형의 그립을 표시합니다:

| 그립 | 위치 | 기능 |
|------|------|------|
| **꼭짓점** | 각 배치된 점에서 | 드래그하여 해당 꼭짓점의 위치 변경; 연결된 모든 세그먼트가 따라서 늘어남 |
| **세그먼트 중간점** | 각 세그먼트의 중심 | 세그먼트의 **두** 끝점을 함께 이동하여 세그먼트 길이와 각도를 유지한 채 이동 |

세그먼트 중간점 그립은 폴리선만의 특징입니다 — 길이를 변경하지 않고 개별 세그먼트를 옆으로 밀 수 있습니다. [Line](../line/)에서는 중간점 그립이 전체 객체에 대한 이동 명령어를 활성화합니다.

전체 폴리선을 이동하는 "전체 이동" 그립은 없습니다. 전체 경로를 이동하려면 [Move](../move/) 명령어를 사용하세요.

## 폴리선 선택

| 방법 | 동작 |
|------|------|
| **클릭** | 클릭이 임의 세그먼트의 히트 테스트 거리 내에 있으면 폴리선 선택 |
| **오른쪽으로 드래그** (엄격) | 모든 꼭짓점이 박스 안에 있어야 함 |
| **왼쪽으로 드래그** (교차) | 박스 경계와 교차하는 임의 세그먼트가 전체 폴리선을 선택 |

폴리선은 하나의 객체이므로, 임의 세그먼트에 닿는 교차 선택은 모든 세그먼트를 선택합니다.

## 지원되는 편집 명령어

폴리선은 모든 일반 변환과 간격띄우기를 지원하지만, 자르기와 연장은 지원하지 않습니다([Line](../line/) 전용):

| 명령어 | 폴리선에 일어나는 일 |
|--------|--------------------:|
| [Move](../move/) | 모든 꼭짓점을 동일한 변위만큼 이동 |
| [Copy](../copy/) | 새 위치에 동일한 폴리선 생성 |
| [Rotate](../rotate/) | 선택한 기준점을 중심으로 모든 꼭짓점 회전 |
| [Mirror](../mirror/) | 대칭축을 기준으로 모든 꼭짓점 반사 |
| [Scale](../scale/) | 기준점에서 모든 꼭짓점을 균일하게 축척 |
| [Offset](../offset/) | 고정된 수직 거리만큼 평행한 폴리선 생성 |
| [Delete](../delete/) | 도면에서 폴리선 제거 |

## 특성

폴리선을 선택하면 특성 패널에 다음이 표시됩니다:

**일반**

| 특성 | 기본값 | 의미 |
|------|--------|------|
| 색상 | 256 (레이어별) | ACI 색상 인덱스 |
| 레이어 | `0` | 레이어 지정 |
| 선종류 | 레이어별 | 명명된 선종류 패턴 |
| 선종류 축척 | 1 | 선종류 패턴의 축척 인수 |
| 두께 | 0 | 돌출 두께 |

**기하학**

| 특성 | 의미 |
|------|------|
| 닫힘 | 마지막 꼭짓점이 첫 번째 꼭짓점으로 다시 연결되는지 여부 |
| 꼭짓점 수 | 꼭짓점의 총 개수 |
| 꼭짓점 | 모든 꼭짓점의 좌표 목록 |

## 폴리선 vs 선 — 언제 무엇을 사용할지

| | 폴리선 | 선 |
|--|--------|-----|
| 객체 수 | 전체 경로에 하나의 `LWPOLYLINE` | 세그먼트당 하나의 `LINE` |
| 닫힌 도형 | 가능(닫기 플래그) | 불가능 |
| 자르기 / 연장 | 불가능 | 가능 — 세그먼트별 |
| 세그먼트 중간점 그립 | 전체 세그먼트를 이동 | 객체에 대한 이동 활성화 |
| 최적 용도 | 전체를 유지할 윤곽선, 도형 | 구성 선, 자를 기하학 |

## DXF — LWPOLYLINE 객체

폴리선은 DXF 파일에 `LWPOLYLINE` 객체로 저장됩니다. 꼭짓점 좌표, 닫기 플래그, 색상, 레이어, 선종류, 선종류 축척, 두께를 포함한 모든 특성은 손실 없이 왕복 저장됩니다. [Rectangle](../rectangle/) 명령어로 그린 직사각형도 `LWPOLYLINE`(닫힌, 네 꼭짓점)으로 저장되며 DXF 수준에서 구별할 수 없습니다.

DXF 호환 응용 프로그램(LibreCAD, FreeCAD 등)의 `LWPOLYLINE` 객체는 편집기에서 완전히 편집 가능한 폴리선으로 다시 읽힙니다.
