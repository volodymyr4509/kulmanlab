# LayerMatch

`LayerMatch` 명령어는 선택한 객체의 레이어를 클릭한 원본 객체의 레이어에 맞게 재지정합니다. 레이어 관리자를 열지 않고도 객체 그룹을 올바른 레이어로 이동하는 가장 빠른 방법입니다.

## 워크플로우

**사전 선택 후 일치**:

1. 레이어를 변경하려는 객체를 선택합니다.
2. `LayerMatch`를 입력하거나 **LayerMatch** 도구 모음 버튼(페인트 아이콘)을 클릭합니다.
3. **원본 객체를 클릭** — 레이어를 복사하려는 객체.
4. 선택한 모든 객체가 즉시 원본 객체의 레이어로 이동합니다.

**활성화 후 선택**:

1. 아무것도 선택하지 않고 `LayerMatch`를 입력하거나 도구 모음 버튼을 클릭합니다.
2. **대상 객체 선택** — 클릭하여 개별 객체를 토글하거나 드래그하여 영역으로 선택합니다.
3. **Enter** 또는 **Space**를 눌러 선택을 확인합니다.
4. **원본 객체를 클릭** — 해당 레이어가 모든 대상에 적용됩니다.

## 키보드 참고

| 키 | 동작 |
|----|------|
| `Enter` / `Space` | 대상 선택을 확인하고 원본 선택 단계로 진행 |
| `Escape` | 초기화 — 대상 선택으로 돌아가거나 완전히 취소 |

## 동작 세부 정보

- `layer` 특성만 변경됩니다 — 색상, 선종류, 선 가중치, 기하학은 변경되지 않습니다.
- 원본 객체 자체는 수정되지 않습니다.
- 원본이 클릭된 후 명령어가 완료됩니다.
- 원본 선택 단계 중 빈 캔버스를 클릭하면 아무 일도 일어나지 않습니다.
