# 초점 이동

`pan` 명령어는 지속적인 드래그-스크롤 모드를 시작합니다 — 캔버스의 아무 곳이나 클릭하고 드래그하여 뷰를 이동합니다. 확대/축소 수준은 변경되지 않습니다. 초점 이동 모드는 `Escape`를 누를 때까지 활성 상태로 유지되므로 한 번의 활성화로 여러 번 드래그할 수 있습니다.

## 뷰 이동

1. 터미널에 `pan`을 입력하거나 **초점 이동** 도구 모음 버튼을 클릭합니다.
2. 캔버스의 아무 곳이나 **클릭하고 드래그**하여 뷰를 이동합니다.
3. 필요한 만큼 놓고 다시 드래그합니다.
4. 초점 이동 모드를 종료하려면 `Escape`를 누릅니다.

## 가운데 클릭 드래그 — 명령어 활성화 없이 초점 이동

가장 빠른 초점 이동 방법: 다른 명령어가 활성화되어 있을 때도 언제든지 **가운데 마우스 버튼을 누른 채 드래그합니다**. 명령어 활성화가 필요 없습니다. 가운데 버튼을 놓으면 이전 상태로 돌아갑니다.

## 초점 이동 vs 확대/축소 컨트롤

| 동작 | 효과 | 초점 이동 모드 필요? |
|------|------|-------------------|
| 왼쪽 클릭 드래그 (초점 이동 모드) | 뷰포트 이동 | 예 |
| 가운데 클릭 드래그 | 뷰포트 이동 | 아니오 — 항상 작동 |
| 스크롤 휠 | 커서 방향으로 확대/축소 | 아니오 — 항상 작동 |
| 가운데 버튼 더블 클릭 | 모든 객체 맞추기 | 아니오 — 항상 작동 |
| [확대](./zoom-in) / [축소](./zoom-out) | 단계별 1.5× 확대/축소 | 아니오 |
| [전체 보기](./fit) | 뷰에 모든 객체 맞추기 | 아니오 |

## 키보드 참고

| 키 | 동작 |
|----|------|
| `Escape` | 초점 이동 모드 종료 |

## 팁

- 스크롤 휠을 사용하여 대상 영역으로 확대/축소한 다음 초점 이동하여 위치를 미세 조정합니다.
- **가운데 마우스 버튼 더블 클릭**은 [전체 보기](./fit)를 즉시 실행합니다 — 잃어버린 뷰를 초기화하는 가장 빠른 방법입니다.
- 명령어 중간에 있다면(예: 선 끝점 배치 중), 명령어를 취소하지 않고 가운데 클릭 드래그로 초점 이동합니다.
