# 스플라인 피트

`splinefit` 명령어는 클릭한 모든 점을 통과하는 3차 스플라인을 그립니다 — 보간 곡선. 곡선이 제어 꼭짓점 방향으로만 끌어당겨지는 [스플라인 CV](./spline-cv)와 달리, 여기서는 곡선이 각 클릭한 좌표를 정확히 통과하도록 강제됩니다. 내부적으로 편집기는 이를 달성하기 위해 제어 꼭짓점에 맞추고, 해당 CV는 DXF 파일에 피트 점과 함께 저장됩니다.

## 피트 점으로 스플라인 그리기

1. 터미널에 `splinefit`을 입력하거나 **스플라인 피트** 도구 모음 버튼을 클릭합니다.
2. **클릭하여 피트 점 배치** — 곡선이 각 점을 통과합니다. 또는 `X,Y`를 입력하고 **Enter**를 눌러 정확한 좌표를 입력합니다.
3. **Enter**를 눌러 완료합니다(최소 2개 점 필요).

```
  ●──────●──────●──────●  ← 곡선이 각 클릭을 정확히 통과
  p1     p2     p3     p4
```

실시간 미리보기는 커서 위치에서 다음 점을 포함하여 커서를 이동할 때 현재 보간된 곡선을 보여줍니다. **Escape**를 누르면 배치된 모든 점이 버려지고 종료됩니다.

## 좌표 입력

클릭 대신 임의 피트 점에 대한 정확한 위치를 입력합니다:

1. X 값을 입력합니다.
2. `,`를 누릅니다 — 터미널에 `[X], [Y{커서}]`가 표시됩니다.
3. Y 값을 입력합니다.
4. **Enter**를 눌러 피트 점을 배치합니다.

## 키보드 참고

| 키 | 동작 |
|----|------|
| `0`–`9`, `.`, `-` | X 좌표 입력 시작 |
| `,` | X를 잠그고 Y 입력으로 이동 |
| `Backspace` | 마지막으로 입력한 문자 삭제 |
| `Enter` | 입력한 좌표 확인 또는 입력 중이지 않고 점이 2개 이상이면 스플라인 완료 |
| `Escape` | 모든 점을 버리고 종료 |

## 그립 편집 — 피트 점으로 재형성

선택한 피트 스플라인은 피트 점마다 하나의 그립을 노출합니다:

| 그립 | 위치 | 기능 |
|------|------|------|
| **피트 점** | 각 클릭한 위치 | 드래그하여 해당 피트 점 이동 — 전체 곡선이 새 위치를 통과하도록 재보간 |

하나의 그립을 드래그하면 인접 세그먼트뿐만 아니라 전체 곡선이 재피팅됩니다. 이것은 꼭짓점을 이동하면 인접 두 세그먼트만 재형성되는 폴리선 그립 편집과 다릅니다.

"전체 스플라인 이동" 그립은 없습니다. 전체 스플라인을 이동하려면 [이동](./move) 명령어를 사용하세요.

## 피트 스플라인 선택

| 방법 | 동작 |
|------|------|
| **클릭** | 클릭이 곡선의 임의 점 근처에 닿으면 선택 |
| **오른쪽으로 드래그** (엄격) | 곡선을 따른 모든 샘플 점이 선택 박스 안에 있어야 함 |
| **왼쪽으로 드래그** (교차) | 선택 박스 경계와 교차하는 곡선의 어느 부분이든 선택 |

## 지원되는 편집 명령어

| 명령어 | 스플라인에 일어나는 일 |
|--------|----------------------|
| [이동](./move) | 모든 피트 점과 재계산된 CV를 동일한 변위만큼 이동 |
| [복사](./copy) | 새 위치에 동일한 스플라인 생성 |
| [회전](./rotate) | 선택한 기준점을 중심으로 모든 피트 점 회전 |
| [대칭](./mirror) | 대칭축을 기준으로 모든 피트 점 반사 |
| [축척](./scale) | 기준점에서 모든 피트 점을 균일하게 축척 |
| [삭제](./delete) | 스플라인 제거 |

스플라인은 **간격띄우기**, **자르기**, **연장**을 지원하지 않습니다.

## 특성

**일반**

| 특성 | 기본값 | 의미 |
|------|--------|------|
| 색상 | 256 (레이어별) | ACI 색상 인덱스 |
| 레이어 | `0` | 레이어 지정 |
| 선종류 | 레이어별 | 명명된 선종류 패턴 |
| 선종류 축척 | 1 | 선종류 패턴의 축척 인수 |
| 두께 | 0 | 돌출 두께 |

**기하학**

| 특성 | 의미 |
|------|------|
| 차수 | 다항식 차수 — 항상 3(3차) |
| 피트 점 | 클릭한 모든 통과 점의 좌표 |
| 제어 꼭짓점 | 곡선을 렌더링하는 데 사용되는 내부적으로 계산된 CV |

## 스플라인 피트 vs 스플라인 CV — 무엇을 사용할지

| | 스플라인 피트 | 스플라인 CV |
|--|-------------|------------|
| 곡선이 점을 통과 | 클릭한 모든 점 정확히 | 첫 번째와 마지막만(고정) |
| 그립 편집 효과 | 피트 점 이동 → 전체 곡선 재보간 | CV 이동 → 새 위치 방향으로 끌어당겨짐 |
| 형상 예측성 | 높음 — 곡선이 클릭을 따름 | 낮음 — 곡선이 CV 뒤에 위치 |
| 최적 용도 | 특정 좌표를 통과해야 하는 곡선 | 부드러운 미적 곡선, 자유 형태 경로 |

## DXF — SPLINE 객체 (피트 점 형식)

피트 스플라인은 DXF 파일에 `SPLINE` 객체로 저장되며, 피트 점 좌표와 계산된 제어 꼭짓점을 모두 저장합니다. `splineFlag`는 `8`(피트 점 스플라인)로 설정되어 재로드하는 응용 프로그램이 어떤 점 집합을 편집 가능한 그립으로 표시할지 알 수 있습니다. 색상, 레이어, 선종류, 선종류 축척, 두께를 포함한 모든 특성은 손실 없이 왕복 저장됩니다. 피트 점 스플라인을 지원하는 DXF 응용 프로그램(LibreCAD, FreeCAD)은 피트 점을 기본 편집 가능 데이터로 표시합니다.
