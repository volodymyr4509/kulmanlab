# 삭제

`delete` 명령어는 도면에서 선택한 객체를 제거합니다. 삭제는 [실행 취소](../undo/) 기록에 기록되며 최대 20단계까지 되돌릴 수 있습니다. 별도의 "삭제 확인" 대화 상자가 없습니다 — 확인은 단 한 번의 키 입력입니다.

## 두 가지 삭제 방법

**사전 선택 후 삭제** — 가장 빠른 방법:

1. 캔버스에서 하나 이상의 객체를 선택합니다.
2. 터미널에 `delete`를 입력하거나, **삭제** 도구 모음 버튼을 클릭하거나, **`Delete` 키를 직접 누릅니다**.

객체가 즉시 제거됩니다 — 추가 확인 단계가 없습니다.

**활성화 후 선택**:

1. 아무것도 선택하지 않고 `delete`를 입력하거나 도구 모음 버튼을 클릭합니다.
2. **객체 선택** — 클릭하여 토글하거나, 드래그하여 영역으로 선택합니다.
3. **Enter**, **Space** 또는 **Delete**를 눌러 선택한 객체를 확인하고 제거합니다.

## Delete 키 단축키

키보드의 `Delete` 키는 **전역 단축키**로 작동합니다 — 현재 선택된 객체가 있으면 Delete 명령어를 터미널에서 열지 않고도 즉시 삭제됩니다. 가장 빠른 단일 단계 삭제 워크플로우입니다:

```
객체 클릭 → Delete 키 누르기 → 완료
```

## 명령어 중 선택

| 방법 | 동작 |
|------|------|
| **클릭** | 커서 아래 객체를 선택에서 토글 |
| **오른쪽으로 드래그** (엄격) | 박스 안에 완전히 있는 객체만 선택 |
| **왼쪽으로 드래그** (교차) | 박스 경계와 교차하는 객체 선택 |
| **Enter** / **Space** / **Delete** | 확인하고 선택된 객체 삭제 |

## 삭제된 객체 복구

삭제는 [실행 취소](../undo/) 명령어(`undo`를 입력하거나 도구 모음 버튼 사용)로 취소할 수 있습니다. 파일당 최대 **20단계**를 되돌릴 수 있으며, 기록은 페이지 새로고침 후에도 유지됩니다. 저장 없이 20번 이상 삭제한 경우 이전 삭제는 복구할 수 없습니다.

## 지원되는 객체

삭제는 모든 객체 유형에서 작동합니다 — 선, 폴리선, 직사각형, 원, 호, 타원, 텍스트, 스플라인, 치수, 지시선 및 기타 모든 것.
