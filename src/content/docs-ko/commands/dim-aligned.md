# Dimension Aligned

`dimaligned` 명령어는 두 점 사이의 **실제 직선 거리**를 측정하는 치수를 배치합니다. 치수 선은 두 점을 연결하는 선에 평행하게 임의 각도로 실행됩니다. 이것이 수평 또는 수직으로만 제한되는 [Dimension Linear](../dim-linear/)와의 주요 차이점입니다.

## 정렬 치수의 구조

```
     ●  p2
    /|
   / |  (치수 보조선 2, 치수 선에 수직)
  /  |
 /←5.00→/
/  /
●  /  (치수 보조선 1, 치수 선에 수직)
p1
```

- **치수 보조선** — 각 측정 점에서 치수 선에 수직으로 그려집니다.
- **치수 선** — p1→p2에 평행하며 커서 위치에 의해 한쪽으로 오프셋됩니다.
- **값** — 실제 유클리드 거리 `|p1 – p2|`.

## 정렬 치수 배치

1. 터미널에 `dimaligned`를 입력하거나 **Dimension Aligned** 도구 모음 버튼을 클릭합니다.
2. **첫 번째 치수 보조선 시작점(p1)을 클릭**하거나, `X,Y`를 입력하고 **Enter**를 눌러 정확한 좌표를 입력합니다.
3. **두 번째 치수 보조선 시작점(p2)을 클릭**합니다. 여기서도 좌표 입력이 작동합니다.
4. 치수 선의 수직 오프셋을 설정하기 위해 **커서를 한쪽으로 이동**합니다.
5. **클릭**하여 배치하거나, 오프셋 거리를 입력하고 **Enter**를 눌러 정확히 배치합니다.

## 입력한 오프셋 거리

p1→p2 선에서 정확한 수직 거리에 치수 선을 고정하려면 배치 중 숫자를 입력합니다:

| 키 | 동작 |
|----|------|
| `0`–`9`, `.` | 오프셋에 숫자 추가 |
| `Backspace` | 마지막으로 입력한 문자 삭제 |
| `Enter` / `Space` | 입력한 오프셋에 배치 |

커서 측면이 치수 선이 어느 쪽에 나타날지 결정합니다.

## 키보드 참고

| 키 | 동작 |
|----|------|
| `0`–`9`, `.`, `-` | X 좌표 입력 시작(p1/p2 단계) 또는 오프셋 거리(배치 단계) |
| `,` | X를 잠그고 Y 입력으로 이동(p1/p2 단계) |
| `Backspace` | 마지막으로 입력한 문자 삭제 |
| `Enter` / `Space` | 입력한 좌표 또는 오프셋 확인 |
| `Escape` | 취소 |

## Dimension Aligned vs Dimension Linear

| | Dimension Aligned | Dimension Linear |
|--|----------|----------|
| 치수 선 각도 | p1→p2에 평행 — 임의 각도 | 항상 수평 또는 수직 |
| 측정 내용 | 실제 유클리드 거리 | X 또는 Y 구성 요소만 |
| H/V 방향 잠금 | 아니오 | 예 — `H` 및 `V` 키 |
| 최적 용도 | 대각선 형상, 경사진 절단 | 직교 레이아웃, 그리드 정렬 부품 |

## 레이블 편집 — 단순 모드

배치된 정렬 치수를 **더블클릭**하면 **단순** 모드로 텍스트 편집기가 열립니다. 편집기에는 현재 렌더링된 값이 미리 입력되어 있어 커서를 놓고 직접 편집할 수 있습니다.

| 기능 | 동작 |
|------|------|
| Bold / Italic / Font / Height | **레이블 전체**에 한 번에 적용 |
| 글자별 서식 | 지원되지 않음 |
| `Enter` | 값을 확정하고 편집기 닫기 |
| 여러 줄 | 지원되지 않음 |

자세한 내용은 [텍스트 편집기 — 단순 모드](../../interface/text-editor/#simple-mode)를 참조하세요.

## 치수 연결

이 정렬 치수의 두 번째 치수 보조선에서 계속되는 치수를 추가하려면 [Dimension Continue](../dim-continue/)를 사용하세요 — 이 정렬 치수와 동일한 측정 각도로 잠깁니다.

## DXF — DIMENSION 객체 (정렬 유형)

정렬 치수는 `dimType = 1`(정렬)로 저장된 `DIMENSION` 객체로 저장됩니다. 치수 보조선 시작점, 치수 선 위치, 텍스트 위치, 측정된 값, 회전, 화살표 스타일, 모든 표시 플래그가 손실 없이 왕복 저장됩니다.
