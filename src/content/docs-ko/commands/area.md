# Area

`area` 명령어는 세 개 이상의 클릭한 점으로 정의된 다각형의 면적과 둘레를 측정하고, 두 결과를 모두 소수점 4자리까지 터미널에 출력합니다. [Distance](../distance/)(직선 길이), [Angle](../angle/)(꼭짓점에서의 내부 각도)과 함께 세 번째 측정 명령어입니다.

## 면적 측정의 구조

```
  ● 첫 번째 점
   \
    \
     ● 두 번째 점
      \
       \             (점선) 닫는 변 미리보기
        ●───────────────┐
      세 번째 점        │  (점선) 커서까지의 다음 변 미리보기
                        ✕ 커서  →  터미널: "Area: 12.3456  Perimeter: 45.6789"
```

- **꼭짓점** — 클릭(또는 입력)한 각 점이 다각형의 꼭짓점이 됩니다. 확정된 변은 실선으로 그려지고 내부는 반투명 강조 표시로 채워집니다.
- **미리보기 변** — 점선은 마지막 꼭짓점에서 커서까지의 대기 중인 변과, 커서에서 첫 번째 꼭짓점으로 돌아가는 닫는 변을 보여줍니다.
- **닫는 변** — 첫 번째 점을 다시 클릭할 필요가 없습니다. Enter를 누르면 다각형이 자동으로 닫힙니다.

## 면적 측정하기

1. 터미널에 `area`를 입력하거나 (Measure 패널 하단 행의) **Area** 도구 모음 버튼을 클릭합니다.
2. **첫 번째 점을 클릭**하거나, `X,Y`를 입력하고 **Enter**를 눌러 정확한 좌표를 입력합니다.
3. 도형 주위를 따라 **각 추가 꼭짓점을 순서대로 클릭**합니다. 각 단계에서 좌표 입력이 작동합니다.
4. 최소 **3개의 점**이 배치되면, 대기 중인 좌표나 거리 입력 없이 **Enter**를 눌러 다각형을 닫고 결과를 계산합니다.
5. 터미널에 `Area: <값>  Perimeter: <값>`이 출력되고, 닫힌 다각형 — 채우기, 윤곽선, 꼭짓점 그립 — 이 캔버스에 강조 표시된 채로 유지됩니다.
6. **아무 곳이나 클릭하거나, 아무 키나 누르거나, `Escape`를 누르면** 결과가 사라지고 명령어가 종료됩니다.

## 각도 잠금과 정확한 거리

첫 번째 꼭짓점을 배치한 후, 설정된 각도 추적 간격(10°, 15°, 20°, 30°, 45° 또는 90°, 도구 모음 드롭다운으로 설정 가능) 중 하나를 향해 이동하면 다음 변이 해당 방향으로 잠깁니다:

- 변 미리보기가 잠긴 방향에 맞춰지고, 앵커 꼭짓점에 각도 추적 표시기가 그려집니다.
- 길이를 입력하고 **Enter**를 누르면 잠긴 방향을 따라 정확히 그 거리에 다음 꼭짓점을 배치합니다.
- 잠긴 상태에서 (길이를 입력하지 않고) 클릭하면 커서가 잠긴 방향에 투영된 위치에 꼭짓점이 배치됩니다.

| 키 | 동작 |
|----|------|
| `0`–`9`, `.` | 변 길이 값에 추가 |
| `-` | 음수 길이 (첫 번째 문자에만 해당) |
| `Backspace` | 마지막으로 입력한 문자 삭제 |
| `Enter` | 입력한 길이로 다음 꼭짓점 배치 |

## 다각형 닫기

- Enter는 **3개 이상**의 꼭짓점이 배치된 경우에만 도형을 닫습니다 — 그보다 적으면 아무 효과가 없습니다.
- 마지막 꼭짓점에서 첫 번째 꼭짓점으로 돌아가는 변이 자동으로 추가되며 면적과 둘레 모두에 계산됩니다.
- 점은 어떤 순서로도(시계 방향 또는 반시계 방향) 배치할 수 있으며 결과는 동일합니다.

## Area vs Distance vs Angle

| | Area | Distance | Angle |
|--|------|------|------|
| 측정 내용 | 다각형의 면적과 둘레 | 직선 길이 | 꼭짓점에서의 내부 각도 |
| 클릭 수 | 3개 이상, Enter로 닫음 | 2 | 3 |
| 결과 형식 | `12.3456  Perimeter: 45.6789` | `12.3456` (단위) | `45.0000°` |
| 캔버스 미리보기 | 점선 닫는 변이 있는 채워진 다각형 | 첫 번째 점에서 커서까지의 선 | 꼭짓점에서 커서까지의 두 선 |
| 결과 이후 | 아무 입력으로나 사라진 후 명령어 종료 | 클릭으로 새 측정 연결 | 클릭으로 새 측정 연결 |
| 최적 용도 | 닫힌 영역, 방이나 패널의 면적 | 간격 또는 세그먼트의 길이 | 두 형상 사이의 개구 각도 |

## 좌표 입력

클릭 대신 어느 꼭짓점에서든 정확한 위치를 입력합니다:

1. X 값을 입력합니다.
2. `,`를 누릅니다 — 터미널에 `[X], [Y{커서}]`가 표시됩니다.
3. Y 값을 입력합니다.
4. **Enter**를 눌러 확인합니다.

## 키보드 참고

| 키 | 동작 |
|----|------|
| `0`–`9`, `.`, `-` | X 좌표 입력 시작, 또는 각도 잠금이 활성화된 경우 변 길이 입력 |
| `,` | X를 잠그고 Y 입력으로 이동 |
| `Backspace` | 마지막으로 입력한 문자 삭제 |
| `Enter` | 입력한 좌표 또는 길이 확인; 꼭짓점이 3개 이상이고 대기 중인 입력이 없으면 다각형을 닫음 |
| `Escape` | 꼭짓점을 선택하는 동안: 점을 버리고 첫 번째 점부터 다시 시작; 결과가 표시된 후: 결과를 지우고 명령어 종료 |

## 참고

- 면적은 신발끈 공식(shoelace formula)으로 계산되며, 클릭 순서와 관계없이 항상 양수 값으로 보고됩니다.
- 자기 교차 다각형(변이 서로 교차하는 경우)도 여전히 숫자 결과를 생성하지만, 그 값은 시각적으로 둘러싸인 영역과 일치하지 않을 수 있습니다 — 의미 있는 면적을 얻으려면 교차하지 않는 클릭 순서를 유지하세요.
- 결과는 **터미널과 캔버스 상의 임시 강조 표시로만** 표시됩니다 — 도면에 영구적인 내용은 추가되지 않습니다.
- Distance 및 Angle과 달리, Area는 새 측정을 자동으로 **연결하지 않습니다** — 결과를 지운 후 다른 다각형을 측정하려면 `area`를 다시 실행하세요.
- 정밀도는 면적과 둘레 모두 항상 소수점 4자리이며, 도면 좌표와 동일한 단위입니다(단위 변환 없음).
