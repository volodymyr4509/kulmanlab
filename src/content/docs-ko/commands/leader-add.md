# 지시선+

`leader+` 명령어는 기존 다중 지시선에 새 화살표 팔을 추가합니다. 새 팔은 지시선의 기존 꺾임에서 클릭한 새 화살표 끝으로 이어집니다. 스타일 — 꺾임 위치, 텍스트, 화살표 유형, 크기 — 은 선택한 지시선에서 상속됩니다.

## 팔 추가

1. 터미널에 `leader+`를 입력합니다.
2. **기존 다중 지시선을 클릭**하여 선택합니다.
3. **새 화살표 끝을 클릭**하거나, `X,Y`를 입력하고 **Enter**를 눌러 정확한 좌표를 입력합니다. 미리보기 선이 커서에서 지시선의 꺾임까지 표시됩니다.

팔이 배치되고 명령어가 활성 상태로 유지됩니다 — 즉시 다른 지시선을 클릭하여 팔을 더 추가할 수 있습니다. **Enter**, **Space** 또는 **Escape**를 눌러 완료합니다.

```
  이전:                        이후:
  ◄── 팔 1                      ◄── 팔 1
       \                               \
        ●──── 꺾임 ──── 텍스트          ●──── 꺾임 ──── 텍스트
                                       /
                                  팔 2 ──►  (클릭한 새 끝)
```

## 끝에 대한 좌표 입력

클릭 대신 정확한 위치를 입력할 수 있습니다:

1. X 값을 입력합니다.
2. `,`를 누릅니다 — 터미널이 X가 잠겼음을 확인합니다.
3. Y 값을 입력합니다.
4. **Enter**를 눌러 배치합니다.

## 키보드 참고

| 키 | 동작 |
|----|------|
| `0`–`9`, `.`, `-` | X 좌표 입력 시작 |
| `,` | X를 잠그고 Y 입력으로 이동 |
| `Enter` | 입력한 좌표를 확인하고 팔 배치 |
| `Enter` / `Space` | 완료(입력이 진행 중이지 않을 때) |
| `Escape` | 취소하고 초기화 |

## 참고

- **다중 지시선** 객체만 선택할 수 있습니다 — 다른 객체 유형을 클릭하면 아무 일도 일어나지 않습니다.
- 새 팔은 기존 꺾임에서 시작합니다; 화살표 끝이 어디에 갈지만 선택합니다.
- 다중 지시선이 가질 수 있는 팔의 수에 제한이 없습니다.
