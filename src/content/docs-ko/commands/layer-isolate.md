# LayerIsolate

`LayerIsolate` 명령어는 선택한 객체에 속하는 레이어를 **제외한** 모든 레이어를 동결합니다. 아무것도 영구적으로 숨기거나 삭제하지 않고도 특정 기하학에 빠르게 집중할 때 사용합니다 — 완료 후 [LayerUnfreezeAll](../layer-unfreeze-all/)로 동결을 해제하세요.

## 두 가지 시작 방법

**사전 선택 후 격리** — 먼저 객체를 선택한 다음 활성화:

1. 캔버스에서 하나 이상의 객체를 선택합니다.
2. 터미널에 `LayerIsolate`를 입력하거나 **LayerIsolate** 도구 모음 버튼을 클릭합니다.
3. 선택한 객체의 레이어는 표시된 상태로 유지됩니다; 다른 모든 레이어는 즉시 동결됩니다.

**활성화 후 선택**:

1. `LayerIsolate`를 입력하거나 도구 모음 버튼을 클릭합니다.
2. **객체 선택** — 개별 객체를 클릭하거나 드래그하여 영역으로 선택합니다.
3. **Enter** 또는 **Space**를 눌러 확인합니다 — 격리가 적용됩니다.

## 키보드 참고

| 키 | 동작 |
|----|------|
| `Enter` / `Space` | 선택을 확인하고 격리 적용 |
| `Escape` | 취소하고 선택 지우기 |

## 동작 세부 정보

- 선택에 **표시되지 않은** 모든 레이어는 동결로 설정됩니다.
- 선택에 **표시된** 레이어는 이전에 동결되어 있었더라도 동결 해제됩니다.
- 격리가 적용된 후 선택이 지워집니다.
- 적용 후 명령어가 자동으로 완료됩니다.

## 격리 취소

[LayerUnfreezeAll](../layer-unfreeze-all/)을 실행하여 한 번에 모든 레이어를 표시 상태로 복원합니다.
