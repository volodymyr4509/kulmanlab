# 지시선−

`leader-` 명령어는 기존 다중 지시선에서 화살표 팔 하나를 제거합니다. 텍스트 레이블, 꺾임, 나머지 팔은 모두 보존됩니다 — 선택한 팔만 삭제됩니다. 팔이 하나뿐인 다중 지시선에서는 팔을 제거할 수 없습니다.

## 팔 제거

1. 터미널에 `leader-`를 입력합니다.
2. 두 개 이상의 팔이 있는 **다중 지시선을 클릭**합니다. 클릭한 지시선에 팔이 하나만 있으면 터미널에 오류가 표시되고 유효한 선택을 기다립니다.
3. **제거하려는 팔 근처로 커서를 이동** — 가장 가까운 팔이 마커로 강조 표시됩니다.
4. **클릭**하여 해당 팔을 제거합니다.

팔이 제거되고 명령어가 활성 상태로 유지됩니다 — 즉시 다른 지시선(또는 같은 지시선)을 클릭하여 팔을 더 제거할 수 있습니다. **Enter**, **Space** 또는 **Escape**를 눌러 완료합니다.

```
  이전:                   이후:
  ◄── 팔 1                 ◄── 팔 1
       \                         \
        ●──── 꺾임 ──── 텍스트    ●──── 꺾임 ──── 텍스트
       /
  팔 2 ──►  ← 이 팔 제거됨
```

## 가장 가까운 팔이 결정되는 방법

명령어는 커서에서 각 팔의 선 세그먼트(꺾임까지의 마지막 팔 점에서 세그먼트 포함)까지의 수직 거리를 측정합니다. 거리가 가장 작은 팔이 강조 표시되고 클릭 시 제거됩니다.

## 키보드 참고

| 키 | 동작 |
|----|------|
| `Enter` / `Space` | 팔 제거 완료 |
| `Escape` | 취소하고 초기화 |

## 참고

- **팔이 하나만 있는** 지시선은 보호됩니다 — 팔을 먼저 추가해야 제거할 수 있습니다.
- 꺾임 위치와 텍스트 내용은 어떤 팔을 제거하든 항상 보존됩니다.
