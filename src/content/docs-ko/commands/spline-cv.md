# Spline CV

`splinecv` 명령어는 제어 꼭짓점(CV)을 배치하여 **3차 B-스플라인**을 그립니다. 결과 곡선은 각 CV 방향으로 끌어당겨지지만 통과하지는 않습니다 — 첫 번째와 마지막 꼭짓점 제외, **고정 매듭**이 곡선을 정확히 앵커합니다. 이를 통해 직관적인 형상 제어가 가능합니다: 꼭짓점을 끌어당겨 모든 점을 통과시키지 않고도 곡선을 그쪽으로 밀 수 있습니다.

## 제어 꼭짓점으로 스플라인 그리기

1. 터미널에 `splinecv`를 입력하거나 **Spline CV** 도구 모음 버튼을 클릭합니다.
2. **클릭하여 제어 꼭짓점 배치** — 각 클릭이 꼭짓점을 추가합니다. 또는 `X,Y`를 입력하고 **Enter**를 눌러 정확한 좌표를 입력합니다.
3. **Enter**를 눌러 완료합니다(최소 2개 꼭짓점 필요).

```
  CV ●         ● CV
      \       /
       \     /    ← CV 방향으로 끌어당겨진 곡선
        \   /         하지만 통과하지 않음
  CV ●   ●   ● CV (시작/끝: 곡선이 여기서 접함)
```

실시간 미리보기는 각 꼭짓점 후 커서를 이동할 때 업데이트되어, 다음 점이 커서 위치에 있다면 스플라인이 어떻게 보일지 보여줍니다. **Escape**를 누르면 배치된 모든 꼭짓점이 버려지고 종료됩니다.

## 좌표 입력

클릭 대신 임의 제어 꼭짓점에 대한 정확한 위치를 입력합니다:

1. X 값을 입력합니다.
2. `,`를 누릅니다 — 터미널에 `[X], [Y{커서}]`가 표시됩니다.
3. Y 값을 입력합니다.
4. **Enter**를 눌러 꼭짓점을 배치합니다.

## 키보드 참고

| 키 | 동작 |
|----|------|
| `0`–`9`, `.`, `-` | X 좌표 입력 시작 |
| `,` | X를 잠그고 Y 입력으로 이동 |
| `Backspace` | 마지막으로 입력한 문자 삭제 |
| `Enter` | 입력한 좌표 확인 또는 입력 중이지 않고 꼭짓점이 2개 이상이면 스플라인 완료 |
| `Escape` | 모든 꼭짓점을 버리고 종료 |

## 그립 편집 — 제어 꼭짓점으로 재형성

선택한 CV 스플라인은 제어 꼭짓점마다 하나의 그립을 노출합니다:

| 그립 | 위치 | 기능 |
|------|------|------|
| **제어 꼭짓점** | 각 CV 위치 | 드래그하여 해당 CV 이동 — 곡선이 새 위치 방향으로 재형성 |

"전체 스플라인 이동" 그립은 없습니다. 전체 스플라인을 이동하려면 [Move](../move/) 명령어를 사용하세요.

## CV 스플라인 선택

| 방법 | 동작 |
|------|------|
| **클릭** | 클릭이 곡선의 임의 점 근처에 닿으면 선택 |
| **오른쪽으로 드래그** (엄격) | 곡선을 따른 모든 샘플 점이 선택 박스 안에 있어야 함 |
| **왼쪽으로 드래그** (교차) | 선택 박스 경계와 교차하는 곡선의 어느 부분이든 선택 |

## 지원되는 편집 명령어

| 명령어 | 스플라인에 일어나는 일 |
|--------|----------------------|
| [Move](../move/) | 모든 제어 꼭짓점을 동일한 변위만큼 이동 |
| [Copy](../copy/) | 새 위치에 동일한 스플라인 생성 |
| [Rotate](../rotate/) | 선택한 기준점을 중심으로 모든 CV 회전 |
| [Mirror](../mirror/) | 대칭축을 기준으로 모든 CV 반사 |
| [Scale](../scale/) | 기준점에서 모든 CV를 균일하게 축척 |
| [Delete](../delete/) | 스플라인 제거 |

스플라인은 **간격띄우기**, **자르기**, **연장**을 지원하지 않습니다.

## 특성

**일반**

| 특성 | 기본값 | 의미 |
|------|--------|------|
| 색상 | 256 (레이어별) | ACI 색상 인덱스 |
| 레이어 | `0` | 레이어 지정 |
| 선종류 | 레이어별 | 명명된 선종류 패턴 |
| 선종류 축척 | 1 | 선종류 패턴의 축척 인수 |
| 두께 | 0 | 돌출 두께 |

**기하학**

| 특성 | 의미 |
|------|------|
| 차수 | 다항식 차수 — 항상 3(3차) |
| 제어 꼭짓점 | 모든 CV의 좌표 |
| 피트 점 | CV 스플라인의 경우 비어 있음; 피트 점 스플라인에서만 채워짐 |

## 스플라인 CV vs 스플라인 피트 — 무엇을 사용할지

| | 스플라인 CV | 스플라인 피트 |
|--|------------|--------------|
| 곡선이 점을 통과 | 첫 번째와 마지막만(고정) | 클릭한 모든 점 정확히 |
| 형상 제어 | CV를 영역 방향으로 끌어당기기 | 곡선이 반드시 닿아야 하는 피트 점 이동 |
| 그립 편집 효과 | CV 이동 → 곡선 끌어당겨짐 | 피트 점 이동 → 곡선 재보간 |
| 최적 용도 | 부드러운 미적 곡선, 자유 형태 경로 | 특정 좌표를 통과해야 하는 곡선 |

## DXF — SPLINE 객체 (제어 꼭짓점 형식)

CV 스플라인은 DXF 파일에 `SPLINE` 객체로 저장되며, 차수, 매듭 벡터, 모든 제어 꼭짓점 좌표를 저장합니다. 색상, 레이어, 선종류, 선종류 축척, 두께를 포함한 모든 특성은 손실 없이 왕복 저장됩니다. `splineFlag`는 `9`(CV 스플라인)로 설정되어 재로드 시 형식이 보존됩니다. CV 데이터가 있는 `SPLINE` 객체를 지원하는 모든 DXF 응용 프로그램에서 올바르게 읽힙니다.
