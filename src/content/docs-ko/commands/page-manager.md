# PageManager

`PageManager` 명령어는 활성 레이아웃의 페이지 설정 대화 상자를 열어 용지 형식, 방향, 도면 단위와 밀리미터 간의 축척을 변경할 수 있게 합니다.

## 페이지 관리자 열기

- 터미널에 `PageManager`를 입력하거나, **또는**
- 하단의 레이아웃 탭을 오른쪽 클릭하고 **페이지 관리자**를 선택합니다.

## 설정

| 설정 | 설명 |
|------|------|
| **용지 형식** | 표준 시트 크기(A4, A3, A2, Letter 등) |
| **방향** | 세로 또는 가로 |
| **축척** | mm당 도면 단위 — 모델 좌표가 용지에 매핑되는 방식 제어 |

용지 크기 또는 축척을 변경하면 레이아웃의 모든 기존 뷰포트가 위치를 올바르게 유지하도록 비례적으로 크기가 재조정됩니다.

## 참고

- PageManager는 용지 레이아웃에만 적용됩니다. 모델 공간에서는 사용할 수 없습니다.
- 새 레이아웃을 추가하려면 하단 탭 표시줄의 **PageManager** 버튼을 클릭합니다.
- 레이아웃의 이름을 바꾸거나 삭제하려면 해당 탭을 오른쪽 클릭합니다.
