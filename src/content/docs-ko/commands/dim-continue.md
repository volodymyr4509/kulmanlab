# 연속 치수

`dimcontinue` 명령어는 기존 치수의 **두 번째 치수 보조선**에서 새 치수를 연결합니다. 각 새 세그먼트는 기준과 동일한 측정 축과 치수 선 오프셋에 배치됩니다. 화살표 크기, 텍스트 높이, 치수 보조선 길이를 포함한 모든 스타일 특성이 기준에서 자동으로 복사됩니다.

## 연결된 치수의 모양

```
  |←— 3.00 —→|←— 2.50 —→|←— 4.00 —→|
  |           |           |           |
  ●           ●           ●           ●
  p1        p2 (기준    p3           p4
           ext2 → 새 시작)
```

각 직사각형은 별도의 `DIMENSION` 객체입니다. 같은 치수 선 위치와 측정 방향을 공유합니다.

## 체인 시작

1. 터미널에 `dimcontinue`를 입력하거나 **연속 치수** 도구 모음 버튼을 클릭합니다.
2. **방금 치수를 배치한 경우** — 명령어가 자동으로 기준으로 선택합니다(클릭 불필요).
3. **최근 치수가 없는 경우** — 임의 기존 치수를 클릭하여 기준으로 사용합니다.
4. **다음 치수 보조선 시작점을 클릭** — 커서를 이동할 때 미리보기가 새 치수를 보여줍니다. 또는 `X,Y`를 입력하고 **Enter**를 눌러 정확한 좌표를 입력합니다.
5. 계속 클릭(또는 입력)하여 체인을 연장합니다. 각 배치된 치수가 자동으로 새 기준이 됩니다.
6. **Enter**, **Space** 또는 **Escape**를 눌러 체인을 완료합니다.

## 기준 치수에서 상속되는 것

| 특성 | 기준에서 상속 |
|------|-------------|
| 측정 방향 / 각도 | 예 — 전체 체인에 대해 잠금 |
| 치수 선 오프셋(측정 점에서의 거리) | 예 |
| 화살표 크기 | 예 |
| 텍스트 높이 | 예 |
| 치수 보조선 오프셋 및 연장 | 예 |
| 텍스트 정렬 | 예 |
| 스타일 이름 | 예 |
| 색상, 레이어 | 상속되지 않음 — 현재 레이어 사용 |

## 측정 방향 잠금

체인의 측정 방향은 **기준 치수의 각도에 고정됩니다**:

- 선형 기준(H) → 모든 연속 치수가 수평 거리(Δ X)를 측정합니다.
- 선형 기준(V) → 모든 연속 치수가 수직 거리(Δ Y)를 측정합니다.
- 임의 각도의 정렬 기준 → 모든 연속 치수가 같은 각도로 측정합니다.

체인 도중에 방향을 변경할 수 없습니다. 다른 방향의 치수를 추가하려면 새 [선형 치수](./dim-linear) 또는 [정렬 치수](./dim-aligned)를 시작하세요.

## 키보드 참고

| 키 | 동작 |
|----|------|
| `0`–`9`, `.`, `-` | X 좌표 입력 시작 |
| `,` | X를 잠그고 Y 입력으로 이동 |
| `Backspace` | 마지막으로 입력한 문자 삭제 |
| `Enter` | 입력한 좌표 확인 또는 입력이 진행 중이지 않으면 체인 완료 |
| `Space` / `Escape` | 체인 완료 |

## 연속 치수 vs 새로 시작

| | 연속 치수 | 선형 / 정렬 치수 |
|--|----------|----------------|
| 시작점 | 마지막 기준의 ext2에 고정 | 어디서든 클릭 |
| 각도 | 기준에 잠금 | 자유 |
| 오프셋 | 기준에서 상속 | 커서 또는 입력으로 설정 |
| 스타일 | 기준에서 상속 | 현재 스타일 |
| 최적 용도 | 행을 따른 누적 측정 | 첫 번째 치수 또는 방향 변경 |

## DXF — DIMENSION 객체

체인의 각 세그먼트는 DXF 파일에 독립적인 `DIMENSION` 객체로 저장됩니다. 파일에서 연결되지 않습니다 — 같은 기준에서 생성되었으므로 특성을 공유하지만, 배치 후 각각 개별적으로 편집할 수 있습니다.
