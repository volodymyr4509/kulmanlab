# 선

`line` 명령어는 DXF 모델에서 별도의 `LINE` 객체로 저장되는 개별 직선 세그먼트를 그립니다. 각 세그먼트를 배치한 후 명령어는 활성 상태를 유지하며 끝점을 새로운 시작점으로 재사용하므로, 한 번에 한 세그먼트씩 연결된 경로를 만들 수 있습니다. [폴리선](../polyline/)과 달리, 연결된 선들은 독립적인 객체로 유지됩니다 — 각 선은 인접 선에 영향을 주지 않고 자르거나, 연장하거나, 삭제할 수 있습니다.

## 선 그리기

1. 터미널에 `line`을 입력하거나 **선** 도구 모음 버튼을 클릭합니다.
2. **시작점을 클릭**하거나, `X,Y`를 입력하고 **Enter**를 눌러 정확한 좌표를 입력합니다.
3. **끝점을 클릭** — 세그먼트가 배치되고 끝점이 다음 시작점이 됩니다. 좌표 입력도 여기서 작동합니다.
4. 계속 클릭(또는 입력)하여 더 많은 세그먼트를 연결합니다.
5. **Enter** 또는 **Escape**를 눌러 종료합니다.

```
  ●──────────●──────────●──────────●
 시작      2번째 클릭  3번째 클릭  Enter로 완료
           (자동으로 다음 시작점이 됨)
```

단일 세그먼트만 필요한 경우 3단계 직후에 **Enter** 또는 **Escape**를 누르세요.

## 좌표 입력

클릭 대신 시작점 또는 이후 점에 대한 정확한 위치를 입력합니다:

1. X 값을 입력합니다(숫자, `.`, 또는 `-`).
2. `,`를 누릅니다 — 터미널에 `[X], [Y{커서}]`가 표시됩니다.
3. Y 값을 입력합니다.
4. **Enter**를 눌러 점을 배치합니다.

## 각도 잠금 및 정확한 길이 입력

점을 배치한 후 커서를 이동하면, 명령어는 45° 스냅 축(0°, 45°, 90°, 135°, …)을 감지합니다. 다음 조건을 만족하면 각도가 **잠깁니다**:

- 커서가 앵커에서 최소 **5 × 그립 크기** 이상 떨어져 있고, **그리고**
- 가장 가까운 축으로부터의 수직 거리가 **1 그립 크기** 이내일 때.

잠긴 상태에서 미리보기가 축에 스냅되고 정확한 길이를 입력할 수 있습니다:

| 키 | 동작 |
|----|------|
| `0`–`9`, `.` | 길이 값에 숫자 추가 |
| `-` | 음수 길이 — 축을 따라 방향을 반전(첫 번째 문자만) |
| `Backspace` | 마지막으로 입력한 문자 삭제 |
| `Enter` | 입력한 거리에 끝점 배치 |

누적된 값은 터미널에 실시간으로 표시됩니다(예: `끝점을 클릭하거나 길이를 입력하세요: 12.5`). 잠긴 상태에서 클릭하면 클릭이 축에 투영되어 끝점이 항상 축 위에 정확히 위치합니다.

앵커 점에 가깝게 돌아오면 잠금이 해제됩니다.

## 키보드 참고

| 키 | 동작 |
|----|------|
| `0`–`9`, `.`, `-` | X 좌표 입력 시작 또는 각도 잠금 시 거리 |
| `,` | X를 잠그고 Y 입력으로 이동 |
| `Backspace` | 마지막으로 입력한 문자 삭제 |
| `Enter` | 입력한 좌표 또는 길이 확인, 또는 아무것도 입력하지 않은 경우 체인 완료 |
| `Escape` | 체인을 완료하고 종료 |

## 그립 편집 — 끝점 늘리기

선택한 선은 세 개의 그립을 표시합니다:

| 그립 | 위치 | 기능 |
|------|------|------|
| **시작** | 첫 번째 끝점 | 드래그하여 위치 변경 — 끝점은 고정됨 |
| **중간점** | 선의 중심 | 전체 선에 대한 **이동** 활성화 |
| **끝** | 두 번째 끝점 | 드래그하여 위치 변경 — 시작점은 고정됨 |

한쪽 끝점을 늘려도 다른 쪽 끝점에는 영향을 주지 않습니다. 이것은 꼭짓점을 이동하면 전체 경로의 모양이 바뀌는 [폴리선](../polyline/) 그립 편집과 다릅니다.

## 선 선택

| 방법 | 동작 |
|------|------|
| **클릭** | 클릭이 세그먼트의 히트 테스트 거리 내에 있으면 선 선택 |
| **오른쪽으로 드래그** (엄격) | 두 끝점이 모두 박스 안에 있을 때만 선 선택 |
| **왼쪽으로 드래그** (교차) | 세그먼트의 어느 부분이든 박스 경계와 교차하면 선 선택 |

## 지원되는 편집 명령어

선은 [자르기](../trim/)와 [연장](../extend/)이 작동하는 **유일한** 객체입니다. 모든 표준 변환 명령어도 적용됩니다:

| 명령어 | 선에 일어나는 일 |
|--------|----------------|
| [이동](../move/) | 두 끝점을 동일한 변위만큼 이동 |
| [복사](../copy/) | 새 위치에 동일한 선 생성 |
| [회전](../rotate/) | 선택한 기준점을 중심으로 두 끝점 회전 |
| [대칭](../mirror/) | 대칭축을 기준으로 두 끝점 반사 |
| [축척](../scale/) | 기준점에서 두 끝점을 균일하게 축척 |
| [간격띄우기](../offset/) | 고정된 수직 거리만큼 평행한 선 생성 |
| [자르기](../trim/) | 교차점에서 선 자르기 — **선만 해당** |
| [연장](../extend/) | 가장 가까운 끝점을 경계까지 늘리기 — **선만 해당** |
| [삭제](../delete/) | 도면에서 선 제거 |

## 특성

선을 선택하면 특성 패널에 DXF `LINE` 레코드가 가진 모든 필드가 표시됩니다:

**일반**

| 특성 | 기본값 | 의미 |
|------|--------|------|
| 색상 | 256 (레이어별) | ACI 색상 인덱스 |
| 레이어 | `0` | 레이어 지정 |
| 선종류 | 레이어별 | 명명된 선종류 패턴 |
| 선종류 축척 | 1 | 선종류 패턴의 축척 인수 |
| 두께 | 0 | 돌출 두께 |

**기하학**

| 특성 | 의미 |
|------|------|
| 시작 X / 시작 Y | 첫 번째 끝점 좌표 |
| 끝 X / 끝 Y | 두 번째 끝점 좌표 |

모든 필드는 명령어를 다시 실행하지 않고도 패널에서 직접 편집할 수 있습니다.

## 선 vs 폴리선 — 언제 무엇을 사용할지

| | 선 | 폴리선 |
|--|------|--------|
| 객체 수 | 세그먼트당 하나의 `LINE` | 전체 경로에 하나의 `LWPOLYLINE` |
| 자르기 / 연장 | 가능 — 세그먼트별 | 불가능 |
| 닫힌 도형 | 불가능 | 가능 (닫기 플래그) |
| 그립 편집 | 개별 끝점 늘리기 | 경로를 따라 꼭짓점 이동 |
| 최적 용도 | 구성 선, 단일 세그먼트, 자를 기하학 | 윤곽선, 전체를 유지할 도형 |

## DXF — LINE 객체

선은 DXF 파일에 `LINE` 객체로 저장됩니다. 모든 특성 — 시작/끝 좌표, 색상, 레이어, 선종류, 선종류 축척, 두께 — 은 손실 없이 왕복 저장됩니다. `LINE` 객체를 포함하는 DXF를 열면 편집기에서 완전히 편집 가능한 `Line` 객체가 됩니다.

편집기에서 그린 선도 저장 시 `LINE` 객체로 기록되므로, LibreCAD, FreeCAD 및 기타 DXF 호환 응용 프로그램에서 읽을 수 있습니다.
