# Align

`align` 명령어는 하나 또는 두 개의 원본/대상 점 쌍을 사용해 선택한 객체를 재배치합니다. 한 쌍이면 [Move](../move/)와 완전히 동일하게 동작합니다(이동만). 두 쌍이면 원본에서 원본으로의 방향이 대상에서 대상으로의 방향과 일치하도록 선택을 회전시키고, 선택적으로 원본 선분 길이가 대상 선분 길이와 일치하도록 크기를 조정할 수도 있습니다 — 이동, 회전, 크기 조정을 하나의 작업으로 수행합니다.

## 두 가지 시작 방법

**사전 선택 후 정렬** — 먼저 객체를 선택한 다음 활성화:

1. 캔버스에서 하나 이상의 객체를 선택합니다.
2. 터미널에 `align`을 입력하거나 **Align** 도구 모음 버튼을 클릭합니다.
3. **첫 번째 원본 점(S1)을 클릭**한 다음 **첫 번째 대상 점(D1)을 클릭**합니다.
4. **두 번째 원본 점(S2)을 클릭**하거나, 지금 바로 이동만 적용하려면 **Enter**를 누릅니다.
5. **두 번째 대상 점(D2)을 클릭**합니다.
6. 크기 조정 프롬프트에 응답합니다: 크기를 조정하려면 **Y**를, 원래 크기를 유지하려면 **N** / **Enter**를 누릅니다.

**활성화 후 선택** — 아무것도 선택하지 않고 명령어 시작:

1. `align`을 입력하거나 도구 모음 버튼을 클릭합니다.
2. **객체 선택** — 클릭하여 개별 객체를 토글하거나, 드래그하여 영역으로 선택합니다.
3. **Enter** 또는 **Space**를 눌러 선택을 확인합니다.
4. 위와 같이 S1 → D1 → S2 → D2 → 크기 조정 프롬프트 순서로 계속합니다.

> 터미널은 모호하지 않을 만큼의 글자만 있으면 됩니다 — 다른 어떤 명령어도 이 두 글자로 시작하지 않으므로 `al`을 입력하고 **Enter**를 누르면 Align이 바로 실행됩니다.

## 정렬의 구조

```
  원본 점 (객체 위):                   대상 점:
  ● S1                                 ● D1
   \                                    \
    ● S2                                 ● D2

  결과: 선택 항목은 S1이 D1에 도달하도록 이동한 다음, S1→S2
  방향이 D1→D2 방향과 일치하도록 D1을 중심으로 회전됩니다 —
  크기 조정을 선택하면 |S1S2|가 |D1D2|가 되도록 크기가
  조정됩니다.
```

각 단계에서 고스트 미리보기가 커서를 따라갑니다: D1을 배치하는 동안 이동 미리보기, D2가 배치되는 동안 점선 회전 미리보기가 표시됩니다.

## 한 점 정렬 (이동만)

D1을 배치한 후, 두 번째 원본 점을 클릭하는 대신 **Enter**를 누릅니다. 선택 항목은 S1→D1 벡터만큼 이동합니다 — 회전이나 크기 조정 없이 — S1을 기준점으로, D1을 대상으로 사용하는 [Move](../move/)와 동일합니다.

## 두 점 정렬 (이동 + 회전 + 선택적 크기 조정)

S2와 D2가 모두 배치되면:

- **회전 각도** — 대상 방향(`D1 → D2`)과 원본 방향(`S1 → S2`) 사이의 차이입니다.
- **크기 조정 프롬프트** — `scale objects to alignment points? [Yes/No] <N>`가 나타나며, 기본값은 **No**입니다:
  - `S1–S2` 거리가 `D1–D2` 거리가 되도록 D1을 중심으로 선택 항목도 균일하게 크기 조정하려면 **Y**를 누릅니다.
  - 원래 크기를 유지하려면 **N** 또는 **Enter**를 누릅니다 — 이동과 회전만 적용됩니다.

크기 조정 프롬프트에서 아무 키나 누르면 정렬이 즉시 적용됩니다 — Yes 또는 No를 선택한 후 별도의 확인 단계는 없습니다.

## 키보드 참고

| 키 | 동작 |
|----|------|
| `Enter` / `Space` | 선택을 확인하고 S1 단계로 진행 |
| `Enter` (S2 단계에서) | 회전 건너뛰기 — S1과 D1을 사용해 이동만 하는 정렬 적용 |
| `Y` | 크기 조정과 함께 정렬 적용 |
| `N` / `Enter` (크기 조정 프롬프트에서) | 크기 조정 없이 정렬 적용 |
| `Escape` | 점을 선택하는 동안: 배치한 점을 버리고 선택 단계로 돌아감; 아무것도 선택되지 않은 경우: 명령어 취소 |

## 명령어 중 선택

명령어가 선택 단계에서 시작할 때:

| 방법 | 동작 |
|------|------|
| **클릭** | 커서 아래 객체를 선택에서 토글 |
| **오른쪽으로 드래그** (엄격) | 박스 안에 완전히 있는 객체 추가 |
| **왼쪽으로 드래그** (교차) | 박스 경계와 교차하는 객체 추가 |
| **Enter** / **Space** | 선택을 확인하고 S1 단계로 이동 |

## 정렬 후

정렬된 객체는 새 위치에서 선택된 상태로 유지되며, 명령어는 자동으로 종료됩니다 — **Align**을 다시 실행하거나, 다시 선택하지 않고 [Move](../move/), [Rotate](../rotate/), [Scale](../scale/)로 전환할 수 있습니다.

## Align vs Move

| | Align | Move |
|--|-------|------|
| 점 쌍 | 1개(이동만) 또는 2개(이동 + 회전 + 크기 조정) | 1개(이동만) |
| 회전 | 두 번째 점 쌍이 있으면 가능 | 불가 |
| 크기 조정 | 두 번째 점 쌍이 있으면 선택적으로 가능 | 불가 |
| 최적 용도 | 기준점을 사용해 한 도형을 다른 도형에 맞추기 | 간단한 재배치 |

## 지원되는 객체

Align은 Move, Rotate, Scale이 지원하는 모든 객체 유형에서 작동합니다 — 이러한 명령어가 사용하는 것과 동일한 `translate`, `rotate`, `scale` 작업이 순서대로 적용되므로 제외되는 것이 없습니다.
