# Mirror

`mirror` 명령어는 두 점 축을 기준으로 반사된 선택한 객체의 미러 복사본을 만듭니다. 원본은 **항상 유지됩니다** — [Move](../move/)이나 [Rotate](../rotate/)과 달리, 대칭은 기존 객체를 수정하지 않고 새 객체만 추가합니다.

## 두 가지 시작 방법

**사전 선택 후 대칭** — 먼저 객체를 선택한 다음 활성화:

1. 캔버스에서 하나 이상의 객체를 선택합니다.
2. 터미널에 `mirror`를 입력하거나 **Mirror** 도구 모음 버튼을 클릭합니다.
3. **대칭축의 첫 번째 점을 클릭**하거나, `X,Y`를 입력하고 **Enter**를 눌러 정확한 좌표를 입력합니다.
4. **두 번째 점을 클릭** — 미러 복사본이 배치되고 명령어가 종료됩니다. 여기서도 좌표 입력이 작동합니다.

**활성화 후 선택** — 아무것도 선택하지 않고 명령어 시작:

1. `mirror`를 입력하거나 도구 모음 버튼을 클릭합니다.
2. **객체 선택** — 클릭하여 토글하거나, 드래그하여 영역으로 선택합니다.
3. **Enter** 또는 **Space**를 눌러 선택을 확인합니다.
4. **첫 번째 점을 클릭**한 다음 **두 번째 점을 클릭**합니다(두 단계에서 좌표 입력 가능).

```
  원본:          대칭축:        결과:
                 |
  [객체 A]  →   |    →    [객체 A] + [미러된 A]
                 |
```

두 번째 축 점을 위치시키는 동안 미러 복사본의 실시간 미리보기가 커서를 따라 이동합니다.

## 대칭축

축은 두 클릭한 점을 통과하는 무한 선입니다. 임의 각도로 설정할 수 있습니다:

- 커서를 **45° 스냅 축**(0°, 45°, 90°, 135°, …) 근처로 이동하면 그 각도로 잠깁니다 — 깔끔한 수평, 수직 또는 대각 반사에 유용합니다.
- 스냅 영역에서 멀리 클릭하면 자유 각도 축이 됩니다.

## 좌표 입력

축 점 단계에서 클릭 대신 정확한 위치를 입력합니다:

1. X 값을 입력합니다.
2. `,`를 누릅니다 — 터미널에 `[X], [Y{커서}]`가 표시됩니다.
3. Y 값을 입력합니다.
4. **Enter**를 눌러 확인합니다.

## 키보드 참고

| 키 | 동작 |
|----|------|
| `Enter` / `Space` | 선택 확인 |
| `0`–`9`, `.`, `-` | X 좌표 입력 시작 |
| `,` | X를 잠그고 Y 입력으로 이동 |
| `Backspace` | 마지막으로 입력한 문자 삭제 |
| `Enter` | 입력한 좌표 확인 |
| `Escape` | 취소하고 초기화 |

## 명령어 중 선택

| 방법 | 동작 |
|------|------|
| **클릭** | 커서 아래 객체 토글 |
| **오른쪽으로 드래그** (엄격) | 박스 안에 완전히 있는 객체 추가 |
| **왼쪽으로 드래그** (교차) | 박스와 교차하는 객체 추가 |
| **Enter** / **Space** | 선택 확인 |

## 미러되는 내용

모든 객체 유형을 지원합니다. 기하학이 수학적으로 축을 기준으로 반사됩니다:

| 객체 | 변경 사항 |
|------|----------|
| 선 | 두 끝점 반사 |
| 원 | 중심 반사; 반지름 변경 없음 |
| 호 | 중심 반사; 시작 및 끝 각도가 축을 기준으로 재계산 |
| 타원 | 중심 반사; 장축 방향이 축을 기준으로 반전 |
| 폴리선 / 직사각형 | 모든 꼭짓점 반사 |
| 텍스트 | 앵커 점 반사; 텍스트 문자열은 **반전되지 않음** |
| 스플라인 | 모든 제어 꼭짓점 / 피트 점 반사 |

## Mirror vs Copy

| | Mirror | Copy |
|--|------|------|
| 원본 | 항상 유지 | 항상 유지 |
| 새 객체 위치 | 축을 기준으로 반사 | 변위 벡터만큼 오프셋 |
| 최적 용도 | 대칭 디자인, 양측 형상 | 임의 방향으로 기하학 반복 |
