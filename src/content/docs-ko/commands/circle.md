# 원

`circle` 명령어는 중심점과 반지름으로 원을 그립니다. 중심점을 클릭한 후 두 번째 점을 클릭하거나 정확한 숫자를 입력하여 반지름을 설정할 수 있습니다 — 두 옵션은 동시에 활성화됩니다.

## 원 그리기

1. 터미널에 `circle`을 입력하거나 **원** 도구 모음 버튼을 클릭합니다.
2. **중심점을 클릭**하거나, `X,Y`를 입력하고 **Enter**를 눌러 정확한 좌표를 입력합니다.
3. 반지름을 설정합니다 — 다음 중 하나를 선택:
   - 캔버스의 **임의 점을 클릭** — 중심에서의 거리가 반지름이 됩니다, 또는
   - **반지름을 입력**하고 **Enter**를 눌러 정확한 값을 입력합니다.

원이 즉시 배치되고 명령어가 종료됩니다.

```
  중심 ●
        \  반지름 선 미리보기
         \
          ● ← 여기를 클릭하거나 숫자 입력
```

반지름 단계에서 실시간 미리보기는 현재 커서 거리에서 원을 표시하고 중심에서 현재 점까지 반지름 선도 그립니다.

## 중심점 좌표 입력

클릭 대신 중심 위치를 입력할 수 있습니다:

1. X 값을 입력합니다.
2. `,`를 누릅니다 — 터미널에 `[X], [Y{커서}]`가 표시됩니다.
3. Y 값을 입력합니다.
4. **Enter**를 눌러 중심을 배치하고 반지름 입력으로 진행합니다.

## 반지름 직접 입력

중심이 배치된 후, 즉시 입력하면 반지름 값이 축적됩니다:

| 키 | 동작 |
|----|------|
| `0`–`9`, `.` | 반지름 값에 숫자 추가 |
| `Backspace` | 마지막으로 입력한 문자 삭제 |
| `Enter` | 입력한 반지름으로 원 배치 |

누적된 값은 터미널 프롬프트에 표시됩니다(예: `원의 반지름을 입력하세요: 25`). 커서가 반지름 선 마커의 방향을 제어하는 동안 미리보기는 입력한 반지름으로 업데이트됩니다.

## 키보드 참고

| 키 | 동작 |
|----|------|
| `0`–`9`, `.`, `-` | X 좌표 입력 시작(중심 단계) 또는 반지름 숫자(반지름 단계) |
| `,` | X를 잠그고 Y 입력으로 이동(중심 단계) |
| `Backspace` | 마지막으로 입력한 문자 삭제 |
| `Enter` | 입력한 좌표 또는 반지름 확인 |
| `Escape` | 취소하고 초기화 |

## 그립 편집 — 반지름 조정

선택한 원은 다섯 개의 그립을 노출합니다:

| 그립 | 위치 | 기능 |
|------|------|------|
| **중심** | 중심점 | 전체 원 이동; 반지름 변경 없음 |
| **왼쪽** | 가장 왼쪽 점(중심 − 반지름) | 드래그하여 새 반지름 = 중심까지의 거리 |
| **오른쪽** | 가장 오른쪽 점(중심 + 반지름) | 드래그하여 새 반지름 = 중심까지의 거리 |
| **위** | 가장 위쪽 점 | 드래그하여 새 반지름 = 중심까지의 거리 |
| **아래** | 가장 아래쪽 점 | 드래그하여 새 반지름 = 중심까지의 거리 |

네 개의 주요 그립은 모두 동일하게 작동합니다 — 새 반지름은 중심에서 드래그 위치까지의 거리와 같습니다. 중심은 고정됩니다.

## 원 선택

| 방법 | 동작 |
|------|------|
| **클릭** | 클릭이 원주 근처에 닿으면 선택 |
| **오른쪽으로 드래그** (엄격) | 전체 경계 사각형(중심 ± 반지름)이 박스 안에 있어야 함 |
| **왼쪽으로 드래그** (교차) | 원주의 어느 부분이든 박스 경계와 교차하거나 닿으면 원 선택 |

## 특성

**일반**

| 특성 | 기본값 | 의미 |
|------|--------|------|
| 색상 | 256 (레이어별) | ACI 색상 인덱스 |
| 레이어 | `0` | 레이어 지정 |
| 선종류 | 레이어별 | 명명된 선종류 패턴 |
| 선종류 축척 | 1 | 선종류 패턴의 축척 인수 |
| 두께 | 0 | 돌출 두께 |

**기하학**

| 특성 | 의미 |
|------|------|
| 중심 X / 중심 Y | 중심점의 좌표 |
| 반지름 | 도면 단위로 표시된 원 반지름 |

## 원 vs 호 — 언제 무엇을 사용할지

| | 원 | 호 |
|--|------|-----|
| 범위 | 전체 360° | 부분 — 시작 및 끝 각도로 정의 |
| 그리기 방법 | 중심 + 반지름 | 곡선의 세 점 |
| 입력 방식 | 반지름 값 | 없음 — 클릭만 가능 |
| 크기 조정 그립 | 4개의 주요 점 | 시작 및 끝 점(각도 + 반지름) |
| 치수 | 반지름: [반지름 치수](../dim-radius/) · 지름: [지름 치수](../dim-diameter/) | [반지름 치수](../dim-radius/) |
| 최적 용도 | 완전한 구멍, 볼트 원, 둥근 형상 | 모깎기, 부분 곡선, 아치형 경로 |

## DXF — CIRCLE 객체

원은 DXF 파일에 `CIRCLE` 객체로 저장됩니다. 중심 좌표, 반지름, 색상, 레이어, 선종류, 선종류 축척, 두께 모두 손실 없이 왕복 저장됩니다. DXF 호환 응용 프로그램은 이것을 표준 원으로 읽습니다.
