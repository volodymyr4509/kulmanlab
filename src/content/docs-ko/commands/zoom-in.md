# Zoom In

`zoomin` 명령어는 현재 확대/축소 수준을 **1.5×**로 곱하고 뷰포트 중간점을 중심으로 즉시 종료됩니다. 커서가 아닌 화면 중심으로 확대/축소되는 스크롤 휠 틱 하나의 도구 모음 버전입니다.

## 확대

**Zoom In** 도구 모음 버튼을 클릭하거나 터미널에 `zoomin`을 입력합니다. 확대/축소가 즉시 적용되고 명령어가 종료됩니다 — 캔버스 클릭이 필요 없습니다.

## 1.5× 단계 작동 방식

| 현재 확대/축소 | 확대 한 번 후 |
|-------------|-------------|
| 1.00× | 1.50× |
| 1.50× | 2.25× |
| 10.00× | 15.00× |
| 6,667× | 10,000× (제한됨) |

확대/축소 수준은 항상 `확대/축소` 레이블 옆 캔버스의 **오른쪽 하단 모서리**에 표시됩니다. 상한은 **10,000×**이며; 이후 단계는 아무 효과가 없습니다.

## 도구 모음 확대 vs 스크롤 휠

| | 확대 버튼 | 스크롤 휠 |
|--|----------|----------|
| 확대/축소 중심 | 뷰포트 중간점 | 커서 위치 |
| 단계 크기 | 클릭당 1.5× | 틱당 ~1.1× |
| 활성화 필요 | 아니오 | 아니오 — 항상 작동 |
| 최적 용도 | 대략적인 탐색 | 정확한 커서 대상 확대/축소 |

## 키보드 참고

이 명령어에는 키보드 단축키가 없습니다. 대신 스크롤 휠을 사용하세요 — 명령어를 활성화하지 않고도 언제든지 작동합니다.
