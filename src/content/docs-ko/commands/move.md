# 이동

`move` 명령어는 기준점에서 대상점까지 선택한 객체를 이동합니다. 모든 선택한 객체에 적용되는 변위는 기준에서 대상까지의 벡터입니다. 이동 후 모든 객체는 새 위치에서 선택된 상태를 유지하여 추가 편집에 사용할 수 있습니다.

## 두 가지 시작 방법

**사전 선택 후 이동** — 먼저 객체를 선택한 다음 활성화:

1. 캔버스에서 하나 이상의 객체를 선택합니다.
2. 터미널에 `move`를 입력하거나 **이동** 도구 모음 버튼을 클릭합니다.
3. **기준점을 클릭**하거나, `X,Y`를 입력하고 **Enter**를 눌러 정확한 좌표를 입력합니다.
4. **대상을 클릭** — 모든 선택한 객체가 기준→대상 벡터만큼 이동합니다. 여기서도 좌표 입력이 작동합니다.

**활성화 후 선택** — 아무것도 선택하지 않고 명령어 시작:

1. `move`를 입력하거나 도구 모음 버튼을 클릭합니다.
2. **객체 선택** — 클릭하여 개별 객체를 토글하거나, 드래그하여 영역으로 선택합니다.
3. **Enter** 또는 **Space**를 눌러 선택을 확인합니다.
4. **기준점을 클릭**한 다음 **대상을 클릭**합니다(두 단계에서 좌표 입력 가능).

```
  이전:                    이후:
  ● 기준                       → ● 대상
  [객체 A]                      [이동된 객체 A]
  [객체 B]                      [이동된 객체 B]
```

모든 선택한 객체의 고스트 미리보기가 기준점에서 대상으로 커서를 따라 결과를 클릭 전에 보여줍니다.

## 좌표 입력

기준점 또는 대상 단계에서 클릭 대신 정확한 위치를 입력합니다:

1. X 값을 입력합니다.
2. `,`를 누릅니다 — 터미널에 `[X], [Y{커서}]`가 표시됩니다.
3. Y 값을 입력합니다.
4. **Enter**를 눌러 확인합니다.

## 각도 잠금 및 정확한 거리

기준점이 설정된 후 명령어는 45° 스냅 축(0°, 45°, 90°, 135°, …)을 감시합니다. 커서가 기준에서 충분히 멀리 있고 축에서 한 그립 폭 이내이면 방향이 **잠깁니다**. 잠긴 상태에서:

- 고스트 미리보기가 축에 스냅됩니다.
- 거리를 입력하고 **Enter**를 누르면 잠긴 방향을 따라 정확히 그만큼 이동합니다.
- 클릭하면 축에 투영되어 대상이 항상 정확히 그 위에 있습니다.

| 키 | 동작 |
|----|------|
| `0`–`9`, `.` | 거리 값에 숫자 추가 |
| `-` | 음수 거리 — 축을 따라 방향 반전(첫 번째 문자만) |
| `Backspace` | 마지막으로 입력한 문자 삭제 |
| `Enter` | 입력한 거리로 이동 적용 |

## 키보드 참고

| 키 | 동작 |
|----|------|
| `Enter` / `Space` | 선택 확인 및 기준점 단계로 진행 |
| `0`–`9`, `.`, `-` | X 좌표 입력 시작 또는 각도 잠금 시 거리 |
| `,` | X를 잠그고 Y 입력으로 이동 |
| `Backspace` | 마지막으로 입력한 문자 삭제 |
| `Enter` | 좌표 확인 또는 입력한 거리로 이동 적용 |
| `Escape` | 취소하고 초기화 |

## 그립에서 이동 활성화

선택한 [선](../line/)의 **중간점 그립**을 클릭하면 중간점을 기준점으로 이미 설정하고 이동 단계가 활성화된 상태로 이동이 자동으로 시작됩니다. 이것은 선택 단계를 거치지 않고 단일 선을 재배치하는 가장 빠른 방법입니다.

## 명령어 중 선택

명령어가 선택 단계에서 시작할 때:

| 방법 | 동작 |
|------|------|
| **클릭** | 커서 아래 객체를 선택에서 토글 |
| **오른쪽으로 드래그** (엄격) | 박스 안에 완전히 있는 객체 추가 |
| **왼쪽으로 드래그** (교차) | 박스 경계와 교차하는 객체 추가 |
| **Enter** / **Space** | 선택을 확인하고 기준점 단계로 이동 |

## 이동 후

이동된 객체는 새 위치에서 선택된 상태로 유지됩니다. 즉, 즉시:
- 더 조금 이동하기 위해 다시 **이동**을 실행할 수 있습니다.
- 다시 선택하지 않고 [복사](../copy/), [회전](../rotate/) 또는 [축척](../scale/)을 실행할 수 있습니다.
- **Delete**를 눌러 제거할 수 있습니다.

## 이동 vs 복사

| | 이동 | 복사 |
|--|------|------|
| 원래 위치 | 비워짐 — 객체가 더 이상 거기에 없음 | 유지됨 — 원본이 제자리에 있음 |
| 결과 수 | 동일한 수의 객체 | 작업당 추가 세트 하나 |
| 이후 선택 | 새 위치에서 이동된 객체 선택 | 새 위치에서 복사된 객체 선택 |
| 최적 용도 | 기하학 재배치 | 기하학 복제 |

## 지원되는 객체

이동은 모든 객체 유형에서 작동합니다: 선, 폴리선, 직사각형, 원, 호, 타원, 텍스트, 스플라인, 치수, 지시선 및 기타 모든 것. 모든 객체는 내부적으로 `translate(dx, dy)`를 구현하므로 제외되는 것이 없습니다.
