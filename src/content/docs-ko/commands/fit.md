# 전체 보기

`fit` 명령어는 도면의 모든 객체의 경계 박스를 계산하고 작은 여백을 두어 모든 객체가 표시되도록 확대/축소 수준과 이동 위치를 모두 조정합니다. DXF 파일을 가져온 후 잃어버린 뷰를 복구하거나 방향을 잡는 가장 빠른 방법입니다.

## 뷰 맞추기

**전체 보기** 도구 모음 버튼을 클릭하거나 터미널에 `fit`을 입력합니다. 뷰가 즉시 조정되고 명령어가 종료됩니다 — 상호 작용이 필요 없습니다.

**가운데 마우스 버튼을 더블 클릭**하면 명령어를 활성화하지 않고도 언제든지 같은 전체 보기 작업이 실행됩니다 — 드로잉 중 잃어버린 뷰를 초기화하는 가장 빠른 단축키입니다.

## 경계 박스 맞추기 작동 방식

1. 전체 보기는 모든 객체를 포함하는 축 정렬 경계 박스를 찾습니다(최소 X, 최대 X, 최소 Y, 최대 Y).
2. 더 높거나 넓은 치수가 여백을 두고 캔버스를 채우도록 확대/축소 수준이 설정됩니다.
3. 뷰가 경계 박스의 중간점을 중심으로 합니다.

| 도면 상태 | 결과 |
|----------|------|
| 높이보다 넓은 경우 | 너비에 의해 확대/축소 제한 |
| 넓이보다 높은 경우 | 높이에 의해 확대/축소 제한 |
| 단일 객체 | 해당 객체 주변에 맞춤 |
| 빈 도면 | 뷰가 변경되지 않음 |

## 전체 보기 vs 수동 확대/축소 컨트롤

| | 전체 보기 | 확대 / 축소 | 스크롤 휠 |
|--|---------|---------|----------|
| 중심 | 모든 객체 | 뷰포트 중간점 | 커서 |
| 단계 크기 | 자동(한 번에) | 단계당 1.5× | 틱당 ~1.1× |
| 최적 용도 | 잃어버린 뷰 복구, 가져오기 후 방향 설정 | 중심에서 단계별 확대/축소 | 정확한 커서 대상 확대/축소 |

## 키보드 참고

이 명령어에는 키보드 단축키가 없습니다. 대신 **가운데 마우스 버튼 더블 클릭** 단축키를 사용하세요.
