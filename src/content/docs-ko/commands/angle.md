# Angle

`angle` 명령어는 세 클릭한 점을 통한 두 선분에 의해 형성된 꼭짓점에서의 내부 각도를 측정합니다. 결과 — 항상 0°에서 180° 사이 — 는 소수점 4자리까지 터미널에 표시됩니다. 두 가지 측정 명령어 중 하나입니다 — [Distance](../distance/)는 대신 직선 길이를 측정합니다.

## 각도 측정의 구조

```
  ● 첫 번째 점(첫 번째 광선의 끝)
   \
    \  첫 번째 광선 미리보기
     \
      ● 꼭짓점(3단계)
     /
    /  두 번째 광선 미리보기(커서까지)
   /
  ● 세 번째 점  →  터미널: "각도: 45.0000°"
```

- **첫 번째 점** — 각도의 한쪽 끝(2단계).
- **꼭짓점** — 각도가 측정되는 모서리(3단계).
- **세 번째 점** — 각도의 다른 쪽 끝(4단계).

## 각도 측정

1. 터미널에 `angle`을 입력하거나 **Angle** 도구 모음 버튼을 클릭합니다.
2. **첫 번째 점을 클릭** — 각도의 한쪽 팔 끝. 또는 `X,Y`를 입력하고 **Enter**를 눌러 정확한 좌표를 입력합니다.
3. **꼭짓점을 클릭** — 두 팔이 만나는 모서리. 여기서도 좌표 입력이 작동합니다.
4. **세 번째 점을 클릭** — 두 번째 팔 끝. 여기서도 좌표 입력이 작동합니다. 이 점을 배치하면 결과가 출력됩니다.
5. 선택적으로 **다시 클릭**하여 해당 클릭을 새 첫 번째 점으로 새 측정을 시작합니다.

## 내부 각도 규칙

명령어는 꼭짓점에서 두 광선의 내적을 사용하여 각도를 계산합니다:

- **항상 내부**: 결과는 더 작은 각도이며 0°에서 180° 사이입니다.
- 끝점을 클릭하는 순서는 결과에 영향을 주지 않습니다 — 꼭짓점 위치만 중요합니다.
- 동일선상의 점(세 점이 모두 한 선 위)은 0° 또는 180°를 반환합니다.

## 측정 연결

결과가 나타난 후 클릭하면 즉시 다음 측정이 시작됩니다 — 클릭한 점이 새 첫 번째 점이 됩니다. 명령어는 `Escape`를 누를 때까지 자동으로 종료되지 않습니다.

## Angle vs Distance

| | Angle | Distance |
|--|------|------|
| 측정 내용 | 꼭짓점에서의 내부 각도 | 직선 길이 |
| 클릭 수 | 3 | 2 |
| 결과 형식 | `45.0000°` | `12.3456` (단위) |
| 캔버스 미리보기 | 꼭짓점에서 두 끝까지의 두 선 | 첫 번째 점에서 커서까지의 선 |
| 최적 용도 | 두 형상 사이의 개구 각도 | 간격 또는 세그먼트의 길이 |

## 좌표 입력

세 점 중 어느 점에서든 클릭 대신 정확한 위치를 입력합니다:

1. X 값을 입력합니다.
2. `,`를 누릅니다 — 터미널에 `[X], [Y{커서}]`가 표시됩니다.
3. Y 값을 입력합니다.
4. **Enter**를 눌러 확인합니다.

## 키보드 참고

| 키 | 동작 |
|----|------|
| `0`–`9`, `.`, `-` | X 좌표 입력 시작 |
| `,` | X를 잠그고 Y 입력으로 이동 |
| `Backspace` | 마지막으로 입력한 문자 삭제 |
| `Enter` | 입력한 좌표 확인 |
| `Escape` | 취소하고 2단계로 초기화 |

## 참고

- 결과는 **터미널에만** 표시됩니다 — 도면에는 아무것도 추가되지 않습니다.
- 정밀도는 항상 도 단위로 소수점 4자리입니다.
