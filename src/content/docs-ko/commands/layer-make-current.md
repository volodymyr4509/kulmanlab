# LayerMakeCurrent

`LayerMakeCurrent` 명령어는 **현재 도면 레이어**를 클릭한 객체가 속한 레이어에 맞게 설정합니다. 새 객체는 그 후 자동으로 해당 레이어에 그려집니다.

## 사용 방법

1. 터미널에 `LayerMakeCurrent`를 입력하거나 **현재로 설정** 도구 모음 버튼(스포이드 아이콘)을 클릭합니다.
2. 캔버스에서 **임의 객체를 클릭**합니다.
3. 현재 레이어가 해당 객체의 레이어와 일치하도록 업데이트됩니다. 명령어가 즉시 완료됩니다.

## 동작 세부 정보

- 빈 캔버스를 클릭하면(객체 히트 없음), 터미널에 `객체를 찾을 수 없습니다`가 표시되고 다시 시도할 수 있도록 명령어가 활성 상태로 유지됩니다.
- 현재 레이어 설정만 변경됩니다 — 객체는 수정되지 않습니다.
- 업데이트된 레이어는 도구 모음의 레이어 선택기에 반영됩니다.
