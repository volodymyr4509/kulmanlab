# Arc

`arc` 명령어는 클릭한 세 점을 통과하는 원호를 그립니다. 호는 세 점 모두를 통과하는 고유한 외접원으로 계산됩니다 — 중심이나 반지름을 직접 지정할 필요가 없습니다. 호는 첫 번째 클릭에서 세 번째 클릭까지 두 번째를 통과하며 이어집니다.

## 호 그리기

1. 터미널에 `arc`를 입력하거나 **Arc** 도구 모음 버튼을 클릭합니다.
2. **첫 번째 점을 클릭** — 호의 한쪽 끝. 또는 `X,Y`를 입력하고 **Enter**를 눌러 정확한 좌표를 입력합니다.
3. **두 번째 점을 클릭** — 호가 통과해야 하는 점(곡률과 방향을 제어). 여기서도 좌표 입력이 작동합니다.
4. **세 번째 점을 클릭** — 호의 다른 쪽 끝. 호가 배치되고 명령어가 종료됩니다. 여기서도 좌표 입력이 작동합니다.

```
           ● (2번째 클릭 — 곡선의 중간점)
          / \
         /   \
        ●     ●
     1번째     3번째
```

세 번째를 위치시키는 동안 선 미리보기가 첫 두 클릭을 연결합니다. 두 번째 클릭부터 실시간 호 미리보기가 커서를 추적합니다.

> **동일선상의 점**: 세 점이 모두 직선 위에 있으면 호를 계산할 수 없으며 객체가 배치되지 않습니다. 두 번째 점을 선에서 벗어나게 이동하고 다시 시도하세요.

## 좌표 입력

세 단계 중 어느 단계에서든 클릭 대신 정확한 위치를 입력할 수 있습니다:

1. X 값을 입력합니다.
2. `,`를 누릅니다 — 터미널에 `[X], [Y{커서}]`가 표시됩니다.
3. Y 값을 입력합니다.
4. **Enter**를 눌러 점을 배치합니다.

## 키보드 참고

| 키 | 동작 |
|----|------|
| `0`–`9`, `.`, `-` | X 좌표 입력 시작 |
| `,` | X를 잠그고 Y 입력으로 이동 |
| `Backspace` | 마지막으로 입력한 문자 삭제 |
| `Enter` | 입력한 좌표 확인 |
| `Escape` | 배치된 모든 점을 버리고 종료 |

## 그립 편집 — 끝점 및 반지름 조정

선택한 호는 세 개의 그립을 노출합니다:

| 그립 | 위치 | 기능 |
|------|------|------|
| **중심** | 외접원의 기하학적 중심 | 전체 호 이동; 반지름과 각도 변경 없음 |
| **시작** | 호의 첫 번째 끝점 | 드래그하여 외접원을 따라 시작점 이동 — 시작 각도와 반지름 모두 변경 |
| **끝** | 호의 마지막 끝점 | 드래그하여 외접원을 따라 끝점 이동 — 끝 각도와 반지름 모두 변경 |

시작 또는 끝 그립을 드래그하면 드래그 위치로 재배치되고 중심에서 새 위치까지의 각도와 반지름이 재계산됩니다. 반대쪽 끝점은 고정됩니다.

## 호 선택

| 방법 | 동작 |
|------|------|
| **클릭** | 클릭이 호 곡선 근처에 닿으면 선택(현이 아님) |
| **오른쪽으로 드래그** (엄격) | 호를 따라 분포된 샘플 점이 모두 박스 안에 있어야 함 |
| **왼쪽으로 드래그** (교차) | 박스 안에 있는 호의 어떤 샘플 점이든 선택 |

## 지원되는 편집 명령어

| 명령어 | 호에 일어나는 일 |
|--------|----------------|
| [Move](../move/) | 중심 이동; 반지름과 각도 변경 없음 |
| [Copy](../copy/) | 새 위치에 동일한 호 생성 |
| [Rotate](../rotate/) | 중심을 회전하고 회전량만큼 시작/끝 각도를 이동 |
| [Mirror](../mirror/) | 중심을 반사하고 대칭축을 기준으로 시작/끝 각도를 반전 |
| [Scale](../scale/) | 중심 위치를 축척하고 반지름에 축척 인수 곱하기 |
| [Offset](../offset/) | 같은 각도 범위에서 더 크거나 작은 반지름의 동심 호 생성 |
| [Delete](../delete/) | 호 제거 |

## 특성

**일반**

| 특성 | 기본값 | 의미 |
|------|--------|------|
| 색상 | 256 (레이어별) | ACI 색상 인덱스 |
| 레이어 | `0` | 레이어 지정 |
| 선종류 | 레이어별 | 명명된 선종류 패턴 |
| 선종류 축척 | 1 | 선종류 패턴의 축척 인수 |
| 두께 | 0 | 돌출 두께 |

**기하학**

| 특성 | 의미 |
|------|------|
| 중심 X / 중심 Y | 외접원의 중심 |
| 반지름 | 외접원의 반지름 |
| 시작 각도 | 호가 시작하는 각도(양의 X 축에서 측정, 도 단위) |
| 끝 각도 | 호가 끝나는 각도(도 단위) |

## Arc vs Circle — 언제 무엇을 사용할지

| | Arc | Circle |
|--|------|-----|
| 범위 | 부분 — 첫 번째에서 세 번째 클릭까지 | 전체 360° |
| 입력 방법 | 곡선의 세 점 | 중심 + 반지름(클릭 또는 입력) |
| 입력 방식 | 각 점에 대한 X,Y 좌표 | 반지름 값(중심도 X,Y 허용) |
| 배치 후 크기 조정 | 시작/끝 그립 드래그 | 임의 주요 그립 드래그 |
| 최적 용도 | 모깎기, 둥근 모서리, 아치형 경로 | 완전한 구멍, 둥근 형상 |

## DXF — ARC 객체

호는 DXF 파일에 `ARC` 객체로 저장되며, 중심 좌표, 반지름, 시작 각도, 끝 각도를 저장합니다. 색상, 레이어, 선종류, 선종류 축척, 두께를 포함한 모든 특성은 손실 없이 왕복 저장됩니다. DXF 호환 응용 프로그램(LibreCAD, FreeCAD 등)은 이것을 표준 호로 읽습니다.
