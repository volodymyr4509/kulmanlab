# Offset

`offset` 명령어는 고정된 수직 거리만큼 객체의 평행 복사본을 만듭니다. 거리를 한 번 입력한 다음 객체를 클릭하고 측면을 선택합니다 — 명령어는 같은 거리를 유지하면서 한 세션에서 여러 객체를 간격띄울 수 있습니다.

지원되는 객체 유형: **선, 원, 호, 타원, 폴리선** (직사각형 포함).

## 간격띄우기 사용

1. 터미널에 `offset`을 입력하거나 **Offset** 도구 모음 버튼을 클릭합니다.
2. **간격띄우기 거리를 입력**하고 **Enter** 또는 **Space**를 누릅니다.
3. 간격띄울 **객체를 클릭** — 지원되지 않는 유형의 객체인 경우 오류 메시지가 나타나며 다른 객체를 클릭할 수 있습니다.
4. 복사본이 나타날 **측면으로 커서를 이동** — 실시간 미리보기가 따라옵니다.
5. **클릭**하여 간격띄워진 복사본을 배치합니다.

각 배치 후 명령어는 **같은 거리**를 유지하며 3단계로 돌아가 다음 간격띄우기 준비를 합니다. **Escape**를 눌러 거리 입력 단계로 초기화합니다.

```
  거리: 10

  ─────────────────    ← 원본 선
  ─────────────────    ← 간격띄워진 복사본 (10단위 아래)
```

## 객체별 간격띄우기 동작

| 객체 | 간격띄우기 계산 방법 |
|------|---------------------|
| **선** | 원본 방향에 수직으로 이동된 평행선 |
| **원** | 동심원; 바깥쪽 클릭 → 더 큰 반지름, 안쪽 클릭 → 더 작은 반지름 |
| **호** | 새 반지름의 동심 호; 동일한 각도 범위 보존 |
| **타원** | 두 반축이 같은 거리만큼 증가 또는 감소 |
| **폴리선** | 각 세그먼트를 독립적으로 간격띄우기; 인접한 간격띄워진 세그먼트는 모서리에서 마이터 처리 |

**원**, **호**, **타원**의 경우: 안쪽 간격띄우기가 반지름이나 반축을 0 이하로 줄이면 간격띄우기가 적용되지 않습니다.

## 키보드 참고

| 키 | 동작 |
|----|------|
| `0`–`9`, `.` | 거리 값에 숫자 추가 |
| `Backspace` | 마지막으로 입력한 문자 삭제 |
| `Enter` / `Space` | 입력한 거리를 확인하고 객체 선택으로 진행 |
| `Escape` | 거리 입력 단계로 초기화 |

## 워크플로우 참고

거리는 **Escape**를 누를 때까지 설정된 상태로 유지됩니다. 이를 통해 같은 간격으로 여러 객체를 간격띄우는 것이 효율적입니다 — 거리를 한 번 입력한 다음 각 객체를 클릭하고 측면을 선택합니다.

## 간격띄우기 vs 복사

| | 간격띄우기 | 복사 |
|--|----------|------|
| 변위 | 객체의 기하학에 수직 | 임의 벡터(기준 → 대상) |
| 지원 객체 | 선, 원, 호, 타원, 폴리선 | 모든 객체 유형 |
| 거리 입력 | 객체 선택 전에 입력 | 선택 후 입력 또는 클릭 |
| 최적 용도 | 평행선, 동심원, 안쪽/바깥쪽 경로 | 임의 위치에 복제 배치 |
