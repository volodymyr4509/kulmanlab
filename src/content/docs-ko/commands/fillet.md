# Fillet

`fillet` 명령어는 주어진 반지름의 접선 호를 삽입하고 각 선을 호가 시작되는 점까지 잘라 두 [Line](../line/) 객체 사이의 모서리를 라운딩합니다. 결과는 두 선을 연결하는 부드럽고 반지름이 있는 모서리입니다.

모깎기는 **선 객체에서만** 작동합니다.

## 모깎기 사용

1. 터미널에 `fillet`을 입력하거나 **Fillet** 도구 모음 버튼을 클릭합니다.
2. **모깎기 반지름을 입력**하고 **Enter**를 누릅니다.
3. **첫 번째 선을 클릭** — 클릭한 부분이 교차점의 어느 쪽을 유지할지 결정합니다.
4. **두 번째 선 위에 커서를 올립니다** — 점선 호 미리보기가 결과 모깎기를 보여줍니다. 유지하려는 측면으로 커서를 이동합니다.
5. **클릭**하여 적용합니다. 두 선이 잘리고 호가 삽입됩니다.

```
  이전:                     이후 모깎기(반지름 r):

  ──────────────              ──────────╮
                │                        ╰────
                │
```

## 교차하는 선의 측면 선택

두 선이 서로 교차할 때, 모깎기는 클릭 위치로 정의된 모서리에 적용됩니다 — 각 선에서 **커서와 같은 측면**의 부분이 유지됩니다.

- 첫 번째 선의 한쪽 끝 근처를 클릭하여 그 절반을 선택합니다.
- 두 번째 선의 원하는 절반으로 커서를 이동합니다 — 점선 미리보기가 실시간으로 업데이트됩니다.

## 명령어가 만드는 것

- 첫 번째 선의 교차점에 가장 가까운 끝점이 접선점 **T1**으로 이동됩니다.
- 두 번째 선의 교차점에 가장 가까운 끝점이 접선점 **T2**로 이동됩니다.
- **T1**에서 **T2**까지 두 선에 접하는 새 호 객체가 삽입됩니다.

삽입된 호는 현재 선 가중치, 색상, 레이어, 선종류 설정을 상속합니다.

## 키보드 참고

| 키 | 동작 |
|----|------|
| `0`–`9`, `.` | 반지름 값에 숫자 추가 |
| `Backspace` | 마지막으로 입력한 문자 삭제 |
| `Enter` | 입력한 반지름을 확인하고 선 선택으로 이동 |
| `Escape` | 취소하고 초기화 |

## 지원되는 객체

| 객체 | 지원 여부 |
|------|----------|
| 선 | 예 — 첫 번째와 두 번째 객체 모두 |
| 호, 원, 타원, 폴리선 | 아니오 |
| 텍스트, 스플라인, 치수, 지시선 | 아니오 |

## Fillet vs Chamfer

| | Fillet | Chamfer |
|--|--------|--------|
| 모서리 유형 | 라운딩 호 | 직선 절단 |
| 입력 | 반지름 하나 | 두 거리(d1, d2) |
| 삽입되는 객체 | 호 | 선 |
| 지원 객체 | 선만 해당 | 선 및 폴리선 |
