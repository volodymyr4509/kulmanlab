# Scale

`scale` 명령어는 기준점을 중심으로 선택한 객체를 균일하게 크기 조정합니다. 기준점으로부터의 모든 거리에 축척 인수가 곱해집니다 — 인수 `2`는 모든 치수를 두 배로 하고, `0.5`는 반으로 줄입니다. 인수는 항상 입력으로 설정합니다; 클릭으로 축척을 설정하는 방법은 없습니다.

## 두 가지 시작 방법

**사전 선택 후 축척** — 먼저 객체를 선택한 다음 활성화:

1. 캔버스에서 하나 이상의 객체를 선택합니다.
2. 터미널에 `scale`을 입력하거나 **Scale** 도구 모음 버튼을 클릭합니다.
3. **기준점을 클릭** — 축척 중 이동하지 않는 고정 점. 또는 `X,Y`를 입력하고 **Enter**를 눌러 정확한 좌표를 입력합니다.
4. **축척 인수를 입력**하고 **Enter**를 누릅니다.

**활성화 후 선택** — 아무것도 선택하지 않고 명령어 시작:

1. `scale`을 입력하거나 도구 모음 버튼을 클릭합니다.
2. **객체 선택** — 클릭하여 토글하거나, 드래그하여 영역으로 선택합니다.
3. **Enter** 또는 **Space**를 눌러 선택을 확인합니다.
4. **기준점을 클릭**(좌표 입력 가능)한 다음 인수를 입력합니다.

```
  기준 ●                기준 ●
        [객체]   →            [더 큰 객체]
  인수 = 2 → ●로부터의 거리가 두 배로
```

## 축척 인수 입력

기준점이 배치된 후 터미널에 `축척 인수를 입력하세요:`가 표시되며 키보드 입력을 기다립니다:

| 키 | 동작 |
|----|------|
| `0`–`9`, `.` | 인수에 숫자 추가 |
| `-` | 음수 인수(첫 번째 문자만 — 반전 후 축척) |
| `Backspace` | 마지막으로 입력한 문자 삭제 |
| `Enter` | 입력한 인수로 축척 적용 |

인수는 0이 아니어야 합니다. 일반적인 값:

| 인수 | 효과 |
|------|------|
| `2` | 모든 치수 두 배로 |
| `0.5` | 모든 치수 반으로 |
| `1.5` | 50% 증가 |
| `-1` | 기준점을 기준으로 대칭(180° 회전과 동일) |

입력하는 동안 실시간 미리보기가 없습니다 — 축척된 결과는 **Enter**를 누른 후에만 나타납니다.

## 키보드 참고

| 키 | 동작 |
|----|------|
| `Enter` / `Space` | 선택 확인 |
| `0`–`9`, `.`, `-` | X 좌표 입력 시작(기준점 단계) 또는 축척 인수(인수 단계) |
| `,` | X를 잠그고 Y 입력으로 이동(기준점 단계) |
| `Backspace` | 마지막으로 입력한 문자 삭제 |
| `Enter` | 좌표 확인 또는 축척 적용 |
| `Escape` | 취소하고 초기화 |

## 명령어 중 선택

| 방법 | 동작 |
|------|------|
| **클릭** | 커서 아래 객체 토글 |
| **오른쪽으로 드래그** (엄격) | 박스 안에 완전히 있는 객체 추가 |
| **왼쪽으로 드래그** (교차) | 박스와 교차하는 객체 추가 |
| **Enter** / **Space** | 선택 확인 |

## 축척되는 내용

모든 객체 유형을 지원합니다. 각 객체는 기준점을 기준으로 기하학을 축척합니다:

| 객체 | 변경 사항 |
|------|----------|
| 선 | 두 끝점이 기준점에서 축척 |
| 원 | 중심이 기준점에서 축척; 반지름에 인수 곱하기 |
| 호 | 중심 축척; 반지름에 인수 곱하기; 각도 변경 없음 |
| 타원 | 중심 축척; 두 반축 길이에 인수 곱하기 |
| 폴리선 / 직사각형 | 모든 꼭짓점이 기준점에서 축척 |
| 텍스트 | 앵커 점 축척; 높이에 인수 곱하기 |
| 스플라인 | 모든 제어 꼭짓점 / 피트 점 축척 |
