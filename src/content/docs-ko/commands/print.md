# 인쇄

`print` 명령어는 **인쇄 관리자**를 엽니다 — 실시간 미리보기 캔버스, 형식 선택기(PNG / JPEG / WebP / PDF), 단색 전환 및 선택적 영역 자르기가 있는 전용 내보내기 창입니다. 실제 프린터로는 아무것도 전송되지 않습니다; 출력은 파일로 다운로드됩니다.

## 인쇄 관리자 여는 방법

**인쇄** 도구 모음 버튼을 클릭하거나 터미널에 `print`를 입력합니다. 인쇄 관리자가 현재 뷰포트의 미리보기를 보여주며 즉시 열립니다.

## 인쇄 관리자 레이아웃

창에는 두 개의 패널이 있습니다:
- **왼쪽 사이드바** — 모든 내보내기 컨트롤.
- **오른쪽 패널** — 설정을 변경할 때 업데이트되는 실시간 미리보기 캔버스.

### 사이드바 컨트롤

| 컨트롤 | 설명 |
|--------|------|
| **영역 변경** | 캔버스에서 사용자 정의 직사각형으로 자르기(아래 참조) |
| **단색** 전환 | 모든 색상 선을 검정으로 변환 — 깔끔한 인쇄 출력을 위해 기본값으로 켜져 있음 |
| **형식** 드롭다운 | PNG, JPEG, WebP 또는 PDF |
| **내보내기** 버튼 | 파일 생성 및 다운로드 |

## 사용자 정의 내보내기 영역 선택

기본적으로 미리보기는 인쇄 관리자를 열었을 때 캔버스에 보였던 내용을 정확히 보여줍니다. 특정 영역을 내보내려면:

1. **영역 변경**을 클릭합니다 — 인쇄 관리자가 숨겨지고 캔버스가 대화형 상태가 됩니다.
2. 내보내기 직사각형의 **첫 번째 모서리를 클릭**합니다.
3. **반대쪽 모서리를 클릭**합니다 — 인쇄 관리자가 선택한 영역이 미리보기에 표시되며 다시 열립니다.

영역 선택 중에 `Escape`를 눌러 취소하고 이전 영역을 복원합니다.

미리보기 캔버스는 선택한 영역의 **정확한 가로세로 비율**에 맞게 동적으로 크기가 조정되므로 미리보기가 픽셀 단위로 정확합니다.

## 내보내기 형식

| 형식 | 최적 용도 | 비고 |
|------|----------|------|
| **PNG** | 손실 없음, 선명한 선 | 흰색 배경, 투명도 없음 |
| **JPEG** | 공유를 위한 작은 파일 | 95% 품질, 약간의 압축 |
| **WebP** | 웹용 가장 작은 파일 | 동일한 95% 품질, JPEG보다 나은 압축 |
| **PDF** | 인쇄 준비 문서 | PDF 컨테이너 내에 150 DPI로 이미지 삽입 |

내보낸 파일은 `kulman-<타임스탬프>.<확장자>` 형태로 이름이 지정되며 자동으로 다운로드됩니다.

## 내보내기 해상도 및 배경

- 최대 해상도: **2000 × 2000 픽셀**, 선택한 영역에 비례하여 크기 조정.
- 배경은 항상 **흰색**입니다.
- **비인쇄** 표시된 레이어는 내보내기에서 제외됩니다.

## 인쇄 vs 내보내기

| 기능 | 인쇄 | 내보내기 |
|------|------|---------|
| 출력 | 래스터 이미지(.png / .jpeg / .webp / .pdf) | 벡터 소스 파일(.dxf / .json) |
| 다른 도구에서 편집 가능 | 아니오 | 예(DXF) |
| 레이어 및 선종류 보존 | 아니오(평면 렌더링) | 예 |
| 치수 및 지시선 캡처 | 예 | JSON만 가능 |

시각적 스냅샷이 필요할 때 **인쇄**를 사용합니다. 편집 가능한 파일이 필요하면 [내보내기](./export)를 사용합니다.

## 키보드 참고

| 키 | 동작 |
|----|------|
| `Escape`(영역 선택 중) | 영역 선택 취소, 이전 영역 복원 |
| `Escape`(인쇄 관리자에서) | 인쇄 관리자 닫기 |

## 관련 명령어
