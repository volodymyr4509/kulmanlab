# ViewportRectangle

`ViewportRectangle` 명령어는 두 개의 반대쪽 모서리를 선택하여 활성 용지 레이아웃에 새 뷰포트를 생성합니다. 레이아웃 공간에서만 사용 가능합니다.

## 뷰포트 생성

1. 화면 하단의 탭을 사용하여 용지 레이아웃으로 전환합니다.
2. 터미널에 `ViewportRectangle`을 입력하거나 **ViewportRectangle** 도구 모음 버튼을 클릭합니다.
3. **첫 번째 모서리를 클릭**하거나, `X,Y`를 입력하고 **Enter**를 눌러 정확한 좌표를 입력합니다.
4. **반대쪽 모서리를 클릭** — 뷰포트가 즉시 배치됩니다. 여기서도 좌표 입력이 작동합니다.

새 뷰포트는 레이아웃의 기본 축척으로 전체 모델을 보여줍니다. 뷰포트 내에서 스크롤 휠을 사용하여 확대/축소하거나, 가운데 버튼을 드래그하여 모델 뷰를 이동합니다.

## 좌표 입력

모서리 단계에서 정확한 좌표를 입력할 수 있습니다:

1. X 값을 입력합니다.
2. `,`를 누릅니다 — 터미널에 `[X], [Y{커서}]`가 표시됩니다.
3. Y 값을 입력합니다.
4. **Enter**를 눌러 점을 배치합니다.

## 키보드 참고

| 키 | 동작 |
|----|------|
| `0`–`9`, `.`, `-` | X 좌표 입력 시작 |
| `,` | X를 잠그고 Y 입력으로 이동 |
| `Enter` | 입력한 좌표 확인 |
| `Escape` | 취소 |

## 뷰포트 편집

뷰포트를 배치한 후 클릭하여 선택합니다:

- **가장자리 또는 모서리를 드래그**하여 크기를 조정합니다.
- **중심 그립을 드래그**하여 이동합니다.
- 컨트롤 바의 **축척 선택기**를 사용하여 정확한 축척을 설정합니다(예: 1:50). 목록에 없는 축척을 입력하려면 드롭다운 하단의 입력 필드에 직접 입력합니다 — 비율 형식(`1:200`, `5:1`) 또는 소수(`0.005`)를 허용하며 **Enter**를 누릅니다.
- 뷰포트를 오른쪽 클릭하고 **잠금**을 사용하여 우발적인 변경을 방지합니다.

## 참고

- ViewportRectangle은 용지 레이아웃 탭이 활성화된 경우에만 사용 가능합니다. 모델 공간에서 실행하면 오류 메시지가 표시되고 종료됩니다.
- 기존 뷰포트를 복사하려면 [ViewportCopy](../viewport-copy/)를 사용하세요.
