# 축소

`zoomout` 명령어는 현재 확대/축소 수준을 **1.5×**로 나누고(약 0.667을 곱하는 것과 동일) 뷰포트 중간점을 중심으로 즉시 종료됩니다. [확대](../zoom-in/)의 역연산입니다.

## 축소

**축소** 도구 모음 버튼을 클릭하거나 터미널에 `zoomout`을 입력합니다. 확대/축소가 즉시 적용되고 명령어가 종료됩니다 — 캔버스 클릭이 필요 없습니다.

## 1.5× 단계 작동 방식

| 현재 확대/축소 | 축소 한 번 후 |
|-------------|------------|
| 1.50× | 1.00× |
| 2.25× | 1.50× |
| 10.00× | 6.67× |
| 0.015× | 0.01× (하한) |

확대/축소 수준은 항상 캔버스의 **오른쪽 하단 모서리**에 표시됩니다. 하한은 **0.01×**이며; 이후 단계는 아무 효과가 없습니다.

## 도구 모음 축소 vs 스크롤 휠

| | 축소 버튼 | 스크롤 휠 |
|--|----------|----------|
| 확대/축소 중심 | 뷰포트 중간점 | 커서 위치 |
| 단계 크기 | 클릭당 1.5× | 틱당 ~1.1× |
| 활성화 필요 | 아니오 | 아니오 — 항상 작동 |
| 최적 용도 | 더 많은 컨텍스트를 보기 위해 뒤로 물러나기 | 부드러운 커서 앵커 축소 |

## 키보드 참고

이 명령어에는 키보드 단축키가 없습니다. 대신 스크롤 휠을 사용하세요 — 명령어를 활성화하지 않고도 언제든지 작동합니다.
