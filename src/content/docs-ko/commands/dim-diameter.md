# 지름 치수

`dimdiameter` 명령어는 호 또는 원에 지름 치수를 배치합니다. 치수 선은 중심 사이를 통과하여 지름 반대편 두 호 점 사이에 걸쳐 있으며 `⌀ <값>`으로 레이블이 붙습니다. 한쪽 가장자리에서 중심까지의 반지름만 주석을 달려면 [반지름 치수](./dim-radius)를 사용하세요.

## 지름 치수의 구조

```
  ●──────────── ⌀ 10.00 ────────────●
  (먼 호 점)              (가까운 호 점 / 텍스트 측면)
```

- **치수 선** — 양쪽 호 교차점에 화살표가 있는 전체 지름에 걸쳐 있습니다.
- **가까운 호 점** — 커서 측면의 원주 점(텍스트 레이블이 위치).
- **먼 호 점** — 지름 반대편 점.
- **레이블** — `⌀` 다음에 지름 값.

## 지름 치수 배치

1. 터미널에 `dimdiameter`를 입력하거나 **지름 치수** 도구 모음 버튼을 클릭합니다.
2. **호 또는 원을 클릭**하여 선택합니다.
3. **커서를 이동**하여 치수 선을 원하는 각도로 회전합니다.
4. **클릭**하여 치수를 배치합니다.

**호** 및 **원** 객체만 선택할 수 있습니다.

## 지름 치수 vs 반지름 치수

| | 지름 치수 | 반지름 치수 |
|--|----------|-----------|
| 측정 내용 | 전체 지름(2 × 반지름) | 반지름(중심에서 가장자리까지) |
| 치수 선 | 가장자리 → 중심 통과 가장자리 | 중심 → 가장자리 |
| 레이블 접두사 | `⌀` | `R` |
| 화살표 | 둘(두 호 점에) | 하나(호 점에) |
| 최적 용도 | 완전한 원형 구멍 또는 축 치수 | 곡선 형상의 한쪽 주석 |

## DXF — DIMENSION 지름 객체

지름 치수는 지름 유형 기하학을 가진 `DIMENSION` 객체로 저장되며, 두 호 점 위치와 측정된 지름 값(2 × 반지름)을 저장합니다. 모든 특성이 손실 없이 왕복 저장됩니다.
