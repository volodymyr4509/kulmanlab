# 모따기

`chamfer` 명령어는 두 [선](./line) 또는 [폴리선](./polyline) 객체 사이에 직선 대각 모서리를 만듭니다. 각 객체를 따라 얼마나 잘라낼지(d1 및 d2)를 지정하면, 명령어가 두 객체를 해당 점까지 잘라 연결 선을 삽입합니다.

같은 거리를 사용하면 대칭적인 45° 절단이 생성되고; 다른 거리를 사용하면 비대칭 경사가 생성됩니다.

모따기는 **선 및 폴리선** 객체에서 작동합니다.

## 모따기 사용

1. 터미널에 `chamfer`를 입력하거나 **모따기** 도구 모음 버튼을 클릭합니다.
2. **첫 번째 모따기 거리를 입력**합니다(d1 — 첫 번째 객체를 따른 거리)하고 **Enter**를 누릅니다.
3. **두 번째 모따기 거리를 입력**합니다(d2 — 두 번째 객체를 따른 거리)하고 **Enter**를 누릅니다.
4. **첫 번째 객체를 클릭** — 클릭한 부분이 교차점의 어느 쪽을 유지할지 결정합니다.
5. **두 번째 객체 위에 커서를 올립니다** — 점선 미리보기가 결과 모따기 절단을 보여줍니다. 유지하려는 측면으로 커서를 이동합니다.
6. **클릭**하여 적용합니다. 두 객체가 잘리고 모따기 선이 삽입됩니다.

```
  이전 (d1=5, d2=8):        이후:

  ──────────────              ──────────╲
                │                        ╲────
                │
```

## 측면 선택

두 선이 서로 교차할 때, 모따기는 클릭 위치로 정의된 모서리에 적용됩니다 — 각 객체에서 **커서와 같은 측면**의 부분이 유지됩니다.

- 첫 번째 객체의 한쪽 끝 근처를 클릭하여 그 절반을 선택합니다.
- 두 번째 객체의 원하는 절반으로 커서를 이동합니다 — 점선 미리보기가 실시간으로 업데이트됩니다.

폴리선의 경우, 클릭 위치가 폴리선의 어떤 **세그먼트**가 참여할지 결정하고, 교차점 측면에서 가장 가까운 꼭짓점이 잘리는 꼭짓점이 됩니다.

## 명령어가 만드는 것

- 첫 번째 객체의 교차점에 가장 가까운 끝점(또는 폴리선 꼭짓점)이 교차점에서 첫 번째 객체를 따라 d1만큼 위치한 **T1**으로 이동됩니다.
- 두 번째 객체의 교차점에 가장 가까운 끝점(또는 폴리선 꼭짓점)이 교차점에서 두 번째 객체를 따라 d2만큼 위치한 **T2**로 이동됩니다.
- **T1**에서 **T2**까지 새 선 객체가 삽입됩니다.

삽입된 선은 현재 선 가중치, 색상, 레이어, 선종류 설정을 상속합니다.

## 키보드 참고

| 키 | 동작 |
|----|------|
| `0`–`9`, `.` | 현재 거리 값에 숫자 추가 |
| `Backspace` | 마지막으로 입력한 문자 삭제 |
| `Enter` | 입력한 거리를 확인하고 진행 |
| `Escape` | 취소하고 초기화 |

## 지원되는 객체

| 객체 | 지원 여부 |
|------|----------|
| 선 | 예 |
| 폴리선 / 직사각형 | 예 |
| 호, 원, 타원 | 아니오 |
| 텍스트, 스플라인, 치수, 지시선 | 아니오 |

## 모따기 vs 모깎기

| | 모따기 | 모깎기 |
|--|--------|--------|
| 모서리 유형 | 직선 절단 | 라운딩 호 |
| 입력 | 두 거리(d1, d2) | 반지름 하나 |
| 삽입되는 객체 | 선 | 호 |
| 지원 객체 | 선 및 폴리선 | 선만 해당 |
