# 지시선

`leader` 명령어는 네 단계로 다중 지시선 주석을 그립니다: 형상에 닿는 화살표, 꺾임에서 구부러지는 지시선, 텍스트 앵커, 입력한 레이블. 모든 주석 명령어 중에서 지시선만이 깜박이는 커서 미리보기가 있는 대화형 텍스트 입력 단계를 포함합니다.

## 다중 지시선의 구조

```
  ◄── 화살표 끝  (2단계 — 형상에 닿음)
      \
       \  지시선
        \
         ●──── 꺾임(3단계) ──── 텍스트 앵커(4단계)
                                레이블 텍스트  (5단계)
```

- **화살표 끝** — 주석이 달리는 형상에 배치되는 뾰족한 끝.
- **꺾임** — 지시선이 텍스트 방향으로 구부러지는 팔꿈치.
- **텍스트 앵커** — 레이블이 위치하는 곳. 텍스트는 자동으로 왼쪽 또는 오른쪽으로 정렬됩니다.

## 지시선 그리기

1. 터미널에 `leader`를 입력하거나 **지시선** 도구 모음 버튼을 클릭합니다.
2. **화살표 끝을 클릭**하거나, `X,Y`를 입력하고 **Enter**를 눌러 정확한 좌표를 입력합니다.
3. **꺾임을 클릭** — 지시선의 구부러짐. 각도가 45° 단위로 잠힙니다; 길이를 입력하고 **Enter**를 눌러 정확히 배치합니다. 또는 절대 좌표를 입력하려면 `X,Y`를 입력합니다.
4. **텍스트 위치를 클릭** — 레이블이 앵커될 곳. 같은 옵션 적용: 클릭, 각도 잠금 + 길이, 또는 `X,Y`.
5. **레이블 텍스트를 입력** — 캔버스 미리보기가 깜박이는 커서와 함께 실시간으로 업데이트됩니다. **Enter**를 눌러 배치합니다.

## 좌표 입력(모든 점 단계)

임의 점 선택 단계(끝, 꺾임, 텍스트 위치)에서 클릭 대신 정확한 좌표를 입력할 수 있습니다:

1. X 값을 입력합니다(숫자, `.`, 또는 `-`).
2. `,`를 누릅니다 — 터미널에 X가 잠겼음을 확인하는 `[X], [Y{커서}]`가 표시됩니다.
3. Y 값을 입력합니다.
4. **Enter**를 눌러 점을 배치합니다.

## 각도 잠금(3단계 및 4단계)

각 배치된 점 후에 명령어는 커서가 충분히 멀리 있을 때 45° 축에 스냅됩니다. 잠긴 상태에서:
- 미리보기가 축에 스냅됩니다.
- 길이를 입력하고 **Enter**를 눌러 정확히 그 거리에 다음 점을 배치합니다.

각도 잠금과 좌표 입력은 상호 배타적입니다 — `,` 없이 숫자를 입력하면 명령어는 이를 거리로 해석합니다(각도 잠금이 활성화되어 있어야 함). 절대 좌표를 입력하려면 X 번호 다음에 쉼표를 입력하여 시작합니다.

## 텍스트 레이블 편집

5단계에서 레이블을 입력하는 동안 배치 전에 텍스트를 탐색하고 편집할 수 있습니다:

| 키 | 동작 |
|----|------|
| 인쇄 가능한 모든 문자 | 커서 위치에 삽입 |
| `←` / `→` | 커서를 왼쪽 또는 오른쪽으로 이동 |
| `Backspace` | 커서 왼쪽 문자 삭제 |
| `Delete` | 커서 오른쪽 문자 삭제 |
| `Enter` | 지시선 배치 |

## 자동 텍스트 방향

텍스트 정렬은 꺾임에 대한 커서 위치에 따라 조정됩니다:

| 커서 위치 | 텍스트 방향 |
|----------|------------|
| 꺾임의 **오른쪽** | 텍스트 앵커에서 왼쪽에서 오른쪽 |
| 꺾임의 **왼쪽** | 오른쪽에서 왼쪽(오른쪽에 앵커됨) |

수동 조정이 필요 없습니다 — 레이블을 원하는 쪽으로 커서를 이동하면 올바르게 정렬됩니다.

## 키보드 참고

**점 단계(끝, 꺾임, 텍스트 위치)**

| 키 | 동작 |
|----|------|
| `0`–`9`, `.`, `-` | X 좌표 입력 시작(그런 다음 `,`로 X 잠금 및 Y 입력) |
| `,` | X 확인 및 Y 입력으로 이동 |
| `0`–`9`, `.`, `-` | 각도 잠금 시 거리 입력 |
| `Backspace` | 마지막으로 입력한 문자 삭제 |
| `Enter` | 입력한 좌표 또는 거리 확인 |

**텍스트 입력 단계**

| 키 | 동작 |
|----|------|
| 인쇄 가능한 문자 | 커서에 삽입 |
| `←` / `→` | 커서 이동 |
| `Backspace` | 왼쪽 삭제 |
| `Delete` | 오른쪽 삭제 |
| `Enter` | 지시선 배치 |

| 키 | 동작 |
|----|------|
| `Escape` | 취소하고 2단계로 초기화 |

## 팔 추가 및 제거

- 기존 지시선에 추가 화살표 팔을 추가하려면: [지시선+](./leader-add)
- 두 개 이상의 팔이 있는 지시선에서 팔을 제거하려면: [지시선−](./leader-remove)

## DXF — 가져오기 전용

**지시선은 가져오기 전용입니다.** DXF 파일의 `MLEADER` 객체는 올바르게 읽혀 표시되지만, 편집기에서 그린 지시선은 DXF 파일을 저장할 때 **기록되지 않습니다**. 시각적 주석에 지시선을 사용하세요; 왕복 워크플로우에 의존하지 마세요.
