# Ellipse

`ellipse` 명령어는 세 번의 클릭을 사용하여 타원을 그립니다: 중심점, 첫 번째(장) 반축의 끝점(임의 각도), 두 번째(단) 반축의 길이. 두 축은 항상 서로 수직입니다 — 두 번째 축 방향은 첫 번째에서 자동으로 도출됩니다.

## 타원 그리기

1. 터미널에 `ellipse`를 입력하거나 **Ellipse** 도구 모음 버튼을 클릭합니다.
2. **중심점을 클릭**하거나, `X,Y`를 입력하고 **Enter**를 눌러 정확한 좌표를 입력합니다.
3. **첫 번째 축 끝점을 클릭** — 첫 번째 반축의 방향과 길이를 모두 설정합니다. 여기서도 좌표 입력이 작동합니다.
4. **두 번째 축 길이를 설정** — 첫 번째 축에 수직으로 커서를 이동한 다음, 클릭하거나 길이를 입력합니다.

```
               ● ← 첫 번째 축 끝점(3단계)
              /
  중심 ●  /  ← 첫 번째 축(임의 각도)
            |
            ● ← 커서 위치로 두 번째 축 길이 설정(4단계)
```

타원은 4단계 후에 배치되고 명령어가 종료됩니다.

## 축 입력 — 클릭, 좌표 또는 입력한 길이

**중심(2단계):** 클릭하거나 정확한 위치를 위해 `X,Y`를 입력합니다.

**첫 번째 축 끝점(3단계):** 클릭하거나 정확한 좌표를 위해 `X,Y`를 입력합니다. 각도 잠금도 45° 단위로 스냅합니다 — 잠긴 상태에서 길이를 입력하고 **Enter**를 누르면 끝점이 정확히 그 거리에 배치됩니다.

**두 번째 축(4단계):** 입력한 길이는 항상 사용 가능합니다 — 각도 잠금이 필요 없습니다. 방향은 이미 첫 번째 축에 수직으로 고정되어 있으므로; 입력은 길이만 설정합니다.

| 키 | 동작 |
|----|------|
| `0`–`9`, `.` | 축 길이에 숫자 추가(두 번째 축 단계) |
| `Backspace` | 마지막으로 입력한 문자 삭제 |
| `Enter` | 입력한 길이에 축 끝점 배치 |

## 좌표 입력(중심 및 첫 번째 축 끝점)

1. X 값을 입력합니다.
2. `,`를 누릅니다 — 터미널에 `[X], [Y{커서}]`가 표시됩니다.
3. Y 값을 입력합니다.
4. **Enter**를 눌러 확인합니다.

## 키보드 참고

| 키 | 동작 |
|----|------|
| `0`–`9`, `.`, `-` | X 좌표 입력 시작(중심/첫 번째 축 단계) 또는 각도 잠금 시 축 길이 |
| `,` | X를 잠그고 Y 입력으로 이동 |
| `Backspace` | 마지막으로 입력한 문자 삭제 |
| `Enter` | 입력한 좌표 또는 길이 확인 |
| `Escape` | 취소하고 초기화 |

## 그립 편집 — 독립적인 축 크기 조정

선택한 타원은 다섯 개의 그립을 노출합니다:

| 그립 | 개수 | 기능 |
|------|------|------|
| **중심** | 1 | 전체 타원 이동; 두 축 변경 없음 |
| **장축 끝점** | 2(긴 축의 반대편 끝) | 드래그하여 장 반축 길이 조정; 단축 절대 크기 유지 |
| **단축 끝점** | 2(짧은 축의 반대편 끝) | 드래그하여 단 반축 길이 조정; 장축 변경 없음 |

장축 및 단축 그립은 독립적입니다 — 명령어를 다시 실행하지 않고도 타원을 재형성할 수 있습니다.

## 타원 선택

| 방법 | 동작 |
|------|------|
| **클릭** | 클릭이 타원 윤곽선 근처에 닿으면 선택 |
| **오른쪽으로 드래그** (엄격) | 타원의 축 정렬 경계 박스가 선택 박스 내에 완전히 있어야 함 |
| **왼쪽으로 드래그** (교차) | 선택 박스 경계와 교차하는 타원 윤곽선의 어느 부분이든 선택 |

## 특성

**일반**

| 특성 | 기본값 | 의미 |
|------|--------|------|
| 색상 | 256 (레이어별) | ACI 색상 인덱스 |
| 레이어 | `0` | 레이어 지정 |
| 선종류 | 레이어별 | 명명된 선종류 패턴 |
| 선종류 축척 | 1 | 선종류 패턴의 축척 인수 |
| 두께 | 0 | 돌출 두께 |

**기하학**

| 특성 | 의미 |
|------|------|
| 중심 X / 중심 Y | 타원의 중심 |
| 장축 X / 장축 Y | 중심에서 장축 끝점까지의 벡터(방향과 길이 모두 인코딩) |
| 축 비율 | 단 반축과 장 반축의 비율(0 < 비율 ≤ 1) |
| 시작 각도 / 끝 각도 | 도 단위의 매개변수 각도; 완전한 타원은 모두 0°/360° |

## Ellipse vs Circle — 언제 무엇을 사용할지

| | Ellipse | Circle |
|--|------|-----|
| 축 | 임의 각도의 두 독립 반축 | 하나의 반지름, 대칭 |
| 회전 | 임의 각도로 배치 가능 | 회전 없음 |
| 입력 방식 | 축별 길이 | 반지름만 |
| 그립 크기 조정 | 장축과 단축 독립적으로 | 네 개의 주요 점 모두 동일 |
| 최적 용도 | 사시도, 타원형 형상, 원근 구멍 | 대칭 둥근 형상 |

## DXF — ELLIPSE 객체

타원은 DXF 파일에 `ELLIPSE` 객체로 저장됩니다. 형식은 중심점, 완전한 장축 벡터(방향 + 길이), 축 비율을 저장합니다. 회전, 형상, 모든 스타일 특성은 손실 없이 왕복 저장됩니다. 원은 퇴화된 타원으로 저장되지 않습니다 — 두 객체 유형은 DXF 모델에서 구별됩니다.
