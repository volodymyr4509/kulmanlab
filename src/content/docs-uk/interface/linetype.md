# Тип лінії

**Тип лінії** визначає шаблон вигляду лінії або полілінії: суцільна, пунктирна, штрихова, штрих-пунктирна або будь-яка інша послідовність штрихів і пробілів. Це зберігається у властивості `LTYPE` об'єкта в DXF.

## Призначення типу лінії

### На рівні об'єкта

1. Виберіть об'єкт.
2. На **панелі властивостей** клацніть поле **Тип лінії**.
3. Виберіть тип з випадаючого списку.

### На рівні шару (рекомендовано)

Встановіть тип лінії шару в **менеджері шарів** — всі об'єкти з типом лінії `ByLayer` автоматично використовуватимуть тип шару. Це зручніше при необхідності зміни типу для групи об'єктів.

## ByLayer та ByBlock

| Значення | Що означає |
|----------|------------|
| **ByLayer** | Об'єкт наслідує тип лінії від свого шару (рекомендовано) |
| **ByBlock** | Об'єкт наслідує тип лінії від блоку (для блоків) |
| Конкретна назва | Тип лінії призначений безпосередньо об'єкту |

## Масштаб типу лінії

Властивість **Масштаб типу лінії** (або `LTSCALE`) визначає, наскільки великим є шаблон на полотні. Значення `1` — стандартний розмір. Збільшіть для більшого шаблону (менша щільність штрихів), зменшіть для дрібнішого.

Глобальний масштаб типів ліній застосовується до всіх об'єктів, а масштаб типу лінії окремого об'єкта множиться на глобальний.

## Стандартні типи ліній DXF

| Назва | Вигляд | Опис |
|-------|--------|------|
| `CONTINUOUS` | ——— | Суцільна лінія |
| `DASHED` | - - - | Рівні штрихи |
| `DOTTED` | . . . | Крапки |
| `DASHDOT` | -.-.  | Штрих-крапка |
| `DASHDOT2` | _._. | Штрих-крапка дрібний |
| `CENTER` | —.— | Центрова лінія |
| `HIDDEN` | — — | Приховані лінії |

## DXF — LTYPE

Типи ліній зберігаються у секції `TABLES` DXF файлу. При відкритті DXF файлу всі визначення типів ліній зберігаються і доступні для вибору.
