# Текстовий редактор

Текстовий редактор KulmanLab CAD надає параметри форматування для текстових об'єктів, розміщених командою [Text](../commands/text). Ці параметри визначають зовнішній вигляд тексту: шрифт, розмір, вирівнювання та інтервал.

## Відкриття текстового редактора

Параметри форматування тексту доступні через:
- Панель **Властивості** при вибраному текстовому об'єкті.
- Меню **Налаштування** → **Текстовий редактор**.

## Параметри форматування

### Шрифт

Вибір гарнітури шрифту з доступних варіантів. KulmanLab CAD підтримує стандартні CAD-шрифти та основні системні шрифти, що забезпечують відображення у браузері.

| Шрифт | Опис |
|-------|------|
| **Standard** | Стандартний CAD-шрифт (за замовчуванням) |
| **Arial / Helvetica** | Сучасний шрифт без засічок |
| **Times / Serif** | Класичний шрифт із засічками |
| **Monospace** | Фіксована ширина символів |

### Висота тексту

Висота текстових символів у одиницях кресленика. Це те ж значення, що й властивість **Висота** у DXF об'єкті TEXT.

| Значення | Типове використання |
|----------|-------------------|
| Мале (2–3) | Мітки деталей, виноски |
| Середнє (5–7) | Заголовки розділів, номери позицій |
| Велике (10+) | Заголовки аркушів, назви проєктів |

### Вирівнювання

Горизонтальне вирівнювання тексту відносно точки вставки:

| Вирівнювання | Точка вставки |
|-------------|--------------|
| **Ліво** | Ліворуч від першого символу (за замовчуванням) |
| **Центр** | По центру рядка тексту |
| **Право** | Праворуч від останнього символу |

### Кут повороту

Кут нахилу тексту у градусах. `0` — горизонтальний текст (зліва направо). Позитивні значення повертають проти годинникової стрілки.

### Інтервал між рядками

Відстань між рядками для багаторядкового тексту. Значення `1.0` — стандартний одинарний інтервал; `1.5` — полуторний; `2.0` — подвійний.

## Застосування до існуючого тексту

1. **Виберіть текстовий об'єкт** (клацніть на нього).
2. Змініть параметри на панелі **Властивості**.
3. Зміни застосовуються негайно.

Для одночасної зміни кількох текстових об'єктів виберіть їх усі, потім змініть параметри — зміни застосовуються до всіх вибраних.

## Параметри за замовчуванням для нових текстів

Параметри форматування в меню Налаштування встановлюють значення за замовчуванням для **наступних** розміщень командою [Text](../commands/text). Поточні тексти на кресленику не змінюються при зміні параметрів за замовчуванням.

## DXF — об'єкти TEXT та STYLE

Параметри тексту зберігаються у двох місцях DXF файлу:

- **Секція TABLES → STYLE**: визначення текстових стилів (шрифт, висота за замовчуванням).
- **Об'єкт TEXT**: фактичний текст з точкою вставки, висотою, кутом та посиланням на стиль.

При відкритті DXF файлу з іншої CAD-програми текстові стилі з цього файлу використовуються для відображення тексту. Якщо шрифт недоступний у браузері, він замінюється найближчим доступним аналогом.
