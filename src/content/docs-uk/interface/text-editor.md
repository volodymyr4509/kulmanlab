# Текстовий редактор

Текстовий редактор відкривається при розміщенні або подвійному клацанні на редагованому об'єкті. Маленький **chip режиму** в заголовку — **rich** (акцентний колір) або **simple** (приглушений) — показує який режим активний для поточного об'єкта.

## Режими редактора

### Режим rich

Використовується: **Text** (мітки MTEXT) та анотації **Multileader**.

| Функція | Поведінка |
|---------|-----------|
| Bold / Italic / Strikethrough | Для окремих символів (застосовується до виділення або до всього об'єкта без виділення) |
| Font та Height | Перевизначення для окремих символів або стандарт всього об'єкта |
| `Enter` | Вставляє тверде перенесення рядка |
| `Shift+←/→` | Розширює або зменшує виділення тексту |
| `Home` / `End` | Перехід на початок / кінець поточного твердого рядка |
| Перенос слів | Підтримується через ручки зміни ширини референса |

### Режим simple

Використовується: **Dimension Linear**, **Dimension Aligned**, **Dimension Angular**, **Dimension Radius**, **Dimension Diameter**.

Редактор заздалегідь заповнений поточною відрендереною міткою розміру, тому ви можете розмістити курсор і редагувати значення безпосередньо.

| Функція | Поведінка |
|---------|-----------|
| Bold / Italic / Font / Height | Доступно — застосовується до **всієї** мітки одночасно |
| Форматування окремих символів | Не підтримується |
| `Enter` | **Підтверджує** значення та закриває редактор (без переносу рядка) |
| Багаторядковість | Не підтримується |
| Перенос слів | Не підтримується |

## Відкриття редактора

| Дія | Результат |
|-----|-----------|
| Команда `text` → клацніть позицію | Створює новий текстовий об'єкт і відкриває редактор (**rich**) |
| Двічі клацніть існуючий об'єкт **Text** | Знову відкриває редактор у режимі **rich** |
| Двічі клацніть існуючий **Multileader** | Відкриває редактор у режимі **rich** |
| Двічі клацніть об'єкт **розміру** | Відкриває редактор у режимі **simple** |
| `Escape` всередині редактора | Закриває редактор і зберігає всі зміни |

## Панель інструментів

Панель інструментів ширяє над обмежувальною рамкою тексту та залишається прив'язаною до об'єкта під час панорамування або масштабування.

### Bold · Italic · Strikethrough

| Кнопка | Скорочення | Що робить |
|--------|-----------|-----------|
| **B** | — | Перемкнути жирний |
| *I* | — | Перемкнути курсив |
| ~~S~~ | — | Перемкнути закреслений |

**Як застосовується перемикання:**

- **З виділенням тексту** — стиль застосовується лише до виділених символів.
- **Без виділення, курсор у існуючому тексті** — перемикає стиль на всьому об'єкті (всіх сегментах).
- **Порожній текст або новий об'єкт** — стиль зберігається на порожньому сегменті та застосовується до кожного символу введеного з цього моменту.

Кнопка відображається підсвіченою (активною) коли кожен символ у поточному виділенні — або символ безпосередньо ліворуч від курсора — має встановлений цей стиль.

### Font

Спадний список містить ретельно підібраний набір кросплатформних гарнітур (Helvetica, Times New Roman, Georgia, Courier New, Verdana, Tahoma, Trebuchet MS, Lucida Console, Impact). Верхня опція **Arial** є стандартною.

- **З виділенням** — перевизначає шрифт лише для виділених символів.
- **Без виділення** — застосовує шрифт до всього об'єкта.

Спадний список відображає шрифт символу ліворуч від курсора коли немає виділення.

### Height

Числове поле встановлює **висоту капіталі** (висота великої літери) в одиницях кресленика.

- **З виділенням** — перевизначає висоту для виділених символів, незалежно від базової висоти об'єкта.
- **Без виділення** — змінює базову висоту об'єкта (застосовується до всіх символів без індивідуального перевизначення висоти).

Поле відображає висоту символу ліворуч від курсора. Залиште порожнім для використання стандарту об'єкта.

## Курсор та навігація

| Клавіша | Дія |
|---------|-----|
| `←` / `→` | Перемістити курсор на один символ ліворуч або праворуч |
| `Home` | Перейти на початок поточного твердого рядка |
| `End` | Перейти на кінець поточного твердого рядка |
| `Shift` + `←` / `→` | Розширити або зменшити виділення |
| `Backspace` | Видалити символ ліворуч (або виділення) |
| `Delete` | Видалити символ праворуч (або виділення) |
| `Enter` | Вставити перенесення рядка |
| `Escape` | Закрити редактор |

Висота курсора автоматично відповідає висоті капіталі сусіднього символу, включаючи менший розмір для нижнього та верхнього індексів.

## Перенос слів

Коли текстовий об'єкт має встановлену **ширину референса**, довгі рядки м'яко переносяться на межах слів щоб вміщуватися в цій ширині.

Для встановлення або зміни ширини референса поки об'єкт вибраний, перетягніть **ручки зміни розміру** — тонкі прямокутники на лівому та правому краях пунктирної обмежувальної рамки. Вміст перетікає в реальному часі під час перетягування.

Встановлення ширини референса в нуль (перетягніть ручки разом або видаліть значення в панелі властивостей) прибирає перенос слів і дозволяє рядкам зростати вільно.

## Багаторядковий текст

Натисніть `Enter` для вставлення твердого переносу рядка. Кожен твердий рядок є незалежним — `Home` та `End` навігують лише в межах поточного твердого рядка.

Тверді переноси рядків та форматування окремих символів зберігаються у форматі MTEXT і виживають повний DXF round-trip.

## Сумісність з DXF

Текстові об'єкти зберігаються як **MTEXT** у файлах DXF. Bold та Italic кодуються як `\L`, `\K`, `\O` та вбудовані перемикачі шрифту (`\f`). Висота на символ кодується як `\H`. Все форматування зберігається при експорті та читається програмами LibreCAD, FreeCAD та іншими DXF-сумісними програмами.
