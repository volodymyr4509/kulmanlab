# Сітка та прив'язка

**Сітка** відображає регулярні опорні лінії на полотні як візуальні орієнтири — аналогічно крапкам або лініям на міліметровому папері. **Прив'язка** (об'єктна прив'язка) автоматично притягує курсор до точних геометричних точок на існуючих об'єктах.

Обидва параметри **незалежні** — ви можете мати сітку без прив'язки, прив'язку без сітки або обидва разом.

## Вмикання та вимикання

| Параметр | Як перемкнути |
|----------|--------------|
| Сітка | Кнопка **Grid** на панелі інструментів або клавіша `G` |
| Прив'язка | Кнопка **Snap** на панелі інструментів або клавіша `S` |

## Сітка

Коли сітка увімкнена, полотно показує регулярну сітку ліній або точок. Відстань між лініями сітки адаптується до поточного масштабу — при наближенні проміжки стають дрібнішими, при відвернення — крупнішими.

Сітка є **лише візуальним орієнтиром** — курсор не прив'язується до ліній сітки автоматично (за це відповідає окрема функція прив'язки до сітки, якщо вона увімкнена).

## Прив'язка до об'єктів

Коли прив'язка увімкнена, курсор автоматично притягується до найближчої геометричної точки при наближенні до неї під час виконання команди:

| Тип прив'язки | Що виявляє |
|--------------|------------|
| **Кінцева точка** | Кінці ліній, вершини поліліній, кінці дуг |
| **Середня точка** | Середина відрізка лінії або полілінії |
| **Центр** | Центр кола або дуги |
| **Перетин** | Точка, де перетинаються дві лінії або об'єкти |
| **Квадрант** | Кардинальні точки кола (ліво, право, верх, низ) |

При прив'язці до точки з'являється символ маркера прив'язки (наприклад, маленький квадрат для кінцевої точки, трикутник для середньої точки).

## Блокування кута vs. прив'язка до об'єктів

| | Блокування кута (без прив'язки) | Прив'язка до об'єктів |
|---|-------------------------------|----------------------|
| Що виявляє | Кути, кратні 45° від поточної точки | Точні геометричні точки на існуючих об'єктах |
| Активується | Переміщення курсора вздовж осі | Наближення до існуючого об'єкта |
| Доступно | Завжди для Line та Polyline | Лише коли увімкнено |

## Поради для точного креслення

1. **Прив'язка + введення координат**: прив'язка до кінцевої точки для початку лінії, потім введення точної довжини.
2. **Вимикання тимчасово**: якщо прив'язка заважає (наприклад, при розміщенні тексту), натисніть `S` для тимчасового вимкнення.
3. **Сітка як орієнтир**: увімкніть сітку для перевірки вирівнювання об'єктів при роботі зі стандартними розмірами.
