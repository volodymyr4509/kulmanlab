# Offset

Команда `offset` створює паралельну копію об'єкта на фіксованій перпендикулярній відстані. Ви вводите відстань один раз, потім клацаєте об'єкти і вибираєте сторону — команда залишається готовою з тією ж відстанню, тому можна зміщувати кілька об'єктів за одну сесію.

Підтримувані типи об'єктів: **Line, Circle, Arc, Ellipse, Polyline** (включно з Rectangle).

## Використання Offset

1. Введіть `offset` у терміналі або натисніть кнопку **Offset** на панелі інструментів.
2. **Введіть відстань зміщення** та натисніть **Enter** або **Пробіл**.
3. **Клацніть об'єкт** для зміщення — якщо об'єкт не підтримуваного типу, з'являється повідомлення про помилку і можна клацнути інший об'єкт.
4. **Перемістіть курсор** на ту сторону де повинна з'явитися копія — живий попередній перегляд слідує за курсором.
5. **Клацніть** для розміщення копії зі зміщенням.

Після кожного розміщення команда повертається до кроку 3 з **тією ж відстанню**, готова до наступного зміщення. Натисніть **Escape** для скидання до кроку введення відстані.

```
  Відстань: 10

  ─────────────────    ← оригінальна лінія
  ─────────────────    ← зміщена копія (10 одиниць нижче)
```

## Поведінка зміщення для кожного типу об'єкта

| Об'єкт | Як обчислюється зміщення |
|--------|--------------------------|
| **Line** | Паралельна лінія зміщена перпендикулярно до оригінального напрямку |
| **Circle** | Концентричне коло; клік назовні → більший радіус, всередині → менший радіус |
| **Arc** | Концентрична дуга з новим радіусом; той самий кутовий охват збережений |
| **Ellipse** | Обидві піввісі збільшені або зменшені на однакову відстань |
| **Polyline** | Кожен сегмент зміщується незалежно; сусідні зміщені сегменти з'єднуються в кутах |

Для **Circle**, **Arc** та **Ellipse**: якщо зміщення всередину зменшило б будь-який радіус або піввісь до нуля або нижче, зміщення не застосовується.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.` | Додати цифру до значення відстані |
| `Backspace` | Видалити останній введений символ |
| `Enter` / `Пробіл` | Підтвердити введену відстань та перейти до вибору об'єкта |
| `Escape` | Скинути до кроку введення відстані |

## Примітка про робочий процес

Відстань залишається встановленою до натискання **Escape**. Це робить зміщення багатьох об'єктів на однаковому інтервалі ефективним — введіть відстань один раз, потім клацайте та вибирайте сторону для кожного об'єкта по черзі.

## Offset проти Copy

| | Offset | Copy |
|---|--------|------|
| Зміщення | Перпендикулярно до геометрії об'єкта | Довільний вектор (база → призначення) |
| Підтримувані об'єкти | Line, Circle, Arc, Ellipse, Polyline | Всі типи об'єктів |
| Введення відстані | Вводиться до вибору об'єкта | Вводиться або клацається після вибору |
| Найкраще для | Паралельні лінії, концентричні кола, вставлені/зовнішні шляхи | Розміщення дублікатів на довільних позиціях |
