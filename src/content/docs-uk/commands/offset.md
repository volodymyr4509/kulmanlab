# Offset

Команда `offset` створює паралельну копію об'єкта на рівній перпендикулярній відстані. Спочатку вводиться відстань, потім клацається об'єкт та вказується бік зміщення.

## Як використовувати

1. Введіть `offset` у терміналі або натисніть кнопку **Offset** на панелі інструментів.
2. **Введіть відстань зміщення** та натисніть **Enter**.
3. **Клацніть об'єкт** для зміщення.
4. **Клацніть бік** — на який бік від об'єкта розміщується копія.
5. Команда залишається активною — клацайте більше об'єктів або натисніть **Escape** для виходу.

## Поведінка per-об'єкт

| Об'єкт | Результат зміщення |
|--------|-------------------|
| **Лінія** | Паралельна лінія рівної довжини на перпендикулярній відстані |
| **Полілінія** | Паралельна полілінія — кожен відрізок зміщується, кути авто-перетинаються |
| **Коло** | Концентричне коло з більшим або меншим радіусом |
| **Дуга** | Концентрична дуга з тим самим кутовим охватом |
| **Еліпс** | Паралельний еліпс зі зміненим співвідношенням осей |

## Захист від нульового радіуса

Для **Кіл** та **Дуг** при зміщенні всередину: якщо відстань перевищує радіус (результат мав би від'ємний радіус), команда **ігнорує операцію** та не створює об'єкт.

## Збереження відстані

Після введення відстані для першого зміщення вона **зберігається**. Наступний клік на об'єкт використовує ту ж відстань — не потрібно вводити її знову. Введіть нове значення, щоб змінити відстань.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.` | Додати цифру до відстані |
| `Backspace` | Видалити останній символ |
| `Enter` | Підтвердити відстань |
| `Escape` | Скасувати та вийти |

## Offset проти Copy

| | Offset | Copy |
|---|--------|------|
| Позиція копії | Паралельно на точну відстань | Зміщено на вектор |
| Підтримувані типи | Line, Polyline, Circle, Arc, Ellipse | Всі типи |
| Геометрія копії | Може відрізнятися (для кіл/еліпсів) | Ідентична до оригіналу |
