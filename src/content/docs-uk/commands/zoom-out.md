# Zoom Out

Команда `zoom out` зменшує масштаб огляду на фіксований **крок 1.5×** (ділить масштаб на 1.5), відображаючи більшу частину кресленика.

## Як запустити

- Натисніть кнопку **Zoom Out** на панелі інструментів — зменшує навколо **центру вікна**.
- **Прокрутіть колесо миші вниз** — зменшує навколо **позиції курсора** (зазвичай зручніше).

> Примітка: немає клавіатурних ярликів для Zoom Out. Для зуму використовуйте коліщатко миші.

## Крок 1.5× (поділ)

| Поточний масштаб | Після Zoom Out |
|-----------------|----------------|
| 1× | 0.667× |
| 0.667× | 0.444× |
| 0.444× | 0.296× |
| … | … |
| ~0.015× | 0.01× (мінімум) |

Мінімальний масштаб: **0.01×** — подальший Zoom Out не змінює масштаб.

## Zoom Out через кнопку проти Zoom через коліщатко

| | Кнопка Zoom Out | Прокрутка коліщатком вниз |
|---|-----------------|---------------------------|
| Центр зуму | Центр вікна | Позиція курсора |
| Крок | ÷1.5 | ÷1.5 |
| Доступна під час команди | Так | Так |
