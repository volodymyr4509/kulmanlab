# Zoom Out

Команда `zoomout` ділить поточний рівень масштабу на **1.5×** (еквівалентно множенню на ~0.667) і виходить негайно, центруючись по середині вікна перегляду. Це інверсія [Zoom In](./zoom-in).

## Зменшення масштабу

Натисніть кнопку **Zoom Out** на панелі інструментів або введіть `zoomout` у терміналі. Масштаб застосовується миттєво і команда виходить — клацання на полотні не потрібне.

## Як працює крок 1.5×

| Поточний масштаб | Після одного Zoom Out |
|-----------------|----------------------|
| 1.50× | 1.00× |
| 2.25× | 1.50× |
| 10.00× | 6.67× |
| 0.015× | 0.01× (мінімум) |

Рівень масштабу завжди показується у **правому нижньому куті** полотна. Нижня межа — **0.01×**; подальші кроки нічого не змінюють.

## Кнопка Zoom Out проти колеса прокрутки

| | Кнопка Zoom Out | Колесо прокрутки |
|---|----------------|-----------------|
| Центр масштабу | Середина вікна перегляду | Позиція курсора |
| Розмір кроку | 1.5× за клік | ~1.1× за клік |
| Потрібна активація | Ні | Ні — працює завжди |
| Найкраще для | Крок назад для перегляду більшого контексту | Плавне зменшення масштабу з прив'язкою до курсора |

## Довідник клавіатури

Немає клавіатурних ярликів для цієї команди. Натомість використовуйте колесо прокрутки — воно працює в будь-який момент без активації будь-якої команди.

## Пов'язані команди перегляду

| Команда | Що робить |
|---------|----------|
| [Zoom In](./zoom-in) | Множить масштаб на 1.5× за крок |
| [Fit](./fit) | Скидає масштаб для показу всіх об'єктів |
| [Pan](./pan) | Зміщує вікно перегляду без масштабування |
