# Array Grid

Команда `ArrayGrid` створює прямокутну сітку копій вибраних об'єктів — введіть кількість рядків, кількість стовпців та відстань між ними, все набрано в терміналі. Початковий вибір займає комірку рядок 0, стовпець 0; кожна інша комірка є перенесеною копією.

## Два способи запуску

**Вибрати спочатку, потім масив** — виберіть об'єкти, потім активуйте:

1. Виберіть один або більше об'єктів на полотні.
2. Введіть `arraygrid` у терміналі (достатньо `arr` — це однозначно) або натисніть кнопку **Array Grid** на панелі інструментів.
3. Введіть кількість **рядків** та натисніть **Enter**.
4. Введіть кількість **стовпців** та натисніть **Enter**.
5. Введіть **відстань між рядками** та натисніть **Enter**.
6. Введіть **відстань між стовпцями** та натисніть **Enter** — сітка створюється негайно.

**Активувати, потім вибрати** — запустіть команду без вибору:

1. Введіть `arraygrid` або натисніть кнопку на панелі інструментів.
2. **Виберіть об'єкти** — клацайте для перемикання окремих об'єктів або перетягуйте для вибору рамкою.
3. Натисніть **Enter** або **Пробіл** для підтвердження вибору.
4. Продовжте з рядки → стовпці → відстань між рядками → відстань між стовпцями, як вище.

```
  2 рядки x 3 стовпці:

  [B] [B] [B]   <- рядок 1 (перенесені копії)
  [A] [A] [A]   <- рядок 0: початковий вибір, копії праворуч
```

> Терміналу потрібно лише стільки літер, скільки достатньо для однозначності — введення `arr` і натискання **Enter** активує Array Grid напряму, оскільки жодна інша назва команди не починається з цих трьох літер (Arc, Area, Align і Angle розходяться раніше).

## Рядки, стовпці та відстань

| Запит | Приймає | Примітки |
|-------|---------|----------|
| Рядки | Додатні цілі числа (1, 2, 3…) | Лише цифри — без десяткової крапки чи знака |
| Стовпці | Додатні цілі числа (1, 2, 3…) | Лише цифри — без десяткової крапки чи знака |
| Відстань між рядками | Число зі знаком (напр. `10`, `-5.5`) | Відстань між рядками; від'ємне значення змінює напрямок |
| Відстань між стовпцями | Число зі знаком (напр. `10`, `-5.5`) | Відстань між стовпцями; від'ємне значення змінює напрямок |

З 1 рядком і 1 стовпцем копії не створюються — команда завершується без зміни креслення.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `Enter` / `Пробіл` | Підтвердити вибір та перейти до запиту рядків |
| `0`–`9` | Ввести цифри для рядків або стовпців |
| `0`–`9`, `.`, `-` | Ввести цифри для відстані між рядками/стовпцями (`-` лише як перший символ) |
| `Backspace` | Видалити останній введений символ поточного запиту |
| `Enter` | Підтвердити поточний запит та перейти до наступного |
| `Escape` | Очистити введені значення рядків/стовпців/відстані та повернутися до фази вибору |

## Вибір об'єктів під час команди

| Метод | Поведінка |
|-------|-----------|
| **Клік** | Перемикає об'єкт під курсором у/з вибору |
| **Перетягнути праворуч** (строгий) | Додає об'єкти, що повністю знаходяться в рамці |
| **Перетягнути ліворуч** (перехресний) | Додає об'єкти, що перетинають межу рамки |
| **Enter** / **Пробіл** | Підтверджує вибір та переходить до запиту рядків |

## Після масиву

Нові копії додаються до кресленика і команда завершується — початковий вибір скидається. Запустіть **Array Grid** знову, або почніть нову команду.

## Array Grid проти Copy

| | Array Grid | Copy |
|---|-----------|------|
| Вибір точок | Немає — рядки, стовпці та відстань вводяться | Базова точка та точка призначення клацаються (або вводяться) |
| Створені копії | Рядки × стовпці − 1 | Рівно 1 за операцію копіювання |
| Розташування | Регулярна прямокутна сітка | Будь-де, з будь-яким зміщенням |
| Найкраще для | Повторення елемента за регулярним шаблоном (отвори, плитка, кріплення) | Один дублікат у довільній позиції |

## Підтримувані об'єкти

Array Grid працює з усіма типами об'єктів. Всі об'єкти реалізують `translate(dx, dy)` внутрішньо — ту саму операцію, яку використовують [Copy](../copy/) та [Move](../move/) — тому жоден не виключений.
