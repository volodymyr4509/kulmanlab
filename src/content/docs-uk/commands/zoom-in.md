# Zoom In

Команда `zoom in` збільшує масштаб огляду на фіксований **крок 1.5×**, наближаючи кресленик. Масштаб показується у правому нижньому куті.

## Як запустити

- Натисніть кнопку **Zoom In** на панелі інструментів — збільшує навколо **центру вікна**.
- **Прокрутіть колесо миші вгору** — збільшує навколо **позиції курсора** (зазвичай зручніше).

> Примітка: немає клавіатурних ярликів для Zoom In. Для зуму використовуйте коліщатко миші.

## Крок 1.5×

| Поточний масштаб | Після Zoom In |
|-----------------|---------------|
| 1× | 1.5× |
| 1.5× | 2.25× |
| 2.25× | 3.375× |
| … | … |
| ~6 667× | 10 000× (максимум) |

Максимальний масштаб: **10 000×** — подальший Zoom In не змінює масштаб.

## Zoom In через кнопку проти Zoom через коліщатко

| | Кнопка Zoom In | Прокрутка коліщатком |
|---|----------------|---------------------|
| Центр зуму | Центр вікна | Позиція курсора |
| Крок | 1.5× | 1.5× |
| Доступна під час команди | Так | Так |
