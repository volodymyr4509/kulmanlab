# Delete

Команда `delete` видаляє всі поточно вибрані об'єкти з кресленика. Дія скасовується командою [Undo](./undo).

## Три способи видалення

**Вибрати спочатку, потім видалити** (найшвидший):
1. Виберіть об'єкти (клацніть або перетягніть рамку).
2. Натисніть клавішу **Delete** на клавіатурі.

**Через панель інструментів або термінал**:
1. Виберіть об'єкти.
2. Введіть `delete` у терміналі або натисніть кнопку **Delete**.

**Активувати, потім вибрати**:
1. Введіть `delete`.
2. Виберіть об'єкти.
3. Натисніть **Enter** або **Пробіл** для підтвердження.

## Клавіша Delete — ярлик

Якщо на полотні є вибрані об'єкти, натискання клавіші **Delete** (або **Backspace**) миттєво їх видаляє без введення команди в термінал.

## Скасування видалення

Використовуйте [Undo](./undo) (Ctrl+Z / Cmd+Z) одразу після видалення для відновлення об'єктів.

## Вибір об'єктів під час команди

| Метод | Поведінка |
|-------|-----------|
| **Клік** | Перемикає об'єкт під курсором |
| **Перетягнути праворуч** (строгий) | Додає об'єкти, що повністю знаходяться в рамці |
| **Перетягнути ліворуч** (перехресний) | Додає об'єкти, що перетинають межу рамки |
| **Enter** / **Пробіл** | Підтверджує вибір та виконує видалення |

## Підтримувані об'єкти

Delete працює з усіма типами об'єктів: Лінія, Полілінія, Прямокутник, Коло, Дуга, Еліпс, Текст, Сплайн, Розміри, Виноски та всі інші.
