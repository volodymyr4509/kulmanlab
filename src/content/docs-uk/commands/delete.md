# Delete

Команда `delete` видаляє вибрані об'єкти з кресленика. Видалення записуються в історію [Undo](../undo/) і можуть бути скасовані до 20 кроків. Немає окремого діалогу підтвердження видалення — підтвердження є одним натисканням клавіші.

## Два способи видалення

**Вибрати спочатку, потім видалити** — найшвидший шлях:

1. Виберіть один або більше об'єктів на полотні.
2. Введіть `delete` у терміналі, натисніть кнопку **Delete** на панелі інструментів, **або натисніть клавішу `Delete`** безпосередньо.

Об'єкти видаляються миттєво — без додаткового кроку підтвердження.

**Активувати, потім вибрати**:

1. Введіть `delete` або натисніть кнопку панелі інструментів (без виділених об'єктів).
2. **Виберіть об'єкти** — клацніть для перемикання, або перетягніть для вибору областю.
3. Натисніть **Enter**, **Пробіл** або **Delete** для підтвердження та видалення вибраних об'єктів.

## Ярлик клавіші Delete

Клавіша `Delete` на клавіатурі діє як **глобальний ярлик** — якщо будь-які об'єкти поточно вибрані, натискання її видаляє їх негайно, навіть без відкриття команди Delete у терміналі. Це найшвидший однокроковий робочий процес видалення:

```
Клацнути об'єкт → натиснути клавішу Delete → готово
```

## Вибір під час команди

| Метод | Поведінка |
|-------|-----------|
| **Клік** | Перемикає об'єкт під курсором у/з вибору |
| **Перетягнути праворуч** (строгий) | Вибирає лише об'єкти повністю всередині рамки |
| **Перетягнути ліворуч** (перехресний) | Вибирає об'єкти що перетинають межу рамки |
| **Enter** / **Пробіл** / **Delete** | Підтверджує та видаляє вибрані об'єкти |

## Відновлення видалених об'єктів

Видалення можна скасувати командою [Undo](../undo/) (введіть `undo` або скористайтеся кнопкою панелі інструментів). До **20 кроків** можна скасувати на кожен файл, і історія зберігається при перезавантаженнях сторінки. Якщо ви перевищили 20 видалень без збереження, ранніші видалення не можна відновити.

## Підтримувані об'єкти

Delete працює з кожним типом об'єктів — Line, Polyline, Rectangle, Circle, Arc, Ellipse, Text, Spline, Dimension, Leader та всіма іншими.
