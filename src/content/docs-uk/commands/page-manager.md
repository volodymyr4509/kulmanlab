# PageManager

Команда `page manager` відкриває менеджер сторінок для налаштування параметрів активного паперового компонування: розміру паперу, орієнтації (портрет/альбом) та масштабу.

## Відкриття менеджера сторінок

Введіть `page manager` у терміналі або натисніть кнопку **PageManager** — відкривається панель налаштувань компонування.

## Доступні налаштування

| Параметр | Опис |
|----------|------|
| Розмір паперу | Стандартні розміри (A4, A3, A2, A1, A0, Letter тощо) або власний |
| Орієнтація | Портрет або альбом |
| Масштаб | Масштаб простору моделі у компонуванні (наприклад, 1:50, 1:100) |

## Паперові компонування

KulmanLab CAD підтримує кілька паперових компонувань на одному кресленику. Кожне компонування може мати власний розмір паперу та масштаб.

Для переключення між компонуваннями використовуйте вкладки внизу екрана.
