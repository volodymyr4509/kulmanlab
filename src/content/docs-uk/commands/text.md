# Text

Команда `text` розміщує багаторядкову текстову мітку. Після клацання позиції на полотні відкривається спливаючий редактор у режимі **rich** — можна вводити вміст, застосовувати жирний/курсив/закреслений для окремих символів, змінювати шрифти та висоти, а також вставляти переноси рядків. Натисніть **Escape** для підтвердження та закриття редактора.

Дивіться сторінку [Текстовий редактор](../../interface/text-editor/) для повного довідника редактора, включаючи порівняння режимів **rich** та **simple**.

## Розміщення текстової мітки

1. Введіть `text` у терміналі або натисніть кнопку **Text** на панелі інструментів.
2. **Клацніть позицію прив'язки** на полотні. Або введіть `X,Y` та натисніть **Enter** для точної координати.
3. Відкривається **спливаючий текстовий редактор** над новою міткою. Введіть вміст.
4. Натисніть **Escape** для підтвердження мітки та закриття редактора.

Стандартна висота — **12 одиниць кресленика**.

## Редагування існуючої мітки

**Двічі клацніть** будь-яку текстову мітку на полотні для повторного відкриття редактора для цієї мітки.

## Введення координат для прив'язки

Замість клацання, введіть точну позицію:

1. Введіть значення X.
2. Натисніть `,` — термінал покаже `[X], [Y{cursor}]`.
3. Введіть значення Y.
4. Натисніть **Enter** для розміщення прив'язки та відкриття редактора.

## Довідник клавіатури

**Фаза прив'язки**

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.`, `-` | Почати введення координати X |
| `,` | Заблокувати X та перейти до введення Y |
| `Backspace` | Видалити останній введений символ |
| `Enter` | Підтвердити введену координату |

**Фаза текстового редактора** (дивіться [Текстовий редактор](../../interface/text-editor/) для повного довідника)

| Клавіша | Дія |
|---------|-----|
| Будь-який друкований символ | Вставити в позиції курсора |
| `Backspace` / `Delete` | Видалити суміжний символ або виділення |
| `Enter` | Вставити перенос рядка |
| `←` / `→` | Перемістити курсор |
| `Home` / `End` | Перейти на початок / кінець твердого рядка |
| `Escape` | Підтвердити та закрити редактор |

## Редагування ручками — зміна позиції

Вибрана текстова мітка показує одну ручку в точці прив'язки:

| Ручка | Розташування | Що робить |
|-------|-------------|-----------|
| **Прив'язка** | Нижній лівий кут тексту | Перетягнути для зміни позиції мітки |

## Вибір тексту

| Метод | Поведінка |
|-------|-----------|
| **Клік** | Вибирає якщо клік потрапляє в повернуту обмежувальну рамку тексту |
| **Перетягнути праворуч** (строгий) | Всі чотири кути обмежувальної рамки мають знаходитися всередині рамки вибору |
| **Перетягнути ліворуч** (перехресний) | Будь-яке перекриття між обмежувальною рамкою тексту та рамкою вибору вибирає його |

## Підтримувані команди редагування

| Команда | Що відбувається з текстом |
|---------|--------------------------|
| [Move](../move/) | Переміщує точку прив'язки |
| [Copy](../copy/) | Створює ідентичну мітку на новій позиції |
| [Rotate](../rotate/) | Обертає позицію прив'язки та додає кут до Кута повороту |
| [Mirror](../mirror/) | Відображає точку прив'язки через вісь дзеркала (рядок тексту не перевертається) |
| [Scale](../scale/) | Масштабує позицію прив'язки та множить висоту на коефіцієнт масштабу |
| [Delete](../delete/) | Видаляє мітку |

Текст **не підтримує** Offset, Trim та Extend.

## Властивості

Коли вибрана текстова мітка, панель властивостей показує:

**Загальні**

| Властивість | Стандарт | Значення |
|-------------|---------|---------|
| Колір | 256 (ByLayer) | Індекс кольору ACI |
| Шар | `0` | Призначення шару |

**Геометрія**

| Властивість | Значення |
|-------------|---------|
| Позиція X / Позиція Y | Координати точки прив'язки |
| Висота | Базова висота тексту в одиницях кресленика (стандарт: **12**) |
| Кут повороту | Поворот проти годинникової стрілки в градусах |

**Властивості**

| Властивість | Значення |
|-------------|---------|
| Вміст | Рядок тексту (вбудовані коди MTEXT збережені) |
| Точка прив'язки | Код вирівнювання (1 = верхній лівий … 9 = нижній правий) |

Текст не має властивостей Тип лінії, Масштаб типу лінії або Товщина.

## DXF — об'єкт MTEXT

Текстові мітки зберігаються як об'єкти **MTEXT** у файлі DXF. Жирний та курсив кодуються за допомогою `\L`, `\K`, `\O` та вбудованих перемикань шрифту (`\f`). Висота на символ кодується як `\H`. Все форматування виживає повний DXF round-trip і читається програмами LibreCAD, FreeCAD та іншими DXF-сумісними програмами.
