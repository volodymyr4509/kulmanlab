# Text

Команда `text` розміщує текстовий об'єкт (**MTEXT**) на кресленику. Клацніть точку прив'язки, введіть текст та натисніть Enter для підтвердження.

## Розміщення тексту

1. Введіть `text` у терміналі або натисніть кнопку **Text** на панелі інструментів.
2. **Клацніть точку прив'язки** (базову точку вирівнювання) або введіть `X,Y` та натисніть **Enter**.
3. **Введіть текст** у відображений вбудований редактор.
4. Натисніть **Enter** або клацніть за межами тексту для підтвердження та розміщення.

Натисніть **Escape** для відміни без розміщення тексту.

## Введення координат для точки прив'язки

1. Введіть значення X.
2. Натисніть `,` — термінал показує `[X], [Y{cursor}]`.
3. Введіть значення Y.
4. Натисніть **Enter** для підтвердження.

## Довідник клавіатури — Фаза точки прив'язки

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.`, `-` | Почати введення координати X |
| `,` | Заблокувати X та перейти до введення Y |
| `Backspace` | Видалити останній введений символ |
| `Enter` | Підтвердити координату |
| `Escape` | Скасувати |

## Довідник клавіатури — Фаза редактора тексту

| Клавіша | Дія |
|---------|-----|
| Будь-який символ | Додати до тексту |
| `←` / `→` | Перемістити курсор |
| `Backspace` | Видалити символ ліворуч |
| `Delete` | Видалити символ праворуч |
| `Enter` | Підтвердити та розмістити текст |
| `Escape` | Скасувати |

## Редагування ручками

Вибраний текст показує **одну ручку** у точці прив'язки. Перетягніть її для переміщення тексту.

## Вибір тексту

Клацання по **обмежувальній рамці** тексту вибирає його (не потрібно клацати по гліфах). Клацання за межами рамки не вибирає.

## Підтримувані команди редагування

| Команда | Що відбувається |
|---------|-----------------|
| [Move](./move) | Переміщує точку прив'язки |
| [Copy](./copy) | Створює копію тексту |
| [Rotate](./rotate) | Обертає текст навколо точки прив'язки |
| [Mirror](./mirror) | Відображає позицію тексту (рядок не перевертається) |
| [Scale](./scale) | Масштабує позицію від базової точки |
| [Delete](./delete) | Видаляє текст |

## Властивості

| Властивість | Значення |
|-------------|----------|
| Колір | Колір тексту |
| Шар | Шар до якого належить |
| Позиція X / Y | Координати точки прив'язки |
| Висота | Висота тексту в одиницях кресленика |
| Кут повороту | Кут нахилу в градусах |
| Вміст | Текстовий рядок |
| Точка прив'язки | Верхній-лівий, Верхній-центр, тощо |

## DXF — об'єкт MTEXT

Текст зберігається як об'єкт `MTEXT` у DXF файлі. Вміст, точка вставки, висота, кут повороту та точка прив'язки проходять round-trip без втрат.
