# Dimension Radius

Команда `dimradius` розміщує розмір радіуса на дузі або колі. Лінія розміру іде від центру до точки на окружності в напрямку курсора, позначаючись `R <значення>`. Для вимірювання повного діаметра замість цього використовуйте [Dimension Diameter](../dim-diameter/).

## Анатомія розміру радіуса

```
  ● (центр)
   \
    \  R 5.00
     \
      ●────── текст (сторона курсора)
   (точка дуги)
```

- **Лінія розміру** — від центру через точку дуги в напрямку курсора, зі стрілкою в точці дуги.
- **Ярлик** — `R` з наступним значенням радіуса.

## Розміщення розміру радіуса

1. Введіть `dimradius` у терміналі або натисніть кнопку **Dimension Radius** на панелі інструментів.
2. **Клацніть дугу або коло** для вибору.
3. **Перемістіть курсор** для орієнтації лінії розміру — точка дуги відстежує напрямок курсора від центру.
4. **Клацніть** для розміщення розміру.

Можна вибрати лише об'єкти типу **Arc** та **Circle**. Клацання будь-якого іншого типу об'єкта не виконує жодної дії.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `Escape` | Скасувати |

## Dimension Radius проти Dimension Diameter

| | Dimension Radius | Dimension Diameter |
|---|-----------------|-------------------|
| Що вимірює | Радіус (від центру до краю) | Діаметр (від краю до краю через центр) |
| Лінія розміру | Від центру до точки дуги | Від точки дуги до точки дуги (через центр) |
| Префікс ярлика | `R` | `⌀` |
| Стрілки | Одна (в точці дуги) | Дві (в обох точках дуги) |
| Найкраще для | Розмір однієї сторони криволінійного елемента | Розмір повних кругових елементів |

## Редагування мітки — simple mode

**Двічі клацніть** розміщений розмір радіуса, щоб відкрити текстовий редактор у режимі **simple**. Редактор заздалегідь заповнений поточним відрендеренним значенням (наприклад, `R 5.00`), тому ви можете розмістити курсор і редагувати його безпосередньо.

| Функція | Поведінка |
|---------|-----------|
| Bold / Italic / Font / Height | Застосовується до **всієї** мітки одночасно |
| Форматування окремих символів | Не підтримується |
| `Enter` | Підтверджує значення та закриває редактор |
| Багаторядковість | Не підтримується |

Дивіться [Текстовий редактор — simple mode](../../interface/text-editor/#simple-mode) для повного довідника.

## DXF — об'єкт DIMENSION з типом радіуса

Розміри радіуса зберігаються як об'єкти `DIMENSION` з геометрією типу радіуса, зберігаючи координати центру, позицію точки дуги та виміряне значення радіуса. Всі властивості проходять round-trip без втрат.
