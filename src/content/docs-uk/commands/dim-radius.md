# Dimension Radius

Команда `dim radius` розміщує розмір, що відображає радіус кола або дуги. Значення розміру відображається з префіксом **R** (наприклад, R25).

## Анатомія розміру радіуса

```
             R25
  ┌──────────────→●  ← точка на дузі/колі
  ●  (центр)
```

Лінія розміру іде від центру кола/дуги до клікнутої точки на ній.

## Як додати розмір радіуса

1. Введіть `dim radius` у терміналі або натисніть кнопку **Dim Radius**.
2. **Клацніть коло або дугу** для вибору.
3. **Клацніть місце розміщення тексту розміру** — лінія розміру малюється від центру до клікнутої точки.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `Escape` | Скасувати та скинути |

## Dim Radius проти Dim Diameter

| | Dim Radius | Dim Diameter |
|---|------------|-------------|
| Що вимірює | Радіус (R) | Діаметр (Ø = 2R) |
| Застосовується до | Кола та дуги | Лише кола |
| Префікс значення | R | Ø |
| Лінія розміру | Центр → точка на колі | Точка → центр → протилежна точка |

## DXF — об'єкт DIMENSION

Зберігається як об'єкт `DIMENSION` з типом "radius". Повний round-trip у DXF-сумісних програмах.
