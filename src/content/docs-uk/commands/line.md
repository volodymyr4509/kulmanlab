# Line

Команда `line` малює окремі прямолінійні відрізки, які зберігаються як окремі об'єкти `LINE` в DXF моделі. Після кожного відрізка команда залишається активною та повторно використовує кінцеву точку як нову початкову, тому ви можете будувати з'єднані шляхи по одному відрізку за раз. На відміну від [Polylіne](../polyline/), ланцюгові лінії залишаються незалежними об'єктами — кожну з них можна обрізати, продовжити або видалити, не впливаючи на сусідні.

## Малювання ліній

1. Введіть `line` у терміналі або натисніть кнопку **Line** на панелі інструментів.
2. **Клацніть початкову точку** або введіть `X,Y` та натисніть **Enter** для точної координати.
3. **Клацніть кінцеву точку** — відрізок розміщується, і кінцева точка стає наступною початковою. Тут також працює введення координат.
4. Продовжуйте клацати (або вводити), щоб ланцюгувати більше відрізків.
5. Натисніть **Enter** або **Escape** для завершення.

```
  ●──────────●──────────●──────────●
 Початок  2-й клік  3-й клік   Enter для завершення
            (автоматично стає наступним початком)
```

Потрібен лише один відрізок? Натисніть **Enter** або **Escape** одразу після кроку 3.

## Введення координат

Замість клацання введіть точну позицію для початку або будь-якої наступної точки:

1. Введіть значення X (цифри, `.` або `-`).
2. Натисніть `,` — термінал покаже `[X], [Y{cursor}]`.
3. Введіть значення Y.
4. Натисніть **Enter** для розміщення точки.

## Блокування кута та точне введення довжини

Коли ви переміщуєте курсор після розміщення точки, команда відстежує вісь прив'язки під кутом 45° (0°, 45°, 90°, 135°, …). Кут **блокується**, коли:

- курсор знаходиться щонайменше на відстані **5 × розмір ручки** від якоря, **та**
- він знаходиться в межах **1 розміру ручки** перпендикулярної відстані від найближчої осі.

При блокуванні попередній перегляд прив'язується до осі, і ви можете ввести точну довжину:

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.` | Додати цифру до значення довжини |
| `-` | Від'ємна довжина — змінює напрямок вздовж осі (лише перший символ) |
| `Backspace` | Видалити останній введений символ |
| `Enter` | Розмістити кінцеву точку на введеній відстані |

Накопичене значення відображається у реальному часі в терміналі (наприклад, `click end point or enter length: 12.5`). Клік при заблокованій осі проєктується на вісь, тому кінцева точка завжди лежить точно на ній.

Повернення курсора близько до якоря вимикає блокування.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.`, `-` | Почати введення координати X або відстані при заблокованому куті |
| `,` | Заблокувати X та перейти до введення Y |
| `Backspace` | Видалити останній введений символ |
| `Enter` | Підтвердити введену координату або довжину, або завершити ланцюг якщо нічого не введено |
| `Escape` | Завершити ланцюг та вийти |

## Редагування ручками — розтягування кінцевих точок

Вибрана лінія показує три ручки:

| Ручка | Розташування | Що робить |
|-------|-------------|-----------|
| **Початок** | Перша кінцева точка | Перетягнути для переміщення — кінець залишається фіксованим |
| **Середина** | Центр лінії | Активує **Move** для всієї лінії |
| **Кінець** | Друга кінцева точка | Перетягнути для переміщення — початок залишається фіксованим |

Розтягування однієї кінцевої точки ніколи не впливає на іншу. Це відрізняється від редагування ручками [Polyline](../polyline/), де переміщення вершини змінює форму всього шляху.

## Вибір ліній

| Метод | Поведінка |
|-------|-----------|
| **Клік** | Вибирає лінію, якщо клік знаходиться в межах відстані виявлення відрізка |
| **Перетягнути праворуч** (строгий) | Лінія вибирається лише якщо обидві кінцеві точки знаходяться всередині рамки |
| **Перетягнути ліворуч** (перехресний) | Лінія вибирається, якщо будь-яка частина відрізка перетинає межу рамки |

## Підтримувані команди редагування

Лінії — **єдиний** об'єкт, на який діють [Trim](../trim/) та [Extend](../extend/). Усі стандартні команди перетворення також застосовуються:

| Команда | Що відбувається з лінією |
|---------|-------------------------|
| [Move](../move/) | Переміщує обидві кінцеві точки на однакове зміщення |
| [Copy](../copy/) | Створює ідентичну лінію на новій позиції |
| [Rotate](../rotate/) | Обертає обидві кінцеві точки навколо вибраної базової точки |
| [Mirror](../mirror/) | Відображає обидві кінцеві точки відносно дзеркальної осі |
| [Scale](../scale/) | Масштабує обидві кінцеві точки рівномірно від базової точки |
| [Offset](../offset/) | Створює паралельну лінію на фіксованій перпендикулярній відстані |
| [Trim](../trim/) | Обрізає лінію на перетинах — **лише лінії** |
| [Extend](../extend/) | Розтягує найближчу кінцеву точку до межі — **лише лінії** |
| [Delete](../delete/) | Видаляє лінію з кресленика |

## Властивості

Коли лінія вибрана, панель властивостей показує всі поля запису DXF `LINE`:

**Загальні**

| Властивість | За замовчуванням | Значення |
|-------------|-----------------|----------|
| Колір | 256 (ByLayer) | Індекс кольору ACI |
| Шар | `0` | Призначення шару |
| Тип лінії | ByLayer | Іменований шаблон типу лінії |
| Масштаб типу лінії | 1 | Коефіцієнт масштабу шаблону типу лінії |
| Товщина | 0 | Товщина екструзії |

**Геометрія**

| Властивість | Значення |
|-------------|----------|
| Початок X / Початок Y | Координати першої кінцевої точки |
| Кінець X / Кінець Y | Координати другої кінцевої точки |

Усі поля можна редагувати безпосередньо на панелі без повторного запуску команди.

## Line проти Polyline — коли що використовувати

| | Line | Polyline |
|---|------|---------|
| Кількість об'єктів | Один `LINE` на відрізок | Один `LWPOLYLINE` для всього шляху |
| Trim / Extend | Так — відрізок за відрізком | Ні |
| Замкнута форма | Ні | Так (прапор закриття) |
| Редагування ручками | Розтягування окремих кінцевих точок | Переміщення будь-якої вершини вздовж шляху |
| Найкраще для | Будівельні лінії, окремі відрізки, геометрія для обрізання | Контури, обриси, форми що зберігаються цілими |

## DXF — об'єкт LINE

Лінії зберігаються як об'єкти `LINE` у DXF файлі. Кожна властивість — координати початку/кінця, колір, шар, тип лінії, масштаб типу лінії та товщина — проходить round-trip без втрат. Коли ви відкриваєте DXF, що містить об'єкти `LINE`, вони стають повністю редагованими об'єктами `Line` в редакторі.

Лінії, намальовані в редакторі, також записуються як об'єкти `LINE` при збереженні, тому вони читаються LibreCAD, FreeCAD та будь-яким іншим DXF-сумісним додатком.
