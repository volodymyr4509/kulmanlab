# Extend

Команда `extend` подовжує найближчий кінець лінії до найближчого перетину з об'єктом-межею. Наведіть курсор на кінець лінії — підсвічується попередній перегляд — та клацніть для подовження.

## Як використовувати

1. Введіть `extend` у терміналі або натисніть кнопку **Extend** на панелі інструментів.
2. **Наведіть** курсор на кінець лінії, яку потрібно подовжити.
3. **Попередній перегляд** показує лінію, подовжену до найближчої межі.
4. **Клацніть** для підтвердження.

Команда залишається активною — наводьте та клацайте додаткові кінці ліній. Натисніть `Escape` або `Enter` для виходу.

## Логіка вибору кінцевої точки

Курсор визначає, який кінець лінії подовжується:

- Якщо курсор ближче до **кінцевої точки** — подовжується від кінцевої точки.
- Якщо курсор ближче до **початкової точки** — подовжується від початкової точки.
- Лінія подовжується у **тому ж напрямку** від вибраного кінця до першого об'єкта-межі.

## Які об'єкти ігноруються як межі

Деякі типи об'єктів **не вважаються межами**, навіть якщо вони перетинають шлях подовження:

| Тип об'єкта | Є межею? |
|-------------|----------|
| Лінія | Так |
| Полілінія | Так |
| Коло | Так |
| Дуга | Так |
| Еліпс | Так |
| Текст / Mtext | **Ні** |
| Виноска (Multileader) | **Ні** |
| Сплайн | **Ні** |

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `Escape` / `Enter` | Вийти з команди Extend |

## Підтримувані об'єкти

| Роль | Підтримуваний тип |
|------|-------------------|
| **Об'єкт, що подовжується** | Лише **Лінія** |
| **Межа** | Будь-який тип крім Тексту, Mtext, Multileader та Сплайну |

## Extend проти Trim

| | Extend | Trim |
|---|--------|------|
| Що робить | Подовжує лінію до межі | Видаляє сегмент лінії |
| Взаємодія | Наведення на кінець + клік | Наведення на сегмент + клік |
| Межа | Будь-який тип крім Тексту/Сплайну | Будь-який тип об'єкта |
| Діє на | Лише лінії | Лише лінії |
