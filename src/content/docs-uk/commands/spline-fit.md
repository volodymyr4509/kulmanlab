# Spline Fit

Команда `spline fit` будує сплайн, що **проходить через кожну клікнуту точку** — на відміну від [Spline CV](./spline-cv), де крива лише притягується до контрольних вершин. Алгоритм підбирає кубічну криву, що інтерполює задані точки.

## Малювання сплайна через точки

1. Введіть `spline fit` у терміналі або натисніть кнопку **Spline Fit** на панелі інструментів.
2. **Клацніть першу точку** або введіть `X,Y` та натисніть **Enter**.
3. **Клацайте наступні точки** — крива оновлюється при кожному кліку, проходячи через всі розміщені точки.
4. Натисніть **Enter** або **Пробіл** для завершення (потрібно щонайменше 2 точки).

Натисніть **Escape** для скасування.

## Введення координат

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.`, `-` | Почати введення координати X |
| `,` | Заблокувати X та перейти до введення Y |
| `Backspace` | Видалити останній введений символ |
| `Enter` / `Пробіл` | Підтвердити координату / завершити розміщення |
| `Escape` | Скасувати та скинути |

## Редагування ручками

Вибраний Spline Fit показує **ручки у кожній точці підгонки**. Перетягніть точку для переміщення — **вся крива перераховується** так щоб і далі проходити через всі точки (включно із щойно переміщеною).

## Вибір Fit-сплайнів

| Метод | Поведінка |
|-------|-----------|
| **Клік на кривій** | Вибирає сплайн |
| **Рамка праворуч** (строга) | Вибирає лише якщо вся обмежувальна рамка всередині |
| **Рамка ліворуч** (перехресна) | Вибирає якщо крива перетинає рамку |

## Підтримувані команди редагування

| Команда | Що відбувається |
|---------|-----------------|
| [Move](./move) | Переміщує всі точки підгонки рівномірно |
| [Copy](./copy) | Дублює весь сплайн |
| [Rotate](./rotate) | Обертає всі точки навколо базової точки |
| [Mirror](./mirror) | Відображає всі точки |
| [Scale](./scale) | Масштабує всі точки від базової точки |
| [Delete](./delete) | Видаляє сплайн |

Offset, Trim та Extend **не підтримуються** для Fit-сплайнів.

## Властивості

| Властивість | Значення |
|-------------|---------|
| Ступінь (Degree) | 3 (кубічний) |
| Точки підгонки (Fit Points) | Список клікнутих точок |
| Контрольні вершини | Обчислюються автоматично з точок підгонки |

## Spline Fit проти Spline CV

| | Spline Fit | Spline CV |
|---|------------|-----------|
| Крива проходить через точки | Так — інтерполює | Ні — притягується |
| Контроль | Прямий (через точки на кривій) | Непрямий (через CV) |
| Гладкість | Може мати осциляцію при щільних точках | Зазвичай гладша |
| Найкраще для | Криві що мають проходити через задані точки | Вільні органічні форми |

## DXF — об'єкт SPLINE

Зберігається як об'єкт `SPLINE` з `splineFlag=8` (Fit-сплайн) та точками підгонки. Повний round-trip у всіх DXF-сумісних програмах.
