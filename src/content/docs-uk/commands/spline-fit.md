# Spline Fit

Команда `splinefit` будує кубічний сплайн, що **проходить через кожну клікнуту точку** — інтерполяційна крива. На відміну від [Spline CV](../spline-cv/), де крива лише притягується до контрольних вершин, тут крива змушена потрапляти в кожну клікнуту координату точно. Всередині редактор підбирає контрольні вершини для цього, і ці CV зберігаються поряд з точками підгонки у файлі DXF.

## Малювання сплайна через точки підгонки

1. Введіть `splinefit` у терміналі або натисніть кнопку **Spline Fit** на панелі інструментів.
2. **Клацайте для розміщення точок підгонки** — крива проходитиме через кожну з них. Або введіть `X,Y` та натисніть **Enter** для точної координати.
3. Натисніть **Enter** для завершення (потрібно щонайменше 2 точки).

```
  ●──────●──────●──────●  ← крива проходить точно через кожен клік
  p1     p2     p3     p4
```

Попередній перегляд у реальному часі показує поточну інтерпольовану криву під час руху курсора, включно з перспективною наступною точкою на позиції курсора. Натисніть **Escape** для відміни всіх розміщених точок і виходу.

## Введення координат

Замість клацання, введіть точну позицію для будь-якої точки підгонки:

1. Введіть значення X.
2. Натисніть `,` — термінал покаже `[X], [Y{cursor}]`.
3. Введіть значення Y.
4. Натисніть **Enter** для розміщення точки підгонки.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.`, `-` | Почати введення координати X |
| `,` | Заблокувати X та перейти до введення Y |
| `Backspace` | Видалити останній введений символ |
| `Enter` | Підтвердити введену координату або завершити сплайн якщо введення не активне та є ≥ 2 точки |
| `Escape` | Скасувати всі точки та вийти |

## Редагування ручками — зміна форми через точки підгонки

Вибраний Fit-сплайн показує одну ручку на кожній точці підгонки:

| Ручка | Розташування | Що робить |
|-------|-------------|-----------|
| **Точка підгонки** | В кожній клікнутій позиції | Перетягнути для переміщення цієї точки підгонки — вся крива перераховується щоб пройти через нову позицію |

Перетягування однієї ручки перераховує всю криву, а не лише суміжні сегменти. Це відрізняється від редагування ручками полілінії, де переміщення вершини змінює лише два суміжні сегменти.

Немає ручки "переміщення всього сплайна". Для переміщення всього сплайна використовуйте команду [Move](../move/).

## Вибір Fit-сплайнів

| Метод | Поведінка |
|-------|-----------|
| **Клік** | Вибирає якщо клік потрапляє поряд з будь-якою точкою кривої |
| **Перетягнути праворуч** (строгий) | Всі вибіркові точки вздовж кривої мають знаходитися всередині рамки вибору |
| **Перетягнути ліворуч** (перехресний) | Будь-яка частина кривої, що перетинає межу рамки вибору, вибирає її |

## Підтримувані команди редагування

| Команда | Що відбувається зі сплайном |
|---------|----------------------------|
| [Move](../move/) | Переміщує всі точки підгонки та перераховані CV на однакове зміщення |
| [Copy](../copy/) | Створює ідентичний сплайн на новій позиції |
| [Rotate](../rotate/) | Обертає всі точки підгонки навколо обраної базової точки |
| [Mirror](../mirror/) | Відображає всі точки підгонки через вісь дзеркала |
| [Scale](../scale/) | Масштабує всі точки підгонки рівномірно від базової точки |
| [Delete](../delete/) | Видаляє сплайн |

Сплайни **не підтримують** Offset, Trim та Extend.

## Властивості

**Загальні**

| Властивість | Стандарт | Значення |
|-------------|---------|---------|
| Колір | 256 (ByLayer) | Індекс кольору ACI |
| Шар | `0` | Призначення шару |
| Тип лінії | ByLayer | Іменований шаблон типу лінії |
| Масштаб типу лінії | 1 | Коефіцієнт масштабу шаблону типу лінії |
| Товщина | 0 | Товщина видавлювання |

**Геометрія**

| Властивість | Значення |
|-------------|---------|
| Ступінь (Degree) | Степінь полінома — завжди 3 (кубічний) |
| Точки підгонки (Fit Points) | Координати всіх клікнутих точок проходження |
| Контрольні вершини | Внутрішньо обчислені CV для відтворення кривої |

## Spline Fit проти Spline CV — що обрати

| | Spline Fit | Spline CV |
|---|------------|-----------|
| Крива проходить через точки | Кожна клікнута точка точно | Лише перша та остання (затиснуті) |
| Ефект редагування ручкою | Точка підгонки рухається → вся крива перераховується | CV рухається → крива притягується до нової позиції |
| Передбачуваність форми | Висока — крива слідує кліками | Нижча — крива відстає від CV |
| Найкраще для | Криві що мають потрапляти в задані координати | Гладкі естетичні криві, вільні шляхи |

## DXF — об'єкт SPLINE (форма з точками підгонки)

Fit-сплайни зберігаються як об'єкти `SPLINE` у файлі DXF, зберігаючи як координати точок підгонки, так і обчислені контрольні вершини. Прапорець `splineFlag` встановлений в `8` (сплайн з точками підгонки), щоб програма при перезавантаженні знала який набір точок відображати як редаговані ручки. Всі властивості — колір, шар, тип лінії, масштаб типу лінії та товщина — проходять round-trip без втрат. DXF-програми, що підтримують сплайни з точками підгонки (LibreCAD, FreeCAD), відображатимуть точки підгонки як основні редаговані дані.
