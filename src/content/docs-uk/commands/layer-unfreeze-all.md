# LayerUnfreezeAll

Команда `layer unfreeze all` розморожує всі шари кресленика за один крок, відновлюючи видимість всіх раніше прихованих шарів.

## Як використовувати

Введіть `layer unfreeze all` у терміналі або натисніть кнопку **LayerUnfreezeAll** — всі шари стають видимими негайно. Жодного вибору об'єктів не потрібно.

## Коли використовувати

Зазвичай після [LayerIsolate](./layer-isolate): ізолюйте шари для фокусованого редагування, потім запустіть LayerUnfreezeAll для відновлення повного огляду.

## Деталі поведінки

| Аспект | Поведінка |
|--------|-----------|
| Що змінюється | Лише прапорець **Заморожено** → розморожено |
| Що **не** змінюється | Прапорці **Заблоковано** та **Друкувати** — незмінні |
| Підказки | Немає — команда виконується негайно |
| Вибір об'єктів | Не потрібен |
