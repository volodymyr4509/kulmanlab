# Copy

Команда `copy` створює транслюючі дублікати вибраних об'єктів та розміщує їх зі зміщенням від базової точки до точки призначення — оригінали залишаються точно на своєму місці. Це ключова відмінність від [Move](../move/): Copy додає нові об'єкти до кресленика; Move переміщує існуючі.

## Два способи запуску

**Вибрати спочатку, потім скопіювати** — виберіть об'єкти, потім активуйте:

1. Виберіть один або більше об'єктів на полотні.
2. Введіть `copy` у терміналі або натисніть кнопку **Copy** на панелі інструментів.
3. **Клацніть базову точку** або введіть `X,Y` та натисніть **Enter** для точної координати.
4. **Клацніть точку призначення** — дублікати з'являються зі зміщенням база→призначення. Введення координат також працює тут.

**Активувати, потім вибрати** — запустіть команду без вибору:

1. Введіть `copy` або натисніть кнопку на панелі інструментів.
2. **Виберіть об'єкти** — клацайте для перемикання окремих об'єктів або перетягуйте для вибору рамкою.
3. Натисніть **Enter** або **Пробіл** для підтвердження вибору.
4. **Клацніть базову точку**, потім **клацніть точку призначення** (введення координат доступне на обох кроках).

```
  До:               Після:
  [об'єкт A]        [об'єкт A]  ← оригінали незмінні
  [об'єкт B]        [об'єкт B]
                    [копія A]   ← нові об'єкти
                    [копія B]
```

Привид попереднього перегляду копій слідує за курсором від базової точки до точки призначення.

## Введення координат

На кроці базової точки або точки призначення введіть точну позицію замість клацання:

1. Введіть значення X.
2. Натисніть `,` — термінал показує `[X], [Y{cursor}]`.
3. Введіть значення Y.
4. Натисніть **Enter** для підтвердження.

## Блокування кута та точна відстань

Після встановлення базової точки команда прив'язується до осей під кутом 45° (0°, 45°, 90°, 135°, …) коли курсор достатньо далеко і поблизу осі. При блокуванні введіть відстань та натисніть **Enter** для розміщення копій точно на цьому зміщенні.

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.` | Додати цифру до значення відстані |
| `-` | Від'ємна відстань — змінює напрямок вздовж осі (лише перший символ) |
| `Backspace` | Видалити останній введений символ |
| `Enter` | Розмістити копії на введеній відстані |

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `Enter` / `Пробіл` | Підтвердити вибір та перейти до фази базової точки |
| `0`–`9`, `.`, `-` | Почати введення координати X або відстані при заблокованому куті |
| `,` | Заблокувати X та перейти до введення Y |
| `Backspace` | Видалити останній введений символ |
| `Enter` | Підтвердити координату або застосувати копію на введеній відстані |
| `Escape` | Скасувати та скинути |

## Вибір об'єктів під час команди

| Метод | Поведінка |
|-------|-----------|
| **Клік** | Перемикає об'єкт під курсором у/з вибору |
| **Перетягнути праворуч** (строгий) | Додає об'єкти, що повністю знаходяться в рамці |
| **Перетягнути ліворуч** (перехресний) | Додає об'єкти, що перетинають межу рамки |
| **Enter** / **Пробіл** | Підтверджує вибір |

## Після копіювання

Оригінали залишаються на своїх місцях — нові копії додаються до кресленика і команда завершується. Для роботи з копіями одразу запустіть Copy або іншу команду на новому виборі.

## Copy проти Move

| | Copy | Move |
|---|------|------|
| Оригінали | Залишаються на місці | Видаляються з оригінальної позиції |
| Кількість результатів | Збільшується на кількість скопійованих об'єктів | Незмінна |
| Найкраще для | Повторювана геометрія, симетричні компонування | Перепозиціонування геометрії |

## Підтримувані об'єкти

Copy працює з усіма типами об'єктів. Всі об'єкти реалізують `translate(dx, dy)` внутрішньо, тому жоден не виключений.
