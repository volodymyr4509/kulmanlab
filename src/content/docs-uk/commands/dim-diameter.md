# Dimension Diameter

Команда `dimdiameter` розміщує розмір діаметра на дузі або колі. Лінія розміру охоплює повний діаметр — проходячи через центр між двома діаметрально протилежними точками дуги — і позначається `⌀ <значення>`. Для розміщення лише радіуса від центру до одного краю використовуйте [Dimension Radius](../dim-radius/).

## Анатомія розміру діаметра

```
  ●──────────── ⌀ 10.00 ────────────●
  (дальня точка дуги)        (ближня точка дуги / сторона тексту)
```

- **Лінія розміру** — охоплює повний діаметр зі стрілками в обох точках перетину дуги.
- **Ближня точка дуги** — точка окружності з боку курсора (де знаходиться текстовий ярлик).
- **Дальня точка дуги** — діаметрально протилежна точка.
- **Ярлик** — `⌀` з наступним значенням діаметра.

## Розміщення розміру діаметра

1. Введіть `dimdiameter` у терміналі або натисніть кнопку **Dimension Diameter** на панелі інструментів.
2. **Клацніть дугу або коло** для вибору.
3. **Перемістіть курсор** для обертання лінії розміру до бажаного кута.
4. **Клацніть** для розміщення розміру.

Можна вибрати лише об'єкти типу **Arc** та **Circle**.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `Escape` | Скасувати |

## Dimension Diameter проти Dimension Radius

| | Dimension Diameter | Dimension Radius |
|---|-------------------|-----------------|
| Що вимірює | Повний діаметр (2 × радіус) | Радіус (від центру до краю) |
| Лінія розміру | Від краю до краю через центр | Від центру до краю |
| Префікс ярлика | `⌀` | `R` |
| Стрілки | Дві (в обох точках дуги) | Одна (в точці дуги) |
| Найкраще для | Повні розміри кругового отвору або вала | Розмір однієї сторони криволінійного елемента |

## Редагування мітки — simple mode

**Двічі клацніть** розміщений розмір діаметра, щоб відкрити текстовий редактор у режимі **simple**. Редактор заздалегідь заповнений поточним відрендеренним значенням (наприклад, `⌀ 10.00`), тому ви можете розмістити курсор і редагувати його безпосередньо.

| Функція | Поведінка |
|---------|-----------|
| Bold / Italic / Font / Height | Застосовується до **всієї** мітки одночасно |
| Форматування окремих символів | Не підтримується |
| `Enter` | Підтверджує значення та закриває редактор |
| Багаторядковість | Не підтримується |

Дивіться [Текстовий редактор — simple mode](../../interface/text-editor/#simple-mode) для повного довідника.

## DXF — об'єкт DIMENSION з типом діаметра

Розміри діаметра зберігаються як об'єкти `DIMENSION` з геометрією типу діаметра, зберігаючи позиції обох точок дуги та виміряне значення діаметра (2 × радіус). Всі властивості проходять round-trip без втрат.
