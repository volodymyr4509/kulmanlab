# Dimension Diameter

Команда `dim diameter` розміщує розмір, що відображає діаметр кола. Значення розміру відображається з префіксом **Ø** (наприклад, Ø50).

## Анатомія розміру діаметра

```
  ●──────●──────●  ← лінія розміру через центр
  (дальня точка дуги) (центр) (ближня точка дуги)
           Ø50
```

Лінія розміру проходить через **центр** кола від одної точки дуги до іншої (ближня та дальня відносно кліку).

## Як додати розмір діаметра

1. Введіть `dim diameter` у терміналі або натисніть кнопку **Dim Diameter**.
2. **Клацніть коло** для вибору.
3. **Клацніть місце розміщення тексту розміру** — лінія розміру проходить через центр кола.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `Escape` | Скасувати та скинути |

## Dim Diameter проти Dim Radius

| | Dim Diameter | Dim Radius |
|---|-------------|------------|
| Застосовується до | Лише кола | Кола та дуги |
| Значення | 2 × радіус | Радіус |
| Позначення | Ø | R |
| Лінія розміру | Через центр (два кінці на колі) | Від центру до одного кінця |

## DXF — об'єкт DIMENSION

Зберігається як об'єкт `DIMENSION` з типом "diameter". Повний round-trip у DXF-сумісних програмах.
