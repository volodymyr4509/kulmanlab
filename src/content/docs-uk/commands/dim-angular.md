# Dimension Angular

Команда `dim angular` вимірює та відображає кут між двома лініями або кутовий охват дуги чи кола. Значення відображається в градусах.

> **Примітка DXF:** Кутові розміри **не** включаються у DXF-експорт. Вони зберігаються лише у форматі JSON. Використовуйте [Export](./export) у форматі JSON для збереження кутових розмірів.

## Два лінії

1. Введіть `dim angular` у терміналі або натисніть кнопку **Dim Angular**.
2. **Клацніть першу лінію**.
3. **Клацніть другу лінію**.
4. **Клацніть місце розміщення тексту** — дугова лінія розміру малюється між двома лініями.

Позиція кліку для тексту (по якій стороні від перетину) визначає, чи вимірюється гострий (менший) чи тупий (більший) кут.

## Дуга

1. Введіть `dim angular`.
2. **Клацніть дугу** — розмір вимірює кутовий охват дуги від початкового до кінцевого кута.
3. **Клацніть місце розміщення тексту**.

## Коло

1. Введіть `dim angular`.
2. **Клацніть коло** — розмір вимірює кут від першої до другої клікнутої точки на колі відносно центру.
3. **Клацніть другу точку на колі** для визначення кута вимірювання.
4. **Клацніть місце розміщення тексту**.

## Динамічне оновлення

Значення розміру оновлюється в реальному часі при переміщенні точок прив'язки за допомогою ручок після розміщення.
