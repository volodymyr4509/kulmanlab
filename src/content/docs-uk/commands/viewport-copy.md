# ViewportCopy

Команда `viewport copy` дублює існуюче вікно перегляду паперового компонування на нову позицію. Нове вікно перегляду успадковує точні налаштування оригіналу.

> **Примітка:** ViewportCopy доступна лише у **просторі аркуша (Layout)**. У просторі моделі команда відображає повідомлення про помилку.

## Попередній вибір або вибір під час команди

**Вибрати спочатку**:
1. Виберіть вікно перегляду, яке потрібно скопіювати.
2. Введіть `viewport copy` або натисніть кнопку **ViewportCopy**.
3. **Клацніть базову точку**.
4. **Клацніть точку призначення** — копія розміщується і команда завершується.

**Активувати, потім вибрати**:
1. Введіть `viewport copy` або натисніть кнопку.
2. **Клацніть вікно перегляду** для вибору.
3. **Клацніть базову точку**, потім **клацніть точку призначення**.

## Введення координат

На кроці базової або точки призначення введіть точну позицію замість клацання:
1. Введіть X, натисніть `,`, введіть Y, натисніть **Enter**.

## Блокування кута та точна відстань

Після встановлення базової точки прив'язується до осей під кутом 45°. Введіть відстань та натисніть **Enter** для копіювання на точне зміщення.

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.` | Додати цифру до відстані |
| `-` | Від'ємна відстань (лише перший символ) |
| `Backspace` | Видалити останній символ |
| `Enter` | Розмістити копію на введеній відстані |

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.`, `-` | Почати введення координати X або відстані |
| `,` | Заблокувати X та перейти до введення Y |
| `Backspace` | Видалити останній введений символ |
| `Enter` | Підтвердити координату або відстань |
| `Escape` | Скасувати та скинути |
