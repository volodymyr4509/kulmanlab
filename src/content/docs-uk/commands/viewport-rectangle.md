# ViewportRectangle

Команда `viewport rectangle` створює прямокутне вікно перегляду в активному паперовому компонуванні, клацаючи два протилежних кути.

> **Примітка:** ViewportRectangle доступна лише у **просторі аркуша (Layout)**. Спробуйте перейти на вкладку аркуша якщо команда відображає помилку у просторі моделі.

## Створення вікна перегляду

1. Переключіться на вкладку **Layout** (паперовий простір).
2. Введіть `viewport rectangle` або натисніть кнопку **ViewportRectangle**.
3. **Клацніть перший кут** або введіть `X,Y` та натисніть **Enter**.
4. **Клацніть протилежний кут** — вікно перегляду створюється і відображає простір моделі.

## Введення координат

1. Введіть X, натисніть `,`, введіть Y, натисніть **Enter**.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.`, `-` | Почати введення координати X |
| `,` | Заблокувати X та перейти до введення Y |
| `Backspace` | Видалити останній введений символ |
| `Enter` | Підтвердити координату |
| `Escape` | Скасувати |

## Редагування вікна перегляду після створення

| Дія | Результат |
|-----|-----------|
| Перетягнути **кут** | Змінити розмір (збільшити/зменшити) |
| Перетягнути **краї** (не кут) | Змінити розмір по одній осі |
| Перетягнути **ручку центру** | Переміщує весь viewport |
| Відкрити **вибір масштабу** | Вибрати зі списку або ввести власне співвідношення (наприклад, `1:50`) |
| **Правий клік → Lock** | Заблокувати/розблокувати масштаб та позицію перегляду |

## ViewportRectangle проти ViewportCopy

| | ViewportRectangle | ViewportCopy |
|---|------------------|-------------|
| Що робить | Створює нове вікно перегляду | Копіює існуюче вікно перегляду |
| Введення | Два кути нового прямокутника | Вибір існуючого + точка призначення |
| Результат | Нове незалежне вікно | Дублікат з тими ж налаштуваннями |
