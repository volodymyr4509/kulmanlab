# Help

Команда `help` відображає вбудований довідник клавіатурних скорочень та посилання на документацію. Натисніть кнопку **Help** на панелі інструментів або введіть `help` у терміналі.

## Клавіатурні скорочення — Навігація

| Клавіша / Дія | Результат |
|---------------|-----------|
| Прокрутка коліщатка миші | Збільшити / Зменшити навколо курсора |
| Перетягнути середньою кнопкою | Прокрутити перегляд (Pan) |
| Двічі клацнути середньою кнопкою | Вмістити весь кресленик (Fit) |

## Клавіатурні скорочення — Команди

| Клавіша | Результат |
|---------|-----------|
| Введіть назву команди | Активувати команду |
| `Delete` | Видалити вибрані об'єкти |
| `Enter` / `Пробіл` | Підтвердити вибір або завершити крок |
| `Escape` | Скасувати поточну команду / скинути вибір |

## Клавіатурні скорочення — Вибір

| Дія | Результат |
|-----|-----------|
| Клік лівою кнопкою на об'єкт | Вибрати об'єкт |
| Перетягнути ліворуч (перехресна рамка) | Вибрати об'єкти, що перетинають рамку |
| Перетягнути праворуч (строга рамка) | Вибрати об'єкти, що повністю в рамці |
| Клік лівою кнопкою на ручку | Активувати редагування ручкою |

## Поради з малювання

- Введіть координати `X,Y` замість кліку для точного введення точок.
- Заблокуйте напрямок по осі 45° — наводьте курсор поки не з'явиться прив'язка.
- Введіть відстань та натисніть Enter після блокування кута для точного переміщення.

## Версія та контакти

На сторінці Help відображається **поточна версія** KulmanLab CAD. Для зв'язку з командою: **kulmanlab@gmail.com**.
