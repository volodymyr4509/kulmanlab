# Dimension Linear

Команда `dimlinear` розміщує горизонтальний або вертикальний розмір між двома початками виносних ліній. Лінія розміру завжди іде або ідеально горизонтально, або ідеально вертикально — вона не може бути розміщена під довільним кутом. Використовуйте [Dimension Aligned](../dim-aligned/) коли потрібен розмір паралельний діагональній лінії.

## Анатомія лінійного розміру

```
  |←————— 5.00 —————→|
  |                   |
  ●  (виносна лінія 1) ●  (виносна лінія 2)
  p1                  p2
```

- **Виносні лінії** — падають від кожної виміряної точки перпендикулярно до лінії розміру.
- **Лінія розміру** — горизонтальна (вимірює відстань X) або вертикальна (вимірює відстань Y).
- **Значення** — проектована відстань вздовж обраної осі, а не справжня відстань між точками.

## Розміщення лінійного розміру

1. Введіть `dimlinear` у терміналі або натисніть кнопку **Dimension Linear** на панелі інструментів.
2. **Клацніть початок першої виносної лінії** (p1), або введіть `X,Y` та натисніть **Enter** для точної координати.
3. **Клацніть початок другої виносної лінії** (p2). Введення координат також працює тут.
4. **Перемістіть курсор** для позиціонування лінії розміру. Орієнтація визначається автоматично з позиції курсора.
5. **Клацніть** для розміщення, або введіть відстань відступу та натисніть **Enter** для точного розміщення.

## Автоматичне визначення орієнтації

Коли орієнтація не зафіксована, команда зчитує позицію курсора відносно двох виміряних точок:

| Позиція курсора | Визначена орієнтація | Що вимірюється |
|-----------------|---------------------|-----------------|
| Вище або нижче точок | Горизонтальна | Δ X між p1 та p2 |
| Ліворуч або праворуч від точок | Вертикальна | Δ Y між p1 та p2 |

Натисніть **H** для фіксації горизонтальної або **V** для фіксації вертикальної в будь-який момент під час фази розміщення. Після фіксації орієнтація не змінюється при русі курсора.

## Введення відступу

Введіть число під час розміщення для фіксації лінії розміру на точній відстані від виміряних точок:

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.` | Додати цифру до відстані відступу |
| `Backspace` | Видалити останній введений символ |
| `Enter` / `Пробіл` | Розмістити на введеній відстані |

Сторона курсора (вище/нижче для горизонтального, ліворуч/праворуч для вертикального) визначає знак — лінія розміру з'являється на стороні де поточно знаходиться курсор.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.`, `-` | Почати введення координати X (фази p1/p2) або відстань відступу (фаза розміщення) |
| `,` | Заблокувати X та перейти до введення Y (фази p1/p2) |
| `H` | Зафіксувати горизонтальну орієнтацію (лише фаза розміщення) |
| `V` | Зафіксувати вертикальну орієнтацію (лише фаза розміщення) |
| `Backspace` | Видалити останній введений символ |
| `Enter` / `Пробіл` | Підтвердити введену координату або відступ |
| `Escape` | Скасувати |

## Dimension Linear проти Dimension Aligned

| | Dimension Linear | Dimension Aligned |
|---|-----------------|------------------|
| Вісь | Завжди H або V | Паралельна до виміряної лінії |
| Що вимірює | Лише компонента X або Y | Справжня Евклідова відстань |
| Клавіші H/V | Так — фіксація орієнтації | Ні — завжди слідує p1→p2 |
| Найкраще для | Ортогональні компонування, плани поверхів | Нахилені елементи, кутові зрізи |

## Редагування мітки — simple mode

**Двічі клацніть** розміщений лінійний розмір, щоб відкрити текстовий редактор у режимі **simple**. Редактор заздалегідь заповнений поточним відрендеренним значенням, тому ви можете розмістити курсор і редагувати його безпосередньо.

| Функція | Поведінка |
|---------|-----------|
| Bold / Italic / Font / Height | Застосовується до **всієї** мітки одночасно |
| Форматування окремих символів | Не підтримується |
| `Enter` | Підтверджує значення та закриває редактор |
| Багаторядковість | Не підтримується |

Дивіться [Текстовий редактор — simple mode](../../interface/text-editor/#simple-mode) для повного довідника.

## Ланцюгові розміри

Для додавання більше розмірів що продовжуються від останньої виносної лінії, використовуйте [Dimension Continue](../dim-continue/) одразу після розміщення цього.

## DXF — об'єкт DIMENSION

Лінійні розміри зберігаються як об'єкти `DIMENSION` з `rotationDeg` встановленим у `0` (горизонтальний) або `90` (вертикальний). Початки виносних ліній, позиція лінії розміру, позиція тексту, виміряне значення, стиль стрілки, висота тексту та всі прапорці відображення проходять round-trip без втрат.
