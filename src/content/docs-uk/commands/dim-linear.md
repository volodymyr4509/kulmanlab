# Dimension Linear

Команда `dim linear` розміщує горизонтальний або вертикальний розмір між двома точками прив'язки.

## Анатомія лінійного розміру

```
       ← текст →
  ┌────────────────┐  ← лінія розміру
  |                |
  ●                ●  ← точки прив'язки
```

- **Точки прив'язки**: дві клікнуті точки геометрії.
- **Виносні лінії**: від точок прив'язки до лінії розміру.
- **Лінія розміру**: горизонтальна або вертикальна, зі стрілками та текстом значення.

## Як додати лінійний розмір

1. Введіть `dim linear` у терміналі або натисніть кнопку **Dim Linear**.
2. **Клацніть першу точку прив'язки** (або введіть `X,Y` + Enter).
3. **Клацніть другу точку прив'язки** (або введіть координати).
4. **Клацніть місце розміщення лінії розміру** — перетягніть для встановлення відступу від геометрії.

## Автоматичне визначення напрямку

| Положення курсора відносно точок | Орієнтація |
|----------------------------------|------------|
| Курсор вище або нижче обох точок | Горизонтальний (вимірює ΔX) |
| Курсор ліворуч або праворуч від обох точок | Вертикальний (вимірює ΔY) |

## Фіксація напрямку клавішами H/V

Після встановлення першої точки прив'язки:
- Натисніть **H** для примусового горизонтального розміру.
- Натисніть **V** для примусового вертикального розміру.

## Введення відступу

Після встановлення обох точок прив'язки введіть числове значення відступу (відстань від геометрії до лінії розміру) та натисніть Enter замість клацання.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.`, `-` | Почати введення координати X або відступу |
| `,` | Заблокувати X та перейти до введення Y |
| `H` | Зафіксувати горизонтальний напрямок |
| `V` | Зафіксувати вертикальний напрямок |
| `Backspace` | Видалити останній введений символ |
| `Enter` | Підтвердити координату або застосувати відступ |
| `Escape` | Скасувати та скинути |

## Ланцюгові розміри

Для продовження розмірного ланцюга від кінцевої точки цього розміру використовуйте [Dim Continue](./dim-continue).

## Dim Linear проти Dim Aligned

| | Dim Linear | Dim Aligned |
|---|------------|-------------|
| Що вимірює | Горизонтальну або вертикальну складову | Пряму відстань між точками |
| Кут лінії розміру | Завжди 0° або 90° | Паралельний вектору між точками |
| Найкраще для | Прямокутна геометрія | Нахилені об'єкти, скоси |

## DXF — об'єкт DIMENSION

Лінійні розміри зберігаються як об'єкти `DIMENSION` з типом "linear". Повний round-trip у DXF-сумісних програмах.
