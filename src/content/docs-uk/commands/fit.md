# Fit

Команда `fit` обчислює обмежувальну рамку всіх об'єктів та встановлює масштаб і позицію перегляду так, щоб весь кресленик був видимий у вікні.

## Як запустити

- Натисніть кнопку **Fit** на панелі інструментів, або
- **Двічі клацніть середньою кнопкою миші** (швидкий ярлик).

> Примітка: немає клавіатурного ярлика — використовуйте подвійний клік середньою кнопкою.

## Алгоритм обмежувальної рамки

1. Збирає всі об'єкти на кресленику.
2. Обчислює **мінімальний прямокутник**, що охоплює всі вершини/центри.
3. Масштабує перегляд так, щоб прямокутник + поле відступу вмістилися у вікно.
4. Центрує перегляд на центрі прямокутника.

| Стан кресленика | Поведінка |
|-----------------|-----------|
| Більш широкий ніж високий | Масштабується по ширині вікна |
| Більш високий ніж широкий | Масштабується по висоті вікна |
| Один об'єкт | Масштабується з відступом навколо нього |
| Порожній кресленик | Перегляд не змінюється |

## Fit проти ручних елементів керування зумом

| | Fit | Zoom In / Zoom Out | Прокрутка коліщатка |
|---|-----|--------------------|---------------------|
| Точність | Автоматична | Кроки по 1.5× | Кроки по 1.5× |
| Позиція | Автоматично по центру | Зум навколо курсора | Зум навколо курсора |
| Найкраще для | Огляд повного кресленика | Навігація до конкретної ділянки | Плавна навігація |

| Команда | Дія |
|---------|-----|
| [Zoom In](../zoom-in/) | Збільшення на 1.5× |
| [Zoom Out](../zoom-out/) | Зменшення на 1.5× |
| [Pan](../pan/) | Прокрутка перегляду без масштабування |
