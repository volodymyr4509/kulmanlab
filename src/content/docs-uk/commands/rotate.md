# Rotate

Команда `rotate` обертає вибрані об'єкти навколо базової точки. Ви вказуєте кут обертання або вводячи число у градусах, або кліком — кут обчислюється з напрямку між базовою точкою та кліком.

## Два способи запуску

**Вибрати спочатку, потім обертати** — виберіть об'єкти, потім активуйте:

1. Виберіть один або більше об'єктів на полотні.
2. Введіть `rotate` у терміналі або натисніть кнопку **Rotate** на панелі інструментів.
3. **Клацніть базову точку** — центр обертання. Або введіть `X,Y` та натисніть **Enter** для точної координати.
4. **Введіть кут та натисніть Enter** або **клацніть** для визначення кута за напрямком курсора.

**Активувати, потім вибрати** — запустіть команду без вибору:

1. Введіть `rotate` або натисніть кнопку на панелі інструментів.
2. **Виберіть об'єкти** — клацайте або перетягуйте.
3. Натисніть **Enter** або **Пробіл** для підтвердження вибору.
4. **Клацніть базову точку** (введення координат доступне), потім вкажіть кут.

```
  До:            Після (90° навколо ●):
                    ╔══╗
  ●  [об'єкт]  →  ● ║    ║
                    ╚══╝
```

Живий привид попереднього перегляду обернутих об'єктів слідує за кутом курсора після встановлення базової точки.

## Встановлення кута

**Введений кут** — введіть число (у градусах) в будь-який момент після встановлення базової точки. Попередній перегляд прив'язується до введеного кута поки ви продовжуєте налаштовувати перед натисканням Enter.

**Кут кліком** — якщо введеного значення немає, клік встановлює кут рівним `atan2(курсорY − базаY, курсорX − базаX)` — напрямку від базової точки до кліку, у градусах.

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.` | Додати цифру до значення кута |
| `-` | Від'ємний кут (лише перший символ) |
| `Backspace` | Видалити останній введений символ |
| `Enter` | Застосувати обертання на введеному куті |

## Напрямок кута

Кути відповідають **конвенції DXF**:

- **Позитивні** значення обертають **проти годинникової стрілки** у координатах кресленика (вісь Y вгору).
- На екрані, де вісь Y інвертована (вниз), позитивні кути виглядають **за годинниковою стрілкою**.

Поширені значення: `90` = чверть оберту, `180` = пів оберту, `-90` = протилежний чверть оберт.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `Enter` / `Пробіл` | Підтвердити вибір |
| `0`–`9`, `.`, `-` | Почати введення координати X (фаза базової точки) або значення кута (фаза кута) |
| `,` | Заблокувати X та перейти до введення Y (фаза базової точки) |
| `Backspace` | Видалити останній введений символ |
| `Enter` | Підтвердити координату або застосувати обертання |
| `Escape` | Скасувати та скинути |

## Вибір об'єктів під час команди

| Метод | Поведінка |
|-------|-----------|
| **Клік** | Перемикає об'єкт під курсором |
| **Перетягнути праворуч** (строгий) | Додає об'єкти, що повністю знаходяться в рамці |
| **Перетягнути ліворуч** (перехресний) | Додає об'єкти, що перетинають межу рамки |
| **Enter** / **Пробіл** | Підтверджує вибір |

## Підтримувані об'єкти

Rotate працює з усіма типами об'єктів. Геометрія кожного об'єкта обертається навколо базової точки — наприклад, Коло переміщує свій центр, радіус залишається незмінним; Дуга переміщує центр та зміщує початковий та кінцевий кути на величину обертання; Текст переміщує точку вставки та додає кут до властивості Rotation Degree.
