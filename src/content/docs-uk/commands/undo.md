# Undo

Команда `undo` скасовує останню редагуючу дію. Кожне виконання Undo скасовує ще один крок назад в історії редагування.

## Скасування дії

- Введіть `undo` у терміналі, або
- Натисніть **Ctrl+Z** (Windows/Linux) або **Cmd+Z** (Mac), або
- Натисніть кнопку **Undo** на панелі інструментів.

## Що можна скасувати

- Малювання об'єктів (Line, Polyline, Circle тощо)
- Редагування об'єктів (Move, Copy, Rotate, Delete тощо)
- Зміни властивостей
- Зміни шарів

## Обмеження

Деякі дії не скасовуються або мають обмежену підтримку скасування — наприклад, очищення сховища через [WipeStorage](./wipestorage).

## Undo проти Redo

| | Undo | Redo |
|---|------|------|
| Напрямок | Назад у часі | Вперед у часі |
| Клавіша | Ctrl+Z / Cmd+Z | Ctrl+Y / Cmd+Y |
| Скасовує | Останню виконану дію | Останню скасовану дію |
