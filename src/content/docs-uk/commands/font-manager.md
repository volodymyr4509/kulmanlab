# Font Manager

Команда `FontManager` відкриває діалог для перегляду та вибору шрифтів, а також для завантаження власних файлів `.ttf` для використання в об'єктах [Text](../text/) та [Multileader](../leader/).

## Відкриття Font Manager

- Введіть `FontManager` у терміналі, **або**
- Натисніть кнопку **Font Manager** на панелі інструментів [текстового редактора](../../interface/text-editor/).

## Групи шрифтів

| Група | Вміст |
|-------|-------|
| **Default** | Вбудований шрифт без засічок — доступний завжди |
| **User** | Ваші власні завантажені шрифти `.ttf` (відображається лише після додавання хоча б одного) |
| **Free** | 15 вбудованих шрифтів Google Fonts (EB Garamond, Fira Code, Inter, Lato, Merriweather, Montserrat, Nunito, Open Sans, Oswald, Playfair Display, Poppins, Raleway, Roboto, Roboto Condensed, Source Code Pro) |
| **System** | Поширені шрифти операційної системи (Courier New, Georgia, Helvetica, Impact, Lucida Console, Tahoma, Times New Roman, Trebuchet MS, Verdana) |

Клацніть будь-який шрифт у списку, щоб переглянути його праворуч — назву, зразок алфавіту, панграму та цифри.

## Завантаження власного шрифту

1. Натисніть **Add Font** у нижній частині діалогу (або введіть `Font+` у терміналі, щоб одразу відкрити вибір файлу).
2. Виберіть файл `.ttf`. Підтримуються лише шрифти TrueType — `.otf` та `.woff`/`.woff2` не підтримуються.
3. Назва файлу (без розширення) стає назвою шрифту в групі **User**. Наприклад, завантаження `MyFont.ttf` додає шрифт з назвою `MyFont`.

Завантажені шрифти зберігаються назавжди в браузері (IndexedDB) і автоматично завантажуються знову при наступному відкритті KulmanLab CAD.

## Видалення власного шрифту

Наведіть курсор на шрифт у групі **User** і натисніть кнопку **×** поруч з ним. Вбудовані шрифти (Default, Free, System) видалити не можна.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `↑` / `↓` | Перемістити вибір вгору або вниз по списку шрифтів |
| `Escape` | Закрити Font Manager |

## Сумісність з DXF

Назва шрифту вбудовується в експортовані об'єкти **MTEXT** як вбудований код форматування, тому DXF, пропущений через KulmanLab CAD, зберігає своє призначення шрифту. Файли власних шрифтів не вбудовуються в DXF — лише назва шрифту. Якщо ви повторно імпортуєте кресленик, що посилається на власний шрифт, який ще не завантажено на цьому пристрої, текст відображається шрифтом Default, доки ви не завантажите шрифт із такою самою назвою.

## Пов'язані команди

- [Text](../text/) — розміщує текстові мітки, до яких застосовується вибір шрифту
- [Match Properties](../match-properties/) — копіює висоту тексту, але не шрифт, між об'єктами
