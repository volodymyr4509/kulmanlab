# Pan

Команда `pan` переміщує огляд кресленика (прокручує без масштабування). Геометрія кресленика залишається незмінною — рухається лише точка огляду.

## Як запустити Pan

**Режим Pan через команду**:
1. Введіть `pan` або натисніть кнопку **Pan** на панелі інструментів.
2. **Клацніть та перетягніть** у будь-якому напрямку.
3. Натисніть **Escape** для виходу з режиму Pan.

**Середня кнопка миші (завжди доступна)**:
- **Утримуйте середню кнопку миші** (колесо) та перетягуйте — працює в будь-який час, навіть під час виконання іншої команди. Повернення до попередньої команди після відпускання.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `Escape` | Вийти з режиму Pan |

> Примітка: немає клавіатурних скорочень для самого переміщення. Для Pan під час будь-якої команди — використовуйте середню кнопку миші.

## Pan проти Zoom

| | Pan | Zoom |
|---|-----|------|
| Що змінюється | Позиція огляду | Масштаб (рівень збільшення) |
| Геометрія | Не змінюється | Не змінюється |
| Ярлик | Середня кнопка миші | Коліщатко миші |

## Поради

- **Прокрутка + Pan**: Коліщатко миші для зуму, середня кнопка для панорамування — найшвидша навігація без перемикання між командами.
- **Подвійний клік середньою**: Замість Pan вручну — подвійний клік середньою кнопкою запускає [Fit](../fit/) для відображення всього кресленика.
- **Pan під час команди**: Утримуйте середню кнопку для тимчасового Pan без переривання поточної команди (наприклад, під час розміщення точок Лінії).
