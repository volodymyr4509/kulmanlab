# Leader

Команда `leader` малює виноску — стрілкову лінію, що веде до текстової мітки. Виноска складається з ламкого шляху точок (плечей) зі стрілкою в першій точці та редагованою текстовою міткою в кінці.

> **Примітка DXF:** Виноски доступні лише для імпорту. Об'єкти MLEADER з файлів DXF правильно зчитуються та відображаються. Однак виноски, намальовані в редакторі, **не зберігаються** при збереженні файлу DXF. Використовуйте [Export](./export) у форматі JSON для збереження виносок.

## Анатомія виноски

```
  ● Точка 1 (стрілка)
   \
    \  плечо
     \
      ● Точка 2
      |__________
                 [Текст мітки]
```

Перша клікнута точка отримує **стрілку**. Остання точка перед Enter — де кріпиться текстова мітка.

## Чотири фази команди

1. **Фаза першої точки** — де розміщується стрілка.
2. **Фаза наступних точок** — побудова плечей виноски. Натисніть Enter для переходу до тексту.
3. **Фаза введення тексту** — вводьте текст мітки.
4. **Підтвердження** — натисніть Enter для розміщення виноски.

## Введення координат

На будь-якій точковій фазі введіть точну позицію замість клацання:

1. Введіть значення X.
2. Натисніть `,` — термінал показує `[X], [Y{cursor}]`.
3. Введіть значення Y.
4. Натисніть **Enter** для підтвердження.

## Блокування кута

Під час фаз введення точок виноска відстежує осі прив'язки під кутом 45°. Тяжіє до 0°, 45°, 90°, 135°, … коли курсор близько. Введіть відстань та натисніть **Enter** для точного розміщення вздовж заблокованого напрямку.

## Редагування текстової мітки

Під час фази введення тексту:

| Клавіша | Дія |
|---------|-----|
| Будь-який символ | Додати до тексту мітки |
| `←` / `→` | Перемістити курсор у тексті |
| `Backspace` | Видалити символ ліворуч від курсора |
| `Delete` | Видалити символ праворуч від курсора |
| `Enter` | Підтвердити текст та розмістити виноску |

## Автоматичний напрямок тексту

Напрямок текстової мітки визначається автоматично залежно від положення останнього плеча:

| Положення | Вирівнювання тексту |
|-----------|-------------------|
| Плечо виходить праворуч | Текст ліворуч від кінцевої точки |
| Плечо виходить ліворуч | Текст праворуч від кінцевої точки |

## Довідник клавіатури — фази точок

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.`, `-` | Почати введення координати X |
| `,` | Заблокувати X та перейти до введення Y |
| `Backspace` | Видалити останній введений символ |
| `Enter` | Підтвердити координату або завершити введення плечей (перейти до тексту) |
| `Escape` | Скасувати та скинути |

## Довідник клавіатури — фаза тексту

| Клавіша | Дія |
|---------|-----|
| Будь-який символ | Додати до тексту мітки |
| `←` / `→` | Перемістити курсор у тексті |
| `Backspace` | Видалити символ ліворуч |
| `Delete` | Видалити символ праворуч |
| `Enter` | Підтвердити та розмістити виноску |
| `Escape` | Скасувати введення тексту |

## Додавання та видалення плечей

Після розміщення виноски можна змінити її плечі:

- **[Leader Add](./leader-add)** — додає нове плече до існуючої виноски.
- **[Leader Remove](./leader-remove)** — видаляє плече з виноски (виноска видаляється якщо залишається одне плече).
