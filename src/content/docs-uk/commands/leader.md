# Leader

Команда `leader` малює анотацію виноски в чотири фази: стрілка що торкається елемента, лінія виноски що згинається в коліні, прив'язка тексту та введена мітка. З усіх команд анотації, Leader є єдиною що включає інтерактивну фазу введення тексту з попереднім переглядом миготливого курсора.

> **Примітка DXF:** Виноски доступні **лише для імпорту**. Об'єкти `MLEADER` з файлів DXF правильно зчитуються та відображаються, але виноски намальовані в редакторі **не записуються** при збереженні DXF файлу. Використовуйте [Export](../export/) у форматі JSON для збереження виносок.

## Анатомія виноски

```
  ◄── кінчик стрілки  (крок 2 — торкається елемента)
      \
       \  лінія виноски
        \
         ●──── коліно (крок 3) ──── прив'язка тексту (крок 4)
                                    Текст мітки  (крок 5)
```

- **Кінчик стрілки** — загострений кінець розміщений на елементі що анотується.
- **Коліно** — кут де лінія виноски згинається в напрямку тексту.
- **Прив'язка тексту** — де позиціонується мітка. Текст вирівнюється ліворуч або праворуч автоматично.

## Малювання виноски

1. Введіть `leader` у терміналі або натисніть кнопку **Leader** на панелі інструментів.
2. **Клацніть кінчик стрілки**, або введіть `X,Y` та натисніть **Enter** для точної координати.
3. **Клацніть коліно** — вигин у виносці. Кут прив'язується до кратних 45°; введіть довжину та натисніть **Enter** для точного розміщення. Або введіть `X,Y` для абсолютної координати.
4. **Клацніть позицію тексту** — де прив'язується мітка. Ті ж варіанти: клік, блокування кута + довжина, або `X,Y`.
5. **Введіть текст мітки** — попередній перегляд на полотні оновлюється в реальному часі з миготливим курсором. Натисніть **Enter** для розміщення.

## Введення координат (всі фази точок)

На будь-якому кроці вибору точки (кінчик, коліно, позиція тексту) можна ввести точну координату замість кліку:

1. Введіть значення X (цифри, `.` або `-`).
2. Натисніть `,` — термінал показує `[X], [Y{cursor}]` підтверджуючи що X зафіксований.
3. Введіть значення Y.
4. Натисніть **Enter** для розміщення точки.

## Блокування кута (кроки 3 та 4)

Після кожної розміщеної точки, команда прив'язується до осей 45° коли курсор достатньо далеко. При блокуванні:
- Попередній перегляд прив'язується до осі.
- Введіть довжину та натисніть **Enter** для розміщення наступної точки на точній відстані.

Блокування кута та введення координат взаємовиключні — якщо ввести цифру без попереднього `,`, команда інтерпретує її як відстань (блокування кута має бути активним). Для введення абсолютної координати натомість, починайте з числа X за яким іде кома.

## Редагування текстової мітки

Під час введення мітки на кроці 5, можна навігувати та редагувати текст перед розміщенням:

| Клавіша | Дія |
|---------|-----|
| Будь-який друкований символ | Вставити в позиції курсора |
| `←` / `→` | Перемістити курсор ліворуч або праворуч |
| `Backspace` | Видалити символ ліворуч від курсора |
| `Delete` | Видалити символ праворуч від курсора |
| `Enter` | Розмістити виноску |

## Автоматичний напрямок тексту

Вирівнювання тексту коригується до позиції курсора відносно коліна:

| Позиція курсора | Напрямок тексту |
|-----------------|----------------|
| **Праворуч** від коліна | Зліва направо від прив'язки тексту |
| **Ліворуч** від коліна | Справа наліво (прив'язка з правої сторони) |

Ручне коригування не потрібне — перемістіть курсор на сторону де хочете мітку і вона вирівнюється правильно.

## Довідник клавіатури

**Фази точок (кінчик, коліно, позиція тексту)**

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.`, `-` | Почати введення координати X (потім `,` для фіксації X та введення Y) |
| `,` | Підтвердити X та перейти до введення Y |
| `0`–`9`, `.`, `-` | Накопичити відстань при блокуванні кута |
| `Backspace` | Видалити останній введений символ |
| `Enter` | Підтвердити введену координату або відстань |

**Фаза введення тексту**

| Клавіша | Дія |
|---------|-----|
| Друкований символ | Вставити в позиції курсора |
| `←` / `→` | Перемістити курсор |
| `Backspace` | Видалити ліворуч |
| `Delete` | Видалити праворуч |
| `Enter` | Розмістити виноску |

| Клавіша | Дія |
|---------|-----|
| `Escape` | Скасувати та скинути до кроку 2 |

## Редагування існуючої виноски

**Двічі клацніть** розміщену виноску, щоб знову відкрити текстовий редактор у режимі **rich**. У режимі rich можна застосовувати жирний, курсив та перевизначення шрифту або висоти для окремих символів, а також вставляти переноси рядків за допомогою `Enter`. Натисніть **Escape** для підтвердження та закриття.

Дивіться [Текстовий редактор — rich mode](../interface/text-editor#rich-mode) для повного довідника.
