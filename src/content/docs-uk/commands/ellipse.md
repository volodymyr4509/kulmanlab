# Ellipse

Команда `ellipse` малює еліпс за трьома кліками: центральна точка, кінець першої (головної) піввісі під будь-яким кутом і довжина другої (допоміжної) піввісі. Дві осі завжди перпендикулярні одна до одної — напрямок другої осі виводиться автоматично з першої.

## Малювання еліпса

1. Введіть `ellipse` у терміналі або натисніть кнопку **Ellipse** на панелі інструментів.
2. **Клацніть центральну точку** або введіть `X,Y` та натисніть **Enter** для точної координати.
3. **Клацніть кінець першої осі** — встановлює і напрямок, і довжину першої піввісі. Введення координат також працює тут.
4. **Встановіть довжину другої осі** — перемістіть курсор перпендикулярно до першої осі, потім клацніть або введіть довжину.

```
               ● ← кінець першої осі (крок 3)
              /
  центр ●  /  ← перша вісь (будь-який кут)
            |
            ● ← курсор тут встановлює довжину другої осі (крок 4)
```

Еліпс розміщується після кроку 4 і команда завершується.

## Введення осі — клік, координата або введення довжини

**Центр (крок 2):** клацніть або введіть `X,Y` для точної позиції.

**Кінець першої осі (крок 3):** клацніть або введіть `X,Y` для точної координати. Блокування кута також прив'язує до кратних 45° — при блокуванні введіть довжину та натисніть **Enter** для розміщення кінцевої точки на точній відстані.

**Друга вісь (крок 4):** введення довжини доступне завжди — блокування кута не потрібне. Напрямок вже зафіксований перпендикулярно до першої осі; введення встановлює лише довжину.

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.` | Додати цифру до довжини осі (фаза другої осі) |
| `Backspace` | Видалити останній введений символ |
| `Enter` | Розмістити кінець осі на введеній довжині |

## Введення координат (центр та кінець першої осі)

1. Введіть значення X.
2. Натисніть `,` — термінал покаже `[X], [Y{cursor}]`.
3. Введіть значення Y.
4. Натисніть **Enter** для підтвердження.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.`, `-` | Почати введення координати X (фази центру/першої осі) або довжини осі при блокуванні кута |
| `,` | Заблокувати X та перейти до введення Y |
| `Backspace` | Видалити останній введений символ |
| `Enter` | Підтвердити введену координату або довжину |
| `Escape` | Скасувати та скинути |

## Редагування ручками — незалежне масштабування осей

Вибраний еліпс показує п'ять ручок:

| Ручка | Кількість | Що робить |
|-------|-----------|-----------|
| **Центр** | 1 | Переміщує весь еліпс; обидві осі незмінні |
| **Кінці головної осі** | 2 (протилежні кінці довшої осі) | Перетягнути для зміни довжини головної піввісі; абсолютний розмір допоміжної осі залишається постійним |
| **Кінці допоміжної осі** | 2 (протилежні кінці коротшої осі) | Перетягнути для зміни довжини допоміжної піввісі; головна вісь незмінна |

Ручки головної та допоміжної осі незалежні — можна змінити форму еліпса без повторного запуску команди.

## Вибір еліпсів

| Метод | Поведінка |
|-------|-----------|
| **Клік** | Вибирає якщо клік потрапляє поряд з контуром еліпса |
| **Перетягнути праворуч** (строгий) | Обмежувальна рамка еліпса, вирівняна по осях, повинна повністю вміщатися в рамці вибору |
| **Перетягнути ліворуч** (перехресний) | Будь-яка частина контуру еліпса, що перетинає межу рамки вибору, вибирає його |

## Властивості

**Загальні**

| Властивість | Стандарт | Значення |
|-------------|---------|---------|
| Колір | 256 (ByLayer) | Індекс кольору ACI |
| Шар | `0` | Призначення шару |
| Тип лінії | ByLayer | Іменований шаблон типу лінії |
| Масштаб типу лінії | 1 | Коефіцієнт масштабу шаблону типу лінії |
| Товщина | 0 | Товщина видавлювання |

**Геометрія**

| Властивість | Значення |
|-------------|---------|
| Центр X / Центр Y | Центр еліпса |
| Вектор головної осі X / Y | Вектор від центру до кінця головної осі (кодує і напрямок, і довжину) |
| Відношення осей | Відношення допоміжної піввісі до головної (0 < відношення ≤ 1) |
| Початковий кут / Кінцевий кут | Параметричні кути в градусах; для повного еліпса обидва 0°/360° |

## Ellipse проти Circle — що обрати

| | Ellipse | Circle |
|---|---------|--------|
| Осі | Дві незалежні піввісі під будь-яким кутом | Один радіус, симетричне |
| Поворот | Може бути розміщений під будь-яким кутом | Без повороту |
| Введення | Довжина на кожну вісь | Лише радіус |
| Масштабування ручками | Головна та допоміжна незалежно | Всі чотири кардинальні точки рівномірно |
| Найкраще для | Косі види, овальні елементи, перспективні отвори | Симетричні круглі елементи |

## DXF — об'єкт ELLIPSE

Еліпси зберігаються як об'єкти `ELLIPSE` у файлі DXF. Формат зберігає центральну точку, повний вектор головної осі (напрямок + довжина) та відношення осей. Поворот, форма та всі властивості стилю проходять round-trip без втрат. Коло **не** зберігається як вироджений еліпс — два типи об'єктів залишаються різними в моделі DXF.
