# Ellipse

Команда `ellipse` малює еліпс шляхом клацання трьох точок: центру, кінця першої (головної) осі та точки, що визначає ширину другої (допоміжної) осі. Нахил першої осі визначає кут нахилу всього еліпса.

## Малювання еліпса

1. Введіть `ellipse` у терміналі або натисніть кнопку **Ellipse** на панелі інструментів.
2. **Клацніть центральну точку** або введіть `X,Y` та натисніть **Enter**.
3. **Клацніть кінець першої осі** — відстань та кут від центру визначають довжину та напрямок головної осі.
4. **Клацніть точку другої осі** — відстань від центру до цього кліку вздовж перпендикуляра до першої осі задає напівширину допоміжної осі.

Еліпс розміщується і команда завершується.

## Введення координат

Для будь-якого кроку введіть точну позицію замість клацання:

1. Введіть значення X.
2. Натисніть `,` — термінал покаже `[X], [Y{cursor}]`.
3. Введіть значення Y.
4. Натисніть **Enter** для розміщення точки.

На кроці другої осі також можна ввести **довжину** безпосередньо (одне число) — вона застосовується як напівширина вздовж поточного кута перпендикуляра.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.`, `-` | Почати введення координати X або довжини осі |
| `,` | Заблокувати X та перейти до введення Y |
| `Backspace` | Видалити останній введений символ |
| `Enter` | Підтвердити введену координату |
| `Escape` | Скасувати та вийти |

## Редагування ручками

Вибраний еліпс показує **5 ручок**:

| Ручка | Розташування | Що робить |
|-------|-------------|-----------|
| **Центр** | Центральна точка | Переміщує весь еліпс |
| **Кінець головної осі +** | Кінець головної осі | Змінює довжину та кут головної осі |
| **Кінець головної осі −** | Протилежний кінець | Незалежне редагування іншого кінця |
| **Кінець допоміжної осі +** | Кінець допоміжної осі | Змінює ширину другої осі |
| **Кінець допоміжної осі −** | Протилежний кінець | Незалежне редагування |

## Вибір еліпсів

| Метод | Поведінка |
|-------|-----------|
| **Клік на контурі** | Вибирає еліпс |
| **Клік всередині** | Не вибирає |
| **Рамка праворуч** (строга) | Вибирає лише якщо вся обмежувальна рамка всередині |
| **Рамка ліворуч** (перехресна) | Вибирає якщо контур перетинає рамку |

## Підтримувані команди редагування

| Команда | Що відбувається |
|---------|-----------------|
| [Move](./move) | Переміщує центр |
| [Copy](./copy) | Створює ідентичний еліпс |
| [Rotate](./rotate) | Обертає центр та всі осі |
| [Mirror](./mirror) | Відображає еліпс |
| [Scale](./scale) | Масштабує обидві осі рівномірно |
| [Delete](./delete) | Видаляє еліпс |

Offset, Trim та Extend **не підтримуються** для еліпсів.

## Властивості

| Властивість | Значення |
|-------------|----------|
| Центр X / Y | Координати центральної точки |
| Кінець осі X / Y | Кінцева точка головної осі відносно центру |
| Співвідношення | Відношення допоміжної осі до головної (0–1) |

## Ellipse проти Circle

| | Ellipse | Circle |
|---|---------|--------|
| Осі | Дві (можуть бути різними) | Одна (радіус) |
| Кут нахилу | Визначається напрямком головної осі | Завжди симетричне |
| Редагування | 5 ручок | 5 ручок (центр + 2 квадранти + 2 квадранти) |

## DXF — об'єкт ELLIPSE

Еліпси зберігаються як об'єкти `ELLIPSE` у DXF файлі зі збереженням точного геометричного визначення (центр, вектор головної осі, співвідношення осей). Коло **не є** виродженим еліпсом у DXF — кола зберігаються окремо як `CIRCLE`.
