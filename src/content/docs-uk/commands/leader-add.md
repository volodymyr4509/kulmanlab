# Leader+

Команда `leader+` додає нове плече зі стрілкою до існуючої мультивиноски. Нове плече тягнеться від існуючої перехідної ділянки виноски до нової вістрю стрілки, яку ви клацаєте. Весь стиль — позиція перехідної ділянки, текст, тип стрілки та розмір — успадковується від вибраної виноски.

## Додавання плеча

1. Введіть `leader+` у терміналі.
2. **Клацніть існуючу мультивиноску** для її вибору.
3. **Клацніть нову вістрю стрілки** або введіть `X,Y` та натисніть **Enter** для точної координати. Лінія попереднього перегляду показує від курсора до перехідної ділянки виноски.

Плече розміщується, і команда залишається активною — можна одразу клацнути іншу виноску для додавання більше плечей. Натисніть **Enter**, **Пробіл** або **Escape** для завершення.

```
  До:                                   Після:
  ◄── плече 1                           ◄── плече 1
       \                                      \
        ●──── перехід ──── текст               ●──── перехід ──── текст
                                              /
                                  плече 2 ──►  (нова вістря яку ви клацнули)
```

## Введення координат вістрю

Замість клацання можна ввести точну позицію:

1. Введіть значення X.
2. Натисніть `,` — термінал підтверджує блокування X.
3. Введіть значення Y.
4. Натисніть **Enter** для розміщення.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.`, `-` | Почати введення координати X |
| `,` | Заблокувати X та перейти до введення Y |
| `Enter` | Підтвердити введену координату та розмістити плече |
| `Enter` / `Пробіл` | Завершити (коли немає введення у процесі) |
| `Escape` | Скасувати та скинути |

## Примітки

- Вибирати можна лише об'єкти **Мультивиноска** — клацання будь-якого іншого типу об'єкта нічого не робить.
- Нове плече починається від існуючої перехідної ділянки; ви визначаєте лише де розміщується вістря стрілки.
- Кількість плечей мультивиноски не обмежена.
