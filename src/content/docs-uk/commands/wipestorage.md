# Wipe Storage

Команда `wipestorage` видаляє всі кресленики та збережені дані з локального сховища браузера. Дія **незворотна** — збережіть важливі кресленики через [Export](../export/) перед використанням.

## Очищення сховища

1. Введіть `wipestorage` у терміналі або натисніть кнопку **Wipe Storage**.
2. Підтвердьте у відображеному діалозі — всі кресленики видаляються і розпочинається новий порожній кресленик.

## Коли використовувати

- Сховище браузера переповнене і додаток дає попередження.
- Потрібно повністю скинути стан для передачі пристрою іншому користувачу.
- Усунення несправностей при корумпованих даних.

> **Альтернатива для аварійних ситуацій**: Якщо додаток аварійно завершується при кожному запуску (наприклад, через надто великі координати) і Wipe Storage недоступний — відкрийте `https://kulmanlab.com/?resetKulmanLocalStorage` для скидання без запуску редактора.

## Wipe Storage проти New File

| | Wipe Storage | New File |
|---|-------------|----------|
| Що видаляє | Всі кресленики з сховища | Нічого — просто відкриває порожнє полотно |
| Поточний кресленик | Видаляється | Зберігається у сховищі |
| Незворотність | Так | Ні |
