# Leader−

Команда `leader-` видаляє одне плече зі стрілкою з існуючої мультивиноски. Текстова мітка, перехідна ділянка та всі інші плечі зберігаються — видаляється лише вибране плече. Мультивиноска з єдиним плечем не може мати це плече видаленим.

## Видалення плеча

1. Введіть `leader-` у терміналі.
2. **Клацніть мультивиноску**, що має два або більше плечей. Якщо клацнута виноска має лише одне плече, термінал показує помилку і чекає на коректний вибір.
3. **Перемістіть курсор біля плеча**, яке потрібно видалити — найближче плече підсвічується маркером.
4. **Клацніть** для видалення цього плеча.

Плече видаляється, і команда залишається активною — можна одразу клацнути іншу виноску (або ту ж) для видалення більше плечей. Натисніть **Enter**, **Пробіл** або **Escape** для завершення.

```
  До:                          Після:
  ◄── плече 1                  ◄── плече 1
       \                              \
        ●──── перехід ──── текст       ●──── перехід ──── текст
       /
  плече 2 ──►  ← це плече видалено
```

## Як визначається найближче плече

Команда вимірює перпендикулярну відстань від курсора до відрізків кожного плеча (включаючи відрізок від останньої точки плеча до перехідної ділянки). Плече з найменшою відстанню підсвічується і буде видалено при кліці.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `Enter` / `Пробіл` | Завершити видалення плечей |
| `Escape` | Скасувати та скинути |

## Примітки

- Виноска з **лише одним плечем** захищена — потрібно спочатку додати плече перш ніж видаляти.
- Позиція перехідної ділянки та вміст тексту завжди зберігаються незалежно від того, яке плече видаляється.
