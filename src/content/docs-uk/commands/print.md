# Print

Команда `print` відкриває **Менеджер друку** — спеціальне вікно експорту з попереднім переглядом, вибором формату (PNG / JPEG / WebP / PDF), перемикачем монохромного режиму та необов'язковим обрізанням ділянки. Нічого не надсилається на фізичний принтер — результат завантажується як файл.

## Відкриття Менеджера друку

Натисніть кнопку **Print** на панелі інструментів або введіть `print` у терміналі. Менеджер друку відкривається негайно, показуючи попередній перегляд поточного вікна перегляду.

## Макет Менеджера друку

Вікно складається з двох панелей:
- **Ліва бічна панель** — всі елементи керування експортом.
- **Права панель** — полотно попереднього перегляду, що оновлюється при зміні налаштувань.

### Елементи керування бічної панелі

| Елемент | Опис |
|---------|------|
| **Change Area** | Обрізати до довільного прямокутника на полотні (дивіться нижче) |
| Перемикач **Monochrome** | Конвертувати всі кольорові лінії в чорні — увімкнено за замовчуванням для чистого друку |
| Список **Format** | PNG, JPEG, WebP або PDF |
| Кнопка **Export** | Генерує та завантажує файл |

## Вибір довільної ділянки для експорту

За замовчуванням попередній перегляд показує точно те, що було видно на полотні при відкритті Менеджера друку. Для експорту конкретної ділянки:

1. Натисніть **Change Area** — Менеджер друку ховається і полотно стає інтерактивним.
2. **Клацніть перший кут** прямокутника експорту.
3. **Клацніть протилежний кут** — Менеджер друку знову відкривається з вибраною ділянкою у попередньому перегляді.

Натисніть `Escape` під час вибору ділянки для скасування та відновлення попередньої ділянки.

Полотно попереднього перегляду динамічно змінює розмір відповідно до **точного співвідношення сторін** вибраної ділянки, тому попередній перегляд є піксельно точним.

## Формати експорту

| Формат | Найкраще для | Примітки |
|--------|-------------|----------|
| **PNG** | Без втрат, чіткі лінії | Білий фон, без прозорості |
| **JPEG** | Менший файл для надсилання | Якість 95%, легке стискання |
| **WebP** | Найменший файл для веб | Та ж якість 95%, краще стискання ніж JPEG |
| **PDF** | Документи готові до друку | Зображення вбудоване при 150 DPI у контейнер PDF |

Експортований файл називається `kulman-<мітка часу>.<розширення>` та завантажується автоматично.

## Роздільна здатність та фон

- Максимальна роздільна здатність: **2000 × 2000 пікселів**, пропорційно до вибраної ділянки.
- Фон завжди **білий**.
- Шари, позначені як **не для друку**, виключаються з експорту.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `Escape` (під час вибору ділянки) | Скасувати вибір ділянки, відновити попередню |
| `Escape` (у Менеджері друку) | Закрити Менеджер друку |
