# LayerMatch

Команда `layer match` переміщує один або більше об'єктів на шар іншого (джерельного) об'єкта. Це аналог [Match Properties](../match-properties/), але лише для шарів.

## Як використовувати

1. Введіть `layer match` або натисніть кнопку **LayerMatch** на панелі інструментів.
2. **Виберіть цільові об'єкти** — клацайте або перетягуйте для вибору об'єктів, шар яких потрібно змінити.
3. Натисніть **Enter** або **Пробіл** для підтвердження вибору цілей.
4. **Клацніть джерельний об'єкт** — об'єкт, шар якого ви хочете використати. Всі цільові об'єкти переміщуються на цей шар і команда завершується.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `Enter` / `Пробіл` | Підтвердити вибір цілей |
| `Escape` | Скасувати та скинути |

## Деталі поведінки

| Аспект | Поведінка |
|--------|-----------|
| Що змінюється | Лише атрибут **шару** — колір, тип лінії та геометрія не змінюються |
| Джерельний об'єкт | Не змінюється |
| Команда завершується | Автоматично після кліку на джерело |

## Пов'язані команди

- [Match Properties](../match-properties/) — копіює всі властивості (не лише шар)
- [Layer Make Current](../layer-make-current/) — встановити поточний шар
