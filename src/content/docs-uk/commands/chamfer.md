# Chamfer

Команда `chamfer` зрізає прямий діагональний кут між двома об'єктами [Line](../line/) або [Polyline](../polyline/). Ви вказуєте наскільки далеко зрізати вздовж кожного об'єкта (d1 та d2), і команда підрізає обидва об'єкти до цих точок та вставляє з'єднувальну лінію між ними.

Використання рівних відстаней дає симетричний зріз 45°; різні відстані дають асиметричну фаску.

Chamfer працює з об'єктами **Line та Polyline**.

## Використання Chamfer

1. Введіть `chamfer` у терміналі або натисніть кнопку **Chamfer** на панелі інструментів.
2. **Введіть першу відстань фаски** (d1 — відстань вздовж першого об'єкта) та натисніть **Enter**.
3. **Введіть другу відстань фаски** (d2 — відстань вздовж другого об'єкта) та натисніть **Enter**.
4. **Клацніть перший об'єкт** — частина яку ви клацаєте визначає яка сторона від будь-якого перетину зберігається.
5. **Наведіть на другий об'єкт** — пунктирна лінія попереднього перегляду показує отриманий зріз фаски. Перемістіть курсор на сторону яку хочете зберегти.
6. **Клацніть** для застосування. Обидва об'єкти підрізаються та лінія фаски вставляється.

```
  До (d1=5, d2=8):            Після:

  ──────────────              ──────────╲
                │                        ╲────
                │
```

## Вибір сторони

Коли дві лінії перетинають одна одну, фаска застосовується до кута, визначеного позиціями кліків — частина кожного об'єкта **на тій же стороні що й курсор** зберігається.

- Клацніть поряд з одним кінцем першого об'єкта для вибору тієї половини.
- Перемістіть курсор до бажаної половини другого об'єкта — пунктирний попередній перегляд оновлюється в реальному часі.

Для Polyline, позиція кліку визначає який **сегмент** полілінії бере участь, і найближча вершина на стороні перетину є тою що підрізається.

## Що створює команда

- Кінцева точка (або вершина полілінії) першого об'єкта, найближча до перетину, переміщується до точки **T1**, розташованої на відстані d1 вздовж першого об'єкта від перетину.
- Кінцева точка (або вершина полілінії) другого об'єкта, найближча до перетину, переміщується до точки **T2**, розташованої на відстані d2 вздовж другого об'єкта від перетину.
- Новий об'єкт Line вставляється від **T1** до **T2**.

Вставлена лінія успадковує поточні налаштування товщини лінії, кольору, шару та типу лінії.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.` | Додати цифру до поточного значення відстані |
| `Backspace` | Видалити останній введений символ |
| `Enter` | Підтвердити введену відстань та перейти далі |
| `Escape` | Скасувати та скинути |

## Підтримувані об'єкти

| Об'єкт | Підтримується |
|--------|--------------|
| Line | Так |
| Polyline / Rectangle | Так |
| Arc, Circle, Ellipse | Ні |
| Text, Spline, Dimension, Leader | Ні |

## Chamfer проти Fillet

| | Chamfer | Fillet |
|---|---------|--------|
| Тип кута | Прямий зріз | Закруглена дуга |
| Введення | Дві відстані (d1, d2) | Один радіус |
| Вставлений об'єкт | Line | Arc |
| Підтримувані об'єкти | Lines та Polylines | Лише Lines |
