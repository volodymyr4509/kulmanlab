# Chamfer

Команда `chamfer` зрізає пряму діагональну фаску на куті між двома лініями або суміжними відрізками [полілінії](./polyline) — аналогічно [Fillet](./fillet), але замість дуги вставляється пряма лінія.

## Як використовувати

1. Введіть `chamfer` у терміналі або натисніть кнопку **Chamfer** на панелі інструментів.
2. **Введіть першу відстань фаски** (відстань від кута вздовж першої лінії) та натисніть **Enter**.
3. **Введіть другу відстань фаски** (відстань від кута вздовж другої лінії) та натисніть **Enter**. Для симетричної фаски введіть те саме значення.
4. **Клацніть першу лінію** — ближче до **сторони, яку хочете зберегти**.
5. **Клацніть другу лінії** — так само ближче до сторони що залишається. Фаска вставляється.

## Вибір сторони

Аналогічно [Fillet](./fillet), для ліній що **вже перетинаються** місце кліку визначає яку сторону від перетину зберегти. Клацніть ту частину кожної лінії, що залишиться після обрізання.

## Що створює команда

| | Значення |
|---|----------|
| **T1** | Точка відміри вздовж першої лінії — де вона підрізається |
| **T2** | Точка відміри вздовж другої лінії — де вона підрізається |
| **Пряма лінія** | Вставляється між T1 та T2, успадковує колір та шар першої лінії |

## Симетрична проти асиметричної фаски

- **Симетрична** (обидві відстані рівні): фаска під 45° для перпендикулярних ліній.
- **Асиметрична** (різні відстані): фаска під гострішим або тупішим кутом.

## Відстань 0

При обох відстанях рівних 0 — лінії обрізаються або подовжуються до гострого кута (те ж що Fillet з R=0), без вставки фаски.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.` | Ввести відстань |
| `Backspace` | Видалити останній символ |
| `Enter` | Підтвердити відстань |
| `Escape` | Скасувати |

## Підтримувані об'єкти

- [Line](./line) — повністю підтримується
- Суміжні відрізки [Polyline](./polyline) — фаска всередині полілінії

## Chamfer проти Fillet

| | Chamfer | Fillet |
|---|---------|--------|
| Що вставляється | Пряма лінія (фаска) | Дотична дуга |
| Введення | Дві відстані | Один радіус |
| Найкраще для | Машинні фаски, зрізи | Плавні кути, галтелі |
