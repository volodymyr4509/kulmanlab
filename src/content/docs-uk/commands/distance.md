# Distance

Команда `distance` вимірює відстань між двома точками, що відображається у терміналі. Жодних об'єктів на полотні не створюється — лише читання.

## Анатомія вимірювання

```
  ● Точка 1 ←——————————→ ● Точка 2
                d = ?
```

Відстань — евклідова довжина між двома клікнутими точками.

## Ланцюгове вимірювання

Після першого вимірювання (Точка 1 → Точка 2) команда залишається активною і чекає наступного кліку. Цей клік стає новою **Точкою 2** (попередня Точка 2 стає новою Точкою 1). Це дозволяє вимірювати відстані вздовж ламаного шляху:

```
  ● T1 →d1→ ● T2 →d2→ ● T3
```

Кожна відстань d1, d2, … відображається окремо у терміналі.

Натисніть `Escape` для завершення ланцюгового вимірювання.

## Distance проти Angle

| | Distance | Angle |
|---|----------|-------|
| Кількість точок | 2 | 3 (зенітна + 2) |
| Вимірює | Довжину між точками | Внутрішній кут між напрямками |
| Результат | Одне число в одиницях кресленика | Одне число в градусах |

## Введення координат

На будь-якому кроці введення точки введіть точну позицію замість клацання:

1. Введіть значення X.
2. Натисніть `,` — термінал показує `[X], [Y{cursor}]`.
3. Введіть значення Y.
4. Натисніть **Enter** для підтвердження.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.`, `-` | Почати введення координати X |
| `,` | Заблокувати X та перейти до введення Y |
| `Backspace` | Видалити останній введений символ |
| `Enter` | Підтвердити координату |
| `Escape` | Скасувати та скинути |

## Примітки

- Результати відображаються **лише у терміналі** — ніяких об'єктів на полотні не створюється.
- Точність: **4 десяткові знаки**.
- Одиниці відповідають одиницям кресленика (немає вбудованого перетворення одиниць).
