# Distance

Команда `distance` вимірює пряму лінію (Евклідову) відстань між двома клікнутими точками та виводить результат у терміналі з точністю 4 десяткові знаки. Це одна з трьох команд вимірювання — [Angle](../angle/) вимірює кутове відкриття у вершині, а [Area](../area/) вимірює площу та периметр багатокутника.

## Анатомія вимірювання відстані

```
  ● перша точка
   \
    \  лінія попереднього перегляду (у реальному часі)
     \
      ● друга точка    →  термінал: "Distance: 12.3456"
```

- **Перша точка** — початок вимірювання.
- **Друга точка** — кінцева точка; її розміщення виводить результат негайно.
- **Результат** — відображається у терміналі, не розміщується на полотні.

## Вимірювання відстані

1. Введіть `distance` у терміналі або натисніть кнопку **Distance** на панелі інструментів.
2. **Клацніть першу точку**, або введіть `X,Y` та натисніть **Enter** для точної координати.
3. **Клацніть другу точку** — виміряна відстань з'являється у терміналі. Введення координат також працює тут.
4. **Клацніть ще раз** (необов'язково) для початку нового вимірювання. Команда залишається активною.

Натисніть `Escape` в будь-який момент для скидання до кроку 2.

## Ланцюгові вимірювання

Після появи результату, клацання негайно починає наступне вимірювання — клікнута точка стає новою першою точкою. Це дозволяє вимірювати послідовність відстаней без повторної активації команди.

## Distance проти Angle

| | Distance | Angle |
|---|---------|-------|
| Що вимірює | Довжина прямої лінії | Внутрішній кут у вершині |
| Кількість кліків | 2 | 3 |
| Формат результату | `12.3456` (одиниці) | `45.0000°` |
| Попередній перегляд на полотні | Лінія від першої точки до курсора | Дві лінії від вершини до курсора |
| Найкраще для | Довжина зазору або сегмента | Кут відкриття між двома елементами |

## Введення координат

Замість клацання, введіть точну позицію для будь-якої точки:

1. Введіть значення X.
2. Натисніть `,` — термінал покаже `[X], [Y{cursor}]`.
3. Введіть значення Y.
4. Натисніть **Enter** для підтвердження.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.`, `-` | Почати введення координати X |
| `,` | Заблокувати X та перейти до введення Y |
| `Backspace` | Видалити останній введений символ |
| `Enter` | Підтвердити введену координату |
| `Escape` | Скасувати та скинути до кроку 2 |

## Примітки

- Результати показуються **лише у терміналі** — нічого не додається до кресленика.
- Результат виражений в тих самих одиницях що й координати кресленика (без перетворення одиниць).
- Точність завжди 4 десяткові знаки.
