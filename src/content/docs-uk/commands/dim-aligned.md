# Dimension Aligned

Команда `dimaligned` розміщує розмір, що вимірює **справжню пряму відстань** між двома точками. Лінія розміру йде паралельно лінії, що з'єднує ці дві точки, тому вона може бути під будь-яким кутом. Це є ключовою відмінністю від [Dimension Linear](./dim-linear), який обмежений горизонтальним або вертикальним.

## Анатомія вирівняного розміру

```
     ●  p2
    /|
   / |  (виносна лінія 2, перпендикулярна до лінії розміру)
  /  |
 /←5.00→/
/  /
●  /  (виносна лінія 1, перпендикулярна до лінії розміру)
p1
```

- **Виносні лінії** — перпендикулярні до лінії розміру, проведені від кожної виміряної точки.
- **Лінія розміру** — паралельна p1→p2, зміщена в одну сторону залежно від позиції курсора.
- **Значення** — справжня Евклідова відстань `|p1 – p2|`.

## Розміщення вирівняного розміру

1. Введіть `dimaligned` у терміналі або натисніть кнопку **Dimension Aligned** на панелі інструментів.
2. **Клацніть початок першої виносної лінії** (p1), або введіть `X,Y` та натисніть **Enter** для точної координати.
3. **Клацніть початок другої виносної лінії** (p2). Введення координат також працює тут.
4. **Перемістіть курсор** в одну сторону для встановлення перпендикулярного відступу лінії розміру.
5. **Клацніть** для розміщення, або введіть відстань відступу та натисніть **Enter** для точного розміщення.

## Введення відступу

Введіть число під час розміщення для фіксації лінії розміру на точній перпендикулярній відстані від лінії p1→p2:

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.` | Додати цифру до відступу |
| `Backspace` | Видалити останній введений символ |
| `Enter` / `Пробіл` | Розмістити на введеному відступі |

Сторона курсора визначає з якої сторони з'явиться лінія розміру.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.`, `-` | Почати введення координати X (фази p1/p2) або відстань відступу (фаза розміщення) |
| `,` | Заблокувати X та перейти до введення Y (фази p1/p2) |
| `Backspace` | Видалити останній введений символ |
| `Enter` / `Пробіл` | Підтвердити введену координату або відступ |
| `Escape` | Скасувати |

## Dimension Aligned проти Dimension Linear

| | Dimension Aligned | Dimension Linear |
|---|------------------|-----------------|
| Кут лінії розміру | Паралельний p1→p2 — будь-який кут | Завжди горизонтальний або вертикальний |
| Що вимірює | Справжня Евклідова відстань | Лише компонента X або Y |
| Блокування H/V орієнтації | Ні | Так — клавіші `H` та `V` |
| Найкраще для | Нахилені елементи, кутові зрізи | Ортогональні компонування, деталі вирівняні по сітці |

## Ланцюгові розміри

Для додавання більше розмірів що продовжуються від другої виносної лінії цього, використовуйте [Dimension Continue](./dim-continue) — він фіксується до того ж кута вимірювання що й цей вирівняний розмір.

## DXF — об'єкт DIMENSION (тип aligned)

Вирівняні розміри зберігаються як об'єкти `DIMENSION` з `dimType = 1` (aligned). Початки виносних ліній, позиція лінії розміру, позиція тексту, виміряне значення, поворот, стиль стрілки та всі прапорці відображення проходять round-trip без втрат.
