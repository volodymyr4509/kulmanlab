# Dimension Aligned

Команда `dim aligned` розміщує розмір, вирівняний по лінії між двома точками прив'язки. На відміну від [Dim Linear](./dim-linear), що вимірює лише горизонтальну або вертикальну складову, Dim Aligned вимірює **пряму відстань** між двома точками під будь-яким кутом.

## Анатомія вирівняного розміру

```
   ●——————●  ← точки прив'язки
    \      \
     ●——→←——●  ← лінія розміру (паралельна до вектора)
      [текст]
```

Лінія розміру **паралельна** до вектора між точками прив'язки.

## Як додати вирівняний розмір

1. Введіть `dim aligned` у терміналі або натисніть кнопку **Dim Aligned**.
2. **Клацніть першу точку прив'язки** (або введіть `X,Y` + Enter).
3. **Клацніть другу точку прив'язки** (або введіть координати).
4. **Клацніть місце розміщення лінії розміру** — перетягніть для встановлення відступу від геометрії.

## Введення відступу

Після встановлення обох точок прив'язки введіть числове значення відступу та натисніть Enter для розміщення точно на цій відстані від геометрії.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.`, `-` | Почати введення координати X або відступу |
| `,` | Заблокувати X та перейти до введення Y |
| `Backspace` | Видалити останній введений символ |
| `Enter` | Підтвердити координату або застосувати відступ |
| `Escape` | Скасувати та скинути |

## Ланцюгові розміри

Для продовження розмірного ланцюга від кінцевої точки цього розміру використовуйте [Dim Continue](./dim-continue).

## Dim Aligned проти Dim Linear

| | Dim Aligned | Dim Linear |
|---|-------------|-----------|
| Що вимірює | Пряма відстань між точками | Горизонтальна або вертикальна складова |
| Кут лінії розміру | Паралельний до вектора між точками | Завжди 0° або 90° |
| Найкраще для | Нахилені об'єкти, скоси, тріангуляція | Відстані паралельні до осей |

## DXF — об'єкт DIMENSION

Зберігається як об'єкт `DIMENSION` з типом "aligned". Повний round-trip у DXF-сумісних програмах.
