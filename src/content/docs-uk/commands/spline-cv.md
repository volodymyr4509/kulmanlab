# Spline CV

Команда `splinecv` будує **кубічний B-сплайн**, розміщуючи контрольні вершини (CV). Отримана крива притягується до кожної CV, але не проходить через них — крім першої та останньої вершини, де **затиснуті вузли** прив'язують криву точно. Це дає інтуїтивний контроль форми: потягніть вершину, щоб відхилити криву до неї, не примушуючи її торкатися кожної точки.

## Малювання сплайна за контрольними вершинами

1. Введіть `splinecv` у терміналі або натисніть кнопку **Spline CV** на панелі інструментів.
2. **Клацайте для розміщення контрольних вершин** — кожен клік додає вершину. Або введіть `X,Y` та натисніть **Enter** для точної координати.
3. Натисніть **Enter** для завершення (потрібно щонайменше 2 вершини).

```
  CV ●         ● CV
      \       /
       \     /    ← крива притягується до CV
        \   /         але не проходить через них
  CV ●   ●   ● CV (початок/кінець: крива торкається тут)
```

Попередній перегляд у реальному часі оновлюється під час руху курсора після кожної вершини, показуючи як виглядатиме сплайн з наступною точкою на позиції курсора. Натисніть **Escape** для відміни всіх розміщених вершин і виходу.

## Введення координат

Замість клацання, введіть точну позицію для будь-якої контрольної вершини:

1. Введіть значення X.
2. Натисніть `,` — термінал покаже `[X], [Y{cursor}]`.
3. Введіть значення Y.
4. Натисніть **Enter** для розміщення вершини.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.`, `-` | Почати введення координати X |
| `,` | Заблокувати X та перейти до введення Y |
| `Backspace` | Видалити останній введений символ |
| `Enter` | Підтвердити введену координату або завершити сплайн якщо введення не активне та є ≥ 2 вершини |
| `Escape` | Скасувати всі вершини та вийти |

## Редагування ручками — зміна форми через контрольні вершини

Вибраний CV-сплайн показує одну ручку на кожній контрольній вершині:

| Ручка | Розташування | Що робить |
|-------|-------------|-----------|
| **Контрольна вершина** | В кожній позиції CV | Перетягнути для переміщення CV — крива змінює форму до нової позиції |

Немає ручки "переміщення всього сплайна". Для переміщення всього сплайна використовуйте команду [Move](../move/).

## Вибір CV-сплайнів

| Метод | Поведінка |
|-------|-----------|
| **Клік** | Вибирає якщо клік потрапляє поряд з будь-якою точкою кривої |
| **Перетягнути праворуч** (строгий) | Всі вибіркові точки вздовж кривої мають знаходитися всередині рамки вибору |
| **Перетягнути ліворуч** (перехресний) | Будь-яка частина кривої, що перетинає межу рамки вибору, вибирає її |

## Підтримувані команди редагування

| Команда | Що відбувається зі сплайном |
|---------|----------------------------|
| [Move](../move/) | Переміщує всі контрольні вершини на однакове зміщення |
| [Copy](../copy/) | Створює ідентичний сплайн на новій позиції |
| [Rotate](../rotate/) | Обертає всі CV навколо обраної базової точки |
| [Mirror](../mirror/) | Відображає всі CV через вісь дзеркала |
| [Scale](../scale/) | Масштабує всі CV рівномірно від базової точки |
| [Delete](../delete/) | Видаляє сплайн |

Сплайни **не підтримують** Offset, Trim та Extend.

## Властивості

**Загальні**

| Властивість | Стандарт | Значення |
|-------------|---------|---------|
| Колір | 256 (ByLayer) | Індекс кольору ACI |
| Шар | `0` | Призначення шару |
| Тип лінії | ByLayer | Іменований шаблон типу лінії |
| Масштаб типу лінії | 1 | Коефіцієнт масштабу шаблону типу лінії |
| Товщина | 0 | Товщина видавлювання |

**Геометрія**

| Властивість | Значення |
|-------------|---------|
| Ступінь (Degree) | Степінь полінома — завжди 3 (кубічний) |
| Контрольні вершини | Координати всіх CV |
| Точки підгонки (Fit Points) | Порожньо для CV-сплайнів; заповнюється лише для сплайнів з точками підгонки |

## Spline CV проти Spline Fit — що обрати

| | Spline CV | Spline Fit |
|---|-----------|------------|
| Крива проходить через точки | Лише перша та остання (затиснуті) | Кожна клікнута точка точно |
| Контроль форми | Тягнути CV до потрібної ділянки | Переміщувати точки підгонки яких крива повинна торкатися |
| Ефект редагування ручкою | CV рухається → крива притягується | Точка підгонки рухається → крива перераховується |
| Найкраще для | Гладкі естетичні криві, вільні шляхи | Криві що мають потрапляти в задані координати |

## DXF — об'єкт SPLINE (форма з контрольними вершинами)

CV-сплайни зберігаються як об'єкти `SPLINE` у файлі DXF, зберігаючи ступінь, вузловий вектор та всі координати контрольних вершин. Всі властивості — колір, шар, тип лінії, масштаб типу лінії та товщина — проходять round-trip без втрат. Прапорець `splineFlag` встановлений в `9` (CV-сплайн), щоб форма зберігалася при перезавантаженні. Будь-яка DXF-програма, що підтримує об'єкти `SPLINE` з даними CV, читає їх коректно.
