# Spline CV

Команда `spline cv` будує NURBS-сплайн, розміщуючи **контрольні вершини** (CV). Крива притягується до вершин, але не обов'язково проходить через них — на відміну від [Spline Fit](./spline-fit), де крива проходить через кожну клікнуту точку.

## Малювання сплайна за CV

1. Введіть `spline cv` у терміналі або натисніть кнопку **Spline CV** на панелі інструментів.
2. **Клацніть першу контрольну вершину** або введіть `X,Y` та натисніть **Enter**.
3. **Клацайте наступні вершини** — крива у реальному часі оновлюється при кожному кліку.
4. Натисніть **Enter** або **Пробіл** для завершення (потрібно щонайменше 2 вершини).

Натисніть **Escape** для скасування всіх розміщених вершин.

## Введення координат

Для будь-якої вершини введіть `X,Y` та натисніть **Enter** для точного розміщення.

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.`, `-` | Почати введення координати X |
| `,` | Заблокувати X та перейти до введення Y |
| `Backspace` | Видалити останній введений символ |
| `Enter` / `Пробіл` | Підтвердити координату / завершити розміщення |
| `Escape` | Скасувати та скинути |

## Редагування ручками

Кожна контрольна вершина показується як **одна ручка**. Перетягніть вершину для зміни форми кривої — прилеглі сегменти оновлюються плавно.

## Вибір CV-сплайнів

| Метод | Поведінка |
|-------|-----------|
| **Клік на кривій** | Вибирає сплайн |
| **Рамка праворуч** (строга) | Вибирає лише якщо вся обмежувальна рамка всередині |
| **Рамка ліворуч** (перехресна) | Вибирає якщо крива перетинає рамку |

## Підтримувані команди редагування

| Команда | Що відбувається |
|---------|-----------------|
| [Move](./move) | Переміщує всі CV рівномірно |
| [Copy](./copy) | Дублює весь сплайн |
| [Rotate](./rotate) | Обертає всі CV навколо базової точки |
| [Mirror](./mirror) | Відображає всі CV |
| [Scale](./scale) | Масштабує всі CV від базової точки |
| [Delete](./delete) | Видаляє сплайн |

Offset, Trim та Extend **не підтримуються** для CV-сплайнів.

## Властивості

| Властивість | Значення |
|-------------|---------|
| Ступінь (Degree) | 3 (кубічний) |
| Контрольні вершини | Список CV-точок |
| Точки підгонки (Fit Points) | Порожньо — не використовуються для CV-сплайнів |

## Spline CV проти Spline Fit

| | Spline CV | Spline Fit |
|---|-----------|-----------|
| Крива проходить через точки | Ні — притягується | Так — інтерполює |
| Контроль | Непрямий (через CV) | Прямий (через точки на кривій) |
| Гладкість | Зазвичай гладша | Може мати осциляцію при щільних точках |
| Найкраще для | Вільні органічні форми | Криві що мають проходити через задані точки |

## DXF — об'єкт SPLINE

Зберігається як об'єкт `SPLINE` з `splineFlag=9` (CV-сплайн), контрольними вершинами, вузловим вектором та ступенем кривої. Повний round-trip у всіх DXF-сумісних програмах.
