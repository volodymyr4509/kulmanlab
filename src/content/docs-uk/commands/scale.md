# Scale

Команда `scale` рівномірно збільшує або зменшує вибрані об'єкти відносно базової точки — всі відстані від базової точки множаться на один і той же коефіцієнт. Базова точка залишається на місці; все інше рухається та змінює розмір пропорційно.

## Два способи запуску

**Вибрати спочатку, потім масштабувати** — виберіть об'єкти, потім активуйте:

1. Виберіть один або більше об'єктів на полотні.
2. Введіть `scale` у терміналі або натисніть кнопку **Scale** на панелі інструментів.
3. **Клацніть базову точку** або введіть `X,Y` та натисніть **Enter** для точної координати.
4. **Введіть коефіцієнт та натисніть Enter** або **клацніть** для встановлення коефіцієнта за відстанню від бази.

**Активувати, потім вибрати** — запустіть команду без вибору:

1. Введіть `scale` або натисніть кнопку на панелі інструментів.
2. **Виберіть об'єкти** — клацайте або перетягуйте.
3. Натисніть **Enter** або **Пробіл** для підтвердження вибору.
4. **Клацніть базову точку** (введення координат доступне), потім вкажіть коефіцієнт.

```
  До:          Після (2×):      Після (0.5×):
  ● [■■■]  →  ● [████████]  →  ● [■]
```

Живий привид попереднього перегляду слідує за коефіцієнтом курсора після встановлення базової точки.

## Введення коефіцієнта масштабування

Введіть числовий коефіцієнт в будь-який момент після встановлення базової точки.

| Введення | Результат |
|----------|-----------|
| `2` | Вдвічі більше (подвоїти) |
| `0.5` | Вдвічі менше (скоротити наполовину) |
| `1.5` | На 50% більше |
| `-1` | Дзеркально відобразити відносно базової точки (масштаб = 1, напрямок перевернутий) |

**Коефіцієнт кліком** — якщо введеного значення немає, клік встановлює коефіцієнт за відстанню від базової точки до кліку, нормалізованою відносно початкового розміру об'єкта.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `Enter` / `Пробіл` | Підтвердити вибір |
| `0`–`9`, `.`, `-` | Почати введення координати X (фаза бази) або значення коефіцієнта |
| `,` | Заблокувати X та перейти до введення Y (фаза бази) |
| `Backspace` | Видалити останній введений символ |
| `Enter` | Підтвердити координату або застосувати масштаб |
| `Escape` | Скасувати та скинути |

## Вибір об'єктів під час команди

| Метод | Поведінка |
|-------|-----------|
| **Клік** | Перемикає об'єкт під курсором |
| **Перетягнути праворуч** (строгий) | Додає об'єкти, що повністю знаходяться в рамці |
| **Перетягнути ліворуч** (перехресний) | Додає об'єкти, що перетинають межу рамки |
| **Enter** / **Пробіл** | Підтверджує вибір |

## Що масштабується

| Об'єкт | Що змінюється |
|--------|--------------|
| Лінія | Обидві кінцеві точки масштабуються відносно бази |
| Коло | Центр масштабується; радіус множиться на коефіцієнт |
| Дуга | Центр масштабується; радіус множиться на коефіцієнт |
| Еліпс | Центр масштабується; обидва радіуси множаться на коефіцієнт |
| Полілінія / Прямокутник | Кожна вершина масштабується відносно бази |
| Текст | Точка вставки масштабується; висота тексту множиться на коефіцієнт |
| Сплайн | Всі контрольні вершини / точки підгонки масштабуються відносно бази |
