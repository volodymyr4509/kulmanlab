# Arc

Команда `arc` малює кругову дугу через три клікнуті точки. Дуга обчислюється як унікальне описане коло, що проходить через всі три точки — не потрібно вказувати центр або радіус безпосередньо. Дуга йде від першого кліку до третього, проходячи через другий.

## Малювання дуги

1. Введіть `arc` у терміналі або натисніть кнопку **Arc** на панелі інструментів.
2. **Клацніть першу точку** — один кінець дуги. Або введіть `X,Y` та натисніть **Enter** для точної координати.
3. **Клацніть другу точку** — точка, через яку має проходити дуга (контролює кривизну та напрямок). Введення координат також працює тут.
4. **Клацніть третю точку** — інший кінець дуги. Дуга розміщується і команда завершується. Введення координат також працює тут.

```
           ● (2-й клік — середня точка на кривій)
          / \
         /   \
        ●     ●
     1-й       3-й
```

Лінія попереднього перегляду з'єднує перші два кліки, поки ви позиціонуєте третій. Починаючи від другого кліку, живий попередній перегляд дуги слідує за курсором.

> **Колінеарні точки**: якщо всі три точки лежать на одній прямій, дугу неможливо обчислити і жоден об'єкт не розміщується. Перемістіть другу точку з лінії та спробуйте знову.

## Введення координат

На будь-якому з трьох кроків ви можете ввести точну позицію замість клацання:

1. Введіть значення X.
2. Натисніть `,` — термінал покаже `[X], [Y{cursor}]`.
3. Введіть значення Y.
4. Натисніть **Enter** для розміщення точки.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.`, `-` | Почати введення координати X |
| `,` | Заблокувати X та перейти до введення Y |
| `Backspace` | Видалити останній введений символ |
| `Enter` | Підтвердити введену координату |
| `Escape` | Скасувати всі розміщені точки та вийти |

## Редагування ручками — коригування кінцевих точок та радіуса

Вибрана дуга показує три ручки:

| Ручка | Розташування | Що робить |
|-------|-------------|-----------|
| **Центр** | Геометричний центр описаного кола | Переміщує всю дугу; радіус та кути незмінні |
| **Початок** | Перша кінцева точка дуги | Перетягнути для переміщення початку по описаному колу — змінює початковий кут та радіус |
| **Кінець** | Остання кінцева точка дуги | Перетягнути для переміщення кінця по описаному колу — змінює кінцевий кут та радіус |

Перетягування ручки початку або кінця переміщує її в позицію перетягування та перераховує кут і радіус від цієї нової позиції відносно центру. Протилежна кінцева точка залишається фіксованою.

## Вибір дуг

| Метод | Поведінка |
|-------|-----------|
| **Клік** | Вибирає якщо клік потрапляє біля кривої дуги (не хорди) |
| **Перетягнути праворуч** (строгий) | Зразкові точки вздовж дуги повинні всі знаходитися всередині рамки |
| **Перетягнути ліворуч** (перехресний) | Будь-яка зразкова точка на дузі всередині рамки вибирає її |

## Підтримувані команди редагування

| Команда | Що відбувається з дугою |
|---------|------------------------|
| [Move](./move) | Переміщує центр; радіус та кути незмінні |
| [Copy](./copy) | Створює ідентичну дугу на новій позиції |
| [Rotate](./rotate) | Обертає центр та зміщує початковий/кінцевий кути на величину обертання |
| [Mirror](./mirror) | Відображає центр та інвертує початковий/кінцевий кути відносно дзеркальної осі |
| [Scale](./scale) | Масштабує позицію центру та множить радіус на коефіцієнт масштабу |
| [Offset](./offset) | Створює концентричну дугу з більшим або меншим радіусом, той самий кутовий охоплення |
| [Delete](./delete) | Видаляє дугу |

## Властивості

**Загальні**

| Властивість | За замовчуванням | Значення |
|-------------|-----------------|----------|
| Колір | 256 (ByLayer) | Індекс кольору ACI |
| Шар | `0` | Призначення шару |
| Тип лінії | ByLayer | Іменований шаблон типу лінії |
| Масштаб типу лінії | 1 | Коефіцієнт масштабу шаблону типу лінії |
| Товщина | 0 | Товщина екструзії |

**Геометрія**

| Властивість | Значення |
|-------------|----------|
| Центр X / Центр Y | Центр описаного кола |
| Радіус | Радіус описаного кола |
| Початковий кут | Кут у градусах де починається дуга (вимірюється від позитивної осі X) |
| Кінцевий кут | Кут у градусах де закінчується дуга |

## Arc проти Circle — коли що використовувати

| | Arc | Circle |
|---|-----|--------|
| Охоплення | Часткове — від першого до третього кліку | Повні 360° |
| Метод введення | Три точки на кривій | Центр + радіус (клік або введення) |
| Введення з клавіатури | Координата X,Y для кожної точки | Значення радіуса (центр також приймає X,Y) |
| Зміна розміру після розміщення | Перетягнути ручки початку/кінця | Перетягнути будь-яку кардинальну ручку |
| Найкраще для | Заокруглення, округлені кути, дугові шляхи | Повні отвори, круглі елементи |

## DXF — об'єкт ARC

Дуги зберігаються як об'єкти `ARC` у DXF файлі, що містять координати центру, радіус, початковий та кінцевий кути. Всі властивості — включаючи колір, шар, тип лінії, масштаб типу лінії та товщину — проходять round-trip без втрат. Будь-який DXF-сумісний додаток (LibreCAD, FreeCAD тощо) читає їх як стандартні дуги.
