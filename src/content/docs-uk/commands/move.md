# Move

Команда `move` переміщує вибрані об'єкти від базової точки до точки призначення. Зміщення, що застосовується до кожного вибраного об'єкта — це вектор від базової до точки призначення. Після переміщення всі об'єкти залишаються вибраними на новій позиції, готові до подальшого редагування.

## Два способи запуску

**Вибрати спочатку, потім перемістити** — виберіть об'єкти, потім активуйте:

1. Виберіть один або більше об'єктів на полотні.
2. Введіть `move` у терміналі або натисніть кнопку **Move** на панелі інструментів.
3. **Клацніть базову точку** або введіть `X,Y` та натисніть **Enter** для точної координати.
4. **Клацніть точку призначення** — всі вибрані об'єкти зміщуються на вектор база→призначення. Введення координат також працює тут.

**Активувати, потім вибрати** — запустіть команду без вибору:

1. Введіть `move` або натисніть кнопку на панелі інструментів.
2. **Виберіть об'єкти** — клацайте для перемикання окремих об'єктів або перетягуйте для вибору рамкою.
3. Натисніть **Enter** або **Пробіл** для підтвердження вибору.
4. **Клацніть базову точку**, потім **клацніть точку призначення** (введення координат доступне на обох кроках).

```
  До:                      Після:
  ● база                     → ● призначення
  [об'єкт A]                    [об'єкт A переміщений]
  [об'єкт B]                    [об'єкт B переміщений]
```

Привид попереднього перегляду всіх вибраних об'єктів слідує за курсором від базової точки до точки призначення, показуючи результат до кліку.

## Введення координат

На кроці базової точки або точки призначення введіть точну позицію замість клацання:

1. Введіть значення X.
2. Натисніть `,` — термінал показує `[X], [Y{cursor}]`.
3. Введіть значення Y.
4. Натисніть **Enter** для підтвердження.

## Блокування кута та точна відстань

Після встановлення базової точки команда відстежує вісь прив'язки під кутом 45° (0°, 45°, 90°, 135°, …). Напрямок **блокується** коли курсор достатньо далеко від бази і в межах одного розміру ручки від осі. При блокуванні:

- Привид попереднього перегляду прив'язується до осі.
- Введіть відстань та натисніть **Enter** для переміщення точно на цю відстань вздовж заблокованого напрямку.
- Клік проєктується на вісь, тому точка призначення завжди лежить точно на ній.

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.` | Додати цифру до значення відстані |
| `-` | Від'ємна відстань — змінює напрямок вздовж осі (лише перший символ) |
| `Backspace` | Видалити останній введений символ |
| `Enter` | Застосувати переміщення на введеній відстані |

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `Enter` / `Пробіл` | Підтвердити вибір та перейти до фази базової точки |
| `0`–`9`, `.`, `-` | Почати введення координати X або відстані при заблокованому куті |
| `,` | Заблокувати X та перейти до введення Y |
| `Backspace` | Видалити останній введений символ |
| `Enter` | Підтвердити координату або застосувати переміщення на введеній відстані |
| `Escape` | Скасувати та скинути |

## Активація Move через ручку

Клацання **ручки середньої точки** вибраної [Лінії](../line/) запускає Move автоматично, з уже встановленою середньою точкою як базовою точкою та активною фазою переміщення. Це найшвидший спосіб перепозиціонувати одну лінію без кроку вибору.

## Вибір об'єктів під час команди

Коли команда починається у фазі вибору:

| Метод | Поведінка |
|-------|-----------|
| **Клік** | Перемикає об'єкт під курсором у/з вибору |
| **Перетягнути праворуч** (строгий) | Додає об'єкти, що повністю знаходяться в рамці |
| **Перетягнути ліворуч** (перехресний) | Додає об'єкти, що перетинають межу рамки |
| **Enter** / **Пробіл** | Підтверджує вибір та переходить до фази базової точки |

## Після переміщення

Переміщені об'єкти залишаються вибраними на новій позиції. Це означає, що ви можете одразу:
- Запустити **Move** знову для тонкого налаштування.
- Запустити [Copy](../copy/), [Rotate](../rotate/) або [Scale](../scale/) без повторного вибору.
- Натиснути **Delete** для їх видалення.

## Move проти Copy

| | Move | Copy |
|---|------|------|
| Оригінальна позиція | Звільнена — об'єктів більше немає там | Збережена — оригінали залишаються на місці |
| Кількість результатів | Та ж кількість об'єктів | Один додатковий набір на операцію |
| Найкраще для | Перепозиціонування геометрії | Дублювання геометрії |

## Підтримувані об'єкти

Move працює з усіма типами об'єктів: Лінія, Полілінія, Прямокутник, Коло, Дуга, Еліпс, Текст, Сплайн, Розміри, Виноски та всі інші.
