# Dimension Continue

Команда `dimcontinue` ланцюжком додає нові розміри від **другої виносної лінії** існуючого розміру. Кожен новий сегмент розміщується вздовж тієї ж осі вимірювання та на тому ж відступі лінії розміру що й базовий. Всі властивості стилю — розмір стрілки, висота тексту, довжини виносних ліній — копіюються з базового автоматично.

## Як виглядають ланцюгові розміри

```
  |←— 3.00 —→|←— 2.50 —→|←— 4.00 —→|
  |           |           |           |
  ●           ●           ●           ●
  p1        p2 (база     p3           p4
           ext2 → новий початок)
```

Кожен прямокутник — окремий об'єкт `DIMENSION`. Вони мають однакову позицію лінії розміру та напрямок вимірювання.

## Початок ланцюга

1. Введіть `dimcontinue` у терміналі або натисніть кнопку **Dimension Continue** на панелі інструментів.
2. **Якщо розмір щойно розміщено** — команда автоматично підхоплює його як базу (клацати не потрібно).
3. **Якщо жодного нещодавнього розміру немає** — клацніть будь-який існуючий розмір для використання як бази.
4. **Клацніть початок наступної виносної лінії** — попередній перегляд показує новий розмір при русі курсора. Або введіть `X,Y` та натисніть **Enter** для точної координати.
5. Продовжуйте клацати (або вводити) для розширення ланцюга. Кожен розміщений розмір автоматично стає новою базою.
6. Натисніть **Enter**, **Пробіл** або **Escape** для завершення ланцюга.

## Що успадковується від базового розміру

| Властивість | Успадковується від бази |
|-------------|-------------------------|
| Напрямок вимірювання / кут | Так — зафіксований для всього ланцюга |
| Відступ лінії розміру (відстань від виміряних точок) | Так |
| Розмір стрілки | Так |
| Висота тексту | Так |
| Відступ та розширення виносної лінії | Так |
| Вирівнювання тексту | Так |
| Ім'я стилю | Так |
| Колір, Шар | Не успадковується — використовується поточний шар |

## Фіксація напрямку вимірювання

Напрямок вимірювання ланцюга **зафіксований до кута базового розміру**:

- Лінійна база (H) → всі продовження вимірюють горизонтальну відстань (Δ X).
- Лінійна база (V) → всі продовження вимірюють вертикальну відстань (Δ Y).
- Вирівнена база під будь-яким кутом → всі продовження вимірюють вздовж того ж кута.

Не можна змінити напрямок в середині ланцюга. Почніть новий [Dimension Linear](./dim-linear) або [Dimension Aligned](./dim-aligned) для вимірювання в іншому напрямку.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.`, `-` | Почати введення координати X |
| `,` | Заблокувати X та перейти до введення Y |
| `Backspace` | Видалити останній введений символ |
| `Enter` | Підтвердити введену координату або завершити ланцюг якщо введення не активне |
| `Пробіл` / `Escape` | Завершити ланцюг |

## Dimension Continue проти початку заново

| | Dimension Continue | Dimension Linear / Aligned |
|---|-------------------|--------------------------|
| Початкова точка | Зафіксована в ext2 останньої бази | Клацніть будь-де |
| Кут | Зафіксований до бази | Вільний |
| Відступ | Успадкований від бази | Встановлюється курсором або введенням |
| Стиль | Успадкований від бази | Поточний стиль |
| Найкраще для | Сукупні вимірювання вздовж ряду | Перший розмір або зміна напрямку |

## Редагування міток після розміщення — simple mode

**Двічі клацніть** будь-який розмір у ланцюгу, щоб відкрити текстовий редактор у режимі **simple**. Кожен сегмент є незалежним і може бути відредагований окремо.

| Функція | Поведінка |
|---------|-----------|
| Bold / Italic / Font / Height | Застосовується до **всієї** мітки одночасно |
| Форматування окремих символів | Не підтримується |
| `Enter` | Підтверджує значення та закриває редактор |
| Багаторядковість | Не підтримується |

Дивіться [Текстовий редактор — simple mode](../interface/text-editor#simple-mode) для повного довідника.

## DXF — об'єкти DIMENSION

Кожен сегмент ланцюга зберігається як незалежний об'єкт `DIMENSION` у файлі DXF. Вони не пов'язані у файлі — вони мають спільні властивості тому що були створені з тієї ж бази, але кожен може бути відредагований окремо після розміщення.
