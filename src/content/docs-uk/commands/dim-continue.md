# Dimension Continue

Команда `dim continue` додає новий розмір, що починається там, де закінчується попередній — створює ланцюг послідовних розмірів на одній лінії розміру.

## Початок ланцюга

Dim Continue автоматично підхоплює **останній розміщений розмір** (Dim Linear або Dim Aligned) — не потрібно клацати на попередній розмір. Якщо жодного розміру ще немає, спочатку розмістіть його за допомогою Dim Linear або Dim Aligned.

## Як продовжити ланцюг

1. Розмістіть перший розмір за допомогою [Dim Linear](./dim-linear) або [Dim Aligned](./dim-aligned).
2. Введіть `dim continue` або натисніть кнопку **Dim Continue**.
3. **Клацніть наступну точку прив'язки** — новий розмір додається від кінцевої точки попереднього до цієї нової точки.
4. Продовжуйте клацати для додавання більше розмірів у ланцюзі.
5. Натисніть **Escape** або **Enter** для завершення.

## Схема ланцюгового розміру

```
  ├──── 30 ────┼──── 45 ────┼──── 25 ────┤
  ●             ●            ●            ●
  (1-й)         (2-й)       (3-й)        (4-й клік)
```

Всі розміри вирівняні по одній лінії розміру. Кожен наступний використовує кінцеву точку попереднього як свою початкову точку.

## Що успадковується від попереднього розміру

| Атрибут | Успадковується? |
|---------|-----------------|
| Орієнтація (горизонт./вертик.) | Так |
| Висота лінії розміру | Так |
| Шар | Так |
| Стиль тексту | Так |

## Фіксація напрямку вимірювання

Якщо базовий розмір — Dim Linear:
- Натисніть **H** для фіксації горизонтального ланцюга.
- Натисніть **V** для фіксації вертикального ланцюга.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.`, `-` | Почати введення координати X |
| `,` | Заблокувати X та перейти до введення Y |
| `H` | Зафіксувати горизонтальний напрямок |
| `V` | Зафіксувати вертикальний напрямок |
| `Backspace` | Видалити останній введений символ |
| `Enter` | Підтвердити координату або завершити ланцюг |
| `Escape` | Завершити ланцюг |

## Dim Continue проти нових розмірів

| | Dim Continue | Новий Dim Linear |
|---|-------------|-----------------|
| Початкова точка | Автоматично — кінцева точка попереднього | Вибирається вручну |
| Вирівнювання | Автоматично по тій же лінії розміру | Задається вручну |
| Найкраще для | Ряди отворів, рівновіддалені елементи | Довільне розміщення |

## DXF — об'єкти DIMENSION

Кожен сегмент ланцюга зберігається як **незалежний** об'єкт `DIMENSION` у DXF. Зв'язок між ними — лише положення (спільні точки прив'язки), а не групування.
