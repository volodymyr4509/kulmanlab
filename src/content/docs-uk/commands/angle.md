# Angle

Команда `angle` вимірює внутрішній кут між двома лініями, заданими трьома точками (зенітна — спільна вершина). Результат відображається лише у терміналі.

## Анатомія вимірювання

```
  Точка 1
   \
    \ ← кут
     ● Зенітна точка
    /
   /
  Точка 2
```

Вимірюється кут від лінії Зенітна→Точка1 до лінії Зенітна→Точка2.

## Конвенція внутрішнього кута

Angle завжди повертає **внутрішній кут** (від 0° до 180°) між двома напрямками, використовуючи скалярний добуток:

- Паралельні лінії (один напрямок): `0.0000°`
- Перпендикулярні лінії: `90.0000°`
- Протилежні напрямки (колінеарні): `180.0000°`

Результат завжди невід'ємний та ніколи не перевищує 180°.

## Ланцюгове вимірювання

Після першого вимірювання команда залишається активною. Наступний клік стає новою **Точкою 1** того ж зеніту, тому можна вимірювати кілька кутів навколо однієї вершини без перезапуску.

Для вимірювання нового кута з іншим зенітом натисніть `Escape` та запустіть `angle` знову.

## Введення координат

На будь-якому кроці введення точки введіть точну позицію замість клацання:

1. Введіть значення X.
2. Натисніть `,` — термінал показує `[X], [Y{cursor}]`.
3. Введіть значення Y.
4. Натисніть **Enter** для підтвердження.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.`, `-` | Почати введення координати X |
| `,` | Заблокувати X та перейти до введення Y |
| `Backspace` | Видалити останній введений символ |
| `Enter` | Підтвердити координату |
| `Escape` | Скасувати та скинути |

## Примітки

- Результати відображаються **лише у терміналі** — ніяких розмірних об'єктів на полотні не створюється.
- Точність: **4 десяткові знаки** (наприклад, `37.5000°`).
- Для вимірювання кута між лініями на кресленику, що вже є, використовуйте [Dim Angular](./dim-angular) — він розміщує кутовий розмір безпосередньо на кресленику.
