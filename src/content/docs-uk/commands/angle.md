# Angle

Команда `angle` вимірює внутрішній кут у вершині, утвореній двома відрізками прямих через три клікнуті точки. Результат — завжди між 0° та 180° — відображається у терміналі з точністю 4 десяткові знаки. Це одна з двох команд вимірювання — [Distance](../distance/) вимірює довжину прямої лінії натомість.

## Анатомія вимірювання кута

```
  ● перша точка (кінець першого променя)
   \
    \  попередній перегляд першого променя
     \
      ● вершина (крок 3)
     /
    /  попередній перегляд другого променя (до курсора)
   /
  ● третя точка  →  термінал: "Angle: 45.0000°"
```

- **Перша точка** — один кінець кута (крок 2).
- **Вершина** — кут де зустрічаються два промені (крок 3).
- **Третя точка** — інший кінець кута (крок 4).

## Вимірювання кута

1. Введіть `angle` у терміналі або натисніть кнопку **Angle** на панелі інструментів.
2. **Клацніть першу точку** — один кінець кута. Або введіть `X,Y` та натисніть **Enter** для точної координати.
3. **Клацніть вершину** — кут де зустрічаються два промені. Введення координат також працює тут.
4. **Клацніть третю точку** — другий кінець кута. Введення координат також працює тут. Розміщення цієї точки виводить результат.
5. **Клацніть ще раз** (необов'язково) для початку нового вимірювання з того кліку як нова перша точка.

## Конвенція внутрішнього кута

Команда обчислює кут, використовуючи скалярний добуток двох променів від вершини:

- **Завжди внутрішній**: результат є меншим кутом між 0° та 180°.
- Порядок кліку кінцевих точок не впливає на результат — важлива лише позиція вершини.
- Колінеарні точки (всі три на одній лінії) повертають 0° або 180°.

## Ланцюгові вимірювання

Після появи результату, клацання негайно починає наступне вимірювання — клікнута точка стає новою першою точкою. Команда ніколи не виходить автоматично поки не натиснути `Escape`.

## Angle проти Distance

| | Angle | Distance |
|---|-------|---------|
| Що вимірює | Внутрішній кут у вершині | Довжина прямої лінії |
| Кількість кліків | 3 | 2 |
| Формат результату | `45.0000°` | `12.3456` (одиниці) |
| Попередній перегляд на полотні | Дві лінії від вершини до обох кінців | Лінія від першої точки до курсора |
| Найкраще для | Кут відкриття між двома елементами | Довжина зазору або сегмента |

## Введення координат

Замість клацання, введіть точну позицію для будь-якої з трьох точок:

1. Введіть значення X.
2. Натисніть `,` — термінал покаже `[X], [Y{cursor}]`.
3. Введіть значення Y.
4. Натисніть **Enter** для підтвердження.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.`, `-` | Почати введення координати X |
| `,` | Заблокувати X та перейти до введення Y |
| `Backspace` | Видалити останній введений символ |
| `Enter` | Підтвердити введену координату |
| `Escape` | Скасувати та скинути до кроку 2 |

## Примітки

- Результати показуються **лише у терміналі** — нічого не додається до кресленика.
- Точність завжди 4 десяткові знаки в градусах.
- Для вимірювання кута між лініями на кресленику, що вже є, використовуйте [Dim Angular](../dim-angular/) — він розміщує кутовий розмір безпосередньо на кресленику.
