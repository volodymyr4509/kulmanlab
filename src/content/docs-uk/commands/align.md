# Align

Команда `align` змінює позицію вибраних об'єктів за однією або двома парами точок джерела/призначення. З однією парою вона працює точно як [Move](../move/) (лише перенесення). З двома парами вона також обертає вибір так, щоб напрямок джерело-до-джерела збігався з напрямком призначення-до-призначення, і опціонально може масштабувати його так, щоб довжина відрізка джерела відповідала довжині відрізка призначення — перенесення, обертання та масштабування в одній операції.

## Два способи запуску

**Вибрати спочатку, потім вирівняти** — виберіть об'єкти, потім активуйте:

1. Виберіть один або більше об'єктів на полотні.
2. Введіть `align` у терміналі або натисніть кнопку **Align** на панелі інструментів.
3. **Клацніть першу точку джерела (S1)**, потім **першу точку призначення (D1)**.
4. **Клацніть другу точку джерела (S2)**, або натисніть **Enter**, щоб одразу застосувати вирівнювання лише з перенесенням.
5. **Клацніть другу точку призначення (D2)**.
6. Дайте відповідь на запит масштабування: натисніть **Y** для масштабування, або **N** / **Enter**, щоб зберегти оригінальний розмір.

**Активувати, потім вибрати** — запустіть команду без вибору:

1. Введіть `align` або натисніть кнопку на панелі інструментів.
2. **Виберіть об'єкти** — клацайте для перемикання окремих об'єктів або перетягуйте для вибору рамкою.
3. Натисніть **Enter** або **Пробіл** для підтвердження вибору.
4. Продовжуйте: S1 → D1 → S2 → D2 → запит масштабування, як описано вище.

> Терміналу потрібно лише стільки літер, скільки достатньо для однозначності — введення `al` та натискання **Enter** активує Align напряму, оскільки жодна інша команда не починається з цих двох літер.

## Анатомія вирівнювання

```
  Точки джерела (на об'єктах):         Точки призначення:
  ● S1                                 ● D1
   \                                    \
    ● S2                                 ● D2

  Результат: вибір переноситься так, щоб S1 опинилася на D1,
  потім обертається навколо D1 так, щоб напрямок S1→S2
  збігався з напрямком D1→D2 — і, якщо ви обираєте
  масштабування, змінюється розмір так, щоб |S1S2| стало |D1D2|.
```

Привид попереднього перегляду слідує за курсором на кожному кроці: попередній перегляд переміщення під час розміщення D1, потім пунктирний обертовий попередній перегляд, коли розміщується D2.

## Вирівнювання за однією точкою (лише перенесення)

Після розміщення D1 натисніть **Enter** замість клацання другою точкою джерела. Вибір переноситься на вектор S1→D1 — без обертання чи масштабування — ідентично [Move](../move/) з S1 як базовою точкою та D1 як призначенням.

## Вирівнювання за двома точками (перенесення + обертання + опціональне масштабування)

Коли S2 і D2 обидві розміщені:

- **Кут обертання** — різниця між напрямком призначення (`D1 → D2`) та напрямком джерела (`S1 → S2`).
- **Запит масштабування** — з'являється `scale objects to alignment points? [Yes/No] <N>`, з **No** за замовчуванням:
  - Натисніть **Y**, щоб також рівномірно масштабувати вибір навколо D1 так, щоб відстань `S1–S2` стала відстанню `D1–D2`.
  - Натисніть **N** або **Enter**, щоб зберегти оригінальний розмір — застосовуються лише перенесення та обертання.

Натискання клавіші на запиті масштабування одразу застосовує вирівнювання — окремого кроку підтвердження після вибору Так чи Ні немає.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `Enter` / `Пробіл` | Підтвердити вибір і перейти до фази S1 |
| `Enter` (на кроці S2) | Пропустити обертання — застосувати вирівнювання лише з перенесенням, використовуючи S1 і D1 |
| `Y` | Застосувати вирівнювання з масштабуванням |
| `N` / `Enter` (на запиті масштабування) | Застосувати вирівнювання без масштабування |
| `Escape` | Під час розміщення точок: відкидає точки та повертається до фази вибору; без вибору: скасовує команду |

## Вибір об'єктів під час команди

Коли команда починається у фазі вибору:

| Метод | Поведінка |
|-------|-----------|
| **Клік** | Перемикає об'єкт під курсором у/з вибору |
| **Перетягнути праворуч** (строгий) | Додає об'єкти, що повністю знаходяться в рамці |
| **Перетягнути ліворуч** (перехресний) | Додає об'єкти, що перетинають межу рамки |
| **Enter** / **Пробіл** | Підтверджує вибір і переходить до фази S1 |

## Після вирівнювання

Вирівняні об'єкти залишаються вибраними на новій позиції, і команда завершується автоматично — запустіть **Align** знову, або перемкніться на [Move](../move/), [Rotate](../rotate/) чи [Scale](../scale/) без повторного вибору.

## Align проти Move

| | Align | Move |
|---|-------|------|
| Пари точок | 1 (лише перенесення) або 2 (перенесення + обертання + масштабування) | 1 (лише перенесення) |
| Обертання | Так, з другою парою точок | Ні |
| Масштабування | Опціональне, з другою парою точок | Ні |
| Найкраще для | Підгонка однієї форми до іншої за опорними точками | Просте перепозиціонування |

## Підтримувані об'єкти

Align працює з усіма типами об'єктів, що підтримуються Move, Rotate та Scale — ті ж операції `translate`, `rotate` і `scale`, які використовують ці команди, застосовуються послідовно, тож жоден тип не виключено.
