# Rectangle

Команда `rectangle` малює прямокутник, паралельний осям, шляхом клацання двох протилежних кутів. Сторони завжди горизонтальні та вертикальні — для повернутих форм скористайтесь [Polyline](./polyline) та [Rotate](./rotate) або намалюйте за допомогою [Line](./line).

Прямокутники зберігаються як замкнені `LWPOLYLINE` з чотирма вершинами — на рівні DXF вони неможливо відрізнити від замкненої [Polyline](./polyline).

## Малювання прямокутника

1. Введіть `rectangle` у терміналі або натисніть кнопку **Rectangle** на панелі інструментів.
2. **Клацніть перший кут** або введіть `X,Y` та натисніть **Enter** для точної координати.
3. **Клацніть протилежний кут** — прямокутник розміщується і команда завершується. Введення координат також працює тут.

```
  ●──────────────────●  ← 2-й клік
  │                  │
  │   попередній     │
  │    перегляд      │
  │                  │
  ●──────────────────  ← 1-й клік (або навпаки)
```

Попередній перегляд у реальному часі показує прямокутник від першого кута до позиції курсора. Будь-який порядок кутів працює — ви можете клацнути будь-який кут першим.

## Введення координат

Для точного розміщення:

1. Введіть значення X.
2. Натисніть `,` — термінал покаже `[X], [Y{cursor}]`.
3. Введіть значення Y.
4. Натисніть **Enter** для розміщення кута та переходу до наступного.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.`, `-` | Почати введення координати X |
| `,` | Заблокувати X та перейти до введення Y |
| `Backspace` | Видалити останній введений символ |
| `Enter` | Підтвердити введену координату |
| `Escape` | Скасувати та вийти |

## Редагування ручками

Оскільки прямокутник — це замкнена полілінія, він має ті ж ручки, що й [Polyline](./polyline):

| Ручка | Розташування | Що робить |
|-------|-------------|-----------|
| **Вершина** | Кожен з 4 кутів | Перетягнути для переміщення кута — обидва прилеглі відрізки розтягуються |
| **Середина відрізка** | Центр кожного з 4 боків | Перетягнути для переміщення всього боку паралельно — змінює ширину або висоту |

> Перетягування вершини більше не зберігає форму прямокутника — після цього це стає загальною полілінією. Для паралельного переміщення боку використовуйте ручку середини відрізка.

## Вибір прямокутників

| Метод | Поведінка |
|-------|-----------|
| **Клік на контурі** | Вибирає прямокутник |
| **Клік всередині** | Не вибирає |
| **Рамка праворуч** (строга) | Вибирає лише якщо весь прямокутник всередині |
| **Рамка ліворуч** (перехресна) | Вибирає якщо контур перетинає рамку |

## Підтримувані команди редагування

| Команда | Що відбувається |
|---------|-----------------|
| [Move](./move) | Переміщує всі 4 вершини |
| [Copy](./copy) | Створює ідентичний прямокутник |
| [Rotate](./rotate) | Обертає всі 4 вершини навколо базової точки |
| [Mirror](./mirror) | Відображає всі 4 вершини |
| [Scale](./scale) | Масштабує всі 4 вершини від базової точки |
| [Offset](./offset) | Створює менший або більший паралельний прямокутник |
| [Delete](./delete) | Видаляє прямокутник |

## Rectangle проти Polyline проти Line

| | Rectangle | Polyline | Line |
|---|-----------|----------|------|
| Кути | Лише 90° | Будь-які | Будь-які |
| Замкнений | Завжди | Опціонально | Ніколи |
| Offset | Так | Так | Так |
| Trim/Extend | Ні | Ні | Так |
| DXF | LWPOLYLINE (замкнена) | LWPOLYLINE | LINE |

## Властивості

Ті ж поля що й для [Polyline](./polyline): колір, шар, тип лінії, масштаб типу лінії, товщина, прапор закриття, вершини.

## DXF — об'єкт LWPOLYLINE

Прямокутники зберігаються як замкнені об'єкти `LWPOLYLINE` з 4 вершинами. При відкритті DXF файлу, що містить прямокутники, намальовані в інших CAD програмах, вони також показуватимуться як замкнені полілінії — немає окремого типу "прямокутника" в DXF.
