# Rectangle

Команда `rectangle` малює прямокутник вирівняний по осях, визначений двома кліками протилежних кутів. Результат зберігається як **замкнений `LWPOLYLINE`** з чотирма вершинами — по одній в кожному куті. Немає окремого типу об'єкта прямокутник: після створення форма поводиться точно як будь-яка інша [Polyline](../polyline/) і кожне редагування полілінії застосовується до неї.

## Малювання прямокутника

1. Введіть `rectangle` у терміналі або натисніть кнопку **Rectangle** на панелі інструментів.
2. **Клацніть перший кут**, або введіть `X,Y` та натисніть **Enter** для точної координати.
3. **Клацніть протилежний кут** — прямокутник розміщується миттєво і команда завершується. Введення координат також працює тут.

```
  ● (перший клік)────────────┐
  |                          |
  |   живий попередній       |
  |   перегляд слідує        |
  |   за курсором після кроку 2  |
  └──────────────────────────● (другий клік)
```

Два кліки можуть бути будь-якою парою діагонально протилежних кутів — верхній лівий + нижній правий, або нижній лівий + верхній правий тощо. Порядок не має значення.

## Введення координат

На будь-якому кроці кута можна ввести точну позицію:

1. Введіть значення X.
2. Натисніть `,` — термінал покаже `[X], [Y{cursor}]`.
3. Введіть значення Y.
4. Натисніть **Enter** для розміщення кута.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.`, `-` | Почати введення координати X |
| `,` | Заблокувати X та перейти до введення Y |
| `Enter` | Підтвердити введену координату |
| `Escape` | Скасувати |

Сторони завжди горизонтальні та вертикальні — немає блокування кута для команди rectangle.

## Редагування ручками — зміна форми після створення

Вибраний прямокутник показує ручки в кожній вершині та в середній точці кожної сторони:

| Ручка | Розташування | Що робить |
|-------|-------------|-----------|
| **Кут** | Кожна з 4 вершин | Перетягнути для переміщення тієї вершини; дві суміжні сторони розтягуються слідуючи — протилежний кут залишається зафіксованим |
| **Середня точка сторони** | Центр кожної з 4 сторін | Перетягнути для переміщення обох кінцевих точок тієї сторони разом, зберігаючи довжину та кут сторони |

Перетягування ручки кута перетворює прямокутник у нечотирикутний чотирикутник. Якщо потрібен прямокутник іншого розміру, перетягніть кут зберігаючи сторони приблизно ортогональними, або видаліть та намалюйте новий.

## Вибір прямокутників

Оскільки прямокутник є полілінією, вибір працює однаково:

| Метод | Поведінка |
|-------|-----------|
| **Клік** | Вибирає якщо клік потрапляє на будь-яку з чотирьох сторін |
| **Перетягнути праворуч** (строгий) | Всі чотири вершини мають знаходитися всередині рамки вибору |
| **Перетягнути ліворуч** (перехресний) | Будь-яка сторона що перетинає межу рамки вибирає весь прямокутник |

## Підтримувані команди редагування

Всі команди редагування полілінії застосовуються. Trim та Extend призначені лише для [Line](../line/) і не працюють з прямокутниками:

| Команда | Що відбувається з прямокутником |
|---------|---------------------------------|
| [Move](../move/) | Переміщує всі чотири вершини на однакове зміщення |
| [Copy](../copy/) | Створює ідентичний прямокутник на новій позиції |
| [Rotate](../rotate/) | Обертає всі чотири вершини навколо обраної базової точки |
| [Mirror](../mirror/) | Відображає всі чотири вершини через вісь дзеркала |
| [Scale](../scale/) | Масштабує всі чотири вершини рівномірно від базової точки |
| [Offset](../offset/) | Створює паралельний (вставлений або зовнішній) прямокутник на фіксованій відстані |
| [Delete](../delete/) | Видаляє прямокутник з кресленика |

## Властивості

Коли прямокутник вибраний, панель властивостей показує ті ж поля що й будь-яка полілінія:

**Загальні**

| Властивість | Стандарт | Значення |
|-------------|---------|---------|
| Колір | 256 (ByLayer) | Індекс кольору ACI |
| Шар | `0` | Призначення шару |
| Тип лінії | ByLayer | Іменований шаблон типу лінії |
| Масштаб типу лінії | 1 | Коефіцієнт масштабу шаблону типу лінії |
| Товщина | 0 | Товщина видавлювання |

**Геометрія**

| Властивість | Значення |
|-------------|---------|
| Замкнений | Завжди `true` для прямокутника |
| Кількість вершин | Завжди `4` для незміненого прямокутника |
| Вершини | Координати всіх чотирьох кутів |

## Rectangle проти Polyline проти Line

| | Rectangle | Polyline | Line |
|---|-----------|---------|------|
| Як малювати | 2 кліки (кути) | Клацнути кожну вершину | Клацнути кожну кінцеву точку |
| Тип об'єкта | Замкнений `LWPOLYLINE` | Відкритий або замкнений `LWPOLYLINE` | `LINE` на сегмент |
| Сторони завжди ортогональні | Так (при створенні) | Ні | Ні |
| Trim / Extend | Ні | Ні | Так |
| Найкраще для | Рамки, прямокутні ділянки | Довільні контури та шляхи | Окремі сегменти, допоміжні лінії |

## DXF — об'єкт LWPOLYLINE

Прямокутники зберігаються як замкнені об'єкти `LWPOLYLINE` з чотирма вершинами. Всі властивості — координати вершин, колір, шар, тип лінії, масштаб типу лінії та товщина — проходять round-trip без втрат.

Немає окремого типу `RECTANGLE` в DXF. При повторному відкритті файлу, форма з'являється як замкнена чотиривершинна полілінія, а не прямокутник. Будь-який переглядач або редактор DXF що підтримує `LWPOLYLINE` (LibreCAD, FreeCAD тощо) відобразить його правильно.
