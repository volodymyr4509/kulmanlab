# New File

Команда `new file` негайно починає новий порожній кресленик. Натисніть значок **нової сторінки** на панелі інструментів або введіть `new file` у терміналі.

> **Попередження:** New File **не запитує підтвердження** та **не зберігає** поточний кресленик автоматично. Будь-яка незбережена робота буде втрачена. Якщо потрібно зберегти поточний кресленик — спочатку скористайтеся [Export](../export/).

## Що містить новий файл

| Аспект | Значення |
|--------|---------|
| Об'єкти | Жодного — порожнє полотно |
| Шари | Лише шар за замовчуванням (Layer 0) |
| Назва файлу | Генерується автоматично: `kulman-Mon01_HH:MM:SS.dxf` |

## New File проти Import

| | New File | Import |
|---|----------|--------|
| Вміст | Порожнє полотно | Вміст вибраного файлу |
| Попередній кресленик | Замінюється (без попередження) | Замінюється (без попередження) |
| Найкраще для | Початок нового кресленика | Відкриття існуючого DXF або JSON |

## Пов'язані команди

- [Files](../files/) — переглядати та відновлювати збережені кресленики
- [Export](../export/) — завантажити поточний кресленик перед початком нового
- [Import](../import/) — відкрити існуючий файл замість початку з нуля
