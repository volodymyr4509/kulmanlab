# Mirror

Команда `mirror` створює дзеркальні копії вибраних об'єктів відносно осі, заданої двома точками. Оригінали **завжди зберігаються** — на відміну від [Move](../move/) або [Rotate](../rotate/), Mirror не змінює існуючі об'єкти; вона лише додає нові.

## Два способи запуску

**Вибрати спочатку, потім відобразити** — виберіть об'єкти, потім активуйте:

1. Виберіть один або більше об'єктів на полотні.
2. Введіть `mirror` у терміналі або натисніть кнопку **Mirror** на панелі інструментів.
3. **Клацніть першу точку** осі дзеркала, або введіть `X,Y` та натисніть **Enter** для точної координати.
4. **Клацніть другу точку** — дзеркальні копії розміщуються і команда завершується. Введення координат також працює тут.

**Активувати, потім вибрати** — запустіть команду без вибору:

1. Введіть `mirror` або натисніть кнопку на панелі інструментів.
2. **Виберіть об'єкти** — клацайте для перемикання або перетягуйте для вибору рамкою.
3. Натисніть **Enter** або **Пробіл** для підтвердження вибору.
4. **Клацніть першу точку**, потім **клацніть другу точку** осі дзеркала (введення координат доступне на обох кроках).

```
  Оригінал:         Вісь:       Результат:
                    |
  [об'єкт A]   →   |   →   [об'єкт A] + [дзеркало A]
                    |
```

Живий попередній перегляд дзеркальних копій слідує за курсором поки ви позиціонуєте другу точку осі.

## Вісь дзеркала

Вісь — нескінченна пряма через дві клікнуті точки. Вона може бути під будь-яким кутом:

- Перемістіть курсор близько до **осі прив'язки під кутом 45°** (0°, 45°, 90°, 135°, …) — вісь блокується до цього кута, що зручно для чистих горизонтальних, вертикальних або діагональних відображень.
- Клацніть поза зоною прив'язки для осі під довільним кутом.

## Введення координат

На будь-якому кроці введення точки осі введіть точну позицію замість клацання:

1. Введіть значення X.
2. Натисніть `,` — термінал показує `[X], [Y{cursor}]`.
3. Введіть значення Y.
4. Натисніть **Enter** для підтвердження.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `Enter` / `Пробіл` | Підтвердити вибір |
| `0`–`9`, `.`, `-` | Почати введення координати X |
| `,` | Заблокувати X та перейти до введення Y |
| `Backspace` | Видалити останній введений символ |
| `Enter` | Підтвердити введену координату |
| `Escape` | Скасувати та скинути |

## Вибір об'єктів під час команди

| Метод | Поведінка |
|-------|-----------|
| **Клік** | Перемикає об'єкт під курсором |
| **Перетягнути праворуч** (строгий) | Додає об'єкти, що повністю знаходяться в рамці |
| **Перетягнути ліворуч** (перехресний) | Додає об'єкти, що перетинають межу рамки |
| **Enter** / **Пробіл** | Підтверджує вибір |

## Що відображається

Підтримуються всі типи об'єктів. Геометрія відображається математично відносно осі:

| Об'єкт | Що змінюється |
|--------|--------------|
| Лінія | Обидві кінцеві точки відображаються |
| Коло | Центр відображається; радіус незмінний |
| Дуга | Центр відображається; початковий та кінцевий кути перераховуються відносно осі |
| Еліпс | Центр відображається; напрямок головної осі відображається відносно осі |
| Полілінія / Прямокутник | Кожна вершина відображається |
| Текст | Точка вставки відображається; текстовий рядок **не** перевертається |
| Сплайн | Всі контрольні вершини / точки підгонки відображаються |

## Mirror проти Copy

| | Mirror | Copy |
|---|--------|------|
| Оригінали | Завжди зберігаються | Завжди зберігаються |
| Позиція нових об'єктів | Відображена відносно осі | Зміщена на вектор переміщення |
| Найкраще для | Симетричні деталі, двосторонні елементи | Повторення геометрії в будь-якому напрямку |
