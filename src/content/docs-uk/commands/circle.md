# Circle

Команда `circle` малює коло, визначене центральною точкою та радіусом. Після клацання центру ви можете встановити радіус або клацнувши другу точку на полотні, або ввівши точне число — обидва варіанти доступні одночасно.

## Малювання кола

1. Введіть `circle` у терміналі або натисніть кнопку **Circle** на панелі інструментів.
2. **Клацніть центральну точку** або введіть `X,Y` та натисніть **Enter** для точної координати.
3. Встановіть радіус — або:
   - **Клацніть будь-яку точку** на полотні — відстань від центру стає радіусом, або
   - **Введіть радіус** та натисніть **Enter** для точного значення.

Коло розміщується негайно і команда завершується.

```
  центр ●
         \  лінія радіуса (попередній перегляд)
          \
           ● ← клацнути тут, або ввести число
```

Під час фази радіуса живий попередній перегляд показує коло на поточній відстані курсора та малює лінію радіуса від центру до поточної точки.

## Введення координат центру

Замість клацання можна ввести позицію центру:

1. Введіть значення X.
2. Натисніть `,` — термінал покаже `[X], [Y{cursor}]`.
3. Введіть значення Y.
4. Натисніть **Enter** для розміщення центру та переходу до введення радіуса.

## Введення радіуса з клавіатури

Після розміщення центру введення одразу формує значення радіуса:

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.` | Додати цифру до значення радіуса |
| `Backspace` | Видалити останній введений символ |
| `Enter` | Розмістити коло із введеним радіусом |

Накопичене значення відображається у запиті терміналу (наприклад, `enter radius of circle: 25`). Попередній перегляд оновлюється, щоб показати введений радіус, поки курсор контролює напрямок маркера лінії радіуса.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.`, `-` | Почати введення координати X (фаза центру) або цифра радіуса (фаза радіуса) |
| `,` | Заблокувати X та перейти до введення Y (фаза центру) |
| `Backspace` | Видалити останній введений символ |
| `Enter` | Підтвердити введену координату або радіус |
| `Escape` | Скасувати та скинути |

## Редагування ручками — зміна радіуса

Вибране коло показує п'ять ручок:

| Ручка | Розташування | Що робить |
|-------|-------------|-----------|
| **Центр** | Центральна точка | Переміщує все коло; радіус залишається незмінним |
| **Ліво** | Крайня ліва точка (центр − радіус) | Перетягнути для встановлення нового радіуса = відстань до центру |
| **Право** | Крайня права точка (центр + радіус) | Перетягнути для встановлення нового радіуса = відстань до центру |
| **Верх** | Найвища точка | Перетягнути для встановлення нового радіуса = відстань до центру |
| **Низ** | Найнижча точка | Перетягнути для встановлення нового радіуса = відстань до центру |

Всі чотири кардинальні ручки поводяться однаково — новий радіус дорівнює відстані від центру до позиції перетягування. Центр залишається фіксованим.

## Вибір кіл

| Метод | Поведінка |
|-------|-----------|
| **Клік** | Вибирає якщо клік потрапляє біля кола |
| **Перетягнути праворуч** (строгий) | Весь обмежувальний квадрат (центр ± радіус) повинен знаходитися всередині рамки |
| **Перетягнути ліворуч** (перехресний) | Будь-яка частина кола, що перетинає або торкається межі рамки, вибирає коло |

## Властивості

**Загальні**

| Властивість | За замовчуванням | Значення |
|-------------|-----------------|----------|
| Колір | 256 (ByLayer) | Індекс кольору ACI |
| Шар | `0` | Призначення шару |
| Тип лінії | ByLayer | Іменований шаблон типу лінії |
| Масштаб типу лінії | 1 | Коефіцієнт масштабу шаблону типу лінії |
| Товщина | 0 | Товщина екструзії |

**Геометрія**

| Властивість | Значення |
|-------------|----------|
| Центр X / Центр Y | Координати центральної точки |
| Радіус | Радіус кола в одиницях кресленика |

## Circle проти Arc — коли що використовувати

| | Circle | Arc |
|---|--------|-----|
| Охоплення | Повні 360° | Часткове — визначається початковим та кінцевим кутом |
| Як малювати | Центр + радіус | Три точки на кривій |
| Введення з клавіатури | Значення радіуса | Немає — лише кліком |
| Ручка зміни розміру | 4 кардинальні точки | Початкова та кінцева точки (кут + радіус) |
| Розміри | Радіус: [Dim Radius](../dim-radius/) · Діаметр: [Dim Diameter](../dim-diameter/) | [Dim Radius](../dim-radius/) |
| Найкраще для | Повні отвори, кола для болтів, круглі елементи | Заокруглення, часткові криві, дугові шляхи |

## DXF — об'єкт CIRCLE

Кола зберігаються як об'єкти `CIRCLE` у DXF файлі. Координати центру, радіус, колір, шар, тип лінії, масштаб типу лінії та товщина — всі проходять round-trip без втрат. Будь-який DXF-сумісний додаток читає їх як стандартні кола.
