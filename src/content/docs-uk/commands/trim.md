# Trim

Команда `trim` видаляє частину [Line](../line/), що знаходиться між двома сусідніми точками перетину, розділяючи лінію на один або два коротші відрізки. Сегмент для відрізання визначається позицією курсора — наведіть на частину яку хочете видалити та клацніть для підрізання.

Trim працює **лише з об'єктами Line**. Для дуг, кіл, поліліній та інших типів об'єктів використовуйте [Delete](../delete/) або редагування ручками замість цього.

## Підрізання лінії

1. Введіть `trim` у терміналі або натисніть кнопку **Trim** на панелі інструментів.
2. **Наведіть на сегмент лінії** який хочете видалити — червоний попередній перегляд підсвічує точно ту частину що буде відрізана.
3. **Клацніть** для видалення цього сегмента.

Команда залишається активною після кожного підрізання, тому можна продовжувати наводити та клацати для відрізання більше сегментів. Натисніть **Escape** для виходу.

```
  До:                           Після відрізання середнього сегмента:

  ──────●──────●──────          ──────●          ●──────
      перетин  перетин            (ліва частина) (права частина)
                                  (середній сегмент видалено)
```

## Як визначається сегмент для підрізання

Команда проектує позицію курсора на лінію з наведенням та знаходить всі точки перетину цієї лінії з іншими об'єктами. Ці параметри перетину ділять лінію на сегменти. Сегмент, чий інтервал містить проекцію курсора, підсвічується і буде видалений при кліку.

- Якщо курсор **перед першим перетином**: видаляється та ведуча частина лінії.
- Якщо курсор **між двома перетинами**: видаляється та середня частина; лінія розділяється на дві.
- Якщо курсор **після останнього перетину**: видаляється та хвостова частина.
- Якщо лінія **не має перетинів** з жодним іншим об'єктом: попередній перегляд не показується і клацання нічого не робить.

## Підтримувані об'єкти

| Об'єкт | Можна підрізати? |
|--------|----------------|
| Line | Так |
| Arc, Circle, Ellipse | Ні |
| Polyline / Rectangle | Ні |
| Text, Spline, Dimension, Leader | Ні |

Об'єкти що використовуються як **межі різання** можуть бути будь-якого типу — лише лінія що підрізається повинна бути об'єктом Line.

## Trim проти Extend

| | Trim | Extend |
|---|------|--------|
| Що робить | Видаляє сегмент лінії | Розтягує кінцеву точку лінії до межі |
| Тригер | Наведення на сегмент для відрізання | Наведення поряд з кінцевою точкою для розтягування |
| Результат | Лінія розділяється або коротшає | Кінцева точка лінії переміщується до межі |
| Обидва | Лише лінії | Лише лінії |
