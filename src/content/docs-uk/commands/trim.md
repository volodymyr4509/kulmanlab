# Trim

Команда `trim` видаляє сегмент лінії між двома перетинаючими об'єктами. Наведіть курсор на відповідний сегмент — з'являється **червоний попередній перегляд** — та клацніть для його видалення.

## Як використовувати

1. Введіть `trim` у терміналі або натисніть кнопку **Trim** на панелі інструментів.
2. **Наведіть** курсор на сегмент лінії між двома перетинаючими об'єктами.
3. **Червоний попередній перегляд** показує що буде видалено.
4. **Клацніть** для видалення сегмента.

Команда залишається активною — наводьте та клацайте додаткові сегменти. Натисніть `Escape` або `Enter` для виходу.

## Як визначається сегмент

Лінія, на яку наводять курсор, розбивається в усіх точках перетину з іншими об'єктами на кресленику. Сегмент, що підсвічується, — це розбитий відрізок, найближчий до позиції курсора.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `Escape` / `Enter` | Вийти з команди Trim |

## Підтримувані об'єкти

| Роль | Підтримуваний тип |
|------|-------------------|
| **Об'єкт, що підрізається** | Лише **Лінія** |
| **Межа різання** | Будь-який тип об'єкта (Лінія, Коло, Дуга, Полілінія, тощо) |

Лише лінії можуть бути укорочені командою Trim. Інші типи об'єктів (Кола, Дуги, Поліінії тощо) **не можна** підрізати — але їх можна використовувати як межі різання для ліній.

## Trim проти Extend

| | Trim | Extend |
|---|------|--------|
| Що робить | Видаляє сегмент лінії | Подовжує лінію до межі |
| Взаємодія | Наведення + клік | Наведення + клік |
| Межа | Будь-який тип об'єкта | Будь-який тип об'єкта (окрім текстів та сплайнів) |
| Діє на | Лише лінії | Лише лінії |
