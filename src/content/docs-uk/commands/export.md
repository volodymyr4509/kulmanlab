# Export

Команда `export` відкриває **Менеджер експорту** — спеціальне вікно для вибору формату та завантаження кресленика на ваш пристрій.

## Відкриття Менеджера експорту

Введіть `export` у терміналі або натисніть кнопку **Export** на панелі інструментів. Менеджер експорту відкривається з двома картками форматів.

## Покроковий процес

1. **Клацніть картку формату** — JSON або DXF.
2. Картка підсвічується — показує вибраний формат.
3. **Клацніть кнопку Export** — файл генерується та завантажується.

## Порівняння форматів

| | JSON | DXF |
|---|------|-----|
| **Розміри** | Зберігаються | **Не** зберігаються |
| **Виноски (Leader)** | Зберігаються | **Не** зберігаються |
| **Геометрія** | Зберігається | Зберігається (LINE, CIRCLE, ARC, ELLIPSE, LWPOLYLINE, SPLINE, TEXT/MTEXT) |
| **Шари** | Зберігаються | Зберігаються |
| **Сумісність** | Лише KulmanLab | Будь-яка CAD-програма |
| **Перенесення між браузерами** | Так (через Import) | Так |

## Що зберігається у DXF

Формат DXF AC1012. Зберігаються об'єкти:

- LINE, CIRCLE, ARC, ELLIPSE, LWPOLYLINE, SPLINE
- TEXT / MTEXT
- Властивості: колір, шар, тип лінії, масштаб типу лінії, товщина
- Шари: назви, колір, тип лінії, стан заморожування/блокування

**Не** зберігаються у DXF:
- Розміри (DIMENSION) — використовуйте JSON
- Виноски (MLEADER) — використовуйте JSON

## Що зберігається у JSON

JSON зберігає **повний стан кресленика**: вся геометрія, всі розміри, всі виноски, всі шари та їх стани. Це єдиний формат для збереження розмірів та виносок.

## Назва файлу

Завантажений файл називається `kulman-<мітка часу>.<розширення>`, наприклад:
- `kulman-2026-01-15_14:32:07.json`
- `kulman-2026-01-15_14:32:07.dxf`

## Export проти Print

| | Export | Print |
|---|--------|-------|
| Формат | DXF або JSON (векторні дані) | PNG, JPEG, WebP або PDF (растрове зображення) |
| Чи можна редагувати результат | Так (у CAD-програмах) | Ні |
| Найкраще для | Обмін даними між програмами | Документація, передача клієнту |
