# Fillet

Команда `fillet` вставляє плавну дотичну дугу між двома лініями або відрізками, обрізаючи або продовжуючи їх так, щоб вони плавно з'єднувалися.

## Як використовувати

1. Введіть `fillet` у терміналі або натисніть кнопку **Fillet** на панелі інструментів.
2. **Введіть радіус** та натисніть **Enter**.
3. **Клацніть першу лінію** — ближче до **сторони, яку хочете зберегти**.
4. **Клацніть другу лінію** — так само ближче до сторони, що залишається. Дуга вставляється, лінії підрізаються.

## Вибір сторони для ліній, що перетинаються

Коли дві лінії вже перетинаються, місце кліку на кожній лінії визначає, яку **сторону** від перетину зберегти:

```
  До:                    Після Fillet (R=5):
  |   |                  |   |
  | × |   ← перетин      |   |
  |   |                       ╮
  ────┼────              ─────╯
```

Клацніть ліворуч від × на горизонтальній лінії та нижче × на вертикальній — і зберігається ліво-нижній кут, вставляється дуга в кут.

## Що створює команда

| | Значення |
|---|----------|
| **T1** | Точка дотику на першій лінії — де перша лінія підрізається |
| **T2** | Точка дотику на другій лінії — де друга лінія підрізається |
| **Дуга** | Вставляється між T1 та T2, успадковує колір та шар першої лінії |

Якщо лінії не перетинаються, вони **продовжуються** до точки дотику перед вставкою дуги.

## Радіус 0

При радіусі 0 кут загострюється — лінії обрізаються або подовжуються до точки їх перетину без вставки дуги.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.` | Ввести радіус |
| `Backspace` | Видалити останній символ |
| `Enter` | Підтвердити радіус |
| `Escape` | Скасувати |
