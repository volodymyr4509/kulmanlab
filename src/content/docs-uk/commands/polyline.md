# Polyline

Команда `polyline` малює з'єднаний шлях із будь-якої кількості прямих відрізків, що зберігаються як один об'єкт `LWPOLYLINE`. Оскільки весь шлях є одним об'єктом, вибір його вибирає кожний відрізок одночасно — переміщайте, обертайте або масштабуйте всю форму за одну операцію. Це ключова відмінність від ланцюгових [Lines](../line/), де кожен відрізок є незалежним об'єктом.

Полілінії також можуть бути **замкнуті**: команда [Rectangle](../rectangle/) використовує той самий об'єкт `LWPOLYLINE` з встановленим прапором закриття.

## Малювання полілінії

1. Введіть `polyline` у терміналі або натисніть кнопку **Polyline** на панелі інструментів.
2. **Клацніть першу точку** або введіть `X,Y` та натисніть **Enter** для точної координати.
3. **Клацніть кожну наступну точку** — кожен клік додає відрізок. Введення координат працює на кожному кроці.
4. Натисніть **Enter** або **Пробіл** для завершення (потрібно щонайменше 2 розміщені точки).

```
  ●──────●
  1-й    2-й
          \
           \  відрізок 3 (у процесі — курсор тут)
            ●  ← клікнути щоб додати, Enter/Пробіл щоб завершити
```

Натискання **Escape** у будь-який час скасовує всі розміщені точки та виходить з команди.

## Введення координат

Замість клацання введіть точну позицію для будь-якої вершини:

1. Введіть значення X.
2. Натисніть `,` — термінал покаже `[X], [Y{cursor}]`.
3. Введіть значення Y.
4. Натисніть **Enter** для розміщення вершини.

## Блокування кута та точна довжина відрізка

Та сама логіка прив'язки під кутом 45°, що і в команді [Line](../line/#блокування-кута-та-точне-введення-довжини), застосовується між будь-якими двома послідовними точками. При блокуванні на осі:

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.` | Додати цифру до довжини відрізка |
| `-` | Від'ємна довжина — змінює напрямок вздовж осі (лише перший символ) |
| `Backspace` | Видалити останній введений символ |
| `Enter` | Розмістити наступну точку на введеній відстані |

Поточна накопичена довжина відображається у запиті терміналу в реальному часі. Клік при блокуванні проєктується на вісь, тому нова вершина потрапляє точно на неї.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `0`–`9`, `.`, `-` | Почати введення координати X або довжини відрізка при заблокованому куті |
| `,` | Заблокувати X та перейти до введення Y |
| `Backspace` | Видалити останній введений символ |
| `Enter` | Підтвердити введену координату або довжину, або завершити полілінію якщо нічого не введено та є ≥ 2 точки |
| `Пробіл` | Завершити полілінію (те ж що Enter коли немає введення у процесі) |
| `Escape` | Скасувати всі точки та вийти |

## Редагування ручками — вершини та середні точки відрізків

Вибрана полілінія показує два типи ручок:

| Ручка | Розташування | Що робить |
|-------|-------------|-----------|
| **Вершина** | На кожній розміщеній точці | Перетягнути для переміщення вершини; всі з'єднані відрізки розтягуються слідом |
| **Середина відрізка** | Центр кожного відрізка | Перетягнути для переміщення **обох** кінцевих точок відрізка разом, зберігаючи його довжину та кут |

Ручка середньої точки відрізка унікальна для поліліній — вона дозволяє ковзати окремий відрізок убік, не змінюючи його довжину. На [Line](../line/) ручка середньої точки натомість активує команду Move для всього об'єкта.

Немає окремої ручки "перемістити всю полілінію". Для переміщення всього шляху використовуйте команду [Move](../move/).

## Вибір поліліній

| Метод | Поведінка |
|-------|-----------|
| **Клік** | Вибирає полілінію якщо клік потрапляє в межі виявлення будь-якого відрізка |
| **Перетягнути праворуч** (строгий) | Всі вершини повинні знаходитися всередині рамки |
| **Перетягнути ліворуч** (перехресний) | Будь-який відрізок, що перетинає межу рамки, вибирає всю полілінію |

Оскільки полілінія є одним об'єктом, перехресний вибір, що торкається будь-якого відрізка, вибирає всі відрізки.

## Підтримувані команди редагування

Полілінії підтримують всі загальні перетворення та зміщення, але **не** обрізання або продовження (вони лише для [Line](../line/)):

| Команда | Що відбувається з полілінією |
|---------|------------------------------|
| [Move](../move/) | Переміщує всі вершини на однакове зміщення |
| [Copy](../copy/) | Створює ідентичну полілінію на новій позиції |
| [Rotate](../rotate/) | Обертає всі вершини навколо вибраної базової точки |
| [Mirror](../mirror/) | Відображає всі вершини відносно дзеркальної осі |
| [Scale](../scale/) | Масштабує всі вершини рівномірно від базової точки |
| [Offset](../offset/) | Створює паралельну полілінію на фіксованій перпендикулярній відстані |
| [Delete](../delete/) | Видаляє полілінію з кресленика |

## Властивості

**Загальні**

| Властивість | За замовчуванням | Значення |
|-------------|-----------------|----------|
| Колір | 256 (ByLayer) | Індекс кольору ACI |
| Шар | `0` | Призначення шару |
| Тип лінії | ByLayer | Іменований шаблон типу лінії |
| Масштаб типу лінії | 1 | Коефіцієнт масштабу шаблону типу лінії |
| Товщина | 0 | Товщина екструзії |

**Геометрія**

| Властивість | Значення |
|-------------|----------|
| Замкнута | Чи з'єднується остання вершина з першою |
| Кількість вершин | Загальна кількість вершин |
| Вершини | Список координат усіх вершин |

## Polyline проти Line — коли що використовувати

| | Polyline | Line |
|---|---------|------|
| Кількість об'єктів | Один `LWPOLYLINE` для всього шляху | Один `LINE` на відрізок |
| Замкнута форма | Так (прапор закриття) | Ні |
| Trim / Extend | Ні | Так — відрізок за відрізком |
| Ручка середньої точки | Переміщує весь відрізок | Активує Move для об'єкта |
| Найкраще для | Контури, обриси, форми що зберігаються цілими | Будівельні лінії, геометрія для обрізання |

## DXF — об'єкт LWPOLYLINE

Полілінії зберігаються як об'єкти `LWPOLYLINE` у DXF файлі. Всі властивості — координати вершин, прапор закриття, колір, шар, тип лінії, масштаб типу лінії та товщина — проходять round-trip без втрат. Прямокутники, намальовані командою [Rectangle](../rectangle/), також зберігаються як `LWPOLYLINE` (замкнені, чотири вершини) і на рівні DXF є нерозрізнимими.

Об'єкти `LWPOLYLINE` з будь-якого DXF-сумісного додатка (LibreCAD, FreeCAD тощо) повертаються назад як повністю редаговані полілінії в редакторі.
