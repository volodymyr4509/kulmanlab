# Match Properties

Команда `MatchProperties` копіює **візуальні властивості та властивості шару** від джерельного об'єкта до одного або більше цільових об'єктів. Передаються лише властивості, спільні між типами джерельного та цільового об'єктів — геометрія ніколи не змінюється.

## Як активувати

Натисніть кнопку **Match Properties** на панелі інструментів (іконка валика) у панелі Стиль, або введіть `MatchProperties` у терміналі.

## Робочий процес

**Активувати спочатку, потім вибрати джерело:**

1. Введіть `MatchProperties` або натисніть кнопку панелі інструментів без попереднього вибору.
2. **Клацніть джерельний об'єкт** — той чиї властивості хочете скопіювати.
3. **Клацніть кожен цільовий об'єкт** для застосування властивостей джерела. Можна клацати кілька об'єктів один за одним.
4. Для застосування до групи одразу, **перетягніть рамку вибору** над цілями.
5. Натисніть **Enter** або **Escape** для завершення.

**Попередньо вибрати джерело, потім активувати:**

1. Клацніть один об'єкт для вибору.
2. Активуйте `MatchProperties`. Вибраний об'єкт автоматично використовується як джерело.
3. Клацніть цільові об'єкти або виберіть перетягуванням, потім **Enter** або **Escape** для завершення.

## Які властивості копіюються

MatchProperties копіює властивості що належать до спільного базового класу між джерелом та ціллю. Як мінімум, **всі типи об'єктів** мають ці властивості:

| Властивість | Опис |
|-------------|------|
| **Колір** | Індекс кольору об'єкта (включно з "By Layer" / "By Block") |
| **Шар** | Шар до якого належить об'єкт |

Коли джерело та ціль мають однаковий тип об'єкта (наприклад, обидва є розмірами), також копіюються додаткові специфічні для типу властивості — наприклад, висота тексту, розмір стрілки, налаштування виносної лінії.

Геометрія (координати, радіус, довжина тощо) ніколи не змінюється.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `Enter` / `Пробіл` | Підтвердити вибір областю або завершити команду |
| `Escape` | Завершити застосування (якщо джерело встановлено) або скасувати |

## Деталі поведінки

- Джерельний об'єкт ніколи не змінюється.
- Кожен клік або вибір перетягуванням застосовує властивості джерела негайно — кроку підтвердження немає.
- Вибір областю слідує стандартним правилам: перетягнути **праворуч** для строгого вибору (повністю охоплений), перетягнути **ліворуч** для вибору перетином (будь-яке перетинання).
- Клацання джерельного об'єкта як цілі ігнорується.
- Для об'єктів з текстом (**Текст**, **Розміри**, **Мультивиноски**) копіюється лише висота тексту — шрифт, жирний, курсив та інші налаштування стилю тексту не копіюються.

## Пов'язані команди

- [LayerMatch](../layer-match/) — переміщує вибрані об'єкти на той же шар що й джерело (лише властивість шару)
- [LayerMakeCurrent](../layer-make-current/) — встановлює поточний шар малювання з клікнутого об'єкта
