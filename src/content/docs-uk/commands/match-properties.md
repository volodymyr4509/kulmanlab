# Match Properties

Команда `match properties` копіює властивості (колір, шар, тип лінії, масштаб типу лінії та специфічні для типу властивості) від одного джерельного об'єкта до одного або більше цільових. Подібна до "форматного пензля" у програмах для роботи з текстом.

## Активація

Натисніть значок **пензля** на панелі інструментів або введіть `match properties` у терміналі.

## Два способи вибору цілей

**Активувати спочатку, потім вибрати** — найгнучкіший спосіб:

1. Введіть `match properties` або натисніть кнопку пензля.
2. **Клацніть джерельний об'єкт** — об'єкт, властивості якого ви хочете скопіювати.
3. **Клацайте цільові об'єкти** — кожен клікнутий об'єкт отримує властивості джерела. Для вибору кількох цілей можна **перетягнути виділяючу рамку**.
4. Натисніть **Escape** для завершення.

**Попередній вибір цілей**:

1. **Виберіть цільові об'єкти** до запуску команди.
2. Введіть `match properties` — команда запитує лише джерело.
3. **Клацніть джерельний об'єкт** — властивості застосовуються і команда завершується.

## Довідник клавіатури

| Клавіша | Дія |
|---------|-----|
| `Escape` | Завершити команду |

## Які властивості копіюються

| Властивість | Копіюється? |
|-------------|-------------|
| Колір | Так |
| Шар | Так |
| Тип лінії | Так |
| Масштаб типу лінії | Так |
| Специфічні для типу (наприклад, розмір тексту) | Так |
| Геометрія (форма, розмір, розміщення) | Ні |

## Деталі поведінки

- Джерельний об'єкт **не змінюється**.
- Команда залишається активною — можна клікати кілька цілей підряд.
- Якщо ціль і джерело одного типу (наприклад, два кола), копіюються також специфічні для типу властивості.
