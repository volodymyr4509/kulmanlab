# KulmanLab CAD — Довідник команд

Ласкаво просимо до довідника команд **KulmanLab CAD**. [KulmanLab CAD](https://kulmanlab.com) — це безкоштовний браузерний CAD-інструмент для креслення, редагування та експорту DXF файлів — без встановлення. Використовуйте бічну панель для перегляду всіх доступних команд, згрупованих за розділами.

## Відновлення

Якщо додаток аварійно завершується при кожному запуску (наприклад, після роботи з надзвичайно великими координатами), ви можете очистити всі локально збережені дані, додавши `?reset` до URL:

```
https://kulmanlab.com/?resetKulmanLocalStorage
```

Це видаляє все з локальної бази даних браузера та розпочинає новий порожній кресленик. Параметр `?reset` автоматично видаляється з URL. Використовуйте це як крайній засіб, коли [Wipe Storage](./commands/wipestorage/) недоступний, оскільки додаток взагалі не завантажується.

## Як працюють команди

Кожна команда слідує одному й тому ж шаблону:

1. **Активуйте** — натисніть кнопку на панелі інструментів або введіть назву команди в терміналі внизу екрана.
2. **Слідуйте запиту** — термінал показує, яке введення очікується наступним.
3. **Завершіть або скасуйте** — більшість команд завершуються автоматично після останнього введення. Натисніть **Escape** будь-коли для скасування.

## Вибір об'єктів

Декілька команд редагування (Move, Copy, Rotate, Mirror, Scale, Delete) мають однакову поведінку вибору:

- **Клацніть** об'єкт для вибору або скасування вибору.
- **Перетягніть праворуч** (зліва направо) для строгого вибору — вибираються лише об'єкти, що повністю знаходяться всередині рамки.
- **Перетягніть ліворуч** (справа наліво) для перехресного вибору — вибирається будь-який об'єкт, що перетинається з рамкою.
- Натисніть **Enter** або **Пробіл** для підтвердження вибору та переходу до наступного кроку.
