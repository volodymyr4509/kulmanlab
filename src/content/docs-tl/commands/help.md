# Help

Binubuksan ang in-app help panel na may buod ng lahat ng keyboard shortcuts, navigation tips, at pangkalahatang paggamit.

## Paano gamitin

I-click ang **Help** toolbar button o i-type ang `help` sa terminal. Agad na magbubukas ang help panel at magsasara ang command.

## Ano ang laman ng help panel

### Navigation
| Shortcut | Aksyon |
|----------|--------|
| `Scroll wheel` | Mag-zoom in / out |
| `Middle click + drag` | I-pan ang view |
| `Double middle click` | Ikasya ang lahat ng entities sa screen |

### Commands
| Shortcut | Aksyon |
|----------|--------|
| Type | Simulan ang pag-type ng pangalan ng command — lilitaw ang mga suggestion sa canvas |
| `Delete` | Tanggalin ang lahat ng napiling entities |
| `Enter` / `Space` | Kumpirmahin ang command o input |
| `Enter` / `Space` (idle) | Ulitin ang huling ginamit na command |
| `Escape` | Kanselahin ang kasalukuyang command o i-clear ang selection |

### Selection
| Shortcut | Aksyon |
|----------|--------|
| `Left click` | Piliin ang isang entity |
| `Drag →` | Piliin lang ang mga entity na **ganap na nasa loob** ng drag area |
| `Drag ←` | Piliin ang anumang entity na **humihipo** sa drag area |
| `Left click grip` | I-edit nang direkta ang isang vertex o endpoint |

### Mga tip sa pagguhit
- **Snap points** — awtomatikong nag-s-snap ang cursor sa mga kalapit na entity endpoints.
- **Sticky angle** — igalaw ang cursor nang sapat na malayo sa isang direksyon para maka-lock sa 45° increments.
- **Exact length** — habang naka-lock ang angle, mag-type ng numero at pindutin ang `Enter` para itakda ang eksaktong distansya.
