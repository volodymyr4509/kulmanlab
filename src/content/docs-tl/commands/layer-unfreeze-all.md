# LayerUnfreezeAll

Nililinis ng `LayerUnfreezeAll` command ang frozen flag sa **bawat layer** sa drawing nang instant. Walang kailangang selection o confirmation — tumatakbo at natatapos ito nang isang hakbang lang.

## Paggamit

I-type ang `LayerUnfreezeAll` sa terminal o i-click ang **Unfreeze All** toolbar button (sun icon). Agad na magiging visible ang lahat ng frozen layers.

## Kailan Gagamitin

Karaniwang ginagamit pagkatapos ng [LayerIsolate](../layer-isolate/) para ibalik ang lahat ng layer sa normal nilang visible state.

## Detalye ng Pag-uugali

- Naaaplay sa lahat ng layer anuman ang kasalukuyang state nila.
- Hindi naaapektuhan ang locked o plot flags — ang frozen flag lang ang nagbabago.
- Agad na natatapos ang command nang walang prompts.
