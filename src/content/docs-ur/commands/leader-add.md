# Leader+

`leader+` کمانڈ موجودہ ملٹی لیڈر میں نیا تیر نما بازو شامل کرتی ہے۔ نیا بازو لیڈر کے موجودہ ڈاگ لیگ سے آپ کے کلک کردہ نئے تیر نوک تک اشارہ کرتا ہے۔ تمام اسٹائلنگ — ڈاگ لیگ پوزیشن، متن، تیر کی قسم، اور سائز — منتخب لیڈر سے وراثت میں ملتی ہے۔

## بازو شامل کرنا

1. ٹرمینل میں `leader+` ٹائپ کریں۔
2. اسے منتخب کرنے کے لیے **موجودہ ملٹی لیڈر پر کلک کریں**۔
3. **نئی تیر نوک پر کلک کریں**، یا درست کوآرڈینیٹ کے لیے `X,Y` ٹائپ کریں اور **Enter** دبائیں۔ کرسر سے لیڈر کے ڈاگ لیگ تک پیش نظارہ لائن ظاہر ہوتی ہے۔

بازو رکھنے کے بعد کمانڈ فعال رہتی ہے — آپ فوری طور پر مزید بازو شامل کرنے کے لیے کسی اور لیڈر پر کلک کر سکتے ہیں۔ ختم کرنے کے لیے **Enter**، **Space**، یا **Escape** دبائیں۔

```
  پہلے:                        بعد:
  ◄── بازو 1                    ◄── بازو 1
       \                               \
        ●──── ڈاگ لیگ ──── متن          ●──── ڈاگ لیگ ──── متن
                                       /
                                  بازو 2 ──►  (آپ نے جہاں کلک کیا)
```

## نوک کے لیے کوآرڈینیٹ اندراج

کلک کرنے کی بجائے، آپ درست پوزیشن ٹائپ کر سکتے ہیں:

1. X قدر ٹائپ کریں۔
2. `,` دبائیں — ٹرمینل تصدیق کرتا ہے کہ X لاک ہو گیا۔
3. Y قدر ٹائپ کریں۔
4. رکھنے کے لیے **Enter** دبائیں۔

## کی بورڈ حوالہ

| کلید | عمل |
|-----|--------|
| `0`–`9`, `.`, `-` | X کوآرڈینیٹ ٹائپ کرنا شروع کریں |
| `,` | X لاک کریں اور Y اندراج پر جائیں |
| `Enter` | ٹائپ کردہ کوآرڈینیٹ تصدیق کریں اور بازو رکھیں |
| `Enter` / `Space` | ختم کریں (جب کوئی ان پٹ جاری نہ ہو) |
| `Escape` | منسوخ کریں اور ری سیٹ کریں |

## نوٹس

- صرف **Multileader** اشیاء منتخب کی جا سکتی ہیں — کسی اور اینٹیٹی قسم پر کلک کرنے سے کچھ نہیں ہوتا۔
- نیا بازو موجودہ ڈاگ لیگ سے شروع ہوتا ہے؛ آپ صرف یہ چنتے ہیں کہ تیر نوک کہاں جائے۔
- ملٹی لیڈر کے باذوؤں کی تعداد کی کوئی حد نہیں۔
