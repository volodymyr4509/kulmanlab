# Copy

`copy` کمانڈ منتخب اشیاء کی ترجمہ شدہ نقل بناتی ہے اور انہیں بنیادی نقطے سے منزل تک آفسیٹ رکھتی ہے — اصل بالکل وہیں رہتے ہیں۔ یہ [Move](../move/) سے ایک اہم فرق ہے: Copy ڈرائنگ میں نئی اشیاء شامل کرتی ہے؛ Move موجودہ کو منتقل کرتی ہے۔

## شروع کرنے کے دو طریقے

**پیشگی منتخب، پھر نقل کریں** — پہلے اشیاء منتخب کریں، پھر فعال کریں:

1. کینوس پر ایک یا زیادہ اشیاء منتخب کریں۔
2. ٹرمینل میں `copy` ٹائپ کریں یا **Copy** ٹول بار بٹن پر کلک کریں۔
3. **بنیادی نقطے پر کلک کریں**، یا درست کوآرڈینیٹ کے لیے `X,Y` ٹائپ کریں اور **Enter** دبائیں۔
4. **منزل پر کلک کریں** — نقلیں بنیاد→منزل آفسیٹ پر ظاہر ہوتی ہیں۔ کوآرڈینیٹ اندراج یہاں بھی کام کرتا ہے۔

**فعال کریں، پھر منتخب کریں** — کچھ منتخب کیے بغیر کمانڈ شروع کریں:

1. `copy` ٹائپ کریں یا ٹول بار بٹن پر کلک کریں۔
2. **اشیاء منتخب کریں** — انفرادی اشیاء ٹوگل کرنے کے لیے کلک کریں، یا علاقے کے لیے گھسیٹیں۔
3. انتخاب کی تصدیق کے لیے **Enter** یا **Space** دبائیں۔
4. **بنیادی نقطے پر کلک کریں**، پھر **منزل پر کلک کریں** (دونوں مراحل پر کوآرڈینیٹ اندراج دستیاب)۔

```
  پہلے:               بعد:
  [شے A]            [شے A]  ← اصل بدلے نہیں
  [شے B]            [شے B]
                    [A کی نقل] ← نئی اشیاء
                    [B کی نقل]
```

نقلوں کا ایک گھوسٹ پیش نظارہ بنیادی نقطے سے منزل تک کرسر کے پیچھے چلتا ہے۔

## کوآرڈینیٹ اندراج

بنیادی نقطے یا منزل مرحلے پر، کلک کرنے کی بجائے ایک درست پوزیشن ٹائپ کریں:

1. X قدر ٹائپ کریں۔
2. `,` دبائیں — ٹرمینل `[X], [Y{cursor}]` دکھاتا ہے۔
3. Y قدر ٹائپ کریں۔
4. تصدیق کے لیے **Enter** دبائیں۔

## زاویہ لاکنگ اور درست فاصلہ

بنیادی نقطہ سیٹ ہونے کے بعد کمانڈ 45° محوروں (0°, 45°, 90°, 135°, …) پر اسنیپ کرتی ہے جب کرسر کافی دور اور محور کے قریب ہو۔ لاک رہتے ہوئے، فاصلہ ٹائپ کریں اور **Enter** دبائیں نقلوں کو بالکل اس آفسیٹ پر رکھنے کے لیے۔

| کلید | عمل |
|-----|--------|
| `0`–`9`, `.` | فاصلے کی قدر میں شامل کریں |
| `-` | منفی فاصلہ — محور کے ساتھ سمت الٹ دیتا ہے (صرف پہلا حرف) |
| `Backspace` | آخری ٹائپ کیا گیا حرف حذف کریں |
| `Enter` | ٹائپ کیے گئے فاصلے پر نقلیں رکھیں |

## کی بورڈ حوالہ

| کلید | عمل |
|-----|--------|
| `Enter` / `Space` | انتخاب تصدیق کریں اور بنیادی نقطہ مرحلے پر جائیں |
| `0`–`9`, `.`, `-` | X کوآرڈینیٹ اندراج شروع کریں، یا زاویہ لاک ہونے پر فاصلہ |
| `,` | X لاک کریں اور Y اندراج پر جائیں |
| `Backspace` | آخری ٹائپ کیا گیا حرف حذف کریں |
| `Enter` | کوآرڈینیٹ تصدیق کریں یا ٹائپ کیے گئے فاصلے پر نقل لاگو کریں |
| `Escape` | منسوخ کریں اور ری سیٹ کریں |

## کمانڈ کے دوران انتخاب

| طریقہ | برتاؤ |
|--------|-----------|
| **کلک** | کرسر کے نیچے شے کو انتخاب میں شامل/خارج کرتا ہے |
| **دائیں گھسیٹیں** (سخت) | باکس کے مکمل اندر اشیاء شامل کرتا ہے |
| **بائیں گھسیٹیں** (کراسنگ) | باکس کی سرحد کو کاٹنے والی اشیاء شامل کرتا ہے |
| **Enter** / **Space** | انتخاب تصدیق کرتا ہے |

## نقل کے بعد

**اصل منتخب رہتے ہیں** — نئی نقلیں ڈرائنگ میں شامل ہوتی ہیں لیکن انتخاب صاف ہو جاتا ہے اور کمانڈ نکل جاتی ہے۔ نقلوں کے ساتھ فوری کام کرنے کے لیے، انتخاب پر Copy دوبارہ چلائیں، یا نئی کمانڈ شروع کریں۔

## Copy بمقابلہ Move

| | Copy | Move |
|---|------|------|
| اصل | جگہ پر رہتے ہیں | اصل پوزیشن سے ہٹا دیے جاتے ہیں |
| نتیجے کی تعداد | نقل کی گئی اشیاء کی تعداد بڑھتی ہے | بدلتی نہیں |
| آپریشن کے بعد | اصل نئی پوزیشن پر منتخب | منتقل اشیاء نئی پوزیشن پر منتخب |
| بہترین | ہندسہ دہرانا، ہموار لے آؤٹ | ہندسہ دوبارہ پوزیشن کرنا |

## معاون اشیاء

Copy ہر شے کی قسم پر کام کرتی ہے۔ تمام اشیاء اندرونی طور پر `translate(dx, dy)` نافذ کرتی ہیں لہذا کوئی خارج نہیں ہے۔
