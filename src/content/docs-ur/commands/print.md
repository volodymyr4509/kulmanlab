# Print

`print` کمانڈ **Print Manager** کھولتی ہے — براہ راست پیش نظارہ کینوس، فارمیٹ سلیکٹر (PNG / JPEG / WebP / PDF)، مونوکروم ٹوگل، اور اختیاری رقبہ کراپ کے ساتھ مخصوص برآمد ونڈو۔ کوئی چیز فیزیکل پرنٹر کو نہیں بھیجی جاتی؛ آؤٹ پٹ فائل کے طور پر ڈاؤن لوڈ ہوتی ہے۔

## Print Manager کھولنا

**Print** ٹول بار بٹن پر کلک کریں یا ٹرمینل میں `print` ٹائپ کریں۔ Print Manager فوری طور پر موجودہ ویوپورٹ کا پیش نظارہ دکھاتا ہے۔

## Print Manager کی ترتیب

ونڈو کے دو پینلز ہیں:
- **بائیں سائیڈ بار** — تمام برآمد کنٹرولز۔
- **دائیں پینل** — براہ راست پیش نظارہ کینوس جو ترتیبات بدلنے پر اپ ڈیٹ ہوتا ہے۔

### سائیڈ بار کنٹرولز

| کنٹرول | تفصیل |
|---------|-------------|
| **Change Area** | کینوس پر کسٹم مستطیل تک کراپ کریں (نیچے دیکھیں) |
| **Monochrome** ٹوگل | تمام رنگین لائنیں کالی کریں — صاف پرنٹ آؤٹ پٹ کے لیے ڈیفالٹ پر |
| **Format** ڈراپ ڈاؤن | PNG، JPEG، WebP، یا PDF |
| **Export** بٹن | فائل بنائیں اور ڈاؤن لوڈ کریں |

## کسٹم برآمد رقبہ منتخب کرنا

ڈیفالٹ کے مطابق پیش نظارہ وہی دکھاتا ہے جو Print Manager کھولتے وقت کینوس پر نظر آ رہا تھا۔ مخصوص علاقہ برآمد کرنے کے لیے:

1. **Change Area** پر کلک کریں — Print Manager چھپ جاتا ہے اور کینوس انٹرایکٹو ہو جاتا ہے۔
2. برآمد مستطیل کا **پہلا کونا کلک کریں**۔
3. **مخالف کونا کلک کریں** — Print Manager منتخب رقبے کے ساتھ پیش نظارہ میں دوبارہ کھلتا ہے۔

رقبہ انتخاب کے دوران `Escape` دبا کر منسوخ کریں اور پچھلا رقبہ بحال کریں۔

پیش نظارہ کینوس منتخب رقبے کے **بالکل پہلو تناسب** سے میل کھانے کے لیے متحرک طور پر سائز تبدیل کرتا ہے، تاکہ پیش نظارہ پکسل درست ہو۔

## برآمد فارمیٹس

| فارمیٹ | بہترین | نوٹس |
|--------|----------|-------|
| **PNG** | بے نقصان، تیز لائنیں | سفید پس منظر، کوئی شفافیت نہیں |
| **JPEG** | شیئرنگ کے لیے چھوٹی فائل | 95% معیار، معمولی کمپریشن |
| **WebP** | ویب کے لیے سب سے چھوٹی فائل | وہی 95% معیار، JPEG سے بہتر کمپریشن |
| **PDF** | پرنٹ کے لیے تیار دستاویزات | PDF کنٹینر میں 150 DPI پر تصویر |

برآمد شدہ فائل کا نام `kulman-<timestamp>.<ext>` ہوتا ہے اور خود بخود ڈاؤن لوڈ ہوتی ہے۔

## برآمد ریزولوشن اور پس منظر

- زیادہ سے زیادہ ریزولوشن: **2000 × 2000 پکسلز**، منتخب رقبے کے متناسب پیمانے میں۔
- پس منظر ہمیشہ **سفید** ہوتا ہے۔
- **غیر پلاٹنگ** کے طور پر نشان زد لیئرز برآمد سے خارج ہوتے ہیں۔

## کی بورڈ حوالہ

| کلید | عمل |
|-----|--------|
| `Escape` (رقبہ انتخاب کے دوران) | رقبہ انتخاب منسوخ کریں، پچھلا رقبہ بحال کریں |
| `Escape` (Print Manager میں) | Print Manager بند کریں |
