# Text

`text` کمانڈ ملٹی لائن متن لیبل رکھتی ہے۔ کینوس پر پوزیشن کلک کرنے کے بعد، ایک پاپ اپ ایڈیٹر **rich** موڈ میں کھلتا ہے — آپ مواد ٹائپ کر سکتے ہیں، فی حرف bold/italic/strikethrough لاگو کر سکتے ہیں، فونٹس اور اونچائیاں بدل سکتے ہیں، اور لائن بریک شامل کر سکتے ہیں۔ کمٹ کرنے اور ایڈیٹر بند کرنے کے لیے **Escape** دبائیں۔

مکمل ایڈیٹر حوالے کے لیے [Text Editor](../interface/text-editor) صفحہ دیکھیں، جس میں **rich** اور **simple** موڈز کا موازنہ شامل ہے۔

## متن لیبل رکھنا

1. ٹرمینل میں `text` ٹائپ کریں یا **Text** ٹول بار بٹن پر کلک کریں۔
2. کینوس پر **لنگر پوزیشن کلک کریں**۔ یا درست کوآرڈینیٹ کے لیے `X,Y` ٹائپ کریں اور **Enter** دبائیں۔
3. نئے لیبل کے اوپر **متن ایڈیٹر پاپ اپ** کھلتا ہے۔ اپنا مواد ٹائپ کریں۔
4. لیبل کمٹ کرنے اور ایڈیٹر بند کرنے کے لیے **Escape** دبائیں۔

ڈیفالٹ اونچائی **12 ڈرائنگ یونٹس** ہے۔

## موجودہ لیبل ترمیم

اس لیبل کے لیے ایڈیٹر دوبارہ کھولنے کے لیے کینوس پر کسی بھی متن لیبل پر **ڈبل کلک** کریں۔

## لنگر کے لیے کوآرڈینیٹ اندراج

کلک کرنے کی بجائے، درست پوزیشن ٹائپ کریں:

1. X قدر ٹائپ کریں۔
2. `,` دبائیں — ٹرمینل `[X], [Y{cursor}]` دکھاتا ہے۔
3. Y قدر ٹائپ کریں۔
4. لنگر رکھنے اور ایڈیٹر کھولنے کے لیے **Enter** دبائیں۔

## کی بورڈ حوالہ

**لنگر مرحلہ**

| کلید | عمل |
|-----|--------|
| `0`–`9`, `.`, `-` | X کوآرڈینیٹ اندراج شروع کریں |
| `,` | X لاک کریں اور Y اندراج پر جائیں |
| `Backspace` | آخری ٹائپ کردہ حرف حذف کریں |
| `Enter` | ٹائپ کردہ کوآرڈینیٹ تصدیق کریں |

**متن ایڈیٹر مرحلہ** (مکمل حوالے کے لیے [Text Editor](../interface/text-editor) دیکھیں)

| کلید | عمل |
|-----|--------|
| کوئی بھی پرنٹ ہونے والا حرف | کرسر پر داخل کریں |
| `Backspace` / `Delete` | ملحقہ حرف یا انتخاب حذف کریں |
| `Enter` | لائن بریک داخل کریں |
| `←` / `→` | کیریٹ حرکت کریں |
| `Home` / `End` | ہارڈ لائن کے آغاز / انتہا پر جائیں |
| `Escape` | کمٹ کریں اور ایڈیٹر بند کریں |

## گرپ ترمیم — دوبارہ پوزیشن

منتخب متن لیبل لنگر نقطے پر ایک گرپ ظاہر کرتا ہے:

| گرپ | پوزیشن | کیا کرتا ہے |
|------|----------|--------------|
| **لنگر** | متن کا نیچے بائیں | لیبل دوبارہ پوزیشن کرنے کے لیے ڈریگ کریں |

## متن منتخب کرنا

| طریقہ | برتاؤ |
|--------|-----------|
| **کلک** | منتخب کرتا ہے اگر کلک متن کے گھمے ہوئے باؤنڈنگ باکس کے اندر ہو |
| **دائیں ڈریگ** (سخت) | باؤنڈنگ باکس کے تمام چار کونے انتخاب رقبے کے اندر ہونے چاہیں |
| **بائیں ڈریگ** (کراسنگ) | متن باؤنڈنگ باکس اور انتخاب رقبے کے درمیان کوئی بھی اوورلیپ اسے منتخب کرے |

## سپورٹ شدہ ترمیم کمانڈز

| کمانڈ | متن پر کیا ہوتا ہے |
|---------|--------------------------|
| [Move](../move/) | لنگر نقطہ حرکت دیتی ہے |
| [Copy](../copy/) | نئی پوزیشن پر ایک جیسا لیبل بناتی ہے |
| [Rotate](../rotate/) | لنگر پوزیشن گھماتی ہے اور Rotation Degree میں زاویہ شامل کرتی ہے |
| [Mirror](../mirror/) | عکس محور پر لنگر نقطہ منعکس کرتی ہے (متن سٹرنگ نہیں پلٹتا) |
| [Scale](../scale/) | لنگر پوزیشن پیمانہ کرتی ہے اور اونچائی پیمانہ عنصر سے ضرب کرتی ہے |
| [Delete](../delete/) | لیبل ہٹاتی ہے |

متن **Offset**، **Trim**، یا **Extend** سپورٹ نہیں کرتا۔

## خصوصیات

جب متن لیبل منتخب ہو خصوصیات پینل دکھاتا ہے:

**عمومی**

| خاصیت | ڈیفالٹ | مطلب |
|----------|---------|---------|
| رنگ | 256 (ByLayer) | ACI رنگ انڈیکس |
| لیئر | `0` | لیئر تفویض |

**جیومیٹری**

| خاصیت | مطلب |
|----------|---------|
| پوزیشن X / پوزیشن Y | لنگر نقطہ کوآرڈینیٹس |
| اونچائی | ڈرائنگ یونٹس میں بنیادی متن اونچائی (ڈیفالٹ: **12**) |
| گھماؤ ڈگری | ڈگریوں میں گھڑی مخالف گھماؤ |

**خصوصیات**

| خاصیت | مطلب |
|----------|---------|
| مواد | متن سٹرنگ (MTEXT انلائن کوڈز محفوظ) |
| منسلک نقطہ | سیدھ کوڈ (1 = اوپر بائیں … 9 = نیچے دائیں) |

متن میں Linetype، Linetype Scale، یا Thickness خصوصیات نہیں ہیں۔

## DXF — MTEXT اینٹیٹی

متن لیبلز DXF فائل میں **MTEXT** اشیاء کے طور پر محفوظ ہوتے ہیں۔ Bold اور italic `\L`، `\K`، `\O`، اور انلائن فونٹ سوئچز (`\f`) کے ذریعے انکوڈ ہوتے ہیں۔ فی حرف اونچائی `\H` کے طور پر انکوڈ ہوتی ہے۔ تمام فارمیٹنگ مکمل DXF راؤنڈ ٹرپ میں باقی رہتی ہے۔
