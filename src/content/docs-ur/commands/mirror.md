# Mirror

`mirror` کمانڈ دو نقطاتی محور پر منعکس منتخب اشیاء کی عکسی نقلیں بناتی ہے۔ اصل اشیاء **ہمیشہ محفوظ رہتی ہیں** — [Move](../move/) یا [Rotate](../rotate/) کے برعکس، Mirror کبھی موجودہ اشیاء تبدیل نہیں کرتی؛ یہ صرف نئی شامل کرتی ہے۔

## شروع کرنے کے دو طریقے

**پہلے منتخب کریں، پھر عکس بنائیں** — پہلے اشیاء منتخب کریں، پھر فعال کریں:

1. کینوس پر ایک یا زیادہ اشیاء منتخب کریں۔
2. ٹرمینل میں `mirror` ٹائپ کریں یا **Mirror** ٹول بار بٹن پر کلک کریں۔
3. عکس محور کے **پہلے نقطے پر کلک کریں**، یا درست کوآرڈینیٹ کے لیے `X,Y` ٹائپ کریں اور **Enter** دبائیں۔
4. **دوسرے نقطے پر کلک کریں** — عکسی نقلیں رکھی جاتی ہیں اور کمانڈ ختم ہوتی ہے۔ کوآرڈینیٹ اندراج یہاں بھی کام کرتا ہے۔

**پہلے فعال کریں، پھر منتخب کریں** — بغیر کچھ منتخب کیے کمانڈ شروع کریں:

1. `mirror` ٹائپ کریں یا ٹول بار بٹن پر کلک کریں۔
2. **اشیاء منتخب کریں** — ٹوگل کرنے کے لیے کلک کریں، یا علاقے کے ذریعے منتخب کرنے کے لیے ڈریگ کریں۔
3. انتخاب تصدیق کرنے کے لیے **Enter** یا **Space** دبائیں۔
4. **پہلے نقطے پر کلک کریں**، پھر عکس محور کے **دوسرے نقطے پر کلک کریں** (دونوں مراحل پر کوآرڈینیٹ اندراج دستیاب ہے)۔

```
  اصل:          عکس محور:        نتیجہ:
                     |
  [اینٹیٹی A]    →    |    →    [اینٹیٹی A] + [عکسی A]
                     |
```

دوسرے محور نقطے کی پوزیشن کے دوران عکسی نقلوں کا براہ راست پیش نظارہ کرسر کی پیروی کرتا ہے۔

## عکس محور

محور دو کلک کردہ نقاط سے گزرنے والی لامحدود لائن ہے۔ یہ کسی بھی زاویے پر ہو سکتی ہے:

- کرسر کو **45° اسنیپ محور** (0°، 45°، 90°، 135°، …) کے قریب لے جائیں اور محور اس زاویے پر لاک ہو جاتا ہے — صاف افقی، عمودی، یا قطری انعکاس کے لیے مفید۔
- اسنیپ زون سے دور کلک کریں مفت زاویے کے محور کے لیے۔

## کوآرڈینیٹ اندراج

کسی بھی محور نقطہ مرحلے پر، کلک کرنے کی بجائے درست پوزیشن ٹائپ کریں:

1. X قدر ٹائپ کریں۔
2. `,` دبائیں — ٹرمینل `[X], [Y{cursor}]` دکھاتا ہے۔
3. Y قدر ٹائپ کریں۔
4. تصدیق کے لیے **Enter** دبائیں۔

## کی بورڈ حوالہ

| کلید | عمل |
|-----|--------|
| `Enter` / `Space` | انتخاب تصدیق کریں |
| `0`–`9`, `.`, `-` | X کوآرڈینیٹ اندراج شروع کریں |
| `,` | X لاک کریں اور Y اندراج پر جائیں |
| `Backspace` | آخری ٹائپ کردہ حرف حذف کریں |
| `Enter` | ٹائپ کردہ کوآرڈینیٹ تصدیق کریں |
| `Escape` | منسوخ کریں اور ری سیٹ کریں |

## کمانڈ کے دوران انتخاب

| طریقہ | برتاؤ |
|--------|-----------|
| **کلک** | کرسر کے نیچے اینٹیٹی ٹوگل کرتا ہے |
| **دائیں ڈریگ** (سخت) | باکس کے مکمل اندر موجود اشیاء شامل کرتا ہے |
| **بائیں ڈریگ** (کراسنگ) | باکس کو چھونے والی اشیاء شامل کرتا ہے |
| **Enter** / **Space** | انتخاب تصدیق کرتا ہے |

## کیا عکس بنتا ہے

ہر اینٹیٹی قسم سپورٹ ہے۔ جیومیٹری ریاضیاتی طور پر محور پر منعکس ہوتی ہے:

| اینٹیٹی | کیا بدلتا ہے |
|--------|-------------|
| لائن | دونوں اختتامی نقاط منعکس |
| دائرہ | مرکز منعکس؛ رداس غیر متغیر |
| قوس | مرکز منعکس؛ ابتدا اور انتہا کے زاویے محور پر دوبارہ حساب |
| بیضہ | مرکز منعکس؛ بڑے محور کی سمت محور پر پلٹی |
| پولی لائن / مستطیل | ہر راس منعکس |
| متن | لنگر نقطہ منعکس؛ متن سٹرنگ **الٹا نہیں** ہوتا |
| اسپلائن | تمام کنٹرول راس / فٹ نقاط منعکس |

## Mirror بمقابلہ Copy

| | Mirror | Copy |
|---|--------|------|
| اصل اشیاء | ہمیشہ محفوظ | ہمیشہ محفوظ |
| نئی اینٹیٹی پوزیشن | محور پر منعکس | بے گھر ویکٹر کے ذریعے آفسیٹ |
| بہترین | متقارن ڈیزائن، دو طرفہ خصوصیات | کسی بھی سمت میں جیومیٹری دہرانا |
