# Font Manager

`FontManager` کمانڈ فونٹس براؤز اور منتخب کرنے کے لیے، اور [Text](../text/) اور [Multileader](../leader/) اینٹیٹیز میں استعمال کے لیے اپنی `.ttf` فائلیں اپ لوڈ کرنے کے لیے ایک ڈائیلاگ کھولتی ہے۔

## Font Manager کھولنا

- ٹرمینل میں `FontManager` ٹائپ کریں، **یا**
- [text editor](../../interface/text-editor/) ٹول بار میں **Font Manager** بٹن پر کلک کریں۔

## فونٹ گروپس

| گروپ | مواد |
|------|------|
| **Default** | بلٹ اِن sans-serif فونٹ — ہمیشہ دستیاب |
| **User** | آپ کے اپنے اپ لوڈ کردہ `.ttf` فونٹس (صرف اس وقت دکھائے جاتے ہیں جب آپ ایک شامل کریں) |
| **Free** | 15 بنڈل شدہ Google Fonts (EB Garamond, Fira Code, Inter, Lato, Merriweather, Montserrat, Nunito, Open Sans, Oswald, Playfair Display, Poppins, Raleway, Roboto, Roboto Condensed, Source Code Pro) |
| **System** | عام OS فونٹس (Courier New, Georgia, Helvetica, Impact, Lucida Console, Tahoma, Times New Roman, Trebuchet MS, Verdana) |

پیش نظارے کے لیے فہرست میں کسی بھی فونٹ پر کلک کریں، جو دائیں جانب دکھایا جائے گا — نام، حروفِ تہجی کا نمونہ، ایک پینگرام، اور ہندسے۔

## کسٹم فونٹ اپ لوڈ کرنا

1. ڈائیلاگ کے نچلے حصے میں **Add Font** پر کلک کریں (یا فائل منتخب کرنے کا ڈائیلاگ براہ راست کھولنے کے لیے ٹرمینل میں `Font+` ٹائپ کریں)۔
2. ایک `.ttf` فائل منتخب کریں۔ صرف TrueType فونٹس کو سپورٹ کیا جاتا ہے — `.otf` اور `.woff`/`.woff2` کو سپورٹ نہیں کیا جاتا۔
3. فائل کا نام (ایکسٹینشن کے بغیر) **User** گروپ میں فونٹ کا نام بن جاتا ہے۔ مثال کے طور پر، `MyFont.ttf` اپ لوڈ کرنے سے `MyFont` نامی فونٹ شامل ہو جاتا ہے۔

اپ لوڈ کردہ فونٹس براؤزر (IndexedDB) میں مستقل طور پر محفوظ ہو جاتے ہیں اور اگلی بار جب آپ KulmanLab CAD کھولتے ہیں تو خود بخود دوبارہ لوڈ ہو جاتے ہیں۔

## کسٹم فونٹ ہٹانا

**User** گروپ میں کسی فونٹ پر ہوور کریں اور اس کے ساتھ موجود **×** بٹن پر کلک کریں۔ بلٹ اِن فونٹس (Default, Free, System) کو ہٹایا نہیں جا سکتا۔

## کی بورڈ حوالہ

| کلید | عمل |
|-----|--------|
| `↑` / `↓` | فونٹ فہرست میں انتخاب اوپر یا نیچے منتقل کریں |
| `Escape` | Font Manager بند کریں |

## DXF مطابقت

فونٹ کا نام ایکسپورٹ شدہ **MTEXT** اینٹیٹیز میں ایک اِن لائن فارمیٹنگ کوڈ کے طور پر ایمبیڈ ہوتا ہے، اس لیے KulmanLab CAD کے ذریعے راؤنڈ ٹرپ کی گئی DXF اپنی فونٹ ایسائنمنٹ برقرار رکھتی ہے۔ کسٹم فونٹ کی *فائلیں* DXF میں ایمبیڈ نہیں ہوتیں — صرف فونٹ کا *نام* ایمبیڈ ہوتا ہے۔ اگر آپ ایسی ڈرائنگ دوبارہ درآمد کرتے ہیں جو کسی ایسے کسٹم فونٹ کا حوالہ دیتی ہے جو آپ نے اس ڈیوائس پر اپ لوڈ نہیں کیا، تو جب تک آپ اسی نام کا فونٹ اپ لوڈ نہ کریں متن ڈیفالٹ فونٹ میں دکھایا جاتا رہے گا۔

## متعلقہ کمانڈز

- [Text](../text/) — متن کے وہ لیبل رکھتی ہے جن پر فونٹ کا انتخاب لاگو ہوتا ہے
- [Match Properties](../match-properties/) — اینٹیٹیز کے درمیان متن کی اونچائی کاپی کرتی ہے، مگر فونٹ نہیں
