# Scale

`scale` کمانڈ منتخب اشیاء کو بنیادی نقطے کے گرد یکساں طور پر پیمانہ کرتی ہے۔ بنیادی نقطے سے تمام فاصلے پیمانہ عنصر سے ضرب ہوتے ہیں — `2` کا عنصر تمام ابعاد دوگنا کرتا ہے، `0.5` انہیں آدھا کرتا ہے۔ عنصر ہمیشہ ٹائپ کر کے داخل کیا جاتا ہے؛ کلک سے پیمانہ سیٹ کرنے کا کوئی طریقہ نہیں۔

## شروع کرنے کے دو طریقے

**پہلے منتخب کریں، پھر پیمانہ کریں** — پہلے اشیاء منتخب کریں، پھر فعال کریں:

1. کینوس پر ایک یا زیادہ اشیاء منتخب کریں۔
2. ٹرمینل میں `scale` ٹائپ کریں یا **Scale** ٹول بار بٹن پر کلک کریں۔
3. **بنیادی نقطے پر کلک کریں** — وہ ثابت نقطہ جو پیمانہ کے دوران نہیں حرکت کرتا۔ یا درست کوآرڈینیٹ کے لیے `X,Y` ٹائپ کریں اور **Enter** دبائیں۔
4. **پیمانہ عنصر ٹائپ کریں** اور **Enter** دبائیں۔

**پہلے فعال کریں، پھر منتخب کریں** — بغیر کچھ منتخب کیے کمانڈ شروع کریں:

1. `scale` ٹائپ کریں یا ٹول بار بٹن پر کلک کریں۔
2. **اشیاء منتخب کریں** — ٹوگل کرنے کے لیے کلک کریں، یا علاقے کے ذریعے منتخب کرنے کے لیے ڈریگ کریں۔
3. انتخاب تصدیق کرنے کے لیے **Enter** یا **Space** دبائیں۔
4. **بنیادی نقطے پر کلک کریں** (کوآرڈینیٹ اندراج دستیاب)، پھر عنصر ٹائپ کریں۔

```
  بنیاد ●                بنیاد ●
        [اینٹیٹی]   →          [بڑی اینٹیٹی]
  عنصر = 2 → ● سے فاصلے دوگنے
```

## پیمانہ عنصر ٹائپ کرنا

بنیادی نقطہ رکھنے کے بعد ٹرمینل `enter scale factor:` دکھاتا ہے اور کی بورڈ ان پٹ کا انتظار کرتا ہے:

| کلید | عمل |
|-----|--------|
| `0`–`9`, `.` | عنصر میں ہندسہ شامل کریں |
| `-` | منفی عنصر (صرف پہلا حرف — پھر پیمانہ کرتا ہے) |
| `Backspace` | آخری ٹائپ کردہ حرف حذف کریں |
| `Enter` | ٹائپ کردہ عنصر پر پیمانہ لاگو کریں |

عنصر صفر نہیں ہو سکتا۔ عام قدریں:

| عنصر | اثر |
|--------|--------|
| `2` | تمام ابعاد دوگنا |
| `0.5` | تمام ابعاد آدھا |
| `1.5` | 50% اضافہ |
| `-1` | بنیادی نقطے پر آئینہ (180° گھماؤ کے مساوی) |

ٹائپ کرتے وقت کوئی براہ راست پیش نظارہ نہیں — **Enter** دبانے کے بعد ہی پیمانہ شدہ نتیجہ ظاہر ہوتا ہے۔

## کی بورڈ حوالہ

| کلید | عمل |
|-----|--------|
| `Enter` / `Space` | انتخاب تصدیق کریں |
| `0`–`9`, `.`, `-` | X کوآرڈینیٹ اندراج شروع کریں (بنیادی نقطہ مرحلہ)، یا پیمانہ عنصر (عنصر مرحلہ) |
| `,` | X لاک کریں اور Y اندراج پر جائیں (بنیادی نقطہ مرحلہ) |
| `Backspace` | آخری ٹائپ کردہ حرف حذف کریں |
| `Enter` | کوآرڈینیٹ تصدیق کریں یا پیمانہ لاگو کریں |
| `Escape` | منسوخ کریں اور ری سیٹ کریں |

## کمانڈ کے دوران انتخاب

| طریقہ | برتاؤ |
|--------|-----------|
| **کلک** | کرسر کے نیچے اینٹیٹی ٹوگل کرتا ہے |
| **دائیں ڈریگ** (سخت) | باکس کے مکمل اندر موجود اشیاء شامل کرتا ہے |
| **بائیں ڈریگ** (کراسنگ) | باکس کو چھونے والی اشیاء شامل کرتا ہے |
| **Enter** / **Space** | انتخاب تصدیق کرتا ہے |

## کیا پیمانہ ہوتا ہے

تمام اینٹیٹی اقسام سپورٹ ہیں۔ ہر اینٹیٹی بنیادی نقطے کے نسبت اپنی جیومیٹری پیمانہ کرتی ہے:

| اینٹیٹی | کیا بدلتا ہے |
|--------|-------------|
| لائن | دونوں اختتامی نقاط بنیادی نقطے سے دور پیمانہ |
| دائرہ | مرکز بنیادی نقطے سے پیمانہ؛ رداس عنصر سے ضرب |
| قوس | مرکز پیمانہ؛ رداس عنصر سے ضرب؛ زاویے غیر متغیر |
| بیضہ | مرکز پیمانہ؛ دونوں نیم محور لمبائیاں عنصر سے ضرب |
| پولی لائن / مستطیل | ہر ورٹیکس بنیادی نقطے سے پیمانہ |
| متن | لنگر نقطہ پیمانہ؛ اونچائی عنصر سے ضرب |
| اسپلائن | تمام کنٹرول ورٹیکسیز / فٹ نقاط پیمانہ |
