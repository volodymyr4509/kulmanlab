# Distance

`distance` کمانڈ دو کلک کیے گئے نقاط کے درمیان سیدھی لائن (یوکلیڈین) فاصلہ ناپتی ہے اور نتیجہ ٹرمینل میں ۴ اعشاریہ مقامات تک پرنٹ کرتی ہے۔ یہ تین پیمائش کمانڈز میں سے ایک ہے — [Angle](../angle/) ورٹیکس پر زاویاتی کھلاؤ ناپتی ہے، اور [Area](../area/) کثیرالاضلاع کا رقبہ اور محیط ناپتی ہے۔

## فاصلہ پیمائش کا خاکہ

```
  ● پہلا نقطہ
   \
    \  پیش نظارہ لائن (لائیو)
     \
      ● دوسرا نقطہ    →  ٹرمینل: "Distance: 12.3456"
```

- **پہلا نقطہ** — پیمائش کی اصل۔
- **دوسرا نقطہ** — اختتامی نقطہ؛ اسے رکھنے سے فوری نتیجہ پرنٹ ہوتا ہے۔
- **نتیجہ** — ٹرمینل میں دکھایا جاتا ہے، کینوس پر نہیں رکھا جاتا۔

## فاصلہ ناپنا

1. ٹرمینل میں `distance` ٹائپ کریں یا **Distance** ٹول بار بٹن پر کلک کریں۔
2. **پہلے نقطے پر کلک کریں**، یا درست کوآرڈینیٹ کے لیے `X,Y` ٹائپ کریں اور **Enter** دبائیں۔
3. **دوسرے نقطے پر کلک کریں** — ناپا گیا فاصلہ ٹرمینل میں ظاہر ہوتا ہے۔ کوآرڈینیٹ اندراج یہاں بھی کام کرتا ہے۔
4. **دوبارہ کلک کریں** (اختیاری) نئی پیمائش شروع کریں۔ کمانڈ فعال رہتی ہے۔

کسی بھی وقت ری سیٹ کرنے کے لیے `Escape` دبائیں۔

## پیمائشیں جوڑنا

نتیجہ دکھانے کے بعد، کلک کرنے سے فوری طور پر اگلی پیمائش شروع ہوتی ہے — کلک کیا گیا نقطہ نیا پہلا نقطہ بن جاتا ہے۔ یہ آپ کو کمانڈ دوبارہ فعال کیے بغیر فاصلوں کی ایک سلسلہ ناپنے دیتا ہے۔

## Distance بمقابلہ Angle

| | Distance | Angle |
|---|---------|-------|
| کیا ناپتا ہے | سیدھی لائن کی لمبائی | ورٹیکس پر داخلی زاویہ |
| کلکس کی تعداد | ۲ | ۳ |
| نتیجے کی شکل | `12.3456` (اکائیاں) | `45.0000°` |
| کینوس پیش نظارہ | پہلے نقطے سے کرسر تک لائن | ورٹیکس سے کرسر تک دو لائنیں |
| بہترین | فاصلے یا سیگمنٹ کی لمبائی | دو خصوصیات کے درمیان زاویہ |

## کوآرڈینیٹ اندراج

کلک کرنے کی بجائے، دونوں نقاط میں سے کسی کے لیے بھی درست پوزیشن ٹائپ کریں:

1. X قدر ٹائپ کریں۔
2. `,` دبائیں — ٹرمینل `[X], [Y{cursor}]` دکھاتا ہے۔
3. Y قدر ٹائپ کریں۔
4. تصدیق کے لیے **Enter** دبائیں۔

## کی بورڈ حوالہ

| کلید | عمل |
|-----|--------|
| `0`–`9`, `.`, `-` | X کوآرڈینیٹ اندراج شروع کریں |
| `,` | X لاک کریں اور Y اندراج پر جائیں |
| `Backspace` | آخری ٹائپ کیا گیا حرف حذف کریں |
| `Enter` | ٹائپ کیا گیا کوآرڈینیٹ تصدیق کریں |
| `Escape` | منسوخ کریں اور مرحلہ ۲ پر ری سیٹ کریں |

## نوٹس

- نتائج صرف **ٹرمینل میں** دکھائے جاتے ہیں — ڈرائنگ میں کچھ شامل نہیں ہوتا۔
- نتیجہ ڈرائنگ کوآرڈینیٹ کی طرح ہی اکائیوں میں ظاہر ہوتا ہے (کوئی اکائی تبدیلی نہیں)۔
- درستگی ہمیشہ ۴ اعشاریہ مقامات ہوتی ہے۔
