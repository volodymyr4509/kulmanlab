# New File

**New File** کمانڈ کینوس صاف کرتی ہے اور نئی خالی ڈرائنگ شروع کرتی ہے۔ ٹائم اسٹیمپ کے ساتھ منفرد فائل نام خود بخود بنتا ہے۔

## نئی فائل بنانے کا طریقہ

File پینل میں **New File** ٹول بار بٹن (نیا صفحہ آئیکن) پر کلک کریں۔ کینوس فوری طور پر صاف ہوتا ہے — کوئی پرامپٹ یا تصدیق ڈائیلاگ نہیں۔

## نئی فائل میں کیا ہوتا ہے

نئی بنائی گئی فائل سے شروع ہوتا ہے:

- کینوس پر **کوئی اشیاء نہیں**۔
- **ایک ڈیفالٹ لیئر** جس کا نام `0`، رنگ سفید اور لائن ٹائپ `Continuous`۔
- **بنایا گیا فائل نام** `kulman-Mon01_HH:MM:SS.dxf` فارمیٹ میں (مثلاً `kulman-May22_14:30:00.dxf`)۔

فائل خود بخود براؤزر اسٹوریج میں محفوظ ہوتی ہے اور [حالیہ فائلز](./files) میں ظاہر ہوتی ہے۔

## خبردار — غیر محفوظ کام حذف ہو جائے گا

**New File** پر کلک کرنے سے موجودہ کینوس کی تمام اشیاء بغیر کسی انتباہ کے حذف ہو جاتی ہیں۔ اگر آپ موجودہ ڈرائنگ رکھنا چاہتے ہیں، تو پہلے اسے [برآمد](./export) کریں۔

## New File بمقابلہ Import

| صورتحال | تجویز کردہ عمل |
|-----------|-------------------|
| سکریچ سے ڈرائنگ شروع کرنا | **New File** |
| موجودہ DXF یا JSON فائل کھولنا | [Import](./import) |
| متغیر پر کام کرنے کے لیے ڈرائنگ کاپی کرنا | موجودہ فائل [برآمد](./export) کریں، پھر نقل [درآمد](./import) کریں |

## متعلقہ کمانڈز

- [Import](./import) — موجودہ DXF یا JSON ڈرائنگ کھولیں
- [Export](./export) — تازہ شروع کرنے سے پہلے ڈرائنگ ڈاؤن لوڈ کریں
- [Files](./files) — براؤزر اسٹوریج سے پچھلی ڈرائنگ بحال کریں
