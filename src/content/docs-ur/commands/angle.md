# Angle

`angle` کمانڈ تین کلک کیے گئے نقاط سے دو لائن سیگمنٹوں کے ورٹیکس پر داخلی زاویہ ناپتی ہے۔ نتیجہ — ہمیشہ 0° اور 180° کے درمیان — ٹرمینل میں ۴ اعشاریہ مقامات تک دکھایا جاتا ہے۔ یہ تین پیمائش کمانڈز میں سے ایک ہے — [Distance](../distance/) سیدھی لائن کی لمبائی ناپتی ہے، اور [Area](../area/) کثیرالاضلاع کا رقبہ اور محیط ناپتی ہے۔

## زاویہ پیمائش کا خاکہ

```
  ● پہلا نقطہ (پہلی شعاع کا اختتام)
   \
    \  پہلی شعاع کا پیش نظارہ
     \
      ● ورٹیکس (مرحلہ ۳)
     /
    /  دوسری شعاع کا پیش نظارہ (کرسر تک)
   /
  ● تیسرا نقطہ  →  ٹرمینل: "Angle: 45.0000°"
```

- **پہلا نقطہ** — زاویے کا ایک اختتام (مرحلہ ۲)۔
- **ورٹیکس** — وہ کونا جہاں زاویہ ناپا جاتا ہے (مرحلہ ۳)۔
- **تیسرا نقطہ** — زاویے کا دوسرا اختتام (مرحلہ ۴)۔

## زاویہ ناپنا

1. ٹرمینل میں `angle` ٹائپ کریں یا **Angle** ٹول بار بٹن پر کلک کریں۔
2. **پہلے نقطے پر کلک کریں** — زاویے کا ایک بازو اختتام۔ یا درست کوآرڈینیٹ کے لیے `X,Y` ٹائپ کریں اور **Enter** دبائیں۔
3. **ورٹیکس پر کلک کریں** — وہ کونا جہاں دونوں بازو ملتے ہیں۔ کوآرڈینیٹ اندراج یہاں بھی کام کرتا ہے۔
4. **تیسرے نقطے پر کلک کریں** — دوسرا بازو اختتام۔ کوآرڈینیٹ اندراج یہاں بھی کام کرتا ہے۔ اس نقطے کو رکھنے سے نتیجہ پرنٹ ہوتا ہے۔
5. **دوبارہ کلک کریں** (اختیاری) اس کلک کو نئے پہلے نقطے کے طور پر نئی پیمائش شروع کریں۔

## داخلی زاویے کا اصول

کمانڈ ورٹیکس سے دو شعاعوں کے ڈاٹ پروڈکٹ سے زاویہ حساب کرتی ہے:

- **ہمیشہ داخلی**: نتیجہ چھوٹا زاویہ ہوتا ہے، 0° اور 180° کے درمیان۔
- اختتامی نقاط کلک کرنے کی ترتیب نتیجے کو متاثر نہیں کرتی — صرف ورٹیکس کی پوزیشن اہم ہے۔
- ہم خط نقاط (تینوں ایک لائن پر) 0° یا 180° واپس کرتے ہیں۔

## پیمائشیں جوڑنا

نتیجہ ظاہر ہونے کے بعد، کلک کرنے سے فوری طور پر اگلی پیمائش شروع ہوتی ہے — کلک کیا گیا نقطہ نیا پہلا نقطہ بن جاتا ہے۔ کمانڈ خود بخود نہیں نکلتی جب تک آپ `Escape` نہ دبائیں۔

## Angle بمقابلہ Distance

| | Angle | Distance |
|---|-------|---------|
| کیا ناپتا ہے | ورٹیکس پر داخلی زاویہ | سیدھی لائن کی لمبائی |
| کلکس کی تعداد | ۳ | ۲ |
| نتیجے کی شکل | `45.0000°` | `12.3456` (اکائیاں) |
| کینوس پیش نظارہ | ورٹیکس سے دونوں اختتامات تک دو لائنیں | پہلے نقطے سے کرسر تک لائن |
| بہترین | دو خصوصیات کے درمیان زاویہ | فاصلے یا سیگمنٹ کی لمبائی |

## کوآرڈینیٹ اندراج

کلک کرنے کی بجائے، تینوں نقاط میں سے کسی کے لیے بھی درست پوزیشن ٹائپ کریں:

1. X قدر ٹائپ کریں۔
2. `,` دبائیں — ٹرمینل `[X], [Y{cursor}]` دکھاتا ہے۔
3. Y قدر ٹائپ کریں۔
4. تصدیق کے لیے **Enter** دبائیں۔

## کی بورڈ حوالہ

| کلید | عمل |
|-----|--------|
| `0`–`9`, `.`, `-` | X کوآرڈینیٹ اندراج شروع کریں |
| `,` | X لاک کریں اور Y اندراج پر جائیں |
| `Backspace` | آخری ٹائپ کیا گیا حرف حذف کریں |
| `Enter` | ٹائپ کیا گیا کوآرڈینیٹ تصدیق کریں |
| `Escape` | منسوخ کریں اور مرحلہ ۲ پر ری سیٹ کریں |

## نوٹس

- نتائج صرف **ٹرمینل میں** دکھائے جاتے ہیں — ڈرائنگ میں کچھ شامل نہیں ہوتا۔
- درستگی ہمیشہ ڈگریوں میں ۴ اعشاریہ مقامات ہوتی ہے۔
