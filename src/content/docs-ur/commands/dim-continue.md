# Dimension Continue

`dimcontinue` کمانڈ موجودہ پیمائش کی **دوسری ایکسٹینشن لائن** سے نئی پیمائشیں جوڑتی ہے۔ ہر نئے سیگمنٹ کو بیس کی طرح ایک ہی پیمائش محور اور ایک ہی پیمائش لائن آفسیٹ پر رکھا جاتا ہے۔ تمام انداز خصوصیات — تیر کا سائز، متن اونچائی، ایکسٹینشن لائن لمبائیاں — بیس سے خود بخود نقل کی جاتی ہیں۔

## جڑی پیمائشیں کیسی دکھتی ہیں

```
  |←— 3.00 —→|←— 2.50 —→|←— 4.00 —→|
  |           |           |           |
  ●           ●           ●           ●
  p1        p2 (بیس    p3           p4
           ext2 → نئی ابتدا)
```

ہر مستطیل ایک علیحدہ `DIMENSION` شے ہے۔ وہ ایک ہی پیمائش لائن پوزیشن اور پیمائش سمت شیئر کرتے ہیں۔

## زنجیر شروع کرنا

1. ٹرمینل میں `dimcontinue` ٹائپ کریں یا **Dimension Continue** ٹول بار بٹن پر کلک کریں۔
2. **اگر پیمائش ابھی رکھی گئی ہو** — کمانڈ اسے خود بخود بیس کے طور پر اٹھاتی ہے (کوئی کلک ضروری نہیں)۔
3. **اگر حالیہ پیمائش موجود نہ ہو** — کسی بھی موجودہ پیمائش پر کلک کریں اسے بیس کے طور پر استعمال کرنے کے لیے۔
4. **اگلی ایکسٹینشن لائن کی اصل پر کلک کریں** — پیش نظارہ نئی پیمائش دکھاتا ہے جیسے آپ کرسر حرکت دیتے ہیں۔ یا `X,Y` ٹائپ کریں اور درست کوآرڈینیٹ کے لیے **Enter** دبائیں۔
5. زنجیر بڑھانے کے لیے کلک (یا ٹائپ) کرتے رہیں۔ ہر رکھی گئی پیمائش خود بخود نیا بیس بن جاتی ہے۔
6. زنجیر ختم کرنے کے لیے **Enter**، **Space**، یا **Escape** دبائیں۔

## بیس پیمائش سے کیا وراثت میں ملتا ہے

| خاصیت | بیس سے وراثت میں |
|----------|---------------------|
| پیمائش سمت / زاویہ | ہاں — پوری زنجیر کے لیے لاک |
| پیمائش لائن آفسیٹ (ناپے گئے نقاط سے فاصلہ) | ہاں |
| تیر کا سائز | ہاں |
| متن اونچائی | ہاں |
| ایکسٹینشن لائن آفسیٹ اور توسیع | ہاں |
| متن سیدھ | ہاں |
| انداز کا نام | ہاں |
| Color، Layer | وراثت میں نہیں — موجودہ پرت استعمال ہوتی ہے |

## پیمائش سمت لاکنگ

زنجیر کی پیمائش سمت **بیس پیمائش کے زاویے پر ثابت** ہے:

- Linear بیس (H) → تمام تسلسل افقی فاصلہ ناپتے ہیں (Δ X)۔
- Linear بیس (V) → تمام تسلسل عمودی فاصلہ ناپتے ہیں (Δ Y)۔
- کسی بھی زاویے پر Aligned بیس → تمام تسلسل اس ایک زاویے کے ساتھ ناپتے ہیں۔

آپ زنجیر کے درمیان میں سمت نہیں بدل سکتے۔ مختلف سمت میں پیمائش کے لیے نئی [Dimension Linear](../dim-linear/) یا [Dimension Aligned](../dim-aligned/) شروع کریں۔

## کی بورڈ حوالہ

| کلید | عمل |
|-----|--------|
| `0`–`9`, `.`, `-` | X کوآرڈینیٹ اندراج شروع کریں |
| `,` | X لاک کریں اور Y اندراج پر جائیں |
| `Backspace` | آخری ٹائپ کیا گیا حرف حذف کریں |
| `Enter` | ٹائپ کیا گیا کوآرڈینیٹ تصدیق کریں، یا کوئی ان پٹ جاری نہ ہونے پر زنجیر ختم کریں |
| `Space` / `Escape` | زنجیر ختم کریں |

## Dimension Continue بمقابلہ تازہ شروع کرنا

| | Dimension Continue | Dimension Linear / Aligned |
|---|-------------------|--------------------------|
| ابتدائی نقطہ | آخری بیس کے ext2 پر ثابت | کہیں بھی کلک کریں |
| زاویہ | بیس پر لاک | آزاد |
| آفسیٹ | بیس سے وراثت میں | کرسر یا ٹائپ سے سیٹ |
| انداز | بیس سے وراثت میں | موجودہ انداز |
| بہترین | ایک قطار میں مجموعی پیمائشیں | پہلی پیمائش یا سمت بدلنا |

## رکھاؤ کے بعد لیبلز ترمیم کرنا — سادہ موڈ

زنجیر میں کسی بھی پیمائش پر **ڈبل کلک** کریں **سادہ** موڈ میں متن ایڈیٹر کھولنے کے لیے۔ ہر سیگمنٹ آزاد ہے اور انفرادی طور پر ترمیم کیا جا سکتا ہے۔

| خصوصیت | سلوک |
|---------|-----------|
| Bold / Italic / Font / Height | **پورے** لیبل پر یکبارگی لاگو ہوتی ہے |
| فی حرف فارمیٹنگ | حمایت یافتہ نہیں |
| `Enter` | قدر محفوظ کرتا ہے اور ایڈیٹر بند کرتا ہے |
| ملٹی لائن | حمایت یافتہ نہیں |

مکمل حوالہ کے لیے [Text Editor — simple mode](../../interface/text-editor/#simple-mode) دیکھیں۔

## DXF — DIMENSION اشیاء

زنجیر کا ہر سیگمنٹ DXF فائل میں ایک آزاد `DIMENSION` شے کے طور پر محفوظ ہوتا ہے۔ وہ فائل میں جڑے نہیں ہیں — وہ خصوصیات شیئر کرتے ہیں کیونکہ ایک ہی بیس سے بنائے گئے تھے، لیکن ہر ایک کو رکھاؤ کے بعد انفرادی طور پر ترمیم کی جا سکتی ہے۔
