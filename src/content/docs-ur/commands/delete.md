# Delete

`delete` کمانڈ ڈرائنگ سے منتخب اشیاء ہٹاتی ہے۔ حذف کاروائیاں [Undo](../undo/) تاریخ میں ریکارڈ ہوتی ہیں اور 20 مراحل تک واپس کی جا سکتی ہیں۔ کوئی علیحدہ "تصدیق حذف" ڈائیلاگ نہیں — تصدیق ایک واحد کی پریس ہے۔

## حذف کرنے کے دو طریقے

**پیشگی منتخب، پھر حذف کریں** — سب سے تیز راستہ:

1. کینوس پر ایک یا زیادہ اشیاء منتخب کریں۔
2. ٹرمینل میں `delete` ٹائپ کریں، **Delete** ٹول بار بٹن پر کلک کریں، **یا براہ راست `Delete` کلید دبائیں**۔

اشیاء فوری طور پر ہٹا دی جاتی ہیں — کوئی اضافی تصدیق مرحلہ نہیں۔

**فعال کریں، پھر منتخب کریں**:

1. `delete` ٹائپ کریں یا ٹول بار بٹن پر کلک کریں (کچھ منتخب کیے بغیر)۔
2. **اشیاء منتخب کریں** — ٹوگل کرنے کے لیے کلک کریں، یا علاقے کے لیے گھسیٹیں۔
3. منتخب اشیاء تصدیق کرنے اور ہٹانے کے لیے **Enter**، **Space**، یا **Delete** دبائیں۔

## Delete کلید شارٹ کٹ

کی بورڈ پر `Delete` کلید ایک **عالمی شارٹ کٹ** کے طور پر کام کرتی ہے — اگر کوئی اشیاء فی الحال منتخب ہیں، تو اسے دبانے سے وہ فوری طور پر حذف ہو جاتی ہیں، یہاں تک کہ ٹرمینل میں Delete کمانڈ کھولے بغیر۔ یہ سب سے تیز واحد مرحلہ حذف ورک فلو ہے:

```
شے کلک کریں → Delete کلید دبائیں → ہو گیا
```

## کمانڈ کے دوران انتخاب

| طریقہ | برتاؤ |
|--------|-----------|
| **کلک** | کرسر کے نیچے شے کو انتخاب میں شامل/خارج کرتا ہے |
| **دائیں گھسیٹیں** (سخت) | صرف وہ اشیاء منتخب کرتا ہے جو باکس کے مکمل اندر ہوں |
| **بائیں گھسیٹیں** (کراسنگ) | باکس کی سرحد کو کاٹنے والی اشیاء منتخب کرتا ہے |
| **Enter** / **Space** / **Delete** | منتخب اشیاء تصدیق اور حذف کرتا ہے |

## حذف کردہ اشیاء بازیاب کرنا

حذف کاروائیاں [Undo](../undo/) کمانڈ سے واپس کی جا سکتی ہیں (`undo` ٹائپ کریں یا ٹول بار بٹن استعمال کریں)۔ فی فائل **20 مراحل** تک واپس کیے جا سکتے ہیں، اور تاریخ صفحہ ری لوڈ کے بعد بھی برقرار رہتی ہے۔ اگر آپ نے محفوظ کیے بغیر 20 حذف کاروائیاں کر لی ہیں، تو پہلے کی حذف کاروائیاں واپس نہیں کی جا سکتیں۔

## معاون اشیاء

Delete ہر شے کی قسم پر کام کرتی ہے — Line، Polyline، Rectangle، Circle، Arc، Ellipse، Text، Spline، Dimension، Leader، اور دیگر تمام۔
