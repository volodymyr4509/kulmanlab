# Dimension Linear

`dimlinear` کمانڈ دو ایکسٹینشن لائن اصل کے درمیان افقی یا عمودی پیمائش رکھتی ہے۔ پیمائش لائن ہمیشہ یا تو بالکل افقی یا بالکل عمودی چلتی ہے — یہ کسی بھی من مانے زاویے پر نہیں رکھی جا سکتی۔ ترچھی لائن کے متوازی پیمائش کے لیے [Dimension Aligned](../dim-aligned/) استعمال کریں۔

## linear پیمائش کا خاکہ

```
  |←————— 5.00 —————→|
  |                   |
  ●  (ext line 1)     ●  (ext line 2)
  p1                  p2
```

- **ایکسٹینشن لائنیں** — ہر ناپے گئے نقطے سے پیمائش لائن کے عمودی گرتی ہیں۔
- **پیمائش لائن** — افقی (X فاصلہ ناپتی ہے) یا عمودی (Y فاصلہ ناپتی ہے)۔
- **قدر** — چنے گئے محور کے ساتھ پیش کردہ فاصلہ، نقطے سے نقطے کا اصل فاصلہ نہیں۔

## linear پیمائش رکھنا

1. ٹرمینل میں `dimlinear` ٹائپ کریں یا **Dimension Linear** ٹول بار بٹن پر کلک کریں۔
2. **پہلی ایکسٹینشن لائن کی اصل پر کلک کریں** (p1)، یا درست کوآرڈینیٹ کے لیے `X,Y` ٹائپ کریں اور **Enter** دبائیں۔
3. **دوسری ایکسٹینشن لائن کی اصل پر کلک کریں** (p2)۔ کوآرڈینیٹ اندراج یہاں بھی کام کرتا ہے۔
4. **کرسر حرکت دیں** پیمائش لائن پوزیشن کریں۔ رخ خود بخود کرسر کی پوزیشن سے پکڑا جاتا ہے۔
5. **کلک کریں** رکھنے کے لیے، یا آفسیٹ فاصلہ ٹائپ کریں اور درست رکھاؤ کے لیے **Enter** دبائیں۔

## خودکار رخ پکڑنا

جب کوئی رخ مجبور نہ ہو، کمانڈ دو ناپے گئے نقاط کے نسبت کرسر کی پوزیشن پڑھتی ہے:

| کرسر پوزیشن | پکڑا گیا رخ | کیا ناپا جاتا ہے |
|-----------------|---------------------|-----------------|
| نقاط کے اوپر یا نیچے | افقی | p1 اور p2 کے درمیان Δ X |
| نقاط کے بائیں یا دائیں | عمودی | p1 اور p2 کے درمیان Δ Y |

رکھاؤ مرحلے کے دوران کسی بھی وقت **H** دبائیں افقی یا **V** عمودی لاک کرنے کے لیے۔ ایک بار لاک ہونے پر، کرسر حرکت کرنے پر رخ نہیں بدلتا۔

## ٹائپ کردہ آفسیٹ فاصلہ

رکھاؤ کے دوران نمبر ٹائپ کریں پیمائش لائن کو ناپے گئے نقاط سے بالکل فاصلے پر ٹھیک کرنے کے لیے:

| کلید | عمل |
|-----|--------|
| `0`–`9`, `.` | آفسیٹ فاصلے میں ہندسہ شامل کریں |
| `Backspace` | آخری ٹائپ کیا گیا حرف حذف کریں |
| `Enter` / `Space` | ٹائپ کیے گئے فاصلے پر رکھیں |

کرسر کی طرف (افقی کے لیے اوپر/نیچے، عمودی کے لیے بائیں/دائیں) اشارہ طے کرتی ہے — پیمائش لائن اس طرف ظاہر ہوتی ہے جہاں کرسر فی الحال ہے۔

## کی بورڈ حوالہ

| کلید | عمل |
|-----|--------|
| `0`–`9`, `.`, `-` | X کوآرڈینیٹ اندراج شروع کریں (p1/p2 مراحل)، یا آفسیٹ فاصلہ (رکھاؤ مرحلہ) |
| `,` | X لاک کریں اور Y اندراج پر جائیں (p1/p2 مراحل) |
| `H` | افقی رخ لاک کریں (صرف رکھاؤ مرحلہ) |
| `V` | عمودی رخ لاک کریں (صرف رکھاؤ مرحلہ) |
| `Backspace` | آخری ٹائپ کیا گیا حرف حذف کریں |
| `Enter` / `Space` | ٹائپ کیا گیا کوآرڈینیٹ یا آفسیٹ تصدیق کریں |
| `Escape` | منسوخ کریں |

## Dimension Linear بمقابلہ Dimension Aligned

| | Dimension Linear | Dimension Aligned |
|---|-----------------|------------------|
| محور | ہمیشہ H یا V | ناپی گئی لائن کے متوازی |
| ناپتا ہے | صرف X یا Y جز | اصل یوکلیڈین فاصلہ |
| H/V کلیدیں | ہاں — رخ لاک کریں | نہیں — ہمیشہ p1→p2 کے ساتھ |
| بہترین | ہم مثلث لے آؤٹ، فرش کے نقشے | ترچھی خصوصیات، زاویہ دار کٹ |

## لیبل ترمیم کرنا — سادہ موڈ

رکھی گئی linear پیمائش پر **ڈبل کلک** کریں **سادہ** موڈ میں متن ایڈیٹر کھولنے کے لیے۔ ایڈیٹر میں موجودہ رینڈر شدہ قدر پہلے سے بھری ہوتی ہے تاکہ آپ کرسر پوزیشن کر کے براہ راست ترمیم کر سکیں۔

| خصوصیت | سلوک |
|---------|-----------|
| Bold / Italic / Font / Height | **پورے** لیبل پر یکبارگی لاگو ہوتی ہے |
| فی حرف فارمیٹنگ | حمایت یافتہ نہیں |
| `Enter` | قدر محفوظ کرتا ہے اور ایڈیٹر بند کرتا ہے |
| ملٹی لائن | حمایت یافتہ نہیں |

مکمل حوالہ کے لیے [Text Editor — simple mode](../../interface/text-editor/#simple-mode) دیکھیں۔

## پیمائشیں جوڑنا

آخری ایکسٹینشن لائن سے جاری رہنے والی مزید پیمائشیں شامل کرنے کے لیے، اسے رکھنے کے فوری بعد [Dimension Continue](../dim-continue/) استعمال کریں۔

## DXF — DIMENSION شے

Linear پیمائشیں DXF میں `rotationDeg` کے ساتھ `DIMENSION` اشیاء کے طور پر محفوظ ہوتی ہیں جو `0` (افقی) یا `90` (عمودی) پر سیٹ ہوتی ہے۔ ایکسٹینشن لائن اصل، پیمائش لائن پوزیشن، متن پوزیشن، ناپی گئی قدر، تیر کا انداز، متن اونچائی، اور تمام ڈسپلے فلیگز بغیر نقصان کے راؤنڈ ٹرپ کرتے ہیں۔
