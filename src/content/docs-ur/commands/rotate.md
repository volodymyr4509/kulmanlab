# Rotate

`rotate` کمانڈ منتخب اشیاء کو بنیادی نقطے کے گرد گھماتی ہے۔ آپ گھماؤ کا زاویہ یا تو ڈگریوں میں نمبر ٹائپ کر کے یا کلک سے — زاویہ بنیادی نقطے اور کلک پوزیشن کے درمیان سمت سے حساب ہوتا ہے۔

## شروع کرنے کے دو طریقے

**پہلے منتخب کریں، پھر گھمائیں** — پہلے اشیاء منتخب کریں، پھر فعال کریں:

1. کینوس پر ایک یا زیادہ اشیاء منتخب کریں۔
2. ٹرمینل میں `rotate` ٹائپ کریں یا **Rotate** ٹول بار بٹن پر کلک کریں۔
3. **بنیادی نقطے پر کلک کریں** — گھماؤ کا مرکز۔ یا درست کوآرڈینیٹ کے لیے `X,Y` ٹائپ کریں اور **Enter** دبائیں۔
4. **زاویہ ٹائپ کریں اور Enter دبائیں**، یا کرسر سمت سے زاویہ سیٹ کرنے کے لیے **کلک کریں**۔

**پہلے فعال کریں، پھر منتخب کریں** — بغیر کچھ منتخب کیے کمانڈ شروع کریں:

1. `rotate` ٹائپ کریں یا ٹول بار بٹن پر کلک کریں۔
2. **اشیاء منتخب کریں** — ٹوگل کرنے کے لیے کلک کریں، یا علاقے کے ذریعے منتخب کرنے کے لیے ڈریگ کریں۔
3. انتخاب تصدیق کرنے کے لیے **Enter** یا **Space** دبائیں۔
4. **بنیادی نقطے پر کلک کریں** (کوآرڈینیٹ اندراج دستیاب)، پھر زاویہ سیٹ کریں۔

بنیادی نقطہ سیٹ ہونے کے بعد گھمی ہوئی اشیاء کا براہ راست گھوسٹ پیش نظارہ کرسر زاویے کی پیروی کرتا ہے۔

## زاویہ سیٹ کرنا

**ٹائپ کردہ زاویہ** — بنیادی نقطہ رکھنے کے بعد کسی بھی وقت نمبر (ڈگریوں میں) ٹائپ کریں۔ Enter دبانے سے پہلے جاری ایڈجسٹمنٹ کے دوران پیش نظارہ ٹائپ کردہ زاویے پر اسنیپ کرتا ہے۔

**کلک زاویہ** — اگر کوئی ٹائپ کردہ قدر موجود نہ ہو، کلک کرنے سے زاویہ `atan2(cursorY − baseY, cursorX − baseX)` سے مساوی ہوتا ہے — بنیادی نقطے سے کلک تک سمت، ڈگریوں میں۔

| کلید | عمل |
|-----|--------|
| `0`–`9`, `.` | زاویہ قدر میں ہندسہ شامل کریں |
| `-` | منفی زاویہ (صرف پہلا حرف) |
| `Backspace` | آخری ٹائپ کردہ حرف حذف کریں |
| `Enter` | ٹائپ کردہ زاویے پر گھماؤ لاگو کریں |

## زاویہ کی سمت

زاویے **DXF کنونشن** کی پیروی کرتے ہیں:

- **مثبت** قدریں ڈرائنگ کوآرڈینیٹس (Y-اوپر) میں **گھڑی کی مخالف سمت** گھماتی ہیں۔
- اسکرین پر، جہاں Y-محور الٹا (Y-نیچے) ہے، مثبت زاویے **گھڑی کی سمت** ظاہر ہوتے ہیں۔

عام قدریں: `90` = چوتھائی موڑ، `180` = آدھا موڑ، `-90` = مخالف چوتھائی موڑ۔

## کی بورڈ حوالہ

| کلید | عمل |
|-----|--------|
| `Enter` / `Space` | انتخاب تصدیق کریں |
| `0`–`9`, `.`, `-` | X کوآرڈینیٹ اندراج شروع کریں (بنیادی نقطہ مرحلہ)، یا زاویہ قدر (زاویہ مرحلہ) |
| `,` | X لاک کریں اور Y اندراج پر جائیں (بنیادی نقطہ مرحلہ) |
| `Backspace` | آخری ٹائپ کردہ حرف حذف کریں |
| `Enter` | کوآرڈینیٹ تصدیق کریں یا گھماؤ لاگو کریں |
| `Escape` | منسوخ کریں اور ری سیٹ کریں |

## کمانڈ کے دوران انتخاب

| طریقہ | برتاؤ |
|--------|-----------|
| **کلک** | کرسر کے نیچے اینٹیٹی ٹوگل کرتا ہے |
| **دائیں ڈریگ** (سخت) | باکس کے مکمل اندر موجود اشیاء شامل کرتا ہے |
| **بائیں ڈریگ** (کراسنگ) | باکس کو چھونے والی اشیاء شامل کرتا ہے |
| **Enter** / **Space** | انتخاب تصدیق کرتا ہے |

## سپورٹ شدہ اشیاء

Rotate ہر اینٹیٹی قسم پر کام کرتی ہے۔ ہر اینٹیٹی کی جیومیٹری بنیادی نقطے کے گرد گھومتی ہے — مثلاً، دائرہ اپنا مرکز حرکت دیتا ہے جبکہ رداس برقرار رہتا ہے؛ قوس اپنا مرکز حرکت دیتی ہے اور ابتدا و انتہا کے زاویے گھماؤ سے بدلتے ہیں؛ متن اینٹیٹی اپنا لنگر نقطہ حرکت دیتی ہے اور گھماؤ ڈگری خاصیت میں زاویہ شامل ہوتا ہے۔
