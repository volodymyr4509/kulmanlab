# Help

تمام کی بورڈ شارٹ کٹس، نیویگیشن تجاویز، اور عمومی استعمال کے خلاصے کے ساتھ ان-ایپ مدد پینل کھولتا ہے۔

## استعمال کا طریقہ

**Help** ٹول بار بٹن پر کلک کریں یا ٹرمینل میں `help` ٹائپ کریں۔ مدد پینل فوری طور پر کھلتا ہے اور کمانڈ نکل جاتی ہے۔

## مدد پینل میں کیا ہے

### نیویگیشن
| شارٹ کٹ | عمل |
|----------|--------|
| `Scroll wheel` | زوم ان / آؤٹ |
| `Middle click + drag` | نظارہ پین کریں |
| `Double middle click` | تمام اشیاء اسکرین پر فٹ کریں |

### کمانڈز
| شارٹ کٹ | عمل |
|----------|--------|
| ٹائپ کریں | کمانڈ کا نام ٹائپ کرنا شروع کریں — تجاویز کینوس پر ظاہر ہوتی ہیں |
| `Delete` | تمام منتخب اشیاء حذف کریں |
| `Enter` / `Space` | کمانڈ یا ان پٹ تصدیق کریں |
| `Enter` / `Space` (خاموش) | آخری استعمال کردہ کمانڈ دہرائیں |
| `Escape` | موجودہ کمانڈ منسوخ کریں یا انتخاب صاف کریں |

### انتخاب
| شارٹ کٹ | عمل |
|----------|--------|
| `Left click` | ایک شے منتخب کریں |
| `Drag →` | صرف وہ اشیاء منتخب کریں جو **مکمل طور پر** گھسیٹنے کے علاقے کے اندر ہوں |
| `Drag ←` | کوئی بھی شے منتخب کریں جو گھسیٹنے کے علاقے کو **چھوئے** |
| `Left click grip` | ورٹیکس یا اختتامی نقطہ براہ راست ترمیم کریں |

### ڈرائنگ تجاویز
- **اسنیپ نقاط** — کرسر خود بخود قریبی شے اختتامی نقاط پر اسنیپ کرتا ہے۔
- **چپکتا زاویہ** — ایک سمت میں کافی دور کرسر حرکت دیں 45° اضافوں پر لاک کرنے کے لیے۔
- **درست لمبائی** — زاویہ لاک ہونے پر، نمبر ٹائپ کریں اور بالکل فاصلہ سیٹ کرنے کے لیے `Enter` دبائیں۔
