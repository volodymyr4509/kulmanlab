# Export

**Export** کمانڈ موجودہ ڈرائنگ آپ کے فائل سسٹم پر ڈاؤن لوڈ کرتی ہے۔ دو فارمیٹ دستیاب ہیں: دیگر CAD ٹولز سے مطابقت کے لیے **DXF** اور KulmanLab CAD میں مکمل صداقت محفوظ کرنے کے لیے **JSON**۔

## برآمد کرنے کا طریقہ

1. File پینل میں **Export** ٹول بار بٹن (ڈاؤن لوڈ آئیکن) پر کلک کریں۔
2. **Export Manager** پاپ اپ کھلتا ہے۔
3. فارمیٹ منتخب کرنے کے لیے فارمیٹ کارڈ پر کلک کریں — **JSON** یا **DXF**۔
4. **Export** بٹن پر کلک کریں۔ فائل خود بخود آپ کے ڈیفالٹ ڈاؤن لوڈ فولڈر میں ڈاؤن لوڈ ہوتی ہے۔

## فارمیٹ کا انتخاب

| فارمیٹ | ایکسٹینشن | بہترین | حدود |
|--------|-----------|----------|-------------|
| **JSON** *(مقامی)* | `.json` | KulmanLab CAD میں دوبارہ کھولنے کے لیے کام محفوظ کرنا | دیگر CAD ٹولز سے مطابق نہیں |
| **DXF** | `.dxf` | FreeCAD، LibreCAD وغیرہ کے ساتھ شیئر کرنا | پیمائشیں اور لیڈرز برآمد نہیں ہوتے |

**JSON کب استعمال کریں:** جب بھی آپ اپنے کام کی مکمل نقل محفوظ کرنا چاہیں۔ JSON KulmanLab کا مقامی فارمیٹ ہے اور ہر شے کو بالکل محفوظ کرتا ہے — پیمائشیں، لیڈرز، اور تمام پرت ڈیٹا سمیت۔

**DXF کب استعمال کریں:** جب آپ کو ڈرائنگ کسی دوسری CAD ایپلیکیشن استعمال کرنے والے کو دینی ہو۔ برآمد کردہ فائل AC1012 DXF فارمیٹ استعمال کرتی ہے اور زیادہ تر DXF-مطابق ٹولز میں کھلتی ہے۔

## فی فارمیٹ کیا برآمد ہوتا ہے

### JSON برآمد

تمام شے کی اقسام شامل ہیں:

- لائنیں، دائرے، قوس، بیضوی، polylines، splines، متن
- پیمائشیں (linear، aligned، continued، radius، diameter)
- Multileaders
- پرت کی تعریفیں اور لائن ٹائپ ٹیبل

### DXF برآمد

صرف ہندسے کی اشیاء شامل ہیں:

- لائنیں، دائرے، قوس، بیضوی، polylines (بطور `LWPOLYLINE` برآمد)، splines، متن
- پرت کی تعریفیں اور لائن ٹائپ ٹیبل

**DXF میں شامل نہیں:** پیمائش اشیاء اور multileaders۔ یہ KulmanLab-مخصوص ڈیٹا ڈھانچے استعمال کرتے ہیں جو معیاری DXF میں درست طریقے سے ظاہر نہیں کیے جا سکتے۔ اگر آپ کی ڈرائنگ میں تشریحیں ہیں، تو JSON یا [Print](./print) استعمال کریں انہیں بصری طور پر پکڑنے کے لیے۔

## برآمد کردہ فائل کا نام

ڈاؤن لوڈ کردہ فائل موجودہ ڈرائنگ فائل کے نام پر رکھی جاتی ہے (مثلاً `myplan_May22_14:30:00.json`)۔ ایکسٹینشن چنے گئے فارمیٹ سے ملانے کے لیے تبدیل ہوتی ہے۔

## Export اور Print کے درمیان فرق

| خصوصیت | Export | Print |
|---------|--------|-------|
| آؤٹ پٹ | ویکٹر سورس فائل (.dxf / .json) | رسٹر تصویر (.png / .jpeg / .webp / .pdf) |
| دیگر ٹولز میں قابل ترمیم | ہاں (DXF) | نہیں |
| پرتیں اور لائن ٹائپز محفوظ | ہاں | نہیں (چپٹا رینڈر) |
| پیمائشیں اور لیڈرز پکڑتا ہے | صرف JSON | ہاں |

قابل ترمیم فائل کے لیے **Export** استعمال کریں۔ بصری عکس کے لیے [Print](./print) استعمال کریں۔
