# Offset

`offset` کمانڈ ایک مقررہ عمودی فاصلے پر اینٹیٹی کی متوازی نقل بناتی ہے۔ آپ فاصلہ ایک بار ٹائپ کرتے ہیں، پھر اشیاء پر کلک کریں اور طرف چنیں — کمانڈ اسی فاصلے پر تیار رہتی ہے تاکہ آپ ایک سیشن میں متعدد اشیاء آفسیٹ کر سکیں۔

سپورٹ شدہ اینٹیٹی اقسام: **Line، Circle، Arc، Ellipse، Polyline** (Rectangles سمیت)۔

## آفسیٹ استعمال کرنا

1. ٹرمینل میں `offset` ٹائپ کریں یا **Offset** ٹول بار بٹن پر کلک کریں۔
2. **آفسیٹ فاصلہ ٹائپ کریں** اور **Enter** یا **Space** دبائیں۔
3. آفسیٹ کے لیے **کوئی اینٹیٹی کلک کریں** — اگر اینٹیٹی سپورٹ شدہ قسم نہیں ہے، تو خطا پیغام ظاہر ہوتا ہے اور آپ مختلف اینٹیٹی کلک کر سکتے ہیں۔
4. **کرسر اس طرف لے جائیں** جہاں نقل ظاہر ہونی چاہیے — براہ راست پیش نظارہ پیچھے آتا ہے۔
5. **کلک کریں** آفسیٹ نقل رکھنے کے لیے۔

ہر رکھاؤ کے بعد کمانڈ **اسی فاصلے** پر مرحلہ ۳ پر واپس آتی ہے، اگلے آفسیٹ کے لیے تیار۔ فاصلہ اندراج مرحلے پر واپس ری سیٹ کرنے کے لیے **Escape** دبائیں۔

```
  فاصلہ: 10

  ─────────────────    ← اصل لائن
  ─────────────────    ← آفسیٹ نقل (10 یونٹ نیچے)
```

## فی اینٹیٹی آفسیٹ برتاؤ

| اینٹیٹی | آفسیٹ کیسے حساب ہوتی ہے |
|--------|---------------------------|
| **لائن** | اصل سمت کے عمود پر منتقل متوازی لائن |
| **دائرہ** | ہم مرکز دائرہ؛ باہر کلک → بڑا رداس، اندر → چھوٹا رداس |
| **قوس** | نئے رداس پر ہم مرکز قوس؛ وہی زاویہ پھیلاؤ محفوظ |
| **بیضہ** | دونوں نیم محور ایک ہی فاصلے سے بڑھے یا کم ہوئے |
| **پولی لائن** | ہر سیگمنٹ آزادانہ آفسیٹ؛ ہمسایہ آفسیٹ سیگمنٹس کونوں پر مائیٹر |

**دائرہ**، **قوس**، اور **بیضہ** کے لیے: اگر اندرونی آفسیٹ کوئی رداس یا نیم محور صفر یا اس سے کم کر دے، تو آفسیٹ لاگو نہیں ہوتی۔

## کی بورڈ حوالہ

| کلید | عمل |
|-----|--------|
| `0`–`9`, `.` | فاصلہ قدر میں ہندسہ شامل کریں |
| `Backspace` | آخری ٹائپ کردہ حرف حذف کریں |
| `Enter` / `Space` | ٹائپ کردہ فاصلہ تصدیق کریں اور اینٹیٹی انتخاب پر جائیں |
| `Escape` | فاصلہ اندراج مرحلے پر ری سیٹ کریں |

## کام کا نوٹ

فاصلہ **Escape** دبانے تک سیٹ رہتا ہے۔ اس سے ایک ہی فاصلے پر کئی اشیاء آفسیٹ کرنا آسان ہو جاتا ہے — فاصلہ ایک بار ٹائپ کریں، پھر ہر اینٹیٹی کے لیے کلک کریں اور طرف چنیں۔

## Offset بمقابلہ Copy

| | Offset | Copy |
|---|--------|------|
| بے گھری | اینٹیٹی کی جیومیٹری کے عمود پر | من مانی ویکٹر (بنیاد → منزل) |
| سپورٹ شدہ اشیاء | Line، Circle، Arc، Ellipse، Polyline | تمام اینٹیٹی اقسام |
| فاصلہ اندراج | اینٹیٹی منتخب کرنے سے پہلے ٹائپ | منتخب کرنے کے بعد ٹائپ یا کلک |
| بہترین | متوازی لائنیں، ہم مرکز دائرے، اندرونی/بیرونی راستے | من مانی پوزیشنز پر نقلیں رکھنا |
