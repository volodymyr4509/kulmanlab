# Array Grid

`ArrayGrid` کمانڈ منتخب اشیاء سے ایک مستطیل گرڈ کی نقلیں بناتی ہے — قطاروں کی تعداد، کالموں کی تعداد اور ان کے درمیان فاصلہ درج کریں، سب کچھ ٹرمینل میں ٹائپ کیا جاتا ہے۔ اصل انتخاب قطار 0، کالم 0 خانے پر رہتا ہے؛ ہر دوسرا خانہ ایک منتقل شدہ نقل ہے۔

## شروع کرنے کے دو طریقے

**پیشگی منتخب، پھر array کریں** — پہلے اشیاء منتخب کریں، پھر فعال کریں:

1. کینوس پر ایک یا زیادہ اشیاء منتخب کریں۔
2. ٹرمینل میں `arraygrid` ٹائپ کریں (صرف `arr` کافی ہے — یہ واضح ہے) یا **Array Grid** ٹول بار بٹن پر کلک کریں۔
3. **قطاروں** کی تعداد ٹائپ کریں اور **Enter** دبائیں۔
4. **کالموں** کی تعداد ٹائپ کریں اور **Enter** دبائیں۔
5. **قطاروں کے درمیان فاصلہ** ٹائپ کریں اور **Enter** دبائیں۔
6. **کالموں کے درمیان فاصلہ** ٹائپ کریں اور **Enter** دبائیں — گرڈ فوراً بن جاتا ہے۔

**فعال کریں، پھر منتخب کریں** — کچھ منتخب کیے بغیر کمانڈ شروع کریں:

1. `arraygrid` ٹائپ کریں یا ٹول بار بٹن پر کلک کریں۔
2. **اشیاء منتخب کریں** — انفرادی اشیاء ٹوگل کرنے کے لیے کلک کریں، یا علاقے کے لیے گھسیٹیں۔
3. انتخاب کی تصدیق کے لیے **Enter** یا **Space** دبائیں۔
4. اوپر کی طرح قطاروں → کالموں → قطار فاصلہ → کالم فاصلہ کے ساتھ جاری رکھیں۔

```
  2 قطاریں x 3 کالم:

  [B] [B] [B]   <- قطار 1 (منتقل شدہ نقلیں)
  [A] [A] [A]   <- قطار 0: اصل انتخاب، نقلیں دائیں طرف
```

> ٹرمینل کو صرف اتنے حروف کی ضرورت ہوتی ہے جتنے واضح ہونے کے لیے کافی ہوں — `arr` ٹائپ کرنا اور **Enter** دبانا براہ راست Array Grid کو فعال کرتا ہے، کیونکہ کوئی دوسری کمانڈ کا نام ان تین حروف سے شروع نہیں ہوتا (Arc، Area، Align، اور Angle سب پہلے ہی الگ ہو جاتے ہیں)۔

## قطاریں، کالم اور فاصلہ

| پرامپٹ | قبول کرتا ہے | نوٹس |
|--------|---------------|------|
| قطاریں | مثبت پورے نمبر (1, 2, 3…) | صرف ہندسے — کوئی اعشاریہ یا نشان نہیں |
| کالم | مثبت پورے نمبر (1, 2, 3…) | صرف ہندسے — کوئی اعشاریہ یا نشان نہیں |
| قطار فاصلہ | نشان والا اعشاریہ (مثلاً `10`, `-5.5`) | قطاروں کے درمیان فاصلہ؛ منفی سمت الٹ دیتا ہے |
| کالم فاصلہ | نشان والا اعشاریہ (مثلاً `10`, `-5.5`) | کالموں کے درمیان فاصلہ؛ منفی سمت الٹ دیتا ہے |

1 قطار اور 1 کالم کے ساتھ، کوئی نقل نہیں بنتی — کمانڈ ڈرائنگ میں تبدیلی کیے بغیر نکل جاتی ہے۔

## کی بورڈ حوالہ

| کلید | عمل |
|-----|--------|
| `Enter` / `Space` | انتخاب تصدیق کریں اور قطار پرامپٹ پر آگے بڑھیں |
| `0`–`9` | قطاروں یا کالموں کے لیے ہندسے درج کریں |
| `0`–`9`, `.`, `-` | قطار/کالم فاصلے کے لیے ہندسے درج کریں (`-` صرف پہلے حرف کے طور پر) |
| `Backspace` | موجودہ پرامپٹ کا آخری ٹائپ کیا گیا حرف حذف کریں |
| `Enter` | موجودہ پرامپٹ کی تصدیق کریں اور اگلے پر آگے بڑھیں |
| `Escape` | ٹائپ کی گئی قطار/کالم/فاصلہ قدریں صاف کریں اور انتخاب مرحلے پر واپس جائیں |

## کمانڈ کے دوران انتخاب

| طریقہ | برتاؤ |
|--------|-----------|
| **کلک** | کرسر کے نیچے شے کو انتخاب میں شامل/خارج کرتا ہے |
| **دائیں گھسیٹیں** (سخت) | باکس کے مکمل اندر اشیاء شامل کرتا ہے |
| **بائیں گھسیٹیں** (کراسنگ) | باکس کی سرحد کو کاٹنے والی اشیاء شامل کرتا ہے |
| **Enter** / **Space** | انتخاب کی تصدیق کرتا ہے اور قطار پرامپٹ پر آگے بڑھتا ہے |

## array کے بعد

نئی نقلیں ڈرائنگ میں شامل ہوتی ہیں اور کمانڈ نکل جاتی ہے — اصل انتخاب صاف ہو جاتا ہے۔ **Array Grid** دوبارہ چلائیں، یا نئی کمانڈ شروع کریں۔

## Array Grid بمقابلہ Copy

| | Array Grid | Copy |
|---|-----------|------|
| پوائنٹ کا انتخاب | کوئی نہیں — قطاریں، کالم اور فاصلہ ٹائپ کیے جاتے ہیں | بنیادی نقطہ اور منزل پر کلک کیا جاتا ہے (یا ٹائپ کیا جاتا ہے) |
| بننے والی نقلیں | قطاریں × کالم − 1 | ہر کاپی آپریشن میں بالکل 1 |
| ترتیب | باقاعدہ مستطیل گرڈ | کہیں بھی، کسی بھی آفسیٹ پر |
| بہترین | باقاعدہ پیٹرن میں ایک یونٹ کو دہرانا (سوراخ، ٹائلیں، فاسٹنرز) | کسی بھی مقام پر ایک واحد نقل |

## معاون اشیاء

Array Grid ہر شے کی قسم پر کام کرتی ہے۔ تمام اشیاء اندرونی طور پر `translate(dx, dy)` نافذ کرتی ہیں — وہی عمل جو [Copy](../copy/) اور [Move](../move/) استعمال کرتے ہیں — لہذا کوئی خارج نہیں ہے۔
