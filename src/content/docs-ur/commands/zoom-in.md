# Zoom In

`zoomin` کمانڈ موجودہ زوم سطح کو **1.5×** سے ضرب دیتی ہے اور ویوپورٹ درمیانی نقطے پر مرکوز فوری طور پر ختم ہوتی ہے۔ یہ اسکرول ویل کے ایک ٹک کا ٹول بار متبادل ہے جو کرسر کی بجائے اسکرین مرکز کی طرف زوم کرتا ہے۔

## زوم ان کرنا

**Zoom In** ٹول بار بٹن پر کلک کریں یا ٹرمینل میں `zoomin` ٹائپ کریں۔ زوم فوری طور پر لاگو ہوتا ہے اور کمانڈ ختم ہوتی ہے — کینوس پر کوئی کلک درکار نہیں۔

## 1.5× قدم کیسے کام کرتا ہے

| موجودہ زوم | ایک Zoom In کے بعد |
|-------------|------------------|
| 1.00× | 1.50× |
| 1.50× | 2.25× |
| 10.00× | 15.00× |
| 6,667× | 10,000× (حد) |

زوم سطح ہمیشہ `zoom` لیبل کے ساتھ کینوس کے **نیچے دائیں کونے** میں دکھائی جاتی ہے۔ اوپری حد **10,000×** ہے؛ مزید قدم کوئی اثر نہیں کرتے۔

## ٹول بار Zoom In بمقابلہ اسکرول ویل

| | Zoom In بٹن | اسکرول ویل |
|---|--------------|-------------|
| زوم مرکز | ویوپورٹ درمیانی نقطہ | کرسر پوزیشن |
| قدم کا سائز | فی کلک 1.5× | فی ٹک ~1.1× |
| فعالیت درکار | نہیں | نہیں — ہمیشہ کام کرتا ہے |
| بہترین | موٹا نیویگیشن | درست، کرسر ہدف زوم |

## کی بورڈ حوالہ

اس کمانڈ کے لیے کوئی کی بورڈ شارٹ کٹ نہیں۔ اسکرول ویل استعمال کریں — یہ کسی بھی کمانڈ فعال کیے بغیر کسی بھی وقت کام کرتا ہے۔
