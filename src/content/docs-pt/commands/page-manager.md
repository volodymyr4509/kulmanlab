# PageManager

O comando `PageManager` abre a caixa de diálogo de configurações de página para o layout ativo, permitindo que você altere o formato do papel, a orientação e a escala entre unidades de desenho e milímetros.

## Abrindo o Page Manager

- Digite `PageManager` no terminal, **ou**
- Clique com o botão direito em uma aba de layout na parte inferior e escolha **Page Manager**.

## Configurações

| Configuração | Descrição |
|-------------|-----------|
| **Formato do papel** | Tamanho de folha padrão (A4, A3, A2, Carta, etc.) |
| **Orientação** | Retrato ou Paisagem |
| **Escala** | Unidades de desenho por mm — controla como as coordenadas do modelo mapeiam para o papel |

Alterar o tamanho do papel ou a escala redimensiona proporcionalmente todos os viewports existentes no layout para que suas posições permaneçam corretas.

## Notas

- PageManager aplica-se apenas a layouts de papel. Não está disponível no espaço modelo.
- Para adicionar um novo layout, clique no botão **+** na barra de abas na parte inferior.
- Para renomear ou excluir um layout, clique com o botão direito em sua aba.
