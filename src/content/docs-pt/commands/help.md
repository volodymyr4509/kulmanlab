# Help

Abre o painel de ajuda in-app com um resumo de todos os atalhos de teclado, dicas de navegação e uso geral.

## Como usar

Clique no botão **Help** na barra de ferramentas ou digite `help` no terminal. O painel de ajuda abre imediatamente e o comando termina.

## O que há no painel de ajuda

### Navegação
| Atalho | Ação |
|--------|------|
| `Roda de scroll` | Zoom in / out |
| `Clique central + arrastar` | Pan da vista |
| `Clique duplo central` | Ajustar todas as entidades à tela |

### Comandos
| Atalho | Ação |
|--------|------|
| Digite | Comece a digitar o nome de um comando — sugestões aparecem no canvas |
| `Delete` | Excluir todas as entidades selecionadas |
| `Enter` / `Espaço` | Confirmar comando ou entrada |
| `Enter` / `Espaço` (inativo) | Repetir o último comando usado |
| `Escape` | Cancelar o comando atual ou limpar seleção |

### Seleção
| Atalho | Ação |
|--------|------|
| `Clique esquerdo` | Selecionar uma entidade |
| `Arrastar →` | Selecionar apenas entidades **completamente dentro** da área de arraste |
| `Arrastar ←` | Selecionar qualquer entidade que **toque** a área de arraste |
| `Clique esquerdo grip` | Editar um vértice ou ponto final diretamente |

### Dicas de desenho
- **Pontos de snap** — o cursor encaixa automaticamente nos pontos finais de entidades próximas.
- **Ângulo fixo** — mova o cursor longe o suficiente numa direção para travar nos incrementos de 45°.
- **Comprimento exato** — enquanto o ângulo está travado, digite um número e pressione `Enter` para definir uma distância exata.

## Versão

A versão atual do aplicativo é mostrada no canto superior esquerdo do painel de ajuda (ex.: `KULMAN 2026.04.15`).
