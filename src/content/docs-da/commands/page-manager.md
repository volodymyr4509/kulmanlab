# PageManager

Kommandoen `PageManager` åbner sideindstillingsdialogen for det aktive layout, så du kan ændre papirformatet, orienteringen og skalaen mellem tegneenheder og millimeter.

## Åbne Page Manager

- Skriv `PageManager` i terminalen, **eller**
- Højreklik en layout-fane forneden og vælg **Page Manager**.

## Indstillinger

| Indstilling | Beskrivelse |
|---------|-------------|
| **Paper format** | Standard arkstørrelse (A4, A3, A2, Letter osv.) |
| **Orientation** | Stående eller liggende |
| **Scale** | Tegneenheder pr. mm — styrer hvordan modelkoordinater kortlægges til papiret |

At ændre papirstørrelsen eller skalaen skalerer alle eksisterende viewports i layoutet proportionalt, så deres positioner forbliver korrekte.

## Bemærkninger

- PageManager gælder kun papirlayouts. Den er ikke tilgængelig i modelrummet.
- For at tilføje et nyt layout, klik **+**-knappen i fanelinjen forneden.
- For at omdøbe eller slette et layout, højreklik dets fane.
