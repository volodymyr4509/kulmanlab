# Text Editor

টেক্সট এডিটর খোলে যখনই আপনি একটি সম্পাদনযোগ্য এন্টিটি স্থাপন বা ডাবল-ক্লিক করেন। হেডারে একটি ছোট **মোড চিপ** — **rich** (অ্যাকসেন্ট রং) বা **simple** (মিউটেড) — দেখায় বর্তমান এন্টিটির জন্য কোন মোড সক্রিয়।

## এডিটর মোড

### Rich মোড

ব্যবহার করে: **Text** (MTEXT লেবেল) এবং **Multileader** টীকা।

| বৈশিষ্ট্য | আচরণ |
|---------|-----------|
| বোল্ড / ইটালিক / স্ট্রাইকথ্রু | প্রতি-অক্ষর (নির্বাচনে প্রযোজ্য, বা নির্বাচন না থাকলে সম্পূর্ণ এন্টিটিতে) |
| ফন্ট এবং উচ্চতা | প্রতি-অক্ষর ওভাররাইড, বা সমগ্র-এন্টিটি ডিফল্ট |
| `Enter` | একটি হার্ড লাইন বিরতি ঢোকায় |
| `Shift+←/→` | টেক্সট নির্বাচন প্রসারিত বা সংকুচিত করে |
| `Home` / `End` | বর্তমান হার্ড লাইনের শুরু/শেষে লাফ দিন |
| ওয়ার্ড র‌্যাপ | রেফারেন্স-প্রস্থ রিসাইজ গ্রিপের মাধ্যমে সমর্থিত |

### Simple মোড

ব্যবহার করে: **Dimension Linear**, **Dimension Aligned**, **Dimension Angular**, **Dimension Radius**, **Dimension Diameter**।

এডিটরটি মাত্রার বর্তমান রেন্ডার করা লেবেল দিয়ে পূর্ব-পূরণ করা থাকে যাতে আপনি কার্সার অবস্থান করে মান সরাসরি সম্পাদনা করতে পারেন।

| বৈশিষ্ট্য | আচরণ |
|---------|-----------|
| বোল্ড / ইটালিক / ফন্ট / উচ্চতা | উপলব্ধ — **সম্পূর্ণ** লেবেলে একসাথে প্রযোজ্য |
| প্রতি-অক্ষর ফরম্যাটিং | সমর্থিত নয় |
| `Enter` | মান **কমিট** করে এবং এডিটর বন্ধ করে (কোনো লাইন বিরতি নয়) |
| বহু-লাইন | সমর্থিত নয় |
| ওয়ার্ড র‌্যাপ | সমর্থিত নয় |

## এডিটর খোলা

| ক্রিয়া | ফলাফল |
|--------|--------|
| `text` কমান্ড → পজিশন ক্লিক | একটি নতুন টেক্সট এন্টিটি তৈরি করে এবং এডিটর খোলে (**rich**) |
| বিদ্যমান **Text** এন্টিটিতে ডাবল-ক্লিক | **rich** মোডে এডিটর পুনরায় খোলে |
| বিদ্যমান **Multileader**-এ ডাবল-ক্লিক | **rich** মোডে এডিটর খোলে |
| **মাত্রা** এন্টিটিতে ডাবল-ক্লিক | **simple** মোডে এডিটর খোলে |
| এডিটরের ভেতরে `Escape` | এডিটর বন্ধ করে এবং সমস্ত পরিবর্তন রাখে |

## টুলবার

টুলবারটি টেক্সটের বাউন্ডিং বাক্সের উপরে ভাসমান এবং আপনি প্যান বা জুম করার সময় এন্টিটির সাথে অ্যাঙ্কর থাকে।

### Bold · Italic · Strikethrough

| বাটন | শর্টকাট | কী করে |
|--------|----------|--------------|
| **B** | — | বোল্ড টগল করুন |
| *I* | — | ইটালিক টগল করুন |
| ~~S~~ | — | স্ট্রাইকথ্রু টগল করুন |

**টগলিং কীভাবে প্রযোজ্য হয়:**

- **টেক্সট নির্বাচন সহ** — স্টাইল ঠিক নির্বাচিত অক্ষরগুলিতে প্রয়োগ হয়।
- **নির্বাচন নেই, বিদ্যমান টেক্সটে কার্সর** — সম্পূর্ণ এন্টিটিতে (সমস্ত সেগমেন্ট) স্টাইল টগল করে।
- **খালি টেক্সট বা নতুন এন্টিটি** — স্টাইল খালি সেগমেন্টে সংরক্ষিত হয় এবং সেই পয়েন্ট থেকে টাইপ করা প্রতিটি অক্ষরে প্রয়োগ হয়।

বাটনটি হাইলাইট (সক্রিয়) দেখায় যখন বর্তমান নির্বাচনের প্রতিটি অক্ষর — বা কার্সরের বাম দিকের অক্ষর — সেই স্টাইল সেট করা থাকে।

### ফন্ট

ড্রপডাউন ক্রস-প্ল্যাটফর্ম টাইপফেসের একটি কিউরেটেড সেট তালিকাভুক্ত করে (Helvetica, Times New Roman, Georgia, Courier New, Verdana, Tahoma, Trebuchet MS, Lucida Console, Impact)। শীর্ষ অপশন **Arial** হলো ডিফল্ট।

- **নির্বাচন সহ** — শুধুমাত্র নির্বাচিত অক্ষরের জন্য ফন্ট ওভাররাইড করে।
- **নির্বাচন নেই** — সম্পূর্ণ এন্টিটিতে ফন্ট প্রয়োগ করে।

নির্বাচন না থাকলে ড্রপডাউন কার্সরের বাম দিকের অক্ষরের ফন্ট প্রতিফলিত করে।

### উচ্চতা

নম্বর ফিল্ড ড্রয়িং ইউনিটে **ক্যাপ উচ্চতা** (একটি বড় হাতের অক্ষরের উচ্চতা) সেট করে।

- **নির্বাচন সহ** — নির্বাচিত অক্ষরের জন্য উচ্চতা ওভাররাইড করে, এন্টিটির বেস উচ্চতা থেকে স্বাধীনভাবে।
- **নির্বাচন নেই** — এন্টিটির বেস উচ্চতা পরিবর্তন করে (পৃথক উচ্চতা ওভাররাইড নেই এমন সমস্ত অক্ষরে প্রযোজ্য)।

ফিল্ড কার্সরের বাম দিকের অক্ষরের উচ্চতা প্রতিফলিত করে। এন্টিটির ডিফল্ট ব্যবহার করতে এটি খালি রাখুন।

## কার্সর এবং নেভিগেশন

| কী | ক্রিয়া |
|-----|--------|
| `←` / `→` | ক্যারেট এক অক্ষর বাম বা ডানে সরান |
| `Home` | বর্তমান হার্ড লাইনের শুরুতে লাফ দিন |
| `End` | বর্তমান হার্ড লাইনের শেষে লাফ দিন |
| `Shift` + `←` / `→` | নির্বাচন প্রসারিত বা সংকুচিত করুন |
| `Backspace` | বাম দিকের অক্ষর (বা নির্বাচন) মুছুন |
| `Delete` | ডান দিকের অক্ষর (বা নির্বাচন) মুছুন |
| `Enter` | লাইন বিরতি ঢোকান |
| `Escape` | এডিটর বন্ধ করুন |

কার্সর উচ্চতা স্বয়ংক্রিয়ভাবে সংলগ্ন অক্ষরের ক্যাপ উচ্চতার সাথে মিলে, সাবস্ক্রিপ্ট এবং সুপারস্ক্রিপ্টের জন্য ব্যবহৃত ছোট আকার সহ।

## কপি, কাট, এবং পেস্ট

| কী | ক্রিয়া |
|-----|--------|
| `Ctrl+C` / `Cmd+C` | নির্বাচিত টেক্সট কপি করুন |
| `Ctrl+X` / `Cmd+X` | নির্বাচিত টেক্সট কাট করুন |
| `Ctrl+V` / `Cmd+V` | কার্সরে পেস্ট করুন |

কপি এবং কাটের জন্য সক্রিয় টেক্সট নির্বাচন প্রয়োজন। পেস্ট করা টেক্সট সবসময় সাধারণ থাকে — এটি কপি করার সময় থাকা ফরম্যাটিং বজায় রাখার পরিবর্তে কার্সরে ইতিমধ্যে থাকা ফরম্যাটিং (বোল্ড, ইটালিক, ফন্ট, উচ্চতা) গ্রহণ করে।

**Rich মোডে**, পেস্ট করা টেক্সটে লাইন বিরতি বজায় থাকে। **Simple মোডে**, লাইন বিরতি সরিয়ে ফেলা হয়, কারণ মাত্রা লেবেল এক-লাইনের হয়।

## ওয়ার্ড র‌্যাপ

একটি টেক্সট এন্টিটিতে **রেফারেন্স প্রস্থ** সেট থাকলে, দীর্ঘ লাইনগুলি সেই প্রস্থের মধ্যে ফিট করতে শব্দ সীমানায় সফট-র‌্যাপ হয়।

এন্টিটি নির্বাচিত থাকাকালীন রেফারেন্স প্রস্থ সেট বা পরিবর্তন করতে **রিসাইজ গ্রিপ** টেনে আনুন — ড্যাশড বাউন্ডিং বাক্সের বাম ও ডান প্রান্তে পাতলা আয়তক্ষেত্র। আপনি টানার সময় বিষয়বস্তু রিয়েল টাইমে রিফ্লো হয়।

রেফারেন্স প্রস্থ শূন্যে সেট করলে (গ্রিপ একসাথে টানুন বা প্রপার্টি প্যানেলে মান মুছুন) ওয়ার্ড র‌্যাপ সরিয়ে লাইনগুলি মুক্তভাবে বাড়তে দেয়।

## বহু-লাইন টেক্সট

একটি হার্ড লাইন বিরতি ঢোকাতে `Enter` চাপুন। প্রতিটি হার্ড লাইন স্বতন্ত্র — `Home` এবং `End` শুধুমাত্র বর্তমান হার্ড লাইনের মধ্যে নেভিগেট করে।

হার্ড লাইন বিরতি এবং প্রতি-অক্ষর ফরম্যাটিং MTEXT ফরম্যাট ব্যবহার করে সংরক্ষিত এবং একটি সম্পূর্ণ DXF রাউন্ড-ট্রিপ টিকে থাকে।

## DXF সামঞ্জস্যতা

টেক্সট এন্টিটিগুলি DXF ফাইলে **MTEXT** হিসেবে সংরক্ষিত। বোল্ড এবং ইটালিক `\L`, `\K`, `\O` এবং ইনলাইন ফন্ট সুইচ (`\f`) হিসেবে এনকোড করা হয়। প্রতি-অক্ষর উচ্চতা `\H` হিসেবে এনকোড করা হয়। সমস্ত ফরম্যাটিং এক্সপোর্টে সংরক্ষিত এবং LibreCAD, FreeCAD এবং অন্য DXF-সামঞ্জস্যপূর্ণ অ্যাপ্লিকেশন দ্বারা পঠনযোগ্য।
