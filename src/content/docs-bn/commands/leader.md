# Leader

`leader` কমান্ড চারটি ধাপে একটি মাল্টিলিডার অ্যানোটেশন আঁকে: একটি ফিচার স্পর্শ করা তীরমুখ, একটি ডগলেগে বাঁকানো লিডার লাইন, একটি টেক্সট অ্যাঙ্কর এবং একটি টাইপ করা লেবেল। সমস্ত অ্যানোটেশন কমান্ডের মধ্যে, Leader একমাত্র যা ব্লিঙ্কিং কার্সর প্রিভিউ সহ ইন্টারেক্টিভ টেক্সট-ইনপুট পর্যায় অন্তর্ভুক্ত করে।

## মাল্টিলিডারের গঠন

```
  ◄── তীরমুখ টিপ  (ধাপ ২ — ফিচার স্পর্শ করে)
      \
       \  লিডার লাইন
        \
         ●──── ডগলেগ (ধাপ ৩) ──── টেক্সট অ্যাঙ্কর (ধাপ ৪)
                                    লেবেল টেক্সট  (ধাপ ৫)
```

- **তীরমুখ টিপ** — অ্যানোটেট করা ফিচারে স্থাপিত নির্দেশ করা প্রান্ত।
- **ডগলেগ** — লিডার লাইন যেখানে টেক্সটের দিকে বাঁকে।
- **টেক্সট অ্যাঙ্কর** — লেবেলটি যেখানে স্থাপিত হয়। টেক্সট স্বয়ংক্রিয়ভাবে বাম বা ডানে সারিবদ্ধ হয়।

## একটি লিডার আঁকা

1. টার্মিনালে `leader` টাইপ করুন বা **Leader** টুলবার বাটনে ক্লিক করুন।
2. **তীরমুখ টিপে ক্লিক করুন**, বা সঠিক কোঅর্ডিনেটের জন্য `X,Y` টাইপ করুন এবং **Enter** চাপুন।
3. **ডগলেগে ক্লিক করুন** — লিডারে বাঁক। কোণ ৪৫° ইনক্রিমেন্টে লক হয়; সঠিক স্থাপনার জন্য একটি দৈর্ঘ্য টাইপ করুন এবং **Enter** চাপুন। অথবা পরম কোঅর্ডিনেট এন্ট্র করতে `X,Y` টাইপ করুন।
4. **টেক্সট পজিশনে ক্লিক করুন** — লেবেলটি যেখানে অ্যাঙ্কর করে। একই বিকল্পগুলি প্রযোজ্য: ক্লিক, কোণ-লক + দৈর্ঘ্য, বা `X,Y`।
5. **লেবেল টেক্সট টাইপ করুন** — ক্যানভাস প্রিভিউ ব্লিঙ্কিং কার্সর সহ লাইভ আপডেট হয়। স্থাপন করতে **Enter** চাপুন।

## কোঅর্ডিনেট এন্ট্রি (সমস্ত পয়েন্ট পর্যায়)

যেকোনো পয়েন্ট-বাছাই ধাপে (টিপ, ডগলেগ, টেক্সট পজিশন) ক্লিক করার বদলে একটি সঠিক কোঅর্ডিনেট টাইপ করতে পারেন:

1. X মান টাইপ করুন (সংখ্যা, `.`, বা `-`)।
2. `,` চাপুন — টার্মিনাল `[X], [Y{কার্সর}]` দেখায় X লক হয়েছে বলে নিশ্চিত করে।
3. Y মান টাইপ করুন।
4. পয়েন্ট স্থাপন করতে **Enter** চাপুন।

## কোণ লকিং (ধাপ ৩ এবং ৪)

প্রতিটি স্থাপিত পয়েন্টের পরে, কার্সর যথেষ্ট দূরে থাকলে কমান্ড ৪৫° অক্ষে স্ন্যাপ করে। লক থাকাকালীন:
- প্রিভিউ অক্ষে স্ন্যাপ করে।
- পরবর্তী পয়েন্টটি সেই সঠিক দূরত্বে স্থাপন করতে একটি দৈর্ঘ্য টাইপ করুন এবং **Enter** চাপুন।

কোণ লকিং এবং কোঅর্ডিনেট এন্ট্রি পারস্পরিক একচেটিয়া — একবার আপনি পূর্ববর্তী `,` ছাড়া একটি সংখ্যা টাইপ করলে, কমান্ড এটিকে দূরত্ব হিসেবে ব্যাখ্যা করে (কোণ লক সক্রিয় থাকতে হবে)। পরিবর্তে পরম কোঅর্ডিনেট এন্ট্র করতে, X নম্বর দিয়ে শুরু করুন তারপর কমা।

## টেক্সট লেবেল সম্পাদনা

ধাপ ৫-এ লেবেল টাইপ করার সময়, স্থাপনের আগে টেক্সট নেভিগেট এবং সম্পাদনা করতে পারেন:

| কী | ক্রিয়া |
|-----|--------|
| যেকোনো প্রিন্টযোগ্য অক্ষর | কার্সর পজিশনে ঢোকান |
| `←` / `→` | কার্সর বাম বা ডানে সরান |
| `Backspace` | কার্সরের বাম দিকের অক্ষর মুছুন |
| `Delete` | কার্সরের ডান দিকের অক্ষর মুছুন |
| `Enter` | লিডার স্থাপন করুন |

## স্বয়ংক্রিয় টেক্সট দিক

ডগলেগের সাপেক্ষে কার্সর পজিশনের উপর ভিত্তি করে টেক্সট সারিবদ্ধতা সামঞ্জস্য হয়:

| কার্সর পজিশন | টেক্সট দিক |
|-----------------|---------------|
| ডগলেগের **ডানে** | টেক্সট অ্যাঙ্কর থেকে বাম-থেকে-ডানে |
| ডগলেগের **বামে** | ডান-থেকে-বামে (ডান দিকে অ্যাঙ্কর করা) |

কোনো ম্যানুয়াল সামঞ্জস্যের দরকার নেই — লেবেলটি যে দিকে চান সেই দিকে কার্সর সরান এবং এটি সঠিকভাবে সারিবদ্ধ হবে।

## কীবোর্ড রেফারেন্স

**পয়েন্ট পর্যায় (টিপ, ডগলেগ, টেক্সট পজিশন)**

| কী | ক্রিয়া |
|-----|--------|
| `0`–`9`, `.`, `-` | X কোঅর্ডিনেট টাইপ শুরু করুন (তারপর X লক করতে এবং Y এন্ট্র করতে `,`) |
| `,` | X নিশ্চিত করুন এবং Y এন্ট্রিতে যান |
| `0`–`9`, `.`, `-` | কোণ-লক থাকলে দূরত্ব গড়ে তুলুন |
| `Backspace` | শেষ টাইপ করা অক্ষর মুছুন |
| `Enter` | টাইপ করা কোঅর্ডিনেট বা দূরত্ব নিশ্চিত করুন |

**টেক্সট ইনপুট পর্যায়**

| কী | ক্রিয়া |
|-----|--------|
| প্রিন্টযোগ্য অক্ষর | কার্সরে ঢোকান |
| `←` / `→` | কার্সর সরান |
| `Backspace` | বাম দিক মুছুন |
| `Delete` | ডান দিক মুছুন |
| `Enter` | লিডার স্থাপন করুন |

| কী | ক্রিয়া |
|-----|--------|
| `Escape` | বাতিল করুন এবং ধাপ ২-এ রিসেট করুন |

## বিদ্যমান লিডার সম্পাদনা করা

স্থাপিত multileader-এ **ডাবল-ক্লিক** করুন **rich** মোডে টেক্সট এডিটর পুনরায় খুলতে। rich মোডে আপনি বোল্ড, ইটালিক এবং প্রতি-অক্ষর ফন্ট বা উচ্চতা ওভাররাইড প্রয়োগ করতে পারেন এবং `Enter` দিয়ে লাইন বিরতি ঢোকাতে পারেন। কমিট করতে এবং বন্ধ করতে **Escape** চাপুন।

সম্পূর্ণ রেফারেন্সের জন্য [Text Editor — rich mode](../../interface/text-editor/#rich-mode) দেখুন।

- দুটি বা তার বেশি আর্ম থাকা লিডার থেকে আর্ম সরাতে: [Leader−](../leader-remove/)

## DXF — শুধু ইম্পোর্ট

**লিডার শুধুমাত্র ইম্পোর্ট-যোগ্য।** DXF ফাইল থেকে `MLEADER` এন্টিটি সঠিকভাবে পড়া এবং প্রদর্শিত হয়, কিন্তু এডিটরে আঁকা লিডারগুলি DXF ফাইল সেভ করার সময় **লেখা হয় না**। ভিজ্যুয়াল অ্যানোটেশনের জন্য লিডার ব্যবহার করুন; রাউন্ড-ট্রিপ ওয়ার্কফ্লোতে এগুলির উপর নির্ভর করা থেকে বিরত থাকুন।
