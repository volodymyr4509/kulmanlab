# Circle

`circle` কমান্ড একটি কেন্দ্র পয়েন্ট এবং একটি ব্যাসার্ধ দ্বারা সংজ্ঞায়িত একটি বৃত্ত আঁকে। কেন্দ্র ক্লিক করার পরে আপনি ক্যানভাসে দ্বিতীয় পয়েন্ট ক্লিক করে বা একটি সঠিক সংখ্যা টাইপ করে ব্যাসার্ধ সেট করতে পারেন — উভয় বিকল্প একই সময়ে লাইভ।

## একটি বৃত্ত আঁকা

1. টার্মিনালে `circle` টাইপ করুন বা **Circle** টুলবার বোতামে ক্লিক করুন।
2. **কেন্দ্র পয়েন্টে ক্লিক করুন**, অথবা সঠিক স্থানাঙ্কের জন্য `X,Y` টাইপ করুন এবং **Enter** চাপুন।
3. ব্যাসার্ধ সেট করুন — হয়:
   - ক্যানভাসে **যেকোনো পয়েন্টে ক্লিক করুন** — কেন্দ্র থেকে দূরত্ব ব্যাসার্ধ হয়, বা
   - **ব্যাসার্ধ টাইপ করুন** এবং সঠিক মানের জন্য **Enter** চাপুন।

বৃত্তটি তাৎক্ষণিকভাবে স্থাপিত হয় এবং কমান্ড প্রস্থান করে।

```
  center ●
          \  ব্যাসার্ধ লাইন পূর্বদর্শন
           \
            ● ← এখানে ক্লিক করুন, বা একটি সংখ্যা টাইপ করুন
```

ব্যাসার্ধ পর্যায়ে, লাইভ পূর্বদর্শন বর্তমান কার্সার দূরত্বে বৃত্ত দেখায় এবং কেন্দ্র থেকে বর্তমান পয়েন্ট পর্যন্ত একটি ব্যাসার্ধ রেখাও আঁকে।

## কেন্দ্র স্থানাঙ্ক এন্ট্রি

ক্লিক করার পরিবর্তে, আপনি কেন্দ্র অবস্থান টাইপ করতে পারেন:

1. X মান টাইপ করুন।
2. `,` চাপুন — টার্মিনাল দেখায় `[X], [Y{cursor}]`।
3. Y মান টাইপ করুন।
4. কেন্দ্র স্থাপন করতে এবং ব্যাসার্ধ ইনপুটে এগিয়ে যেতে **Enter** চাপুন।

## টাইপ করা ব্যাসার্ধ ইনপুট

কেন্দ্র স্থাপনের পরে, অবিলম্বে টাইপ করা একটি ব্যাসার্ধ মান তৈরি করে:

| কী | ক্রিয়া |
|-----|--------|
| `0`–`9`, `.` | ব্যাসার্ধ মানে অঙ্ক যোগ করুন |
| `Backspace` | শেষ টাইপ করা অক্ষর মুছুন |
| `Enter` | টাইপ করা ব্যাসার্ধে বৃত্ত স্থাপন করুন |

সংগৃহীত মান টার্মিনাল প্রম্পটে দেখানো হয় (যেমন `enter radius of circle: 25`)। পূর্বদর্শন কার্সার ব্যাসার্ধ মার্কারের দিক নিয়ন্ত্রণ করার সময় টাইপ করা ব্যাসার্ধ দেখাতে আপডেট হয়।

## কীবোর্ড রেফারেন্স

| কী | ক্রিয়া |
|-----|--------|
| `0`–`9`, `.`, `-` | X স্থানাঙ্ক এন্ট্রি শুরু করুন (কেন্দ্র পর্যায়), বা ব্যাসার্ধ অঙ্ক (ব্যাসার্ধ পর্যায়) |
| `,` | X লক করুন এবং Y এন্ট্রিতে যান (কেন্দ্র পর্যায়) |
| `Backspace` | শেষ টাইপ করা অক্ষর মুছুন |
| `Enter` | টাইপ করা স্থানাঙ্ক বা ব্যাসার্ধ নিশ্চিত করুন |
| `Escape` | বাতিল করুন এবং রিসেট করুন |

## Grip সম্পাদনা — ব্যাসার্ধ পরিবর্তন করা

একটি নির্বাচিত বৃত্ত পাঁচটি গ্রিপ প্রকাশ করে:

| Grip | অবস্থান | এটি কী করে |
|------|----------|--------------|
| **Center** | কেন্দ্র পয়েন্ট | পুরো বৃত্ত সরায়; ব্যাসার্ধ অপরিবর্তিত থাকে |
| **Left** | সর্বাধিক বাম পয়েন্ট (center − radius) | নতুন ব্যাসার্ধ = কেন্দ্রে দূরত্ব সেট করতে টানুন |
| **Right** | সর্বাধিক ডান পয়েন্ট (center + radius) | নতুন ব্যাসার্ধ = কেন্দ্রে দূরত্ব সেট করতে টানুন |
| **Top** | সর্বোচ্চ পয়েন্ট | নতুন ব্যাসার্ধ = কেন্দ্রে দূরত্ব সেট করতে টানুন |
| **Bottom** | সর্বনিম্ন পয়েন্ট | নতুন ব্যাসার্ধ = কেন্দ্রে দূরত্ব সেট করতে টানুন |

চারটি কার্ডিনাল গ্রিপ একইভাবে আচরণ করে — নতুন ব্যাসার্ধ কেন্দ্র থেকে টেনে আনার অবস্থান পর্যন্ত দূরত্বের সমান। কেন্দ্র স্থির থাকে।

## বৃত্ত নির্বাচন করা

| পদ্ধতি | আচরণ |
|--------|-----------|
| **ক্লিক** | ক্লিকটি পরিধির কাছে পড়লে নির্বাচন করে |
| **ডানে টানুন** (কঠোর) | সম্পূর্ণ বাউন্ডিং স্কোয়ার (center ± radius) বাক্সের ভেতরে থাকতে হবে |
| **বামে টানুন** (ক্রসিং) | পরিধির যেকোনো অংশ যা বাক্সের সীমানা অতিক্রম বা স্পর্শ করে তা বৃত্ত নির্বাচন করে |

## DXF — CIRCLE সত্তা

বৃত্তগুলি DXF ফাইলে `CIRCLE` সত্তা হিসেবে সংরক্ষিত হয়। কেন্দ্র স্থানাঙ্ক, ব্যাসার্ধ, রঙ, লেয়ার, লাইনটাইপ, লাইনটাইপ স্কেল এবং বেধ সব কোনো ক্ষতি ছাড়াই রাউন্ড-ট্রিপ করে। যেকোনো DXF-সামঞ্জস্যপূর্ণ অ্যাপ্লিকেশন এগুলিকে স্ট্যান্ডার্ড বৃত্ত হিসেবে পড়ে।
