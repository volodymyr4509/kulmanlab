# Import

**Import** কমান্ড আপনার স্থানীয় ফাইল সিস্টেম থেকে KulmanLab CAD-এ একটি বিদ্যমান ড্রয়িং লোড করে। মানক **DXF** ফরম্যাট এবং KulmanLab-এর নিজস্ব **JSON** ফরম্যাট উভয়ই সমর্থিত।

## একটি ফাইল আমদানির পদ্ধতি

1. স্ক্রিনের শীর্ষে File প্যানেলে **Import** টুলবার বাটনে (ফোল্ডার আইকন) ক্লিক করুন।
2. আপনার ব্রাউজারের ফাইল পিকার খুলবে। আপনার ড্রয়িং ফাইলে নেভিগেট করুন এবং সেটি নির্বাচন করুন।
3. ড্রয়িং সঙ্গে সঙ্গে ক্যানভাসে লোড হয়। ভিউপোর্ট স্বয়ংক্রিয়ভাবে সমস্ত এন্টিটি ফিট করে।

বিকল্পভাবে, আপনি সরাসরি ক্যানভাসে একটি ফাইল ড্র্যাগ অ্যান্ড ড্রপ করতে পারেন।

## সমর্থিত ফাইল ফরম্যাট

| ফরম্যাট | এক্সটেনশন | কখন ব্যবহার করবেন |
|--------|-----------|-------------|
| **DXF** | `.dxf` | AutoCAD, FreeCAD, LibreCAD বা অন্যান্য CAD টুল থেকে ড্রয়িং |
| **JSON** *(নেটিভ)* | `.json` | KulmanLab CAD থেকে পূর্বে সংরক্ষিত ড্রয়িং — পূর্ণ বিশ্বস্ততা |

## DXF থেকে কী আমদানি হয়

KulmanLab নিম্নলিখিত DXF এন্টিটি টাইপগুলি পার্স করে:

| এন্টিটি টাইপ | DXF কোড | নোট |
|-------------|----------|-------|
| লাইন | `LINE` | |
| বৃত্ত | `CIRCLE` | |
| চাপ | `ARC` | |
| উপবৃত্ত | `ELLIPSE` | |
| পলিলাইন | `LWPOLYLINE` | |
| স্প্লাইন | `SPLINE` | |
| টেক্সট | `TEXT`, `MTEXT` | |
| ডাইমেনশন | `DIMENSION` | |
| মাল্টিলিডার | `MULTILEADER` | |

লেয়ার সংজ্ঞা এবং লাইনটাইপ টেবিলও DXF ফাইল থেকে আমদানি করা হয় যখন উপস্থিত থাকে।

অসমর্থিত DXF টাইপ ব্যবহারকারী এন্টিটিগুলি নীরবে এড়িয়ে যায় — বাকি ড্রয়িং এখনও লোড হয়।

## ফাইলের নামকরণ এবং স্টোরেজ

আপনি যখন একটি ফাইল আমদানি করেন, ফাইলের নামে একটি টাইমস্ট্যাম্প যোগ করা হয় (উদাহরণস্বরূপ `myplan_May22_14:30:00.dxf`)। এটি আপনাকে নাম সংঘর্ষ ছাড়াই [Recent Files](../files/)-এ একই ড্রয়িংয়ের একাধিক সংস্করণ রাখতে দেয়। যদি ফাইলের নামে ইতিমধ্যে একটি টাইমস্ট্যাম্প থাকে, তবে এটি যেমন আছে তেমনই ব্যবহৃত হয়।

আমদানির পরে ড্রয়িং স্বয়ংক্রিয়ভাবে ব্রাউজার স্টোরেজে (IndexedDB) সংরক্ষিত হয়, তাই এটি [Files](../files/) প্যানেলে প্রদর্শিত হয় এবং পেজ রিলোডে টিকে থাকে।

## বর্তমান ড্রয়িংয়ের কী হয়

আমদানি করলে বর্তমান ক্যানভাস প্রতিস্থাপিত হয়। কোনো মার্জ বা অ্যাপেন্ড নেই। যদি আপনার অসংরক্ষিত পরিবর্তন থাকে, তবে আগে বর্তমান ড্রয়িং [export](../export/) করুন।

## স্টার্টআপে

KulmanLab পেজ লোড হলে স্বয়ংক্রিয়ভাবে সর্বাধিক সম্প্রতি সম্পাদিত ফাইলটি পুনরায় খোলে। যদি কোনো সংরক্ষিত ফাইল না থাকে, একটি ডিফল্ট নমুনা ড্রয়িং লোড করা হয়।

## সমস্যা সমাধান

| সমস্যা | সম্ভাব্য কারণ | সমাধান |
|---------|-------------|-----|
| আমদানির পরে ক্যানভাস খালি | DXF এন্টিটিগুলি অসমর্থিত টাইপ ব্যবহার করে (যেমন HATCH, INSERT) | এন্টিটিগুলি এড়িয়ে গেছে — টার্মিনালে "no entities found" বার্তা খুঁজুন |
| Import বাটন কাজ করে না | ব্রাউজার ফাইল পিকার ব্লক করেছে | আবার বাটনে ক্লিক করুন; কিছু ব্রাউজারে নতুন ব্যবহারকারী অঙ্গভঙ্গি প্রয়োজন |
| ডাইমেনশন ভুল দেখাচ্ছে | অ-মানক ডাইমেনশন জ্যামিতি লেখা একটি টুল থেকে DXF | বর্তমান DXF সংস্করণ ব্যবহার করে উৎস অ্যাপ থেকে পুনরায় এক্সপোর্ট করুন |
