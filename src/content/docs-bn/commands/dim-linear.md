# Dimension Linear

`dimlinear` কমান্ড দুটি এক্সটেনশন-লাইন মূলের মধ্যে একটি অনুভূমিক বা উল্লম্ব মাত্রা স্থাপন করে। মাত্রা রেখা সর্বদা নিখুঁতভাবে অনুভূমিক বা উল্লম্ব চলে — এটি যেকোনো কোণে স্থাপন করা যায় না। একটি তির্যক রেখার সমান্তরালে মাত্রার প্রয়োজন হলে [Dimension Aligned](./dim-aligned) ব্যবহার করুন।

## একটি রৈখিক মাত্রার গঠন

```
  |←————— 5.00 —————→|
  |                   |
  ●  (ext লাইন 1)     ●  (ext লাইন 2)
  p1                  p2
```

- **এক্সটেনশন লাইন** — মাত্রা রেখায় লম্ব প্রতিটি পরিমাপ করা পয়েন্ট থেকে নামে।
- **মাত্রা রেখা** — অনুভূমিক (X দূরত্ব পরিমাপ করে) বা উল্লম্ব (Y দূরত্ব পরিমাপ করে)।
- **মান** — বেছে নেওয়া অক্ষ বরাবর প্রক্ষেপিত দূরত্ব, সত্যিকারের পয়েন্ট-থেকে-পয়েন্ট দূরত্ব নয়।

## একটি রৈখিক মাত্রা স্থাপন করা

1. টার্মিনালে `dimlinear` টাইপ করুন বা **Dimension Linear** টুলবার বোতামে ক্লিক করুন।
2. **প্রথম এক্সটেনশন লাইন মূলে ক্লিক করুন** (p1), অথবা সঠিক স্থানাঙ্কের জন্য `X,Y` টাইপ করুন এবং **Enter** চাপুন।
3. **দ্বিতীয় এক্সটেনশন লাইন মূলে ক্লিক করুন** (p2)। স্থানাঙ্ক এন্ট্রিও এখানে কাজ করে।
4. মাত্রা রেখা অবস্থান করতে **কার্সার সরান**। অভিমুখ কার্সার অবস্থান থেকে স্বয়ংক্রিয়ভাবে সনাক্ত করা হয়।
5. স্থাপন করতে **ক্লিক করুন**, বা সঠিক স্থাপনার জন্য একটি অফসেট দূরত্ব টাইপ করুন এবং **Enter** চাপুন।

## স্বয়ংক্রিয় অভিমুখ সনাক্তকরণ

কোনো অভিমুখ বাধ্য না করা হলে, কমান্ড দুটি পরিমাপ করা পয়েন্টের সাপেক্ষে কার্সার অবস্থান পড়ে:

| কার্সার অবস্থান | সনাক্ত করা অভিমুখ | কী পরিমাপ করা হয় |
|-----------------|---------------------|-----------------|
| পয়েন্টগুলির উপরে বা নীচে | অনুভূমিক | p1 এবং p2-এর মধ্যে Δ X |
| পয়েন্টগুলির বামে বা ডানে | উল্লম্ব | p1 এবং p2-এর মধ্যে Δ Y |

স্থাপন পর্যায়ে যেকোনো সময় অনুভূমিক লক করতে **H** বা উল্লম্ব লক করতে **V** চাপুন। একবার লক হলে, কার্সার সরানোর সাথে অভিমুখ পরিবর্তন হয় না।

## টাইপ করা অফসেট দূরত্ব

পরিমাপ করা পয়েন্ট থেকে সঠিক দূরত্বে মাত্রা রেখা স্থির করতে স্থাপনের সময় একটি সংখ্যা টাইপ করুন:

| কী | ক্রিয়া |
|-----|--------|
| `0`–`9`, `.` | অফসেট দূরত্বে অঙ্ক যোগ করুন |
| `Backspace` | শেষ টাইপ করা অক্ষর মুছুন |
| `Enter` / `Space` | টাইপ করা দূরত্বে স্থাপন করুন |

কার্সার পাশ (অনুভূমিকের জন্য উপর/নীচ, উল্লম্বের জন্য বাম/ডান) চিহ্ন নির্ধারণ করে — মাত্রা রেখা বর্তমানে কার্সার যে পাশে আছে সেই পাশে প্রদর্শিত হয়।

## কীবোর্ড রেফারেন্স

| কী | ক্রিয়া |
|-----|--------|
| `0`–`9`, `.`, `-` | X স্থানাঙ্ক এন্ট্রি শুরু করুন (p1/p2 পর্যায়), বা অফসেট দূরত্ব (স্থাপন পর্যায়) |
| `,` | X লক করুন এবং Y এন্ট্রিতে যান (p1/p2 পর্যায়) |
| `H` | অনুভূমিক অভিমুখ লক করুন (শুধুমাত্র স্থাপন পর্যায়) |
| `V` | উল্লম্ব অভিমুখ লক করুন (শুধুমাত্র স্থাপন পর্যায়) |
| `Backspace` | শেষ টাইপ করা অক্ষর মুছুন |
| `Enter` / `Space` | টাইপ করা স্থানাঙ্ক বা অফসেট নিশ্চিত করুন |
| `Escape` | বাতিল করুন |

## Dimension Linear বনাম Dimension Aligned

| | Dimension Linear | Dimension Aligned |
|---|-----------------|------------------|
| অক্ষ | সর্বদা H বা V | পরিমাপ করা রেখার সমান্তরাল |
| পরিমাপ করে | শুধুমাত্র X বা Y উপাদান | সত্যিকারের ইউক্লিডিয়ান দূরত্ব |
| H/V কী | হ্যাঁ — অভিমুখ লক করে | না — সর্বদা p1→p2 অনুসরণ করে |
| সবচেয়ে ভালো | অর্থোগোনাল লেআউট, ফ্লোর প্ল্যান | তির্যক বৈশিষ্ট্য, কোণীয় কাটা |

## লেবেল সম্পাদনা — সরল মোড

স্থাপিত রৈখিক মাত্রায় **ডাবল-ক্লিক** করুন **সরল** মোডে টেক্সট এডিটর খুলতে। এডিটরটি বর্তমান রেন্ডার করা মান দিয়ে পূর্ব-পূরণ করা থাকে যাতে আপনি কার্সার অবস্থান করে সরাসরি সম্পাদনা করতে পারেন।

| বৈশিষ্ট্য | আচরণ |
|---------|-----------|
| Bold / Italic / Font / Height | **সম্পূর্ণ** লেবেলে একসাথে প্রযোজ্য |
| প্রতি-অক্ষর ফরম্যাটিং | সমর্থিত নয় |
| `Enter` | মান কমিট করে এবং এডিটর বন্ধ করে |
| বহু-লাইন | সমর্থিত নয় |

সম্পূর্ণ রেফারেন্সের জন্য [Text Editor — simple mode](../interface/text-editor#simple-mode) দেখুন।

## মাত্রা শৃঙ্খলিত করা

শেষ এক্সটেনশন লাইন থেকে চলমান আরও মাত্রা যোগ করতে, এটি স্থাপনের পরে অবিলম্বে [Dimension Continue](./dim-continue) ব্যবহার করুন।

## DXF — DIMENSION সত্তা

রৈখিক মাত্রাগুলি `rotationDeg` সেট `0` (অনুভূমিক) বা `90` (উল্লম্ব) সহ `DIMENSION` সত্তা হিসেবে সংরক্ষিত। এক্সটেনশন লাইন মূল, মাত্রা লাইন অবস্থান, টেক্সট অবস্থান, পরিমাপ করা মান, তীরচিহ্নের শৈলী, টেক্সট উচ্চতা এবং সমস্ত প্রদর্শন ফ্ল্যাগ কোনো ক্ষতি ছাড়াই রাউন্ড-ট্রিপ করে।
