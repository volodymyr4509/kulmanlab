# Scale

`scale` কমান্ড নির্বাচিত এন্টিটিগুলিকে একটি বেস পয়েন্টের চারপাশে সমানভাবে রিসাইজ করে। বেস পয়েন্ট থেকে সমস্ত দূরত্ব স্কেল ফ্যাক্টর দ্বারা গুণ করা হয় — `2` ফ্যাক্টর সমস্ত মাত্রা দ্বিগুণ করে, `0.5` অর্ধেক করে। ফ্যাক্টর সর্বদা টাইপ করে এন্ট্র করা হয়; কোনো ক্লিক-টু-সেট-স্কেল নেই।

## শুরু করার দুটি উপায়

**আগে নির্বাচন করুন, তারপর স্কেল করুন** — প্রথমে এন্টিটি নির্বাচন করুন, তারপর সক্রিয় করুন:

1. ক্যানভাসে এক বা একাধিক এন্টিটি নির্বাচন করুন।
2. টার্মিনালে `scale` টাইপ করুন বা **Scale** টুলবার বাটনে ক্লিক করুন।
3. **বেস পয়েন্টে ক্লিক করুন** — স্কেলিংয়ের সময় যে স্থির পয়েন্টটি নড়বে না। বা সঠিক কোঅর্ডিনেটের জন্য `X,Y` টাইপ করুন এবং **Enter** চাপুন।
4. **স্কেল ফ্যাক্টর টাইপ করুন** এবং **Enter** চাপুন।

**সক্রিয় করুন, তারপর নির্বাচন করুন** — কিছু নির্বাচন ছাড়া কমান্ড শুরু করুন:

1. `scale` টাইপ করুন বা টুলবার বাটনে ক্লিক করুন।
2. **অবজেক্ট নির্বাচন করুন** — টগল করতে ক্লিক করুন, বা এলাকা দ্বারা নির্বাচন করতে টেনে আনুন।
3. নির্বাচন নিশ্চিত করতে **Enter** বা **Space** চাপুন।
4. **বেস পয়েন্টে ক্লিক করুন** (কোঅর্ডিনেট এন্ট্রি উপলব্ধ), তারপর ফ্যাক্টর টাইপ করুন।

```
  বেস ●                বেস ●
        [এন্টিটি]   →          [বড় এন্টিটি]
  ফ্যাক্টর = 2 → ● থেকে দূরত্ব দ্বিগুণ হয়
```

## স্কেল ফ্যাক্টর টাইপ করা

বেস পয়েন্ট স্থাপনের পরে টার্মিনাল `enter scale factor:` দেখায় এবং কীবোর্ড ইনপুটের জন্য অপেক্ষা করে:

| কী | ক্রিয়া |
|-----|--------|
| `0`–`9`, `.` | ফ্যাক্টরে সংখ্যা যোগ করুন |
| `-` | নেতিবাচক ফ্যাক্টর (শুধু প্রথম অক্ষর — বিপরীত করে তারপর স্কেল করে) |
| `Backspace` | শেষ টাইপ করা অক্ষর মুছুন |
| `Enter` | টাইপ করা ফ্যাক্টরে স্কেল প্রয়োগ করুন |

ফ্যাক্টর অবশ্যই শূন্য ছাড়া হতে হবে। সাধারণ মান:

| ফ্যাক্টর | প্রভাব |
|--------|--------|
| `2` | সমস্ত মাত্রা দ্বিগুণ |
| `0.5` | সমস্ত মাত্রা অর্ধেক |
| `1.5` | ৫০% বৃদ্ধি |
| `-1` | বেস পয়েন্ট জুড়ে মিরর (১৮০° ঘূর্ণনের সমান) |

টাইপ করার সময় কোনো লাইভ প্রিভিউ নেই — স্কেল করা ফলাফল শুধুমাত্র **Enter** চাপার পরে দেখা যায়।

## কীবোর্ড রেফারেন্স

| কী | ক্রিয়া |
|-----|--------|
| `Enter` / `Space` | নির্বাচন নিশ্চিত করুন |
| `0`–`9`, `.`, `-` | X কোঅর্ডিনেট এন্ট্রি শুরু করুন (বেস-পয়েন্ট পর্যায়), বা স্কেল ফ্যাক্টর (ফ্যাক্টর পর্যায়) |
| `,` | X লক করুন এবং Y এন্ট্রিতে যান (বেস-পয়েন্ট পর্যায়) |
| `Backspace` | শেষ টাইপ করা অক্ষর মুছুন |
| `Enter` | কোঅর্ডিনেট নিশ্চিত করুন বা স্কেল প্রয়োগ করুন |
| `Escape` | বাতিল করুন এবং রিসেট করুন |

## কমান্ড চলাকালীন নির্বাচন

| পদ্ধতি | আচরণ |
|--------|-----------|
| **ক্লিক** | কার্সরের নিচের এন্টিটি টগল করে |
| **ডানে টানুন** (কঠোর) | বাক্সের সম্পূর্ণ ভেতরে এন্টিটি যোগ করে |
| **বামে টানুন** (ক্রসিং) | বাক্সের সাথে ছেদকারী এন্টিটি যোগ করে |
| **Enter** / **Space** | নির্বাচন নিশ্চিত করে |

## কী স্কেল হয়

সমস্ত এন্টিটি টাইপ সমর্থিত। প্রতিটি এন্টিটি বেস পয়েন্টের সাপেক্ষে তার জ্যামিতি স্কেল করে:

| এন্টিটি | কী পরিবর্তন হয় |
|--------|-------------|
| Line | উভয় শেষ পয়েন্ট বেস পয়েন্ট থেকে স্কেল করা |
| Circle | কেন্দ্র বেস পয়েন্ট থেকে স্কেল; ব্যাসার্ধ ফ্যাক্টর দ্বারা গুণিত |
| Arc | কেন্দ্র স্কেল; ব্যাসার্ধ ফ্যাক্টর দ্বারা গুণিত; কোণ অপরিবর্তিত |
| Ellipse | কেন্দ্র স্কেল; উভয় সেমি-অক্ষের দৈর্ঘ্য ফ্যাক্টর দ্বারা গুণিত |
| Polyline / Rectangle | প্রতিটি ভার্টেক্স বেস পয়েন্ট থেকে স্কেল |
| Text | অ্যাঙ্কর পয়েন্ট স্কেল; উচ্চতা স্কেল ফ্যাক্টর দ্বারা গুণিত |
| Spline | সমস্ত কন্ট্রোল ভার্টেক্স / ফিট পয়েন্ট স্কেল |
