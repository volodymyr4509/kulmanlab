# Text

`text` কমান্ড একটি বহু-লাইন টেক্সট লেবেল স্থাপন করে। ক্যানভাসে একটি পজিশন ক্লিক করার পরে, একটি পপআপ এডিটর **rich** মোডে খোলে — আপনি বিষয়বস্তু টাইপ করতে, প্রতি-অক্ষর বোল্ড/ইটালিক/স্ট্রাইকথ্রু প্রয়োগ করতে, ফন্ট ও উচ্চতা পরিবর্তন করতে এবং লাইন বিরতি ঢোকাতে পারেন। কমিট করতে এবং এডিটর বন্ধ করতে **Escape** চাপুন।

সম্পূর্ণ এডিটর রেফারেন্সের জন্য [Text Editor](../../interface/text-editor/) পেজ দেখুন, যেখানে **rich** এবং **simple** মোডের তুলনা অন্তর্ভুক্ত রয়েছে।

## একটি টেক্সট লেবেল স্থাপন করা

1. টার্মিনালে `text` টাইপ করুন বা **Text** টুলবার বাটনে ক্লিক করুন।
2. ক্যানভাসে **অ্যাঙ্কর পজিশনে ক্লিক করুন**। বা সঠিক কোঅর্ডিনেটের জন্য `X,Y` টাইপ করুন এবং **Enter** চাপুন।
3. নতুন লেবেলের উপরে **টেক্সট এডিটর পপআপ** খোলে। আপনার বিষয়বস্তু টাইপ করুন।
4. লেবেল কমিট করতে এবং এডিটর বন্ধ করতে **Escape** চাপুন।

ডিফল্ট উচ্চতা হলো **১২ ড্রয়িং ইউনিট**।

## বিদ্যমান লেবেল সম্পাদনা

ক্যানভাসে যেকোনো টেক্সট লেবেলে **ডাবল-ক্লিক** করলে সেই লেবেলের এডিটর পুনরায় খোলে।

## অ্যাঙ্করের জন্য কোঅর্ডিনেট এন্ট্রি

ক্লিক করার বদলে, একটি সঠিক পজিশন টাইপ করুন:

1. X মান টাইপ করুন।
2. `,` চাপুন — টার্মিনাল `[X], [Y{কার্সর}]` দেখায়।
3. Y মান টাইপ করুন।
4. অ্যাঙ্কর স্থাপন করতে এবং এডিটর খুলতে **Enter** চাপুন।

## কীবোর্ড রেফারেন্স

**অ্যাঙ্কর পর্যায়**

| কী | ক্রিয়া |
|-----|--------|
| `0`–`9`, `.`, `-` | X কোঅর্ডিনেট এন্ট্রি শুরু করুন |
| `,` | X লক করুন এবং Y এন্ট্রিতে যান |
| `Backspace` | শেষ টাইপ করা অক্ষর মুছুন |
| `Enter` | টাইপ করা কোঅর্ডিনেট নিশ্চিত করুন |

**টেক্সট এডিটর পর্যায়** (সম্পূর্ণ রেফারেন্সের জন্য [Text Editor](../../interface/text-editor/) দেখুন)

| কী | ক্রিয়া |
|-----|--------|
| যেকোনো প্রিন্টযোগ্য অক্ষর | কার্সরে ঢোকান |
| `Backspace` / `Delete` | সংলগ্ন অক্ষর বা নির্বাচন মুছুন |
| `Enter` | লাইন বিরতি ঢোকান |
| `←` / `→` | ক্যারেট সরান |
| `Home` / `End` | হার্ড লাইনের শুরু / শেষে লাফ দিন |
| `Escape` | কমিট করুন এবং এডিটর বন্ধ করুন |

## গ্রিপ সম্পাদনা — পুনরায় স্থাপন

একটি নির্বাচিত টেক্সট লেবেল অ্যাঙ্কর পয়েন্টে একটি গ্রিপ দেখায়:

| গ্রিপ | পজিশন | কী করে |
|------|----------|--------------|
| **অ্যাঙ্কর** | টেক্সটের নিচে-বামে | লেবেল পুনরায় স্থাপন করতে টেনে আনুন |

## টেক্সট নির্বাচন

| পদ্ধতি | আচরণ |
|--------|-----------|
| **ক্লিক** | ক্লিক টেক্সটের ঘোরানো বাউন্ডিং বাক্সের মধ্যে পড়লে নির্বাচিত হয় |
| **ডানে টানুন** (কঠোর) | বাউন্ডিং বাক্সের সমস্ত চারটি কোণ নির্বাচন এলাকার ভেতরে থাকতে হবে |
| **বামে টানুন** (ক্রসিং) | টেক্সট বাউন্ডিং বাক্স এবং নির্বাচন এলাকার মধ্যে যেকোনো ওভারল্যাপ নির্বাচিত করে |

## সমর্থিত সম্পাদনা কমান্ড

| কমান্ড | টেক্সটে কী হয় |
|---------|--------------------------|
| [Move](../move/) | অ্যাঙ্কর পয়েন্ট সরায় |
| [Copy](../copy/) | নতুন পজিশনে একটি অভিন্ন লেবেল তৈরি করে |
| [Rotate](../rotate/) | অ্যাঙ্কর পজিশন ঘোরায় এবং Rotation Degree তে কোণ যোগ করে |
| [Mirror](../mirror/) | মিরর অক্ষ জুড়ে অ্যাঙ্কর পয়েন্ট প্রতিফলিত করে (টেক্সট স্ট্রিং উল্টানো হয় না) |
| [Scale](../scale/) | অ্যাঙ্কর পজিশন স্কেল করে এবং উচ্চতা স্কেল ফ্যাক্টর দ্বারা গুণ করে |
| [Delete](../delete/) | লেবেল সরায় |

টেক্সট **Offset**, **Trim**, বা **Extend** সমর্থন করে না।

## প্রপার্টি

একটি টেক্সট লেবেল নির্বাচিত হলে প্রপার্টি প্যানেল দেখায়:

**সাধারণ**

| প্রপার্টি | ডিফল্ট | অর্থ |
|----------|---------|---------|
| Color | 256 (ByLayer) | ACI রঙের সূচক |
| Layer | `0` | লেয়ার অ্যাসাইনমেন্ট |

**জ্যামিতি**

| প্রপার্টি | অর্থ |
|----------|---------|
| Position X / Position Y | অ্যাঙ্কর পয়েন্ট কোঅর্ডিনেট |
| Height | ড্রয়িং ইউনিটে বেস টেক্সট উচ্চতা (ডিফল্ট: **১২**) |
| Rotation Degree | ডিগ্রিতে ঘড়ির বিপরীতে ঘূর্ণন |

**প্রপার্টি**

| প্রপার্টি | অর্থ |
|----------|---------|
| Content | টেক্সট স্ট্রিং (MTEXT ইনলাইন কোড সংরক্ষিত) |
| Attachment Point | অ্যালাইনমেন্ট কোড (১ = উপরে-বামে … ৯ = নিচে-ডানে) |

টেক্সটে Linetype, Linetype Scale, বা Thickness প্রপার্টি নেই।

## DXF — MTEXT এন্টিটি

টেক্সট লেবেলগুলি DXF ফাইলে **MTEXT** এন্টিটি হিসেবে সংরক্ষিত। বোল্ড এবং ইটালিক `\L`, `\K`, `\O` এবং ইনলাইন ফন্ট সুইচ (`\f`) ব্যবহার করে এনকোড করা হয়। প্রতি-অক্ষর উচ্চতা `\H` হিসেবে এনকোড করা হয়। সমস্ত ফরম্যাটিং একটি সম্পূর্ণ DXF রাউন্ড-ট্রিপ টিকে থাকে এবং LibreCAD, FreeCAD এবং অন্য DXF-সামঞ্জস্যপূর্ণ অ্যাপ্লিকেশন দ্বারা পঠনযোগ্য।
