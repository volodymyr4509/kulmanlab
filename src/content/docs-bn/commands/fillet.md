# Fillet

`fillet` কমান্ড একটি নির্দিষ্ট ব্যাসার্ধের একটি স্পর্শক চাপ ঢোকিয়ে এবং প্রতিটি লাইনকে চাপ শুরু হওয়ার বিন্দুতে ট্রিম করে দুটি [Line](../line/) সত্তার মধ্যে কোণটি গোলাকার করে। ফলাফল হল উভয় লাইন সংযুক্ত করে একটি মসৃণ, গোলাকার কোণ।

Fillet শুধুমাত্র **Line সত্তায়** কাজ করে।

## Fillet ব্যবহার করা

1. টার্মিনালে `fillet` টাইপ করুন বা **Fillet** টুলবার বোতামে ক্লিক করুন।
2. **fillet ব্যাসার্ধ টাইপ করুন** এবং **Enter** চাপুন।
3. **প্রথম লাইনে ক্লিক করুন** — আপনি যেখানে ক্লিক করেন তা নির্ধারণ করে কোনো ছেদ বিন্দুর কোন পাশ রাখা হবে।
4. **দ্বিতীয় লাইনের উপর হোভার করুন** — একটি ড্যাশড চাপ পূর্বদর্শন ফলস্বরূপ ফিলেট দেখায়। আপনি যে পাশটি রাখতে চান সেখানে কার্সার সরান।
5. **ক্লিক করুন** প্রয়োগ করতে। উভয় লাইন ট্রিম করা হয় এবং চাপ ঢোকানো হয়।

```
  আগে:                     ব্যাসার্ধ r দিয়ে fillet-এর পরে:

  ──────────────              ──────────╮
                │                        ╰────
                │
```

## ছেদকারী লাইনের জন্য পাশ নির্বাচন

যখন দুটি লাইন একে অপরকে অতিক্রম করে, ক্লিক অবস্থান দ্বারা নির্ধারিত কোণে fillet প্রয়োগ করা হয় — প্রতিটি লাইনের **কার্সারের একই পাশের** অংশ রাখা হয়।

- প্রথম লাইনের একটি প্রান্তের কাছে ক্লিক করুন সেই অর্ধেক নির্বাচন করতে।
- দ্বিতীয় লাইনের পছন্দের অর্ধেকে কার্সার সরান — ড্যাশড পূর্বদর্শন লাইভ আপডেট হয়।

## কমান্ড কী তৈরি করে

- প্রথম লাইনের ছেদ বিন্দুর নিকটতম শেষবিন্দু স্পর্শক বিন্দু **T1**-এ স্থানান্তরিত হয়।
- দ্বিতীয় লাইনের ছেদ বিন্দুর নিকটতম শেষবিন্দু স্পর্শক বিন্দু **T2**-এ স্থানান্তরিত হয়।
- **T1** থেকে **T2**-তে উভয় লাইনের স্পর্শক একটি নতুন Arc সত্তা ঢোকানো হয়।

ঢোকানো চাপ বর্তমান lineweight, রঙ, লেয়ার এবং linetype সেটিং উত্তরাধিকার করে।

## কীবোর্ড রেফারেন্স

| কী | ক্রিয়া |
|-----|--------|
| `0`–`9`, `.` | ব্যাসার্ধ মানে অঙ্ক যোগ করুন |
| `Backspace` | শেষ টাইপ করা অক্ষর মুছুন |
| `Enter` | টাইপ করা ব্যাসার্ধ নিশ্চিত করুন এবং লাইন নির্বাচনে যান |
| `Escape` | বাতিল করুন এবং রিসেট করুন |

## সমর্থিত সত্তা

| সত্তা | সমর্থিত |

| Arc, Circle, Ellipse, Polyline | না |
| Text, Spline, Dimension, Leader | না |
