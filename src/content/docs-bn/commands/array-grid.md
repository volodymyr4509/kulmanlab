# Array Grid

`ArrayGrid` কমান্ড নির্বাচিত সত্তা থেকে একটি আয়তাকার গ্রিডের কপি তৈরি করে — সারির সংখ্যা, কলামের সংখ্যা এবং তাদের মধ্যে দূরত্ব দিন, সবকিছু টার্মিনালে টাইপ করা হয়। মূল নির্বাচন সারি 0, কলাম 0 সেল দখল করে; বাকি প্রতিটি সেল একটি অনুবাদিত কপি।

## শুরু করার দুটি উপায়

**প্রি-সিলেক্ট করুন, তারপর array করুন** — প্রথমে সত্তা নির্বাচন করুন, তারপর সক্রিয় করুন:

1. ক্যানভাসে এক বা একাধিক সত্তা নির্বাচন করুন।
2. টার্মিনালে `arraygrid` টাইপ করুন (শুধু `arr` যথেষ্ট — এটি স্পষ্ট) অথবা **Array Grid** টুলবার বোতামে ক্লিক করুন।
3. **সারির** সংখ্যা টাইপ করুন এবং **Enter** চাপুন।
4. **কলামের** সংখ্যা টাইপ করুন এবং **Enter** চাপুন।
5. **সারির মধ্যে দূরত্ব** টাইপ করুন এবং **Enter** চাপুন।
6. **কলামের মধ্যে দূরত্ব** টাইপ করুন এবং **Enter** চাপুন — গ্রিড অবিলম্বে তৈরি হয়।

**সক্রিয় করুন, তারপর নির্বাচন করুন** — কিছু নির্বাচন না করে কমান্ড শুরু করুন:

1. `arraygrid` টাইপ করুন বা টুলবার বোতামে ক্লিক করুন।
2. **বস্তু নির্বাচন করুন** — পৃথক সত্তা টগল করতে ক্লিক করুন, বা এলাকা দ্বারা নির্বাচন করতে টানুন।
3. নির্বাচন নিশ্চিত করতে **Enter** বা **Space** চাপুন।
4. উপরের মতো সারি → কলাম → সারি দূরত্ব → কলাম দূরত্ব দিয়ে চালিয়ে যান।

```
  ২ সারি x ৩ কলাম:

  [B] [B] [B]   <- সারি 1 (অনুবাদিত কপি)
  [A] [A] [A]   <- সারি 0: মূল নির্বাচন, কপি ডানদিকে
```

> টার্মিনালের কেবল স্পষ্ট হওয়ার জন্য যথেষ্ট অক্ষরের প্রয়োজন — `arr` টাইপ করা এবং **Enter** চাপা সরাসরি Array Grid সক্রিয় করে, কারণ অন্য কোনো কমান্ডের নাম এই তিনটি অক্ষর দিয়ে শুরু হয় না (Arc, Area, Align, এবং Angle সবই আগে থেকেই আলাদা হয়ে যায়)।

## সারি, কলাম এবং দূরত্ব

| প্রম্পট | গ্রহণ করে | নোট |
|---------|-----------|-----|
| সারি | ধনাত্মক পূর্ণ সংখ্যা (1, 2, 3…) | শুধুমাত্র অঙ্ক — কোনো দশমিক বিন্দু বা চিহ্ন নেই |
| কলাম | ধনাত্মক পূর্ণ সংখ্যা (1, 2, 3…) | শুধুমাত্র অঙ্ক — কোনো দশমিক বিন্দু বা চিহ্ন নেই |
| সারি দূরত্ব | চিহ্নযুক্ত দশমিক (যেমন `10`, `-5.5`) | সারির মধ্যে দূরত্ব; ঋণাত্মক দিক বিপরীত করে |
| কলাম দূরত্ব | চিহ্নযুক্ত দশমিক (যেমন `10`, `-5.5`) | কলামের মধ্যে দূরত্ব; ঋণাত্মক দিক বিপরীত করে |

১টি সারি এবং ১টি কলামের সাথে, কোনো কপি তৈরি হয় না — কমান্ড ড্রয়িং পরিবর্তন না করেই প্রস্থান করে।

## কীবোর্ড রেফারেন্স

| কী | ক্রিয়া |
|-----|--------|
| `Enter` / `Space` | নির্বাচন নিশ্চিত করুন এবং সারি প্রম্পটে এগিয়ে যান |
| `0`–`9` | সারি বা কলামের জন্য অঙ্ক লিখুন |
| `0`–`9`, `.`, `-` | সারি/কলাম দূরত্বের জন্য অঙ্ক লিখুন (`-` শুধুমাত্র প্রথম অক্ষর হিসেবে) |
| `Backspace` | বর্তমান প্রম্পটের শেষ টাইপ করা অক্ষর মুছুন |
| `Enter` | বর্তমান প্রম্পট নিশ্চিত করুন এবং পরবর্তীতে এগিয়ে যান |
| `Escape` | টাইপ করা সারি/কলাম/দূরত্ব মান পরিষ্কার করুন এবং নির্বাচন পর্যায়ে ফিরে যান |

## কমান্ড চলাকালীন নির্বাচন

| পদ্ধতি | আচরণ |
|--------|-----------|
| **ক্লিক** | কার্সারের নীচের সত্তাকে নির্বাচনে টগল করে |
| **ডানে টানুন** (কঠোর) | বাক্সের সম্পূর্ণ ভেতরে থাকা সত্তা যোগ করে |
| **বামে টানুন** (ক্রসিং) | বাক্সের সীমানা অতিক্রম করা সত্তা যোগ করে |
| **Enter** / **Space** | নির্বাচন নিশ্চিত করে এবং সারি প্রম্পটে এগিয়ে যায় |

## array এর পরে

নতুন কপিগুলি ড্রয়িংয়ে যোগ করা হয় এবং কমান্ড প্রস্থান করে — মূল নির্বাচন পরিষ্কার হয়ে যায়। **Array Grid** আবার চালান, বা একটি নতুন কমান্ড শুরু করুন।

## Array Grid বনাম Copy

| | Array Grid | Copy |
|---|-----------|------|
| পয়েন্ট নির্বাচন | কোনোটি নয় — সারি, কলাম এবং দূরত্ব টাইপ করা হয় | ভিত্তি পয়েন্ট এবং গন্তব্যে ক্লিক করা হয় (বা টাইপ করা হয়) |
| তৈরি করা কপি | সারি × কলাম − 1 | প্রতি কপি অপারেশনে ঠিক 1টি |
| লেআউট | নিয়মিত আয়তাকার গ্রিড | যেকোনো জায়গায়, যেকোনো অফসেটে |
| সবচেয়ে ভালো | নিয়মিত প্যাটার্নে একটি ইউনিট পুনরাবৃত্তি করা (গর্ত, টাইলস, ফাস্টেনার) | যেকোনো স্থানে একটি একক ডুপ্লিকেট |

## সমর্থিত সত্তা

Array Grid প্রতিটি সত্তার ধরনে কাজ করে। সমস্ত সত্তা অভ্যন্তরীণভাবে `translate(dx, dy)` বাস্তবায়ন করে — [Copy](../copy/) এবং [Move](../move/) যে একই অপারেশন ব্যবহার করে — তাই কেউই বাদ পড়ে না।
