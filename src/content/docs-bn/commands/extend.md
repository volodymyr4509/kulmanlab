# Extend

`extend` কমান্ড একটি [Line](../line/)-এর নিকটতম শেষবিন্দুকে ড্রয়িংয়ের অন্য সত্তার সাথে এটি গঠন করা নিকটতম ছেদে প্রসারিত করে। আপনি যে শেষবিন্দুটি প্রসারিত করতে চান তার কাছে হোভার করুন — একটি পূর্বদর্শন প্রসারিত লাইন দেখায় — তারপর প্রয়োগ করতে ক্লিক করুন।

Extend শুধুমাত্র **Line সত্তায়** কাজ করে। যে সীমানায় লাইন প্রসারিত হয় তা Text, Mtext, Multileader এবং Spline ছাড়া যেকোনো সত্তার ধরন হতে পারে।

## একটি লাইন প্রসারিত করা

1. টার্মিনালে `extend` টাইপ করুন বা **Extend** টুলবার বোতামে ক্লিক করুন।
2. **একটি লাইনের একটি প্রান্তের কাছে হোভার করুন** — পূর্বদর্শন সেই দিকে নিকটতম সীমানায় প্রসারিত লাইন দেখায়।
3. এক্সটেনশন প্রয়োগ করতে **ক্লিক করুন**।

কমান্ডটি প্রতিটি এক্সটেনশনের পরে সক্রিয় থাকে যাতে আপনি ধারাবাহিকভাবে একাধিক লাইন প্রসারিত করতে পারেন। প্রস্থান করতে **Escape** চাপুন।

```
  আগে:                      পরে:

  ──────           |           ──────────────|
  (ছোট লাইন)     (সীমানা)  (সীমানায় প্রসারিত)
```

## শেষবিন্দু কীভাবে বেছে নেওয়া হয়

কমান্ড দেখে কার্সার কোন শেষবিন্দুর কাছাকাছি:

- কার্সার **শেষ পয়েন্টের কাছাকাছি** → শেষটি লাইনের দিক বরাবর এগিয়ে প্রসারিত।
- কার্সার **স্টার্ট পয়েন্টের কাছাকাছি** → স্টার্ট পিছনের দিকে প্রসারিত (বিপরীত দিকে)।

বেছে নেওয়া শেষবিন্দু থেকে লাইনের দিক বরাবর একটি রশ্মি নিক্ষেপ করা হয়, এবং সেই রশ্মি বরাবর যেকোনো অন্য সত্তার সাথে **নিকটতম ছেদ** (লাইন নিজেই এবং উপেক্ষিত ধরন ছাড়া) নতুন শেষবিন্দু হয়।

সেই দিকে কোনো ছেদ পাওয়া না গেলে, কোনো পূর্বদর্শন প্রদর্শিত হয় না এবং ক্লিক করলে কিছুই হয় না।

## সীমানা বাদ দেওয়া

নিম্নলিখিত সত্তার ধরনগুলি সীমানা হিসেবে উপেক্ষা করা হয় — একটি লাইন তাদের সাথে মিলতে প্রসারিত হয় না:

- Text / Mtext
- Multileader
- Spline

অন্যান্য সব ধরন (Line, Arc, Circle, Ellipse, Polyline, Dimension) বৈধ সীমানা হিসেবে কাজ করে।

| সত্তা | প্রসারিত করা যাবে? |
|--------|----------------|
| Line | হ্যাঁ |
| Arc, Circle, Ellipse | না |
| Polyline / Rectangle | না |
| Text, Spline, Dimension, Leader | না |

## Extend বনাম Trim

| | Extend | Trim |
|---|--------|------|
| এটি কী করে | একটি লাইনের শেষবিন্দু সীমানায় প্রসারিত করে | একটি লাইনের সেগমেন্ট সরিয়ে দেয় |
| ট্রিগার | প্রসারিত করতে শেষবিন্দুর কাছে হোভার করুন | কাটতে সেগমেন্টের উপর হোভার করুন |
