# New File

**New File** কমান্ড ক্যানভাস পরিষ্কার করে এবং একটি নতুন ফাঁকা ড্রয়িং শুরু করে। টাইমস্ট্যাম্প সহ একটি অনন্য ফাইলের নাম স্বয়ংক্রিয়ভাবে তৈরি হয়।

## একটি নতুন ফাইল কীভাবে তৈরি করবেন

ফাইল প্যানেলে **New File** টুলবার বাটনে (নতুন-পেজ আইকন) ক্লিক করুন। ক্যানভাস সঙ্গে সঙ্গে পরিষ্কার হয় — কোনো প্রম্পট বা নিশ্চিতকরণ ডায়ালগ নেই।

## নতুন ফাইলে কী থাকে

নতুন তৈরি ফাইলে শুরুতে থাকে:

- ক্যানভাসে **কোনো এন্টিটি নেই**।
- সাদা রঙ এবং `Continuous` লাইনটাইপ সহ `0` নামের **একটি ডিফল্ট লেয়ার**।
- `kulman-Mon01_HH:MM:SS.dxf` ফরম্যাটে একটি **তৈরি করা ফাইলের নাম** (যেমন `kulman-May22_14:30:00.dxf`)।

ফাইলটি স্বয়ংক্রিয়ভাবে ব্রাউজার স্টোরেজে সেভ হয় এবং [Recent Files](../files/) এ দেখা যায়।

## সতর্কতা — সংরক্ষণ না করা কাজ মুছে যায়

**New File** ক্লিক করলে সতর্কতা ছাড়াই বর্তমান ক্যানভাসের সমস্ত এন্টিটি মুছে যায়। বর্তমান ড্রয়িং রাখতে চাইলে, প্রথমে [export](../export/) করুন।

## New File বনাম Import — কখন কোনটি

|-----------|-------------------|
| স্ক্র্যাচ থেকে ড্রয়িং শুরু করা | **New File** |
