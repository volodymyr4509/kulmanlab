# Spline CV

`splinecv` কমান্ড কন্ট্রোল ভার্টেক্স (CV) স্থাপন করে একটি **কিউবিক B-spline** আঁকে। ফলস্বরূপ বক্ররেখা প্রতিটি CV-র দিকে আকর্ষিত হয় কিন্তু সেগুলির মধ্য দিয়ে যায় না — প্রথম ও শেষ ভার্টেক্স বাদে, যেখানে **ক্ল্যাম্পড নট** বক্ররেখাকে ঠিক সেখানে অ্যাঙ্কর করে। এটি স্বজ্ঞাত আকৃতি নিয়ন্ত্রণ দেয়: প্রতিটি বিন্দু স্পর্শ না করে বক্ররেখা সেদিকে ঠেলতে একটি ভার্টেক্স টানুন।

## কন্ট্রোল ভার্টেক্স দিয়ে স্প্লাইন আঁকা

1. টার্মিনালে `splinecv` টাইপ করুন বা **Spline CV** টুলবার বাটনে ক্লিক করুন।
2. **কন্ট্রোল ভার্টেক্স স্থাপন করতে ক্লিক করুন** — প্রতিটি ক্লিক একটি ভার্টেক্স যোগ করে। বা সঠিক কোঅর্ডিনেটের জন্য `X,Y` টাইপ করুন এবং **Enter** চাপুন।
3. শেষ করতে **Enter** চাপুন (কমপক্ষে ২টি ভার্টেক্স প্রয়োজন)।

```
  CV ●         ● CV
      \       /
       \     /    ← CV-র দিকে আকর্ষিত বক্ররেখা
        \   /         কিন্তু সেগুলির মধ্য দিয়ে যায় না
  CV ●   ●   ● CV (শুরু/শেষ: বক্ররেখা এখানে স্পর্শ করে)
```

প্রতিটি ভার্টেক্সের পরে কার্সর সরানোর সময় লাইভ প্রিভিউ আপডেট হয়, কার্সর পজিশনে পরবর্তী পয়েন্ট সহ স্প্লাইন কেমন দেখাবে তা দেখায়। সমস্ত স্থাপিত ভার্টেক্স বাদ দিতে এবং বের হতে **Escape** চাপুন।

## কোঅর্ডিনেট এন্ট্রি

ক্লিক করার বদলে, যেকোনো কন্ট্রোল ভার্টেক্সের জন্য একটি সঠিক পজিশন টাইপ করুন:

1. X মান টাইপ করুন।
2. `,` চাপুন — টার্মিনাল `[X], [Y{কার্সর}]` দেখায়।
3. Y মান টাইপ করুন।
4. ভার্টেক্স স্থাপন করতে **Enter** চাপুন।

## কীবোর্ড রেফারেন্স

| কী | ক্রিয়া |
|-----|--------|
| `0`–`9`, `.`, `-` | X কোঅর্ডিনেট এন্ট্রি শুরু করুন |
| `,` | X লক করুন এবং Y এন্ট্রিতে যান |
| `Backspace` | শেষ টাইপ করা অক্ষর মুছুন |
| `Enter` | টাইপ করা কোঅর্ডিনেট নিশ্চিত করুন, বা কিছু না থাকলে এবং ≥ ২ ভার্টেক্স থাকলে স্প্লাইন শেষ করুন |
| `Escape` | সমস্ত ভার্টেক্স বাদ দিন এবং বের হন |

## গ্রিপ সম্পাদনা — কন্ট্রোল ভার্টেক্স দিয়ে পুনরায় আকার দেওয়া

একটি নির্বাচিত CV স্প্লাইন প্রতিটি কন্ট্রোল ভার্টেক্সে একটি গ্রিপ দেখায়:

| গ্রিপ | পজিশন | কী করে |
|------|----------|--------------|
| **কন্ট্রোল ভার্টেক্স** | প্রতিটি CV পজিশনে | সেই CV সরাতে টেনে আনুন — বক্ররেখা নতুন পজিশনের দিকে পুনরায় আকার পায় |

"পুরো স্প্লাইন সরান" গ্রিপ নেই। সম্পূর্ণ স্প্লাইন স্থানান্তর করতে, [Move](../move/) কমান্ড ব্যবহার করুন।

## CV স্প্লাইন নির্বাচন

| পদ্ধতি | আচরণ |
|--------|-----------|
| **ক্লিক** | বক্ররেখার কাছাকাছি কোনো পয়েন্টে ক্লিক হলে নির্বাচিত হয় |
| **ডানে টানুন** (কঠোর) | বক্ররেখা বরাবর সমস্ত নমুনা পয়েন্ট নির্বাচন বাক্সের ভেতরে থাকতে হবে |
| **বামে টানুন** (ক্রসিং) | নির্বাচন বাক্সের সীমানা অতিক্রমকারী বক্ররেখার যেকোনো অংশ নির্বাচিত করে |

## Spline CV বনাম Spline Fit — কোনটি ব্যবহার করবেন

| | Spline CV | Spline Fit |
|---|-----------|------------|
| বক্ররেখা পয়েন্টের মধ্য দিয়ে যায় | শুধুমাত্র প্রথম ও শেষ (ক্ল্যাম্পড) | প্রতিটি ক্লিক করা পয়েন্ট ঠিকমতো |
| আকৃতি নিয়ন্ত্রণ | CV একটি অঞ্চলের দিকে টানুন | বক্ররেখা স্পর্শ করতে হবে এমন ফিট পয়েন্ট সরান |
| গ্রিপ সম্পাদনা প্রভাব | CV সরে → বক্ররেখা আকর্ষিত | ফিট পয়েন্ট সরে → বক্ররেখা পুনরায় ইন্টারপোলেট |
| সেরা জন্য | মসৃণ নান্দনিক বক্ররেখা, মুক্ত-ফর্ম পথ | নির্দিষ্ট কোঅর্ডিনেট স্পর্শ করতে হবে এমন বক্ররেখা |

## DXF — SPLINE এন্টিটি (কন্ট্রোল-ভার্টেক্স ফর্ম)

CV স্প্লাইনগুলি DXF ফাইলে `SPLINE` এন্টিটি হিসেবে সেভ হয়, ডিগ্রি, নট ভেক্টর এবং সমস্ত কন্ট্রোল ভার্টেক্স কোঅর্ডিনেট সংরক্ষণ করে। সমস্ত প্রপার্টি — রঙ, লেয়ার, লাইনটাইপ, লাইনটাইপ স্কেল এবং পুরুত্ব — ক্ষতি ছাড়াই রাউন্ড-ট্রিপ করে। `splineFlag` `9` (CV স্প্লাইন) সেট করা হয় যাতে রিলোডে ফর্ম সংরক্ষিত থাকে। CV ডেটা সহ `SPLINE` এন্টিটি সমর্থনকারী যেকোনো DXF অ্যাপ্লিকেশন এগুলি সঠিকভাবে পড়ে।
