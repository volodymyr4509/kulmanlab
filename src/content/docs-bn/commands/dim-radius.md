# Dimension Radius

`dimradius` কমান্ড একটি আর্ক বা বৃত্তে একটি ব্যাসার্ধ মাত্রা স্থাপন করে। মাত্রা রেখা কার্সারের দিকে পরিধির একটি বিন্দুতে কেন্দ্র থেকে চলে, `R <মান>` লেবেল করা হয়। পরিবর্তে একটি সম্পূর্ণ ব্যাস মাত্রা দিতে, [Dimension Diameter](./dim-diameter) ব্যবহার করুন।

## একটি ব্যাসার্ধ মাত্রার গঠন

```
  ● (কেন্দ্র)
   \
    \  R 5.00
     \
      ●────── টেক্সট (কার্সার পাশ)
   (আর্ক পয়েন্ট)
```

- **মাত্রা রেখা** — কেন্দ্র থেকে কার্সারের দিকে একটি আর্ক পয়েন্টের মধ্য দিয়ে, আর্কে তীরমাথা সহ।
- **লেবেল** — `R` তারপর ব্যাসার্ধের মান।

## একটি ব্যাসার্ধ মাত্রা স্থাপন করা

1. টার্মিনালে `dimradius` টাইপ করুন বা **Dimension Radius** টুলবার বোতামে ক্লিক করুন।
2. এটি নির্বাচন করতে **একটি আর্ক বা বৃত্তে ক্লিক করুন**।
3. মাত্রা রেখা অভিমুখ করতে **কার্সার সরান** — আর্ক-পয়েন্ট কেন্দ্র থেকে কার্সারের দিক ট্র্যাক করে।
4. মাত্রা স্থাপন করতে **ক্লিক করুন**।

শুধুমাত্র **Arc** এবং **Circle** সত্তা নির্বাচন করা যায়। অন্য যেকোনো সত্তার ধরনে ক্লিক করলে কিছুই হয় না।

|-----|--------|
| `Escape` | বাতিল করুন |

## Dimension Radius বনাম Dimension Diameter

| | Dimension Radius | Dimension Diameter |
|---|-----------------|-------------------|
| পরিমাপ করে | ব্যাসার্ধ (কেন্দ্র থেকে প্রান্ত) | ব্যাস (কেন্দ্রের মধ্য দিয়ে প্রান্ত থেকে প্রান্ত) |
| মাত্রা রেখা | কেন্দ্র → আর্ক পয়েন্ট | আর্ক পয়েন্ট → আর্ক পয়েন্ট (কেন্দ্রের মধ্য দিয়ে) |
| লেবেল প্রিফিক্স | `R` | `⌀` |
| তীরমাথা | একটি (আর্ক পয়েন্টে) | দুটি (উভয় আর্ক পয়েন্টে) |
| সবচেয়ে ভালো | একটি বাঁকা বৈশিষ্ট্যের একটি পাশ টীকা করা | সম্পূর্ণ বৃত্তাকার মাত্রা টীকা করা |

## লেবেল সম্পাদনা — সরল মোড

স্থাপিত ব্যাসার্ধ মাত্রায় **ডাবল-ক্লিক** করুন **সরল** মোডে টেক্সট এডিটর খুলতে। এডিটরটি বর্তমান রেন্ডার করা মান (যেমন `R 5.00`) দিয়ে পূর্ব-পূরণ করা থাকে যাতে আপনি কার্সার অবস্থান করে সরাসরি সম্পাদনা করতে পারেন।

| বৈশিষ্ট্য | আচরণ |
|---------|-----------|
| Bold / Italic / Font / Height | **সম্পূর্ণ** লেবেলে একসাথে প্রযোজ্য |
| প্রতি-অক্ষর ফরম্যাটিং | সমর্থিত নয় |
| `Enter` | মান কমিট করে এবং এডিটর বন্ধ করে |
| বহু-লাইন | সমর্থিত নয় |

সম্পূর্ণ রেফারেন্সের জন্য [Text Editor — simple mode](../interface/text-editor#simple-mode) দেখুন।

## DXF — DIMENSION ব্যাসার্ধ সত্তা

ব্যাসার্ধ মাত্রাগুলি ব্যাসার্ধ-ধরনের জ্যামিতি সহ `DIMENSION` সত্তা হিসেবে সংরক্ষিত হয়, কেন্দ্র স্থানাঙ্ক, আর্ক-পয়েন্ট অবস্থান এবং পরিমাপ করা ব্যাসার্ধের মান সঞ্চয় করে। সমস্ত বৈশিষ্ট্য কোনো ক্ষতি ছাড়াই রাউন্ড-ট্রিপ করে।
