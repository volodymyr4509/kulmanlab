# LayerMakeCurrent

`LayerMakeCurrent` কমান্ড **বর্তমান ড্রয়িং লেয়ার** কে ক্লিক করা এন্টিটি যে লেয়ারে আছে সেটিতে সেট করে। নতুন এন্টিটিগুলি তারপর স্বয়ংক্রিয়ভাবে সেই লেয়ারে আঁকা হবে।

## ব্যবহার

1. টার্মিনালে `LayerMakeCurrent` টাইপ করুন বা **Make Current** টুলবার বাটনে (আইড্রপার আইকন) ক্লিক করুন।
2. ক্যানভাসে **যেকোনো এন্টিটিতে ক্লিক করুন**।
3. বর্তমান লেয়ার সেই এন্টিটির লেয়ারের সাথে মিলিয়ে আপডেট হয়। কমান্ডটি সঙ্গে সঙ্গে শেষ হয়।

## আচরণের বিবরণ

- যদি আপনি খালি ক্যানভাসে ক্লিক করেন (কোনো এন্টিটি খুঁজে পাওয়া না যায়), টার্মিনালে `no object found` দেখায় এবং কমান্ডটি সক্রিয় থাকে যাতে আপনি আবার চেষ্টা করতে পারেন।
- শুধুমাত্র বর্তমান লেয়ার সেটিং পরিবর্তন হয় — কোনো এন্টিটি পরিবর্তিত হয় না।
- আপডেট করা লেয়ার টুলবারের লেয়ার সিলেক্টরে প্রতিফলিত হয়।
