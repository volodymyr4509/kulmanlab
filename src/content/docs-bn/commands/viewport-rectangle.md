# ViewportRectangle

`ViewportRectangle` কমান্ড দুটি বিপরীত কোণ বেছে সক্রিয় পেপার লেআউটে একটি নতুন ভিউপোর্ট তৈরি করে। শুধুমাত্র লেআউট স্পেসে উপলব্ধ।

## একটি ভিউপোর্ট তৈরি করা

1. স্ক্রিনের নিচে ট্যাব ব্যবহার করে একটি পেপার লেআউটে যান।
2. টার্মিনালে `ViewportRectangle` টাইপ করুন বা **Viewport Rectangle** টুলবার বাটনে ক্লিক করুন।
3. **প্রথম কোণে ক্লিক করুন**, বা সঠিক কোঅর্ডিনেটের জন্য `X,Y` টাইপ করুন এবং **Enter** চাপুন।
4. **বিপরীত কোণে ক্লিক করুন** — ভিউপোর্ট সঙ্গে সঙ্গে স্থাপিত হয়। কোঅর্ডিনেট এন্ট্রি এখানেও কাজ করে।

নতুন ভিউপোর্ট লেআউটের ডিফল্ট স্কেলে সম্পূর্ণ মডেল দেখায়। মডেল ভিউ জুম করতে ভিউপোর্টের ভেতরে স্ক্রোল হুইল ব্যবহার করুন, বা প্যান করতে মিডল-ড্র্যাগ করুন।

## কোঅর্ডিনেট এন্ট্রি

যেকোনো কোণ ধাপে আপনি একটি সঠিক কোঅর্ডিনেট টাইপ করতে পারেন:

1. X মান টাইপ করুন।
2. `,` চাপুন — টার্মিনাল `[X], [Y{কার্সর}]` দেখায়।
3. Y মান টাইপ করুন।
4. পয়েন্ট স্থাপন করতে **Enter** চাপুন।

## কীবোর্ড রেফারেন্স

| কী | ক্রিয়া |
|-----|--------|
| `0`–`9`, `.`, `-` | X কোঅর্ডিনেট এন্ট্রি শুরু করুন |
| `,` | X লক করুন এবং Y এন্ট্রিতে যান |
| `Enter` | টাইপ করা কোঅর্ডিনেট নিশ্চিত করুন |
| `Escape` | বাতিল করুন |

## একটি ভিউপোর্ট সম্পাদনা

একটি ভিউপোর্ট স্থাপনের পরে, নির্বাচন করতে ক্লিক করুন:

- আকার পরিবর্তন করতে **প্রান্ত বা কোণ টেনে আনুন**।
- সরাতে **সেন্টার গ্রিপ টেনে আনুন**।
- সঠিক স্কেল সেট করতে কন্ট্রোল বারে **স্কেল সিলেক্টর** ব্যবহার করুন (যেমন ১:৫০)। তালিকায় না থাকা স্কেল এন্ট্র করতে, ড্রপডাউনের নিচে ইনপুট ফিল্ডে সরাসরি টাইপ করুন — অনুপাত ফরম্যাট (`1:200`, `5:1`) বা সাধারণ দশমিক (`0.005`) গ্রহণ করে, তারপর **Enter** চাপুন।
- দুর্ঘটনাজনিত পরিবর্তন রোধ করতে একটি ভিউপোর্টে ডান-ক্লিক করুন এবং **Lock** ব্যবহার করুন।

## নোট

- ViewportRectangle শুধুমাত্র একটি পেপার লেআউট ট্যাব সক্রিয় থাকলে উপলব্ধ। মডেল স্পেসে চালালে একটি ত্রুটি বার্তা দেখায় এবং শেষ হয়।
- বিদ্যমান ভিউপোর্ট কপি করতে, [ViewportCopy](../viewport-copy/) ব্যবহার করুন।
