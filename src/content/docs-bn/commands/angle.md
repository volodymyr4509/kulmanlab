# Angle

`angle` কমান্ড তিনটি ক্লিক করা পয়েন্টের মাধ্যমে দুটি লাইন সেগমেন্ট দ্বারা গঠিত একটি শীর্ষবিন্দুতে অভ্যন্তরীণ কোণ পরিমাপ করে। ফলাফল — সর্বদা 0° এবং 180°-এর মধ্যে — 4 দশমিক স্থানে টার্মিনালে প্রদর্শিত হয়। এটি তিনটি পরিমাপ কমান্ডের একটি — [Distance](../distance/) সরল-রেখার দৈর্ঘ্য পরিমাপ করে, এবং [Area](../area/) একটি বহুভুজের ক্ষেত্রফল ও পরিসীমা পরিমাপ করে।

## একটি কোণ পরিমাপের গঠন

```
  ● প্রথম পয়েন্ট (প্রথম রশ্মির প্রান্ত)
   \
    \  প্রথম রশ্মির পূর্বদর্শন
     \
      ● শীর্ষবিন্দু (ধাপ 3)
     /
    /  দ্বিতীয় রশ্মির পূর্বদর্শন (কার্সারে)
   /
  ● তৃতীয় পয়েন্ট  →  টার্মিনাল: "Angle: 45.0000°"
```

- **প্রথম পয়েন্ট** — কোণের একটি প্রান্ত (ধাপ 2)।
- **শীর্ষবিন্দু** — যে কোণে কোণটি পরিমাপ করা হয় (ধাপ 3)।
- **তৃতীয় পয়েন্ট** — কোণের অন্য প্রান্ত (ধাপ 4)।

## একটি কোণ পরিমাপ করা

1. টার্মিনালে `angle` টাইপ করুন বা **Angle** টুলবার বোতামে ক্লিক করুন।
2. **প্রথম পয়েন্টে ক্লিক করুন** — কোণের একটি বাহু প্রান্ত। অথবা সঠিক স্থানাঙ্কের জন্য `X,Y` টাইপ করুন এবং **Enter** চাপুন।
3. **শীর্ষবিন্দুতে ক্লিক করুন** — যেখানে দুটি বাহু মিলিত হয়। স্থানাঙ্ক এন্ট্রিও এখানে কাজ করে।
4. **তৃতীয় পয়েন্টে ক্লিক করুন** — দ্বিতীয় বাহু প্রান্ত। স্থানাঙ্ক এন্ট্রিও এখানে কাজ করে। এই পয়েন্ট স্থাপন করলে ফলাফল মুদ্রণ হয়।
5. **আবার ক্লিক করুন** (ঐচ্ছিক) সেই ক্লিককে নতুন প্রথম পয়েন্ট হিসেবে একটি নতুন পরিমাপ শুরু করতে।

## অভ্যন্তরীণ কোণ নিয়ম

কমান্ডটি শীর্ষবিন্দু থেকে দুটি রশ্মির ডট গুণফল ব্যবহার করে কোণ গণনা করে:

- **সর্বদা অভ্যন্তরীণ**: ফলাফল হল ছোট কোণ, 0° এবং 180°-এর মধ্যে।
- আপনি প্রান্ত পয়েন্টগুলি যে ক্রমে ক্লিক করেন তা ফলাফলকে প্রভাবিত করে না — শুধুমাত্র শীর্ষবিন্দু অবস্থান গুরুত্বপূর্ণ।
- সরেখ পয়েন্ট (সব তিনটি একটি লাইনে) 0° বা 180° দেয়।

## পরিমাপ শৃঙ্খলিত করা

ফলাফল প্রদর্শিত হওয়ার পরে, অবিলম্বে ক্লিক করলে পরবর্তী পরিমাপ শুরু হয় — ক্লিক করা পয়েন্টটি নতুন প্রথম পয়েন্ট হয়। আপনি `Escape` চাপা না পর্যন্ত কমান্ডটি কখনও স্বয়ংক্রিয়ভাবে প্রস্থান করে না।

## Angle বনাম Distance

| | Angle | Distance |
|---|-------|---------|
| এটি কী পরিমাপ করে | একটি শীর্ষবিন্দুতে অভ্যন্তরীণ কোণ | সরল-রেখার দৈর্ঘ্য |
| ক্লিকের সংখ্যা | 3 | 2 |
| ফলাফলের বিন্যাস | `45.0000°` | `12.3456` (একক) |
| ক্যানভাস পূর্বদর্শন | শীর্ষবিন্দু থেকে উভয় প্রান্তে দুটি লাইন | প্রথম পয়েন্ট থেকে কার্সারে লাইন |
| সবচেয়ে ভালো | দুটি বৈশিষ্ট্যের মধ্যে খোলার কোণ | একটি ফাঁক বা সেগমেন্টের দৈর্ঘ্য |

## স্থানাঙ্ক এন্ট্রি

ক্লিক করার পরিবর্তে, তিনটি পয়েন্টের যেকোনো একটির জন্য একটি সঠিক অবস্থান টাইপ করুন:

1. X মান টাইপ করুন।
2. `,` চাপুন — টার্মিনাল দেখায় `[X], [Y{cursor}]`।
3. Y মান টাইপ করুন।
4. নিশ্চিত করতে **Enter** চাপুন।

## কীবোর্ড রেফারেন্স

| কী | ক্রিয়া |
|-----|--------|
| `0`–`9`, `.`, `-` | X স্থানাঙ্ক এন্ট্রি শুরু করুন |
| `,` | X লক করুন এবং Y এন্ট্রিতে যান |
| `Backspace` | শেষ টাইপ করা অক্ষর মুছুন |
| `Enter` | টাইপ করা স্থানাঙ্ক নিশ্চিত করুন |
| `Escape` | বাতিল করুন এবং ধাপ 2-এ রিসেট করুন |

## নোট

- ফলাফল শুধুমাত্র **টার্মিনালে** দেখানো হয় — ড্রয়িংয়ে কিছু যোগ করা হয় না।
- নির্ভুলতা সর্বদা ডিগ্রিতে 4 দশমিক স্থান।
