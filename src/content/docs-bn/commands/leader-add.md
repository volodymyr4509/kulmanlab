# Leader+

`leader+` কমান্ড একটি বিদ্যমান মাল্টিলিডারে নতুন তীরমুখ আর্ম যোগ করে। নতুন আর্মটি লিডারের বিদ্যমান ডগলেগ থেকে আপনার ক্লিক করা নতুন তীরমুখ টিপের দিকে নির্দেশ করে। সমস্ত স্টাইলিং — ডগলেগ পজিশন, টেক্সট, তীরমুখের ধরন এবং আকার — নির্বাচিত লিডার থেকে উত্তরাধিকারসূত্রে পাওয়া যায়।

## একটি আর্ম যোগ করা

1. টার্মিনালে `leader+` টাইপ করুন।
2. এটি নির্বাচন করতে **একটি বিদ্যমান মাল্টিলিডারে ক্লিক করুন**।
3. **নতুন তীরমুখ টিপে ক্লিক করুন**, বা সঠিক কোঅর্ডিনেটের জন্য `X,Y` টাইপ করুন এবং **Enter** চাপুন। কার্সর থেকে লিডারের ডগলেগ পর্যন্ত একটি প্রিভিউ লাইন দেখায়।

আর্মটি স্থাপন হয় এবং কমান্ড সক্রিয় থাকে — আপনি সঙ্গে সঙ্গে আরো আর্ম যোগ করতে অন্য লিডারে ক্লিক করতে পারেন। শেষ করতে **Enter**, **Space**, বা **Escape** চাপুন।

```
  আগে:                          পরে:
  ◄── আর্ম ১                    ◄── আর্ম ১
       \                               \
        ●──── ডগলেগ ──── টেক্সট         ●──── ডগলেগ ──── টেক্সট
                                        /
                                   আর্ম ২ ──►  (আপনার ক্লিক করা নতুন টিপ)
```

## টিপের জন্য কোঅর্ডিনেট এন্ট্রি

ক্লিক করার বদলে, আপনি একটি সঠিক পজিশন টাইপ করতে পারেন:

1. X মান টাইপ করুন।
2. `,` চাপুন — টার্মিনাল X লক হয়েছে বলে নিশ্চিত করে।
3. Y মান টাইপ করুন।
4. স্থাপন করতে **Enter** চাপুন।

## কীবোর্ড রেফারেন্স

| কী | ক্রিয়া |
|-----|--------|
| `0`–`9`, `.`, `-` | X কোঅর্ডিনেট টাইপ শুরু করুন |
| `,` | X লক করুন এবং Y এন্ট্রিতে যান |
| `Enter` | টাইপ করা কোঅর্ডিনেট নিশ্চিত করুন এবং আর্ম স্থাপন করুন |
| `Enter` / `Space` | শেষ করুন (যখন কোনো ইনপুট চলছে না) |
| `Escape` | বাতিল করুন এবং রিসেট করুন |

## নোট

- শুধুমাত্র **Multileader** এন্টিটি নির্বাচন করা যাবে — অন্য কোনো এন্টিটি টাইপে ক্লিক করলে কিছুই হয় না।
- নতুন আর্মটি বিদ্যমান ডগলেগ থেকে শুরু হয়; আপনি শুধু তীরমুখ টিপ কোথায় যাবে তা বেছে নেন।
- একটি মাল্টিলিডারে আর্মের সংখ্যার কোনো সীমা নেই।
