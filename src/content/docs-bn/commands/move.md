# Move

`move` কমান্ড নির্বাচিত এন্টিটিগুলিকে একটি বেস পয়েন্ট থেকে গন্তব্য পয়েন্টে স্থানান্তর করে। প্রতিটি নির্বাচিত এন্টিটিতে প্রয়োগ করা বিচ্যুতি হলো বেস থেকে গন্তব্য পর্যন্ত ভেক্টর। মুভের পরে সমস্ত এন্টিটি তাদের নতুন পজিশনে নির্বাচিত থাকে, আরো সম্পাদনার জন্য প্রস্তুত।

## শুরু করার দুটি উপায়

**আগে নির্বাচন করুন, তারপর সরান** — প্রথমে এন্টিটি নির্বাচন করুন, তারপর সক্রিয় করুন:

1. ক্যানভাসে এক বা একাধিক এন্টিটি নির্বাচন করুন।
2. টার্মিনালে `move` টাইপ করুন বা **Move** টুলবার বাটনে ক্লিক করুন।
3. **বেস পয়েন্টে ক্লিক করুন**, বা সঠিক কোঅর্ডিনেটের জন্য `X,Y` টাইপ করুন এবং **Enter** চাপুন।
4. **গন্তব্যে ক্লিক করুন** — সমস্ত নির্বাচিত এন্টিটি বেস→গন্তব্য ভেক্টর দ্বারা স্থানান্তরিত হয়। কোঅর্ডিনেট এন্ট্রি এখানেও কাজ করে।

**সক্রিয় করুন, তারপর নির্বাচন করুন** — কিছু নির্বাচন ছাড়া কমান্ড শুরু করুন:

1. `move` টাইপ করুন বা টুলবার বাটনে ক্লিক করুন।
2. **অবজেক্ট নির্বাচন করুন** — পৃথক এন্টিটি টগল করতে ক্লিক করুন, বা এলাকা দ্বারা নির্বাচন করতে টেনে আনুন।
3. নির্বাচন নিশ্চিত করতে **Enter** বা **Space** চাপুন।
4. **বেস পয়েন্টে ক্লিক করুন**, তারপর **গন্তব্যে ক্লিক করুন** (উভয় ধাপে কোঅর্ডিনেট এন্ট্রি উপলব্ধ)।

```
  আগে:                    পরে:
  ● বেস                       → ● গন্তব্য
  [এন্টিটি A]                      [এন্টিটি A সরানো]
  [এন্টিটি B]                      [এন্টিটি B সরানো]
```

বেস পয়েন্ট থেকে গন্তব্যে সমস্ত নির্বাচিত এন্টিটির একটি ঘোস্ট প্রিভিউ কার্সর অনুসরণ করে, ক্লিকের আগে ফলাফল দেখায়।

## কোঅর্ডিনেট এন্ট্রি

বেস-পয়েন্ট বা গন্তব্য ধাপে, ক্লিক করার বদলে একটি সঠিক পজিশন টাইপ করুন:

1. X মান টাইপ করুন।
2. `,` চাপুন — টার্মিনাল `[X], [Y{কার্সর}]` দেখায়।
3. Y মান টাইপ করুন।
4. নিশ্চিত করতে **Enter** চাপুন।

## কোণ লকিং এবং সঠিক দূরত্ব

বেস পয়েন্ট সেট হওয়ার পরে কমান্ড একটি ৪৫° স্ন্যাপ অক্ষ (০°, ৪৫°, ৯০°, ১৩৫°, …) খোঁজে। কার্সর বেস থেকে যথেষ্ট দূরে এবং একটি গ্রিপ প্রস্থের মধ্যে অক্ষের কাছে থাকলে দিকটি **লক হয়**। লক থাকাকালীন:

- ঘোস্ট প্রিভিউ অক্ষে স্ন্যাপ করে।
- লক করা দিক বরাবর ঠিক সেই দূরত্বে যেতে একটি দূরত্ব টাইপ করুন এবং **Enter** চাপুন।
- ক্লিক করলে অক্ষে প্রজেক্ট হয়, তাই গন্তব্য সবসময় ঠিক সেই অক্ষে থাকে।

| কী | ক্রিয়া |
|-----|--------|
| `0`–`9`, `.` | দূরত্ব মানে যোগ করুন |
| `-` | নেতিবাচক দূরত্ব — অক্ষ বরাবর দিক বিপরীত করে (শুধু প্রথম অক্ষর) |
| `Backspace` | শেষ টাইপ করা অক্ষর মুছুন |
| `Enter` | টাইপ করা দূরত্বে মুভ প্রয়োগ করুন |

## কীবোর্ড রেফারেন্স

| কী | ক্রিয়া |
|-----|--------|
| `Enter` / `Space` | নির্বাচন নিশ্চিত করুন এবং বেস-পয়েন্ট পর্যায়ে যান |
| `0`–`9`, `.`, `-` | X কোঅর্ডিনেট এন্ট্রি শুরু করুন, বা কোণ-লক থাকলে দূরত্ব |
| `,` | X লক করুন এবং Y এন্ট্রিতে যান |
| `Backspace` | শেষ টাইপ করা অক্ষর মুছুন |
| `Enter` | কোঅর্ডিনেট নিশ্চিত করুন বা টাইপ করা দূরত্বে মুভ প্রয়োগ করুন |
| `Escape` | বাতিল করুন এবং রিসেট করুন |

## একটি গ্রিপ থেকে Move সক্রিয় করা

নির্বাচিত [Line](../line/) এর **মধ্যবিন্দু গ্রিপে** ক্লিক করলে স্বয়ংক্রিয়ভাবে Move চালু হয়, মধ্যবিন্দু ইতিমধ্যে বেস পয়েন্ট হিসেবে সেট করা এবং মুভ পর্যায় সক্রিয়। নির্বাচন ধাপ না করে একটি একক লাইন পুনরায় স্থাপন করার এটি দ্রুততম উপায়।

## কমান্ড চলাকালীন নির্বাচন

কমান্ড নির্বাচন পর্যায়ে শুরু হলে:

| পদ্ধতি | আচরণ |
|--------|-----------|
| **ক্লিক** | কার্সরের নিচের এন্টিটি নির্বাচনে যোগ/বাদ দেয় |
| **ডানে টানুন** (কঠোর) | বাক্সের সম্পূর্ণ ভেতরে এন্টিটি যোগ করে |
| **বামে টানুন** (ক্রসিং) | বাক্সের সীমানা অতিক্রমকারী এন্টিটি যোগ করে |
| **Enter** / **Space** | নির্বাচন নিশ্চিত করে এবং বেস-পয়েন্ট পর্যায়ে যায় |

## মুভের পরে

সরানো এন্টিটিগুলি তাদের নতুন পজিশনে নির্বাচিত থাকে। এর মানে আপনি সঙ্গে সঙ্গে পারেন:
- তাদের আরো সরাতে আবার **Move** চালান।
- পুনরায় নির্বাচন না করে [Copy](../copy/), [Rotate](../rotate/), বা [Scale](../scale/) চালান।
- সেগুলি সরাতে **Delete** চাপুন।

## সমর্থিত এন্টিটি

Move প্রতিটি এন্টিটি টাইপে কাজ করে: Line, Polyline, Rectangle, Circle, Arc, Ellipse, Text, Spline, Dimension, Leader এবং অন্য সব।
