# Rotate

`rotate` কমান্ড নির্বাচিত এন্টিটিগুলিকে একটি বেস পয়েন্টের চারপাশে ঘোরায়। আপনি ঘূর্ণন কোণ ডিগ্রিতে সংখ্যা টাইপ করে বা ক্লিক করে নির্দিষ্ট করেন — কোণটি বেস পয়েন্ট এবং ক্লিক পজিশনের মধ্যবর্তী দিক থেকে গণনা করা হয়।

## শুরু করার দুটি উপায়

**আগে নির্বাচন করুন, তারপর ঘোরান** — প্রথমে এন্টিটি নির্বাচন করুন, তারপর সক্রিয় করুন:

1. ক্যানভাসে এক বা একাধিক এন্টিটি নির্বাচন করুন।
2. টার্মিনালে `rotate` টাইপ করুন বা **Rotate** টুলবার বাটনে ক্লিক করুন।
3. **বেস পয়েন্টে ক্লিক করুন** — ঘূর্ণনের কেন্দ্র। বা সঠিক কোঅর্ডিনেটের জন্য `X,Y` টাইপ করুন এবং **Enter** চাপুন।
4. **একটি কোণ টাইপ করুন এবং Enter চাপুন**, বা কার্সরের দিক থেকে কোণ সেট করতে **ক্লিক করুন**।

**সক্রিয় করুন, তারপর নির্বাচন করুন** — কিছু নির্বাচন ছাড়া কমান্ড শুরু করুন:

1. `rotate` টাইপ করুন বা টুলবার বাটনে ক্লিক করুন।
2. **অবজেক্ট নির্বাচন করুন** — টগল করতে ক্লিক করুন, বা এলাকা দ্বারা নির্বাচন করতে টেনে আনুন।
3. নির্বাচন নিশ্চিত করতে **Enter** বা **Space** চাপুন।
4. **বেস পয়েন্টে ক্লিক করুন** (কোঅর্ডিনেট এন্ট্রি উপলব্ধ), তারপর কোণ সেট করুন।

```
  আগে:             পরে (● এর চারপাশে ৯০° ঘূর্ণন):
                      ╔══╗
  ●  [এন্টিটি]   →  ● ║    ║
                      ╚══╝
```

বেস পয়েন্ট সেট হওয়ার পরে ঘুরানো এন্টিটির একটি লাইভ ঘোস্ট প্রিভিউ কার্সর কোণ অনুসরণ করে।

## কোণ সেট করা

**টাইপ করা কোণ** — বেস পয়েন্ট স্থাপনের পরে যেকোনো সময় একটি সংখ্যা (ডিগ্রিতে) টাইপ করুন। Enter চাপার আগে সামঞ্জস্য করতে থাকলে প্রিভিউ টাইপ করা কোণে স্ন্যাপ করে।

**ক্লিক কোণ** — কোনো টাইপ করা মান না থাকলে, ক্লিক করলে কোণ `atan2(কার্সরY − বেসY, কার্সরX − বেসX)` এ সমান হয় — বেস পয়েন্ট থেকে ক্লিক পর্যন্ত দিক, ডিগ্রিতে।

| কী | ক্রিয়া |
|-----|--------|
| `0`–`9`, `.` | কোণ মানে সংখ্যা যোগ করুন |
| `-` | নেতিবাচক কোণ (শুধু প্রথম অক্ষর) |
| `Backspace` | শেষ টাইপ করা অক্ষর মুছুন |
| `Enter` | টাইপ করা কোণে ঘূর্ণন প্রয়োগ করুন |

## কোণের দিক

কোণগুলি **DXF কনভেনশন** অনুসরণ করে:

- **ধনাত্মক** মান ড্রয়িং কোঅর্ডিনেটে (Y-আপ) **ঘড়ির বিপরীতে** ঘোরায়।
- স্ক্রিনে, যেখানে Y-অক্ষ উল্টানো (Y-ডাউন), ধনাত্মক কোণ **ঘড়ির দিকে** দেখায়।

সাধারণ মান: `90` = কোয়ার্টার টার্ন, `180` = হাফ টার্ন, `-90` = বিপরীত কোয়ার্টার টার্ন।

## কীবোর্ড রেফারেন্স

| কী | ক্রিয়া |
|-----|--------|
| `Enter` / `Space` | নির্বাচন নিশ্চিত করুন |
| `0`–`9`, `.`, `-` | X কোঅর্ডিনেট এন্ট্রি শুরু করুন (বেস-পয়েন্ট পর্যায়), বা কোণ মান (কোণ পর্যায়) |
| `,` | X লক করুন এবং Y এন্ট্রিতে যান (বেস-পয়েন্ট পর্যায়) |
| `Backspace` | শেষ টাইপ করা অক্ষর মুছুন |
| `Enter` | কোঅর্ডিনেট নিশ্চিত করুন বা ঘূর্ণন প্রয়োগ করুন |
| `Escape` | বাতিল করুন এবং রিসেট করুন |

## কমান্ড চলাকালীন নির্বাচন

| পদ্ধতি | আচরণ |
|--------|-----------|
| **ক্লিক** | কার্সরের নিচের এন্টিটি টগল করে |
| **ডানে টানুন** (কঠোর) | বাক্সের সম্পূর্ণ ভেতরে এন্টিটি যোগ করে |
| **বামে টানুন** (ক্রসিং) | বাক্সের সাথে ছেদকারী এন্টিটি যোগ করে |
| **Enter** / **Space** | নির্বাচন নিশ্চিত করে |

## সমর্থিত এন্টিটি

Rotate প্রতিটি এন্টিটি টাইপে কাজ করে। প্রতিটি এন্টিটির জ্যামিতি বেস পয়েন্টের চারপাশে ঘোরানো হয় — উদাহরণস্বরূপ, একটি Circle তার কেন্দ্র সরায় যখন ব্যাসার্ধ একই থাকে; একটি Arc তার কেন্দ্র সরায় এবং ঘূর্ণনের পরিমাণ দ্বারা শুরু ও শেষ কোণ সরায়; একটি Text এন্টিটি তার অ্যাঙ্কর পয়েন্ট সরায় এবং তার Rotation Degree প্রপার্টিতে কোণ যোগ করে।
