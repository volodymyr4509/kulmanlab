# Dimension Aligned

`dimaligned` কমান্ড একটি মাত্রা স্থাপন করে যা দুটি পয়েন্টের মধ্যে **সত্যিকারের সরল-রেখার দূরত্ব** পরিমাপ করে। মাত্রা রেখা দুটি পয়েন্ট সংযোগকারী রেখার সমান্তরালে চলে, তাই এটি যেকোনো কোণে থাকতে পারে। এটি [Dimension Linear](../dim-linear/)-এর সাথে মূল পার্থক্য, যা অনুভূমিক বা উল্লম্বে সীমাবদ্ধ।

## একটি সারিবদ্ধ মাত্রার গঠন

```
     ●  p2
    /|
   / |  (এক্সট লাইন 2, মাত্রা রেখায় লম্ব)
  /  |
 /←5.00→/
/  /
●  /  (এক্সট লাইন 1, মাত্রা রেখায় লম্ব)
p1
```

- **এক্সটেনশন লাইন** — মাত্রা রেখায় লম্ব, প্রতিটি পরিমাপ করা পয়েন্ট থেকে আঁকা।
- **মাত্রা রেখা** — p1→p2-এর সমান্তরাল, কার্সার অবস্থান দ্বারা একপাশে অফসেট করা।
- **মান** — সত্যিকারের ইউক্লিডিয়ান দূরত্ব `|p1 – p2|`।

## একটি সারিবদ্ধ মাত্রা স্থাপন করা

1. টার্মিনালে `dimaligned` টাইপ করুন বা **Dimension Aligned** টুলবার বোতামে ক্লিক করুন।
2. **প্রথম এক্সটেনশন লাইন মূলে ক্লিক করুন** (p1), অথবা সঠিক স্থানাঙ্কের জন্য `X,Y` টাইপ করুন এবং **Enter** চাপুন।
3. **দ্বিতীয় এক্সটেনশন লাইন মূলে ক্লিক করুন** (p2)। স্থানাঙ্ক এন্ট্রিও এখানে কাজ করে।
4. মাত্রা রেখার লম্ব অফসেট সেট করতে **কার্সার সরান**।
5. স্থাপন করতে **ক্লিক করুন**, বা সঠিক স্থাপনার জন্য একটি অফসেট দূরত্ব টাইপ করুন এবং **Enter** চাপুন।

## টাইপ করা অফসেট দূরত্ব

p1→p2 রেখা থেকে সঠিক লম্ব দূরত্বে মাত্রা রেখা স্থির করতে স্থাপনের সময় একটি সংখ্যা টাইপ করুন:

| কী | ক্রিয়া |
|-----|--------|
| `0`–`9`, `.` | অফসেটে অঙ্ক যোগ করুন |
| `Backspace` | শেষ টাইপ করা অক্ষর মুছুন |
| `Enter` / `Space` | টাইপ করা অফসেটে স্থাপন করুন |

কার্সার পাশ নির্ধারণ করে মাত্রা রেখা কোন পাশে দেখা যাবে।

## কীবোর্ড রেফারেন্স

| কী | ক্রিয়া |
|-----|--------|
| `0`–`9`, `.`, `-` | X স্থানাঙ্ক এন্ট্রি শুরু করুন (p1/p2 পর্যায়), বা অফসেট দূরত্ব (স্থাপন পর্যায়) |
| `,` | X লক করুন এবং Y এন্ট্রিতে যান (p1/p2 পর্যায়) |
| `Backspace` | শেষ টাইপ করা অক্ষর মুছুন |
| `Enter` / `Space` | টাইপ করা স্থানাঙ্ক বা অফসেট নিশ্চিত করুন |
| `Escape` | বাতিল করুন |

## Dimension Aligned বনাম Dimension Linear

| | Dimension Aligned | Dimension Linear |
|---|------------------|-----------------|
| মাত্রা রেখার কোণ | p1→p2-এর সমান্তরাল — যেকোনো কোণ | সর্বদা অনুভূমিক বা উল্লম্ব |
| পরিমাপ করে | সত্যিকারের ইউক্লিডিয়ান দূরত্ব | শুধুমাত্র X বা Y উপাদান |
| H/V অভিমুখ লক | না | হ্যাঁ — `H` এবং `V` কী |
| সবচেয়ে ভালো | তির্যক বৈশিষ্ট্য, কোণীয় কাটা | অর্থোগোনাল লেআউট, গ্রিড-সারিবদ্ধ অংশ |

## লেবেল সম্পাদনা — সরল মোড

স্থাপিত সারিবদ্ধ মাত্রায় **ডাবল-ক্লিক** করুন **সরল** মোডে টেক্সট এডিটর খুলতে। এডিটরটি বর্তমান রেন্ডার করা মান দিয়ে পূর্ব-পূরণ করা থাকে যাতে আপনি কার্সার অবস্থান করে সরাসরি সম্পাদনা করতে পারেন।

| বৈশিষ্ট্য | আচরণ |
|---------|-----------|
| Bold / Italic / Font / Height | **সম্পূর্ণ** লেবেলে একসাথে প্রযোজ্য |
| প্রতি-অক্ষর ফরম্যাটিং | সমর্থিত নয় |
| `Enter` | মান কমিট করে এবং এডিটর বন্ধ করে |
| বহু-লাইন | সমর্থিত নয় |

সম্পূর্ণ রেফারেন্সের জন্য [Text Editor — simple mode](../../interface/text-editor/#simple-mode) দেখুন।

## মাত্রা শৃঙ্খলিত করা

এই মাত্রার দ্বিতীয় এক্সটেনশন লাইন থেকে চলমান আরও মাত্রা যোগ করতে, [Dimension Continue](../dim-continue/) ব্যবহার করুন — এটি এই সারিবদ্ধ মাত্রার মতো একই পরিমাপ কোণে লক করে।

## DXF — DIMENSION সত্তা (সারিবদ্ধ ধরন)

সারিবদ্ধ মাত্রাগুলি `dimType = 1` (সারিবদ্ধ) সহ `DIMENSION` সত্তা হিসেবে সংরক্ষিত হয়। এক্সটেনশন লাইন মূল, মাত্রা রেখা অবস্থান, টেক্সট অবস্থান, পরিমাপ করা মান, ঘূর্ণন, তীরচিহ্নের শৈলী এবং সমস্ত প্রদর্শন ফ্ল্যাগ কোনো ক্ষতি ছাড়াই রাউন্ড-ট্রিপ করে।
