# Dimension Continue

`dimcontinue` কমান্ড একটি বিদ্যমান মাত্রার **দ্বিতীয় এক্সটেনশন লাইন** থেকে নতুন মাত্রা শৃঙ্খলিত করে। প্রতিটি নতুন সেগমেন্ট বেসের মতো একই পরিমাপ অক্ষ এবং একই মাত্রা-লাইন অফসেটে স্থাপিত হয়। সমস্ত স্টাইল বৈশিষ্ট্য — তীরচিহ্নের আকার, টেক্সট উচ্চতা, এক্সটেনশন লাইনের দৈর্ঘ্য — বেস থেকে স্বয়ংক্রিয়ভাবে কপি করা হয়।

## শৃঙ্খলিত মাত্রাগুলি কেমন দেখায়

```
  |←— 3.00 —→|←— 2.50 —→|←— 4.00 —→|
  |           |           |           |
  ●           ●           ●           ●
  p1        p2 (বেস    p3           p4
           ext2 → নতুন শুরু)
```

প্রতিটি আয়তক্ষেত্র একটি পৃথক `DIMENSION` সত্তা। তারা একই মাত্রা-লাইন অবস্থান এবং পরিমাপ দিক ভাগ করে।

## একটি শৃঙ্খল শুরু করা

1. টার্মিনালে `dimcontinue` টাইপ করুন বা **Dimension Continue** টুলবার বোতামে ক্লিক করুন।
2. **যদি একটি মাত্রা সবেমাত্র স্থাপিত হয়** — কমান্ড স্বয়ংক্রিয়ভাবে এটি বেস হিসেবে নেয় (কোনো ক্লিকের প্রয়োজন নেই)।
3. **যদি কোনো সাম্প্রতিক মাত্রা না থাকে** — এটি বেস হিসেবে ব্যবহার করতে যেকোনো বিদ্যমান মাত্রায় ক্লিক করুন।
4. **পরবর্তী এক্সটেনশন লাইন মূলে ক্লিক করুন** — কার্সার সরানোর সাথে একটি পূর্বদর্শন নতুন মাত্রা দেখায়। অথবা সঠিক স্থানাঙ্কের জন্য `X,Y` টাইপ করুন এবং **Enter** চাপুন।
5. শৃঙ্খল প্রসারিত করতে ক্লিক (বা টাইপ) করতে থাকুন। প্রতিটি স্থাপিত মাত্রা স্বয়ংক্রিয়ভাবে নতুন বেস হয়।
6. শৃঙ্খল শেষ করতে **Enter**, **Space**, বা **Escape** চাপুন।

## বেস মাত্রা থেকে কী উত্তরাধিকার করা হয়

| বৈশিষ্ট্য | বেস থেকে উত্তরাধিকার |
|----------|---------------------|
| পরিমাপ দিক / কোণ | হ্যাঁ — পুরো শৃঙ্খলের জন্য লক করা |
| মাত্রা লাইন অফসেট (পরিমাপ পয়েন্ট থেকে দূরত্ব) | হ্যাঁ |
| তীরচিহ্নের আকার | হ্যাঁ |
| টেক্সট উচ্চতা | হ্যাঁ |
| এক্সটেনশন লাইন অফসেট এবং এক্সটেনশন | হ্যাঁ |
| টেক্সট সারিবদ্ধতা | হ্যাঁ |
| স্টাইলের নাম | হ্যাঁ |
| Color, Layer | উত্তরাধিকার করা হয় না — বর্তমান লেয়ার ব্যবহার করে |

## পরিমাপ দিক লকিং

শৃঙ্খলের পরিমাপ দিক **বেস মাত্রার কোণে স্থির**:

- রৈখিক বেস (H) → সমস্ত ধারাবাহিকতা অনুভূমিক দূরত্ব (Δ X) পরিমাপ করে।
- রৈখিক বেস (V) → সমস্ত ধারাবাহিকতা উল্লম্ব দূরত্ব (Δ Y) পরিমাপ করে।
- যেকোনো কোণে সারিবদ্ধ বেস → সমস্ত ধারাবাহিকতা সেই একই কোণ বরাবর পরিমাপ করে।

আপনি মধ্য-শৃঙ্খলে দিক পরিবর্তন করতে পারবেন না। ভিন্ন দিকে মাত্রা দিতে একটি নতুন [Dimension Linear](../dim-linear/) বা [Dimension Aligned](../dim-aligned/) শুরু করুন।

## কীবোর্ড রেফারেন্স

| কী | ক্রিয়া |
|-----|--------|
| `0`–`9`, `.`, `-` | X স্থানাঙ্ক এন্ট্রি শুরু করুন |
| `,` | X লক করুন এবং Y এন্ট্রিতে যান |
| `Backspace` | শেষ টাইপ করা অক্ষর মুছুন |
| `Enter` | টাইপ করা স্থানাঙ্ক নিশ্চিত করুন, বা কোনো ইনপুট চলছে না থাকলে শৃঙ্খল শেষ করুন |
| `Space` / `Escape` | শৃঙ্খল শেষ করুন |

## Dimension Continue বনাম নতুনভাবে শুরু করা

| | Dimension Continue | Dimension Linear / Aligned |
|---|-------------------|--------------------------|
| শুরু বিন্দু | শেষ বেসের ext2-এ স্থির | যেকোনো জায়গায় ক্লিক করুন |
| কোণ | বেসে লক | মুক্ত |
| অফসেট | বেস থেকে উত্তরাধিকার | কার্সার বা টাইপ করা দ্বারা সেট |
| স্টাইল | বেস থেকে উত্তরাধিকার | বর্তমান স্টাইল |
| সবচেয়ে ভালো | একটি সারিতে ক্রমবর্ধমান পরিমাপ | প্রথম মাত্রা বা দিক পরিবর্তন |

## স্থাপনার পরে লেবেল সম্পাদনা — সরল মোড

শৃঙ্খলের যেকোনো মাত্রায় **ডাবল-ক্লিক** করুন **সরল** মোডে টেক্সট এডিটর খুলতে। প্রতিটি সেগমেন্ট স্বাধীন এবং পৃথকভাবে সম্পাদনা করা যায়।

| বৈশিষ্ট্য | আচরণ |
|---------|-----------|
| Bold / Italic / Font / Height | **সম্পূর্ণ** লেবেলে একসাথে প্রযোজ্য |
| প্রতি-অক্ষর ফরম্যাটিং | সমর্থিত নয় |
| `Enter` | মান কমিট করে এবং এডিটর বন্ধ করে |
| বহু-লাইন | সমর্থিত নয় |

সম্পূর্ণ রেফারেন্সের জন্য [Text Editor — simple mode](../../interface/text-editor/#simple-mode) দেখুন।

## DXF — DIMENSION সত্তা

শৃঙ্খলের প্রতিটি সেগমেন্ট DXF ফাইলে একটি স্বাধীন `DIMENSION` সত্তা হিসেবে সংরক্ষিত। এগুলি ফাইলে সংযুক্ত নয় — তারা একই বেস থেকে তৈরি হওয়ায় বৈশিষ্ট্য ভাগ করে, কিন্তু প্রতিটি স্থাপনার পরে পৃথকভাবে সম্পাদনা করা যায়।
