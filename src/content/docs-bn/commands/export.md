# Export

**Export** কমান্ড বর্তমান ড্রয়িং আপনার ফাইল সিস্টেমে ডাউনলোড করে। দুটি ফরম্যাট উপলব্ধ: অন্যান্য CAD টুলের সাথে সামঞ্জস্যের জন্য **DXF** এবং KulmanLab CAD-এর মধ্যে সম্পূর্ণ-বিশ্বস্ততার সেভের জন্য **JSON**।

## কীভাবে এক্সপোর্ট করবেন

1. File প্যানেলে **Export** টুলবার বোতামে (ডাউনলোড আইকন) ক্লিক করুন।
2. **Export Manager** পপআপ খোলে।
3. ফরম্যাট নির্বাচন করতে একটি ফরম্যাট কার্ডে ক্লিক করুন — **JSON** বা **DXF**।
4. **Export** বোতামে ক্লিক করুন। ফাইলটি স্বয়ংক্রিয়ভাবে আপনার ডিফল্ট ডাউনলোড ফোল্ডারে ডাউনলোড হয়।

## একটি ফরম্যাট বেছে নেওয়া

| ফরম্যাট | এক্সটেনশন | সবচেয়ে ভালো | সীমাবদ্ধতা |
|--------|-----------|----------|-------------|
| **JSON** *(নেটিভ)* | `.json` | KulmanLab CAD-এ পুনরায় খোলার জন্য কাজ সেভ করা | অন্যান্য CAD টুলের সাথে সামঞ্জস্যপূর্ণ নয় |
| **DXF** | `.dxf` | FreeCAD, LibreCAD, ইত্যাদির সাথে শেয়ার করা | মাত্রা এবং লিডার এক্সপোর্ট করা হয় না |

**JSON কখন ব্যবহার করবেন:** যখনই আপনি আপনার কাজের একটি সম্পূর্ণ কপি সেভ করতে চান। JSON হল KulmanLab-এর নেটিভ ফরম্যাট এবং প্রতিটি সত্তা ঠিক সংরক্ষণ করে — মাত্রা, লিডার এবং সমস্ত লেয়ার ডেটা সহ।

**DXF কখন ব্যবহার করবেন:** যখন আপনাকে অন্য CAD অ্যাপ্লিকেশন ব্যবহার করছেন এমন কাউকে ড্রয়িং হ্যান্ড অফ করতে হবে। এক্সপোর্ট করা ফাইল AC1012 DXF ফরম্যাট ব্যবহার করে এবং বেশিরভাগ DXF-সামঞ্জস্যপূর্ণ টুলে খোলা যায়।

## প্রতিটি ফরম্যাটে কী এক্সপোর্ট করা হয়

### JSON এক্সপোর্ট

সমস্ত সত্তার ধরন অন্তর্ভুক্ত:

- লাইন, বৃত্ত, আর্ক, উপবৃত্ত, পলিলাইন, স্প্লাইন, টেক্সট
- মাত্রা (রৈখিক, সারিবদ্ধ, অব্যাহত, ব্যাসার্ধ, ব্যাস)
- মাল্টিলিডার
- লেয়ার সংজ্ঞা এবং লাইনটাইপ টেবিল

### DXF এক্সপোর্ট

জ্যামিতি-শুধুমাত্র সত্তা অন্তর্ভুক্ত:

- লাইন, বৃত্ত, আর্ক, উপবৃত্ত, পলিলাইন (`LWPOLYLINE` হিসেবে এক্সপোর্ট), স্প্লাইন, টেক্সট
- লেয়ার সংজ্ঞা এবং লাইনটাইপ টেবিল

**DXF-এ অন্তর্ভুক্ত নয়:** মাত্রা সত্তা এবং মাল্টিলিডার। এগুলি KulmanLab-নির্দিষ্ট ডেটা কাঠামো ব্যবহার করে যা স্ট্যান্ডার্ড DXF-এ বিশ্বস্তভাবে প্রতিনিধিত্ব করা যায় না। যদি আপনার ড্রয়িংয়ে টীকা থাকে, JSON ব্যবহার করুন বা সেগুলি দৃশ্যমানভাবে ক্যাপচার করতে [Print](../print/) ব্যবহার করুন।

## এক্সপোর্ট করা ফাইলের নাম

ডাউনলোড করা ফাইলটি বর্তমান ড্রয়িং ফাইলের নামে নামকরণ করা হয় (যেমন `myplan_May22_14:30:00.json`)। এক্সটেনশন বেছে নেওয়া ফরম্যাটের সাথে মেলাতে পরিবর্তিত হয়।

## Export এবং Print-এর মধ্যে পার্থক্য

| বৈশিষ্ট্য | Export | Print |
|---------|--------|-------|
| আউটপুট | ভেক্টর উৎস ফাইল (.dxf / .json) | রাস্টার ছবি (.png / .jpeg / .webp / .pdf) |
| অন্যান্য টুলে সম্পাদনাযোগ্য | হ্যাঁ (DXF) | না |
| লেয়ার ও লাইনটাইপ সংরক্ষণ করে | হ্যাঁ | না (সমতলভাবে রেন্ডার করা) |
| মাত্রা ও লিডার ক্যাপচার করে | শুধুমাত্র JSON | হ্যাঁ |

একটি সম্পাদনাযোগ্য ফাইলের প্রয়োজন হলে **Export** ব্যবহার করুন। একটি ভিজ্যুয়াল স্ন্যাপশটের প্রয়োজন হলে [Print](../print/) ব্যবহার করুন।
