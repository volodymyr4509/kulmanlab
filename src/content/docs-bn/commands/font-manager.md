# Font Manager

`FontManager` কমান্ড ফন্ট ব্রাউজ ও নির্বাচন করার জন্য, এবং [Text](../text/) ও [Multileader](../leader/) এন্টিটিতে ব্যবহারের জন্য নিজের `.ttf` ফাইল আপলোড করার জন্য একটি ডায়ালগ খোলে।

## Font Manager খোলা

- টার্মিনালে `FontManager` টাইপ করুন, **অথবা**
- [text editor](../../interface/text-editor/) টুলবারে **Font Manager** বাটনে ক্লিক করুন।

## ফন্ট গ্রুপ

| গ্রুপ | বিষয়বস্তু |
|-------|-------------|
| **Default** | বিল্ট-ইন sans-serif ফন্ট — সবসময় উপলব্ধ |
| **User** | আপনার নিজের আপলোড করা `.ttf` ফন্ট (শুধুমাত্র একটি যোগ করার পরে দেখানো হয়) |
| **Free** | ১৫টি বান্ডিল করা Google Fonts (EB Garamond, Fira Code, Inter, Lato, Merriweather, Montserrat, Nunito, Open Sans, Oswald, Playfair Display, Poppins, Raleway, Roboto, Roboto Condensed, Source Code Pro) |
| **System** | সাধারণ OS ফন্ট (Courier New, Georgia, Helvetica, Impact, Lucida Console, Tahoma, Times New Roman, Trebuchet MS, Verdana) |

তালিকার যেকোনো ফন্টে ক্লিক করলে ডানদিকে প্রিভিউ দেখা যায় — নাম, বর্ণমালার নমুনা, একটি প্যানগ্রাম এবং সংখ্যা।

## কাস্টম ফন্ট আপলোড করা

1. ডায়ালগের নিচের অংশে **Add Font**-এ ক্লিক করুন (অথবা ফাইল পিকার সরাসরি খুলতে টার্মিনালে `Font+` টাইপ করুন)।
2. একটি `.ttf` ফাইল বেছে নিন। শুধুমাত্র TrueType ফন্ট সমর্থিত — `.otf` এবং `.woff`/`.woff2` সমর্থিত নয়।
3. ফাইলের নাম (এক্সটেনশন ছাড়া) **User** গ্রুপে ফন্টের নাম হয়ে যায়। উদাহরণস্বরূপ, `MyFont.ttf` আপলোড করলে `MyFont` নামে একটি ফন্ট যোগ হয়।

আপলোড করা ফন্টগুলি ব্রাউজারে (IndexedDB) স্থায়ীভাবে সংরক্ষিত হয় এবং আপনি পরবর্তীতে KulmanLab CAD খুললে স্বয়ংক্রিয়ভাবে পুনরায় লোড হয়।

## কাস্টম ফন্ট মুছে ফেলা

**User** গ্রুপে একটি ফন্টের উপর হোভার করুন এবং তার পাশের **×** বাটনে ক্লিক করুন। বিল্ট-ইন ফন্ট (Default, Free, System) মুছে ফেলা যায় না।

## কীবোর্ড রেফারেন্স

| কী | ক্রিয়া |
|-----|--------|
| `↑` / `↓` | ফন্ট তালিকায় নির্বাচন উপরে বা নিচে সরান |
| `Escape` | Font Manager বন্ধ করুন |

## DXF সামঞ্জস্যতা

ফন্টের নাম রপ্তানি করা **MTEXT** এন্টিটিতে একটি ইনলাইন ফরম্যাটিং কোড হিসেবে এমবেড করা থাকে, তাই KulmanLab CAD-এর মাধ্যমে রাউন্ড-ট্রিপ করা একটি DXF তার ফন্ট অ্যাসাইনমেন্ট বজায় রাখে। কাস্টম ফন্টের *ফাইল* DXF-এ এমবেড করা হয় না — শুধুমাত্র ফন্টের *নাম* এমবেড করা হয়। আপনি যদি এমন একটি ড্রয়িং পুনরায় আমদানি করেন যা এমন একটি কাস্টম ফন্টের রেফারেন্স দেয় যা আপনি এই ডিভাইসে আপলোড করেননি, তাহলে আপনি একই নামের ফন্ট আপলোড না করা পর্যন্ত টেক্সট ডিফল্ট ফন্টে রেন্ডার হবে।
