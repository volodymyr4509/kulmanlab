# Line

`line` কমান্ড পৃথক সরল-রেখা সেগমেন্ট আঁকে যা DXF মডেলে আলাদা `LINE` এন্টিটি হিসেবে সংরক্ষিত হয়। প্রতিটি সেগমেন্টের পরে কমান্ড সক্রিয় থাকে এবং শেষ পয়েন্টকে নতুন শুরু পয়েন্ট হিসেবে পুনরায় ব্যবহার করে, তাই আপনি একটি একটি সেগমেন্ট দিয়ে সংযুক্ত পথ তৈরি করতে পারেন। [Polyline](../polyline/) এর বিপরীতে, চেইন করা লাইনগুলি স্বতন্ত্র এন্টিটি থাকে — প্রতিটি প্রতিবেশীদের প্রভাবিত না করে ট্রিম, প্রসারিত বা মুছে ফেলা যায়।

## লাইন আঁকা

1. টার্মিনালে `line` টাইপ করুন বা **Line** টুলবার বাটনে ক্লিক করুন।
2. **শুরু পয়েন্টে ক্লিক করুন**, বা সঠিক কোঅর্ডিনেটের জন্য `X,Y` টাইপ করুন এবং **Enter** চাপুন।
3. **শেষ পয়েন্টে ক্লিক করুন** — সেগমেন্ট স্থাপন হয় এবং শেষ পয়েন্ট পরবর্তী শুরু পয়েন্ট হয়। কোঅর্ডিনেট এন্ট্রি এখানেও কাজ করে।
4. আরো সেগমেন্ট চেইন করতে ক্লিক (বা টাইপ) করতে থাকুন।
5. থামতে **Enter** বা **Escape** চাপুন।

```
  ●──────────●──────────●──────────●
 শুরু     ২য় ক্লিক  ৩য় ক্লিক   শেষ করতে Enter
            (স্বয়ংক্রিয়ভাবে পরবর্তী শুরু হয়)
```

শুধুমাত্র একটি সেগমেন্ট দরকার? ধাপ ৩ এর পরেই **Enter** বা **Escape** চাপুন।

## কোঅর্ডিনেট এন্ট্রি

ক্লিক করার বদলে, শুরু বা যেকোনো পরবর্তী পয়েন্টের জন্য একটি সঠিক পজিশন টাইপ করুন:

1. X মান টাইপ করুন (সংখ্যা, `.`, বা `-`)।
2. `,` চাপুন — টার্মিনাল `[X], [Y{কার্সর}]` দেখায়।
3. Y মান টাইপ করুন।
4. পয়েন্ট স্থাপন করতে **Enter** চাপুন।

## কোণ লকিং এবং সঠিক দৈর্ঘ্য ইনপুট

একটি পয়েন্ট স্থাপনের পরে কার্সর সরানোর সময়, কমান্ড একটি ৪৫° স্ন্যাপ অক্ষ (০°, ৪৫°, ৯০°, ১৩৫°, …) খোঁজে। কোণটি **লক হয়** যখন:

- কার্সর অ্যাঙ্কর থেকে কমপক্ষে **৫ × গ্রিপ সাইজ** দূরে, **এবং**
- এটি নিকটতম অক্ষ থেকে **১ গ্রিপ সাইজ** এর মধ্যে লম্ব দূরত্বে।

লক থাকলে প্রিভিউ অক্ষে স্ন্যাপ করে এবং আপনি সঠিক দৈর্ঘ্য এন্ট্র করতে পারেন:

| কী | ক্রিয়া |
|-----|--------|
| `0`–`9`, `.` | দৈর্ঘ্য মানে সংখ্যা যোগ করুন |
| `-` | নেতিবাচক দৈর্ঘ্য — অক্ষ বরাবর দিক বিপরীত করে (শুধু প্রথম অক্ষর) |
| `Backspace` | শেষ টাইপ করা অক্ষর মুছুন |
| `Enter` | টাইপ করা দূরত্বে শেষ পয়েন্ট স্থাপন করুন |

সংগৃহীত মান টার্মিনালে লাইভ দেখানো হয় (যেমন `click end point or enter length: 12.5`)। লক থাকাকালীন ক্লিক করলে ক্লিকটি অক্ষে প্রজেক্ট করা হয়, তাই শেষ পয়েন্ট সবসময় ঠিক সেই অক্ষে থাকে।

অ্যাঙ্কর পয়েন্টের কাছে ফিরে যাওয়া লকটি নিষ্ক্রিয় করে।

## কীবোর্ড রেফারেন্স

| কী | ক্রিয়া |
|-----|--------|
| `0`–`9`, `.`, `-` | X কোঅর্ডিনেট এন্ট্রি শুরু করুন, বা কোণ-লক থাকলে দূরত্ব |
| `,` | X লক করুন এবং Y এন্ট্রিতে যান |
| `Backspace` | শেষ টাইপ করা অক্ষর মুছুন |
| `Enter` | টাইপ করা কোঅর্ডিনেট বা দৈর্ঘ্য নিশ্চিত করুন, বা কিছু টাইপ না করলে চেইন শেষ করুন |
| `Escape` | চেইন শেষ করুন এবং বের হন |

## গ্রিপ সম্পাদনা — শেষ পয়েন্ট প্রসারিত করা

একটি নির্বাচিত লাইন তিনটি গ্রিপ দেখায়:

| গ্রিপ | কোথায় | কী করে |
|------|-------|--------------|
| **শুরু** | প্রথম শেষ পয়েন্ট | পুনরায় স্থাপন করতে টেনে আনুন — শেষ স্থির থাকে |
| **মধ্যবিন্দু** | লাইনের কেন্দ্র | সম্পূর্ণ লাইনের জন্য **Move** সক্রিয় করে |
| **শেষ** | দ্বিতীয় শেষ পয়েন্ট | পুনরায় স্থাপন করতে টেনে আনুন — শুরু স্থির থাকে |

একটি শেষ পয়েন্ট প্রসারিত করলে অন্যটি কখনো প্রভাবিত হয় না। এটি [Polyline](../polyline/) গ্রিপ সম্পাদনা থেকে আলাদা, যেখানে একটি ভার্টেক্স সরালে সম্পূর্ণ পথটি পুনরায় আকার পায়।

## লাইন নির্বাচন

| পদ্ধতি | আচরণ |
|--------|-----------|
| **ক্লিক** | সেগমেন্টের হিট-টেস্ট দূরত্বের মধ্যে ক্লিক হলে লাইন নির্বাচিত হয় |
| **ডানে টানুন** (কঠোর) | উভয় শেষ পয়েন্ট বাক্সের ভেতরে পড়লেই লাইন নির্বাচিত হয় |
| **বামে টানুন** (ক্রসিং) | সেগমেন্টের যেকোনো অংশ বাক্সের সীমানা অতিক্রম করলে লাইন নির্বাচিত হয় |

## প্রপার্টি

একটি লাইন নির্বাচিত হলে প্রপার্টি প্যানেলে DXF `LINE` রেকর্ডের প্রতিটি ফিল্ড দেখায়:

**সাধারণ**

| প্রপার্টি | ডিফল্ট | অর্থ |
|----------|---------|---------|
| Color | 256 (ByLayer) | ACI রঙের সূচক |
| Layer | `0` | লেয়ার অ্যাসাইনমেন্ট |
| Linetype | ByLayer | নামযুক্ত লাইনটাইপ প্যাটার্ন |
| Linetype Scale | 1 | লাইনটাইপ প্যাটার্নের স্কেল ফ্যাক্টর |
| Thickness | 0 | এক্সট্রুশন পুরুত্ব |

**জ্যামিতি**

| প্রপার্টি | অর্থ |
|----------|---------|
| Start X / Start Y | প্রথম শেষ পয়েন্ট কোঅর্ডিনেট |
| End X / End Y | দ্বিতীয় শেষ পয়েন্ট কোঅর্ডিনেট |

কমান্ড পুনরায় না চালিয়ে প্যানেলে সরাসরি সমস্ত ফিল্ড সম্পাদনযোগ্য।

## Line বনাম Polyline — কোনটি ব্যবহার করবেন

| | Line | Polyline |
|---|------|---------|
| এন্টিটির সংখ্যা | প্রতি সেগমেন্টে একটি `LINE` | সম্পূর্ণ পথের জন্য একটি `LWPOLYLINE` |
| Trim / Extend | হ্যাঁ — সেগমেন্ট অনুযায়ী | না |
| বদ্ধ আকার | না | হ্যাঁ (ক্লোজ ফ্ল্যাগ) |
| গ্রিপ সম্পাদনা | পৃথক শেষ পয়েন্ট প্রসারিত করুন | পথের সাথে যেকোনো ভার্টেক্স সরান |
| সেরা জন্য | কনস্ট্রাকশন লাইন, একক সেগমেন্ট, ট্রিম করার জ্যামিতি | কনট্যুর, আউটলাইন, সম্পূর্ণ রাখা আকার |

## DXF — LINE এন্টিটি

লাইনগুলি DXF ফাইলে `LINE` এন্টিটি হিসেবে সেভ হয়। প্রতিটি প্রপার্টি — শুরু/শেষ কোঅর্ডিনেট, রঙ, লেয়ার, লাইনটাইপ, লাইনটাইপ স্কেল এবং পুরুত্ব — ক্ষতি ছাড়াই রাউন্ড-ট্রিপ করে। যখন আপনি `LINE` এন্টিটি ধারণকারী DXF খোলেন সেগুলি এডিটরে সম্পূর্ণরূপে সম্পাদনযোগ্য `Line` অবজেক্ট হিসেবে পরিণত হয়।

এডিটরে আঁকা লাইনগুলিও সেভে `LINE` এন্টিটি হিসেবে লেখা হয়, তাই LibreCAD, FreeCAD এবং অন্য যেকোনো DXF-সামঞ্জস্যপূর্ণ অ্যাপ্লিকেশন দ্বারা পঠনযোগ্য।
