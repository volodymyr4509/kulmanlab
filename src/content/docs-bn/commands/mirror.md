# Mirror

`mirror` কমান্ড নির্বাচিত এন্টিটির মিরর করা কপি তৈরি করে দুই-পয়েন্ট অক্ষ জুড়ে প্রতিফলিত করে। মূলগুলি **সর্বদা রাখা হয়** — [Move](./move) বা [Rotate](./rotate) এর বিপরীতে, Mirror কখনো বিদ্যমান এন্টিটি পরিবর্তন করে না; এটি শুধুমাত্র নতুন যোগ করে।

## শুরু করার দুটি উপায়

**আগে নির্বাচন করুন, তারপর মিরর করুন** — প্রথমে এন্টিটি নির্বাচন করুন, তারপর সক্রিয় করুন:

1. ক্যানভাসে এক বা একাধিক এন্টিটি নির্বাচন করুন।
2. টার্মিনালে `mirror` টাইপ করুন বা **Mirror** টুলবার বাটনে ক্লিক করুন।
3. মিরর অক্ষের **প্রথম পয়েন্টে ক্লিক করুন**, বা সঠিক কোঅর্ডিনেটের জন্য `X,Y` টাইপ করুন এবং **Enter** চাপুন।
4. **দ্বিতীয় পয়েন্টে ক্লিক করুন** — মিরর করা কপি স্থাপন হয় এবং কমান্ড শেষ হয়। কোঅর্ডিনেট এন্ট্রি এখানেও কাজ করে।

**সক্রিয় করুন, তারপর নির্বাচন করুন** — কিছু নির্বাচন ছাড়া কমান্ড শুরু করুন:

1. `mirror` টাইপ করুন বা টুলবার বাটনে ক্লিক করুন।
2. **অবজেক্ট নির্বাচন করুন** — টগল করতে ক্লিক করুন, বা এলাকা দ্বারা নির্বাচন করতে টেনে আনুন।
3. নির্বাচন নিশ্চিত করতে **Enter** বা **Space** চাপুন।
4. মিরর অক্ষের **প্রথম পয়েন্টে ক্লিক করুন**, তারপর **দ্বিতীয় পয়েন্টে ক্লিক করুন** (উভয় ধাপে কোঅর্ডিনেট এন্ট্রি উপলব্ধ)।

```
  মূল:          মিরর অক্ষ:        ফলাফল:
                 |
  [এন্টিটি A]   →    |    →    [এন্টিটি A] + [মিরর করা A]
                 |
```

দ্বিতীয় অক্ষ পয়েন্ট অবস্থান করার সময় মিরর করা কপির একটি লাইভ প্রিভিউ কার্সর অনুসরণ করে।

## মিরর অক্ষ

অক্ষটি দুটি ক্লিক করা পয়েন্টের মধ্য দিয়ে যাওয়া একটি অসীম রেখা। এটি যেকোনো কোণে হতে পারে:

- কার্সরকে একটি **৪৫° স্ন্যাপ অক্ষের** (০°, ৪৫°, ৯০°, ১৩৫°, …) কাছে নিয়ে যান এবং অক্ষটি সেই কোণে লক হয় — পরিষ্কার অনুভূমিক, উল্লম্ব বা কোণাকুণি প্রতিফলনের জন্য উপযোগী।
- স্ন্যাপ জোনের বাইরে ক্লিক করুন মুক্ত-কোণ অক্ষের জন্য।

## কোঅর্ডিনেট এন্ট্রি

যেকোনো অক্ষ-পয়েন্ট ধাপে, ক্লিক করার বদলে একটি সঠিক পজিশন টাইপ করুন:

1. X মান টাইপ করুন।
2. `,` চাপুন — টার্মিনাল `[X], [Y{কার্সর}]` দেখায়।
3. Y মান টাইপ করুন।
4. নিশ্চিত করতে **Enter** চাপুন।

## কীবোর্ড রেফারেন্স

| কী | ক্রিয়া |
|-----|--------|
| `Enter` / `Space` | নির্বাচন নিশ্চিত করুন |
| `0`–`9`, `.`, `-` | X কোঅর্ডিনেট এন্ট্রি শুরু করুন |
| `,` | X লক করুন এবং Y এন্ট্রিতে যান |
| `Backspace` | শেষ টাইপ করা অক্ষর মুছুন |
| `Enter` | টাইপ করা কোঅর্ডিনেট নিশ্চিত করুন |
| `Escape` | বাতিল করুন এবং রিসেট করুন |

## কমান্ড চলাকালীন নির্বাচন

| পদ্ধতি | আচরণ |
|--------|-----------|
| **ক্লিক** | কার্সরের নিচের এন্টিটি টগল করে |
| **ডানে টানুন** (কঠোর) | বাক্সের সম্পূর্ণ ভেতরে এন্টিটি যোগ করে |
| **বামে টানুন** (ক্রসিং) | বাক্সের সাথে ছেদকারী এন্টিটি যোগ করে |
| **Enter** / **Space** | নির্বাচন নিশ্চিত করে |

## কী মিরর হয়

প্রতিটি এন্টিটি টাইপ সমর্থিত। জ্যামিতি গণিতভাবে অক্ষ জুড়ে প্রতিফলিত হয়:

| এন্টিটি | কী পরিবর্তন হয় |
|--------|-------------|
| Line | উভয় শেষ পয়েন্ট প্রতিফলিত |
| Circle | কেন্দ্র প্রতিফলিত; ব্যাসার্ধ অপরিবর্তিত |
| Arc | কেন্দ্র প্রতিফলিত; শুরু এবং শেষ কোণ অক্ষ জুড়ে পুনরায় গণনা |
| Ellipse | কেন্দ্র প্রতিফলিত; প্রধান অক্ষের দিক অক্ষ জুড়ে ফ্লিপ |
| Polyline / Rectangle | প্রতিটি ভার্টেক্স প্রতিফলিত |
| Text | অ্যাঙ্কর পয়েন্ট প্রতিফলিত; টেক্সট স্ট্রিং **বিপরীত হয় না** |
| Spline | সমস্ত কন্ট্রোল ভার্টেক্স / ফিট পয়েন্ট প্রতিফলিত |
