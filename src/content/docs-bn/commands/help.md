# Help

সমস্ত কীবোর্ড শর্টকাট, নেভিগেশন টিপস এবং সাধারণ ব্যবহারের সারসংক্ষেপ সহ ইন-অ্যাপ সহায়তা প্যানেল খোলে।

## কীভাবে ব্যবহার করবেন

**Help** টুলবার বোতামে ক্লিক করুন বা টার্মিনালে `help` টাইপ করুন। সহায়তা প্যানেল তাৎক্ষণিকভাবে খোলে এবং কমান্ড প্রস্থান করে।

## সহায়তা প্যানেলে কী আছে

### নেভিগেশন
| শর্টকাট | ক্রিয়া |
|----------|--------|
| `Scroll wheel` | জুম ইন / আউট |
| `Middle click + drag` | ভিউ প্যান করুন |
| `Double middle click` | সমস্ত সত্তা স্ক্রিনে ফিট করুন |

### কমান্ড
| শর্টকাট | ক্রিয়া |
|----------|--------|
| টাইপ | একটি কমান্ডের নাম টাইপ করা শুরু করুন — পরামর্শ ক্যানভাসে প্রদর্শিত হয় |
| `Delete` | সমস্ত নির্বাচিত সত্তা মুছুন |
| `Enter` / `Space` | কমান্ড বা ইনপুট নিশ্চিত করুন |
| `Enter` / `Space` (নিষ্ক্রিয়) | শেষ ব্যবহৃত কমান্ড পুনরাবৃত্তি করুন |
| `Escape` | বর্তমান কমান্ড বাতিল করুন বা নির্বাচন পরিষ্কার করুন |

### নির্বাচন
| শর্টকাট | ক্রিয়া |
|----------|--------|
| `Left click` | একটি সত্তা নির্বাচন করুন |
| `Drag →` | শুধুমাত্র টেনে আনা এলাকার **সম্পূর্ণ ভেতরে** থাকা সত্তা নির্বাচন করুন |
| `Drag ←` | যেকোনো সত্তা নির্বাচন করুন যা টেনে আনা এলাকা **স্পর্শ করে** |
| `Left click grip` | সরাসরি একটি শীর্ষবিন্দু বা শেষবিন্দু সম্পাদনা করুন |

### ড্রয়িং টিপস
- **স্ন্যাপ পয়েন্ট** — কার্সার স্বয়ংক্রিয়ভাবে কাছাকাছি সত্তার শেষবিন্দুতে স্ন্যাপ করে।
- **স্টিকি অ্যাঙ্গেল** — 45° বৃদ্ধিতে লক করতে একটি দিকে যথেষ্ট দূরে কার্সার সরান।
- **সঠিক দৈর্ঘ্য** — কোণ লক হওয়ার সময়, একটি সঠিক দূরত্ব সেট করতে একটি সংখ্যা টাইপ করুন এবং `Enter` চাপুন।
