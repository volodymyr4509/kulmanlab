# Ellipse

`ellipse` কমান্ড তিনটি ক্লিক ব্যবহার করে একটি উপবৃত্ত আঁকে: একটি কেন্দ্র পয়েন্ট, যেকোনো কোণে প্রথম (প্রধান) অর্ধ-অক্ষের শেষবিন্দু এবং দ্বিতীয় (অপ্রধান) অর্ধ-অক্ষের দৈর্ঘ্য। দুটি অক্ষ সর্বদা একে অপরের লম্ব — দ্বিতীয় অক্ষের দিক প্রথম থেকে স্বয়ংক্রিয়ভাবে নির্ধারিত হয়।

## একটি উপবৃত্ত আঁকা

1. টার্মিনালে `ellipse` টাইপ করুন বা **Ellipse** টুলবার বোতামে ক্লিক করুন।
2. **কেন্দ্র পয়েন্টে ক্লিক করুন**, অথবা সঠিক স্থানাঙ্কের জন্য `X,Y` টাইপ করুন এবং **Enter** চাপুন।
3. **প্রথম অক্ষ শেষবিন্দুতে ক্লিক করুন** — প্রথম অর্ধ-অক্ষের দিক এবং দৈর্ঘ্য উভয় সেট করে। স্থানাঙ্ক এন্ট্রিও এখানে কাজ করে।
4. **দ্বিতীয় অক্ষের দৈর্ঘ্য সেট করুন** — প্রথম অক্ষে লম্বভাবে কার্সার সরান, তারপর ক্লিক করুন বা একটি দৈর্ঘ্য টাইপ করুন।

```
               ● ← প্রথম অক্ষ শেষবিন্দু (ধাপ 3)
              /
  center ●  /  ← প্রথম অক্ষ (যেকোনো কোণ)
            |
            ● ← এখানে কার্সার দ্বিতীয় অক্ষের দৈর্ঘ্য সেট করে (ধাপ 4)
```

ধাপ 4-এর পরে উপবৃত্তটি স্থাপিত হয় এবং কমান্ড প্রস্থান করে।

## অক্ষ ইনপুট — ক্লিক, স্থানাঙ্ক বা টাইপ করা দৈর্ঘ্য

**কেন্দ্র (ধাপ 2):** একটি সঠিক অবস্থানের জন্য ক্লিক করুন বা `X,Y` টাইপ করুন।

**প্রথম অক্ষ শেষবিন্দু (ধাপ 3):** ক্লিক করুন, বা সঠিক স্থানাঙ্কের জন্য `X,Y` টাইপ করুন। কোণ লকিং 45° বৃদ্ধিতে স্ন্যাপ করে — লক হওয়ার সময়, ঠিক সেই দূরত্বে শেষবিন্দু স্থাপন করতে একটি দৈর্ঘ্য টাইপ করুন এবং **Enter** চাপুন।

**দ্বিতীয় অক্ষ (ধাপ 4):** টাইপ করা দৈর্ঘ্য সর্বদা উপলব্ধ — কোণ লকের প্রয়োজন নেই। দিক ইতিমধ্যে প্রথম অক্ষে লম্ব স্থির; টাইপ করা শুধুমাত্র দৈর্ঘ্য সেট করে।

| কী | ক্রিয়া |
|-----|--------|
| `0`–`9`, `.` | অক্ষের দৈর্ঘ্যে অঙ্ক যোগ করুন (দ্বিতীয় অক্ষ পর্যায়) |
| `Backspace` | শেষ টাইপ করা অক্ষর মুছুন |
| `Enter` | টাইপ করা দৈর্ঘ্যে অক্ষ শেষবিন্দু স্থাপন করুন |

## স্থানাঙ্ক এন্ট্রি (কেন্দ্র এবং প্রথম অক্ষ শেষবিন্দু)

1. X মান টাইপ করুন।
2. `,` চাপুন — টার্মিনাল দেখায় `[X], [Y{cursor}]`।
3. Y মান টাইপ করুন।
4. নিশ্চিত করতে **Enter** চাপুন।

## কীবোর্ড রেফারেন্স

| কী | ক্রিয়া |
|-----|--------|
| `0`–`9`, `.`, `-` | X স্থানাঙ্ক এন্ট্রি শুরু করুন (কেন্দ্র/প্রথম-অক্ষ পর্যায়), বা কোণ-লক হলে অক্ষের দৈর্ঘ্য |
| `,` | X লক করুন এবং Y এন্ট্রিতে যান |
| `Backspace` | শেষ টাইপ করা অক্ষর মুছুন |
| `Enter` | টাইপ করা স্থানাঙ্ক বা দৈর্ঘ্য নিশ্চিত করুন |
| `Escape` | বাতিল করুন এবং রিসেট করুন |

## Grip সম্পাদনা — স্বাধীন অক্ষ আকার পরিবর্তন

একটি নির্বাচিত উপবৃত্ত পাঁচটি গ্রিপ প্রকাশ করে:

| Grip | সংখ্যা | এটি কী করে |
|------|-------|--------------|
| **Center** | 1 | পুরো উপবৃত্ত সরায়; উভয় অক্ষ অপরিবর্তিত |
| **প্রধান অক্ষ শেষবিন্দু** | 2 (দীর্ঘ অক্ষের বিপরীত প্রান্ত) | প্রধান অর্ধ-অক্ষের দৈর্ঘ্য পরিবর্তন করতে টানুন; অপ্রধান অক্ষের নিরঙ্কুশ আকার স্থির থাকে |
| **অপ্রধান অক্ষ শেষবিন্দু** | 2 (ছোট অক্ষের বিপরীত প্রান্ত) | অপ্রধান অর্ধ-অক্ষের দৈর্ঘ্য পরিবর্তন করতে টানুন; প্রধান অক্ষ অপরিবর্তিত |

প্রধান এবং অপ্রধান অক্ষ গ্রিপ স্বাধীন — আপনি কমান্ড পুনরায় না চালিয়ে উপবৃত্তের আকার পরিবর্তন করতে পারেন।

## উপবৃত্ত নির্বাচন করা

| পদ্ধতি | আচরণ |
|--------|-----------|
| **ক্লিক** | ক্লিকটি উপবৃত্তের রূপরেখার কাছে পড়লে নির্বাচন করে |
| **ডানে টানুন** (কঠোর) | উপবৃত্তের অক্ষ-সারিবদ্ধ বাউন্ডিং বাক্স সম্পূর্ণরূপে নির্বাচন বাক্সের ভেতরে ফিট করতে হবে |
| **বামে টানুন** (ক্রসিং) | উপবৃত্তের রূপরেখার যেকোনো অংশ যা নির্বাচন বাক্সের সীমানা অতিক্রম করে তা নির্বাচন করে |

## Ellipse বনাম Circle — কোনটি ব্যবহার করবেন

| | Ellipse | Circle |
|---|---------|--------|
| অক্ষ | যেকোনো কোণে দুটি স্বাধীন অর্ধ-অক্ষ | একটি ব্যাসার্ধ, প্রতিসম |
| ঘূর্ণন | যেকোনো কোণে স্থাপন করা যায় | কোনো ঘূর্ণন নেই |
| টাইপ করা ইনপুট | প্রতি-অক্ষ দৈর্ঘ্য | শুধুমাত্র ব্যাসার্ধ |
| গ্রিপ আকার পরিবর্তন | প্রধান এবং অপ্রধান স্বাধীনভাবে | সমানভাবে সব চারটি কার্ডিনাল পয়েন্ট |
| সবচেয়ে ভালো | তির্যক দৃশ্য, ডিম্বাকৃতি বৈশিষ্ট্য, পরিপ্রেক্ষিত গর্ত | প্রতিসম গোলাকার বৈশিষ্ট্য |

## DXF — ELLIPSE সত্তা

উপবৃত্তগুলি DXF ফাইলে `ELLIPSE` সত্তা হিসেবে সংরক্ষিত হয়। ফরম্যাটটি কেন্দ্র পয়েন্ট, সম্পূর্ণ প্রধান অক্ষ ভেক্টর (দিক + দৈর্ঘ্য) এবং অক্ষ অনুপাত সঞ্চয় করে। ঘূর্ণন, আকৃতি এবং সমস্ত স্টাইল বৈশিষ্ট্য কোনো ক্ষতি ছাড়াই রাউন্ড-ট্রিপ করে। একটি বৃত্ত একটি অবনমিত উপবৃত্ত হিসেবে সংরক্ষিত হয় **না** — দুটি সত্তার ধরন DXF মডেলে স্বতন্ত্র থাকে।
