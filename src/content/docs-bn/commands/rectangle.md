# Rectangle

`rectangle` কমান্ড দুটি বিপরীত কোণ ক্লিক দিয়ে নির্ধারিত একটি অক্ষ-সমান্তরাল আয়তক্ষেত্র আঁকে। ফলাফল চারটি ভার্টেক্স সহ একটি **বদ্ধ `LWPOLYLINE`** হিসেবে সংরক্ষিত হয় — প্রতিটি কোণে একটি। কোনো ডেডিকেটেড আয়তক্ষেত্র এন্টিটি টাইপ নেই: তৈরির পরে আকারটি ঠিক অন্য যেকোনো [Polyline](../polyline/) এর মতো আচরণ করে এবং প্রতিটি পলিলাইন সম্পাদনা এটিতে প্রযোজ্য।

## একটি আয়তক্ষেত্র আঁকা

1. টার্মিনালে `rectangle` টাইপ করুন বা **Rectangle** টুলবার বাটনে ক্লিক করুন।
2. **প্রথম কোণে ক্লিক করুন**, বা সঠিক কোঅর্ডিনেটের জন্য `X,Y` টাইপ করুন এবং **Enter** চাপুন।
3. **বিপরীত কোণে ক্লিক করুন** — আয়তক্ষেত্র তাৎক্ষণিকভাবে স্থাপন হয় এবং কমান্ড শেষ হয়। কোঅর্ডিনেট এন্ট্রি এখানেও কাজ করে।

```
  ● (প্রথম ক্লিক)────────────┐
  |                          |
  |   লাইভ প্রিভিউ ধাপ ২ এর  |
  |   পরে কার্সর অনুসরণ করে  |
  └──────────────────────────● (দ্বিতীয় ক্লিক)
```

দুটি ক্লিক যেকোনো জোড়া কোণাকুণি বিপরীত কোণ হতে পারে — উপরের-বাম + নিচের-ডান, বা নিচের-বাম + উপরের-ডান, ইত্যাদি। ক্রম গুরুত্বপূর্ণ নয়।

## কোঅর্ডিনেট এন্ট্রি

যেকোনো কোণ ধাপে আপনি একটি সঠিক পজিশন টাইপ করতে পারেন:

1. X মান টাইপ করুন।
2. `,` চাপুন — টার্মিনাল `[X], [Y{কার্সর}]` দেখায়।
3. Y মান টাইপ করুন।
4. কোণ স্থাপন করতে **Enter** চাপুন।

## কীবোর্ড রেফারেন্স

| কী | ক্রিয়া |
|-----|--------|
| `0`–`9`, `.`, `-` | X কোঅর্ডিনেট এন্ট্রি শুরু করুন |
| `,` | X লক করুন এবং Y এন্ট্রিতে যান |
| `Enter` | টাইপ করা কোঅর্ডিনেট নিশ্চিত করুন |
| `Escape` | বাতিল করুন |

বাহুগুলি সর্বদা অনুভূমিক এবং উল্লম্ব — rectangle কমান্ডের জন্য কোনো কোণ লকিং নেই।

## গ্রিপ সম্পাদনা — তৈরির পরে পুনরায় আকার দেওয়া

একটি নির্বাচিত আয়তক্ষেত্র প্রতিটি ভার্টেক্সে এবং প্রতিটি বাহুর মিডপয়েন্টে গ্রিপ দেখায়:

| গ্রিপ | পজিশন | কী করে |
|------|----------|--------------|
| **কোণ** | ৪টি ভার্টেক্সের প্রতিটিতে | সেই ভার্টেক্স সরাতে টেনে আনুন; দুটি সংলগ্ন বাহু অনুসরণ করে প্রসারিত হয় — বিপরীত কোণ স্থির থাকে |
| **বাহু মিডপয়েন্ট** | ৪টি বাহুর প্রতিটির কেন্দ্রে | সেই বাহুর উভয় শেষ পয়েন্ট একসাথে স্থানান্তর করতে টেনে আনুন, বাহুর দৈর্ঘ্য ও কোণ রেখে |

একটি কোণ গ্রিপ টেনে আনলে আয়তক্ষেত্র একটি অ-আয়তাকার চতুর্ভুজে পরিণত হয়। যদি আপনার শুধু ভিন্ন আকারের আয়তক্ষেত্র দরকার হয়, বাহুগুলি মোটামুটি লম্বানুভূমিক রেখে কোণ টেনে আনুন, বা মুছে নতুন একটি আঁকুন।

## আয়তক্ষেত্র নির্বাচন

আয়তক্ষেত্র একটি পলিলাইন হওয়ায়, নির্বাচন একইভাবে কাজ করে:

| পদ্ধতি | আচরণ |
|--------|-----------|
| **ক্লিক** | চারটি বাহুর যেকোনো একটিতে ক্লিক পড়লে নির্বাচিত হয় |
| **ডানে টানুন** (কঠোর) | চারটি ভার্টেক্স সব নির্বাচন বাক্সের ভেতরে থাকতে হবে |
| **বামে টানুন** (ক্রসিং) | বাক্সের সীমানা অতিক্রমকারী যেকোনো বাহু পুরো আয়তক্ষেত্র নির্বাচিত করে |

## সমর্থিত সম্পাদনা কমান্ড

সমস্ত পলিলাইন সম্পাদনা কমান্ড প্রযোজ্য। Trim এবং Extend [Line](../line/)-শুধু এবং আয়তক্ষেত্রে কাজ করে না:

| কমান্ড | আয়তক্ষেত্রে কী হয় |
|---------|-------------------------------|
| [Move](../move/) | সমস্ত চারটি ভার্টেক্স একই বিচ্যুতিতে স্থানান্তরিত করে |
| [Copy](../copy/) | নতুন পজিশনে একটি অভিন্ন আয়তক্ষেত্র তৈরি করে |
| [Rotate](../rotate/) | বেছে নেওয়া বেস পয়েন্টের চারপাশে সমস্ত চারটি ভার্টেক্স ঘোরায় |
| [Mirror](../mirror/) | মিরর অক্ষ জুড়ে সমস্ত চারটি ভার্টেক্স প্রতিফলিত করে |
| [Scale](../scale/) | বেস পয়েন্ট থেকে সমস্ত চারটি ভার্টেক্স সমানভাবে স্কেল করে |
| [Offset](../offset/) | নির্দিষ্ট দূরত্বে একটি সমান্তরাল (ইনসেট বা আউটসেট) আয়তক্ষেত্র তৈরি করে |
| [Delete](../delete/) | আয়তক্ষেত্র সরায় |

## প্রপার্টি

একটি আয়তক্ষেত্র নির্বাচিত হলে প্রপার্টি প্যানেল যেকোনো পলিলাইনের মতো একই ফিল্ড দেখায়:

**সাধারণ**

| প্রপার্টি | ডিফল্ট | অর্থ |
|----------|---------|---------|
| Color | 256 (ByLayer) | ACI রঙের সূচক |
| Layer | `0` | লেয়ার অ্যাসাইনমেন্ট |
| Linetype | ByLayer | নামযুক্ত লাইনটাইপ প্যাটার্ন |
| Linetype Scale | 1 | লাইনটাইপ প্যাটার্নের স্কেল ফ্যাক্টর |
| Thickness | 0 | এক্সট্রুশন পুরুত্ব |

**জ্যামিতি**

| প্রপার্টি | অর্থ |
|----------|---------|
| Closed | আয়তক্ষেত্রের জন্য সর্বদা `true` |
| Vertex Count | অপরিবর্তিত আয়তক্ষেত্রের জন্য সর্বদা `4` |
| Vertices | সমস্ত চারটি কোণের কোঅর্ডিনেট |

| | Rectangle | Polyline | Line |

| এন্টিটি টাইপ | বদ্ধ `LWPOLYLINE` | খোলা বা বদ্ধ `LWPOLYLINE` | প্রতি সেগমেন্টে `LINE` |
| বাহু সর্বদা লম্বানুভূমিক | হ্যাঁ (তৈরিতে) | না | না |
| Trim / Extend | না | না | হ্যাঁ |
| সেরা জন্য | বাক্স, ফ্রেম, আয়তাকার এলাকা | যেকোনো আউটলাইন এবং পথ | পৃথক সেগমেন্ট, কনস্ট্রাকশন লাইন |

## DXF — LWPOLYLINE এন্টিটি

আয়তক্ষেত্রগুলি চারটি ভার্টেক্স সহ বদ্ধ `LWPOLYLINE` এন্টিটি হিসেবে সেভ হয়। সমস্ত প্রপার্টি — ভার্টেক্স কোঅর্ডিনেট, রঙ, লেয়ার, লাইনটাইপ, লাইনটাইপ স্কেল এবং পুরুত্ব — ক্ষতি ছাড়াই রাউন্ড-ট্রিপ করে।

DXF-এ কোনো ডেডিকেটেড `RECTANGLE` টাইপ নেই। ফাইল পুনরায় খোলা হলে, আকারটি আয়তক্ষেত্রের পরিবর্তে একটি বদ্ধ চার-ভার্টেক্স পলিলাইন হিসেবে দেখা যায়। `LWPOLYLINE` সমর্থনকারী যেকোনো DXF ভিউয়ার বা এডিটর (LibreCAD, FreeCAD, ইত্যাদি) এটি সঠিকভাবে প্রদর্শন করবে।
