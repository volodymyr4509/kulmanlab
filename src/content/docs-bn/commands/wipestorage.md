# Wipe Storage

`wipestorage` কমান্ড KulmanLab CAD-এর জন্য **ব্রাউজারে সংরক্ষিত সমস্ত ডেটা** স্থায়ীভাবে মুছে দেয় — প্রতিটি সংরক্ষিত ফাইল, লেয়ার ও লাইনটাইপ টেবিল এবং আনডো ইতিহাস। পেজ পরে স্বয়ংক্রিয়ভাবে রিলোড হয়।

:::danger অপরিবর্তনীয়
এই কার্যক্রম বাতিল করা যায় না। ব্রাউজারে সংরক্ষিত সমস্ত ফাইল মুছে যায়। এই কমান্ড চালানোর আগে রাখতে চাওয়া যেকোনো ড্রয়িং `.json` বা `.dxf` ফাইল হিসেবে এক্সপোর্ট করুন।
:::

## কখন ব্যবহার করবেন

- ব্রাউজার স্টোরেজ দূষিত হয়েছে এবং অ্যাপ ফাইল লোড বা সেভ করতে ব্যর্থ হচ্ছে।
- আপনি অ্যাপটি সম্পূর্ণভাবে একটি পরিষ্কার অবস্থায় রিসেট করতে চান।
- আপনি ব্রাউজার বা ডিভাইস পরিবর্তন করছেন এবং আর স্থানীয় কপির দরকার নেই।

## কীভাবে চালাবেন

1. টার্মিনালে `wipestorage` টাইপ করুন এবং **Enter** চাপুন।
2. টার্মিনাল প্রম্পট করে: *Wipe all browser local storage? Type YES to confirm*
3. `YES` (যেকোনো বড়/ছোট হাতে) টাইপ করুন এবং **Enter** চাপুন।

অ্যাপ ডেটাবেস মুছে দেয় এবং পেজ রিলোড করে। যদি আপনি `YES` ছাড়া অন্য কিছু টাইপ করেন এবং **Enter** চাপেন, বা **Escape** চাপেন, কমান্ড বাতিল হয় এবং কিছুই মুছে যায় না।

| প্রতিটি ফাইলের জন্য আনডো / রিডো ইতিহাস | হ্যাঁ |
