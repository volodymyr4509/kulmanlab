# Offset

`offset` কমান্ড একটি নির্দিষ্ট লম্ব দূরত্বে একটি এন্টিটির সমান্তরাল কপি তৈরি করে। আপনি একবার দূরত্ব টাইপ করেন, তারপর এন্টিটি ক্লিক করুন এবং একটি দিক বেছে নিন — কমান্ডটি একই দূরত্বে প্রস্তুত থাকে যাতে আপনি এক সেশনে একাধিক অবজেক্ট অফসেট করতে পারেন।

সমর্থিত এন্টিটি টাইপ: **Line, Circle, Arc, Ellipse, Polyline** (Rectangle সহ)।

## অফসেট ব্যবহার

1. টার্মিনালে `offset` টাইপ করুন বা **Offset** টুলবার বাটনে ক্লিক করুন।
2. **অফসেট দূরত্ব টাইপ করুন** এবং **Enter** বা **Space** চাপুন।
3. অফসেট করতে **একটি এন্টিটিতে ক্লিক করুন** — এন্টিটিটি সমর্থিত টাইপ না হলে, একটি ত্রুটি বার্তা দেখায় এবং আপনি একটি ভিন্ন এন্টিটি ক্লিক করতে পারেন।
4. কপি কোন দিকে থাকবে সেই দিকে **কার্সর সরান** — একটি লাইভ প্রিভিউ অনুসরণ করে।
5. অফসেট কপি স্থাপন করতে **ক্লিক করুন**।

প্রতিটি স্থাপনের পরে কমান্ড **একই দূরত্বে** ধাপ ৩-এ ফিরে আসে, পরবর্তী অফসেটের জন্য প্রস্তুত। দূরত্ব-এন্ট্রি ধাপে ফিরে যেতে **Escape** চাপুন।

```
  দূরত্ব: 10

  ─────────────────    ← মূল লাইন
  ─────────────────    ← অফসেট কপি (নিচে ১০ ইউনিট)
```

## প্রতি-এন্টিটি অফসেট আচরণ

| এন্টিটি | অফসেট কীভাবে গণনা হয় |
|--------|---------------------------|
| **Line** | মূল দিকের লম্বে পরিবর্তিত সমান্তরাল লাইন |
| **Circle** | কেন্দ্রীক সার্কেল; বাইরে ক্লিক → বড় ব্যাসার্ধ, ভেতরে → ছোট ব্যাসার্ধ |
| **Arc** | নতুন ব্যাসার্ধে কেন্দ্রীক আর্ক; একই কোণীয় বিস্তার সংরক্ষিত |
| **Ellipse** | উভয় সেমি-অক্ষ একই দূরত্বে বৃদ্ধি বা হ্রাস |
| **Polyline** | প্রতিটি সেগমেন্ট স্বাধীনভাবে অফসেট; সংলগ্ন অফসেট সেগমেন্ট কোণায় মাইটার করা |

**Circle**, **Arc**, এবং **Ellipse** এর জন্য: ভেতরের দিকের অফসেট যদি কোনো ব্যাসার্ধ বা সেমি-অক্ষ শূন্য বা তার কম করে, তাহলে অফসেট প্রয়োগ হয় না।

## কীবোর্ড রেফারেন্স

| কী | ক্রিয়া |
|-----|--------|
| `0`–`9`, `.` | দূরত্ব মানে সংখ্যা যোগ করুন |
| `Backspace` | শেষ টাইপ করা অক্ষর মুছুন |
| `Enter` / `Space` | টাইপ করা দূরত্ব নিশ্চিত করুন এবং এন্টিটি নির্বাচনে যান |
| `Escape` | দূরত্ব-এন্ট্রি ধাপে রিসেট করুন |

## কর্মপ্রবাহ নোট

**Escape** না চাপা পর্যন্ত দূরত্ব সেট থাকে। এটি একই স্পেসিংয়ে অনেক এন্টিটি অফসেট করা দক্ষ করে তোলে — একবার দূরত্ব টাইপ করুন, তারপর প্রতিটি এন্টিটির জন্য ক্লিক করুন এবং দিক বেছে নিন।
