# Copy

`copy` কমান্ড নির্বাচিত সত্তার অনুবাদিত ডুপ্লিকেট তৈরি করে এবং সেগুলিকে একটি ভিত্তি পয়েন্ট থেকে গন্তব্যে অফসেট স্থাপন করে — মূলগুলি ঠিক যেখানে আছে সেখানে থাকে। এটি [Move](./move)-এর সাথে একটি মূল পার্থক্য: Copy ড্রয়িংয়ে নতুন সত্তা যোগ করে; Move বিদ্যমানগুলি স্থানান্তরিত করে।

## শুরু করার দুটি উপায়

**প্রি-সিলেক্ট করুন, তারপর কপি করুন** — প্রথমে সত্তা নির্বাচন করুন, তারপর সক্রিয় করুন:

1. ক্যানভাসে এক বা একাধিক সত্তা নির্বাচন করুন।
2. টার্মিনালে `copy` টাইপ করুন বা **Copy** টুলবার বোতামে ক্লিক করুন।
3. **ভিত্তি পয়েন্টে ক্লিক করুন**, অথবা সঠিক স্থানাঙ্কের জন্য `X,Y` টাইপ করুন এবং **Enter** চাপুন।
4. **গন্তব্যে ক্লিক করুন** — ডুপ্লিকেটগুলি base→destination অফসেটে প্রদর্শিত হয়। স্থানাঙ্ক এন্ট্রিও এখানে কাজ করে।

**সক্রিয় করুন, তারপর নির্বাচন করুন** — কিছু নির্বাচন না করে কমান্ড শুরু করুন:

1. `copy` টাইপ করুন বা টুলবার বোতামে ক্লিক করুন।
2. **বস্তু নির্বাচন করুন** — পৃথক সত্তা টগল করতে ক্লিক করুন, বা এলাকা দ্বারা নির্বাচন করতে টানুন।
3. নির্বাচন নিশ্চিত করতে **Enter** বা **Space** চাপুন।
4. **ভিত্তি পয়েন্টে ক্লিক করুন**, তারপর **গন্তব্যে ক্লিক করুন** (উভয় ধাপে স্থানাঙ্ক এন্ট্রি উপলব্ধ)।

```
  আগে:               পরে:
  [সত্তা A]            [সত্তা A]  ← মূলগুলি অস্পর্শিত
  [সত্তা B]            [সত্তা B]
                        [A-এর কপি] ← নতুন সত্তা
                        [B-এর কপি]
```

ভিত্তি পয়েন্ট থেকে গন্তব্য পর্যন্ত কপিগুলির একটি ঘোস্ট পূর্বদর্শন কার্সার অনুসরণ করে।

## স্থানাঙ্ক এন্ট্রি

ভিত্তি-পয়েন্ট বা গন্তব্য ধাপে, ক্লিক করার পরিবর্তে একটি সঠিক অবস্থান টাইপ করুন:

1. X মান টাইপ করুন।
2. `,` চাপুন — টার্মিনাল দেখায় `[X], [Y{cursor}]`।
3. Y মান টাইপ করুন।
4. নিশ্চিত করতে **Enter** চাপুন।

## কোণ লকিং এবং সঠিক দূরত্ব

ভিত্তি পয়েন্ট সেট হওয়ার পরে কমান্ড 45° অক্ষে (0°, 45°, 90°, 135°, …) স্ন্যাপ করে যখন কার্সার যথেষ্ট দূরে এবং অক্ষের কাছাকাছি থাকে। লক হওয়ার সময়, একটি দূরত্ব টাইপ করুন এবং **Enter** চাপুন ঠিক সেই অফসেটে কপি স্থাপন করতে।

| কী | ক্রিয়া |
|-----|--------|
| `0`–`9`, `.` | দূরত্ব মানে যোগ করুন |
| `-` | নেতিবাচক দূরত্ব — অক্ষ বরাবর দিক বিপরীত করে (শুধুমাত্র প্রথম অক্ষর) |
| `Backspace` | শেষ টাইপ করা অক্ষর মুছুন |
| `Enter` | টাইপ করা দূরত্বে কপি স্থাপন করুন |

## কীবোর্ড রেফারেন্স

| কী | ক্রিয়া |
|-----|--------|
| `Enter` / `Space` | নির্বাচন নিশ্চিত করুন এবং ভিত্তি-পয়েন্ট পর্যায়ে এগিয়ে যান |
| `0`–`9`, `.`, `-` | X স্থানাঙ্ক এন্ট্রি শুরু করুন, বা কোণ-লক হলে দূরত্ব |
| `,` | X লক করুন এবং Y এন্ট্রিতে যান |
| `Backspace` | শেষ টাইপ করা অক্ষর মুছুন |
| `Enter` | স্থানাঙ্ক নিশ্চিত করুন বা টাইপ করা দূরত্বে কপি প্রয়োগ করুন |
| `Escape` | বাতিল করুন এবং রিসেট করুন |

## কমান্ড চলাকালীন নির্বাচন

| পদ্ধতি | আচরণ |
|--------|-----------|
| **ক্লিক** | কার্সারের নীচের সত্তাকে নির্বাচনে টগল করে |
| **ডানে টানুন** (কঠোর) | বাক্সের সম্পূর্ণ ভেতরে থাকা সত্তা যোগ করে |
| **বামে টানুন** (ক্রসিং) | বাক্সের সীমানা অতিক্রম করা সত্তা যোগ করে |
| **Enter** / **Space** | নির্বাচন নিশ্চিত করে |

## কপির পরে

**মূলগুলি নির্বাচিত থাকে** — নতুন কপিগুলি ড্রয়িংয়ে যোগ করা হয় কিন্তু নির্বাচন পরিষ্কার হয় এবং কমান্ড প্রস্থান করে। কপিগুলির সাথে অবিলম্বে কাজ করতে, নির্বাচনে Copy আবার চালান, বা একটি নতুন কমান্ড শুরু করুন।

## Copy বনাম Move

| | Copy | Move |
|---|------|------|
| মূলগুলি | যেখানে আছে থাকে | মূল অবস্থান থেকে সরানো হয় |
| ফলাফল সংখ্যা | কপি করা সত্তার সংখ্যা দ্বারা বৃদ্ধি পায় | অপরিবর্তিত |
| অপারেশনের পরে | মূলগুলি এখনও নির্বাচিত | নতুন অবস্থানে সরানো সত্তা নির্বাচিত |
| সবচেয়ে ভালো | জ্যামিতি পুনরাবৃত্তি, প্রতিসম লেআউট | জ্যামিতি পুনঃস্থাপন |

## সমর্থিত সত্তা

Copy প্রতিটি সত্তার ধরনে কাজ করে। সমস্ত সত্তা অভ্যন্তরীণভাবে `translate(dx, dy)` বাস্তবায়ন করে তাই কেউই বাদ পড়ে না।
