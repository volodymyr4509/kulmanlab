# Spline Fit

`splinefit` কমান্ড একটি কিউবিক স্প্লাইন আঁকে যা আপনার ক্লিক করা প্রতিটি পয়েন্টের মধ্য দিয়ে যায় — একটি ইন্টারপোলেটিং বক্ররেখা। [Spline CV](../spline-cv/) এর বিপরীতে, যেখানে বক্ররেখা শুধুমাত্র কন্ট্রোল ভার্টেক্সের দিকে আকর্ষিত হয়, এখানে বক্ররেখা প্রতিটি ক্লিক করা কোঅর্ডিনেট ঠিকমতো স্পর্শ করতে বাধ্য। ভেতরে এডিটর এটি অর্জন করতে কন্ট্রোল ভার্টেক্স ফিট করে, এবং সেই CV-গুলি DXF ফাইলে ফিট পয়েন্টের পাশাপাশি সংরক্ষিত হয়।

## ফিট পয়েন্ট দিয়ে স্প্লাইন আঁকা

1. টার্মিনালে `splinefit` টাইপ করুন বা **Spline Fit** টুলবার বাটনে ক্লিক করুন।
2. **ফিট পয়েন্ট স্থাপন করতে ক্লিক করুন** — বক্ররেখা প্রতিটির মধ্য দিয়ে যাবে। বা সঠিক কোঅর্ডিনেটের জন্য `X,Y` টাইপ করুন এবং **Enter** চাপুন।
3. শেষ করতে **Enter** চাপুন (কমপক্ষে ২টি পয়েন্ট প্রয়োজন)।

```
  ●──────●──────●──────●  ← বক্ররেখা প্রতিটি ক্লিকের মধ্য দিয়ে ঠিকমতো যায়
  p1     p2     p3     p4
```

কার্সর সরানোর সময় লাইভ প্রিভিউ বর্তমান ইন্টারপোলেটেড বক্ররেখা দেখায়, কার্সর পজিশনে সম্ভাব্য পরবর্তী পয়েন্ট সহ। সমস্ত স্থাপিত পয়েন্ট বাদ দিতে এবং বের হতে **Escape** চাপুন।

## কোঅর্ডিনেট এন্ট্রি

ক্লিক করার বদলে, যেকোনো ফিট পয়েন্টের জন্য একটি সঠিক পজিশন টাইপ করুন:

1. X মান টাইপ করুন।
2. `,` চাপুন — টার্মিনাল `[X], [Y{কার্সর}]` দেখায়।
3. Y মান টাইপ করুন।
4. ফিট পয়েন্ট স্থাপন করতে **Enter** চাপুন।

## কীবোর্ড রেফারেন্স

| কী | ক্রিয়া |
|-----|--------|
| `0`–`9`, `.`, `-` | X কোঅর্ডিনেট এন্ট্রি শুরু করুন |
| `,` | X লক করুন এবং Y এন্ট্রিতে যান |
| `Backspace` | শেষ টাইপ করা অক্ষর মুছুন |
| `Enter` | টাইপ করা কোঅর্ডিনেট নিশ্চিত করুন, বা কিছু না থাকলে এবং ≥ ২ পয়েন্ট থাকলে স্প্লাইন শেষ করুন |
| `Escape` | সমস্ত পয়েন্ট বাদ দিন এবং বের হন |

## গ্রিপ সম্পাদনা — ফিট পয়েন্ট দিয়ে পুনরায় আকার দেওয়া

একটি নির্বাচিত ফিট স্প্লাইন প্রতিটি ফিট পয়েন্টে একটি গ্রিপ দেখায়:

| গ্রিপ | পজিশন | কী করে |
|------|----------|--------------|
| **ফিট পয়েন্ট** | প্রতিটি ক্লিক করা পজিশনে | সেই ফিট পয়েন্ট সরাতে টেনে আনুন — নতুন পজিশনের মধ্য দিয়ে যেতে সম্পূর্ণ বক্ররেখা পুনরায় ইন্টারপোলেট হয় |

একটি গ্রিপ টেনে আনলে পুরো বক্ররেখা পুনরায় ফিট হয়, শুধু সংলগ্ন সেগমেন্ট নয়। এটি পলিলাইন গ্রিপ সম্পাদনা থেকে আলাদা, যেখানে একটি ভার্টেক্স সরানো শুধুমাত্র দুটি সংলগ্ন সেগমেন্ট পুনরায় আকার দেয়।

"পুরো স্প্লাইন সরান" গ্রিপ নেই। সম্পূর্ণ স্প্লাইন স্থানান্তর করতে, [Move](../move/) কমান্ড ব্যবহার করুন।

## ফিট স্প্লাইন নির্বাচন

| পদ্ধতি | আচরণ |
|--------|-----------|
| **ক্লিক** | বক্ররেখার কাছাকাছি কোনো পয়েন্টে ক্লিক হলে নির্বাচিত হয় |
| **ডানে টানুন** (কঠোর) | বক্ররেখা বরাবর সমস্ত নমুনা পয়েন্ট নির্বাচন বাক্সের ভেতরে থাকতে হবে |
| **বামে টানুন** (ক্রসিং) | নির্বাচন বাক্সের সীমানা অতিক্রমকারী বক্ররেখার যেকোনো অংশ নির্বাচিত করে |

## সমর্থিত সম্পাদনা কমান্ড

|---------|---------------------------|
| [Move](../move/) | সমস্ত ফিট পয়েন্ট এবং পুনরায় গণনা করা CV একই বিচ্যুতিতে স্থানান্তরিত করে |
| [Copy](../copy/) | নতুন পজিশনে একটি অভিন্ন স্প্লাইন তৈরি করে |

## Spline Fit বনাম Spline CV — কোনটি ব্যবহার করবেন

| | Spline Fit | Spline CV |
|---|------------|-----------|
| বক্ররেখা পয়েন্টের মধ্য দিয়ে যায় | প্রতিটি ক্লিক করা পয়েন্ট ঠিকমতো | শুধুমাত্র প্রথম ও শেষ (ক্ল্যাম্পড) |
| গ্রিপ সম্পাদনা প্রভাব | ফিট পয়েন্ট সরে → সম্পূর্ণ বক্ররেখা পুনরায় ইন্টারপোলেট | CV সরে → বক্ররেখা নতুন পজিশনের দিকে আকর্ষিত |
| আকৃতির পূর্বানুমানযোগ্যতা | উচ্চ — বক্ররেখা ক্লিক অনুসরণ করে | কম — বক্ররেখা CV-র পিছিয়ে থাকে |
| সেরা জন্য | নির্দিষ্ট কোঅর্ডিনেট স্পর্শ করতে হবে এমন বক্ররেখা | মসৃণ নান্দনিক বক্ররেখা, মুক্ত-ফর্ম পথ |

## DXF — SPLINE এন্টিটি (ফিট-পয়েন্ট ফর্ম)

ফিট স্প্লাইনগুলি DXF ফাইলে `SPLINE` এন্টিটি হিসেবে সেভ হয়, ফিট পয়েন্ট কোঅর্ডিনেট এবং গণনা করা কন্ট্রোল ভার্টেক্স উভয়ই সংরক্ষণ করে। `splineFlag` `8` (ফিট-পয়েন্ট স্প্লাইন) সেট করা হয় যাতে রিলোড অ্যাপ্লিকেশন জানে কোন সেট পয়েন্ট সম্পাদনযোগ্য গ্রিপ হিসেবে প্রদর্শন করতে হবে। সমস্ত প্রপার্টি — রঙ, লেয়ার, লাইনটাইপ, লাইনটাইপ স্কেল এবং পুরুত্ব — ক্ষতি ছাড়াই রাউন্ড-ট্রিপ করে। ফিট-পয়েন্ট স্প্লাইন সমর্থনকারী DXF অ্যাপ্লিকেশন (LibreCAD, FreeCAD) প্রাথমিক সম্পাদনযোগ্য ডেটা হিসেবে ফিট পয়েন্ট প্রদর্শন করবে।
