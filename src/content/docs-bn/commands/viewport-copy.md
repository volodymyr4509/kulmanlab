# ViewportCopy

`ViewportCopy` কমান্ড একটি ভিউপোর্ট নতুন পজিশনে কপি করে, এর স্কেল এবং মডেল সেন্টার সংরক্ষণ করে। শুধুমাত্র লেআউট স্পেসে উপলব্ধ।

## একটি ভিউপোর্ট কপি করা

1. একটি পেপার লেআউট ট্যাবে যান।
2. ঐচ্ছিকভাবে একটি ভিউপোর্ট নির্বাচন করতে ক্লিক করুন।
3. টার্মিনালে `ViewportCopy` টাইপ করুন বা **Viewport Copy** টুলবার বাটনে ক্লিক করুন।
4. কোনো ভিউপোর্ট পূর্ব-নির্বাচিত না থাকলে, কপি করতে **ভিউপোর্টে ক্লিক করুন**।
5. **বেস পয়েন্টে ক্লিক করুন** — বিচ্যুতির রেফারেন্স। বা সঠিক কোঅর্ডিনেটের জন্য `X,Y` টাইপ করুন এবং **Enter** চাপুন।
6. **গন্তব্যে ক্লিক করুন** — ভিউপোর্ট বেস→গন্তব্য অফসেটে স্থাপিত হয়। বা কোঅর্ডিনেট এন্ট্রি / কোণ লক ব্যবহার করুন।

স্থাপনার পরে, কমান্ড সক্রিয় থাকে — একই ভিউপোর্টের আরেকটি কপি স্থাপন করতে অন্য গন্তব্যে ক্লিক করুন। শেষ করতে **Enter**, **Space**, বা **Escape** চাপুন।

## কোঅর্ডিনেট এন্ট্রি

বেস-পয়েন্ট এবং গন্তব্য ধাপে ক্লিক করার বদলে একটি সঠিক কোঅর্ডিনেট টাইপ করতে পারেন:

1. X মান টাইপ করুন।
2. `,` চাপুন — টার্মিনাল `[X], [Y{কার্সর}]` দেখায়।
3. Y মান টাইপ করুন।
4. নিশ্চিত করতে **Enter** চাপুন।

## কোণ লকিং এবং সঠিক দূরত্ব

বেস পয়েন্ট সেট হওয়ার পরে, কার্সর সারিবদ্ধ হলে কমান্ড ৪৫° অক্ষে (০°, ৪৫°, ৯০°, ১৩৫°, …) স্ন্যাপ করে। লক থাকাকালীন:

- প্রিভিউ অক্ষে স্ন্যাপ করে।
- লক করা দিক বরাবর ঠিক সেই অফসেটে কপি স্থাপন করতে একটি দূরত্ব টাইপ করুন এবং **Enter** চাপুন।

| কী | ক্রিয়া |
|-----|--------|
| `0`–`9`, `.` | দূরত্ব মানে সংখ্যা যোগ করুন |
| `-` | নেতিবাচক দূরত্ব (দিক বিপরীত করে; শুধু প্রথম অক্ষর) |
| `Backspace` | শেষ টাইপ করা অক্ষর মুছুন |
| `Enter` | টাইপ করা দূরত্বে কপি স্থাপন করুন |

## কীবোর্ড রেফারেন্স

| কী | ক্রিয়া |
|-----|--------|
| `0`–`9`, `.`, `-` | X কোঅর্ডিনেট এন্ট্রি শুরু করুন, বা কোণ-লক থাকলে দূরত্ব |
| `,` | X লক করুন এবং Y এন্ট্রিতে যান |
| `Enter` | টাইপ করা কোঅর্ডিনেট বা দূরত্ব নিশ্চিত করুন |
| `Enter` / `Space` | শেষ করুন (কোনো ইনপুট চলছে না থাকলে) |
| `Escape` | বাতিল করুন এবং রিসেট করুন |

## নোট

- ViewportCopy শুধুমাত্র একটি পেপার লেআউট ট্যাব সক্রিয় থাকলে উপলব্ধ।
- কপি করা ভিউপোর্ট মূলের মতো একই স্কেল, মডেল সেন্টার, লক অবস্থা এবং মাত্রা উত্তরাধিকারসূত্রে পায়।
- স্ক্র্যাচ থেকে নতুন ভিউপোর্ট তৈরি করতে, [ViewportRectangle](../viewport-rectangle/) ব্যবহার করুন।
