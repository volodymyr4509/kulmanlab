# 線種（Linetype）

ツールバーの**線種**チップは、作図するすべての新しいエンティティに割り当てられる破線パターンを制御します。クリックしてピッカードロップダウンを開きます。

## オプション

| 値 | 意味 |
|----|------|
| **From Layer（レイヤーから）** | エンティティはそのレイヤーに定義された線種を継承します。DXF では `ByLayer` として表示されます。 |
| **ByBlock** | エンティティは属するブロックの線種を継承します。ブロックの外側では視覚的な効果はありません。 |
| **Continuous（連続）** | 破線パターンのない実線。 |
| **名前付き線種** | 現在の DXF ファイルから読み込まれた任意の線種（例：`DASHED`、`CENTER`、`HIDDEN`、`PHANTOM` など）。ドロップダウンには各パターンのライブプレビューとその定義文字列が表示されます。 |

## 適用方法

選択した線種は変更後に作成されるすべてのエンティティに適用されます。既存のエンティティには遡って適用されません。

既存のエンティティの線種を変更するには、選択してプロパティパネルの **Linetype** フィールドを編集するか、[MatchProperties](../commands/match-properties) を使用して別のエンティティからコピーします。

## 線種スケール

各エンティティには**線種スケール**プロパティ（デフォルト `1`）もあります。破線パターンはこの係数で乗算されます。`2` の値で破線が 2 倍長くなり、`0.5` で半分の長さになります。エンティティを選択した後にプロパティパネルで編集できます。

## 使用可能な線種

ドロップダウンには現在読み込まれた DXF ファイルに存在する線種のみがリストされます。新しく作成されたファイルには `ByLayer`、`ByBlock`、`Continuous` のみが含まれます。DXF をインポートすると、ファイルの `$LTYPE` テーブルに定義されているすべての線種が使用可能になります。

リストにない特定の線種（例：`DASHED2`）が必要な場合は、それを含む DXF ファイルをインポートしてください — その後、現在のセッションのピッカーに線種が表示されます。

## DXF 互換性

線種名はエンティティレコードに文字列として保存されます。`ByLayer` と `ByBlock` は標準の DXF センチネル値です。すべての名前付き線種とその破線パターンはエクスポート時に正確に保持され、LibreCAD、FreeCAD、その他の DXF 対応アプリケーションでロスなく双方向対応しています。
