# テキストエディター

テキストエディターは、`text` コマンドで新しいテキストラベルを配置するとき、または既存のテキストエンティティをダブルクリックしたときに開きます。複数行コンテンツ、文字ごとのフォーマット、ワードラップをサポートします。

## エディターを開く

| 操作 | 結果 |
|-----|------|
| `text` コマンド → 位置をクリック | 新しいテキストエンティティを作成してエディターを開く |
| 既存のテキストエンティティをダブルクリック | そのエンティティのエディターを再度開く |
| エディター内で `Escape` | エディターを閉じてすべての変更を保持 |

## ツールバー

ツールバーはテキストのバウンディングボックスの上に浮いており、パンやズームしてもエンティティにアンカーされたままです。

### 太字・斜体・取り消し線

| ボタン | ショートカット | 機能 |
|-------|---------|------|
| **B** | — | 太字を切り替え |
| *I* | — | 斜体を切り替え |
| ~~S~~ | — | 取り消し線を切り替え |

**切り替えの適用方法：**

- **テキストが選択されている場合** — 選択した文字にのみスタイルを適用します。
- **選択なし、既存テキスト内にカーソル** — エンティティ全体（すべてのセグメント）のスタイルを切り替えます。
- **空のテキストまたは新しいエンティティ** — 空のセグメントにスタイルを保存して、そこからタイプするすべての文字に適用します。

ボタンはハイライト（アクティブ）表示されます：現在の選択範囲内のすべての文字、またはカーソルのすぐ左の文字がそのスタイルに設定されている場合。

### フォント

ドロップダウンにはクロスプラットフォームの書体の厳選セット（Helvetica、Times New Roman、Georgia、Courier New、Verdana、Tahoma、Trebuchet MS、Lucida Console、Impact）がリストされます。上部のオプション **Arial** がデフォルトです。

- **選択あり** — 選択した文字のみのフォントをオーバーライドします。
- **選択なし** — エンティティ全体にフォントを適用します。

選択がない場合、ドロップダウンはカーソルの左の文字のフォントを反映します。

### 高さ

数値フィールドは作図単位での**キャップ高さ**（大文字の高さ）を設定します。

- **選択あり** — 選択した文字の高さをオーバーライドします。エンティティのベース高さとは独立しています。
- **選択なし** — エンティティのベース高さを変更します（個別の高さオーバーライドを持たないすべての文字に適用）。

フィールドはカーソルの左の文字の高さを反映します。エンティティのデフォルトを使用する場合は空白のままにします。

## カーソルとナビゲーション

| キー | 操作 |
|-----|------|
| `←` / `→` | キャレットを 1 文字左右に移動 |
| `Home` | 現在のハードラインの先頭にジャンプ |
| `End` | 現在のハードラインの末尾にジャンプ |
| `Shift` + `←` / `→` | 選択範囲を拡大または縮小 |
| `Backspace` | 左の文字（または選択範囲）を削除 |
| `Delete` | 右の文字（または選択範囲）を削除 |
| `Enter` | 改行を挿入 |
| `Escape` | エディターを閉じる |

カーソルの高さは、下付き文字や上付き文字で使用される小さいサイズを含め、隣接する文字のキャップ高さに自動的に一致します。

## ワードラップ

テキストエンティティに**基準幅**が設定されている場合、長い行はその幅に収まるように単語の境界でソフトラップされます。

エンティティが選択されている状態で基準幅を設定または変更するには、破線のバウンディングボックスの左右端にある薄い長方形の**リサイズグリップ**をドラッグします。ドラッグしながらコンテンツがリアルタイムで再フローされます。

基準幅をゼロに設定すると（グリップを合わせるか、プロパティパネルで値を削除）ワードラップが解除されて行が自由に延びます。

## 複数行テキスト

`Enter` を押してハード改行を挿入します。各ハード行は独立しています — `Home` と `End` は現在のハード行内のみでナビゲートします。

ハード改行と文字ごとのフォーマットは MTEXT 形式を使用して保存され、完全な DXF ラウンドトリップで保持されます。

## DXF 互換性

テキストエンティティは DXF ファイルに **MTEXT** として保存されます。太字と斜体は `\L`、`\K`、`\O`、インラインフォント切り替え（`\f`）としてエンコードされます。文字ごとの高さは `\H` としてエンコードされます。すべてのフォーマットはエクスポート時に保持され、LibreCAD、FreeCAD、その他の DXF 対応アプリケーションで読み込めます。
