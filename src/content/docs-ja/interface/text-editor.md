# テキストエディター

テキストエディターは、編集可能なエンティティを配置またはダブルクリックしたときに開きます。ヘッダーに小さな**モードチップ** — **rich**（アクセントカラー）または **simple**（ミュート）— が表示され、現在のエンティティでアクティブなモードを示します。

## エディターモード

### リッチモード

使用対象：**Text**（MTEXT ラベル）および **Multileader** 注釈。

| 機能 | 動作 |
|------|------|
| Bold / Italic / Strikethrough | 文字ごと（選択に適用、または選択なしの場合はエンティティ全体） |
| Font および Height | 文字ごとのオーバーライド、またはエンティティ全体のデフォルト |
| `Enter` | ハード改行を挿入 |
| `Shift+←/→` | テキスト選択を拡大または縮小 |
| `Home` / `End` | 現在のハードラインの先頭/末尾にジャンプ |
| ワードラップ | 基準幅リサイズグリップ経由でサポート |

### シンプルモード

使用対象：**Dimension Linear**、**Dimension Aligned**、**Dimension Angular**、**Dimension Radius**、**Dimension Diameter**。

エディターには現在の寸法の表示ラベルがあらかじめ入力されているため、カーソルを移動して値を直接編集できます。

| 機能 | 動作 |
|------|------|
| Bold / Italic / Font / Height | 利用可能 — **ラベル全体**に一度に適用 |
| 文字ごとのフォーマット | サポートされていません |
| `Enter` | 値を**確定**してエディターを閉じる（改行なし） |
| 複数行 | サポートされていません |
| ワードラップ | サポートされていません |

## エディターを開く

| 操作 | 結果 |
|------|------|
| `text` コマンド → 位置をクリック | 新しいテキストエンティティを作成してエディターを開く（**リッチ**） |
| 既存の **Text** エンティティをダブルクリック | **リッチ**モードでエディターを再度開く |
| 既存の **Multileader** をダブルクリック | **リッチ**モードでエディターを開く |
| **寸法**エンティティをダブルクリック | **シンプル**モードでエディターを開く |
| エディター内で `Escape` | エディターを閉じてすべての変更を保持 |

## ツールバー

ツールバーはテキストのバウンディングボックスの上に浮いており、パンやズームしてもエンティティにアンカーされたままです。

### 太字・斜体・取り消し線

| ボタン | ショートカット | 機能 |
|-------|---------|------|
| **B** | — | 太字を切り替え |
| *I* | — | 斜体を切り替え |
| ~~S~~ | — | 取り消し線を切り替え |

**切り替えの適用方法：**

- **テキストが選択されている場合** — 選択した文字にのみスタイルを適用します。
- **選択なし、既存テキスト内にカーソル** — エンティティ全体（すべてのセグメント）のスタイルを切り替えます。
- **空のテキストまたは新しいエンティティ** — 空のセグメントにスタイルを保存して、そこからタイプするすべての文字に適用します。

ボタンはハイライト（アクティブ）表示されます：現在の選択範囲内のすべての文字、またはカーソルのすぐ左の文字がそのスタイルに設定されている場合。

### フォント

ドロップダウンは、利用可能な書体を **Default**（組み込みのサンセリフフォント）、**User**（アップロードした自分のフォント。ある場合）、**Free**（同梱の Google Fonts のセット）、**System**（Helvetica、Times New Roman、Georgia、Courier New、Verdana、Tahoma、Trebuchet MS、Lucida Console、Impact などの一般的な OS フォント）にグループ分けします。

- **選択あり** — 選択した文字のみのフォントをオーバーライドします。
- **選択なし** — エンティティ全体にフォントを適用します。

選択がない場合、ドロップダウンはカーソルの左の文字のフォントを反映します。

組み込みのリストに限定されません — ツールバーの **Font Manager（フォントマネージャー）** ボタンをクリックすると、独自の `.ttf` ファイルをアップロードして **User** グループに追加できます。詳細は [Font Manager](../../commands/font-manager/) を参照してください。

### 高さ

数値フィールドは作図単位での**キャップ高さ**（大文字の高さ）を設定します。

- **選択あり** — 選択した文字の高さをオーバーライドします。エンティティのベース高さとは独立しています。
- **選択なし** — エンティティのベース高さを変更します（個別の高さオーバーライドを持たないすべての文字に適用）。

フィールドはカーソルの左の文字の高さを反映します。エンティティのデフォルトを使用する場合は空白のままにします。

## カーソルとナビゲーション

| キー | 操作 |
|-----|------|
| `←` / `→` | キャレットを 1 文字左右に移動 |
| `Home` | 現在のハードラインの先頭にジャンプ |
| `End` | 現在のハードラインの末尾にジャンプ |
| `Shift` + `←` / `→` | 選択範囲を拡大または縮小 |
| `Backspace` | 左の文字（または選択範囲）を削除 |
| `Delete` | 右の文字（または選択範囲）を削除 |
| `Enter` | 改行を挿入 |
| `Escape` | エディターを閉じる |

カーソルの高さは、下付き文字や上付き文字で使用される小さいサイズを含め、隣接する文字のキャップ高さに自動的に一致します。

## コピー、切り取り、貼り付け

| キー | 操作 |
|-----|------|
| `Ctrl+C` / `Cmd+C` | 選択したテキストをコピー |
| `Ctrl+X` / `Cmd+X` | 選択したテキストを切り取り |
| `Ctrl+V` / `Cmd+V` | カーソル位置に貼り付け |

コピーと切り取りにはテキスト選択が必要です。貼り付けたテキストは常にプレーンです — コピー時の書式を引き継ぐのではなく、カーソル位置に既にある書式（太字、斜体、フォント、高さ）を適用します。

**リッチモード**では、貼り付けたテキスト内の改行が保持されます。**シンプルモード**では、寸法ラベルは単一行のため、改行は削除されます。

## ワードラップ

テキストエンティティに**基準幅**が設定されている場合、長い行はその幅に収まるように単語の境界でソフトラップされます。

エンティティが選択されている状態で基準幅を設定または変更するには、破線のバウンディングボックスの左右端にある薄い長方形の**リサイズグリップ**をドラッグします。ドラッグしながらコンテンツがリアルタイムで再フローされます。

基準幅をゼロに設定すると（グリップを合わせるか、プロパティパネルで値を削除）ワードラップが解除されて行が自由に延びます。

## 複数行テキスト

`Enter` を押してハード改行を挿入します。各ハード行は独立しています — `Home` と `End` は現在のハード行内のみでナビゲートします。

ハード改行と文字ごとのフォーマットは MTEXT 形式を使用して保存され、完全な DXF ラウンドトリップで保持されます。

## DXF 互換性

テキストエンティティは DXF ファイルに **MTEXT** として保存されます。太字と斜体は `\L`、`\K`、`\O`、インラインフォント切り替え（`\f`）としてエンコードされます。文字ごとの高さは `\H` としてエンコードされます。すべてのフォーマットはエクスポート時に保持され、LibreCAD、FreeCAD、その他の DXF 対応アプリケーションで読み込めます。
