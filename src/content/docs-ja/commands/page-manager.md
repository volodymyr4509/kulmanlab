# PageManager（ページマネージャー）

`PageManager` コマンドはアクティブなレイアウトのページ設定ダイアログを開き、用紙フォーマット、向き、作図単位とミリメートルのスケールを変更できます。

## ページマネージャーを開く

- ターミナルに `PageManager` と入力する、**または**
- 下部のレイアウトタブを右クリックして**ページマネージャー**を選択します。

## 設定項目

| 設定 | 説明 |
|-----|------|
| **用紙フォーマット** | 標準用紙サイズ（A4、A3、A2、Letter など） |
| **向き** | 縦向きまたは横向き |
| **スケール** | mm あたりの作図単位 — モデル座標が用紙にどのようにマッピングされるかを制御 |

用紙サイズまたはスケールを変更すると、レイアウト内のすべての既存ビューポートが比例してリスケールされ、位置が正しく保たれます。

## 注記

- PageManager は用紙レイアウトにのみ適用されます。モデル空間では使用できません。
- 新しいレイアウトを追加するには、下部のタブバーの **+** ボタンをクリックします。
- レイアウトの名前変更または削除には、タブを右クリックします。
