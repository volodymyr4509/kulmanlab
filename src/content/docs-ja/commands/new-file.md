# New File（新規ファイル）

**New File** コマンドはキャンバスをクリアして新しい空の図面を開始します。タイムスタンプ付きの一意のファイル名が自動的に生成されます。

## 新規ファイルの作成方法

File パネルの **New File** ツールバーボタン（新規ページアイコン）をクリックします。プロンプトや確認ダイアログなしにキャンバスが即座にクリアされます。

## 新規ファイルの内容

新しく作成されたファイルは以下の状態で開始します：

- キャンバス上に**エンティティなし**。
- 色が白で線種が `Continuous` の `0` という名前の**デフォルトレイヤー** 1 つ。
- `kulman-Mon01_HH:MM:SS.dxf` 形式の**生成されたファイル名**（例：`kulman-May22_14:30:00.dxf`）。

ファイルはブラウザーストレージに自動保存され、[最近のファイル](../files/)に表示されます。

## 警告 — 未保存の作業は破棄されます

**New File** をクリックすると、現在のキャンバス上のすべてのエンティティが警告なしに破棄されます。現在の図面を保持したい場合は、先に[エクスポート](../export/)してください。

## New File と Import の使い分け

| 状況 | 推奨操作 |
|-----|---------|
| 最初から図面を作成する | **New File** |
| 既存の DXF または JSON ファイルを開く | [Import](../import/) |
| 図面をコピーしてバリアントを作成する | 現在のファイルを[エクスポート](../export/)してから、コピーを[インポート](../import/) |

## 関連コマンド

- [Import](../import/) — 既存の DXF または JSON 図面を開く
- [Export](../export/) — 新規作成前に図面をダウンロード
- [Files](../files/) — ブラウザーストレージから以前の図面を復元
