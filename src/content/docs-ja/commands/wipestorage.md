# Wipe Storage（ストレージのクリア）

`wipestorage` コマンドは KulmanLab CAD 用に**ブラウザーに保存されているすべてのデータ** — 保存済みのすべてのファイル、レイヤーと線種テーブル、元に戻す履歴 — を完全に削除します。その後ページが自動的にリロードされます。

:::danger 元に戻せません
この操作は取り消せません。ブラウザーに保存されているすべてのファイルが削除されます。このコマンドを実行する前に、保持したい図面を `.json` または `.dxf` ファイルとしてエクスポートしてください。
:::

## 使用するタイミング

- ブラウザーストレージが壊れていてアプリのファイルの読み込みまたは保存が失敗する。
- アプリを完全にクリーンな状態にリセットしたい。
- ブラウザーまたはデバイスを切り替えて、ローカルコピーが不要になった。

## 実行方法

1. ターミナルに `wipestorage` と入力して **Enter** を押します。
2. ターミナルがプロンプトを表示します：*Wipe all browser local storage? Type YES to confirm*
3. `YES`（大文字小文字問わず）と入力して **Enter** を押します。

アプリはデータベースを削除してページをリロードします。`YES` 以外の文字を入力して **Enter** を押すか、**Escape** を押すと、コマンドはキャンセルされて何も削除されません。

## 削除されるもの

| データ | 削除 |
|-----|-----|
| ブラウザーに保存されているすべてのファイル | あり |
| 各ファイルのレイヤーと線種テーブル | あり |
| 各ファイルの元に戻す / やり直し履歴 | あり |

影響を受けるのは**このブラウザー**にローカルに保存されているデータのみです。`.json` または `.dxf` としてエクスポート済みのファイルは影響を受けません。
