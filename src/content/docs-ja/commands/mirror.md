# Mirror（鏡像）

`mirror` コマンドは 2 点で定義した軸で選択エンティティの鏡像コピーを作成します。元のエンティティは**常に保持されます** — [Move](./move) や [Rotate](./rotate) とは異なり、Mirror は既存のエンティティを変更せず、新しいエンティティを追加するのみです。

## 2 つの開始方法

**先に選択してから鏡像** — 先にエンティティを選択してから実行します：

1. キャンバス上で 1 つ以上のエンティティを選択します。
2. ターミナルに `mirror` と入力するか、**Mirror** ツールバーボタンをクリックします。
3. **鏡像軸の最初の点をクリック**するか、`X,Y` と入力して **Enter** を押し正確な座標を指定します。
4. **2 番目の点をクリック** — 鏡像コピーが配置されてコマンドが終了します。ここでも座標入力が使用できます。

**コマンドを実行してから選択** — 何も選択しない状態でコマンドを起動します：

1. `mirror` と入力するか、ツールバーボタンをクリックします。
2. **オブジェクトを選択** — クリックして選択切り替え、またはドラッグしてエリア選択します。
3. **Enter** または **Space** を押して選択を確定します。
4. **最初の点をクリック**、次に**2 番目の点をクリック**して鏡像軸を指定します（両ステップで座標入力可能）。

```
  元:           鏡像軸:       結果:
                |
  [エンティティ A]  →  |  →  [エンティティ A] + [鏡像 A]
                |
```

2 番目の軸点を位置決めしている間、鏡像コピーのライブプレビューがカーソルに追従します。

## 鏡像軸

軸は 2 つのクリック点を通る無限直線です。任意の角度にできます：

- カーソルを **45° スナップ軸**（0°, 45°, 90°, 135°, …）に近づけると軸がその角度にロックされます — 水平、垂直、または斜め方向の正確な反転に便利です。
- スナップゾーンから離れた場所でクリックすると自由な角度の軸になります。

## 座標入力

軸点ステップのどちらかで、クリックの代わりに正確な位置を入力できます：

1. X 値を入力します。
2. `,` を押します — ターミナルに `[X], [Y{カーソル}]` と表示されます。
3. Y 値を入力します。
4. **Enter** を押して確定します。

## キーボードリファレンス

| キー | 操作 |
|-----|------|
| `Enter` / `Space` | 選択を確定 |
| `0`〜`9`, `.`, `-` | X 座標入力を開始 |
| `,` | X をロックして Y 入力に移動 |
| `Backspace` | 最後に入力した文字を削除 |
| `Enter` | 入力した座標を確定 |
| `Escape` | キャンセルしてリセット |

## コマンド中の選択

| 方法 | 動作 |
|-----|------|
| **クリック** | カーソル下のエンティティを選択切り替え |
| **右ドラッグ**（厳密） | ボックス内に完全に収まるエンティティを追加 |
| **左ドラッグ**（交差） | ボックスと交差するエンティティを追加 |
| **Enter** / **Space** | 選択を確定 |

## 鏡像されるもの

すべてのエンティティタイプがサポートされています。ジオメトリは軸で数学的に反転されます：

| エンティティ | 変更内容 |
|---------|---------|
| 直線 | 両端点を反転 |
| 円 | 中心を反転；半径は変わらない |
| 円弧 | 中心を反転；開始角度と終了角度を軸で再計算 |
| 楕円 | 中心を反転；長軸方向を軸で反転 |
| ポリライン / 長方形 | すべての頂点を反転 |
| テキスト | アンカー点を反転；テキスト文字列は**反転しない** |
| スプライン | すべての制御頂点 / フィット点を反転 |

## Mirror と Copy の比較

| | Mirror | Copy |
|---|--------|------|
| 元のエンティティ | 常に保持 | 常に保持 |
| 新しいエンティティの位置 | 軸で反転 | 変位ベクトルでオフセット |
| 適している用途 | 対称設計、二側面の形状 | 任意の方向への図形の複製 |
