# Leader−

`leader-` コマンドは既存のマルチ引出線から矢印アームを 1 本削除します。テキストラベル、ドッグレッグ、および残りのアームはすべて保持されます — 選択されたアームのみが削除されます。アームが 1 本しかないマルチ引出線からはアームを削除できません。

## アームの削除

1. ターミナルに `leader-` と入力します。
2. **2 本以上のアームを持つマルチ引出線をクリック**します。クリックした引出線のアームが 1 本のみの場合、ターミナルにエラーが表示され、有効な選択を待ちます。
3. **削除したいアームの近くにカーソルを移動** — 最も近いアームにマーカーが表示されてハイライトされます。
4. **クリック**してそのアームを削除します。

アームが削除されてもコマンドはアクティブのままで、すぐに別の引出線（または同じ引出線）をクリックしてさらにアームを削除できます。**Enter**、**Space**、または **Escape** を押して終了します。

```
  前:                   後:
  ◄── アーム 1           ◄── アーム 1
       \                       \
        ●──── ドッグレッグ ──── テキスト  ●──── ドッグレッグ ──── テキスト
       /
  アーム 2 ──►  ← このアームが削除される
```

## 最も近いアームの判定方法

コマンドはカーソルから各アームの線分（最後のアーム点からドッグレッグまでのセグメントを含む）への垂直距離を測定します。最も小さい距離のアームがハイライト表示され、クリックで削除されます。

## キーボードリファレンス

| キー | 操作 |
|-----|------|
| `Enter` / `Space` | アーム削除を終了 |
| `Escape` | キャンセルしてリセット |

## 注記

- **アームが 1 本だけ**の引出線は保護されています — 削除する前に先にアームを追加する必要があります。
- どのアームを削除しても、ドッグレッグ位置とテキスト内容は常に保持されます。
