# Rectangle（長方形）

`rectangle` コマンドは対角の 2 点クリックで定義される軸平行な長方形を作図します。結果は各コーナーに 1 つずつ、4 頂点を持つ**閉じた `LWPOLYLINE`**として保存されます。専用の長方形エンティティタイプはありません：作成後は他のあらゆる[ポリライン](../polyline/)と同じように動作し、すべてのポリライン編集が適用されます。

## 長方形の作図

1. ターミナルに `rectangle` と入力するか、**Rectangle** ツールバーボタンをクリックします。
2. **最初のコーナーをクリック**するか、`X,Y` と入力して **Enter** を押し正確な座標を指定します。
3. **対角のコーナーをクリック** — 長方形が即座に配置されてコマンドが終了します。ここでも座標入力が使用できます。

```
  ● (最初のクリック)────────────┐
  |                            |
  |   ステップ 2 以降は          |
  |   カーソルに追従するプレビュー  |
  └──────────────────────────── ● (2 回目のクリック)
```

2 回のクリックは対角のコーナーの任意の組み合わせで構いません — 左上＋右下、または左下＋右上など。順序は関係ありません。

## 座標入力

いずれかのコーナーステップで正確な位置を入力できます：

1. X 値を入力します。
2. `,` を押します — ターミナルに `[X], [Y{カーソル}]` と表示されます。
3. Y 値を入力します。
4. **Enter** を押してコーナーを配置します。

## キーボードリファレンス

| キー | 操作 |
|-----|------|
| `0`〜`9`, `.`, `-` | X 座標入力を開始 |
| `,` | X をロックして Y 入力に移動 |
| `Enter` | 入力した座標を確定 |
| `Escape` | キャンセル |

辺は常に水平および垂直です — 長方形コマンドには角度ロックはありません。

## グリップ編集 — 作成後の変形

選択した長方形にはすべての頂点と各辺の中点にグリップが表示されます：

| グリップ | 位置 | 機能 |
|---------|------|------|
| **コーナー** | 4 頂点のそれぞれ | ドラッグしてその頂点を移動；隣接する 2 辺が追従して伸縮 — 対角のコーナーは固定 |
| **辺の中点** | 4 辺のそれぞれの中央 | ドラッグしてその辺の両端点を一緒に移動、辺の長さと角度を維持 |

コーナーグリップをドラッグすると、長方形が非長方形の四角形になります。サイズが異なる長方形だけが必要な場合は、辺がほぼ直交する状態でコーナーをドラッグするか、削除して新しい長方形を作図してください。

## 長方形の選択

長方形はポリラインなので、選択も同様の方法で動作します：

| 方法 | 動作 |
|-----|------|
| **クリック** | 4 辺のいずれかでクリックが当たった場合に選択 |
| **右ドラッグ**（厳密） | 4 頂点すべてが選択ボックス内に収まる必要がある |
| **左ドラッグ**（交差） | ボックス境界を横切る任意の辺が長方形全体を選択 |

## サポートされる編集コマンド

すべてのポリライン編集コマンドが適用されます。トリムと延長は[直線](../line/)専用で、長方形には機能しません：

| コマンド | 長方形への効果 |
|---------|------------|
| [Move](../move/) | 4 頂点すべてを同じ変位で移動 |
| [Copy](../copy/) | 新しい位置に同一の長方形を作成 |
| [Rotate](../rotate/) | 選択した基点を中心に 4 頂点すべてを回転 |
| [Mirror](../mirror/) | 鏡像軸で 4 頂点すべてを反転 |
| [Scale](../scale/) | 基点から 4 頂点すべてを均一に拡大縮小 |
| [Offset](../offset/) | 固定距離で平行な（インセットまたはアウトセット）長方形を作成 |
| [Delete](../delete/) | 図面から長方形を削除 |

## プロパティ

長方形を選択するとプロパティパネルにポリラインと同じフィールドが表示されます：

**一般**

| プロパティ | デフォルト | 説明 |
|---------|---------|------|
| Color | 256 (ByLayer) | ACI カラーインデックス |
| Layer | `0` | レイヤー割り当て |
| Linetype | ByLayer | 名前付き線種パターン |
| Linetype Scale | 1 | 線種パターンのスケール係数 |
| Thickness | 0 | 押し出し厚さ |

**ジオメトリ**

| プロパティ | 説明 |
|---------|------|
| Closed | 長方形の場合は常に `true` |
| Vertex Count | 変更されていない長方形の場合は常に `4` |
| Vertices | 4 コーナーすべての座標 |

## 長方形、ポリライン、直線の比較

| | 長方形 | ポリライン | 直線 |
|---|---------|---------|------|
| 作図方法 | 2 クリック（コーナー） | 各頂点をクリック | 各端点をクリック |
| エンティティタイプ | 閉じた `LWPOLYLINE` | 開いたまたは閉じた `LWPOLYLINE` | セグメントごとに `LINE` |
| 辺が常に直交 | あり（作成時） | なし | なし |
| トリム / 延長 | なし | なし | あり |
| 適している用途 | ボックス、フレーム、長方形エリア | 任意のアウトラインとパス | 個別セグメント、補助線 |

## DXF — LWPOLYLINE エンティティ

長方形は閉じた 4 頂点の `LWPOLYLINE` エンティティとして保存されます。すべてのプロパティ（頂点座標、色、レイヤー、線種、線種スケール、厚さ）はロスなく双方向対応しています。

DXF には専用の `RECTANGLE` タイプは存在しません。ファイルを再度開くと、図形は長方形ではなく閉じた 4 頂点ポリラインとして表示されます。`LWPOLYLINE` をサポートする任意の DXF ビューアまたはエディター（LibreCAD、FreeCAD など）で正しく表示されます。
