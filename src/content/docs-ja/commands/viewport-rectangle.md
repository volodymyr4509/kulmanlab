# ViewportRectangle（ビューポート長方形）

`ViewportRectangle` コマンドは対角の 2 点を選択することでアクティブな用紙レイアウトに新しいビューポートを作成します。用紙レイアウト空間でのみ使用できます。

## ビューポートの作成

1. 画面下部のタブを使用して用紙レイアウトに切り替えます。
2. ターミナルに `ViewportRectangle` と入力するか、**Viewport Rectangle** ツールバーボタンをクリックします。
3. **最初のコーナーをクリック**するか、`X,Y` と入力して **Enter** を押し正確な座標を指定します。
4. **対角のコーナーをクリック** — ビューポートが即座に配置されます。ここでも座標入力が使用できます。

新しいビューポートはレイアウトのデフォルトスケールでモデル全体を表示します。ビューポート内でスクロールホイールを使用してズーム、中ボタンドラッグでモデルビューをパンできます。

## 座標入力

いずれかのコーナーステップで正確な座標を入力できます：

1. X 値を入力します。
2. `,` を押します — ターミナルに `[X], [Y{カーソル}]` と表示されます。
3. Y 値を入力します。
4. **Enter** を押して点を配置します。

## キーボードリファレンス

| キー | 操作 |
|-----|------|
| `0`〜`9`, `.`, `-` | X 座標入力を開始 |
| `,` | X をロックして Y 入力に移動 |
| `Enter` | 入力した座標を確定 |
| `Escape` | キャンセル |

## ビューポートの編集

ビューポートを配置した後、クリックして選択します：

- **端またはコーナーをドラッグ**してリサイズします。
- **中心グリップをドラッグ**して移動します。
- コントロールバーの**スケールセレクター**を使用して正確なスケールを設定します（例：1:50）。リストにないスケールを入力するには、ドロップダウンの下部の入力フィールドに直接入力します — 比率形式（`1:200`、`5:1`）または小数（`0.005`）を受け付けて、**Enter** を押します。
- ビューポートを右クリックして**ロック**を使用すると、誤った変更を防止できます。

## 注記

- ViewportRectangle は用紙レイアウトタブがアクティブな場合にのみ使用できます。モデル空間で実行するとエラーメッセージが表示されて終了します。
- 既存のビューポートをコピーするには [ViewportCopy](../viewport-copy/) を使用してください。
