# Fillet

`fillet` コマンドは指定した半径の接線円弧を挿入し、各直線を円弧が始まる点まで切り戻すことで、2 つの [Line](../line/) エンティティ間の角を丸めます。結果は両直線を滑らかに結ぶ丸みのついたコーナーです。

フィレットは **Line エンティティのみ**で機能します。

## フィレットの使用手順

1. ターミナルに `fillet` と入力するか、**Fillet** ツールバーボタンをクリックします。
2. **フィレット半径を入力**して **Enter** を押します。
3. **最初の直線をクリック** — クリックした部分が交点のどちら側を保持するかを決定します。
4. **2 番目の直線の上にカーソルを移動** — 破線の円弧プレビューでフィレットの結果が表示されます。保持したい側にカーソルを移動します。
5. **クリック**して適用。両直線がトリムされ円弧が挿入されます。

```
  前:                     後（フィレット半径 r）:

  ──────────────              ──────────╮
                │                        ╰────
                │
```

## 交差する直線の側面選択

2 本の直線が互いに交差する場合、フィレットはクリック位置によって定義されるコーナーに適用されます — 各直線の**カーソルと同じ側**の部分が保持されます。

- 最初の直線の一端近くをクリックしてその半分を選択します。
- 2 番目の直線の希望する半分にカーソルを移動すると、破線プレビューがリアルタイムで更新されます。

## コマンドが作成するもの

- 最初の直線の交点に最も近い端点が接点 **T1** に移動します。
- 2 番目の直線の交点に最も近い端点が接点 **T2** に移動します。
- **T1** から **T2** へ、両直線に接する新しい Arc エンティティが挿入されます。

挿入された円弧は現在の線幅・色・レイヤー・線種設定を継承します。

## キーボードリファレンス

| キー | 操作 |
|-----|------|
| `0`〜`9`, `.` | 半径値に数字を追加 |
| `Backspace` | 最後に入力した文字を削除 |
| `Enter` | 入力した半径を確定して直線選択に進む |
| `Escape` | キャンセルしてリセット |

## サポートされるエンティティ

| エンティティ | サポート |
|--------|---------|
| Line | あり — 最初と 2 番目のエンティティとして |
| Arc, Circle, Ellipse, Polyline | なし |
| Text, Spline, Dimension, Leader | なし |

## Fillet と Chamfer の比較

| | Fillet | Chamfer |
|---|--------|---------|
| 角の種類 | 丸い円弧 | 直線カット |
| 入力 | 1 つの半径 | 2 つの距離（d1, d2） |
| 挿入エンティティ | 円弧 | 直線 |
| サポートエンティティ | Line のみ | Line と Polyline |
