# LayerMatch

`LayerMatch` コマンドは選択したエンティティのレイヤーを、クリックしたソースエンティティのレイヤーに合わせて変更します。レイヤーマネージャーを開かずに、オブジェクトのグループを正しいレイヤーに移動する最も素早い方法です。

## ワークフロー

**先に選択してからマッチ**：

1. レイヤーを変更したいエンティティを選択します。
2. `LayerMatch` と入力するか、**Layer Match** ツールバーボタン（ペイントアイコン）をクリックします。
3. **ソースオブジェクトをクリック** — コピーしたいレイヤーのエンティティをクリックします。
4. 選択したすべてのエンティティが即座にソースオブジェクトのレイヤーに移動します。

**コマンドを実行してから選択**：

1. 何も選択していない状態で `LayerMatch` と入力するか、ツールバーボタンをクリックします。
2. **対象オブジェクトを選択** — クリックして個々のエンティティを選択切り替え、またはドラッグしてエリア選択します。
3. **Enter** または **Space** を押して選択を確定します。
4. **ソースオブジェクトをクリック** — そのレイヤーがすべての対象に適用されます。

## キーボードリファレンス

| キー | 操作 |
|-----|------|
| `Enter` / `Space` | 対象選択を確定してソース選択フェーズに進む |
| `Escape` | リセット — 対象選択に戻るか、完全にキャンセル |

## 動作の詳細

- 変更されるのは `layer` プロパティのみです — 色、線種、線幅、ジオメトリは変更されません。
- ソースオブジェクト自体は変更されません。
- ソースをクリックした後、コマンドは終了します。
- ソース選択フェーズで空のキャンバスをクリックしても何も起こりません。
