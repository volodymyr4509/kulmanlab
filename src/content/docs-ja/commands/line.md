# Line

`line` コマンドは DXF モデル内で独立した `LINE` エンティティとして保存される直線セグメントを作図します。各セグメントを配置した後もコマンドはアクティブのまま、その端点を新しい始点として再利用するので、連続したパスを 1 セグメントずつ構築できます。[Polyline](../polyline/)とは異なり、連鎖した直線は独立したエンティティのまま — それぞれを隣接するものに影響を与えずにトリム、延長、または削除できます。

## 直線の作図

1. ターミナルに `line` と入力するか、**Line** ツールバーボタンをクリックします。
2. **始点をクリック**するか、`X,Y` と入力して **Enter** を押し正確な座標を指定します。
3. **終点をクリック** — セグメントが配置されて終点が次の始点になります。ここでも座標入力が使用できます。
4. さらにセグメントを連鎖するにはクリック（または入力）を続けます。
5. **Enter** または **Escape** を押して終了します。

```
  ●──────────●──────────●──────────●
 始点     2 回目クリック  3 回目クリック   Enter で終了
            (自動的に次の始点になる)
```

セグメントが 1 本だけ必要な場合は、ステップ 3 の直後に **Enter** または **Escape** を押します。

## 座標入力

クリックの代わりに始点または以降のいずれかの点に正確な位置を入力できます：

1. X 値を入力します（数字、`.`、または `-`）。
2. `,` を押します — ターミナルに `[X], [Y{カーソル}]` と表示されます。
3. Y 値を入力します。
4. **Enter** を押して点を配置します。

## 角度ロックと正確な長さ入力

点を配置した後にカーソルを移動すると、コマンドは 45° スナップ軸（0°, 45°, 90°, 135°, …）を検出します。次の条件を満たすと角度が**ロック**されます：

- カーソルがアンカーから少なくとも **5 × グリップサイズ**離れている、**かつ**
- 最も近い軸からの垂直距離が **1 グリップサイズ**以内。

ロック中はプレビューが軸にスナップし、正確な長さを入力できます：

| キー | 操作 |
|-----|------|
| `0`〜`9`, `.` | 長さ値に数字を追加 |
| `-` | 負の長さ — 軸方向を反転（最初の文字のみ） |
| `Backspace` | 最後に入力した文字を削除 |
| `Enter` | 入力した距離で終点を配置 |

累積値はターミナルにリアルタイムで表示されます（例：`click end point or enter length: 12.5`）。ロック中にクリックすると、クリックが軸に投影されるので終点は常に軸上に正確に配置されます。

アンカー点の近くに戻るとロックが解除されます。

## キーボードリファレンス

| キー | 操作 |
|-----|------|
| `0`〜`9`, `.`, `-` | X 座標入力を開始、または角度ロック中の距離 |
| `,` | X をロックして Y 入力に移動 |
| `Backspace` | 最後に入力した文字を削除 |
| `Enter` | 入力した座標または長さを確定、何も入力していない場合は連鎖を終了 |
| `Escape` | 連鎖を終了して終了 |

## グリップ編集 — 端点のストレッチ

選択した直線には 3 つのグリップが表示されます：

| グリップ | 位置 | 機能 |
|---------|------|------|
| **始点** | 最初の端点 | ドラッグして位置変更 — 終点は固定 |
| **中点** | 直線の中央 | 直線全体の**移動**を起動 |
| **終点** | 2 番目の端点 | ドラッグして位置変更 — 始点は固定 |

一方の端点をストレッチしても、もう一方は影響を受けません。これは[Polyline](../polyline/)のグリップ編集とは異なります — ポリラインでは頂点を移動するとパス全体が変形します。

## 直線の選択

| 方法 | 動作 |
|-----|------|
| **クリック** | セグメントのヒットテスト距離内にクリックした場合に直線を選択 |
| **右ドラッグ**（厳密） | 両端点がボックス内に収まる場合のみ選択 |
| **左ドラッグ**（交差） | セグメントの任意の部分がボックス境界を横切る場合に選択 |

## サポートされる編集コマンド

直線は[Trim](../trim/)と[Extend](../extend/)が機能する**唯一**のエンティティです。すべての標準変形コマンドも適用できます：

| コマンド | 直線への効果 |
|---------|------------|
| [Move](../move/) | 両端点を同じ変位で移動 |
| [Copy](../copy/) | 新しい位置に同一の直線を作成 |
| [Rotate](../rotate/) | 選択した基点を中心に両端点を回転 |
| [Mirror](../mirror/) | 鏡像軸で両端点を反転 |
| [Scale](../scale/) | 基点から両端点を均一に拡大縮小 |
| [Offset](../offset/) | 固定の垂直距離で平行な直線を作成 |
| [Trim](../trim/) | 交差点で直線をカット — **直線のみ** |
| [Extend](../extend/) | 最も近い端点を境界まで伸ばす — **直線のみ** |
| [Delete](../delete/) | 図面から直線を削除 |

## プロパティ

直線を選択するとプロパティパネルに DXF `LINE` レコードが持つすべてのフィールドが表示されます：

**一般**

| プロパティ | デフォルト | 説明 |
|---------|---------|------|
| Color | 256 (ByLayer) | ACI カラーインデックス |
| Layer | `0` | レイヤー割り当て |
| Linetype | ByLayer | 名前付き線種パターン |
| Linetype Scale | 1 | 線種パターンのスケール係数 |
| Thickness | 0 | 押し出し厚さ |

**ジオメトリ**

| プロパティ | 説明 |
|---------|------|
| Start X / Start Y | 最初の端点座標 |
| End X / End Y | 2 番目の端点座標 |

すべてのフィールドはコマンドを再実行せずに、パネルで直接編集できます。

## Line と Polyline — 使い分け

| | Line | Polyline |
|---|------|---------|
| エンティティ数 | セグメントごとに 1 つの `LINE` | パス全体で 1 つの `LWPOLYLINE` |
| トリム / 延長 | あり — セグメントごと | なし |
| 閉じた図形 | なし | あり（クローズフラグ） |
| グリップ編集 | 個々の端点をストレッチ | パス上の任意の頂点を移動 |
| 適している用途 | 補助線、単一セグメント、トリムする図形 | 輪郭、アウトライン、全体を保持する図形 |

## DXF — LINE エンティティ

直線は DXF ファイルに `LINE` エンティティとして保存されます。すべてのプロパティ（始点/終点座標、色、レイヤー、線種、線種スケール、厚さ）はロスなく双方向対応しています。`LINE` エンティティを含む DXF を開くと、エディターで完全に編集可能な `Line` オブジェクトになります。

エディターで作図した直線も保存時に `LINE` エンティティとして書き出されるので、LibreCAD、FreeCAD、その他の DXF 対応アプリケーションで読み込めます。
