# Zoom In（ズームイン）

`zoomin` コマンドは現在のズームレベルを **1.5×** 倍にしてすぐに終了します。ビューポートの中点を中心にズームします。カーソルではなく画面の中央に向けてズームするスクロールホイール 1 ティックのツールバー版です。

## ズームイン方法

**Zoom In** ツールバーボタンをクリックするか、ターミナルに `zoomin` と入力します。ズームは即座に適用されてコマンドが終了します — キャンバスのクリックは不要です。

## 1.5× ステップの動作

| 現在のズーム | Zoom In 1 回後 |
|------------|----------------|
| 1.00× | 1.50× |
| 1.50× | 2.25× |
| 10.00× | 15.00× |
| 6,667× | 10,000×（上限） |

ズームレベルは常に `zoom` ラベルの隣の**キャンバス右下**に表示されます。上限は **10,000×** で、それ以上はステップしても何も起こりません。

## ツールバーの Zoom In とスクロールホイールの比較

| | Zoom In ボタン | スクロールホイール |
|---|--------------|-------------|
| ズームの中心 | ビューポートの中点 | カーソル位置 |
| ステップサイズ | クリックごとに 1.5× | ティックごとに約 1.1× |
| 起動が必要 | いいえ | いいえ — 常に使用可能 |
| 適している用途 | 粗いナビゲーション | 精密なカーソル指定ズーム |

## キーボードリファレンス

このコマンドにキーボードショートカットはありません。代わりにスクロールホイールを使用してください — コマンドを起動せずに常に使用できます。
