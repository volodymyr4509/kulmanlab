# Spline Fit（スプラインフィット）

`splinefit` コマンドはクリックしたすべての点を通る 3 次スプライン（補間曲線）を作図します。[スプライン CV](./spline-cv) では制御頂点に向けて曲線が引き寄せられるだけですが、ここでは曲線がクリックした各座標を正確に通るように強制されます。内部的にエディターはこれを達成するために制御頂点をフィットし、それらの CV はフィット点と共に DXF ファイルに保存されます。

## フィット点でスプラインを作図

1. ターミナルに `splinefit` と入力するか、**Spline Fit** ツールバーボタンをクリックします。
2. **クリックしてフィット点を配置** — 曲線は各点を通過します。または `X,Y` と入力して **Enter** を押し正確な座標を指定します。
3. **Enter** を押して終了します（最低 2 点が必要）。

```
  ●──────●──────●──────●  ← 曲線は各クリックを正確に通過
  p1     p2     p3     p4
```

カーソルを移動すると現在の補間曲線（カーソル位置の見込みの次の点を含む）を示すライブプレビューが表示されます。**Escape** を押すと配置済みの点がすべて破棄されてコマンドが終了します。

## 座標入力

任意のフィット点に対してクリックの代わりに正確な位置を入力できます：

1. X 値を入力します。
2. `,` を押します — ターミナルに `[X], [Y{カーソル}]` と表示されます。
3. Y 値を入力します。
4. **Enter** を押してフィット点を配置します。

## キーボードリファレンス

| キー | 操作 |
|-----|------|
| `0`〜`9`, `.`, `-` | X 座標入力を開始 |
| `,` | X をロックして Y 入力に移動 |
| `Backspace` | 最後に入力した文字を削除 |
| `Enter` | 入力した座標を確定、または入力が進行中でなく ≥ 2 点ある場合はスプラインを終了 |
| `Escape` | すべての点を破棄して終了 |

## グリップ編集 — フィット点による変形

選択したフィットスプラインにはフィット点ごとに 1 つのグリップが表示されます：

| グリップ | 位置 | 機能 |
|---------|------|------|
| **フィット点** | 各クリック位置 | ドラッグしてそのフィット点を移動 — 曲線全体が新しい位置を通るように再補間 |

1 つのグリップをドラッグすると隣接するセグメントだけでなく曲線全体が再フィットされます。これはポリラインのグリップ編集とは異なります — ポリラインでは頂点を移動すると隣接する 2 つのセグメントのみが変形します。

「スプライン全体を移動」グリップはありません。スプライン全体を移動するには [Move](./move) コマンドを使用してください。

## フィットスプラインの選択

| 方法 | 動作 |
|-----|------|
| **クリック** | 曲線上の任意の点の近くにクリックした場合に選択 |
| **右ドラッグ**（厳密） | 曲線に沿ったすべてのサンプル点が選択ボックス内に収まる必要がある |
| **左ドラッグ**（交差） | 選択ボックスの境界を横切る曲線の任意の部分で選択 |

## サポートされる編集コマンド

| コマンド | スプラインへの効果 |
|---------|----------------|
| [Move](./move) | すべてのフィット点と再計算された CV を同じ変位で移動 |
| [Copy](./copy) | 新しい位置に同一のスプラインを作成 |
| [Rotate](./rotate) | 選択した基点を中心にすべてのフィット点を回転 |
| [Mirror](./mirror) | 鏡像軸ですべてのフィット点を反転 |
| [Scale](./scale) | 基点からすべてのフィット点を均一にスケール |
| [Delete](./delete) | スプラインを削除 |

スプラインは **Offset**、**Trim**、**Extend** をサポートしません。

## プロパティ

**一般**

| プロパティ | デフォルト | 説明 |
|---------|---------|------|
| Color | 256 (ByLayer) | ACI カラーインデックス |
| Layer | `0` | レイヤー割り当て |
| Linetype | ByLayer | 名前付き線種パターン |
| Linetype Scale | 1 | 線種パターンのスケール係数 |
| Thickness | 0 | 押し出し厚さ |

**ジオメトリ**

| プロパティ | 説明 |
|---------|------|
| Degree | 多項式次数 — 常に 3（3 次） |
| Fit Points | クリックした通過点すべての座標 |
| Control Vertices | 曲線のレンダリングに使用する内部的に計算された CV |

## スプラインフィットとスプライン CV の比較 — 使い分け

| | スプラインフィット | スプライン CV |
|---|------------|-----------|
| 曲線が点を通過 | クリックした各点を正確に通過 | 最初と最後のみ（クランプト） |
| グリップ編集の効果 | フィット点移動 → 曲線全体が再補間 | CV 移動 → 曲線が新しい位置に引き寄せられる |
| 形状の予測可能性 | 高い — 曲線がクリックに追従 | 低め — 曲線が CV に遅れて追従 |
| 適している用途 | 特定の座標を通過しなければならない曲線 | 滑らかな美的曲線、フリーフォームパス |

## DXF — SPLINE エンティティ（フィット点形式）

フィットスプラインは DXF ファイルに `SPLINE` エンティティとして保存され、フィット点座標と計算された制御頂点の両方を格納します。`splineFlag` は `8`（フィット点スプライン）に設定されているので、リロードするアプリケーションがどの点のセットを編集可能グリップとして表示するかを知ることができます。すべてのプロパティ（色、レイヤー、線種、線種スケール、厚さ）はロスなく双方向対応しています。フィット点スプラインをサポートする DXF アプリケーション（LibreCAD、FreeCAD）はフィット点を主要な編集可能データとして表示します。
