# Help

すべてのキーボードショートカット・ナビゲーションのヒント・一般的な使用方法のまとめを含むアプリ内ヘルプパネルを開きます。

## 使用方法

**Help** ツールバーボタンをクリックするか、ターミナルに `help` と入力します。ヘルプパネルがすぐに開き、コマンドが終了します。

## ヘルプパネルの内容

### ナビゲーション
| ショートカット | 操作 |
|----------|--------|
| `スクロールホイール` | ズームイン / アウト |
| `ミドルクリック + ドラッグ` | ビューをパン |
| `ミドルダブルクリック` | すべてのエンティティを画面に収める |

### コマンド
| ショートカット | 操作 |
|----------|--------|
| 入力 | コマンド名の入力を開始 — キャンバスに候補が表示されます |
| `Delete` | 選択したすべてのエンティティを削除 |
| `Enter` / `Space` | コマンドまたは入力を確定 |
| `Enter` / `Space`（アイドル時） | 最後に使用したコマンドを繰り返す |
| `Escape` | 現在のコマンドをキャンセルまたは選択を解除 |

### 選択
| ショートカット | 操作 |
|----------|--------|
| `左クリック` | エンティティを選択 |
| `右にドラッグ` | ドラッグ範囲**内に完全に収まる**エンティティのみを選択 |
| `左にドラッグ` | ドラッグ範囲に**接触する**すべてのエンティティを選択 |
| `グリップを左クリック` | 頂点または端点を直接編集 |

### 作図のヒント
- **スナップ点** — カーソルは近くのエンティティ端点に自動的にスナップします。
- **角度スティッキー** — ある方向に十分遠くカーソルを移動すると 45° 刻みにロックします。
- **正確な長さ** — 角度がロックされた状態で数値を入力して `Enter` を押すと正確な距離を設定できます。

## バージョン
