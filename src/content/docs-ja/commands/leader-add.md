# Leader+

`leader+` コマンドは既存のマルチ引出線に新しい矢印アームを追加します。新しいアームは引出線の既存のドッグレッグから、クリックして指定する新しい矢印の先端まで伸びます。スタイル（ドッグレッグ位置、テキスト、矢印の種類、サイズ）はすべて選択した引出線から継承されます。

## アームの追加

1. ターミナルに `leader+` と入力します。
2. **既存のマルチ引出線をクリック**して選択します。
3. **新しい矢印の先端をクリック**するか、`X,Y` と入力して **Enter** を押し正確な座標を指定します。カーソルから引出線のドッグレッグまでのプレビューラインが表示されます。

アームが配置されてもコマンドはアクティブのままで、すぐに別の引出線をクリックしてさらにアームを追加できます。**Enter**、**Space**、または **Escape** を押して終了します。

```
  前:                        後:
  ◄── アーム 1                ◄── アーム 1
       \                            \
        ●──── ドッグレッグ ──── テキスト  ●──── ドッグレッグ ──── テキスト
                                        /
                               アーム 2 ──►  (クリックした新しい先端)
```

## 先端の座標入力

クリックの代わりに正確な位置を入力できます：

1. X 値を入力します。
2. `,` を押します — ターミナルが X がロックされたことを確認します。
3. Y 値を入力します。
4. **Enter** を押して配置します。

## キーボードリファレンス

| キー | 操作 |
|-----|------|
| `0`〜`9`, `.`, `-` | X 座標の入力を開始 |
| `,` | X をロックして Y 入力に移動 |
| `Enter` | 入力した座標を確定してアームを配置 |
| `Enter` / `Space` | 終了（入力が進行中でない場合） |
| `Escape` | キャンセルしてリセット |

## 注記

- **Multileader** エンティティのみ選択できます — 他のエンティティタイプをクリックしても何も起こりません。
- 新しいアームは既存のドッグレッグから始まります。矢印の先端の位置のみを指定します。
- マルチ引出線のアーム数に制限はありません。
