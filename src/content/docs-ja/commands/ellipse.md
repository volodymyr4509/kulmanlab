# Ellipse

`ellipse` コマンドは 3 クリックで楕円を描きます: 中心点、任意の角度での最初の（長）半軸の端点、2 番目の（短）半軸の長さ。2 つの軸は常に互いに垂直です — 2 番目の軸方向は最初の軸から自動的に導かれます。

## 楕円の描画手順

1. ターミナルに `ellipse` と入力するか、**Ellipse** ツールバーボタンをクリックします。
2. **中心点をクリック**するか、`X,Y` と入力して **Enter** を押すと正確な座標を指定できます。
3. **最初の軸端点をクリック** — 最初の半軸の方向と長さの両方を設定します。ここでも座標入力が使用できます。
4. **2 番目の軸長さを設定** — カーソルを最初の軸に垂直に移動し、クリックするか長さを入力します。

```
               ● ← 最初の軸端点（ステップ 3）
              /
  中心 ●  /  ← 最初の軸（任意の角度）
            |
            ● ← カーソルここで 2 番目の軸長さを設定（ステップ 4）
```

ステップ 4 の後に楕円が配置され、コマンドが終了します。

## 軸入力 — クリック・座標・長さ入力

**中心（ステップ 2）:** クリックするか、正確な位置のために `X,Y` を入力します。

**最初の軸端点（ステップ 3）:** クリックするか、正確な座標のために `X,Y` を入力します。角度ロックも 45° 刻みにスナップします — ロック中に長さを入力して **Enter** を押すと端点をその正確な距離に配置できます。

**2 番目の軸（ステップ 4）:** テキスト入力は常に利用可能 — 角度ロックは不要です。方向はすでに最初の軸に垂直に固定されており、入力は長さのみを設定します。

| キー | 操作 |
|-----|------|
| `0`〜`9`, `.` | 軸長さに数字を追加（2 番目の軸フェーズ） |
| `Backspace` | 最後に入力した文字を削除 |
| `Enter` | 入力した長さで軸端点を配置 |

## 座標入力（中心と最初の軸端点）

1. X 値を入力します。
2. `,` を押します — ターミナルに `[X], [Y{カーソル}]` と表示されます。
3. Y 値を入力します。
4. **Enter** を押して確定します。

## キーボードリファレンス

| キー | 操作 |
|-----|------|
| `0`〜`9`, `.`, `-` | X 座標入力の開始（中心/最初の軸フェーズ）、または角度ロック時の軸長さ |
| `,` | X を確定して Y 入力に移行 |
| `Backspace` | 最後に入力した文字を削除 |
| `Enter` | 入力した座標または長さを確定 |
| `Escape` | キャンセルしてリセット |

## グリップ編集 — 独立した軸リサイズ

選択した楕円には 5 つのグリップが表示されます:

| グリップ | 数 | 機能 |
|------|-------|------|
| **中心** | 1 | 楕円全体を移動。両軸は変わらない |
| **長軸端点** | 2（長軸の両端） | ドラッグして長半軸の長さをリサイズ。短軸の絶対サイズは変わらない |
| **短軸端点** | 2（短軸の両端） | ドラッグして短半軸の長さをリサイズ。長軸は変わらない |

長軸と短軸のグリップは独立しています — コマンドを再実行せずに楕円を変形できます。

## 楕円の選択

| 方法 | 動作 |
|--------|-----------|
| **クリック** | 楕円の輪郭付近にクリックが当たると選択 |
| **右ドラッグ**（厳密） | 楕円の軸平行バウンディングボックスが選択ボックス内に完全に収まる場合に選択 |
| **左ドラッグ**（交差） | 楕円の輪郭の一部が選択ボックス境界と交差すると選択 |

## サポートされる編集コマンド

| コマンド | 楕円への影響 |
|---------|-----------| 
| [移動](../move/) | 中心を移動。両軸は変わらない |
| [コピー](../copy/) | 新しい中心に同一の楕円を作成 |
| [回転](../rotate/) | 中心位置と長軸ベクトルを同じ角度だけ回転 |
| [鏡像](../mirror/) | 中心を反射し、鏡軸に対して長軸方向を再計算 |
| [尺度変更](../scale/) | 中心位置を拡縮し、両半軸の長さに係数を乗じる |
| [オフセット](../offset/) | 固定距離だけ外側または内側にオフセットした同心楕円を作成 |
| [削除](../delete/) | 楕円を削除 |

## プロパティ

**一般**

| プロパティ | デフォルト | 意味 |
|----------|---------|------|
| Color | 256 (ByLayer) | ACI カラーインデックス |
| Layer | `0` | レイヤー割り当て |
| Linetype | ByLayer | 線種パターン名 |
| Linetype Scale | 1 | 線種パターンの尺度係数 |
| Thickness | 0 | 押し出し厚さ |

**ジオメトリ**

| プロパティ | 意味 |
|----------|------|
| Center X / Center Y | 楕円の中心 |
| Major Axis X / Major Axis Y | 中心から長軸端点へのベクトル（方向と長さの両方をエンコード） |
| Axis Ratio | 短半軸と長半軸の比（0 < 比 ≤ 1） |
| Start Angle / End Angle | 度数でのパラメータ角。完全な楕円では両方 0°/360° |

## Ellipse と Circle — 使い分け

| | Ellipse | Circle |
|---|---------|--------|
| 軸 | 任意の角度での 2 つの独立した半軸 | 1 つの半径、対称 |
| 回転 | 任意の角度に配置可能 | 回転なし |
| テキスト入力 | 軸ごとの長さ | 半径のみ |
| グリップリサイズ | 長軸と短軸を独立して | 4 方向グリップがすべて等しく |
| 最適用途 | 斜め図、楕円形フィーチャ、透視穴 | 対称な円形フィーチャ |

## DXF — ELLIPSE エンティティ

楕円は DXF ファイルに `ELLIPSE` エンティティとして保存されます。フォーマットには中心点・完全な長軸ベクトル（方向 + 長さ）・軸比率が格納されます。回転・形状・すべてのスタイルプロパティがロスなくラウンドトリップします。円は退化した楕円として保存されません — 2 つのエンティティタイプは DXF モデルで区別されます。
