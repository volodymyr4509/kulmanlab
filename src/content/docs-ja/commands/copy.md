# Copy

`copy` コマンドは選択したエンティティの複製を作成し、基点から目的地へのオフセットで配置します — 元のエンティティはそのまま残ります。これが [Move](../move/) との唯一の重要な違いです: Copy は図面に新しいエンティティを追加し、Move は既存のものを移動します。

## 2 つの開始方法

**事前選択してからコピー** — 先にエンティティを選択:

1. キャンバスで 1 つ以上のエンティティを選択します。
2. ターミナルに `copy` と入力するか、**Copy** ツールバーボタンをクリックします。
3. **基点をクリック**するか、`X,Y` と入力して **Enter** を押すと正確な座標を指定できます。
4. **目的地をクリック** — 基点→目的地オフセットで複製が表示されます。ここでも座標入力が使用できます。

**起動してから選択** — 何も選択せずにコマンドを開始:

1. `copy` と入力するかツールバーボタンをクリックします。
2. **オブジェクトを選択** — クリックして個別のエンティティを切り替えるか、ドラッグして範囲選択します。
3. **Enter** または **Space** を押して選択を確定します。
4. **基点をクリック**してから**目的地をクリック**します（両ステップで座標入力利用可能）。

```
  前:               後:
  [エンティティ A]    [エンティティ A]  ← 元はそのまま
  [エンティティ B]    [エンティティ B]
                    [A のコピー] ← 新しいエンティティ
                    [B のコピー]
```

基点から目的地へ複製のゴーストプレビューがカーソルを追います。

## 座標入力

基点または目的地のステップで、クリックの代わりに正確な位置を入力できます:

1. X 値を入力します。
2. `,` を押します — ターミナルに `[X], [Y{カーソル}]` と表示されます。
3. Y 値を入力します。
4. **Enter** を押して確定します。

## 角度ロックと正確な距離

基点が設定されると、カーソルが軸から十分遠く、近くにある場合は 45° 軸（0°, 45°, 90°, 135°, …）にスナップします。ロック中に距離を入力して **Enter** を押すと、複製をその正確なオフセットに配置できます。

| キー | 操作 |
|-----|------|
| `0`〜`9`, `.` | 距離値に数字を追加 |
| `-` | 負の距離 — 軸に沿って方向を反転（最初の文字のみ） |
| `Backspace` | 最後に入力した文字を削除 |
| `Enter` | 入力した距離で複製を配置 |

## キーボードリファレンス

| キー | 操作 |
|-----|------|
| `Enter` / `Space` | 選択を確定して基点フェーズに進む |
| `0`〜`9`, `.`, `-` | X 座標入力の開始、または角度ロック時の距離 |
| `,` | X を確定して Y 入力に移行 |
| `Backspace` | 最後に入力した文字を削除 |
| `Enter` | 座標を確定または入力した距離でコピーを適用 |
| `Escape` | キャンセルしてリセット |

## コマンド中の選択

| 方法 | 動作 |
|--------|-----------|
| **クリック** | カーソル下のエンティティを選択/解除 |
| **右ドラッグ**（厳密） | ボックス内に完全に入っているエンティティを追加 |
| **左ドラッグ**（交差） | ボックス境界と交差するエンティティを追加 |
| **Enter** / **Space** | 選択を確定 |

## コピー後の動作

**元のエンティティは選択されたまま**です — 新しい複製は図面に追加されますが、選択は解除されてコマンドが終了します。複製をすぐに操作するには、選択に対して再度 Copy を実行するか、新しいコマンドを開始します。

## Copy と Move の比較

| | Copy | Move |
|---|------|------|
| 元のエンティティ | そのままの位置に残る | 元の位置から削除される |
| 結果の数 | コピーしたエンティティ分だけ増加 | 変わらない |
| 操作後の状態 | 元のエンティティが引き続き選択 | 新しい位置で移動したエンティティが選択 |
| 最適用途 | ジオメトリの繰り返し、対称レイアウト | ジオメトリの位置変更 |

## サポートされるエンティティ

Copy はすべてのエンティティタイプで機能します。すべてのエンティティは内部的に `translate(dx, dy)` を実装しているため、除外されるものはありません。
