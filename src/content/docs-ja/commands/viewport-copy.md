# ViewportCopy（ビューポートコピー）

`ViewportCopy` コマンドはビューポートをスケールとモデルの中心を保持したまま新しい位置にコピーします。用紙レイアウト空間でのみ使用できます。

## ビューポートのコピー

1. 用紙レイアウトタブに切り替えます。
2. オプションでビューポートをクリックして事前選択します。
3. ターミナルに `ViewportCopy` と入力するか、**Viewport Copy** ツールバーボタンをクリックします。
4. ビューポートが事前選択されていない場合は、**コピーするビューポートをクリック**します。
5. **基点をクリック** — 変位の基準点。または `X,Y` と入力して **Enter** を押し正確な座標を指定します。
6. **目的地をクリック** — ビューポートが基点→目的地オフセットに配置されます。座標入力 / 角度ロックも使用できます。

配置後もコマンドはアクティブのままで — 別の目的地をクリックして同じビューポートのコピーをさらに配置できます。**Enter**、**Space**、または **Escape** を押して終了します。

## 座標入力

基点と目的地ステップで、クリックの代わりに正確な座標を入力できます：

1. X 値を入力します。
2. `,` を押します — ターミナルに `[X], [Y{カーソル}]` と表示されます。
3. Y 値を入力します。
4. **Enter** を押して確定します。

## 角度ロックと正確な距離

基点が設定された後、コマンドはカーソルが揃うと 45° 軸（0°, 45°, 90°, 135°, …）にスナップします。ロック中：

- プレビューが軸にスナップします。
- 距離を入力して **Enter** を押すと、ロックした方向に正確にそのオフセットでコピーを配置します。

| キー | 操作 |
|-----|------|
| `0`〜`9`, `.` | 距離値に数字を追加 |
| `-` | 負の距離（方向を反転；最初の文字のみ） |
| `Backspace` | 最後に入力した文字を削除 |
| `Enter` | 入力した距離でコピーを配置 |

## キーボードリファレンス

| キー | 操作 |
|-----|------|
| `0`〜`9`, `.`, `-` | X 座標入力を開始、または角度ロック中の距離 |
| `,` | X をロックして Y 入力に移動 |
| `Enter` | 入力した座標または距離を確定 |
| `Enter` / `Space` | 終了（入力が進行中でない場合） |
| `Escape` | キャンセルしてリセット |

## 注記

- ViewportCopy は用紙レイアウトタブがアクティブな場合にのみ使用できます。
- コピーされたビューポートは元のビューポートと同じスケール、モデルの中心、ロック状態、寸法を継承します。
- 新しいビューポートをゼロから作成するには [ViewportRectangle](./viewport-rectangle) を使用してください。
