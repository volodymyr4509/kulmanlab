# Trim（トリム）

`trim` コマンドは 2 つの隣接する交差点の間にある[Line](../line/)の部分を削除し、直線を 1 つまたは 2 つの短いセグメントに分割します。カットするセグメントはカーソル位置によって決まります — 削除したい部分の上にカーソルを移動してクリックでトリムします。

Trim は**Line エンティティのみ**で機能します。円弧、円、ポリライン、その他のエンティティタイプには [Delete](../delete/) またはグリップ編集を使用してください。

## 直線のトリム

1. ターミナルに `trim` と入力するか、**Trim** ツールバーボタンをクリックします。
2. **削除したい直線セグメントの上にカーソルを移動** — 赤いプレビューでカットされる部分が正確にハイライトされます。
3. **クリック**してそのセグメントを削除します。

各トリム後もコマンドはアクティブのままなので、カーソルを移動してクリックを続けてさらにセグメントをカットできます。**Escape** を押して終了します。

```
  前:                     交差点の間のセグメントをトリム後:

  ──────●──────●──────        ──────●          ●──────
      交差    交差           （左部分）        （右部分）
                              （中間セグメントが削除された）
```

## トリムセグメントの決定方法

コマンドはカーソル位置をホバーした直線に投影し、その直線が他のエンティティと持つすべての交差点を検索します。これらの交差パラメーターが直線をセグメントに分割します。カーソルの投影を含む区間のセグメントがハイライトされ、クリックで削除されます。

- カーソルが**最初の交差点より前**：その先頭部分が削除されます。
- カーソルが**2 つの交差点の間**：その中間部分が削除され、直線が 2 つに分割されます。
- カーソルが**最後の交差点より後**：その末尾部分が削除されます。
- 直線が他のエンティティと**交差点を持たない**：プレビューは表示されず、クリックしても何も起こりません。

## キーボードリファレンス

| キー | 操作 |
|-----|------|
| `Escape` | トリムモードを終了 |

## サポートされるエンティティ

| エンティティ | トリム可能？ |
|---------|----------|
| 直線 | あり |
| 円弧、円、楕円 | なし |
| ポリライン / 長方形 | なし |
| テキスト、スプライン、寸法、引出線 | なし |

**カット境界**として使用されるエンティティは任意のタイプにできます — トリムされる直線のみが Line エンティティである必要があります。

## Trim と Extend の比較

| | Trim | Extend |
|---|------|--------|
| 機能 | 直線のセグメントを削除 | 直線の端点を境界まで伸ばす |
| トリガー | カットするセグメントの上にカーソル | 延長する端点の近くにカーソル |
| 結果 | 直線が分割または短くなる | 直線の端点が境界まで移動 |
| 両方 | 直線のみ | 直線のみ |
