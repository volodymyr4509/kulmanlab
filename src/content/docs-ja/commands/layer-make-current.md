# LayerMakeCurrent

`LayerMakeCurrent` コマンドは、クリックしたエンティティが属するレイヤーを**現在の作図レイヤー**として設定します。以降に作成されるエンティティはそのレイヤーに自動的に配置されます。

## 使用手順

1. ターミナルに `LayerMakeCurrent` と入力するか、**Make Current** ツールバーボタン（スポイトアイコン）をクリックします。
2. キャンバス上の**任意のエンティティをクリック**します。
3. カレントレイヤーがそのエンティティのレイヤーに更新されます。コマンドはすぐに終了します。

## 動作の詳細

- 空のキャンバス（エンティティが見つからない場所）をクリックした場合、ターミナルに `no object found` と表示され、コマンドはアクティブのまま再試行できます。
- 変更されるのはカレントレイヤーの設定のみです — エンティティは変更されません。
- 更新されたレイヤーはツールバーのレイヤーセレクターに反映されます。
