# Angle

`angle` コマンドは、3 点のクリックによって形成される 2 つの直線セグメントの頂点における内角を計測します。結果は常に 0° 〜 180° の範囲で、小数点以下 4 桁でターミナルに表示されます。2 つの計測コマンドの一方であり、[Distance](../distance/) は直線距離を計測します。

## 角度計測の構造

```
  ● 最初の点（最初の光線の端点）
   \
    \  最初の光線プレビュー
     \
      ● 頂点（ステップ 3）
     /
    /  2 番目の光線プレビュー（カーソルまで）
   /
  ● 3 番目の点  →  ターミナル: "Angle: 45.0000°"
```

- **最初の点** — 角度の一方の端（ステップ 2）。
- **頂点** — 角度を計測するコーナー（ステップ 3）。
- **3 番目の点** — 角度のもう一方の端（ステップ 4）。

## 角度の計測手順

1. ターミナルに `angle` と入力するか、**Angle** ツールバーボタンをクリックします。
2. **最初の点をクリック** — 角度の一方の端。または `X,Y` と入力して **Enter** を押すと正確な座標を指定できます。
3. **頂点をクリック** — 2 つのアームが交わるコーナー。ここでも座標入力が使用できます。
4. **3 番目の点をクリック** — 2 番目のアームの端。ここでも座標入力が使用できます。この点を配置すると結果が表示されます。
5. **再度クリック**（任意）して、そのクリックを新しい最初の点として新しい計測を開始します。

## 内角の計算規則

コマンドは頂点からの 2 つの光線の内積を使って角度を計算します:

- **常に内角**: 結果は 0° 〜 180° の小さい方の角度です。
- 端点をクリックする順序は結果に影響しません — 頂点の位置のみが重要です。
- 3 点が同一直線上にある場合は 0° または 180° を返します。

## 連続計測

結果が表示された後、クリックするとすぐに次の計測が開始されます — クリックした点が新しい最初の点になります。`Escape` を押すまでコマンドは自動終了しません。

## Angle と Distance の比較

| | Angle | Distance |
|---|-------|---------|
| 計測対象 | 頂点での内角 | 直線距離 |
| クリック数 | 3 | 2 |
| 結果の形式 | `45.0000°` | `12.3456`（単位） |
| キャンバスプレビュー | 頂点から両端への 2 本の線 | 最初の点からカーソルまでの線 |
| 最適用途 | 2 つのフィーチャ間の開き角 | 隙間またはセグメントの長さ |

## 座標入力

クリックの代わりに、3 点のいずれかに正確な位置を入力できます:

1. X 値を入力します。
2. `,` を押します — ターミナルに `[X], [Y{カーソル}]` と表示されます。
3. Y 値を入力します。
4. **Enter** を押して確定します。

## キーボードリファレンス

| キー | 操作 |
|-----|------|
| `0`〜`9`, `.`, `-` | X 座標入力の開始 |
| `,` | X を確定して Y 入力に移行 |
| `Backspace` | 最後に入力した文字を削除 |
| `Enter` | 入力した座標を確定 |
| `Escape` | キャンセルしてステップ 2 にリセット |

## 注意事項

- 結果は**ターミナルのみ**に表示されます — 図面には何も追加されません。
- 精度は常に度数で小数点以下 4 桁です。
