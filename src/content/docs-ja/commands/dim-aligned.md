# Dimension Aligned

`dimaligned` コマンドは 2 点間の**真の直線距離**を計測する寸法を配置します。寸法線は 2 点を結ぶ線に平行して配置されるため、任意の角度に対応できます。これが水平または垂直に制限される [Dimension Linear](./dim-linear) との主な違いです。

## 整列寸法の構造

```
     ●  p2
    /|
   / |  （引出線 2、寸法線に垂直）
  /  |
 /←5.00→/
/  /
●  /  （引出線 1、寸法線に垂直）
p1
```

- **引出線** — 各計測点から寸法線に垂直に引かれます。
- **寸法線** — p1→p2 に平行し、カーソル位置に応じて一方の側にオフセットされます。
- **値** — 真のユークリッド距離 `|p1 – p2|`。

## 整列寸法の配置手順

1. ターミナルに `dimaligned` と入力するか、**Dimension Aligned** ツールバーボタンをクリックします。
2. **最初の引出線の起点**（p1）をクリックするか、`X,Y` と入力して **Enter** を押すと正確な座標を指定できます。
3. **2 番目の引出線の起点**（p2）をクリックします。ここでも座標入力が使用できます。
4. **カーソルを移動**して一方の側への寸法線の垂直オフセットを設定します。
5. **クリック**して配置するか、オフセット距離を入力して **Enter** を押すと正確に配置できます。

## オフセット距離の入力

配置中に数値を入力すると、p1→p2 の線から正確な垂直距離に寸法線を固定できます:

| キー | 操作 |
|-----|------|
| `0`〜`9`, `.` | オフセットに数字を追加 |
| `Backspace` | 最後に入力した文字を削除 |
| `Enter` / `Space` | 入力したオフセットで配置 |

カーソルの側によって、寸法線がどちら側に表示されるかが決まります。

## キーボードリファレンス

| キー | 操作 |
|-----|------|
| `0`〜`9`, `.`, `-` | X 座標入力の開始（p1/p2 フェーズ）、またはオフセット距離（配置フェーズ） |
| `,` | X を確定して Y 入力に移行（p1/p2 フェーズ） |
| `Backspace` | 最後に入力した文字を削除 |
| `Enter` / `Space` | 入力した座標またはオフセットを確定 |
| `Escape` | キャンセル |

## Dimension Aligned と Dimension Linear の比較

| | Dimension Aligned | Dimension Linear |
|---|------------------|-----------------|
| 寸法線の角度 | p1→p2 に平行 — 任意の角度 | 常に水平または垂直 |
| 計測対象 | 真のユークリッド距離 | X または Y 成分のみ |
| H/V 向き固定 | なし | あり — `H` と `V` キー |
| 最適用途 | 斜めフィーチャ、傾斜カット | 直交レイアウト、グリッド整列部品 |

## 寸法の連続

この寸法の 2 番目の引出線から続く寸法を追加するには、[Dimension Continue](./dim-continue) を使用します — この整列寸法と同じ計測角度でロックされます。

## DXF — DIMENSION エンティティ（整列タイプ）

整列寸法は `dimType = 1`（整列）の `DIMENSION` エンティティとして DXF ファイルに保存されます。引出線の起点・寸法線の位置・テキスト位置・計測値・回転・矢印スタイル・すべての表示フラグがロスなくラウンドトリップします。
