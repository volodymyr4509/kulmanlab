# Circle

`circle` コマンドは中心点と半径で定義された円を描きます。中心をクリックした後、2 番目の点をキャンバスでクリックするか、正確な数値を入力することで半径を設定できます — 両方のオプションが同時に利用できます。

## 円の描画手順

1. ターミナルに `circle` と入力するか、**Circle** ツールバーボタンをクリックします。
2. **中心点をクリック**するか、`X,Y` と入力して **Enter** を押すと正確な座標を指定できます。
3. 半径を設定します — 以下のいずれか:
   - **任意の点をクリック** — 中心からの距離が半径になります。または
   - **半径を入力**して **Enter** を押すと正確な値を指定できます。

円はすぐに配置され、コマンドが終了します。

```
  中心 ●
          \  半径ラインプレビュー
           \
            ● ← ここをクリックするか、数値を入力
```

半径フェーズ中は、現在のカーソル距離での円のライブプレビューと、中心から現在の点までの半径ラインが表示されます。

## 中心座標入力

クリックの代わりに中心位置を入力できます:

1. X 値を入力します。
2. `,` を押します — ターミナルに `[X], [Y{カーソル}]` と表示されます。
3. Y 値を入力します。
4. **Enter** を押して中心を配置し、半径入力に進みます。

## 半径のテキスト入力

中心が配置された後、すぐに入力を始めると半径値が構築されます:

| キー | 操作 |
|-----|------|
| `0`〜`9`, `.` | 半径値に数字を追加 |
| `Backspace` | 最後に入力した文字を削除 |
| `Enter` | 入力した半径で円を配置 |

累積値はターミナルのプロンプトに表示されます（例: `enter radius of circle: 25`）。プレビューは入力した半径に更新され、カーソルは半径ラインマーカーの方向を制御します。

## キーボードリファレンス

| キー | 操作 |
|-----|------|
| `0`〜`9`, `.`, `-` | X 座標入力の開始（中心フェーズ）または半径数字（半径フェーズ） |
| `,` | X を確定して Y 入力に移行（中心フェーズ） |
| `Backspace` | 最後に入力した文字を削除 |
| `Enter` | 入力した座標または半径を確定 |
| `Escape` | キャンセルしてリセット |

## グリップ編集 — 半径のリサイズ

選択した円には 5 つのグリップが表示されます:

| グリップ | 位置 | 機能 |
|------|----------|------|
| **中心** | 中心点 | 円全体を移動。半径は変わらない |
| **左** | 最も左の点（中心 − 半径） | ドラッグすると新しい半径 = 中心までの距離 |
| **右** | 最も右の点（中心 + 半径） | ドラッグすると新しい半径 = 中心までの距離 |
| **上** | 最も上の点 | ドラッグすると新しい半径 = 中心までの距離 |
| **下** | 最も下の点 | ドラッグすると新しい半径 = 中心までの距離 |

4 方向グリップはすべて同じ動作をします — 新しい半径はドラッグ位置から中心までの距離です。中心は固定されます。

## 円の選択

| 方法 | 動作 |
|--------|-----------|
| **クリック** | 円周付近にクリックが当たると選択 |
| **右ドラッグ**（厳密） | バウンディングスクエア全体（中心 ± 半径）がボックス内に収まる場合に選択 |
| **左ドラッグ**（交差） | 円周の一部がボックス境界と交差または接触すると選択 |

## サポートされる編集コマンド

| コマンド | 円への影響 |
|---------|----------|
| [Move](../move/) | 中心を移動。半径は変わらない |
| [Copy](../copy/) | 新しい中心に同一の円を作成 |
| [Rotate](../rotate/) | 基点を中心に円の中心を回転。半径は変わらない |
| [Mirror](../mirror/) | 鏡軸を基準に中心を反射。半径は変わらない |
| [Scale](../scale/) | 中心位置を拡縮し、半径に係数を乗じる |
| [Offset](../offset/) | 大きいまたは小さい半径の同心円を作成 |
| [Delete](../delete/) | 円を削除 |

## プロパティ

**一般**

| プロパティ | デフォルト | 意味 |
|----------|---------|------|
| Color | 256 (ByLayer) | ACI カラーインデックス |
| Layer | `0` | レイヤー割り当て |
| Linetype | ByLayer | 線種パターン名 |
| Linetype Scale | 1 | 線種パターンの尺度係数 |
| Thickness | 0 | 押し出し厚さ |

**ジオメトリ**

| プロパティ | 意味 |
|----------|------|
| Center X / Center Y | 中心点の座標 |
| Radius | 図面単位での円の半径 |

## Circle と Arc — 使い分け

| | Circle | Arc |
|---|--------|-----|
| 範囲 | 完全な 360° | 部分的 — スタート角とエンド角で定義 |
| 描画方法 | 中心 + 半径 | 曲線上の 3 点 |
| テキスト入力 | 半径値 | なし — クリックのみ |
| リサイズグリップ | 4 方向グリップ | スタートとエンド（角度 + 半径） |
| 寸法 | 半径: [Dimension Radius](../dim-radius/) · 直径: [Dimension Diameter](../dim-diameter/) | [Dimension Radius](../dim-radius/) |
| 最適用途 | 完全な穴、ボルト円、円形フィーチャ | フィレット、部分曲線、アーチ状パス |

## DXF — CIRCLE エンティティ

円は DXF ファイルに `CIRCLE` エンティティとして保存されます。中心座標・半径・色・レイヤー・線種・線種尺度・厚さはすべてロスなくラウンドトリップします。DXF 互換アプリケーションはこれらを標準の円として読み込みます。
