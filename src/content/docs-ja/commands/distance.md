# Distance

`distance` コマンドはクリックした 2 点間の直線（ユークリッド）距離を計測し、小数点以下 4 桁でターミナルに結果を表示します。2 つの計測コマンドの一方であり、[Angle](../angle/) は代わりに頂点での角度を計測します。

## 距離計測の構造

```
  ● 最初の点
   \
    \  プレビューライン（ライブ）
     \
      ● 2 番目の点    →  ターミナル: "Distance: 12.3456"
```

- **最初の点** — 計測の起点。
- **2 番目の点** — 終点。この点を配置すると即座に結果が表示されます。
- **結果** — ターミナルに表示。キャンバスには何も配置されません。

## 距離の計測手順

1. ターミナルに `distance` と入力するか、**Distance** ツールバーボタンをクリックします。
2. **最初の点をクリック**するか、`X,Y` と入力して **Enter** を押すと正確な座標を指定できます。
3. **2 番目の点をクリック** — 計測した距離がターミナルに表示されます。ここでも座標入力が使用できます。
4. **再度クリック**（任意）して新しい計測を開始します。コマンドはアクティブのままです。

いつでも `Escape` を押してステップ 2 にリセットできます。

## 連続計測

結果が表示された後、クリックするとすぐに次の計測が開始されます — クリックした点が新しい最初の点になります。コマンドを再起動せずに一連の距離を計測できます。

## Distance と Angle の比較

| | Distance | Angle |
|---|---------|-------|
| 計測対象 | 直線距離 | 頂点での内角 |
| クリック数 | 2 | 3 |
| 結果の形式 | `12.3456`（単位） | `45.0000°` |
| キャンバスプレビュー | 最初の点からカーソルまでの線 | 頂点からカーソルまでの 2 本の線 |
| 最適用途 | 隙間またはセグメントの長さ | 2 つのフィーチャ間の開き角 |

## 座標入力

クリックの代わりに、いずれかの点に正確な位置を入力できます:

1. X 値を入力します。
2. `,` を押します — ターミナルに `[X], [Y{カーソル}]` と表示されます。
3. Y 値を入力します。
4. **Enter** を押して確定します。

## キーボードリファレンス

| キー | 操作 |
|-----|------|
| `0`〜`9`, `.`, `-` | X 座標入力の開始 |
| `,` | X を確定して Y 入力に移行 |
| `Backspace` | 最後に入力した文字を削除 |
| `Enter` | 入力した座標を確定 |
| `Escape` | キャンセルしてステップ 2 にリセット |

## 注意事項

- 結果は**ターミナルのみ**に表示されます — 図面には何も追加されません。
- 結果は図面座標と同じ単位で表されます（単位変換はありません）。
- 精度は常に小数点以下 4 桁です。
