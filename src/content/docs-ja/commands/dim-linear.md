# Dimension Linear

`dimlinear` コマンドは 2 つの引出線の起点間に水平または垂直の寸法を配置します。寸法線は常に完全な水平または垂直に配置されます — 任意の角度には配置できません。斜めの線に平行な寸法が必要な場合は [Dimension Aligned](../dim-aligned/) を使用してください。

## 水平垂直寸法の構造

```
  |←————— 5.00 —————→|
  |                   |
  ●  （引出線 1）     ●  （引出線 2）
  p1                  p2
```

- **引出線** — 各計測点から寸法線に垂直に引かれます。
- **寸法線** — 水平（X 距離を計測）または垂直（Y 距離を計測）。
- **値** — 実際の点間距離ではなく、選択した軸に沿って投影された距離。

## 水平垂直寸法の配置手順

1. ターミナルに `dimlinear` と入力するか、**Dimension Linear** ツールバーボタンをクリックします。
2. **最初の引出線の起点**（p1）をクリックするか、`X,Y` と入力して **Enter** を押すと正確な座標を指定できます。
3. **2 番目の引出線の起点**（p2）をクリックします。ここでも座標入力が使用できます。
4. **カーソルを移動**して寸法線を配置します。向きはカーソル位置から自動的に検出されます。
5. **クリック**して配置するか、オフセット距離を入力して **Enter** を押すと正確に配置できます。

## 向きの自動検出

向きが強制されていない場合、コマンドは 2 つの計測点に対するカーソル位置を読み取ります:

| カーソル位置 | 検出された向き | 計測対象 |
|-----------------|---------------------|-----------------|
| 点の上または下 | 水平 | p1 と p2 の Δ X |
| 点の左または右 | 垂直 | p1 と p2 の Δ Y |

配置フェーズ中いつでも **H** を押して水平にロック、**V** を押して垂直にロックできます。ロックすると、カーソルを移動しても向きは変わりません。

## オフセット距離の入力

配置中に数値を入力すると、計測点から正確な距離に寸法線を固定できます:

| キー | 操作 |
|-----|------|
| `0`〜`9`, `.` | オフセット距離に数字を追加 |
| `Backspace` | 最後に入力した文字を削除 |
| `Enter` / `Space` | 入力した距離で配置 |

カーソルの側（水平の場合は上/下、垂直の場合は左/右）が符号を決定します — 寸法線はカーソルが現在ある側に表示されます。

## キーボードリファレンス

| キー | 操作 |
|-----|------|
| `0`〜`9`, `.`, `-` | X 座標入力の開始（p1/p2 フェーズ）、またはオフセット距離（配置フェーズ） |
| `,` | X を確定して Y 入力に移行（p1/p2 フェーズ） |
| `H` | 水平向きをロック（配置フェーズのみ） |
| `V` | 垂直向きをロック（配置フェーズのみ） |
| `Backspace` | 最後に入力した文字を削除 |
| `Enter` / `Space` | 入力した座標またはオフセットを確定 |
| `Escape` | キャンセル |

## Dimension Linear と Dimension Aligned の比較

| | Dimension Linear | Dimension Aligned |
|---|-----------------|------------------|
| 軸 | 常に H または V | 計測した線に平行 |
| 計測対象 | X または Y 成分のみ | 真のユークリッド距離 |
| H/V キー | あり — 向きをロック | なし — 常に p1→p2 に従う |
| 最適用途 | 直交レイアウト、間取り図 | 斜めフィーチャ、傾斜カット |

## ラベルの編集 — シンプルモード

配置された水平垂直寸法を**ダブルクリック**すると、**シンプル**モードでテキストエディターが開きます。エディターには現在の表示値があらかじめ入力されているため、カーソルを移動して直接編集できます。

| 機能 | 動作 |
|------|------|
| Bold / Italic / Font / Height | **ラベル全体**に一度に適用 |
| 文字ごとのフォーマット | サポートされていません |
| `Enter` | 値を確定してエディターを閉じる |
| 複数行 | サポートされていません |

詳細は [テキストエディター — シンプルモード](../../interface/text-editor/#simple-mode) を参照してください。

## 寸法の連続

最後の引出線から続く寸法を追加するには、この寸法を配置した直後に [Dimension Continue](../dim-continue/) を使用します。

## DXF — DIMENSION エンティティ

水平垂直寸法は `rotationDeg` が `0`（水平）または `90`（垂直）に設定された `DIMENSION` エンティティとして DXF ファイルに保存されます。引出線の起点・寸法線の位置・テキスト位置・計測値・矢印スタイル・テキスト高さ・すべての表示フラグがロスなくラウンドトリップします。
