# Dimension Angular

`DimensionAngular` コマンドは図面に**角度寸法**円弧注釈を配置します。2 直線間の角度・円弧の範囲・円のセクターを計測してラベルを付けます。

## 起動方法

マークアップパネルの **Dimension Angular** ツールバーボタンをクリックするか、ターミナルに `DimensionAngular` と入力します。

## 3 つの入力モード

最初のクリックで使用するモードが決まります:

### 2 直線

1. **最初の直線をクリック。** カーソルの位置によって直線のどちら側が使用されるかが決まります。
2. **2 番目の直線をクリック。** 2 本の直線は交差していなければなりません（交点は自動的に計算されます。画面上に表示されている必要はありません）。
3. **クリックして**寸法円弧を配置します。カーソルを移動して半径と注釈を付けるセクターを選択します — 頂点のどちら側にカーソルがあるかによって注釈が追従します。

平行な直線は角度寸法を形成できません。直線が交差しない場合、コマンドは 2 番目のクリックを無視します。

### 円弧

1. **円弧をクリック。** 寸法は円弧の中心を頂点として、円弧のスタート角からエンド角まで即座に作成されます。
2. **クリックして**希望する半径に寸法円弧を配置します。

### 円

1. **円をクリック。** 最初の角度端点が円の最近傍点にスナップします。
2. **円上の 2 番目の点をクリック**して 2 番目の角度端点を定義します。
3. **クリックして**寸法円弧を配置します。

## キーボードリファレンス

| キー | 操作 |
|-----|------|
| `Escape` | キャンセルして最初のピックに戻る |

## 動作の詳細

- 寸法円弧は常に配置した頂点の側に描かれます — カーソルを頂点の反対側に移動すると補角に切り替わります。
- 計測された角度は度数で表示され、配置中にカーソルを移動するとリアルタイムで更新されます。
- 結果の注釈は現在のレイヤーに格納された完全な `DimensionAngular` エンティティです。外観プロパティ（矢印サイズ、テキスト高さ、引出線の長さ）はプロパティパネルで調整できます。
- 角度寸法は JSON エクスポートに含まれますが、DXF エクスポーターではサポートされていません。

## ラベルの編集 — シンプルモード

配置された角度寸法を**ダブルクリック**すると、**シンプル**モードでテキストエディターが開きます。エディターには現在の表示値があらかじめ入力されているため、カーソルを移動して直接編集できます。

| 機能 | 動作 |
|------|------|
| Bold / Italic / Font / Height | **ラベル全体**に一度に適用 |
| 文字ごとのフォーマット | サポートされていません |
| `Enter` | 値を確定してエディターを閉じる |
| 複数行 | サポートされていません |

詳細は [テキストエディター — シンプルモード](../../interface/text-editor/#simple-mode) を参照してください。
