# Zoom Out（ズームアウト）

`zoomout` コマンドは現在のズームレベルを **1.5×** で割り（約 0.667 倍に相当）、ビューポートの中点を中心にすぐに終了します。[Zoom In](./zoom-in) の逆です。

## ズームアウト方法

**Zoom Out** ツールバーボタンをクリックするか、ターミナルに `zoomout` と入力します。ズームは即座に適用されてコマンドが終了します — キャンバスのクリックは不要です。

## 1.5× ステップの動作

| 現在のズーム | Zoom Out 1 回後 |
|------------|----------------|
| 1.50× | 1.00× |
| 2.25× | 1.50× |
| 10.00× | 6.67× |
| 0.015× | 0.01×（下限） |

ズームレベルは常にキャンバスの**右下**に表示されます。下限は **0.01×** で、それ以上はステップしても何も起こりません。

## ツールバーの Zoom Out とスクロールホイールの比較

| | Zoom Out ボタン | スクロールホイール |
|---|----------------|-------------|
| ズームの中心 | ビューポートの中点 | カーソル位置 |
| ステップサイズ | クリックごとに 1.5× | ティックごとに約 1.1× |
| 起動が必要 | いいえ | いいえ — 常に使用可能 |
| 適している用途 | より多くのコンテキストを見るためにステップバック | 滑らかなカーソルアンカーズームアウト |

## キーボードリファレンス

このコマンドにキーボードショートカットはありません。代わりにスクロールホイールを使用してください — コマンドを起動せずに常に使用できます。
