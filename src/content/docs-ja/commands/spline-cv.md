# Spline CV（スプライン CV）

`splinecv` コマンドは制御頂点（CV）を配置することで**3 次 B スプライン**を作図します。結果の曲線は各 CV に引き寄せられますが、CV を通過しません — ただし最初と最後の頂点は例外で、**クランプトノット**が曲線をそこに正確にアンカーします。これにより直感的な形状制御が可能になります：曲線をその点に強制的に接触させることなく、頂点を引いて曲線をそちらに向けることができます。

## 制御頂点でスプラインを作図

1. ターミナルに `splinecv` と入力するか、**Spline CV** ツールバーボタンをクリックします。
2. **クリックして制御頂点を配置** — クリックするたびに頂点が追加されます。または `X,Y` と入力して **Enter** を押し正確な座標を指定します。
3. **Enter** を押して終了します（最低 2 頂点が必要）。

```
  CV ●         ● CV
      \       /
       \     /    ← CV に引き寄せられる曲線
        \   /         ただし通過しない
  CV ●   ●   ● CV (始点/終点: 曲線はここを通過)
```

各頂点の後にカーソルを移動すると、次の点がカーソル位置にある場合のスプラインの見た目を示すライブプレビューが更新されます。**Escape** を押すと配置済みの頂点がすべて破棄されてコマンドが終了します。

## 座標入力

任意の制御頂点に対してクリックの代わりに正確な位置を入力できます：

1. X 値を入力します。
2. `,` を押します — ターミナルに `[X], [Y{カーソル}]` と表示されます。
3. Y 値を入力します。
4. **Enter** を押して頂点を配置します。

## キーボードリファレンス

| キー | 操作 |
|-----|------|
| `0`〜`9`, `.`, `-` | X 座標入力を開始 |
| `,` | X をロックして Y 入力に移動 |
| `Backspace` | 最後に入力した文字を削除 |
| `Enter` | 入力した座標を確定、または入力が進行中でなく ≥ 2 頂点ある場合はスプラインを終了 |
| `Escape` | すべての頂点を破棄して終了 |

## グリップ編集 — 制御頂点による変形

選択した CV スプラインには制御頂点ごとに 1 つのグリップが表示されます：

| グリップ | 位置 | 機能 |
|---------|------|------|
| **制御頂点** | 各 CV の位置 | ドラッグしてその CV を移動 — 曲線が新しい位置に引き寄せられる方向に変形 |

「スプライン全体を移動」グリップはありません。スプライン全体を移動するには [Move](../move/) コマンドを使用してください。

## CV スプラインの選択

| 方法 | 動作 |
|-----|------|
| **クリック** | 曲線上の任意の点の近くにクリックした場合に選択 |
| **右ドラッグ**（厳密） | 曲線に沿ったすべてのサンプル点が選択ボックス内に収まる必要がある |
| **左ドラッグ**（交差） | 選択ボックスの境界を横切る曲線の任意の部分で選択 |

## サポートされる編集コマンド

| コマンド | スプラインへの効果 |
|---------|----------------|
| [Move](../move/) | すべての制御頂点を同じ変位で移動 |
| [Copy](../copy/) | 新しい位置に同一のスプラインを作成 |
| [Rotate](../rotate/) | 選択した基点を中心にすべての CV を回転 |
| [Mirror](../mirror/) | 鏡像軸ですべての CV を反転 |
| [Scale](../scale/) | 基点からすべての CV を均一にスケール |
| [Delete](../delete/) | スプラインを削除 |

スプラインは **Offset**、**Trim**、**Extend** をサポートしません。

## プロパティ

**一般**

| プロパティ | デフォルト | 説明 |
|---------|---------|------|
| Color | 256 (ByLayer) | ACI カラーインデックス |
| Layer | `0` | レイヤー割り当て |
| Linetype | ByLayer | 名前付き線種パターン |
| Linetype Scale | 1 | 線種パターンのスケール係数 |
| Thickness | 0 | 押し出し厚さ |

**ジオメトリ**

| プロパティ | 説明 |
|---------|------|
| Degree | 多項式次数 — 常に 3（3 次） |
| Control Vertices | すべての CV の座標 |
| Fit Points | CV スプラインの場合は空；フィット点スプラインの場合のみ入力 |

## スプライン CV とスプラインフィットの比較 — 使い分け

| | スプライン CV | スプラインフィット |
|---|-----------|------------|
| 曲線が点を通過 | 最初と最後のみ（クランプト） | クリックした各点を正確に通過 |
| 形状制御 | CV を領域に向けて引く | 曲線が通過しなければならないフィット点を移動 |
| グリップ編集の効果 | CV 移動 → 曲線が引き寄せられる | フィット点移動 → 曲線が再補間 |
| 適している用途 | 滑らかな美的曲線、フリーフォームパス | 特定の座標を通過しなければならない曲線 |

## DXF — SPLINE エンティティ（制御頂点形式）

CV スプラインは DXF ファイルに `SPLINE` エンティティとして保存され、次数、ノットベクトル、すべての制御頂点座標を格納します。すべてのプロパティ（色、レイヤー、線種、線種スケール、厚さ）はロスなく双方向対応しています。`splineFlag` は `9`（CV スプライン）に設定されているので、リロード時に形式が保持されます。CV データを持つ `SPLINE` エンティティをサポートする任意の DXF アプリケーションがこれらを正しく読み込みます。
