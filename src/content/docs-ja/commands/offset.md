# Offset（オフセット）

`offset` コマンドは固定の垂直距離でエンティティの平行コピーを作成します。距離を一度入力したら、エンティティをクリックして側面を選択します — コマンドは同じ距離を保持したまま待機するので、1 セッションで複数のオブジェクトをオフセットできます。

サポートされるエンティティタイプ：**直線、円、円弧、楕円、ポリライン**（長方形を含む）。

## オフセットの使用方法

1. ターミナルに `offset` と入力するか、**Offset** ツールバーボタンをクリックします。
2. **オフセット距離を入力**して **Enter** または **Space** を押します。
3. **オフセットするエンティティをクリック** — サポートされていないエンティティタイプの場合、エラーメッセージが表示されて別のエンティティをクリックできます。
4. **コピーを表示する側にカーソルを移動** — ライブプレビューが追従します。
5. **クリック**してオフセットコピーを配置します。

配置後、コマンドは**同じ距離**でステップ 3 に戻り、次のオフセットの準備ができています。**Escape** を押して距離入力ステップにリセットします。

```
  距離: 10

  ─────────────────    ← 元の直線
  ─────────────────    ← オフセットコピー（10 単位下）
```

## エンティティごとのオフセット動作

| エンティティ | オフセットの計算方法 |
|---------|----------------|
| **直線** | 元の方向に垂直にシフトした平行直線 |
| **円** | 同心円；外側をクリック → 大きい半径、内側をクリック → 小さい半径 |
| **円弧** | 新しい半径での同心円弧；同じ角度スパンを保持 |
| **楕円** | 両半軸を同じ距離だけ増減 |
| **ポリライン** | 各セグメントを独立してオフセット；隣接するオフセットセグメントはコーナーでマイター処理 |

**円**、**円弧**、**楕円**の場合：内側へのオフセットで半径または半軸がゼロ以下になる場合、オフセットは適用されません。

## キーボードリファレンス

| キー | 操作 |
|-----|------|
| `0`〜`9`, `.` | 距離値に数字を追加 |
| `Backspace` | 最後に入力した文字を削除 |
| `Enter` / `Space` | 入力した距離を確定してエンティティ選択に進む |
| `Escape` | 距離入力ステップにリセット |

## ワークフローのヒント

**Escape** を押すまで距離は設定されたままです。これにより同じ間隔で多くのエンティティをオフセットするのが効率的になります — 距離を一度入力して、各エンティティをクリックして側面を選択するだけです。

## Offset と Copy の比較

| | Offset | Copy |
|---|--------|------|
| 変位 | エンティティのジオメトリに垂直 | 任意のベクトル（基点 → 目的地） |
| サポートエンティティ | 直線、円、円弧、楕円、ポリライン | すべてのエンティティタイプ |
| 距離入力 | エンティティ選択前に入力 | 選択後に入力またはクリック |
| 適している用途 | 平行線、同心円、インセット/アウトセットパス | 任意の位置への複製配置 |
