# Dimension Diameter

`dimdiameter` コマンドは円弧または円に直径寸法を配置します。寸法線は中心を通って直径方向に対向する 2 つの円弧点の間に広がり、`⌀ <値>` とラベル付けされます。中心から一方の端のみを注釈付けするには [Dimension Radius](./dim-radius) を使用してください。

## 直径寸法の構造

```
  ●──────────── ⌀ 10.00 ────────────●
  （遠い円弧点）              （近い円弧点 / テキスト側）
```

- **寸法線** — 両方の円弧交点に矢印を付け、完全な直径に広がります。
- **近い円弧点** — カーソル側の円周上の点（テキストラベルが配置される場所）。
- **遠い円弧点** — 直径方向に対向する点。
- **ラベル** — `⌀` の後に直径値。

## 直径寸法の配置手順

1. ターミナルに `dimdiameter` と入力するか、**Dimension Diameter** ツールバーボタンをクリックします。
2. **円弧または円をクリック**して選択します。
3. **カーソルを移動**して寸法線を希望する角度に回転します。
4. **クリック**して寸法を配置します。

選択できるのは **Arc** と **Circle** エンティティのみです。

## Dimension Diameter と Dimension Radius の比較

| | Dimension Diameter | Dimension Radius |
|---|-------------------|-----------------|
| 計測対象 | 完全な直径（半径の 2 倍） | 半径（中心から端まで） |
| 寸法線 | 端 → 端（中心を通る） | 中心 → 端 |
| ラベル接頭辞 | `⌀` | `R` |
| 矢印 | 2 つ（両方の円弧点に） | 1 つ（円弧点に） |
| 最適用途 | 完全な円形穴またはシャフトの寸法 | 曲線フィーチャの片側の注釈 |

## DXF — DIMENSION 直径エンティティ

直径寸法は直径タイプジオメトリの `DIMENSION` エンティティとして DXF ファイルに保存され、両円弧点の位置と計測直径値（半径 × 2）が格納されます。すべてのプロパティがロスなくラウンドトリップします。
