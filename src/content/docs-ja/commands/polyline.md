# Polyline（ポリライン）

`polyline` コマンドは任意数の直線セグメントからなる接続パスを、1 つの `LWPOLYLINE` エンティティとして保存します。パス全体が 1 つのオブジェクトなので、選択するとすべてのセグメントが同時に選択されます — 1 回の操作で図形全体を移動、回転、または拡大縮小できます。これが各セグメントが独立したエンティティとなる連鎖した[直線](../line/)との主な違いです。

ポリラインは**閉じる**こともできます：[長方形](../rectangle/)コマンドはクローズフラグが設定された同じ `LWPOLYLINE` エンティティを使用します。

## ポリラインの作図

1. ターミナルに `polyline` と入力するか、**Polyline** ツールバーボタンをクリックします。
2. **最初の点をクリック**するか、`X,Y` と入力して **Enter** を押し正確な座標を指定します。
3. **後続の各点をクリック** — クリックするたびにセグメントが追加されます。各ステップで座標入力が使用できます。
4. **Enter** または **Space** を押して終了します（最低 2 点が必要）。

```
  ●──────●
  1 点目  2 点目
          \
           \  セグメント 3（進行中 — カーソルはここ）
            ●  ← クリックして追加、Enter/Space で終了
```

**Escape** を押すと、配置済みの点がすべて破棄されてコマンドが終了します。

## 座標入力

任意の頂点に対してクリックの代わりに正確な位置を入力できます：

1. X 値を入力します。
2. `,` を押します — ターミナルに `[X], [Y{カーソル}]` と表示されます。
3. Y 値を入力します。
4. **Enter** を押して頂点を配置します。

## 角度ロックと正確なセグメント長

連続する 2 点間で[直線](../line/#角度ロックと正確な長さ入力)コマンドと同じ 45° スナップロジックが適用されます。軸にロックされているとき：

| キー | 操作 |
|-----|------|
| `0`〜`9`, `.` | セグメント長に数字を追加 |
| `-` | 負の長さ — 軸方向を反転（最初の文字のみ） |
| `Backspace` | 最後に入力した文字を削除 |
| `Enter` | 入力した距離で次の点を配置 |

現在の累積長さはターミナルプロンプトにリアルタイムで表示されます。ロック中にクリックすると軸に投影されるので、新しい頂点は常に軸上に正確に配置されます。

## キーボードリファレンス

| キー | 操作 |
|-----|------|
| `0`〜`9`, `.`, `-` | X 座標入力を開始、または角度ロック中のセグメント長 |
| `,` | X をロックして Y 入力に移動 |
| `Backspace` | 最後に入力した文字を削除 |
| `Enter` | 入力した座標または長さを確定、何も入力せず ≥ 2 点ある場合はポリラインを終了 |
| `Space` | ポリラインを終了（入力が進行中でない場合は Enter と同じ） |
| `Escape` | すべての点を破棄して終了 |

## グリップ編集 — 頂点とセグメント中点

選択したポリラインには 2 種類のグリップが表示されます：

| グリップ | 位置 | 機能 |
|---------|------|------|
| **頂点** | 各配置点 | ドラッグしてその頂点を移動；接続するすべてのセグメントが追従して伸縮 |
| **セグメント中点** | 各セグメントの中央 | ドラッグしてそのセグメントの**両**端点を一緒に移動、セグメントの長さと角度を維持 |

セグメント中点グリップはポリライン固有のもの — 個々のセグメントを長さを変えずに横方向にスライドさせることができます。[直線](../line/)では中点グリップを使うとエンティティ全体の Move コマンドが起動します。

ポリライン全体を移動する単一の「移動」グリップはありません。パス全体を移動するには [Move](../move/) コマンドを使用してください。

## ポリラインの選択

| 方法 | 動作 |
|-----|------|
| **クリック** | 任意のセグメントのヒットテスト距離内にクリックした場合にポリラインを選択 |
| **右ドラッグ**（厳密） | すべての頂点がボックス内に収まる必要がある |
| **左ドラッグ**（交差） | ボックス境界を横切る任意のセグメントがポリライン全体を選択 |

ポリラインは 1 つのエンティティなので、任意のセグメントに触れる交差選択ですべてのセグメントが選択されます。

## サポートされる編集コマンド

ポリラインはすべての一般的な変形とオフセットをサポートしますが、**トリムと延長はサポートしません**（それらは[直線](../line/)専用です）：

| コマンド | ポリラインへの効果 |
|---------|----------------|
| [Move](../move/) | すべての頂点を同じ変位で移動 |
| [Copy](../copy/) | 新しい位置に同一のポリラインを作成 |
| [Rotate](../rotate/) | 選択した基点を中心にすべての頂点を回転 |
| [Mirror](../mirror/) | 鏡像軸ですべての頂点を反転 |
| [Scale](../scale/) | 基点からすべての頂点を均一に拡大縮小 |
| [Offset](../offset/) | 固定の垂直距離で平行なポリラインを作成 |
| [Delete](../delete/) | 図面からポリラインを削除 |

## プロパティ

ポリラインを選択するとプロパティパネルに以下が表示されます：

**一般**

| プロパティ | デフォルト | 説明 |
|---------|---------|------|
| Color | 256 (ByLayer) | ACI カラーインデックス |
| Layer | `0` | レイヤー割り当て |
| Linetype | ByLayer | 名前付き線種パターン |
| Linetype Scale | 1 | 線種パターンのスケール係数 |
| Thickness | 0 | 押し出し厚さ |

**ジオメトリ**

| プロパティ | 説明 |
|---------|------|
| Closed | 最後の頂点が最初の頂点に接続しているか |
| Vertex Count | 頂点の総数 |
| Vertices | すべての頂点の座標リスト |

## ポリラインと直線の比較 — 使い分け

| | ポリライン | 直線 |
|---|---------|------|
| エンティティ数 | パス全体で 1 つの `LWPOLYLINE` | セグメントごとに 1 つの `LINE` |
| 閉じた図形 | あり（クローズフラグ） | なし |
| トリム / 延長 | なし | あり — セグメントごと |
| セグメント中点グリップ | セグメント全体を移動 | エンティティの Move を起動 |
| 適している用途 | アウトライン、輪郭、全体を保持する図形 | 補助線、トリムする図形 |

## DXF — LWPOLYLINE エンティティ

ポリラインは DXF ファイルに `LWPOLYLINE` エンティティとして保存されます。すべてのプロパティ（頂点座標、クローズフラグ、色、レイヤー、線種、線種スケール、厚さ）はロスなく双方向対応しています。[長方形](../rectangle/)コマンドで作図した長方形も `LWPOLYLINE`（閉じた 4 頂点）として保存され、DXF レベルでは区別できません。

DXF 対応アプリケーション（LibreCAD、FreeCAD など）からの `LWPOLYLINE` エンティティは、エディターで完全に編集可能なポリラインとして読み込まれます。
