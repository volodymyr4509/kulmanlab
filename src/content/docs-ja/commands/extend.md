# Extend

`extend` コマンドは [Line](../line/) の最近傍端点を、図面内の他のエンティティとの最も近い交点まで延長します。延長したい端点付近にカーソルを合わせると — プレビューで延長後の直線が表示され — クリックで適用します。

Extend は **Line エンティティのみ**で機能します。直線が向かう境界となるエンティティは Text、Mtext、Multileader、Spline を除く任意のエンティティタイプが使用できます。

## 直線の延長手順

1. ターミナルに `extend` と入力するか、**Extend** ツールバーボタンをクリックします。
2. **直線の一端付近にカーソルを合わせる** — その方向の最近傍境界まで延長した直線のプレビューが表示されます。
3. **クリック**して延長を適用します。

各延長の後もコマンドはアクティブのままなので、複数の直線を連続して延長できます。終了するには **Escape** を押します。

```
  前:                      後:

  ──────           |           ──────────────|
  （短い直線）     （境界）  （境界まで延長された直線）
```

## 端点の選択方法

コマンドはカーソルがどの端点に近いかを確認します:

- カーソルが**終点に近い** → 終点が直線方向に沿って前方に延長されます。
- カーソルが**始点に近い** → 始点が逆方向（反対方向）に延長されます。

選択した端点から直線方向にレイが投射され、そのレイ上の他のエンティティとの**最も近い交点**（直線自体と無視されるタイプを除く）が新しい端点になります。

その方向に交点が見つからない場合、プレビューは表示されず、クリックしても何も起こりません。

## 境界の除外

以下のエンティティタイプは境界として無視されます — 直線はこれらに向かって延長されません:

- Text / Mtext
- Multileader
- Spline

その他すべてのタイプ（Line、Arc、Circle、Ellipse、Polyline、Dimension）は有効な境界として機能します。

## キーボードリファレンス

| キー | 操作 |
|-----|------|
| `Escape` | 延長モードを終了 |

## サポートされるエンティティ

| エンティティ | 延長可能? |
|--------|----------------|
| Line | あり |
| Arc, Circle, Ellipse | なし |
| Polyline / Rectangle | なし |
| Text, Spline, Dimension, Leader | なし |

## Extend と Trim の比較

| | Extend | Trim |
|---|--------|------|
| 機能 | 直線の端点を境界まで延長 | 直線のセグメントを削除 |
| トリガー | 延長する端点付近にカーソルを合わせる | カットするセグメントにカーソルを合わせる |
| 結果 | 直線の端点が外側に移動 | 直線が分割または短縮 |
| 共通点 | 直線のみ | 直線のみ |
