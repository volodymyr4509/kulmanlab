# Help

Avaa sovelluksen sisäisen ohjepaneelin, joka sisältää yhteenvedon kaikista pikanäppäimistä, navigointivinkeistä ja yleisestä käytöstä.

## Käyttäminen

Napsauta **Help**-painiketta työkalurivillä tai kirjoita `help` terminaaliin. Ohjepaneeli avautuu välittömästi ja komento päättyy.

## Mitä ohjepaneeli sisältää

### Navigointi
| Pikanäppäin | Toiminto |
|----------|--------|
| `Rullan pyöritys` | Zoomaa sisään / ulos |
| `Keskiklikkaus + veto` | Panoroi näkymää |
| `Kaksois-keskiklikkaus` | Sovita kaikki entiteetit näytölle |

### Komennot
| Pikanäppäin | Toiminto |
|----------|--------|
| Kirjoita | Aloita komennon nimen kirjoittaminen — ehdotukset ilmestyvät piirtoalueelle |
| `Delete` | Poista kaikki valitut entiteetit |
| `Enter` / `Space` | Vahvista komento tai syöte |
| `Enter` / `Space` (jouten) | Toista viimeksi käytetty komento |
| `Escape` | Peruuta nykyinen komento tai tyhjennä valinta |

### Valinta
| Pikanäppäin | Toiminto |
|----------|--------|
| `Vasen klikkaus` | Valitse entiteetti |
| `Veto →` | Valitse vain **kokonaan vedon alueen sisällä** olevat entiteetit |
| `Veto ←` | Valitse mikä tahansa entiteetti, joka **koskettaa** vedon aluetta |
| `Vasen klikkaus kahvassa` | Muokkaa kärkeä tai päätepistettä suoraan |

### Piirtämisen vinkit
- **Tarttumapisteet** — kohdistin tarttuu automaattisesti lähellä oleviin entiteettien päätepisteisiin.
- **Tarttuva kulma** — siirrä kohdistinta tarpeeksi kauas yhteen suuntaan lukitaksesi 45°:n lisäyksiin.
- **Tarkka pituus** — kun kulma on lukittu, kirjoita numero ja paina `Enter` asettaaksesi tarkan etäisyyden.
