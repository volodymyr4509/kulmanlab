# Help

`help` komutu, yerleşik klavye kısayolları referansını ve belge bağlantılarını görüntüler. Araç çubuğundaki **Help** düğmesine basın veya terminale `help` yazın.

## Klavye Kısayolları — Gezinme

| Tuş / İşlem | Sonuç |
|------------|-------|
| Fare tekerleği kaydırma | İmleç etrafında Yakınlaştır / Uzaklaştır |
| Orta düğmeyle sürükle | Görünümü kaydır (Pan) |
| Orta düğmeye çift tıkla | Tüm çizimi sığdır (Fit) |

## Klavye Kısayolları — Komutlar

| Tuş | Sonuç |
|-----|-------|
| Komut adını yaz | Komutu etkinleştirir |
| `Delete` | Seçili nesneleri siler |
| `Enter` / `Boşluk` | Seçimi onaylar veya adımı tamamlar |
| `Escape` | Geçerli komutu iptal eder / seçimi sıfırlar |

## Klavye Kısayolları — Seçim

| İşlem | Sonuç |
|-------|-------|
| Nesneye sol tıkla | Nesneyi seçer |
| Sola sürükle (çapraz çerçeve) | Çerçeveyle kesişen nesneleri seçer |
| Sağa sürükle (katı çerçeve) | Tamamen çerçeve içindeki nesneleri seçer |
| Tutamaca sol tıkla | Tutamaç düzenlemeyi etkinleştirir |

## Çizim İpuçları

- Noktalara tıklamak yerine kesin nokta girişi için `X,Y` koordinatları yazın.
- 45° eksen yönünü kilitleyin — kilitleme göründüğünde imleci yaklaştırın.
- Kesin hareket için açıyı kilitledikten sonra mesafeyi yazın ve Enter tuşuna basın.

## Sürüm ve İletişim
