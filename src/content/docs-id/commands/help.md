# Help

Membuka panel bantuan dalam aplikasi dengan ringkasan semua pintasan keyboard, tips navigasi, dan penggunaan umum.

## Cara menggunakan

Klik tombol toolbar **Help** atau ketik `help` di terminal. Panel bantuan langsung terbuka dan perintah keluar.

## Apa yang ada di panel bantuan

### Navigasi
| Pintasan | Aksi |
|----------|--------|
| `Scroll wheel` | Zoom masuk / keluar |
| `Middle click + drag` | Geser tampilan |
| `Double middle click` | Sesuaikan semua entitas ke layar |

### Perintah
| Pintasan | Aksi |
|----------|--------|
| Ketik | Mulai mengetik nama perintah — saran muncul di kanvas |
| `Delete` | Hapus semua entitas yang dipilih |
| `Enter` / `Space` | Konfirmasi perintah atau input |
| `Enter` / `Space` (idle) | Ulangi perintah terakhir yang digunakan |
| `Escape` | Batalkan perintah saat ini atau hapus seleksi |

### Seleksi
| Pintasan | Aksi |
|----------|--------|
| `Left click` | Pilih entitas |
| `Drag →` | Pilih hanya entitas yang **sepenuhnya berada di dalam** area seret |
| `Drag ←` | Pilih entitas mana pun yang **menyentuh** area seret |
| `Left click grip` | Edit vertex atau titik akhir secara langsung |

### Tips menggambar
- **Titik snap** — kursor secara otomatis snap ke titik akhir entitas terdekat.
- **Sudut tetap** — gerakkan kursor cukup jauh ke satu arah untuk mengunci ke kelipatan 45°.
- **Panjang tepat** — saat sudut terkunci, ketik angka dan tekan `Enter` untuk mengatur jarak tepat.
