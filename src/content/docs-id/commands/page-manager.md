# PageManager

Perintah `PageManager` membuka dialog pengaturan halaman untuk layout aktif, memungkinkan Anda mengubah format kertas, orientasi, dan skala antara satuan gambar dan milimeter.

## Membuka Page Manager

- Ketik `PageManager` di terminal, **atau**
- Klik kanan tab layout di bagian bawah dan pilih **Page Manager**.

## Pengaturan

| Pengaturan | Deskripsi |
|---------|-------------|
| **Format kertas** | Ukuran lembar standar (A4, A3, A2, Letter, dll.) |
| **Orientasi** | Potret atau Lanskap |
| **Skala** | Satuan gambar per mm — mengontrol cara koordinat model dipetakan ke kertas |

Mengubah ukuran kertas atau skala menskalakan ulang semua viewport yang ada di layout secara proporsional sehingga posisinya tetap benar.

## Catatan

- PageManager hanya berlaku untuk layout kertas. Tidak tersedia di model space.
- Untuk menambahkan layout baru, klik tombol **+** di bar tab di bagian bawah.
- Untuk mengubah nama atau menghapus layout, klik kanan tab-nya.
