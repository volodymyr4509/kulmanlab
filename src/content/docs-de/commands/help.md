# Help

Öffnet das In-App-Hilfe-Panel mit einer Übersicht aller Tastenkombinationen, Navigationshinweise und allgemeinen Nutzungshinweisen.

## So verwenden Sie es

Klicken Sie auf die Schaltfläche **Help** in der Symbolleiste oder geben Sie `help` im Terminal ein. Das Hilfe-Panel öffnet sich sofort und der Befehl endet.

## Inhalt des Hilfe-Panels

### Navigation
| Tastenkombination | Aktion |
|-------------------|--------|
| `Scroll wheel` | Vergrößern / Verkleinern |
| `Middle click + drag` | Ansicht verschieben |
| `Double middle click` | Alle Elemente an den Bildschirm anpassen |

### Befehle
| Tastenkombination | Aktion |
|-------------------|--------|
| Tippen | Befehlsnamen eingeben — Vorschläge erscheinen auf der Zeichenfläche |
| `Delete` | Alle ausgewählten Elemente löschen |
| `Enter` / `Space` | Befehl oder Eingabe bestätigen |
| `Enter` / `Space` (im Leerlauf) | Letzten verwendeten Befehl wiederholen |
| `Escape` | Aktuellen Befehl abbrechen oder Auswahl aufheben |

### Auswahl
| Tastenkombination | Aktion |
|-------------------|--------|
| `Left click` | Element auswählen |
| `Drag →` | Nur Elemente auswählen, die **vollständig innerhalb** des Ziehbereichs liegen |
| `Drag ←` | Jedes Element auswählen, das den Ziehbereich **berührt** |
| `Left click grip` | Scheitelpunkt oder Endpunkt direkt bearbeiten |

### Zeichentipps
- **Einrastpunkte** — der Mauszeiger rastet automatisch an nahe gelegenen Elementendpunkten ein.
- **Winkelsperre** — Mauszeiger weit genug in eine Richtung bewegen, um auf 45°-Schritte zu sperren.
- **Exakte Länge** — während der Winkel gesperrt ist, eine Zahl eingeben und `Enter` drücken, um einen genauen Abstand festzulegen.
