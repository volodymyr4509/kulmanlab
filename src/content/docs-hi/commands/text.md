# Text

`text` कमांड बहु-पंक्ति टेक्स्ट लेबल रखती है। कैनवास पर एक स्थिति क्लिक करने के बाद, एक पॉपअप एडिटर **रिच** मोड में खुलता है — आप सामग्री टाइप कर सकते हैं, प्रति-अक्षर बोल्ड/इटैलिक/स्ट्राइकथ्रू लागू कर सकते हैं, फ़ॉन्ट और ऊँचाई बदल सकते हैं, और लाइन ब्रेक डाल सकते हैं। कमिट और एडिटर बंद करने के लिए **Escape** दबाएं।

पूर्ण एडिटर संदर्भ के लिए [Text Editor](../../interface/text-editor/) पृष्ठ देखें, जिसमें **रिच** और **सरल** मोड की तुलना शामिल है।

## टेक्स्ट लेबल रखना

1. टर्मिनल में `text` टाइप करें या **Text** टूलबार बटन पर क्लिक करें।
2. कैनवास पर **एंकर स्थिति पर क्लिक करें**। या सटीक निर्देशांक के लिए `X,Y` टाइप करें और **Enter** दबाएं।
3. **टेक्स्ट एडिटर पॉपअप** नए लेबल के ऊपर खुलता है। अपनी सामग्री टाइप करें।
4. लेबल कमिट करने और एडिटर बंद करने के लिए **Escape** दबाएं।

डिफ़ॉल्ट ऊँचाई **12 ड्राइंग इकाइयाँ** है।

## मौजूदा लेबल संपादित करना

किसी भी टेक्स्ट लेबल पर उस लेबल के लिए एडिटर फिर से खोलने के लिए **डबल-क्लिक** करें।

## एंकर के लिए निर्देशांक प्रविष्टि

क्लिक करने के बजाय, सटीक स्थिति टाइप करें:

1. X मान टाइप करें।
2. `,` दबाएं — टर्मिनल `[X], [Y{cursor}]` दिखाता है।
3. Y मान टाइप करें।
4. एंकर रखने और एडिटर खोलने के लिए **Enter** दबाएं।

## कीबोर्ड संदर्भ

**एंकर चरण**

| Key | Action |
|-----|--------|
| `0`–`9`, `.`, `-` | X निर्देशांक प्रविष्टि शुरू करें |
| `,` | X लॉक करें और Y प्रविष्टि पर जाएं |
| `Backspace` | अंतिम टाइप किया गया अक्षर हटाएं |
| `Enter` | टाइप किए गए निर्देशांक की पुष्टि करें |

**टेक्स्ट एडिटर चरण** (पूर्ण संदर्भ के लिए [Text Editor](../../interface/text-editor/) देखें)

| Key | Action |
|-----|--------|
| कोई भी प्रिंट करने योग्य अक्षर | कर्सर पर डालें |
| `Backspace` / `Delete` | आसन्न अक्षर या चयन हटाएं |
| `Enter` | लाइन ब्रेक डालें |
| `←` / `→` | कैरेट मूव करें |
| `Home` / `End` | कठिन पंक्ति के प्रारंभ / अंत पर जाएं |
| `Escape` | कमिट करें और एडिटर बंद करें |

## ग्रिप संपादन — पुनःस्थित करना

एक चयनित टेक्स्ट लेबल में एंकर बिंदु पर एक ग्रिप होता है:

| ग्रिप | स्थान | क्या करता है |
|------|----------|--------------|
| **Anchor** | टेक्स्ट का निचला-बाएं | लेबल पुनःस्थित करने के लिए ड्रैग करें |

## टेक्स्ट चुनना

| विधि | व्यवहार |
|--------|-----------|
| **क्लिक** | यदि क्लिक टेक्स्ट के घुमाए हुए बाउंडिंग बॉक्स के अंदर है तो चुनता है |
| **दाईं ओर ड्रैग** (सख्त) | बाउंडिंग बॉक्स के सभी चार कोने चयन क्षेत्र के अंदर होने चाहिए |
| **बाईं ओर ड्रैग** (क्रॉसिंग) | टेक्स्ट बाउंडिंग बॉक्स और चयन क्षेत्र के बीच कोई भी ओवरलैप उसे चुनता है |

## समर्थित संपादन कमांड

| कमांड | टेक्स्ट पर क्या होता है |
|---------|--------------------------|
| [Move](../move/) | एंकर बिंदु मूव करता है |
| [Copy](../copy/) | नई स्थिति पर समान लेबल बनाता है |
| [Rotate](../rotate/) | एंकर स्थिति घुमाता है और Rotation Degree में कोण जोड़ता है |
| [Mirror](../mirror/) | एंकर बिंदु को मिरर अक्ष पर परावर्तित करता है (टेक्स्ट स्ट्रिंग पलटी नहीं जाती) |
| [Scale](../scale/) | एंकर स्थिति स्केल करता है और ऊँचाई को स्केल फैक्टर से गुणा करता है |
| [Delete](../delete/) | लेबल हटाता है |

टेक्स्ट **Offset**, **Trim**, या **Extend** का समर्थन नहीं करता।

## गुण

जब टेक्स्ट लेबल चुना जाता है तो गुण पैनल दिखाता है:

**सामान्य**

| गुण | डिफ़ॉल्ट | अर्थ |
|----------|---------|---------|
| Color | 256 (ByLayer) | ACI रंग सूचकांक |
| Layer | `0` | लेयर असाइनमेंट |

**ज्यामिति**

| गुण | अर्थ |
|----------|---------|
| Position X / Position Y | एंकर बिंदु निर्देशांक |
| Height | ड्राइंग इकाइयों में बेस टेक्स्ट ऊँचाई (डिफ़ॉल्ट: **12**) |
| Rotation Degree | डिग्री में काउंटर-क्लॉकवाइज़ घुमाव |

**गुण**

| गुण | अर्थ |
|----------|---------|
| Content | टेक्स्ट स्ट्रिंग (MTEXT इनलाइन कोड संरक्षित) |
| Attachment Point | संरेखण कोड (1 = ऊपर-बाएं … 9 = नीचे-दाएं) |

टेक्स्ट में Linetype, Linetype Scale, या Thickness गुण नहीं हैं।

## DXF — MTEXT ऑब्जेक्ट

टेक्स्ट लेबल DXF फ़ाइल में **MTEXT** ऑब्जेक्ट के रूप में संग्रहीत होते हैं। बोल्ड और इटैलिक `\L`, `\K`, `\O`, और इनलाइन फ़ॉन्ट स्विच (`\f`) का उपयोग करके एन्कोड किए जाते हैं। प्रति-अक्षर ऊँचाई `\H` के रूप में एन्कोड की जाती है। सभी फ़ॉर्मेटिंग पूर्ण DXF राउंड-ट्रिप में बची रहती है और LibreCAD, FreeCAD और अन्य DXF-संगत एप्लिकेशन द्वारा पढ़ी जा सकती है।
