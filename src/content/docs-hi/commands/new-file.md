# New File

**New File** कमांड कैनवास साफ़ करती है और नई खाली ड्राइंग शुरू करती है। टाइमस्टैंप के साथ एक अनूठा फ़ाइल नाम स्वचालित रूप से बनाया जाता है।

## नई फ़ाइल कैसे बनाएं

File पैनल में **New File** टूलबार बटन (नया पेज आइकन) पर क्लिक करें। कैनवास तुरंत साफ़ होता है — कोई प्रॉम्प्ट या पुष्टि डायलॉग नहीं।

## नई फ़ाइल में क्या होता है

नई बनाई गई फ़ाइल के साथ शुरुआत:

- कैनवास पर **कोई ऑब्जेक्ट नहीं**।
- **एक डिफ़ॉल्ट लेयर** जिसका नाम `0` है, रंग सफ़ेद और लाइनटाइप `Continuous` के साथ।
- `kulman-Mon01_HH:MM:SS.dxf` प्रारूप में **जेनरेट किया गया फ़ाइल नाम** (जैसे `kulman-May22_14:30:00.dxf`)।

फ़ाइल स्वचालित रूप से ब्राउज़र स्टोरेज में सेव होती है और [Recent Files](../files/) में दिखाई देती है।

## चेतावनी — असेव काम हटा दिया जाएगा

**New File** पर क्लिक करने से बिना चेतावनी के वर्तमान कैनवास के सभी ऑब्जेक्ट हटा दिए जाते हैं। यदि आप वर्तमान ड्राइंग रखना चाहते हैं, तो पहले उसे [export](../export/) करें।

## New File बनाम Import

| स्थिति | अनुशंसित क्रिया |
|--------|--------------|
| शुरू से ड्राइंग बनाना | **New File** |
| मौजूदा DXF या JSON फ़ाइल खोलना | [Import](../import/) |
| किसी वेरिएंट पर काम करने के लिए ड्राइंग कॉपी करना | वर्तमान फ़ाइल [Export](../export/) करें, फिर कॉपी [Import](../import/) करें |

## संबंधित कमांड

- [Import](../import/) — मौजूदा DXF या JSON ड्राइंग खोलें
- [Export](../export/) — नया शुरू करने से पहले ड्राइंग डाउनलोड करें
- [Files](../files/) — ब्राउज़र स्टोरेज से पिछली ड्राइंग पुनर्स्थापित करें
