# Help

इन-ऐप हेल्प पैनल खोलता है जिसमें सभी कीबोर्ड शॉर्टकट का सारांश, नेविगेशन टिप्स, और सामान्य उपयोग जानकारी होती है।

## कैसे उपयोग करें

**Help** टूलबार बटन पर क्लिक करें या टर्मिनल में `help` टाइप करें। हेल्प पैनल तुरंत खुलता है और कमांड बाहर निकलती है।

## हेल्प पैनल में क्या है

### नेविगेशन
| शॉर्टकट | एक्शन |
|---------|-------|
| `Scroll wheel` | ज़ूम इन / आउट |
| `Middle click + drag` | व्यू पैन करें |
| `Double middle click` | सभी ऑब्जेक्ट स्क्रीन पर फिट करें |

### कमांड
| शॉर्टकट | एक्शन |
|---------|-------|
| Type | कमांड नाम टाइप करना शुरू करें — सुझाव कैनवास पर दिखाई देते हैं |
| `Delete` | सभी चयनित ऑब्जेक्ट हटाएं |
| `Enter` / `Space` | कमांड या इनपुट की पुष्टि करें |
| `Enter` / `Space` (idle) | अंतिम उपयोग की गई कमांड दोहराएं |
| `Escape` | वर्तमान कमांड रद्द करें या चयन साफ़ करें |

### चयन
| शॉर्टकट | एक्शन |
|---------|-------|
| `Left click` | एक ऑब्जेक्ट चुनें |
| `Drag →` | केवल ड्रैग क्षेत्र के **पूरी तरह अंदर** ऑब्जेक्ट चुनें |
| `Drag ←` | कोई भी ऑब्जेक्ट जो ड्रैग क्षेत्र को **छूता** है चुनें |
| `Left click grip` | सीधे वर्टेक्स या एंडपॉइंट संपादित करें |

### ड्राइंग टिप्स
- **स्नैप पॉइंट** — कर्सर स्वचालित रूप से पास के ऑब्जेक्ट एंडपॉइंट पर स्नैप होता है।
- **स्टिकी एंगल** — 45° वृद्धियों पर लॉक करने के लिए एक दिशा में कर्सर को पर्याप्त दूर मूव करें।
- **सटीक लंबाई** — कोण लॉक होने पर, एक सटीक दूरी सेट करने के लिए एक नंबर टाइप करें और `Enter` दबाएं।
