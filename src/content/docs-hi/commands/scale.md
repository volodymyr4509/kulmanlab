# Scale

`scale` कमांड चयनित ऑब्जेक्ट को आधार बिंदु के चारों ओर समान रूप से आकार बदलती है। आधार बिंदु से सभी दूरियां स्केल फैक्टर से गुणा होती हैं — `2` का फैक्टर सभी आयाम दोगुना करता है, `0.5` आधा करता है। फैक्टर हमेशा टाइप करके दर्ज होता है; क्लिक-से-स्केल नहीं है।

## शुरू करने के दो तरीके

**पहले चुनें, फिर स्केल करें** — पहले ऑब्जेक्ट चुनें, फिर सक्रिय करें:

1. कैनवास पर एक या अधिक ऑब्जेक्ट चुनें।
2. टर्मिनल में `scale` टाइप करें या **Scale** टूलबार बटन पर क्लिक करें।
3. **आधार बिंदु पर क्लिक करें** — वह स्थिर बिंदु जो स्केलिंग के दौरान नहीं मूव करता। या सटीक निर्देशांक के लिए `X,Y` टाइप करें और **Enter** दबाएं।
4. **स्केल फैक्टर टाइप करें** और **Enter** दबाएं।

**पहले सक्रिय करें, फिर चुनें** — कुछ भी चुने बिना कमांड शुरू करें:

1. `scale` टाइप करें या टूलबार बटन पर क्लिक करें।
2. **ऑब्जेक्ट चुनें** — टॉगल करने के लिए क्लिक करें, या क्षेत्र से चुनने के लिए ड्रैग करें।
3. चयन की पुष्टि करने के लिए **Enter** या **Space** दबाएं।
4. **आधार बिंदु पर क्लिक करें** (निर्देशांक प्रविष्टि उपलब्ध), फिर फैक्टर टाइप करें।

```
  Base ●                Base ●
        [entity]   →          [larger entity]
  factor = 2 → distances from ● are doubled
```

## स्केल फैक्टर टाइप करना

आधार बिंदु रखने के बाद टर्मिनल `enter scale factor:` दिखाता है और कीबोर्ड इनपुट की प्रतीक्षा करता है:

| Key | Action |
|-----|--------|
| `0`–`9`, `.` | फैक्टर में अंक जोड़ें |
| `-` | नकारात्मक फैक्टर (केवल पहला अक्षर — उलटता है फिर स्केल करता है) |
| `Backspace` | अंतिम टाइप किया गया अक्षर हटाएं |
| `Enter` | टाइप किए गए फैक्टर पर स्केल लागू करें |

फैक्टर शून्येतर होना चाहिए। सामान्य मान:

| फैक्टर | प्रभाव |
|--------|--------|
| `2` | सभी आयाम दोगुना करता है |
| `0.5` | सभी आयाम आधा करता है |
| `1.5` | 50% से बढ़ाता है |
| `-1` | आधार बिंदु पर मिरर करता है (180° रोटेशन के बराबर) |

टाइप करते समय कोई लाइव प्रीव्यू नहीं है — स्केल किया गया परिणाम केवल **Enter** दबाने के बाद दिखाई देता है।

## कीबोर्ड संदर्भ

| Key | Action |
|-----|--------|
| `Enter` / `Space` | चयन की पुष्टि करें |
| `0`–`9`, `.`, `-` | X निर्देशांक प्रविष्टि शुरू करें (आधार-बिंदु चरण), या स्केल फैक्टर (फैक्टर चरण) |
| `,` | X लॉक करें और Y प्रविष्टि पर जाएं (आधार-बिंदु चरण) |
| `Backspace` | अंतिम टाइप किया गया अक्षर हटाएं |
| `Enter` | निर्देशांक की पुष्टि करें या स्केल लागू करें |
| `Escape` | रद्द करें और रीसेट करें |

## कमांड के दौरान चयन

| विधि | व्यवहार |
|--------|-----------|
| **क्लिक** | कर्सर के नीचे ऑब्जेक्ट को टॉगल करता है |
| **दाईं ओर ड्रैग** (सख्त) | बॉक्स के पूरी तरह अंदर ऑब्जेक्ट जोड़ता है |
| **बाईं ओर ड्रैग** (क्रॉसिंग) | बॉक्स को काटने वाले ऑब्जेक्ट जोड़ता है |
| **Enter** / **Space** | चयन की पुष्टि करता है |

## क्या स्केल किया जाता है

सभी ऑब्जेक्ट प्रकार समर्थित हैं। प्रत्येक ऑब्जेक्ट आधार बिंदु के सापेक्ष अपनी ज्यामिति स्केल करता है:

| ऑब्जेक्ट | क्या बदलता है |
|--------|-------------|
| Line | दोनों एंडपॉइंट आधार बिंदु से स्केल किए जाते हैं |
| Circle | केंद्र आधार बिंदु से स्केल; त्रिज्या फैक्टर से गुणा |
| Arc | केंद्र स्केल; त्रिज्या फैक्टर से गुणा; कोण अपरिवर्तित |
| Ellipse | केंद्र स्केल; दोनों अर्ध-अक्ष लंबाई फैक्टर से गुणा |
| Polyline / Rectangle | हर वर्टेक्स आधार बिंदु से स्केल |
| Text | एंकर बिंदु स्केल; ऊँचाई फैक्टर से गुणा |
| Spline | सभी कंट्रोल वर्टेक्स / फिट पॉइंट स्केल |
