# Array Grid

`ArrayGrid` कमांड चयनित ऑब्जेक्ट से एक आयताकार ग्रिड की कॉपी बनाती है — पंक्तियों की संख्या, स्तंभों की संख्या और उनके बीच की दूरी दर्ज करें, सब कुछ टर्मिनल में टाइप किया जाता है। मूल चयन पंक्ति 0, स्तंभ 0 सेल पर रहता है; बाकी हर सेल एक अनुवादित कॉपी है।

## शुरू करने के दो तरीके

**पहले चुनें, फिर array करें** — पहले ऑब्जेक्ट चुनें, फिर सक्रिय करें:

1. कैनवास पर एक या अधिक ऑब्जेक्ट चुनें।
2. टर्मिनल में `arraygrid` टाइप करें (केवल `arr` ही काफी है — यह स्पष्ट है) या **Array Grid** टूलबार बटन पर क्लिक करें।
3. **पंक्तियों** की संख्या टाइप करें और **Enter** दबाएं।
4. **स्तंभों** की संख्या टाइप करें और **Enter** दबाएं।
5. **पंक्तियों के बीच की दूरी** टाइप करें और **Enter** दबाएं।
6. **स्तंभों के बीच की दूरी** टाइप करें और **Enter** दबाएं — ग्रिड तुरंत बन जाती है।

**पहले सक्रिय करें, फिर चुनें** — कुछ भी चुने बिना कमांड शुरू करें:

1. `arraygrid` टाइप करें या टूलबार बटन पर क्लिक करें।
2. **ऑब्जेक्ट चुनें** — व्यक्तिगत ऑब्जेक्ट टॉगल करने के लिए क्लिक करें, या क्षेत्र से चुनने के लिए ड्रैग करें।
3. चयन की पुष्टि करने के लिए **Enter** या **Space** दबाएं।
4. ऊपर की तरह पंक्तियों → स्तंभों → पंक्ति दूरी → स्तंभ दूरी के साथ जारी रखें।

```
  2 पंक्तियाँ x 3 स्तंभ:

  [B] [B] [B]   <- पंक्ति 1 (अनुवादित कॉपी)
  [A] [A] [A]   <- पंक्ति 0: मूल चयन, कॉपी दाईं ओर
```

> टर्मिनल को स्पष्ट होने के लिए केवल उतने ही अक्षरों की आवश्यकता होती है — `arr` टाइप करना और **Enter** दबाना सीधे Array Grid को सक्रिय करता है, क्योंकि कोई अन्य कमांड नाम इन तीन अक्षरों से शुरू नहीं होता (Arc, Area, Align, और Angle सभी पहले ही अलग हो जाते हैं)।

## पंक्तियाँ, स्तंभ और दूरी

| प्रॉम्प्ट | स्वीकार करता है | नोट्स |
|-----------|----------------|-------|
| पंक्तियाँ | धनात्मक पूर्ण संख्याएँ (1, 2, 3…) | केवल अंक — कोई दशमलव बिंदु या चिह्न नहीं |
| स्तंभ | धनात्मक पूर्ण संख्याएँ (1, 2, 3…) | केवल अंक — कोई दशमलव बिंदु या चिह्न नहीं |
| पंक्ति दूरी | चिह्नित दशमलव (जैसे `10`, `-5.5`) | पंक्तियों के बीच की दूरी; ऋणात्मक दिशा उलट देता है |
| स्तंभ दूरी | चिह्नित दशमलव (जैसे `10`, `-5.5`) | स्तंभों के बीच की दूरी; ऋणात्मक दिशा उलट देता है |

1 पंक्ति और 1 स्तंभ के साथ, कोई कॉपी नहीं बनाई जाती — कमांड ड्राइंग में कोई बदलाव किए बिना बाहर निकल जाती है।

## कीबोर्ड संदर्भ

| Key | Action |
|-----|--------|
| `Enter` / `Space` | चयन की पुष्टि करें और पंक्ति प्रॉम्प्ट पर आगे बढ़ें |
| `0`–`9` | पंक्तियों या स्तंभों के लिए अंक दर्ज करें |
| `0`–`9`, `.`, `-` | पंक्ति/स्तंभ दूरी के लिए अंक दर्ज करें (`-` केवल पहले अक्षर के रूप में) |
| `Backspace` | वर्तमान प्रॉम्प्ट का अंतिम टाइप किया गया अक्षर हटाएं |
| `Enter` | वर्तमान प्रॉम्प्ट की पुष्टि करें और अगले पर आगे बढ़ें |
| `Escape` | टाइप किए गए पंक्ति/स्तंभ/दूरी मान साफ़ करें और चयन चरण पर लौटें |

## कमांड के दौरान चयन

| विधि | व्यवहार |
|--------|-----------|
| **क्लिक** | कर्सर के नीचे ऑब्जेक्ट को चयन में टॉगल करता है |
| **दाईं ओर ड्रैग** (सख्त) | बॉक्स के पूरी तरह अंदर ऑब्जेक्ट जोड़ता है |
| **बाईं ओर ड्रैग** (क्रॉसिंग) | बॉक्स की सीमा को काटने वाले ऑब्जेक्ट जोड़ता है |
| **Enter** / **Space** | चयन की पुष्टि करता है और पंक्ति प्रॉम्प्ट पर आगे बढ़ता है |

## Array के बाद

नई कॉपी ड्राइंग में जोड़ी जाती हैं और कमांड बाहर निकल जाती है — मूल चयन साफ़ हो जाता है। **Array Grid** फिर से चलाएं, या एक नई कमांड शुरू करें।

## Array Grid बनाम Copy

| | Array Grid | Copy |
|---|-----------|------|
| पॉइंट चयन | कोई नहीं — पंक्तियाँ, स्तंभ और दूरी टाइप की जाती हैं | आधार बिंदु और गंतव्य पर क्लिक किया जाता है (या टाइप किया जाता है) |
| बनाई गई कॉपी | पंक्तियाँ × स्तंभ − 1 | प्रति कॉपी ऑपरेशन ठीक 1 |
| लेआउट | नियमित आयताकार ग्रिड | कहीं भी, किसी भी ऑफसेट पर |
| सबसे अच्छा | नियमित पैटर्न में एक इकाई को दोहराना (छेद, टाइलें, फास्टनर) | किसी मनमाने स्थान पर एक एकल डुप्लिकेट |

## समर्थित ऑब्जेक्ट

Array Grid हर ऑब्जेक्ट प्रकार पर काम करता है। सभी ऑब्जेक्ट आंतरिक रूप से `translate(dx, dy)` लागू करते हैं — वही ऑपरेशन जो [Copy](../copy/) और [Move](../move/) उपयोग करते हैं — इसलिए कोई भी बाहर नहीं है।
