# Dimension Linear

`dimlinear` कमांड दो एक्सटेंशन-लाइन उत्पत्ति के बीच एक क्षैतिज या ऊर्ध्वाधर माप रखती है। माप रेखा हमेशा या तो पूरी तरह क्षैतिज या पूरी तरह ऊर्ध्वाधर चलती है — इसे मनमाने कोण पर नहीं रखा जा सकता। विकर्ण रेखा के समानांतर माप के लिए [Dimension Aligned](./dim-aligned) का उपयोग करें।

## रेखीय माप की संरचना

```
  |←————— 5.00 —————→|
  |                   |
  ●  (ext line 1)     ●  (ext line 2)
  p1                  p2
```

- **एक्सटेंशन लाइन** — प्रत्येक मापे गए बिंदु से माप रेखा के लंबवत।
- **माप रेखा** — क्षैतिज (X दूरी मापती है) या ऊर्ध्वाधर (Y दूरी मापती है)।
- **मान** — चुने गए अक्ष के साथ प्रक्षेपित दूरी, वास्तविक बिंदु-से-बिंदु दूरी नहीं।

## रेखीय माप रखना

1. टर्मिनल में `dimlinear` टाइप करें या **Dimension Linear** टूलबार बटन पर क्लिक करें।
2. **पहली एक्सटेंशन लाइन उत्पत्ति पर क्लिक करें** (p1), या सटीक निर्देशांक के लिए `X,Y` टाइप करें और **Enter** दबाएं।
3. **दूसरी एक्सटेंशन लाइन उत्पत्ति पर क्लिक करें** (p2)। यहाँ भी निर्देशांक प्रविष्टि काम करती है।
4. माप रेखा की स्थिति निर्धारित करने के लिए **कर्सर मूव करें**। अभिमुखता कर्सर स्थिति से स्वचालित रूप से पता चलती है।
5. **क्लिक करें** रखने के लिए, या सटीक प्लेसमेंट के लिए ऑफसेट दूरी टाइप करें और **Enter** दबाएं।

## स्वचालित अभिमुखता पहचान

जब कोई अभिमुखता बाध्य नहीं है, कमांड दो मापे गए बिंदुओं के सापेक्ष कर्सर स्थिति पढ़ती है:

| कर्सर स्थिति | पहचानी गई अभिमुखता | क्या मापा जाता है |
|-----------------|---------------------|-----------------|
| बिंदुओं के ऊपर या नीचे | क्षैतिज | p1 और p2 के बीच Δ X |
| बिंदुओं के बाएं या दाएं | ऊर्ध्वाधर | p1 और p2 के बीच Δ Y |

प्लेसमेंट चरण के दौरान किसी भी समय क्षैतिज लॉक करने के लिए **H** या ऊर्ध्वाधर के लिए **V** दबाएं। एक बार लॉक होने पर, कर्सर मूव करने से अभिमुखता नहीं बदलती।

## टाइप की गई ऑफसेट दूरी

प्लेसमेंट के दौरान एक संख्या टाइप करके माप रेखा को मापे गए बिंदुओं से सटीक दूरी पर रखें:

| Key | Action |
|-----|--------|
| `0`–`9`, `.` | ऑफसेट दूरी में अंक जोड़ें |
| `Backspace` | अंतिम टाइप किया गया अक्षर हटाएं |
| `Enter` / `Space` | टाइप की गई दूरी पर रखें |

कर्सर की साइड (क्षैतिज के लिए ऊपर/नीचे, ऊर्ध्वाधर के लिए बाएं/दाएं) चिह्न निर्धारित करती है — माप रेखा उस साइड पर दिखाई देती है जिस पर वर्तमान में कर्सर है।

## कीबोर्ड संदर्भ

| Key | Action |
|-----|--------|
| `0`–`9`, `.`, `-` | X निर्देशांक प्रविष्टि शुरू करें (p1/p2 चरण), या ऑफसेट दूरी (प्लेसमेंट चरण) |
| `,` | X लॉक करें और Y प्रविष्टि पर जाएं (p1/p2 चरण) |
| `H` | क्षैतिज अभिमुखता लॉक करें (केवल प्लेसमेंट चरण) |
| `V` | ऊर्ध्वाधर अभिमुखता लॉक करें (केवल प्लेसमेंट चरण) |
| `Backspace` | अंतिम टाइप किया गया अक्षर हटाएं |
| `Enter` / `Space` | टाइप किए गए निर्देशांक या ऑफसेट की पुष्टि करें |
| `Escape` | रद्द करें |

## Dimension Linear बनाम Dimension Aligned

| | Dimension Linear | Dimension Aligned |
|---|-----------------|------------------|
| अक्ष | हमेशा H या V | मापी गई रेखा के समानांतर |
| मापता है | केवल X या Y घटक | सही यूक्लिडियन दूरी |
| H/V कुंजियाँ | हाँ — अभिमुखता लॉक करें | नहीं — हमेशा p1→p2 का अनुसरण करती है |
| सबसे अच्छा | लंबवत लेआउट, फ्लोर प्लान | विकर्ण फीचर, कोणीय कट |

## लेबल संपादित करना — सरल मोड

रखी गई रेखीय माप पर **डबल-क्लिक** करें **सरल** मोड में टेक्स्ट एडिटर खोलने के लिए। एडिटर में वर्तमान रेंडर की गई वैल्यू पहले से भरी होती है ताकि आप कर्सर की स्थिति निर्धारित करके सीधे संपादित कर सकें।

| विशेषता | व्यवहार |
|---------|-----------|
| Bold / Italic / Font / Height | **पूरे** लेबल पर एक साथ लागू होती है |
| प्रति-अक्षर फ़ॉर्मेटिंग | समर्थित नहीं |
| `Enter` | वैल्यू सहेजता है और एडिटर बंद करता है |
| बहु-पंक्ति | समर्थित नहीं |

पूर्ण संदर्भ के लिए [Text Editor — simple mode](../interface/text-editor#simple-mode) देखें।

## माप चेन करना

इस माप की अंतिम एक्सटेंशन लाइन से जारी रखते हुए अधिक माप जोड़ने के लिए, इसे रखने के तुरंत बाद [Dimension Continue](./dim-continue) का उपयोग करें।

## DXF — DIMENSION ऑब्जेक्ट

रेखीय माप `DIMENSION` ऑब्जेक्ट के रूप में सहेजे जाते हैं, `rotationDeg` के साथ `0` (क्षैतिज) या `90` (ऊर्ध्वाधर) पर सेट। एक्सटेंशन लाइन उत्पत्ति, माप रेखा स्थिति, टेक्स्ट स्थिति, मापा गया मान, तीर स्टाइल, टेक्स्ट ऊँचाई, और सभी प्रदर्शन फ्लैग बिना किसी नुकसान के राउंड-ट्रिप करते हैं।
