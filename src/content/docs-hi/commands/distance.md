# Distance

`distance` कमांड दो क्लिक किए गए बिंदुओं के बीच सीधी-रेखा (यूक्लिडियन) दूरी मापती है और टर्मिनल में 4 दशमलव स्थानों तक परिणाम प्रिंट करती है। यह तीन माप कमांड में से एक है — [Angle](../angle/) एक शीर्ष पर कोणीय उद्घाटन मापता है, और [Area](../area/) एक बहुभुज का क्षेत्रफल और परिधि मापता है।

## दूरी माप की संरचना

```
  ● पहला बिंदु
   \
    \  प्रीव्यू रेखा (live)
     \
      ● दूसरा बिंदु    →  terminal: "Distance: 12.3456"
```

- **पहला बिंदु** — माप का उद्गम।
- **दूसरा बिंदु** — एंडपॉइंट; रखने पर तुरंत परिणाम प्रिंट होता है।
- **परिणाम** — टर्मिनल में दिखाया जाता है, कैनवास पर नहीं।

## दूरी मापना

1. टर्मिनल में `distance` टाइप करें या **Distance** टूलबार बटन पर क्लिक करें।
2. **पहले बिंदु पर क्लिक करें**, या सटीक निर्देशांक के लिए `X,Y` टाइप करें और **Enter** दबाएं।
3. **दूसरे बिंदु पर क्लिक करें** — मापी गई दूरी टर्मिनल में दिखाई देती है। यहाँ भी निर्देशांक प्रविष्टि काम करती है।
4. **फिर क्लिक करें** (वैकल्पिक) नई माप शुरू करने के लिए। कमांड सक्रिय रहती है।

किसी भी समय चरण 2 पर रीसेट करने के लिए `Escape` दबाएं।

## माप श्रृंखला

परिणाम दिखाने के बाद, क्लिक करने पर तुरंत अगली माप शुरू होती है — क्लिक किया गया बिंदु नया पहला बिंदु बन जाता है। इससे आप कमांड को पुनः सक्रिय किए बिना दूरियों की श्रृंखला माप सकते हैं।

## Distance बनाम Angle

| | Distance | Angle |
|---|---------|-------|
| क्या मापता है | सीधी-रेखा लंबाई | एक शीर्ष पर आंतरिक कोण |
| क्लिक की संख्या | 2 | 3 |
| परिणाम प्रारूप | `12.3456` (यूनिट) | `45.0000°` |
| कैनवास प्रीव्यू | पहले बिंदु से कर्सर तक रेखा | शीर्ष से कर्सर तक दो रेखाएं |
| सबसे अच्छा | किसी अंतराल या खंड की लंबाई के लिए | दो फीचर के बीच उद्घाटन कोण के लिए |

## निर्देशांक प्रविष्टि

क्लिक करने के बजाय किसी भी बिंदु के लिए सटीक स्थिति टाइप करें:

1. X मान टाइप करें।
2. `,` दबाएं — टर्मिनल `[X], [Y{cursor}]` दिखाता है।
3. Y मान टाइप करें।
4. पुष्टि करने के लिए **Enter** दबाएं।

## कीबोर्ड संदर्भ

| Key | Action |
|-----|--------|
| `0`–`9`, `.`, `-` | X निर्देशांक प्रविष्टि शुरू करें |
| `,` | X लॉक करें और Y प्रविष्टि पर जाएं |
| `Backspace` | अंतिम टाइप किया गया अक्षर हटाएं |
| `Enter` | टाइप किए गए निर्देशांक की पुष्टि करें |
| `Escape` | रद्द करें और चरण 2 पर रीसेट करें |

## नोट्स

- परिणाम **केवल टर्मिनल में** दिखाए जाते हैं — ड्राइंग में कुछ नहीं जोड़ा जाता।
- परिणाम ड्राइंग निर्देशांक के समान इकाइयों में व्यक्त होता है (कोई इकाई रूपांतरण नहीं)।
- सटीकता हमेशा 4 दशमलव स्थान होती है।
