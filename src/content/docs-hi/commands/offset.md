# Offset

`offset` कमांड एक निश्चित लंबवत दूरी पर एक ऑब्जेक्ट की समानांतर कॉपी बनाती है। आप दूरी एक बार टाइप करते हैं, फिर ऑब्जेक्ट क्लिक करें और एक साइड चुनें — कमांड उसी दूरी पर तैयार रहती है इसलिए आप एक सत्र में कई ऑब्जेक्ट ऑफसेट कर सकते हैं।

समर्थित ऑब्जेक्ट प्रकार: **Line, Circle, Arc, Ellipse, Polyline** (Rectangles सहित)।

## ऑफसेट का उपयोग

1. टर्मिनल में `offset` टाइप करें या **Offset** टूलबार बटन पर क्लिक करें।
2. **ऑफसेट दूरी टाइप करें** और **Enter** या **Space** दबाएं।
3. ऑफसेट करने के लिए **एक ऑब्जेक्ट पर क्लिक करें** — यदि ऑब्जेक्ट समर्थित प्रकार का नहीं है, तो एक त्रुटि संदेश दिखाई देता है और आप एक अलग ऑब्जेक्ट पर क्लिक कर सकते हैं।
4. **कर्सर को उस साइड पर ले जाएं** जहाँ कॉपी दिखनी चाहिए — एक लाइव प्रीव्यू अनुसरण करता है।
5. ऑफसेट कॉपी रखने के लिए **क्लिक करें**।

प्रत्येक प्लेसमेंट के बाद कमांड **उसी दूरी** पर चरण 3 पर वापस आती है, अगले ऑफसेट के लिए तैयार। दूरी-प्रविष्टि चरण पर वापस रीसेट करने के लिए **Escape** दबाएं।

```
  Distance: 10

  ─────────────────    ← original line
  ─────────────────    ← offset copy (10 units below)
```

## प्रति-ऑब्जेक्ट ऑफसेट व्यवहार

| ऑब्जेक्ट | ऑफसेट की गणना कैसे होती है |
|--------|---------------------------|
| **Line** | मूल दिशा के लंबवत शिफ्ट की गई समानांतर रेखा |
| **Circle** | केंद्रित वृत्त; बाहर क्लिक करें → बड़ी त्रिज्या, अंदर → छोटी त्रिज्या |
| **Arc** | नई त्रिज्या पर केंद्रित चाप; समान कोणीय विस्तार संरक्षित |
| **Ellipse** | दोनों अर्ध-अक्ष समान दूरी से बढ़ाए या घटाए जाते हैं |
| **Polyline** | प्रत्येक खंड स्वतंत्र रूप से ऑफसेट; आसन्न ऑफसेट खंड कोनों पर मिटर्ड |

**Circle**, **Arc**, और **Ellipse** के लिए: यदि अंदरी ऑफसेट किसी भी त्रिज्या या अर्ध-अक्ष को शून्य या उससे नीचे कर देता है, तो ऑफसेट लागू नहीं होता।

## कीबोर्ड संदर्भ

| Key | Action |
|-----|--------|
| `0`–`9`, `.` | दूरी मान में अंक जोड़ें |
| `Backspace` | अंतिम टाइप किया गया अक्षर हटाएं |
| `Enter` / `Space` | टाइप की गई दूरी की पुष्टि करें और ऑब्जेक्ट चयन पर आगे बढ़ें |
| `Escape` | दूरी-प्रविष्टि चरण पर रीसेट करें |

## वर्कफ़्लो नोट

दूरी **Escape** दबाने तक सेट रहती है। यह एक ही स्पेसिंग पर कई ऑब्जेक्ट ऑफसेट करना कुशल बनाता है — दूरी एक बार टाइप करें, फिर क्रम में प्रत्येक ऑब्जेक्ट के लिए क्लिक करें और साइड चुनें।
