# Dimension Aligned

`dimaligned` कमांड एक माप रखती है जो दो बिंदुओं के बीच **सही सीधी-रेखा दूरी** मापती है। माप रेखा दो बिंदुओं को जोड़ने वाली रेखा के समानांतर चलती है, इसलिए यह किसी भी कोण पर हो सकती है। यह [Dimension Linear](../dim-linear/) से मुख्य अंतर है, जो क्षैतिज या ऊर्ध्वाधर तक सीमित है।

## संरेखित माप की संरचना

```
     ●  p2
    /|
   / |  (ext line 2, perpendicular to dim line)
  /  |
 /←5.00→/
/  /
●  /  (ext line 1, perpendicular to dim line)
p1
```

- **एक्सटेंशन लाइन** — माप रेखा के लंबवत, प्रत्येक मापे गए बिंदु से।
- **माप रेखा** — p1→p2 के समानांतर, कर्सर स्थिति से एक साइड पर ऑफसेट।
- **मान** — सही यूक्लिडियन दूरी `|p1 – p2|`।

## संरेखित माप रखना

1. टर्मिनल में `dimaligned` टाइप करें या **Dimension Aligned** टूलबार बटन पर क्लिक करें।
2. **पहली एक्सटेंशन लाइन उत्पत्ति पर क्लिक करें** (p1), या सटीक निर्देशांक के लिए `X,Y` टाइप करें और **Enter** दबाएं।
3. **दूसरी एक्सटेंशन लाइन उत्पत्ति पर क्लिक करें** (p2)। यहाँ भी निर्देशांक प्रविष्टि काम करती है।
4. माप रेखा का लंबवत ऑफसेट सेट करने के लिए **कर्सर को एक साइड पर ले जाएं**।
5. **क्लिक करें** रखने के लिए, या सटीक प्लेसमेंट के लिए ऑफसेट दूरी टाइप करें और **Enter** दबाएं।

## टाइप की गई ऑफसेट दूरी

p1→p2 रेखा से सटीक लंबवत दूरी पर माप रेखा रखने के लिए प्लेसमेंट के दौरान एक संख्या टाइप करें:

| Key | Action |
|-----|--------|
| `0`–`9`, `.` | ऑफसेट में अंक जोड़ें |
| `Backspace` | अंतिम टाइप किया गया अक्षर हटाएं |
| `Enter` / `Space` | टाइप किए गए ऑफसेट पर रखें |

कर्सर की साइड निर्धारित करती है कि माप रेखा किस साइड दिखाई देती है।

## कीबोर्ड संदर्भ

| Key | Action |
|-----|--------|
| `0`–`9`, `.`, `-` | X निर्देशांक प्रविष्टि शुरू करें (p1/p2 चरण), या ऑफसेट दूरी (प्लेसमेंट चरण) |
| `,` | X लॉक करें और Y प्रविष्टि पर जाएं (p1/p2 चरण) |
| `Backspace` | अंतिम टाइप किया गया अक्षर हटाएं |
| `Enter` / `Space` | टाइप किए गए निर्देशांक या ऑफसेट की पुष्टि करें |
| `Escape` | रद्द करें |

## Dimension Aligned बनाम Dimension Linear

| | Dimension Aligned | Dimension Linear |
|---|------------------|-----------------|
| माप रेखा कोण | p1→p2 के समानांतर — कोई भी कोण | हमेशा क्षैतिज या ऊर्ध्वाधर |
| मापता है | सही यूक्लिडियन दूरी | केवल X या Y घटक |
| H/V अभिमुखता लॉक | नहीं | हाँ — `H` और `V` कुंजियाँ |
| सबसे अच्छा | विकर्ण फीचर, कोणीय कट | लंबवत लेआउट, ग्रिड-संरेखित भाग |

## लेबल संपादित करना — सरल मोड

रखी गई संरेखित माप पर **डबल-क्लिक** करें **सरल** मोड में टेक्स्ट एडिटर खोलने के लिए। एडिटर में वर्तमान रेंडर की गई वैल्यू पहले से भरी होती है ताकि आप कर्सर की स्थिति निर्धारित करके सीधे संपादित कर सकें।

| विशेषता | व्यवहार |
|---------|-----------|
| Bold / Italic / Font / Height | **पूरे** लेबल पर एक साथ लागू होती है |
| प्रति-अक्षर फ़ॉर्मेटिंग | समर्थित नहीं |
| `Enter` | वैल्यू सहेजता है और एडिटर बंद करता है |
| बहु-पंक्ति | समर्थित नहीं |

पूर्ण संदर्भ के लिए [Text Editor — simple mode](../../interface/text-editor/#simple-mode) देखें।

## माप चेन करना

इस संरेखित माप की दूसरी एक्सटेंशन लाइन से जारी रखते हुए अधिक माप जोड़ने के लिए, [Dimension Continue](../dim-continue/) का उपयोग करें — यह इस संरेखित माप के समान माप कोण पर लॉक हो जाती है।

## DXF — DIMENSION ऑब्जेक्ट (संरेखित प्रकार)

संरेखित माप `DIMENSION` ऑब्जेक्ट के रूप में `dimType = 1` (संरेखित) के साथ सहेजे जाते हैं। एक्सटेंशन लाइन उत्पत्ति, माप रेखा स्थिति, टेक्स्ट स्थिति, मापा गया मान, रोटेशन, तीर स्टाइल, और सभी प्रदर्शन फ्लैग बिना किसी नुकसान के राउंड-ट्रिप करते हैं।
