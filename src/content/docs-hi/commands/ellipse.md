# Ellipse

`ellipse` कमांड तीन क्लिक का उपयोग करके दीर्घवृत्त खींचती है: एक केंद्र बिंदु, किसी भी कोण पर पहले (प्रमुख) अर्ध-अक्ष का एंडपॉइंट, और दूसरे (लघु) अर्ध-अक्ष की लंबाई। दो अक्ष हमेशा एक दूसरे के लंबवत होते हैं — दूसरी अक्ष की दिशा पहले से स्वचालित रूप से ली जाती है।

## दीर्घवृत्त बनाना

1. टर्मिनल में `ellipse` टाइप करें या **Ellipse** टूलबार बटन पर क्लिक करें।
2. **केंद्र बिंदु पर क्लिक करें**, या सटीक निर्देशांक के लिए `X,Y` टाइप करें और **Enter** दबाएं।
3. **पहले अक्ष के एंडपॉइंट पर क्लिक करें** — पहले अर्ध-अक्ष की दिशा और लंबाई दोनों सेट करता है। यहाँ भी निर्देशांक प्रविष्टि काम करती है।
4. **दूसरी अक्ष की लंबाई सेट करें** — कर्सर को पहली अक्ष के लंबवत ले जाएं, फिर क्लिक करें या लंबाई टाइप करें।

```
               ● ← first axis endpoint (step 3)
              /
  center ●  /  ← first axis (any angle)
            |
            ● ← cursor here sets second axis length (step 4)
```

चरण 4 के बाद दीर्घवृत्त रखा जाता है और कमांड बाहर निकल जाती है।

## अक्ष इनपुट — क्लिक, निर्देशांक, या टाइप की गई लंबाई

**केंद्र (चरण 2):** क्लिक करें या सटीक स्थिति के लिए `X,Y` टाइप करें।

**पहला अक्ष एंडपॉइंट (चरण 3):** क्लिक करें, या सटीक निर्देशांक के लिए `X,Y` टाइप करें। कोण लॉकिंग 45° वृद्धि पर भी स्नैप करती है — लॉक होने पर, एक लंबाई टाइप करें और उस सटीक दूरी पर एंडपॉइंट रखने के लिए **Enter** दबाएं।

**दूसरी अक्ष (चरण 4):** टाइप की गई लंबाई हमेशा उपलब्ध है — कोण लॉक आवश्यक नहीं। दिशा पहले से पहली अक्ष के लंबवत तय है; टाइप करने से केवल लंबाई सेट होती है।

| Key | Action |
|-----|--------|
| `0`–`9`, `.` | अक्ष लंबाई में अंक जोड़ें (दूसरी अक्ष चरण) |
| `Backspace` | अंतिम टाइप किया गया अक्षर हटाएं |
| `Enter` | टाइप की गई लंबाई पर अक्ष एंडपॉइंट रखें |

## निर्देशांक प्रविष्टि (केंद्र और पहला अक्ष एंडपॉइंट)

1. X मान टाइप करें।
2. `,` दबाएं — टर्मिनल `[X], [Y{cursor}]` दिखाता है।
3. Y मान टाइप करें।
4. पुष्टि करने के लिए **Enter** दबाएं।

## कीबोर्ड संदर्भ

| Key | Action |
|-----|--------|
| `0`–`9`, `.`, `-` | X निर्देशांक प्रविष्टि शुरू करें (केंद्र/पहला-अक्ष चरण), या कोण-लॉक होने पर अक्ष लंबाई |
| `,` | X लॉक करें और Y प्रविष्टि पर जाएं |
| `Backspace` | अंतिम टाइप किया गया अक्षर हटाएं |
| `Enter` | टाइप किए गए निर्देशांक या लंबाई की पुष्टि करें |
| `Escape` | रद्द करें और रीसेट करें |

## ग्रिप संपादन — स्वतंत्र अक्ष आकार बदलना

एक चयनित दीर्घवृत्त में पाँच ग्रिप होते हैं:

| ग्रिप | गिनती | क्या करता है |
|------|-------|--------------|
| **Center** | 1 | पूरे दीर्घवृत्त को मूव करता है; दोनों अक्ष अपरिवर्तित |
| **Major axis endpoints** | 2 (लंबी अक्ष के विपरीत सिरे) | प्रमुख अर्ध-अक्ष की लंबाई बदलने के लिए ड्रैग करें; लघु अक्ष का निरपेक्ष आकार स्थिर रहता है |
| **Minor axis endpoints** | 2 (छोटी अक्ष के विपरीत सिरे) | लघु अर्ध-अक्ष की लंबाई बदलने के लिए ड्रैग करें; प्रमुख अक्ष अपरिवर्तित |

प्रमुख और लघु अक्ष ग्रिप स्वतंत्र हैं — आप कमांड दोबारा चलाए बिना दीर्घवृत्त को फिर से आकार दे सकते हैं।

## दीर्घवृत्त चुनना

| विधि | व्यवहार |
|--------|-----------|
| **क्लिक** | यदि क्लिक दीर्घवृत्त आउटलाइन के पास है तो चुनता है |
| **दाईं ओर ड्रैग** (सख्त) | दीर्घवृत्त का अक्ष-संरेखित बाउंडिंग बॉक्स पूरी तरह चयन बॉक्स के अंदर होना चाहिए |
| **बाईं ओर ड्रैग** (क्रॉसिंग) | दीर्घवृत्त आउटलाइन का कोई भी भाग जो चयन बॉक्स की सीमा पार करे उसे चुनता है |

## गुण

**सामान्य**

| गुण | डिफ़ॉल्ट | अर्थ |
|----------|---------|---------|
| Color | 256 (ByLayer) | ACI रंग सूचकांक |
| Layer | `0` | लेयर असाइनमेंट |
| Linetype | ByLayer | नामित लाइनटाइप पैटर्न |
| Linetype Scale | 1 | लाइनटाइप पैटर्न पर स्केल फैक्टर |
| Thickness | 0 | एक्सट्रूज़न मोटाई |

**ज्यामिति**

| गुण | अर्थ |
|----------|---------|
| Center X / Center Y | दीर्घवृत्त का केंद्र |
| Major Axis X / Major Axis Y | केंद्र से प्रमुख अक्ष एंडपॉइंट तक वेक्टर (दिशा और लंबाई दोनों एन्कोड करता है) |
| Axis Ratio | लघु अर्ध-अक्ष से प्रमुख अर्ध-अक्ष का अनुपात (0 < अनुपात ≤ 1) |
| Start Angle / End Angle | डिग्री में पैरामीट्रिक कोण; पूर्ण दीर्घवृत्त के लिए दोनों 0°/360° |

## Ellipse बनाम Circle — कब किसका उपयोग करें

| | Ellipse | Circle |
|---|---------|--------|
| अक्ष | किसी भी कोण पर दो स्वतंत्र अर्ध-अक्ष | एक त्रिज्या, सममित |
| घुमाव | किसी भी कोण पर रखा जा सकता है | कोई घुमाव नहीं |
| टाइप किया गया इनपुट | प्रति अक्ष लंबाई | केवल त्रिज्या |
| ग्रिप से आकार बदलें | प्रमुख और लघु स्वतंत्र रूप से | सभी चार कार्डिनल बिंदु समान रूप से |
| सबसे अच्छा | तिरछे दृश्य, अंडाकार फीचर, परिप्रेक्ष्य छेद | सममित गोल फीचर |

## DXF — ELLIPSE ऑब्जेक्ट

दीर्घवृत्त DXF फ़ाइल में `ELLIPSE` ऑब्जेक्ट के रूप में सहेजे जाते हैं। प्रारूप केंद्र बिंदु, पूर्ण प्रमुख अक्ष वेक्टर (दिशा + लंबाई), और अक्ष अनुपात संग्रहीत करता है। घुमाव, आकार और सभी स्टाइल गुण बिना किसी नुकसान के राउंड-ट्रिप करते हैं। एक वृत्त पतित दीर्घवृत्त के रूप में संग्रहीत **नहीं** होता — दो ऑब्जेक्ट प्रकार DXF मॉडल में अलग रहते हैं।
