# Dimension Continue

`dimcontinue` कमांड मौजूदा माप की **दूसरी एक्सटेंशन लाइन** से नए माप जोड़ती है। प्रत्येक नया खंड बेस के समान माप अक्ष और माप-रेखा ऑफसेट पर रखा जाता है। सभी स्टाइल गुण — तीर आकार, टेक्स्ट ऊँचाई, एक्सटेंशन लाइन लंबाई — बेस से स्वचालित रूप से कॉपी किए जाते हैं।

## चेन किए गए माप कैसे दिखते हैं

```
  |←— 3.00 —→|←— 2.50 —→|←— 4.00 —→|
  |           |           |           |
  ●           ●           ●           ●
  p1        p2 (base    p3           p4
           ext2 → new start)
```

प्रत्येक आयत एक अलग `DIMENSION` ऑब्जेक्ट है। वे समान माप-रेखा स्थिति और माप दिशा साझा करते हैं।

## श्रृंखला शुरू करना

1. टर्मिनल में `dimcontinue` टाइप करें या **Dimension Continue** टूलबार बटन पर क्लिक करें।
2. **यदि हाल ही में माप रखा गया था** — कमांड इसे स्वचालित रूप से बेस के रूप में उठाती है (कोई क्लिक आवश्यक नहीं)।
3. **यदि कोई हालिया माप नहीं है** — किसी मौजूदा माप को बेस के रूप में उपयोग करने के लिए क्लिक करें।
4. **अगली एक्सटेंशन लाइन उत्पत्ति पर क्लिक करें** — प्रीव्यू कर्सर मूव करने पर नया माप दिखाता है। या `X,Y` टाइप करें और **Enter** दबाएं सटीक निर्देशांक के लिए।
5. श्रृंखला एक्सटेंड करने के लिए क्लिक (या टाइप) करते रहें। प्रत्येक रखा गया माप स्वचालित रूप से नया बेस बन जाता है।
6. श्रृंखला समाप्त करने के लिए **Enter**, **Space**, या **Escape** दबाएं।

## बेस माप से क्या विरासत मिलती है

| गुण | बेस से विरासत |
|----------|---------------------|
| माप दिशा / कोण | हाँ — पूरी श्रृंखला के लिए लॉक |
| माप रेखा ऑफसेट (मापे गए बिंदुओं से दूरी) | हाँ |
| तीर आकार | हाँ |
| टेक्स्ट ऊँचाई | हाँ |
| एक्सटेंशन लाइन ऑफसेट और एक्सटेंशन | हाँ |
| टेक्स्ट संरेखण | हाँ |
| स्टाइल नाम | हाँ |
| Color, Layer | विरासत नहीं — वर्तमान लेयर का उपयोग |

## माप दिशा लॉकिंग

श्रृंखला की माप दिशा **बेस माप के कोण पर तय** है:

- रेखीय बेस (H) → सभी जारी माप क्षैतिज दूरी (Δ X) मापते हैं।
- रेखीय बेस (V) → सभी जारी माप ऊर्ध्वाधर दूरी (Δ Y) मापते हैं।
- किसी भी कोण पर संरेखित बेस → सभी जारी माप उसी कोण के साथ मापते हैं।

आप श्रृंखला के बीच में दिशा नहीं बदल सकते। अलग दिशा में माप के लिए एक नई [Dimension Linear](../dim-linear/) या [Dimension Aligned](../dim-aligned/) शुरू करें।

## कीबोर्ड संदर्भ

| Key | Action |
|-----|--------|
| `0`–`9`, `.`, `-` | X निर्देशांक प्रविष्टि शुरू करें |
| `,` | X लॉक करें और Y प्रविष्टि पर जाएं |
| `Backspace` | अंतिम टाइप किया गया अक्षर हटाएं |
| `Enter` | टाइप किए गए निर्देशांक की पुष्टि करें, या यदि कोई इनपुट प्रगति में नहीं है तो श्रृंखला समाप्त करें |
| `Space` / `Escape` | श्रृंखला समाप्त करें |

## Dimension Continue बनाम नया शुरू करना

| | Dimension Continue | Dimension Linear / Aligned |
|---|-------------------|--------------------------|
| प्रारंभिक बिंदु | अंतिम बेस की ext2 पर तय | कहीं भी क्लिक करें |
| कोण | बेस पर लॉक | मुक्त |
| ऑफसेट | बेस से विरासत | कर्सर या टाइप से सेट |
| स्टाइल | बेस से विरासत | वर्तमान स्टाइल |
| सबसे अच्छा | एक पंक्ति के साथ संचयी माप | पहला माप या दिशा बदलना |

## प्लेसमेंट के बाद लेबल संपादित करना — सरल मोड

श्रृंखला में किसी भी माप पर **डबल-क्लिक** करें **सरल** मोड में टेक्स्ट एडिटर खोलने के लिए। प्रत्येक खंड स्वतंत्र है और व्यक्तिगत रूप से संपादित किया जा सकता है।

| विशेषता | व्यवहार |
|---------|-----------|
| Bold / Italic / Font / Height | **पूरे** लेबल पर एक साथ लागू होती है |
| प्रति-अक्षर फ़ॉर्मेटिंग | समर्थित नहीं |
| `Enter` | वैल्यू सहेजता है और एडिटर बंद करता है |
| बहु-पंक्ति | समर्थित नहीं |

पूर्ण संदर्भ के लिए [Text Editor — simple mode](../../interface/text-editor/#simple-mode) देखें।

## DXF — DIMENSION ऑब्जेक्ट

श्रृंखला में प्रत्येक खंड DXF फ़ाइल में एक स्वतंत्र `DIMENSION` ऑब्जेक्ट के रूप में संग्रहीत होता है। वे फ़ाइल में जुड़े नहीं हैं — वे गुण साझा करते हैं क्योंकि वे एक ही बेस से बनाए गए थे, लेकिन प्रत्येक को रखने के बाद व्यक्तिगत रूप से संपादित किया जा सकता है।
