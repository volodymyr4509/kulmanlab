# Copy

`copy` कमांड चयनित ऑब्जेक्ट के अनुवादित डुप्लिकेट बनाती है और उन्हें आधार बिंदु से गंतव्य तक ऑफसेट करती है — मूल ठीक वहीं रहते हैं। यह [Move](./move) से एक मुख्य अंतर है: Copy ड्राइंग में नए ऑब्जेक्ट जोड़ता है; Move मौजूदा को स्थानांतरित करता है।

## शुरू करने के दो तरीके

**पहले चुनें, फिर कॉपी करें** — पहले ऑब्जेक्ट चुनें, फिर सक्रिय करें:

1. कैनवास पर एक या अधिक ऑब्जेक्ट चुनें।
2. टर्मिनल में `copy` टाइप करें या **Copy** टूलबार बटन पर क्लिक करें।
3. **आधार बिंदु पर क्लिक करें**, या सटीक निर्देशांक के लिए `X,Y` टाइप करें और **Enter** दबाएं।
4. **गंतव्य पर क्लिक करें** — डुप्लिकेट आधार→गंतव्य ऑफसेट पर दिखाई देते हैं। यहाँ भी निर्देशांक प्रविष्टि काम करती है।

**पहले सक्रिय करें, फिर चुनें** — कुछ भी चुने बिना कमांड शुरू करें:

1. `copy` टाइप करें या टूलबार बटन पर क्लिक करें।
2. **ऑब्जेक्ट चुनें** — व्यक्तिगत ऑब्जेक्ट टॉगल करने के लिए क्लिक करें, या क्षेत्र से चुनने के लिए ड्रैग करें।
3. चयन की पुष्टि करने के लिए **Enter** या **Space** दबाएं।
4. **आधार बिंदु पर क्लिक करें**, फिर **गंतव्य पर क्लिक करें** (दोनों चरणों पर निर्देशांक प्रविष्टि उपलब्ध)।

```
  Before:               After:
  [entity A]            [entity A]  ← originals untouched
  [entity B]            [entity B]
                        [copy of A] ← new entities
                        [copy of B]
```

कॉपी का एक घोस्ट प्रीव्यू आधार बिंदु से गंतव्य तक कर्सर का अनुसरण करता है।

## निर्देशांक प्रविष्टि

आधार-बिंदु या गंतव्य चरण पर, क्लिक करने के बजाय सटीक स्थिति टाइप करें:

1. X मान टाइप करें।
2. `,` दबाएं — टर्मिनल `[X], [Y{cursor}]` दिखाता है।
3. Y मान टाइप करें।
4. पुष्टि करने के लिए **Enter** दबाएं।

## कोण लॉकिंग और सटीक दूरी

आधार बिंदु सेट होने के बाद कमांड 45° अक्षों (0°, 45°, 90°, 135°, …) पर स्नैप करती है जब कर्सर काफी दूर हो और अक्ष के पास हो। लॉक होने पर, एक दूरी टाइप करें और ठीक उस ऑफसेट पर कॉपी रखने के लिए **Enter** दबाएं।

| Key | Action |
|-----|--------|
| `0`–`9`, `.` | दूरी मान में जोड़ें |
| `-` | नकारात्मक दूरी — अक्ष के साथ दिशा उलटता है (केवल पहला अक्षर) |
| `Backspace` | अंतिम टाइप किया गया अक्षर हटाएं |
| `Enter` | टाइप की गई दूरी पर कॉपी रखें |

## कीबोर्ड संदर्भ

| Key | Action |
|-----|--------|
| `Enter` / `Space` | चयन की पुष्टि करें और आधार-बिंदु चरण पर आगे बढ़ें |
| `0`–`9`, `.`, `-` | X निर्देशांक प्रविष्टि शुरू करें, या कोण-लॉक होने पर दूरी |
| `,` | X लॉक करें और Y प्रविष्टि पर जाएं |
| `Backspace` | अंतिम टाइप किया गया अक्षर हटाएं |
| `Enter` | निर्देशांक की पुष्टि करें या टाइप की गई दूरी पर कॉपी लागू करें |
| `Escape` | रद्द करें और रीसेट करें |

## कमांड के दौरान चयन

| विधि | व्यवहार |
|--------|-----------|
| **क्लिक** | कर्सर के नीचे ऑब्जेक्ट को चयन में टॉगल करता है |
| **दाईं ओर ड्रैग** (सख्त) | बॉक्स के पूरी तरह अंदर ऑब्जेक्ट जोड़ता है |
| **बाईं ओर ड्रैग** (क्रॉसिंग) | बॉक्स की सीमा को काटने वाले ऑब्जेक्ट जोड़ता है |
| **Enter** / **Space** | चयन की पुष्टि करता है |

## कॉपी के बाद

**मूल चयनित रहते हैं** — नई कॉपी ड्राइंग में जोड़ी जाती हैं लेकिन चयन साफ़ हो जाता है और कमांड बाहर निकल जाती है। कॉपी के साथ तुरंत काम करने के लिए, चयन पर फिर से Copy चलाएं, या एक नई कमांड शुरू करें।

## Copy बनाम Move

| | Copy | Move |
|---|------|------|
| मूल | अपनी जगह पर रहते हैं | मूल स्थिति से हटाए जाते हैं |
| परिणाम गिनती | कॉपी किए गए ऑब्जेक्ट की संख्या से बढ़ता है | अपरिवर्तित |
| ऑपरेशन के बाद | मूल अभी भी चयनित | नई स्थिति पर मूव किए गए ऑब्जेक्ट चयनित |
| सबसे अच्छा | ज्यामिति दोहराना, सममित लेआउट | ज्यामिति पुनःस्थित करना |

## समर्थित ऑब्जेक्ट

Copy हर ऑब्जेक्ट प्रकार पर काम करता है। सभी ऑब्जेक्ट आंतरिक रूप से `translate(dx, dy)` लागू करते हैं इसलिए कोई भी बाहर नहीं है।
