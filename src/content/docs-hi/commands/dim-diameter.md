# Dimension Diameter

`dimdiameter` कमांड एक चाप या वृत्त पर व्यास माप रखती है। माप रेखा पूर्ण व्यास तक फैली है — केंद्र से होकर दो व्यासत: विपरीत चाप बिंदुओं के बीच — और `⌀ <मान>` से लेबल। केंद्र से एक किनारे तक केवल त्रिज्या एनोटेट करने के लिए, [Dimension Radius](../dim-radius/) का उपयोग करें।

## व्यास माप की संरचना

```
  ●──────────── ⌀ 10.00 ────────────●
  (far arc point)              (near arc point / text side)
```

- **माप रेखा** — पूर्ण व्यास तक फैली, दोनों चाप-प्रतिच्छेदन बिंदुओं पर तीर के साथ।
- **निकट चाप बिंदु** — कर्सर साइड पर परिधि बिंदु (जहाँ टेक्स्ट लेबल है)।
- **दूर चाप बिंदु** — व्यासत: विपरीत बिंदु।
- **लेबल** — `⌀` के बाद व्यास मान।

## व्यास माप रखना

1. टर्मिनल में `dimdiameter` टाइप करें या **Dimension Diameter** टूलबार बटन पर क्लिक करें।
2. इसे चुनने के लिए **एक चाप या वृत्त पर क्लिक करें**।
3. माप रेखा को वांछित कोण पर घुमाने के लिए **कर्सर मूव करें**।
4. माप रखने के लिए **क्लिक करें**।

केवल **Arc** और **Circle** ऑब्जेक्ट चुने जा सकते हैं।

| Key | Action |

## Dimension Diameter बनाम Dimension Radius

| | Dimension Diameter | Dimension Radius |
|---|-------------------|-----------------|
| मापता है | पूर्ण व्यास (2 × त्रिज्या) | त्रिज्या (केंद्र से किनारे तक) |
| माप रेखा | किनारे → केंद्र से होकर किनारे | केंद्र → किनारे |
| लेबल उपसर्ग | `⌀` | `R` |
| तीर | दो (दोनों चाप बिंदुओं पर) | एक (चाप बिंदु पर) |
| सबसे अच्छा | पूर्ण वृत्ताकार छेद या शाफ्ट माप | एक घुमाव फीचर की एक साइड एनोटेट करना |

## लेबल संपादित करना — सरल मोड

रखी गई व्यास माप पर **डबल-क्लिक** करें **सरल** मोड में टेक्स्ट एडिटर खोलने के लिए। एडिटर में वर्तमान रेंडर की गई वैल्यू (जैसे `⌀ 10.00`) पहले से भरी होती है ताकि आप कर्सर की स्थिति निर्धारित करके सीधे संपादित कर सकें।

| विशेषता | व्यवहार |
|---------|-----------|
| Bold / Italic / Font / Height | **पूरे** लेबल पर एक साथ लागू होती है |
| प्रति-अक्षर फ़ॉर्मेटिंग | समर्थित नहीं |
| `Enter` | वैल्यू सहेजता है और एडिटर बंद करता है |
| बहु-पंक्ति | समर्थित नहीं |

पूर्ण संदर्भ के लिए [Text Editor — simple mode](../../interface/text-editor/#simple-mode) देखें।

## DXF — DIMENSION व्यास ऑब्जेक्ट

व्यास माप व्यास-प्रकार ज्यामिति के साथ `DIMENSION` ऑब्जेक्ट के रूप में सहेजे जाते हैं, दोनों चाप-बिंदु स्थिति और मापी गई व्यास मान (2 × त्रिज्या) संग्रहीत करते हैं। सभी गुण बिना किसी नुकसान के राउंड-ट्रिप करते हैं।
