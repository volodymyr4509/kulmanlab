# Leader−

`leader-` कमांड मौजूदा मल्टीलीडर से एक तीर भुजा हटाती है। टेक्स्ट लेबल, डॉगलेग और सभी बाकी भुजाएं सुरक्षित रहती हैं — केवल चयनित भुजा हटाई जाती है। केवल एक भुजा वाले मल्टीलीडर से भुजा नहीं हटाई जा सकती।

## भुजा हटाना

1. टर्मिनल में `leader-` टाइप करें।
2. दो या अधिक भुजाओं वाले **मल्टीलीडर पर क्लिक करें**। यदि क्लिक किए गए लीडर में केवल एक भुजा है, तो टर्मिनल त्रुटि दिखाता है और वैध चयन की प्रतीक्षा करता है।
3. **जिस भुजा को हटाना है उसके पास कर्सर ले जाएं** — सबसे पास वाली भुजा एक मार्कर से हाइलाइट होती है।
4. उस भुजा को हटाने के लिए **क्लिक करें**।

भुजा हटा दी जाती है और कमांड सक्रिय रहती है — आप तुरंत और भुजाएं हटाने के लिए किसी अन्य लीडर पर (या उसी पर) क्लिक कर सकते हैं। समाप्त करने के लिए **Enter**, **Space**, या **Escape** दबाएं।

```
  Before:                   After:
  ◄── arm 1                 ◄── arm 1
       \                          \
        ●──── dogleg ──── text     ●──── dogleg ──── text
       /
  arm 2 ──►  ← this arm removed
```

## सबसे पास वाली भुजा कैसे निर्धारित होती है

कमांड प्रत्येक भुजा के रेखा खंडों (डॉगलेग से अंतिम भुजा बिंदु तक के खंड सहित) से कर्सर की लंबवत दूरी मापती है। सबसे कम दूरी वाली भुजा हाइलाइट होती है और क्लिक पर हटाई जाएगी।

## कीबोर्ड संदर्भ

| Key | Action |
|-----|--------|
| `Enter` / `Space` | भुजाएं हटाना समाप्त करें |
| `Escape` | रद्द करें और रीसेट करें |

## नोट्स

- **केवल एक भुजा** वाला लीडर सुरक्षित है — पहले एक भुजा जोड़नी होगी भुजा हटाने से पहले।
- डॉगलेग स्थिति और टेक्स्ट सामग्री हमेशा सुरक्षित रहती है चाहे कोई भी भुजा हटाई जाए।
