# Spline CV

`splinecv` कमांड कंट्रोल वर्टेक्स (CVs) रखकर एक **क्यूबिक B-स्प्लाइन** खींचती है। परिणामी वक्र प्रत्येक CV की ओर आकर्षित होता है लेकिन उनसे होकर नहीं गुजरता — सिवाय पहले और आखिरी वर्टेक्स के, जहाँ **clamped knots** वक्र को ठीक वहाँ एंकर करते हैं। यह आपको सहज आकार नियंत्रण देता है: एक वर्टेक्स खींचें और वक्र उसकी ओर झुके बिना हर बिंदु को छुए।

## कंट्रोल वर्टेक्स से स्प्लाइन बनाना

1. टर्मिनल में `splinecv` टाइप करें या **Spline CV** टूलबार बटन पर क्लिक करें।
2. **कंट्रोल वर्टेक्स रखने के लिए क्लिक करें** — प्रत्येक क्लिक एक वर्टेक्स जोड़ता है। या सटीक निर्देशांक के लिए `X,Y` टाइप करें और **Enter** दबाएं।
3. समाप्त करने के लिए **Enter** दबाएं (कम से कम 2 वर्टेक्स आवश्यक)।

```
  CV ●         ● CV
      \       /
       \     /    ← curve pulled toward CVs
        \   /         but not passing through them
  CV ●   ●   ● CV (start/end: curve touches here)
```

लाइव प्रीव्यू प्रत्येक वर्टेक्स के बाद कर्सर मूव करने पर अपडेट होता है, दिखाता है कि कर्सर स्थिति पर अगले बिंदु के साथ स्प्लाइन कैसी दिखेगी। सभी रखे गए वर्टेक्स हटाने और बाहर निकलने के लिए **Escape** दबाएं।

## निर्देशांक प्रविष्टि

क्लिक करने के बजाय, किसी भी कंट्रोल वर्टेक्स के लिए सटीक स्थिति टाइप करें:

1. X मान टाइप करें।
2. `,` दबाएं — टर्मिनल `[X], [Y{cursor}]` दिखाता है।
3. Y मान टाइप करें।
4. वर्टेक्स रखने के लिए **Enter** दबाएं।

## कीबोर्ड संदर्भ

| Key | Action |
|-----|--------|
| `0`–`9`, `.`, `-` | X निर्देशांक प्रविष्टि शुरू करें |
| `,` | X लॉक करें और Y प्रविष्टि पर जाएं |
| `Backspace` | अंतिम टाइप किया गया अक्षर हटाएं |
| `Enter` | टाइप किए गए निर्देशांक की पुष्टि करें, या यदि कोई इनपुट प्रगति में नहीं है और ≥ 2 वर्टेक्स हैं तो स्प्लाइन समाप्त करें |
| `Escape` | सभी वर्टेक्स हटाएं और बाहर निकलें |

## ग्रिप संपादन — कंट्रोल वर्टेक्स से फिर से आकार देना

एक चयनित CV स्प्लाइन प्रत्येक कंट्रोल वर्टेक्स पर एक ग्रिप दिखाती है:

| ग्रिप | स्थान | क्या करता है |
|------|----------|--------------|
| **Control vertex** | प्रत्येक CV स्थिति पर | उस CV को मूव करने के लिए ड्रैग करें — वक्र नई स्थिति की ओर फिर से आकार लेता है |

"पूरी स्प्लाइन मूव करें" ग्रिप नहीं है। पूरी स्प्लाइन को अनुवादित करने के लिए [Move](../move/) कमांड का उपयोग करें।

## CV स्प्लाइन चुनना

| विधि | व्यवहार |
|--------|-----------|
| **क्लिक** | यदि क्लिक वक्र पर किसी भी बिंदु के पास है तो चुनता है |
| **दाईं ओर ड्रैग** (सख्त) | वक्र के साथ सभी नमूना बिंदु चयन बॉक्स के अंदर होने चाहिए |
| **बाईं ओर ड्रैग** (क्रॉसिंग) | वक्र का कोई भी भाग जो चयन बॉक्स की सीमा पार करे उसे चुनता है |

## समर्थित संपादन कमांड

| कमांड | स्प्लाइन पर क्या होता है |
|---------|---------------------------|
| [Move](../move/) | सभी कंट्रोल वर्टेक्स को समान विस्थापन से अनुवादित करता है |
| [Copy](../copy/) | नई स्थिति पर समान स्प्लाइन बनाता है |
| [Rotate](../rotate/) | सभी CVs को चुने हुए आधार बिंदु के चारों ओर घुमाता है |
| [Mirror](../mirror/) | सभी CVs को मिरर अक्ष पर परावर्तित करता है |
| [Scale](../scale/) | सभी CVs को आधार बिंदु से समान रूप से स्केल करता है |
| [Delete](../delete/) | स्प्लाइन हटाता है |

स्प्लाइन **Offset**, **Trim**, या **Extend** का समर्थन नहीं करती।

**सामान्य**

| गुण | डिफ़ॉल्ट | अर्थ |
|----------|---------|---------|

| Layer | `0` | लेयर असाइनमेंट |

| Linetype Scale | 1 | लाइनटाइप पैटर्न पर स्केल फैक्टर |
| Thickness | 0 | एक्सट्रूज़न मोटाई |

**ज्यामिति**

| गुण | अर्थ |
|----------|---------|
| Degree | बहुपद घात — हमेशा 3 (क्यूबिक) |
| Control Vertices | सभी CVs के निर्देशांक |
| Fit Points | CV स्प्लाइन के लिए खाली; केवल फिट-पॉइंट स्प्लाइन के लिए आबाद |

## Spline CV बनाम Spline Fit — कौन सा उपयोग करें

| | Spline CV | Spline Fit |
|---|-----------|------------|
| वक्र बिंदुओं से होकर गुजरता है | केवल पहले और आखिरी (clamped) | हर क्लिक किया गया बिंदु ठीक |
| आकार नियंत्रण | CVs को एक क्षेत्र की ओर खींचें | फिट पॉइंट मूव करें जिसे वक्र छूना चाहिए |
| ग्रिप संपादन प्रभाव | CV मूव → वक्र आकर्षित | फिट पॉइंट मूव → वक्र पुनः-इंटरपोलेटेड |
| सबसे अच्छा | सुचारू सौंदर्यात्मक वक्र, मुक्त-रूप पथ | वक्र जो विशिष्ट निर्देशांक छूने चाहिए |

## DXF — SPLINE ऑब्जेक्ट (कंट्रोल-वर्टेक्स रूप)

CV स्प्लाइन DXF फ़ाइल में `SPLINE` ऑब्जेक्ट के रूप में सहेजी जाती हैं, घात, नॉट वेक्टर और सभी कंट्रोल वर्टेक्स निर्देशांक संग्रहीत करती हैं। सभी गुण — रंग, लेयर, लाइनटाइप, लाइनटाइप स्केल, और मोटाई — बिना किसी नुकसान के राउंड-ट्रिप करते हैं। `splineFlag` `9` (CV स्प्लाइन) पर सेट होता है ताकि फिर से लोड करने पर रूप संरक्षित रहे। CV डेटा के साथ `SPLINE` ऑब्जेक्ट का समर्थन करने वाला कोई भी DXF एप्लिकेशन इन्हें सही तरीके से पढ़ता है।
