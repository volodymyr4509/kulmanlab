# Circle

`circle` कमांड केंद्र बिंदु और त्रिज्या से परिभाषित वृत्त बनाती है। केंद्र क्लिक करने के बाद आप त्रिज्या या तो कैनवास पर दूसरा बिंदु क्लिक करके या सटीक संख्या टाइप करके सेट कर सकते हैं — दोनों विकल्प एक साथ लाइव हैं।

## वृत्त बनाना

1. टर्मिनल में `circle` टाइप करें या **Circle** टूलबार बटन पर क्लिक करें।
2. **केंद्र बिंदु पर क्लिक करें**, या सटीक निर्देशांक के लिए `X,Y` टाइप करें और **Enter** दबाएं।
3. त्रिज्या सेट करें — या तो:
   - कैनवास पर **कोई भी बिंदु क्लिक करें** — केंद्र से दूरी त्रिज्या बन जाती है, या
   - **त्रिज्या टाइप करें** और सटीक मान के लिए **Enter** दबाएं।

वृत्त तुरंत रखा जाता है और कमांड बाहर निकल जाती है।

```
  center ●
          \  radius line preview
           \
            ● ← click here, or type a number
```

त्रिज्या चरण में, लाइव प्रीव्यू वर्तमान कर्सर दूरी पर वृत्त दिखाता है और केंद्र से वर्तमान बिंदु तक त्रिज्या रेखा भी खींचता है।

## केंद्र निर्देशांक प्रविष्टि

क्लिक करने के बजाय, केंद्र स्थिति टाइप करें:

1. X मान टाइप करें।
2. `,` दबाएं — टर्मिनल `[X], [Y{cursor}]` दिखाता है।
3. Y मान टाइप करें।
4. केंद्र रखने और त्रिज्या इनपुट पर आगे बढ़ने के लिए **Enter** दबाएं।

## टाइप की गई त्रिज्या इनपुट

केंद्र रखने के बाद, टाइप करना तुरंत त्रिज्या मान बनाता है:

| Key | Action |
|-----|--------|
| `0`–`9`, `.` | त्रिज्या मान में अंक जोड़ें |
| `Backspace` | अंतिम टाइप किया गया अक्षर हटाएं |
| `Enter` | टाइप की गई त्रिज्या पर वृत्त रखें |

संचित मान टर्मिनल प्रॉम्प्ट में दिखाया जाता है (जैसे `enter radius of circle: 25`)। जबकि कर्सर त्रिज्या रेखा मार्कर की दिशा नियंत्रित करता है, प्रीव्यू टाइप की गई त्रिज्या दिखाने के लिए अपडेट होता है।

## कीबोर्ड संदर्भ

| Key | Action |
|-----|--------|
| `0`–`9`, `.`, `-` | X निर्देशांक प्रविष्टि शुरू करें (केंद्र चरण), या त्रिज्या अंक (त्रिज्या चरण) |
| `,` | X लॉक करें और Y प्रविष्टि पर जाएं (केंद्र चरण) |
| `Backspace` | अंतिम टाइप किया गया अक्षर हटाएं |
| `Enter` | टाइप किए गए निर्देशांक या त्रिज्या की पुष्टि करें |
| `Escape` | रद्द करें और रीसेट करें |

## ग्रिप संपादन — त्रिज्या बदलना

एक चयनित वृत्त में पाँच ग्रिप होते हैं:

| ग्रिप | स्थान | क्या करता है |
|------|----------|--------------|
| **Center** | केंद्र बिंदु | पूरे वृत्त को मूव करता है; त्रिज्या अपरिवर्तित रहती है |
| **Left** | सबसे बाईं तरफ का बिंदु (केंद्र − त्रिज्या) | नई त्रिज्या = केंद्र से दूरी सेट करने के लिए ड्रैग करें |
| **Right** | सबसे दाईं तरफ का बिंदु (केंद्र + त्रिज्या) | नई त्रिज्या = केंद्र से दूरी सेट करने के लिए ड्रैग करें |
| **Top** | सबसे ऊपर का बिंदु | नई त्रिज्या = केंद्र से दूरी सेट करने के लिए ड्रैग करें |
| **Bottom** | सबसे नीचे का बिंदु | नई त्रिज्या = केंद्र से दूरी सेट करने के लिए ड्रैग करें |

सभी चार कार्डिनल ग्रिप समान व्यवहार करते हैं — नई त्रिज्या केंद्र से ड्रैग स्थिति तक की दूरी के बराबर होती है। केंद्र स्थिर रहता है।

## वृत्त चुनना

| विधि | व्यवहार |
|--------|-----------|
| **क्लिक** | यदि क्लिक परिधि के पास है तो चुनता है |
| **दाईं ओर ड्रैग** (सख्त) | पूरा बाउंडिंग स्क्वायर (केंद्र ± त्रिज्या) बॉक्स के अंदर होना चाहिए |
| **बाईं ओर ड्रैग** (क्रॉसिंग) | परिधि का कोई भी भाग जो बॉक्स की सीमा को पार करे वृत्त को चुनता है |

## Circle बनाम Arc — कब किसका उपयोग करें

| | Circle | Arc |

| विस्तार | पूर्ण 360° | आंशिक — स्टार्ट और एंड कोण से परिभाषित |
| कैसे बनाएं | केंद्र + त्रिज्या | वक्र पर तीन बिंदु |
| टाइप किया गया इनपुट | त्रिज्या मान | कोई नहीं — केवल क्लिक |

| माप | त्रिज्या: [Dim Radius](../dim-radius/) · व्यास: [Dim Diameter](../dim-diameter/) | [Dim Radius](../dim-radius/) |
| सबसे अच्छा | पूर्ण छेद, बोल्ट वृत्त, गोल फीचर | फ़िलेट, आंशिक वक्र, धनुषाकार पथ |

## DXF — CIRCLE ऑब्जेक्ट

वृत्त DXF फ़ाइल में `CIRCLE` ऑब्जेक्ट के रूप में सहेजे जाते हैं। केंद्र निर्देशांक, त्रिज्या, रंग, लेयर, लाइनटाइप, लाइनटाइप स्केल, और मोटाई सभी बिना किसी नुकसान के राउंड-ट्रिप करते हैं। कोई भी DXF-संगत एप्लिकेशन इन्हें मानक वृत्त के रूप में पढ़ता है।
