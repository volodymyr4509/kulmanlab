# Print

`print` कमांड **Print Manager** खोलती है — एक समर्पित निर्यात विंडो जिसमें लाइव प्रीव्यू कैनवास, फ़ॉर्मेट सेलेक्टर (PNG / JPEG / WebP / PDF), मोनोक्रोम टॉगल, और वैकल्पिक क्षेत्र क्रॉप होता है। कुछ भी भौतिक प्रिंटर को नहीं भेजा जाता; आउटपुट फ़ाइल के रूप में डाउनलोड होता है।

## Print Manager खोलना

**Print** टूलबार बटन पर क्लिक करें या टर्मिनल में `print` टाइप करें। Print Manager तुरंत वर्तमान व्यूपोर्ट का प्रीव्यू दिखाते हुए खुलता है।

## Print Manager लेआउट

विंडो में दो पैनल हैं:
- **बाईं साइडबार** — सभी निर्यात नियंत्रण।
- **दायां पैनल** — लाइव प्रीव्यू कैनवास जो सेटिंग्स बदलने पर अपडेट होता है।

### साइडबार नियंत्रण

| नियंत्रण | विवरण |
|---------|-------|
| **Change Area** | कैनवास पर कस्टम आयत पर क्रॉप करें (नीचे देखें) |
| **Monochrome** टॉगल | सभी रंगीन रेखाओं को काले में परिवर्तित करें — स्वच्छ प्रिंट आउटपुट के लिए डिफ़ॉल्ट रूप से चालू |
| **Format** ड्रॉपडाउन | PNG, JPEG, WebP, या PDF |
| **Export** बटन | फ़ाइल जेनरेट करें और डाउनलोड करें |

## कस्टम निर्यात क्षेत्र चुनना

डिफ़ॉल्ट रूप से प्रीव्यू वही दिखाता है जो Print Manager खोलने पर कैनवास पर दृश्यमान था। एक विशिष्ट क्षेत्र निर्यात करने के लिए:

1. **Change Area** पर क्लिक करें — Print Manager छुपता है और कैनवास इंटरएक्टिव हो जाता है।
2. निर्यात आयत का **पहला कोना क्लिक करें**।
3. **विपरीत कोना क्लिक करें** — Print Manager चयनित क्षेत्र प्रीव्यू के साथ फिर से खुलता है।

क्षेत्र चयन के दौरान रद्द करने और पिछला क्षेत्र पुनर्स्थापित करने के लिए `Escape` दबाएं।

प्रीव्यू कैनवास चयनित क्षेत्र के **सटीक आस्पेक्ट रेशियो** से मेल खाने के लिए गतिशील रूप से आकार बदलता है, इसलिए प्रीव्यू पिक्सल-सटीक है।

## निर्यात प्रारूप

| प्रारूप | सबसे अच्छा | नोट्स |
|--------|----------|-------|
| **PNG** | लॉसलेस, तीखी रेखाएं | सफ़ेद पृष्ठभूमि, कोई पारदर्शिता नहीं |
| **JPEG** | शेयरिंग के लिए छोटी फ़ाइल | 95% गुणवत्ता, थोड़ा संपीड़न |
| **WebP** | वेब के लिए सबसे छोटी फ़ाइल | समान 95% गुणवत्ता, JPEG से बेहतर संपीड़न |
| **PDF** | प्रिंट-रेडी दस्तावेज़ | PDF कंटेनर के अंदर 150 DPI पर एम्बेड की गई छवि |

निर्यात की गई फ़ाइल का नाम `kulman-<timestamp>.<ext>` है और स्वचालित रूप से डाउनलोड होती है।

## निर्यात रिज़ॉल्यूशन और पृष्ठभूमि

- अधिकतम रिज़ॉल्यूशन: **2000 × 2000 पिक्सल**, चयनित क्षेत्र के अनुपात में स्केल।
- पृष्ठभूमि हमेशा **सफ़ेद** होती है।
- **नॉन-प्लॉटिंग** चिह्नित लेयर निर्यात से बाहर होती हैं।

## कीबोर्ड संदर्भ

| Key | Action |
|-----|--------|
| `Escape` (क्षेत्र चयन के दौरान) | क्षेत्र चयन रद्द करें, पिछला क्षेत्र पुनर्स्थापित करें |
| `Escape` (Print Manager में) | Print Manager बंद करें |
