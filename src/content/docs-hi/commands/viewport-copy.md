# ViewportCopy

`ViewportCopy` कमांड व्यूपोर्ट को उसके स्केल और मॉडल केंद्र सुरक्षित रखते हुए नई स्थिति पर कॉपी करती है। केवल लेआउट स्पेस में उपलब्ध।

## व्यूपोर्ट कॉपी करना

1. पेपर लेआउट टैब पर स्विच करें।
2. वैकल्पिक रूप से व्यूपोर्ट को पहले से चुनने के लिए क्लिक करें।
3. टर्मिनल में `ViewportCopy` टाइप करें या **Viewport Copy** टूलबार बटन पर क्लिक करें।
4. यदि कोई व्यूपोर्ट पहले से नहीं चुना था, **कॉपी करने के लिए व्यूपोर्ट पर क्लिक करें**।
5. **बेस पॉइंट पर क्लिक करें** — विस्थापन के लिए संदर्भ। या सटीक निर्देशांक के लिए `X,Y` टाइप करें और **Enter** दबाएं।
6. **गंतव्य पर क्लिक करें** — व्यूपोर्ट बेस→गंतव्य ऑफसेट पर रखा जाता है। या निर्देशांक प्रविष्टि / कोण लॉक का उपयोग करें।

रखने के बाद, कमांड सक्रिय रहती है — उसी व्यूपोर्ट की एक और कॉपी रखने के लिए किसी अन्य गंतव्य पर क्लिक करें। समाप्त करने के लिए **Enter**, **Space**, या **Escape** दबाएं।

## निर्देशांक प्रविष्टि

बेस पॉइंट और गंतव्य चरणों पर क्लिक करने के बजाय सटीक निर्देशांक टाइप कर सकते हैं:

1. X मान टाइप करें।
2. `,` दबाएं — टर्मिनल `[X], [Y{cursor}]` दिखाता है।
3. Y मान टाइप करें।
4. पुष्टि करने के लिए **Enter** दबाएं।

## कोण लॉकिंग और सटीक दूरी

बेस पॉइंट सेट होने के बाद, जब कर्सर संरेखित होता है तो कमांड 45° अक्षों (0°, 45°, 90°, 135°, …) पर स्नैप करती है। लॉक होने पर:

- प्रीव्यू अक्ष पर स्नैप होता है।
- दूरी टाइप करें और लॉक दिशा में ठीक उस ऑफसेट पर कॉपी रखने के लिए **Enter** दबाएं।

| Key | Action |
|-----|--------|
| `0`–`9`, `.` | दूरी मान में अंक जोड़ें |
| `-` | नकारात्मक दूरी (दिशा उलटती है; केवल पहला अक्षर) |
| `Backspace` | अंतिम टाइप किया गया अक्षर हटाएं |
| `Enter` | टाइप की गई दूरी पर कॉपी रखें |

## कीबोर्ड संदर्भ

| Key | Action |
|-----|--------|
| `0`–`9`, `.`, `-` | X निर्देशांक प्रविष्टि शुरू करें, या कोण-लॉक होने पर दूरी |
| `,` | X लॉक करें और Y प्रविष्टि पर जाएं |
| `Enter` | टाइप किए गए निर्देशांक या दूरी की पुष्टि करें |
| `Enter` / `Space` | समाप्त करें (जब कोई इनपुट जारी न हो) |
| `Escape` | रद्द करें और रीसेट करें |

## नोट्स

- ViewportCopy केवल तब उपलब्ध है जब पेपर लेआउट टैब सक्रिय हो।
- कॉपी किया गया व्यूपोर्ट मूल के समान स्केल, मॉडल केंद्र, लॉक स्थिति, और आयाम इनहेरिट करता है।
- नया व्यूपोर्ट शुरू से बनाने के लिए, [ViewportRectangle](../viewport-rectangle/) का उपयोग करें।
