# Leader+

`leader+` कमांड मौजूदा मल्टीलीडर में एक नई तीर भुजा जोड़ती है। नई भुजा लीडर के मौजूदा डॉगलेग से एक नई तीर टिप की ओर इशारा करती है जिसे आप क्लिक करते हैं। सभी स्टाइल — डॉगलेग स्थिति, टेक्स्ट, तीर प्रकार, और आकार — चयनित लीडर से विरासत में मिलते हैं।

## भुजा जोड़ना

1. टर्मिनल में `leader+` टाइप करें।
2. एक **मौजूदा मल्टीलीडर पर क्लिक करें** उसे चुनने के लिए।
3. **नई तीर टिप पर क्लिक करें**, या सटीक निर्देशांक के लिए `X,Y` टाइप करें और **Enter** दबाएं। कर्सर से लीडर के डॉगलेग तक एक प्रीव्यू रेखा दिखाई देती है।

भुजा रखी जाती है और कमांड सक्रिय रहती है — आप तुरंत और भुजाएं जोड़ने के लिए किसी अन्य लीडर पर क्लिक कर सकते हैं। समाप्त करने के लिए **Enter**, **Space**, या **Escape** दबाएं।

```
  Before:                        After:
  ◄── arm 1                      ◄── arm 1
       \                               \
        ●──── dogleg ──── text          ●──── dogleg ──── text
                                       /
                                  arm 2 ──►  (new tip you clicked)
```

## टिप के लिए निर्देशांक प्रविष्टि

क्लिक करने के बजाय, सटीक स्थिति टाइप कर सकते हैं:

1. X मान टाइप करें।
2. `,` दबाएं — टर्मिनल X लॉक होने की पुष्टि करता है।
3. Y मान टाइप करें।
4. रखने के लिए **Enter** दबाएं।

## कीबोर्ड संदर्भ

| Key | Action |
|-----|--------|
| `0`–`9`, `.`, `-` | X निर्देशांक टाइप करना शुरू करें |
| `,` | X लॉक करें और Y प्रविष्टि पर जाएं |
| `Enter` | टाइप किए गए निर्देशांक की पुष्टि करें और भुजा रखें |
| `Enter` / `Space` | समाप्त करें (जब कोई इनपुट प्रगति में न हो) |
| `Escape` | रद्द करें और रीसेट करें |

## नोट्स

- केवल **Multileader** ऑब्जेक्ट चुने जा सकते हैं — किसी अन्य ऑब्जेक्ट प्रकार पर क्लिक करने से कुछ नहीं होता।
- नई भुजा मौजूदा डॉगलेग से निकलती है; आप केवल चुनते हैं कि तीर टिप कहाँ जाए।
- एक मल्टीलीडर में कितनी भुजाएं हो सकती हैं, इसकी कोई सीमा नहीं है।
