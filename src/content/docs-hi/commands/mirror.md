# Mirror

`mirror` कमांड चयनित ऑब्जेक्ट की मिरर की गई कॉपी बनाती है जो एक दो-बिंदु अक्ष पर परावर्तित होती है। मूल **हमेशा रखे जाते हैं** — [Move](../move/) या [Rotate](../rotate/) के विपरीत, Mirror कभी भी मौजूदा ऑब्जेक्ट को संशोधित नहीं करता; यह केवल नए जोड़ता है।

## शुरू करने के दो तरीके

**पहले चुनें, फिर मिरर करें** — पहले ऑब्जेक्ट चुनें, फिर सक्रिय करें:

1. कैनवास पर एक या अधिक ऑब्जेक्ट चुनें।
2. टर्मिनल में `mirror` टाइप करें या **Mirror** टूलबार बटन पर क्लिक करें।
3. **मिरर अक्ष के पहले बिंदु पर क्लिक करें**, या सटीक निर्देशांक के लिए `X,Y` टाइप करें और **Enter** दबाएं।
4. **दूसरे बिंदु पर क्लिक करें** — मिरर की गई कॉपी रखी जाती हैं और कमांड बाहर निकल जाती है। यहाँ भी निर्देशांक प्रविष्टि काम करती है।

**पहले सक्रिय करें, फिर चुनें** — कुछ भी चुने बिना कमांड शुरू करें:

1. `mirror` टाइप करें या टूलबार बटन पर क्लिक करें।
2. **ऑब्जेक्ट चुनें** — टॉगल करने के लिए क्लिक करें, या क्षेत्र से चुनने के लिए ड्रैग करें।
3. चयन की पुष्टि करने के लिए **Enter** या **Space** दबाएं।
4. **पहले बिंदु पर क्लिक करें**, फिर मिरर अक्ष के **दूसरे बिंदु पर क्लिक करें** (दोनों चरणों पर निर्देशांक प्रविष्टि उपलब्ध)।

```
  Original:          Mirror axis:        Result:
                     |
  [entity A]    →    |    →    [entity A] + [mirrored A]
                     |
```

दूसरे अक्ष बिंदु की स्थिति निर्धारित करते समय मिरर की गई कॉपी का एक लाइव प्रीव्यू कर्सर का अनुसरण करता है।

## मिरर अक्ष

अक्ष दो क्लिक किए गए बिंदुओं से होकर एक अनंत रेखा है। यह किसी भी कोण पर हो सकती है:

- कर्सर को **45° स्नैप अक्ष** (0°, 45°, 90°, 135°, …) के पास ले जाएं और अक्ष उस कोण पर लॉक हो जाती है — साफ़ क्षैतिज, ऊर्ध्वाधर या विकर्ण प्रतिबिंब के लिए उपयोगी।
- स्नैप ज़ोन से दूर क्लिक करें मुक्त-कोण अक्ष के लिए।

## निर्देशांक प्रविष्टि

किसी भी अक्ष-बिंदु चरण पर, क्लिक करने के बजाय सटीक स्थिति टाइप करें:

1. X मान टाइप करें।
2. `,` दबाएं — टर्मिनल `[X], [Y{cursor}]` दिखाता है।
3. Y मान टाइप करें।
4. पुष्टि करने के लिए **Enter** दबाएं।

## कीबोर्ड संदर्भ

| Key | Action |
|-----|--------|
| `Enter` / `Space` | चयन की पुष्टि करें |
| `0`–`9`, `.`, `-` | X निर्देशांक प्रविष्टि शुरू करें |
| `,` | X लॉक करें और Y प्रविष्टि पर जाएं |
| `Backspace` | अंतिम टाइप किया गया अक्षर हटाएं |
| `Enter` | टाइप किए गए निर्देशांक की पुष्टि करें |
| `Escape` | रद्द करें और रीसेट करें |

## कमांड के दौरान चयन

| विधि | व्यवहार |
|--------|-----------|
| **क्लिक** | कर्सर के नीचे ऑब्जेक्ट को टॉगल करता है |
| **दाईं ओर ड्रैग** (सख्त) | बॉक्स के पूरी तरह अंदर ऑब्जेक्ट जोड़ता है |
| **बाईं ओर ड्रैग** (क्रॉसिंग) | बॉक्स को काटने वाले ऑब्जेक्ट जोड़ता है |
| **Enter** / **Space** | चयन की पुष्टि करता है |

## क्या मिरर किया जाता है

हर ऑब्जेक्ट प्रकार समर्थित है। ज्यामिति गणितीय रूप से अक्ष पर परावर्तित होती है:

| ऑब्जेक्ट | क्या बदलता है |
|--------|-------------|
| Line | दोनों एंडपॉइंट परावर्तित |
| Circle | केंद्र परावर्तित; त्रिज्या अपरिवर्तित |
| Arc | केंद्र परावर्तित; स्टार्ट और एंड कोण अक्ष पर पुनर्गणित |
| Ellipse | केंद्र परावर्तित; प्रमुख अक्ष दिशा अक्ष पर पलटी |
| Polyline / Rectangle | हर वर्टेक्स परावर्तित |
| Text | एंकर बिंदु परावर्तित; टेक्स्ट स्ट्रिंग **उलटी नहीं** होती |
| Spline | सभी कंट्रोल वर्टेक्स / फिट पॉइंट परावर्तित |
