# Dimension Radius

`dimradius` कमांड एक चाप या वृत्त पर त्रिज्या माप रखती है। माप रेखा केंद्र से कर्सर की दिशा में परिधि पर एक बिंदु तक चलती है, `R <मान>` से लेबल। पूर्ण व्यास मापने के लिए, [Dimension Diameter](../dim-diameter/) का उपयोग करें।

## त्रिज्या माप की संरचना

```
  ● (center)
   \
    \  R 5.00
     \
      ●────── text (cursor side)
   (arc point)
```

- **माप रेखा** — केंद्र से चाप बिंदु से कर्सर की ओर, चाप पर तीर के साथ।
- **लेबल** — `R` के बाद त्रिज्या मान।

## त्रिज्या माप रखना

1. टर्मिनल में `dimradius` टाइप करें या **Dimension Radius** टूलबार बटन पर क्लिक करें।
2. इसे चुनने के लिए **एक चाप या वृत्त पर क्लिक करें**।
3. माप रेखा अभिमुख करने के लिए **कर्सर मूव करें** — चाप-बिंदु केंद्र से कर्सर की दिशा में ट्रैक करता है।
4. माप रखने के लिए **क्लिक करें**।

केवल **Arc** और **Circle** ऑब्जेक्ट चुने जा सकते हैं। किसी अन्य ऑब्जेक्ट प्रकार पर क्लिक करने से कुछ नहीं होता।

## कीबोर्ड संदर्भ

| Key | Action |
|-----|--------|
| `Escape` | रद्द करें |

## Dimension Radius बनाम Dimension Diameter

| | Dimension Radius | Dimension Diameter |
|---|-----------------|-------------------|
| मापता है | त्रिज्या (केंद्र से किनारे तक) | व्यास (केंद्र से होकर किनारे से किनारे तक) |
| माप रेखा | केंद्र → चाप बिंदु | चाप बिंदु → चाप बिंदु (केंद्र से होकर) |
| लेबल उपसर्ग | `R` | `⌀` |
| तीर | एक (चाप बिंदु पर) | दो (दोनों चाप बिंदुओं पर) |
| सबसे अच्छा | एक घुमाव फीचर की एक साइड एनोटेट करना | पूर्ण वृत्ताकार माप एनोटेट करना |

## लेबल संपादित करना — सरल मोड

रखी गई त्रिज्या माप पर **डबल-क्लिक** करें **सरल** मोड में टेक्स्ट एडिटर खोलने के लिए। एडिटर में वर्तमान रेंडर की गई वैल्यू (जैसे `R 5.00`) पहले से भरी होती है ताकि आप कर्सर की स्थिति निर्धारित करके सीधे संपादित कर सकें।

| विशेषता | व्यवहार |
|---------|-----------|
| Bold / Italic / Font / Height | **पूरे** लेबल पर एक साथ लागू होती है |
| प्रति-अक्षर फ़ॉर्मेटिंग | समर्थित नहीं |
| `Enter` | वैल्यू सहेजता है और एडिटर बंद करता है |
| बहु-पंक्ति | समर्थित नहीं |

पूर्ण संदर्भ के लिए [Text Editor — simple mode](../../interface/text-editor/#simple-mode) देखें।

## DXF — DIMENSION त्रिज्या ऑब्जेक्ट

त्रिज्या माप त्रिज्या-प्रकार ज्यामिति के साथ `DIMENSION` ऑब्जेक्ट के रूप में सहेजे जाते हैं, केंद्र निर्देशांक, चाप-बिंदु स्थिति और मापी गई त्रिज्या मान संग्रहीत करते हैं। सभी गुण बिना किसी नुकसान के राउंड-ट्रिप करते हैं।
