# القلب

يُنشئ أمر `mirror` نسخاً معكوسة من العناصر المحددة منعكسةً عبر محور نقطتين. **تُحتفظ بالأصول دائماً** — على خلاف [Move](../move/) أو [Rotate](../rotate/)، لا يُعدِّل القلب العناصر الموجودة أبداً؛ بل يُضيف عناصر جديدة فحسب.

## طريقتان للبدء

**التحديد المسبق ثم القلب** — حدد العناصر أولاً ثم فعِّل الأمر:

1. حدد عنصراً أو أكثر على اللوحة.
2. اكتب `mirror` في المحطة أو انقر على زر **Mirror** في شريط الأدوات.
3. **انقر على النقطة الأولى** لمحور الانعكاس، أو اكتب `X,Y` واضغط **Enter** للإحداثي الدقيق.
4. **انقر على النقطة الثانية** — تُوضع النسخ المعكوسة ويخرج الأمر. إدخال الإحداثيات يعمل هنا أيضاً.

**التفعيل ثم التحديد** — ابدأ الأمر مع عدم تحديد أي شيء:

1. اكتب `mirror` أو انقر على زر شريط الأدوات.
2. **حدد الكائنات** — انقر للتبديل، أو اسحب للتحديد بالمنطقة.
3. اضغط **Enter** أو **Space** لتأكيد التحديد.
4. **انقر على النقطة الأولى**، ثم **انقر على النقطة الثانية** لمحور الانعكاس (إدخال الإحداثيات متاح في كلتا الخطوتين).

```
  أصلي:          محور الانعكاس:        النتيجة:
                      |
  [عنصر أ]    →      |    →    [عنصر أ] + [عنصر أ معكوس]
                      |
```

معاينة حية للنسخ المعكوسة تتبع المؤشر بينما تضع نقطة المحور الثانية.

## محور الانعكاس

المحور خط لا نهائي يمر بالنقطتين المنقورتين. يمكن أن يكون بأي زاوية:

- حرّك المؤشر قريباً من **محور التثبيت بزاوية 45°** (0°، 45°، 90°، 135°، ...) ويُثبَّت المحور على تلك الزاوية — مفيد للانعكاسات الأفقية أو الرأسية أو القطرية النظيفة.
- انقر بعيداً عن منطقة التثبيت للحصول على محور بزاوية حرة.

## إدخال الإحداثيات

في خطوة أي نقطة محور، اكتب موضعاً دقيقاً بدلاً من النقر:

1. اكتب قيمة X.
2. اضغط `,` — تُظهر المحطة `[X], [Y{cursor}]`.
3. اكتب قيمة Y.
4. اضغط **Enter** للتأكيد.

## مرجع لوحة المفاتيح

| المفتاح | الإجراء |
|---------|---------|
| `Enter` / `Space` | تأكيد التحديد |
| `0`–`9`، `.`، `-` | بدء إدخال إحداثي X |
| `,` | تثبيت X والانتقال إلى إدخال Y |
| `Backspace` | حذف آخر حرف مكتوب |
| `Enter` | تأكيد الإحداثي المكتوب |
| `Escape` | إلغاء وإعادة التعيين |

## التحديد أثناء الأمر

| الطريقة | السلوك |
|---------|--------|
| **النقر** | يبدّل العنصر تحت المؤشر |
| **السحب يميناً** (صارم) | يُضيف العناصر الواقعة بالكامل داخل المربع |
| **السحب يساراً** (متقاطع) | يُضيف العناصر التي تتقاطع مع المربع |
| **Enter** / **Space** | يؤكد التحديد |

## ما الذي يُعكس

كل نوع عنصر مدعوم. تُعكس الهندسة عبر المحور رياضياً:

| العنصر | ما الذي يتغير |
|--------|-------------|
| الخط | كلتا نقطتَي النهاية مُعكستان |
| الدائرة | المركز مُعكس؛ نصف القطر دون تغيير |
| القوس | المركز مُعكس؛ تُعاد حساب زاويتَي البداية والنهاية عبر المحور |
| القطع الناقص | المركز مُعكس؛ اتجاه المحور الرئيسي مقلوب عبر المحور |
| الخط المتعدد / المستطيل | كل رأس مُعكس |
| النص | نقطة التثبيت مُعكسة؛ سلسلة النص **غير** معكوسة |
| المنحنى | جميع نقاط التحكم / نقاط التناسب مُعكسة |
