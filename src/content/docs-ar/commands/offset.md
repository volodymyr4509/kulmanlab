# الإزاحة

يُنشئ أمر `offset` نسخة متوازية من عنصر على مسافة عمودية ثابتة. تكتب المسافة مرة واحدة، ثم تنقر على العناصر وتختار جانباً — يبقى الأمر جاهزاً على المسافة ذاتها لتتمكن من إزاحة كائنات متعددة في جلسة واحدة.

أنواع العناصر المدعومة: **Line وCircle وArc وEllipse وPolyline** (بما فيها المستطيلات).

## استخدام الإزاحة

1. اكتب `offset` في المحطة أو انقر على زر **Offset** في شريط الأدوات.
2. **اكتب مسافة الإزاحة** واضغط **Enter** أو **Space**.
3. **انقر على عنصر** لإزاحته — إذا لم يكن من نوع مدعوم، تظهر رسالة خطأ ويمكنك النقر على عنصر آخر.
4. **حرّك المؤشر** إلى الجانب الذي يجب أن تظهر فيه النسخة — معاينة حية تتبع المؤشر.
5. **انقر** لوضع النسخة المُزاحة.

بعد كل وضع يعود الأمر إلى الخطوة 3 على **المسافة ذاتها**، جاهزاً للإزاحة التالية. اضغط **Escape** لإعادة التعيين إلى خطوة إدخال المسافة.

```
  المسافة: 10

  ─────────────────    ← الخط الأصلي
  ─────────────────    ← النسخة المُزاحة (10 وحدات للأسفل)
```

## سلوك الإزاحة لكل عنصر

| العنصر | كيف تُحسب الإزاحة |
|--------|------------------|
| **الخط** | خط موازٍ مُزاح عمودياً على الاتجاه الأصلي |
| **الدائرة** | دائرة متحدة المركز؛ نقر خارجي → نصف قطر أكبر، نقر داخلي → نصف قطر أصغر |
| **القوس** | قوس متحد المركز بنصف قطر جديد؛ الامتداد الزاوي محفوظ |
| **القطع الناقص** | يزداد أو يتناقص نصفا المحورين بالمسافة ذاتها |
| **الخط المتعدد** | كل مقطع يُزاح بشكل مستقل؛ المقاطع المُزاحة المجاورة تُحسب طرفياً عند الزوايا |

بالنسبة لـ**الدائرة** و**القوس** و**القطع الناقص**: إذا كانت الإزاحة الداخلية ستُقلِّل أي نصف قطر أو نصف محور إلى الصفر أو ما دونه، لا تُطبَّق الإزاحة.

## مرجع لوحة المفاتيح

| المفتاح | الإجراء |
|---------|---------|
| `0`–`9`، `.` | إلحاق رقم بقيمة المسافة |
| `Backspace` | حذف آخر حرف مكتوب |
| `Enter` / `Space` | تأكيد المسافة المكتوبة والانتقال إلى اختيار العنصر |
| `Escape` | إعادة التعيين إلى خطوة إدخال المسافة |

## ملاحظة حول سير العمل

تبقى المسافة مُعيَّنة حتى تضغط **Escape**. هذا يجعل إزاحة عناصر عديدة بالتباعد ذاته فعّالاً — اكتب المسافة مرة واحدة، ثم انقر واختر جانباً لكل عنصر بالتتالي.

## الإزاحة مقابل النسخ

| | الإزاحة | النسخ |
|---|--------|------|
| الإزاحة | عمودية على هندسة العنصر | شعاع تعسفي (أساس → وجهة) |
| العناصر المدعومة | الخط والدائرة والقوس والقطع الناقص والخط المتعدد | كل أنواع العناصر |
| إدخال المسافة | مكتوبة قبل اختيار العنصر | مكتوبة أو منقورة بعد الاختيار |
| الأنسب لـ | الخطوط المتوازية والدوائر المتحدة المركز والمسارات الداخلية/الخارجية | وضع النسخ في مواضع تعسفية |
