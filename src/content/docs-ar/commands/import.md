# Import

يُحمِّل أمر **Import** رسماً موجوداً من نظام الملفات المحلي إلى KulmanLab CAD. يُدعم كلٌّ من تنسيق **DXF** القياسي وتنسيق **JSON** الخاص بـ KulmanLab.

## كيفية استيراد ملف

1. انقر على زر **Import** في شريط الأدوات (أيقونة المجلد) في لوحة الملف في أعلى الشاشة.
2. يفتح منتقي الملفات في متصفحك. تصفح إلى ملف الرسم وحدده.
3. يُحمَّل الرسم على اللوحة فوراً. يُلائم إطار العرض جميع العناصر تلقائياً.

بدلاً من ذلك، يمكنك سحب ملف وإفلاته مباشرةً على اللوحة.

## تنسيقات الملف المدعومة

| التنسيق | الامتداد | متى تستخدمه |
|---------|----------|------------|
| **DXF** | `.dxf` | رسومات من FreeCAD وLibreCAD وأدوات CAD أخرى |
| **JSON** *(أصلي)* | `.json` | رسومات محفوظة مسبقاً من KulmanLab CAD — دقة كاملة |

## ما يُستورَد من DXF

يُحلِّل KulmanLab أنواع عناصر DXF التالية:

| نوع العنصر | رمز DXF | ملاحظات |
|------------|---------|---------|
| الخط | `LINE` | |
| الدائرة | `CIRCLE` | |
| القوس | `ARC` | |
| القطع الناقص | `ELLIPSE` | |
| الخط المتعدد | `LWPOLYLINE` | |
| المنحنى | `SPLINE` | |
| النص | `TEXT`، `MTEXT` | |
| البُعد | `DIMENSION` | |
| خط الإشارة المتعدد | `MULTILEADER` | |

تعريفات الطبقات وجداول نوع الخط تُستورَد أيضاً من ملف DXF عند وجودها.

العناصر التي تستخدم أنواع DXF غير مدعومة تُتخطى بهدوء — يُحمَّل بقية الرسم كما هو.

## تسمية الملف والتخزين

عند استيراد ملف، يُلحَق طابع زمني باسم الملف (مثل `myplan_May22_14:30:00.dxf`). يتيح هذا الاحتفاظ بإصدارات متعددة للرسم ذاته في [الملفات الأخيرة](../files/) دون تعارض في الأسماء. إذا كان اسم الملف يحتوي بالفعل على طابع زمني، يُستخدم كما هو.

يُحفظ الرسم تلقائياً في تخزين المتصفح (IndexedDB) بعد الاستيراد، لذا يظهر في لوح [الملفات](../files/) ويبقى عبر إعادات تحميل الصفحة.

## ما يحدث للرسم الحالي

الاستيراد يستبدل اللوحة الحالية. لا يوجد دمج أو إلحاق. إذا كان لديك تغييرات غير محفوظة، [صدِّر](../export/) الرسم الحالي أولاً.

## عند بدء التشغيل

يُعيد KulmanLab تلقائياً فتح آخر ملف تم تعديله عند تحميل الصفحة. إذا لم توجد ملفات محفوظة، يُحمَّل رسم نموذجي افتراضي.

## استكشاف الأخطاء وإصلاحها

| المشكلة | السبب المحتمل | الحل |
|---------|-------------|------|
| اللوحة فارغة بعد الاستيراد | عناصر DXF تستخدم أنواعاً غير مدعومة (مثل HATCH وINSERT) | العناصر تُخطَّت — تحقق من رسالة "no entities found" في المحطة |
| زر الاستيراد لا يفعل شيئاً | المتصفح حجب منتقي الملفات | انقر على الزر مرة أخرى؛ بعض المتصفحات تتطلب إيماءة مستخدم جديدة |
| الأبعاد تبدو خاطئة | DXF من أداة تكتب هندسة بُعد غير قياسية | أعد التصدير من التطبيق المصدر باستخدام إصدار DXF حديث |
