# Dimension Continue

يُسلسل أمر `dimcontinue` أبعاداً جديدة من **خط التمديد الثاني** لبُعد موجود. يُوضع كل مقطع جديد على محور القياس ذاته وعلى إزاحة خط البُعد ذاتها كالأساس. تُنسخ جميع خصائص النمط — حجم السهم وارتفاع النص وأطوال خطوط التمديد — من الأساس تلقائياً.

## كيف تبدو الأبعاد المتسلسلة

```
  |←— 3.00 —→|←— 2.50 —→|←— 4.00 —→|
  |           |           |           |
  ●           ●           ●           ●
  p1        p2 (أساس    p3           p4
           ext2 → بداية جديدة)
```

كل مستطيل عنصر `DIMENSION` مستقل. يتشاركون موضع خط البُعد ذاته واتجاه القياس ذاته.

## بدء تسلسل

1. اكتب `dimcontinue` في المحطة أو انقر على زر **Dimension Continue** في شريط الأدوات.
2. **إذا وُضع بُعد للتو** — يلتقطه الأمر تلقائياً كأساس (لا حاجة للنقر).
3. **إذا لم يوجد بُعد حديث** — انقر على أي بُعد موجود لاستخدامه كأساس.
4. **انقر على أصل خط التمديد التالي** — معاينة تُظهر البُعد الجديد أثناء تحريك المؤشر. أو اكتب `X,Y` واضغط **Enter** للإحداثي الدقيق.
5. استمر في النقر (أو الكتابة) لتمديد التسلسل. كل بُعد موضوع يصبح الأساس الجديد تلقائياً.
6. اضغط **Enter** أو **Space** أو **Escape** لإنهاء التسلسل.

## ما يُوروَث من البُعد الأساسي

| الخاصية | موروث من الأساس |
|---------|----------------|
| اتجاه القياس / الزاوية | نعم — مُثبَّت لكامل التسلسل |
| إزاحة خط البُعد (المسافة من النقاط المقيسة) | نعم |
| حجم السهم | نعم |
| ارتفاع النص | نعم |
| إزاحة خط التمديد وامتداده | نعم |
| محاذاة النص | نعم |
| اسم النمط | نعم |
| اللون والطبقة | غير موروث — يستخدم الطبقة الحالية |

## تثبيت اتجاه القياس

اتجاه قياس التسلسل **مُثبَّت على زاوية البُعد الأساسي**:

- أساس خطي (H) → جميع الاستمرارات تقيس المسافة الأفقية (Δ X).
- أساس خطي (V) → جميع الاستمرارات تقيس المسافة الرأسية (Δ Y).
- أساس محاذٍ بأي زاوية → جميع الاستمرارات تقيس على تلك الزاوية ذاتها.

لا يمكن تغيير الاتجاه في منتصف التسلسل. ابدأ [بُعداً خطياً](../dim-linear/) جديداً أو [بُعداً محاذياً](../dim-aligned/) للقياس في اتجاه مختلف.

## مرجع لوحة المفاتيح

| المفتاح | الإجراء |
|---------|---------|
| `0`–`9`، `.`، `-` | بدء إدخال إحداثي X |
| `,` | تثبيت X والانتقال إلى إدخال Y |
| `Backspace` | حذف آخر حرف مكتوب |
| `Enter` | تأكيد الإحداثي المكتوب، أو إنهاء التسلسل إذا لم يكن هناك إدخال جارٍ |
| `Space` / `Escape` | إنهاء التسلسل |

## البُعد المستمر مقابل البدء من جديد

| | البُعد المستمر | البُعد الخطي / المحاذي |
|---|-------------------|--------------------------|
| نقطة البداية | مُثبَّتة عند ext2 للأساس الأخير | انقر في أي مكان |
| الزاوية | مُثبَّتة على الأساس | حرة |
| الإزاحة | موروثة من الأساس | تُعيَّن بالمؤشر أو مكتوبة |
| النمط | موروث من الأساس | النمط الحالي |
| الأنسب لـ | القياسات التراكمية على طول صف | البُعد الأول أو تغيير الاتجاه |

## تحرير التسميات بعد الوضع — الوضع البسيط

**انقر مزدوجاً** على أي بُعد في التسلسل لفتح محرر النص في الوضع **البسيط**. كل مقطع مستقل ويمكن تحريره بشكل فردي.

| الميزة | السلوك |
|--------|--------|
| Bold / Italic / Font / Height | تُطبَّق على **التسمية بأكملها** دفعةً واحدة |
| التنسيق لكل حرف | غير مدعوم |
| `Enter` | يُثبِّت القيمة ويغلق المحرر |
| متعدد الأسطر | غير مدعوم |

راجع [محرر النص — الوضع البسيط](../../interface/text-editor/#simple-mode) للمرجع الكامل.

## DXF — عناصر DIMENSION

كل مقطع في التسلسل يُخزَّن كعنصر `DIMENSION` مستقل في ملف DXF. لا يتشابكون في الملف — يتشاركون الخصائص لأنهم أُنشئوا من الأساس ذاته، لكن يمكن تحرير كل منهم بشكل مستقل بعد الوضع.
