# النص

يضع أمر `text` تسمية نصية متعددة الأسطر. بعد النقر على موضع على اللوحة، يفتح محرر منبثق في الوضع **الغني** — يمكنك كتابة المحتوى وتطبيق الخط العريض/المائل/المشطوب لكل حرف، وتغيير الخطوط والارتفاعات، وإدراج فواصل الأسطر. اضغط **Escape** للتأكيد وإغلاق المحرر.

راجع صفحة [محرر النص](../interface/text-editor) للمرجع الكامل للمحرر، بما يشمل مقارنة بين الوضعَين **الغني** و**البسيط**.

## وضع تسمية نصية

1. اكتب `text` في المحطة أو انقر على زر **Text** في شريط الأدوات.
2. **انقر على موضع التثبيت** على اللوحة. أو اكتب `X,Y` واضغط **Enter** للإحداثي الدقيق.
3. يفتح **محرر النص المنبثق** فوق التسمية الجديدة. اكتب محتواك.
4. اضغط **Escape** لتأكيد التسمية وإغلاق المحرر.

الارتفاع الافتراضي هو **12 وحدة رسم**.

## تحرير تسمية موجودة

**انقر مزدوجاً** على أي تسمية نصية على اللوحة لإعادة فتح المحرر لتلك التسمية.

## إدخال إحداثي نقطة التثبيت

بدلاً من النقر، اكتب موضعاً دقيقاً:

1. اكتب قيمة X.
2. اضغط `,` — تُظهر المحطة `[X], [Y{cursor}]`.
3. اكتب قيمة Y.
4. اضغط **Enter** لوضع نقطة التثبيت وفتح المحرر.

## مرجع لوحة المفاتيح

**مرحلة نقطة التثبيت**

| المفتاح | الإجراء |
|---------|---------|
| `0`–`9`، `.`، `-` | بدء إدخال إحداثي X |
| `,` | تثبيت X والانتقال إلى إدخال Y |
| `Backspace` | حذف آخر حرف مكتوب |
| `Enter` | تأكيد الإحداثي المكتوب |

**مرحلة محرر النص** (راجع [محرر النص](../interface/text-editor) للمرجع الكامل)

| المفتاح | الإجراء |
|---------|---------|
| أي حرف قابل للطباعة | إدراج عند موضع المؤشر |
| `Backspace` / `Delete` | حذف الحرف المجاور أو التحديد |
| `Enter` | إدراج فاصل سطر |
| `←` / `→` | تحريك علامة الإقحام |
| `Home` / `End` | القفز إلى بداية/نهاية السطر الصارم |
| `Escape` | تأكيد وإغلاق المحرر |

## تحرير نقاط التحكم — إعادة التموضع

يُظهر النص المحدد نقطة تحكم واحدة عند نقطة التثبيت:

| نقطة التحكم | الموضع | ما تفعله |
|------------|--------|---------|
| **التثبيت** | أسفل يسار النص | اسحب لإعادة تموضع التسمية |

## تحديد النصوص

| الطريقة | السلوك |
|---------|--------|
| **النقر** | يُحدَّد إذا وقع النقر ضمن المربع المحيط المدوَّر للنص |
| **السحب يميناً** (صارم) | يجب أن تقع الزوايا الأربع للمربع المحيط داخل منطقة التحديد |
| **السحب يساراً** (متقاطع) | أي تداخل بين المربع المحيط للنص ومنطقة التحديد يُحدِّده |

## أوامر التحرير المدعومة

| الأمر | ما يحدث للنص |
|-------|------------|
| [Move](../move/) | يُحرك نقطة التثبيت |
| [Copy](../copy/) | يُنشئ تسمية مطابقة في موضع جديد |
| [Rotate](../rotate/) | يُدوِّر موضع نقطة التثبيت ويُضيف الزاوية إلى Rotation Degree |
| [Mirror](../mirror/) | يعكس نقطة التثبيت عبر محور الانعكاس (لا تُعكس سلسلة النص) |
| [Scale](../scale/) | يُغيِّر مقياس موضع نقطة التثبيت ويضرب الارتفاع في عامل التحجيم |
| [Delete](../delete/) | يحذف التسمية |

النص لا يدعم **Offset** أو **Trim** أو **Extend**.

## الخصائص

عند تحديد تسمية نصية، يُظهر لوح الخصائص:

**عام**

| الخاصية | الافتراضي | المعنى |
|---------|----------|--------|
| اللون | 256 (ByLayer) | فهرس لون ACI |
| الطبقة | `0` | تعيين الطبقة |

**الهندسة**

| الخاصية | المعنى |
|---------|--------|
| Position X / Position Y | إحداثيات نقطة التثبيت |
| Height | ارتفاع النص الأساسي بوحدات الرسم (الافتراضي: **12**) |
| Rotation Degree | دوران عكس اتجاه عقارب الساعة بالدرجات |

**الخصائص**

| الخاصية | المعنى |
|---------|--------|
| Content | سلسلة النص (رموز MTEXT المضمَّنة محفوظة) |
| Attachment Point | رمز المحاذاة (1 = أعلى اليسار ... 9 = أسفل اليمين) |

النص ليس له خصائص Linetype أو Linetype Scale أو Thickness.

## DXF — عنصر MTEXT

تُخزَّن التسميات النصية كعناصر **MTEXT** في ملف DXF. يُشفَّر الخط العريض والمائل باستخدام `\L` و`\K` و`\O` ومبدِّلات الخط المضمَّنة (`\f`). يُشفَّر ارتفاع كل حرف بـ`\H`. جميع التنسيقات تنجو من دورة DXF الكاملة وتكون قابلة للقراءة في LibreCAD وFreeCAD وغيرهما من التطبيقات المتوافقة مع DXF.
