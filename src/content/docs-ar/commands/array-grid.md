# Array Grid

يُنشئ أمر `ArrayGrid` شبكة مستطيلة من النسخ من العناصر المحددة — أدخل عدد الصفوف وعدد الأعمدة والمسافة بينها، كل ذلك مكتوب في المحطة. يشغل التحديد الأصلي خانة الصف 0، العمود 0؛ وكل خانة أخرى هي نسخة مُزاحة.

## طريقتان للبدء

**التحديد المسبق ثم Array** — حدد العناصر أولاً ثم فعِّل الأمر:

1. حدد عنصراً أو أكثر على اللوحة.
2. اكتب `arraygrid` في المحطة (يكفي `arr` — غير ملتبس) أو انقر على زر **Array Grid** في شريط الأدوات.
3. اكتب عدد **الصفوف** واضغط **Enter**.
4. اكتب عدد **الأعمدة** واضغط **Enter**.
5. اكتب **المسافة بين الصفوف** واضغط **Enter**.
6. اكتب **المسافة بين الأعمدة** واضغط **Enter** — تُنشأ الشبكة فوراً.

**التفعيل ثم التحديد** — ابدأ الأمر مع عدم تحديد أي شيء:

1. اكتب `arraygrid` أو انقر على زر شريط الأدوات.
2. **حدد الكائنات** — انقر لتبديل تحديد العناصر المنفردة، أو اسحب للتحديد بالمنطقة.
3. اضغط **Enter** أو **Space** لتأكيد التحديد.
4. تابع بالصفوف ← الأعمدة ← المسافة بين الصفوف ← المسافة بين الأعمدة كما هو أعلاه.

```
  صفان × 3 أعمدة:

  [B] [B] [B]   <- الصف 1 (نسخ مُزاحة)
  [A] [A] [A]   <- الصف 0: التحديد الأصلي، النسخ إلى اليمين
```

> لا تحتاج المحطة إلا للأحرف الكافية لعدم الالتباس — كتابة `arr` والضغط على **Enter** يُفعِّل Array Grid مباشرة، لأنه لا يوجد اسم أمر آخر يبدأ بهذه الأحرف الثلاثة (Arc وArea وAlign وAngle تتباين جميعها في وقت أبكر).

## الصفوف والأعمدة والمسافة

| المطالبة | تقبل | ملاحظات |
|----------|------|---------|
| الصفوف | أعداد صحيحة موجبة (1، 2، 3…) | أرقام فقط — بدون فاصلة عشرية أو إشارة |
| الأعمدة | أعداد صحيحة موجبة (1، 2، 3…) | أرقام فقط — بدون فاصلة عشرية أو إشارة |
| المسافة بين الصفوف | عدد عشري بإشارة (مثل `10`، `-5.5`) | المسافة بين الصفوف؛ السالب يعكس الاتجاه |
| المسافة بين الأعمدة | عدد عشري بإشارة (مثل `10`، `-5.5`) | المسافة بين الأعمدة؛ السالب يعكس الاتجاه |

مع صف واحد وعمود واحد، لا يتم إنشاء أي نسخ — يخرج الأمر دون تغيير الرسم.

## مرجع لوحة المفاتيح

| المفتاح | الإجراء |
|---------|---------|
| `Enter` / `Space` | تأكيد التحديد والانتقال إلى مطالبة الصفوف |
| `0`–`9` | إدخال أرقام للصفوف أو الأعمدة |
| `0`–`9`، `.`، `-` | إدخال أرقام لمسافة الصفوف/الأعمدة (`-` كحرف أول فقط) |
| `Backspace` | حذف آخر حرف مكتوب في المطالبة الحالية |
| `Enter` | تأكيد المطالبة الحالية والانتقال إلى التالية |
| `Escape` | مسح قيم الصفوف/الأعمدة/المسافة المكتوبة والعودة إلى مرحلة التحديد |

## التحديد أثناء الأمر

| الطريقة | السلوك |
|---------|--------|
| **النقر** | يبدّل العنصر تحت المؤشر داخل/خارج التحديد |
| **السحب يميناً** (صارم) | يُضيف العناصر الواقعة بالكامل داخل المربع |
| **السحب يساراً** (متقاطع) | يُضيف العناصر التي تتقاطع مع حدود المربع |
| **Enter** / **Space** | يؤكد التحديد وينتقل إلى مطالبة الصفوف |

## بعد Array

تُضاف النسخ الجديدة إلى الرسم ويخرج الأمر — يُمسح التحديد الأصلي. شغِّل **Array Grid** مرة أخرى، أو ابدأ أمراً جديداً.

## Array Grid مقابل Copy

| | Array Grid | Copy |
|---|-----------|------|
| اختيار النقاط | لا يوجد — الصفوف والأعمدة والمسافة تُكتب | يتم النقر (أو الكتابة) على نقطة الأساس والوجهة |
| النسخ المُنشأة | الصفوف × الأعمدة − 1 | نسخة واحدة بالضبط لكل عملية نسخ |
| التخطيط | شبكة مستطيلة منتظمة | في أي مكان، بأي إزاحة |
| الأنسب لـ | تكرار وحدة في نمط منتظم (ثقوب، بلاط، مثبتات) | نسخة واحدة في موضع عشوائي |

## العناصر المدعومة

يعمل Array Grid مع كل نوع عنصر. جميع العناصر تُطبِّق `translate(dx, dy)` داخلياً — نفس العملية التي يستخدمها [Copy](../copy/) و[Move](../move/) — لذا لا يُستثنى منها أي شيء.
