# ملف جديد

يمسح أمر **New File** اللوحة ويبدأ رسماً فارغاً جديداً. يُنشأ اسم ملف فريد بطابع زمني تلقائياً.

## كيفية إنشاء ملف جديد

انقر على زر **New File** في شريط الأدوات (أيقونة صفحة جديدة) في لوحة الملف. تُمسح اللوحة فوراً — لا توجد موجهات أو مربعات حوار تأكيد.

## ما يحتويه الملف الجديد

يبدأ الملف المُنشأ حديثاً بما يلي:

- **لا عناصر** على اللوحة.
- **طبقة افتراضية واحدة** باسم `0` بلون أبيض ونوع خط `Continuous`.
- **اسم ملف مُنشأ** بتنسيق `kulman-Mon01_HH:MM:SS.dxf` (مثل `kulman-May22_14:30:00.dxf`).

يُحفظ الملف في تخزين المتصفح تلقائياً ويظهر في [الملفات الأخيرة](./files).

## تحذير — تُفقد الأعمال غير المحفوظة

النقر على **New File** يتجاهل جميع العناصر الموجودة على اللوحة الحالية دون تحذير. إذا أردت الاحتفاظ بالرسم الحالي، [صدِّره](./export) أولاً.

## متى تستخدم New File مقابل Import

| الحالة | الإجراء الموصى به |
|--------|-----------------|
| بدء رسم من الصفر | **New File** |
| فتح ملف DXF أو JSON موجود | [Import](./import) |
| نسخ رسم للعمل على نسخة بديلة | [صدِّر](./export) الملف الحالي، ثم [استورد](./import) النسخة |

## الأوامر ذات الصلة

- [Export](./export) — تنزيل الرسم قبل البدء من جديد
- [Files](./files) — استعادة رسم سابق من تخزين المتصفح
