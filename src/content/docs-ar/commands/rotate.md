# التدوير

يُدوِّر أمر `rotate` العناصر المحددة حول نقطة أساس. تُحدِّد زاوية الدوران إما بكتابة رقم بالدرجات أو بالنقر — تُحسَب الزاوية من الاتجاه بين نقطة الأساس وموضع النقر.

## طريقتان للبدء

**التحديد المسبق ثم التدوير** — حدد العناصر أولاً ثم فعِّل الأمر:

1. حدد عنصراً أو أكثر على اللوحة.
2. اكتب `rotate` في المحطة أو انقر على زر **Rotate** في شريط الأدوات.
3. **انقر على نقطة الأساس** — مركز الدوران. أو اكتب `X,Y` واضغط **Enter** للإحداثي الدقيق.
4. **اكتب زاوية واضغط Enter**، أو **انقر** لتعيين الزاوية من اتجاه المؤشر.

**التفعيل ثم التحديد** — ابدأ الأمر مع عدم تحديد أي شيء:

1. اكتب `rotate` أو انقر على زر شريط الأدوات.
2. **حدد الكائنات** — انقر للتبديل، أو اسحب للتحديد بالمنطقة.
3. اضغط **Enter** أو **Space** لتأكيد التحديد.
4. **انقر على نقطة الأساس** (إدخال الإحداثيات متاح)، ثم عيِّن الزاوية.

```
  قبل:            بعد (90° دوران حول ●):
                     ╔══╗
  ●  [عنصر]   →   ● ║    ║
                     ╚══╝
```

تتبع معاينة شبحية حية للعناصر المُدارة زاوية المؤشر بعد تعيين نقطة الأساس.

## تعيين الزاوية

**الزاوية المكتوبة** — اكتب رقماً (بالدرجات) في أي وقت بعد وضع نقطة الأساس. تُلتقط المعاينة على الزاوية المكتوبة بينما تستمر في الضبط قبل الضغط على Enter.

**النقر للزاوية** — إذا لم تكن هناك قيمة مكتوبة، يُعيِّن النقر الزاوية مساويةً لـ`atan2(cursorY − baseY, cursorX − baseX)` — الاتجاه من نقطة الأساس إلى موضع النقر، بالدرجات.

| المفتاح | الإجراء |
|---------|---------|
| `0`–`9`، `.` | إلحاق رقم بقيمة الزاوية |
| `-` | زاوية سالبة (الحرف الأول فقط) |
| `Backspace` | حذف آخر حرف مكتوب |
| `Enter` | تطبيق الدوران على الزاوية المكتوبة |

## اتجاه الزاوية

تتبع الزوايا **اتفاقية DXF**:

- القيم **الموجبة** تُدوِّر **عكس اتجاه عقارب الساعة** في إحداثيات الرسم (Y للأعلى).
- على الشاشة، حيث المحور Y معكوس (Y للأسفل)، تبدو الزوايا الموجبة في **اتجاه عقارب الساعة**.

قيم شائعة: `90` = ربع دوران، `180` = نصف دوران، `-90` = ربع دوران معاكس.

## مرجع لوحة المفاتيح

| المفتاح | الإجراء |
|---------|---------|
| `Enter` / `Space` | تأكيد التحديد |
| `0`–`9`، `.`، `-` | بدء إدخال إحداثي X (مرحلة نقطة الأساس)، أو قيمة الزاوية (مرحلة الزاوية) |
| `,` | تثبيت X والانتقال إلى إدخال Y (مرحلة نقطة الأساس) |
| `Backspace` | حذف آخر حرف مكتوب |
| `Enter` | تأكيد الإحداثي أو تطبيق الدوران |
| `Escape` | إلغاء وإعادة التعيين |

## التحديد أثناء الأمر

| الطريقة | السلوك |
|---------|--------|
| **النقر** | يبدّل العنصر تحت المؤشر |
| **السحب يميناً** (صارم) | يُضيف العناصر الواقعة بالكامل داخل المربع |
| **السحب يساراً** (متقاطع) | يُضيف العناصر التي تتقاطع مع المربع |
| **Enter** / **Space** | يؤكد التحديد |

## العناصر المدعومة

يعمل التدوير على كل نوع عنصر. تُدار هندسة كل عنصر حول نقطة الأساس — على سبيل المثال، الدائرة تنقل مركزها بينما يبقى نصف القطر ذاته؛ القوس ينقل مركزه ويُزيح زاويتَي البداية والنهاية بمقدار الدوران؛ عنصر النص ينقل نقطة تثبيته ويُضيف الزاوية إلى خاصية Rotation Degree.
