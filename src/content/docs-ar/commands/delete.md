# الحذف

يحذف أمر `delete` العناصر المحددة من الرسم. تُسجَّل عمليات الحذف في سجل [Undo](./undo) ويمكن عكسها بما يصل إلى 20 خطوة. لا يوجد حوار "تأكيد الحذف" منفصل — التأكيد بضغطة مفتاح واحدة.

## طريقتان للحذف

**التحديد المسبق ثم الحذف** — أسرع مسار:

1. حدد عنصراً أو أكثر على اللوحة.
2. اكتب `delete` في المحطة، أو انقر على زر **Delete** في شريط الأدوات، **أو اضغط مفتاح `Delete`** مباشرةً.

تُزال العناصر فوراً — دون خطوة تأكيد إضافية.

**التفعيل ثم التحديد**:

1. اكتب `delete` أو انقر على زر شريط الأدوات (مع عدم تحديد أي شيء).
2. **حدد الكائنات** — انقر للتبديل، أو اسحب للتحديد بالمنطقة.
3. اضغط **Enter** أو **Space** أو **Delete** لتأكيد وحذف العناصر المحددة.

## اختصار مفتاح Delete

يعمل مفتاح `Delete` على لوحة المفاتيح كـ**اختصار عام** — إذا كانت أي عناصر محددة حالياً، فإن ضغطه يحذفها فوراً، حتى دون فتح أمر Delete في المحطة. هذا أسرع سير عمل للحذف في خطوة واحدة:

```
انقر على عنصر → اضغط مفتاح Delete → تم
```

## التحديد أثناء الأمر

| الطريقة | السلوك |
|---------|--------|
| **النقر** | يبدّل العنصر تحت المؤشر داخل/خارج التحديد |
| **السحب يميناً** (صارم) | يُحدِّد العناصر الواقعة بالكامل داخل المربع فقط |
| **السحب يساراً** (متقاطع) | يُحدِّد العناصر التي تتقاطع مع حدود المربع |
| **Enter** / **Space** / **Delete** | يؤكد ويحذف العناصر المحددة |

## استرداد العناصر المحذوفة

عمليات الحذف قابلة للتراجع بأمر [Undo](./undo) (اكتب `undo` أو استخدم زر شريط الأدوات). يمكن عكس ما يصل إلى **20 خطوة** لكل ملف، ويستمر السجل عبر إعادات تحميل الصفحة. إذا كنت قد تجاوزت 20 عملية حذف دون حفظ، لا يمكن استرداد عمليات الحذف الأقدم.

## العناصر المدعومة

يعمل الحذف على كل نوع عنصر — Line وPolyline وRectangle وCircle وArc وEllipse وText وSpline وDimension وLeader وغيرها.
