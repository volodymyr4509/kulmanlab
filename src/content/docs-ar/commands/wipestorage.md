# مسح التخزين

يحذف أمر `wipestorage` نهائياً **جميع البيانات المخزَّنة في المتصفح** لتطبيق KulmanLab CAD — كل ملف محفوظ وجداول الطبقات وأنواع الخطوط وسجل التراجع. تُعاد تحميل الصفحة تلقائياً بعد ذلك.

:::danger لا رجعة فيه
لا يمكن التراجع عن هذا الإجراء. تُحذف جميع الملفات المخزَّنة في المتصفح. صدِّر أي رسومات تريد الاحتفاظ بها كملفات `.json` أو `.dxf` قبل تشغيل هذا الأمر.
:::

## متى تستخدمه

- تخزين المتصفح تالف والتطبيق يفشل في التحميل أو حفظ الملفات.
- تريد إعادة تعيين التطبيق بالكامل إلى حالة نظيفة.
- تغيِّر المتصفح أو الجهاز ولم تعد تحتاج إلى النسخة المحلية.

## كيفية تشغيله

1. اكتب `wipestorage` في المحطة واضغط **Enter**.
2. تطلب المحطة: *Wipe all browser local storage? Type YES to confirm*
3. اكتب `YES` (بأي تهجئة) واضغط **Enter**.

يحذف التطبيق قاعدة البيانات ويُعيد تحميل الصفحة. إذا كتبت أي شيء غير `YES` وضغطت **Enter**، أو ضغطت **Escape**، يُلغى الأمر ولا يُحذف أي شيء.

## ما يُحذف

| البيانات | مُحذوف |
|---------|-------|
| جميع الملفات المحفوظة في المتصفح | نعم |
| جداول الطبقات وأنواع الخطوط لكل ملف | نعم |
| سجل التراجع/الإعادة لكل ملف | نعم |

تتأثر فقط البيانات المخزَّنة محلياً في **هذا المتصفح**. الملفات التي صدَّرتها بالفعل بصيغة `.json` أو `.dxf` لا تتأثر.
