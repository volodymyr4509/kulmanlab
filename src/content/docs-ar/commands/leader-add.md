# Leader+

يُضيف أمر `leader+` ذراع سهم جديدة إلى خط إشارة متعدد موجود. تمتد الذراع الجديدة من مرفق خط الإشارة الحالي إلى رأس سهم جديد تنقر عليه. يُوروَث جميع التنسيق — موضع المرفق والنص ونوع السهم وحجمه — من خط الإشارة المحدد.

## إضافة ذراع

1. اكتب `leader+` في المحطة.
2. **انقر على خط إشارة متعدد موجود** لتحديده.
3. **انقر على رأس السهم الجديد**، أو اكتب `X,Y` واضغط **Enter** للإحداثي الدقيق. خط معاينة يُظهر من المؤشر إلى مرفق خط الإشارة.

تُوضع الذراع ويبقى الأمر نشطاً — يمكنك فوراً النقر على خط إشارة آخر لإضافة ذراع أخرى. اضغط **Enter** أو **Space** أو **Escape** للإنهاء.

```
  قبل:                        بعد:
  ◄── الذراع 1                ◄── الذراع 1
       \                             \
        ●──── المرفق ──── النص         ●──── المرفق ──── النص
                                      /
                                 الذراع 2 ──►  (الرأس الجديد الذي نقرت عليه)
```

## إدخال الإحداثيات للرأس

بدلاً من النقر، يمكنك كتابة موضع دقيق:

1. اكتب قيمة X.
2. اضغط `,` — تُؤكِّد المحطة تثبيت X.
3. اكتب قيمة Y.
4. اضغط **Enter** للوضع.

## مرجع لوحة المفاتيح

| المفتاح | الإجراء |
|---------|---------|
| `0`–`9`، `.`، `-` | بدء كتابة إحداثي X |
| `,` | تثبيت X والانتقال إلى إدخال Y |
| `Enter` | تأكيد الإحداثي المكتوب ووضع الذراع |
| `Enter` / `Space` | إنهاء (عندما لا يكون هناك إدخال جارٍ) |
| `Escape` | إلغاء وإعادة التعيين |

## ملاحظات

- فقط عناصر **خط الإشارة المتعدد** يمكن تحديدها — النقر على أي نوع عنصر آخر لا يفعل شيئاً.
- الذراع الجديدة تنطلق من المرفق الحالي؛ أنت فقط تختار موضع رأس السهم.
- لا يوجد حد لعدد الأذرع التي يمكن لخط إشارة متعدد أن يملكها.
