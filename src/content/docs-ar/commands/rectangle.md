# Rectangle

يرسم أمر `rectangle` مستطيلاً موازياً للمحاور محدداً بنقرتين على زاويتين متقابلتين. تُخزَّن النتيجة كـ**`LWPOLYLINE` مغلق** بأربعة رؤوس — رأس عند كل زاوية. لا يوجد نوع كيان مخصص للمستطيل: بعد الإنشاء يتصرف الشكل تماماً مثل أي [Polyline](../polyline/) آخر وتنطبق عليه جميع عمليات تحرير الخط المتعدد.

## رسم مستطيل

1. اكتب `rectangle` في المحطة أو انقر على زر **Rectangle** في شريط الأدوات.
2. **انقر على الزاوية الأولى**، أو اكتب `X,Y` واضغط **Enter** للإحداثي الدقيق.
3. **انقر على الزاوية المقابلة** — يُوضع المستطيل فوراً ويخرج الأمر. إدخال الإحداثيات يعمل هنا أيضاً.

```
  ● (النقرة الأولى)────────────────┐
  |                                 |
  |   معاينة حية تتبع              |
  |   المؤشر بعد الخطوة 2          |
  └─────────────────────────────────● (النقرة الثانية)
```

يمكن أن تكون النقرتان أي زوج من الزوايا المتقابلة قطرياً — أعلى اليسار + أسفل اليمين، أو أسفل اليسار + أعلى اليمين، إلخ. الترتيب لا يهم.

## إدخال الإحداثيات

في خطوة أي زاوية يمكنك كتابة موضع دقيق:

1. اكتب قيمة X.
2. اضغط `,` — تُظهر المحطة `[X], [Y{cursor}]`.
3. اكتب قيمة Y.
4. اضغط **Enter** لوضع الزاوية.

## مرجع لوحة المفاتيح

| المفتاح | الإجراء |
|---------|---------|
| `0`–`9`، `.`، `-` | بدء إدخال إحداثي X |
| `,` | تثبيت X والانتقال إلى إدخال Y |
| `Enter` | تأكيد الإحداثي المكتوب |
| `Escape` | إلغاء |

الأضلاع تكون دائماً أفقية ورأسية — لا يوجد تثبيت زاوية لأمر Rectangle.

## تحرير نقاط التحكم — إعادة التشكيل بعد الإنشاء

يُظهر المستطيل المحدد نقاط تحكم عند كل رأس وعند منتصف كل ضلع:

| نقطة التحكم | الموضع | ما تفعله |
|------------|--------|---------|
| **الزاوية** | كل من الرؤوس الأربعة | اسحب لتحريك هذا الرأس؛ يمتد الضلعان المجاوران لمتابعته — تبقى الزاوية المقابلة ثابتة |
| **منتصف الضلع** | مركز كل من الأضلاع الأربعة | اسحب لنقل كلتا نقطتَي نهاية ذلك الضلع معاً، مع الحفاظ على طول الضلع واتجاهه |

سحب نقطة تحكم الزاوية يحوّل المستطيل إلى شبه منحرف غير مستطيل. إذا كنت تحتاج فقط مستطيلاً بحجم مختلف، اسحب الزاوية مع إبقاء الأضلاع متعامدة تقريباً، أو احذفه وارسم واحداً جديداً.

## تحديد المستطيلات

لأن المستطيل خط متعدد، يعمل التحديد بالطريقة ذاتها:

| الطريقة | السلوك |
|---------|--------|
| **النقر** | يُحدَّد إذا وقع النقر على أي من الأضلاع الأربعة |
| **السحب يميناً** (صارم) | يجب أن تقع الرؤوس الأربعة داخل مربع التحديد |
| **السحب يساراً** (متقاطع) | أي ضلع يتقاطع مع حدود المربع يُحدِّد المستطيل بأكمله |

## أوامر التحرير المدعومة

تنطبق جميع أوامر تحرير الخط المتعدد. القص والتمديد خاصان بـ[Line](../line/) ولا يعملان على المستطيلات:

| الأمر | ما يحدث للمستطيل |
|-------|----------------|
| [Move](../move/) | يُزيح الرؤوس الأربعة بالإزاحة ذاتها |
| [Copy](../copy/) | يُنشئ مستطيلاً مطابقاً في موضع جديد |
| [Rotate](../rotate/) | يُدوِّر الرؤوس الأربعة حول نقطة الأساس المختارة |
| [Mirror](../mirror/) | يعكس الرؤوس الأربعة عبر محور الانعكاس |
| [Scale](../scale/) | يُغيِّر مقياس الرؤوس الأربعة بشكل موحد من نقطة الأساس |
| [Offset](../offset/) | يُنشئ مستطيلاً موازياً (داخلي أو خارجي) على مسافة ثابتة |
| [Delete](../delete/) | يحذف المستطيل من الرسم |

## الخصائص

عند تحديد مستطيل، يُظهر لوح الخصائص الحقول ذاتها كأي خط متعدد:

**عام**

| الخاصية | الافتراضي | المعنى |
|---------|----------|--------|
| اللون | 256 (ByLayer) | فهرس لون ACI |
| الطبقة | `0` | تعيين الطبقة |
| نوع الخط | ByLayer | نمط نوع الخط المسمى |
| مقياس نوع الخط | 1 | عامل التحجيم على نمط نوع الخط |
| السُّمك | 0 | سُمك البثق |

**الهندسة**

| الخاصية | المعنى |
|---------|--------|
| Closed | دائماً `true` للمستطيل |
| Vertex Count | دائماً `4` للمستطيل غير المعدَّل |
| Vertices | إحداثيات الزوايا الأربع |

## المستطيل مقابل الخط المتعدد مقابل الخط

| | المستطيل | الخط المتعدد | الخط |
|---|-----------|---------|------|
| طريقة الرسم | نقرتان (زوايا) | انقر كل رأس | انقر كل نقطة نهاية |
| نوع الكيان | `LWPOLYLINE` مغلق | `LWPOLYLINE` مفتوح أو مغلق | `LINE` لكل مقطع |
| الأضلاع متعامدة دائماً | نعم (عند الإنشاء) | لا | لا |
| Trim / Extend | لا | لا | نعم |
| الأنسب لـ | الصناديق والأطر والمناطق المستطيلة | المحيطات والمسارات التعسفية | المقاطع المنفردة وخطوط الإنشاء |

## DXF — عنصر LWPOLYLINE

تُحفظ المستطيلات كعناصر `LWPOLYLINE` مغلقة بأربعة رؤوس. جميع الخصائص — إحداثيات الرؤوس واللون والطبقة ونوع الخط ومقياسه والسُّمك — تنتقل دون خسارة.

لا يوجد نوع `RECTANGLE` مخصص في DXF. عند إعادة فتح الملف، يظهر الشكل كخط متعدد مغلق بأربعة رؤوس وليس كمستطيل. أي مشاهد أو محرر DXF يدعم `LWPOLYLINE` (LibreCAD وFreeCAD وغيرهما) سيعرضه بشكل صحيح.
