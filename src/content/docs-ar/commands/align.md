# Align

يعيد أمر `align` وضع العناصر المحددة باستخدام زوج واحد أو زوجين من نقاط المصدر/الوجهة. بزوج واحد، يعمل تماماً مثل [Move](../move/) (نقل فقط). بزوجين، يقوم أيضاً بتدوير التحديد بحيث يتطابق اتجاه المصدر-إلى-المصدر مع اتجاه الوجهة-إلى-الوجهة، ويمكنه اختيارياً تحجيمه بحيث يتطابق طول مقطع المصدر مع طول مقطع الوجهة — نقل وتدوير وتحجيم في عملية واحدة.

## طريقتان للبدء

**التحديد المسبق ثم المحاذاة** — حدد العناصر أولاً ثم فعِّل الأمر:

1. حدد عنصراً أو أكثر على اللوحة.
2. اكتب `align` في المحطة أو انقر على زر **Align** في شريط الأدوات.
3. **انقر على أول نقطة مصدر (S1)**، ثم **على أول نقطة وجهة (D1)**.
4. **انقر على ثاني نقطة مصدر (S2)**، أو اضغط **Enter** لتطبيق محاذاة نقل فقط الآن.
5. **انقر على ثاني نقطة وجهة (D2)**.
6. أجب عن مطالبة التحجيم: اضغط **Y** للتحجيم، أو **N** / **Enter** للحفاظ على الحجم الأصلي.

**التفعيل ثم التحديد** — ابدأ الأمر مع عدم تحديد أي شيء:

1. اكتب `align` أو انقر على زر شريط الأدوات.
2. **حدد الكائنات** — انقر لتبديل تحديد العناصر المنفردة، أو اسحب للتحديد بالمنطقة.
3. اضغط **Enter** أو **Space** لتأكيد التحديد.
4. تابع بـ S1 → D1 → S2 → D2 → مطالبة التحجيم كما أعلاه.

> تحتاج المحطة فقط إلى عدد الأحرف الكافي لعدم الالتباس — كتابة `al` والضغط على **Enter** تُفعِّل Align مباشرة، لأنه لا يوجد أمر آخر يبدأ بهذين الحرفين.

## تشريح محاذاة

```
  نقاط المصدر (على العناصر):           نقاط الوجهة:
  ● S1                                 ● D1
   \                                    \
    ● S2                                 ● D2

  النتيجة: يُنقل التحديد بحيث تحطّ S1 على D1، ثم يُدار حول D1
  بحيث يتطابق اتجاه S1→S2 مع اتجاه D1→D2 — وإذا اخترت التحجيم،
  يُعاد ضبط الحجم بحيث يصبح |S1S2| مساوياً لـ |D1D2|.
```

تتبع معاينة شبحية المؤشرَ في كل خطوة: معاينة نقل أثناء وضع D1، ثم معاينة مدورة (متقطعة) بمجرد أن يبدأ تحديد موضع D2.

## محاذاة بنقطة واحدة (نقل فقط)

بعد وضع D1، اضغط **Enter** بدلاً من النقر على نقطة مصدر ثانية. يُنقل التحديد بمقدار الشعاع S1→D1 — دون تدوير أو تحجيم — مطابق تماماً لأمر [Move](../move/) باستخدام S1 كنقطة أساس وD1 كوجهة.

## محاذاة بنقطتين (نقل + تدوير + تحجيم اختياري)

بمجرد وضع كل من S2 وD2:

- **زاوية التدوير** — الفرق بين اتجاه الوجهة (`D1 → D2`) واتجاه المصدر (`S1 → S2`).
- **مطالبة التحجيم** — تظهر `scale objects to alignment points? [Yes/No] <N>`، مع **No** كافتراضي:
  - اضغط **Y** لتحجيم التحديد أيضاً بشكل موحد حول D1 بحيث تصبح المسافة `S1–S2` هي المسافة `D1–D2`.
  - اضغط **N** أو **Enter** للحفاظ على الحجم الأصلي — يُطبَّق النقل والتدوير فقط.

الضغط على أي مفتاح عند مطالبة التحجيم يُطبِّق المحاذاة فوراً — لا توجد خطوة تأكيد منفصلة بعد اختيار نعم أو لا.

## مرجع لوحة المفاتيح

| المفتاح | الإجراء |
|---------|---------|
| `Enter` / `Space` | تأكيد التحديد والانتقال إلى مرحلة S1 |
| `Enter` (في خطوة S2) | تخطي التدوير — تطبيق محاذاة نقل فقط باستخدام S1 وD1 |
| `Y` | تطبيق المحاذاة مع التحجيم |
| `N` / `Enter` (عند مطالبة التحجيم) | تطبيق المحاذاة دون تحجيم |
| `Escape` | أثناء وضع النقاط: يتجاهل النقاط ويعود إلى مرحلة التحديد؛ مع عدم وجود تحديد: يلغي الأمر |

## التحديد أثناء الأمر

عند بدء الأمر في مرحلة التحديد:

| الطريقة | السلوك |
|---------|--------|
| **النقر** | يبدّل العنصر تحت المؤشر داخل/خارج التحديد |
| **السحب يميناً** (صارم) | يُضيف العناصر الواقعة بالكامل داخل المربع |
| **السحب يساراً** (متقاطع) | يُضيف العناصر التي تتقاطع مع حدود المربع |
| **Enter** / **Space** | يؤكد التحديد وينتقل إلى مرحلة S1 |

## بعد المحاذاة

تبقى العناصر المحاذاة محددة في موضعها الجديد، وينتهي الأمر تلقائياً — شغِّل **Align** مرة أخرى، أو انتقل إلى [Move](../move/) أو [Rotate](../rotate/) أو [Scale](../scale/) دون إعادة التحديد.

## Align مقابل Move

| | Align | Move |
|---|-------|------|
| أزواج النقاط | 1 (نقل فقط) أو 2 (نقل + تدوير + تحجيم) | 1 (نقل فقط) |
| التدوير | نعم، بزوج نقاط ثانٍ | لا |
| التحجيم | اختياري، بزوج نقاط ثانٍ | لا |
| الأنسب لـ | مطابقة شكل على آخر باستخدام نقاط مرجعية | إعادة تموضع بسيطة |

## العناصر المدعومة

يعمل Align على كل نوع عنصر تدعمه Move وRotate وScale — تُطبَّق بالتتابع نفس عمليات `translate` و`rotate` و`scale` التي تستخدمها تلك الأوامر، لذا لا يُستثنى منها أي شيء.
