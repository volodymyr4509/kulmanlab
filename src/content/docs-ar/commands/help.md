# Help

يفتح لوح المساعدة داخل التطبيق الذي يحتوي على ملخص لجميع اختصارات لوحة المفاتيح ونصائح التنقل والاستخدام العام.

## كيفية الاستخدام

انقر على زر **Help** في شريط الأدوات أو اكتب `help` في المحطة. يفتح لوح المساعدة فوراً ويخرج الأمر.

## محتويات لوح المساعدة

### التنقل
| الاختصار | الإجراء |
|---------|---------|
| `عجلة التمرير` | التكبير / التصغير |
| `نقر بالزر الأوسط + سحب` | تحريك إطار العرض |
| `نقر مزدوج بالزر الأوسط` | احتواء جميع العناصر في الشاشة |

### الأوامر
| الاختصار | الإجراء |
|---------|---------|
| الكتابة | ابدأ بكتابة اسم الأمر — تظهر الاقتراحات على اللوحة |
| `Delete` | حذف جميع العناصر المحددة |
| `Enter` / `Space` | تأكيد الأمر أو الإدخال |
| `Enter` / `Space` (في وضع الخمول) | تكرار آخر أمر مُستخدَم |
| `Escape` | إلغاء الأمر الحالي أو مسح التحديد |

### التحديد
| الاختصار | الإجراء |
|---------|---------|
| `نقر بالزر الأيسر` | تحديد عنصر |
| `سحب →` | تحديد العناصر **الموجودة بالكامل** داخل منطقة السحب فقط |
| `سحب ←` | تحديد أي عنصر **يتقاطع** مع منطقة السحب |
| `نقر بالزر الأيسر على نقطة تحكم` | تعديل رأس أو نقطة نهاية مباشرةً |

### نصائح الرسم
- **نقاط الالتقاط** — يلتقط المؤشر تلقائياً بنقاط النهاية القريبة للعناصر.
- **الزاوية الثابتة** — حرِّك المؤشر بعيداً بما يكفي في اتجاه واحد للقفل على مضاعفات 45°.
- **الطول الدقيق** — بينما الزاوية مقفلة، اكتب رقماً واضغط `Enter` لتحديد مسافة دقيقة.

## الإصدار

يُعرض إصدار التطبيق الحالي في الزاوية العلوية اليسرى من لوح المساعدة (مثل `KULMAN 2026.04.15`).
