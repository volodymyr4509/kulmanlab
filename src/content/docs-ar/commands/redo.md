# الإعادة

يتقدم أمر `redo` للأمام عبر سجل التراجع، مُعيداً تطبيق الإجراءات التي عُكست بواسطة [Undo](./undo). الإعادة متاحة فقط عندما تكون قد تراجعت بـUndo ولم تُجرِ بعدُ أي تغيير جديد.

## كيفية الإعادة

- اكتب `redo` في المحطة، أو
- انقر على زر **Redo** في شريط الأدوات.

كل استدعاء يُعيد تطبيق إجراء مُتراجَع عنه واحداً سابقاً. استدعِه مراراً للتقدم عبر جميع إدخالات الإعادة المتاحة.

## سلوك مكدس الإعادة

| التفصيل | السلوك |
|---------|--------|
| متاح بعد | خطوة [Undo](./undo) واحدة أو أكثر |
| يُمسح بواسطة | **أي إجراء رسم جديد** — إضافة أو تحرير أو حذف عنصر |
| التخزين | المتصفح، لكل ملف — يستمر عبر إعادة التحميل (طالما لم يُتخَذ أي إجراء جديد قبل إعادة التحميل) |
| أقصى عمق | حتى 20 إدخالاً (المجموعة ذاتها مع Undo) |

حالما يُرسم عنصر جديد أو يُحذف أو يُعدَّل، يُمسح مكدس الإعادة ولا يمكن استرداد تلك الإدخالات. فقط الإجراءات المُتراجَع عنها التي لم تُستبدَل بعمل جديد يمكن إعادتها.

## الإعادة مقابل التراجع

| | الإعادة | التراجع |
|---|------|------|
| الاتجاه | يتقدم **للأمام** عبر الإدخالات المُتراجَع عنها | يتقدم **للخلف** عبر السجل |
| متاح عندما | بعد تراجع واحد على الأقل، مع عدم اتخاذ أي إجراء جديد | يوجد إجراء مسجَّل واحد على الأقل |
| يُمسح بواسطة | أي إجراء رسم جديد | لا شيء |

زر الإعادة في شريط الأدوات يكون معتماً عندما لا توجد إدخالات لإعادتها. استخدم [Undo](./undo) أولاً لإنشاء إدخالات إعادة.
