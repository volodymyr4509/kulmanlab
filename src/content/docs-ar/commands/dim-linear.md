# البُعد الخطي

يضع أمر `dimlinear` بُعداً أفقياً أو رأسياً بين نقطتَي أصل خطي تمديد. يعمل خط البُعد دائماً إما أفقياً تماماً أو رأسياً تماماً — لا يمكن وضعه بزاوية تعسفية. استخدم [البُعد المحاذي](../dim-aligned/) عندما تحتاج بُعداً موازياً لخط قطري.

## تشريح البُعد الخطي

```
  |←————— 5.00 —————→|
  |                   |
  ●  (خط تمديد 1)     ●  (خط تمديد 2)
  p1                  p2
```

- **خطوط التمديد** — تنخفض من كل نقطة مقيسة عمودياً على خط البُعد.
- **خط البُعد** — أفقي (يقيس مسافة X) أو رأسي (يقيس مسافة Y).
- **القيمة** — المسافة المُسقطة على المحور المختار، لا المسافة الفعلية من نقطة إلى نقطة.

## وضع بُعد خطي

1. اكتب `dimlinear` في المحطة أو انقر على زر **Dimension Linear** في شريط الأدوات.
2. **انقر على أصل خط التمديد الأول** (p1)، أو اكتب `X,Y` واضغط **Enter** للإحداثي الدقيق.
3. **انقر على أصل خط التمديد الثاني** (p2). إدخال الإحداثيات يعمل هنا أيضاً.
4. **حرّك المؤشر** لتموضع خط البُعد. يُكشف الاتجاه تلقائياً من موضع المؤشر.
5. **انقر** للوضع، أو اكتب مسافة إزاحة واضغط **Enter** للوضع الدقيق.

## الكشف التلقائي للاتجاه

عندما لا يُفرض اتجاه، يقرأ الأمر موضع المؤشر نسبةً إلى النقطتين المقيستين:

| موضع المؤشر | الاتجاه المكشوف | ما يُقاس |
|------------|----------------|---------|
| فوق أو أسفل النقاط | أفقي | Δ X بين p1 وp2 |
| يسار أو يمين النقاط | رأسي | Δ Y بين p1 وp2 |

اضغط **H** لتثبيت الأفقي أو **V** لتثبيت الرأسي في أي وقت أثناء مرحلة الوضع. بمجرد التثبيت، لا يتغير الاتجاه مع تحريك المؤشر.

## مسافة الإزاحة المكتوبة

اكتب رقماً أثناء الوضع لتثبيت خط البُعد على مسافة دقيقة من النقاط المقيسة:

| المفتاح | الإجراء |
|---------|---------|
| `0`–`9`، `.` | إلحاق رقم بمسافة الإزاحة |
| `Backspace` | حذف آخر حرف مكتوب |
| `Enter` / `Space` | وضع على المسافة المكتوبة |

الجانب الذي يقع فيه المؤشر (فوق/أسفل للأفقي، يسار/يمين للرأسي) يحدد الإشارة — يظهر خط البُعد على الجانب الذي يوجد فيه المؤشر حالياً.

## مرجع لوحة المفاتيح

| المفتاح | الإجراء |
|---------|---------|
| `0`–`9`، `.`، `-` | بدء إدخال إحداثي X (مراحل p1/p2)، أو مسافة الإزاحة (مرحلة الوضع) |
| `,` | تثبيت X والانتقال إلى إدخال Y (مراحل p1/p2) |
| `H` | تثبيت الاتجاه الأفقي (مرحلة الوضع فقط) |
| `V` | تثبيت الاتجاه الرأسي (مرحلة الوضع فقط) |
| `Backspace` | حذف آخر حرف مكتوب |
| `Enter` / `Space` | تأكيد الإحداثي أو الإزاحة المكتوبة |
| `Escape` | إلغاء |

## البُعد الخطي مقابل البُعد المحاذي

| | البُعد الخطي | البُعد المحاذي |
|---|-----------------|------------------|
| المحور | دائماً H أو V | موازٍ للخط المقيس |
| يقيس | مكوّن X أو Y فقط | المسافة الإقليدية الفعلية |
| مفاتيح H/V | نعم — تثبيت الاتجاه | لا — يتبع دائماً p1→p2 |
| الأنسب لـ | التخطيطات المتعامدة والمسقط الأرضي | الميزات القطرية والقطوع المائلة |

## تحرير التسمية — الوضع البسيط

**انقر مزدوجاً** على بُعد خطي موضوع لفتح محرر النص في الوضع **البسيط**. يُملأ المحرر مسبقاً بالقيمة المعروضة الحالية حتى تتمكن من تموضع المؤشر وتحريرها مباشرةً.

| الميزة | السلوك |
|--------|--------|
| Bold / Italic / Font / Height | تُطبَّق على **التسمية بأكملها** دفعةً واحدة |
| التنسيق لكل حرف | غير مدعوم |
| `Enter` | يُثبِّت القيمة ويغلق المحرر |
| متعدد الأسطر | غير مدعوم |

راجع [محرر النص — الوضع البسيط](../interface/text-editor#simple-mode) للمرجع الكامل.

## تسلسل الأبعاد

لإضافة أبعاد أخرى مستمرة من آخر خط تمديد، استخدم [Dimension Continue](../dim-continue/) مباشرةً بعد وضع هذا البُعد.

## DXF — عنصر DIMENSION

تُحفظ الأبعاد الخطية كعناصر `DIMENSION` مع تعيين `rotationDeg` على `0` (أفقي) أو `90` (رأسي). أصول خطوط التمديد وموضع خط البُعد وموضع النص والقيمة المقيسة ونمط السهم وارتفاع النص وجميع علامات العرض تنتقل دون خسارة.
