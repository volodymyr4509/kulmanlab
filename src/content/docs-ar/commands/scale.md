# Scale

يُغيِّر أمر `scale` حجم العناصر المحددة بشكل موحد حول نقطة أساس. تُضرب جميع المسافات من نقطة الأساس في عامل المقياس — عامل `2` يُضاعف جميع الأبعاد، `0.5` يُنصِّفها. العامل يُدخَل دائماً بالكتابة؛ لا يوجد تعيين المقياس بالنقر.

## طريقتان للبدء

**التحديد المسبق ثم تغيير المقياس** — حدد العناصر أولاً ثم فعِّل الأمر:

1. حدد عنصراً أو أكثر على اللوحة.
2. اكتب `scale` في المحطة أو انقر على زر **Scale** في شريط الأدوات.
3. **انقر على نقطة الأساس** — النقطة الثابتة التي لا تتحرك أثناء تغيير المقياس. أو اكتب `X,Y` واضغط **Enter** للإحداثي الدقيق.
4. **اكتب عامل المقياس** واضغط **Enter**.

**التفعيل ثم التحديد** — ابدأ الأمر مع عدم تحديد أي شيء:

1. اكتب `scale` أو انقر على زر شريط الأدوات.
2. **حدد الكائنات** — انقر للتبديل، أو اسحب للتحديد بالمنطقة.
3. اضغط **Enter** أو **Space** لتأكيد التحديد.
4. **انقر على نقطة الأساس** (إدخال الإحداثيات متاح)، ثم اكتب العامل.

```
  أساس ●                أساس ●
        [عنصر]   →            [عنصر أكبر]
  العامل = 2 → المسافات من ● تتضاعف
```

## كتابة عامل المقياس

بعد وضع نقطة الأساس تُظهر المحطة `enter scale factor:` وتنتظر إدخال لوحة المفاتيح:

| المفتاح | الإجراء |
|---------|---------|
| `0`–`9`، `.` | إلحاق رقم بالعامل |
| `-` | عامل سالب (الحرف الأول فقط — يعكس ثم يُغيِّر المقياس) |
| `Backspace` | حذف آخر حرف مكتوب |
| `Enter` | تطبيق المقياس على العامل المكتوب |

يجب أن يكون العامل غير صفري. قيم شائعة:

| العامل | التأثير |
|--------|---------|
| `2` | يُضاعف جميع الأبعاد |
| `0.5` | يُنصِّف جميع الأبعاد |
| `1.5` | يزيد بنسبة 50% |
| `-1` | يعكس عبر نقطة الأساس (مساوٍ لدوران 180°) |

لا توجد معاينة حية أثناء الكتابة — تظهر النتيجة المُغيَّرة مقياسها فقط بعد الضغط على **Enter**.

## مرجع لوحة المفاتيح

| المفتاح | الإجراء |
|---------|---------|
| `Enter` / `Space` | تأكيد التحديد |
| `0`–`9`، `.`، `-` | بدء إدخال إحداثي X (مرحلة نقطة الأساس)، أو عامل المقياس (مرحلة العامل) |
| `,` | تثبيت X والانتقال إلى إدخال Y (مرحلة نقطة الأساس) |
| `Backspace` | حذف آخر حرف مكتوب |
| `Enter` | تأكيد الإحداثي أو تطبيق المقياس |
| `Escape` | إلغاء وإعادة التعيين |

## التحديد أثناء الأمر

| الطريقة | السلوك |
|---------|--------|
| **النقر** | يبدّل العنصر تحت المؤشر |
| **السحب يميناً** (صارم) | يُضيف العناصر الواقعة بالكامل داخل المربع |
| **السحب يساراً** (متقاطع) | يُضيف العناصر التي تتقاطع مع المربع |
| **Enter** / **Space** | يؤكد التحديد |

## ما الذي يُغيَّر مقياسه

كل أنواع العناصر مدعومة. يُغيِّر كل عنصر هندستَه نسبةً إلى نقطة الأساس:

| العنصر | ما الذي يتغير |
|--------|-------------|
| الخط | كلتا نقطتَي النهاية مُغيَّرتان مقياسهما بعيداً عن نقطة الأساس |
| الدائرة | المركز مُغيَّر مقياسه من نقطة الأساس؛ نصف القطر مضروب في العامل |
| القوس | المركز مُغيَّر مقياسه؛ نصف القطر مضروب في العامل؛ الزوايا دون تغيير |
| القطع الناقص | المركز مُغيَّر مقياسه؛ طولا نصفَي المحورين مضروبان في العامل |
| الخط المتعدد / المستطيل | كل رأس مُغيَّر مقياسه من نقطة الأساس |
| النص | نقطة التثبيت مُغيَّرة مقياسها؛ الارتفاع مضروب في العامل |
| المنحنى | جميع نقاط التحكم / نقاط التناسب مُغيَّرة مقياسها |
