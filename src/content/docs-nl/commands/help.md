# Help

Opent het Help-paneel in de app met een overzicht van alle toetsenbordsnelkoppelingen, navigatietips en algemeen gebruik.

## Gebruik

Klik op de werkbalkknop **Help** of typ `help` in de terminal. Het Help-paneel opent onmiddellijk en het commando sluit af.

## Wat staat er in het Help-paneel

### Navigatie
| Snelkoppeling | Actie |
|----------|--------|
| `Scrollwiel` | In- / uitzoomen |
| `Middelste klik + slepen` | De weergave verschuiven |
| `Dubbele middelste klik` | Alle entiteiten passend op het scherm |

### Commando's
| Snelkoppeling | Actie |
|----------|--------|
| Typen | Begin een commandonaam te typen — suggesties verschijnen op het canvas |
| `Delete` | Verwijder alle geselecteerde entiteiten |
| `Enter` / `Spatie` | Commando of invoer bevestigen |
| `Enter` / `Spatie` (inactief) | Herhaal het laatst gebruikte commando |
| `Escape` | Huidig commando annuleren of selectie wissen |

### Selectie
| Snelkoppeling | Actie |
|----------|--------|
| `Linkerklik` | Selecteer een entiteit |
| `Slepen →` | Selecteer alleen entiteiten die **volledig binnen** het sleepgebied vallen |
| `Slepen ←` | Selecteer elke entiteit die het sleepgebied **raakt** |
| `Linkerklik op grip` | Bewerk een hoekpunt of eindpunt direct |

### Tekentips
- **Snappunten** — de cursor snapt automatisch naar nabijgelegen eindpunten van entiteiten.
- **Vaste hoek** — beweeg de cursor ver genoeg in één richting om te vergrendelen op stappen van 45°.
- **Exacte lengte** — terwijl de hoek vergrendeld is, typ een getal en druk op **Enter** om een exacte afstand in te stellen.
