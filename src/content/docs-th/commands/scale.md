# Scale

คำสั่ง `scale` ปรับขนาดวัตถุที่เลือกสม่ำเสมอรอบจุดฐาน ระยะทางทั้งหมดจากจุดฐานจะถูกคูณด้วยตัวคูณสเกล — ตัวคูณ `2` จะทำให้ขนาดเป็นสองเท่า, `0.5` ทำให้เป็นครึ่งหนึ่ง ต้องป้อนตัวคูณโดยการพิมพ์เสมอ ไม่มีการคลิกเพื่อตั้งสเกล

## สองวิธีเริ่มต้น

**เลือกก่อน แล้วปรับสเกล** — เลือกวัตถุก่อน แล้วเปิดใช้งาน:

1. เลือกวัตถุหนึ่งหรือมากกว่าบนแคนวาส
2. พิมพ์ `scale` ในเทอร์มินัลหรือคลิกปุ่ม **Scale** บนแถบเครื่องมือ
3. **คลิกจุดฐาน** — จุดที่ไม่เคลื่อนระหว่างการปรับสเกล หรือพิมพ์ `X,Y` แล้วกด **Enter** เพื่อป้อนพิกัดที่แน่นอน
4. **พิมพ์ตัวคูณสเกล** แล้วกด **Enter**

**เปิดใช้งานก่อน แล้วเลือก** — เริ่มคำสั่งโดยไม่มีการเลือก:

1. พิมพ์ `scale` หรือคลิกปุ่มแถบเครื่องมือ
2. **เลือกวัตถุ** — คลิกเพื่อสลับ หรือลากเพื่อเลือกตามพื้นที่
3. กด **Enter** หรือ **Space** เพื่อยืนยันการเลือก
4. **คลิกจุดฐาน** (การป้อนพิกัดพร้อมใช้) แล้วพิมพ์ตัวคูณ

```
  ฐาน ●                ฐาน ●
        [วัตถุ]   →          [วัตถุที่ใหญ่กว่า]
  ตัวคูณ = 2 → ระยะจาก ● จะเป็นสองเท่า
```

## การพิมพ์ตัวคูณสเกล

หลังจากวางจุดฐาน เทอร์มินัลจะแสดง `enter scale factor:` และรอการป้อนแป้นพิมพ์:

| ปุ่ม | การกระทำ |
|-----|----------|
| `0`–`9`, `.` | เพิ่มตัวเลขเข้าค่าตัวคูณ |
| `-` | ตัวคูณติดลบ (อักขระตัวแรกเท่านั้น — กลับด้านแล้วปรับสเกล) |
| `Backspace` | ลบอักขระที่พิมพ์ล่าสุด |
| `Enter` | ใช้สเกลที่ตัวคูณที่พิมพ์ |

ตัวคูณต้องไม่เป็นศูนย์ ค่าทั่วไป:

| ตัวคูณ | ผล |
|--------|-----|
| `2` | ขนาดเป็นสองเท่าทุกมิติ |
| `0.5` | ขนาดเป็นครึ่งหนึ่งทุกมิติ |
| `1.5` | เพิ่มขึ้น 50% |
| `-1` | สะท้อนผ่านจุดฐาน (เทียบเท่าการหมุน 180°) |

ไม่มีตัวอย่างสดขณะพิมพ์ — ผลลัพธ์ที่ปรับสเกลแล้วจะปรากฏหลังจากกด **Enter** เท่านั้น

## อ้างอิงแป้นพิมพ์

| ปุ่ม | การกระทำ |
|-----|----------|
| `Enter` / `Space` | ยืนยันการเลือก |
| `0`–`9`, `.`, `-` | เริ่มป้อนพิกัด X (ระยะจุดฐาน) หรือตัวคูณสเกล (ระยะตัวคูณ) |
| `,` | ล็อก X และย้ายไปป้อน Y (ระยะจุดฐาน) |
| `Backspace` | ลบอักขระที่พิมพ์ล่าสุด |
| `Enter` | ยืนยันพิกัดหรือใช้สเกล |
| `Escape` | ยกเลิกและรีเซ็ต |

## การเลือกระหว่างคำสั่ง

| วิธี | พฤติกรรม |
|------|-----------|
| **คลิก** | สลับวัตถุใต้เคอร์เซอร์ |
| **ลากไปขวา** (เข้มงวด) | เพิ่มวัตถุที่อยู่ในกล่องทั้งหมด |
| **ลากไปซ้าย** (ตัดผ่าน) | เพิ่มวัตถุที่ตัดขอบกล่อง |
| **Enter** / **Space** | ยืนยันการเลือก |

## สิ่งที่ถูกปรับสเกล

รองรับวัตถุทุกประเภท แต่ละวัตถุปรับสเกลเรขาคณิตสัมพันธ์กับจุดฐาน:

| วัตถุ | สิ่งที่เปลี่ยนแปลง |
|-------|-----------------|
| Line | จุดสิ้นสุดทั้งสองปรับสเกลออกจากจุดฐาน |
| Circle | ศูนย์กลางปรับสเกลจากจุดฐาน รัศมีคูณด้วยตัวคูณ |
| Arc | ศูนย์กลางปรับสเกล รัศมีคูณด้วยตัวคูณ มุมไม่เปลี่ยน |
| Ellipse | ศูนย์กลางปรับสเกล ความยาวกึ่งแกนทั้งสองคูณด้วยตัวคูณ |
| Polyline / Rectangle | จุดยอดทุกจุดปรับสเกลจากจุดฐาน |
| Text | จุดยึดปรับสเกล ความสูงคูณด้วยตัวคูณ |
| Spline | จุดควบคุม/จุดพอดีทั้งหมดปรับสเกล |
