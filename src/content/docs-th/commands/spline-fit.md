# Spline Fit

คำสั่ง `splinefit` วาด spline ลูกบาศก์ที่ผ่านทุกจุดที่คุณคลิก — เป็นเส้นโค้งการแทรกค่า ต่างจาก [Spline CV](../spline-cv/) ที่เส้นโค้งเพียงดึงดูดไปทางจุดควบคุม ที่นี่เส้นโค้งถูกบังคับให้แตะทุกพิกัดที่คลิกพอดี ภายในโปรแกรมแก้ไขจะ fit จุดควบคุมเพื่อให้ได้ผลนี้ และ CVs เหล่านั้นถูกเก็บไว้พร้อมกับ fit points ในไฟล์ DXF

## การวาด Spline ผ่าน Fit Points

1. พิมพ์ `splinefit` ในเทอร์มินัลหรือคลิกปุ่ม **Spline Fit** บนแถบเครื่องมือ
2. **คลิกเพื่อวาง fit points** — เส้นโค้งจะผ่านแต่ละจุด หรือพิมพ์ `X,Y` แล้วกด **Enter** เพื่อป้อนพิกัดที่แน่นอน
3. กด **Enter** เพื่อเสร็จสิ้น (ต้องการอย่างน้อย 2 จุด)

```
  ●──────●──────●──────●  ← เส้นโค้งผ่านทุกการคลิกพอดี
  p1     p2     p3     p4
```

ตัวอย่างสดแสดงเส้นโค้งการแทรกค่าปัจจุบันขณะเคลื่อนเคอร์เซอร์ รวมถึงจุดถัดไปที่เป็นไปได้ที่ตำแหน่งเคอร์เซอร์ กด **Escape** เพื่อยกเลิกจุดทั้งหมดที่วางไว้และออก

## การป้อนพิกัด

แทนการคลิก พิมพ์ตำแหน่งที่แน่นอนสำหรับ fit point ใดก็ได้:

1. พิมพ์ค่า X
2. กด `,` — เทอร์มินัลแสดง `[X], [Y{cursor}]`
3. พิมพ์ค่า Y
4. กด **Enter** เพื่อวาง fit point

## อ้างอิงแป้นพิมพ์

| ปุ่ม | การกระทำ |
|-----|----------|
| `0`–`9`, `.`, `-` | เริ่มป้อนพิกัด X |
| `,` | ล็อก X และย้ายไปป้อน Y |
| `Backspace` | ลบอักขระที่พิมพ์ล่าสุด |
| `Enter` | ยืนยันพิกัดที่พิมพ์ หรือเสร็จสิ้น spline ถ้าไม่มีการป้อนและมีจุด ≥ 2 |
| `Escape` | ยกเลิกจุดทั้งหมดและออก |

## การแก้ไขกริป — ปรับรูปร่างผ่าน Fit Points

Fit spline ที่เลือกจะแสดงกริปหนึ่งอันต่อ fit point:

| กริป | ตำแหน่ง | สิ่งที่ทำ |
|------|---------|---------|
| **Fit point** | ที่ตำแหน่งที่คลิกแต่ละจุด | ลากเพื่อย้าย fit point นั้น — เส้นโค้งทั้งหมดจะ re-interpolate เพื่อผ่านตำแหน่งใหม่ |

การลากกริปหนึ่งอัน re-fit เส้นโค้งทั้งหมด ไม่ใช่แค่ส่วนที่อยู่ติดกัน ซึ่งต่างจากการแก้ไขกริปของโพลีไลน์ที่การย้ายจุดยอดจะปรับรูปร่างเฉพาะสองส่วนที่อยู่ติดกัน

ไม่มีกริป "ย้าย spline ทั้งหมด" หากต้องการเลื่อน spline ทั้งหมด ให้ใช้คำสั่ง [Move](../move/)

## การเลือก Fit Spline

| วิธี | พฤติกรรม |
|------|-----------|
| **คลิก** | เลือกถ้าคลิกลงใกล้จุดใดๆ บนเส้นโค้ง |
| **ลากไปขวา** (เข้มงวด) | จุดตัวอย่างทั้งหมดตามเส้นโค้งต้องอยู่ในกล่องเลือก |
| **ลากไปซ้าย** (ตัดผ่าน) | ส่วนใดของเส้นโค้งที่ตัดขอบกล่องเลือกจะเลือก spline นั้น |

## Spline Fit vs Spline CV — ควรใช้อันไหน

| | Spline Fit | Spline CV |
|---|------------|-----------|
| เส้นโค้งผ่านจุด | ทุกจุดที่คลิกพอดี | เฉพาะจุดแรกและสุดท้าย (clamped) |
| ผลการแก้ไขกริป | Fit point เคลื่อน → เส้นโค้งทั้งหมด re-interpolated | CV เคลื่อน → เส้นโค้งดึงดูดไปทางตำแหน่งใหม่ |
| ความคาดเดาได้ของรูปร่าง | สูง — เส้นโค้งตามการคลิก | ต่ำกว่า — เส้นโค้งล้าหลัง CVs |
| เหมาะสำหรับ | เส้นโค้งที่ต้องแตะพิกัดเฉพาะ | เส้นโค้งสวยงาม เส้นทางอิสระ |

## DXF — วัตถุ SPLINE (รูปแบบ fit-point)

Fit spline จะถูกบันทึกเป็นวัตถุ `SPLINE` ในไฟล์ DXF โดยเก็บทั้งพิกัด fit point และจุดควบคุมที่คำนวณแล้ว `splineFlag` ถูกตั้งเป็น `8` (fit-point spline) เพื่อให้แอปพลิเคชันที่โหลดซ้ำรู้ว่าจะแสดงจุดใดเป็นกริปที่แก้ไขได้ คุณสมบัติทั้งหมด — สี เลเยอร์ linetype สเกล linetype และความหนา — จะถ่ายโอนข้อมูลไปกลับโดยไม่สูญหาย แอปพลิเคชัน DXF ที่รองรับ fit-point splines (LibreCAD, FreeCAD) จะแสดง fit points เป็นข้อมูลที่แก้ไขได้หลัก
