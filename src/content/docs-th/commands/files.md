# Files

คำสั่ง **Files** เปิด **แผงไฟล์ล่าสุด** — รายการภาพวาดทั้งหมดที่ถูกบันทึกไปยังหน่วยความจำในเครื่องของเบราว์เซอร์ ใช้เพื่อเปิดภาพวาดก่อนหน้าอีกครั้งหรือลบไฟล์ที่ไม่ต้องการอีกต่อไป

## วิธีเปิดแผงไฟล์ล่าสุด

คลิกปุ่ม **Files** บนแถบเครื่องมือ (ไอคอนประวัติ) ในแผง File ที่ด้านบนของหน้าจอ แผงเปิดด้านซ้ายของแคนวาส

## แผงไฟล์ล่าสุด

แต่ละรายการในรายการแสดงชื่อไฟล์และปุ่มดำเนินการสองปุ่ม:

| ปุ่ม | การกระทำ |
|------|----------|
| ✓ **Restore** | โหลดภาพวาดบนแคนวาส — แทนที่เนื้อหาปัจจุบัน |
| ✕ **Delete** | ลบไฟล์ออกจากหน่วยความจำเบราว์เซอร์อย่างถาวร |

หากยังไม่มีไฟล์ที่บันทึก แผงแสดง "No files saved"

## วิธีจัดเก็บไฟล์

KulmanLab บันทึกภาพวาดไปยัง **IndexedDB** — ฐานข้อมูลในเบราว์เซอร์ที่คงอยู่ ซึ่งหมายความว่า:

- ไฟล์รอดชีวิตจากการโหลดหน้าและรีสตาร์ทเบราว์เซอร์
- ไฟล์ถูกจัดเก็บ **ในเครื่องของคุณ** — ไม่ถูกอัปโหลดไปยังเซิร์ฟเวอร์ใด
- แต่ละเบราว์เซอร์และอุปกรณ์มีหน่วยเก็บข้อมูลอิสระ ภาพวาดที่บันทึกใน Chrome บนคอมพิวเตอร์เครื่องหนึ่งไม่สามารถใช้งานใน Firefox หรือบนเครื่องอื่น
- การล้างข้อมูลไซต์ของเบราว์เซอร์จะลบไฟล์ที่บันทึกทั้งหมด

## การโหลดไฟล์อัตโนมัติเมื่อเริ่มต้น

เมื่อคุณเปิด KulmanLab CAD แอปจะโหลด **ไฟล์ที่แก้ไขล่าสุด** จากหน่วยเก็บข้อมูลโดยอัตโนมัติ คุณไม่ต้องเปิดด้วยตนเองจากแผง Files ทุกครั้ง

## การจัดการพื้นที่เก็บข้อมูล

ไม่มีขีดจำกัดจำนวนภาพวาดที่บันทึกได้ แต่หน่วยเก็บข้อมูลเบราว์เซอร์มีจำกัด หากพบคำเตือนเกี่ยวกับพื้นที่เก็บข้อมูล ให้ลบไฟล์เก่าๆ จากแผง Files

หากต้องการลบภาพวาดที่บันทึกทั้งหมดพร้อมกัน ใช้คำสั่ง [WipeStorage](../wipestorage/)

## ชื่อไฟล์และการกำหนดเวอร์ชัน

แต่ละไฟล์มีการประทับเวลาในชื่อ (เช่น `myplan_May22_14:30:00.dxf`) การประทับเวลาถูกเพิ่มโดยอัตโนมัติเมื่อคุณ [import](../import/) ไฟล์ เพื่อให้แน่ใจว่าการนำเข้าไฟล์ต้นทางเดิมซ้ำจะสร้างรายการใหม่แทนที่จะเขียนทับเวอร์ชันก่อนหน้า

## คำสั่งที่เกี่ยวข้อง

- [Import](../import/) — โหลดภาพวาดจากระบบไฟล์ของคุณไปยังหน่วยความจำเบราว์เซอร์
- [Export](../export/) — ดาวน์โหลดภาพวาดไปยังระบบไฟล์ของคุณ
- [New File](../new-file/) — เริ่มภาพวาดเปล่า (บันทึกอัตโนมัติด้วย)
- [WipeStorage](../wipestorage/) — ล้างไฟล์ที่บันทึกทั้งหมดจากหน่วยความจำเบราว์เซอร์
