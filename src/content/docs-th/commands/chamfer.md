# Chamfer

คำสั่ง `chamfer` ตัดมุมเฉียงตรงระหว่างวัตถุ [Line](../line/) หรือ [Polyline](../polyline/) สองชิ้น คุณระบุว่าจะตัดกลับตามแต่ละวัตถุแค่ไหน (d1 และ d2) และคำสั่งจะตัดทั้งสองวัตถุไปยังจุดเหล่านั้นและแทรกเส้นเชื่อมระหว่างกัน

การใช้ระยะเท่ากันจะสร้างการตัด 45° แบบสมมาตร ระยะต่างกันจะสร้างการเฉลียงแบบไม่สมมาตร

Chamfer ทำงานบนวัตถุ **Line และ Polyline**

## การใช้ chamfer

1. พิมพ์ `chamfer` ในเทอร์มินัลหรือคลิกปุ่ม **Chamfer** บนแถบเครื่องมือ
2. **พิมพ์ระยะ chamfer แรก** (d1 — ระยะตามวัตถุแรก) แล้วกด **Enter**
3. **พิมพ์ระยะ chamfer ที่สอง** (d2 — ระยะตามวัตถุที่สอง) แล้วกด **Enter**
4. **คลิกวัตถุแรก** — ส่วนที่คุณคลิกกำหนดว่าด้านใดของจุดตัดจะถูกเก็บไว้
5. **วางเมาส์เหนือวัตถุที่สอง** — เส้นประแสดงผลลัพธ์การตัด chamfer ย้ายเคอร์เซอร์ไปด้านที่ต้องการเก็บไว้
6. **คลิก** เพื่อใช้งาน ทั้งสองวัตถุถูกตัดและเส้น chamfer ถูกแทรก

```
  ก่อน (d1=5, d2=8):        หลัง:

  ──────────────              ──────────╲
                │                        ╲────
                │
```

## การเลือกด้าน

เมื่อสองเส้นตัดกัน chamfer ถูกใช้กับมุมที่กำหนดโดยตำแหน่งคลิก — ส่วนของแต่ละวัตถุ **ด้านเดียวกับเคอร์เซอร์** จะถูกเก็บไว้

- คลิกใกล้ปลายด้านหนึ่งของวัตถุแรกเพื่อเลือกครึ่งนั้น
- ย้ายเคอร์เซอร์ไปยังครึ่งที่ต้องการของวัตถุที่สอง — ตัวอย่างเส้นประอัปเดตสด

สำหรับ Polyline ตำแหน่งคลิกกำหนด **ส่วน** ของโพลีไลน์ที่เข้าร่วม และจุดยอดที่ใกล้ที่สุดในด้านจุดตัดคือจุดที่ถูกตัด

## สิ่งที่คำสั่งสร้าง

- จุดสิ้นสุดของวัตถุแรก (หรือจุดยอดโพลีไลน์) ที่ใกล้กับจุดตัดถูกย้ายไปยังจุด **T1** ซึ่งอยู่ d1 ตามวัตถุแรกจากจุดตัด
- จุดสิ้นสุดของวัตถุที่สอง (หรือจุดยอดโพลีไลน์) ที่ใกล้กับจุดตัดถูกย้ายไปยังจุด **T2** ซึ่งอยู่ d2 ตามวัตถุที่สองจากจุดตัด
- วัตถุ Line ใหม่ถูกแทรกจาก **T1** ถึง **T2**

เส้นที่แทรกรับค่าความหนาเส้น สี เลเยอร์ และประเภทเส้นปัจจุบัน

## อ้างอิงแป้นพิมพ์

| ปุ่ม | การกระทำ |
|-----|----------|
| `0`–`9`, `.` | เพิ่มตัวเลขเข้าไปในค่าระยะปัจจุบัน |
| `Backspace` | ลบอักขระที่พิมพ์ล่าสุด |
| `Enter` | ยืนยันระยะที่พิมพ์และดำเนินการต่อ |
| `Escape` | ยกเลิกและรีเซ็ต |

## วัตถุที่รองรับ

| วัตถุ | รองรับ |
|-------|--------|
| Line | ใช่ |
| Polyline / Rectangle | ใช่ |
| Arc, Circle, Ellipse | ไม่ |
| Text, Spline, Dimension, Leader | ไม่ |

## Chamfer vs Fillet

| | Chamfer | Fillet |
|---|---------|--------|
| ประเภทมุม | การตัดตรง | ส่วนโค้งมน |
| ป้อน | สองระยะ (d1, d2) | รัศมีหนึ่งค่า |
| วัตถุที่แทรก | Line | Arc |
| วัตถุที่รองรับ | Lines และ Polylines | Lines เท่านั้น |
