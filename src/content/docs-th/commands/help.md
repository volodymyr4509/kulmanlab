# Help

เปิดแผงช่วยเหลือในแอปพร้อมสรุปทางลัดแป้นพิมพ์ทั้งหมด เคล็ดลับการนำทาง และการใช้งานทั่วไป

## วิธีใช้

คลิกปุ่ม **Help** บนแถบเครื่องมือหรือพิมพ์ `help` ในเทอร์มินัล แผงช่วยเหลือเปิดทันทีและคำสั่งออก

## เนื้อหาในแผงช่วยเหลือ

### การนำทาง
| ทางลัด | การกระทำ |
|--------|----------|
| `Scroll wheel` | ซูมเข้า / ออก |
| `Middle click + drag` | แพนมุมมอง |
| `Double middle click` | จัดวัตถุทั้งหมดให้พอดีกับหน้าจอ |

### คำสั่ง
| ทางลัด | การกระทำ |
|--------|----------|
| พิมพ์ | เริ่มพิมพ์ชื่อคำสั่ง — คำแนะนำปรากฏบนแคนวาส |
| `Delete` | ลบวัตถุที่เลือกทั้งหมด |
| `Enter` / `Space` | ยืนยันคำสั่งหรือป้อนข้อมูล |
| `Enter` / `Space` (ว่าง) | ทำซ้ำคำสั่งที่ใช้ล่าสุด |
| `Escape` | ยกเลิกคำสั่งปัจจุบันหรือล้างการเลือก |

### การเลือก
| ทางลัด | การกระทำ |
|--------|----------|
| `Left click` | เลือกวัตถุ |
| `Drag →` | เลือกเฉพาะวัตถุที่อยู่ **ภายใน** พื้นที่ลาก |
| `Drag ←` | เลือกวัตถุที่ **สัมผัส** พื้นที่ลาก |
| `Left click grip` | แก้ไขจุดยอดหรือจุดสิ้นสุดโดยตรง |

### เคล็ดลับการวาด
- **จุดสแนป** — เคอร์เซอร์สแนปไปยังจุดสิ้นสุดของวัตถุที่อยู่ใกล้โดยอัตโนมัติ
- **มุมเหนียว** — ย้ายเคอร์เซอร์ไกลพอในทิศทางหนึ่งเพื่อล็อกกับ 45°
- **ความยาวที่แน่นอน** — ขณะที่มุมล็อก พิมพ์ตัวเลขแล้วกด `Enter` เพื่อตั้งระยะที่แน่นอน
