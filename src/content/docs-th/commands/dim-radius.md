# Dimension Radius

คำสั่ง `dimradius` วางมิติรัศมีบนส่วนโค้งหรือวงกลม เส้นมิติวิ่งจากศูนย์กลางไปยังจุดบนเส้นรอบวงในทิศทางของเคอร์เซอร์ ระบุด้วย `R <ค่า>` หากต้องการกำหนดเส้นผ่านศูนย์กลางเต็มแทน ใช้ [Dimension Diameter](../dim-diameter/)

## กายวิภาคของมิติรัศมี

```
  ● (ศูนย์กลาง)
   \
    \  R 5.00
     \
      ●────── ข้อความ (ด้านเคอร์เซอร์)
   (จุดส่วนโค้ง)
```

- **เส้นมิติ** — จากศูนย์กลางผ่านจุดส่วนโค้งไปยังเคอร์เซอร์ มีหัวลูกศรที่ส่วนโค้ง
- **ป้าย** — `R` ตามด้วยค่ารัศมี

## การวางมิติรัศมี

1. พิมพ์ `dimradius` ในเทอร์มินัลหรือคลิกปุ่ม **Dimension Radius** บนแถบเครื่องมือ
2. **คลิกส่วนโค้งหรือวงกลม** เพื่อเลือก
3. **ย้ายเคอร์เซอร์** เพื่อจัดทิศทางเส้นมิติ — จุดส่วนโค้งติดตามทิศทางเคอร์เซอร์จากศูนย์กลาง
4. **คลิก** เพื่อวางมิติ

เฉพาะวัตถุ **Arc** และ **Circle** เท่านั้นที่สามารถเลือกได้ การคลิกวัตถุประเภทอื่นไม่มีผล

## อ้างอิงแป้นพิมพ์

| ปุ่ม | การกระทำ |
|-----|----------|
| `Escape` | ยกเลิก |

## Dimension Radius vs Dimension Diameter

| | Dimension Radius | Dimension Diameter |
|---|-----------------|-------------------|
| วัด | รัศมี (ศูนย์กลางถึงขอบ) | เส้นผ่านศูนย์กลาง (ขอบถึงขอบผ่านศูนย์กลาง) |
| เส้นมิติ | ศูนย์กลาง → จุดส่วนโค้ง | จุดส่วนโค้ง → จุดส่วนโค้ง (ผ่านศูนย์กลาง) |
| นำหน้าป้าย | `R` | `⌀` |
| หัวลูกศร | หนึ่ง (ที่จุดส่วนโค้ง) | สอง (ที่จุดส่วนโค้งทั้งสอง) |
| เหมาะสำหรับ | ระบุด้านหนึ่งของลักษณะโค้ง | ระบุมิติวงกลมเต็ม |

## การแก้ไขป้าย — simple mode

**ดับเบิลคลิก** มิติรัศมีที่วางแล้วเพื่อเปิดโปรแกรมแก้ไขข้อความในโหมด **simple** โปรแกรมแก้ไขถูกเติมค่าที่แสดงปัจจุบัน (เช่น `R 5.00`) ล่วงหน้าเพื่อให้คุณวางเคอร์เซอร์และแก้ไขได้โดยตรง

| ฟีเจอร์ | การทำงาน |
|---------|----------|
| Bold / Italic / Font / Height | ใช้กับ**ป้ายทั้งหมด**พร้อมกัน |
| การจัดรูปแบบต่อตัวอักษร | ไม่รองรับ |
| `Enter` | ยืนยันค่าและปิดโปรแกรมแก้ไข |
| หลายบรรทัด | ไม่รองรับ |

ดู [Text Editor — simple mode](../../interface/text-editor/#simple-mode) สำหรับข้อมูลอ้างอิงทั้งหมด

## DXF — วัตถุ DIMENSION ประเภท radius

มิติรัศมีถูกบันทึกเป็นวัตถุ `DIMENSION` ที่มีเรขาคณิตประเภท radius จัดเก็บพิกัดศูนย์กลาง ตำแหน่งจุดส่วนโค้ง และค่ารัศมีที่วัด คุณสมบัติทั้งหมดรอบเที่ยวโดยไม่สูญเสียข้อมูล
