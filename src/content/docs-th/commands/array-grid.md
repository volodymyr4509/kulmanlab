# Array Grid

คำสั่ง `ArrayGrid` สร้างกริดสี่เหลี่ยมของสำเนาจากวัตถุที่เลือก — ป้อนจำนวนแถว จำนวนคอลัมน์ และระยะห่างระหว่างกัน ทั้งหมดพิมพ์ในเทอร์มินัล การเลือกเดิมครอบครองเซลล์แถว 0 คอลัมน์ 0 ส่วนเซลล์อื่นๆ ล้วนเป็นสำเนาที่แปลตำแหน่ง

## สองวิธีเริ่มต้น

**เลือกก่อน แล้ว array** — เลือกวัตถุก่อน แล้วเปิดใช้งาน:

1. เลือกวัตถุหนึ่งหรือมากกว่าบนแคนวาส
2. พิมพ์ `arraygrid` ในเทอร์มินัล (แค่ `arr` ก็เพียงพอ — ไม่กำกวม) หรือคลิกปุ่ม **Array Grid** บนแถบเครื่องมือ
3. พิมพ์จำนวน **แถว** แล้วกด **Enter**
4. พิมพ์จำนวน **คอลัมน์** แล้วกด **Enter**
5. พิมพ์ **ระยะห่างระหว่างแถว** แล้วกด **Enter**
6. พิมพ์ **ระยะห่างระหว่างคอลัมน์** แล้วกด **Enter** — กริดจะถูกสร้างขึ้นทันที

**เปิดใช้งานก่อน แล้วเลือก** — เริ่มคำสั่งโดยไม่มีการเลือก:

1. พิมพ์ `arraygrid` หรือคลิกปุ่มแถบเครื่องมือ
2. **เลือกวัตถุ** — คลิกเพื่อสลับวัตถุแต่ละชิ้น หรือลากเพื่อเลือกตามพื้นที่
3. กด **Enter** หรือ **Space** เพื่อยืนยันการเลือก
4. ดำเนินต่อด้วยแถว → คอลัมน์ → ระยะห่างแถว → ระยะห่างคอลัมน์ ตามด้านบน

```
  2 แถว x 3 คอลัมน์:

  [B] [B] [B]   <- แถว 1 (สำเนาที่แปลตำแหน่ง)
  [A] [A] [A]   <- แถว 0: การเลือกเดิม สำเนาไปทางขวา
```

> เทอร์มินัลต้องการเพียงตัวอักษรเท่าที่จำเป็นเพื่อไม่ให้กำกวม — การพิมพ์ `arr` แล้วกด **Enter** จะเปิดใช้งาน Array Grid โดยตรง เนื่องจากไม่มีชื่อคำสั่งอื่นใดขึ้นต้นด้วยสามตัวอักษรนั้น (Arc, Area, Align และ Angle ล้วนแยกออกไปก่อนหน้านั้น)

## แถว คอลัมน์ และระยะห่าง

| พรอมต์ | รับค่า | หมายเหตุ |
|--------|--------|----------|
| แถว | จำนวนเต็มบวก (1, 2, 3…) | ตัวเลขเท่านั้น — ไม่มีจุดทศนิยมหรือเครื่องหมาย |
| คอลัมน์ | จำนวนเต็มบวก (1, 2, 3…) | ตัวเลขเท่านั้น — ไม่มีจุดทศนิยมหรือเครื่องหมาย |
| ระยะห่างแถว | ทศนิยมมีเครื่องหมาย (เช่น `10`, `-5.5`) | ระยะห่างระหว่างแถว; ค่าลบจะกลับทิศทาง |
| ระยะห่างคอลัมน์ | ทศนิยมมีเครื่องหมาย (เช่น `10`, `-5.5`) | ระยะห่างระหว่างคอลัมน์; ค่าลบจะกลับทิศทาง |

เมื่อมี 1 แถวและ 1 คอลัมน์ จะไม่มีการสร้างสำเนา — คำสั่งจะออกโดยไม่เปลี่ยนแปลงภาพวาด

## อ้างอิงแป้นพิมพ์

| ปุ่ม | การกระทำ |
|-----|----------|
| `Enter` / `Space` | ยืนยันการเลือกและดำเนินการไปยังพรอมต์แถว |
| `0`–`9` | ป้อนตัวเลขสำหรับแถวหรือคอลัมน์ |
| `0`–`9`, `.`, `-` | ป้อนตัวเลขสำหรับระยะห่างแถว/คอลัมน์ (`-` เป็นอักขระตัวแรกเท่านั้น) |
| `Backspace` | ลบอักขระที่พิมพ์ล่าสุดของพรอมต์ปัจจุบัน |
| `Enter` | ยืนยันพรอมต์ปัจจุบันและดำเนินการไปยังพรอมต์ถัดไป |
| `Escape` | ล้างค่าแถว/คอลัมน์/ระยะห่างที่พิมพ์และกลับไปยังขั้นตอนการเลือก |

## การเลือกระหว่างคำสั่ง

| วิธี | พฤติกรรม |
|------|-----------|
| **คลิก** | สลับวัตถุใต้เคอร์เซอร์เข้า/ออกจากการเลือก |
| **ลากไปขวา** (เข้มงวด) | เพิ่มวัตถุที่อยู่ในกล่องทั้งหมด |
| **ลากไปซ้าย** (ตัดผ่าน) | เพิ่มวัตถุที่ตัดขอบกล่อง |
| **Enter** / **Space** | ยืนยันการเลือกและดำเนินการไปยังพรอมต์แถว |

## หลังจาก array

สำเนาใหม่ถูกเพิ่มลงในภาพวาดและคำสั่งออก — การเลือกเดิมถูกล้าง รัน **Array Grid** อีกครั้ง หรือเริ่มคำสั่งใหม่

## Array Grid เทียบกับ Copy

| | Array Grid | Copy |
|---|-----------|------|
| การเลือกจุด | ไม่มี — แถว คอลัมน์ และระยะห่างถูกพิมพ์ | คลิก (หรือพิมพ์) จุดฐานและปลายทาง |
| สำเนาที่สร้างขึ้น | แถว × คอลัมน์ − 1 | พอดี 1 ต่อการดำเนินการสำเนา |
| เค้าโครง | กริดสี่เหลี่ยมสม่ำเสมอ | ที่ใดก็ได้ ที่ออฟเซ็ตใดก็ได้ |
| เหมาะสำหรับ | ทำซ้ำหน่วยหนึ่งตามรูปแบบสม่ำเสมอ (รู กระเบื้อง ตัวยึด) | สำเนาเดียวในตำแหน่งใดก็ได้ |

## วัตถุที่รองรับ

Array Grid ทำงานกับวัตถุทุกประเภท วัตถุทั้งหมดใช้ `translate(dx, dy)` ภายใน — การดำเนินการเดียวกับที่ [Copy](../copy/) และ [Move](../move/) ใช้ — ดังนั้นจึงไม่มีการยกเว้น
