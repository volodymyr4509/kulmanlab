# Dimension Diameter

คำสั่ง `dimdiameter` วางมิติเส้นผ่านศูนย์กลางบนส่วนโค้งหรือวงกลม เส้นมิติขยายตลอดเส้นผ่านศูนย์กลาง — ผ่านศูนย์กลางระหว่างสองจุดตรงข้ามของส่วนโค้ง — และระบุด้วย `⌀ <ค่า>` หากต้องการระบุเฉพาะรัศมีจากศูนย์กลางไปยังขอบหนึ่ง ใช้ [Dimension Radius](./dim-radius)

## กายวิภาคของมิติเส้นผ่านศูนย์กลาง

```
  ●──────────── ⌀ 10.00 ────────────●
  (จุดส่วนโค้งไกล)              (จุดส่วนโค้งใกล้ / ด้านข้อความ)
```

- **เส้นมิติ** — ขยายตลอดเส้นผ่านศูนย์กลาง มีหัวลูกศรที่จุดตัดส่วนโค้งทั้งสอง
- **จุดส่วนโค้งใกล้** — จุดเส้นรอบวงด้านเคอร์เซอร์ (ที่ป้ายข้อความวาง)
- **จุดส่วนโค้งไกล** — จุดตรงข้ามเส้นผ่านศูนย์กลาง
- **ป้าย** — `⌀` ตามด้วยค่าเส้นผ่านศูนย์กลาง

## การวางมิติเส้นผ่านศูนย์กลาง

1. พิมพ์ `dimdiameter` ในเทอร์มินัลหรือคลิกปุ่ม **Dimension Diameter** บนแถบเครื่องมือ
2. **คลิกส่วนโค้งหรือวงกลม** เพื่อเลือก
3. **ย้ายเคอร์เซอร์** เพื่อหมุนเส้นมิติไปมุมที่ต้องการ
4. **คลิก** เพื่อวางมิติ

เฉพาะวัตถุ **Arc** และ **Circle** เท่านั้นที่สามารถเลือกได้

## อ้างอิงแป้นพิมพ์

| ปุ่ม | การกระทำ |
|-----|----------|
| `Escape` | ยกเลิก |

## Dimension Diameter vs Dimension Radius

| | Dimension Diameter | Dimension Radius |
|---|-------------------|-----------------|
| วัด | เส้นผ่านศูนย์กลางเต็ม (2 × รัศมี) | รัศมี (ศูนย์กลางถึงขอบ) |
| เส้นมิติ | ขอบ → ขอบ ผ่านศูนย์กลาง | ศูนย์กลาง → ขอบ |
| นำหน้าป้าย | `⌀` | `R` |
| หัวลูกศร | สอง (ที่จุดส่วนโค้งทั้งสอง) | หนึ่ง (ที่จุดส่วนโค้ง) |
| เหมาะสำหรับ | มิติรูเต็มหรือเพลากลม | ระบุด้านหนึ่งของลักษณะโค้ง |

## การแก้ไขป้าย — simple mode

**ดับเบิลคลิก** มิติเส้นผ่านศูนย์กลางที่วางแล้วเพื่อเปิดโปรแกรมแก้ไขข้อความในโหมด **simple** โปรแกรมแก้ไขถูกเติมค่าที่แสดงปัจจุบัน (เช่น `⌀ 10.00`) ล่วงหน้าเพื่อให้คุณวางเคอร์เซอร์และแก้ไขได้โดยตรง

| ฟีเจอร์ | การทำงาน |
|---------|----------|
| Bold / Italic / Font / Height | ใช้กับ**ป้ายทั้งหมด**พร้อมกัน |
| การจัดรูปแบบต่อตัวอักษร | ไม่รองรับ |
| `Enter` | ยืนยันค่าและปิดโปรแกรมแก้ไข |
| หลายบรรทัด | ไม่รองรับ |

ดู [Text Editor — simple mode](../interface/text-editor#simple-mode) สำหรับข้อมูลอ้างอิงทั้งหมด

## DXF — วัตถุ DIMENSION ประเภท diameter

มิติเส้นผ่านศูนย์กลางถูกบันทึกเป็นวัตถุ `DIMENSION` ที่มีเรขาคณิตประเภท diameter จัดเก็บตำแหน่งจุดส่วนโค้งทั้งสองและค่าเส้นผ่านศูนย์กลางที่วัด (2 × รัศมี) คุณสมบัติทั้งหมดรอบเที่ยวโดยไม่สูญเสียข้อมูล
