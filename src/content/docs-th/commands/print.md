# Print

คำสั่ง `print` เปิด **Print Manager** — หน้าต่างส่งออกพร้อมแคนวาสตัวอย่างแบบสด ตัวเลือกรูปแบบ (PNG / JPEG / WebP / PDF) สลับขาวดำ และการครอบพื้นที่ตามต้องการ ไม่มีการส่งงานไปยังเครื่องพิมพ์จริง ผลลัพธ์จะถูกดาวน์โหลดเป็นไฟล์

## การเปิด Print Manager

คลิกปุ่ม **Print** บนแถบเครื่องมือหรือพิมพ์ `print` ในเทอร์มินัล Print Manager จะเปิดขึ้นทันทีพร้อมแสดงตัวอย่างของวิวพอร์ตปัจจุบัน

## เค้าโครง Print Manager

หน้าต่างมีสองแผง:
- **แถบด้านซ้าย** — ตัวควบคุมการส่งออกทั้งหมด
- **แผงด้านขวา** — แคนวาสตัวอย่างแบบสดที่อัปเดตตามการเปลี่ยนการตั้งค่า

### ตัวควบคุมในแถบด้านซ้าย

| ตัวควบคุม | คำอธิบาย |
|-----------|-----------|
| **Change Area** | ครอบพื้นที่สี่เหลี่ยมที่กำหนดเองบนแคนวาส (ดูด้านล่าง) |
| สลับ **Monochrome** | แปลงเส้นสีทั้งหมดเป็นสีดำ — เปิดใช้งานโดยค่าเริ่มต้นเพื่อผลลัพธ์การพิมพ์ที่ชัดเจน |
| ดรอปดาวน์ **Format** | PNG, JPEG, WebP หรือ PDF |
| ปุ่ม **Export** | สร้างและดาวน์โหลดไฟล์ |

## การเลือกพื้นที่ส่งออกแบบกำหนดเอง

โดยค่าเริ่มต้น ตัวอย่างแสดงสิ่งที่มองเห็นบนแคนวาสเมื่อเปิด Print Manager หากต้องการส่งออกพื้นที่เฉพาะ:

1. คลิก **Change Area** — Print Manager จะซ่อนตัวและแคนวาสจะโต้ตอบได้
2. **คลิกมุมแรก** ของสี่เหลี่ยมส่งออก
3. **คลิกมุมตรงข้าม** — Print Manager จะเปิดขึ้นใหม่พร้อมพื้นที่ที่เลือกในตัวอย่าง

กด `Escape` ระหว่างการเลือกพื้นที่เพื่อยกเลิกและกู้คืนพื้นที่ก่อนหน้า

แคนวาสตัวอย่างปรับขนาดแบบไดนามิกเพื่อให้ตรงกับ **อัตราส่วนภาพที่แน่นอน** ของพื้นที่ที่เลือก ดังนั้นตัวอย่างจึงแม่นยำระดับพิกเซล

## รูปแบบการส่งออก

| รูปแบบ | เหมาะสำหรับ | หมายเหตุ |
|--------|------------|---------|
| **PNG** | ไม่สูญเสียข้อมูล เส้นคมชัด | พื้นหลังสีขาว ไม่มีความโปร่งใส |
| **JPEG** | ไฟล์ขนาดเล็กสำหรับแชร์ | คุณภาพ 95% มีการบีบอัดเล็กน้อย |
| **WebP** | ไฟล์ขนาดเล็กที่สุดสำหรับเว็บ | คุณภาพ 95% เหมือนกัน บีบอัดดีกว่า JPEG |
| **PDF** | เอกสารพร้อมพิมพ์ | รูปภาพฝังที่ 150 DPI ภายในคอนเทนเนอร์ PDF |

ไฟล์ที่ส่งออกจะตั้งชื่อว่า `kulman-<timestamp>.<ext>` และดาวน์โหลดอัตโนมัติ

## ความละเอียดและพื้นหลังการส่งออก

- ความละเอียดสูงสุด: **2000 × 2000 พิกเซล** ปรับสัดส่วนตามพื้นที่ที่เลือก
- พื้นหลังเป็น **สีขาว** เสมอ
- เลเยอร์ที่ทำเครื่องหมายว่า **non-plotting** จะถูกยกเว้นจากการส่งออก

## อ้างอิงแป้นพิมพ์

| ปุ่ม | การกระทำ |
|-----|----------|
| `Escape` (ระหว่างการเลือกพื้นที่) | ยกเลิกการเลือกพื้นที่ กู้คืนพื้นที่ก่อนหน้า |
| `Escape` (ใน Print Manager) | ปิด Print Manager |
