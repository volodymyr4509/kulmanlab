# Arc

คำสั่ง `arc` วาดส่วนโค้งวงกลมผ่านสามจุดที่คุณคลิก ส่วนโค้งถูกคำนวณเป็นวงกลมรอบนอกเฉพาะที่ผ่านจุดทั้งสาม — ไม่จำเป็นต้องระบุจุดศูนย์กลางหรือรัศมีโดยตรง ส่วนโค้งวิ่งจากคลิกแรกไปยังคลิกที่สาม ผ่านคลิกที่สอง

## การวาดส่วนโค้ง

1. พิมพ์ `arc` ในเทอร์มินัลหรือคลิกปุ่ม **Arc** บนแถบเครื่องมือ
2. **คลิกจุดแรก** — ปลายหนึ่งของส่วนโค้ง หรือพิมพ์ `X,Y` แล้วกด **Enter** เพื่อป้อนพิกัดที่แน่นอน
3. **คลิกจุดที่สอง** — จุดที่ส่วนโค้งต้องผ่าน (ควบคุมความโค้งและทิศทาง) การป้อนพิกัดก็ใช้ได้ที่นี่เช่นกัน
4. **คลิกจุดที่สาม** — ปลายอีกด้านของส่วนโค้ง ส่วนโค้งถูกวางและคำสั่งออก การป้อนพิกัดก็ใช้ได้ที่นี่เช่นกัน

```
           ● (คลิกที่ 2 — จุดกึ่งกลางบนเส้นโค้ง)
          / \
         /   \
        ●     ●
     ที่ 1       ที่ 3
```

ตัวอย่างเส้นเชื่อมคลิกสองครั้งแรกขณะที่คุณวางตำแหน่งคลิกที่สาม จากคลิกที่สองเป็นต้นไป ตัวอย่างส่วนโค้งสดจะติดตามเคอร์เซอร์

> **จุดร่วมเส้น**: ถ้าจุดทั้งสามอยู่บนเส้นตรง ส่วนโค้งไม่สามารถคำนวณได้และไม่มีวัตถุถูกวาง ย้ายจุดที่สองออกจากเส้นแล้วลองใหม่

## การป้อนพิกัด

ในขั้นตอนใดก็ได้จากสามขั้นตอน คุณสามารถพิมพ์ตำแหน่งที่แน่นอนแทนการคลิก:

1. พิมพ์ค่า X
2. กด `,` — เทอร์มินัลแสดง `[X], [Y{cursor}]`
3. พิมพ์ค่า Y
4. กด **Enter** เพื่อวางจุด

## อ้างอิงแป้นพิมพ์

| ปุ่ม | การกระทำ |
|-----|----------|
| `0`–`9`, `.`, `-` | เริ่มป้อนพิกัด X |
| `,` | ล็อก X และย้ายไปป้อน Y |
| `Backspace` | ลบอักขระที่พิมพ์ล่าสุด |
| `Enter` | ยืนยันพิกัดที่พิมพ์ |
| `Escape` | ทิ้งจุดทั้งหมดที่วางและออก |

## การแก้ไขกริป — ปรับจุดสิ้นสุดและรัศมี

ส่วนโค้งที่เลือกแสดงกริปสามตัว:

| กริป | ตำแหน่ง | สิ่งที่ทำ |
|------|---------|-----------|
| **Center** | ศูนย์กลางทางเรขาคณิตของวงกลมรอบนอก | ย้ายส่วนโค้งทั้งหมด รัศมีและมุมไม่เปลี่ยน |
| **Start** | จุดสิ้นสุดแรกบนส่วนโค้ง | ลากเพื่อย้ายจุดเริ่มต้นตามวงกลมรอบนอก — เปลี่ยนทั้งมุมเริ่มต้นและรัศมี |
| **End** | จุดสิ้นสุดสุดท้ายบนส่วนโค้ง | ลากเพื่อย้ายจุดสิ้นสุดตามวงกลมรอบนอก — เปลี่ยนทั้งมุมสิ้นสุดและรัศมี |

การลากกริปเริ่มต้นหรือสิ้นสุดจะย้ายตำแหน่งไปยังตำแหน่งที่ลากและคำนวณมุมและรัศมีใหม่จากตำแหน่งใหม่นั้นสัมพันธ์กับศูนย์กลาง จุดสิ้นสุดตรงข้ามคงที่

## การเลือกส่วนโค้ง

| วิธี | พฤติกรรม |
|------|-----------|
| **คลิก** | เลือกถ้าคลิกอยู่ใกล้เส้นโค้ง (ไม่ใช่คอร์ด) |
| **ลากไปขวา** (เข้มงวด) | จุดตัวอย่างที่กระจายตามส่วนโค้งต้องอยู่ในกล่องทั้งหมด |
| **ลากไปซ้าย** (ตัดผ่าน) | จุดตัวอย่างใดๆ บนส่วนโค้งที่อยู่ในกล่องจะเลือกมัน |

## คำสั่งแก้ไขที่รองรับ

| คำสั่ง | สิ่งที่เกิดขึ้นกับส่วนโค้ง |
|--------|--------------------------|
| [Move](./move) | เลื่อนศูนย์กลาง รัศมีและมุมไม่เปลี่ยน |
| [Copy](./copy) | สร้างส่วนโค้งที่เหมือนกันในตำแหน่งใหม่ |
| [Rotate](./rotate) | หมุนศูนย์กลางและเลื่อนมุมเริ่มต้น/สิ้นสุดตามปริมาณการหมุน |
| [Mirror](./mirror) | สะท้อนศูนย์กลางและกลับมุมเริ่มต้น/สิ้นสุดข้ามแกนสะท้อน |
| [Scale](./scale) | ปรับมาตราส่วนตำแหน่งศูนย์กลางและคูณรัศมีด้วยตัวคูณมาตราส่วน |
| [Offset](./offset) | สร้างส่วนโค้งร่วมศูนย์กลางที่รัศมีใหญ่หรือเล็กกว่า มุมเดิม |
| [Delete](./delete) | ลบส่วนโค้ง |

## คุณสมบัติ

**ทั่วไป**

| คุณสมบัติ | ค่าเริ่มต้น | ความหมาย |
|-----------|-----------|----------|
| Color | 256 (ByLayer) | ดัชนีสี ACI |
| Layer | `0` | การกำหนดเลเยอร์ |
| Linetype | ByLayer | รูปแบบเส้นที่มีชื่อ |
| Linetype Scale | 1 | ตัวคูณมาตราส่วนสำหรับรูปแบบเส้น |
| Thickness | 0 | ความหนาของการรีดออก |

**เรขาคณิต**

| คุณสมบัติ | ความหมาย |
|-----------|----------|
| Center X / Center Y | ศูนย์กลางของวงกลมรอบนอก |
| Radius | รัศมีของวงกลมรอบนอก |
| Start Angle | มุมเป็นองศาที่ส่วนโค้งเริ่มต้น (วัดจากแกน X บวก) |
| End Angle | มุมเป็นองศาที่ส่วนโค้งสิ้นสุด |

## Arc vs Circle — เมื่อไหรควรใช้อะไร

| | Arc | Circle |
|---|-----|--------|
| ช่วง | บางส่วน — คลิกแรกถึงคลิกที่สาม | เต็ม 360° |
| วิธีป้อน | สามจุดบนเส้นโค้ง | ศูนย์กลาง + รัศมี (คลิกหรือพิมพ์) |
| ป้อนแบบพิมพ์ | พิกัด X,Y สำหรับแต่ละจุด | ค่ารัศมี (ศูนย์กลางก็รับ X,Y) |
| ปรับขนาดหลังวาง | ลากกริปเริ่มต้น/สิ้นสุด | ลากกริปจุดหลัก |
| เหมาะสำหรับ | การ Fillet มุมโค้ง เส้นทางโค้ง | รูเต็ม ลักษณะกลม |

## DXF — วัตถุ ARC

ส่วนโค้งถูกบันทึกเป็นวัตถุ `ARC` ในไฟล์ DXF จัดเก็บพิกัดศูนย์กลาง รัศมี มุมเริ่มต้น และมุมสิ้นสุด คุณสมบัติทั้งหมด — รวมถึงสี เลเยอร์ ประเภทเส้น มาตราส่วนประเภทเส้น และความหนา — รอบเที่ยวโดยไม่สูญเสียข้อมูล แอปพลิเคชันที่รองรับ DXF (LibreCAD, FreeCAD ฯลฯ) อ่านสิ่งเหล่านี้เป็นส่วนโค้งมาตรฐาน
