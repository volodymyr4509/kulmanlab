# Zoom Out

คำสั่ง `zoomout` หารระดับซูมปัจจุบันด้วย **1.5 เท่า** (เทียบเท่ากับการคูณด้วย ~0.667) และออกทันที ศูนย์กลางที่จุดกึ่งกลาง viewport เป็นการย้อนกลับของ [Zoom In](../zoom-in/)

## การซูมออก

คลิกปุ่ม **Zoom Out** บนแถบเครื่องมือหรือพิมพ์ `zoomout` ในเทอร์มินัล การซูมจะถูกใช้ทันทีและคำสั่งออก — ไม่ต้องคลิกบนแคนวาส

## วิธีทำงานของขั้นตอน 1.5 เท่า

| ระดับซูมปัจจุบัน | หลัง Zoom Out หนึ่งครั้ง |
|----------------|------------------------|
| 1.50× | 1.00× |
| 2.25× | 1.50× |
| 10.00× | 6.67× |
| 0.015× | 0.01× (ขีดจำกัดต่ำสุด) |

ระดับซูมจะแสดงที่ **มุมล่างขวา** ของแคนวาส ขีดจำกัดล่างคือ **0.01×** ขั้นตอนเพิ่มเติมจะไม่ทำอะไร

## ปุ่ม Zoom Out vs Scroll Wheel

| | ปุ่ม Zoom Out | Scroll Wheel |
|---|--------------|-------------|
| ศูนย์กลางซูม | จุดกึ่งกลาง viewport | ตำแหน่งเคอร์เซอร์ |
| ขนาดขั้นตอน | 1.5× ต่อคลิก | ~1.1× ต่อขีก |
| ต้องเปิดใช้งาน | ไม่ | ไม่ — ทำงานตลอดเวลา |
| เหมาะสำหรับ | การถอยหลังเพื่อดูบริบทมากขึ้น | การซูมออกที่เรียบเนียนยึดกับเคอร์เซอร์ |

## อ้างอิงแป้นพิมพ์

ไม่มีแป้นพิมพ์ลัดสำหรับคำสั่งนี้ ใช้ scroll wheel แทน — ทำงานได้ตลอดเวลาโดยไม่ต้องเปิดใช้งานคำสั่งใดๆ
