# Dimension Angular

คำสั่ง `DimensionAngular` วาง **คำอธิบายมิติมุม** บนภาพวาด วัดและระบุมุมระหว่างสองเส้น ช่วงของส่วนโค้ง หรือเซกเตอร์ของวงกลม

## วิธีเปิดใช้งาน

คลิกปุ่ม **Dimension Angular** บนแถบเครื่องมือในแผง Markup หรือพิมพ์ `DimensionAngular` ในเทอร์มินัล

## สามโหมดป้อนข้อมูล

คลิกแรกกำหนดโหมดที่ใช้:

### สองเส้น

1. **คลิกเส้นแรก** ตำแหน่งเคอร์เซอร์กำหนดว่าจะใช้ด้านใดของเส้น
2. **คลิกเส้นที่สอง** สองเส้นต้องตัดกัน (จุดตัดถูกคำนวณโดยอัตโนมัติ ไม่จำเป็นต้องมองเห็นบนหน้าจอ)
3. **คลิกเพื่อวาง** ส่วนโค้งมิติ ย้ายเคอร์เซอร์เพื่อเลือกรัศมีและเซกเตอร์มุมที่ระบุ — คำอธิบายติดตามเคอร์เซอร์ไปด้านใดก็ได้ของจุดยอดที่คุณอยู่

เส้นขนานไม่สามารถสร้างมิติมุมได้ คำสั่งจะเพิกเฉยคลิกที่สองถ้าเส้นไม่ตัดกัน

### ส่วนโค้ง

1. **คลิกส่วนโค้ง** มิติถูกสร้างทันทีจากมุมเริ่มต้นถึงมุมสิ้นสุดของส่วนโค้ง โดยใช้ศูนย์กลางของส่วนโค้งเป็นจุดยอด
2. **คลิกเพื่อวาง** ส่วนโค้งมิติที่รัศมีที่ต้องการ

### วงกลม

1. **คลิกวงกลม** จุดสิ้นสุดมุมแรกสแนปไปยังจุดที่ใกล้ที่สุดบนวงกลม
2. **คลิกจุดที่สอง** บนวงกลมเพื่อกำหนดจุดสิ้นสุดมุมที่สอง
3. **คลิกเพื่อวาง** ส่วนโค้งมิติ

## อ้างอิงแป้นพิมพ์

| ปุ่ม | การกระทำ |
|-----|----------|
| `Escape` | ยกเลิกและกลับไปยังการเลือกแรก |

## รายละเอียดพฤติกรรม

- ส่วนโค้งมิติวาดเสมอด้านจุดยอดที่คุณวาง — ย้ายเคอร์เซอร์ข้ามจุดยอดเพื่อพลิกไปยังมุมเสริม
- มุมที่วัดแสดงเป็นองศาและอัปเดตสดขณะย้ายเคอร์เซอร์ระหว่างการวาง
- คำอธิบายที่ได้เป็นวัตถุ `DimensionAngular` เต็มรูปแบบที่จัดเก็บบนเลเยอร์ปัจจุบัน คุณสมบัติรูปลักษณ์ (ขนาดลูกศร ความสูงข้อความ ความยาวเส้นขยาย) สามารถปรับในแผงคุณสมบัติ
- มิติมุมรวมอยู่ในการส่งออก JSON แต่ไม่รองรับโดยตัวส่งออก DXF

## การแก้ไขป้าย — simple mode

**ดับเบิลคลิก** มิติมุมที่วางแล้วเพื่อเปิดโปรแกรมแก้ไขข้อความในโหมด **simple** โปรแกรมแก้ไขถูกเติมค่าที่แสดงปัจจุบันล่วงหน้าเพื่อให้คุณวางเคอร์เซอร์และแก้ไขได้โดยตรง

| ฟีเจอร์ | การทำงาน |
|---------|----------|
| Bold / Italic / Font / Height | ใช้กับ**ป้ายทั้งหมด**พร้อมกัน |
| การจัดรูปแบบต่อตัวอักษร | ไม่รองรับ |
| `Enter` | ยืนยันค่าและปิดโปรแกรมแก้ไข |
| หลายบรรทัด | ไม่รองรับ |

ดู [Text Editor — simple mode](../interface/text-editor#simple-mode) สำหรับข้อมูลอ้างอิงทั้งหมด
