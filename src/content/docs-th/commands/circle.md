# Circle

คำสั่ง `circle` วาดวงกลมที่กำหนดด้วยจุดศูนย์กลางและรัศมี หลังจากคลิกศูนย์กลาง คุณสามารถตั้งค่ารัศมีโดยการคลิกจุดที่สองบนแคนวาสหรือพิมพ์ตัวเลขที่แน่นอน — ทั้งสองตัวเลือกใช้งานได้พร้อมกัน

## การวาดวงกลม

1. พิมพ์ `circle` ในเทอร์มินัลหรือคลิกปุ่ม **Circle** บนแถบเครื่องมือ
2. **คลิกจุดศูนย์กลาง** หรือพิมพ์ `X,Y` แล้วกด **Enter** เพื่อป้อนพิกัดที่แน่นอน
3. ตั้งค่ารัศมี — ทำได้สองแบบ:
   - **คลิกจุดใดก็ได้** บนแคนวาส — ระยะจากศูนย์กลางกลายเป็นรัศมี หรือ
   - **พิมพ์รัศมี** แล้วกด **Enter** เพื่อป้อนค่าที่แน่นอน

วงกลมถูกวางทันทีและคำสั่งออก

```
  center ●
          \  ตัวอย่างเส้นรัศมี
           \
            ● ← คลิกที่นี่ หรือพิมพ์ตัวเลข
```

ในขณะที่อยู่ในระยะรัศมี ตัวอย่างสดแสดงวงกลมที่ระยะเคอร์เซอร์ปัจจุบันและยังวาดเส้นรัศมีจากศูนย์กลางไปยังจุดปัจจุบัน

## การป้อนพิกัดศูนย์กลาง

แทนที่จะคลิก คุณสามารถพิมพ์ตำแหน่งศูนย์กลาง:

1. พิมพ์ค่า X
2. กด `,` — เทอร์มินัลแสดง `[X], [Y{cursor}]`
3. พิมพ์ค่า Y
4. กด **Enter** เพื่อวางศูนย์กลางและดำเนินการป้อนรัศมี

## การป้อนรัศมีแบบพิมพ์

หลังจากวางศูนย์กลาง การพิมพ์จะสร้างค่ารัศมีขึ้นทันที:

| ปุ่ม | การกระทำ |
|-----|----------|
| `0`–`9`, `.` | เพิ่มตัวเลขเข้าไปในค่ารัศมี |
| `Backspace` | ลบอักขระที่พิมพ์ล่าสุด |
| `Enter` | วางวงกลมที่รัศมีที่พิมพ์ |

ค่าสะสมแสดงในเทอร์มินัล (เช่น `enter radius of circle: 25`) ตัวอย่างอัปเดตแสดงรัศมีที่พิมพ์ขณะเคอร์เซอร์ควบคุมทิศทางของเครื่องหมายเส้นรัศมี

## อ้างอิงแป้นพิมพ์

| ปุ่ม | การกระทำ |
|-----|----------|
| `0`–`9`, `.`, `-` | เริ่มป้อนพิกัด X (ระยะศูนย์กลาง) หรือตัวเลขรัศมี (ระยะรัศมี) |
| `,` | ล็อก X และย้ายไปป้อน Y (ระยะศูนย์กลาง) |
| `Backspace` | ลบอักขระที่พิมพ์ล่าสุด |
| `Enter` | ยืนยันพิกัดหรือรัศมีที่พิมพ์ |
| `Escape` | ยกเลิกและรีเซ็ต |

## การแก้ไขกริป — ปรับขนาดรัศมี

วงกลมที่เลือกแสดงกริปห้าตัว:

| กริป | ตำแหน่ง | สิ่งที่ทำ |
|------|---------|-----------|
| **Center** | จุดศูนย์กลาง | ย้ายวงกลมทั้งหมด รัศมีไม่เปลี่ยน |
| **Left** | จุดซ้ายสุด (ศูนย์กลาง − รัศมี) | ลากเพื่อตั้งรัศมีใหม่ = ระยะถึงศูนย์กลาง |
| **Right** | จุดขวาสุด (ศูนย์กลาง + รัศมี) | ลากเพื่อตั้งรัศมีใหม่ = ระยะถึงศูนย์กลาง |
| **Top** | จุดบนสุด | ลากเพื่อตั้งรัศมีใหม่ = ระยะถึงศูนย์กลาง |
| **Bottom** | จุดล่างสุด | ลากเพื่อตั้งรัศมีใหม่ = ระยะถึงศูนย์กลาง |

กริปจุดหลักทั้งสี่ทำงานเหมือนกัน — รัศมีใหม่เท่ากับระยะจากศูนย์กลางไปยังตำแหน่งที่ลาก ศูนย์กลางคงที่

## การเลือกวงกลม

| วิธี | พฤติกรรม |
|------|-----------|
| **คลิก** | เลือกถ้าคลิกอยู่ใกล้เส้นรอบวง |
| **ลากไปขวา** (เข้มงวด) | กล่องสี่เหลี่ยมล้อมรอบ (ศูนย์กลาง ± รัศมี) ต้องอยู่ในกล่องทั้งหมด |
| **ลากไปซ้าย** (ตัดผ่าน) | ส่วนใดของเส้นรอบวงที่ตัดหรือสัมผัสขอบกล่องจะเลือกวงกลม |

## Circle vs Arc — เมื่อไหรควรใช้อะไร

| | Circle | Arc |
|---|--------|-----|
| ช่วง | เต็ม 360° | บางส่วน — กำหนดด้วยมุมเริ่มต้นและสิ้นสุด |
| วิธีวาด | ศูนย์กลาง + รัศมี | สามจุดบนเส้นโค้ง |
| ป้อนแบบพิมพ์ | ค่ารัศมี | ไม่มี — คลิกเท่านั้น |
| กริปปรับขนาด | 4 จุดหลัก | จุดเริ่มต้นและสิ้นสุด (มุม + รัศมี) |
| การกำหนดมิติ | รัศมี: [Dim Radius](../dim-radius/) · เส้นผ่านศูนย์กลาง: [Dim Diameter](../dim-diameter/) | [Dim Radius](../dim-radius/) |
| เหมาะสำหรับ | รูเต็ม วงกลมสลัก ลักษณะกลม | การ Fillet เส้นโค้งบางส่วน เส้นทางโค้ง |

## DXF — วัตถุ CIRCLE

วงกลมถูกบันทึกเป็นวัตถุ `CIRCLE` ในไฟล์ DXF พิกัดศูนย์กลาง รัศมี สี เลเยอร์ ประเภทเส้น มาตราส่วนประเภทเส้น และความหนาทั้งหมดรอบเที่ยวโดยไม่สูญเสียข้อมูล แอปพลิเคชันที่รองรับ DXF อ่านสิ่งเหล่านี้เป็นวงกลมมาตรฐาน
