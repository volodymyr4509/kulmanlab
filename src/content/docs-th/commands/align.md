# Align

คำสั่ง `align` จัดตำแหน่งวัตถุที่เลือกใหม่โดยใช้คู่จุดต้นทาง/ปลายทางหนึ่งหรือสองคู่ เมื่อใช้หนึ่งคู่ จะทำงานเหมือน [Move](../move/) ทุกประการ (ย้ายอย่างเดียว) เมื่อใช้สองคู่ จะหมุนวัตถุที่เลือกด้วยเพื่อให้ทิศทางต้นทาง-ไปยัง-ต้นทางตรงกับทิศทางปลายทาง-ไปยัง-ปลายทาง และสามารถปรับขนาด (ไม่บังคับ) ให้ความยาวส่วนต้นทางตรงกับความยาวส่วนปลายทางได้ — ย้าย หมุน และปรับขนาดในการดำเนินการเดียว

## สองวิธีเริ่มต้น

**เลือกก่อน แล้วจัดแนว** — เลือกวัตถุก่อน แล้วเปิดใช้งาน:

1. เลือกวัตถุหนึ่งหรือมากกว่าบนแคนวาส
2. พิมพ์ `align` ในเทอร์มินัลหรือคลิกปุ่ม **Align** บนแถบเครื่องมือ
3. **คลิกจุดต้นทางแรก (S1)** แล้ว **คลิกจุดปลายทางแรก (D1)**
4. **คลิกจุดต้นทางที่สอง (S2)** หรือกด **Enter** เพื่อใช้การจัดแนวแบบย้ายอย่างเดียวทันที
5. **คลิกจุดปลายทางที่สอง (D2)**
6. ตอบพร้อมท์การปรับขนาด: กด **Y** เพื่อปรับขนาด หรือ **N** / **Enter** เพื่อคงขนาดเดิม

**เปิดใช้งานก่อน แล้วเลือก** — เริ่มคำสั่งโดยไม่มีการเลือก:

1. พิมพ์ `align` หรือคลิกปุ่มแถบเครื่องมือ
2. **เลือกวัตถุ** — คลิกเพื่อสลับวัตถุแต่ละชิ้น หรือลากเพื่อเลือกตามพื้นที่
3. กด **Enter** หรือ **Space** เพื่อยืนยันการเลือก
4. ดำเนินต่อด้วย S1 → D1 → S2 → D2 → พร้อมท์การปรับขนาดตามด้านบน

> เทอร์มินัลต้องการเพียงตัวอักษรที่เพียงพอเพื่อไม่ให้กำกวม — การพิมพ์ `al` แล้วกด **Enter** จะเปิดใช้งาน Align ได้โดยตรง เนื่องจากไม่มีคำสั่งอื่นใดที่ขึ้นต้นด้วยสองตัวอักษรนี้

## กายวิภาคของการจัดแนว

```
  จุดต้นทาง (บนวัตถุ):                 จุดปลายทาง:
  ● S1                                 ● D1
   \                                    \
    ● S2                                 ● D2

  ผลลัพธ์: วัตถุที่เลือกจะถูกย้ายให้ S1 ไปตกที่ D1 จากนั้นหมุนรอบ
  D1 ให้ทิศทาง S1→S2 ตรงกับทิศทาง D1→D2 — และหากคุณเลือกที่จะ
  ปรับขนาด จะถูกปรับขนาดให้ |S1S2| กลายเป็น |D1D2|
```

ตัวอย่างเงาจะติดตามเคอร์เซอร์ในทุกขั้นตอน: ตัวอย่างการย้ายขณะวาง D1 จากนั้นตัวอย่างการหมุนแบบเส้นประเมื่อกำลังวาง D2

## การจัดแนวจุดเดียว (ย้ายอย่างเดียว)

หลังจากวาง D1 แล้ว ให้กด **Enter** แทนการคลิกจุดต้นทางที่สอง วัตถุที่เลือกจะถูกย้ายตามเวกเตอร์ S1→D1 — โดยไม่มีการหมุนหรือปรับขนาด — เหมือนกับ [Move](../move/) ที่ใช้ S1 เป็นจุดฐานและ D1 เป็นปลายทางทุกประการ

## การจัดแนวสองจุด (ย้าย + หมุน + ปรับขนาดไม่บังคับ)

เมื่อ S2 และ D2 ถูกวางแล้วทั้งคู่:

- **มุมหมุน** — ผลต่างระหว่างทิศทางปลายทาง (`D1 → D2`) และทิศทางต้นทาง (`S1 → S2`)
- **พร้อมท์การปรับขนาด** — จะแสดง `scale objects to alignment points? [Yes/No] <N>` โดยค่าเริ่มต้นคือ **No**:
  - กด **Y** เพื่อปรับขนาดวัตถุที่เลือกรอบ D1 อย่างสม่ำเสมอด้วย เพื่อให้ระยะ `S1–S2` กลายเป็นระยะ `D1–D2`
  - กด **N** หรือ **Enter** เพื่อคงขนาดเดิม — จะใช้เฉพาะการย้ายและการหมุนเท่านั้น

การกดปุ่มใด ๆ ที่พร้อมท์การปรับขนาดจะใช้การจัดแนวทันที — ไม่มีขั้นตอนยืนยันแยกต่างหากหลังจากเลือก Yes หรือ No

## อ้างอิงแป้นพิมพ์

| ปุ่ม | การกระทำ |
|-----|----------|
| `Enter` / `Space` | ยืนยันการเลือกและไปยังระยะ S1 |
| `Enter` (ที่ขั้นตอน S2) | ข้ามการหมุน — ใช้การจัดแนวแบบย้ายอย่างเดียวโดยใช้ S1 และ D1 |
| `Y` | ใช้การจัดแนวพร้อมการปรับขนาด |
| `N` / `Enter` (ที่พร้อมท์การปรับขนาด) | ใช้การจัดแนวโดยไม่ปรับขนาด |
| `Escape` | ขณะวางจุด: ยกเลิกจุดที่วางไว้และกลับสู่ระยะการเลือก; เมื่อไม่มีการเลือก: ยกเลิกคำสั่ง |

## การเลือกระหว่างคำสั่ง

เมื่อคำสั่งเริ่มในระยะการเลือก:

| วิธี | พฤติกรรม |
|------|-----------|
| **คลิก** | สลับวัตถุใต้เคอร์เซอร์เข้า/ออกจากการเลือก |
| **ลากไปขวา** (เข้มงวด) | เพิ่มวัตถุที่อยู่ในกล่องทั้งหมด |
| **ลากไปซ้าย** (ตัดผ่าน) | เพิ่มวัตถุที่ตัดขอบกล่อง |
| **Enter** / **Space** | ยืนยันการเลือกและย้ายไปยังระยะ S1 |

## หลังจากจัดแนว

วัตถุที่จัดแนวแล้วยังคงถูกเลือกที่ตำแหน่งใหม่ และคำสั่งจะสิ้นสุดโดยอัตโนมัติ — รัน **Align** อีกครั้ง หรือสลับไปยัง [Move](../move/), [Rotate](../rotate/), หรือ [Scale](../scale/) โดยไม่ต้องเลือกใหม่

## Align เทียบกับ Move

| | Align | Move |
|---|-------|------|
| คู่จุด | 1 คู่ (ย้ายอย่างเดียว) หรือ 2 คู่ (ย้าย + หมุน + ปรับขนาด) | 1 คู่ (ย้ายอย่างเดียว) |
| การหมุน | ได้ หากมีคู่จุดที่สอง | ไม่ได้ |
| การปรับขนาด | ไม่บังคับ หากมีคู่จุดที่สอง | ไม่ได้ |
| เหมาะสำหรับ | จัดรูปทรงหนึ่งให้เข้ากับอีกรูปทรงโดยใช้จุดอ้างอิง | การวางตำแหน่งใหม่แบบง่าย |

## วัตถุที่รองรับ

Align ทำงานกับวัตถุทุกประเภทที่ Move, Rotate และ Scale รองรับ — การดำเนินการ `translate`, `rotate` และ `scale` แบบเดียวกับที่คำสั่งเหล่านั้นใช้จะถูกนำไปใช้ตามลำดับ ดังนั้นจึงไม่มีการยกเว้น
