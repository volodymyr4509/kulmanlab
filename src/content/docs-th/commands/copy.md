# Copy

คำสั่ง `copy` สร้างสำเนาที่แปลตำแหน่งของวัตถุที่เลือกและวางในออฟเซ็ตจากจุดฐานไปยังปลายทาง — ต้นฉบับยังอยู่ในตำแหน่งเดิม นี่คือความแตกต่างสำคัญจาก [Move](../move/): Copy เพิ่มวัตถุใหม่ลงในภาพวาด ส่วน Move ย้ายที่อยู่ของวัตถุที่มีอยู่

## สองวิธีเริ่มต้น

**เลือกก่อน แล้วสำเนา** — เลือกวัตถุก่อน แล้วเปิดใช้งาน:

1. เลือกวัตถุหนึ่งหรือมากกว่าบนแคนวาส
2. พิมพ์ `copy` ในเทอร์มินัลหรือคลิกปุ่ม **Copy** บนแถบเครื่องมือ
3. **คลิกจุดฐาน** หรือพิมพ์ `X,Y` แล้วกด **Enter** เพื่อป้อนพิกัดที่แน่นอน
4. **คลิกปลายทาง** — สำเนาปรากฏที่ออฟเซ็ตฐาน→ปลายทาง การป้อนพิกัดก็ใช้ได้ที่นี่เช่นกัน

**เปิดใช้งานก่อน แล้วเลือก** — เริ่มคำสั่งโดยไม่มีการเลือก:

1. พิมพ์ `copy` หรือคลิกปุ่มแถบเครื่องมือ
2. **เลือกวัตถุ** — คลิกเพื่อสลับวัตถุแต่ละชิ้น หรือลากเพื่อเลือกตามพื้นที่
3. กด **Enter** หรือ **Space** เพื่อยืนยันการเลือก
4. **คลิกจุดฐาน** แล้ว **คลิกปลายทาง** (การป้อนพิกัดใช้ได้ทั้งสองขั้นตอน)

```
  ก่อน:               หลัง:
  [วัตถุ A]            [วัตถุ A]  ← ต้นฉบับไม่เปลี่ยน
  [วัตถุ B]            [วัตถุ B]
                       [สำเนาของ A] ← วัตถุใหม่
                       [สำเนาของ B]
```

ตัวอย่างเงาของสำเนาติดตามเคอร์เซอร์จากจุดฐานไปยังปลายทาง

## การป้อนพิกัด

ที่ขั้นตอนจุดฐานหรือปลายทาง พิมพ์ตำแหน่งที่แน่นอนแทนการคลิก:

1. พิมพ์ค่า X
2. กด `,` — เทอร์มินัลแสดง `[X], [Y{cursor}]`
3. พิมพ์ค่า Y
4. กด **Enter** เพื่อยืนยัน

## การล็อกมุมและระยะที่แน่นอน

หลังจากตั้งจุดฐาน คำสั่งจะสแนปกับแกน 45° (0°, 45°, 90°, 135°, …) เมื่อเคอร์เซอร์อยู่ไกลพอและใกล้แกน ขณะล็อก พิมพ์ระยะแล้วกด **Enter** เพื่อวางสำเนาในออฟเซ็ตนั้นพอดี

| ปุ่ม | การกระทำ |
|-----|----------|
| `0`–`9`, `.` | เพิ่มเข้าไปในค่าระยะ |
| `-` | ระยะติดลบ — กลับทิศทางตามแกน (อักขระตัวแรกเท่านั้น) |
| `Backspace` | ลบอักขระที่พิมพ์ล่าสุด |
| `Enter` | วางสำเนาที่ระยะที่พิมพ์ |

## อ้างอิงแป้นพิมพ์

| ปุ่ม | การกระทำ |
|-----|----------|
| `Enter` / `Space` | ยืนยันการเลือกและดำเนินการไปยังระยะจุดฐาน |
| `0`–`9`, `.`, `-` | เริ่มป้อนพิกัด X หรือระยะเมื่อล็อกมุม |
| `,` | ล็อก X และย้ายไปป้อน Y |
| `Backspace` | ลบอักขระที่พิมพ์ล่าสุด |
| `Enter` | ยืนยันพิกัดหรือใช้สำเนาที่ระยะที่พิมพ์ |
| `Escape` | ยกเลิกและรีเซ็ต |

## การเลือกระหว่างคำสั่ง

| วิธี | พฤติกรรม |
|------|-----------|
| **คลิก** | สลับวัตถุใต้เคอร์เซอร์เข้า/ออกจากการเลือก |
| **ลากไปขวา** (เข้มงวด) | เพิ่มวัตถุที่อยู่ในกล่องทั้งหมด |
| **ลากไปซ้าย** (ตัดผ่าน) | เพิ่มวัตถุที่ตัดขอบกล่อง |
| **Enter** / **Space** | ยืนยันการเลือก |

## หลังจากสำเนา

**ต้นฉบับยังคงถูกเลือก** — สำเนาใหม่ถูกเพิ่มลงในภาพวาดแต่การเลือกถูกล้างและคำสั่งออก หากต้องการทำงานกับสำเนาทันที ให้รัน Copy อีกครั้งบนการเลือก หรือเริ่มคำสั่งใหม่

## Copy vs Move

| | Copy | Move |
|---|------|------|
| ต้นฉบับ | อยู่ในตำแหน่งเดิม | ถูกลบออกจากตำแหน่งเดิม |
| จำนวนผลลัพธ์ | เพิ่มขึ้นตามจำนวนวัตถุที่คัดลอก | ไม่เปลี่ยน |
| หลังการทำงาน | ต้นฉบับยังถูกเลือก | วัตถุที่ย้ายถูกเลือกในตำแหน่งใหม่ |
| เหมาะสำหรับ | ทำซ้ำเรขาคณิต เค้าโครงสมมาตร | ย้ายตำแหน่งเรขาคณิต |

## วัตถุที่รองรับ

Copy ทำงานกับวัตถุทุกประเภท วัตถุทั้งหมดใช้ `translate(dx, dy)` ภายในดังนั้นไม่มีการยกเว้น
