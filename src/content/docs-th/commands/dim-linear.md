# Dimension Linear

คำสั่ง `dimlinear` วางมิติแนวนอนหรือแนวตั้งระหว่างจุดกำเนิดสองเส้นขยาย เส้นมิติวิ่งแนวนอนหรือแนวตั้งสมบูรณ์เสมอ — ไม่สามารถวางในมุมใดก็ได้ ใช้ [Dimension Aligned](./dim-aligned) เมื่อต้องการมิติขนานกับเส้นเฉียง

## กายวิภาคของมิติ linear

```
  |←————— 5.00 —————→|
  |                   |
  ●  (เส้นขยาย 1)     ●  (เส้นขยาย 2)
  p1                  p2
```

- **เส้นขยาย** — ลงจากแต่ละจุดที่วัดตั้งฉากกับเส้นมิติ
- **เส้นมิติ** — แนวนอน (วัดระยะ X) หรือแนวตั้ง (วัดระยะ Y)
- **ค่า** — ระยะที่ฉายตามแกนที่เลือก ไม่ใช่ระยะจุดต่อจุดจริง

## การวาง linear dimension

1. พิมพ์ `dimlinear` ในเทอร์มินัลหรือคลิกปุ่ม **Dimension Linear** บนแถบเครื่องมือ
2. **คลิกจุดกำเนิดเส้นขยายแรก** (p1) หรือพิมพ์ `X,Y` แล้วกด **Enter** เพื่อป้อนพิกัดที่แน่นอน
3. **คลิกจุดกำเนิดเส้นขยายที่สอง** (p2) การป้อนพิกัดก็ใช้ได้ที่นี่เช่นกัน
4. **ย้ายเคอร์เซอร์** เพื่อวางตำแหน่งเส้นมิติ ทิศทางถูกตรวจจับอัตโนมัติจากตำแหน่งเคอร์เซอร์
5. **คลิก** เพื่อวาง หรือพิมพ์ระยะออฟเซ็ตแล้วกด **Enter** เพื่อวางอย่างแม่นยำ

## การตรวจจับทิศทางอัตโนมัติ

เมื่อไม่มีการบังคับทิศทาง คำสั่งอ่านตำแหน่งเคอร์เซอร์สัมพันธ์กับสองจุดที่วัด:

| ตำแหน่งเคอร์เซอร์ | ทิศทางที่ตรวจจับ | สิ่งที่วัด |
|------------------|-----------------|------------|
| ด้านบนหรือล่างจุด | แนวนอน | Δ X ระหว่าง p1 และ p2 |
| ด้านซ้ายหรือขวาของจุด | แนวตั้ง | Δ Y ระหว่าง p1 และ p2 |

กด **H** เพื่อล็อกแนวนอนหรือ **V** เพื่อล็อกแนวตั้งได้ตลอดเวลาระหว่างระยะวาง เมื่อล็อกแล้ว ทิศทางไม่เปลี่ยนขณะย้ายเคอร์เซอร์

## ระยะออฟเซ็ตแบบพิมพ์

พิมพ์ตัวเลขระหว่างการวางเพื่อตรึงเส้นมิติที่ระยะที่แน่นอนจากจุดที่วัด:

| ปุ่ม | การกระทำ |
|-----|----------|
| `0`–`9`, `.` | เพิ่มตัวเลขเข้าไปในระยะออฟเซ็ต |
| `Backspace` | ลบอักขระที่พิมพ์ล่าสุด |
| `Enter` / `Space` | วางที่ระยะที่พิมพ์ |

ด้านเคอร์เซอร์ (บน/ล่างสำหรับแนวนอน ซ้าย/ขวาสำหรับแนวตั้ง) กำหนดเครื่องหมาย — เส้นมิติปรากฏด้านที่เคอร์เซอร์อยู่ปัจจุบัน

## อ้างอิงแป้นพิมพ์

| ปุ่ม | การกระทำ |
|-----|----------|
| `0`–`9`, `.`, `-` | เริ่มป้อนพิกัด X (ระยะ p1/p2) หรือระยะออฟเซ็ต (ระยะวาง) |
| `,` | ล็อก X และย้ายไปป้อน Y (ระยะ p1/p2) |
| `H` | ล็อกทิศทางแนวนอน (เฉพาะระยะวาง) |
| `V` | ล็อกทิศทางแนวตั้ง (เฉพาะระยะวาง) |
| `Backspace` | ลบอักขระที่พิมพ์ล่าสุด |
| `Enter` / `Space` | ยืนยันพิกัดหรือออฟเซ็ตที่พิมพ์ |
| `Escape` | ยกเลิก |

## Dimension Linear vs Dimension Aligned

| | Dimension Linear | Dimension Aligned |
|---|-----------------|------------------|
| แกน | แนวนอนหรือแนวตั้งเสมอ | ขนานกับเส้นที่วัด |
| วัด | ส่วนประกอบ X หรือ Y เท่านั้น | ระยะ Euclidean จริง |
| ปุ่ม H/V | ใช่ — ล็อกทิศทาง | ไม่ — ติดตาม p1→p2 เสมอ |
| เหมาะสำหรับ | เค้าโครงตั้งฉาก แผนพื้น | ลักษณะเฉียง การตัดมุม |

## การต่อเชื่อมมิติ

หากต้องการเพิ่มมิติต่อเนื่องจากเส้นขยายสุดท้าย ใช้ [Dimension Continue](./dim-continue) ทันทีหลังจากวางมิตินี้

## DXF — วัตถุ DIMENSION

มิติ linear ถูกบันทึกเป็นวัตถุ `DIMENSION` ที่มี `rotationDeg` ตั้งเป็น `0` (แนวนอน) หรือ `90` (แนวตั้ง) จุดกำเนิดเส้นขยาย ตำแหน่งเส้นมิติ ตำแหน่งข้อความ ค่าที่วัด รูปแบบลูกศร ความสูงข้อความ และแฟล็กแสดงผลทั้งหมดรอบเที่ยวโดยไม่สูญเสียข้อมูล
