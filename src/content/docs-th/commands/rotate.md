# Rotate

คำสั่ง `rotate` หมุนวัตถุที่เลือกรอบจุดฐาน คุณระบุมุมการหมุนโดยพิมพ์ตัวเลขเป็นองศาหรือโดยการคลิก — มุมจะคำนวณจากทิศทางระหว่างจุดฐานและตำแหน่งที่คลิก

## สองวิธีเริ่มต้น

**เลือกก่อน แล้วหมุน** — เลือกวัตถุก่อน แล้วเปิดใช้งาน:

1. เลือกวัตถุหนึ่งหรือมากกว่าบนแคนวาส
2. พิมพ์ `rotate` ในเทอร์มินัลหรือคลิกปุ่ม **Rotate** บนแถบเครื่องมือ
3. **คลิกจุดฐาน** — ศูนย์กลางการหมุน หรือพิมพ์ `X,Y` แล้วกด **Enter** เพื่อป้อนพิกัดที่แน่นอน
4. **พิมพ์มุมแล้วกด Enter** หรือ **คลิก** เพื่อตั้งมุมจากทิศทางเคอร์เซอร์

**เปิดใช้งานก่อน แล้วเลือก** — เริ่มคำสั่งโดยไม่มีการเลือก:

1. พิมพ์ `rotate` หรือคลิกปุ่มแถบเครื่องมือ
2. **เลือกวัตถุ** — คลิกเพื่อสลับ หรือลากเพื่อเลือกตามพื้นที่
3. กด **Enter** หรือ **Space** เพื่อยืนยันการเลือก
4. **คลิกจุดฐาน** (การป้อนพิกัดพร้อมใช้) แล้วตั้งมุม

```
  ก่อน:               หลัง (หมุน 90° รอบ ●):
                          ╔══╗
  ●  [วัตถุ]    →    ● ║    ║
                          ╚══╝
```

ตัวอย่างเงาของวัตถุที่หมุนแล้วติดตามมุมเคอร์เซอร์หลังจากตั้งจุดฐาน

## การตั้งมุม

**มุมที่พิมพ์** — พิมพ์ตัวเลข (เป็นองศา) ได้ตลอดเวลาหลังวางจุดฐาน ตัวอย่างสแนปกับมุมที่พิมพ์ขณะที่คุณยังปรับก่อนกด Enter

**มุมที่คลิก** — ถ้าไม่มีค่าที่พิมพ์ การคลิกจะตั้งมุมเท่ากับ `atan2(cursorY − baseY, cursorX − baseX)` — ทิศทางจากจุดฐานถึงการคลิก เป็นองศา

| ปุ่ม | การกระทำ |
|-----|----------|
| `0`–`9`, `.` | เพิ่มตัวเลขเข้าค่ามุม |
| `-` | มุมติดลบ (อักขระตัวแรกเท่านั้น) |
| `Backspace` | ลบอักขระที่พิมพ์ล่าสุด |
| `Enter` | ใช้การหมุนที่มุมที่พิมพ์ |

## ทิศทางมุม

มุมเป็นไปตาม **ข้อกำหนด DXF**:

- ค่า **บวก** หมุน **ทวนเข็มนาฬิกา** ในพิกัดการวาด (Y ขึ้น)
- บนหน้าจอ ที่แกน Y กลับด้าน (Y ลง) มุมบวกจะปรากฏเป็น **ตามเข็มนาฬิกา**

ค่าทั่วไป: `90` = หมุนหนึ่งในสี่รอบ, `180` = หมุนครึ่งรอบ, `-90` = หมุนหนึ่งในสี่รอบอีกทิศ

## อ้างอิงแป้นพิมพ์

| ปุ่ม | การกระทำ |
|-----|----------|
| `Enter` / `Space` | ยืนยันการเลือก |
| `0`–`9`, `.`, `-` | เริ่มป้อนพิกัด X (ระยะจุดฐาน) หรือค่ามุม (ระยะมุม) |
| `,` | ล็อก X และย้ายไปป้อน Y (ระยะจุดฐาน) |
| `Backspace` | ลบอักขระที่พิมพ์ล่าสุด |
| `Enter` | ยืนยันพิกัดหรือใช้การหมุน |
| `Escape` | ยกเลิกและรีเซ็ต |

## การเลือกระหว่างคำสั่ง

| วิธี | พฤติกรรม |
|------|-----------|
| **คลิก** | สลับวัตถุใต้เคอร์เซอร์ |
| **ลากไปขวา** (เข้มงวด) | เพิ่มวัตถุที่อยู่ในกล่องทั้งหมด |
| **ลากไปซ้าย** (ตัดผ่าน) | เพิ่มวัตถุที่ตัดขอบกล่อง |
| **Enter** / **Space** | ยืนยันการเลือก |

## วัตถุที่รองรับ

Rotate ทำงานกับวัตถุทุกประเภท เรขาคณิตของวัตถุแต่ละชิ้นจะหมุนรอบจุดฐาน — ตัวอย่างเช่น วงกลมจะย้ายจุดศูนย์กลางขณะที่รัศมีเท่าเดิม ส่วนโค้งจะย้ายจุดศูนย์กลางและเปลี่ยนมุมเริ่มต้นและสิ้นสุดตามปริมาณการหมุน วัตถุ Text จะย้ายจุดยึดและเพิ่มมุมให้กับคุณสมบัติ Rotation Degree
