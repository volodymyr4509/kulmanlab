# Trim

คำสั่ง `trim` ลบส่วนของ [Line](./line) ที่อยู่ระหว่างจุดตัดสองจุดที่อยู่ติดกัน โดยแบ่งเส้นออกเป็นหนึ่งหรือสองส่วนที่สั้นกว่า ส่วนที่จะตัดถูกกำหนดโดยตำแหน่งเคอร์เซอร์ — วาง cursor บนส่วนที่ต้องการลบแล้วคลิกเพื่อตัด

Trim ทำงานกับ **Line entities เท่านั้น** สำหรับ arcs, circles, polylines และประเภทวัตถุอื่นๆ ให้ใช้ [Delete](./delete) หรือการแก้ไขกริปแทน

## การตัดเส้น

1. พิมพ์ `trim` ในเทอร์มินัลหรือคลิกปุ่ม **Trim** บนแถบเครื่องมือ
2. **วาง cursor บนส่วนของเส้น** ที่ต้องการลบ — ตัวอย่างสีแดงจะไฮไลต์ส่วนที่จะถูกตัดพอดี
3. **คลิก** เพื่อลบส่วนนั้น

คำสั่งยังคงทำงานหลังจากแต่ละการตัด ดังนั้นคุณสามารถ hover และคลิกต่อไปเพื่อตัดส่วนเพิ่มเติม กด **Escape** เพื่อออก

```
  ก่อน:                         หลังตัดส่วนกลาง:

  ──────●──────●──────          ──────●          ●──────
      ตัดกัน   ตัดกัน          (ส่วนซ้าย)    (ส่วนขวา)
                                            (ส่วนกลางถูกลบ)
```

## วิธีกำหนดส่วนที่ตัด

คำสั่งฉายตำแหน่งเคอร์เซอร์ลงบนเส้นที่ hover และหาจุดตัดทั้งหมดที่เส้นมีกับวัตถุอื่น พารามิเตอร์จุดตัดเหล่านี้แบ่งเส้นออกเป็นส่วนต่างๆ ส่วนที่ช่วงของมันมีการฉาย cursor จะถูกไฮไลต์และจะถูกลบเมื่อคลิก

- ถ้าเคอร์เซอร์อยู่ **ก่อนจุดตัดแรก**: ส่วนนำของเส้นนั้นถูกลบ
- ถ้าเคอร์เซอร์อยู่ **ระหว่างจุดตัดสองจุด**: ส่วนกลางนั้นถูกลบ เส้นแบ่งออกเป็นสอง
- ถ้าเคอร์เซอร์อยู่ **หลังจุดตัดสุดท้าย**: ส่วนท้ายนั้นถูกลบ
- ถ้าเส้น **ไม่มีจุดตัด** กับวัตถุใดๆ: ไม่มีตัวอย่างและการคลิกไม่ทำอะไร

## อ้างอิงแป้นพิมพ์

| ปุ่ม | การกระทำ |
|-----|----------|
| `Escape` | ออกจากโหมด trim |

## วัตถุที่รองรับ

| วัตถุ | ตัดได้หรือไม่ |
|-------|--------------|
| Line | ใช่ |
| Arc, Circle, Ellipse | ไม่ |
| Polyline / Rectangle | ไม่ |
| Text, Spline, Dimension, Leader | ไม่ |

วัตถุที่ใช้เป็น **ขอบตัด** สามารถเป็นประเภทใดก็ได้ — เฉพาะเส้นที่กำลังตัดเท่านั้นที่ต้องเป็น Line entity

## Trim vs Extend

| | Trim | Extend |
|---|------|--------|
| สิ่งที่ทำ | ลบส่วนของเส้น | ยืดจุดสิ้นสุดของเส้นไปถึงขอบ |
| ตัวกระตุ้น | วาง cursor บนส่วนที่จะตัด | วาง cursor ใกล้จุดสิ้นสุดที่จะยืด |
| ผลลัพธ์ | เส้นแบ่งหรือสั้นลง | จุดสิ้นสุดของเส้นย้ายไปที่ขอบ |
| ทั้งสองอย่าง | เส้นเท่านั้น | เส้นเท่านั้น |
