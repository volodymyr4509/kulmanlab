# Export

คำสั่ง **Export** ดาวน์โหลดภาพวาดปัจจุบันไปยังระบบไฟล์ของคุณ มีสองรูปแบบ: **DXF** สำหรับความเข้ากันได้กับเครื่องมือ CAD อื่น และ **JSON** สำหรับการบันทึกความเที่ยงตรงเต็มรูปแบบภายใน KulmanLab CAD

## วิธีส่งออก

1. คลิกปุ่ม **Export** บนแถบเครื่องมือ (ไอคอนดาวน์โหลด) ในแผง File
2. หน้าต่างป๊อปอัพ **Export Manager** เปิด
3. คลิกการ์ดรูปแบบเพื่อเลือกรูปแบบ — **JSON** หรือ **DXF**
4. คลิกปุ่ม **Export** ไฟล์ดาวน์โหลดไปยังโฟลเดอร์ดาวน์โหลดเริ่มต้นของคุณโดยอัตโนมัติ

## การเลือกรูปแบบ

| รูปแบบ | นามสกุล | เหมาะสำหรับ | ข้อจำกัด |
|--------|---------|------------|---------|
| **JSON** *(native)* | `.json` | บันทึกงานเพื่อเปิดใน KulmanLab CAD | ไม่รองรับกับเครื่องมือ CAD อื่น |
| **DXF** | `.dxf` | แชร์กับ FreeCAD, LibreCAD ฯลฯ | มิติและ leader ไม่ถูกส่งออก |

**เมื่อใช้ JSON:** ทุกครั้งที่ต้องการบันทึกสำเนาสมบูรณ์ของงาน JSON เป็นรูปแบบดั้งเดิมของ KulmanLab และเก็บรักษาทุกวัตถุอย่างแน่นอน — รวมถึงมิติ leader และข้อมูลเลเยอร์ทั้งหมด

**เมื่อใช้ DXF:** เมื่อต้องการส่งมอบภาพวาดให้ผู้ใช้แอปพลิเคชัน CAD อื่น ไฟล์ที่ส่งออกใช้รูปแบบ DXF AC1012 และสามารถเปิดในเครื่องมือที่รองรับ DXF ส่วนใหญ่

## สิ่งที่ส่งออกต่อรูปแบบ

### การส่งออก JSON

วัตถุทุกประเภทรวมอยู่:

- Lines, circles, arcs, ellipses, polylines, splines, text
- Dimensions (linear, aligned, continued, radius, diameter)
- Multileaders
- คำจำกัดความเลเยอร์และตาราง linetype

### การส่งออก DXF

วัตถุเฉพาะเรขาคณิตรวมอยู่:

- Lines, circles, arcs, ellipses, polylines (ส่งออกเป็น `LWPOLYLINE`), splines, text
- คำจำกัดความเลเยอร์และตาราง linetype

**ไม่รวมใน DXF:** วัตถุมิติและ multileader สิ่งเหล่านี้ใช้โครงสร้างข้อมูลเฉพาะ KulmanLab ที่ไม่สามารถแสดงได้อย่างถูกต้องใน DXF มาตรฐาน หากภาพวาดของคุณมีคำอธิบายประกอบ ให้ใช้ JSON หรือ [Print](../print/) เพื่อจับภาพเป็นภาพ

## ชื่อไฟล์ที่ส่งออก

ไฟล์ที่ดาวน์โหลดตั้งชื่อตามไฟล์ภาพวาดปัจจุบัน (เช่น `myplan_May22_14:30:00.json`) นามสกุลเปลี่ยนตามรูปแบบที่เลือก

## ความแตกต่างระหว่าง Export และ Print

| คุณลักษณะ | Export | Print |
|-----------|--------|-------|
| ผลลัพธ์ | ไฟล์ต้นทางเวกเตอร์ (.dxf / .json) | รูปภาพ raster (.png / .jpeg / .webp / .pdf) |
| แก้ไขได้ในเครื่องมืออื่น | ใช่ (DXF) | ไม่ |
| เก็บรักษาเลเยอร์และ linetypes | ใช่ | ไม่ (เรนเดอร์แบบเรียบ) |
| จับมิติและ leader | JSON เท่านั้น | ใช่ |

ใช้ **Export** เมื่อต้องการไฟล์ที่แก้ไขได้ ใช้ [Print](../print/) เมื่อต้องการสแนปช็อตภาพ

## คำสั่งที่เกี่ยวข้อง

- [Import](../import/) — เปิดไฟล์ DXF หรือ JSON
- [Print](../print/) — ส่งออกแคนวาสเป็นรูปภาพ PNG, JPEG, WebP หรือ PDF
- [Files](../files/) — เรียกดูภาพวาดที่บันทึกในหน่วยความจำเบราว์เซอร์
