# Extend

คำสั่ง `extend` ยืดจุดสิ้นสุดที่ใกล้ที่สุดของ [Line](./line) ไปยังจุดตัดที่ใกล้ที่สุดที่จะเกิดกับวัตถุอื่นในภาพวาด วางเมาส์ใกล้จุดสิ้นสุดที่ต้องการยืด — ตัวอย่างแสดงเส้นที่ยืดแล้ว — แล้วคลิกเพื่อใช้งาน

Extend ทำงานกับ **วัตถุ Line เท่านั้น** ขอบเขตที่เส้นยืดไปยังสามารถเป็นวัตถุประเภทใดก็ได้ยกเว้น Text, Mtext, Multileader และ Spline

## การยืดเส้น

1. พิมพ์ `extend` ในเทอร์มินัลหรือคลิกปุ่ม **Extend** บนแถบเครื่องมือ
2. **วางเมาส์ใกล้ปลายด้านหนึ่งของเส้น** — ตัวอย่างแสดงเส้นที่ยืดไปยังขอบเขตที่ใกล้ที่สุดในทิศทางนั้น
3. **คลิก** เพื่อใช้งานการยืด

คำสั่งยังคงทำงานหลังจากการยืดแต่ละครั้งเพื่อให้คุณยืดหลายเส้นต่อเนื่อง กด **Escape** เพื่อออก

```
  ก่อน:                      หลัง:

  ──────           |           ──────────────|
  (เส้นสั้น)     (ขอบเขต)  (ยืดไปยังขอบเขต)
```

## วิธีเลือกจุดสิ้นสุด

คำสั่งดูว่าเคอร์เซอร์อยู่ใกล้จุดสิ้นสุดใดมากกว่า:

- เคอร์เซอร์ **ใกล้จุดสิ้นสุด** → จุดสิ้นสุดถูกยืดไปข้างหน้าตามทิศทางเส้น
- เคอร์เซอร์ **ใกล้จุดเริ่มต้น** → จุดเริ่มต้นถูกยืดไปข้างหลัง (ทิศตรงข้าม)

รังสีถูกส่งออกจากจุดสิ้นสุดที่เลือกตามทิศทางเส้น และ **จุดตัดที่ใกล้ที่สุด** ตามรังสีนั้นกับวัตถุอื่น (ยกเว้นเส้นเองและประเภทที่ถูกเพิกเฉย) กลายเป็นจุดสิ้นสุดใหม่

หากไม่พบจุดตัดในทิศทางนั้น ไม่มีตัวอย่างปรากฏและการคลิกไม่มีผล

## การยกเว้นขอบเขต

ประเภทวัตถุต่อไปนี้ถูกเพิกเฉยเป็นขอบเขต — เส้นไม่ยืดไปพบพวกมัน:

- Text / Mtext
- Multileader
- Spline

ประเภทอื่นทั้งหมด (Line, Arc, Circle, Ellipse, Polyline, Dimension) ทำหน้าที่เป็นขอบเขตที่ถูกต้อง

## อ้างอิงแป้นพิมพ์

| ปุ่ม | การกระทำ |
|-----|----------|
| `Escape` | ออกจากโหมด extend |

## วัตถุที่รองรับ

| วัตถุ | สามารถยืดได้? |
|-------|-------------|
| Line | ใช่ |
| Arc, Circle, Ellipse | ไม่ |
| Polyline / Rectangle | ไม่ |
| Text, Spline, Dimension, Leader | ไม่ |

## Extend vs Trim

| | Extend | Trim |
|---|--------|------|
| สิ่งที่ทำ | ยืดจุดสิ้นสุดของเส้นไปยังขอบเขต | ลบส่วนของเส้น |
| ทริกเกอร์ | วางเมาส์ใกล้จุดสิ้นสุดที่จะยืด | วางเมาส์เหนือส่วนที่จะตัด |
| ผลลัพธ์ | จุดสิ้นสุดเส้นย้ายออก | เส้นแยกหรือสั้นลง |
| ทั้งคู่ | Lines เท่านั้น | Lines เท่านั้น |
