# Dimension Continue

คำสั่ง `dimcontinue` ต่อเชื่อมมิติใหม่จาก **เส้นขยายที่สอง** ของมิติที่มีอยู่ แต่ละส่วนใหม่ถูกวางตามแกนการวัดเดียวกันและที่ออฟเซ็ตเส้นมิติเดียวกับฐาน คุณสมบัติสไตล์ทั้งหมด — ขนาดลูกศร ความสูงข้อความ ความยาวเส้นขยาย — ถูกคัดลอกจากฐานโดยอัตโนมัติ

## ลักษณะของมิติต่อเชื่อม

```
  |←— 3.00 —→|←— 2.50 —→|←— 4.00 —→|
  |           |           |           |
  ●           ●           ●           ●
  p1        p2 (ฐาน    p3           p4
           ext2 → จุดเริ่มต้นใหม่)
```

แต่ละสี่เหลี่ยมเป็นวัตถุ `DIMENSION` แยกกัน พวกมันแชร์ตำแหน่งเส้นมิติและทิศทางการวัดเดียวกัน

## การเริ่มสาย

1. พิมพ์ `dimcontinue` ในเทอร์มินัลหรือคลิกปุ่ม **Dimension Continue** บนแถบเครื่องมือ
2. **ถ้าวางมิติเพิ่งก่อนหน้า** — คำสั่งเลือกโดยอัตโนมัติเป็นฐาน (ไม่ต้องคลิก)
3. **ถ้าไม่มีมิติล่าสุด** — คลิกมิติที่มีอยู่เพื่อใช้เป็นฐาน
4. **คลิกจุดกำเนิดเส้นขยายถัดไป** — ตัวอย่างแสดงมิติใหม่ขณะย้ายเคอร์เซอร์ หรือพิมพ์ `X,Y` แล้วกด **Enter** เพื่อป้อนพิกัดที่แน่นอน
5. คลิก (หรือพิมพ์) ต่อไปเพื่อขยายสาย มิติที่วางแต่ละตัวกลายเป็นฐานใหม่โดยอัตโนมัติ
6. กด **Enter**, **Space** หรือ **Escape** เพื่อจบสาย

## สิ่งที่รับทอดจากมิติฐาน

| คุณสมบัติ | รับทอดจากฐาน |
|-----------|-------------|
| ทิศทาง/มุมการวัด | ใช่ — ล็อกสำหรับสายทั้งหมด |
| ออฟเซ็ตเส้นมิติ (ระยะจากจุดที่วัด) | ใช่ |
| ขนาดลูกศร | ใช่ |
| ความสูงข้อความ | ใช่ |
| ออฟเซ็ตและส่วนขยายของเส้นขยาย | ใช่ |
| การจัดข้อความ | ใช่ |
| ชื่อสไตล์ | ใช่ |
| Color, Layer | ไม่รับทอด — ใช้เลเยอร์ปัจจุบัน |

## การล็อกทิศทางการวัด

ทิศทางการวัดของสายถูก **ตรึงกับมุมของมิติฐาน**:

- ฐาน Linear (H) → การต่อเนื่องทั้งหมดวัดระยะแนวนอน (Δ X)
- ฐาน Linear (V) → การต่อเนื่องทั้งหมดวัดระยะแนวตั้ง (Δ Y)
- ฐาน Aligned ในมุมใดก็ได้ → การต่อเนื่องทั้งหมดวัดตามมุมเดียวกัน

คุณไม่สามารถเปลี่ยนทิศทางกลางสาย เริ่ม [Dimension Linear](../dim-linear/) หรือ [Dimension Aligned](../dim-aligned/) ใหม่เพื่อกำหนดมิติในทิศทางต่างกัน

## อ้างอิงแป้นพิมพ์

| ปุ่ม | การกระทำ |
|-----|----------|
| `0`–`9`, `.`, `-` | เริ่มป้อนพิกัด X |
| `,` | ล็อก X และย้ายไปป้อน Y |
| `Backspace` | ลบอักขระที่พิมพ์ล่าสุด |
| `Enter` | ยืนยันพิกัดที่พิมพ์ หรือจบสายถ้าไม่มีการป้อน |
| `Space` / `Escape` | จบสาย |

## Dimension Continue vs เริ่มใหม่

| | Dimension Continue | Dimension Linear / Aligned |
|---|-------------------|--------------------------|
| จุดเริ่มต้น | ตรึงที่ ext2 ของฐานล่าสุด | คลิกที่ใดก็ได้ |
| มุม | ล็อกกับฐาน | อิสระ |
| ออฟเซ็ต | รับทอดจากฐาน | ตั้งด้วยเคอร์เซอร์หรือพิมพ์ |
| สไตล์ | รับทอดจากฐาน | สไตล์ปัจจุบัน |
| เหมาะสำหรับ | การวัดสะสมตามแถว | มิติแรกหรือเปลี่ยนทิศทาง |

## การแก้ไขป้ายหลังการวาง — simple mode

**ดับเบิลคลิก** มิติใดในห่วงโซ่เพื่อเปิดโปรแกรมแก้ไขข้อความในโหมด **simple** แต่ละส่วนเป็นอิสระและสามารถแก้ไขได้แยกกัน

| ฟีเจอร์ | การทำงาน |
|---------|----------|
| Bold / Italic / Font / Height | ใช้กับ**ป้ายทั้งหมด**พร้อมกัน |
| การจัดรูปแบบต่อตัวอักษร | ไม่รองรับ |
| `Enter` | ยืนยันค่าและปิดโปรแกรมแก้ไข |
| หลายบรรทัด | ไม่รองรับ |

ดู [Text Editor — simple mode](../../interface/text-editor/#simple-mode) สำหรับข้อมูลอ้างอิงทั้งหมด

## DXF — วัตถุ DIMENSION

แต่ละส่วนในสายถูกจัดเก็บเป็นวัตถุ `DIMENSION` อิสระในไฟล์ DXF พวกมันไม่ได้เชื่อมโยงในไฟล์ — พวกมันแชร์คุณสมบัติเพราะถูกสร้างจากฐานเดียวกัน แต่แต่ละตัวสามารถแก้ไขได้แยกกันหลังจากวาง
