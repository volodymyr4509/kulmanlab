# Spline CV

คำสั่ง `splinecv` วาด **B-spline ลูกบาศก์** โดยวางจุดควบคุม (CVs) เส้นโค้งที่ได้จะดึงดูดไปทาง CV แต่ละตัวแต่ไม่ผ่านพวกมัน — ยกเว้นที่จุดยอดแรกและสุดท้าย ที่ **clamped knots** ยึดเส้นโค้งให้ตรงนั้นพอดี วิธีนี้ให้การควบคุมรูปร่างที่เป็นธรรมชาติ: ดึง CV เพื่อผลักเส้นโค้งไปทางนั้นโดยไม่บังคับให้แตะทุกจุด

## การวาด Spline ด้วยจุดควบคุม

1. พิมพ์ `splinecv` ในเทอร์มินัลหรือคลิกปุ่ม **Spline CV** บนแถบเครื่องมือ
2. **คลิกเพื่อวางจุดควบคุม** — แต่ละคลิกเพิ่มจุดยอด หรือพิมพ์ `X,Y` แล้วกด **Enter** เพื่อป้อนพิกัดที่แน่นอน
3. กด **Enter** เพื่อเสร็จสิ้น (ต้องการอย่างน้อย 2 จุดยอด)

```
  CV ●         ● CV
      \       /
       \     /    ← เส้นโค้งดึงดูดไปทาง CVs
        \   /         แต่ไม่ผ่านพวกมัน
  CV ●   ●   ● CV (เริ่ม/จบ: เส้นโค้งแตะที่นี่)
```

ตัวอย่างสดอัปเดตขณะเคลื่อนเคอร์เซอร์หลังแต่ละจุด แสดงลักษณะของ spline พร้อมจุดถัดไปที่ตำแหน่งเคอร์เซอร์ กด **Escape** เพื่อยกเลิกจุดทั้งหมดที่วางไว้และออก

## การป้อนพิกัด

แทนการคลิก พิมพ์ตำแหน่งที่แน่นอนสำหรับจุดควบคุมใดก็ได้:

1. พิมพ์ค่า X
2. กด `,` — เทอร์มินัลแสดง `[X], [Y{cursor}]`
3. พิมพ์ค่า Y
4. กด **Enter** เพื่อวางจุดยอด

## อ้างอิงแป้นพิมพ์

| ปุ่ม | การกระทำ |
|-----|----------|
| `0`–`9`, `.`, `-` | เริ่มป้อนพิกัด X |
| `,` | ล็อก X และย้ายไปป้อน Y |
| `Backspace` | ลบอักขระที่พิมพ์ล่าสุด |
| `Enter` | ยืนยันพิกัดที่พิมพ์ หรือเสร็จสิ้น spline ถ้าไม่มีการป้อนและมีจุดยอด ≥ 2 |
| `Escape` | ยกเลิกจุดทั้งหมดและออก |

## การแก้ไขกริป — ปรับรูปร่างผ่านจุดควบคุม

CV spline ที่เลือกจะแสดงกริปหนึ่งอันต่อจุดควบคุม:

| กริป | ตำแหน่ง | สิ่งที่ทำ |
|------|---------|---------|
| **จุดควบคุม** | ที่ตำแหน่ง CV แต่ละตัว | ลากเพื่อย้าย CV นั้น — เส้นโค้งจะปรับรูปร่างไปทางตำแหน่งใหม่ |

ไม่มีกริป "ย้าย spline ทั้งหมด" หากต้องการเลื่อน spline ทั้งหมด ให้ใช้คำสั่ง [Move](../move/)

## การเลือก CV Spline

| วิธี | พฤติกรรม |
|------|-----------|
| **คลิก** | เลือกถ้าคลิกลงใกล้จุดใดๆ บนเส้นโค้ง |
| **ลากไปขวา** (เข้มงวด) | จุดตัวอย่างทั้งหมดตามเส้นโค้งต้องอยู่ในกล่องเลือก |
| **ลากไปซ้าย** (ตัดผ่าน) | ส่วนใดของเส้นโค้งที่ตัดขอบกล่องเลือกจะเลือก spline นั้น |

## คุณสมบัติ

**ทั่วไป**

| คุณสมบัติ | ค่าเริ่มต้น | ความหมาย |
|-----------|------------|---------|
| Color | 256 (ByLayer) | ดัชนีสี ACI |
| Layer | `0` | การกำหนดเลเยอร์ |
| Linetype | ByLayer | รูปแบบเส้นประที่มีชื่อ |
| Linetype Scale | 1 | ตัวคูณสเกลบนรูปแบบเส้นประ |
| Thickness | 0 | ความหนาของการยื่น |

**เรขาคณิต**

| คุณสมบัติ | ความหมาย |
|-----------|---------|
| Degree | ดีกรีพหุนาม — เป็น 3 (ลูกบาศก์) เสมอ |
| Control Vertices | พิกัดของ CVs ทั้งหมด |
| Fit Points | ว่างสำหรับ CV spline ใช้กับ fit-point spline เท่านั้น |

## Spline CV vs Spline Fit — ควรใช้อันไหน

| | Spline CV | Spline Fit |
|---|-----------|------------|
| เส้นโค้งผ่านจุด | เฉพาะจุดแรกและสุดท้าย (clamped) | ทุกจุดที่คลิกพอดี |
| การควบคุมรูปร่าง | ดึง CVs ไปทางพื้นที่ | ย้ายจุด fit ที่เส้นโค้งต้องแตะ |
| ผลการแก้ไขกริป | CV เคลื่อน → เส้นโค้งดึงดูด | Fit point เคลื่อน → เส้นโค้ง re-interpolated |
| เหมาะสำหรับ | เส้นโค้งสวยงาม เส้นทางอิสระ | เส้นโค้งที่ต้องแตะพิกัดเฉพาะ |

## DXF — วัตถุ SPLINE (รูปแบบจุดควบคุม)

CV spline จะถูกบันทึกเป็นวัตถุ `SPLINE` ในไฟล์ DXF โดยเก็บดีกรี เวกเตอร์ knot และพิกัดจุดควบคุมทั้งหมด คุณสมบัติทั้งหมด — สี เลเยอร์ linetype สเกล linetype และความหนา — จะถ่ายโอนข้อมูลไปกลับโดยไม่สูญหาย `splineFlag` ถูกตั้งเป็น `9` (CV spline) เพื่อให้รูปแบบคงอยู่เมื่อโหลดซ้ำ แอปพลิเคชัน DXF ที่รองรับวัตถุ `SPLINE` พร้อมข้อมูล CV จะอ่านได้อย่างถูกต้อง
