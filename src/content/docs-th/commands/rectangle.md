# Rectangle

คำสั่ง `rectangle` วาดสี่เหลี่ยมแนวแกนที่กำหนดโดยการคลิกสองมุมตรงข้าม ผลลัพธ์จะถูกเก็บเป็น **`LWPOLYLINE` แบบปิด** ที่มีสี่จุดยอด — หนึ่งจุดที่แต่ละมุม ไม่มีประเภทสี่เหลี่ยมเฉพาะ: หลังสร้างรูปทรงจะทำงานเหมือน [Polyline](../polyline/) ทุกประการ และทุกการแก้ไขโพลีไลน์ใช้ได้กับมัน

## การวาดสี่เหลี่ยม

1. พิมพ์ `rectangle` ในเทอร์มินัลหรือคลิกปุ่ม **Rectangle** บนแถบเครื่องมือ
2. **คลิกมุมแรก** หรือพิมพ์ `X,Y` แล้วกด **Enter** เพื่อป้อนพิกัดที่แน่นอน
3. **คลิกมุมตรงข้าม** — สี่เหลี่ยมจะถูกวางทันทีและคำสั่งออก การป้อนพิกัดก็ใช้ได้ที่นี่เช่นกัน

```
  ● (คลิกแรก)────────────┐
  |                       |
  |   ตัวอย่างสดตามเคอร์เซอร์  |
  |   หลังขั้นตอน 2          |
  └──────────────────────● (คลิกที่สอง)
```

การคลิกสองครั้งสามารถเป็นมุมทแยงคู่ใดก็ได้ — บนซ้าย + ล่างขวา หรือล่างซ้าย + บนขวา เป็นต้น ลำดับไม่สำคัญ

## การป้อนพิกัด

ที่ขั้นตอนมุมใดก็ได้ คุณสามารถพิมพ์ตำแหน่งที่แน่นอน:

1. พิมพ์ค่า X
2. กด `,` — เทอร์มินัลแสดง `[X], [Y{cursor}]`
3. พิมพ์ค่า Y
4. กด **Enter** เพื่อวางมุม

## อ้างอิงแป้นพิมพ์

| ปุ่ม | การกระทำ |
|-----|----------|
| `0`–`9`, `.`, `-` | เริ่มป้อนพิกัด X |
| `,` | ล็อก X และย้ายไปป้อน Y |
| `Enter` | ยืนยันพิกัดที่พิมพ์ |
| `Escape` | ยกเลิก |

ด้านต่างๆ จะแนวนอนและแนวตั้งเสมอ — ไม่มีการล็อกมุมสำหรับคำสั่งสี่เหลี่ยม

## การแก้ไขกริป — การปรับรูปร่างหลังสร้าง

สี่เหลี่ยมที่เลือกจะแสดงกริปที่จุดยอดทุกจุดและที่จุดกึ่งกลางของแต่ละด้าน:

| กริป | ตำแหน่ง | สิ่งที่ทำ |
|------|---------|---------|
| **มุม** | แต่ละจุดยอดทั้ง 4 | ลากเพื่อย้ายจุดยอดนั้น สองด้านที่อยู่ติดกันจะยืดตาม — มุมตรงข้ามอยู่นิ่ง |
| **จุดกึ่งกลางด้าน** | กึ่งกลางของแต่ละด้านทั้ง 4 | ลากเพื่อเลื่อนจุดสิ้นสุดทั้งสองของด้านนั้นพร้อมกัน รักษาความยาวและมุมของด้าน |

การลากกริปมุมจะเปลี่ยนสี่เหลี่ยมให้กลายเป็นรูปสี่เหลี่ยมที่ไม่ใช่สี่เหลี่ยมมุมฉาก หากต้องการสี่เหลี่ยมขนาดต่างกันให้ลากมุมโดยรักษาด้านให้ตั้งฉากโดยประมาณ หรือลบแล้ววาดใหม่

## การเลือกสี่เหลี่ยม

เนื่องจากสี่เหลี่ยมเป็นโพลีไลน์ การเลือกจึงทำงานเหมือนกัน:

| วิธี | พฤติกรรม |
|------|-----------|
| **คลิก** | เลือกถ้าคลิกลงบนด้านใดด้านหนึ่งในสี่ด้าน |
| **ลากไปขวา** (เข้มงวด) | จุดยอดทั้งสี่ต้องอยู่ในกล่องเลือก |
| **ลากไปซ้าย** (ตัดผ่าน) | ด้านใดที่ตัดขอบกล่องจะเลือกสี่เหลี่ยมทั้งหมด |

## คำสั่งแก้ไขที่รองรับ

คำสั่งแก้ไขโพลีไลน์ทั้งหมดใช้ได้ Trim และ Extend ใช้ได้กับ [Line](../line/) เท่านั้น ไม่ทำงานกับสี่เหลี่ยม:

| คำสั่ง | สิ่งที่เกิดขึ้นกับสี่เหลี่ยม |
|--------|--------------------------|
| [Move](../move/) | เลื่อนจุดยอดทั้งสี่ด้วยการกระจัดเดียวกัน |
| [Copy](../copy/) | สร้างสี่เหลี่ยมเหมือนกันในตำแหน่งใหม่ |
| [Rotate](../rotate/) | หมุนจุดยอดทั้งสี่รอบจุดฐานที่เลือก |
| [Mirror](../mirror/) | สะท้อนจุดยอดทั้งสี่ผ่านแกนกระจก |
| [Scale](../scale/) | ปรับขนาดจุดยอดทั้งสี่สม่ำเสมอจากจุดฐาน |
| [Offset](../offset/) | สร้างสี่เหลี่ยมขนานที่ระยะคงที่ (ด้านใน/ด้านนอก) |
| [Delete](../delete/) | ลบสี่เหลี่ยมออกจากภาพวาด |

## คุณสมบัติ

เมื่อเลือกสี่เหลี่ยม แผงคุณสมบัติจะแสดงฟิลด์เดียวกับโพลีไลน์:

**ทั่วไป**

| คุณสมบัติ | ค่าเริ่มต้น | ความหมาย |
|-----------|------------|---------|
| Color | 256 (ByLayer) | ดัชนีสี ACI |
| Layer | `0` | การกำหนดเลเยอร์ |
| Linetype | ByLayer | รูปแบบเส้นประที่มีชื่อ |
| Linetype Scale | 1 | ตัวคูณสเกลบนรูปแบบเส้นประ |
| Thickness | 0 | ความหนาของการยื่น |

**เรขาคณิต**

| คุณสมบัติ | ความหมาย |
|-----------|---------|
| Closed | `true` เสมอสำหรับสี่เหลี่ยม |
| Vertex Count | `4` เสมอสำหรับสี่เหลี่ยมที่ไม่ได้แก้ไข |
| Vertices | พิกัดของมุมทั้งสี่ |

## สี่เหลี่ยม vs โพลีไลน์ vs เส้น

| | Rectangle | Polyline | Line |
|---|-----------|---------|------|
| วิธีวาด | 2 คลิก (มุม) | คลิกแต่ละจุดยอด | คลิกแต่ละจุดสิ้นสุด |
| ประเภทวัตถุ | `LWPOLYLINE` แบบปิด | `LWPOLYLINE` แบบเปิดหรือปิด | `LINE` ต่อส่วน |
| ด้านตั้งฉากเสมอ | ใช่ (ขณะสร้าง) | ไม่ | ไม่ |
| Trim / Extend | ไม่ | ไม่ | ใช่ |
| เหมาะสำหรับ | กล่อง กรอบ พื้นที่สี่เหลี่ยม | เส้นรูปร่างและเส้นทางต่างๆ | ส่วนเดี่ยว เส้นก่อสร้าง |

## DXF — วัตถุ LWPOLYLINE

สี่เหลี่ยมจะถูกบันทึกเป็นวัตถุ `LWPOLYLINE` แบบปิดที่มีสี่จุดยอด คุณสมบัติทั้งหมด — พิกัดจุดยอด สี เลเยอร์ linetype สเกล linetype และความหนา — จะถ่ายโอนข้อมูลไปกลับโดยไม่สูญหาย

ไม่มีประเภท `RECTANGLE` ใน DXF เมื่อเปิดไฟล์อีกครั้ง รูปทรงจะปรากฏเป็นโพลีไลน์สี่จุดยอดแบบปิดแทนสี่เหลี่ยม โปรแกรมดู DXF หรือโปรแกรมแก้ไขใดๆ ที่รองรับ `LWPOLYLINE` (LibreCAD, FreeCAD เป็นต้น) จะแสดงได้อย่างถูกต้อง
