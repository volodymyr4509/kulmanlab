# ViewportRectangle

คำสั่ง `ViewportRectangle` สร้าง viewport ใหม่ในเลย์เอาต์กระดาษที่ใช้งานอยู่โดยเลือกสองมุมตรงข้าม พร้อมใช้เฉพาะในพื้นที่เลย์เอาต์เท่านั้น

## การสร้าง Viewport

1. สลับไปยังเลย์เอาต์กระดาษโดยใช้แท็บที่ด้านล่างของหน้าจอ
2. พิมพ์ `ViewportRectangle` ในเทอร์มินัลหรือคลิกปุ่ม **Viewport Rectangle** บนแถบเครื่องมือ
3. **คลิกมุมแรก** หรือพิมพ์ `X,Y` แล้วกด **Enter** เพื่อป้อนพิกัดที่แน่นอน
4. **คลิกมุมตรงข้าม** — viewport จะถูกวางทันที การป้อนพิกัดก็ใช้ได้ที่นี่เช่นกัน

viewport ใหม่จะแสดงโมเดลทั้งหมดที่สเกลเริ่มต้นของเลย์เอาต์ ใช้ scroll wheel ภายใน viewport เพื่อซูม หรือลากด้วยปุ่มกลางเพื่อเลื่อนวิวโมเดล

## การป้อนพิกัด

ที่ขั้นตอนมุมใดก็ได้ คุณสามารถพิมพ์พิกัดที่แน่นอน:

1. พิมพ์ค่า X
2. กด `,` — เทอร์มินัลแสดง `[X], [Y{cursor}]`
3. พิมพ์ค่า Y
4. กด **Enter** เพื่อวางจุด

## อ้างอิงแป้นพิมพ์

| ปุ่ม | การกระทำ |
|-----|----------|
| `0`–`9`, `.`, `-` | เริ่มป้อนพิกัด X |
| `,` | ล็อก X และย้ายไปป้อน Y |
| `Enter` | ยืนยันพิกัดที่พิมพ์ |
| `Escape` | ยกเลิก |

## การแก้ไข Viewport

หลังวาง viewport คลิกเพื่อเลือก:

- **ลากขอบหรือมุม** เพื่อปรับขนาด
- **ลากกริปตรงกลาง** เพื่อย้าย
- ใช้ **ตัวเลือกสเกล** ในแถบควบคุมเพื่อตั้งสเกลที่แน่นอน (เช่น 1:50) หากต้องการป้อนสเกลที่ไม่อยู่ในรายการ ให้พิมพ์โดยตรงในช่องป้อนที่ด้านล่างของดรอปดาวน์ — รับรูปแบบอัตราส่วน (`1:200`, `5:1`) หรือทศนิยมธรรมดา (`0.005`) แล้วกด **Enter**
- คลิกขวา viewport แล้วใช้ **Lock** เพื่อป้องกันการเปลี่ยนแปลงโดยไม่ตั้งใจ
