# Distance

คำสั่ง `distance` วัดระยะเส้นตรง (Euclidean) ระหว่างสองจุดที่คลิกและพิมพ์ผลลัพธ์ในเทอร์มินัลถึง 4 ตำแหน่งทศนิยม เป็นหนึ่งในสองคำสั่งวัด — [Angle](../angle/) วัดมุมเปิดที่จุดยอดแทน

## กายวิภาคของการวัดระยะ

```
  ● จุดแรก
   \
    \  เส้นตัวอย่าง (สด)
     \
      ● จุดที่สอง    →  เทอร์มินัล: "Distance: 12.3456"
```

- **จุดแรก** — จุดกำเนิดของการวัด
- **จุดที่สอง** — จุดสิ้นสุด การวางจะพิมพ์ผลลัพธ์ทันที
- **ผลลัพธ์** — แสดงในเทอร์มินัล ไม่ถูกวางบนแคนวาส

## การวัดระยะ

1. พิมพ์ `distance` ในเทอร์มินัลหรือคลิกปุ่ม **Distance** บนแถบเครื่องมือ
2. **คลิกจุดแรก** หรือพิมพ์ `X,Y` แล้วกด **Enter** เพื่อป้อนพิกัดที่แน่นอน
3. **คลิกจุดที่สอง** — ระยะที่วัดปรากฏในเทอร์มินัล การป้อนพิกัดก็ใช้ได้ที่นี่เช่นกัน
4. **คลิกอีกครั้ง** (ไม่บังคับ) เพื่อเริ่มการวัดใหม่ คำสั่งยังคงทำงานอยู่

กด `Escape` ได้ทุกเวลาเพื่อรีเซ็ตไปยังขั้นตอนที่ 2

## การวัดแบบต่อเนื่อง

หลังจากผลลัพธ์แสดง การคลิกจะเริ่มการวัดถัดไปทันที — จุดที่คลิกกลายเป็นจุดแรกใหม่ ซึ่งช่วยให้คุณวัดลำดับระยะโดยไม่ต้องเปิดใช้งานคำสั่งใหม่

## Distance vs Angle

| | Distance | Angle |
|---|---------|-------|
| สิ่งที่วัด | ความยาวเส้นตรง | มุมภายในที่จุดยอด |
| จำนวนคลิก | 2 | 3 |
| รูปแบบผลลัพธ์ | `12.3456` (หน่วย) | `45.0000°` |
| ตัวอย่างบนแคนวาส | เส้นจากจุดแรกไปยังเคอร์เซอร์ | สองเส้นจากจุดยอดไปยังเคอร์เซอร์ |
| เหมาะสำหรับ | ความยาวของช่องว่างหรือส่วน | มุมเปิดระหว่างสองลักษณะ |

## การป้อนพิกัด

แทนที่จะคลิก พิมพ์ตำแหน่งที่แน่นอนสำหรับจุดใดก็ได้:

1. พิมพ์ค่า X
2. กด `,` — เทอร์มินัลแสดง `[X], [Y{cursor}]`
3. พิมพ์ค่า Y
4. กด **Enter** เพื่อยืนยัน

## อ้างอิงแป้นพิมพ์

| ปุ่ม | การกระทำ |
|-----|----------|
| `0`–`9`, `.`, `-` | เริ่มป้อนพิกัด X |
| `,` | ล็อก X และย้ายไปป้อน Y |
| `Backspace` | ลบอักขระที่พิมพ์ล่าสุด |
| `Enter` | ยืนยันพิกัดที่พิมพ์ |
| `Escape` | ยกเลิกและรีเซ็ตไปยังขั้นตอนที่ 2 |

## หมายเหตุ

- ผลลัพธ์แสดงใน **เทอร์มินัลเท่านั้น** — ไม่มีอะไรถูกเพิ่มลงในภาพวาด
- ผลลัพธ์แสดงในหน่วยเดียวกับพิกัดภาพวาด (ไม่มีการแปลงหน่วย)
- ความแม่นยำเสมอ 4 ตำแหน่งทศนิยม
