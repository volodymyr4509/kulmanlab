# Ellipse

คำสั่ง `ellipse` วาดวงรีโดยใช้สามคลิก: จุดศูนย์กลาง จุดสิ้นสุดของกึ่งแกนแรก (หลัก) ในมุมใดก็ได้ และความยาวของกึ่งแกนที่สอง (รอง) สองแกนตั้งฉากกันเสมอ — ทิศทางแกนที่สองถูกกำหนดโดยอัตโนมัติจากแกนแรก

## การวาดวงรี

1. พิมพ์ `ellipse` ในเทอร์มินัลหรือคลิกปุ่ม **Ellipse** บนแถบเครื่องมือ
2. **คลิกจุดศูนย์กลาง** หรือพิมพ์ `X,Y` แล้วกด **Enter** เพื่อป้อนพิกัดที่แน่นอน
3. **คลิกจุดสิ้นสุดแกนแรก** — กำหนดทั้งทิศทางและความยาวของกึ่งแกนแรก การป้อนพิกัดก็ใช้ได้ที่นี่เช่นกัน
4. **ตั้งค่าความยาวแกนที่สอง** — ย้ายเคอร์เซอร์ตั้งฉากกับแกนแรก แล้วคลิกหรือพิมพ์ความยาว

```
               ● ← จุดสิ้นสุดแกนแรก (ขั้นตอนที่ 3)
              /
  center ●  /  ← แกนแรก (มุมใดก็ได้)
            |
            ● ← เคอร์เซอร์ที่นี่ตั้งค่าความยาวแกนที่สอง (ขั้นตอนที่ 4)
```

วงรีถูกวางหลังจากขั้นตอนที่ 4 และคำสั่งออก

## การป้อนแกน — คลิก พิกัด หรือความยาวแบบพิมพ์

**ศูนย์กลาง (ขั้นตอนที่ 2):** คลิกหรือพิมพ์ `X,Y` เพื่อป้อนตำแหน่งที่แน่นอน

**จุดสิ้นสุดแกนแรก (ขั้นตอนที่ 3):** คลิก หรือพิมพ์ `X,Y` เพื่อป้อนพิกัดที่แน่นอน การล็อกมุมก็สแนปกับ 45° — ขณะล็อก พิมพ์ความยาวแล้วกด **Enter** เพื่อวางจุดสิ้นสุดที่ระยะที่แน่นอน

**แกนที่สอง (ขั้นตอนที่ 4):** การพิมพ์ความยาวใช้ได้เสมอ — ไม่ต้องล็อกมุม ทิศทางตรึงตั้งฉากกับแกนแรกแล้ว การพิมพ์กำหนดเฉพาะความยาว

| ปุ่ม | การกระทำ |
|-----|----------|
| `0`–`9`, `.` | เพิ่มตัวเลขเข้าไปในความยาวแกน (ระยะแกนที่สอง) |
| `Backspace` | ลบอักขระที่พิมพ์ล่าสุด |
| `Enter` | วางจุดสิ้นสุดแกนที่ความยาวที่พิมพ์ |

## การป้อนพิกัด (ศูนย์กลางและจุดสิ้นสุดแกนแรก)

1. พิมพ์ค่า X
2. กด `,` — เทอร์มินัลแสดง `[X], [Y{cursor}]`
3. พิมพ์ค่า Y
4. กด **Enter** เพื่อยืนยัน

## อ้างอิงแป้นพิมพ์

| ปุ่ม | การกระทำ |
|-----|----------|
| `0`–`9`, `.`, `-` | เริ่มป้อนพิกัด X (ระยะศูนย์กลาง/แกนแรก) หรือความยาวแกนเมื่อล็อกมุม |
| `,` | ล็อก X และย้ายไปป้อน Y |
| `Backspace` | ลบอักขระที่พิมพ์ล่าสุด |
| `Enter` | ยืนยันพิกัดหรือความยาวที่พิมพ์ |
| `Escape` | ยกเลิกและรีเซ็ต |

## การแก้ไขกริป — การปรับขนาดแกนอิสระ

วงรีที่เลือกแสดงกริปห้าตัว:

| กริป | จำนวน | สิ่งที่ทำ |
|------|-------|-----------|
| **Center** | 1 | ย้ายวงรีทั้งหมด ทั้งสองแกนไม่เปลี่ยน |
| **Major axis endpoints** | 2 (ปลายตรงข้ามของแกนยาว) | ลากเพื่อปรับขนาดความยาวกึ่งแกนหลัก ขนาดสมบูรณ์ของแกนรองคงที่ |
| **Minor axis endpoints** | 2 (ปลายตรงข้ามของแกนสั้น) | ลากเพื่อปรับขนาดความยาวกึ่งแกนรอง แกนหลักไม่เปลี่ยน |

กริปแกนหลักและรองเป็นอิสระ — คุณสามารถปรับรูปร่างวงรีโดยไม่ต้องรันคำสั่งซ้ำ

## การเลือกวงรี

| วิธี | พฤติกรรม |
|------|-----------|
| **คลิก** | เลือกถ้าคลิกอยู่ใกล้เส้นขอบวงรี |
| **ลากไปขวา** (เข้มงวด) | กล่องล้อมรอบที่ตรงแกนของวงรีต้องพอดีภายในกล่องเลือก |
| **ลากไปซ้าย** (ตัดผ่าน) | ส่วนใดของเส้นขอบวงรีที่ตัดขอบกล่องเลือกจะเลือกมัน |

## คุณสมบัติ

**ทั่วไป**

| คุณสมบัติ | ค่าเริ่มต้น | ความหมาย |
|-----------|-----------|----------|
| Color | 256 (ByLayer) | ดัชนีสี ACI |
| Layer | `0` | การกำหนดเลเยอร์ |
| Linetype | ByLayer | รูปแบบเส้นที่มีชื่อ |
| Linetype Scale | 1 | ตัวคูณมาตราส่วนสำหรับรูปแบบเส้น |
| Thickness | 0 | ความหนาของการรีดออก |

**เรขาคณิต**

| คุณสมบัติ | ความหมาย |
|-----------|----------|
| Center X / Center Y | จุดศูนย์กลางของวงรี |
| Major Axis X / Major Axis Y | เวกเตอร์จากศูนย์กลางไปยังจุดสิ้นสุดแกนหลัก (เข้ารหัสทั้งทิศทางและความยาว) |
| Axis Ratio | อัตราส่วนกึ่งแกนรองต่อกึ่งแกนหลัก (0 < อัตราส่วน ≤ 1) |
| Start Angle / End Angle | มุมพารามิเตอร์เป็นองศา ทั้งสองคือ 0°/360° สำหรับวงรีเต็ม |

## Ellipse vs Circle — เมื่อไหรควรใช้อะไร

| | Ellipse | Circle |
|---|---------|--------|
| แกน | สองกึ่งแกนอิสระในมุมใดก็ได้ | รัศมีเดียว สมมาตร |
| การหมุน | วางได้ในมุมใดก็ได้ | ไม่มีการหมุน |
| ป้อนแบบพิมพ์ | ความยาวต่อแกน | รัศมีเท่านั้น |
| กริปปรับขนาด | แกนหลักและรองอิสระ | จุดหลักทั้งสี่เท่ากัน |
| เหมาะสำหรับ | มุมมองเฉียง ลักษณะรูปรี รูเปอร์สเปกทีฟ | ลักษณะกลมสมมาตร |

## DXF — วัตถุ ELLIPSE

วงรีถูกบันทึกเป็นวัตถุ `ELLIPSE` ในไฟล์ DXF รูปแบบจัดเก็บจุดศูนย์กลาง เวกเตอร์แกนหลักเต็ม (ทิศทาง + ความยาว) และอัตราส่วนแกน การหมุน รูปร่าง และคุณสมบัติสไตล์ทั้งหมดรอบเที่ยวโดยไม่สูญเสียข้อมูล วงกลม **ไม่** ถูกจัดเก็บเป็นวงรีเสื่อม — ทั้งสองประเภทวัตถุยังคงแตกต่างกันในโมเดล DXF
