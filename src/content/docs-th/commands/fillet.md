# Fillet

คำสั่ง `fillet` ทำมุมโค้งระหว่างวัตถุ [Line](../line/) สองชิ้นโดยแทรกส่วนโค้งสัมผัสที่มีรัศมีที่กำหนดและตัดแต่ละเส้นกลับไปยังจุดที่ส่วนโค้งเริ่ม ผลลัพธ์คือมุมที่เรียบและมีรัศมีเชื่อมต่อทั้งสองเส้น

Fillet ทำงานกับ **วัตถุ Line เท่านั้น**

## การใช้ fillet

1. พิมพ์ `fillet` ในเทอร์มินัลหรือคลิกปุ่ม **Fillet** บนแถบเครื่องมือ
2. **พิมพ์รัศมี fillet** แล้วกด **Enter**
3. **คลิกเส้นแรก** — ส่วนที่คุณคลิกกำหนดว่าด้านใดของจุดตัดจะถูกเก็บไว้
4. **วางเมาส์เหนือเส้นที่สอง** — ตัวอย่างส่วนโค้งประแสดง fillet ที่ได้ ย้ายเคอร์เซอร์ไปด้านที่ต้องการเก็บไว้
5. **คลิก** เพื่อใช้งาน ทั้งสองเส้นถูกตัดและส่วนโค้งถูกแทรก

```
  ก่อน:                     หลัง fillet (รัศมี r):

  ──────────────              ──────────╮
                │                        ╰────
                │
```

## การเลือกด้านสำหรับเส้นที่ตัดกัน

เมื่อสองเส้นตัดกัน fillet ถูกใช้กับมุมที่กำหนดโดยตำแหน่งคลิก — ส่วนของแต่ละเส้น **ด้านเดียวกับเคอร์เซอร์** จะถูกเก็บไว้

- คลิกใกล้ปลายด้านหนึ่งของเส้นแรกเพื่อเลือกครึ่งนั้น
- ย้ายเคอร์เซอร์ไปยังครึ่งที่ต้องการของเส้นที่สอง — ตัวอย่างประอัปเดตสด

## สิ่งที่คำสั่งสร้าง

- จุดสิ้นสุดของเส้นแรกที่ใกล้กับจุดตัดถูกย้ายไปยังจุดสัมผัส **T1**
- จุดสิ้นสุดของเส้นที่สองที่ใกล้กับจุดตัดถูกย้ายไปยังจุดสัมผัส **T2**
- วัตถุ Arc ใหม่ถูกแทรกจาก **T1** ถึง **T2** สัมผัสกับทั้งสองเส้น

ส่วนโค้งที่แทรกรับค่าความหนาเส้น สี เลเยอร์ และประเภทเส้นปัจจุบัน

## อ้างอิงแป้นพิมพ์

| ปุ่ม | การกระทำ |
|-----|----------|
| `0`–`9`, `.` | เพิ่มตัวเลขเข้าไปในค่ารัศมี |
| `Backspace` | ลบอักขระที่พิมพ์ล่าสุด |
| `Enter` | ยืนยันรัศมีที่พิมพ์และย้ายไปเลือกเส้น |
| `Escape` | ยกเลิกและรีเซ็ต |

## วัตถุที่รองรับ

| วัตถุ | รองรับ |
|-------|--------|
| Line | ใช่ — ทั้งวัตถุแรกและที่สอง |
| Arc, Circle, Ellipse, Polyline | ไม่ |
| Text, Spline, Dimension, Leader | ไม่ |

## Fillet vs Chamfer

| | Fillet | Chamfer |
|---|--------|---------|
| ประเภทมุม | ส่วนโค้งมน | การตัดตรง |
| ป้อน | รัศมีหนึ่งค่า | สองระยะ (d1, d2) |
| วัตถุที่แทรก | Arc | Line |
| วัตถุที่รองรับ | Lines เท่านั้น | Lines และ Polylines |
