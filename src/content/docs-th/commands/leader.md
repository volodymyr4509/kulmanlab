# Leader

คำสั่ง `leader` วาดคำอธิบาย multileader สี่ขั้นตอน: หัวลูกศรที่สัมผัสลักษณะ เส้น leader ที่งอที่ dogleg ตัวยึดข้อความ และป้ายที่พิมพ์ จากคำสั่งคำอธิบายทั้งหมด Leader เป็นคำสั่งเดียวที่รวมระยะป้อนข้อความแบบโต้ตอบพร้อมตัวอย่างเคอร์เซอร์กะพริบ

## กายวิภาคของ multileader

```
  ◄── tip หัวลูกศร  (ขั้นตอนที่ 2 — สัมผัสลักษณะ)
      \
       \  เส้น leader
        \
         ●──── dogleg (ขั้นตอนที่ 3) ──── ตัวยึดข้อความ (ขั้นตอนที่ 4)
                                    ข้อความป้าย  (ขั้นตอนที่ 5)
```

- **Tip หัวลูกศร** — ปลายแหลมที่วางที่ลักษณะที่ระบุ
- **Dogleg** — ข้อศอกที่เส้น leader งอไปยังข้อความ
- **ตัวยึดข้อความ** — ตำแหน่งที่ป้ายอยู่ ข้อความจัดซ้ายหรือขวาโดยอัตโนมัติ

## การวาด leader

1. พิมพ์ `leader` ในเทอร์มินัลหรือคลิกปุ่ม **Leader** บนแถบเครื่องมือ
2. **คลิก tip หัวลูกศร** หรือพิมพ์ `X,Y` แล้วกด **Enter** เพื่อป้อนพิกัดที่แน่นอน
3. **คลิก dogleg** — การงอใน leader มุมล็อกกับ 45° พิมพ์ความยาวแล้วกด **Enter** เพื่อวางอย่างแม่นยำ หรือพิมพ์ `X,Y` เพื่อป้อนพิกัดสัมบูรณ์
4. **คลิกตำแหน่งข้อความ** — ตำแหน่งที่ป้ายยึด ตัวเลือกเดียวกันใช้ได้: คลิก ล็อกมุม + ความยาว หรือ `X,Y`
5. **พิมพ์ข้อความป้าย** — ตัวอย่างบนแคนวาสอัปเดตสดพร้อมเคอร์เซอร์กะพริบ กด **Enter** เพื่อวาง

## การป้อนพิกัด (ทุกระยะจุด)

ที่ขั้นตอนเลือกจุดใด (tip, dogleg, ตำแหน่งข้อความ) คุณสามารถพิมพ์พิกัดที่แน่นอนแทนการคลิก:

1. พิมพ์ค่า X (ตัวเลข `.` หรือ `-`)
2. กด `,` — เทอร์มินัลแสดง `[X], [Y{cursor}]` ยืนยันว่า X ถูกล็อก
3. พิมพ์ค่า Y
4. กด **Enter** เพื่อวางจุด

## การล็อกมุม (ขั้นตอนที่ 3 และ 4)

หลังจากแต่ละจุดที่วาง คำสั่งสแนปกับแกน 45° เมื่อเคอร์เซอร์อยู่ไกลพอ ขณะล็อก:
- ตัวอย่างสแนปกับแกน
- พิมพ์ความยาวแล้วกด **Enter** เพื่อวางจุดถัดไปที่ระยะที่แน่นอน

การล็อกมุมและการป้อนพิกัดไม่สามารถใช้พร้อมกัน — เมื่อพิมพ์ตัวเลขโดยไม่มี `,` ก่อนหน้า คำสั่งตีความว่าเป็นระยะ (ต้องล็อกมุมอยู่) หากต้องการป้อนพิกัดสัมบูรณ์แทน ให้เริ่มด้วยตัวเลข X ตามด้วยเครื่องหมายจุลภาค

## การแก้ไขข้อความป้าย

ขณะพิมพ์ป้ายในขั้นตอนที่ 5 คุณสามารถนำทางและแก้ไขข้อความก่อนวาง:

| ปุ่ม | การกระทำ |
|-----|----------|
| อักขระที่พิมพ์ได้ | แทรกที่ตำแหน่งเคอร์เซอร์ |
| `←` / `→` | ย้ายเคอร์เซอร์ซ้ายหรือขวา |
| `Backspace` | ลบอักขระทางซ้ายของเคอร์เซอร์ |
| `Delete` | ลบอักขระทางขวาของเคอร์เซอร์ |
| `Enter` | วาง leader |

## ทิศทางข้อความอัตโนมัติ

การจัดข้อความปรับตามตำแหน่งเคอร์เซอร์สัมพันธ์กับ dogleg:

| ตำแหน่งเคอร์เซอร์ | ทิศทางข้อความ |
|-----------------|--------------|
| **ขวา** ของ dogleg | ซ้ายไปขวาจากตัวยึดข้อความ |
| **ซ้าย** ของ dogleg | ขวาไปซ้าย (ยึดด้านขวา) |

ไม่ต้องปรับด้วยตนเอง — ย้ายเคอร์เซอร์ไปด้านที่ต้องการป้ายและจัดตรงถูกต้อง

## อ้างอิงแป้นพิมพ์

**ระยะจุด (tip, dogleg, ตำแหน่งข้อความ)**

| ปุ่ม | การกระทำ |
|-----|----------|
| `0`–`9`, `.`, `-` | เริ่มพิมพ์พิกัด X (แล้ว `,` เพื่อล็อก X และป้อน Y) |
| `,` | ยืนยัน X และย้ายไปป้อน Y |
| `0`–`9`, `.`, `-` | สร้างระยะเมื่อล็อกมุม |
| `Backspace` | ลบอักขระที่พิมพ์ล่าสุด |
| `Enter` | ยืนยันพิกัดหรือระยะที่พิมพ์ |

**ระยะป้อนข้อความ**

| ปุ่ม | การกระทำ |
|-----|----------|
| อักขระที่พิมพ์ได้ | แทรกที่เคอร์เซอร์ |
| `←` / `→` | ย้ายเคอร์เซอร์ |
| `Backspace` | ลบซ้าย |
| `Delete` | ลบขวา |
| `Enter` | วาง leader |

| ปุ่ม | การกระทำ |
|-----|----------|
| `Escape` | ยกเลิกและรีเซ็ตไปยังขั้นตอนที่ 2 |

## การแก้ไข leader ที่มีอยู่

**ดับเบิลคลิก** multileader ที่วางแล้วเพื่อเปิดโปรแกรมแก้ไขข้อความอีกครั้งในโหมด **rich** ในโหมด rich คุณสามารถใช้ตัวหนา ตัวเอียง และการแทนที่ฟอนต์หรือขนาดต่อตัวอักษร และแทรกการขึ้นบรรทัดใหม่ด้วย `Enter` กด **Escape** เพื่อยืนยันและปิด

ดู [Text Editor — rich mode](../../interface/text-editor/#rich-mode) สำหรับข้อมูลอ้างอิงทั้งหมด

## การเพิ่มและลบแขน

- หากต้องการเพิ่มแขนหัวลูกศรพิเศษให้กับ leader ที่มีอยู่: [Leader+](../leader-add/)
- หากต้องการลบแขนจาก leader ที่มีสองแขนหรือมากกว่า: [Leader−](../leader-remove/)

## DXF — นำเข้าเท่านั้น

**Leaders นำเข้าเท่านั้น** วัตถุ `MLEADER` จากไฟล์ DXF ถูกอ่านและแสดงอย่างถูกต้อง แต่ leaders ที่วาดในตัวแก้ไข **ไม่ถูกเขียน** เมื่อบันทึกไฟล์ DXF ใช้ leaders สำหรับคำอธิบายภาพ หลีกเลี่ยงการพึ่งพาพวกมันสำหรับเวิร์กโฟลว์รอบเที่ยว
