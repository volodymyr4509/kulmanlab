# Delete

คำสั่ง `delete` ลบวัตถุที่เลือกออกจากภาพวาด การลบถูกบันทึกในประวัติ [Undo](../undo/) และสามารถย้อนกลับได้สูงสุด 20 ขั้นตอน ไม่มีกล่องยืนยันการลบแยกต่างหาก — การยืนยันเป็นการกดปุ่มครั้งเดียว

## สองวิธีลบ

**เลือกก่อน แล้วลบ** — เส้นทางที่เร็วที่สุด:

1. เลือกวัตถุหนึ่งหรือมากกว่าบนแคนวาส
2. พิมพ์ `delete` ในเทอร์มินัล คลิกปุ่ม **Delete** บนแถบเครื่องมือ **หรือกดปุ่ม `Delete`** โดยตรง

วัตถุถูกลบทันที — ไม่มีขั้นตอนยืนยันเพิ่มเติม

**เปิดใช้งานก่อน แล้วเลือก**:

1. พิมพ์ `delete` หรือคลิกปุ่มแถบเครื่องมือ (โดยไม่มีการเลือก)
2. **เลือกวัตถุ** — คลิกเพื่อสลับ หรือลากเพื่อเลือกตามพื้นที่
3. กด **Enter**, **Space** หรือ **Delete** เพื่อยืนยันและลบวัตถุที่เลือก

## ทางลัดปุ่ม Delete

ปุ่ม `Delete` บนแป้นพิมพ์ทำหน้าที่เป็น **ทางลัดทั่วไป** — ถ้ามีวัตถุที่เลือกอยู่ การกดมันจะลบทันที แม้ไม่ได้เปิดคำสั่ง Delete ในเทอร์มินัล นี่คือเวิร์กโฟลว์การลบขั้นตอนเดียวที่เร็วที่สุด:

```
คลิกวัตถุ → กดปุ่ม Delete → เสร็จ
```

## การเลือกระหว่างคำสั่ง

| วิธี | พฤติกรรม |
|------|-----------|
| **คลิก** | สลับวัตถุใต้เคอร์เซอร์เข้า/ออกจากการเลือก |
| **ลากไปขวา** (เข้มงวด) | เลือกเฉพาะวัตถุที่อยู่ในกล่องทั้งหมด |
| **ลากไปซ้าย** (ตัดผ่าน) | เลือกวัตถุที่ตัดขอบกล่อง |
| **Enter** / **Space** / **Delete** | ยืนยันและลบวัตถุที่เลือก |

## การกู้คืนวัตถุที่ลบ

การลบสามารถเลิกทำได้ด้วยคำสั่ง [Undo](../undo/) (พิมพ์ `undo` หรือใช้ปุ่มแถบเครื่องมือ) สามารถย้อนกลับได้สูงสุด **20 ขั้นตอน** ต่อไฟล์ และประวัติคงอยู่ข้ามการโหลดหน้า หากคุณเกิน 20 การลบโดยไม่บันทึก การลบเก่าๆ ไม่สามารถกู้คืนได้

## วัตถุที่รองรับ

Delete ทำงานกับวัตถุทุกประเภท — Line, Polyline, Rectangle, Circle, Arc, Ellipse, Text, Spline, Dimension, Leader และอื่นๆ ทั้งหมด
