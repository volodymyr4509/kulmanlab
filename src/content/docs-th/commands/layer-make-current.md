# LayerMakeCurrent

คำสั่ง `LayerMakeCurrent` ตั้งค่า **เลเยอร์การวาดปัจจุบัน** ให้เป็นเลเยอร์ที่วัตถุที่คลิกสังกัด วัตถุใหม่จะถูกวาดบนเลเยอร์นั้นโดยอัตโนมัติ

## การใช้งาน

1. พิมพ์ `LayerMakeCurrent` ในเทอร์มินัลหรือคลิกปุ่ม **Make Current** บนแถบเครื่องมือ (ไอคอนหลอดดูด)
2. **คลิกวัตถุใดก็ได้** บนแคนวาส
3. เลเยอร์ปัจจุบันถูกอัปเดตให้ตรงกับเลเยอร์ของวัตถุนั้น คำสั่งเสร็จสิ้นทันที

## รายละเอียดพฤติกรรม

- หากคลิกบนแคนวาสว่าง (ไม่มีวัตถุ) เทอร์มินัลแสดง `no object found` และคำสั่งยังคงทำงานเพื่อให้คุณลองใหม่
- เฉพาะการตั้งค่าเลเยอร์ปัจจุบันเท่านั้นที่เปลี่ยน — ไม่มีวัตถุถูกแก้ไข
- เลเยอร์ที่อัปเดตสะท้อนในตัวเลือกเลเยอร์บนแถบเครื่องมือ
