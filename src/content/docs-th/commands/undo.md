# Undo

คำสั่ง `undo` ย้อนกลับการเปลี่ยนแปลงล่าสุดในภาพวาด — หนึ่งขั้นตอนต่อการเรียกใช้ การเพิ่ม การลบ หรือการแก้ไขวัตถุแต่ละรายการจะถูกบันทึกเป็นรายการประวัติแยกกัน Undo จะย้อนกลับผ่านรายการเหล่านี้ตามลำดับย้อนกลับ

## วิธี Undo

- พิมพ์ `undo` ในเทอร์มินัล หรือ
- คลิกปุ่ม **Undo** บนแถบเครื่องมือ

แต่ละครั้งที่เรียกใช้จะย้อนกลับการกระทำที่บันทึกหนึ่งรายการ เรียกซ้ำเพื่อย้อนกลับไปอีก

## พฤติกรรมของประวัติ

| รายละเอียด | ค่า |
|-----------|-----|
| ขั้นตอนต่อไฟล์ | สูงสุด **20** |
| การจัดเก็บ | เบราว์เซอร์ (IndexedDB / localStorage) ต่อชื่อไฟล์ |
| คงอยู่หลังโหลดหน้าใหม่ | ใช่ — ประวัติจะถูกกู้คืนเมื่อเปิดไฟล์อีกครั้ง |
| การกระทำใหม่หลัง Undo | ล้างรายการ Redo ทั้งหมดที่อยู่ข้างหน้าตำแหน่งปัจจุบัน |
| รายการเก่าสุดเมื่อเต็ม | ถูกลบเพื่อสร้างพื้นที่สำหรับการเปลี่ยนแปลงใหม่สุด |

การกลายพันธุ์วัตถุทุกอย่างจะถูกบันทึก: การวาดวัตถุใหม่ การลบวัตถุ การแก้ไขกริปจุดสิ้นสุด การใช้ Move, Rotate, Scale, Mirror, Trim, Extend และ Offset ทั้งหมดสร้างรายการประวัติ

## Undo vs Redo

| | Undo | Redo |
|---|------|------|
| ทิศทาง | ย้อนกลับในประวัติ | เดินหน้าในรายการที่ถูก Undo |
| พร้อมใช้เมื่อ | มีการกระทำที่บันทึกไว้อย่างน้อยหนึ่งรายการ | มี Undo อย่างน้อยหนึ่งครั้งและยังไม่มีการกระทำใหม่ |
| ถูกล้างโดย | ไม่มี — ประวัติสะสมจนถึงขีดจำกัด 20 ขั้นตอน | การกระทำวาดใหม่ใดๆ |

ใช้ [Redo](../redo/) เพื่อนำการกระทำที่ถูก Undo กลับมา ปุ่มบนแถบเครื่องมือจะเป็นสีเทาเมื่อทิศทางนั้นไม่พร้อมใช้
