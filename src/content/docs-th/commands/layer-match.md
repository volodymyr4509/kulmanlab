# LayerMatch

คำสั่ง `LayerMatch` กำหนดเลเยอร์ของวัตถุที่เลือกใหม่ให้ตรงกับเลเยอร์ของวัตถุต้นทางที่คุณคลิก เป็นวิธีที่เร็วที่สุดในการย้ายกลุ่มวัตถุไปยังเลเยอร์ที่ถูกต้องโดยไม่ต้องเปิด Layer Manager

## เวิร์กโฟลว์

**เลือกก่อน แล้วจับคู่**:

1. เลือกวัตถุที่ต้องการเปลี่ยนเลเยอร์
2. พิมพ์ `LayerMatch` หรือคลิกปุ่ม **Layer Match** บนแถบเครื่องมือ (ไอคอนแปรง)
3. **คลิกวัตถุต้นทาง** — วัตถุที่ต้องการคัดลอกเลเยอร์
4. วัตถุที่เลือกทั้งหมดย้ายไปยังเลเยอร์ของวัตถุต้นทางทันที

**เปิดใช้งานก่อน แล้วเลือก**:

1. พิมพ์ `LayerMatch` หรือคลิกปุ่มแถบเครื่องมือโดยไม่มีการเลือก
2. **เลือกวัตถุเป้าหมาย** — คลิกเพื่อสลับวัตถุแต่ละชิ้นหรือลากเพื่อเลือกตามพื้นที่
3. กด **Enter** หรือ **Space** เพื่อยืนยันการเลือก
4. **คลิกวัตถุต้นทาง** — เลเยอร์ของมันถูกใช้กับวัตถุเป้าหมายทั้งหมด

## อ้างอิงแป้นพิมพ์

| ปุ่ม | การกระทำ |
|-----|----------|
| `Enter` / `Space` | ยืนยันการเลือกเป้าหมายและดำเนินการไปยังระยะเลือกต้นทาง |
| `Escape` | รีเซ็ต — กลับไปยังการเลือกเป้าหมายหรือยกเลิกทั้งหมด |

## รายละเอียดพฤติกรรม

- เฉพาะคุณสมบัติ `layer` เท่านั้นที่เปลี่ยน — สี linetype lineweight และเรขาคณิตไม่ถูกแตะต้อง
- วัตถุต้นทางไม่ถูกแก้ไข
- คำสั่งเสร็จสิ้นหลังจากคลิกต้นทาง
- การคลิกบนแคนวาสว่างระหว่างระยะเลือกต้นทางไม่มีผล
