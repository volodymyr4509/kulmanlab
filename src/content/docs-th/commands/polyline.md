# Polyline

คำสั่ง `polyline` วาดเส้นทางที่เชื่อมต่อของส่วนตรงจำนวนเท่าใดก็ได้ ทั้งหมดจัดเก็บเป็นวัตถุ `LWPOLYLINE` เดียว เนื่องจากเส้นทางทั้งหมดเป็นวัตถุเดียว การเลือกจะเลือกทุกส่วนพร้อมกัน — ย้าย หมุน หรือปรับมาตราส่วนรูปทรงทั้งหมดในการดำเนินการเดียว นี่คือความแตกต่างสำคัญจาก [Lines](../line/) ที่ต่อเชื่อมกัน ที่แต่ละส่วนเป็นวัตถุอิสระ

Polylines สามารถ **ปิด** ได้: คำสั่ง [Rectangle](../rectangle/) ใช้วัตถุ `LWPOLYLINE` เดียวกันที่มีแฟล็กปิด

## การวาด polyline

1. พิมพ์ `polyline` ในเทอร์มินัลหรือคลิกปุ่ม **Polyline** บนแถบเครื่องมือ
2. **คลิกจุดแรก** หรือพิมพ์ `X,Y` แล้วกด **Enter** เพื่อป้อนพิกัดที่แน่นอน
3. **คลิกจุดถัดไปแต่ละจุด** — แต่ละคลิกเพิ่มส่วน การป้อนพิกัดใช้ได้ทุกขั้นตอน
4. กด **Enter** หรือ **Space** เพื่อเสร็จสิ้น (ต้องวางอย่างน้อย 2 จุด)

```
  ●──────●
  ที่ 1    ที่ 2
          \
           \  ส่วนที่ 3 (กำลังดำเนินการ — เคอร์เซอร์ที่นี่)
            ●  ← คลิกเพื่อเพิ่ม Enter/Space เพื่อเสร็จสิ้น
```

การกด **Escape** ได้ตลอดเวลาจะทิ้งจุดทั้งหมดที่วางและออกจากคำสั่ง

## การป้อนพิกัด

แทนที่จะคลิก พิมพ์ตำแหน่งที่แน่นอนสำหรับจุดยอดใดก็ได้:

1. พิมพ์ค่า X
2. กด `,` — เทอร์มินัลแสดง `[X], [Y{cursor}]`
3. พิมพ์ค่า Y
4. กด **Enter** เพื่อวางจุดยอด

## การล็อกมุมและความยาวส่วนที่แน่นอน

ตรรกะสแนป 45° เดียวกับคำสั่ง [Line](../line/#angle-locking-and-exact-length-input) ใช้ระหว่างสองจุดต่อเนื่อง เมื่อล็อกกับแกน:

| ปุ่ม | การกระทำ |
|-----|----------|
| `0`–`9`, `.` | เพิ่มตัวเลขเข้าไปในความยาวส่วน |
| `-` | ความยาวติดลบ — กลับทิศทางตามแกน (อักขระตัวแรกเท่านั้น) |
| `Backspace` | ลบอักขระที่พิมพ์ล่าสุด |
| `Enter` | วางจุดถัดไปที่ระยะที่พิมพ์ |

ความยาวสะสมปัจจุบันแสดงในเทอร์มินัลแบบเรียลไทม์ การคลิกขณะล็อกจะฉายลงบนแกนดังนั้นจุดยอดใหม่ตกพอดีบนแกน

## อ้างอิงแป้นพิมพ์

| ปุ่ม | การกระทำ |
|-----|----------|
| `0`–`9`, `.`, `-` | เริ่มป้อนพิกัด X หรือความยาวส่วนเมื่อล็อกมุม |
| `,` | ล็อก X และย้ายไปป้อน Y |
| `Backspace` | ลบอักขระที่พิมพ์ล่าสุด |
| `Enter` | ยืนยันพิกัดหรือความยาวที่พิมพ์ หรือเสร็จสิ้น polyline ถ้าไม่มีอะไรพิมพ์และมีจุด ≥ 2 |
| `Space` | เสร็จสิ้น polyline (เหมือน Enter เมื่อไม่มีการป้อน) |
| `Escape` | ทิ้งจุดทั้งหมดและออก |

## การแก้ไขกริป — จุดยอดและจุดกึ่งกลางส่วน

polyline ที่เลือกแสดงกริปสองประเภท:

| กริป | ตำแหน่ง | สิ่งที่ทำ |
|------|---------|-----------|
| **Vertex** | ที่แต่ละจุดที่วาง | ลากเพื่อย้ายจุดยอดนั้น ส่วนที่เชื่อมต่อทั้งหมดยืดตาม |
| **Segment midpoint** | จุดกึ่งกลางของแต่ละส่วน | ลากเพื่อเลื่อน **ทั้งสอง** จุดสิ้นสุดของส่วนนั้นพร้อมกัน รักษาความยาวและมุมของส่วน |

กริปจุดกึ่งกลางส่วนเป็นเอกลักษณ์ของ polyline — ให้คุณเลื่อนส่วนแต่ละส่วนด้านข้างโดยไม่เปลี่ยนความยาว บน [Line](../line/) กริปจุดกึ่งกลางจะเปิดใช้งานคำสั่ง Move สำหรับวัตถุทั้งหมดแทน

ไม่มีกริป "ย้าย polyline ทั้งหมด" หากต้องการย้ายเส้นทางทั้งหมด ใช้คำสั่ง [Move](../move/)

## การเลือก polylines

| วิธี | พฤติกรรม |
|------|-----------|
| **คลิก** | เลือก polyline ถ้าคลิกอยู่ภายในระยะทดสอบของส่วนใดก็ได้ |
| **ลากไปขวา** (เข้มงวด) | จุดยอดทั้งหมดต้องอยู่ในกล่อง |
| **ลากไปซ้าย** (ตัดผ่าน) | ส่วนใดที่ตัดขอบกล่องจะเลือก polyline ทั้งหมด |

เนื่องจาก polyline เป็นวัตถุเดียว การเลือกตัดผ่านที่สัมผัสส่วนใดจะเลือกส่วนทั้งหมด

## คำสั่งแก้ไขที่รองรับ

Polylines รองรับการแปลงทั่วไปทั้งหมดและ offset แต่ **ไม่** trim หรือ extend (เฉพาะ [Line](../line/)):

| คำสั่ง | สิ่งที่เกิดขึ้นกับ polyline |
|--------|--------------------------|
| [Move](../move/) | เลื่อนจุดยอดทั้งหมดด้วยการกระจัดเดียวกัน |
| [Copy](../copy/) | สร้าง polyline ที่เหมือนกันในตำแหน่งใหม่ |
| [Rotate](../rotate/) | หมุนจุดยอดทั้งหมดรอบจุดฐานที่เลือก |
| [Mirror](../mirror/) | สะท้อนจุดยอดทั้งหมดข้ามแกนสะท้อน |
| [Scale](../scale/) | ปรับมาตราส่วนจุดยอดทั้งหมดสม่ำเสมอจากจุดฐาน |
| [Offset](../offset/) | สร้าง polyline ขนานที่ระยะตั้งฉากคงที่ |
| [Delete](../delete/) | ลบ polyline ออกจากภาพวาด |

## คุณสมบัติ

**ทั่วไป**

| คุณสมบัติ | ค่าเริ่มต้น | ความหมาย |
|-----------|-----------|----------|
| Color | 256 (ByLayer) | ดัชนีสี ACI |
| Layer | `0` | การกำหนดเลเยอร์ |
| Linetype | ByLayer | รูปแบบเส้นที่มีชื่อ |
| Linetype Scale | 1 | ตัวคูณมาตราส่วนสำหรับรูปแบบเส้น |
| Thickness | 0 | ความหนาของการรีดออก |

**เรขาคณิต**

| คุณสมบัติ | ความหมาย |
|-----------|----------|
| Closed | ว่าจุดยอดสุดท้ายเชื่อมกลับไปยังจุดแรกหรือไม่ |
| Vertex Count | จำนวนจุดยอดทั้งหมด |
| Vertices | รายการพิกัดของจุดยอดทั้งหมด |

## Polyline vs Line — เมื่อไหรควรใช้อะไร

| | Polyline | Line |
|---|---------|------|
| จำนวนวัตถุ | หนึ่ง `LWPOLYLINE` สำหรับเส้นทางทั้งหมด | หนึ่ง `LINE` ต่อส่วน |
| รูปทรงปิด | ใช่ (แฟล็กปิด) | ไม่ |
| Trim / Extend | ไม่ | ใช่ — ทีละส่วน |
| กริปจุดกึ่งกลางส่วน | เลื่อนส่วนทั้งหมด | เปิดใช้งาน Move สำหรับวัตถุ |
| เหมาะสำหรับ | โครงร่าง เส้นขอบ รูปทรงที่เก็บไว้ทั้งหมด | เส้นก่อสร้าง เรขาคณิตที่จะตัด |

## DXF — วัตถุ LWPOLYLINE

Polylines ถูกบันทึกเป็นวัตถุ `LWPOLYLINE` ในไฟล์ DXF คุณสมบัติทั้งหมด — พิกัดจุดยอด แฟล็กปิด สี เลเยอร์ ประเภทเส้น มาตราส่วนประเภทเส้น และความหนา — รอบเที่ยวโดยไม่สูญเสียข้อมูล Rectangles ที่วาดด้วยคำสั่ง [Rectangle](../rectangle/) ยังบันทึกเป็น `LWPOLYLINE` (ปิด สี่จุดยอด) และแยกไม่ออกในระดับ DXF
