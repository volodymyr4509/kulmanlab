# Redo

คำสั่ง `redo` เดินหน้าในประวัติการ Undo โดยนำการกระทำที่ถูกยกเลิกด้วย [Undo](../undo/) กลับมาใช้ Redo พร้อมใช้เฉพาะเมื่อคุณถอยหลังด้วย Undo และยังไม่ได้ทำการเปลี่ยนแปลงใหม่

## วิธี Redo

- พิมพ์ `redo` ในเทอร์มินัล หรือ
- คลิกปุ่ม **Redo** บนแถบเครื่องมือ

แต่ละครั้งที่เรียกใช้จะนำการกระทำที่ถูก Undo ก่อนหน้าหนึ่งรายการกลับมา เรียกซ้ำเพื่อเดินหน้าในรายการ Redo ที่มีทั้งหมด

## พฤติกรรมของสแตก Redo

| รายละเอียด | พฤติกรรม |
|-----------|-----------|
| พร้อมใช้หลังจาก | ขั้นตอน [Undo](../undo/) หนึ่งหรือมากกว่า |
| ถูกล้างโดย | **การกระทำวาดใหม่ใดๆ** — การเพิ่ม แก้ไข หรือลบวัตถุ |
| การจัดเก็บ | เบราว์เซอร์ ต่อไฟล์ — คงอยู่หลังโหลดหน้าใหม่ (ตราบเท่าที่ไม่มีการกระทำใหม่ก่อนโหลดซ้ำ) |
| ความลึกสูงสุด | สูงสุด 20 รายการ (กลุ่มเดียวกับ Undo) |

เมื่อวาดวัตถุใหม่ ลบ หรือแก้ไขวัตถุ สแตก Redo จะถูกล้างและรายการเหล่านั้นไม่สามารถกู้คืนได้ เฉพาะการกระทำที่ถูก Undo และยังไม่ถูกแทนที่ด้วยงานใหม่เท่านั้นที่สามารถ Redo ได้

## Redo vs Undo

| | Redo | Undo |
|---|------|------|
| ทิศทาง | เดินหน้าในรายการที่ถูก Undo | ถอยหลังในประวัติ |
| พร้อมใช้เมื่อ | หลัง Undo อย่างน้อยหนึ่งครั้ง โดยไม่มีการกระทำใหม่ | มีการกระทำที่บันทึกไว้อย่างน้อยหนึ่งรายการ |
| ถูกล้างโดย | การกระทำวาดใหม่ใดๆ | ไม่มี |

ปุ่ม Redo บนแถบเครื่องมือจะเป็นสีเทาเมื่อไม่มีรายการที่จะ Redo ใช้ [Undo](../undo/) ก่อนเพื่อสร้างรายการ Redo
