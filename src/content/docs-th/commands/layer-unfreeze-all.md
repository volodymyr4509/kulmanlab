# LayerUnfreezeAll

คำสั่ง `LayerUnfreezeAll` ล้างแฟล็กซ่อนบน **ทุกเลเยอร์** ในภาพวาดทันที ไม่ต้องเลือกหรือยืนยัน — รันและเสร็จสิ้นในขั้นตอนเดียว

## การใช้งาน

พิมพ์ `LayerUnfreezeAll` ในเทอร์มินัลหรือคลิกปุ่ม **Unfreeze All** บนแถบเครื่องมือ (ไอคอนดวงอาทิตย์) เลเยอร์ที่ซ่อนทั้งหมดมองเห็นได้ทันที

## เมื่อใช้

โดยปกติใช้หลังจาก [LayerIsolate](../layer-isolate/) เพื่อกู้คืนเลเยอร์ทั้งหมดให้กลับสู่สถานะมองเห็นปกติ

## รายละเอียดพฤติกรรม

- ใช้กับเลเยอร์ทั้งหมดโดยไม่คำนึงถึงสถานะปัจจุบัน
- ไม่กระทบแฟล็กล็อกหรือพล็อต — เฉพาะแฟล็กซ่อนเท่านั้นที่เปลี่ยน
- คำสั่งเสร็จสิ้นทันทีโดยไม่มีพรอมต์
