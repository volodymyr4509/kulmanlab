# PageManager

คำสั่ง `PageManager` เปิดกล่องโต้ตอบการตั้งค่าหน้าสำหรับเค้าโครงที่ใช้งาน ให้คุณเปลี่ยนรูปแบบกระดาษ ทิศทาง และมาตราส่วนระหว่างหน่วยวาดและมิลลิเมตร

## การเปิด Page Manager

- พิมพ์ `PageManager` ในเทอร์มินัล **หรือ**
- คลิกขวาแท็บเค้าโครงที่ด้านล่างและเลือก **Page Manager**

## การตั้งค่า

| การตั้งค่า | คำอธิบาย |
|-----------|----------|
| **Paper format** | ขนาดแผ่นมาตรฐาน (A4, A3, A2, Letter ฯลฯ) |
| **Orientation** | แนวตั้งหรือแนวนอน |
| **Scale** | หน่วยวาดต่อมม — ควบคุมวิธีการแมปพิกัดโมเดลกับกระดาษ |

การเปลี่ยนขนาดกระดาษหรือมาตราส่วนจะปรับขนาดวิวพอร์ตที่มีอยู่ทั้งหมดในเค้าโครงตามสัดส่วนเพื่อให้ตำแหน่งของพวกมันยังคงถูกต้อง

## หมายเหตุ

- PageManager ใช้กับเค้าโครงกระดาษเท่านั้น ไม่สามารถใช้งานใน model space
- หากต้องการเพิ่มเค้าโครงใหม่ คลิกปุ่ม **+** ในแถบแท็บที่ด้านล่าง
- หากต้องการเปลี่ยนชื่อหรือลบเค้าโครง คลิกขวาแท็บของมัน
