# Wipe Storage

คำสั่ง `wipestorage` ลบ **ข้อมูลทั้งหมดที่เก็บในเบราว์เซอร์** สำหรับ KulmanLab CAD อย่างถาวร — ทุกไฟล์ที่บันทึก ตารางเลเยอร์และ linetype และประวัติ Undo หน้าจะโหลดใหม่โดยอัตโนมัติหลังจากนั้น

:::danger ไม่สามารถย้อนกลับได้
การกระทำนี้ไม่สามารถย้อนกลับได้ ไฟล์ทั้งหมดที่เก็บในเบราว์เซอร์จะถูกลบ ส่งออกภาพวาดที่ต้องการเก็บเป็นไฟล์ `.json` หรือ `.dxf` ก่อนรันคำสั่งนี้
:::

## เมื่อใดควรใช้

- การจัดเก็บเบราว์เซอร์เสียหายและแอปไม่สามารถโหลดหรือบันทึกไฟล์ได้
- ต้องการรีเซ็ตแอปให้กลับสู่สภาพเดิม
- เปลี่ยนเบราว์เซอร์หรืออุปกรณ์และไม่ต้องการสำเนาท้องถิ่นอีกต่อไป

## วิธีรัน

1. พิมพ์ `wipestorage` ในเทอร์มินัลแล้วกด **Enter**
2. เทอร์มินัลแสดง: *Wipe all browser local storage? Type YES to confirm*
3. พิมพ์ `YES` (ตัวพิมพ์ใหญ่/เล็กใดก็ได้) แล้วกด **Enter**

แอปจะลบฐานข้อมูลและโหลดหน้าใหม่ ถ้าพิมพ์สิ่งอื่นที่ไม่ใช่ `YES` แล้วกด **Enter** หรือกด **Escape** คำสั่งจะถูกยกเลิกและไม่มีอะไรถูกลบ

## สิ่งที่ถูกลบ

| ข้อมูล | ถูกลบ |
|--------|-------|
| ไฟล์ทั้งหมดที่บันทึกในเบราว์เซอร์ | ใช่ |
| ตารางเลเยอร์และ linetype สำหรับแต่ละไฟล์ | ใช่ |
| ประวัติ Undo / Redo สำหรับแต่ละไฟล์ | ใช่ |

เฉพาะข้อมูลที่เก็บในท้องถิ่นใน **เบราว์เซอร์นี้** เท่านั้นที่ได้รับผลกระทบ ไฟล์ที่ส่งออกเป็น `.json` หรือ `.dxf` แล้วจะไม่ถูกแตะต้อง
