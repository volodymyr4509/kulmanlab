# Move

คำสั่ง `move` เลื่อนวัตถุที่เลือกจากจุดฐานไปยังจุดปลายทาง การกระจัดที่ใช้กับวัตถุที่เลือกทุกชิ้นคือเวกเตอร์จากฐานถึงปลายทาง หลังจากย้ายวัตถุทั้งหมดยังคงถูกเลือกที่ตำแหน่งใหม่ พร้อมสำหรับการแก้ไขเพิ่มเติม

## สองวิธีเริ่มต้น

**เลือกก่อน แล้วย้าย** — เลือกวัตถุก่อน แล้วเปิดใช้งาน:

1. เลือกวัตถุหนึ่งหรือมากกว่าบนแคนวาส
2. พิมพ์ `move` ในเทอร์มินัลหรือคลิกปุ่ม **Move** บนแถบเครื่องมือ
3. **คลิกจุดฐาน** หรือพิมพ์ `X,Y` แล้วกด **Enter** เพื่อป้อนพิกัดที่แน่นอน
4. **คลิกปลายทาง** — วัตถุที่เลือกทั้งหมดเลื่อนด้วยเวกเตอร์ฐาน→ปลายทาง การป้อนพิกัดก็ใช้ได้ที่นี่เช่นกัน

**เปิดใช้งานก่อน แล้วเลือก** — เริ่มคำสั่งโดยไม่มีการเลือก:

1. พิมพ์ `move` หรือคลิกปุ่มแถบเครื่องมือ
2. **เลือกวัตถุ** — คลิกเพื่อสลับวัตถุแต่ละชิ้น หรือลากเพื่อเลือกตามพื้นที่
3. กด **Enter** หรือ **Space** เพื่อยืนยันการเลือก
4. **คลิกจุดฐาน** แล้ว **คลิกปลายทาง** (การป้อนพิกัดใช้ได้ทั้งสองขั้นตอน)

```
  ก่อน:                    หลัง:
  ● ฐาน                       → ● ปลายทาง
  [วัตถุ A]                      [วัตถุ A ที่ย้าย]
  [วัตถุ B]                      [วัตถุ B ที่ย้าย]
```

ตัวอย่างเงาของวัตถุที่เลือกทั้งหมดติดตามเคอร์เซอร์จากจุดฐานถึงปลายทาง แสดงผลลัพธ์ก่อนคลิก

## การป้อนพิกัด

ที่ขั้นตอนจุดฐานหรือปลายทาง พิมพ์ตำแหน่งที่แน่นอนแทนการคลิก:

1. พิมพ์ค่า X
2. กด `,` — เทอร์มินัลแสดง `[X], [Y{cursor}]`
3. พิมพ์ค่า Y
4. กด **Enter** เพื่อยืนยัน

## การล็อกมุมและระยะที่แน่นอน

หลังจากตั้งจุดฐาน คำสั่งตรวจสอบแกนสแนป 45° (0°, 45°, 90°, 135°, …) ทิศทาง **ล็อก** เมื่อเคอร์เซอร์อยู่ไกลจากฐานและภายในหนึ่งความกว้างกริปของแกน ขณะล็อก:

- ตัวอย่างเงาสแนปกับแกน
- พิมพ์ระยะแล้วกด **Enter** เพื่อย้ายตามทิศทางที่ล็อกนั้นพอดี
- การคลิกฉายลงบนแกน ดังนั้นปลายทางอยู่บนแกนเสมอ

| ปุ่ม | การกระทำ |
|-----|----------|
| `0`–`9`, `.` | เพิ่มเข้าไปในค่าระยะ |
| `-` | ระยะติดลบ — กลับทิศทางตามแกน (อักขระตัวแรกเท่านั้น) |
| `Backspace` | ลบอักขระที่พิมพ์ล่าสุด |
| `Enter` | ใช้การย้ายที่ระยะที่พิมพ์ |

## อ้างอิงแป้นพิมพ์

| ปุ่ม | การกระทำ |
|-----|----------|
| `Enter` / `Space` | ยืนยันการเลือกและดำเนินการไปยังระยะจุดฐาน |
| `0`–`9`, `.`, `-` | เริ่มป้อนพิกัด X หรือระยะเมื่อล็อกมุม |
| `,` | ล็อก X และย้ายไปป้อน Y |
| `Backspace` | ลบอักขระที่พิมพ์ล่าสุด |
| `Enter` | ยืนยันพิกัดหรือใช้การย้ายที่ระยะที่พิมพ์ |
| `Escape` | ยกเลิกและรีเซ็ต |

## การเปิดใช้งาน Move จากกริป

การคลิก **กริปจุดกึ่งกลาง** ของ [Line](./line) ที่เลือกจะเปิดใช้งาน Move โดยอัตโนมัติ โดยมีจุดกึ่งกลางตั้งเป็นจุดฐานแล้วและระยะย้ายทำงานอยู่ นี่คือวิธีที่เร็วที่สุดในการวางตำแหน่งเส้นเดี่ยวโดยไม่ต้องผ่านขั้นตอนการเลือก

## การเลือกระหว่างคำสั่ง

เมื่อคำสั่งเริ่มในระยะการเลือก:

| วิธี | พฤติกรรม |
|------|-----------|
| **คลิก** | สลับวัตถุใต้เคอร์เซอร์เข้า/ออกจากการเลือก |
| **ลากไปขวา** (เข้มงวด) | เพิ่มวัตถุที่อยู่ในกล่องทั้งหมด |
| **ลากไปซ้าย** (ตัดผ่าน) | เพิ่มวัตถุที่ตัดขอบกล่อง |
| **Enter** / **Space** | ยืนยันการเลือกและย้ายไปยังระยะจุดฐาน |

## หลังจากย้าย

วัตถุที่ย้ายยังคงถูกเลือกที่ตำแหน่งใหม่ ซึ่งหมายความว่าคุณสามารถ:
- รัน **Move** อีกครั้งเพื่อขยับเพิ่มเติม
- รัน [Copy](./copy), [Rotate](./rotate) หรือ [Scale](./scale) โดยไม่ต้องเลือกใหม่
- กด **Delete** เพื่อลบพวกมัน

## Move vs Copy

| | Move | Copy |
|---|------|------|
| ตำแหน่งเดิม | ว่าง — วัตถุไม่อยู่ที่นั่นอีกต่อไป | เก็บไว้ — ต้นฉบับอยู่ในตำแหน่งเดิม |
| จำนวนผลลัพธ์ | จำนวนวัตถุเดิม | หนึ่งชุดเพิ่มเติมต่อการดำเนินการ |
| การเลือกหลัง | วัตถุที่ย้ายถูกเลือกที่ตำแหน่งใหม่ | วัตถุที่คัดลอกถูกเลือกที่ตำแหน่งใหม่ |
| เหมาะสำหรับ | วางตำแหน่งเรขาคณิต | ทำสำเนาเรขาคณิต |

## วัตถุที่รองรับ

Move ทำงานกับวัตถุทุกประเภท: Line, Polyline, Rectangle, Circle, Arc, Ellipse, Text, Spline, Dimension, Leader และอื่นๆ ทั้งหมด วัตถุทุกชิ้นใช้ `translate(dx, dy)` ดังนั้นไม่มีการยกเว้น
