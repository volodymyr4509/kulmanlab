# Area

คำสั่ง `area` วัดพื้นที่ที่ปิดล้อมและเส้นรอบวงของรูปหลายเหลี่ยมที่กำหนดด้วยจุดที่คลิกตั้งแต่สามจุดขึ้นไป และพิมพ์ผลลัพธ์ทั้งสองในเทอร์มินัลถึง 4 ตำแหน่งทศนิยม เป็นคำสั่งวัดที่สาม ควบคู่กับ [Distance](../distance/) (ความยาวเส้นตรง) และ [Angle](../angle/) (มุมภายในที่จุดยอด)

## กายวิภาคของการวัดพื้นที่

```
  ● จุดแรก
   \
    \
     ● จุดที่สอง
      \
       \             (เส้นประ) ตัวอย่างขอบปิด
        ●───────────────┐
      จุดที่สาม         │  (เส้นประ) ตัวอย่างขอบถัดไปไปยังเคอร์เซอร์
                        ✕ เคอร์เซอร์  →  เทอร์มินัล: "Area: 12.3456  Perimeter: 45.6789"
```

- **จุดยอด** — แต่ละจุดที่คลิก (หรือพิมพ์) จะกลายเป็นจุดยอดของรูปหลายเหลี่ยม ขอบที่ยืนยันแล้วจะวาดด้วยเส้นทึบ และภายในจะเติมด้วยการไฮไลต์แบบโปร่งแสง
- **ขอบตัวอย่าง** — เส้นประแสดงขอบที่ค้างอยู่จากจุดยอดสุดท้ายไปยังเคอร์เซอร์ และขอบปิดจากเคอร์เซอร์กลับไปยังจุดยอดแรก
- **ขอบปิด** — คุณไม่ต้องคลิกจุดแรกซ้ำอีก การกด Enter จะปิดรูปหลายเหลี่ยมโดยอัตโนมัติ

## การวัดพื้นที่

1. พิมพ์ `area` ในเทอร์มินัลหรือคลิกปุ่ม **Area** บนแถบเครื่องมือ (แถวล่างของแผง Measure)
2. **คลิกจุดแรก** หรือพิมพ์ `X,Y` แล้วกด **Enter** เพื่อป้อนพิกัดที่แน่นอน
3. **คลิกจุดยอดเพิ่มเติมแต่ละจุด** ตามลำดับรอบรูปทรง การป้อนพิกัดใช้ได้ในทุกขั้นตอน
4. เมื่อวางอย่างน้อย **3 จุด** แล้ว ให้กด **Enter** (โดยไม่มีการป้อนพิกัดหรือระยะที่ค้างอยู่) เพื่อปิดรูปหลายเหลี่ยมและคำนวณผลลัพธ์
5. เทอร์มินัลจะพิมพ์ `Area: <ค่า>  Perimeter: <ค่า>` และรูปหลายเหลี่ยมที่ปิดแล้ว — การเติมสี เส้นขอบ และจุดจับที่จุดยอด — จะยังคงไฮไลต์อยู่บนแคนวาส
6. **คลิกที่ใดก็ได้ กดปุ่มใดก็ได้ หรือกด `Escape`** เพื่อปิดผลลัพธ์และสิ้นสุดคำสั่ง

## การล็อกมุมและระยะที่แน่นอน

หลังจากวางจุดยอดแรกแล้ว การเคลื่อนไปในทิศทางของการเพิ่มการติดตามมุมที่กำหนดค่าไว้ (10°, 15°, 20°, 30°, 45° หรือ 90° ปรับได้ผ่านเมนูแบบเลื่อนลงบนแถบเครื่องมือ) จะล็อกขอบถัดไปไปยังทิศทางนั้น:

- ตัวอย่างขอบจะยึดติดกับทิศทางที่ล็อก และตัวบ่งชี้การติดตามมุมจะถูกวาดที่จุดยอดยึด
- พิมพ์ความยาวและกด **Enter** เพื่อวางจุดยอดถัดไปตามระยะนั้นพอดีตามแนวทิศทางที่ล็อก
- การคลิกขณะล็อก (โดยไม่พิมพ์ความยาว) จะวางจุดยอดที่ตำแหน่งฉายของเคอร์เซอร์บนทิศทางที่ล็อก

| ปุ่ม | การกระทำ |
|-----|----------|
| `0`–`9`, `.` | เพิ่มเข้าไปในค่าความยาวขอบ |
| `-` | ความยาวติดลบ (เป็นอักขระแรกเท่านั้น) |
| `Backspace` | ลบอักขระที่พิมพ์ล่าสุด |
| `Enter` | วางจุดยอดถัดไปที่ความยาวที่พิมพ์ |

## การปิดรูปหลายเหลี่ยม

- Enter จะปิดรูปทรงก็ต่อเมื่อวาง **3 จุดยอดขึ้นไป** แล้วเท่านั้น — หากน้อยกว่านั้นจะไม่มีผลใด ๆ
- ขอบจากจุดยอดสุดท้ายกลับไปยังจุดแรกจะถูกเพิ่มโดยอัตโนมัติและนับรวมทั้งในพื้นที่และเส้นรอบวง
- สามารถวางจุดในลำดับใดก็ได้ (ตามเข็มนาฬิกาหรือทวนเข็มนาฬิกา) — ผลลัพธ์จะเหมือนกัน

## Area เทียบกับ Distance เทียบกับ Angle

| | Area | Distance | Angle |
|---|------|---------|-------|
| สิ่งที่วัด | พื้นที่และเส้นรอบวงของรูปหลายเหลี่ยม | ความยาวเส้นตรง | มุมภายในที่จุดยอด |
| จำนวนคลิก | 3 จุดขึ้นไป ปิดด้วย Enter | 2 | 3 |
| รูปแบบผลลัพธ์ | `12.3456  Perimeter: 45.6789` | `12.3456` (หน่วย) | `45.0000°` |
| ตัวอย่างบนแคนวาส | รูปหลายเหลี่ยมที่เติมสีพร้อมขอบปิดเป็นเส้นประ | เส้นจากจุดแรกไปยังเคอร์เซอร์ | สองเส้นจากจุดยอดไปยังเคอร์เซอร์ |
| หลังผลลัพธ์ | ปิดได้ด้วยการป้อนข้อมูลใด ๆ จากนั้นคำสั่งจะสิ้นสุด | คลิกเพื่อต่อเชื่อมการวัดใหม่ | คลิกเพื่อต่อเชื่อมการวัดใหม่ |
| เหมาะสำหรับ | พื้นที่ปิดล้อม พื้นที่ห้องหรือแผง | ความยาวของช่องว่างหรือส่วน | มุมเปิดระหว่างสองลักษณะ |

## การป้อนพิกัด

แทนที่จะคลิก พิมพ์ตำแหน่งที่แน่นอนสำหรับจุดยอดใดก็ได้:

1. พิมพ์ค่า X
2. กด `,` — เทอร์มินัลแสดง `[X], [Y{cursor}]`
3. พิมพ์ค่า Y
4. กด **Enter** เพื่อยืนยัน

## อ้างอิงแป้นพิมพ์

| ปุ่ม | การกระทำ |
|-----|----------|
| `0`–`9`, `.`, `-` | เริ่มป้อนพิกัด X หรือป้อนความยาวขอบเมื่อล็อกมุมทำงานอยู่ |
| `,` | ล็อก X และย้ายไปป้อน Y |
| `Backspace` | ลบอักขระที่พิมพ์ล่าสุด |
| `Enter` | ยืนยันพิกัดหรือความยาวที่พิมพ์ เมื่อมี 3 จุดยอดขึ้นไปและไม่มีการป้อนที่ค้างอยู่ จะปิดรูปหลายเหลี่ยม |
| `Escape` | ขณะเลือกจุดยอด: ยกเลิกจุดที่วางไว้และเริ่มใหม่ที่จุดแรก หลังจากผลลัพธ์แสดง: ปิดผลลัพธ์และสิ้นสุดคำสั่ง |

## หมายเหตุ

- พื้นที่คำนวณด้วยสูตร shoelace และรายงานเป็นค่าบวกเสมอ ไม่ว่าลำดับการคลิกจะเป็นอย่างไร
- รูปหลายเหลี่ยมที่ตัดกันเอง (ขอบที่ตัดกัน) ยังคงให้ผลลัพธ์เป็นตัวเลข แต่ค่าอาจไม่ตรงกับพื้นที่ที่ปิดล้อมในเชิงภาพ — รักษาลำดับการคลิกที่ไม่ตัดกันเพื่อให้ได้พื้นที่ที่มีความหมาย
- ผลลัพธ์แสดงเฉพาะใน **เทอร์มินัลและเป็นการไฮไลต์ชั่วคราวบนแคนวาส** — ไม่มีอะไรถูกเพิ่มลงในภาพวาดอย่างถาวร
- ต่างจาก Distance และ Angle ตรงที่ Area **จะไม่** ต่อเชื่อมไปยังการวัดใหม่โดยอัตโนมัติ — หลังจากปิดผลลัพธ์แล้ว ให้เรียก `area` อีกครั้งเพื่อวัดรูปหลายเหลี่ยมอื่น
- ความแม่นยำเสมอ 4 ตำแหน่งทศนิยมทั้งสำหรับพื้นที่และเส้นรอบวง โดยใช้หน่วยเดียวกับพิกัดภาพวาด (ไม่มีการแปลงหน่วย)
