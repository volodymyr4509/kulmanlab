# ViewportCopy

คำสั่ง `ViewportCopy` คัดลอก viewport ไปยังตำแหน่งใหม่ โดยรักษาสเกลและศูนย์กลางโมเดล พร้อมใช้เฉพาะในพื้นที่เลย์เอาต์เท่านั้น

## การคัดลอก Viewport

1. สลับไปยังแท็บเลย์เอาต์กระดาษ
2. คลิก viewport เพื่อเลือกล่วงหน้าได้ (ไม่บังคับ)
3. พิมพ์ `ViewportCopy` ในเทอร์มินัลหรือคลิกปุ่ม **Viewport Copy** บนแถบเครื่องมือ
4. ถ้าไม่มีการเลือก viewport ล่วงหน้า **คลิก viewport** ที่ต้องการคัดลอก
5. **คลิกจุดฐาน** — อ้างอิงสำหรับการกระจัด หรือพิมพ์ `X,Y` แล้วกด **Enter** เพื่อป้อนพิกัดที่แน่นอน
6. **คลิกปลายทาง** — viewport จะถูกวางที่ออฟเซ็ตฐาน→ปลายทาง หรือใช้การป้อนพิกัด / การล็อกมุม

หลังจากวาง คำสั่งยังคงทำงาน — คลิกปลายทางอื่นเพื่อวางสำเนาอีกชุดของ viewport เดิม กด **Enter**, **Space** หรือ **Escape** เพื่อเสร็จสิ้น

## การป้อนพิกัด

ที่ขั้นตอนจุดฐานและปลายทาง คุณสามารถพิมพ์พิกัดที่แน่นอนแทนการคลิก:

1. พิมพ์ค่า X
2. กด `,` — เทอร์มินัลแสดง `[X], [Y{cursor}]`
3. พิมพ์ค่า Y
4. กด **Enter** เพื่อยืนยัน

## การล็อกมุมและระยะที่แน่นอน

หลังจากตั้งจุดฐาน คำสั่งจะสแนปกับแกน 45° (0°, 45°, 90°, 135°, …) เมื่อเคอร์เซอร์อยู่แนว ขณะล็อก:

- ตัวอย่างสแนปกับแกน
- พิมพ์ระยะแล้วกด **Enter** เพื่อวางสำเนาที่ออฟเซ็ตนั้นพอดีตามทิศทางที่ล็อก

| ปุ่ม | การกระทำ |
|-----|----------|
| `0`–`9`, `.` | เพิ่มตัวเลขเข้าค่าระยะ |
| `-` | ระยะติดลบ (กลับทิศทาง อักขระตัวแรกเท่านั้น) |
| `Backspace` | ลบอักขระที่พิมพ์ล่าสุด |
| `Enter` | วางสำเนาที่ระยะที่พิมพ์ |

## อ้างอิงแป้นพิมพ์

| ปุ่ม | การกระทำ |
|-----|----------|
| `0`–`9`, `.`, `-` | เริ่มป้อนพิกัด X หรือระยะเมื่อล็อกมุม |
| `,` | ล็อก X และย้ายไปป้อน Y |
| `Enter` | ยืนยันพิกัดหรือระยะที่พิมพ์ |
| `Enter` / `Space` | เสร็จสิ้น (เมื่อไม่มีการป้อนอยู่) |
| `Escape` | ยกเลิกและรีเซ็ต |

## หมายเหตุ

- ViewportCopy พร้อมใช้เฉพาะเมื่อแท็บเลย์เอาต์กระดาษทำงานอยู่
- viewport ที่คัดลอกสืบทอดสเกล ศูนย์กลางโมเดล สถานะล็อก และขนาดเดียวกับต้นฉบับ
- หากต้องการสร้าง viewport ใหม่ตั้งแต่ต้น ให้ใช้ [ViewportRectangle](../viewport-rectangle/)
