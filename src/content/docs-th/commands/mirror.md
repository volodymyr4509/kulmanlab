# Mirror

คำสั่ง `mirror` สร้างสำเนาที่สะท้อนของวัตถุที่เลือกข้ามแกนสองจุด ต้นฉบับถูก **เก็บไว้เสมอ** — ต่างจาก [Move](./move) หรือ [Rotate](./rotate) Mirror ไม่แก้ไขวัตถุที่มีอยู่ เพียงเพิ่มวัตถุใหม่

## สองวิธีเริ่มต้น

**เลือกก่อน แล้ว mirror** — เลือกวัตถุก่อน แล้วเปิดใช้งาน:

1. เลือกวัตถุหนึ่งหรือมากกว่าบนแคนวาส
2. พิมพ์ `mirror` ในเทอร์มินัลหรือคลิกปุ่ม **Mirror** บนแถบเครื่องมือ
3. **คลิกจุดแรก** ของแกนสะท้อน หรือพิมพ์ `X,Y` แล้วกด **Enter** เพื่อป้อนพิกัดที่แน่นอน
4. **คลิกจุดที่สอง** — สำเนาที่สะท้อนถูกวางและคำสั่งออก การป้อนพิกัดก็ใช้ได้ที่นี่เช่นกัน

**เปิดใช้งานก่อน แล้วเลือก** — เริ่มคำสั่งโดยไม่มีการเลือก:

1. พิมพ์ `mirror` หรือคลิกปุ่มแถบเครื่องมือ
2. **เลือกวัตถุ** — คลิกเพื่อสลับ หรือลากเพื่อเลือกตามพื้นที่
3. กด **Enter** หรือ **Space** เพื่อยืนยันการเลือก
4. **คลิกจุดแรก** แล้ว **คลิกจุดที่สอง** ของแกนสะท้อน (การป้อนพิกัดใช้ได้ทั้งสองขั้นตอน)

```
  ต้นฉบับ:          แกนสะท้อน:        ผลลัพธ์:
                     |
  [วัตถุ A]    →    |    →    [วัตถุ A] + [A ที่สะท้อน]
                     |
```

ตัวอย่างสดของสำเนาที่สะท้อนติดตามเคอร์เซอร์ขณะวางจุดแกนที่สอง

## แกนสะท้อน

แกนเป็นเส้นอนันต์ผ่านสองจุดที่คลิก สามารถอยู่ในมุมใดก็ได้:

- ย้ายเคอร์เซอร์ใกล้ **แกนสแนป 45°** (0°, 45°, 90°, 135°, …) และแกนล็อกกับมุมนั้น — มีประโยชน์สำหรับการสะท้อนแนวนอน แนวตั้ง หรือเฉียงที่สะอาด
- คลิกห่างจากโซนสแนปเพื่อแกนมุมอิสระ

## การป้อนพิกัด

ที่ขั้นตอนจุดแกนใด พิมพ์ตำแหน่งที่แน่นอนแทนการคลิก:

1. พิมพ์ค่า X
2. กด `,` — เทอร์มินัลแสดง `[X], [Y{cursor}]`
3. พิมพ์ค่า Y
4. กด **Enter** เพื่อยืนยัน

## อ้างอิงแป้นพิมพ์

| ปุ่ม | การกระทำ |
|-----|----------|
| `Enter` / `Space` | ยืนยันการเลือก |
| `0`–`9`, `.`, `-` | เริ่มป้อนพิกัด X |
| `,` | ล็อก X และย้ายไปป้อน Y |
| `Backspace` | ลบอักขระที่พิมพ์ล่าสุด |
| `Enter` | ยืนยันพิกัดที่พิมพ์ |
| `Escape` | ยกเลิกและรีเซ็ต |

## การเลือกระหว่างคำสั่ง

| วิธี | พฤติกรรม |
|------|-----------|
| **คลิก** | สลับวัตถุใต้เคอร์เซอร์ |
| **ลากไปขวา** (เข้มงวด) | เพิ่มวัตถุที่อยู่ในกล่องทั้งหมด |
| **ลากไปซ้าย** (ตัดผ่าน) | เพิ่มวัตถุที่ตัดกล่อง |
| **Enter** / **Space** | ยืนยันการเลือก |

## สิ่งที่ถูกสะท้อน

วัตถุทุกประเภทรองรับ เรขาคณิตถูกสะท้อนข้ามแกนทางคณิตศาสตร์:

| วัตถุ | สิ่งที่เปลี่ยน |
|-------|--------------|
| Line | จุดสิ้นสุดทั้งสองถูกสะท้อน |
| Circle | ศูนย์กลางถูกสะท้อน รัศมีไม่เปลี่ยน |
| Arc | ศูนย์กลางถูกสะท้อน มุมเริ่มต้น/สิ้นสุดถูกคำนวณใหม่ข้ามแกน |
| Ellipse | ศูนย์กลางถูกสะท้อน ทิศทางแกนหลักพลิกข้ามแกน |
| Polyline / Rectangle | จุดยอดทุกจุดถูกสะท้อน |
| Text | จุดยึดถูกสะท้อน สตริงข้อความ **ไม่** ถูกกลับด้าน |
| Spline | จุดควบคุม/จุดพอดีทั้งหมดถูกสะท้อน |

## Mirror vs Copy

| | Mirror | Copy |
|---|--------|------|
| ต้นฉบับ | เก็บไว้เสมอ | เก็บไว้เสมอ |
| ตำแหน่งวัตถุใหม่ | สะท้อนข้ามแกน | ออฟเซ็ตด้วยเวกเตอร์การกระจัด |
| เหมาะสำหรับ | การออกแบบสมมาตร ลักษณะสองข้าง | ทำซ้ำเรขาคณิตในทิศทางใดก็ได้ |
