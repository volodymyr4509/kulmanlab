# New File

คำสั่ง **New File** ล้างแคนวาสและเริ่มภาพวาดเปล่าใหม่ ชื่อไฟล์ที่ไม่ซ้ำกันพร้อมการประทับเวลาถูกสร้างโดยอัตโนมัติ

## วิธีสร้างไฟล์ใหม่

คลิกปุ่ม **New File** บนแถบเครื่องมือ (ไอคอนหน้าใหม่) ในแผง File แคนวาสล้างทันที — ไม่มีพรอมต์หรือกล่องยืนยัน

## เนื้อหาในไฟล์ใหม่

ไฟล์ที่สร้างใหม่เริ่มต้นด้วย:

- **ไม่มีวัตถุ** บนแคนวาส
- **เลเยอร์เริ่มต้นหนึ่งเลเยอร์** ชื่อ `0` ที่มีสีขาวและ linetype `Continuous`
- **ชื่อไฟล์ที่สร้าง** ในรูปแบบ `kulman-Mon01_HH:MM:SS.dxf` (เช่น `kulman-May22_14:30:00.dxf`)

ไฟล์ถูกบันทึกในหน่วยความจำเบราว์เซอร์โดยอัตโนมัติและปรากฏใน [Recent Files](../files/)

## คำเตือน — งานที่ยังไม่ได้บันทึกถูกทิ้ง

การคลิก **New File** ทิ้งวัตถุทั้งหมดบนแคนวาสปัจจุบันโดยไม่มีคำเตือน หากต้องการเก็บภาพวาดปัจจุบัน [ส่งออก](../export/) ก่อน

## เมื่อใช้ New File vs Import

| สถานการณ์ | การดำเนินการที่แนะนำ |
|-----------|-------------------|
| เริ่มภาพวาดตั้งแต่ต้น | **New File** |
| เปิดไฟล์ DXF หรือ JSON ที่มีอยู่ | [Import](../import/) |
| คัดลอกภาพวาดเพื่อทำงานบนตัวแปร | [Export](../export/) ไฟล์ปัจจุบัน แล้ว [Import](../import/) สำเนา |

## คำสั่งที่เกี่ยวข้อง

- [Import](../import/) — เปิดภาพวาด DXF หรือ JSON ที่มีอยู่
- [Export](../export/) — ดาวน์โหลดภาพวาดก่อนเริ่มใหม่
- [Files](../files/) — กู้คืนภาพวาดก่อนหน้าจากหน่วยความจำเบราว์เซอร์
