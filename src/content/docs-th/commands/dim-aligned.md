# Dimension Aligned

คำสั่ง `dimaligned` วางมิติที่วัด **ระยะเส้นตรงจริง** ระหว่างสองจุด เส้นมิติวิ่งขนานกับเส้นที่เชื่อมสองจุด ดังนั้นจึงอยู่ในมุมใดก็ได้ นี่คือความแตกต่างสำคัญจาก [Dimension Linear](../dim-linear/) ที่จำกัดเฉพาะแนวนอนหรือแนวตั้ง

## กายวิภาคของมิติ aligned

```
     ●  p2
    /|
   / |  (เส้นขยาย 2 ตั้งฉากกับเส้นมิติ)
  /  |
 /←5.00→/
/  /
●  /  (เส้นขยาย 1 ตั้งฉากกับเส้นมิติ)
p1
```

- **เส้นขยาย** — ตั้งฉากกับเส้นมิติ วาดจากแต่ละจุดที่วัด
- **เส้นมิติ** — ขนานกับ p1→p2 ออฟเซ็ตไปด้านหนึ่งตามตำแหน่งเคอร์เซอร์
- **ค่า** — ระยะ Euclidean จริง `|p1 – p2|`

## การวาง aligned dimension

1. พิมพ์ `dimaligned` ในเทอร์มินัลหรือคลิกปุ่ม **Dimension Aligned** บนแถบเครื่องมือ
2. **คลิกจุดกำเนิดเส้นขยายแรก** (p1) หรือพิมพ์ `X,Y` แล้วกด **Enter** เพื่อป้อนพิกัดที่แน่นอน
3. **คลิกจุดกำเนิดเส้นขยายที่สอง** (p2) การป้อนพิกัดก็ใช้ได้ที่นี่เช่นกัน
4. **ย้ายเคอร์เซอร์** ไปด้านหนึ่งเพื่อตั้งออฟเซ็ตตั้งฉากของเส้นมิติ
5. **คลิก** เพื่อวาง หรือพิมพ์ระยะออฟเซ็ตแล้วกด **Enter** เพื่อวางอย่างแม่นยำ

## ระยะออฟเซ็ตแบบพิมพ์

พิมพ์ตัวเลขระหว่างการวางเพื่อตรึงเส้นมิติที่ระยะตั้งฉากที่แน่นอนจากเส้น p1→p2:

| ปุ่ม | การกระทำ |
|-----|----------|
| `0`–`9`, `.` | เพิ่มตัวเลขเข้าไปในออฟเซ็ต |
| `Backspace` | ลบอักขระที่พิมพ์ล่าสุด |
| `Enter` / `Space` | วางที่ออฟเซ็ตที่พิมพ์ |

ด้านเคอร์เซอร์กำหนดว่าเส้นมิติจะปรากฏด้านใด

## อ้างอิงแป้นพิมพ์

| ปุ่ม | การกระทำ |
|-----|----------|
| `0`–`9`, `.`, `-` | เริ่มป้อนพิกัด X (ระยะ p1/p2) หรือระยะออฟเซ็ต (ระยะวาง) |
| `,` | ล็อก X และย้ายไปป้อน Y (ระยะ p1/p2) |
| `Backspace` | ลบอักขระที่พิมพ์ล่าสุด |
| `Enter` / `Space` | ยืนยันพิกัดหรือออฟเซ็ตที่พิมพ์ |
| `Escape` | ยกเลิก |

## Dimension Aligned vs Dimension Linear

| | Dimension Aligned | Dimension Linear |
|---|------------------|-----------------|
| มุมเส้นมิติ | ขนานกับ p1→p2 — มุมใดก็ได้ | แนวนอนหรือแนวตั้งเสมอ |
| วัด | ระยะ Euclidean จริง | ส่วนประกอบ X หรือ Y เท่านั้น |
| การล็อก H/V | ไม่ | ใช่ — ปุ่ม `H` และ `V` |
| เหมาะสำหรับ | ลักษณะเฉียง การตัดมุม | เค้าโครงตั้งฉาก ชิ้นส่วนตามกริด |

## การแก้ไขป้าย — simple mode

**ดับเบิลคลิก** มิติ aligned ที่วางแล้วเพื่อเปิดโปรแกรมแก้ไขข้อความในโหมด **simple** โปรแกรมแก้ไขถูกเติมค่าที่แสดงปัจจุบันล่วงหน้าเพื่อให้คุณวางเคอร์เซอร์และแก้ไขได้โดยตรง

| ฟีเจอร์ | การทำงาน |
|---------|----------|
| Bold / Italic / Font / Height | ใช้กับ**ป้ายทั้งหมด**พร้อมกัน |
| การจัดรูปแบบต่อตัวอักษร | ไม่รองรับ |
| `Enter` | ยืนยันค่าและปิดโปรแกรมแก้ไข |
| หลายบรรทัด | ไม่รองรับ |

ดู [Text Editor — simple mode](../interface/text-editor#simple-mode) สำหรับข้อมูลอ้างอิงทั้งหมด

## การต่อเชื่อมมิติ

หากต้องการเพิ่มมิติต่อเนื่องจากเส้นขยายที่สองของมิตินี้ ใช้ [Dimension Continue](../dim-continue/) — จะล็อกมุมการวัดเดียวกับมิติ aligned นี้

## DXF — วัตถุ DIMENSION (ประเภท aligned)

มิติ aligned ถูกบันทึกเป็นวัตถุ `DIMENSION` ที่มี `dimType = 1` (aligned) จุดกำเนิดเส้นขยาย ตำแหน่งเส้นมิติ ตำแหน่งข้อความ ค่าที่วัด การหมุน รูปแบบลูกศร และแฟล็กแสดงผลทั้งหมดรอบเที่ยวโดยไม่สูญเสียข้อมูล
