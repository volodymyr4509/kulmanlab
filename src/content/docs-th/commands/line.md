# Line

คำสั่ง `line` วาดส่วนของเส้นตรงแต่ละส่วนที่จัดเก็บเป็นวัตถุ `LINE` แยกกันในโมเดล DXF หลังจากแต่ละส่วน คำสั่งจะยังคงทำงานและใช้จุดสิ้นสุดเป็นจุดเริ่มต้นใหม่ ดังนั้นคุณสามารถสร้างเส้นทางที่เชื่อมต่อกันทีละส่วน ต่างจาก [Polyline](../polyline/) เส้นตรงที่ต่อเชื่อมกันยังคงเป็นวัตถุอิสระ — แต่ละเส้นสามารถตัด ยืด หรือลบได้โดยไม่กระทบเพื่อนบ้าน

## การวาดเส้น

1. พิมพ์ `line` ในเทอร์มินัลหรือคลิกปุ่ม **Line** บนแถบเครื่องมือ
2. **คลิกจุดเริ่มต้น** หรือพิมพ์ `X,Y` แล้วกด **Enter** เพื่อป้อนพิกัดที่แน่นอน
3. **คลิกจุดสิ้นสุด** — ส่วนถูกวางและจุดสิ้นสุดกลายเป็นจุดเริ่มต้นใหม่ การป้อนพิกัดก็ใช้ได้ที่นี่เช่นกัน
4. คลิก (หรือพิมพ์) ต่อไปเพื่อต่อเชื่อมส่วนเพิ่มเติม
5. กด **Enter** หรือ **Escape** เพื่อหยุด

```
  ●──────────●──────────●──────────●
 start     คลิกที่ 2  คลิกที่ 3   Enter เพื่อจบ
            (กลายเป็นจุดเริ่มต้นถัดไปอัตโนมัติ)
```

ต้องการแค่ส่วนเดียว? กด **Enter** หรือ **Escape** ทันทีหลังจากขั้นตอนที่ 3

## การป้อนพิกัด

แทนที่จะคลิก ให้พิมพ์ตำแหน่งที่แน่นอนสำหรับจุดเริ่มต้นหรือจุดถัดไป:

1. พิมพ์ค่า X (ตัวเลข `.` หรือ `-`)
2. กด `,` — เทอร์มินัลแสดง `[X], [Y{cursor}]`
3. พิมพ์ค่า Y
4. กด **Enter** เพื่อวางจุด

## การล็อกมุมและการป้อนความยาวที่แน่นอน

เมื่อคุณเลื่อนเคอร์เซอร์หลังจากวางจุด คำสั่งจะตรวจสอบแกนสแนป 45° (0°, 45°, 90°, 135°, …) มุม **ล็อก** เมื่อ:

- เคอร์เซอร์อยู่ห่างจากจุดยึดอย่างน้อย **5 × ขนาดกริป** **และ**
- อยู่ภายใน **1 ขนาดกริป** ของระยะห่างตั้งฉากจากแกนที่ใกล้ที่สุด

เมื่อล็อกแล้ว ตัวอย่างจะสแนปกับแกนและคุณสามารถป้อนความยาวที่แน่นอน:

| ปุ่ม | การกระทำ |
|-----|----------|
| `0`–`9`, `.` | เพิ่มตัวเลขเข้าไปในค่าความยาว |
| `-` | ความยาวติดลบ — กลับทิศทางตามแกน (อักขระตัวแรกเท่านั้น) |
| `Backspace` | ลบอักขระที่พิมพ์ล่าสุด |
| `Enter` | วางจุดสิ้นสุดที่ระยะที่พิมพ์ |

ค่าสะสมแสดงสดในเทอร์มินัล (เช่น `click end point or enter length: 12.5`) คลิกขณะล็อกและคลิกจะถูกฉายลงบนแกน ดังนั้นจุดสิ้นสุดจะอยู่บนแกนเสมอ

การเลื่อนกลับใกล้จุดยึดจะยกเลิกการล็อก

## อ้างอิงแป้นพิมพ์

| ปุ่ม | การกระทำ |
|-----|----------|
| `0`–`9`, `.`, `-` | เริ่มป้อนพิกัด X หรือระยะเมื่อล็อกมุม |
| `,` | ล็อก X และย้ายไปป้อน Y |
| `Backspace` | ลบอักขระที่พิมพ์ล่าสุด |
| `Enter` | ยืนยันพิกัดหรือความยาวที่พิมพ์ หรือจบสายถ้าไม่มีอะไรพิมพ์ |
| `Escape` | จบสายและออก |

## การแก้ไขกริป — ยืดจุดสิ้นสุด

เส้นที่เลือกแสดงกริปสามตัว:

| กริป | ตำแหน่ง | สิ่งที่ทำ |
|------|---------|-----------|
| **Start** | จุดสิ้นสุดแรก | ลากเพื่อย้าย — จุดสิ้นสุดยังคงตรึงอยู่ |
| **Midpoint** | จุดกึ่งกลางของเส้น | เปิดใช้งาน **Move** สำหรับเส้นทั้งเส้น |
| **End** | จุดสิ้นสุดที่สอง | ลากเพื่อย้าย — จุดเริ่มต้นยังคงตรึงอยู่ |

การยืดจุดสิ้นสุดหนึ่งไม่กระทบอีกจุด ซึ่งต่างจากการแก้ไขกริปของ [Polyline](../polyline/) ที่การย้ายจุดยอดจะปรับรูปร่างเส้นทางทั้งหมด

## การเลือกเส้น

| วิธี | พฤติกรรม |
|------|-----------|
| **คลิก** | เลือกเส้นถ้าคลิกอยู่ภายในระยะทดสอบของส่วน |
| **ลากไปขวา** (เข้มงวด) | เส้นถูกเลือกเฉพาะเมื่อจุดสิ้นสุดทั้งสองอยู่ในกล่อง |
| **ลากไปซ้าย** (ตัดผ่าน) | เส้นถูกเลือกถ้าส่วนใดส่วนหนึ่งตัดผ่านขอบกล่อง |

## คำสั่งแก้ไขที่รองรับ

เส้นเป็นวัตถุ **เดียว** ที่ [Trim](../trim/) และ [Extend](../extend/) ทำงานได้ คำสั่งการแปลงมาตรฐานทั้งหมดก็ใช้ได้:

| คำสั่ง | สิ่งที่เกิดขึ้นกับเส้น |
|--------|----------------------|
| [Move](../move/) | เลื่อนจุดสิ้นสุดทั้งสองด้วยการกระจัดเดียวกัน |
| [Copy](../copy/) | สร้างเส้นที่เหมือนกันในตำแหน่งใหม่ |
| [Rotate](../rotate/) | หมุนจุดสิ้นสุดทั้งสองรอบจุดฐานที่เลือก |
| [Mirror](../mirror/) | สะท้อนจุดสิ้นสุดทั้งสองข้ามแกนสะท้อน |
| [Scale](../scale/) | ปรับมาตราส่วนจุดสิ้นสุดทั้งสองสม่ำเสมอจากจุดฐาน |
| [Offset](../offset/) | สร้างเส้นขนานที่ระยะห่างตั้งฉากคงที่ |
| [Trim](../trim/) | ตัดเส้นที่จุดตัด — **เส้นเท่านั้น** |
| [Extend](../extend/) | ยืดจุดสิ้นสุดที่ใกล้ที่สุดเพื่อถึงขอบเขต — **เส้นเท่านั้น** |
| [Delete](../delete/) | ลบเส้นออกจากภาพวาด |

## คุณสมบัติ

เมื่อเลือกเส้น แผงคุณสมบัติแสดงทุกช่องที่บันทึก DXF `LINE` มี:

**ทั่วไป**

| คุณสมบัติ | ค่าเริ่มต้น | ความหมาย |
|-----------|-----------|----------|
| Color | 256 (ByLayer) | ดัชนีสี ACI |
| Layer | `0` | การกำหนดเลเยอร์ |
| Linetype | ByLayer | รูปแบบเส้นที่มีชื่อ |
| Linetype Scale | 1 | ตัวคูณมาตราส่วนสำหรับรูปแบบเส้น |
| Thickness | 0 | ความหนาของการรีดออก |

**เรขาคณิต**

| คุณสมบัติ | ความหมาย |
|-----------|----------|
| Start X / Start Y | พิกัดจุดสิ้นสุดแรก |
| End X / End Y | พิกัดจุดสิ้นสุดที่สอง |

ทุกช่องสามารถแก้ไขได้โดยตรงในแผงโดยไม่ต้องรันคำสั่งซ้ำ

## Line vs Polyline — เมื่อไหรควรใช้อะไร

| | Line | Polyline |
|---|------|---------|
| จำนวนวัตถุ | หนึ่ง `LINE` ต่อส่วน | หนึ่ง `LWPOLYLINE` สำหรับเส้นทางทั้งหมด |
| Trim / Extend | ใช่ — ทีละส่วน | ไม่ |
| รูปทรงปิด | ไม่ | ใช่ (แฟล็กปิด) |
| การแก้ไขกริป | ยืดจุดสิ้นสุดแต่ละจุด | ย้ายจุดยอดใดก็ได้ตามเส้นทาง |
| เหมาะสำหรับ | เส้นก่อสร้าง ส่วนเดี่ยว เรขาคณิตที่จะตัด | เส้นขอบ โครงร่าง รูปทรงที่เก็บไว้ทั้งหมด |

## DXF — วัตถุ LINE

เส้นถูกบันทึกเป็นวัตถุ `LINE` ในไฟล์ DXF คุณสมบัติทุกอย่าง — พิกัดเริ่มต้น/สิ้นสุด สี เลเยอร์ ประเภทเส้น มาตราส่วนประเภทเส้น และความหนา — รอบเที่ยวโดยไม่สูญเสียข้อมูล เมื่อคุณเปิด DXF ที่มีวัตถุ `LINE` พวกมันจะกลายเป็นวัตถุ `Line` ที่แก้ไขได้ในตัวแก้ไข

เส้นที่วาดในตัวแก้ไขยังถูกเขียนเป็นวัตถุ `LINE` เมื่อบันทึก ดังนั้นจึงอ่านได้โดย LibreCAD, FreeCAD และแอปพลิเคชันที่รองรับ DXF อื่นๆ
