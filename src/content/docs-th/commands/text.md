# Text

คำสั่ง `text` วางป้ายข้อความหลายบรรทัด หลังจากคลิกตำแหน่งบนแคนวาส โปรแกรมแก้ไขป๊อปอัปจะเปิดขึ้นในโหมด **rich** — คุณสามารถพิมพ์เนื้อหา ใช้ตัวหนา/ตัวเอียง/ขีดทับต่อตัวอักษร เปลี่ยนฟอนต์และขนาด และแทรกการขึ้นบรรทัดใหม่ กด **Escape** เพื่อยืนยันและปิดโปรแกรมแก้ไข

ดูหน้า [Text Editor](../../interface/text-editor/) สำหรับข้อมูลอ้างอิงโปรแกรมแก้ไขทั้งหมด รวมถึงการเปรียบเทียบโหมด **rich** และ **simple**

## การวางป้ายข้อความ

1. พิมพ์ `text` ในเทอร์มินัลหรือคลิกปุ่ม **Text** บนแถบเครื่องมือ
2. **คลิกตำแหน่งยึด** บนแคนวาส หรือพิมพ์ `X,Y` แล้วกด **Enter** เพื่อป้อนพิกัดที่แน่นอน
3. **ป๊อปอัปโปรแกรมแก้ไขข้อความ** จะเปิดเหนือป้ายใหม่ พิมพ์เนื้อหาของคุณ
4. กด **Escape** เพื่อยืนยันป้ายและปิดโปรแกรมแก้ไข

ความสูงเริ่มต้นคือ **12 หน่วยการวาด**

## การแก้ไขป้ายที่มีอยู่

**ดับเบิลคลิก** ป้ายข้อความใดก็ได้บนแคนวาสเพื่อเปิดโปรแกรมแก้ไขสำหรับป้ายนั้น

## การป้อนพิกัดสำหรับจุดยึด

แทนการคลิก พิมพ์ตำแหน่งที่แน่นอน:

1. พิมพ์ค่า X
2. กด `,` — เทอร์มินัลแสดง `[X], [Y{cursor}]`
3. พิมพ์ค่า Y
4. กด **Enter** เพื่อวางจุดยึดและเปิดโปรแกรมแก้ไข

## อ้างอิงแป้นพิมพ์

**ระยะจุดยึด**

| ปุ่ม | การกระทำ |
|-----|----------|
| `0`–`9`, `.`, `-` | เริ่มป้อนพิกัด X |
| `,` | ล็อก X และย้ายไปป้อน Y |
| `Backspace` | ลบอักขระที่พิมพ์ล่าสุด |
| `Enter` | ยืนยันพิกัดที่พิมพ์ |

**ระยะโปรแกรมแก้ไขข้อความ** (ดู [Text Editor](../../interface/text-editor/) สำหรับข้อมูลอ้างอิงทั้งหมด)

| ปุ่ม | การกระทำ |
|-----|----------|
| อักขระที่พิมพ์ได้ใดๆ | แทรกที่เคอร์เซอร์ |
| `Backspace` / `Delete` | ลบอักขระที่อยู่ติดกันหรือการเลือก |
| `Enter` | แทรกการขึ้นบรรทัดใหม่ |
| `←` / `→` | ย้าย caret |
| `Home` / `End` | ข้ามไปยังต้น/ปลายของบรรทัดแข็ง |
| `Escape` | ยืนยันและปิดโปรแกรมแก้ไข |

## การแก้ไขกริป — การวางตำแหน่งใหม่

ป้ายข้อความที่เลือกจะแสดงกริปหนึ่งอันที่จุดยึด:

| กริป | ตำแหน่ง | สิ่งที่ทำ |
|------|---------|---------|
| **Anchor** | ล่างซ้ายของข้อความ | ลากเพื่อย้ายตำแหน่งป้าย |

## การเลือกข้อความ

| วิธี | พฤติกรรม |
|------|-----------|
| **คลิก** | เลือกถ้าคลิกลงภายในกล่องขอบเขตของข้อความที่หมุน |
| **ลากไปขวา** (เข้มงวด) | มุมทั้งสี่ของกล่องขอบเขตต้องอยู่ในพื้นที่เลือก |
| **ลากไปซ้าย** (ตัดผ่าน) | การทับซ้อนใดๆ ระหว่างกล่องขอบเขตข้อความและพื้นที่เลือกจะเลือกมัน |

## คำสั่งแก้ไขที่รองรับ

| คำสั่ง | สิ่งที่เกิดขึ้นกับข้อความ |
|--------|----------------------|
| [Move](../move/) | ย้ายจุดยึด |
| [Copy](../copy/) | สร้างป้ายเหมือนกันในตำแหน่งใหม่ |
| [Rotate](../rotate/) | หมุนตำแหน่งจุดยึดและเพิ่มมุมให้กับ Rotation Degree |
| [Mirror](../mirror/) | สะท้อนจุดยึดผ่านแกนกระจก (สตริงข้อความไม่กลับด้าน) |
| [Scale](../scale/) | ปรับสเกลตำแหน่งจุดยึดและคูณความสูงด้วยตัวคูณสเกล |
| [Delete](../delete/) | ลบป้าย |

Text ไม่รองรับ **Offset**, **Trim** หรือ **Extend**

## คุณสมบัติ

เมื่อเลือกป้ายข้อความ แผงคุณสมบัติจะแสดง:

**ทั่วไป**

| คุณสมบัติ | ค่าเริ่มต้น | ความหมาย |
|-----------|------------|---------|
| Color | 256 (ByLayer) | ดัชนีสี ACI |
| Layer | `0` | การกำหนดเลเยอร์ |

**เรขาคณิต**

| คุณสมบัติ | ความหมาย |
|-----------|---------|
| Position X / Position Y | พิกัดจุดยึด |
| Height | ความสูงข้อความพื้นฐานในหน่วยการวาด (ค่าเริ่มต้น: **12**) |
| Rotation Degree | การหมุนทวนเข็มนาฬิกาเป็นองศา |

**คุณสมบัติ**

| คุณสมบัติ | ความหมาย |
|-----------|---------|
| Content | สตริงข้อความ (รหัส MTEXT inline ถูกเก็บไว้) |
| Attachment Point | รหัสการจัดวาง (1 = บนซ้าย … 9 = ล่างขวา) |

Text ไม่มีคุณสมบัติ Linetype, Linetype Scale หรือ Thickness

## DXF — วัตถุ MTEXT

ป้ายข้อความจะถูกเก็บเป็นวัตถุ **MTEXT** ในไฟล์ DXF ตัวหนาและตัวเอียงถูกเข้ารหัสโดยใช้ `\L`, `\K`, `\O` และการสลับฟอนต์ inline (`\f`) ความสูงต่อตัวอักษรถูกเข้ารหัสเป็น `\H` การจัดรูปแบบทั้งหมดจะรอดผ่านการถ่ายโอนข้อมูล DXF และอ่านได้โดย LibreCAD, FreeCAD และแอปพลิเคชันที่เข้ากันได้กับ DXF อื่นๆ
