# Font Manager

คำสั่ง `FontManager` เปิดกล่องโต้ตอบสำหรับเรียกดูและเลือกฟอนต์ และสำหรับอัปโหลดไฟล์ `.ttf` ของคุณเองเพื่อใช้ในเอนทิตี [Text](../text/) และ [Multileader](../leader/)

## การเปิด Font Manager

- พิมพ์ `FontManager` ในเทอร์มินัล **หรือ**
- คลิกปุ่ม **Font Manager** บนแถบเครื่องมือของ [text editor](../../interface/text-editor/)

## กลุ่มฟอนต์

| กลุ่ม | เนื้อหา |
|-------|----------|
| **Default** | ฟอนต์ sans-serif ในตัว — พร้อมใช้งานเสมอ |
| **User** | ฟอนต์ `.ttf` ที่คุณอัปโหลดเอง (จะแสดงก็ต่อเมื่อคุณเพิ่มฟอนต์อย่างน้อยหนึ่งรายการ) |
| **Free** | ฟอนต์ Google Fonts ที่มาพร้อมระบบ 15 แบบ (EB Garamond, Fira Code, Inter, Lato, Merriweather, Montserrat, Nunito, Open Sans, Oswald, Playfair Display, Poppins, Raleway, Roboto, Roboto Condensed, Source Code Pro) |
| **System** | ฟอนต์ทั่วไปของระบบปฏิบัติการ (Courier New, Georgia, Helvetica, Impact, Lucida Console, Tahoma, Times New Roman, Trebuchet MS, Verdana) |

คลิกฟอนต์ใดก็ได้ในรายการเพื่อดูตัวอย่างทางด้านขวา — ชื่อ ตัวอย่างตัวอักษร ประโยคแพนแกรม และตัวเลข

## การอัปโหลดฟอนต์กำหนดเอง

1. คลิก **Add Font** ที่ส่วนท้ายของกล่องโต้ตอบ (หรือพิมพ์ `Font+` ในเทอร์มินัลเพื่อเปิดตัวเลือกไฟล์โดยตรง)
2. เลือกไฟล์ `.ttf` รองรับเฉพาะฟอนต์ TrueType เท่านั้น — ไม่รองรับ `.otf` และ `.woff`/`.woff2`
3. ชื่อไฟล์ (ไม่รวมนามสกุล) จะกลายเป็นชื่อฟอนต์ในกลุ่ม **User** ตัวอย่างเช่น การอัปโหลด `MyFont.ttf` จะเพิ่มฟอนต์ชื่อ `MyFont`

ฟอนต์ที่อัปโหลดจะถูกบันทึกไว้อย่างถาวรในเบราว์เซอร์ (IndexedDB) และโหลดขึ้นมาใหม่โดยอัตโนมัติในครั้งถัดไปที่คุณเปิด KulmanLab CAD

## การลบฟอนต์กำหนดเอง

วางเมาส์เหนือฟอนต์ในกลุ่ม **User** แล้วคลิกปุ่ม **×** ที่อยู่ข้างๆ ฟอนต์ในตัว (Default, Free, System) ไม่สามารถลบได้

## อ้างอิงแป้นพิมพ์

| ปุ่ม | การกระทำ |
|-----|--------|
| `↑` / `↓` | เลื่อนตัวเลือกขึ้นหรือลงในรายการฟอนต์ |
| `Escape` | ปิด Font Manager |

## ความเข้ากันได้กับ DXF

ชื่อฟอนต์จะถูกฝังอยู่ในเอนทิตี **MTEXT** ที่ส่งออก ในรูปแบบโค้ดจัดรูปแบบแบบอินไลน์ ดังนั้น DXF ที่ผ่านการนำเข้า-ส่งออกซ้ำผ่าน KulmanLab CAD จะยังคงการกำหนดฟอนต์ไว้ *ไฟล์* ฟอนต์กำหนดเองจะไม่ถูกฝังอยู่ใน DXF — มีเพียง *ชื่อ* ฟอนต์เท่านั้นที่ถูกฝัง หากคุณนำเข้าภาพวาดที่อ้างอิงถึงฟอนต์กำหนดเองที่คุณยังไม่ได้อัปโหลดบนอุปกรณ์นี้อีกครั้ง ข้อความจะแสดงผลด้วยฟอนต์เริ่มต้นจนกว่าคุณจะอัปโหลดฟอนต์ที่มีชื่อเดียวกันนั้น

## คำสั่งที่เกี่ยวข้อง

- [Text](../text/) — วางป้ายข้อความที่การเลือกฟอนต์นำไปใช้
- [Match Properties](../match-properties/) — คัดลอกความสูงข้อความ แต่ไม่รวมฟอนต์ ระหว่างวัตถุ
