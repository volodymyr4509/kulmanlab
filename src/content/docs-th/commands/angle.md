# Angle

คำสั่ง `angle` วัดมุมภายในที่จุดยอดที่เกิดจากส่วนของเส้นสองส่วนผ่านสามจุดที่คลิก ผลลัพธ์ — เสมออยู่ระหว่าง 0° ถึง 180° — แสดงในเทอร์มินัลถึง 4 ตำแหน่งทศนิยม เป็นหนึ่งในสามคำสั่งวัด — [Distance](../distance/) วัดความยาวเส้นตรง และ [Area](../area/) วัดพื้นที่และเส้นรอบวงของรูปหลายเหลี่ยม

## กายวิภาคของการวัดมุม

```
  ● จุดแรก (ปลายของรัศมีแรก)
   \
    \  ตัวอย่างรัศมีแรก
     \
      ● จุดยอด (ขั้นตอนที่ 3)
     /
    /  ตัวอย่างรัศมีที่สอง (ไปยังเคอร์เซอร์)
   /
  ● จุดที่สาม  →  เทอร์มินัล: "Angle: 45.0000°"
```

- **จุดแรก** — ปลายหนึ่งของมุม (ขั้นตอนที่ 2)
- **จุดยอด** — มุมที่วัดมุม (ขั้นตอนที่ 3)
- **จุดที่สาม** — ปลายอีกด้านของมุม (ขั้นตอนที่ 4)

## การวัดมุม

1. พิมพ์ `angle` ในเทอร์มินัลหรือคลิกปุ่ม **Angle** บนแถบเครื่องมือ
2. **คลิกจุดแรก** — ปลายแขนหนึ่งของมุม หรือพิมพ์ `X,Y` แล้วกด **Enter** เพื่อป้อนพิกัดที่แน่นอน
3. **คลิกจุดยอด** — มุมที่แขนสองแขนพบกัน การป้อนพิกัดก็ใช้ได้ที่นี่เช่นกัน
4. **คลิกจุดที่สาม** — ปลายแขนที่สอง การป้อนพิกัดก็ใช้ได้ที่นี่เช่นกัน การวางจุดนี้จะพิมพ์ผลลัพธ์
5. **คลิกอีกครั้ง** (ไม่บังคับ) เพื่อเริ่มการวัดใหม่จากคลิกนั้นเป็นจุดแรกใหม่

## ข้อตกลงมุมภายใน

คำสั่งคำนวณมุมโดยใช้ผลคูณจุดของรัศมีสองเส้นจากจุดยอด:

- **ภายในเสมอ**: ผลลัพธ์คือมุมที่เล็กกว่า ระหว่าง 0° ถึง 180°
- ลำดับที่คุณคลิกจุดปลายไม่กระทบผลลัพธ์ — เฉพาะตำแหน่งจุดยอดเท่านั้นที่สำคัญ
- จุดร่วมเส้น (สามจุดบนเส้นหนึ่ง) ให้ผลลัพธ์ 0° หรือ 180°

## การวัดแบบต่อเนื่อง

หลังจากผลลัพธ์ปรากฏ การคลิกจะเริ่มการวัดถัดไปทันที — จุดที่คลิกกลายเป็นจุดแรกใหม่ คำสั่งไม่ออกอัตโนมัติจนกว่าคุณจะกด `Escape`

## Angle vs Distance

| | Angle | Distance |
|---|-------|---------|
| สิ่งที่วัด | มุมภายในที่จุดยอด | ความยาวเส้นตรง |
| จำนวนคลิก | 3 | 2 |
| รูปแบบผลลัพธ์ | `45.0000°` | `12.3456` (หน่วย) |
| ตัวอย่างบนแคนวาส | สองเส้นจากจุดยอดถึงปลาย | เส้นจากจุดแรกไปยังเคอร์เซอร์ |
| เหมาะสำหรับ | มุมเปิดระหว่างสองลักษณะ | ความยาวของช่องว่างหรือส่วน |

## การป้อนพิกัด

แทนที่จะคลิก พิมพ์ตำแหน่งที่แน่นอนสำหรับจุดใดก็ได้จากสามจุด:

1. พิมพ์ค่า X
2. กด `,` — เทอร์มินัลแสดง `[X], [Y{cursor}]`
3. พิมพ์ค่า Y
4. กด **Enter** เพื่อยืนยัน

## อ้างอิงแป้นพิมพ์

| ปุ่ม | การกระทำ |
|-----|----------|
| `0`–`9`, `.`, `-` | เริ่มป้อนพิกัด X |
| `,` | ล็อก X และย้ายไปป้อน Y |
| `Backspace` | ลบอักขระที่พิมพ์ล่าสุด |
| `Enter` | ยืนยันพิกัดที่พิมพ์ |
| `Escape` | ยกเลิกและรีเซ็ตไปยังขั้นตอนที่ 2 |

## หมายเหตุ

- ผลลัพธ์แสดงใน **เทอร์มินัลเท่านั้น** — ไม่มีอะไรถูกเพิ่มลงในภาพวาด
- ความแม่นยำเสมอ 4 ตำแหน่งทศนิยมในหน่วยองศา
