# Zoom In

คำสั่ง `zoomin` คูณระดับซูมปัจจุบันด้วย **1.5 เท่า** และออกทันที ศูนย์กลางที่จุดกึ่งกลาง viewport เป็นเทียบเท่าของ scroll wheel หนึ่งขีกซูมไปที่ศูนย์กลางหน้าจอแทนที่จะเป็นเคอร์เซอร์

## การซูมเข้า

คลิกปุ่ม **Zoom In** บนแถบเครื่องมือหรือพิมพ์ `zoomin` ในเทอร์มินัล การซูมจะถูกใช้ทันทีและคำสั่งออก — ไม่ต้องคลิกบนแคนวาส

## วิธีทำงานของขั้นตอน 1.5 เท่า

| ระดับซูมปัจจุบัน | หลัง Zoom In หนึ่งครั้ง |
|----------------|----------------------|
| 1.00× | 1.50× |
| 1.50× | 2.25× |
| 10.00× | 15.00× |
| 6,667× | 10,000× (จำกัด) |

ระดับซูมจะแสดงที่ **มุมล่างขวา** ของแคนวาสถัดจากป้าย `zoom` ขีดจำกัดบนคือ **10,000×** ขั้นตอนเพิ่มเติมจะไม่ทำอะไร

## ปุ่ม Zoom In vs Scroll Wheel

| | ปุ่ม Zoom In | Scroll Wheel |
|---|--------------|-------------|
| ศูนย์กลางซูม | จุดกึ่งกลาง viewport | ตำแหน่งเคอร์เซอร์ |
| ขนาดขั้นตอน | 1.5× ต่อคลิก | ~1.1× ต่อขีก |
| ต้องเปิดใช้งาน | ไม่ | ไม่ — ทำงานตลอดเวลา |
| เหมาะสำหรับ | การนำทางหยาบ | การซูมที่แม่นยำไปยังเคอร์เซอร์ |

## อ้างอิงแป้นพิมพ์

ไม่มีแป้นพิมพ์ลัดสำหรับคำสั่งนี้ ใช้ scroll wheel แทน — ทำงานได้ตลอดเวลาโดยไม่ต้องเปิดใช้งานคำสั่งใดๆ
