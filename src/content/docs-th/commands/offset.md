# Offset

คำสั่ง `offset` สร้างสำเนาขนานของวัตถุที่ระยะตั้งฉากคงที่ คุณพิมพ์ระยะครั้งเดียว แล้วคลิกวัตถุและเลือกด้าน — คำสั่งยังคงพร้อมที่ระยะเดิมเพื่อให้คุณ offset หลายวัตถุในเซสชั่นเดียว

ประเภทวัตถุที่รองรับ: **Line, Circle, Arc, Ellipse, Polyline** (รวมถึง Rectangles)

## การใช้ offset

1. พิมพ์ `offset` ในเทอร์มินัลหรือคลิกปุ่ม **Offset** บนแถบเครื่องมือ
2. **พิมพ์ระยะ offset** แล้วกด **Enter** หรือ **Space**
3. **คลิกวัตถุ** เพื่อ offset — ถ้าวัตถุไม่ใช่ประเภทที่รองรับ ข้อความแสดงข้อผิดพลาดปรากฏและคุณสามารถคลิกวัตถุอื่น
4. **ย้ายเคอร์เซอร์** ไปด้านที่สำเนาควรปรากฏ — ตัวอย่างสดติดตาม
5. **คลิก** เพื่อวางสำเนา offset

หลังจากแต่ละการวาง คำสั่งกลับสู่ขั้นตอนที่ 3 ที่ **ระยะเดิม** พร้อมสำหรับ offset ถัดไป กด **Escape** เพื่อรีเซ็ตกลับไปยังขั้นตอนป้อนระยะ

```
  ระยะ: 10

  ─────────────────    ← เส้นต้นฉบับ
  ─────────────────    ← สำเนา offset (10 หน่วยด้านล่าง)
```

## พฤติกรรม offset ต่อวัตถุ

| วัตถุ | วิธีคำนวณ offset |
|-------|---------------|
| **Line** | เส้นขนานเลื่อนตั้งฉากกับทิศทางต้นฉบับ |
| **Circle** | วงกลมร่วมศูนย์กลาง คลิกนอก → รัศมีใหญ่กว่า ใน → รัศมีเล็กกว่า |
| **Arc** | ส่วนโค้งร่วมศูนย์กลางที่รัศมีใหม่ ช่วงมุมเดิมยังคง |
| **Ellipse** | กึ่งแกนทั้งสองเพิ่มหรือลดด้วยระยะเดิม |
| **Polyline** | แต่ละส่วน offset อิสระ ส่วน offset ที่อยู่ติดกันถูกตัดที่มุม |

สำหรับ **Circle**, **Arc** และ **Ellipse**: ถ้า offset เข้าจะลดรัศมีหรือกึ่งแกนใดๆ ลงเหลือศูนย์หรือต่ำกว่า offset จะไม่ถูกใช้

## อ้างอิงแป้นพิมพ์

| ปุ่ม | การกระทำ |
|-----|----------|
| `0`–`9`, `.` | เพิ่มตัวเลขเข้าไปในค่าระยะ |
| `Backspace` | ลบอักขระที่พิมพ์ล่าสุด |
| `Enter` / `Space` | ยืนยันระยะที่พิมพ์และดำเนินการเลือกวัตถุ |
| `Escape` | รีเซ็ตไปยังขั้นตอนป้อนระยะ |

## หมายเหตุเวิร์กโฟลว์

ระยะคงที่จนกว่าคุณจะกด **Escape** ซึ่งทำให้มีประสิทธิภาพในการ offset วัตถุหลายชิ้นที่ระยะห่างเดิม — พิมพ์ระยะครั้งเดียว แล้วคลิกและเลือกด้านสำหรับแต่ละวัตถุ

## Offset vs Copy

| | Offset | Copy |
|---|--------|------|
| การกระจัด | ตั้งฉากกับเรขาคณิตของวัตถุ | เวกเตอร์ใดก็ได้ (ฐาน → ปลายทาง) |
| วัตถุที่รองรับ | Line, Circle, Arc, Ellipse, Polyline | วัตถุทุกประเภท |
| การป้อนระยะ | พิมพ์ก่อนเลือกวัตถุ | พิมพ์หรือคลิกหลังเลือก |
| เหมาะสำหรับ | เส้นขนาน วงกลมร่วมศูนย์กลาง เส้นทางเข้า/ออก | วางสำเนาในตำแหน่งใดก็ได้ |
