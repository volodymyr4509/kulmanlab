# Leader+

คำสั่ง `leader+` เพิ่มแขนหัวลูกศรใหม่ให้กับ multileader ที่มีอยู่ แขนใหม่ชี้จาก dogleg ที่มีอยู่ของ leader ไปยัง tip หัวลูกศรใหม่ที่คุณคลิก สไตล์ทั้งหมด — ตำแหน่ง dogleg ข้อความ ประเภทหัวลูกศร และขนาด — ถูกรับทอดจาก leader ที่เลือก

## การเพิ่มแขน

1. พิมพ์ `leader+` ในเทอร์มินัล
2. **คลิก multileader ที่มีอยู่** เพื่อเลือก
3. **คลิก tip หัวลูกศรใหม่** หรือพิมพ์ `X,Y` แล้วกด **Enter** เพื่อป้อนพิกัดที่แน่นอน เส้นตัวอย่างแสดงจากเคอร์เซอร์ไปยัง dogleg ของ leader

แขนถูกวางและคำสั่งยังคงทำงาน — คุณสามารถคลิก leader อื่นเพื่อเพิ่มแขนเพิ่มเติมได้ทันที กด **Enter**, **Space** หรือ **Escape** เพื่อเสร็จสิ้น

```
  ก่อน:                        หลัง:
  ◄── แขน 1                      ◄── แขน 1
       \                               \
        ●──── dogleg ──── ข้อความ          ●──── dogleg ──── ข้อความ
                                       /
                                  แขน 2 ──►  (tip ใหม่ที่คุณคลิก)
```

## การป้อนพิกัดสำหรับ tip

แทนที่จะคลิก คุณสามารถพิมพ์ตำแหน่งที่แน่นอน:

1. พิมพ์ค่า X
2. กด `,` — เทอร์มินัลยืนยันว่า X ถูกล็อก
3. พิมพ์ค่า Y
4. กด **Enter** เพื่อวาง

## อ้างอิงแป้นพิมพ์

| ปุ่ม | การกระทำ |
|-----|----------|
| `0`–`9`, `.`, `-` | เริ่มพิมพ์พิกัด X |
| `,` | ล็อก X และย้ายไปป้อน Y |
| `Enter` | ยืนยันพิกัดที่พิมพ์และวางแขน |
| `Enter` / `Space` | เสร็จสิ้น (เมื่อไม่มีการป้อน) |
| `Escape` | ยกเลิกและรีเซ็ต |

## หมายเหตุ

- เฉพาะวัตถุ **Multileader** เท่านั้นที่สามารถเลือกได้ — การคลิกวัตถุประเภทอื่นไม่มีผล
- แขนใหม่มาจาก dogleg ที่มีอยู่ คุณเลือกเฉพาะตำแหน่ง tip หัวลูกศร
- ไม่มีขีดจำกัดจำนวนแขนที่ multileader สามารถมีได้
