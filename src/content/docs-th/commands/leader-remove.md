# Leader−

คำสั่ง `leader-` ลบแขนหัวลูกศรหนึ่งแขนออกจาก multileader ที่มีอยู่ ป้ายข้อความ dogleg และแขนที่เหลือทั้งหมดถูกเก็บไว้ — เฉพาะแขนที่เลือกเท่านั้นที่ถูกลบ multileader ที่มีแขนเพียงแขนเดียวไม่สามารถลบแขนออกได้

## การลบแขน

1. พิมพ์ `leader-` ในเทอร์มินัล
2. **คลิก multileader** ที่มีสองแขนหรือมากกว่า ถ้า leader ที่คลิกมีแขนเพียงแขนเดียว เทอร์มินัลแสดงข้อผิดพลาดและรอการเลือกที่ถูกต้อง
3. **ย้ายเคอร์เซอร์ใกล้แขน** ที่ต้องการลบ — แขนที่ใกล้ที่สุดถูกไฮไลต์ด้วยเครื่องหมาย
4. **คลิก** เพื่อลบแขนนั้น

แขนถูกลบและคำสั่งยังคงทำงาน — คุณสามารถคลิก leader อื่น (หรืออันเดียวกัน) เพื่อลบแขนเพิ่มเติมได้ทันที กด **Enter**, **Space** หรือ **Escape** เพื่อเสร็จสิ้น

```
  ก่อน:                   หลัง:
  ◄── แขน 1                 ◄── แขน 1
       \                          \
        ●──── dogleg ──── ข้อความ     ●──── dogleg ──── ข้อความ
       /
  แขน 2 ──►  ← แขนนี้ถูกลบ
```

## วิธีกำหนดแขนที่ใกล้ที่สุด

คำสั่งวัดระยะห่างตั้งฉากจากเคอร์เซอร์ถึงส่วนของเส้นแต่ละแขน (รวมถึงส่วนจากจุดแขนสุดท้ายถึง dogleg) แขนที่มีระยะห่างน้อยที่สุดถูกไฮไลต์และจะถูกลบเมื่อคลิก

## อ้างอิงแป้นพิมพ์

| ปุ่ม | การกระทำ |
|-----|----------|
| `Enter` / `Space` | เสร็จสิ้นการลบแขน |
| `Escape` | ยกเลิกและรีเซ็ต |

## หมายเหตุ

- leader ที่มี **แขนเพียงแขนเดียว** ได้รับการป้องกัน — คุณต้องเพิ่มแขนก่อนจึงลบแขนได้
- ตำแหน่ง dogleg และเนื้อหาข้อความถูกเก็บไว้เสมอโดยไม่คำนึงว่าลบแขนใด
