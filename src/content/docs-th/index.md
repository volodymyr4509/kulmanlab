# KulmanLab CAD — คู่มืออ้างอิงคำสั่ง

ยินดีต้อนรับสู่คู่มืออ้างอิงคำสั่ง **KulmanLab CAD** [KulmanLab CAD](https://kulmanlab.com) เป็นเครื่องมือ CAD ฟรีที่ทำงานในเบราว์เซอร์สำหรับวาด แก้ไข และส่งออกไฟล์ DXF — ไม่ต้องติดตั้ง ใช้แถบด้านข้างเพื่อเรียกดูคำสั่งทั้งหมดที่จัดกลุ่มตามแผงควบคุม

## รูปทรง

| คำสั่ง | สิ่งที่ทำ |
|---------|-----------|
| [Line](./commands/line/) | วาดเส้นตรงระหว่างสองจุด |
| [Polyline](./commands/polyline/) | วาดเส้นทางหลายส่วนแบบเปิด |
| [Rectangle](./commands/rectangle/) | วาดสี่เหลี่ยมผืนผ้าแนวแกน |
| [Circle](./commands/circle/) | วาดวงกลมด้วยจุดศูนย์กลางและรัศมี |
| [Arc](./commands/arc/) | วาดส่วนโค้งผ่านสามจุด |
| [Ellipse](./commands/ellipse/) | วาดวงรีด้วยจุดศูนย์กลางและสองแกน |
| [Text](./commands/text/) | วางป้ายข้อความบนแคนวาส |

| [Spline Fit](./commands/spline-fit/) | วาดสไปลน์ที่ผ่านจุดที่คลิก |

## การกู้คืน

หากแอปพลิเคชันล้มเหลวเมื่อเริ่มต้น (เช่น หลังจากทำงานกับพิกัดขนาดใหญ่มาก) คุณสามารถล้างข้อมูลที่จัดเก็บในเครื่องทั้งหมดโดยเพิ่ม `?reset` ต่อท้าย URL:

```
https://kulmanlab.com/?resetKulmanLocalStorage
```

การกระทำนี้จะลบทุกอย่างออกจากฐานข้อมูลในเครื่องของเบราว์เซอร์และเริ่มต้นภาพวาดเปล่าใหม่ พารามิเตอร์ `?reset` จะลบตัวเองออกจาก URL โดยอัตโนมัติ ใช้วิธีนี้เป็นทางเลือกสุดท้ายเมื่อ [WipeStorage](./commands/wipestorage/) ไม่สามารถเข้าถึงได้เนื่องจากแอปไม่โหลด

## วิธีการทำงานของคำสั่ง

ทุกคำสั่งทำตามรูปแบบเดียวกัน:

1. **เปิดใช้งาน** — คลิกปุ่มแถบเครื่องมือหรือพิมพ์ชื่อคำสั่งในเทอร์มินัลที่ด้านล่างของหน้าจอ
2. **ทำตามคำแนะนำ** — เทอร์มินัลแสดงสิ่งที่ต้องป้อนถัดไป
3. **เสร็จสิ้นหรือยกเลิก** — คำสั่งส่วนใหญ่เสร็จสิ้นโดยอัตโนมัติหลังจากรับข้อมูลสุดท้าย กด **Escape** ได้ทุกเวลาเพื่อยกเลิก

## การเลือกวัตถุ

คำสั่งแก้ไขหลายรายการ (Move, Copy, Rotate, Mirror, Scale, Delete) มีพฤติกรรมการเลือกเหมือนกัน:

- **คลิก** วัตถุเพื่อเลือกหรือยกเลิกการเลือก
- **ลากไปทางขวา** (ซ้ายไปขวา) เพื่อการเลือกแบบเข้มงวด — เลือกเฉพาะวัตถุที่อยู่ภายในกล่องเท่านั้น
- **ลากไปทางซ้าย** (ขวาไปซ้าย) เพื่อการเลือกแบบตัดผ่าน — เลือกวัตถุใดก็ตามที่ตัดผ่านกล่อง
- กด **Enter** หรือ **Space** เพื่อยืนยันการเลือกและดำเนินการขั้นตอนถัดไป
