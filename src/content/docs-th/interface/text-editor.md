# Text Editor

โปรแกรมแก้ไขข้อความจะเปิดขึ้นทุกครั้งที่คุณวางป้ายข้อความใหม่ด้วยคำสั่ง `text` หรือดับเบิลคลิกวัตถุข้อความที่มีอยู่ รองรับเนื้อหาหลายบรรทัด การจัดรูปแบบต่อตัวอักษร และ word-wrap

## การเปิดโปรแกรมแก้ไข

| การกระทำ | ผลลัพธ์ |
|---------|---------|
| คำสั่ง `text` → คลิกตำแหน่ง | สร้างวัตถุข้อความใหม่และเปิดโปรแกรมแก้ไข |
| ดับเบิลคลิกวัตถุข้อความที่มีอยู่ | เปิดโปรแกรมแก้ไขสำหรับวัตถุนั้นอีกครั้ง |
| `Escape` ภายในโปรแกรมแก้ไข | ปิดโปรแกรมแก้ไขและเก็บการเปลี่ยนแปลงทั้งหมด |

## แถบเครื่องมือ

แถบเครื่องมือลอยอยู่เหนือกล่องขอบเขตของข้อความและยังยึดกับวัตถุขณะเลื่อนหรือซูม

### ตัวหนา · ตัวเอียง · ขีดทับ

| ปุ่ม | ทางลัด | สิ่งที่ทำ |
|------|--------|---------|
| **B** | — | สลับตัวหนา |
| *I* | — | สลับตัวเอียง |
| ~~S~~ | — | สลับขีดทับ |

**วิธีการสลับ:**

- **มีการเลือกข้อความ** — สไตล์จะถูกใช้กับตัวอักษรที่เลือกเท่านั้น
- **ไม่มีการเลือก เคอร์เซอร์ในข้อความที่มีอยู่** — สลับสไตล์บนวัตถุทั้งหมด (ทุกส่วน)
- **ข้อความว่างหรือวัตถุใหม่** — สไตล์ถูกเก็บบนส่วนว่างและใช้กับทุกตัวอักษรที่พิมพ์จากจุดนั้นเป็นต้นไป

ปุ่มจะถูกไฮไลต์ (ใช้งาน) เมื่อทุกตัวอักษรในการเลือกปัจจุบัน — หรือตัวอักษรทางซ้ายของเคอร์เซอร์ — มีสไตล์นั้นตั้งไว้

### ฟอนต์

ดรอปดาวน์แสดงชุดตัวอักษรข้ามแพลตฟอร์มที่คัดสรรไว้ (Helvetica, Times New Roman, Georgia, Courier New, Verdana, Tahoma, Trebuchet MS, Lucida Console, Impact) ตัวเลือกบนสุด **Arial** เป็นค่าเริ่มต้น

- **มีการเลือก** — แทนที่ฟอนต์สำหรับตัวอักษรที่เลือกเท่านั้น
- **ไม่มีการเลือก** — ใช้ฟอนต์กับวัตถุทั้งหมด

ดรอปดาวน์สะท้อนฟอนต์ของตัวอักษรทางซ้ายของเคอร์เซอร์เมื่อไม่มีการเลือก

### ความสูง

ช่องตัวเลขตั้ง **ความสูงตัวพิมพ์ใหญ่** (ความสูงของอักษรตัวใหญ่) ในหน่วยการวาด

- **มีการเลือก** — แทนที่ความสูงสำหรับตัวอักษรที่เลือก โดยไม่คำนึงถึงความสูงพื้นฐานของวัตถุ
- **ไม่มีการเลือก** — เปลี่ยนความสูงพื้นฐานของวัตถุ (ใช้กับตัวอักษรทั้งหมดที่ไม่มีการแทนที่ความสูงเฉพาะ)

ช่องสะท้อนความสูงของตัวอักษรทางซ้ายของเคอร์เซอร์ ปล่อยว่างเพื่อใช้ค่าเริ่มต้นของวัตถุ

## เคอร์เซอร์และการนำทาง

| ปุ่ม | การกระทำ |
|-----|----------|
| `←` / `→` | ย้าย caret หนึ่งตัวอักษรไปซ้ายหรือขวา |
| `Home` | ข้ามไปยังต้นของบรรทัดแข็งปัจจุบัน |
| `End` | ข้ามไปยังปลายของบรรทัดแข็งปัจจุบัน |
| `Shift` + `←` / `→` | ขยายหรือย่อการเลือก |
| `Backspace` | ลบตัวอักษรทางซ้าย (หรือการเลือก) |
| `Delete` | ลบตัวอักษรทางขวา (หรือการเลือก) |
| `Enter` | แทรกการขึ้นบรรทัดใหม่ |
| `Escape` | ปิดโปรแกรมแก้ไข |

ความสูงเคอร์เซอร์จะตรงกับความสูงตัวพิมพ์ใหญ่ของตัวอักษรที่อยู่ติดกันโดยอัตโนมัติ รวมถึงขนาดที่เล็กกว่าที่ใช้สำหรับ subscript และ superscript

## Word Wrap

เมื่อวัตถุข้อความมี **ความกว้างอ้างอิง** ตั้งไว้ บรรทัดที่ยาวจะถูก soft-wrap ที่ขอบคำเพื่อให้พอดีภายในความกว้างนั้น

หากต้องการตั้งหรือเปลี่ยนความกว้างอ้างอิงขณะเลือกวัตถุ ลาก **กริปปรับขนาด** — สี่เหลี่ยมบางๆ ที่ขอบซ้ายและขวาของกล่องขอบเขตประ เนื้อหาจะ reflow แบบเรียลไทม์ขณะลาก

การตั้งความกว้างอ้างอิงเป็นศูนย์ (ลากกริปชนกันหรือลบค่าในแผงคุณสมบัติ) จะลบ word wrap และให้บรรทัดขยายได้อย่างอิสระ

## ข้อความหลายบรรทัด

กด `Enter` เพื่อแทรกการขึ้นบรรทัดแข็ง แต่ละบรรทัดแข็งเป็นอิสระ — `Home` และ `End` นำทางภายในบรรทัดแข็งปัจจุบันเท่านั้น

การขึ้นบรรทัดแข็งและการจัดรูปแบบต่อตัวอักษรจะถูกเก็บโดยใช้รูปแบบ MTEXT และรอดผ่านการถ่ายโอนข้อมูล DXF เต็มรูปแบบ

## ความเข้ากันได้กับ DXF

วัตถุข้อความจะถูกเก็บเป็น **MTEXT** ในไฟล์ DXF ตัวหนาและตัวเอียงถูกเข้ารหัสเป็น `\L`, `\K`, `\O` และการสลับฟอนต์ inline (`\f`) ความสูงต่อตัวอักษรถูกเข้ารหัสเป็น `\H` การจัดรูปแบบทั้งหมดจะถูกเก็บรักษาเมื่อส่งออกและอ่านได้โดย LibreCAD, FreeCAD และแอปพลิเคชันที่เข้ากันได้กับ DXF อื่นๆ
