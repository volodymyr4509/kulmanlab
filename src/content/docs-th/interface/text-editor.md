# Text Editor

โปรแกรมแก้ไขข้อความจะเปิดขึ้นเมื่อคุณวางหรือดับเบิลคลิก entity ที่แก้ไขได้ **chip โหมด** ขนาดเล็กในส่วนหัว — **rich** (สีเน้น) หรือ **simple** (สีหม่น) — แสดงโหมดที่ใช้งานสำหรับ entity ปัจจุบัน

## โหมดของโปรแกรมแก้ไข

### Rich mode

ใช้โดย: **Text** (ป้ายข้อความ MTEXT) และคำอธิบาย **Multileader**

| ฟีเจอร์ | การทำงาน |
|---------|----------|
| Bold / Italic / Strikethrough | ต่อตัวอักษร (ใช้กับส่วนที่เลือก หรือ entity ทั้งหมดถ้าไม่มีการเลือก) |
| Font และ Height | การแทนที่ต่อตัวอักษร หรือค่าเริ่มต้นทั้ง entity |
| `Enter` | แทรกการขึ้นบรรทัดแบบ hard |
| `Shift+←/→` | ขยายหรือย่อการเลือกข้อความ |
| `Home` / `End` | ข้ามไปต้น/ปลายบรรทัดแข็งปัจจุบัน |
| Word wrap | รองรับผ่าน grip ปรับขนาดความกว้างอ้างอิง |

### Simple mode

ใช้โดย: **Dimension Linear**, **Dimension Aligned**, **Dimension Angular**, **Dimension Radius**, **Dimension Diameter**

โปรแกรมแก้ไขถูกเติมป้ายที่แสดงปัจจุบันของ dimension ล่วงหน้าเพื่อให้คุณวางเคอร์เซอร์และแก้ไขค่าได้โดยตรง

| ฟีเจอร์ | การทำงาน |
|---------|----------|
| Bold / Italic / Font / Height | มีให้ใช้ — ใช้กับ**ป้ายทั้งหมด**พร้อมกัน |
| การจัดรูปแบบต่อตัวอักษร | ไม่รองรับ |
| `Enter` | **ยืนยัน**ค่าและปิดโปรแกรมแก้ไข (ไม่มีการขึ้นบรรทัด) |
| หลายบรรทัด | ไม่รองรับ |
| Word wrap | ไม่รองรับ |

## การเปิดโปรแกรมแก้ไข

| การกระทำ | ผลลัพธ์ |
|---------|---------|
| คำสั่ง `text` → คลิกตำแหน่ง | สร้าง entity ข้อความใหม่และเปิดโปรแกรมแก้ไข (**rich**) |
| ดับเบิลคลิก entity **Text** ที่มีอยู่ | เปิดโปรแกรมแก้ไขในโหมด **rich** อีกครั้ง |
| ดับเบิลคลิก **Multileader** ที่มีอยู่ | เปิดโปรแกรมแก้ไขในโหมด **rich** |
| ดับเบิลคลิก entity **dimension** | เปิดโปรแกรมแก้ไขในโหมด **simple** |
| `Escape` ภายในโปรแกรมแก้ไข | ปิดโปรแกรมแก้ไขและเก็บการเปลี่ยนแปลงทั้งหมด |

## แถบเครื่องมือ

แถบเครื่องมือลอยอยู่เหนือกล่องขอบเขตของข้อความและยังยึดกับวัตถุขณะเลื่อนหรือซูม

### ตัวหนา · ตัวเอียง · ขีดทับ

| ปุ่ม | ทางลัด | สิ่งที่ทำ |
|------|--------|---------|
| **B** | — | สลับตัวหนา |
| *I* | — | สลับตัวเอียง |
| ~~S~~ | — | สลับขีดทับ |

**วิธีการสลับ:**

- **มีการเลือกข้อความ** — สไตล์จะถูกใช้กับตัวอักษรที่เลือกเท่านั้น
- **ไม่มีการเลือก เคอร์เซอร์ในข้อความที่มีอยู่** — สลับสไตล์บนวัตถุทั้งหมด (ทุกส่วน)
- **ข้อความว่างหรือวัตถุใหม่** — สไตล์ถูกเก็บบนส่วนว่างและใช้กับทุกตัวอักษรที่พิมพ์จากจุดนั้นเป็นต้นไป

ปุ่มจะถูกไฮไลต์ (ใช้งาน) เมื่อทุกตัวอักษรในการเลือกปัจจุบัน — หรือตัวอักษรทางซ้ายของเคอร์เซอร์ — มีสไตล์นั้นตั้งไว้

### ฟอนต์

ดรอปดาวน์แบ่งฟอนต์ที่มีให้ใช้งานออกเป็นกลุ่ม **Default** (ฟอนต์ sans-serif ในตัว), **User** (ฟอนต์ที่คุณอัปโหลดเอง ถ้ามี), **Free** (ชุดฟอนต์ Google Fonts ที่มาพร้อมระบบ) และ **System** (ฟอนต์ทั่วไปของระบบปฏิบัติการ เช่น Helvetica, Times New Roman, Georgia, Courier New, Verdana, Tahoma, Trebuchet MS, Lucida Console และ Impact)

- **มีการเลือก** — แทนที่ฟอนต์สำหรับตัวอักษรที่เลือกเท่านั้น
- **ไม่มีการเลือก** — ใช้ฟอนต์กับวัตถุทั้งหมด

ไม่ได้จำกัดอยู่แค่รายการในตัว — คลิกปุ่ม **Font Manager** บนแถบเครื่องมือเพื่ออัปโหลดไฟล์ `.ttf` ของคุณเองและเพิ่มเข้าไปในกลุ่ม **User** ดูรายละเอียดที่ [Font Manager](../../commands/font-manager/)

### ความสูง

ช่องตัวเลขตั้ง **ความสูงตัวพิมพ์ใหญ่** (ความสูงของอักษรตัวใหญ่) ในหน่วยการวาด

- **มีการเลือก** — แทนที่ความสูงสำหรับตัวอักษรที่เลือก โดยไม่คำนึงถึงความสูงพื้นฐานของวัตถุ
- **ไม่มีการเลือก** — เปลี่ยนความสูงพื้นฐานของวัตถุ (ใช้กับตัวอักษรทั้งหมดที่ไม่มีการแทนที่ความสูงเฉพาะ)

## เคอร์เซอร์และการนำทาง

| ปุ่ม | การกระทำ |
|-----|----------|
| `←` / `→` | ย้าย caret หนึ่งตัวอักษรไปซ้ายหรือขวา |
| `Home` | ข้ามไปยังต้นของบรรทัดแข็งปัจจุบัน |
| `End` | ข้ามไปยังปลายของบรรทัดแข็งปัจจุบัน |
| `Shift` + `←` / `→` | ขยายหรือย่อการเลือก |
| `Backspace` | ลบตัวอักษรทางซ้าย (หรือการเลือก) |
| `Delete` | ลบตัวอักษรทางขวา (หรือการเลือก) |
| `Enter` | แทรกการขึ้นบรรทัดใหม่ |
| `Escape` | ปิดโปรแกรมแก้ไข |

ความสูงเคอร์เซอร์จะตรงกับความสูงตัวพิมพ์ใหญ่ของตัวอักษรที่อยู่ติดกันโดยอัตโนมัติ รวมถึงขนาดที่เล็กกว่าที่ใช้สำหรับ subscript และ superscript

## คัดลอก ตัด และวาง

| ปุ่ม | การกระทำ |
|-----|----------|
| `Ctrl+C` / `Cmd+C` | คัดลอกข้อความที่เลือก |
| `Ctrl+X` / `Cmd+X` | ตัดข้อความที่เลือก |
| `Ctrl+V` / `Cmd+V` | วางที่ตำแหน่งเคอร์เซอร์ |

การคัดลอกและตัดต้องมีการเลือกข้อความที่ใช้งานอยู่ ข้อความที่วางจะเป็นข้อความธรรมดาเสมอ — โดยจะรับรูปแบบ (ตัวหนา ตัวเอียง ฟอนต์ ความสูง) ที่มีอยู่แล้วที่ตำแหน่งเคอร์เซอร์ แทนที่จะคงรูปแบบที่มีตอนคัดลอก

ใน **Rich mode** การขึ้นบรรทัดใหม่ในข้อความที่วางจะถูกเก็บไว้ ใน **Simple mode** การขึ้นบรรทัดใหม่จะถูกลบออก เนื่องจากป้ายมิติเป็นแบบบรรทัดเดียว

## Word Wrap

เมื่อวัตถุข้อความมี **ความกว้างอ้างอิง** ตั้งไว้ บรรทัดที่ยาวจะถูก soft-wrap ที่ขอบคำเพื่อให้พอดีภายในความกว้างนั้น

หากต้องการตั้งหรือเปลี่ยนความกว้างอ้างอิงขณะเลือกวัตถุ ลาก **grip ปรับขนาด** — สี่เหลี่ยมบางๆ ที่ขอบซ้ายและขวาของกล่องขอบเขตประ เนื้อหาจะ reflow แบบเรียลไทม์ขณะลาก

การตั้งความกว้างอ้างอิงเป็นศูนย์ (ลากกริปชนกันหรือลบค่าในแผงคุณสมบัติ) จะลบ word wrap และให้บรรทัดขยายได้อย่างอิสระ

## ข้อความหลายบรรทัด

กด `Enter` เพื่อแทรกการขึ้นบรรทัดแข็ง แต่ละบรรทัดแข็งเป็นอิสระ — `Home` และ `End` นำทางภายในบรรทัดแข็งปัจจุบันเท่านั้น

การขึ้นบรรทัดแข็งและการจัดรูปแบบต่อตัวอักษรจะถูกเก็บโดยใช้รูปแบบ MTEXT และรอดผ่านการถ่ายโอนข้อมูล DXF เต็มรูปแบบ

## ความเข้ากันได้กับ DXF

วัตถุข้อความจะถูกเก็บเป็น **MTEXT** ในไฟล์ DXF ตัวหนาและตัวเอียงถูกเข้ารหัสเป็น `\L`, `\K`, `\O` และการสลับฟอนต์ inline (`\f`) ความสูงต่อตัวอักษรถูกเข้ารหัสเป็น `\H` การจัดรูปแบบทั้งหมดจะถูกเก็บรักษาเมื่อส่งออกและอ่านได้โดย LibreCAD, FreeCAD และแอปพลิเคชันที่เข้ากันได้กับ DXF อื่นๆ
