# Grid & Snap

ปุ่มสลับสองปุ่มในแถบควบคุมให้คุณวางกริดอ้างอิงและล็อกเคอร์เซอร์ไปยังจุดตัดกันขณะวาด

| ปุ่ม | สิ่งที่ทำ |
|------|---------|
| **Grid** | แสดงกริดจุดหรือเส้นบนแคนวาส |
| **Snap** | ล็อกเคอร์เซอร์ไปยังจุดกริดที่ใกล้ที่สุดเมื่อไม่มีสแนปเรขาคณิตที่ใกล้กว่า |

ปุ่มสลับทั้งสองอิสระกัน — คุณสามารถแสดงกริดโดยไม่สแนป สแนปโดยไม่แสดงกริด หรือใช้ทั้งสองพร้อมกัน

## การเปิดใช้งาน Grid และ Snap

คลิก **Grid** หรือ **Snap** ในแถบเครื่องมือควบคุม สถานะที่ใช้งานอยู่จะถูกไฮไลต์ การตั้งค่าจะถูกบันทึกระหว่างเซสชัน

เมื่อเปิดใช้ **Snap** กริดจะสลับการแสดงจากเส้นเป็น **จุด** โดยอัตโนมัติ — จุดเหล่านั้นทำเครื่องหมายจุดที่แน่นอนที่เคอร์เซอร์จะสแนปไป

## ระยะห่างกริดแบบปรับตัว

ระยะห่างกริดปรับตัวโดยอัตโนมัติขณะซูมเพื่อให้เส้นกริดอยู่ห่างกันในระยะที่สบายบนหน้าจอ (~40 px) ขั้นตอนเสมอเป็นตัวเลข "กลม" — เป็นทวีคูณของ 1, 2 หรือ 5 ที่กำลังใดก็ได้ของสิบ:

| ตัวอย่างซูม / สเกลโมเดล | ขั้นตอนกริด |
|------------------------|------------|
| ซูมออก (พื้นที่กว้าง) | 100, 500, 1000 … |
| ซูมปานกลาง | 10, 20, 50 … |
| ซูมเข้า (รายละเอียดละเอียด) | 1, 2, 5 … |
| ใกล้มาก | 0.1, 0.2, 0.5 … |

หมายความว่าทุกจุดสแนปจะอยู่บนพิกัดกลมในโมเดลสเปซ — ไม่มีออฟเซ็ตทศนิยมสะสม

## ลำดับความสำคัญของสแนป

**สแนปจุดสิ้นสุดและจุดตัดมีความสำคัญสูงกว่ากริดเสมอ** เคอร์เซอร์จะสแนปกับจุดกริดเฉพาะเมื่อไม่อยู่ใกล้ตัวเลือกสแนปเรขาคณิต (จุดสิ้นสุด จุดกึ่งกลาง ศูนย์กลาง หรือจุดตัด)

หมายความว่าคุณสามารถวาดโดยเปิดสแนปกับกริดและยังสแนปได้อย่างแม่นยำกับเรขาคณิตที่มีอยู่เมื่อเคอร์เซอร์เข้าใกล้พอ กริดเป็นทางเลือกสำรอง ไม่ใช่การแทนที่

## โหมดเลย์เอาต์

- **โมเดลสเปซ** — จุดหรือเส้นเติมพื้นที่แคนวาสที่มองเห็นได้ทั้งหมด
- **พื้นที่เลย์เอาต์ (กระดาษ)** — จุดถูกตัดที่สี่เหลี่ยมกระดาษและไม่ขยายออกนอก
- **ภายใน viewport** — กริดตามระบบพิกัดโมเดลที่สเกลของ viewport ดังนั้นจุดจึงตรงกับหน่วยโมเดลเดิมโดยไม่คำนึงถึงการขยายของ viewport

## ขั้นตอนการทำงานทั่วไป

1. เปิด **Grid** และ **Snap** ก่อนเริ่มภาพวาดที่ต้องการระยะห่างปกติ
2. ซูมไปยังระดับที่ขั้นตอนกริดตรงกับส่วนเพิ่มที่ต้องการ (เช่น ซูมจนจุดห่างกัน 10 หน่วย)
3. วาด — เคอร์เซอร์จะสแนปกับจุดกริดโดยอัตโนมัติ เรขาคณิตที่มีอยู่ยังสแนปตามปกติเมื่อคุณอยู่ใกล้
4. ปิด **Snap** เมื่อต้องการเคลื่อนเคอร์เซอร์อิสระหรือต้องการสแนปกับเรขาคณิตเท่านั้น
