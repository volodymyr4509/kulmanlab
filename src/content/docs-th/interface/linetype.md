# Linetype

**ชิป linetype** ในแถบเครื่องมือควบคุมรูปแบบเส้นประที่กำหนดให้กับวัตถุใหม่ทุกชิ้นที่คุณวาด คลิกเพื่อเปิดดรอปดาวน์ตัวเลือก

## ตัวเลือก

| ค่า | ความหมาย |
|-----|---------|
| **From Layer** | วัตถุสืบทอด linetype ที่กำหนดบนเลเยอร์ของมัน แสดงเป็น `ByLayer` ใน DXF |
| **ByBlock** | วัตถุสืบทอด linetype ของบล็อกที่มันอยู่ ไม่มีผลที่มองเห็นได้นอกบล็อก |
| **Continuous** | เส้นต่อเนื่องไม่ขาด — ไม่มีรูปแบบเส้นประ |
| **Named linetypes** | Linetype ใดๆ ที่โหลดจากไฟล์ DXF ปัจจุบัน (เช่น `DASHED`, `CENTER`, `HIDDEN`, `PHANTOM`, …) ดรอปดาวน์แสดงตัวอย่างสดของแต่ละรูปแบบและสตริงนิยาม |

## วิธีใช้งาน

linetype ที่เลือกจะถูกใช้กับวัตถุทุกชิ้นที่สร้างหลังจากการเปลี่ยนแปลง ไม่มีผลย้อนหลังกับวัตถุที่มีอยู่

หากต้องการเปลี่ยน linetype ของวัตถุที่มีอยู่ เลือกพวกมันและแก้ไขช่อง **Linetype** ในแผงคุณสมบัติ หรือใช้ [MatchProperties](../../commands/match-properties/) เพื่อคัดลอกจากวัตถุอื่น

## สเกล Linetype

วัตถุแต่ละชิ้นยังมีคุณสมบัติ **Linetype Scale** (ค่าเริ่มต้น `1`) รูปแบบเส้นประจะถูกคูณด้วยตัวคูณนี้ ค่า `2` ทำให้เส้นประยาวเป็นสองเท่า `0.5` ทำให้สั้นครึ่งหนึ่ง แก้ไขในแผงคุณสมบัติหลังจากเลือกวัตถุ

## Linetypes ที่พร้อมใช้

ดรอปดาวน์แสดงเฉพาะ linetypes ที่มีอยู่ในไฟล์ DXF ที่โหลดปัจจุบัน ไฟล์ที่สร้างใหม่มีเฉพาะ `ByLayer`, `ByBlock` และ `Continuous` เมื่อนำเข้า DXF linetypes ทั้งหมดที่กำหนดในตาราง `$LTYPE` ของไฟล์จะพร้อมใช้

หากต้องการ linetype เฉพาะ (เช่น `DASHED2`) ที่ไม่อยู่ในรายการ ให้นำเข้าไฟล์ DXF ที่มีมัน — linetype จะปรากฏในตัวเลือกสำหรับเซสชันปัจจุบัน

## ความเข้ากันได้กับ DXF

ชื่อ linetype จะถูกเก็บเป็นสตริงในระเบียนวัตถุ `ByLayer` และ `ByBlock` เป็นค่า sentinel DXF มาตรฐาน linetypes ที่มีชื่อทั้งหมดและรูปแบบเส้นประจะถูกเก็บรักษาอย่างแน่นอนเมื่อส่งออกและถ่ายโอนข้อมูลไปกลับโดยไม่สูญหายใน LibreCAD, FreeCAD และแอปพลิเคชันที่เข้ากันได้กับ DXF อื่นๆ
