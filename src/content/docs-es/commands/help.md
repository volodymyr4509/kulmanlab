# Help

Abre el panel de ayuda integrado con un resumen de todos los atajos de teclado, consejos de navegación y uso general.

## Cómo usar

Haz clic en el botón **Help** de la barra de herramientas o escribe `help` en el terminal. El panel de ayuda se abre inmediatamente y el comando termina.

## Qué hay en el panel de ayuda

### Navegación
| Atajo | Acción |
|-------|--------|
| `Scroll wheel` | Acercar / alejar |
| `Middle click + drag` | Desplazar la vista |
| `Double middle click` | Ajustar todas las entidades a la pantalla |

### Comandos
| Atajo | Acción |
|-------|--------|
| Escribir | Empieza a escribir un nombre de comando — las sugerencias aparecen en el lienzo |
| `Delete` | Eliminar todas las entidades seleccionadas |
| `Enter` / `Space` | Confirmar comando o entrada |
| `Enter` / `Space` (inactivo) | Repetir el último comando usado |
| `Escape` | Cancelar el comando actual o borrar la selección |

### Selección
| Atajo | Acción |
|-------|--------|
| `Left click` | Seleccionar una entidad |
| `Drag →` | Seleccionar solo entidades **completamente dentro** del área de arrastre |
| `Drag ←` | Seleccionar cualquier entidad que **toque** el área de arrastre |
| `Left click grip` | Editar un vértice o extremo directamente |

### Consejos de dibujo
- **Puntos de ajuste** — el cursor se ajusta automáticamente a los extremos cercanos de las entidades.
- **Ángulo fijo** — mueve el cursor lo suficientemente lejos en una dirección para bloquearlo a incrementos de 45°.
- **Longitud exacta** — mientras el ángulo está bloqueado, escribe un número y presiona `Enter` para establecer una distancia exacta.

## Versión

La versión actual de la aplicación se muestra en la esquina superior izquierda del panel de ayuda (p. ej. `KULMAN 2026.04.15`).
