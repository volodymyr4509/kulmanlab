# Help

Otwiera wbudowany panel pomocy z podsumowaniem wszystkich skrótów klawiaturowych, wskazówek nawigacyjnych i ogólnego użytkowania.

## Jak używać

Kliknij przycisk **Help** na pasku narzędzi lub wpisz `help` w terminalu. Panel pomocy otwiera się natychmiast i polecenie kończy się.

## Co zawiera panel pomocy

### Nawigacja
| Skrót | Akcja |
|-------|-------|
| `Kółko myszy` | Powiększ / Pomniejsz |
| `Środkowy przycisk + przeciągnij` | Panoramuj widok |
| `Dwukrotne kliknięcie środkowym przyciskiem` | Dopasuj wszystkie elementy do ekranu |

### Polecenia
| Skrót | Akcja |
|-------|-------|
| Pisanie | Zacznij wpisywać nazwę polecenia — podpowiedzi pojawiają się na płótnie |
| `Delete` | Usuń wszystkie zaznaczone elementy |
| `Enter` / `Spacja` | Potwierdź polecenie lub wejście |
| `Enter` / `Spacja` (w trybie bezczynności) | Powtórz ostatnio używane polecenie |
| `Escape` | Anuluj bieżące polecenie lub wyczyść zaznaczenie |

### Zaznaczanie
| Skrót | Akcja |
|-------|-------|
| `Lewy przycisk myszy` | Zaznacz element |
| `Przeciągnij →` | Zaznacz tylko elementy **w całości wewnątrz** obszaru przeciągania |
| `Przeciągnij ←` | Zaznacz dowolny element **dotykający** obszaru przeciągania |
| `Lewy przycisk myszy na uchwycie` | Edytuj wierzchołek lub punkt końcowy bezpośrednio |

### Wskazówki rysunkowe
- **Punkty przyciągania** — kursor automatycznie przyciąga się do pobliskich punktów końcowych elementów.
- **Sticky kąt** — przesuń kursor wystarczająco daleko w jednym kierunku, aby zablokować do przyrostów 45°.
- **Dokładna długość** — podczas zablokowania kąta wpisz liczbę i naciśnij `Enter`, aby ustawić dokładną odległość.
