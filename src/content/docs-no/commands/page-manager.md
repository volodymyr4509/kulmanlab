# PageManager

Kommandoen `PageManager` åpner sideinnstillingsdialogen for det aktive layoutet, slik at du kan endre papirformatet, orienteringen og skalaen mellom tegneenheter og millimeter.

## Åpne Page Manager

- Skriv `PageManager` i terminalen, **eller**
- Høyreklikk en layout-fane nederst og velg **Page Manager**.

## Innstillinger

| Innstilling | Beskrivelse |
|---------|-------------|
| **Papirformat** | Standard arkstørrelse (A4, A3, A2, Letter osv.) |
| **Orientering** | Stående eller liggende |
| **Skala** | Tegneenheter per mm — styrer hvordan modellkoordinater tilordnes papiret |

Å endre papirstørrelsen eller skalaen skalerer alle eksisterende viewporter i layoutet proporsjonalt slik at posisjonene deres forblir korrekte.

## Merknader

- PageManager gjelder kun papirlayouter. Den er ikke tilgjengelig i modellrommet.
- For å legge til et nytt layout, klikk **+**-knappen i fanelinjen nederst.
- For å gi nytt navn til eller slette et layout, høyreklikk fanen.
