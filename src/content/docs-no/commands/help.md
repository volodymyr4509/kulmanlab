# Help

Åpner hjelpepanelet i appen med et sammendrag av alle tastatursnarveier, navigasjonstips og generell bruk.

## Slik bruker du den

Klikk på **Help**-knappen i verktøylinjen eller skriv `help` i terminalen. Hjelpepanelet åpnes umiddelbart og kommandoen avsluttes.

## Hva som er i hjelpepanelet

### Navigasjon
| Snarvei | Handling |
|----------|--------|
| `Rullehjul` | Zoom inn / ut |
| `Midtklikk + dra` | Panorer visningen |
| `Dobbelt midtklikk` | Tilpass alle entiteter til skjermen |

### Kommandoer
| Snarvei | Handling |
|----------|--------|
| Skriv | Begynn å skrive et kommandonavn — forslag vises på lerretet |
| `Delete` | Slett alle markerte entiteter |
| `Enter` / `Space` | Bekreft kommando eller inntasting |
| `Enter` / `Space` (i ro) | Gjenta sist brukte kommando |
| `Escape` | Avbryt gjeldende kommando eller fjern markering |

### Markering
| Snarvei | Handling |
|----------|--------|
| `Venstreklikk` | Marker en entitet |
| `Dra →` | Marker kun entiteter **helt inni** dra-området |
| `Dra ←` | Marker enhver entitet som **berører** dra-området |
| `Venstreklikk grep` | Rediger et hjørne eller endepunkt direkte |

### Tegnetips
- **Snap-punkter** — markøren fester seg automatisk til nærliggende entitetsendepunkter.
- **Klebrig vinkel** — flytt markøren langt nok i én retning for å låse til 45°-inkrementer.
- **Eksakt lengde** — mens vinkelen er låst, skriv et tall og trykk `Enter` for å sette en eksakt avstand.
