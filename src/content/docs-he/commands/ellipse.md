# Ellipse

פקודת `ellipse` משרטטת אליפסה באמצעות שלוש לחיצות: נקודת מרכז, נקודת הקצה של חצי-הציר הראשון (הגדול) בכל זווית, ואורך חצי-הציר השני (הקטן). שני הצירים תמיד ניצבים זה לזה — כיוון הציר השני נגזר אוטומטית מהראשון.

## שרטוט אליפסה

1. הקלידו `ellipse` במסוף או לחצו על כפתור סרגל הכלים **Ellipse**.
2. **לחצו על נקודת המרכז**, או הקלידו `X,Y` ולחצו **Enter** לקואורדינטה מדויקת.
3. **לחצו על נקודת הקצה של הציר הראשון** — קובעת גם את הכיוון וגם את האורך של חצי-הציר הראשון. הזנת קואורדינטות עובדת גם כאן.
4. **הגדירו את אורך הציר השני** — הזיזו את הסמן ניצב לציר הראשון, ואז לחצו או הקלידו אורך.

```
               ● ← נקודת קצה הציר הראשון (שלב 3)
              /
  מרכז ●  /  ← ציר ראשון (כל זווית)
            |
            ● ← הסמן כאן קובע את אורך הציר השני (שלב 4)
```

האליפסה ממוקמת לאחר שלב 4 והפקודה יוצאת.

## הזנת ציר — לחיצה, קואורדינטה או אורך מוקלד

**מרכז (שלב 2):** לחצו או הקלידו `X,Y` למיקום מדויק.

**נקודת קצה הציר הראשון (שלב 3):** לחצו, או הקלידו `X,Y` לקואורדינטה מדויקת. נעילת זווית גם היא נצמדת למרווחים של 45° — בעת נעילה, הקלידו אורך ולחצו **Enter** למיקום נקודת הקצה במרחק המדויק הזה.

**ציר שני (שלב 4):** אורך מוקלד תמיד זמין — אין צורך בנעילת זווית. הכיוון כבר קבוע ניצב לציר הראשון; ההקלדה קובעת רק את האורך.

| מקש | פעולה |
|-----|--------|
| `0`–`9`, `.` | הוספת ספרה לאורך הציר (שלב הציר השני) |
| `Backspace` | מחיקת התו האחרון שהוקלד |
| `Enter` | מיקום נקודת קצה הציר באורך שהוקלד |

## הזנת קואורדינטות (מרכז ונקודת קצה הציר הראשון)

1. הקלידו את ערך X.
2. לחצו `,` — המסוף מציג `[X], [Y{cursor}]`.
3. הקלידו את ערך Y.
4. לחצו **Enter** לאישור.

## מרגע לוח המקשים

| מקש | פעולה |
|-----|--------|
| `0`–`9`, `.`, `-` | התחלת הזנת קואורדינטת X (שלבי מרכז/ציר-ראשון), או אורך ציר בעת נעילת זווית |
| `,` | נעילת X ומעבר להזנת Y |
| `Backspace` | מחיקת התו האחרון שהוקלד |
| `Enter` | אישור קואורדינטה או אורך שהוקלדו |
| `Escape` | ביטול ואיפוס |

## עריכת נקודות אחיזה — שינוי גודל צירים עצמאי

אליפסה נבחרת חושפת חמש נקודות אחיזה:

| נקודת אחיזה | כמות | מה היא עושה |
|------|-------|--------------|
| **מרכז** | 1 | מזיזה את כל האליפסה; שני הצירים ללא שינוי |
| **נקודות קצה ציר ראשי** | 2 (קצוות מנוגדים של הציר הארוך יותר) | גררו כדי לשנות את אורך חצי-הציר הראשי; הגודל המוחלט של הציר המשני נשאר קבוע |
| **נקודות קצה ציר משני** | 2 (קצוות מנוגדים של הציר הקצר יותר) | גררו כדי לשנות את אורך חצי-הציר המשני; הציר הראשי ללא שינוי |

נקודות האחיזה של הציר הראשי והמשני עצמאיות — תוכלו לעצב מחדש את האליפסה בלי להריץ את הפקודה מחדש.

## בחירת אליפסות

| שיטה | התנהגות |
|--------|-----------|
| **לחיצה** | בוחרת אם הלחיצה נופלת קרוב למתאר האליפסה |
| **גרירה ימינה** (קפדנית) | תיבת התיחום מיושרת-הצירים של האליפסה חייבת להתאים במלואה בתוך תיבת הבחירה |
| **גרירה שמאלה** (חוצה) | כל חלק ממתאר האליפסה שחוצה את גבול תיבת הבחירה בוחר אותה |

## מאפיינים

**כללי**

| מאפיין | ברירת מחדל | משמעות |
|----------|---------|---------|
| צבע | 256 (ByLayer) | אינדקס צבע ACI |
| שכבה | `0` | שיוך שכבה |
| סוג קו | ByLayer | תבנית סוג קו בשם |
| קנה מידה סוג קו | 1 | גורם קנה מידה על תבנית סוג הקו |
| עובי | 0 | עובי אקסטרוזיה |

**גיאומטריה**

| מאפיין | משמעות |
|----------|---------|
| Center X / Center Y | מרכז האליפסה |
| Major Axis X / Major Axis Y | וקטור מהמרכז לנקודת הקצה של הציר הראשי (מקודד גם כיוון וגם אורך) |
| Axis Ratio | יחס בין חצי-הציר המשני לחצי-הציר הראשי (0 < יחס ≤ 1) |
| Start Angle / End Angle | זוויות פרמטריות במעלות; שתיהן 0°/360° עבור אליפסה מלאה |

## Ellipse מול Circle — מתי להשתמש בכל אחת

| | Ellipse | Circle |
|---|---------|--------|
| צירים | שני חצאי-צירים עצמאיים בכל זווית | רדיוס אחד, סימטרי |
| סיבוב | ניתן למיקום בכל זווית | ללא סיבוב |
| קלט מוקלד | אורך לכל ציר | רדיוס בלבד |
| שינוי גודל נקודת אחיזה | ראשי ומשני באופן עצמאי | כל ארבע הנקודות הקרדינליות באופן שווה |
| אידיאלי עבור | תצוגות אלכסוניות, מאפיינים אובליים, חורים בפרספקטיבה | מאפיינים עגולים סימטריים |

## DXF — ישות ELLIPSE

אליפסות נשמרות כישויות `ELLIPSE` בקובץ ה-DXF. הפורמט שומר את נקודת המרכז, את וקטור הציר הראשי המלא (כיוון + אורך) ואת יחס הצירים. סיבוב, צורה וכל מאפייני הסגנון נשמרים ללא אובדן בהעברה הדדית. מעגל **אינו** נשמר כאליפסה מנוונת — שני סוגי הישויות נשארים נבדלים במודל ה-DXF.
