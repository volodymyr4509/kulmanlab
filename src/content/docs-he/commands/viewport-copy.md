# ViewportCopy

הפקודה `ViewportCopy` מעתיקה חלון תצוגה למיקום חדש, תוך שמירה על קנה המידה ומרכז המודל שלו. זמינה רק במרחב פריסה.

## שכפול חלון תצוגה

1. עברו ללשונית פריסת נייר.
2. אופציונלית לחצו על חלון תצוגה כדי לבחור בו מראש.
3. הקלידו `ViewportCopy` במסוף או לחצו על כפתור סרגל הכלים **Viewport Copy**.
4. אם לא נבחר חלון תצוגה מראש, **לחצו על חלון התצוגה** לשכפול.
5. **לחצו על נקודת הבסיס** — הייחוס להזזה. או הקלידו `X,Y` ולחצו **Enter** לקואורדינטה מדויקת.
6. **לחצו על היעד** — חלון התצוגה ממוקם בהיסט בסיס→יעד. או השתמשו בהזנת קואורדינטות / נעילת זווית.

לאחר המיקום, הפקודה נשארת פעילה — לחצו על יעד אחר כדי למקם עותק נוסף של אותו חלון תצוגה. לחצו **Enter**, **Space** או **Escape** כדי לסיים.

## הזנת קואורדינטות

בשלבי נקודת הבסיס והיעד תוכלו להקליד קואורדינטה מדויקת במקום ללחוץ:

1. הקלידו את ערך X.
2. לחצו `,` — המסוף מציג `[X], [Y{cursor}]`.
3. הקלידו את ערך Y.
4. לחצו **Enter** לאישור.

## נעילת זווית ומרחק מדויק

לאחר שנקודת הבסיס הוגדרה, הפקודה נצמדת לצירי 45° (0°, 45°, 90°, 135°, …) כאשר הסמן מיושר. בעת נעילה:

- התצוגה המקדימה נצמדת לציר.
- הקלידו מרחק ולחצו **Enter** כדי למקם את העותק בדיוק בהיסט הזה לאורך הכיוון הנעול.

| מקש | פעולה |
|-----|--------|
| `0`–`9`, `.` | הוספת ספרה לערך המרחק |
| `-` | מרחק שלילי (הופך כיוון; רק כתו הראשון) |
| `Backspace` | מחיקת התו האחרון שהוקלד |
| `Enter` | מיקום העותק במרחק שהוקלד |

## מרגע לוח המקשים

| מקש | פעולה |
|-----|--------|
| `0`–`9`, `.`, `-` | התחלת הזנת קואורדינטת X, או מרחק בעת נעילת זווית |
| `,` | נעילת X ומעבר להזנת Y |
| `Enter` | אישור קואורדינטה או מרחק שהוקלדו |
| `Enter` / `Space` | סיום (כשאין קלט בתהליך) |
| `Escape` | ביטול ואיפוס |

## הערות

- ViewportCopy זמינה רק כאשר לשונית פריסת נייר פעילה.
- חלון התצוגה המשוכפל יורש את אותו קנה מידה, מרכז מודל, מצב נעילה ומידות כמו המקורי.
- כדי ליצור חלון תצוגה חדש מאפס, השתמשו ב-[ViewportRectangle](../viewport-rectangle/).
