# Line

פקודת `line` משרטטת קטעי קו ישרים בודדים, הנשמרים כישויות `LINE` נפרדות במודל ה-DXF. לאחר כל קטע הפקודה נשארת פעילה ומשתמשת מחדש בנקודת הקצה כנקודת התחלה חדשה, כך שתוכלו לבנות נתיבים מחוברים קטע אחר קטע. בניגוד ל-[Polyline](../polyline/), קווים משורשרים נשארים ישויות עצמאיות — כל אחד ניתן לגיזום, הארכה או מחיקה בלי להשפיע על השכנים שלו.

## שרטוט קווים

1. הקלידו `line` במסוף או לחצו על כפתור סרגל הכלים **Line**.
2. **לחצו על נקודת ההתחלה**, או הקלידו `X,Y` ולחצו **Enter** לקואורדינטה מדויקת.
3. **לחצו על נקודת הסיום** — הקטע ממוקם ונקודת הקצה הופכת לנקודת ההתחלה הבאה. הזנת קואורדינטות עובדת גם כאן.
4. המשיכו ללחוץ (או להקליד) כדי לשרשר עוד קטעים.
5. לחצו **Enter** או **Escape** כדי לעצור.

```
  ●──────────●──────────●──────────●
 התחלה    לחיצה 2     לחיצה 3    Enter לסיום
            (הופכת אוטומטית להתחלה הבאה)
```

צריכים רק קטע בודד? לחצו **Enter** או **Escape** מיד לאחר שלב 3.

## הזנת קואורדינטות

במקום ללחוץ, הקלידו מיקום מדויק להתחלה או כל נקודה עוקבת:

1. הקלידו את ערך X (ספרות, `.` או `-`).
2. לחצו `,` — המסוף מציג `[X], [Y{cursor}]`.
3. הקלידו את ערך Y.
4. לחצו **Enter** למיקום הנקודה.

## נעילת זווית והזנת אורך מדויק

בזמן שאתם מזיזים את הסמן לאחר מיקום נקודה, הפקודה עוקבת אחר ציר נעילה של 45° (0°, 45°, 90°, 135°, …). הזווית **נעולה** כאשר:

- הסמן נמצא במרחק של לפחות **5 × גודל נקודת אחיזה** מהעוגן, **וגם**
- הוא בטווח של **גודל נקודת אחיזה אחת** מרחק ניצב מהציר הקרוב ביותר.

בעת נעילה התצוגה המקדימה נצמדת לציר ותוכלו להזין אורך מדויק:

| מקש | פעולה |
|-----|--------|
| `0`–`9`, `.` | הוספת ספרה לערך האורך |
| `-` | אורך שלילי — הופך כיוון לאורך הציר (רק כתו הראשון) |
| `Backspace` | מחיקת התו האחרון שהוקלד |
| `Enter` | מיקום נקודת הקצה במרחק שהוקלד |

הערך הצבור מוצג בזמן אמת במסוף (לדוגמה `click end point or enter length: 12.5`). לחיצה בעת נעילה מקרינה את הלחיצה על הציר, כך שנקודת הקצה תמיד נמצאת עליו בדיוק.

חזרה קרוב לנקודת העוגן מנתקת את הנעילה.

## מרגע לוח המקשים

| מקש | פעולה |
|-----|--------|
| `0`–`9`, `.`, `-` | התחלת הזנת קואורדינטת X, או מרחק בעת נעילת זווית |
| `,` | נעילת X ומעבר להזנת Y |
| `Backspace` | מחיקת התו האחרון שהוקלד |
| `Enter` | אישור קואורדינטה או אורך שהוקלדו, או סיום השרשרת אם לא הוקלד דבר |
| `Escape` | סיום השרשרת ויציאה |

## עריכת נקודות אחיזה — מתיחת נקודות קצה

קו נבחר מציג שלוש נקודות אחיזה:

| נקודת אחיזה | היכן | מה היא עושה |
|------|-------|--------------|
| **התחלה** | נקודת הקצה הראשונה | גררו כדי למקם מחדש — הסוף נשאר קבוע |
| **אמצע** | מרכז הקו | מפעילה **Move** לכל הקו |
| **סוף** | נקודת הקצה השנייה | גררו כדי למקם מחדש — ההתחלה נשארת קבועה |

מתיחת נקודת קצה אחת לעולם לא משפיעה על השנייה. זה שונה מעריכת נקודות אחיזה ב-[Polyline](../polyline/), שבה הזזת קודקוד מעצבת מחדש את כל הנתיב.

## בחירת קווים

| שיטה | התנהגות |
|--------|-----------|
| **לחיצה** | בוחרת את הקו אם הלחיצה נמצאת בטווח בדיקת פגיעה מהקטע |
| **גרירה ימינה** (קפדנית) | הקו נבחר רק אם שתי נקודות הקצה נופלות בתוך התיבה |
| **גרירה שמאלה** (חוצה) | הקו נבחר אם כל חלק מהקטע חוצה את גבול התיבה |

## פקודות עריכה נתמכות

Lines הן הישות ה**יחידה** שעליה [Trim](../trim/) ו-[Extend](../extend/) פועלות. כל פקודות הטרנספורמציה הסטנדרטיות חלות גם כן:

| פקודה | מה קורה לקו |
|---------|------------------------|
| [Move](../move/) | מזיזה את שתי נקודות הקצה באותה הזזה |
| [Copy](../copy/) | יוצרת קו זהה במיקום חדש |
| [Rotate](../rotate/) | מסובבת את שתי נקודות הקצה סביב נקודת הבסיס הנבחרת |
| [Mirror](../mirror/) | משקפת את שתי נקודות הקצה על פני ציר השיקוף |
| [Scale](../scale/) | משנה קנה מידה של שתי נקודות הקצה באופן אחיד מנקודת הבסיס |
| [Offset](../offset/) | יוצרת קו מקביל במרחק ניצב קבוע |
| [Trim](../trim/) | חותכת את הקו בחיתוכים — **קווים בלבד** |
| [Extend](../extend/) | מארכת את נקודת הקצה הקרובה ביותר לגבול — **קווים בלבד** |
| [Delete](../delete/) | מסירה את הקו מהשרטוט |

## מאפיינים

כאשר קו נבחר, פאנל המאפיינים מציג כל שדה שרשומת ה-`LINE` ב-DXF נושאת:

**כללי**

| מאפיין | ברירת מחדל | משמעות |
|----------|---------|---------|
| צבע | 256 (ByLayer) | אינדקס צבע ACI |
| שכבה | `0` | שיוך שכבה |
| סוג קו | ByLayer | תבנית סוג קו בשם |
| קנה מידה סוג קו | 1 | גורם קנה מידה על תבנית סוג הקו |
| עובי | 0 | עובי אקסטרוזיה |

**גיאומטריה**

| מאפיין | משמעות |
|----------|---------|
| Start X / Start Y | קואורדינטות נקודת הקצה הראשונה |
| End X / End Y | קואורדינטות נקודת הקצה השנייה |

כל השדות ניתנים לעריכה ישירות בפאנל בלי להריץ את הפקודה מחדש.

## Line מול Polyline — מתי להשתמש בכל אחת

| | Line | Polyline |
|---|------|---------|
| מספר ישויות | `LINE` אחד לכל קטע | `LWPOLYLINE` אחד לכל הנתיב |
| Trim / Extend | כן — קטע אחר קטע | לא |
| צורה סגורה | לא | כן (דגל סגירה) |
| עריכת נקודות אחיזה | מתיחת נקודות קצה בודדות | הזזת כל קודקוד לאורך הנתיב |
| אידיאלי עבור | קווי בנייה, קטעים בודדים, גיאומטריה שתגזרו | קווי מתאר, מתארים, צורות שתשמרו שלמות |

## DXF — ישות LINE

קווים נשמרים כישויות `LINE` בקובץ ה-DXF. כל מאפיין — קואורדינטות התחלה/סיום, צבע, שכבה, סוג קו, קנה מידה של סוג קו ועובי — נשמר ללא אובדן בהעברה הדדית. כאשר אתם פותחים DXF המכיל ישויות `LINE`, הן הופכות לאובייקטי `Line` הניתנים לעריכה מלאה בעורך.

קווים המשורטטים בעורך נכתבים גם כישויות `LINE` בעת השמירה, כך שהם קריאים על ידי LibreCAD, FreeCAD וכל יישום אחר התומך ב-DXF.
