# Delete

פקודת `delete` מסירה ישויות נבחרות מהשרטוט. מחיקות נרשמות בהיסטוריית [Undo](../undo/) וניתן לבטלן עד 20 שלבים. אין דיאלוג "אישור מחיקה" נפרד — האישור הוא לחיצת מקש בודדת.

## שתי דרכים למחוק

**בחירה מראש, ואז מחיקה** — הנתיב המהיר ביותר:

1. בחרו ישות אחת או יותר על הקנבס.
2. הקלידו `delete` במסוף, לחצו על כפתור סרגל הכלים **Delete**, **או לחצו על מקש `Delete`** ישירות.

הישויות מוסרות מיידית — ללא שלב אישור נוסף.

**הפעלה, ואז בחירה**:

1. הקלידו `delete` או לחצו על כפתור סרגל הכלים (ללא בחירה).
2. **בחרו אובייקטים** — לחצו כדי לשנות בחירה, או גררו לבחירה לפי אזור.
3. לחצו **Enter**, **Space** או **Delete** כדי לאשר ולהסיר את הישויות הנבחרות.

## קיצור מקש Delete

מקש `Delete` במקלדת פועל כ**קיצור גלובלי** — אם ישויות כלשהן נבחרות כרגע, לחיצה עליו מוחקת אותן מיידית, גם ללא פתיחת פקודת Delete במסוף. זהו תהליך המחיקה החד-שלבי המהיר ביותר:

```
לחיצה על ישות → לחיצה על מקש Delete → סיום
```

## בחירה במהלך הפקודה

| שיטה | התנהגות |
|--------|-----------|
| **לחיצה** | משנה בחירה של הישות מתחת לסמן |
| **גרירה ימינה** (קפדנית) | בוחרת רק ישויות שנמצאות במלואן בתוך התיבה |
| **גרירה שמאלה** (חוצה) | בוחרת ישויות שחותכות את גבול התיבה |
| **Enter** / **Space** / **Delete** | מאשרת ומוחקת את הישויות הנבחרות |

## שחזור ישויות שנמחקו

מחיקות ניתנות לביטול עם פקודת [Undo](../undo/) (הקלידו `undo` או השתמשו בכפתור סרגל הכלים). ניתן לבטל עד **20 שלבים** לכל קובץ, וההיסטוריה נשמרת לאורך רענוני עמוד. אם חרגתם מ-20 מחיקות בלי לשמור, מחיקות מוקדמות יותר לא ניתנות לשחזור.

## ישויות נתמכות

Delete פועלת על כל סוג ישות — Line, Polyline, Rectangle, Circle, Arc, Ellipse, Text, Spline, Dimension, Leader וכל האחרים.
