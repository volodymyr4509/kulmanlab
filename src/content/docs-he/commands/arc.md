# Arc

פקודת `arc` משרטטת קשת מעגלית דרך שלוש נקודות שאתם לוחצים עליהן. הקשת מחושבת כמעגל ההיקפי הייחודי העובר דרך שלוש הנקודות — אין צורך לציין מרכז או רדיוס ישירות. הקשת רצה מהלחיצה הראשונה ללחיצה השלישית, ועוברת דרך השנייה.

## שרטוט קשת

1. הקלידו `arc` במסוף או לחצו על כפתור סרגל הכלים **Arc**.
2. **לחצו על הנקודה הראשונה** — קצה אחד של הקשת. או הקלידו `X,Y` ולחצו **Enter** לקואורדינטה מדויקת.
3. **לחצו על הנקודה השנייה** — נקודה שהקשת חייבת לעבור דרכה (קובעת עקמומיות וכיוון). הזנת קואורדינטות עובדת גם כאן.
4. **לחצו על הנקודה השלישית** — הקצה השני של הקשת. הקשת ממוקמת והפקודה יוצאת. הזנת קואורדינטות עובדת גם כאן.

```
           ● (לחיצה שנייה — אמצע העקומה)
          / \
         /   \
        ●     ●
     ראשונה   שלישית
```

תצוגה מקדימה של קו מחברת בין שתי הלחיצות הראשונות בזמן שאתם ממקמים את השלישית. מהלחיצה השנייה ואילך תצוגה מקדימה חיה של הקשת עוקבת אחר הסמן.

> **נקודות קוליניאריות**: אם שלוש הנקודות נמצאות על קו ישר, לא ניתן לחשב את הקשת ואף ישות לא מוצבת. הזיזו את הנקודה השנייה מהקו ונסו שוב.

## הזנת קואורדינטות

בכל אחד משלושת השלבים תוכלו להקליד מיקום מדויק במקום ללחוץ:

1. הקלידו את ערך X.
2. לחצו `,` — המסוף מציג `[X], [Y{cursor}]`.
3. הקלידו את ערך Y.
4. לחצו **Enter** למיקום הנקודה.

## מרגע לוח המקשים

| מקש | פעולה |
|-----|--------|
| `0`–`9`, `.`, `-` | התחלת הזנת קואורדינטת X |
| `,` | נעילת X ומעבר להזנת Y |
| `Backspace` | מחיקת התו האחרון שהוקלד |
| `Enter` | אישור הקואורדינטה שהוקלדה |
| `Escape` | ביטול כל הנקודות שהוצבו ויציאה |

## עריכת נקודות אחיזה — התאמת קצוות ורדיוס

קשת נבחרת חושפת שלוש נקודות אחיזה:

| נקודת אחיזה | מיקום | מה היא עושה |
|------|----------|--------------|
| **מרכז** | המרכז הגיאומטרי של מעגל ההיקף | מזיזה את כל הקשת; רדיוס וזוויות ללא שינוי |
| **התחלה** | נקודת הקצה הראשונה על הקשת | גררו כדי להזיז את ההתחלה לאורך מעגל ההיקף — משנה גם את זווית ההתחלה וגם את הרדיוס |
| **סוף** | נקודת הקצה האחרונה על הקשת | גררו כדי להזיז את הסוף לאורך מעגל ההיקף — משנה גם את זווית הסוף וגם את הרדיוס |

גרירת נקודת אחיזה של התחלה או סוף ממקמת אותה מחדש למיקום הגרירה ומחשבת מחדש גם את הזווית וגם את הרדיוס מהמיקום החדש הזה ביחס למרכז. נקודת הקצה הנגדית נשארת קבועה.

## בחירת קשתות

| שיטה | התנהגות |
|--------|-----------|
| **לחיצה** | בוחרת אם הלחיצה נופלת קרוב לעקומת הקשת (לא למיתר) |
| **גרירה ימינה** (קפדנית) | נקודות דגימה המפוזרות לאורך הקשת חייבות כולן להימצא בתוך התיבה |
| **גרירה שמאלה** (חוצה) | כל נקודת דגימה על הקשת שנופלת בתוך התיבה בוחרת אותה |

## מאפיינים

**כללי**

| מאפיין | ברירת מחדל | משמעות |
|----------|---------|---------|
| צבע | 256 (ByLayer) | אינדקס צבע ACI |
| שכבה | `0` | שיוך שכבה |
| סוג קו | ByLayer | תבנית סוג קו בשם |
| קנה מידה סוג קו | 1 | גורם קנה מידה על תבנית סוג הקו |
| עובי | 0 | עובי אקסטרוזיה |

**גיאומטריה**

| מאפיין | משמעות |
|----------|---------|
| Center X / Center Y | מרכז מעגל ההיקף |
| Radius | רדיוס מעגל ההיקף |
| Start Angle | זווית במעלות שבה הקשת מתחילה (נמדדת מציר ה-X החיובי) |
| End Angle | זווית במעלות שבה הקשת מסתיימת |

## Arc מול Circle — מתי להשתמש בכל אחת

| | Arc | Circle |
|---|-----|--------|
| טווח | חלקי — מהלחיצה הראשונה לשלישית | 360° מלא |
| שיטת קלט | שלוש נקודות על העקומה | מרכז + רדיוס (לחיצה או הקלדה) |
| קלט מוקלד | קואורדינטת X,Y לכל נקודה | ערך רדיוס (המרכז מקבל גם X,Y) |
| שינוי גודל לאחר מיקום | גרירת נקודות אחיזה של התחלה/סוף | גרירת כל נקודת אחיזה קרדינלית |
| אידיאלי עבור | עיגולים, פינות מעוגלות, נתיבים קמורים | חורים מלאים, מאפיינים עגולים |

## DXF — ישות ARC

קשתות נשמרות כישויות `ARC` בקובץ ה-DXF, השומרות קואורדינטות מרכז, רדיוס, זווית התחלה וזווית סוף. כל המאפיינים — כולל צבע, שכבה, סוג קו, קנה מידה של סוג קו ועובי — נשמרים ללא אובדן בהעברה הדדית. כל יישום התומך ב-DXF (LibreCAD, FreeCAD וכו') קורא אותן כקשתות תקניות.
