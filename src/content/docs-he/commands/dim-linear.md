# Dimension Linear

פקודת `dimlinear` ממקמת מידה אופקית או אנכית בין שני מקורות קו עזר. קו המידה תמיד רץ באופן אופקי מושלם או אנכי מושלם — הוא לא יכול להיות ממוקם בזווית שרירותית. השתמשו ב-[Dimension Aligned](../dim-aligned/) כאשר אתם זקוקים למידה מקבילה לקו אלכסוני.

## אנטומיה של מידה ליניארית

```
  |←————— 5.00 —————→|
  |                   |
  ●  (קו עזר 1)       ●  (קו עזר 2)
  p1                  p2
```

- **קווי עזר** — יורדים מכל נקודה נמדדת ניצבים לקו המידה.
- **קו מידה** — אופקי (מודד מרחק X) או אנכי (מודד מרחק Y).
- **ערך** — המרחק המוקרן לאורך הציר הנבחר, לא המרחק האמיתי בין הנקודות.

## מיקום מידה ליניארית

1. הקלידו `dimlinear` במסוף או לחצו על כפתור סרגל הכלים **Dimension Linear**.
2. **לחצו על מקור קו העזר הראשון** (p1), או הקלידו `X,Y` ולחצו **Enter** לקואורדינטה מדויקת.
3. **לחצו על מקור קו העזר השני** (p2). הזנת קואורדינטות עובדת גם כאן.
4. **הזיזו את הסמן** למיקום קו המידה. הכיוון מזוהה אוטומטית ממיקום הסמן.
5. **לחצו** למיקום, או הקלידו מרחק היסט ולחצו **Enter** למיקום מדויק.

## זיהוי כיוון אוטומטי

כאשר לא נכפה כיוון, הפקודה קוראת את מיקום הסמן ביחס לשתי הנקודות הנמדדות:

| מיקום סמן | כיוון שזוהה | מה נמדד |
|-----------------|---------------------|-----------------|
| מעל או מתחת לנקודות | אופקי | Δ X בין p1 ל-p2 |
| משמאל או מימין לנקודות | אנכי | Δ Y בין p1 ל-p2 |

לחצו **H** לנעילת אופקי או **V** לנעילת אנכי בכל עת במהלך שלב המיקום. לאחר הנעילה, הכיוון לא משתנה בזמן שאתם מזיזים את הסמן.

## הזנת מרחק היסט מוקלדת

הקלידו מספר במהלך המיקום כדי לקבע את קו המידה במרחק מדויק מהנקודות הנמדדות:

| מקש | פעולה |
|-----|--------|
| `0`–`9`, `.` | הוספת ספרה למרחק ההיסט |
| `Backspace` | מחיקת התו האחרון שהוקלד |
| `Enter` / `Space` | מיקום במרחק שהוקלד |

צד הסמן (מעל/מתחת עבור אופקי, שמאל/ימין עבור אנכי) קובע את הסימן — קו המידה מופיע בצד שבו הסמן נמצא כרגע.

## מרגע לוח המקשים

| מקש | פעולה |
|-----|--------|
| `0`–`9`, `.`, `-` | התחלת הזנת קואורדינטת X (שלבי p1/p2), או מרחק היסט (שלב מיקום) |
| `,` | נעילת X ומעבר להזנת Y (שלבי p1/p2) |
| `H` | נעילת כיוון אופקי (שלב מיקום בלבד) |
| `V` | נעילת כיוון אנכי (שלב מיקום בלבד) |
| `Backspace` | מחיקת התו האחרון שהוקלד |
| `Enter` / `Space` | אישור קואורדינטה או היסט שהוקלדו |
| `Escape` | ביטול |

## Dimension Linear מול Dimension Aligned

| | Dimension Linear | Dimension Aligned |
|---|-----------------|------------------|
| ציר | תמיד H או V | מקביל לקו הנמדד |
| מודדת | רק רכיב X או Y | מרחק אוקלידי אמיתי |
| מקשי H/V | כן — נועלים כיוון | לא — תמיד עוקב אחר p1→p2 |
| אידיאלי עבור | פריסות אורתוגונליות, תוכניות קומה | מאפיינים אלכסוניים, חיתוכים בזווית |

## עריכת התווית — מצב simple

**לחיצה כפולה** על מידה ליניארית שהוצבה פותחת את עורך הטקסט במצב **simple**. העורך מאותחל מראש עם הערך המוצג הנוכחי כך שתוכלו למקם את הסמן ולערוך אותו ישירות.

| תכונה | התנהגות |
|---------|-----------|
| מודגש / נטוי / גופן / גובה | חלים על **כל** התווית בבת אחת |
| עיצוב לפי תו | לא נתמך |
| `Enter` | מאשר את הערך וסוגר את העורך |
| מרובת שורות | לא נתמך |

ראו [עורך טקסט — מצב simple](../../interface/text-editor/#simple-mode) להפניה המלאה.

## שרשור מידות

כדי להוסיף עוד מידות הממשיכות מקו העזר האחרון, השתמשו ב-[Dimension Continue](../dim-continue/) מיד לאחר מיקום זו.

## DXF — ישות DIMENSION

מידות ליניאריות נשמרות כישויות `DIMENSION` עם `rotationDeg` מוגדר ל-`0` (אופקי) או `90` (אנכי). מקורות קווי העזר, מיקום קו המידה, מיקום הטקסט, הערך הנמדד, סגנון החץ, גובה הטקסט וכל דגלי התצוגה נשמרים ללא אובדן בהעברה הדדית.
