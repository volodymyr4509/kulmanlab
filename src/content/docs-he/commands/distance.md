# Distance

פקודת `distance` מודדת את המרחק בקו ישר (אוקלידי) בין שתי נקודות שנלחצו ומדפיסה את התוצאה במסוף עד 4 ספרות אחרי הנקודה. זוהי אחת משלוש פקודות מדידה — [Angle](../angle/) מודדת את הפתיחה הזוויתית בקודקוד, ו-[Area](../area/) מודדת את השטח התחום וההיקף של מצולע.

## אנטומיה של מדידת מרחק

```
  ● נקודה ראשונה
   \
    \  קו תצוגה מקדימה (חי)
     \
      ● נקודה שנייה    →  מסוף: "Distance: 12.3456"
```

- **נקודה ראשונה** — מקור המדידה.
- **נקודה שנייה** — נקודת סיום; מיקומה מדפיס את התוצאה מיידית.
- **תוצאה** — מוצגת במסוף, לא ממוקמת על הקנבס.

## מדידת מרחק

1. הקלידו `distance` במסוף או לחצו על כפתור סרגל הכלים **Distance**.
2. **לחצו על הנקודה הראשונה**, או הקלידו `X,Y` ולחצו **Enter** לקואורדינטה מדויקת.
3. **לחצו על הנקודה השנייה** — המרחק הנמדד מופיע במסוף. הזנת קואורדינטות עובדת גם כאן.
4. **לחצו שוב** (אופציונלי) כדי להתחיל מדידה חדשה. הפקודה נשארת פעילה.

לחצו `Escape` בכל עת כדי לחזור לשלב 2.

## שרשור מדידות

לאחר שהתוצאה מוצגת, לחיצה מתחילה מיד את המדידה הבאה — הנקודה שנלחצה הופכת לנקודה הראשונה החדשה. כך תוכלו למדוד רצף של מרחקים ללא הפעלה מחדש של הפקודה.

## Distance מול Angle

| | Distance | Angle |
|---|---------|-------|
| מה היא מודדת | אורך קו ישר | זווית פנימית בקודקוד |
| מספר לחיצות | 2 | 3 |
| פורמט התוצאה | `12.3456` (יחידות) | `45.0000°` |
| תצוגה מקדימה על הקנבס | קו מהנקודה הראשונה לסמן | שני קווים מהקודקוד לסמן |
| אידיאלי עבור | אורך של פער או קטע | זווית פתיחה בין שני מאפיינים |

## הזנת קואורדינטות

במקום ללחוץ, תוכלו להקליד מיקום מדויק לכל אחת מהנקודות:

1. הקלידו את ערך X.
2. לחצו `,` — המסוף מציג `[X], [Y{cursor}]`.
3. הקלידו את ערך Y.
4. לחצו **Enter** לאישור.

## מרגע לוח המקשים

| מקש | פעולה |
|-----|--------|
| `0`–`9`, `.`, `-` | התחלת הזנת קואורדינטת X |
| `,` | נעילת X ומעבר להזנת Y |
| `Backspace` | מחיקת התו האחרון שהוקלד |
| `Enter` | אישור הקואורדינטה שהוקלדה |
| `Escape` | ביטול וחזרה לשלב 2 |

## הערות

- התוצאות מוצגות **רק במסוף** — שום דבר אינו מתווסף לשרטוט.
- התוצאה מבוטאת באותן יחידות כמו קואורדינטות השרטוט (ללא המרת יחידות).
- הדיוק הוא תמיד 4 ספרות אחרי הנקודה.
