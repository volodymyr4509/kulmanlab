# Import

הפקודה **Import** טוענת שרטוט קיים ממערכת הקבצים המקומית שלכם ל-KulmanLab CAD. גם פורמט **DXF** התקני וגם פורמט ה-**JSON** הייחודי של KulmanLab נתמכים.

## איך לייבא קובץ

1. לחצו על כפתור סרגל הכלים **Import** (סמל תיקייה) בפאנל File בראש המסך.
2. בורר הקבצים של הדפדפן נפתח. נווטו לקובץ השרטוט שלכם ובחרו בו.
3. השרטוט נטען לקנבס מיידית. חלון התצוגה מתאים את כל הישויות אוטומטית.

לחלופין, תוכלו לגרור ולשחרר קובץ ישירות על הקנבס.

## פורמטי קבצים נתמכים

| פורמט | סיומת | מתי להשתמש |
|--------|-----------|-------------|
| **DXF** | `.dxf` | שרטוטים מ-FreeCAD, LibreCAD או כלי CAD אחרים |
| **JSON** *(native)* | `.json` | שרטוטים שנשמרו בעבר מ-KulmanLab CAD — נאמנות מלאה |

## מה מיובא מ-DXF

KulmanLab מפענח את סוגי ישויות ה-DXF הבאים:

| סוג ישות | קוד DXF | הערות |
|-------------|----------|-------|
| Line | `LINE` | |
| Circle | `CIRCLE` | |
| Arc | `ARC` | |
| Ellipse | `ELLIPSE` | |
| Polyline | `LWPOLYLINE` | |
| Spline | `SPLINE` | |
| Text | `TEXT`, `MTEXT` | |
| Dimension | `DIMENSION` | |
| Multileader | `MULTILEADER` | |

הגדרות שכבות וטבלאות סוגי קו מיובאות גם הן מקובץ ה-DXF כאשר הן קיימות.

ישויות המשתמשות בסוגי DXF לא נתמכים מדולגות בשקט — שאר השרטוט עדיין נטען.

## שיוך שמות קבצים ואחסון

כאשר אתם מייבאים קובץ, חותמת זמן מתווספת לשם הקובץ (לדוגמה `myplan_May22_14:30:00.dxf`). זה מאפשר לכם לשמור מספר גרסאות של אותו שרטוט ב[קבצים אחרונים](../files/) ללא התנגשויות שמות. אם שם הקובץ כבר מכיל חותמת זמן, הוא משמש כפי שהוא.

השרטוט נשמר אוטומטית באחסון הדפדפן (IndexedDB) לאחר הייבוא, כך שהוא מופיע בפאנל [Files](../files/) ושורד רענוני עמוד.

## מה קורה לשרטוט הנוכחי

ייבוא מחליף את הקנבס הנוכחי. אין מיזוג או הוספה. אם יש לכם שינויים לא שמורים, [ייצאו](../export/) את השרטוט הנוכחי תחילה.

## בהפעלה

KulmanLab פותח אוטומטית מחדש את הקובץ שנערך לאחרונה בעת טעינת העמוד. אם אין קבצים שמורים, שרטוט דוגמה ברירת מחדל נטען.

## פתרון בעיות

| בעיה | סיבה סבירה | תיקון |
|---------|-------------|-----|
| הקנבס ריק לאחר ייבוא | ישויות DXF משתמשות בסוגים לא נתמכים (למשל HATCH, INSERT) | הישויות דולגו — בדקו את הודעת "no entities found" במסוף |
| כפתור Import לא עושה כלום | הדפדפן חסם את בורר הקבצים | לחצו על הכפתור שוב; חלק מהדפדפנים דורשים תנועת משתמש טרייה |
| מידות נראות שגויות | DXF מכלי שכותב גיאומטריית מידות לא סטנדרטית | ייצאו מחדש מהיישום המקורי באמצעות גרסת DXF עדכנית |
