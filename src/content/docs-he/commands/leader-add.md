# Leader+

פקודת `leader+` מוסיפה זרוע חדשה של ראש חץ למולטי-לידר קיים. הזרוע החדשה מצביעה מהכתף הקיימת של הלידר לקצה ראש חץ חדש שאתם לוחצים עליו. כל העיצוב — מיקום הכתף, הטקסט, סוג ראש החץ והגודל — יורש מהלידר הנבחר.

## הוספת זרוע

1. הקלידו `leader+` במסוף.
2. **לחצו על מולטי-לידר קיים** כדי לבחור בו.
3. **לחצו על קצה ראש החץ החדש**, או הקלידו `X,Y` ולחצו **Enter** לקואורדינטה מדויקת. קו תצוגה מקדימה מוצג מהסמן לכתף של הלידר.

הזרוע ממוקמת והפקודה נשארת פעילה — תוכלו מיד ללחוץ על לידר אחר כדי להוסיף עוד זרועות. לחצו **Enter**, **Space** או **Escape** כדי לסיים.

```
  לפני:                          אחרי:
  ◄── זרוע 1                    ◄── זרוע 1
       \                               \
        ●──── כתף ──── טקסט            ●──── כתף ──── טקסט
                                       /
                                  זרוע 2 ──►  (הקצה החדש שלחצתם עליו)
```

## הזנת קואורדינטות לקצה

במקום ללחוץ, תוכלו להקליד מיקום מדויק:

1. הקלידו את ערך X.
2. לחצו `,` — המסוף מאשר ש-X נעול.
3. הקלידו את ערך Y.
4. לחצו **Enter** למיקום.

## מרגע לוח המקשים

| מקש | פעולה |
|-----|--------|
| `0`–`9`, `.`, `-` | התחלת הקלדת קואורדינטת X |
| `,` | נעילת X ומעבר להזנת Y |
| `Enter` | אישור הקואורדינטה שהוקלדה ומיקום הזרוע |
| `Enter` / `Space` | סיום (כשאין קלט בתהליך) |
| `Escape` | ביטול ואיפוס |

## הערות

- ניתן לבחור רק ישויות **Multileader** — לחיצה על כל סוג ישות אחר לא עושה דבר.
- הזרוע החדשה מקורה בכתף הקיימת; אתם בוחרים רק לאן הולך קצה ראש החץ.
- אין הגבלה על מספר הזרועות שיכולות להיות למולטי-לידר.
