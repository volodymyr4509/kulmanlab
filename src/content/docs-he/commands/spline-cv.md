# Spline CV

פקודת `splinecv` משרטטת **B-spline קובי** על ידי מיקום נקודות בקרה (CVs). העקומה המתקבלת נמשכת לכיוון כל CV אך לא עוברת דרכן — למעט בנקודת הבקרה הראשונה והאחרונה, שבהן **קשירת מפרקים** מעגנת את העקומה בדיוק. זה נותן לכם שליטה אינטואיטיבית על הצורה: משכו נקודת בקרה כדי לדחוף את העקומה לכיוונה בלי לאלץ אותה לגעת בכל נקודה.

## שרטוט spline על ידי נקודות בקרה

1. הקלידו `splinecv` במסוף או לחצו על כפתור סרגל הכלים **Spline CV**.
2. **לחצו למיקום נקודות בקרה** — כל לחיצה מוסיפה נקודת בקרה. או הקלידו `X,Y` ולחצו **Enter** לקואורדינטה מדויקת.
3. לחצו **Enter** לסיום (נדרשות לפחות 2 נקודות בקרה).

```
  CV ●         ● CV
      \       /
       \     /    ← העקומה נמשכת לכיוון ה-CVs
        \   /         אך לא עוברת דרכן
  CV ●   ●   ● CV (התחלה/סוף: העקומה נוגעת כאן)
```

התצוגה המקדימה החיה מתעדכנת בזמן שאתם מזיזים את הסמן לאחר כל נקודת בקרה, ומציגה איך ה-spline ייראה עם הנקודה הבאה במיקום הסמן. לחצו **Escape** כדי לבטל את כל נקודות הבקרה שהוצבו ולצאת.

## הזנת קואורדינטות

במקום ללחוץ, הקלידו מיקום מדויק לכל נקודת בקרה:

1. הקלידו את ערך X.
2. לחצו `,` — המסוף מציג `[X], [Y{cursor}]`.
3. הקלידו את ערך Y.
4. לחצו **Enter** למיקום נקודת הבקרה.

## מרגע לוח המקשים

| מקש | פעולה |
|-----|--------|
| `0`–`9`, `.`, `-` | התחלת הזנת קואורדינטת X |
| `,` | נעילת X ומעבר להזנת Y |
| `Backspace` | מחיקת התו האחרון שהוקלד |
| `Enter` | אישור הקואורדינטה שהוקלדה, או סיום ה-spline אם אין קלט בתהליך ויש ≥ 2 נקודות בקרה |
| `Escape` | ביטול כל נקודות הבקרה ויציאה |

## עריכת נקודות אחיזה — עיצוב מחדש דרך נקודות בקרה

CV spline נבחר חושף נקודת אחיזה אחת לכל נקודת בקרה:

| נקודת אחיזה | מיקום | מה היא עושה |
|------|----------|--------------|
| **נקודת בקרה** | בכל מיקום CV | גררו כדי להזיז את ה-CV הזו — העקומה מעצבת מחדש לכיוון המיקום החדש |

אין נקודת אחיזה "להזיז את כל ה-spline". כדי להזיז את כל ה-spline, השתמשו בפקודת [Move](../move/).

## בחירת CV splines

| שיטה | התנהגות |
|--------|-----------|
| **לחיצה** | בוחרת אם הלחיצה נופלת קרוב לכל נקודה על העקומה |
| **גרירה ימינה** (קפדנית) | כל נקודות הדגימה לאורך העקומה חייבות ליפול בתוך תיבת הבחירה |
| **גרירה שמאלה** (חוצה) | כל חלק מהעקומה שחוצה את גבול תיבת הבחירה בוחר אותה |

## מאפיינים

**כללי**

## Spline CV מול Spline Fit — באיזה להשתמש

| | Spline CV | Spline Fit |
|---|-----------|------------|
| העקומה עוברת דרך נקודות | ראשונה ואחרונה בלבד (קשור) | כל נקודה שנלחצה בדיוק |
| שליטה בצורה | משיכת CVs לכיוון אזור | הזזת נקודות התאמה שהעקומה חייבת לגעת בהן |
| אפקט עריכת נקודת אחיזה | הזזת CV → העקומה נמשכת | הזזת נקודת התאמה → העקומה מחושבת מחדש |
| אידיאלי עבור | עקומות אסתטיות חלקות, נתיבים חופשיים | עקומות שחייבות לפגוע בקואורדינטות ספציפיות |

## DXF — ישות SPLINE (צורת נקודת בקרה)

CV splines נשמרות כישויות `SPLINE` בקובץ ה-DXF, השומרות את הדרגה, וקטור המפרקים וכל קואורדינטות נקודות הבקרה. כל המאפיינים — צבע, שכבה, סוג קו, קנה מידה של סוג קו ועובי — נשמרים ללא אובדן בהעברה הדדית. `splineFlag` מוגדר ל-`9` (CV spline) כך שהצורה נשמרת בטעינה מחדש. כל יישום DXF התומך בישויות `SPLINE` עם נתוני CV קורא אותן כראוי.
