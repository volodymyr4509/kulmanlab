# New File

הפקודה **New File** מנקה את הקנבס ומתחילה שרטוט ריק חדש. שם קובץ ייחודי עם חותמת זמן נוצר אוטומטית.

## איך ליצור קובץ חדש

לחצו על כפתור סרגל הכלים **New File** (סמל עמוד חדש) בפאנל File. הקנבס מתנקה מיידית — ללא הנחיות או דיאלוגי אישור.

## מה הקובץ החדש מכיל

קובץ שנוצר לאחרונה מתחיל עם:

- **ללא ישויות** על הקנבס.
- **שכבת ברירת מחדל אחת** בשם `0` עם צבע לבן וסוג קו `Continuous`.
- **שם קובץ שנוצר** בפורמט `kulman-Mon01_HH:MM:SS.dxf` (לדוגמה `kulman-May22_14:30:00.dxf`).

הקובץ נשמר באחסון הדפדפן אוטומטית ומופיע ב[קבצים אחרונים](../files/).

## אזהרה — עבודה לא שמורה נמחקת

לחיצה על **New File** מוחקת את כל הישויות בקנבס הנוכחי ללא אזהרה. אם ברצונכם לשמור את השרטוט הנוכחי, [ייצאו](../export/) אותו קודם.

## מתי להשתמש ב-New File מול Import

| מצב | פעולה מומלצת |
|-----------|-------------------|
| התחלת שרטוט מאפס | **New File** |
| פתיחת קובץ DXF או JSON קיים | [Import](../import/) |
| העתקת שרטוט כדי לעבוד על גרסה | [ייצאו](../export/) את הקובץ הנוכחי, ואז [ייבאו](../import/) את ההעתק |

## פקודות קשורות

- [Import](../import/) — פתיחת שרטוט DXF או JSON קיים
- [Export](../export/) — הורדת השרטוט לפני התחלה מחדש
- [Files](../files/) — שחזור שרטוט קודם מאחסון הדפדפן
