# Array Grid

פקודת `ArrayGrid` יוצרת רשת מלבנית של עותקים מהישויות הנבחרות — הזינו את מספר השורות, מספר העמודות והמרווח ביניהן, הכול מוקלד במסוף. הבחירה המקורית תופסת את התא שורה 0, עמודה 0; כל תא אחר הוא עותק מוזז.

## שתי דרכים להתחיל

**בחירה מראש, ואז Array** — בחרו ישויות תחילה, ואז הפעילו:

1. בחרו ישות אחת או יותר על הקנבס.
2. הקלידו `arraygrid` במסוף (מספיק `arr` — חד-משמעי) או לחצו על כפתור סרגל הכלים **Array Grid**.
3. הקלידו את מספר **השורות** ולחצו **Enter**.
4. הקלידו את מספר **העמודות** ולחצו **Enter**.
5. הקלידו את **המרווח בין השורות** ולחצו **Enter**.
6. הקלידו את **המרווח בין העמודות** ולחצו **Enter** — הרשת נוצרת מיד.

**הפעלה, ואז בחירה** — התחילו את הפקודה ללא בחירה:

1. הקלידו `arraygrid` או לחצו על כפתור סרגל הכלים.
2. **בחרו אובייקטים** — לחצו כדי לשנות בחירה לכל ישות, או גררו לבחירה לפי אזור.
3. לחצו **Enter** או **Space** כדי לאשר את הבחירה.
4. המשיכו עם שורות ← עמודות ← מרווח שורות ← מרווח עמודות כמו למעלה.

```
  2 שורות x 3 עמודות:

  [B] [B] [B]   <- שורה 1 (עותקים מוזזים)
  [A] [A] [A]   <- שורה 0: בחירה מקורית, עותקים ימינה
```

> המסוף זקוק רק לאותיות מספיקות כדי להיות חד-משמעי — הקלדת `arr` ולחיצה על **Enter** מפעילה את Array Grid ישירות, מכיוון שאין שם פקודה אחר שמתחיל בשלוש אותיות אלה (Arc, Area, Align ו-Angle כולם מתפצלים מוקדם יותר).

## שורות, עמודות ומרווח

| הנחיה | מקבלת | הערות |
|-------|-------|-------|
| שורות | מספרים שלמים חיוביים (1, 2, 3…) | ספרות בלבד — ללא נקודה עשרונית או סימן |
| עמודות | מספרים שלמים חיוביים (1, 2, 3…) | ספרות בלבד — ללא נקודה עשרונית או סימן |
| מרווח שורות | עשרוני עם סימן (למשל `10`, `-5.5`) | המרחק בין שורות; ערך שלילי הופך את הכיוון |
| מרווח עמודות | עשרוני עם סימן (למשל `10`, `-5.5`) | המרחק בין עמודות; ערך שלילי הופך את הכיוון |

עם שורה אחת ועמודה אחת, לא נוצרים עותקים — הפקודה יוצאת מבלי לשנות את השרטוט.

## מרגע לוח המקשים

| מקש | פעולה |
|-----|--------|
| `Enter` / `Space` | מאשר את הבחירה ומתקדם להנחיית השורות |
| `0`–`9` | הזנת ספרות לשורות או עמודות |
| `0`–`9`, `.`, `-` | הזנת ספרות למרווח שורות/עמודות (`-` רק כתו הראשון) |
| `Backspace` | מחיקת התו האחרון שהוקלד בהנחיה הנוכחית |
| `Enter` | מאשר את ההנחיה הנוכחית ומתקדם לבאה |
| `Escape` | מנקה את ערכי השורות/עמודות/מרווח שהוקלדו וחוזר לשלב הבחירה |

## בחירה במהלך הפקודה

| שיטה | התנהגות |
|--------|-----------|
| **לחיצה** | משנה בחירה של הישות מתחת לסמן |
| **גרירה ימינה** (קפדנית) | מוסיפה ישויות שנמצאות במלואן בתוך התיבה |
| **גרירה שמאלה** (חוצה) | מוסיפה ישויות שחותכות את גבול התיבה |
| **Enter** / **Space** | מאשר את הבחירה ומתקדם להנחיית השורות |

## אחרי ה-Array

העותקים החדשים מתווספים לשרטוט והפקודה יוצאת — הבחירה המקורית מתנקה. הריצו שוב את **Array Grid**, או התחילו פקודה חדשה.

## Array Grid מול Copy

| | Array Grid | Copy |
|---|-----------|------|
| בחירת נקודות | אין — שורות, עמודות ומרווח מוקלדים | נקודת הבסיס והיעד נלחצות (או מוקלדות) |
| עותקים שנוצרים | שורות × עמודות − 1 | בדיוק 1 לכל פעולת העתקה |
| פריסה | רשת מלבנית סדירה | בכל מקום, בכל היסט |
| הכי טוב עבור | חזרה על יחידה בתבנית סדירה (חורים, אריחים, אלמנטי חיבור) | שכפול בודד במיקום שרירותי |

## ישויות נתמכות

Array Grid פועלת על כל סוג ישות. כל הישויות מיישמות `translate(dx, dy)` באופן פנימי — אותה פעולה שבה משתמשות [Copy](../copy/) ו-[Move](../move/) — כך שאף אחת אינה מוחרגת.
