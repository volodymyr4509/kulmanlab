# Mirror

פקודת `mirror` יוצרת עותקים משוקפים של ישויות נבחרות המשתקפות על פני ציר דו-נקודתי. המקוריים **תמיד נשמרים** — בניגוד ל-[Move](../move/) או [Rotate](../rotate/), Mirror לעולם לא משנה ישויות קיימות; היא רק מוסיפה חדשות.

## שתי דרכים להתחיל

**בחירה מראש, ואז שיקוף** — בחרו ישויות תחילה, ואז הפעילו:

1. בחרו ישות אחת או יותר על הקנבס.
2. הקלידו `mirror` במסוף או לחצו על כפתור סרגל הכלים **Mirror**.
3. **לחצו על הנקודה הראשונה** של ציר השיקוף, או הקלידו `X,Y` ולחצו **Enter** לקואורדינטה מדויקת.
4. **לחצו על הנקודה השנייה** — עותקים משוקפים ממוקמים והפקודה יוצאת. הזנת קואורדינטות עובדת גם כאן.

**הפעלה, ואז בחירה** — התחילו את הפקודה ללא בחירה:

1. הקלידו `mirror` או לחצו על כפתור סרגל הכלים.
2. **בחרו אובייקטים** — לחצו כדי לשנות בחירה, או גררו לבחירה לפי אזור.
3. לחצו **Enter** או **Space** כדי לאשר את הבחירה.
4. **לחצו על הנקודה הראשונה**, ולאחר מכן **לחצו על הנקודה השנייה** של ציר השיקוף (הזנת קואורדינטות זמינה בשני השלבים).

```
  מקורי:              ציר שיקוף:          תוצאה:
                     |
  [ישות A]      →    |    →    [ישות A] + [A משוקפת]
                     |
```

תצוגה מקדימה חיה של העותקים המשוקפים עוקבת אחר הסמן בזמן שאתם ממקמים את נקודת הציר השנייה.

## ציר השיקוף

הציר הוא קו אינסופי דרך שתי הנקודות שנלחצו. הוא יכול להיות בכל זווית:

- הזיזו את הסמן קרוב ל**ציר נעילה של 45°** (0°, 45°, 90°, 135°, …) והציר נעול לזווית הזו — שימושי לשיקופים אופקיים, אנכיים או אלכסוניים נקיים.
- לחצו הרחק מאזור הנעילה לציר בזווית חופשית.

## הזנת קואורדינטות

בכל שלב של נקודת הציר, הקלידו מיקום מדויק במקום ללחוץ:

1. הקלידו את ערך X.
2. לחצו `,` — המסוף מציג `[X], [Y{cursor}]`.
3. הקלידו את ערך Y.
4. לחצו **Enter** לאישור.

## מרגע לוח המקשים

| מקש | פעולה |
|-----|--------|
| `Enter` / `Space` | אישור הבחירה |
| `0`–`9`, `.`, `-` | התחלת הזנת קואורדינטת X |
| `,` | נעילת X ומעבר להזנת Y |
| `Backspace` | מחיקת התו האחרון שהוקלד |
| `Enter` | אישור הקואורדינטה שהוקלדה |
| `Escape` | ביטול ואיפוס |

## בחירה במהלך הפקודה

| שיטה | התנהגות |
|--------|-----------|
| **לחיצה** | משנה בחירה של הישות מתחת לסמן |
| **גרירה ימינה** (קפדנית) | מוסיפה ישויות שנמצאות במלואן בתוך התיבה |
| **גרירה שמאלה** (חוצה) | מוסיפה ישויות שחותכות את התיבה |
| **Enter** / **Space** | מאשר את הבחירה |

## מה משתקף

כל סוג ישות נתמך. גיאומטריה משתקפת מתמטית על פני הציר:

| ישות | מה משתנה |
|--------|-------------|
| Line | שתי נקודות הקצה משוקפות |
| Circle | המרכז משוקף; רדיוס ללא שינוי |
| Arc | המרכז משוקף; זוויות התחלה וסוף מחושבות מחדש על פני הציר |
| Ellipse | המרכז משוקף; כיוון הציר הראשי הפוך על פני הציר |
| Polyline / Rectangle | כל קודקוד משוקף |
| Text | נקודת העיגון משוקפת; מחרוזת הטקסט **לא** הופכת |
| Spline | כל נקודות הבקרה / נקודות ההתאמה משוקפות |
