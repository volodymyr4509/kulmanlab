# Help

פותח את פאנל העזרה בתוך האפליקציה עם סיכום של כל קיצורי המקלדת, טיפים לניווט ושימוש כללי.

## איך להשתמש

לחצו על כפתור סרגל הכלים **Help** או הקלידו `help` במסוף. פאנל העזרה נפתח מיידית והפקודה יוצאת.

## מה נמצא בפאנל העזרה

### ניווט
| קיצור | פעולה |
|----------|--------|
| `Scroll wheel` | הגדלה / הקטנה |
| `Middle click + drag` | הזזת התצוגה |
| `Double middle click` | התאמת כל הישויות למסך |

### פקודות
| קיצור | פעולה |
|----------|--------|
| הקלדה | התחילו להקליד שם פקודה — הצעות מופיעות על הקנבס |
| `Delete` | מחיקת כל הישויות הנבחרות |
| `Enter` / `Space` | אישור פקודה או קלט |
| `Enter` / `Space` (במנוחה) | חזרה על הפקודה האחרונה שהופעלה |
| `Escape` | ביטול הפקודה הנוכחית או ניקוי הבחירה |

### בחירה
| קיצור | פעולה |
|----------|--------|
| `Left click` | בחירת ישות |
| `Drag →` | בחירת רק ישויות **הנמצאות במלואן** באזור הגרירה |
| `Drag ←` | בחירת כל ישות ש**נוגעת** באזור הגרירה |
| `Left click grip` | עריכת קודקוד או נקודת קצה ישירות |

### טיפים לשרטוט
- **נקודות הצמדה** — הסמן נצמד אוטומטית לנקודות קצה קרובות של ישויות.
- **זווית דביקה** — הזיזו את הסמן מספיק רחוק בכיוון אחד כדי לנעול למרווחים של 45°.
- **אורך מדויק** — בעת נעילת הזווית, הקלידו מספר ולחצו `Enter` כדי להגדיר מרחק מדויק.
