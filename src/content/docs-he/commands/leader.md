# Leader

פקודת `leader` משרטטת הערת מולטי-לידר בארבעה שלבים: ראש חץ הנוגע במאפיין, קו לידר שמתעקל בכתף (dogleg), עוגן טקסט, וטקסט מוקלד. מכל פקודות ההערה, Leader היא היחידה הכוללת שלב הזנת טקסט אינטראקטיבי עם תצוגה מקדימה של סמן מהבהב.

## אנטומיה של מולטי-לידר

```
  ◄── קצה ראש חץ  (שלב 2 — נוגע במאפיין)
      \
       \  קו לידר
        \
         ●──── כתף (שלב 3) ──── עוגן טקסט (שלב 4)
                                    טקסט תווית  (שלב 5)
```

- **קצה ראש חץ** — הקצה המחודד המוצב במאפיין המוער.
- **כתף (dogleg)** — המרפק שבו קו הלידר מתעקל לכיוון הטקסט.
- **עוגן טקסט** — היכן שהתווית ממוקמת. הטקסט מיושר שמאלה או ימינה אוטומטית.

## שרטוט לידר

1. הקלידו `leader` במסוף או לחצו על כפתור סרגל הכלים **Leader**.
2. **לחצו על קצה ראש החץ**, או הקלידו `X,Y` ולחצו **Enter** לקואורדינטה מדויקת.
3. **לחצו על הכתף** — העיקול בלידר. הזווית נצמדת למרווחים של 45°; הקלידו אורך ולחצו **Enter** למיקום מדויק. או הקלידו `X,Y` להזנת קואורדינטה מוחלטת.
4. **לחצו על מיקום הטקסט** — היכן שהתווית מעוגנת. אותן אפשרויות חלות: לחיצה, נעילת-זווית + אורך, או `X,Y`.
5. **הקלידו את טקסט התווית** — התצוגה המקדימה על הקנבס מתעדכנת בזמן אמת עם סמן מהבהב. לחצו **Enter** למיקום.

## הזנת קואורדינטות (כל שלבי הנקודה)

בכל שלב של בחירת נקודה (קצה, כתף, מיקום טקסט) תוכלו להקליד קואורדינטה מדויקת במקום ללחוץ:

1. הקלידו את ערך X (ספרות, `.` או `-`).
2. לחצו `,` — המסוף מציג `[X], [Y{cursor}]` ומאשר ש-X נעול.
3. הקלידו את ערך Y.
4. לחצו **Enter** למיקום הנקודה.

## נעילת זווית (שלבים 3 ו-4)

לאחר כל נקודה שהוצבה, הפקודה נצמדת לצירי 45° כאשר הסמן רחוק מספיק. בעת נעילה:
- התצוגה המקדימה נצמדת לציר.
- הקלידו אורך ולחצו **Enter** למיקום הנקודה הבאה במרחק המדויק הזה.

נעילת זווית והזנת קואורדינטות מוציאות זו את זו — ברגע שתקלידו ספרה בלי `,` קודם, הפקודה מפרשת אותה כמרחק (נעילת זווית חייבת להיות פעילה). כדי להזין קואורדינטה מוחלטת במקום זאת, התחילו במספר X ואחריו פסיק.

## עריכת תווית הטקסט

בזמן הקלדת התווית בשלב 5, תוכלו לנווט ולערוך את הטקסט לפני המיקום:

| מקש | פעולה |
|-----|--------|
| כל תו הניתן להדפסה | הוספה במיקום הסמן |
| `←` / `→` | הזזת הסמן שמאלה או ימינה |
| `Backspace` | מחיקת התו משמאל לסמן |
| `Delete` | מחיקת התו מימין לסמן |
| `Enter` | מיקום הלידר |

## כיוון טקסט אוטומטי

יישור הטקסט מתאים עצמו למיקום הסמן ביחס לכתף:

| מיקום סמן | כיוון טקסט |
|-----------------|---------------|
| מ**ימין** לכתף | משמאל לימין מעוגן הטקסט |
| מ**שמאל** לכתף | מימין לשמאל (מעוגן בצד הימני) |

אין צורך בהתאמה ידנית — הזיזו את הסמן לצד שבו אתם רוצים את התווית והיא מתיישרת נכון.

## מרגע לוח המקשים

**שלבי נקודה (קצה, כתף, מיקום טקסט)**

| מקש | פעולה |
|-----|--------|
| `0`–`9`, `.`, `-` | התחלת הקלדת קואורדינטת X (ואז `,` לנעילת X והזנת Y) |
| `,` | אישור X ומעבר להזנת Y |
| `0`–`9`, `.`, `-` | בניית מרחק בעת נעילת זווית |
| `Backspace` | מחיקת התו האחרון שהוקלד |
| `Enter` | אישור קואורדינטה או מרחק שהוקלדו |

**שלב הזנת טקסט**

| מקש | פעולה |
|-----|--------|
| תו הניתן להדפסה | הוספה בסמן |
| `←` / `→` | הזזת הסמן |
| `Backspace` | מחיקה שמאלה |
| `Delete` | מחיקה ימינה |
| `Enter` | מיקום הלידר |

| מקש | פעולה |
|-----|--------|
| `Escape` | ביטול וחזרה לשלב 2 |

## עריכת לידר קיים

**לחיצה כפולה** על מולטי-לידר שהוצב פותחת מחדש את עורך הטקסט במצב **rich**. במצב rich תוכלו להחיל מודגש, נטוי ועקיפות גופן או גובה לפי תו, ולהכניס מעברי שורה עם `Enter`. לחצו **Escape** כדי לאשר ולסגור.

ראו [עורך טקסט — מצב rich](../../interface/text-editor/#rich-mode) להפניה המלאה.

## DXF — ייבוא בלבד

**Leaders הם ייבוא בלבד.** ישויות `MLEADER` מקבצי DXF נקראות ומוצגות כראוי, אך leaders המשורטטים בעורך **לא נכתבים** בעת שמירת קובץ DXF. השתמשו ב-leaders להערות ויזואליות; הימנעו מהסתמכות עליהם לתהליכי עבודה של העברה הדדית.
