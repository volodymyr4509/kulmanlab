# Rotate

פקודת `rotate` מסובבת ישויות נבחרות סביב נקודת בסיס. אתם מציינים את זווית הסיבוב באמצעות הקלדת מספר במעלות או על ידי לחיצה — הזווית מחושבת מהכיוון בין נקודת הבסיס למיקום הלחיצה.

## שתי דרכים להתחיל

**בחירה מראש, ואז סיבוב** — בחרו ישויות תחילה, ואז הפעילו:

1. בחרו ישות אחת או יותר על הקנבס.
2. הקלידו `rotate` במסוף או לחצו על כפתור סרגל הכלים **Rotate**.
3. **לחצו על נקודת הבסיס** — מרכז הסיבוב. או הקלידו `X,Y` ולחצו **Enter** לקואורדינטה מדויקת.
4. **הקלידו זווית ולחצו Enter**, או **לחצו** כדי להגדיר את הזווית מכיוון הסמן.

**הפעלה, ואז בחירה** — התחילו את הפקודה ללא בחירה:

1. הקלידו `rotate` או לחצו על כפתור סרגל הכלים.
2. **בחרו אובייקטים** — לחצו כדי לשנות בחירה, או גררו לבחירה לפי אזור.
3. לחצו **Enter** או **Space** כדי לאשר את הבחירה.
4. **לחצו על נקודת הבסיס** (הזנת קואורדינטות זמינה), ואז הגדירו את הזווית.

```
  לפני:            אחרי (סיבוב של 90° סביב ●):
                        ╔══╗
  ●  [ישות]      →   ● ║    ║
                        ╚══╝
```

תצוגה מקדימה רפאית חיה של הישויות המסובבות עוקבת אחר זווית הסמן לאחר שנקודת הבסיס הוגדרה.

## הגדרת הזווית

**זווית מוקלדת** — הקלידו מספר (במעלות) בכל עת לאחר שנקודת הבסיס הוצבה. התצוגה המקדימה נצמדת לזווית שהוקלדה בזמן שאתם ממשיכים להתאים לפני לחיצה על Enter.

**זווית לחיצה** — אם אין ערך מוקלד, לחיצה מגדירה את הזווית שווה ל-`atan2(cursorY − baseY, cursorX − baseX)` — הכיוון מנקודת הבסיס ללחיצה, במעלות.

| מקש | פעולה |
|-----|--------|
| `0`–`9`, `.` | הוספת ספרה לערך הזווית |
| `-` | זווית שלילית (רק כתו הראשון) |
| `Backspace` | מחיקת התו האחרון שהוקלד |
| `Enter` | החלת הסיבוב בזווית שהוקלדה |

## כיוון הזווית

הזוויות עוקבות אחר **מוסכמת ה-DXF**:

- ערכים **חיוביים** מסובבים **נגד כיוון השעון** בקואורדינטות השרטוט (Y-למעלה).
- על המסך, שבו ציר ה-Y הפוך (Y-למטה), זוויות חיוביות נראות **בכיוון השעון**.

ערכים נפוצים: `90` = רבע סיבוב, `180` = חצי סיבוב, `-90` = רבע סיבוב הפוך.

## מרגע לוח המקשים

| מקש | פעולה |
|-----|--------|
| `Enter` / `Space` | אישור הבחירה |
| `0`–`9`, `.`, `-` | התחלת הזנת קואורדינטת X (שלב נקודת בסיס), או ערך זווית (שלב זווית) |
| `,` | נעילת X ומעבר להזנת Y (שלב נקודת בסיס) |
| `Backspace` | מחיקת התו האחרון שהוקלד |
| `Enter` | אישור קואורדינטה או החלת סיבוב |
| `Escape` | ביטול ואיפוס |

## בחירה במהלך הפקודה

| שיטה | התנהגות |
|--------|-----------|
| **לחיצה** | משנה בחירה של הישות מתחת לסמן |
| **גרירה ימינה** (קפדנית) | מוסיפה ישויות שנמצאות במלואן בתוך התיבה |
| **גרירה שמאלה** (חוצה) | מוסיפה ישויות שחותכות את התיבה |
| **Enter** / **Space** | מאשר את הבחירה |

## ישויות נתמכות

Rotate פועלת על כל סוג ישות. הגיאומטריה של כל ישות מסובבת סביב נקודת הבסיס — לדוגמה, Circle מזיזה את המרכז שלה בעוד הרדיוס נשאר זהה; Arc מזיזה את המרכז שלה ומזיזה את זוויות ההתחלה והסיום שלה בכמות הסיבוב; ישות Text מזיזה את נקודת העיגון שלה ומוסיפה את הזווית למאפיין Rotation Degree שלה.
