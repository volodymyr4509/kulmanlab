# Leader−

פקודת `leader-` מסירה זרוע אחת של ראש חץ ממולטי-לידר קיים. תווית הטקסט, הכתף וכל הזרועות הנותרות נשמרות — רק הזרוע הנבחרת נמחקת. ממולטי-לידר עם זרוע אחת בלבד לא ניתן להסיר את הזרוע.

## הסרת זרוע

1. הקלידו `leader-` במסוף.
2. **לחצו על מולטי-לידר** שיש לו שתיים או יותר זרועות. אם ללידר שנלחץ יש רק זרוע אחת, המסוף מציג שגיאה וממתין לבחירה תקפה.
3. **הזיזו את הסמן קרוב לזרוע** שברצונכם להסיר — הזרוע הקרובה ביותר מודגשת עם סמן.
4. **לחצו** כדי להסיר את הזרוע הזו.

הזרוע מוסרת והפקודה נשארת פעילה — תוכלו מיד ללחוץ על לידר אחר (או אותו אחד) כדי להסיר עוד זרועות. לחצו **Enter**, **Space** או **Escape** כדי לסיים.

```
  לפני:                     אחרי:
  ◄── זרוע 1                 ◄── זרוע 1
       \                          \
        ●──── כתף ──── טקסט        ●──── כתף ──── טקסט
       /
  זרוע 2 ──►  ← זרוע זו הוסרה
```

## איך נבחרת הזרוע הקרובה ביותר

הפקודה מודדת את המרחק הניצב מהסמן לקטעי הקו של כל זרוע (כולל הקטע מנקודת הזרוע האחרונה לכתף). הזרוע עם המרחק הקטן ביותר מודגשת ותוסר בלחיצה.

## מרגע לוח המקשים

| מקש | פעולה |
|-----|--------|
| `Enter` / `Space` | סיום הסרת הזרועות |
| `Escape` | ביטול ואיפוס |

## הערות

- לידר עם **זרוע אחת בלבד** מוגן — עליכם להוסיף זרוע קודם לפני שתוכלו להסיר אחת.
- מיקום הכתף ותוכן הטקסט תמיד נשמרים ללא קשר לאיזו זרוע הוסרה.
