# Move

פקודת `move` מזיזה ישויות נבחרות מנקודת בסיס לנקודת יעד. ההזזה המוחלת על כל ישות נבחרת היא הווקטור מבסיס ליעד. לאחר ההזזה כל הישויות נשארות נבחרות במיקומן החדש, מוכנות לעריכות נוספות.

## שתי דרכים להתחיל

**בחירה מראש, ואז הזזה** — בחרו ישויות תחילה, ואז הפעילו:

1. בחרו ישות אחת או יותר על הקנבס.
2. הקלידו `move` במסוף או לחצו על כפתור סרגל הכלים **Move**.
3. **לחצו על נקודת הבסיס**, או הקלידו `X,Y` ולחצו **Enter** לקואורדינטה מדויקת.
4. **לחצו על היעד** — כל הישויות הנבחרות זזות לפי וקטור בסיס→יעד. הזנת קואורדינטות עובדת גם כאן.

**הפעלה, ואז בחירה** — התחילו את הפקודה ללא בחירה:

1. הקלידו `move` או לחצו על כפתור סרגל הכלים.
2. **בחרו אובייקטים** — לחצו כדי לשנות בחירה לישויות בודדות, או גררו לבחירה לפי אזור.
3. לחצו **Enter** או **Space** כדי לאשר את הבחירה.
4. **לחצו על נקודת הבסיס**, ולאחר מכן **לחצו על היעד** (הזנת קואורדינטות זמינה בשני השלבים).

```
  לפני:                      אחרי:
  ● בסיס                       → ● יעד
  [ישות A]                      [ישות A הוזזה]
  [ישות B]                      [ישות B הוזזה]
```

תצוגה מקדימה רפאית של כל הישויות הנבחרות עוקבת אחר הסמן מנקודת הבסיס ליעד, ומציגה את התוצאה לפני שאתם לוחצים.

## הזנת קואורדינטות

בשלב נקודת הבסיס או היעד, הקלידו מיקום מדויק במקום ללחוץ:

1. הקלידו את ערך X.
2. לחצו `,` — המסוף מציג `[X], [Y{cursor}]`.
3. הקלידו את ערך Y.
4. לחצו **Enter** לאישור.

## נעילת זווית ומרחק מדויק

לאחר שנקודת הבסיס הוגדרה, הפקודה עוקבת אחר ציר נעילה של 45° (0°, 45°, 90°, 135°, …). הכיוון **נעול** כאשר הסמן רחוק מספיק מהבסיס ובטווח רוחב נקודת אחיזה אחת מהציר. בעת נעילה:

- התצוגה המקדימה הרפאית נצמדת לציר.
- הקלידו מרחק ולחצו **Enter** כדי להזיז בדיוק את המרחק הזה בכיוון הנעול.
- לחיצה מקרינה על הציר, כך שהיעד תמיד נמצא עליו בדיוק.

| מקש | פעולה |
|-----|--------|
| `0`–`9`, `.` | הוספה לערך המרחק |
| `-` | מרחק שלילי — הופך כיוון לאורך הציר (רק כתו הראשון) |
| `Backspace` | מחיקת התו האחרון שהוקלד |
| `Enter` | החלת ההזזה במרחק שהוקלד |

## מרגע לוח המקשים

| מקש | פעולה |
|-----|--------|
| `Enter` / `Space` | אישור הבחירה והמעבר לשלב נקודת הבסיס |
| `0`–`9`, `.`, `-` | התחלת הזנת קואורדינטת X, או מרחק בעת נעילת זווית |
| `,` | נעילת X ומעבר להזנת Y |
| `Backspace` | מחיקת התו האחרון שהוקלד |
| `Enter` | אישור קואורדינטה או החלת הזזה במרחק שהוקלד |
| `Escape` | ביטול ואיפוס |

## הפעלת Move מנקודת אחיזה

לחיצה על **נקודת האחיזה האמצעית** של [Line](../line/) נבחר מפעילה Move אוטומטית, כאשר האמצע כבר מוגדר כנקודת בסיס ושלב ההזזה פעיל. זו הדרך המהירה ביותר למקם מחדש קו בודד בלי לעבור דרך שלב הבחירה.

## בחירה במהלך הפקודה

כאשר הפקודה מתחילה בשלב הבחירה:

| שיטה | התנהגות |
|--------|-----------|
| **לחיצה** | משנה בחירה של הישות מתחת לסמן |
| **גרירה ימינה** (קפדנית) | מוסיפה ישויות שנמצאות במלואן בתוך התיבה |
| **גרירה שמאלה** (חוצה) | מוסיפה ישויות שחותכות את גבול התיבה |
| **Enter** / **Space** | מאשר את הבחירה ועובר לשלב נקודת הבסיס |

## לאחר ההזזה

הישויות המוזזות נשארות נבחרות במיקומן החדש. זה אומר שתוכלו מיד:
- להריץ **Move** שוב כדי להזיז אותן עוד.
- להריץ [Copy](../copy/), [Rotate](../rotate/) או [Scale](../scale/) בלי לבחור מחדש.
- ללחוץ **Delete** כדי להסיר אותן.

## Move מול Copy

| | Move | Copy |
|---|------|------|
| מיקום מקורי | מפונה — הישויות כבר לא שם | נשמר — המקוריים נשארים במקומם |
| ספירת תוצאה | אותו מספר ישויות | סט נוסף אחד לכל פעולה |
| בחירה לאחר מכן | ישויות מוזזות נבחרות במיקום החדש | ישויות מועתקות נבחרות במיקום החדש |
| אידיאלי עבור | מיקום מחדש של גיאומטריה | שכפול גיאומטריה |

## ישויות נתמכות

Move פועלת על כל סוג ישות: Line, Polyline, Rectangle, Circle, Arc, Ellipse, Text, Spline, Dimension, Leader וכל האחרים. כל הישויות מיישמות `translate(dx, dy)` כך שאף אחת אינה מוחרגת.
