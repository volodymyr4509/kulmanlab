# Angle

פקודת `angle` מודדת את הזווית הפנימית בקודקוד הנוצר על ידי שני קטעי קו דרך שלוש נקודות שנלחצו. התוצאה — תמיד בין 0° ל-180° — מוצגת במסוף עד 4 ספרות אחרי הנקודה. זוהי אחת משלוש פקודות מדידה — [Distance](../distance/) מודדת אורך קו ישר, ו-[Area](../area/) מודדת את השטח התחום וההיקף של מצולע.

## אנטומיה של מדידת זווית

```
  ● נקודה ראשונה (קצה קרן ראשונה)
   \
    \  תצוגה מקדימה של הקרן הראשונה
     \
      ● קודקוד (שלב 3)
     /
    /  תצוגה מקדימה של הקרן השנייה (לסמן)
   /
  ● נקודה שלישית  →  מסוף: "Angle: 45.0000°"
```

- **נקודה ראשונה** — קצה אחד של הזווית (שלב 2).
- **קודקוד** — הפינה שבה נמדדת הזווית (שלב 3).
- **נקודה שלישית** — הקצה השני של הזווית (שלב 4).

## מדידת זווית

1. הקלידו `angle` במסוף או לחצו על כפתור סרגל הכלים **Angle**.
2. **לחצו על הנקודה הראשונה** — קצה זרוע אחת של הזווית. או הקלידו `X,Y` ולחצו **Enter** לקואורדינטה מדויקת.
3. **לחצו על הקודקוד** — הפינה שבה שתי הזרועות נפגשות. הזנת קואורדינטות עובדת גם כאן.
4. **לחצו על הנקודה השלישית** — קצה הזרוע השנייה. הזנת קואורדינטות עובדת גם כאן. מיקום נקודה זו מדפיס את התוצאה.
5. **לחצו שוב** (אופציונלי) כדי להתחיל מדידה חדשה מאותה לחיצה כנקודה ראשונה חדשה.

## מוסכמת הזווית הפנימית

הפקודה מחשבת את הזווית באמצעות מכפלה סקלרית של שתי הקרניים מהקודקוד:

- **תמיד פנימית**: התוצאה היא הזווית הקטנה יותר, בין 0° ל-180°.
- הסדר שבו אתם לוחצים על נקודות הקצה אינו משפיע על התוצאה — רק מיקום הקודקוד חשוב.
- נקודות קוליניאריות (שלושתן על קו אחד) מחזירות 0° או 180°.

## שרשור מדידות

לאחר שהתוצאה מופיעה, לחיצה מתחילה מיד את המדידה הבאה — הנקודה שנלחצה הופכת לנקודה הראשונה החדשה. הפקודה לעולם לא יוצאת אוטומטית עד שתלחצו `Escape`.

## Angle מול Distance

| | Angle | Distance |
|---|-------|---------|
| מה היא מודדת | זווית פנימית בקודקוד | אורך קו ישר |
| מספר לחיצות | 3 | 2 |
| פורמט התוצאה | `45.0000°` | `12.3456` (יחידות) |
| תצוגה מקדימה על הקנבס | שני קווים מהקודקוד לשני הקצוות | קו מהנקודה הראשונה לסמן |
| אידיאלי עבור | זווית פתיחה בין שני מאפיינים | אורך של פער או קטע |

## הזנת קואורדינטות

במקום ללחוץ, תוכלו להקליד מיקום מדויק לכל אחת משלוש הנקודות:

1. הקלידו את ערך X.
2. לחצו `,` — המסוף מציג `[X], [Y{cursor}]`.
3. הקלידו את ערך Y.
4. לחצו **Enter** לאישור.

## מרגע לוח המקשים

| מקש | פעולה |
|-----|--------|
| `0`–`9`, `.`, `-` | התחלת הזנת קואורדינטת X |
| `,` | נעילת X ומעבר להזנת Y |
| `Backspace` | מחיקת התו האחרון שהוקלד |
| `Enter` | אישור הקואורדינטה שהוקלדה |
| `Escape` | ביטול וחזרה לשלב 2 |

## הערות

- התוצאות מוצגות **רק במסוף** — שום דבר אינו מתווסף לשרטוט.
- הדיוק הוא תמיד 4 ספרות אחרי הנקודה במעלות.
