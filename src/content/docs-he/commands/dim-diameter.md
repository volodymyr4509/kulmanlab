# Dimension Diameter

פקודת `dimdiameter` ממקמת מידת קוטר על קשת או מעגל. קו המידה משתרע על פני הקוטר המלא — עובר דרך המרכז בין שתי נקודות קשת מנוגדות דיאמטרלית — ומתויג `⌀ <ערך>`. כדי לתייג רק את הרדיוס מהמרכז לקצה אחד, השתמשו ב-[Dimension Radius](../dim-radius/).

## אנטומיה של מידת קוטר

```
  ●──────────── ⌀ 10.00 ────────────●
  (נקודת קשת רחוקה)              (נקודת קשת קרובה / צד הטקסט)
```

- **קו מידה** — משתרע על פני הקוטר המלא, עם ראשי חץ בשתי נקודות חיתוך הקשת.
- **נקודת קשת קרובה** — נקודת ההיקף בצד הסמן (שם יושבת תווית הטקסט).
- **נקודת קשת רחוקה** — הנקודה המנוגדת דיאמטרלית.
- **תווית** — `⌀` ואחריו ערך הקוטר.

## מיקום מידת קוטר

1. הקלידו `dimdiameter` במסוף או לחצו על כפתור סרגל הכלים **Dimension Diameter**.
2. **לחצו על קשת או מעגל** כדי לבחור בו.
3. **הזיזו את הסמן** כדי לסובב את קו המידה לזווית הרצויה.
4. **לחצו** למיקום המידה.

ניתן לבחור רק ישויות **Arc** ו-**Circle**.

| מקש | פעולה |
|-----|--------|
| `Escape` | ביטול |

## Dimension Diameter מול Dimension Radius

| | Dimension Diameter | Dimension Radius |
|---|-------------------|-----------------|
| מודדת | קוטר מלא (2 × רדיוס) | רדיוס (מרכז לקצה) |
| קו מידה | קצה → קצה דרך המרכז | מרכז → קצה |
| קידומת תווית | `⌀` | `R` |
| ראשי חץ | שניים (בשתי נקודות הקשת) | אחד (בנקודת הקשת) |
| אידיאלי עבור | מידות חור עגול מלא או פיר | תיוג צד אחד של מאפיין מעוקל |

## עריכת התווית — מצב simple

**לחיצה כפולה** על מידת קוטר שהוצבה פותחת את עורך הטקסט במצב **simple**. העורך מאותחל מראש עם הערך המוצג הנוכחי (למשל `⌀ 10.00`) כך שתוכלו למקם את הסמן ולערוך אותו ישירות.

| תכונה | התנהגות |
|---------|-----------|
| מודגש / נטוי / גופן / גובה | חלים על **כל** התווית בבת אחת |
| עיצוב לפי תו | לא נתמך |
| `Enter` | מאשר את הערך וסוגר את העורך |
| מרובת שורות | לא נתמך |

ראו [עורך טקסט — מצב simple](../../interface/text-editor/#simple-mode) להפניה המלאה.

## DXF — ישות מידת קוטר DIMENSION

מידות קוטר נשמרות כישויות `DIMENSION` עם גיאומטריה מסוג קוטר, השומרות גם את מיקומי נקודות הקשת וגם את ערך הקוטר הנמדד (2 × רדיוס). כל המאפיינים נשמרים ללא אובדן בהעברה הדדית.
