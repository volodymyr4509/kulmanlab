# Wipe Storage

הפקודה `wipestorage` מוחקת לצמיתות **את כל הנתונים השמורים בדפדפן** עבור KulmanLab CAD — כל קובץ שמור, טבלאות שכבות וסוגי קו, והיסטוריית ביטול. העמוד נטען מחדש אוטומטית לאחר מכן.

:::danger בלתי הפיך
פעולה זו לא ניתנת לביטול. כל הקבצים השמורים בדפדפן נמחקים. ייצאו כל שרטוט שברצונכם לשמור כקובץ `.json` או `.dxf` לפני הרצת הפקודה הזו.
:::

## מתי להשתמש בזה

- אחסון הדפדפן פגום והאפליקציה נכשלת בטעינה או שמירה של קבצים.
- אתם רוצים לאפס את האפליקציה לחלוטין למצב נקי.
- אתם עוברים בין דפדפנים או מכשירים ואינכם זקוקים עוד לעותק המקומי.

## איך להריץ אותה

1. הקלידו `wipestorage` במסוף ולחצו **Enter**.
2. המסוף מציג הנחיה: *Wipe all browser local storage? Type YES to confirm*
3. הקלידו `YES` (בכל צירוף אותיות רישיות/קטנות) ולחצו **Enter**.

האפליקציה מוחקת את מסד הנתונים וטוענת מחדש את העמוד. אם תקלידו כל דבר אחר מלבד `YES` ותלחצו **Enter**, או תלחצו **Escape**, הפקודה מבוטלת ושום דבר לא נמחק.

## מה נמחק

| נתון | נמחק |
|------|---------|
| כל הקבצים השמורים בדפדפן | כן |
| טבלאות שכבות וסוגי קו לכל קובץ | כן |
| היסטוריית ביטול/ביצוע חוזר לכל קובץ | כן |

רק נתונים שנשמרו מקומית ב**דפדפן זה** מושפעים. קבצים שכבר ייצאתם כ-`.json` או `.dxf` לא נפגעים.
