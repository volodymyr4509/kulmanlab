# Dimension Aligned

פקודת `dimaligned` ממקמת מידה המודדת את **המרחק האמיתי בקו ישר** בין שתי נקודות. קו המידה רץ מקביל לקו המחבר בין שתי הנקודות, כך שהוא יכול להיות בכל זווית. זהו ההבדל המרכזי מ-[Dimension Linear](../dim-linear/), המוגבל לאופקי או אנכי.

## אנטומיה של מידה מיושרת

```
     ●  p2
    /|
   / |  (קו עזר 2, ניצב לקו המידה)
  /  |
 /←5.00→/
/  /
●  /  (קו עזר 1, ניצב לקו המידה)
p1
```

- **קווי עזר** — ניצבים לקו המידה, משורטטים מכל נקודה נמדדת.
- **קו מידה** — מקביל ל-p1→p2, מוזז לצד אחד לפי מיקום הסמן.
- **ערך** — המרחק האוקלידי האמיתי `|p1 – p2|`.

## מיקום מידה מיושרת

1. הקלידו `dimaligned` במסוף או לחצו על כפתור סרגל הכלים **Dimension Aligned**.
2. **לחצו על מקור קו העזר הראשון** (p1), או הקלידו `X,Y` ולחצו **Enter** לקואורדינטה מדויקת.
3. **לחצו על מקור קו העזר השני** (p2). הזנת קואורדינטות עובדת גם כאן.
4. **הזיזו את הסמן** לצד אחד כדי להגדיר את ההיסט הניצב של קו המידה.
5. **לחצו** למיקום, או הקלידו מרחק היסט ולחצו **Enter** למיקום מדויק.

## הזנת מרחק היסט מוקלדת

הקלידו מספר במהלך המיקום כדי לקבע את קו המידה במרחק ניצב מדויק מהקו p1→p2:

| מקש | פעולה |
|-----|--------|
| `0`–`9`, `.` | הוספת ספרה להיסט |
| `Backspace` | מחיקת התו האחרון שהוקלד |
| `Enter` / `Space` | מיקום בהיסט שהוקלד |

צד הסמן קובע באיזה צד קו המידה יופיע.

## מרגע לוח המקשים

| מקש | פעולה |
|-----|--------|
| `0`–`9`, `.`, `-` | התחלת הזנת קואורדינטת X (שלבי p1/p2), או מרחק היסט (שלב מיקום) |
| `,` | נעילת X ומעבר להזנת Y (שלבי p1/p2) |
| `Backspace` | מחיקת התו האחרון שהוקלד |
| `Enter` / `Space` | אישור קואורדינטה או היסט שהוקלדו |
| `Escape` | ביטול |

## Dimension Aligned מול Dimension Linear

| | Dimension Aligned | Dimension Linear |
|---|------------------|-----------------|
| זווית קו המידה | מקביל ל-p1→p2 — כל זווית | תמיד אופקי או אנכי |
| מודדת | מרחק אוקלידי אמיתי | רק רכיב X או Y |
| נעילת כיוון H/V | לא | כן — מקשי `H` ו-`V` |
| אידיאלי עבור | מאפיינים אלכסוניים, חיתוכים בזווית | פריסות אורתוגונליות, חלקים מיושרי-רשת |

## עריכת התווית — מצב simple

**לחיצה כפולה** על מידה מיושרת שהוצבה פותחת את עורך הטקסט במצב **simple**. העורך מאותחל מראש עם הערך המוצג הנוכחי כך שתוכלו למקם את הסמן ולערוך אותו ישירות.

| תכונה | התנהגות |
|---------|-----------|
| מודגש / נטוי / גופן / גובה | חלים על **כל** התווית בבת אחת |
| עיצוב לפי תו | לא נתמך |
| `Enter` | מאשר את הערך וסוגר את העורך |
| מרובת שורות | לא נתמך |

ראו [עורך טקסט — מצב simple](../../interface/text-editor/#simple-mode) להפניה המלאה.

## שרשור מידות

כדי להוסיף עוד מידות הממשיכות מקו העזר השני של זו, השתמשו ב-[Dimension Continue](../dim-continue/) — הוא ננעל לאותה זווית מדידה כמו מידה מיושרת זו.

## DXF — ישות DIMENSION (סוג מיושר)

מידות מיושרות נשמרות כישויות `DIMENSION` עם `dimType = 1` (מיושר). מקורות קווי העזר, מיקום קו המידה, מיקום הטקסט, הערך הנמדד, הסיבוב, סגנון החץ וכל דגלי התצוגה נשמרים ללא אובדן בהעברה הדדית.
