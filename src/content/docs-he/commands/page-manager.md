# PageManager

הפקודה `PageManager` פותחת את דיאלוג הגדרות העמוד עבור הפריסה הפעילה, המאפשר לכם לשנות את פורמט הנייר, הכיוון וקנה המידה בין יחידות שרטוט למילימטרים.

## פתיחת Page Manager

- הקלידו `PageManager` במסוף, **או**
- לחצו לחיצה ימנית על לשונית פריסה בתחתית ובחרו **Page Manager**.

## הגדרות

| הגדרה | תיאור |
|---------|-------------|
| **פורמט נייר** | גודל גיליון תקני (A4, A3, A2, Letter וכו') |
| **כיוון** | לאורך (Portrait) או לרוחב (Landscape) |
| **קנה מידה** | יחידות שרטוט למ"מ — קובע איך קואורדינטות המודל ממופות לנייר |

שינוי גודל הנייר או קנה המידה משנה את הקנה מידה של כל חלונות התצוגה הקיימים בפריסה באופן פרופורציונלי כך שמיקומיהם נשארים נכונים.

## הערות

- PageManager חלה רק על פריסות נייר. היא לא זמינה במרחב המודל.
- כדי להוסיף פריסה חדשה, לחצו על כפתור ה-**+** בשורת הלשוניות בתחתית.
- כדי לשנות שם או למחוק פריסה, לחצו לחיצה ימנית על הלשונית שלה.
