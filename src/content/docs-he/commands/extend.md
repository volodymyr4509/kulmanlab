# Extend

פקודת `extend` מארכת את נקודת הקצה הקרובה ביותר של [Line](../line/) לחיתוך הקרוב ביותר שהיא הייתה יוצרת עם ישות אחרת בשרטוט. רחפו קרוב לנקודת הקצה שברצונכם להאריך — תצוגה מקדימה מציגה את הקו המוארך — ואז לחצו כדי להחיל.

Extend פועלת רק על **ישויות Line**. הגבולות שהקו מתארך לכיוונם יכולים להיות כל סוג ישות אחר מלבד Text, Mtext, Multileader ו-Spline.

## הארכת קו

1. הקלידו `extend` במסוף או לחצו על כפתור סרגל הכלים **Extend**.
2. **רחפו קרוב לאחד מקצוות הקו** — התצוגה המקדימה מציגה את הקו מוארך לגבול הקרוב ביותר בכיוון זה.
3. **לחצו** להחלת ההארכה.

הפקודה נשארת פעילה לאחר כל הארכה כך שתוכלו להאריך מספר קווים ברצף. לחצו **Escape** ליציאה.

```
  לפני:                         אחרי:

  ──────           |           ──────────────|
  (קו קצר)         (גבול)       (מוארך לגבול)
```

## איך נבחרת נקודת הקצה

הפקודה בוחנת לאיזו נקודת קצה הסמן קרוב יותר:

- הסמן **קרוב יותר לנקודת הסיום** → הסיום מתארך קדימה לאורך כיוון הקו.
- הסמן **קרוב יותר לנקודת ההתחלה** → ההתחלה מתארכת אחורה (בכיוון ההפוך).

קרן נורית מנקודת הקצה שנבחרה לאורך כיוון הקו, וה**חיתוך הקרוב ביותר** לאורך אותה קרן עם ישות אחרת (למעט הקו עצמו והסוגים המוחרגים) הופך לנקודת הקצה החדשה.

אם לא נמצא חיתוך בכיוון זה, לא מופיעה תצוגה מקדימה ולחיצה לא עושה דבר.

## החרגות גבול

סוגי הישויות הבאים מוחרגים כגבולות — קו לא מתארך כדי לפגוש אותם:

- Text / Mtext
- Multileader
- Spline

כל שאר הסוגים (Line, Arc, Circle, Ellipse, Polyline, Dimension) משמשים כגבולות תקפים.

## מרגע לוח המקשים

| מקש | פעולה |
|-----|--------|
| `Escape` | יציאה ממצב extend |

## ישויות נתמכות

| ישות | ניתן להאריך? |
|--------|----------------|
| Line | כן |
| Arc, Circle, Ellipse | לא |
| Polyline / Rectangle | לא |
| Text, Spline, Dimension, Leader | לא |
