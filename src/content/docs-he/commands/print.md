# Print

הפקודה `print` פותחת את **Print Manager** — חלון ייצוא ייעודי עם קנבס תצוגה מקדימה חי, בורר פורמט (PNG / JPEG / WebP / PDF), מתג שחור-לבן וחיתוך אזור אופציונלי. שום דבר לא נשלח למדפסת פיזית; הפלט מורד כקובץ.

## פתיחת Print Manager

לחצו על כפתור סרגל הכלים **Print** או הקלידו `print` במסוף. Print Manager נפתח מיידית ומציג תצוגה מקדימה של חלון התצוגה הנוכחי.

## פריסת Print Manager

לחלון יש שני פאנלים:
- **סרגל צד שמאלי** — כל בקרות הייצוא.
- **פאנל ימני** — קנבס תצוגה מקדימה חי המתעדכן בזמן שאתם משנים הגדרות.

### בקרות סרגל הצד

| בקרה | תיאור |
|---------|-------------|
| **Change Area** | חיתוך למלבן מותאם אישית על הקנבס (ראו למטה) |
| מתג **Monochrome** | המרת כל הקווים הצבועים לשחור — פעיל כברירת מחדל לפלט הדפסה נקי |
| תפריט נפתח **Format** | PNG, JPEG, WebP או PDF |
| כפתור **Export** | יצירת הקובץ והורדתו |

## בחירת אזור ייצוא מותאם אישית

כברירת מחדל, התצוגה המקדימה מציגה בדיוק את מה שהיה גלוי על הקנבס כאשר פתחתם את Print Manager. כדי לייצא אזור ספציפי:

1. לחצו **Change Area** — Print Manager מוסתר והקנבס הופך לאינטראקטיבי.
2. **לחצו על הפינה הראשונה** של מלבן הייצוא.
3. **לחצו על הפינה הנגדית** — Print Manager נפתח מחדש עם האזור הנבחר בתצוגה המקדימה.

לחצו `Escape` במהלך בחירת האזור כדי לבטל ולשחזר את האזור הקודם.

קנבס התצוגה המקדימה משנה גודל באופן דינמי כדי להתאים ל**יחס הרוחב-גובה המדויק** של האזור הנבחר, כך שהתצוגה המקדימה מדויקת לפיקסל.

## פורמטי ייצוא

| פורמט | אידיאלי עבור | הערות |
|--------|----------|-------|
| **PNG** | ללא אובדן, קווים חדים | רקע לבן, ללא שקיפות |
| **JPEG** | קובץ קטן יותר לשיתוף | איכות 95%, דחיסה קלה |
| **WebP** | הקובץ הקטן ביותר לאינטרנט | אותה איכות 95%, דחיסה טובה יותר מ-JPEG |
| **PDF** | מסמכים מוכנים להדפסה | תמונה מוטמעת ב-150 DPI בתוך מכולת PDF |

הקובץ המיוצא נקרא `kulman-<timestamp>.<ext>` ומורד אוטומטית.

## רזולוציית ייצוא ורקע

- רזולוציה מקסימלית: **2000 × 2000 פיקסלים**, בקנה מידה פרופורציונלי לאזור הנבחר.
- הרקע תמיד **לבן**.
- שכבות המסומנות כ**לא-מודפסות** מוחרגות מהייצוא.

## מרגע לוח המקשים

| מקש | פעולה |
|-----|--------|
| `Escape` (במהלך בחירת אזור) | ביטול בחירת האזור, שחזור האזור הקודם |
| `Escape` (ב-Print Manager) | סגירת Print Manager |
