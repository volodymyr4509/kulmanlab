# Circle

פקודת `circle` משרטטת מעגל המוגדר על ידי נקודת מרכז ורדיוס. לאחר לחיצה על המרכז תוכלו להגדיר את הרדיוס על ידי לחיצה על נקודה שנייה על הקנבס או על ידי הקלדת מספר מדויק — שתי האפשרויות פעילות בו-זמנית.

## שרטוט מעגל

1. הקלידו `circle` במסוף או לחצו על כפתור סרגל הכלים **Circle**.
2. **לחצו על נקודת המרכז**, או הקלידו `X,Y` ולחצו **Enter** לקואורדינטה מדויקת.
3. הגדירו את הרדיוס — באחת משתי הדרכים:
   - **לחצו על כל נקודה** על הקנבס — המרחק מהמרכז הופך לרדיוס, או
   - **הקלידו את הרדיוס** ולחצו **Enter** לערך מדויק.

המעגל ממוקם מיידית והפקודה יוצאת.

```
  מרכז ●
          \  תצוגה מקדימה של קו רדיוס
           \
            ● ← לחצו כאן, או הקלידו מספר
```

בזמן שלב הרדיוס, התצוגה המקדימה החיה מציגה את המעגל במרחק הסמן הנוכחי וגם מציירת קו רדיוס מהמרכז לנקודה הנוכחית.

## הזנת קואורדינטת מרכז

במקום ללחוץ, תוכלו להקליד את מיקום המרכז:

1. הקלידו את ערך X.
2. לחצו `,` — המסוף מציג `[X], [Y{cursor}]`.
3. הקלידו את ערך Y.
4. לחצו **Enter** למיקום המרכז ולהתקדם להזנת הרדיוס.

## הזנת רדיוס מוקלדת

לאחר שהמרכז הוצב, הקלדה בונה מיד ערך רדיוס:

| מקש | פעולה |
|-----|--------|
| `0`–`9`, `.` | הוספת ספרה לערך הרדיוס |
| `Backspace` | מחיקת התו האחרון שהוקלד |
| `Enter` | מיקום המעגל ברדיוס שהוקלד |

הערך הצבור מוצג בהנחיית המסוף (למשל `enter radius of circle: 25`). התצוגה המקדימה מתעדכנת כדי להציג את הרדיוס שהוקלד בזמן שהסמן שולט בכיוון סמן קו הרדיוס.

## מרגע לוח המקשים

| מקש | פעולה |
|-----|--------|
| `0`–`9`, `.`, `-` | התחלת הזנת קואורדינטת X (שלב מרכז), או ספרת רדיוס (שלב רדיוס) |
| `,` | נעילת X ומעבר להזנת Y (שלב מרכז) |
| `Backspace` | מחיקת התו האחרון שהוקלד |
| `Enter` | אישור קואורדינטה או רדיוס שהוקלדו |
| `Escape` | ביטול ואיפוס |

## עריכת נקודות אחיזה — שינוי גודל הרדיוס

מעגל נבחר חושף חמש נקודות אחיזה:

| נקודת אחיזה | מיקום | מה היא עושה |
|------|----------|--------------|
| **מרכז** | נקודת המרכז | מזיזה את כל המעגל; הרדיוס נשאר ללא שינוי |
| **שמאל** | הנקודה השמאלית ביותר (מרכז − רדיוס) | גררו כדי להגדיר רדיוס חדש = מרחק למרכז |
| **ימין** | הנקודה הימנית ביותר (מרכז + רדיוס) | גררו כדי להגדיר רדיוס חדש = מרחק למרכז |
| **עליון** | הנקודה העליונה ביותר | גררו כדי להגדיר רדיוס חדש = מרחק למרכז |
| **תחתון** | הנקודה התחתונה ביותר | גררו כדי להגדיר רדיוס חדש = מרחק למרכז |

כל ארבע נקודות האחיזה הקרדינליות מתנהגות באופן זהה — הרדיוס החדש שווה למרחק מהמרכז למיקום הגרירה. המרכז נשאר קבוע.

## בחירת מעגלים

| שיטה | התנהגות |
|--------|-----------|
| **לחיצה** | בוחרת אם הלחיצה נופלת קרוב להיקף |
| **גרירה ימינה** (קפדנית) | כל הריבוע התוחם (מרכז ± רדיוס) חייב להימצא בתוך התיבה |
| **גרירה שמאלה** (חוצה) | כל חלק מההיקף שחוצה או נוגע בגבול התיבה בוחר את המעגל |

## Circle מול Arc — מתי להשתמש בכל אחת

| | Circle | Arc |
|---|--------|-----|
| טווח | 360° מלא | חלקי — מוגדר על ידי זווית התחלה וסוף |
| איך לשרטט | מרכז + רדיוס | שלוש נקודות על העקומה |
| קלט מוקלד | ערך רדיוס | אין — לחיצה בלבד |
| נקודת אחיזה לשינוי גודל | 4 נקודות קרדינליות | נקודות התחלה וסוף (זווית + רדיוס) |
| מידוד | רדיוס: [Dim Radius](../dim-radius/) · קוטר: [Dim Diameter](../dim-diameter/) | [Dim Radius](../dim-radius/) |
| אידיאלי עבור | חורים מלאים, מעגלי ברגים, מאפיינים עגולים | פינות מעוגלות, עקומות חלקיות, נתיבים קמורים |

## DXF — ישות CIRCLE

מעגלים נשמרים כישויות `CIRCLE` בקובץ ה-DXF. קואורדינטות מרכז, רדיוס, צבע, שכבה, סוג קו, קנה מידה של סוג קו ועובי — כולם נשמרים ללא אובדן בהעברה הדדית. כל יישום התומך ב-DXF קורא אותם כמעגלים תקניים.
