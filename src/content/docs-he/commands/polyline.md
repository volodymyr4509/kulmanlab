# Polyline

פקודת `polyline` משרטטת נתיב מחובר של כל מספר של קטעים ישרים, כולם נשמרים כישות `LWPOLYLINE` אחת. מכיוון שכל הנתיב הוא אובייקט אחד, בחירתו בוחרת כל קטע בבת אחת — הזיזו, סובבו או שנו קנה מידה של כל הצורה בפעולה אחת. זהו ההבדל המרכזי מקווים משורשרים ([Lines](../line/)), שבהם כל קטע הוא ישות עצמאית.

Polylines יכולות להיות גם **סגורות**: פקודת [Rectangle](../rectangle/) משתמשת באותה ישות `LWPOLYLINE` עם דגל סגירה מוגדר.

## שרטוט polyline

1. הקלידו `polyline` במסוף או לחצו על כפתור סרגל הכלים **Polyline**.
2. **לחצו על הנקודה הראשונה**, או הקלידו `X,Y` ולחצו **Enter** לקואורדינטה מדויקת.
3. **לחצו על כל נקודה עוקבת** — כל לחיצה מוסיפה קטע. הזנת קואורדינטות עובדת בכל שלב.
4. לחצו **Enter** או **Space** כדי לסיים (דורש לפחות 2 נקודות שהוצבו).

```
  ●──────●
  1ה    2ה
          \
           \  קטע 3 (בתהליך — סמן כאן)
            ●  ← לחיצה להוספה, Enter/Space לסיום
```

לחיצה על **Escape** בכל עת מבטלת את כל הנקודות שהוצבו ויוצאת מהפקודה.

## הזנת קואורדינטות

במקום ללחוץ, הקלידו מיקום מדויק לכל קודקוד:

1. הקלידו את ערך X.
2. לחצו `,` — המסוף מציג `[X], [Y{cursor}]`.
3. הקלידו את ערך Y.
4. לחצו **Enter** למיקום הקודקוד.

## נעילת זווית ואורך קטע מדויק

אותה לוגיקת נעילה של 45° כמו פקודת [Line](../line/#angle-locking-and-exact-length-input) חלה בין כל שתי נקודות עוקבות. בעת נעילה לציר:

| מקש | פעולה |
|-----|--------|
| `0`–`9`, `.` | הוספת ספרה לאורך הקטע |
| `-` | אורך שלילי — הופך כיוון לאורך הציר (רק כתו הראשון) |
| `Backspace` | מחיקת התו האחרון שהוקלד |
| `Enter` | מיקום הנקודה הבאה במרחק שהוקלד |

האורך הצבור הנוכחי מופיע בהנחיית המסוף בזמן אמת. לחיצה בעת נעילה מקרינה על הציר כך שהקודקוד החדש נוחת עליו בדיוק.

## מרגע לוח המקשים

| מקש | פעולה |
|-----|--------|
| `0`–`9`, `.`, `-` | התחלת הזנת קואורדינטת X, או אורך קטע בעת נעילת זווית |
| `,` | נעילת X ומעבר להזנת Y |
| `Backspace` | מחיקת התו האחרון שהוקלד |
| `Enter` | אישור קואורדינטה או אורך שהוקלדו, או סיום הפוליליין אם לא הוקלד דבר ויש ≥ 2 נקודות |
| `Space` | סיום הפוליליין (זהה ל-Enter כשאין קלט בתהליך) |
| `Escape` | ביטול כל הנקודות ויציאה |

## עריכת נקודות אחיזה — קודקודים ואמצעי-קטעים

פוליליין נבחר מציג שני סוגי נקודות אחיזה:

| נקודת אחיזה | מיקום | מה היא עושה |
|------|----------|--------------|
| **קודקוד** | בכל נקודה שהוצבה | גררו כדי למקם מחדש את הקודקוד; כל הקטעים המחוברים נמתחים בעקבותיו |
| **אמצע קטע** | מרכז כל קטע | גררו כדי להזיז **שתי** נקודות הקצה של הקטע יחד, תוך שמירה על אורך וזווית הקטע |

נקודת האחיזה של אמצע הקטע ייחודית לפוליליינים — היא מאפשרת לכם להחליק קטע בודד הצידה בלי לשנות את אורכו. ב-[Line](../line/), נקודת האחיזה האמצעית מפעילה במקום זאת את פקודת Move לכל הישות.

אין נקודת אחיזה בודדת "להזיז את כל הפוליליין". כדי להזיז את כל הנתיב השתמשו בפקודת [Move](../move/).

## בחירת פוליליינים

| שיטה | התנהגות |
|--------|-----------|
| **לחיצה** | בוחרת את הפוליליין אם הלחיצה נמצאת בטווח בדיקת פגיעה מכל קטע |
| **גרירה ימינה** (קפדנית) | כל הקודקודים חייבים ליפול בתוך התיבה |
| **גרירה שמאלה** (חוצה) | כל קטע שחוצה את גבול התיבה בוחר את כל הפוליליין |

מכיוון שפוליליין הוא ישות אחת, בחירה חוצה שנוגעת בכל קטע בוחרת את כל הקטעים.

## פקודות עריכה נתמכות

Polylines תומכות בכל הטרנספורמציות הכלליות ו-offset, אך **לא** trim או extend (אלה עבור [Line](../line/) בלבד):

| פקודה | מה קורה לפוליליין |
|---------|------------------------------|
| [Move](../move/) | מזיזה את כל הקודקודים באותה הזזה |
| [Copy](../copy/) | יוצרת פוליליין זהה במיקום חדש |
| [Rotate](../rotate/) | מסובבת את כל הקודקודים סביב נקודת הבסיס הנבחרת |
| [Mirror](../mirror/) | משקפת את כל הקודקודים על פני ציר השיקוף |
| [Scale](../scale/) | משנה קנה מידה של כל הקודקודים באופן אחיד מנקודת הבסיס |
| [Offset](../offset/) | יוצרת פוליליין מקביל במרחק ניצב קבוע |
| [Delete](../delete/) | מסירה את הפוליליין מהשרטוט |

## מאפיינים

כאשר פוליליין נבחר, פאנל המאפיינים מציג:

**כללי**

| מאפיין | ברירת מחדל | משמעות |
|----------|---------|---------|
| צבע | 256 (ByLayer) | אינדקס צבע ACI |

**גיאומטריה**

| מאפיין | משמעות |
|----------|---------|
| Closed | האם הקודקוד האחרון מתחבר בחזרה לראשון |

| Vertices | רשימת קואורדינטות של כל הקודקודים |

## Polyline מול Line — מתי להשתמש בכל אחת

| | Polyline | Line |
|---|---------|------|
| מספר ישויות | `LWPOLYLINE` אחד לכל הנתיב | `LINE` אחד לכל קטע |
| צורה סגורה | כן (דגל סגירה) | לא |
| Trim / Extend | לא | כן — קטע אחר קטע |
| נקודת אחיזה אמצע-קטע | מזיזה את כל הקטע | מפעילה Move לישות |
| אידיאלי עבור | מתארים, קווי מתאר, צורות שתשמרו שלמות | קווי בנייה, גיאומטריה שתגזרו |

## DXF — ישות LWPOLYLINE

Polylines נשמרות כישויות `LWPOLYLINE` בקובץ ה-DXF. כל המאפיינים — קואורדינטות קודקוד, דגל סגירה, צבע, שכבה, סוג קו, קנה מידה של סוג קו ועובי — נשמרים ללא אובדן בהעברה הדדית. Rectangles המשורטטים עם פקודת [Rectangle](../rectangle/) נשמרים גם הם כ-`LWPOLYLINE` (סגור, ארבעה קודקודים) ולא ניתנים להבחנה ברמת ה-DXF.

ישויות `LWPOLYLINE` מכל יישום התומך ב-DXF (LibreCAD, FreeCAD וכו') נקראות בחזרה כפוליליינים הניתנים לעריכה מלאה בעורך.
