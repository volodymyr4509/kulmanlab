# Scale

פקודת `scale` משנה את גודלן של ישויות נבחרות באופן אחיד סביב נקודת בסיס. כל המרחקים מנקודת הבסיס מוכפלים בגורם קנה המידה — גורם של `2` מכפיל את כל המידות פי שניים, `0.5` מחצה אותן. הגורם תמיד מוזן על ידי הקלדה; אין לחיצה להגדרת קנה מידה.

## שתי דרכים להתחיל

**בחירה מראש, ואז שינוי קנה מידה** — בחרו ישויות תחילה, ואז הפעילו:

1. בחרו ישות אחת או יותר על הקנבס.
2. הקלידו `scale` במסוף או לחצו על כפתור סרגל הכלים **Scale**.
3. **לחצו על נקודת הבסיס** — הנקודה הקבועה שלא זזה בזמן שינוי קנה המידה. או הקלידו `X,Y` ולחצו **Enter** לקואורדינטה מדויקת.
4. **הקלידו את גורם קנה המידה** ולחצו **Enter**.

**הפעלה, ואז בחירה** — התחילו את הפקודה ללא בחירה:

1. הקלידו `scale` או לחצו על כפתור סרגל הכלים.
2. **בחרו אובייקטים** — לחצו כדי לשנות בחירה, או גררו לבחירה לפי אזור.
3. לחצו **Enter** או **Space** כדי לאשר את הבחירה.
4. **לחצו על נקודת הבסיס** (הזנת קואורדינטות זמינה), ואז הקלידו את הגורם.

```
  בסיס ●                בסיס ●
        [ישות]   →          [ישות גדולה יותר]
  גורם = 2 → מרחקים מ-● מוכפלים פי שניים
```

## הקלדת גורם קנה המידה

לאחר שנקודת הבסיס הוצבה, המסוף מציג `enter scale factor:` וממתין לקלט מקלדת:

| מקש | פעולה |
|-----|--------|
| `0`–`9`, `.` | הוספת ספרה לגורם |
| `-` | גורם שלילי (רק כתו הראשון — הופך ואז משנה קנה מידה) |
| `Backspace` | מחיקת התו האחרון שהוקלד |
| `Enter` | החלת שינוי קנה המידה בגורם שהוקלד |

הגורם חייב להיות שונה מאפס. ערכים נפוצים:

| גורם | אפקט |
|--------|--------|
| `2` | מכפיל את כל המידות פי שניים |
| `0.5` | מחצה את כל המידות |
| `1.5` | מגדיל ב-50% |
| `-1` | משקף על פני נקודת הבסיס (שווה לסיבוב של 180°) |

אין תצוגה מקדימה חיה בזמן ההקלדה — התוצאה בקנה המידה החדש מופיעה רק לאחר לחיצה על **Enter**.

## מרגע לוח המקשים

| מקש | פעולה |
|-----|--------|
| `Enter` / `Space` | אישור הבחירה |
| `0`–`9`, `.`, `-` | התחלת הזנת קואורדינטת X (שלב נקודת בסיס), או גורם קנה מידה (שלב גורם) |
| `,` | נעילת X ומעבר להזנת Y (שלב נקודת בסיס) |
| `Backspace` | מחיקת התו האחרון שהוקלד |
| `Enter` | אישור קואורדינטה או החלת קנה מידה |
| `Escape` | ביטול ואיפוס |

## בחירה במהלך הפקודה

| שיטה | התנהגות |
|--------|-----------|
| **לחיצה** | משנה בחירה של הישות מתחת לסמן |
| **גרירה ימינה** (קפדנית) | מוסיפה ישויות שנמצאות במלואן בתוך התיבה |
| **גרירה שמאלה** (חוצה) | מוסיפה ישויות שחותכות את התיבה |
| **Enter** / **Space** | מאשר את הבחירה |

## מה משנה קנה מידה

כל סוגי הישויות נתמכים. כל ישות משנה קנה מידה של הגיאומטריה שלה ביחס לנקודת הבסיס:

| ישות | מה משתנה |
|--------|-------------|
| Line | שתי נקודות הקצה משנות קנה מידה מנקודת הבסיס |
| Circle | המרכז משנה קנה מידה מנקודת הבסיס; הרדיוס מוכפל בגורם |
| Arc | המרכז משנה קנה מידה; הרדיוס מוכפל בגורם; הזוויות ללא שינוי |
| Ellipse | המרכז משנה קנה מידה; שני אורכי חצאי-הצירים מוכפלים בגורם |
| Polyline / Rectangle | כל קודקוד משנה קנה מידה מנקודת הבסיס |
| Text | נקודת העיגון משנה קנה מידה; הגובה מוכפל בגורם |
| Spline | כל נקודות הבקרה / נקודות ההתאמה משנות קנה מידה |
