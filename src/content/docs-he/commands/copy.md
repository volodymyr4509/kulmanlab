# Copy

פקודת `copy` יוצרת עותקים מוזזים של ישויות נבחרות וממקמת אותם עם היסט מנקודת בסיס ליעד — המקוריים נשארים בדיוק במקום שהם נמצאים. זהו ההבדל המרכזי היחיד מ-[Move](../move/): Copy מוסיפה ישויות חדשות לשרטוט; Move מזיזה ישויות קיימות.

## שתי דרכים להתחיל

**בחירה מראש, ואז העתקה** — בחרו ישויות תחילה, ואז הפעילו:

1. בחרו ישות אחת או יותר על הקנבס.
2. הקלידו `copy` במסוף או לחצו על כפתור סרגל הכלים **Copy**.
3. **לחצו על נקודת הבסיס**, או הקלידו `X,Y` ולחצו **Enter** לקואורדינטה מדויקת.
4. **לחצו על היעד** — עותקים מופיעים בהיסט בסיס→יעד. הזנת קואורדינטות עובדת גם כאן.

**הפעלה, ואז בחירה** — התחילו את הפקודה ללא בחירה:

1. הקלידו `copy` או לחצו על כפתור סרגל הכלים.
2. **בחרו אובייקטים** — לחצו כדי לשנות בחירה לכל ישות, או גררו לבחירה לפי אזור.
3. לחצו **Enter** או **Space** כדי לאשר את הבחירה.
4. **לחצו על נקודת הבסיס**, ולאחר מכן **לחצו על היעד** (הזנת קואורדינטות זמינה בשני השלבים).

```
  לפני:                  אחרי:
  [ישות A]               [ישות A]  ← מקוריים ללא שינוי
  [ישות B]               [ישות B]
                         [עותק של A] ← ישויות חדשות
                         [עותק של B]
```

תצוגה מקדימה רפאית של העותקים עוקבת אחר הסמן מנקודת הבסיס ליעד.

## הזנת קואורדינטות

בשלב נקודת הבסיס או היעד, הקלידו מיקום מדויק במקום ללחוץ:

1. הקלידו את ערך X.
2. לחצו `,` — המסוף מציג `[X], [Y{cursor}]`.
3. הקלידו את ערך Y.
4. לחצו **Enter** לאישור.

## נעילת זווית ומרחק מדויק

לאחר שנקודת הבסיס הוגדרה, הפקודה נצמדת לצירי 45° (0°, 45°, 90°, 135°, …) כאשר הסמן רחוק מספיק וקרוב לציר. בעת נעילה, הקלידו מרחק ולחצו **Enter** כדי למקם את העותקים בהיסט המדויק הזה.

| מקש | פעולה |
|-----|--------|
| `0`–`9`, `.` | הוספה לערך המרחק |
| `-` | מרחק שלילי — הופך כיוון לאורך הציר (רק כתו הראשון) |
| `Backspace` | מחיקת התו האחרון שהוקלד |
| `Enter` | מיקום העותקים במרחק שהוקלד |

## מרגע לוח המקשים

| מקש | פעולה |
|-----|--------|
| `Enter` / `Space` | אישור הבחירה והמעבר לשלב נקודת הבסיס |
| `0`–`9`, `.`, `-` | התחלת הזנת קואורדינטת X, או מרחק בעת נעילת זווית |
| `,` | נעילת X ומעבר להזנת Y |
| `Backspace` | מחיקת התו האחרון שהוקלד |
| `Enter` | אישור קואורדינטה או החלת העתקה במרחק שהוקלד |
| `Escape` | ביטול ואיפוס |

## בחירה במהלך הפקודה

| שיטה | התנהגות |
|--------|-----------|
| **לחיצה** | משנה בחירה של הישות מתחת לסמן |
| **גרירה ימינה** (קפדנית) | מוסיפה ישויות שנמצאות במלואן בתוך התיבה |
| **גרירה שמאלה** (חוצה) | מוסיפה ישויות שחותכות את גבול התיבה |
| **Enter** / **Space** | מאשר את הבחירה |

## לאחר ההעתקה

**המקוריים נשארים נבחרים** — העותקים החדשים מתווספים לשרטוט אך הבחירה מתנקה והפקודה יוצאת. כדי לעבוד עם העותקים מיידית, הריצו Copy שוב על הבחירה, או התחילו פקודה חדשה.

## Copy מול Move

| | Copy | Move |
|---|------|------|
| מקוריים | נשארים במקומם | מוסרים מהמיקום המקורי |
| ספירת תוצאה | גדלה לפי מספר הישויות שהועתקו | ללא שינוי |
| לאחר הפעולה | המקוריים עדיין נבחרים | ישויות מוזזות נבחרות במיקום החדש |
| אידיאלי עבור | חזרה על גיאומטריה, פריסות סימטריות | מיקום מחדש של גיאומטריה |

## ישויות נתמכות

Copy פועלת על כל סוג ישות. כל הישויות מיישמות `translate(dx, dy)` באופן פנימי כך שאף אחת אינה מוחרגת.
