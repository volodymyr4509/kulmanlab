# Dimension Radius

פקודת `dimradius` ממקמת מידת רדיוס על קשת או מעגל. קו המידה רץ מהמרכז לנקודה על ההיקף בכיוון הסמן, מתויג `R <ערך>`. כדי לתייג קוטר מלא במקום זאת, השתמשו ב-[Dimension Diameter](../dim-diameter/).

## אנטומיה של מידת רדיוס

```
  ● (מרכז)
   \
    \  R 5.00
     \
      ●────── טקסט (צד הסמן)
   (נקודת קשת)
```

- **קו מידה** — מהמרכז דרך נקודת קשת לכיוון הסמן, עם ראש חץ בקשת.
- **תווית** — `R` ואחריו ערך הרדיוס.

## מיקום מידת רדיוס

1. הקלידו `dimradius` במסוף או לחצו על כפתור סרגל הכלים **Dimension Radius**.
2. **לחצו על קשת או מעגל** כדי לבחור בו.
3. **הזיזו את הסמן** כדי לכוון את קו המידה — נקודת הקשת עוקבת אחר כיוון הסמן מהמרכז.
4. **לחצו** למיקום המידה.

ניתן לבחור רק ישויות **Arc** ו-**Circle**. לחיצה על כל סוג ישות אחר לא עושה דבר.

## Dimension Radius מול Dimension Diameter

| | Dimension Radius | Dimension Diameter |
|---|-----------------|-------------------|
| מודדת | רדיוס (מרכז לקצה) | קוטר (קצה לקצה דרך המרכז) |
| קו מידה | מרכז → נקודת קשת | נקודת קשת → נקודת קשת (דרך המרכז) |
| קידומת תווית | `R` | `⌀` |
| ראשי חץ | אחד (בנקודת הקשת) | שניים (בשתי נקודות הקשת) |
| אידיאלי עבור | תיוג צד אחד של מאפיין מעוקל | תיוג מידות מעגליות מלאות |

## עריכת התווית — מצב simple

**לחיצה כפולה** על מידת רדיוס שהוצבה פותחת את עורך הטקסט במצב **simple**. העורך מאותחל מראש עם הערך המוצג הנוכחי (למשל `R 5.00`) כך שתוכלו למקם את הסמן ולערוך אותו ישירות.

| תכונה | התנהגות |
|---------|-----------|
| מודגש / נטוי / גופן / גובה | חלים על **כל** התווית בבת אחת |
| עיצוב לפי תו | לא נתמך |
| `Enter` | מאשר את הערך וסוגר את העורך |
| מרובת שורות | לא נתמך |

ראו [עורך טקסט — מצב simple](../../interface/text-editor/#simple-mode) להפניה המלאה.

## DXF — ישות מידת רדיוס DIMENSION

מידות רדיוס נשמרות כישויות `DIMENSION` עם גיאומטריה מסוג רדיוס, השומרות את קואורדינטות המרכז, מיקום נקודת הקשת וערך הרדיוס הנמדד. כל המאפיינים נשמרים ללא אובדן בהעברה הדדית.
