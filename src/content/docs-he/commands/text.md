# Text

פקודת `text` ממקמת תווית טקסט מרובת-שורות. לאחר שאתם לוחצים על מיקום בקנבס, עורך קופץ נפתח במצב **rich** — תוכלו להקליד תוכן, להחיל מודגש/נטוי/קו-חוצה לפי תו, לשנות גופנים וגבהים, ולהכניס מעברי שורה. לחצו **Escape** כדי לאשר ולסגור את העורך.

ראו את עמוד [עורך הטקסט](../../interface/text-editor/) להפניה המלאה של העורך, כולל השוואה בין מצבי **rich** ו-**simple**.

## מיקום תווית טקסט

1. הקלידו `text` במסוף או לחצו על כפתור סרגל הכלים **Text**.
2. **לחצו על מיקום העיגון** על הקנבס. או הקלידו `X,Y` ולחצו **Enter** לקואורדינטה מדויקת.
3. **חלון עורך הטקסט הקופץ** נפתח מעל התווית החדשה. הקלידו את התוכן שלכם.
4. לחצו **Escape** כדי לאשר את התווית ולסגור את העורך.

הגובה ברירת המחדל הוא **12 יחידות שרטוט**.

## עריכת תווית קיימת

**לחיצה כפולה** על כל תווית טקסט על הקנבס פותחת מחדש את העורך עבור אותה תווית.

## הזנת קואורדינטות לעוגן

במקום ללחוץ, הקלידו מיקום מדויק:

1. הקלידו את ערך X.
2. לחצו `,` — המסוף מציג `[X], [Y{cursor}]`.
3. הקלידו את ערך Y.
4. לחצו **Enter** למיקום העוגן ופתיחת העורך.

## מרגע לוח המקשים

**שלב העוגן**

| מקש | פעולה |
|-----|--------|
| `0`–`9`, `.`, `-` | התחלת הזנת קואורדינטת X |
| `,` | נעילת X ומעבר להזנת Y |
| `Backspace` | מחיקת התו האחרון שהוקלד |
| `Enter` | אישור הקואורדינטה שהוקלדה |

**שלב עורך הטקסט** (ראו [עורך הטקסט](../../interface/text-editor/) להפניה המלאה)

| מקש | פעולה |
|-----|--------|
| כל תו הניתן להדפסה | הוספה בסמן |
| `Backspace` / `Delete` | מחיקת התו הסמוך או הבחירה |
| `Enter` | הוספת מעבר שורה |
| `←` / `→` | הזזת הסמן |
| `Home` / `End` | קפיצה להתחלה / סוף שורה קשיחה |
| `Escape` | אישור וסגירת העורך |

## עריכת נקודות אחיזה — מיקום מחדש

תווית טקסט נבחרת חושפת נקודת אחיזה אחת בנקודת העיגון:

| נקודת אחיזה | מיקום | מה היא עושה |
|------|----------|--------------|
| **עוגן** | שמאל-תחתון של הטקסט | גררו כדי למקם מחדש את התווית |

## בחירת טקסט

| שיטה | התנהגות |
|--------|-----------|
| **לחיצה** | בוחרת אם הלחיצה נופלת בתוך תיבת התיחום המסובבת של הטקסט |
| **גרירה ימינה** (קפדנית) | כל ארבע פינות תיבת התיחום חייבות ליפול בתוך אזור הבחירה |
| **גרירה שמאלה** (חוצה) | כל חפיפה בין תיבת התיחום של הטקסט לאזור הבחירה בוחרת אותה |

## פקודות עריכה נתמכות

| פקודה | מה קורה לטקסט |
|---------|--------------------------|
| [Move](../move/) | מזיזה את נקודת העיגון |
| [Copy](../copy/) | יוצרת תווית זהה במיקום חדש |
| [Rotate](../rotate/) | מסובבת את מיקום העיגון ומוסיפה את הזווית ל-Rotation Degree |
| [Mirror](../mirror/) | משקפת את נקודת העיגון על פני ציר השיקוף (מחרוזת הטקסט לא הופכת) |
| [Scale](../scale/) | משנה קנה מידה של מיקום העיגון ומכפילה את הגובה בגורם קנה המידה |
| [Delete](../delete/) | מסירה את התווית |

Text אינה תומכת ב-**Offset**, **Trim** או **Extend**.

## מאפיינים

כאשר תווית טקסט נבחרת, פאנל המאפיינים מציג:

**כללי**

| מאפיין | ברירת מחדל | משמעות |
|----------|---------|---------|
| צבע | 256 (ByLayer) | אינדקס צבע ACI |
| שכבה | `0` | שיוך שכבה |

**גיאומטריה**

| מאפיין | משמעות |
|----------|---------|
| Position X / Position Y | קואורדינטות נקודת העיגון |
| Height | גובה טקסט בסיסי ביחידות שרטוט (ברירת מחדל: **12**) |
| Rotation Degree | סיבוב נגד כיוון השעון במעלות |

**מאפיינים**

| מאפיין | משמעות |
|----------|---------|
| Content | מחרוזת הטקסט (קודי MTEXT מוטבעים נשמרים) |
| Attachment Point | קוד יישור (1 = שמאל-עליון … 9 = ימין-תחתון) |

לטקסט אין מאפייני Linetype, Linetype Scale או Thickness.

## DXF — ישות MTEXT

תוויות טקסט נשמרות כישויות **MTEXT** בקובץ ה-DXF. מודגש ונטוי מקודדים באמצעות `\L`, `\K`, `\O` והחלפות גופן מוטבעות (`\f`). גובה לפי תו מקודד כ-`\H`. כל העיצוב שורד העברה הדדית מלאה של DXF וקריא על ידי LibreCAD, FreeCAD ויישומים אחרים התומכים ב-DXF.
