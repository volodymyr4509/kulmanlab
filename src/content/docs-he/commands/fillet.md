# Fillet

פקודת `fillet` מעגלת את הפינה בין שתי ישויות [Line](../line/) על ידי הכנסת קשת משיקה ברדיוס נתון וגזירת כל קו בחזרה לנקודה שבה הקשת מתחילה. התוצאה היא פינה חלקה ומעוגלת המחברת בין שני הקווים.

Fillet פועלת רק על **ישויות Line**.

## שימוש ב-Fillet

1. הקלידו `fillet` במסוף או לחצו על כפתור סרגל הכלים **Fillet**.
2. **הקלידו את רדיוס ה-fillet** ולחצו **Enter**.
3. **לחצו על הקו הראשון** — החלק שאתם לוחצים עליו קובע איזה צד של כל חיתוך נשמר.
4. **רחפו מעל הקו השני** — תצוגה מקדימה של קשת מקווקוות מציגה את ה-fillet שינבע. הזיזו את הסמן לצד שברצונכם לשמור.
5. **לחצו** כדי להחיל. שני הקווים נגזרים והקשת מוכנסת.

```
  לפני:                        אחרי fillet (רדיוס r):

  ──────────────              ──────────╮
                │                        ╰────
                │
```

## בחירת צד לקווים מצטלבים

כאשר שני קווים חוצים זה את זה, ה-fillet מוחל על הפינה המוגדרת על ידי מיקומי הלחיצה — החלק של כל קו ב**אותו צד כמו הסמן** נשמר.

- לחצו קרוב לאחד הקצוות של הקו הראשון כדי לבחור את החצי ההוא.
- הזיזו את הסמן לחצי הרצוי של הקו השני — התצוגה המקדימה המקווקוות מתעדכנת בזמן אמת.

## מה הפקודה יוצרת

- נקודת הקצה של הקו הראשון הקרובה ביותר לחיתוך מוזזת לנקודת ההשקה **T1**.
- נקודת הקצה של הקו השני הקרובה ביותר לחיתוך מוזזת לנקודת ההשקה **T2**.
- ישות Arc חדשה מוכנסת מ-**T1** ל-**T2**, משיקה לשני הקווים.

הקשת המוכנסת יורשת את הגדרות עובי הקו, הצבע, השכבה וסוג הקו הנוכחיים.

## מרגע לוח המקשים

| מקש | פעולה |
|-----|--------|
| `0`–`9`, `.` | הוספת ספרה לערך הרדיוס |
| `Backspace` | מחיקת התו האחרון שהוקלד |
| `Enter` | אישור הרדיוס שהוקלד ומעבר לבחירת קו |
| `Escape` | ביטול ואיפוס |

## ישויות נתמכות

| ישות | נתמך |
|--------|-----------|
| Line | כן — כישות ראשונה ושנייה |
| Arc, Circle, Ellipse, Polyline | לא |
| Text, Spline, Dimension, Leader | לא |
