# Offset

פקודת `offset` יוצרת עותק מקביל של ישות במרחק ניצב קבוע. אתם מקלידים את המרחק פעם אחת, ואז לוחצים על ישויות ובוחרים צד — הפקודה נשארת מוכנה באותו מרחק כך שתוכלו לעשות offset למספר אובייקטים בישיבה אחת.

סוגי ישויות נתמכים: **Line, Circle, Arc, Ellipse, Polyline** (כולל Rectangles).

## שימוש ב-offset

1. הקלידו `offset` במסוף או לחצו על כפתור סרגל הכלים **Offset**.
2. **הקלידו את מרחק ה-offset** ולחצו **Enter** או **Space**.
3. **לחצו על ישות** לביצוע offset — אם הישות אינה מסוג נתמך, הודעת שגיאה מופיעה ותוכלו ללחוץ על ישות אחרת.
4. **הזיזו את הסמן** לצד שבו העותק צריך להופיע — תצוגה מקדימה חיה עוקבת.
5. **לחצו** למיקום עותק ה-offset.

לאחר כל מיקום הפקודה חוזרת לשלב 3 באותו **מרחק**, מוכנה ל-offset הבא. לחצו **Escape** לאיפוס בחזרה לשלב הזנת המרחק.

```
  מרחק: 10

  ─────────────────    ← קו מקורי
  ─────────────────    ← עותק offset (10 יחידות למטה)
```

## התנהגות offset לפי סוג ישות

| ישות | איך ה-offset מחושב |
|--------|---------------------------|
| **Line** | קו מקביל מוזז ניצב לכיוון המקורי |
| **Circle** | מעגל קונצנטרי; לחיצה בחוץ → רדיוס גדול יותר, בפנים → רדיוס קטן יותר |
| **Arc** | קשת קונצנטרית ברדיוס חדש; אותו טווח זווית נשמר |
| **Ellipse** | שני חצאי-הצירים גדלים או קטנים באותו מרחק |
| **Polyline** | כל קטע מקבל offset באופן עצמאי; קטעי offset סמוכים משופעים בפינות |

עבור **Circle**, **Arc** ו-**Ellipse**: אם ה-offset הפנימי היה מקטין רדיוס או חצי-ציר לאפס או מתחת, ה-offset לא מוחל.

## מרגע לוח המקשים

| מקש | פעולה |
|-----|--------|
| `0`–`9`, `.` | הוספת ספרה לערך המרחק |
| `Backspace` | מחיקת התו האחרון שהוקלד |
| `Enter` / `Space` | אישור המרחק שהוקלד והמעבר לבחירת ישות |
| `Escape` | איפוס בחזרה לשלב הזנת המרחק |

## הערת תהליך עבודה

המרחק נשאר מוגדר עד שתלחצו **Escape**. זה הופך את זה ליעיל לעשות offset למספר ישויות באותו מרווח — הקלידו את המרחק פעם אחת, ואז לחצו ובחרו צד לכל ישות בתורה.

## Offset מול Copy

| | Offset | Copy |
|---|--------|------|
| הזזה | ניצב לגיאומטריה של הישות | וקטור שרירותי (בסיס → יעד) |
| ישויות נתמכות | Line, Circle, Arc, Ellipse, Polyline | כל סוגי הישויות |
| הזנת מרחק | מוקלד לפני בחירת הישות | מוקלד או נלחץ אחרי הבחירה |
