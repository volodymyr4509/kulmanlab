# Chamfer

פקודת `chamfer` חותכת פינה אלכסונית ישרה בין שתי ישויות [Line](../line/) או [Polyline](../polyline/). אתם מציינים כמה רחוק לחתוך לאורך כל ישות (d1 ו-d2), והפקודה גוזרת את שתי הישויות לנקודות אלה ומכניסה קו מחבר ביניהן.

שימוש במרחקים שווים מייצר חיתוך סימטרי בזווית 45°; מרחקים שונים מייצרים שיפוע אסימטרי.

Chamfer פועלת על ישויות **Line ו-Polyline**.

## שימוש ב-Chamfer

1. הקלידו `chamfer` במסוף או לחצו על כפתור סרגל הכלים **Chamfer**.
2. **הקלידו את מרחק ה-Chamfer הראשון** (d1 — מרחק לאורך הישות הראשונה) ולחצו **Enter**.
3. **הקלידו את מרחק ה-Chamfer השני** (d2 — מרחק לאורך הישות השנייה) ולחצו **Enter**.
4. **לחצו על הישות הראשונה** — החלק שאתם לוחצים עליו קובע איזה צד של כל חיתוך נשמר.
5. **רחפו מעל הישות השנייה** — תצוגה מקדימה מקווקוות מציגה את חיתוך ה-chamfer שינבע. הזיזו את הסמן לצד שברצונכם לשמור.
6. **לחצו** כדי להחיל. שתי הישויות נגזרות וקו ה-chamfer מוכנס.

```
  לפני (d1=5, d2=8):          אחרי:

  ──────────────              ──────────╲
                │                        ╲────
                │
```

## בחירת צד

כאשר שני קווים חוצים זה את זה, ה-chamfer מוחל על הפינה המוגדרת על ידי מיקומי הלחיצה — החלק של כל ישות ב**אותו צד כמו הסמן** נשמר.

- לחצו קרוב לאחד הקצוות של הישות הראשונה כדי לבחור את החצי ההוא.
- הזיזו את הסמן לחצי הרצוי של הישות השנייה — התצוגה המקדימה המקווקוות מתעדכנת בזמן אמת.

עבור Polylines, מיקום הלחיצה קובע איזה **קטע** של הפוליליין משתתף, והקודקוד הקרוב ביותר בצד החיתוך הוא זה שנגזר.

## מה הפקודה יוצרת

- נקודת הקצה (או קודקוד הפוליליין) הקרובה ביותר לחיתוך של הישות הראשונה מוזזת לנקודה **T1**, הממוקמת d1 לאורך הישות הראשונה מהחיתוך.
- נקודת הקצה (או קודקוד הפוליליין) הקרובה ביותר לחיתוך של הישות השנייה מוזזת לנקודה **T2**, הממוקמת d2 לאורך הישות השנייה מהחיתוך.
- ישות Line חדשה מוכנסת מ-**T1** ל-**T2**.

הקו המוכנס יורש את הגדרות עובי הקו, הצבע, השכבה וסוג הקו הנוכחיים.

## מרגע לוח המקשים

| מקש | פעולה |
|-----|--------|
| `0`–`9`, `.` | הוספת ספרה לערך המרחק הנוכחי |
| `Backspace` | מחיקת התו האחרון שהוקלד |
| `Enter` | אישור המרחק שהוקלד והמשך |
| `Escape` | ביטול ואיפוס |

## ישויות נתמכות

| ישות | נתמך |
|--------|-----------|
| Line | כן |
| Polyline / Rectangle | כן |
| Arc, Circle, Ellipse | לא |
| Text, Spline, Dimension, Leader | לא |
