# Undo

הפקודה `undo` מבטלת את השינוי האחרון בשרטוט — צעד אחד לכל הפעלה. כל הוספה, מחיקה או עריכה של ישויות נרשמת כרשומת היסטוריה נפרדת. Undo צועדת אחורה דרך רשומות אלה בסדר הפוך.

## איך לבטל

- הקלידו `undo` במסוף, או
- לחצו על כפתור סרגל הכלים **Undo**.

כל הפעלה מבטלת פעולה רשומה אחת. הפעילו אותה שוב ושוב כדי לצעוד אחורה יותר.

## התנהגות היסטוריה

| פרט | ערך |
|--------|-------|
| צעדים לכל קובץ | עד **20** |
| אחסון | דפדפן (IndexedDB / localStorage), לכל שם קובץ |
| שורדת רענון עמוד | כן — ההיסטוריה משוחזרת כשאתם פותחים מחדש את הקובץ |
| פעולה חדשה לאחר ביטול | מנקה את כל רשומות ה-redo שלפני המיקום הנוכחי |
| הרשומה הישנה ביותר כשמלא | נמחקת כדי לפנות מקום לשינוי החדש ביותר |

כל שינוי ישות נרשם: שרטוט ישויות חדשות, מחיקת ישויות, עריכת נקודות אחיזה של נקודות קצה, החלת Move, Rotate, Scale, Mirror, Trim, Extend ו-Offset — כולם יוצרים רשומות היסטוריה.

## Undo מול Redo

| | Undo | Redo |
|---|------|------|
| כיוון | צעד **אחורה** דרך ההיסטוריה | צעד **קדימה** דרך רשומות שבוטלו |
| זמינה כאשר | קיימת לפחות פעולה אחת רשומה | בוצע לפחות Undo אחד וטרם בוצעה פעולה חדשה |
| מתנקה על ידי | שום דבר — ההיסטוריה מצטברת עד למגבלת 20 הצעדים | כל פעולת שרטוט חדשה |

השתמשו ב-[Redo](../redo/) כדי להחיל מחדש פעולה שבוטלה. כפתורי סרגל הכלים מאופרים כאשר הכיוון המתאים לא זמין.
