# Spline Fit

פקודת `splinefit` משרטטת spline קובי העובר דרך כל נקודה שאתם לוחצים עליה — עקומה מתאמת. בניגוד ל-[Spline CV](../spline-cv/), שבה העקומה רק נמשכת לכיוון נקודות בקרה, כאן העקומה נאלצת לפגוע בכל קואורדינטה שנלחצה בדיוק. באופן פנימי העורך מתאים נקודות בקרה כדי להשיג זאת, וה-CVs האלה נשמרות לצד נקודות ההתאמה בקובץ ה-DXF.

## שרטוט spline דרך נקודות התאמה

1. הקלידו `splinefit` במסוף או לחצו על כפתור סרגל הכלים **Spline Fit**.
2. **לחצו למיקום נקודות התאמה** — העקומה תעבור דרך כל אחת. או הקלידו `X,Y` ולחצו **Enter** לקואורדינטה מדויקת.
3. לחצו **Enter** לסיום (נדרשות לפחות 2 נקודות).

```
  ●──────●──────●──────●  ← העקומה עוברת בדיוק דרך כל לחיצה
  p1     p2     p3     p4
```

התצוגה המקדימה החיה מציגה את העקומה המתואמת הנוכחית בזמן שאתם מזיזים את הסמן, כולל הנקודה הבאה הפוטנציאלית במיקום הסמן. לחצו **Escape** כדי לבטל את כל הנקודות שהוצבו ולצאת.

## הזנת קואורדינטות

במקום ללחוץ, הקלידו מיקום מדויק לכל נקודת התאמה:

1. הקלידו את ערך X.
2. לחצו `,` — המסוף מציג `[X], [Y{cursor}]`.
3. הקלידו את ערך Y.
4. לחצו **Enter** למיקום נקודת ההתאמה.

## מרגע לוח המקשים

| מקש | פעולה |
|-----|--------|
| `0`–`9`, `.`, `-` | התחלת הזנת קואורדינטת X |
| `,` | נעילת X ומעבר להזנת Y |
| `Backspace` | מחיקת התו האחרון שהוקלד |
| `Enter` | אישור הקואורדינטה שהוקלדה, או סיום ה-spline אם אין קלט בתהליך ויש ≥ 2 נקודות |
| `Escape` | ביטול כל הנקודות ויציאה |

## עריכת נקודות אחיזה — עיצוב מחדש דרך נקודות התאמה

fit spline נבחר חושף נקודת אחיזה אחת לכל נקודת התאמה:

| נקודת אחיזה | מיקום | מה היא עושה |
|------|----------|--------------|
| **נקודת התאמה** | בכל מיקום שנלחץ | גררו כדי להזיז את נקודת ההתאמה — כל העקומה מחושבת מחדש כדי לעבור דרך המיקום החדש |

גרירת נקודת אחיזה אחת מחשבת מחדש את כל העקומה, לא רק את הקטעים הסמוכים. זה שונה מעריכת נקודות אחיזה בפוליליין, שבה הזזת קודקוד מעצבת מחדש רק את שני הקטעים הסמוכים.

אין נקודת אחיזה "להזיז את כל ה-spline". כדי להזיז את כל ה-spline, השתמשו בפקודת [Move](../move/).

## בחירת fit splines

| שיטה | התנהגות |
|--------|-----------|
| **לחיצה** | בוחרת אם הלחיצה נופלת קרוב לכל נקודה על העקומה |
| **גרירה ימינה** (קפדנית) | כל נקודות הדגימה לאורך העקומה חייבות ליפול בתוך תיבת הבחירה |
| **גרירה שמאלה** (חוצה) | כל חלק מהעקומה שחוצה את גבול תיבת הבחירה בוחר אותה |

## Spline Fit מול Spline CV — באיזה להשתמש

| | Spline Fit | Spline CV |
|---|------------|-----------|
| העקומה עוברת דרך נקודות | כל נקודה שנלחצה בדיוק | ראשונה ואחרונה בלבד (קשור) |
| אפקט עריכת נקודת אחיזה | הזזת נקודת התאמה → כל העקומה מחושבת מחדש | הזזת CV → העקומה נמשכת למיקום החדש |
| ניבוי הצורה | גבוה — העקומה עוקבת אחר הלחיצות | נמוך יותר — העקומה מפגרת אחרי ה-CVs |
| אידיאלי עבור | עקומות שחייבות לפגוע בקואורדינטות ספציפיות | עקומות אסתטיות חלקות, נתיבים חופשיים |

## DXF — ישות SPLINE (צורת נקודת התאמה)

Fit splines נשמרות כישויות `SPLINE` בקובץ ה-DXF, השומרות גם את קואורדינטות נקודות ההתאמה וגם את נקודות הבקרה המחושבות. `splineFlag` מוגדר ל-`8` (fit-point spline) כך שיישום הטוען מחדש יודע איזה סט נקודות להציג כנקודות אחיזה ניתנות לעריכה. כל המאפיינים — צבע, שכבה, סוג קו, קנה מידה של סוג קו ועובי — נשמרים ללא אובדן בהעברה הדדית. יישומי DXF התומכים ב-fit-point splines (LibreCAD, FreeCAD) יציגו את נקודות ההתאמה כנתונים הניתנים לעריכה העיקריים.
