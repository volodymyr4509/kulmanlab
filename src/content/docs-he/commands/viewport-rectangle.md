# ViewportRectangle

הפקודה `ViewportRectangle` יוצרת חלון תצוגה חדש בפריסת הנייר הפעילה על ידי בחירת שתי פינות מנוגדות. זמינה רק במרחב פריסה.

## יצירת חלון תצוגה

1. עברו לפריסת נייר באמצעות הלשונית בתחתית המסך.
2. הקלידו `ViewportRectangle` במסוף או לחצו על כפתור סרגל הכלים **Viewport Rectangle**.
3. **לחצו על הפינה הראשונה**, או הקלידו `X,Y` ולחצו **Enter** לקואורדינטה מדויקת.
4. **לחצו על הפינה הנגדית** — חלון התצוגה ממוקם מיידית. הזנת קואורדינטות עובדת גם כאן.

חלון התצוגה החדש מציג את המודל המלא בקנה המידה ברירת המחדל של הפריסה. השתמשו בגלגלת הגלילה בתוך חלון התצוגה כדי לזום, או בגרירת הלחצן האמצעי כדי להזיז את תצוגת המודל.

## הזנת קואורדינטות

בכל שלב פינה תוכלו להקליד קואורדינטה מדויקת:

1. הקלידו את ערך X.
2. לחצו `,` — המסוף מציג `[X], [Y{cursor}]`.
3. הקלידו את ערך Y.
4. לחצו **Enter** למיקום הנקודה.

## מרגע לוח המקשים

| מקש | פעולה |
|-----|--------|
| `0`–`9`, `.`, `-` | התחלת הזנת קואורדינטת X |
| `,` | נעילת X ומעבר להזנת Y |
| `Enter` | אישור הקואורדינטה שהוקלדה |
| `Escape` | ביטול |

## עריכת חלון תצוגה

לאחר מיקום חלון תצוגה, לחצו עליו כדי לבחור בו:

- **גררו את הקצוות או הפינות** לשינוי גודל.
- **גררו את נקודת האחיזה המרכזית** כדי להזיז אותו.
- השתמשו ב**בורר קנה המידה** בשורת הבקרה כדי להגדיר קנה מידה מדויק (למשל 1:50). כדי להזין קנה מידה שאינו ברשימה, הקלידו אותו ישירות בשדה הקלט בתחתית התפריט הנפתח — מקבל פורמט יחס (`1:200`, `5:1`) או מספר עשרוני פשוט (`0.005`), ואז לחצו **Enter**.
- לחצו לחיצה ימנית על חלון תצוגה והשתמשו ב-**Lock** כדי למנוע שינויים בשוגג.

## הערות

- ViewportRectangle זמינה רק כאשר לשונית פריסת נייר פעילה. הרצתה במרחב מודל מציגה הודעת שגיאה ויוצאת.
- כדי לשכפל חלון תצוגה קיים, השתמשו ב-[ViewportCopy](../viewport-copy/).
