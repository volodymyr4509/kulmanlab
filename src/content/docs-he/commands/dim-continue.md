# Dimension Continue

פקודת `dimcontinue` משרשרת מידות חדשות מ**קו העזר השני** של מידה קיימת. כל קטע חדש ממוקם לאורך אותו ציר מדידה ובאותו היסט קו מידה כמו הבסיס. כל מאפייני הסגנון — גודל חץ, גובה טקסט, אורכי קווי עזר — מועתקים מהבסיס אוטומטית.

## איך נראות מידות משורשרות

```
  |←— 3.00 —→|←— 2.50 —→|←— 4.00 —→|
  |           |           |           |
  ●           ●           ●           ●
  p1        p2 (בסיס     p3           p4
           ext2 → התחלה חדשה)
```

כל מלבן הוא ישות `DIMENSION` נפרדת. הן חולקות את אותו מיקום קו מידה וכיוון מדידה.

## התחלת שרשרת

1. הקלידו `dimcontinue` במסוף או לחצו על כפתור סרגל הכלים **Dimension Continue**.
2. **אם מידה הוצבה זה עתה** — הפקודה קולטת אותה אוטומטית כבסיס (ללא צורך בלחיצה).
3. **אם אין מידה אחרונה** — לחצו על מידה קיימת כלשהי כדי להשתמש בה כבסיס.
4. **לחצו על מקור קו העזר הבא** — תצוגה מקדימה מציגה את המידה החדשה בזמן שאתם מזיזים את הסמן. או הקלידו `X,Y` ולחצו **Enter** לקואורדינטה מדויקת.
5. המשיכו ללחוץ (או להקליד) כדי להרחיב את השרשרת. כל מידה שהוצבה הופכת לבסיס החדש אוטומטית.
6. לחצו **Enter**, **Space** או **Escape** כדי לסיים את השרשרת.

## מה יורש ממידת הבסיס

| מאפיין | יורש מהבסיס |
|----------|---------------------|
| כיוון/זווית מדידה | כן — נעול לכל השרשרת |
| היסט קו המידה (מרחק מהנקודות הנמדדות) | כן |
| גודל חץ | כן |
| גובה טקסט | כן |
| היסט והארכת קו עזר | כן |
| יישור טקסט | כן |
| שם סגנון | כן |
| צבע, שכבה | לא יורש — משתמש בשכבה הנוכחית |

## נעילת כיוון המדידה

כיוון המדידה של השרשרת **קבוע לזווית של מידת הבסיס**:

- בסיס ליניארי (H) → כל ההמשכים מודדים מרחק אופקי (Δ X).
- בסיס ליניארי (V) → כל ההמשכים מודדים מרחק אנכי (Δ Y).
- בסיס מיושר בכל זווית → כל ההמשכים מודדים לאורך אותה זווית.

לא ניתן לשנות את הכיוון באמצע השרשרת. התחילו [Dimension Linear](../dim-linear/) או [Dimension Aligned](../dim-aligned/) חדש כדי למדוד בכיוון אחר.

## מרגע לוח המקשים

| מקש | פעולה |
|-----|--------|
| `0`–`9`, `.`, `-` | התחלת הזנת קואורדינטת X |
| `,` | נעילת X ומעבר להזנת Y |
| `Backspace` | מחיקת התו האחרון שהוקלד |
| `Enter` | אישור קואורדינטה שהוקלדה, או סיום השרשרת אם אין קלט בתהליך |
| `Space` / `Escape` | סיום השרשרת |

## Dimension Continue מול התחלה מחדש

| | Dimension Continue | Dimension Linear / Aligned |
|---|-------------------|--------------------------|
| נקודת התחלה | קבועה ב-ext2 של הבסיס האחרון | לחיצה בכל מקום |
| זווית | נעולה לבסיס | חופשית |
| היסט | יורש מהבסיס | נקבע על ידי הסמן או הקלדה |
| סגנון | יורש מהבסיס | הסגנון הנוכחי |
| אידיאלי עבור | מדידות מצטברות לאורך שורה | מידה ראשונה או שינוי כיוון |

## עריכת תוויות לאחר המיקום — מצב simple

**לחיצה כפולה** על כל מידה בשרשרת פותחת את עורך הטקסט במצב **simple**. כל קטע עצמאי וניתן לעריכה בנפרד.

| תכונה | התנהגות |
|---------|-----------|
| מודגש / נטוי / גופן / גובה | חלים על **כל** התווית בבת אחת |
| עיצוב לפי תו | לא נתמך |
| `Enter` | מאשר את הערך וסוגר את העורך |
| מרובת שורות | לא נתמך |

ראו [עורך טקסט — מצב simple](../../interface/text-editor/#simple-mode) להפניה המלאה.

## DXF — ישויות DIMENSION

כל קטע בשרשרת נשמר כישות `DIMENSION` עצמאית בקובץ ה-DXF. הן אינן מקושרות בקובץ — הן חולקות מאפיינים כי נוצרו מאותו בסיס, אך כל אחת ניתנת לעריכה בנפרד לאחר המיקום.
