# Text Editor

עורך הטקסט נפתח כאשר אתם ממקמים או לוחצים לחיצה כפולה על ישות הניתנת לעריכה. **תג מצב** קטן בכותרת — **rich** (בצבע ההדגשה) או **simple** (מעומעם) — מציג איזה מצב פעיל עבור הישות הנוכחית.

## מצבי העורך

### מצב Rich

בשימוש עבור: **Text** (תוויות MTEXT) והערות **Multileader**.

| תכונה | התנהגות |
|---------|-----------|
| מודגש / נטוי / קו-חוצה | לפי תו (חל על הבחירה, או על כל הישות אם אין בחירה) |
| גופן וגובה | עקיפה לפי תו, או ברירת מחדל לכל הישות |
| Alignment (Left / Center / Right / Justify) | **טקסט בלבד** — לא זמין עבור Multileader |
| `Enter` | מכניס מעבר שורה קשיח |
| `Shift+←/→` | מרחיב או מכווץ בחירת טקסט |
| `Home` / `End` | קפיצה להתחלה / סוף השורה הקשיחה הנוכחית |
| גלישת מילים | נתמכת דרך נקודות אחיזה לשינוי גודל רוחב-ייחוס |

### מצב Simple

בשימוש עבור: **Dimension Linear**, **Dimension Aligned**, **Dimension Angular**, **Dimension Radius**, **Dimension Diameter**.

העורך מאותחל מראש עם התווית המוצגת הנוכחית של המידה כך שתוכלו למקם את הסמן ולערוך את הערך ישירות.

| תכונה | התנהגות |
|---------|-----------|
| מודגש / נטוי / גופן / גובה | זמין — חל על **כל** התווית בבת אחת |
| עיצוב לפי תו | לא נתמך |
| `Enter` | **מאשר** את הערך וסוגר את העורך (ללא מעבר שורה) |
| מרובת שורות | לא נתמך |
| גלישת מילים | לא נתמך |

## פתיחת העורך

| פעולה | תוצאה |
|--------|--------|
| פקודת `text` → לחיצה על מיקום | יוצרת ישות טקסט חדשה ופותחת את העורך (**rich**) |
| לחיצה כפולה על ישות **Text** קיימת | פותחת מחדש את העורך במצב **rich** |
| לחיצה כפולה על **Multileader** קיים | פותחת את העורך במצב **rich** |
| לחיצה כפולה על ישות **מידה** | פותחת את העורך במצב **simple** |
| `Escape` בתוך העורך | סוגר את העורך ושומר את כל השינויים |

## סרגל כלים

סרגל הכלים צף מעל תיבת התיחום של הטקסט ונשאר מעוגן לישות בזמן שאתם עושים pan או zoom.

### מודגש · נטוי · קו-חוצה

| כפתור | קיצור | מה הוא עושה |
|--------|----------|--------------|
| **B** | — | הפעלה/כיבוי של מודגש |
| *I* | — | הפעלה/כיבוי של נטוי |
| ~~S~~ | — | הפעלה/כיבוי של קו-חוצה |

**איך ההפעלה חלה:**

- **עם בחירת טקסט** — הסגנון חל אך ורק על התווים הנבחרים.
- **ללא בחירה, סמן בטקסט קיים** — מפעיל/מכבה את הסגנון על כל הישות (כל הקטעים).
- **טקסט ריק או ישות חדשה** — הסגנון נשמר על הקטע הריק ומוחל על כל תו שתקלידו מנקודה זו ואילך.

הכפתור מוצג מודגש (פעיל) כאשר לכל תו בבחירה הנוכחית — או לתו מיד משמאל לסמן — יש את הסגנון הזה מוגדר.

### גופן

התפריט הנפתח מקבץ את הגופנים הזמינים ל-**Default** (הסאנס-סריף המובנה), **User** (הגופנים שהעליתם בעצמכם, אם יש), **Free** (סט של גופני Google Fonts מובנים), ו-**System** (גופני מערכת הפעלה נפוצים כמו Helvetica, Times New Roman, Georgia, Courier New, Verdana, Tahoma, Trebuchet MS, Lucida Console ו-Impact).

- **עם בחירה** — עוקף את הגופן רק עבור התווים הנבחרים.
- **ללא בחירה** — מחיל את הגופן על כל הישות.

התפריט הנפתח משקף את הגופן של התו משמאל לסמן כאשר אין בחירה.

לא מוגבל לרשימה המובנית — לחצו על כפתור **Font Manager** בסרגל הכלים כדי להעלות קובץ `.ttf` משלכם ולהוסיף אותו לקבוצת **User**. ראו [Font Manager](../../commands/font-manager/) לפרטים.

### גובה

שדה המספר קובע את **גובה האותיות הגדולות** (cap height) ביחידות שרטוט.

- **עם בחירה** — עוקף את הגובה עבור התווים הנבחרים, באופן עצמאי מגובה הבסיס של הישות.
- **ללא בחירה** — משנה את גובה הבסיס של הישות (חל על כל התווים שאין להם עקיפת גובה אישית).

השדה משקף את הגובה של התו משמאל לסמן. השאירו אותו ריק כדי להשתמש בברירת המחדל של הישות.

### יישור

ארבעה כפתורים — **Align Left**, **Align Center**, **Align Right**, **Justify** — קובעים את יישור הפסקה. זמין רק עבור ישויות **Text**; תוויות Multileader ומידות לא מציגות כפתורים אלה.

- לחיצה על כפתור מיישרת מחדש כל שורה בתוך תיבת התיחום הקיימת של הישות — היא לא מזיזה את נקודת ההוספה ולא משנה את גודל התיבה.
- לחיצה על הכפתור הפעיל כבר מנקה את העקיפה, וחוזרת לעמודה המשתמעת מנקודת העיגון של הישות.
- **Justify** מותח את הרווח בין מילים כך שכל שורה תמלא את מלוא רוחב השורה.

## סמן וניווט

| מקש | פעולה |
|-----|--------|
| `←` / `→` | הזזת הסמן תו אחד שמאלה או ימינה |
| `Home` | קפיצה להתחלת השורה הקשיחה הנוכחית |
| `End` | קפיצה לסוף השורה הקשיחה הנוכחית |
| `Shift` + `←` / `→` | הרחבה או כיווץ הבחירה |
| `Backspace` | מחיקת התו משמאל (או הבחירה) |
| `Delete` | מחיקת התו מימין (או הבחירה) |
| `Enter` | הוספת מעבר שורה |
| `Escape` | סגירת העורך |

גובה הסמן מתאים אוטומטית לגובה האותיות הגדולות של התו הסמוך, כולל הגודל הקטן יותר בשימוש לכתב תחתי ועילי.

## העתקה, גזירה והדבקה

| מקש | פעולה |
|-----|--------|
| `Ctrl+C` / `Cmd+C` | העתקת הטקסט הנבחר |
| `Ctrl+X` / `Cmd+X` | גזירת הטקסט הנבחר |
| `Ctrl+V` / `Cmd+V` | הדבקה בסמן |

העתקה וגזירה דורשות בחירת טקסט פעילה. טקסט מודבק תמיד פשוט — הוא מקבל את העיצוב (מודגש, נטוי, גופן, גובה) שכבר נמצא בסמן במקום לשאת את העיצוב שהיה לו בעת ההעתקה.

ב**מצב rich**, מעברי שורה בטקסט המודבק נשמרים. ב**מצב simple**, מעברי שורה מוסרים, מכיוון שתוויות מידות הן שורה בודדת.

## גלישת מילים

כאשר לישות טקסט יש **רוחב ייחוס** מוגדר, שורות ארוכות עוברות גלישה רכה בגבולות מילים כדי להתאים לרוחב הזה.

כדי להגדיר או לשנות את רוחב הייחוס בזמן שהישות נבחרת, גררו את **נקודות האחיזה לשינוי גודל** — המלבנים הדקים בקצוות השמאלי והימני של תיבת התיחום המקווקוות. התוכן זורם מחדש בזמן אמת בזמן שאתם גוררים.

הגדרת רוחב הייחוס לאפס (גררו את נקודות האחיזה יחד או מחקו את הערך בפאנל המאפיינים) מסירה את גלישת המילים ומאפשרת לשורות לגדול בחופשיות.

## טקסט מרובה שורות

לחצו `Enter` כדי להכניס מעבר שורה קשיח. כל שורה קשיחה עצמאית — `Home` ו-`End` מנווטים רק בתוך השורה הקשיחה הנוכחית.

מעברי שורה קשיחים ועיצוב לפי תו נשמרים באמצעות פורמט MTEXT ושורדים העברה הדדית מלאה של DXF.

## תאימות DXF

ישויות טקסט נשמרות כ-**MTEXT** בקבצי DXF. מודגש ונטוי מקודדים כ-`\L`, `\K`, `\O` והחלפות גופן מוטבעות (`\f`). גובה לפי תו מקודד כ-`\H`. כל העיצוב נשמר בייצוא וקריא על ידי LibreCAD, FreeCAD ויישומים אחרים התומכים ב-DXF.
