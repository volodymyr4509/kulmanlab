# ViewportCopy

`ViewportCopy` ਕਮਾਂਡ ਵਿਊਪੋਰਟ ਦੇ ਸਕੇਲ ਅਤੇ ਮਾਡਲ ਸੈਂਟਰ ਨੂੰ ਸੁਰੱਖਿਅਤ ਰੱਖਦੇ ਹੋਏ ਇਸਨੂੰ ਨਵੀਂ ਸਥਿਤੀ 'ਤੇ ਕਾਪੀ ਕਰਦੀ ਹੈ। ਸਿਰਫ਼ ਲੇਆਉਟ ਸਪੇਸ ਵਿੱਚ ਉਪਲਬਧ ਹੈ।

## ਵਿਊਪੋਰਟ ਕਾਪੀ ਕਰਨਾ

1. ਪੇਪਰ ਲੇਆਉਟ ਟੈਬ 'ਤੇ ਜਾਓ।
2. ਚਾਹੋ ਤਾਂ ਇਸਨੂੰ ਪਹਿਲਾਂ ਤੋਂ ਚੁਣਨ ਲਈ ਇੱਕ ਵਿਊਪੋਰਟ 'ਤੇ ਕਲਿੱਕ ਕਰੋ।
3. ਟਰਮੀਨਲ ਵਿੱਚ `ViewportCopy` ਟਾਈਪ ਕਰੋ ਜਾਂ **Viewport Copy** ਟੂਲਬਾਰ ਬਟਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ।
4. ਜੇ ਕੋਈ ਵਿਊਪੋਰਟ ਪਹਿਲਾਂ ਤੋਂ ਚੁਣਿਆ ਨਹੀਂ ਗਿਆ ਸੀ, ਤਾਂ ਕਾਪੀ ਕਰਨ ਲਈ **ਵਿਊਪੋਰਟ 'ਤੇ ਕਲਿੱਕ ਕਰੋ**।
5. **ਬੇਸ ਬਿੰਦੂ 'ਤੇ ਕਲਿੱਕ ਕਰੋ** — ਵਿਸਥਾਪਨ ਦਾ ਹਵਾਲਾ ਬਿੰਦੂ। ਜਾਂ ਸਹੀ ਕੋਆਰਡੀਨੇਟ ਲਈ `X,Y` ਟਾਈਪ ਕਰੋ ਅਤੇ **Enter** ਦਬਾਓ।
6. **ਟਿਕਾਣੇ 'ਤੇ ਕਲਿੱਕ ਕਰੋ** — ਵਿਊਪੋਰਟ ਬੇਸ→ਟਿਕਾਣਾ ਆਫਸੈੱਟ 'ਤੇ ਰੱਖਿਆ ਜਾਂਦਾ ਹੈ। ਜਾਂ ਕੋਆਰਡੀਨੇਟ ਐਂਟਰੀ / ਐਂਗਲ ਲਾਕ ਵਰਤੋ।

ਰੱਖਣ ਤੋਂ ਬਾਅਦ, ਕਮਾਂਡ ਸਰਗਰਮ ਰਹਿੰਦੀ ਹੈ — ਉਸੇ ਵਿਊਪੋਰਟ ਦੀ ਇੱਕ ਹੋਰ ਕਾਪੀ ਰੱਖਣ ਲਈ ਕਿਸੇ ਹੋਰ ਟਿਕਾਣੇ 'ਤੇ ਕਲਿੱਕ ਕਰੋ। ਸਮਾਪਤ ਕਰਨ ਲਈ **Enter**, **Space**, ਜਾਂ **Escape** ਦਬਾਓ।

## ਕੋਆਰਡੀਨੇਟ ਐਂਟਰੀ

ਬੇਸ-ਬਿੰਦੂ ਅਤੇ ਟਿਕਾਣੇ ਦੇ ਸਟੈਪਾਂ 'ਤੇ ਤੁਸੀਂ ਕਲਿੱਕ ਕਰਨ ਦੀ ਬਜਾਏ ਇੱਕ ਸਹੀ ਕੋਆਰਡੀਨੇਟ ਟਾਈਪ ਕਰ ਸਕਦੇ ਹੋ:

1. X ਮੁੱਲ ਟਾਈਪ ਕਰੋ।
2. `,` ਦਬਾਓ — ਟਰਮੀਨਲ `[X], [Y{cursor}]` ਦਿਖਾਉਂਦਾ ਹੈ।
3. Y ਮੁੱਲ ਟਾਈਪ ਕਰੋ।
4. ਪੁਸ਼ਟੀ ਕਰਨ ਲਈ **Enter** ਦਬਾਓ।

## ਐਂਗਲ ਲਾਕਿੰਗ ਅਤੇ ਸਹੀ ਦੂਰੀ

ਬੇਸ ਬਿੰਦੂ ਸੈੱਟ ਹੋਣ ਤੋਂ ਬਾਅਦ, ਜਦੋਂ ਕਰਸਰ ਇਕਸਾਰ ਹੁੰਦਾ ਹੈ ਤਾਂ ਕਮਾਂਡ 45° ਧੁਰਿਆਂ (0°, 45°, 90°, 135°, …) 'ਤੇ ਸਨੈਪ ਕਰਦੀ ਹੈ। ਲਾਕ ਹੋਣ ਵੇਲੇ:

- ਪ੍ਰੀਵਿਊ ਧੁਰੇ 'ਤੇ ਸਨੈਪ ਕਰਦਾ ਹੈ।
- ਲਾਕ ਕੀਤੀ ਦਿਸ਼ਾ ਦੇ ਨਾਲ ਠੀਕ ਉਸ ਆਫਸੈੱਟ 'ਤੇ ਕਾਪੀ ਰੱਖਣ ਲਈ ਇੱਕ ਦੂਰੀ ਟਾਈਪ ਕਰੋ ਅਤੇ **Enter** ਦਬਾਓ।

| ਕੀ | ਕਾਰਵਾਈ |
|-----|--------|
| `0`–`9`, `.` | ਦੂਰੀ ਮੁੱਲ ਵਿੱਚ ਅੰਕ ਜੋੜੋ |
| `-` | ਨੈਗੇਟਿਵ ਦੂਰੀ (ਦਿਸ਼ਾ ਉਲਟਾਉਂਦੀ ਹੈ; ਸਿਰਫ਼ ਪਹਿਲਾ ਅੱਖਰ) |
| `Backspace` | ਆਖਰੀ ਟਾਈਪ ਕੀਤਾ ਅੱਖਰ ਮਿਟਾਓ |
| `Enter` | ਟਾਈਪ ਕੀਤੀ ਦੂਰੀ 'ਤੇ ਕਾਪੀ ਰੱਖੋ |

## ਕੀਬੋਰਡ ਰੈਫਰੈਂਸ

| ਕੀ | ਕਾਰਵਾਈ |
|-----|--------|
| `0`–`9`, `.`, `-` | X ਕੋਆਰਡੀਨੇਟ ਐਂਟਰੀ ਸ਼ੁਰੂ ਕਰੋ, ਜਾਂ ਐਂਗਲ-ਲਾਕ ਹੋਣ 'ਤੇ ਦੂਰੀ |
| `,` | X ਲਾਕ ਕਰੋ ਅਤੇ Y ਐਂਟਰੀ 'ਤੇ ਜਾਓ |
| `Enter` | ਟਾਈਪ ਕੀਤਾ ਕੋਆਰਡੀਨੇਟ ਜਾਂ ਦੂਰੀ ਪੁਸ਼ਟੀ ਕਰੋ |
| `Enter` / `Space` | ਸਮਾਪਤ ਕਰੋ (ਜਦੋਂ ਕੋਈ ਇਨਪੁੱਟ ਜਾਰੀ ਨਾ ਹੋਵੇ) |
| `Escape` | ਰੱਦ ਕਰੋ ਅਤੇ ਰੀਸੈੱਟ ਕਰੋ |

## ਨੋਟਸ

- ViewportCopy ਸਿਰਫ਼ ਉਦੋਂ ਉਪਲਬਧ ਹੁੰਦਾ ਹੈ ਜਦੋਂ ਪੇਪਰ ਲੇਆਉਟ ਟੈਬ ਸਰਗਰਮ ਹੋਵੇ।
- ਕਾਪੀ ਕੀਤਾ ਵਿਊਪੋਰਟ ਮੂਲ ਦੇ ਸਮਾਨ ਸਕੇਲ, ਮਾਡਲ ਸੈਂਟਰ, ਲਾਕਡ ਸਥਿਤੀ, ਅਤੇ ਮਾਪ ਵਿਰਾਸਤ ਵਿੱਚ ਲੈਂਦਾ ਹੈ।
- ਸਿਫ਼ਰ ਤੋਂ ਨਵਾਂ ਵਿਊਪੋਰਟ ਬਣਾਉਣ ਲਈ, [ViewportRectangle](../viewport-rectangle/) ਵਰਤੋ।
