# Scale

`scale` ਕਮਾਂਡ ਚੁਣੀਆਂ ਐਂਟਿਟੀਆਂ ਨੂੰ ਇੱਕ ਬੇਸ ਪੁਆਇੰਟ ਦੁਆਲੇ ਇਕਸਾਰ ਢੰਗ ਨਾਲ ਮੁੜ-ਆਕਾਰ ਦਿੰਦੀ ਹੈ। ਬੇਸ ਪੁਆਇੰਟ ਤੋਂ ਸਾਰੀਆਂ ਦੂਰੀਆਂ ਨੂੰ ਸਕੇਲ ਫੈਕਟਰ ਨਾਲ ਗੁਣਾ ਕੀਤਾ ਜਾਂਦਾ ਹੈ — `2` ਦਾ ਫੈਕਟਰ ਸਾਰੇ ਮਾਪ ਦੁੱਗਣੇ ਕਰਦਾ ਹੈ, `0.5` ਉਹਨਾਂ ਨੂੰ ਅੱਧਾ ਕਰਦਾ ਹੈ। ਫੈਕਟਰ ਹਮੇਸ਼ਾ ਟਾਈਪ ਕਰਕੇ ਦਾਖਲ ਕੀਤਾ ਜਾਂਦਾ ਹੈ; ਕੋਈ ਕਲਿੱਕ-ਟੂ-ਸੈੱਟ-ਸਕੇਲ ਨਹੀਂ।

## ਸ਼ੁਰੂ ਕਰਨ ਦੇ ਦੋ ਤਰੀਕੇ

**ਪਹਿਲਾਂ ਚੁਣੋ, ਫਿਰ ਸਕੇਲ ਕਰੋ** — ਪਹਿਲਾਂ ਐਂਟਿਟੀਆਂ ਚੁਣੋ, ਫਿਰ ਸਰਗਰਮ ਕਰੋ:

1. ਕੈਨਵਸ 'ਤੇ ਇੱਕ ਜਾਂ ਵੱਧ ਐਂਟਿਟੀਆਂ ਚੁਣੋ।
2. ਟਰਮੀਨਲ ਵਿੱਚ `scale` ਟਾਈਪ ਕਰੋ ਜਾਂ **Scale** ਟੂਲਬਾਰ ਬਟਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ।
3. **ਬੇਸ ਪੁਆਇੰਟ 'ਤੇ ਕਲਿੱਕ ਕਰੋ** — ਉਹ ਸਥਿਰ ਬਿੰਦੂ ਜੋ ਸਕੇਲਿੰਗ ਦੌਰਾਨ ਹਿੱਲਦਾ ਨਹੀਂ। ਜਾਂ ਸਹੀ ਕੋਆਰਡੀਨੇਟ ਲਈ `X,Y` ਟਾਈਪ ਕਰੋ ਅਤੇ **Enter** ਦਬਾਓ।
4. **ਸਕੇਲ ਫੈਕਟਰ ਟਾਈਪ ਕਰੋ** ਅਤੇ **Enter** ਦਬਾਓ।

**ਸਰਗਰਮ ਕਰੋ, ਫਿਰ ਚੁਣੋ** — ਕੁਝ ਵੀ ਚੁਣੇ ਬਿਨਾਂ ਕਮਾਂਡ ਸ਼ੁਰੂ ਕਰੋ:

1. `scale` ਟਾਈਪ ਕਰੋ ਜਾਂ ਟੂਲਬਾਰ ਬਟਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ।
2. **ਆਬਜੈਕਟ ਚੁਣੋ** — ਟੌਗਲ ਕਰਨ ਲਈ ਕਲਿੱਕ ਕਰੋ, ਜਾਂ ਖੇਤਰ ਦੁਆਰਾ ਚੁਣਨ ਲਈ ਖਿੱਚੋ।
3. ਚੋਣ ਪੁਸ਼ਟੀ ਕਰਨ ਲਈ **Enter** ਜਾਂ **Space** ਦਬਾਓ।
4. **ਬੇਸ ਪੁਆਇੰਟ 'ਤੇ ਕਲਿੱਕ ਕਰੋ** (ਕੋਆਰਡੀਨੇਟ ਐਂਟਰੀ ਉਪਲਬਧ), ਫਿਰ ਫੈਕਟਰ ਟਾਈਪ ਕਰੋ।

```
  Base ●                Base ●
        [entity]   →          [larger entity]
  factor = 2 → distances from ● are doubled
```

## ਸਕੇਲ ਫੈਕਟਰ ਟਾਈਪ ਕਰਨਾ

ਬੇਸ ਪੁਆਇੰਟ ਰੱਖੇ ਜਾਣ ਤੋਂ ਬਾਅਦ ਟਰਮੀਨਲ `enter scale factor:` ਦਿਖਾਉਂਦਾ ਹੈ ਅਤੇ ਕੀਬੋਰਡ ਇਨਪੁਟ ਦੀ ਉਡੀਕ ਕਰਦਾ ਹੈ:

| ਕੀ | ਕਾਰਵਾਈ |
|-----|--------|
| `0`–`9`, `.` | ਫੈਕਟਰ ਵਿੱਚ ਅੰਕ ਜੋੜੋ |
| `-` | ਨਕਾਰਾਤਮਕ ਫੈਕਟਰ (ਸਿਰਫ਼ ਪਹਿਲਾ ਅੱਖਰ — ਪਹਿਲਾਂ ਉਲਟਾਉਂਦਾ ਹੈ ਫਿਰ ਸਕੇਲ ਕਰਦਾ ਹੈ) |
| `Backspace` | ਆਖਰੀ ਟਾਈਪ ਕੀਤਾ ਅੱਖਰ ਮਿਟਾਓ |
| `Enter` | ਟਾਈਪ ਕੀਤੇ ਫੈਕਟਰ 'ਤੇ ਸਕੇਲ ਲਾਗੂ ਕਰੋ |

ਫੈਕਟਰ ਸਿਫ਼ਰ ਨਹੀਂ ਹੋਣਾ ਚਾਹੀਦਾ। ਆਮ ਮੁੱਲ:

| ਫੈਕਟਰ | ਪ੍ਰਭਾਵ |
|--------|--------|
| `2` | ਸਾਰੇ ਮਾਪ ਦੁੱਗਣੇ ਕਰਦਾ ਹੈ |
| `0.5` | ਸਾਰੇ ਮਾਪ ਅੱਧੇ ਕਰਦਾ ਹੈ |
| `1.5` | 50% ਵਧਾਉਂਦਾ ਹੈ |
| `-1` | ਬੇਸ ਪੁਆਇੰਟ ਦੇ ਪਾਰ ਪ੍ਰਤੀਬਿੰਬਿਤ ਕਰਦਾ ਹੈ (180° ਘੁੰਮਾਓ ਦੇ ਬਰਾਬਰ) |

ਟਾਈਪ ਕਰਦੇ ਸਮੇਂ ਕੋਈ ਲਾਈਵ ਪ੍ਰੀਵਿਊ ਨਹੀਂ ਹੁੰਦਾ — ਸਕੇਲ ਕੀਤਾ ਨਤੀਜਾ **Enter** ਦਬਾਉਣ ਤੋਂ ਬਾਅਦ ਹੀ ਦਿਖਾਈ ਦਿੰਦਾ ਹੈ।

## ਕੀਬੋਰਡ ਰੈਫਰੈਂਸ

| ਕੀ | ਕਾਰਵਾਈ |
|-----|--------|
| `Enter` / `Space` | ਚੋਣ ਪੁਸ਼ਟੀ ਕਰੋ |
| `0`–`9`, `.`, `-` | X ਕੋਆਰਡੀਨੇਟ ਐਂਟਰੀ ਸ਼ੁਰੂ ਕਰੋ (ਬੇਸ-ਪੁਆਇੰਟ ਪੜਾਅ), ਜਾਂ ਸਕੇਲ ਫੈਕਟਰ (ਫੈਕਟਰ ਪੜਾਅ) |
| `,` | X ਲਾਕ ਕਰੋ ਅਤੇ Y ਐਂਟਰੀ 'ਤੇ ਜਾਓ (ਬੇਸ-ਪੁਆਇੰਟ ਪੜਾਅ) |
| `Backspace` | ਆਖਰੀ ਟਾਈਪ ਕੀਤਾ ਅੱਖਰ ਮਿਟਾਓ |
| `Enter` | ਕੋਆਰਡੀਨੇਟ ਪੁਸ਼ਟੀ ਕਰੋ ਜਾਂ ਸਕੇਲ ਲਾਗੂ ਕਰੋ |
| `Escape` | ਰੱਦ ਕਰੋ ਅਤੇ ਰੀਸੈੱਟ ਕਰੋ |

## ਕਮਾਂਡ ਦੌਰਾਨ ਚੋਣ

| ਢੰਗ | ਵਿਵਹਾਰ |
|--------|-----------|
| **ਕਲਿੱਕ** | ਕਰਸਰ ਹੇਠਾਂ ਐਂਟਿਟੀ ਟੌਗਲ ਕਰਦਾ ਹੈ |
| **ਸੱਜੇ ਖਿੱਚੋ** (ਸਖ਼ਤ) | ਬਾਕਸ ਦੇ ਪੂਰੀ ਤਰ੍ਹਾਂ ਅੰਦਰ ਵਾਲੀਆਂ ਐਂਟਿਟੀਆਂ ਜੋੜਦਾ ਹੈ |
| **ਖੱਬੇ ਖਿੱਚੋ** (ਕ੍ਰਾਸਿੰਗ) | ਬਾਕਸ ਨੂੰ ਕੱਟਣ ਵਾਲੀਆਂ ਐਂਟਿਟੀਆਂ ਜੋੜਦਾ ਹੈ |
| **Enter** / **Space** | ਚੋਣ ਪੁਸ਼ਟੀ ਕਰਦਾ ਹੈ |

## ਕੀ ਸਕੇਲ ਹੁੰਦਾ ਹੈ

ਸਾਰੀਆਂ ਐਂਟਿਟੀ ਕਿਸਮਾਂ ਸਮਰਥਿਤ ਹਨ। ਹਰੇਕ ਐਂਟਿਟੀ ਆਪਣੀ ਜਿਓਮੈਟਰੀ ਨੂੰ ਬੇਸ ਪੁਆਇੰਟ ਦੇ ਸਾਪੇਖ ਸਕੇਲ ਕਰਦੀ ਹੈ:

| ਐਂਟਿਟੀ | ਕੀ ਬਦਲਦਾ ਹੈ |
|--------|-------------|
| Line | ਦੋਵੇਂ ਅੰਤ ਬਿੰਦੂ ਬੇਸ ਪੁਆਇੰਟ ਤੋਂ ਦੂਰ ਸਕੇਲ ਹੁੰਦੇ ਹਨ |
| Circle | ਕੇਂਦਰ ਬੇਸ ਪੁਆਇੰਟ ਤੋਂ ਸਕੇਲ ਹੁੰਦਾ ਹੈ; ਰੇਡੀਅਸ ਫੈਕਟਰ ਨਾਲ ਗੁਣਾ ਹੁੰਦਾ ਹੈ |
| Arc | ਕੇਂਦਰ ਸਕੇਲ ਹੁੰਦਾ ਹੈ; ਰੇਡੀਅਸ ਫੈਕਟਰ ਨਾਲ ਗੁਣਾ ਹੁੰਦਾ ਹੈ; ਕੋਣ ਬਦਲੇ ਨਹੀਂ ਰਹਿੰਦੇ |
| Ellipse | ਕੇਂਦਰ ਸਕੇਲ ਹੁੰਦਾ ਹੈ; ਦੋਵੇਂ ਸੈਮੀ-ਧੁਰੇ ਦੀ ਲੰਬਾਈ ਫੈਕਟਰ ਨਾਲ ਗੁਣਾ ਹੁੰਦੀ ਹੈ |
| Polyline / Rectangle | ਹਰੇਕ ਵਰਟੈਕਸ ਬੇਸ ਪੁਆਇੰਟ ਤੋਂ ਸਕੇਲ ਹੁੰਦਾ ਹੈ |
| Text | ਐਂਕਰ ਪੁਆਇੰਟ ਸਕੇਲ ਹੁੰਦਾ ਹੈ; ਉਚਾਈ ਫੈਕਟਰ ਨਾਲ ਗੁਣਾ ਹੁੰਦੀ ਹੈ |
| Spline | ਸਾਰੇ ਕੰਟਰੋਲ ਵਰਟਿਸ / ਫਿੱਟ ਪੁਆਇੰਟ ਸਕੇਲ ਹੁੰਦੇ ਹਨ |
