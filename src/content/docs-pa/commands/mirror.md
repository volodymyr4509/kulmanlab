# Mirror

`mirror` ਕਮਾਂਡ ਚੁਣੀਆਂ ਐਂਟਿਟੀਆਂ ਦੀਆਂ ਪ੍ਰਤੀਬਿੰਬਿਤ ਕਾਪੀਆਂ ਬਣਾਉਂਦੀ ਹੈ ਜੋ ਦੋ-ਬਿੰਦੂ ਧੁਰੇ ਦੇ ਪਾਰ ਪ੍ਰਤੀਬਿੰਬਿਤ ਹੁੰਦੀਆਂ ਹਨ। ਮੂਲ **ਹਮੇਸ਼ਾ ਰੱਖੇ ਜਾਂਦੇ ਹਨ** — [Move](../move/) ਜਾਂ [Rotate](../rotate/) ਦੇ ਉਲਟ, Mirror ਕਦੇ ਵੀ ਮੌਜੂਦਾ ਐਂਟਿਟੀਆਂ ਨੂੰ ਸੋਧਦਾ ਨਹੀਂ; ਇਹ ਸਿਰਫ਼ ਨਵੀਆਂ ਜੋੜਦਾ ਹੈ।

## ਸ਼ੁਰੂ ਕਰਨ ਦੇ ਦੋ ਤਰੀਕੇ

**ਪਹਿਲਾਂ ਚੁਣੋ, ਫਿਰ ਪ੍ਰਤੀਬਿੰਬਿਤ ਕਰੋ** — ਪਹਿਲਾਂ ਐਂਟਿਟੀਆਂ ਚੁਣੋ, ਫਿਰ ਸਰਗਰਮ ਕਰੋ:

1. ਕੈਨਵਸ 'ਤੇ ਇੱਕ ਜਾਂ ਵੱਧ ਐਂਟਿਟੀਆਂ ਚੁਣੋ।
2. ਟਰਮੀਨਲ ਵਿੱਚ `mirror` ਟਾਈਪ ਕਰੋ ਜਾਂ **Mirror** ਟੂਲਬਾਰ ਬਟਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ।
3. ਮਿਰਰ ਧੁਰੇ ਦੇ **ਪਹਿਲੇ ਬਿੰਦੂ 'ਤੇ ਕਲਿੱਕ ਕਰੋ**, ਜਾਂ ਸਹੀ ਕੋਆਰਡੀਨੇਟ ਲਈ `X,Y` ਟਾਈਪ ਕਰੋ ਅਤੇ **Enter** ਦਬਾਓ।
4. **ਦੂਜੇ ਬਿੰਦੂ 'ਤੇ ਕਲਿੱਕ ਕਰੋ** — ਪ੍ਰਤੀਬਿੰਬਿਤ ਕਾਪੀਆਂ ਰੱਖੀਆਂ ਜਾਂਦੀਆਂ ਹਨ ਅਤੇ ਕਮਾਂਡ ਬੰਦ ਹੋ ਜਾਂਦੀ ਹੈ। ਕੋਆਰਡੀਨੇਟ ਐਂਟਰੀ ਇੱਥੇ ਵੀ ਕੰਮ ਕਰਦੀ ਹੈ।

**ਸਰਗਰਮ ਕਰੋ, ਫਿਰ ਚੁਣੋ** — ਕੁਝ ਵੀ ਚੁਣੇ ਬਿਨਾਂ ਕਮਾਂਡ ਸ਼ੁਰੂ ਕਰੋ:

1. `mirror` ਟਾਈਪ ਕਰੋ ਜਾਂ ਟੂਲਬਾਰ ਬਟਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ।
2. **ਆਬਜੈਕਟ ਚੁਣੋ** — ਟੌਗਲ ਕਰਨ ਲਈ ਕਲਿੱਕ ਕਰੋ, ਜਾਂ ਖੇਤਰ ਦੁਆਰਾ ਚੁਣਨ ਲਈ ਖਿੱਚੋ।
3. ਚੋਣ ਪੁਸ਼ਟੀ ਕਰਨ ਲਈ **Enter** ਜਾਂ **Space** ਦਬਾਓ।
4. ਮਿਰਰ ਧੁਰੇ ਦੇ **ਪਹਿਲੇ ਬਿੰਦੂ 'ਤੇ ਕਲਿੱਕ ਕਰੋ**, ਫਿਰ **ਦੂਜੇ ਬਿੰਦੂ 'ਤੇ ਕਲਿੱਕ ਕਰੋ** (ਦੋਵੇਂ ਪੜਾਵਾਂ 'ਤੇ ਕੋਆਰਡੀਨੇਟ ਐਂਟਰੀ ਉਪਲਬਧ)।

```
  Original:          Mirror axis:        Result:
                     |
  [entity A]    →    |    →    [entity A] + [mirrored A]
                     |
```

ਦੂਜੇ ਧੁਰਾ ਬਿੰਦੂ ਨੂੰ ਰੱਖਦੇ ਸਮੇਂ ਪ੍ਰਤੀਬਿੰਬਿਤ ਕਾਪੀਆਂ ਦਾ ਇੱਕ ਲਾਈਵ ਪ੍ਰੀਵਿਊ ਕਰਸਰ ਦਾ ਪਿੱਛਾ ਕਰਦਾ ਹੈ।

## ਮਿਰਰ ਧੁਰਾ

ਧੁਰਾ ਦੋ ਕਲਿੱਕ ਕੀਤੇ ਬਿੰਦੂਆਂ ਵਿੱਚੋਂ ਲੰਘਦੀ ਇੱਕ ਅਨੰਤ ਲਾਈਨ ਹੈ। ਇਹ ਕਿਸੇ ਵੀ ਕੋਣ 'ਤੇ ਹੋ ਸਕਦੀ ਹੈ:

- ਕਰਸਰ ਨੂੰ ਇੱਕ **45° ਸਨੈਪ ਧੁਰੇ** (0°, 45°, 90°, 135°, …) ਦੇ ਨੇੜੇ ਲਿਜਾਓ ਅਤੇ ਧੁਰਾ ਉਸ ਕੋਣ 'ਤੇ ਲਾਕ ਹੋ ਜਾਂਦਾ ਹੈ — ਸਾਫ਼ ਹੌਰੀਜ਼ੌਂਟਲ, ਵਰਟੀਕਲ, ਜਾਂ ਡਾਇਗਨਲ ਪ੍ਰਤੀਬਿੰਬਾਂ ਲਈ ਲਾਭਦਾਇਕ।
- ਸਨੈਪ ਜ਼ੋਨ ਤੋਂ ਦੂਰ ਕਲਿੱਕ ਕਰਨ ਨਾਲ ਇੱਕ ਸੁਤੰਤਰ-ਕੋਣ ਧੁਰਾ ਬਣਦਾ ਹੈ।

## ਕੋਆਰਡੀਨੇਟ ਐਂਟਰੀ

ਕਿਸੇ ਵੀ ਧੁਰਾ-ਬਿੰਦੂ ਪੜਾਅ 'ਤੇ, ਕਲਿੱਕ ਕਰਨ ਦੀ ਬਜਾਏ ਸਹੀ ਸਥਿਤੀ ਟਾਈਪ ਕਰੋ:

1. X ਮੁੱਲ ਟਾਈਪ ਕਰੋ।
2. `,` ਦਬਾਓ — ਟਰਮੀਨਲ `[X], [Y{cursor}]` ਦਿਖਾਉਂਦਾ ਹੈ।
3. Y ਮੁੱਲ ਟਾਈਪ ਕਰੋ।
4. ਪੁਸ਼ਟੀ ਕਰਨ ਲਈ **Enter** ਦਬਾਓ।

## ਕੀਬੋਰਡ ਰੈਫਰੈਂਸ

| ਕੀ | ਕਾਰਵਾਈ |
|-----|--------|
| `Enter` / `Space` | ਚੋਣ ਪੁਸ਼ਟੀ ਕਰੋ |
| `0`–`9`, `.`, `-` | X ਕੋਆਰਡੀਨੇਟ ਐਂਟਰੀ ਸ਼ੁਰੂ ਕਰੋ |
| `,` | X ਲਾਕ ਕਰੋ ਅਤੇ Y ਐਂਟਰੀ 'ਤੇ ਜਾਓ |
| `Backspace` | ਆਖਰੀ ਟਾਈਪ ਕੀਤਾ ਅੱਖਰ ਮਿਟਾਓ |
| `Enter` | ਟਾਈਪ ਕੀਤਾ ਕੋਆਰਡੀਨੇਟ ਪੁਸ਼ਟੀ ਕਰੋ |
| `Escape` | ਰੱਦ ਕਰੋ ਅਤੇ ਰੀਸੈੱਟ ਕਰੋ |

## ਕਮਾਂਡ ਦੌਰਾਨ ਚੋਣ

| ਢੰਗ | ਵਿਵਹਾਰ |
|--------|-----------|
| **ਕਲਿੱਕ** | ਕਰਸਰ ਹੇਠਾਂ ਐਂਟਿਟੀ ਟੌਗਲ ਕਰਦਾ ਹੈ |
| **ਸੱਜੇ ਖਿੱਚੋ** (ਸਖ਼ਤ) | ਬਾਕਸ ਦੇ ਪੂਰੀ ਤਰ੍ਹਾਂ ਅੰਦਰ ਵਾਲੀਆਂ ਐਂਟਿਟੀਆਂ ਜੋੜਦਾ ਹੈ |
| **ਖੱਬੇ ਖਿੱਚੋ** (ਕ੍ਰਾਸਿੰਗ) | ਬਾਕਸ ਨੂੰ ਕੱਟਣ ਵਾਲੀਆਂ ਐਂਟਿਟੀਆਂ ਜੋੜਦਾ ਹੈ |
| **Enter** / **Space** | ਚੋਣ ਪੁਸ਼ਟੀ ਕਰਦਾ ਹੈ |

## ਕੀ ਪ੍ਰਤੀਬਿੰਬਿਤ ਹੁੰਦਾ ਹੈ

ਹਰੇਕ ਐਂਟਿਟੀ ਕਿਸਮ ਸਮਰਥਿਤ ਹੈ। ਜਿਓਮੈਟਰੀ ਗਣਿਤਕ ਤੌਰ 'ਤੇ ਧੁਰੇ ਦੇ ਪਾਰ ਪ੍ਰਤੀਬਿੰਬਿਤ ਕੀਤੀ ਜਾਂਦੀ ਹੈ:

| ਐਂਟਿਟੀ | ਕੀ ਬਦਲਦਾ ਹੈ |
|--------|-------------|
| Line | ਦੋਵੇਂ ਅੰਤ ਬਿੰਦੂ ਪ੍ਰਤੀਬਿੰਬਿਤ |
| Circle | ਕੇਂਦਰ ਪ੍ਰਤੀਬਿੰਬਿਤ; ਰੇਡੀਅਸ ਬਦਲਿਆ ਨਹੀਂ |
| Arc | ਕੇਂਦਰ ਪ੍ਰਤੀਬਿੰਬਿਤ; ਸ਼ੁਰੂ ਅਤੇ ਅੰਤ ਕੋਣਾਂ ਦੀ ਧੁਰੇ ਦੇ ਪਾਰ ਮੁੜ-ਗਣਨਾ |
| Ellipse | ਕੇਂਦਰ ਪ੍ਰਤੀਬਿੰਬਿਤ; ਮੇਜਰ ਧੁਰੇ ਦੀ ਦਿਸ਼ਾ ਧੁਰੇ ਦੇ ਪਾਰ ਫਲਿੱਪ |
| Polyline / Rectangle | ਹਰੇਕ ਵਰਟੈਕਸ ਪ੍ਰਤੀਬਿੰਬਿਤ |
| Text | ਐਂਕਰ ਪੁਆਇੰਟ ਪ੍ਰਤੀਬਿੰਬਿਤ; ਟੈਕਸਟ ਸਟਰਿੰਗ **ਉਲਟੀ ਨਹੀਂ** ਹੁੰਦੀ |
| Spline | ਸਾਰੇ ਕੰਟਰੋਲ ਵਰਟਿਸ / ਫਿੱਟ ਪੁਆਇੰਟ ਪ੍ਰਤੀਬਿੰਬਿਤ |
