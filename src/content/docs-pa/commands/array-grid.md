# Array Grid

`ArrayGrid` ਕਮਾਂਡ ਚੁਣੀਆਂ ਐਂਟਿਟੀਆਂ ਤੋਂ ਇੱਕ ਆਇਤਾਕਾਰ ਗਰਿੱਡ ਦੀਆਂ ਕਾਪੀਆਂ ਬਣਾਉਂਦੀ ਹੈ — ਕਤਾਰਾਂ ਦੀ ਗਿਣਤੀ, ਕਾਲਮਾਂ ਦੀ ਗਿਣਤੀ, ਅਤੇ ਉਹਨਾਂ ਦੇ ਵਿਚਕਾਰ ਦੀ ਦੂਰੀ ਦਰਜ ਕਰੋ, ਸਭ ਕੁਝ ਟਰਮੀਨਲ ਵਿੱਚ ਟਾਈਪ ਕੀਤਾ ਜਾਂਦਾ ਹੈ। ਮੂਲ ਚੋਣ ਕਤਾਰ 0, ਕਾਲਮ 0 ਸੈੱਲ 'ਤੇ ਰਹਿੰਦੀ ਹੈ; ਹਰ ਹੋਰ ਸੈੱਲ ਇੱਕ ਸਥਾਨਾਂਤਰਿਤ ਕਾਪੀ ਹੈ।

## ਸ਼ੁਰੂ ਕਰਨ ਦੇ ਦੋ ਤਰੀਕੇ

**ਪਹਿਲਾਂ ਚੁਣੋ, ਫਿਰ array ਕਰੋ** — ਪਹਿਲਾਂ ਐਂਟਿਟੀਆਂ ਚੁਣੋ, ਫਿਰ ਸਰਗਰਮ ਕਰੋ:

1. ਕੈਨਵਸ 'ਤੇ ਇੱਕ ਜਾਂ ਵੱਧ ਐਂਟਿਟੀਆਂ ਚੁਣੋ।
2. ਟਰਮੀਨਲ ਵਿੱਚ `arraygrid` ਟਾਈਪ ਕਰੋ (ਸਿਰਫ਼ `arr` ਹੀ ਕਾਫ਼ੀ ਹੈ — ਇਹ ਸਪਸ਼ਟ ਹੈ) ਜਾਂ **Array Grid** ਟੂਲਬਾਰ ਬਟਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ।
3. **ਕਤਾਰਾਂ** ਦੀ ਗਿਣਤੀ ਟਾਈਪ ਕਰੋ ਅਤੇ **Enter** ਦਬਾਓ।
4. **ਕਾਲਮਾਂ** ਦੀ ਗਿਣਤੀ ਟਾਈਪ ਕਰੋ ਅਤੇ **Enter** ਦਬਾਓ।
5. **ਕਤਾਰਾਂ ਵਿਚਕਾਰ ਦੂਰੀ** ਟਾਈਪ ਕਰੋ ਅਤੇ **Enter** ਦਬਾਓ।
6. **ਕਾਲਮਾਂ ਵਿਚਕਾਰ ਦੂਰੀ** ਟਾਈਪ ਕਰੋ ਅਤੇ **Enter** ਦਬਾਓ — ਗਰਿੱਡ ਤੁਰੰਤ ਬਣ ਜਾਂਦਾ ਹੈ।

**ਸਰਗਰਮ ਕਰੋ, ਫਿਰ ਚੁਣੋ** — ਕੁਝ ਵੀ ਚੁਣੇ ਬਿਨਾਂ ਕਮਾਂਡ ਸ਼ੁਰੂ ਕਰੋ:

1. `arraygrid` ਟਾਈਪ ਕਰੋ ਜਾਂ ਟੂਲਬਾਰ ਬਟਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ।
2. **ਆਬਜੈਕਟ ਚੁਣੋ** — ਵਿਅਕਤੀਗਤ ਐਂਟਿਟੀਆਂ ਟੌਗਲ ਕਰਨ ਲਈ ਕਲਿੱਕ ਕਰੋ, ਜਾਂ ਖੇਤਰ ਦੁਆਰਾ ਚੁਣਨ ਲਈ ਖਿੱਚੋ।
3. ਚੋਣ ਪੁਸ਼ਟੀ ਕਰਨ ਲਈ **Enter** ਜਾਂ **Space** ਦਬਾਓ।
4. ਉੱਪਰ ਵਾਂਗ ਕਤਾਰਾਂ → ਕਾਲਮਾਂ → ਕਤਾਰ ਦੂਰੀ → ਕਾਲਮ ਦੂਰੀ ਨਾਲ ਜਾਰੀ ਰੱਖੋ।

```
  2 ਕਤਾਰਾਂ x 3 ਕਾਲਮ:

  [B] [B] [B]   <- ਕਤਾਰ 1 (ਸਥਾਨਾਂਤਰਿਤ ਕਾਪੀਆਂ)
  [A] [A] [A]   <- ਕਤਾਰ 0: ਮੂਲ ਚੋਣ, ਕਾਪੀਆਂ ਸੱਜੇ ਪਾਸੇ
```

> ਟਰਮੀਨਲ ਨੂੰ ਸਿਰਫ਼ ਇੰਨੇ ਅੱਖਰਾਂ ਦੀ ਲੋੜ ਹੁੰਦੀ ਹੈ ਜੋ ਸਪਸ਼ਟ ਹੋਣ ਲਈ ਕਾਫ਼ੀ ਹੋਣ — `arr` ਟਾਈਪ ਕਰਨਾ ਅਤੇ **Enter** ਦਬਾਉਣਾ ਸਿੱਧੇ Array Grid ਨੂੰ ਸਰਗਰਮ ਕਰਦਾ ਹੈ, ਕਿਉਂਕਿ ਕੋਈ ਹੋਰ ਕਮਾਂਡ ਨਾਮ ਇਹਨਾਂ ਤਿੰਨ ਅੱਖਰਾਂ ਨਾਲ ਸ਼ੁਰੂ ਨਹੀਂ ਹੁੰਦਾ (Arc, Area, Align, ਅਤੇ Angle ਸਭ ਪਹਿਲਾਂ ਹੀ ਵੱਖ ਹੋ ਜਾਂਦੇ ਹਨ)।

## ਕਤਾਰਾਂ, ਕਾਲਮ ਅਤੇ ਦੂਰੀ

| ਪ੍ਰੋਂਪਟ | ਸਵੀਕਾਰ ਕਰਦਾ ਹੈ | ਨੋਟਸ |
|---------|----------------|------|
| ਕਤਾਰਾਂ | ਸਕਾਰਾਤਮਕ ਪੂਰੇ ਨੰਬਰ (1, 2, 3…) | ਸਿਰਫ਼ ਅੰਕ — ਕੋਈ ਦਸ਼ਮਲਵ ਬਿੰਦੂ ਜਾਂ ਚਿੰਨ੍ਹ ਨਹੀਂ |
| ਕਾਲਮ | ਸਕਾਰਾਤਮਕ ਪੂਰੇ ਨੰਬਰ (1, 2, 3…) | ਸਿਰਫ਼ ਅੰਕ — ਕੋਈ ਦਸ਼ਮਲਵ ਬਿੰਦੂ ਜਾਂ ਚਿੰਨ੍ਹ ਨਹੀਂ |
| ਕਤਾਰ ਦੂਰੀ | ਚਿੰਨ੍ਹਿਤ ਦਸ਼ਮਲਵ (ਜਿਵੇਂ `10`, `-5.5`) | ਕਤਾਰਾਂ ਵਿਚਕਾਰ ਦੂਰੀ; ਨਕਾਰਾਤਮਕ ਦਿਸ਼ਾ ਉਲਟਾਉਂਦਾ ਹੈ |
| ਕਾਲਮ ਦੂਰੀ | ਚਿੰਨ੍ਹਿਤ ਦਸ਼ਮਲਵ (ਜਿਵੇਂ `10`, `-5.5`) | ਕਾਲਮਾਂ ਵਿਚਕਾਰ ਦੂਰੀ; ਨਕਾਰਾਤਮਕ ਦਿਸ਼ਾ ਉਲਟਾਉਂਦਾ ਹੈ |

1 ਕਤਾਰ ਅਤੇ 1 ਕਾਲਮ ਨਾਲ, ਕੋਈ ਕਾਪੀ ਨਹੀਂ ਬਣਦੀ — ਕਮਾਂਡ ਡਰਾਇੰਗ ਨੂੰ ਬਦਲੇ ਬਿਨਾਂ ਬੰਦ ਹੋ ਜਾਂਦੀ ਹੈ।

## array ਤੋਂ ਬਾਅਦ

ਨਵੀਆਂ ਕਾਪੀਆਂ ਡਰਾਇੰਗ ਵਿੱਚ ਜੋੜੀਆਂ ਜਾਂਦੀਆਂ ਹਨ ਅਤੇ ਕਮਾਂਡ ਬੰਦ ਹੋ ਜਾਂਦੀ ਹੈ — ਮੂਲ ਚੋਣ ਸਾਫ਼ ਹੋ ਜਾਂਦੀ ਹੈ। **Array Grid** ਦੁਬਾਰਾ ਚਲਾਓ, ਜਾਂ ਇੱਕ ਨਵੀਂ ਕਮਾਂਡ ਸ਼ੁਰੂ ਕਰੋ।

## ਸਮਰਥਿਤ ਐਂਟਿਟੀਆਂ

Array Grid ਹਰੇਕ ਐਂਟਿਟੀ ਕਿਸਮ 'ਤੇ ਕੰਮ ਕਰਦੀ ਹੈ। ਸਾਰੀਆਂ ਐਂਟਿਟੀਆਂ ਅੰਦਰੂਨੀ ਤੌਰ 'ਤੇ `translate(dx, dy)` ਲਾਗੂ ਕਰਦੀਆਂ ਹਨ — ਉਹੀ ਓਪਰੇਸ਼ਨ ਜੋ [Copy](../copy/) ਅਤੇ [Move](../move/) ਵਰਤਦੇ ਹਨ — ਇਸ ਲਈ ਕੋਈ ਵੀ ਬਾਹਰ ਨਹੀਂ ਹੈ।
