# Spline CV

`splinecv` ਕਮਾਂਡ ਕੰਟਰੋਲ ਵਰਟਿਸਿਸ (CVs) ਰੱਖ ਕੇ ਇੱਕ **ਕਿਊਬਿਕ B-spline** ਖਿੱਚਦੀ ਹੈ। ਨਤੀਜੇ ਵਾਲਾ ਕਰਵ ਹਰੇਕ CV ਵੱਲ ਖਿੱਚਿਆ ਜਾਂਦਾ ਹੈ ਪਰ ਉਹਨਾਂ ਵਿੱਚੋਂ ਨਹੀਂ ਲੰਘਦਾ — ਸਿਵਾਏ ਬਿਲਕੁਲ ਪਹਿਲੇ ਅਤੇ ਆਖਰੀ ਵਰਟੈਕਸ ਦੇ, ਜਿੱਥੇ **clamped knots** ਕਰਵ ਨੂੰ ਬਿਲਕੁਲ ਐਂਕਰ ਕਰਦੇ ਹਨ। ਇਹ ਤੁਹਾਨੂੰ ਸਹਿਜ ਆਕਾਰ ਕੰਟਰੋਲ ਦਿੰਦਾ ਹੈ: ਹਰ ਪੁਆਇੰਟ ਨੂੰ ਛੂਹਣ ਲਈ ਮਜਬੂਰ ਕੀਤੇ ਬਿਨਾਂ ਕਰਵ ਨੂੰ ਇੱਕ ਵਰਟੈਕਸ ਵੱਲ ਧੱਕਣ ਲਈ ਉਸਨੂੰ ਖਿੱਚੋ।

## ਕੰਟਰੋਲ ਵਰਟਿਸਿਸ ਦੁਆਰਾ ਸਪਲਾਈਨ ਖਿੱਚਣਾ

1. ਟਰਮੀਨਲ ਵਿੱਚ `splinecv` ਟਾਈਪ ਕਰੋ ਜਾਂ **Spline CV** ਟੂਲਬਾਰ ਬਟਨ ਉੱਤੇ ਕਲਿੱਕ ਕਰੋ।
2. **ਕੰਟਰੋਲ ਵਰਟਿਸਿਸ ਰੱਖਣ ਲਈ ਕਲਿੱਕ ਕਰੋ** — ਹਰੇਕ ਕਲਿੱਕ ਇੱਕ ਵਰਟੈਕਸ ਜੋੜਦਾ ਹੈ। ਜਾਂ ਸਟੀਕ ਕੋਆਰਡੀਨੇਟ ਲਈ `X,Y` ਟਾਈਪ ਕਰੋ ਅਤੇ **Enter** ਦਬਾਓ।
3. ਪੂਰਾ ਕਰਨ ਲਈ **Enter** ਦਬਾਓ (ਘੱਟੋ-ਘੱਟ 2 ਵਰਟਿਸਿਸ ਲੋੜੀਂਦੀਆਂ ਹਨ)।

```
  CV ●         ● CV
      \       /
       \     /    ← curve pulled toward CVs
        \   /         but not passing through them
  CV ●   ●   ● CV (start/end: curve touches here)
```

ਲਾਈਵ ਪ੍ਰੀਵਿਊ ਹਰੇਕ ਵਰਟੈਕਸ ਤੋਂ ਬਾਅਦ ਕਰਸਰ ਹਿਲਾਉਣ ਦੇ ਨਾਲ ਅੱਪਡੇਟ ਹੁੰਦਾ ਹੈ, ਦਿਖਾਉਂਦਾ ਹੈ ਕਿ ਕਰਸਰ ਪੋਜ਼ੀਸ਼ਨ ਤੇ ਅਗਲੇ ਪੁਆਇੰਟ ਨਾਲ ਸਪਲਾਈਨ ਕਿਵੇਂ ਦਿਖੇਗੀ। ਸਾਰੇ ਰੱਖੇ ਵਰਟਿਸਿਸ ਨੂੰ ਰੱਦ ਕਰਨ ਅਤੇ ਬਾਹਰ ਨਿਕਲਣ ਲਈ **Escape** ਦਬਾਓ।

## ਕੋਆਰਡੀਨੇਟ ਐਂਟਰੀ

ਕਲਿੱਕ ਕਰਨ ਦੀ ਬਜਾਏ, ਕਿਸੇ ਵੀ ਕੰਟਰੋਲ ਵਰਟੈਕਸ ਲਈ ਸਟੀਕ ਸਥਿਤੀ ਟਾਈਪ ਕਰੋ:

1. X ਮੁੱਲ ਟਾਈਪ ਕਰੋ।
2. `,` ਦਬਾਓ — ਟਰਮੀਨਲ `[X], [Y{cursor}]` ਦਿਖਾਉਂਦਾ ਹੈ।
3. Y ਮੁੱਲ ਟਾਈਪ ਕਰੋ।
4. ਵਰਟੈਕਸ ਰੱਖਣ ਲਈ **Enter** ਦਬਾਓ।

## ਕੀਬੋਰਡ ਰੈਫਰੈਂਸ

| ਕੀ | ਐਕਸ਼ਨ |
|-----|--------|
| `0`–`9`, `.`, `-` | X ਕੋਆਰਡੀਨੇਟ ਐਂਟਰੀ ਸ਼ੁਰੂ ਕਰੋ |
| `,` | X ਲੌਕ ਕਰੋ ਅਤੇ Y ਐਂਟਰੀ ਵੱਲ ਜਾਓ |
| `Backspace` | ਆਖਰੀ ਟਾਈਪ ਕੀਤਾ ਅੱਖਰ ਮਿਟਾਓ |
| `Enter` | ਟਾਈਪ ਕੀਤੇ ਕੋਆਰਡੀਨੇਟ ਦੀ ਪੁਸ਼ਟੀ ਕਰੋ, ਜਾਂ ਜੇ ਕੋਈ ਇਨਪੁੱਟ ਪ੍ਰਗਤੀ ਵਿੱਚ ਨਹੀਂ ਹੈ ਅਤੇ ≥ 2 ਵਰਟਿਸਿਸ ਮੌਜੂਦ ਹਨ ਤਾਂ ਸਪਲਾਈਨ ਪੂਰਾ ਕਰੋ |
| `Escape` | ਸਾਰੇ ਵਰਟਿਸਿਸ ਰੱਦ ਕਰੋ ਅਤੇ ਬਾਹਰ ਨਿਕਲੋ |

## ਗ੍ਰਿਪ ਸੰਪਾਦਨ — ਕੰਟਰੋਲ ਵਰਟਿਸਿਸ ਰਾਹੀਂ ਮੁੜ-ਆਕਾਰ ਦੇਣਾ

ਇੱਕ ਚੁਣੀ ਗਈ CV ਸਪਲਾਈਨ ਹਰੇਕ ਕੰਟਰੋਲ ਵਰਟੈਕਸ ਲਈ ਇੱਕ ਗ੍ਰਿਪ ਦਿਖਾਉਂਦੀ ਹੈ:

| ਗ੍ਰਿਪ | ਸਥਿਤੀ | ਇਹ ਕੀ ਕਰਦਾ ਹੈ |
|------|----------|--------------|
| **ਕੰਟਰੋਲ ਵਰਟੈਕਸ** | ਹਰੇਕ CV ਸਥਿਤੀ ਤੇ | ਉਸ CV ਨੂੰ ਹਿਲਾਉਣ ਲਈ ਖਿੱਚੋ — ਕਰਵ ਨਵੀਂ ਸਥਿਤੀ ਵੱਲ ਮੁੜ-ਆਕਾਰ ਹੁੰਦਾ ਹੈ |

ਕੋਈ "ਪੂਰੀ ਸਪਲਾਈਨ ਹਿਲਾਓ" ਗ੍ਰਿਪ ਨਹੀਂ ਹੈ। ਪੂਰੀ ਸਪਲਾਈਨ ਨੂੰ ਟ੍ਰਾਂਸਲੇਟ ਕਰਨ ਲਈ, [Move](../move/) ਕਮਾਂਡ ਵਰਤੋ।

## DXF — SPLINE ਐਂਟਿਟੀ (ਕੰਟਰੋਲ-ਵਰਟੈਕਸ ਫਾਰਮ)

CV ਸਪਲਾਈਨਾਂ DXF ਫਾਈਲ ਵਿੱਚ `SPLINE` ਐਂਟਿਟੀਜ਼ ਵਜੋਂ ਸੇਵ ਹੁੰਦੀਆਂ ਹਨ, ਡਿਗਰੀ, knot ਵੈਕਟਰ, ਅਤੇ ਸਾਰੇ ਕੰਟਰੋਲ ਵਰਟੈਕਸ ਕੋਆਰਡੀਨੇਟ ਸਟੋਰ ਕਰਦੀਆਂ ਹਨ। ਸਾਰੀਆਂ ਵਿਸ਼ੇਸ਼ਤਾਵਾਂ — ਰੰਗ, ਲੇਅਰ, ਲਾਈਨਟਾਈਪ, ਲਾਈਨਟਾਈਪ ਸਕੇਲ, ਅਤੇ ਮੋਟਾਈ — ਬਿਨਾਂ ਨੁਕਸਾਨ ਦੇ ਰਾਊਂਡ-ਟ੍ਰਿਪ ਹੁੰਦੀਆਂ ਹਨ। `splineFlag` ਨੂੰ `9` (CV ਸਪਲਾਈਨ) ਸੈੱਟ ਕੀਤਾ ਜਾਂਦਾ ਹੈ ਤਾਂ ਜੋ ਰੀਲੋਡ ਤੇ ਫਾਰਮ ਸੁਰੱਖਿਅਤ ਰਹੇ। ਕੋਈ ਵੀ DXF ਐਪਲੀਕੇਸ਼ਨ ਜੋ CV ਡਾਟਾ ਵਾਲੇ `SPLINE` ਐਂਟਿਟੀਜ਼ ਸਮਰਥਨ ਕਰਦੀ ਹੈ ਇਹਨਾਂ ਨੂੰ ਸਹੀ ਢੰਗ ਨਾਲ ਪੜ੍ਹਦੀ ਹੈ।
