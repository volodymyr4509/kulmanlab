# Help

ਸਾਰੇ ਕੀਬੋਰਡ ਸ਼ਾਰਟਕੱਟਾਂ, ਨੈਵੀਗੇਸ਼ਨ ਟਿਪਸ, ਅਤੇ ਆਮ ਵਰਤੋਂ ਦੇ ਸੰਖੇਪ ਨਾਲ ਐਪ-ਅੰਦਰੂਨੀ ਹੈਲਪ ਪੈਨਲ ਖੋਲ੍ਹਦਾ ਹੈ।

## ਕਿਵੇਂ ਵਰਤੀਏ

**Help** ਟੂਲਬਾਰ ਬਟਨ ਉੱਤੇ ਕਲਿੱਕ ਕਰੋ ਜਾਂ ਟਰਮੀਨਲ ਵਿੱਚ `help` ਟਾਈਪ ਕਰੋ। ਹੈਲਪ ਪੈਨਲ ਤੁਰੰਤ ਖੁੱਲ੍ਹਦਾ ਹੈ ਅਤੇ ਕਮਾਂਡ ਬਾਹਰ ਨਿਕਲ ਜਾਂਦੀ ਹੈ।

## ਹੈਲਪ ਪੈਨਲ ਵਿੱਚ ਕੀ ਹੈ

### ਨੈਵੀਗੇਸ਼ਨ
| ਸ਼ਾਰਟਕੱਟ | ਐਕਸ਼ਨ |
|----------|--------|
| `Scroll wheel` | ਜ਼ੂਮ ਇਨ / ਆਊਟ |
| `Middle click + drag` | ਦ੍ਰਿਸ਼ ਪੈਨ ਕਰੋ |
| `Double middle click` | ਸਾਰੀਆਂ ਐਂਟਿਟੀਜ਼ ਸਕ੍ਰੀਨ ਤੇ ਫਿੱਟ ਕਰੋ |

### ਕਮਾਂਡਾਂ
| ਸ਼ਾਰਟਕੱਟ | ਐਕਸ਼ਨ |
|----------|--------|
| ਟਾਈਪ ਕਰੋ | ਕਮਾਂਡ ਨਾਮ ਟਾਈਪ ਕਰਨਾ ਸ਼ੁਰੂ ਕਰੋ — ਸੁਝਾਅ ਕੈਨਵਸ ਉੱਤੇ ਦਿਖਾਈ ਦਿੰਦੇ ਹਨ |
| `Delete` | ਸਾਰੀਆਂ ਚੁਣੀਆਂ ਐਂਟਿਟੀਜ਼ ਮਿਟਾਓ |
| `Enter` / `Space` | ਕਮਾਂਡ ਜਾਂ ਇਨਪੁੱਟ ਦੀ ਪੁਸ਼ਟੀ ਕਰੋ |
| `Enter` / `Space` (ਵਿਹਲਾ) | ਆਖਰੀ ਵਰਤੀ ਗਈ ਕਮਾਂਡ ਦੁਹਰਾਓ |
| `Escape` | ਮੌਜੂਦਾ ਕਮਾਂਡ ਰੱਦ ਕਰੋ ਜਾਂ ਚੋਣ ਸਾਫ਼ ਕਰੋ |

### ਚੋਣ
| ਸ਼ਾਰਟਕੱਟ | ਐਕਸ਼ਨ |
|----------|--------|
| `Left click` | ਇੱਕ ਐਂਟਿਟੀ ਚੁਣੋ |
| `Drag →` | ਸਿਰਫ਼ ਉਹ ਐਂਟਿਟੀਜ਼ ਚੁਣੋ ਜੋ ਡ੍ਰੈਗ ਖੇਤਰ ਦੇ **ਪੂਰੀ ਤਰ੍ਹਾਂ ਅੰਦਰ** ਹਨ |
| `Drag ←` | ਕੋਈ ਵੀ ਐਂਟਿਟੀ ਚੁਣੋ ਜੋ ਡ੍ਰੈਗ ਖੇਤਰ ਨੂੰ **ਛੂੰਹਦੀ** ਹੈ |
| `Left click grip` | ਵਰਟੈਕਸ ਜਾਂ ਐਂਡਪੁਆਇੰਟ ਸਿੱਧਾ ਸੰਪਾਦਿਤ ਕਰੋ |

### ਡਰਾਇੰਗ ਟਿਪਸ
- **ਸਨੈਪ ਪੁਆਇੰਟ** — ਕਰਸਰ ਆਪਣੇ ਆਪ ਨੇੜਲੇ ਐਂਟਿਟੀ ਐਂਡਪੁਆਇੰਟਾਂ ਵੱਲ ਸਨੈਪ ਹੁੰਦਾ ਹੈ।
- **ਸਟਿੱਕੀ ਐਂਗਲ** — 45° ਵਾਧਿਆਂ ਵਿੱਚ ਲੌਕ ਹੋਣ ਲਈ ਕਰਸਰ ਨੂੰ ਇੱਕ ਦਿਸ਼ਾ ਵਿੱਚ ਕਾਫ਼ੀ ਦੂਰ ਲਿਜਾਓ।
- **ਸਟੀਕ ਲੰਬਾਈ** — ਜਦੋਂ ਐਂਗਲ ਲੌਕ ਹੋਵੇ, ਸਟੀਕ ਦੂਰੀ ਸੈੱਟ ਕਰਨ ਲਈ ਇੱਕ ਨੰਬਰ ਟਾਈਪ ਕਰੋ ਅਤੇ **Enter** ਦਬਾਓ।
