# Ellipse

`ellipse` ਕਮਾਂਡ ਤਿੰਨ ਕਲਿੱਕਾਂ ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਇੱਕ ਐਲਿਪਸ ਖਿੱਚਦੀ ਹੈ: ਇੱਕ ਕੇਂਦਰ ਬਿੰਦੂ, ਕਿਸੇ ਵੀ ਕੋਣ 'ਤੇ ਪਹਿਲੇ (ਮੇਜਰ) ਸੈਮੀ-ਧੁਰੇ ਦਾ ਅੰਤ ਬਿੰਦੂ, ਅਤੇ ਦੂਜੇ (ਮਾਈਨਰ) ਸੈਮੀ-ਧੁਰੇ ਦੀ ਲੰਬਾਈ। ਦੋਵੇਂ ਧੁਰੇ ਹਮੇਸ਼ਾ ਇੱਕ-ਦੂਜੇ ਦੇ ਲੰਬਵਤ ਹੁੰਦੇ ਹਨ — ਦੂਜੇ ਧੁਰੇ ਦੀ ਦਿਸ਼ਾ ਆਪਣੇ ਆਪ ਪਹਿਲੇ ਧੁਰੇ ਤੋਂ ਲਈ ਜਾਂਦੀ ਹੈ।

## ਐਲਿਪਸ ਖਿੱਚਣਾ

1. ਟਰਮੀਨਲ ਵਿੱਚ `ellipse` ਟਾਈਪ ਕਰੋ ਜਾਂ **Ellipse** ਟੂਲਬਾਰ ਬਟਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ।
2. **ਕੇਂਦਰ ਬਿੰਦੂ 'ਤੇ ਕਲਿੱਕ ਕਰੋ**, ਜਾਂ ਸਹੀ ਕੋਆਰਡੀਨੇਟ ਲਈ `X,Y` ਟਾਈਪ ਕਰੋ ਅਤੇ **Enter** ਦਬਾਓ।
3. **ਪਹਿਲੇ ਧੁਰੇ ਦੇ ਅੰਤ ਬਿੰਦੂ 'ਤੇ ਕਲਿੱਕ ਕਰੋ** — ਇਹ ਪਹਿਲੇ ਸੈਮੀ-ਧੁਰੇ ਦੀ ਦਿਸ਼ਾ ਅਤੇ ਲੰਬਾਈ ਦੋਵੇਂ ਸੈੱਟ ਕਰਦਾ ਹੈ। ਕੋਆਰਡੀਨੇਟ ਐਂਟਰੀ ਇੱਥੇ ਵੀ ਕੰਮ ਕਰਦੀ ਹੈ।
4. **ਦੂਜੇ ਧੁਰੇ ਦੀ ਲੰਬਾਈ ਸੈੱਟ ਕਰੋ** — ਕਰਸਰ ਨੂੰ ਪਹਿਲੇ ਧੁਰੇ ਦੇ ਲੰਬਵਤ ਹਿਲਾਓ, ਫਿਰ ਕਲਿੱਕ ਕਰੋ ਜਾਂ ਇੱਕ ਲੰਬਾਈ ਟਾਈਪ ਕਰੋ।

```
               ● ← ਪਹਿਲੇ ਧੁਰੇ ਦਾ ਅੰਤ ਬਿੰਦੂ (ਸਟੈਪ 3)
              /
  ਕੇਂਦਰ ●  /  ← ਪਹਿਲਾ ਧੁਰਾ (ਕੋਈ ਵੀ ਕੋਣ)
            |
            ● ← ਕਰਸਰ ਇੱਥੇ ਦੂਜੇ ਧੁਰੇ ਦੀ ਲੰਬਾਈ ਸੈੱਟ ਕਰਦਾ ਹੈ (ਸਟੈਪ 4)
```

ਸਟੈਪ 4 ਤੋਂ ਬਾਅਦ ਐਲਿਪਸ ਰੱਖੀ ਜਾਂਦੀ ਹੈ ਅਤੇ ਕਮਾਂਡ ਬੰਦ ਹੋ ਜਾਂਦੀ ਹੈ।

## ਧੁਰਾ ਇਨਪੁਟ — ਕਲਿੱਕ, ਕੋਆਰਡੀਨੇਟ, ਜਾਂ ਟਾਈਪ ਕੀਤੀ ਲੰਬਾਈ

**ਕੇਂਦਰ (ਸਟੈਪ 2):** ਸਹੀ ਸਥਿਤੀ ਲਈ ਕਲਿੱਕ ਕਰੋ ਜਾਂ `X,Y` ਟਾਈਪ ਕਰੋ।

**ਪਹਿਲੇ ਧੁਰੇ ਦਾ ਅੰਤ ਬਿੰਦੂ (ਸਟੈਪ 3):** ਕਲਿੱਕ ਕਰੋ, ਜਾਂ ਸਹੀ ਕੋਆਰਡੀਨੇਟ ਲਈ `X,Y` ਟਾਈਪ ਕਰੋ। ਐਂਗਲ ਲਾਕਿੰਗ 45° ਦੇ ਵਾਧੇ 'ਤੇ ਵੀ ਸਨੈਪ ਕਰਦੀ ਹੈ — ਲਾਕ ਹੋਣ ਵੇਲੇ, ਅੰਤ ਬਿੰਦੂ ਨੂੰ ਠੀਕ ਉਸ ਦੂਰੀ 'ਤੇ ਰੱਖਣ ਲਈ ਇੱਕ ਲੰਬਾਈ ਟਾਈਪ ਕਰੋ ਅਤੇ **Enter** ਦਬਾਓ।

**ਦੂਜਾ ਧੁਰਾ (ਸਟੈਪ 4):** ਟਾਈਪ ਕੀਤੀ ਲੰਬਾਈ ਹਮੇਸ਼ਾ ਉਪਲਬਧ ਹੁੰਦੀ ਹੈ — ਐਂਗਲ ਲਾਕ ਦੀ ਲੋੜ ਨਹੀਂ। ਦਿਸ਼ਾ ਪਹਿਲਾਂ ਹੀ ਪਹਿਲੇ ਧੁਰੇ ਦੇ ਲੰਬਵਤ ਸਥਿਰ ਹੁੰਦੀ ਹੈ; ਟਾਈਪ ਕਰਨ ਨਾਲ ਸਿਰਫ਼ ਲੰਬਾਈ ਸੈੱਟ ਹੁੰਦੀ ਹੈ।

| ਕੀ | ਕਾਰਵਾਈ |
|-----|--------|
| `0`–`9`, `.` | ਧੁਰੇ ਦੀ ਲੰਬਾਈ ਵਿੱਚ ਅੰਕ ਜੋੜੋ (ਦੂਜੇ ਧੁਰੇ ਦਾ ਪੜਾਅ) |
| `Backspace` | ਆਖਰੀ ਟਾਈਪ ਕੀਤਾ ਅੱਖਰ ਮਿਟਾਓ |
| `Enter` | ਟਾਈਪ ਕੀਤੀ ਲੰਬਾਈ 'ਤੇ ਧੁਰੇ ਦਾ ਅੰਤ ਬਿੰਦੂ ਰੱਖੋ |

## ਕੋਆਰਡੀਨੇਟ ਐਂਟਰੀ (ਕੇਂਦਰ ਅਤੇ ਪਹਿਲੇ ਧੁਰੇ ਦਾ ਅੰਤ ਬਿੰਦੂ)

1. X ਮੁੱਲ ਟਾਈਪ ਕਰੋ।
2. `,` ਦਬਾਓ — ਟਰਮੀਨਲ `[X], [Y{cursor}]` ਦਿਖਾਉਂਦਾ ਹੈ।
3. Y ਮੁੱਲ ਟਾਈਪ ਕਰੋ।
4. ਪੁਸ਼ਟੀ ਕਰਨ ਲਈ **Enter** ਦਬਾਓ।

## ਕੀਬੋਰਡ ਰੈਫਰੈਂਸ

| ਕੀ | ਕਾਰਵਾਈ |
|-----|--------|
| `0`–`9`, `.`, `-` | X ਕੋਆਰਡੀਨੇਟ ਐਂਟਰੀ ਸ਼ੁਰੂ ਕਰੋ (ਕੇਂਦਰ/ਪਹਿਲਾ-ਧੁਰਾ ਪੜਾਅ), ਜਾਂ ਐਂਗਲ-ਲਾਕ ਹੋਣ 'ਤੇ ਧੁਰੇ ਦੀ ਲੰਬਾਈ |
| `,` | X ਲਾਕ ਕਰੋ ਅਤੇ Y ਐਂਟਰੀ 'ਤੇ ਜਾਓ |
| `Backspace` | ਆਖਰੀ ਟਾਈਪ ਕੀਤਾ ਅੱਖਰ ਮਿਟਾਓ |
| `Enter` | ਟਾਈਪ ਕੀਤਾ ਕੋਆਰਡੀਨੇਟ ਜਾਂ ਲੰਬਾਈ ਪੁਸ਼ਟੀ ਕਰੋ |
| `Escape` | ਰੱਦ ਕਰੋ ਅਤੇ ਰੀਸੈੱਟ ਕਰੋ |

## ਗ੍ਰਿਪ ਸੰਪਾਦਨ — ਸੁਤੰਤਰ ਧੁਰਾ ਮੁੜ-ਆਕਾਰ

ਇੱਕ ਚੁਣੀ ਹੋਈ ਐਲਿਪਸ ਪੰਜ ਗ੍ਰਿਪ ਦਿਖਾਉਂਦੀ ਹੈ:

| ਗ੍ਰਿਪ | ਗਿਣਤੀ | ਇਹ ਕੀ ਕਰਦਾ ਹੈ |
|------|-------|--------------|
| **ਕੇਂਦਰ** | 1 | ਪੂਰੀ ਐਲਿਪਸ ਹਿਲਾਉਂਦਾ ਹੈ; ਦੋਵੇਂ ਧੁਰੇ ਬਦਲੇ ਨਹੀਂ ਰਹਿੰਦੇ |
| **ਮੇਜਰ ਧੁਰੇ ਦੇ ਅੰਤ ਬਿੰਦੂ** | 2 (ਲੰਬੇ ਧੁਰੇ ਦੇ ਉਲਟ ਸਿਰੇ) | ਮੇਜਰ ਸੈਮੀ-ਧੁਰੇ ਦੀ ਲੰਬਾਈ ਮੁੜ-ਆਕਾਰ ਦੇਣ ਲਈ ਖਿੱਚੋ; ਮਾਈਨਰ ਧੁਰੇ ਦਾ ਅਸਲ ਆਕਾਰ ਸਥਿਰ ਰਹਿੰਦਾ ਹੈ |
| **ਮਾਈਨਰ ਧੁਰੇ ਦੇ ਅੰਤ ਬਿੰਦੂ** | 2 (ਛੋਟੇ ਧੁਰੇ ਦੇ ਉਲਟ ਸਿਰੇ) | ਮਾਈਨਰ ਸੈਮੀ-ਧੁਰੇ ਦੀ ਲੰਬਾਈ ਮੁੜ-ਆਕਾਰ ਦੇਣ ਲਈ ਖਿੱਚੋ; ਮੇਜਰ ਧੁਰਾ ਬਦਲਿਆ ਨਹੀਂ ਰਹਿੰਦਾ |

ਮੇਜਰ ਅਤੇ ਮਾਈਨਰ ਧੁਰਾ ਗ੍ਰਿਪ ਸੁਤੰਤਰ ਹਨ — ਤੁਸੀਂ ਕਮਾਂਡ ਦੁਬਾਰਾ ਚਲਾਏ ਬਿਨਾਂ ਐਲਿਪਸ ਦਾ ਆਕਾਰ ਬਦਲ ਸਕਦੇ ਹੋ।

## DXF — ELLIPSE ਐਂਟਿਟੀ

ਐਲਿਪਸਾਂ ਨੂੰ DXF ਫਾਈਲ ਵਿੱਚ `ELLIPSE` ਐਂਟਿਟੀਆਂ ਦੇ ਰੂਪ ਵਿੱਚ ਸੰਭਾਲਿਆ ਜਾਂਦਾ ਹੈ। ਇਹ ਫਾਰਮੈਟ ਕੇਂਦਰ ਬਿੰਦੂ, ਪੂਰਾ ਮੇਜਰ ਧੁਰਾ ਵੈਕਟਰ (ਦਿਸ਼ਾ + ਲੰਬਾਈ), ਅਤੇ ਧੁਰਾ ਅਨੁਪਾਤ ਸਟੋਰ ਕਰਦਾ ਹੈ। ਘੁੰਮਾਓ, ਆਕਾਰ, ਅਤੇ ਸਾਰੀਆਂ ਸਟਾਈਲ ਵਿਸ਼ੇਸ਼ਤਾਵਾਂ ਬਿਨਾਂ ਕਿਸੇ ਨੁਕਸਾਨ ਦੇ ਰਾਊਂਡ-ਟ੍ਰਿਪ ਕਰਦੀਆਂ ਹਨ। ਇੱਕ ਸਰਕਲ ਨੂੰ ਡੀਜਨਰੇਟ ਐਲਿਪਸ ਦੇ ਰੂਪ ਵਿੱਚ **ਨਹੀਂ** ਸਟੋਰ ਕੀਤਾ ਜਾਂਦਾ — ਦੋਵੇਂ ਐਂਟਿਟੀ ਕਿਸਮਾਂ DXF ਮਾਡਲ ਵਿੱਚ ਵੱਖਰੀਆਂ ਰਹਿੰਦੀਆਂ ਹਨ।
