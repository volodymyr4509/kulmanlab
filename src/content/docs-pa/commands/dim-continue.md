# Dimension Continue

`dimcontinue` ਕਮਾਂਡ ਇੱਕ ਮੌਜੂਦਾ ਡਾਇਮੈਂਸ਼ਨ ਦੀ **ਦੂਜੀ ਐਕਸਟੈਂਸ਼ਨ ਲਾਈਨ** ਤੋਂ ਨਵੇਂ ਡਾਇਮੈਂਸ਼ਨ ਚੇਨ ਕਰਦੀ ਹੈ। ਹਰੇਕ ਨਵਾਂ ਸੈਗਮੈਂਟ ਬੇਸ ਵਾਂਗ ਉਸੇ ਮਾਪ ਧੁਰੇ 'ਤੇ ਅਤੇ ਉਸੇ ਡਾਇਮੈਂਸ਼ਨ-ਲਾਈਨ ਆਫਸੈੱਟ 'ਤੇ ਰੱਖਿਆ ਜਾਂਦਾ ਹੈ। ਸਾਰੀਆਂ ਸਟਾਈਲ ਵਿਸ਼ੇਸ਼ਤਾਵਾਂ — ਤੀਰ ਆਕਾਰ, ਟੈਕਸਟ ਉਚਾਈ, ਐਕਸਟੈਂਸ਼ਨ ਲਾਈਨ ਲੰਬਾਈਆਂ — ਬੇਸ ਤੋਂ ਆਪਣੇ-ਆਪ ਕਾਪੀ ਹੁੰਦੀਆਂ ਹਨ।

## ਚੇਨ ਕੀਤੇ ਡਾਇਮੈਂਸ਼ਨ ਕਿਵੇਂ ਦਿਖਦੇ ਹਨ

```
  |←— 3.00 —→|←— 2.50 —→|←— 4.00 —→|
  |           |           |           |
  ●           ●           ●           ●
  p1        p2 (base    p3           p4
           ext2 → new start)
```

ਹਰੇਕ ਆਇਤ ਇੱਕ ਵੱਖਰੀ `DIMENSION` ਐਂਟਿਟੀ ਹੈ। ਇਹ ਸਾਰੀਆਂ ਇੱਕੋ ਡਾਇਮੈਂਸ਼ਨ-ਲਾਈਨ ਸਥਿਤੀ ਅਤੇ ਮਾਪ ਦਿਸ਼ਾ ਸਾਂਝੀ ਕਰਦੀਆਂ ਹਨ।

## ਇੱਕ ਚੇਨ ਸ਼ੁਰੂ ਕਰਨਾ

1. ਟਰਮੀਨਲ ਵਿੱਚ `dimcontinue` ਟਾਈਪ ਕਰੋ ਜਾਂ **Dimension Continue** ਟੂਲਬਾਰ ਬਟਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ।
2. **ਜੇ ਹੁਣੇ ਹੀ ਇੱਕ ਡਾਇਮੈਂਸ਼ਨ ਰੱਖਿਆ ਗਿਆ ਹੈ** — ਕਮਾਂਡ ਇਸਨੂੰ ਆਪਣੇ-ਆਪ ਬੇਸ ਵਜੋਂ ਲੈ ਲੈਂਦੀ ਹੈ (ਕਲਿੱਕ ਦੀ ਲੋੜ ਨਹੀਂ)।
3. **ਜੇ ਕੋਈ ਹਾਲੀਆ ਡਾਇਮੈਂਸ਼ਨ ਮੌਜੂਦ ਨਹੀਂ ਹੈ** — ਇਸਨੂੰ ਬੇਸ ਵਜੋਂ ਵਰਤਣ ਲਈ ਕਿਸੇ ਵੀ ਮੌਜੂਦਾ ਡਾਇਮੈਂਸ਼ਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ।
4. **ਅਗਲੇ ਐਕਸਟੈਂਸ਼ਨ ਲਾਈਨ ਮੂਲ 'ਤੇ ਕਲਿੱਕ ਕਰੋ** — ਜਿਵੇਂ ਤੁਸੀਂ ਕਰਸਰ ਹਿਲਾਉਂਦੇ ਹੋ, ਇੱਕ ਪ੍ਰੀਵਿਊ ਨਵਾਂ ਡਾਇਮੈਂਸ਼ਨ ਦਿਖਾਉਂਦਾ ਹੈ। ਜਾਂ ਸਟੀਕ ਨਿਰਦੇਸ਼ਾਂਕ ਲਈ `X,Y` ਟਾਈਪ ਕਰੋ ਅਤੇ **Enter** ਦਬਾਓ।
5. ਚੇਨ ਵਧਾਉਣ ਲਈ ਕਲਿੱਕ (ਜਾਂ ਟਾਈਪ) ਕਰਦੇ ਰਹੋ। ਹਰੇਕ ਰੱਖਿਆ ਗਿਆ ਡਾਇਮੈਂਸ਼ਨ ਆਪਣੇ-ਆਪ ਨਵਾਂ ਬੇਸ ਬਣ ਜਾਂਦਾ ਹੈ।
6. ਚੇਨ ਖਤਮ ਕਰਨ ਲਈ **Enter**, **Space**, ਜਾਂ **Escape** ਦਬਾਓ।

|----------|---------------------|

| ਡਾਇਮੈਂਸ਼ਨ ਲਾਈਨ ਆਫਸੈੱਟ (ਮਾਪੇ ਗਏ ਬਿੰਦੂਆਂ ਤੋਂ ਦੂਰੀ) | ਹਾਂ |

## ਮਾਪ ਦਿਸ਼ਾ ਲਾਕਿੰਗ

ਚੇਨ ਦੀ ਮਾਪ ਦਿਸ਼ਾ **ਬੇਸ ਡਾਇਮੈਂਸ਼ਨ ਦੇ ਕੋਣ 'ਤੇ ਸਥਿਰ** ਹੁੰਦੀ ਹੈ:

- ਲੀਨੀਅਰ ਬੇਸ (H) → ਸਾਰੇ ਜਾਰੀ ਭਾਗ ਹਰੀਜ਼ੌਂਟਲ ਦੂਰੀ (Δ X) ਮਾਪਦੇ ਹਨ।
- ਲੀਨੀਅਰ ਬੇਸ (V) → ਸਾਰੇ ਜਾਰੀ ਭਾਗ ਵਰਟੀਕਲ ਦੂਰੀ (Δ Y) ਮਾਪਦੇ ਹਨ।
- ਕਿਸੇ ਵੀ ਕੋਣ 'ਤੇ ਅਲਾਈਨਡ ਬੇਸ → ਸਾਰੇ ਜਾਰੀ ਭਾਗ ਉਸੇ ਕੋਣ ਦੇ ਨਾਲ ਮਾਪਦੇ ਹਨ।

ਤੁਸੀਂ ਚੇਨ ਦੇ ਵਿਚਕਾਰ ਦਿਸ਼ਾ ਨਹੀਂ ਬਦਲ ਸਕਦੇ। ਵੱਖਰੀ ਦਿਸ਼ਾ ਵਿੱਚ ਡਾਇਮੈਂਸ਼ਨ ਕਰਨ ਲਈ ਨਵਾਂ [Dimension Linear](../dim-linear/) ਜਾਂ [Dimension Aligned](../dim-aligned/) ਸ਼ੁਰੂ ਕਰੋ।

## ਰੱਖਣ ਤੋਂ ਬਾਅਦ ਲੇਬਲ ਦਾ ਸੰਪਾਦਨ — ਸਿੰਪਲ ਮੋਡ

ਚੇਨ ਵਿੱਚ ਕਿਸੇ ਵੀ ਡਾਇਮੈਂਸ਼ਨ 'ਤੇ **ਡਬਲ-ਕਲਿੱਕ** ਕਰਨ ਨਾਲ ਟੈਕਸਟ ਐਡੀਟਰ **ਸਿੰਪਲ** ਮੋਡ ਵਿੱਚ ਖੁੱਲ੍ਹਦਾ ਹੈ। ਹਰੇਕ ਸੈਗਮੈਂਟ ਸੁਤੰਤਰ ਹੈ ਅਤੇ ਵੱਖਰੇ ਤੌਰ 'ਤੇ ਸੰਪਾਦਿਤ ਕੀਤਾ ਜਾ ਸਕਦਾ ਹੈ।

| ਵਿਸ਼ੇਸ਼ਤਾ | ਵਿਵਹਾਰ |
|---------|-----------|
| Bold / Italic / Font / Height | **ਪੂਰੇ** ਲੇਬਲ 'ਤੇ ਇੱਕਸਾਰ ਲਾਗੂ ਹੁੰਦੇ ਹਨ |
| ਪ੍ਰਤੀ-ਅੱਖਰ ਫਾਰਮੈਟਿੰਗ | ਸਮਰਥਿਤ ਨਹੀਂ |
| `Enter` | ਮੁੱਲ ਕਮਿੱਟ ਕਰਦਾ ਹੈ ਅਤੇ ਐਡੀਟਰ ਬੰਦ ਕਰਦਾ ਹੈ |
| ਮਲਟੀ-ਲਾਈਨ | ਸਮਰਥਿਤ ਨਹੀਂ |

## DXF — DIMENSION ਐਂਟਿਟੀਆਂ

ਚੇਨ ਵਿੱਚ ਹਰੇਕ ਸੈਗਮੈਂਟ DXF ਫਾਈਲ ਵਿੱਚ ਇੱਕ ਸੁਤੰਤਰ `DIMENSION` ਐਂਟਿਟੀ ਵਜੋਂ ਸੰਭਾਲਿਆ ਜਾਂਦਾ ਹੈ। ਇਹ ਫਾਈਲ ਵਿੱਚ ਆਪਸ ਵਿੱਚ ਜੁੜੇ ਨਹੀਂ ਹੁੰਦੇ — ਇਹ ਵਿਸ਼ੇਸ਼ਤਾਵਾਂ ਸਾਂਝੀਆਂ ਕਰਦੇ ਹਨ ਕਿਉਂਕਿ ਇਹ ਇੱਕੋ ਬੇਸ ਤੋਂ ਬਣਾਏ ਗਏ ਸਨ, ਪਰ ਰੱਖਣ ਤੋਂ ਬਾਅਦ ਹਰੇਕ ਨੂੰ ਵੱਖਰੇ ਤੌਰ 'ਤੇ ਸੰਪਾਦਿਤ ਕੀਤਾ ਜਾ ਸਕਦਾ ਹੈ।
