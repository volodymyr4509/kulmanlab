# Delete

`delete` ਕਮਾਂਡ ਚੁਣੀਆਂ ਐਂਟਿਟੀਆਂ ਨੂੰ ਡਰਾਇੰਗ ਵਿੱਚੋਂ ਹਟਾਉਂਦੀ ਹੈ। ਮਿਟਾਉਣ ਦੀਆਂ ਕਾਰਵਾਈਆਂ [Undo](./undo) ਇਤਿਹਾਸ ਵਿੱਚ ਦਰਜ ਹੁੰਦੀਆਂ ਹਨ ਅਤੇ 20 ਕਦਮਾਂ ਤੱਕ ਵਾਪਸ ਕੀਤੀਆਂ ਜਾ ਸਕਦੀਆਂ ਹਨ। ਕੋਈ ਵੱਖਰਾ "ਪੁਸ਼ਟੀ ਡਿਲੀਟ" ਡਾਇਲਾਗ ਨਹੀਂ ਹੈ — ਪੁਸ਼ਟੀ ਸਿਰਫ਼ ਇੱਕ ਕੀ ਪ੍ਰੈਸ ਹੈ।

## ਮਿਟਾਉਣ ਦੇ ਦੋ ਤਰੀਕੇ

**ਪਹਿਲਾਂ ਚੁਣੋ, ਫਿਰ ਮਿਟਾਓ** — ਸਭ ਤੋਂ ਤੇਜ਼ ਰਾਹ:

1. ਕੈਨਵਸ 'ਤੇ ਇੱਕ ਜਾਂ ਵੱਧ ਐਂਟਿਟੀਆਂ ਚੁਣੋ।
2. ਟਰਮੀਨਲ ਵਿੱਚ `delete` ਟਾਈਪ ਕਰੋ, **Delete** ਟੂਲਬਾਰ ਬਟਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ, **ਜਾਂ ਸਿੱਧਾ `Delete` ਕੀ ਦਬਾਓ**।

ਐਂਟਿਟੀਆਂ ਤੁਰੰਤ ਹਟਾਈਆਂ ਜਾਂਦੀਆਂ ਹਨ — ਕੋਈ ਵਾਧੂ ਪੁਸ਼ਟੀ ਕਦਮ ਨਹੀਂ।

**ਸਰਗਰਮ ਕਰੋ, ਫਿਰ ਚੁਣੋ**:

1. `delete` ਟਾਈਪ ਕਰੋ ਜਾਂ ਟੂਲਬਾਰ ਬਟਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ (ਕੁਝ ਵੀ ਚੁਣੇ ਬਿਨਾਂ)।
2. **ਆਬਜੈਕਟ ਚੁਣੋ** — ਟੌਗਲ ਕਰਨ ਲਈ ਕਲਿੱਕ ਕਰੋ, ਜਾਂ ਖੇਤਰ ਦੁਆਰਾ ਚੁਣਨ ਲਈ ਖਿੱਚੋ।
3. ਚੁਣੀਆਂ ਐਂਟਿਟੀਆਂ ਦੀ ਪੁਸ਼ਟੀ ਕਰਨ ਅਤੇ ਹਟਾਉਣ ਲਈ **Enter**, **Space**, ਜਾਂ **Delete** ਦਬਾਓ।

## Delete ਕੀ ਸ਼ਾਰਟਕੱਟ

ਕੀਬੋਰਡ 'ਤੇ `Delete` ਕੀ ਇੱਕ **ਗਲੋਬਲ ਸ਼ਾਰਟਕੱਟ** ਵਜੋਂ ਕੰਮ ਕਰਦੀ ਹੈ — ਜੇ ਵਰਤਮਾਨ ਵਿੱਚ ਕੋਈ ਐਂਟਿਟੀਆਂ ਚੁਣੀਆਂ ਹੋਈਆਂ ਹਨ, ਇਸਨੂੰ ਦਬਾਉਣ ਨਾਲ ਉਹ ਤੁਰੰਤ ਮਿਟਾ ਦਿੱਤੀਆਂ ਜਾਂਦੀਆਂ ਹਨ, ਟਰਮੀਨਲ ਵਿੱਚ Delete ਕਮਾਂਡ ਖੋਲ੍ਹੇ ਬਿਨਾਂ ਵੀ। ਇਹ ਸਭ ਤੋਂ ਤੇਜ਼ ਇੱਕ-ਕਦਮ ਮਿਟਾਉਣ ਦਾ ਵਰਕਫ਼ਲੋ ਹੈ:

```
Click entity → press Delete key → done
```

## ਕਮਾਂਡ ਦੌਰਾਨ ਚੋਣ

| ਢੰਗ | ਵਿਵਹਾਰ |
|--------|-----------|
| **ਕਲਿੱਕ** | ਕਰਸਰ ਹੇਠਾਂ ਐਂਟਿਟੀ ਨੂੰ ਚੋਣ ਵਿੱਚ/ਬਾਹਰ ਟੌਗਲ ਕਰਦਾ ਹੈ |
| **ਸੱਜੇ ਖਿੱਚੋ** (ਸਖ਼ਤ) | ਸਿਰਫ਼ ਬਾਕਸ ਦੇ ਪੂਰੀ ਤਰ੍ਹਾਂ ਅੰਦਰ ਵਾਲੀਆਂ ਐਂਟਿਟੀਆਂ ਚੁਣਦਾ ਹੈ |
| **ਖੱਬੇ ਖਿੱਚੋ** (ਕ੍ਰਾਸਿੰਗ) | ਬਾਕਸ ਦੀ ਸੀਮਾ ਨੂੰ ਕੱਟਣ ਵਾਲੀਆਂ ਐਂਟਿਟੀਆਂ ਚੁਣਦਾ ਹੈ |
| **Enter** / **Space** / **Delete** | ਚੁਣੀਆਂ ਐਂਟਿਟੀਆਂ ਦੀ ਪੁਸ਼ਟੀ ਅਤੇ ਮਿਟਾਉਂਦਾ ਹੈ |

## ਮਿਟਾਈਆਂ ਐਂਟਿਟੀਆਂ ਮੁੜ-ਪ੍ਰਾਪਤ ਕਰਨਾ

ਮਿਟਾਉਣ ਦੀਆਂ ਕਾਰਵਾਈਆਂ [Undo](./undo) ਕਮਾਂਡ (`undo` ਟਾਈਪ ਕਰੋ ਜਾਂ ਟੂਲਬਾਰ ਬਟਨ ਵਰਤੋ) ਨਾਲ ਵਾਪਸ ਕੀਤੀਆਂ ਜਾ ਸਕਦੀਆਂ ਹਨ। ਪ੍ਰਤੀ ਫਾਈਲ **20 ਕਦਮਾਂ** ਤੱਕ ਵਾਪਸ ਕੀਤਾ ਜਾ ਸਕਦਾ ਹੈ, ਅਤੇ ਇਤਿਹਾਸ ਪੇਜ ਰੀਲੋਡ ਦੇ ਪਾਰ ਬਰਕਰਾਰ ਰਹਿੰਦਾ ਹੈ। ਜੇ ਤੁਸੀਂ ਸੇਵ ਕੀਤੇ ਬਿਨਾਂ 20 ਮਿਟਾਉਣ ਦੀਆਂ ਕਾਰਵਾਈਆਂ ਤੋਂ ਵੱਧ ਗਏ ਹੋ, ਪਹਿਲਾਂ ਦੀਆਂ ਮਿਟਾਉਣ ਦੀਆਂ ਕਾਰਵਾਈਆਂ ਮੁੜ-ਪ੍ਰਾਪਤ ਨਹੀਂ ਹੋ ਸਕਦੀਆਂ।

## ਸਮਰਥਿਤ ਐਂਟਿਟੀਆਂ

Delete ਹਰੇਕ ਐਂਟਿਟੀ ਕਿਸਮ 'ਤੇ ਕੰਮ ਕਰਦੀ ਹੈ — Line, Polyline, Rectangle, Circle, Arc, Ellipse, Text, Spline, Dimension, Leader, ਅਤੇ ਹੋਰ ਸਾਰੀਆਂ।
