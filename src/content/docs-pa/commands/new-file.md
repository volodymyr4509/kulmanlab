# New File

**New File** ਕਮਾਂਡ ਕੈਨਵਸ ਸਾਫ਼ ਕਰਦੀ ਹੈ ਅਤੇ ਨਵੀਂ ਖਾਲੀ ਡਰਾਇੰਗ ਸ਼ੁਰੂ ਕਰਦੀ ਹੈ। ਟਾਈਮਸਟੈਂਪ ਵਾਲਾ ਇੱਕ ਵਿਲੱਖਣ ਫਾਈਲ ਨਾਮ ਆਪਣੇ ਆਪ ਬਣਾਇਆ ਜਾਂਦਾ ਹੈ।

## ਨਵੀਂ ਫਾਈਲ ਕਿਵੇਂ ਬਣਾਈਏ

File ਪੈਨਲ ਵਿੱਚ **New File** ਟੂਲਬਾਰ ਬਟਨ (ਨਵਾਂ-ਪੇਜ ਆਈਕਨ) 'ਤੇ ਕਲਿੱਕ ਕਰੋ। ਕੈਨਵਸ ਤੁਰੰਤ ਸਾਫ਼ ਹੋ ਜਾਂਦਾ ਹੈ — ਕੋਈ ਪ੍ਰੋਂਪਟ ਜਾਂ ਪੁਸ਼ਟੀ ਡਾਇਲਾਗ ਨਹੀਂ।

## ਨਵੀਂ ਫਾਈਲ ਵਿੱਚ ਕੀ ਹੁੰਦਾ ਹੈ

ਇੱਕ ਨਵੀਂ ਬਣਾਈ ਫਾਈਲ ਇਹਨਾਂ ਨਾਲ ਸ਼ੁਰੂ ਹੁੰਦੀ ਹੈ:

- ਕੈਨਵਸ 'ਤੇ **ਕੋਈ ਐਂਟਿਟੀ ਨਹੀਂ**।
- `0` ਨਾਮ ਵਾਲੀ **ਇੱਕ ਡਿਫਾਲਟ ਲੇਅਰ**, ਸਫ਼ੈਦ ਰੰਗ ਅਤੇ `Continuous` ਲਾਈਨਟਾਈਪ ਨਾਲ।
- `kulman-Mon01_HH:MM:SS.dxf` ਫਾਰਮੈਟ ਵਿੱਚ ਇੱਕ **ਜਨਰੇਟ ਕੀਤਾ ਫਾਈਲ ਨਾਮ** (ਉਦਾਹਰਨ ਲਈ `kulman-May22_14:30:00.dxf`)।

ਫਾਈਲ ਆਪਣੇ ਆਪ ਬ੍ਰਾਊਜ਼ਰ ਸਟੋਰੇਜ ਵਿੱਚ ਸੇਵ ਹੁੰਦੀ ਹੈ ਅਤੇ [Recent Files](./files) ਵਿੱਚ ਦਿਖਾਈ ਦਿੰਦੀ ਹੈ।

## ਚੇਤਾਵਨੀ — ਅਣਸੇਵ ਕੀਤਾ ਕੰਮ ਮਿਟਾ ਦਿੱਤਾ ਜਾਂਦਾ ਹੈ

**New File** 'ਤੇ ਕਲਿੱਕ ਕਰਨ ਨਾਲ ਮੌਜੂਦਾ ਕੈਨਵਸ ਦੀਆਂ ਸਾਰੀਆਂ ਐਂਟਿਟੀਆਂ ਬਿਨਾਂ ਕਿਸੇ ਚੇਤਾਵਨੀ ਦੇ ਮਿਟਾ ਦਿੱਤੀਆਂ ਜਾਂਦੀਆਂ ਹਨ। ਜੇ ਤੁਸੀਂ ਮੌਜੂਦਾ ਡਰਾਇੰਗ ਰੱਖਣਾ ਚਾਹੁੰਦੇ ਹੋ, ਤਾਂ ਪਹਿਲਾਂ ਇਸਨੂੰ [export](./export) ਕਰੋ।
