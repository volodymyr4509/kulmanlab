# Print

`print` ਕਮਾਂਡ **Print Manager** ਖੋਲ੍ਹਦੀ ਹੈ — ਇੱਕ ਸਮਰਪਿਤ ਐਕਸਪੋਰਟ ਵਿੰਡੋ ਜਿਸ ਵਿੱਚ ਲਾਈਵ ਪ੍ਰੀਵਿਊ ਕੈਨਵਸ, ਫਾਰਮੈਟ ਸਿਲੈਕਟਰ (PNG / JPEG / WebP / PDF), ਮੋਨੋਕ੍ਰੋਮ ਟੌਗਲ, ਅਤੇ ਵਿਕਲਪਿਕ ਖੇਤਰ ਕ੍ਰੌਪ ਹੁੰਦਾ ਹੈ। ਕੁਝ ਵੀ ਕਿਸੇ ਭੌਤਿਕ ਪ੍ਰਿੰਟਰ ਨੂੰ ਨਹੀਂ ਭੇਜਿਆ ਜਾਂਦਾ; ਆਉਟਪੁੱਟ ਇੱਕ ਫਾਈਲ ਵਜੋਂ ਡਾਊਨਲੋਡ ਹੁੰਦਾ ਹੈ।

## Print Manager ਖੋਲ੍ਹਣਾ

**Print** ਟੂਲਬਾਰ ਬਟਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ ਜਾਂ ਟਰਮੀਨਲ ਵਿੱਚ `print` ਟਾਈਪ ਕਰੋ। Print Manager ਤੁਰੰਤ ਮੌਜੂਦਾ ਵਿਊਪੋਰਟ ਦਾ ਪ੍ਰੀਵਿਊ ਦਿਖਾਉਂਦੇ ਹੋਏ ਖੁੱਲ੍ਹਦਾ ਹੈ।

## Print Manager ਲੇਆਉਟ

ਵਿੰਡੋ ਵਿੱਚ ਦੋ ਪੈਨਲ ਹਨ:
- **ਖੱਬੀ ਸਾਈਡਬਾਰ** — ਸਾਰੇ ਐਕਸਪੋਰਟ ਨਿਯੰਤਰਣ।
- **ਸੱਜਾ ਪੈਨਲ** — ਲਾਈਵ ਪ੍ਰੀਵਿਊ ਕੈਨਵਸ ਜੋ ਸੈਟਿੰਗਾਂ ਬਦਲਣ 'ਤੇ ਅੱਪਡੇਟ ਹੁੰਦਾ ਹੈ।

### ਸਾਈਡਬਾਰ ਨਿਯੰਤਰਣ

| ਨਿਯੰਤਰਣ | ਵੇਰਵਾ |
|---------|-------|
| **Change Area** | ਕੈਨਵਸ 'ਤੇ ਕਸਟਮ ਆਇਤ 'ਤੇ ਕ੍ਰੌਪ ਕਰੋ (ਹੇਠਾਂ ਵੇਖੋ) |
| **Monochrome** ਟੌਗਲ | ਸਾਰੀਆਂ ਰੰਗਦਾਰ ਲਾਈਨਾਂ ਨੂੰ ਕਾਲੇ ਵਿੱਚ ਬਦਲੋ — ਸਾਫ਼ ਪ੍ਰਿੰਟ ਆਉਟਪੁੱਟ ਲਈ ਡਿਫਾਲਟ ਰੂਪ ਵਿੱਚ ਚਾਲੂ |
| **Format** ਡ੍ਰੌਪਡਾਊਨ | PNG, JPEG, WebP, ਜਾਂ PDF |
| **Export** ਬਟਨ | ਫਾਈਲ ਜਨਰੇਟ ਕਰੋ ਅਤੇ ਡਾਊਨਲੋਡ ਕਰੋ |

## ਕਸਟਮ ਐਕਸਪੋਰਟ ਖੇਤਰ ਚੁਣਨਾ

ਡਿਫਾਲਟ ਰੂਪ ਵਿੱਚ ਪ੍ਰੀਵਿਊ ਬਿਲਕੁਲ ਉਹੀ ਦਿਖਾਉਂਦਾ ਹੈ ਜੋ Print Manager ਖੋਲ੍ਹਣ ਵੇਲੇ ਕੈਨਵਸ 'ਤੇ ਦਿਖਾਈ ਦੇ ਰਿਹਾ ਸੀ। ਕਿਸੇ ਖਾਸ ਖੇਤਰ ਨੂੰ ਐਕਸਪੋਰਟ ਕਰਨ ਲਈ:

1. **Change Area** 'ਤੇ ਕਲਿੱਕ ਕਰੋ — Print Manager ਲੁਕ ਜਾਂਦਾ ਹੈ ਅਤੇ ਕੈਨਵਸ ਇੰਟਰਐਕਟਿਵ ਹੋ ਜਾਂਦਾ ਹੈ।
2. ਐਕਸਪੋਰਟ ਆਇਤ ਦੇ **ਪਹਿਲੇ ਕੋਨੇ 'ਤੇ ਕਲਿੱਕ ਕਰੋ**।
3. **ਉਲਟੇ ਕੋਨੇ 'ਤੇ ਕਲਿੱਕ ਕਰੋ** — Print Manager ਚੁਣੇ ਹੋਏ ਖੇਤਰ ਦੇ ਪ੍ਰੀਵਿਊ ਨਾਲ ਦੁਬਾਰਾ ਖੁੱਲ੍ਹਦਾ ਹੈ।

ਖੇਤਰ ਚੋਣ ਦੌਰਾਨ ਰੱਦ ਕਰਨ ਅਤੇ ਪਿਛਲਾ ਖੇਤਰ ਬਹਾਲ ਕਰਨ ਲਈ `Escape` ਦਬਾਓ।

ਪ੍ਰੀਵਿਊ ਕੈਨਵਸ ਚੁਣੇ ਹੋਏ ਖੇਤਰ ਦੇ **ਸਟੀਕ ਆਸਪੈਕਟ ਰੇਸ਼ੋ** ਨਾਲ ਮੇਲ ਖਾਣ ਲਈ ਗਤੀਸ਼ੀਲ ਰੂਪ ਵਿੱਚ ਆਕਾਰ ਬਦਲਦਾ ਹੈ, ਇਸ ਲਈ ਪ੍ਰੀਵਿਊ ਪਿਕਸਲ-ਸਟੀਕ ਹੁੰਦਾ ਹੈ।

## ਐਕਸਪੋਰਟ ਫਾਰਮੈਟ

| ਫਾਰਮੈਟ | ਸਭ ਤੋਂ ਵਧੀਆ | ਨੋਟਸ |
|--------|----------|-------|
| **PNG** | ਲੌਸਲੈੱਸ, ਤਿੱਖੀਆਂ ਲਾਈਨਾਂ | ਸਫ਼ੈਦ ਬੈਕਗ੍ਰਾਊਂਡ, ਕੋਈ ਪਾਰਦਰਸ਼ਤਾ ਨਹੀਂ |
| **JPEG** | ਸ਼ੇਅਰ ਕਰਨ ਲਈ ਛੋਟੀ ਫਾਈਲ | 95% ਕੁਆਲਿਟੀ, ਥੋੜ੍ਹਾ ਕੰਪ੍ਰੈਸ਼ਨ |
| **WebP** | ਵੈੱਬ ਲਈ ਸਭ ਤੋਂ ਛੋਟੀ ਫਾਈਲ | ਉਹੀ 95% ਕੁਆਲਿਟੀ, JPEG ਨਾਲੋਂ ਬਿਹਤਰ ਕੰਪ੍ਰੈਸ਼ਨ |
| **PDF** | ਪ੍ਰਿੰਟ-ਤਿਆਰ ਦਸਤਾਵੇਜ਼ | PDF ਕੰਟੇਨਰ ਦੇ ਅੰਦਰ 150 DPI 'ਤੇ ਏਮਬੈੱਡ ਕੀਤੀ ਤਸਵੀਰ |

ਐਕਸਪੋਰਟ ਕੀਤੀ ਫਾਈਲ ਦਾ ਨਾਮ `kulman-<timestamp>.<ext>` ਹੁੰਦਾ ਹੈ ਅਤੇ ਇਹ ਆਪਣੇ ਆਪ ਡਾਊਨਲੋਡ ਹੁੰਦੀ ਹੈ।

## ਐਕਸਪੋਰਟ ਰਿਜ਼ੋਲਿਊਸ਼ਨ ਅਤੇ ਬੈਕਗ੍ਰਾਊਂਡ

- ਵੱਧ ਤੋਂ ਵੱਧ ਰਿਜ਼ੋਲਿਊਸ਼ਨ: **2000 × 2000 ਪਿਕਸਲ**, ਚੁਣੇ ਹੋਏ ਖੇਤਰ ਦੇ ਅਨੁਪਾਤ ਵਿੱਚ ਸਕੇਲ ਕੀਤਾ ਗਿਆ।
- ਬੈਕਗ੍ਰਾਊਂਡ ਹਮੇਸ਼ਾ **ਸਫ਼ੈਦ** ਹੁੰਦਾ ਹੈ।
- **ਨਾਨ-ਪਲਾਟਿੰਗ** ਵਜੋਂ ਚਿੰਨ੍ਹਿਤ ਲੇਅਰਾਂ ਐਕਸਪੋਰਟ ਵਿੱਚੋਂ ਬਾਹਰ ਰਹਿ ਜਾਂਦੀਆਂ ਹਨ।
