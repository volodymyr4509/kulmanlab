# Dimension Linear

`dimlinear` ਕਮਾਂਡ ਦੋ ਐਕਸਟੈਂਸ਼ਨ-ਲਾਈਨ ਮੂਲਾਂ ਵਿਚਕਾਰ ਇੱਕ ਹਰੀਜ਼ੌਂਟਲ ਜਾਂ ਵਰਟੀਕਲ ਡਾਇਮੈਂਸ਼ਨ ਰੱਖਦੀ ਹੈ। ਡਾਇਮੈਂਸ਼ਨ ਲਾਈਨ ਹਮੇਸ਼ਾ ਪੂਰੀ ਤਰ੍ਹਾਂ ਹਰੀਜ਼ੌਂਟਲ ਜਾਂ ਪੂਰੀ ਤਰ੍ਹਾਂ ਵਰਟੀਕਲ ਚਲਦੀ ਹੈ — ਇਸਨੂੰ ਮਨਮਾਨੇ ਕੋਣ 'ਤੇ ਨਹੀਂ ਰੱਖਿਆ ਜਾ ਸਕਦਾ। ਜਦੋਂ ਤੁਹਾਨੂੰ ਇੱਕ ਡਾਇਗਨਲ ਲਾਈਨ ਦੇ ਸਮਾਨਾਂਤਰ ਡਾਇਮੈਂਸ਼ਨ ਚਾਹੀਦਾ ਹੋਵੇ ਤਾਂ [Dimension Aligned](../dim-aligned/) ਵਰਤੋ।

## ਇੱਕ ਲੀਨੀਅਰ ਡਾਇਮੈਂਸ਼ਨ ਦੀ ਸੰਰਚਨਾ

```
  |←————— 5.00 —————→|
  |                   |
  ●  (ext line 1)     ●  (ext line 2)
  p1                  p2
```

- **ਐਕਸਟੈਂਸ਼ਨ ਲਾਈਨਾਂ** — ਹਰੇਕ ਮਾਪੇ ਗਏ ਬਿੰਦੂ ਤੋਂ ਡਾਇਮੈਂਸ਼ਨ ਲਾਈਨ ਦੇ ਲੰਬਵਤ ਹੇਠਾਂ ਜਾਂਦੀਆਂ ਹਨ।
- **ਡਾਇਮੈਂਸ਼ਨ ਲਾਈਨ** — ਹਰੀਜ਼ੌਂਟਲ (X ਦੂਰੀ ਮਾਪਦੀ ਹੈ) ਜਾਂ ਵਰਟੀਕਲ (Y ਦੂਰੀ ਮਾਪਦੀ ਹੈ)।
- **ਮੁੱਲ** — ਚੁਣੇ ਗਏ ਧੁਰੇ ਦੇ ਨਾਲ ਪ੍ਰੋਜੈਕਟਿਡ ਦੂਰੀ, ਨਾ ਕਿ ਅਸਲ ਬਿੰਦੂ-ਤੋਂ-ਬਿੰਦੂ ਦੂਰੀ।

## ਇੱਕ ਲੀਨੀਅਰ ਡਾਇਮੈਂਸ਼ਨ ਰੱਖਣਾ

1. ਟਰਮੀਨਲ ਵਿੱਚ `dimlinear` ਟਾਈਪ ਕਰੋ ਜਾਂ **Dimension Linear** ਟੂਲਬਾਰ ਬਟਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ।
2. **ਪਹਿਲੇ ਐਕਸਟੈਂਸ਼ਨ ਲਾਈਨ ਮੂਲ 'ਤੇ ਕਲਿੱਕ ਕਰੋ** (p1), ਜਾਂ ਸਟੀਕ ਨਿਰਦੇਸ਼ਾਂਕ ਲਈ `X,Y` ਟਾਈਪ ਕਰੋ ਅਤੇ **Enter** ਦਬਾਓ।
3. **ਦੂਜੇ ਐਕਸਟੈਂਸ਼ਨ ਲਾਈਨ ਮੂਲ 'ਤੇ ਕਲਿੱਕ ਕਰੋ** (p2)। ਇੱਥੇ ਵੀ ਨਿਰਦੇਸ਼ਾਂਕ ਐਂਟਰੀ ਕੰਮ ਕਰਦੀ ਹੈ।
4. **ਕਰਸਰ ਹਿਲਾਓ** ਡਾਇਮੈਂਸ਼ਨ ਲਾਈਨ ਦੀ ਸਥਿਤੀ ਤੈਅ ਕਰਨ ਲਈ। ਓਰੀਐਂਟੇਸ਼ਨ ਕਰਸਰ ਸਥਿਤੀ ਤੋਂ ਆਪਣੇ-ਆਪ ਪਛਾਣੀ ਜਾਂਦੀ ਹੈ।
5. ਰੱਖਣ ਲਈ **ਕਲਿੱਕ ਕਰੋ**, ਜਾਂ ਸਟੀਕ ਪਲੇਸਮੈਂਟ ਲਈ ਇੱਕ ਆਫਸੈੱਟ ਦੂਰੀ ਟਾਈਪ ਕਰੋ ਅਤੇ **Enter** ਦਬਾਓ।

## ਆਪਣੇ-ਆਪ ਓਰੀਐਂਟੇਸ਼ਨ ਪਛਾਣ

ਜਦੋਂ ਕੋਈ ਓਰੀਐਂਟੇਸ਼ਨ ਜ਼ਬਰਦਸਤੀ ਲਾਗੂ ਨਹੀਂ ਕੀਤੀ ਜਾਂਦੀ, ਕਮਾਂਡ ਦੋ ਮਾਪੇ ਗਏ ਬਿੰਦੂਆਂ ਦੇ ਸੰਬੰਧ ਵਿੱਚ ਕਰਸਰ ਸਥਿਤੀ ਪੜ੍ਹਦੀ ਹੈ:

| ਕਰਸਰ ਸਥਿਤੀ | ਪਛਾਣੀ ਗਈ ਓਰੀਐਂਟੇਸ਼ਨ | ਕੀ ਮਾਪਿਆ ਜਾਂਦਾ ਹੈ |
|-----------------|---------------------|-----------------|
| ਬਿੰਦੂਆਂ ਦੇ ਉੱਪਰ ਜਾਂ ਹੇਠਾਂ | ਹਰੀਜ਼ੌਂਟਲ | p1 ਅਤੇ p2 ਵਿਚਕਾਰ Δ X |
| ਬਿੰਦੂਆਂ ਦੇ ਖੱਬੇ ਜਾਂ ਸੱਜੇ | ਵਰਟੀਕਲ | p1 ਅਤੇ p2 ਵਿਚਕਾਰ Δ Y |

ਪਲੇਸਮੈਂਟ ਪੜਾਅ ਦੌਰਾਨ ਕਿਸੇ ਵੀ ਸਮੇਂ ਹਰੀਜ਼ੌਂਟਲ ਲਾਕ ਕਰਨ ਲਈ **H** ਦਬਾਓ ਜਾਂ ਵਰਟੀਕਲ ਲਾਕ ਕਰਨ ਲਈ **V** ਦਬਾਓ। ਇੱਕ ਵਾਰ ਲਾਕ ਹੋਣ 'ਤੇ, ਕਰਸਰ ਹਿਲਾਉਣ ਨਾਲ ਓਰੀਐਂਟੇਸ਼ਨ ਬਦਲਦੀ ਨਹੀਂ।

## ਟਾਈਪ ਕੀਤੀ ਆਫਸੈੱਟ ਦੂਰੀ

ਪਲੇਸਮੈਂਟ ਦੌਰਾਨ ਇੱਕ ਨੰਬਰ ਟਾਈਪ ਕਰਕੇ ਡਾਇਮੈਂਸ਼ਨ ਲਾਈਨ ਨੂੰ ਮਾਪੇ ਗਏ ਬਿੰਦੂਆਂ ਤੋਂ ਸਟੀਕ ਦੂਰੀ 'ਤੇ ਸਥਿਰ ਕਰੋ:

| ਕੁੰਜੀ | ਕਾਰਵਾਈ |
|-----|--------|
| `0`–`9`, `.` | ਆਫਸੈੱਟ ਦੂਰੀ ਵਿੱਚ ਅੰਕ ਜੋੜੋ |
| `Backspace` | ਆਖਰੀ ਟਾਈਪ ਕੀਤਾ ਅੱਖਰ ਮਿਟਾਓ |
| `Enter` / `Space` | ਟਾਈਪ ਕੀਤੀ ਦੂਰੀ 'ਤੇ ਰੱਖੋ |

ਕਰਸਰ ਪਾਸਾ (ਹਰੀਜ਼ੌਂਟਲ ਲਈ ਉੱਪਰ/ਹੇਠਾਂ, ਵਰਟੀਕਲ ਲਈ ਖੱਬੇ/ਸੱਜੇ) ਚਿੰਨ੍ਹ ਤੈਅ ਕਰਦਾ ਹੈ — ਡਾਇਮੈਂਸ਼ਨ ਲਾਈਨ ਉਸ ਪਾਸੇ ਦਿਖਾਈ ਦਿੰਦੀ ਹੈ ਜਿਸ ਪਾਸੇ ਕਰਸਰ ਇਸ ਸਮੇਂ ਹੈ।

## ਕੀਬੋਰਡ ਸੰਦਰਭ

| ਕੁੰਜੀ | ਕਾਰਵਾਈ |
|-----|--------|
| `0`–`9`, `.`, `-` | X ਨਿਰਦੇਸ਼ਾਂਕ ਐਂਟਰੀ ਸ਼ੁਰੂ ਕਰੋ (p1/p2 ਪੜਾਅ), ਜਾਂ ਆਫਸੈੱਟ ਦੂਰੀ (ਪਲੇਸਮੈਂਟ ਪੜਾਅ) |
| `,` | X ਲਾਕ ਕਰੋ ਅਤੇ Y ਐਂਟਰੀ ਵੱਲ ਜਾਓ (p1/p2 ਪੜਾਅ) |
| `H` | ਹਰੀਜ਼ੌਂਟਲ ਓਰੀਐਂਟੇਸ਼ਨ ਲਾਕ ਕਰੋ (ਸਿਰਫ਼ ਪਲੇਸਮੈਂਟ ਪੜਾਅ) |
| `V` | ਵਰਟੀਕਲ ਓਰੀਐਂਟੇਸ਼ਨ ਲਾਕ ਕਰੋ (ਸਿਰਫ਼ ਪਲੇਸਮੈਂਟ ਪੜਾਅ) |
| `Backspace` | ਆਖਰੀ ਟਾਈਪ ਕੀਤਾ ਅੱਖਰ ਮਿਟਾਓ |
| `Enter` / `Space` | ਟਾਈਪ ਕੀਤੇ ਨਿਰਦੇਸ਼ਾਂਕ ਜਾਂ ਆਫਸੈੱਟ ਦੀ ਪੁਸ਼ਟੀ ਕਰੋ |
| `Escape` | ਰੱਦ ਕਰੋ |

## Dimension Linear ਬਨਾਮ Dimension Aligned

| | Dimension Linear | Dimension Aligned |
|---|-----------------|------------------|
| ਧੁਰਾ | ਹਮੇਸ਼ਾ H ਜਾਂ V | ਮਾਪੀ ਗਈ ਲਾਈਨ ਦੇ ਸਮਾਨਾਂਤਰ |
| ਮਾਪਦਾ ਹੈ | ਸਿਰਫ਼ X ਜਾਂ Y ਹਿੱਸਾ | ਅਸਲ ਯੂਕਲਿਡੀਅਨ ਦੂਰੀ |
| H/V ਕੁੰਜੀਆਂ | ਹਾਂ — ਓਰੀਐਂਟੇਸ਼ਨ ਲਾਕ ਕਰਦੀਆਂ ਹਨ | ਨਹੀਂ — ਹਮੇਸ਼ਾ p1→p2 ਦੀ ਪਾਲਣਾ ਕਰਦਾ ਹੈ |
| ਇਸ ਲਈ ਵਧੀਆ | ਆਰਥੋਗਨਲ ਲੇਆਉਟ, ਫਲੋਰ ਪਲਾਨ | ਡਾਇਗਨਲ ਵਿਸ਼ੇਸ਼ਤਾਵਾਂ, ਕੋਣੀ ਕੱਟ |

## ਲੇਬਲ ਦਾ ਸੰਪਾਦਨ — ਸਿੰਪਲ ਮੋਡ

ਇੱਕ ਰੱਖੇ ਗਏ ਲੀਨੀਅਰ ਡਾਇਮੈਂਸ਼ਨ 'ਤੇ **ਡਬਲ-ਕਲਿੱਕ** ਕਰਨ ਨਾਲ ਟੈਕਸਟ ਐਡੀਟਰ **ਸਿੰਪਲ** ਮੋਡ ਵਿੱਚ ਖੁੱਲ੍ਹਦਾ ਹੈ। ਐਡੀਟਰ ਮੌਜੂਦਾ ਰੈਂਡਰ ਕੀਤੇ ਮੁੱਲ ਨਾਲ ਪਹਿਲਾਂ ਤੋਂ ਭਰਿਆ ਹੁੰਦਾ ਹੈ ਤਾਂ ਜੋ ਤੁਸੀਂ ਕਰਸਰ ਨੂੰ ਸਥਿਤ ਕਰਕੇ ਸਿੱਧਾ ਸੰਪਾਦਿਤ ਕਰ ਸਕੋ।

| ਵਿਸ਼ੇਸ਼ਤਾ | ਵਿਵਹਾਰ |
|---------|-----------|
| Bold / Italic / Font / Height | **ਪੂਰੇ** ਲੇਬਲ 'ਤੇ ਇੱਕਸਾਰ ਲਾਗੂ ਹੁੰਦੇ ਹਨ |
| ਪ੍ਰਤੀ-ਅੱਖਰ ਫਾਰਮੈਟਿੰਗ | ਸਮਰਥਿਤ ਨਹੀਂ |
| `Enter` | ਮੁੱਲ ਕਮਿੱਟ ਕਰਦਾ ਹੈ ਅਤੇ ਐਡੀਟਰ ਬੰਦ ਕਰਦਾ ਹੈ |
| ਮਲਟੀ-ਲਾਈਨ | ਸਮਰਥਿਤ ਨਹੀਂ |

ਪੂਰੇ ਰੈਫਰੈਂਸ ਲਈ [Text Editor — simple mode](../../interface/text-editor/#simple-mode) ਵੇਖੋ।

## ਡਾਇਮੈਂਸ਼ਨਾਂ ਦੀ ਚੇਨਿੰਗ

ਆਖਰੀ ਐਕਸਟੈਂਸ਼ਨ ਲਾਈਨ ਤੋਂ ਜਾਰੀ ਹੋਰ ਡਾਇਮੈਂਸ਼ਨ ਜੋੜਨ ਲਈ, ਇਸਨੂੰ ਰੱਖਣ ਤੋਂ ਤੁਰੰਤ ਬਾਅਦ [Dimension Continue](../dim-continue/) ਵਰਤੋ।

## DXF — DIMENSION ਐਂਟਿਟੀ

ਲੀਨੀਅਰ ਡਾਇਮੈਂਸ਼ਨ `DIMENSION` ਐਂਟਿਟੀਆਂ ਦੇ ਰੂਪ ਵਿੱਚ ਸੰਭਾਲੇ ਜਾਂਦੇ ਹਨ ਜਿਹਨਾਂ ਦਾ `rotationDeg` `0` (ਹਰੀਜ਼ੌਂਟਲ) ਜਾਂ `90` (ਵਰਟੀਕਲ) ਸੈੱਟ ਹੁੰਦਾ ਹੈ। ਐਕਸਟੈਂਸ਼ਨ ਲਾਈਨ ਮੂਲ, ਡਾਇਮੈਂਸ਼ਨ ਲਾਈਨ ਸਥਿਤੀ, ਟੈਕਸਟ ਸਥਿਤੀ, ਮਾਪਿਆ ਗਿਆ ਮੁੱਲ, ਤੀਰ ਸ਼ੈਲੀ, ਟੈਕਸਟ ਉਚਾਈ, ਅਤੇ ਸਾਰੇ ਡਿਸਪਲੇ ਫਲੈਗ ਬਿਨਾਂ ਕਿਸੇ ਨੁਕਸਾਨ ਦੇ ਰਾਊਂਡ-ਟ੍ਰਿਪ ਕਰਦੇ ਹਨ।
