# Import

**Import** ਕਮਾਂਡ ਤੁਹਾਡੇ ਲੋਕਲ ਫਾਈਲ ਸਿਸਟਮ ਤੋਂ ਮੌਜੂਦਾ ਡਰਾਇੰਗ KulmanLab CAD ਵਿੱਚ ਲੋਡ ਕਰਦੀ ਹੈ। ਮਿਆਰੀ **DXF** ਫਾਰਮੈਟ ਅਤੇ KulmanLab ਦਾ ਆਪਣਾ **JSON** ਫਾਰਮੈਟ ਦੋਵੇਂ ਸਮਰਥਿਤ ਹਨ।

## ਫਾਈਲ ਇੰਪੋਰਟ ਕਿਵੇਂ ਕਰੀਏ

1. ਸਕ੍ਰੀਨ ਦੇ ਸਿਖਰ 'ਤੇ File ਪੈਨਲ ਵਿੱਚ **Import** ਟੂਲਬਾਰ ਬਟਨ (ਫੋਲਡਰ ਆਈਕਨ) 'ਤੇ ਕਲਿੱਕ ਕਰੋ।
2. ਤੁਹਾਡੇ ਬ੍ਰਾਊਜ਼ਰ ਦਾ ਫਾਈਲ ਪਿਕਰ ਖੁੱਲ੍ਹਦਾ ਹੈ। ਆਪਣੀ ਡਰਾਇੰਗ ਫਾਈਲ 'ਤੇ ਨੈਵੀਗੇਟ ਕਰੋ ਅਤੇ ਇਸਨੂੰ ਚੁਣੋ।
3. ਡਰਾਇੰਗ ਤੁਰੰਤ ਕੈਨਵਸ 'ਤੇ ਲੋਡ ਹੁੰਦੀ ਹੈ। ਵਿਊਪੋਰਟ ਆਪਣੇ ਆਪ ਸਾਰੀਆਂ ਐਂਟਿਟੀਆਂ ਫਿੱਟ ਕਰਦਾ ਹੈ।

ਬਦਲਵੇਂ ਰੂਪ ਵਿੱਚ, ਤੁਸੀਂ ਇੱਕ ਫਾਈਲ ਨੂੰ ਸਿੱਧਾ ਕੈਨਵਸ 'ਤੇ ਡਰੈਗ ਅਤੇ ਡ੍ਰੌਪ ਕਰ ਸਕਦੇ ਹੋ।

## ਸਮਰਥਿਤ ਫਾਈਲ ਫਾਰਮੈਟ

| ਫਾਰਮੈਟ | ਐਕਸਟੈਂਸ਼ਨ | ਕਦੋਂ ਵਰਤੀਏ |
|--------|-----------|-------------|
| **DXF** | `.dxf` | AutoCAD, FreeCAD, LibreCAD, ਜਾਂ ਹੋਰ CAD ਟੂਲਾਂ ਤੋਂ ਡਰਾਇੰਗਾਂ |
| **JSON** *(ਨੇਟਿਵ)* | `.json` | ਪਹਿਲਾਂ KulmanLab CAD ਤੋਂ ਸੇਵ ਕੀਤੀਆਂ ਡਰਾਇੰਗਾਂ — ਪੂਰੀ ਫਿਡੈਲਿਟੀ |

## DXF ਤੋਂ ਕੀ ਇੰਪੋਰਟ ਹੁੰਦਾ ਹੈ

KulmanLab ਹੇਠ ਲਿਖੀਆਂ DXF ਐਂਟਿਟੀ ਕਿਸਮਾਂ ਪਾਰਸ ਕਰਦਾ ਹੈ:

| ਐਂਟਿਟੀ ਕਿਸਮ | DXF ਕੋਡ | ਨੋਟਸ |
|-------------|----------|-------|
| ਲਾਈਨ | `LINE` | |
| ਸਰਕਲ | `CIRCLE` | |
| ਆਰਕ | `ARC` | |
| ਐਲਿਪਸ | `ELLIPSE` | |
| ਪੌਲੀਲਾਈਨ | `LWPOLYLINE` | |
| ਸਪਲਾਈਨ | `SPLINE` | |
| ਟੈਕਸਟ | `TEXT`, `MTEXT` | |
| ਡਾਈਮੈਂਸ਼ਨ | `DIMENSION` | |
| ਮਲਟੀਲੀਡਰ | `MULTILEADER` | |

ਲੇਅਰ ਪਰਿਭਾਸ਼ਾਵਾਂ ਅਤੇ ਲਾਈਨਟਾਈਪ ਟੇਬਲ ਵੀ, ਮੌਜੂਦ ਹੋਣ 'ਤੇ, DXF ਫਾਈਲ ਵਿੱਚੋਂ ਇੰਪੋਰਟ ਹੁੰਦੇ ਹਨ।

ਅਸਮਰਥਿਤ DXF ਕਿਸਮਾਂ ਵਰਤਣ ਵਾਲੀਆਂ ਐਂਟਿਟੀਆਂ ਚੁੱਪਚਾਪ ਛੱਡ ਦਿੱਤੀਆਂ ਜਾਂਦੀਆਂ ਹਨ — ਬਾਕੀ ਡਰਾਇੰਗ ਫਿਰ ਵੀ ਲੋਡ ਹੁੰਦੀ ਹੈ।

## ਫਾਈਲ ਨਾਮਕਰਨ ਅਤੇ ਸਟੋਰੇਜ

ਜਦੋਂ ਤੁਸੀਂ ਕੋਈ ਫਾਈਲ ਇੰਪੋਰਟ ਕਰਦੇ ਹੋ, ਫਾਈਲ ਨਾਮ ਵਿੱਚ ਇੱਕ ਟਾਈਮਸਟੈਂਪ ਜੋੜਿਆ ਜਾਂਦਾ ਹੈ (ਉਦਾਹਰਨ ਲਈ `myplan_May22_14:30:00.dxf`)। ਇਸ ਨਾਲ ਤੁਸੀਂ [Recent Files](../files/) ਵਿੱਚ ਇੱਕੋ ਡਰਾਇੰਗ ਦੇ ਕਈ ਵਰਜਨ ਬਿਨਾਂ ਨਾਮ-ਟਕਰਾਅ ਦੇ ਰੱਖ ਸਕਦੇ ਹੋ। ਜੇ ਫਾਈਲ ਨਾਮ ਵਿੱਚ ਪਹਿਲਾਂ ਹੀ ਟਾਈਮਸਟੈਂਪ ਹੈ, ਤਾਂ ਇਹ ਉਵੇਂ ਹੀ ਵਰਤਿਆ ਜਾਂਦਾ ਹੈ।

ਡਰਾਇੰਗ ਇੰਪੋਰਟ ਤੋਂ ਬਾਅਦ ਆਪਣੇ ਆਪ ਬ੍ਰਾਊਜ਼ਰ ਸਟੋਰੇਜ (IndexedDB) ਵਿੱਚ ਸੇਵ ਹੋ ਜਾਂਦੀ ਹੈ, ਇਸ ਲਈ ਇਹ [Files](../files/) ਪੈਨਲ ਵਿੱਚ ਦਿਖਾਈ ਦਿੰਦੀ ਹੈ ਅਤੇ ਪੇਜ ਰੀਲੋਡ ਹੋਣ ਤੋਂ ਬਾਅਦ ਵੀ ਬਣੀ ਰਹਿੰਦੀ ਹੈ।

## ਮੌਜੂਦਾ ਡਰਾਇੰਗ ਦਾ ਕੀ ਹੁੰਦਾ ਹੈ

ਇੰਪੋਰਟ ਕਰਨ ਨਾਲ ਮੌਜੂਦਾ ਕੈਨਵਸ ਬਦਲ ਜਾਂਦਾ ਹੈ। ਕੋਈ ਮਰਜ ਜਾਂ ਜੋੜ ਨਹੀਂ ਹੁੰਦਾ। ਜੇ ਤੁਹਾਡੇ ਕੋਲ ਅਣਸੇਵ ਕੀਤੀਆਂ ਤਬਦੀਲੀਆਂ ਹਨ, ਤਾਂ ਪਹਿਲਾਂ ਮੌਜੂਦਾ ਡਰਾਇੰਗ [export](../export/) ਕਰੋ।

## ਸਟਾਰਟਅੱਪ 'ਤੇ

KulmanLab ਪੇਜ ਲੋਡ ਹੋਣ 'ਤੇ ਆਪਣੇ ਆਪ ਸਭ ਤੋਂ ਹਾਲ ਹੀ ਵਿੱਚ ਸੰਪਾਦਿਤ ਫਾਈਲ ਦੁਬਾਰਾ ਖੋਲ੍ਹਦਾ ਹੈ। ਜੇ ਕੋਈ ਸੇਵ ਕੀਤੀ ਫਾਈਲ ਮੌਜੂਦ ਨਹੀਂ ਹੈ, ਤਾਂ ਇੱਕ ਡਿਫਾਲਟ ਨਮੂਨਾ ਡਰਾਇੰਗ ਲੋਡ ਹੁੰਦੀ ਹੈ।
