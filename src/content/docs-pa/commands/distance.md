# Distance

`distance` ਕਮਾਂਡ ਦੋ ਕਲਿੱਕ ਕੀਤੇ ਪੁਆਇੰਟਾਂ ਵਿਚਕਾਰ ਸਿੱਧੀ-ਲਾਈਨ (ਯੂਕਲੀਡੀਅਨ) ਦੂਰੀ ਮਾਪਦੀ ਹੈ ਅਤੇ ਨਤੀਜਾ ਟਰਮੀਨਲ ਵਿੱਚ 4 ਦਸ਼ਮਲਵ ਸਥਾਨਾਂ ਤੱਕ ਪ੍ਰਿੰਟ ਕਰਦੀ ਹੈ। ਇਹ ਦੋ ਮਾਪ ਕਮਾਂਡਾਂ ਵਿੱਚੋਂ ਇੱਕ ਹੈ — [Angle](./angle) ਇਸਦੀ ਬਜਾਏ ਵਰਟੈਕਸ ਉੱਤੇ ਕੋਣੀ ਖੁੱਲ੍ਹਣ ਮਾਪਦੀ ਹੈ।

## ਦੂਰੀ ਮਾਪ ਦੀ ਬਣਤਰ

```
  ● first point
   \
    \  preview line (live)
     \
      ● second point    →  terminal: "Distance: 12.3456"
```

- **ਪਹਿਲਾ ਪੁਆਇੰਟ** — ਮਾਪ ਦਾ ਮੂਲ।
- **ਦੂਜਾ ਪੁਆਇੰਟ** — ਐਂਡਪੁਆਇੰਟ; ਇਸਨੂੰ ਰੱਖਣ ਨਾਲ ਤੁਰੰਤ ਨਤੀਜਾ ਪ੍ਰਿੰਟ ਹੁੰਦਾ ਹੈ।
- **ਨਤੀਜਾ** — ਟਰਮੀਨਲ ਵਿੱਚ ਦਿਖਾਇਆ ਜਾਂਦਾ ਹੈ, ਕੈਨਵਸ ਉੱਤੇ ਨਹੀਂ ਰੱਖਿਆ ਜਾਂਦਾ।

## ਦੂਰੀ ਮਾਪਣਾ

1. ਟਰਮੀਨਲ ਵਿੱਚ `distance` ਟਾਈਪ ਕਰੋ ਜਾਂ **Distance** ਟੂਲਬਾਰ ਬਟਨ ਉੱਤੇ ਕਲਿੱਕ ਕਰੋ।
2. **ਪਹਿਲੇ ਪੁਆਇੰਟ ਉੱਤੇ ਕਲਿੱਕ ਕਰੋ**, ਜਾਂ ਸਟੀਕ ਕੋਆਰਡੀਨੇਟ ਲਈ `X,Y` ਟਾਈਪ ਕਰੋ ਅਤੇ **Enter** ਦਬਾਓ।
3. **ਦੂਜੇ ਪੁਆਇੰਟ ਉੱਤੇ ਕਲਿੱਕ ਕਰੋ** — ਮਾਪੀ ਗਈ ਦੂਰੀ ਟਰਮੀਨਲ ਵਿੱਚ ਦਿਖਾਈ ਦਿੰਦੀ ਹੈ। ਕੋਆਰਡੀਨੇਟ ਐਂਟਰੀ ਇੱਥੇ ਵੀ ਕੰਮ ਕਰਦੀ ਹੈ।
4. **ਦੁਬਾਰਾ ਕਲਿੱਕ ਕਰੋ** (ਵਿਕਲਪਿਕ) ਨਵਾਂ ਮਾਪ ਸ਼ੁਰੂ ਕਰਨ ਲਈ। ਕਮਾਂਡ ਸਰਗਰਮ ਰਹਿੰਦੀ ਹੈ।

ਕਿਸੇ ਵੀ ਸਮੇਂ ਕਦਮ 2 ਤੇ ਰੀਸੈੱਟ ਕਰਨ ਲਈ `Escape` ਦਬਾਓ।

## ਮਾਪਾਂ ਦੀ ਚੇਨਿੰਗ

ਨਤੀਜਾ ਦਿਖਣ ਤੋਂ ਬਾਅਦ, ਤੁਰੰਤ ਕਲਿੱਕ ਕਰਨ ਨਾਲ ਅਗਲਾ ਮਾਪ ਸ਼ੁਰੂ ਹੁੰਦਾ ਹੈ — ਕਲਿੱਕ ਕੀਤਾ ਪੁਆਇੰਟ ਨਵਾਂ ਪਹਿਲਾ ਪੁਆਇੰਟ ਬਣ ਜਾਂਦਾ ਹੈ। ਇਹ ਤੁਹਾਨੂੰ ਕਮਾਂਡ ਨੂੰ ਦੁਬਾਰਾ ਸਰਗਰਮ ਕੀਤੇ ਬਿਨਾਂ ਦੂਰੀਆਂ ਦਾ ਇੱਕ ਕ੍ਰਮ ਮਾਪਣ ਦਿੰਦਾ ਹੈ।

## ਕੋਆਰਡੀਨੇਟ ਐਂਟਰੀ

ਕਲਿੱਕ ਕਰਨ ਦੀ ਬਜਾਏ, ਦੋਵਾਂ ਵਿੱਚੋਂ ਕਿਸੇ ਵੀ ਪੁਆਇੰਟ ਲਈ ਸਟੀਕ ਸਥਿਤੀ ਟਾਈਪ ਕਰੋ:

1. X ਮੁੱਲ ਟਾਈਪ ਕਰੋ।
2. `,` ਦਬਾਓ — ਟਰਮੀਨਲ `[X], [Y{cursor}]` ਦਿਖਾਉਂਦਾ ਹੈ।
3. Y ਮੁੱਲ ਟਾਈਪ ਕਰੋ।
4. ਪੁਸ਼ਟੀ ਕਰਨ ਲਈ **Enter** ਦਬਾਓ।

## ਕੀਬੋਰਡ ਰੈਫਰੈਂਸ

| ਕੀ | ਐਕਸ਼ਨ |
|-----|--------|
| `0`–`9`, `.`, `-` | X ਕੋਆਰਡੀਨੇਟ ਐਂਟਰੀ ਸ਼ੁਰੂ ਕਰੋ |
| `,` | X ਲੌਕ ਕਰੋ ਅਤੇ Y ਐਂਟਰੀ ਵੱਲ ਜਾਓ |
| `Backspace` | ਆਖਰੀ ਟਾਈਪ ਕੀਤਾ ਅੱਖਰ ਮਿਟਾਓ |
| `Enter` | ਟਾਈਪ ਕੀਤੇ ਕੋਆਰਡੀਨੇਟ ਦੀ ਪੁਸ਼ਟੀ ਕਰੋ |
| `Escape` | ਰੱਦ ਕਰੋ ਅਤੇ ਕਦਮ 2 ਤੇ ਰੀਸੈੱਟ ਕਰੋ |

## ਨੋਟਸ

- ਨਤੀਜੇ ਸਿਰਫ਼ **ਟਰਮੀਨਲ ਵਿੱਚ** ਦਿਖਾਏ ਜਾਂਦੇ ਹਨ — ਡਰਾਇੰਗ ਵਿੱਚ ਕੁਝ ਵੀ ਸ਼ਾਮਲ ਨਹੀਂ ਹੁੰਦਾ।
- ਨਤੀਜਾ ਡਰਾਇੰਗ ਕੋਆਰਡੀਨੇਟਾਂ ਦੇ ਸਮਾਨ ਯੂਨਿਟਾਂ ਵਿੱਚ ਪ੍ਰਗਟ ਕੀਤਾ ਜਾਂਦਾ ਹੈ (ਕੋਈ ਯੂਨਿਟ ਪਰਿਵਰਤਨ ਨਹੀਂ)।
- ਸ਼ੁੱਧਤਾ ਹਮੇਸ਼ਾ 4 ਦਸ਼ਮਲਵ ਸਥਾਨ ਹੁੰਦੀ ਹੈ।
