# Offset

`offset` ਕਮਾਂਡ ਇੱਕ ਸਥਿਰ ਲੰਬਵਤ ਦੂਰੀ 'ਤੇ ਇੱਕ ਐਂਟਿਟੀ ਦੀ ਸਮਾਂਤਰ ਕਾਪੀ ਬਣਾਉਂਦੀ ਹੈ। ਤੁਸੀਂ ਦੂਰੀ ਇੱਕ ਵਾਰ ਟਾਈਪ ਕਰਦੇ ਹੋ, ਫਿਰ ਐਂਟਿਟੀਆਂ 'ਤੇ ਕਲਿੱਕ ਕਰਦੇ ਹੋ ਅਤੇ ਇੱਕ ਸਾਈਡ ਚੁਣਦੇ ਹੋ — ਕਮਾਂਡ ਉਸੇ ਦੂਰੀ 'ਤੇ ਤਿਆਰ ਰਹਿੰਦੀ ਹੈ ਤਾਂ ਜੋ ਤੁਸੀਂ ਇੱਕੋ ਸੈਸ਼ਨ ਵਿੱਚ ਕਈ ਐਂਟਿਟੀਆਂ ਨੂੰ ਆਫਸੈੱਟ ਕਰ ਸਕੋ।

ਸਮਰਥਿਤ ਐਂਟਿਟੀ ਕਿਸਮਾਂ: **Line, Circle, Arc, Ellipse, Polyline** (Rectangles ਸਮੇਤ)।

## Offset ਦੀ ਵਰਤੋਂ

1. ਟਰਮੀਨਲ ਵਿੱਚ `offset` ਟਾਈਪ ਕਰੋ ਜਾਂ **Offset** ਟੂਲਬਾਰ ਬਟਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ।
2. **ਆਫਸੈੱਟ ਦੂਰੀ ਟਾਈਪ ਕਰੋ** ਅਤੇ **Enter** ਜਾਂ **Space** ਦਬਾਓ।
3. ਆਫਸੈੱਟ ਕਰਨ ਲਈ **ਇੱਕ ਐਂਟਿਟੀ 'ਤੇ ਕਲਿੱਕ ਕਰੋ** — ਜੇ ਐਂਟਿਟੀ ਸਮਰਥਿਤ ਕਿਸਮ ਦੀ ਨਹੀਂ ਹੈ, ਤਾਂ ਇੱਕ ਗਲਤੀ ਸੁਨੇਹਾ ਦਿਖਾਈ ਦਿੰਦਾ ਹੈ ਅਤੇ ਤੁਸੀਂ ਕਿਸੇ ਹੋਰ ਐਂਟਿਟੀ 'ਤੇ ਕਲਿੱਕ ਕਰ ਸਕਦੇ ਹੋ।
4. ਕਾਪੀ ਜਿਸ ਪਾਸੇ ਦਿਖਾਈ ਦੇਣੀ ਚਾਹੀਦੀ ਹੈ ਉੱਥੇ **ਕਰਸਰ ਲੈ ਜਾਓ** — ਇੱਕ ਲਾਈਵ ਪ੍ਰੀਵਿਊ ਪਿੱਛੇ-ਪਿੱਛੇ ਚੱਲਦਾ ਹੈ।
5. ਆਫਸੈੱਟ ਕਾਪੀ ਰੱਖਣ ਲਈ **ਕਲਿੱਕ ਕਰੋ**।

ਹਰੇਕ ਪਲੇਸਮੈਂਟ ਤੋਂ ਬਾਅਦ ਕਮਾਂਡ **ਉਸੇ ਦੂਰੀ** 'ਤੇ ਸਟੈਪ 3 'ਤੇ ਵਾਪਸ ਆ ਜਾਂਦੀ ਹੈ, ਅਗਲੇ ਆਫਸੈੱਟ ਲਈ ਤਿਆਰ। ਦੂਰੀ-ਐਂਟਰੀ ਪੜਾਅ 'ਤੇ ਵਾਪਸ ਰੀਸੈੱਟ ਕਰਨ ਲਈ **Escape** ਦਬਾਓ।

```
  Distance: 10

  ─────────────────    ← ਮੂਲ ਲਾਈਨ
  ─────────────────    ← ਆਫਸੈੱਟ ਕਾਪੀ (10 ਯੂਨਿਟ ਹੇਠਾਂ)
```

## ਹਰੇਕ ਐਂਟਿਟੀ ਲਈ ਆਫਸੈੱਟ ਵਿਵਹਾਰ

| ਐਂਟਿਟੀ | ਆਫਸੈੱਟ ਦੀ ਗਣਨਾ ਕਿਵੇਂ ਹੁੰਦੀ ਹੈ |
|--------|---------------------------|
| **Line** | ਮੂਲ ਦਿਸ਼ਾ ਦੇ ਲੰਬਵਤ ਸ਼ਿਫਟ ਕੀਤੀ ਸਮਾਂਤਰ ਲਾਈਨ |
| **Circle** | ਸਮਕੇਂਦਰੀ ਸਰਕਲ; ਬਾਹਰ ਕਲਿੱਕ ਕਰੋ → ਵੱਡਾ ਰੇਡੀਅਸ, ਅੰਦਰ → ਛੋਟਾ ਰੇਡੀਅਸ |
| **Arc** | ਨਵੇਂ ਰੇਡੀਅਸ 'ਤੇ ਸਮਕੇਂਦਰੀ ਆਰਕ; ਇੱਕੋ ਕੋਣੀ ਸਪੈਨ ਸੁਰੱਖਿਅਤ |
| **Ellipse** | ਦੋਵੇਂ ਸੈਮੀ-ਧੁਰੇ ਇੱਕੋ ਦੂਰੀ ਨਾਲ ਵਧਾਏ ਜਾਂ ਘਟਾਏ ਜਾਂਦੇ ਹਨ |
| **Polyline** | ਹਰੇਕ ਖੰਡ ਸੁਤੰਤਰ ਰੂਪ ਵਿੱਚ ਆਫਸੈੱਟ ਹੁੰਦਾ ਹੈ; ਨਾਲ ਲੱਗਦੇ ਆਫਸੈੱਟ ਖੰਡ ਕੋਨਿਆਂ 'ਤੇ ਮਾਈਟਰਡ ਹੁੰਦੇ ਹਨ |

**Circle**, **Arc**, ਅਤੇ **Ellipse** ਲਈ: ਜੇ ਅੰਦਰਲਾ ਆਫਸੈੱਟ ਕਿਸੇ ਵੀ ਰੇਡੀਅਸ ਜਾਂ ਸੈਮੀ-ਧੁਰੇ ਨੂੰ ਜ਼ੀਰੋ ਜਾਂ ਉਸ ਤੋਂ ਹੇਠਾਂ ਕਰ ਦੇਵੇ, ਤਾਂ ਆਫਸੈੱਟ ਲਾਗੂ ਨਹੀਂ ਹੁੰਦਾ।

## ਕੀਬੋਰਡ ਸੰਦਰਭ

| ਕੀ | ਕਾਰਵਾਈ |
|-----|--------|
| `0`–`9`, `.` | ਦੂਰੀ ਮੁੱਲ ਵਿੱਚ ਅੰਕ ਜੋੜੋ |
| `Backspace` | ਆਖਰੀ ਟਾਈਪ ਕੀਤਾ ਅੱਖਰ ਮਿਟਾਓ |
| `Enter` / `Space` | ਟਾਈਪ ਕੀਤੀ ਦੂਰੀ ਦੀ ਪੁਸ਼ਟੀ ਕਰੋ ਅਤੇ ਐਂਟਿਟੀ ਚੋਣ 'ਤੇ ਅੱਗੇ ਵਧੋ |
| `Escape` | ਦੂਰੀ-ਐਂਟਰੀ ਪੜਾਅ 'ਤੇ ਰੀਸੈੱਟ ਕਰੋ |

## ਵਰਕਫਲੋ ਨੋਟ

ਦੂਰੀ **Escape** ਦਬਾਉਣ ਤੱਕ ਸੈੱਟ ਰਹਿੰਦੀ ਹੈ। ਇਹ ਇੱਕੋ ਸਪੇਸਿੰਗ 'ਤੇ ਕਈ ਐਂਟਿਟੀਆਂ ਨੂੰ ਆਫਸੈੱਟ ਕਰਨਾ ਕੁਸ਼ਲ ਬਣਾਉਂਦਾ ਹੈ — ਦੂਰੀ ਇੱਕ ਵਾਰ ਟਾਈਪ ਕਰੋ, ਫਿਰ ਹਰੇਕ ਐਂਟਿਟੀ ਲਈ ਵਾਰੀ-ਵਾਰੀ ਕਲਿੱਕ ਕਰੋ ਅਤੇ ਸਾਈਡ ਚੁਣੋ।
