# Leader+

`leader+` ਕਮਾਂਡ ਮੌਜੂਦਾ ਮਲਟੀਲੀਡਰ ਵਿੱਚ ਇੱਕ ਨਵੀਂ ਤੀਰ ਭੁਜਾ ਜੋੜਦੀ ਹੈ। ਨਵੀਂ ਭੁਜਾ ਲੀਡਰ ਦੇ ਮੌਜੂਦਾ ਡੌਗਲੈੱਗ ਤੋਂ ਤੁਹਾਡੇ ਵੱਲੋਂ ਕਲਿੱਕ ਕੀਤੀ ਗਈ ਨਵੀਂ ਤੀਰ ਟਿਪ ਵੱਲ ਇਸ਼ਾਰਾ ਕਰਦੀ ਹੈ। ਸਾਰੀ ਸਟਾਈਲਿੰਗ — ਡੌਗਲੈੱਗ ਸਥਿਤੀ, ਟੈਕਸਟ, ਤੀਰ ਦੀ ਕਿਸਮ ਅਤੇ ਆਕਾਰ — ਚੁਣੇ ਗਏ ਲੀਡਰ ਤੋਂ ਵਿਰਾਸਤ ਵਿੱਚ ਮਿਲਦੀ ਹੈ।

## ਭੁਜਾ ਜੋੜਨਾ

1. ਟਰਮੀਨਲ ਵਿੱਚ `leader+` ਟਾਈਪ ਕਰੋ।
2. ਇਸਨੂੰ ਚੁਣਨ ਲਈ **ਮੌਜੂਦਾ ਮਲਟੀਲੀਡਰ ਉੱਤੇ ਕਲਿੱਕ ਕਰੋ**।
3. **ਨਵੀਂ ਤੀਰ ਟਿਪ ਉੱਤੇ ਕਲਿੱਕ ਕਰੋ**, ਜਾਂ ਸਟੀਕ ਨਿਰਦੇਸ਼ਾਂਕ ਲਈ `X,Y` ਟਾਈਪ ਕਰੋ ਅਤੇ **Enter** ਦਬਾਓ। ਕਰਸਰ ਤੋਂ ਲੀਡਰ ਦੇ ਡੌਗਲੈੱਗ ਤੱਕ ਇੱਕ ਪ੍ਰੀਵਿਊ ਲਾਈਨ ਦਿਖਾਈ ਦਿੰਦੀ ਹੈ।

ਭੁਜਾ ਰੱਖੀ ਜਾਂਦੀ ਹੈ ਅਤੇ ਕਮਾਂਡ ਸਰਗਰਮ ਰਹਿੰਦੀ ਹੈ — ਤੁਸੀਂ ਹੋਰ ਭੁਜਾਵਾਂ ਜੋੜਨ ਲਈ ਤੁਰੰਤ ਕਿਸੇ ਹੋਰ ਲੀਡਰ ਉੱਤੇ ਕਲਿੱਕ ਕਰ ਸਕਦੇ ਹੋ। ਸਮਾਪਤ ਕਰਨ ਲਈ **Enter**, **Space**, ਜਾਂ **Escape** ਦਬਾਓ।

```
  Before:                        After:
  ◄── arm 1                      ◄── arm 1
       \                               \
        ●──── dogleg ──── text          ●──── dogleg ──── text
                                       /
                                  arm 2 ──►  (new tip you clicked)
```

## ਟਿਪ ਲਈ ਨਿਰਦੇਸ਼ਾਂਕ ਐਂਟਰੀ

ਕਲਿੱਕ ਕਰਨ ਦੀ ਬਜਾਏ, ਤੁਸੀਂ ਇੱਕ ਸਟੀਕ ਸਥਿਤੀ ਟਾਈਪ ਕਰ ਸਕਦੇ ਹੋ:

1. X ਮੁੱਲ ਟਾਈਪ ਕਰੋ।
2. `,` ਦਬਾਓ — ਟਰਮੀਨਲ X ਲਾਕ ਹੋਣ ਦੀ ਪੁਸ਼ਟੀ ਕਰਦਾ ਹੈ।
3. Y ਮੁੱਲ ਟਾਈਪ ਕਰੋ।
4. ਰੱਖਣ ਲਈ **Enter** ਦਬਾਓ।

## ਕੀਬੋਰਡ ਸੰਦਰਭ

| ਕੁੰਜੀ | ਕਾਰਵਾਈ |
|-----|--------|
| `0`–`9`, `.`, `-` | X ਨਿਰਦੇਸ਼ਾਂਕ ਟਾਈਪ ਕਰਨਾ ਸ਼ੁਰੂ ਕਰੋ |
| `,` | X ਲਾਕ ਕਰੋ ਅਤੇ Y ਐਂਟਰੀ ਵੱਲ ਜਾਓ |
| `Enter` | ਟਾਈਪ ਕੀਤੇ ਨਿਰਦੇਸ਼ਾਂਕ ਦੀ ਪੁਸ਼ਟੀ ਕਰੋ ਅਤੇ ਭੁਜਾ ਰੱਖੋ |
| `Enter` / `Space` | ਸਮਾਪਤ ਕਰੋ (ਜਦੋਂ ਕੋਈ ਇਨਪੁੱਟ ਜਾਰੀ ਨਾ ਹੋਵੇ) |
| `Escape` | ਰੱਦ ਕਰੋ ਅਤੇ ਰੀਸੈੱਟ ਕਰੋ |

## ਨੋਟਸ

- ਸਿਰਫ਼ **Multileader** ਇਕਾਈਆਂ ਹੀ ਚੁਣੀਆਂ ਜਾ ਸਕਦੀਆਂ ਹਨ — ਕਿਸੇ ਹੋਰ ਇਕਾਈ ਕਿਸਮ ਉੱਤੇ ਕਲਿੱਕ ਕਰਨ ਨਾਲ ਕੁਝ ਨਹੀਂ ਹੁੰਦਾ।
- ਨਵੀਂ ਭੁਜਾ ਮੌਜੂਦਾ ਡੌਗਲੈੱਗ ਤੋਂ ਸ਼ੁਰੂ ਹੁੰਦੀ ਹੈ; ਤੁਸੀਂ ਸਿਰਫ਼ ਇਹ ਚੁਣਦੇ ਹੋ ਕਿ ਤੀਰ ਟਿਪ ਕਿੱਥੇ ਜਾਵੇ।
- ਇੱਕ ਮਲਟੀਲੀਡਰ ਵਿੱਚ ਕਿੰਨੀਆਂ ਭੁਜਾਵਾਂ ਹੋ ਸਕਦੀਆਂ ਹਨ, ਇਸਦੀ ਕੋਈ ਸੀਮਾ ਨਹੀਂ ਹੈ।
