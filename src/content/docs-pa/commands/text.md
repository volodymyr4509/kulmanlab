# Text

`text` ਕਮਾਂਡ ਇੱਕ ਮਲਟੀ-ਲਾਈਨ ਟੈਕਸਟ ਲੇਬਲ ਰੱਖਦੀ ਹੈ। ਕੈਨਵਸ 'ਤੇ ਕਿਸੇ ਸਥਿਤੀ 'ਤੇ ਕਲਿੱਕ ਕਰਨ ਤੋਂ ਬਾਅਦ, ਇੱਕ ਪੌਪਅੱਪ ਐਡੀਟਰ **rich** ਮੋਡ ਵਿੱਚ ਖੁੱਲ੍ਹਦਾ ਹੈ — ਤੁਸੀਂ ਸਮੱਗਰੀ ਟਾਈਪ ਕਰ ਸਕਦੇ ਹੋ, ਹਰੇਕ ਅੱਖਰ 'ਤੇ ਬੋਲਡ/ਇਟੈਲਿਕ/ਸਟ੍ਰਾਈਕਥਰੂ ਲਾਗੂ ਕਰ ਸਕਦੇ ਹੋ, ਫੌਂਟ ਅਤੇ ਉਚਾਈ ਬਦਲ ਸਕਦੇ ਹੋ, ਅਤੇ ਲਾਈਨ ਬ੍ਰੇਕ ਪਾ ਸਕਦੇ ਹੋ। ਕਮਿੱਟ ਕਰਨ ਅਤੇ ਐਡੀਟਰ ਬੰਦ ਕਰਨ ਲਈ **Escape** ਦਬਾਓ।

ਪੂਰੇ ਐਡੀਟਰ ਰੈਫਰੈਂਸ ਲਈ, **rich** ਅਤੇ **simple** ਮੋਡਾਂ ਦੀ ਤੁਲਨਾ ਸਮੇਤ, [Text Editor](../interface/text-editor) ਪੇਜ ਵੇਖੋ।

## ਟੈਕਸਟ ਲੇਬਲ ਰੱਖਣਾ

1. ਟਰਮੀਨਲ ਵਿੱਚ `text` ਟਾਈਪ ਕਰੋ ਜਾਂ **Text** ਟੂਲਬਾਰ ਬਟਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ।
2. ਕੈਨਵਸ 'ਤੇ **ਐਂਕਰ ਸਥਿਤੀ 'ਤੇ ਕਲਿੱਕ ਕਰੋ**। ਜਾਂ ਸਹੀ ਕੋਆਰਡੀਨੇਟ ਲਈ `X,Y` ਟਾਈਪ ਕਰੋ ਅਤੇ **Enter** ਦਬਾਓ।
3. ਨਵੇਂ ਲੇਬਲ ਦੇ ਉੱਪਰ **ਟੈਕਸਟ ਐਡੀਟਰ ਪੌਪਅੱਪ** ਖੁੱਲ੍ਹਦਾ ਹੈ। ਆਪਣੀ ਸਮੱਗਰੀ ਟਾਈਪ ਕਰੋ।
4. ਲੇਬਲ ਕਮਿੱਟ ਕਰਨ ਅਤੇ ਐਡੀਟਰ ਬੰਦ ਕਰਨ ਲਈ **Escape** ਦਬਾਓ।

ਡਿਫਾਲਟ ਉਚਾਈ **12 ਡਰਾਇੰਗ ਯੂਨਿਟ** ਹੈ।

## ਮੌਜੂਦਾ ਲੇਬਲ ਦਾ ਸੰਪਾਦਨ

ਕੈਨਵਸ 'ਤੇ ਕਿਸੇ ਵੀ ਟੈਕਸਟ ਲੇਬਲ 'ਤੇ **ਡਬਲ-ਕਲਿੱਕ** ਕਰਨ ਨਾਲ ਉਸ ਲੇਬਲ ਲਈ ਐਡੀਟਰ ਦੁਬਾਰਾ ਖੁੱਲ੍ਹਦਾ ਹੈ।

## ਐਂਕਰ ਲਈ ਕੋਆਰਡੀਨੇਟ ਐਂਟਰੀ

ਕਲਿੱਕ ਕਰਨ ਦੀ ਬਜਾਏ, ਇੱਕ ਸਹੀ ਸਥਿਤੀ ਟਾਈਪ ਕਰੋ:

1. X ਮੁੱਲ ਟਾਈਪ ਕਰੋ।
2. `,` ਦਬਾਓ — ਟਰਮੀਨਲ `[X], [Y{cursor}]` ਦਿਖਾਉਂਦਾ ਹੈ।
3. Y ਮੁੱਲ ਟਾਈਪ ਕਰੋ।
4. ਐਂਕਰ ਰੱਖਣ ਅਤੇ ਐਡੀਟਰ ਖੋਲ੍ਹਣ ਲਈ **Enter** ਦਬਾਓ।

## ਕੀਬੋਰਡ ਰੈਫਰੈਂਸ

**ਐਂਕਰ ਪੜਾਅ**

| ਕੀ | ਕਾਰਵਾਈ |
|-----|--------|
| `0`–`9`, `.`, `-` | X ਕੋਆਰਡੀਨੇਟ ਐਂਟਰੀ ਸ਼ੁਰੂ ਕਰੋ |
| `,` | X ਲਾਕ ਕਰੋ ਅਤੇ Y ਐਂਟਰੀ 'ਤੇ ਜਾਓ |
| `Backspace` | ਆਖਰੀ ਟਾਈਪ ਕੀਤਾ ਅੱਖਰ ਮਿਟਾਓ |
| `Enter` | ਟਾਈਪ ਕੀਤਾ ਕੋਆਰਡੀਨੇਟ ਪੁਸ਼ਟੀ ਕਰੋ |

**ਟੈਕਸਟ ਐਡੀਟਰ ਪੜਾਅ** (ਪੂਰੇ ਰੈਫਰੈਂਸ ਲਈ [Text Editor](../interface/text-editor) ਵੇਖੋ)

| ਕੀ | ਕਾਰਵਾਈ |
|-----|--------|
| ਕੋਈ ਵੀ ਛਾਪਣਯੋਗ ਅੱਖਰ | ਕਰਸਰ 'ਤੇ ਪਾਓ |
| `Backspace` / `Delete` | ਨਾਲ ਲੱਗਦਾ ਅੱਖਰ ਜਾਂ ਚੋਣ ਮਿਟਾਓ |
| `Enter` | ਲਾਈਨ ਬ੍ਰੇਕ ਪਾਓ |
| `←` / `→` | ਕੈਰੇਟ ਹਿਲਾਓ |
| `Home` / `End` | ਹਾਰਡ ਲਾਈਨ ਦੇ ਸ਼ੁਰੂ / ਅੰਤ 'ਤੇ ਜਾਓ |
| `Escape` | ਕਮਿੱਟ ਕਰੋ ਅਤੇ ਐਡੀਟਰ ਬੰਦ ਕਰੋ |

**ਆਮ**

## DXF — MTEXT ਐਂਟਿਟੀ

ਟੈਕਸਟ ਲੇਬਲ DXF ਫਾਈਲ ਵਿੱਚ **MTEXT** ਐਂਟਿਟੀਆਂ ਦੇ ਰੂਪ ਵਿੱਚ ਸੰਭਾਲੇ ਜਾਂਦੇ ਹਨ। ਬੋਲਡ ਅਤੇ ਇਟੈਲਿਕ ਨੂੰ `\L`, `\K`, `\O`, ਅਤੇ ਇਨਲਾਈਨ ਫੌਂਟ ਸਵਿੱਚਾਂ (`\f`) ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਏਨਕੋਡ ਕੀਤਾ ਜਾਂਦਾ ਹੈ। ਹਰੇਕ ਅੱਖਰ ਦੀ ਉਚਾਈ `\H` ਵਜੋਂ ਏਨਕੋਡ ਕੀਤੀ ਜਾਂਦੀ ਹੈ। ਸਾਰੀ ਫਾਰਮੈਟਿੰਗ ਪੂਰੀ DXF ਰਾਊਂਡ-ਟ੍ਰਿਪ ਵਿੱਚੋਂ ਬਚੀ ਰਹਿੰਦੀ ਹੈ ਅਤੇ LibreCAD, FreeCAD, ਅਤੇ ਹੋਰ DXF-ਅਨੁਕੂਲ ਐਪਲੀਕੇਸ਼ਨਾਂ ਦੁਆਰਾ ਪੜ੍ਹਨਯੋਗ ਹੈ।
