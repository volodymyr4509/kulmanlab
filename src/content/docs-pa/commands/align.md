# Align

`align` ਕਮਾਂਡ ਇੱਕ ਜਾਂ ਦੋ ਸੋਰਸ/ਡੈਸਟੀਨੇਸ਼ਨ ਪੁਆਇੰਟ ਜੋੜਿਆਂ ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਚੁਣੀਆਂ ਐਂਟਿਟੀਆਂ ਨੂੰ ਦੁਬਾਰਾ ਸਥਾਪਿਤ ਕਰਦੀ ਹੈ। ਇੱਕ ਜੋੜੇ ਨਾਲ ਇਹ ਬਿਲਕੁਲ [Move](../move/) ਵਾਂਗ ਕੰਮ ਕਰਦੀ ਹੈ (ਸਿਰਫ਼ ਟ੍ਰਾਂਸਲੇਟ)। ਦੋ ਜੋੜਿਆਂ ਨਾਲ ਇਹ ਚੋਣ ਨੂੰ ਵੀ ਇਸ ਤਰ੍ਹਾਂ ਰੋਟੇਟ ਕਰਦੀ ਹੈ ਕਿ ਸੋਰਸ-ਤੋਂ-ਸੋਰਸ ਦਿਸ਼ਾ ਡੈਸਟੀਨੇਸ਼ਨ-ਤੋਂ-ਡੈਸਟੀਨੇਸ਼ਨ ਦਿਸ਼ਾ ਨਾਲ ਮੇਲ ਖਾਵੇ, ਅਤੇ ਵਿਕਲਪਿਕ ਤੌਰ 'ਤੇ ਇਸਨੂੰ ਇਸ ਤਰ੍ਹਾਂ ਸਕੇਲ ਕਰ ਸਕਦੀ ਹੈ ਕਿ ਸੋਰਸ ਸੈਗਮੈਂਟ ਦੀ ਲੰਬਾਈ ਡੈਸਟੀਨੇਸ਼ਨ ਸੈਗਮੈਂਟ ਦੀ ਲੰਬਾਈ ਨਾਲ ਮੇਲ ਖਾਵੇ — ਇੱਕ ਹੀ ਕਾਰਵਾਈ ਵਿੱਚ ਟ੍ਰਾਂਸਲੇਟ, ਰੋਟੇਟ, ਅਤੇ ਸਕੇਲ।

## ਸ਼ੁਰੂ ਕਰਨ ਦੇ ਦੋ ਤਰੀਕੇ

**ਪਹਿਲਾਂ ਚੁਣੋ, ਫਿਰ ਅਲਾਈਨ ਕਰੋ** — ਪਹਿਲਾਂ ਐਂਟਿਟੀਆਂ ਚੁਣੋ, ਫਿਰ ਸਰਗਰਮ ਕਰੋ:

1. ਕੈਨਵਸ 'ਤੇ ਇੱਕ ਜਾਂ ਵੱਧ ਐਂਟਿਟੀਆਂ ਚੁਣੋ।
2. ਟਰਮੀਨਲ ਵਿੱਚ `align` ਟਾਈਪ ਕਰੋ ਜਾਂ **Align** ਟੂਲਬਾਰ ਬਟਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ।
3. **ਪਹਿਲੇ ਸੋਰਸ ਪੁਆਇੰਟ (S1) 'ਤੇ ਕਲਿੱਕ ਕਰੋ**, ਫਿਰ **ਪਹਿਲੇ ਡੈਸਟੀਨੇਸ਼ਨ ਪੁਆਇੰਟ (D1) 'ਤੇ**।
4. **ਦੂਜੇ ਸੋਰਸ ਪੁਆਇੰਟ (S2) 'ਤੇ ਕਲਿੱਕ ਕਰੋ**, ਜਾਂ ਹੁਣੇ ਹੀ ਸਿਰਫ਼ ਟ੍ਰਾਂਸਲੇਟ ਵਾਲੀ ਅਲਾਈਨਮੈਂਟ ਲਾਗੂ ਕਰਨ ਲਈ **Enter** ਦਬਾਓ।
5. **ਦੂਜੇ ਡੈਸਟੀਨੇਸ਼ਨ ਪੁਆਇੰਟ (D2) 'ਤੇ ਕਲਿੱਕ ਕਰੋ**।
6. ਸਕੇਲ ਪ੍ਰੌਂਪਟ ਦਾ ਜਵਾਬ ਦਿਓ: ਸਕੇਲ ਕਰਨ ਲਈ **Y** ਦਬਾਓ, ਜਾਂ ਮੂਲ ਆਕਾਰ ਬਣਾਈ ਰੱਖਣ ਲਈ **N** / **Enter** ਦਬਾਓ।

**ਸਰਗਰਮ ਕਰੋ, ਫਿਰ ਚੁਣੋ** — ਕੁਝ ਵੀ ਚੁਣੇ ਬਿਨਾਂ ਕਮਾਂਡ ਸ਼ੁਰੂ ਕਰੋ:

1. `align` ਟਾਈਪ ਕਰੋ ਜਾਂ ਟੂਲਬਾਰ ਬਟਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ।
2. **ਆਬਜੈਕਟ ਚੁਣੋ** — ਵਿਅਕਤੀਗਤ ਐਂਟਿਟੀਆਂ ਟੌਗਲ ਕਰਨ ਲਈ ਕਲਿੱਕ ਕਰੋ, ਜਾਂ ਖੇਤਰ ਦੁਆਰਾ ਚੁਣਨ ਲਈ ਖਿੱਚੋ।
3. ਚੋਣ ਪੁਸ਼ਟੀ ਕਰਨ ਲਈ **Enter** ਜਾਂ **Space** ਦਬਾਓ।
4. ਉੱਪਰ ਦੱਸੇ ਅਨੁਸਾਰ S1 → D1 → S2 → D2 → ਸਕੇਲ ਪ੍ਰੌਂਪਟ ਨਾਲ ਜਾਰੀ ਰੱਖੋ।

> ਟਰਮੀਨਲ ਨੂੰ ਸਿਰਫ਼ ਇੰਨੇ ਅੱਖਰਾਂ ਦੀ ਲੋੜ ਹੁੰਦੀ ਹੈ ਜਿੰਨੇ ਅਸਪਸ਼ਟਤਾ ਨਾ ਹੋਣ ਲਈ ਕਾਫ਼ੀ ਹੋਣ — `al` ਟਾਈਪ ਕਰਕੇ **Enter** ਦਬਾਉਣ ਨਾਲ Align ਸਿੱਧਾ ਸਰਗਰਮ ਹੋ ਜਾਂਦੀ ਹੈ, ਕਿਉਂਕਿ ਕੋਈ ਵੀ ਹੋਰ ਕਮਾਂਡ ਇਹਨਾਂ ਦੋ ਅੱਖਰਾਂ ਨਾਲ ਸ਼ੁਰੂ ਨਹੀਂ ਹੁੰਦੀ।

## ਇੱਕ ਅਲਾਈਨਮੈਂਟ ਦੀ ਬਣਤਰ

```
  ਸੋਰਸ ਪੁਆਇੰਟ (ਐਂਟਿਟੀਆਂ 'ਤੇ):          ਡੈਸਟੀਨੇਸ਼ਨ ਪੁਆਇੰਟ:
  ● S1                                 ● D1
   \                                    \
    ● S2                                 ● D2

  ਨਤੀਜਾ: ਚੋਣ ਨੂੰ ਇਸ ਤਰ੍ਹਾਂ ਟ੍ਰਾਂਸਲੇਟ ਕੀਤਾ ਜਾਂਦਾ ਹੈ ਕਿ S1, D1 'ਤੇ
  ਆ ਜਾਵੇ, ਫਿਰ D1 ਦੇ ਦੁਆਲੇ ਇਸ ਤਰ੍ਹਾਂ ਰੋਟੇਟ ਕੀਤਾ ਜਾਂਦਾ ਹੈ ਕਿ S1→S2
  ਦਿਸ਼ਾ D1→D2 ਦਿਸ਼ਾ ਨਾਲ ਮੇਲ ਖਾਵੇ — ਅਤੇ, ਜੇ ਤੁਸੀਂ ਸਕੇਲ ਕਰਨਾ
  ਚੁਣਦੇ ਹੋ, ਤਾਂ |S1S2|, |D1D2| ਬਣਨ ਲਈ ਆਕਾਰ ਬਦਲਿਆ ਜਾਂਦਾ ਹੈ।
```

ਹਰ ਪੜਾਅ 'ਤੇ ਇੱਕ ਭੂਤ ਪ੍ਰੀਵਿਊ ਕਰਸਰ ਦਾ ਪਿੱਛਾ ਕਰਦਾ ਹੈ: D1 ਰੱਖਦੇ ਸਮੇਂ ਇੱਕ ਮੂਵ ਪ੍ਰੀਵਿਊ, ਫਿਰ D2 ਨੂੰ ਸਥਾਪਿਤ ਕਰਦੇ ਸਮੇਂ ਇੱਕ ਡੈਸ਼ਡ ਘੁੰਮਾਇਆ ਪ੍ਰੀਵਿਊ।

## ਇੱਕ ਪੁਆਇੰਟ ਦੀ ਅਲਾਈਨਮੈਂਟ (ਸਿਰਫ਼ ਟ੍ਰਾਂਸਲੇਟ)

D1 ਰੱਖੇ ਜਾਣ ਤੋਂ ਬਾਅਦ, ਦੂਜੇ ਸੋਰਸ ਪੁਆਇੰਟ 'ਤੇ ਕਲਿੱਕ ਕਰਨ ਦੀ ਬਜਾਏ **Enter** ਦਬਾਓ। ਚੋਣ ਨੂੰ S1→D1 ਵੈਕਟਰ ਨਾਲ ਟ੍ਰਾਂਸਲੇਟ ਕੀਤਾ ਜਾਂਦਾ ਹੈ — ਬਿਨਾਂ ਕਿਸੇ ਰੋਟੇਸ਼ਨ ਜਾਂ ਸਕੇਲ ਦੇ — ਇਹ ਬਿਲਕੁਲ S1 ਨੂੰ ਬੇਸ ਪੁਆਇੰਟ ਅਤੇ D1 ਨੂੰ ਮੰਜ਼ਿਲ ਵਜੋਂ ਵਰਤਦੇ ਹੋਏ ਇੱਕ [Move](../move/) ਵਰਗਾ ਹੈ।

## ਦੋ ਪੁਆਇੰਟਾਂ ਦੀ ਅਲਾਈਨਮੈਂਟ (ਟ੍ਰਾਂਸਲੇਟ + ਰੋਟੇਟ + ਵਿਕਲਪਿਕ ਸਕੇਲ)

ਜਦੋਂ S2 ਅਤੇ D2 ਦੋਵੇਂ ਰੱਖੇ ਜਾ ਚੁੱਕੇ ਹੋਣ:

- **ਰੋਟੇਸ਼ਨ ਐਂਗਲ** — ਡੈਸਟੀਨੇਸ਼ਨ ਦਿਸ਼ਾ (`D1 → D2`) ਅਤੇ ਸੋਰਸ ਦਿਸ਼ਾ (`S1 → S2`) ਵਿਚਕਾਰ ਅੰਤਰ।
- **ਸਕੇਲ ਪ੍ਰੌਂਪਟ** — `scale objects to alignment points? [Yes/No] <N>` ਦਿਖਾਈ ਦਿੰਦਾ ਹੈ, ਜਿਸ ਵਿੱਚ ਡਿਫੌਲਟ **No** ਹੈ:
  - ਚੋਣ ਨੂੰ D1 ਦੇ ਦੁਆਲੇ ਇਕਸਾਰਤਾ ਨਾਲ ਵੀ ਸਕੇਲ ਕਰਨ ਲਈ **Y** ਦਬਾਓ ਤਾਂ ਜੋ `S1–S2` ਦੀ ਦੂਰੀ `D1–D2` ਦੀ ਦੂਰੀ ਬਣ ਜਾਵੇ।
  - ਮੂਲ ਆਕਾਰ ਬਣਾਈ ਰੱਖਣ ਲਈ **N** ਜਾਂ **Enter** ਦਬਾਓ — ਸਿਰਫ਼ ਟ੍ਰਾਂਸਲੇਟ ਅਤੇ ਰੋਟੇਸ਼ਨ ਲਾਗੂ ਹੁੰਦੇ ਹਨ।

ਸਕੇਲ ਪ੍ਰੌਂਪਟ 'ਤੇ ਕੋਈ ਵੀ ਕੀ ਦਬਾਉਣ ਨਾਲ ਅਲਾਈਨਮੈਂਟ ਤੁਰੰਤ ਲਾਗੂ ਹੋ ਜਾਂਦੀ ਹੈ — Yes ਜਾਂ No ਚੁਣਨ ਤੋਂ ਬਾਅਦ ਕੋਈ ਵੱਖਰਾ ਪੁਸ਼ਟੀਕਰਨ ਪੜਾਅ ਨਹੀਂ ਹੁੰਦਾ।

## ਕੀਬੋਰਡ ਰੈਫਰੈਂਸ

| ਕੀ | ਕਾਰਵਾਈ |
|-----|--------|
| `Enter` / `Space` | ਚੋਣ ਪੁਸ਼ਟੀ ਕਰੋ ਅਤੇ S1 ਪੜਾਅ 'ਤੇ ਅੱਗੇ ਵਧੋ |
| `Enter` (S2 ਪੜਾਅ 'ਤੇ) | ਰੋਟੇਸ਼ਨ ਛੱਡੋ — S1 ਅਤੇ D1 ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਸਿਰਫ਼ ਟ੍ਰਾਂਸਲੇਟ ਵਾਲੀ ਅਲਾਈਨਮੈਂਟ ਲਾਗੂ ਕਰੋ |
| `Y` | ਸਕੇਲ ਨਾਲ ਅਲਾਈਨਮੈਂਟ ਲਾਗੂ ਕਰੋ |
| `N` / `Enter` (ਸਕੇਲ ਪ੍ਰੌਂਪਟ 'ਤੇ) | ਬਿਨਾਂ ਸਕੇਲ ਦੇ ਅਲਾਈਨਮੈਂਟ ਲਾਗੂ ਕਰੋ |
| `Escape` | ਪੁਆਇੰਟ ਰੱਖਦੇ ਸਮੇਂ: ਰੱਖੇ ਪੁਆਇੰਟਾਂ ਨੂੰ ਰੱਦ ਕਰਦਾ ਹੈ ਅਤੇ ਚੋਣ ਪੜਾਅ 'ਤੇ ਵਾਪਸ ਜਾਂਦਾ ਹੈ; ਕੁਝ ਵੀ ਨਾ ਚੁਣੇ ਹੋਣ 'ਤੇ: ਕਮਾਂਡ ਰੱਦ ਕਰਦਾ ਹੈ |

## ਕਮਾਂਡ ਦੌਰਾਨ ਚੋਣ

ਜਦੋਂ ਕਮਾਂਡ ਚੋਣ ਪੜਾਅ ਵਿੱਚ ਸ਼ੁਰੂ ਹੁੰਦੀ ਹੈ:

| ਢੰਗ | ਵਿਵਹਾਰ |
|--------|-----------|
| **ਕਲਿੱਕ** | ਕਰਸਰ ਹੇਠਾਂ ਐਂਟਿਟੀ ਨੂੰ ਚੋਣ ਵਿੱਚ/ਬਾਹਰ ਟੌਗਲ ਕਰਦਾ ਹੈ |
| **ਸੱਜੇ ਖਿੱਚੋ** (ਸਖ਼ਤ) | ਬਾਕਸ ਦੇ ਪੂਰੀ ਤਰ੍ਹਾਂ ਅੰਦਰ ਵਾਲੀਆਂ ਐਂਟਿਟੀਆਂ ਜੋੜਦਾ ਹੈ |
| **ਖੱਬੇ ਖਿੱਚੋ** (ਕ੍ਰਾਸਿੰਗ) | ਬਾਕਸ ਦੀ ਸੀਮਾ ਨੂੰ ਕੱਟਣ ਵਾਲੀਆਂ ਐਂਟਿਟੀਆਂ ਜੋੜਦਾ ਹੈ |
| **Enter** / **Space** | ਚੋਣ ਪੁਸ਼ਟੀ ਕਰਦਾ ਹੈ ਅਤੇ S1 ਪੜਾਅ 'ਤੇ ਜਾਂਦਾ ਹੈ |

## ਅਲਾਈਨ ਕਰਨ ਤੋਂ ਬਾਅਦ

ਅਲਾਈਨ ਕੀਤੀਆਂ ਐਂਟਿਟੀਆਂ ਆਪਣੀ ਨਵੀਂ ਸਥਿਤੀ 'ਤੇ ਚੁਣੀਆਂ ਰਹਿੰਦੀਆਂ ਹਨ, ਅਤੇ ਕਮਾਂਡ ਆਪਣੇ ਆਪ ਖਤਮ ਹੋ ਜਾਂਦੀ ਹੈ — **Align** ਦੁਬਾਰਾ ਚਲਾਓ, ਜਾਂ ਦੁਬਾਰਾ ਚੁਣੇ ਬਿਨਾਂ [Move](../move/), [Rotate](../rotate/), ਜਾਂ [Scale](../scale/) 'ਤੇ ਸਵਿੱਚ ਕਰੋ।

## Align ਬਨਾਮ Move

| | Align | Move |
|---|-------|------|
| ਪੁਆਇੰਟ ਜੋੜੇ | 1 (ਸਿਰਫ਼ ਟ੍ਰਾਂਸਲੇਟ) ਜਾਂ 2 (ਟ੍ਰਾਂਸਲੇਟ + ਰੋਟੇਟ + ਸਕੇਲ) | 1 (ਸਿਰਫ਼ ਟ੍ਰਾਂਸਲੇਟ) |
| ਰੋਟੇਸ਼ਨ | ਹਾਂ, ਦੂਜੇ ਪੁਆਇੰਟ ਜੋੜੇ ਨਾਲ | ਨਹੀਂ |
| ਸਕੇਲਿੰਗ | ਦੂਜੇ ਪੁਆਇੰਟ ਜੋੜੇ ਨਾਲ ਵਿਕਲਪਿਕ | ਨਹੀਂ |
| ਸਭ ਤੋਂ ਵਧੀਆ | ਹਵਾਲਾ ਪੁਆਇੰਟਾਂ ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਇੱਕ ਆਕਾਰ ਨੂੰ ਦੂਜੇ ਵਿੱਚ ਫਿੱਟ ਕਰਨਾ | ਸਧਾਰਨ ਮੁੜ-ਸਥਾਪਨਾ |

## ਸਮਰਥਿਤ ਐਂਟਿਟੀਆਂ

Align, Move, Rotate, ਅਤੇ Scale ਦੁਆਰਾ ਸਮਰਥਿਤ ਹਰੇਕ ਐਂਟਿਟੀ ਕਿਸਮ 'ਤੇ ਕੰਮ ਕਰਦੀ ਹੈ — ਇਹਨਾਂ ਕਮਾਂਡਾਂ ਦੁਆਰਾ ਵਰਤੇ ਜਾਂਦੇ ਉਹੀ `translate`, `rotate`, ਅਤੇ `scale` ਓਪਰੇਸ਼ਨ ਕ੍ਰਮ ਵਿੱਚ ਲਾਗੂ ਹੁੰਦੇ ਹਨ, ਇਸ ਲਈ ਕੋਈ ਵੀ ਬਾਹਰ ਨਹੀਂ ਹੈ।
