# Dimension Aligned

`dimaligned` ਕਮਾਂਡ ਇੱਕ ਡਾਇਮੈਂਸ਼ਨ ਰੱਖਦੀ ਹੈ ਜੋ ਦੋ ਬਿੰਦੂਆਂ ਵਿਚਕਾਰ **ਅਸਲ ਸਿੱਧੀ-ਲਾਈਨ ਦੂਰੀ** ਮਾਪਦੀ ਹੈ। ਡਾਇਮੈਂਸ਼ਨ ਲਾਈਨ ਦੋਹਾਂ ਬਿੰਦੂਆਂ ਨੂੰ ਜੋੜਨ ਵਾਲੀ ਲਾਈਨ ਦੇ ਸਮਾਨਾਂਤਰ ਚੱਲਦੀ ਹੈ, ਇਸ ਲਈ ਇਹ ਕਿਸੇ ਵੀ ਕੋਣ 'ਤੇ ਹੋ ਸਕਦੀ ਹੈ। ਇਹੀ [Dimension Linear](../dim-linear/) ਤੋਂ ਮੁੱਖ ਫਰਕ ਹੈ, ਜੋ ਹਰੀਜ਼ੌਂਟਲ ਜਾਂ ਵਰਟੀਕਲ ਤੱਕ ਸੀਮਿਤ ਹੈ।

## ਇੱਕ ਅਲਾਈਨਡ ਡਾਇਮੈਂਸ਼ਨ ਦੀ ਸੰਰਚਨਾ

```
     ●  p2
    /|
   / |  (ext line 2, perpendicular to dim line)
  /  |
 /←5.00→/
/  /
●  /  (ext line 1, perpendicular to dim line)
p1
```

- **ਐਕਸਟੈਂਸ਼ਨ ਲਾਈਨਾਂ** — ਡਾਇਮੈਂਸ਼ਨ ਲਾਈਨ ਦੇ ਲੰਬਵਤ, ਹਰੇਕ ਮਾਪੇ ਗਏ ਬਿੰਦੂ ਤੋਂ ਖਿੱਚੀਆਂ ਜਾਂਦੀਆਂ ਹਨ।
- **ਡਾਇਮੈਂਸ਼ਨ ਲਾਈਨ** — p1→p2 ਦੇ ਸਮਾਨਾਂਤਰ, ਕਰਸਰ ਸਥਿਤੀ ਦੁਆਰਾ ਇੱਕ ਪਾਸੇ ਆਫਸੈੱਟ ਕੀਤੀ ਗਈ।
- **ਮੁੱਲ** — ਅਸਲ ਯੂਕਲਿਡੀਅਨ ਦੂਰੀ `|p1 – p2|`।

## ਇੱਕ ਅਲਾਈਨਡ ਡਾਇਮੈਂਸ਼ਨ ਰੱਖਣਾ

1. ਟਰਮੀਨਲ ਵਿੱਚ `dimaligned` ਟਾਈਪ ਕਰੋ ਜਾਂ **Dimension Aligned** ਟੂਲਬਾਰ ਬਟਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ।
2. **ਪਹਿਲੇ ਐਕਸਟੈਂਸ਼ਨ ਲਾਈਨ ਮੂਲ 'ਤੇ ਕਲਿੱਕ ਕਰੋ** (p1), ਜਾਂ ਸਟੀਕ ਨਿਰਦੇਸ਼ਾਂਕ ਲਈ `X,Y` ਟਾਈਪ ਕਰੋ ਅਤੇ **Enter** ਦਬਾਓ।
3. **ਦੂਜੇ ਐਕਸਟੈਂਸ਼ਨ ਲਾਈਨ ਮੂਲ 'ਤੇ ਕਲਿੱਕ ਕਰੋ** (p2)। ਇੱਥੇ ਵੀ ਨਿਰਦੇਸ਼ਾਂਕ ਐਂਟਰੀ ਕੰਮ ਕਰਦੀ ਹੈ।
4. ਡਾਇਮੈਂਸ਼ਨ ਲਾਈਨ ਦਾ ਲੰਬਵਤ ਆਫਸੈੱਟ ਸੈੱਟ ਕਰਨ ਲਈ **ਕਰਸਰ ਨੂੰ ਇੱਕ ਪਾਸੇ ਹਿਲਾਓ**।
5. ਰੱਖਣ ਲਈ **ਕਲਿੱਕ ਕਰੋ**, ਜਾਂ ਸਟੀਕ ਪਲੇਸਮੈਂਟ ਲਈ ਇੱਕ ਆਫਸੈੱਟ ਦੂਰੀ ਟਾਈਪ ਕਰੋ ਅਤੇ **Enter** ਦਬਾਓ।

## ਟਾਈਪ ਕੀਤੀ ਆਫਸੈੱਟ ਦੂਰੀ

ਪਲੇਸਮੈਂਟ ਦੌਰਾਨ ਇੱਕ ਨੰਬਰ ਟਾਈਪ ਕਰਕੇ ਡਾਇਮੈਂਸ਼ਨ ਲਾਈਨ ਨੂੰ p1→p2 ਲਾਈਨ ਤੋਂ ਸਟੀਕ ਲੰਬਵਤ ਦੂਰੀ 'ਤੇ ਸਥਿਰ ਕਰੋ:

| ਕੁੰਜੀ | ਕਾਰਵਾਈ |
|-----|--------|
| `0`–`9`, `.` | ਆਫਸੈੱਟ ਵਿੱਚ ਅੰਕ ਜੋੜੋ |
| `Backspace` | ਆਖਰੀ ਟਾਈਪ ਕੀਤਾ ਅੱਖਰ ਮਿਟਾਓ |
| `Enter` / `Space` | ਟਾਈਪ ਕੀਤੇ ਆਫਸੈੱਟ 'ਤੇ ਰੱਖੋ |

ਕਰਸਰ ਪਾਸਾ ਤੈਅ ਕਰਦਾ ਹੈ ਕਿ ਡਾਇਮੈਂਸ਼ਨ ਲਾਈਨ ਕਿਸ ਪਾਸੇ ਦਿਖਾਈ ਦਿੰਦੀ ਹੈ।

## ਕੀਬੋਰਡ ਸੰਦਰਭ

| ਕੁੰਜੀ | ਕਾਰਵਾਈ |
|-----|--------|
| `0`–`9`, `.`, `-` | X ਨਿਰਦੇਸ਼ਾਂਕ ਐਂਟਰੀ ਸ਼ੁਰੂ ਕਰੋ (p1/p2 ਪੜਾਅ), ਜਾਂ ਆਫਸੈੱਟ ਦੂਰੀ (ਪਲੇਸਮੈਂਟ ਪੜਾਅ) |
| `,` | X ਲਾਕ ਕਰੋ ਅਤੇ Y ਐਂਟਰੀ ਵੱਲ ਜਾਓ (p1/p2 ਪੜਾਅ) |
| `Backspace` | ਆਖਰੀ ਟਾਈਪ ਕੀਤਾ ਅੱਖਰ ਮਿਟਾਓ |
| `Enter` / `Space` | ਟਾਈਪ ਕੀਤੇ ਨਿਰਦੇਸ਼ਾਂਕ ਜਾਂ ਆਫਸੈੱਟ ਦੀ ਪੁਸ਼ਟੀ ਕਰੋ |
| `Escape` | ਰੱਦ ਕਰੋ |

## Dimension Aligned ਬਨਾਮ Dimension Linear

| | Dimension Aligned | Dimension Linear |
|---|------------------|-----------------|
| ਡਾਇਮੈਂਸ਼ਨ ਲਾਈਨ ਕੋਣ | p1→p2 ਦੇ ਸਮਾਨਾਂਤਰ — ਕੋਈ ਵੀ ਕੋਣ | ਹਮੇਸ਼ਾ ਹਰੀਜ਼ੌਂਟਲ ਜਾਂ ਵਰਟੀਕਲ |
| ਮਾਪਦਾ ਹੈ | ਅਸਲ ਯੂਕਲਿਡੀਅਨ ਦੂਰੀ | ਸਿਰਫ਼ X ਜਾਂ Y ਹਿੱਸਾ |
| H/V ਓਰੀਐਂਟੇਸ਼ਨ ਲਾਕ | ਨਹੀਂ | ਹਾਂ — `H` ਅਤੇ `V` ਕੁੰਜੀਆਂ |
| ਇਸ ਲਈ ਵਧੀਆ | ਡਾਇਗਨਲ ਵਿਸ਼ੇਸ਼ਤਾਵਾਂ, ਕੋਣੀ ਕੱਟ | ਆਰਥੋਗਨਲ ਲੇਆਉਟ, ਗ੍ਰਿੱਡ-ਅਲਾਈਨਡ ਪਾਰਟਸ |

## ਲੇਬਲ ਦਾ ਸੰਪਾਦਨ — ਸਿੰਪਲ ਮੋਡ

ਇੱਕ ਰੱਖੇ ਗਏ ਅਲਾਈਨਡ ਡਾਇਮੈਂਸ਼ਨ 'ਤੇ **ਡਬਲ-ਕਲਿੱਕ** ਕਰਨ ਨਾਲ ਟੈਕਸਟ ਐਡੀਟਰ **ਸਿੰਪਲ** ਮੋਡ ਵਿੱਚ ਖੁੱਲ੍ਹਦਾ ਹੈ। ਐਡੀਟਰ ਮੌਜੂਦਾ ਰੈਂਡਰ ਕੀਤੇ ਮੁੱਲ ਨਾਲ ਪਹਿਲਾਂ ਤੋਂ ਭਰਿਆ ਹੁੰਦਾ ਹੈ ਤਾਂ ਜੋ ਤੁਸੀਂ ਕਰਸਰ ਨੂੰ ਸਥਿਤ ਕਰਕੇ ਸਿੱਧਾ ਸੰਪਾਦਿਤ ਕਰ ਸਕੋ।

| ਵਿਸ਼ੇਸ਼ਤਾ | ਵਿਵਹਾਰ |
|---------|-----------|
| Bold / Italic / Font / Height | **ਪੂਰੇ** ਲੇਬਲ 'ਤੇ ਇੱਕਸਾਰ ਲਾਗੂ ਹੁੰਦੇ ਹਨ |
| ਪ੍ਰਤੀ-ਅੱਖਰ ਫਾਰਮੈਟਿੰਗ | ਸਮਰਥਿਤ ਨਹੀਂ |
| `Enter` | ਮੁੱਲ ਕਮਿੱਟ ਕਰਦਾ ਹੈ ਅਤੇ ਐਡੀਟਰ ਬੰਦ ਕਰਦਾ ਹੈ |
| ਮਲਟੀ-ਲਾਈਨ | ਸਮਰਥਿਤ ਨਹੀਂ |

ਪੂਰੇ ਰੈਫਰੈਂਸ ਲਈ [Text Editor — simple mode](../../interface/text-editor/#simple-mode) ਵੇਖੋ।

## ਡਾਇਮੈਂਸ਼ਨਾਂ ਦੀ ਚੇਨਿੰਗ

ਇਸ ਦੀ ਦੂਜੀ ਐਕਸਟੈਂਸ਼ਨ ਲਾਈਨ ਤੋਂ ਜਾਰੀ ਹੋਰ ਡਾਇਮੈਂਸ਼ਨ ਜੋੜਨ ਲਈ, [Dimension Continue](../dim-continue/) ਵਰਤੋ — ਇਹ ਇਸ ਅਲਾਈਨਡ ਡਾਇਮੈਂਸ਼ਨ ਦੇ ਬਰਾਬਰ ਹੀ ਮਾਪ ਕੋਣ 'ਤੇ ਲਾਕ ਹੁੰਦੀ ਹੈ।

## DXF — DIMENSION ਐਂਟਿਟੀ (ਅਲਾਈਨਡ ਕਿਸਮ)

ਅਲਾਈਨਡ ਡਾਇਮੈਂਸ਼ਨ `DIMENSION` ਐਂਟਿਟੀਆਂ ਦੇ ਰੂਪ ਵਿੱਚ ਸੰਭਾਲੇ ਜਾਂਦੇ ਹਨ ਜਿਹਨਾਂ ਦਾ `dimType = 1` (ਅਲਾਈਨਡ) ਹੁੰਦਾ ਹੈ। ਐਕਸਟੈਂਸ਼ਨ ਲਾਈਨ ਮੂਲ, ਡਾਇਮੈਂਸ਼ਨ ਲਾਈਨ ਸਥਿਤੀ, ਟੈਕਸਟ ਸਥਿਤੀ, ਮਾਪਿਆ ਗਿਆ ਮੁੱਲ, ਰੋਟੇਸ਼ਨ, ਤੀਰ ਸ਼ੈਲੀ, ਅਤੇ ਸਾਰੇ ਡਿਸਪਲੇ ਫਲੈਗ ਬਿਨਾਂ ਕਿਸੇ ਨੁਕਸਾਨ ਦੇ ਰਾਊਂਡ-ਟ੍ਰਿਪ ਕਰਦੇ ਹਨ।
