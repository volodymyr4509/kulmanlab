# Dimension Diameter

`dimdiameter` ਕਮਾਂਡ ਇੱਕ ਆਰਕ ਜਾਂ ਸਰਕਲ 'ਤੇ ਡਾਇਆਮੀਟਰ ਡਾਇਮੈਂਸ਼ਨ ਰੱਖਦੀ ਹੈ। ਡਾਇਮੈਂਸ਼ਨ ਲਾਈਨ ਪੂਰੇ ਡਾਇਆਮੀਟਰ ਨੂੰ ਫੈਲਾਉਂਦੀ ਹੈ — ਦੋ ਡਾਇਆਮੈਟ੍ਰਿਕਲੀ ਵਿਰੋਧੀ ਆਰਕ ਬਿੰਦੂਆਂ ਵਿਚਕਾਰ ਕੇਂਦਰ ਵਿੱਚੋਂ ਲੰਘਦੀ ਹੋਈ — ਅਤੇ `⌀ <value>` ਲੇਬਲ ਕੀਤੀ ਜਾਂਦੀ ਹੈ। ਕੇਂਦਰ ਤੋਂ ਸਿਰਫ਼ ਇੱਕ ਕਿਨਾਰੇ ਤੱਕ ਰੇਡੀਅਸ ਦਰਸਾਉਣ ਲਈ [Dimension Radius](../dim-radius/) ਵਰਤੋ।

## ਇੱਕ ਡਾਇਆਮੀਟਰ ਡਾਇਮੈਂਸ਼ਨ ਦੀ ਸੰਰਚਨਾ

```
  ●──────────── ⌀ 10.00 ────────────●
  (far arc point)              (near arc point / text side)
```

- **ਡਾਇਮੈਂਸ਼ਨ ਲਾਈਨ** — ਪੂਰੇ ਡਾਇਆਮੀਟਰ ਨੂੰ ਫੈਲਾਉਂਦੀ ਹੈ, ਦੋਵੇਂ ਆਰਕ-ਇੰਟਰਸੈਕਸ਼ਨ ਬਿੰਦੂਆਂ 'ਤੇ ਤੀਰਾਂ ਨਾਲ।
- **ਨੇੜਲਾ ਆਰਕ ਬਿੰਦੂ** — ਕਰਸਰ ਦੇ ਪਾਸੇ ਵਾਲਾ ਪਰੀਧੀ ਬਿੰਦੂ (ਜਿੱਥੇ ਟੈਕਸਟ ਲੇਬਲ ਬੈਠਦਾ ਹੈ)।
- **ਦੂਰ ਦਾ ਆਰਕ ਬਿੰਦੂ** — ਡਾਇਆਮੈਟ੍ਰਿਕਲੀ ਵਿਰੋਧੀ ਬਿੰਦੂ।
- **ਲੇਬਲ** — `⌀` ਦੇ ਬਾਅਦ ਡਾਇਆਮੀਟਰ ਮੁੱਲ।

## ਇੱਕ ਡਾਇਆਮੀਟਰ ਡਾਇਮੈਂਸ਼ਨ ਰੱਖਣਾ

1. ਟਰਮੀਨਲ ਵਿੱਚ `dimdiameter` ਟਾਈਪ ਕਰੋ ਜਾਂ **Dimension Diameter** ਟੂਲਬਾਰ ਬਟਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ।
2. ਇਸਨੂੰ ਚੁਣਨ ਲਈ **ਇੱਕ ਆਰਕ ਜਾਂ ਸਰਕਲ 'ਤੇ ਕਲਿੱਕ ਕਰੋ**।
3. ਡਾਇਮੈਂਸ਼ਨ ਲਾਈਨ ਨੂੰ ਲੋੜੀਂਦੇ ਕੋਣ 'ਤੇ ਘੁਮਾਉਣ ਲਈ **ਕਰਸਰ ਹਿਲਾਓ**।
4. ਡਾਇਮੈਂਸ਼ਨ ਰੱਖਣ ਲਈ **ਕਲਿੱਕ ਕਰੋ**।

ਸਿਰਫ਼ **Arc** ਅਤੇ **Circle** ਇਕਾਈਆਂ ਚੁਣੀਆਂ ਜਾ ਸਕਦੀਆਂ ਹਨ।

## Dimension Diameter ਬਨਾਮ Dimension Radius

| | Dimension Diameter | Dimension Radius |
|---|-------------------|-----------------|
| ਮਾਪਦਾ ਹੈ | ਪੂਰਾ ਡਾਇਆਮੀਟਰ (2 × ਰੇਡੀਅਸ) | ਰੇਡੀਅਸ (ਕੇਂਦਰ ਤੋਂ ਕਿਨਾਰੇ ਤੱਕ) |
| ਡਾਇਮੈਂਸ਼ਨ ਲਾਈਨ | ਕਿਨਾਰਾ → ਕਿਨਾਰਾ, ਕੇਂਦਰ ਵਿੱਚੋਂ | ਕੇਂਦਰ → ਕਿਨਾਰਾ |
| ਲੇਬਲ ਪ੍ਰੀਫਿਕਸ | `⌀` | `R` |
| ਤੀਰ | ਦੋ (ਦੋਵੇਂ ਆਰਕ ਬਿੰਦੂਆਂ 'ਤੇ) | ਇੱਕ (ਆਰਕ ਬਿੰਦੂ 'ਤੇ) |
| ਇਸ ਲਈ ਵਧੀਆ | ਪੂਰੇ ਸਰਕੂਲਰ ਹੋਲ ਜਾਂ ਸ਼ਾਫਟ ਡਾਇਮੈਂਸ਼ਨ | ਇੱਕ ਕਰਵਡ ਵਿਸ਼ੇਸ਼ਤਾ ਦੇ ਇੱਕ ਪਾਸੇ ਨੂੰ ਦਰਸਾਉਣਾ |

## ਲੇਬਲ ਦਾ ਸੰਪਾਦਨ — ਸਿੰਪਲ ਮੋਡ

ਇੱਕ ਰੱਖੇ ਗਏ ਡਾਇਆਮੀਟਰ ਡਾਇਮੈਂਸ਼ਨ 'ਤੇ **ਡਬਲ-ਕਲਿੱਕ** ਕਰਨ ਨਾਲ ਟੈਕਸਟ ਐਡੀਟਰ **ਸਿੰਪਲ** ਮੋਡ ਵਿੱਚ ਖੁੱਲ੍ਹਦਾ ਹੈ। ਐਡੀਟਰ ਮੌਜੂਦਾ ਰੈਂਡਰ ਕੀਤੇ ਮੁੱਲ (ਜਿਵੇਂ `⌀ 10.00`) ਨਾਲ ਪਹਿਲਾਂ ਤੋਂ ਭਰਿਆ ਹੁੰਦਾ ਹੈ ਤਾਂ ਜੋ ਤੁਸੀਂ ਕਰਸਰ ਨੂੰ ਸਥਿਤ ਕਰਕੇ ਸਿੱਧਾ ਸੰਪਾਦਿਤ ਕਰ ਸਕੋ।

| ਵਿਸ਼ੇਸ਼ਤਾ | ਵਿਵਹਾਰ |
|---------|-----------|
| Bold / Italic / Font / Height | **ਪੂਰੇ** ਲੇਬਲ 'ਤੇ ਇੱਕਸਾਰ ਲਾਗੂ ਹੁੰਦੇ ਹਨ |
| ਪ੍ਰਤੀ-ਅੱਖਰ ਫਾਰਮੈਟਿੰਗ | ਸਮਰਥਿਤ ਨਹੀਂ |
| `Enter` | ਮੁੱਲ ਕਮਿੱਟ ਕਰਦਾ ਹੈ ਅਤੇ ਐਡੀਟਰ ਬੰਦ ਕਰਦਾ ਹੈ |
| ਮਲਟੀ-ਲਾਈਨ | ਸਮਰਥਿਤ ਨਹੀਂ |

ਪੂਰੇ ਰੈਫਰੈਂਸ ਲਈ [Text Editor — simple mode](../../interface/text-editor/#simple-mode) ਵੇਖੋ।

## DXF — DIMENSION ਡਾਇਆਮੀਟਰ ਐਂਟਿਟੀ

ਡਾਇਆਮੀਟਰ ਡਾਇਮੈਂਸ਼ਨ `DIMENSION` ਐਂਟਿਟੀਆਂ ਦੇ ਰੂਪ ਵਿੱਚ ਸੰਭਾਲੇ ਜਾਂਦੇ ਹਨ ਜਿਹਨਾਂ ਵਿੱਚ ਡਾਇਆਮੀਟਰ-ਕਿਸਮ ਦੀ ਜਿਓਮੈਟਰੀ ਹੁੰਦੀ ਹੈ, ਜੋ ਦੋਵੇਂ ਆਰਕ-ਬਿੰਦੂ ਸਥਿਤੀਆਂ ਅਤੇ ਮਾਪਿਆ ਗਿਆ ਡਾਇਆਮੀਟਰ ਮੁੱਲ (2 × ਰੇਡੀਅਸ) ਸੰਭਾਲਦੀ ਹੈ। ਸਾਰੀਆਂ ਵਿਸ਼ੇਸ਼ਤਾਵਾਂ ਬਿਨਾਂ ਕਿਸੇ ਨੁਕਸਾਨ ਦੇ ਰਾਊਂਡ-ਟ੍ਰਿਪ ਕਰਦੀਆਂ ਹਨ।
