# LayerMatch

`LayerMatch` ਕਮਾਂਡ ਚੁਣੀਆਂ ਗਈਆਂ ਇਕਾਈਆਂ ਦੀ ਲੇਅਰ ਨੂੰ ਉਸ ਸਰੋਤ ਇਕਾਈ ਦੀ ਲੇਅਰ ਨਾਲ ਮੇਲ ਕਰਨ ਲਈ ਮੁੜ-ਨਿਰਧਾਰਿਤ ਕਰਦੀ ਹੈ ਜਿਸ ਉੱਤੇ ਤੁਸੀਂ ਕਲਿੱਕ ਕਰਦੇ ਹੋ। ਇਹ Layer Manager ਖੋਲ੍ਹੇ ਬਿਨਾਂ ਇਕਾਈਆਂ ਦੇ ਸਮੂਹ ਨੂੰ ਸਹੀ ਲੇਅਰ 'ਤੇ ਲਿਜਾਣ ਦਾ ਸਭ ਤੋਂ ਤੇਜ਼ ਤਰੀਕਾ ਹੈ।

## ਵਰਕਫ਼ਲੋ

**ਪਹਿਲਾਂ ਚੁਣੋ, ਫਿਰ ਮੇਲ ਕਰੋ**:

1. ਉਹ ਇਕਾਈਆਂ ਚੁਣੋ ਜਿਨ੍ਹਾਂ ਦੀ ਲੇਅਰ ਤੁਸੀਂ ਬਦਲਣਾ ਚਾਹੁੰਦੇ ਹੋ।
2. `LayerMatch` ਟਾਈਪ ਕਰੋ ਜਾਂ **Layer Match** ਟੂਲਬਾਰ ਬਟਨ (ਪੇਂਟ ਆਈਕਨ) 'ਤੇ ਕਲਿੱਕ ਕਰੋ।
3. **ਸਰੋਤ ਇਕਾਈ 'ਤੇ ਕਲਿੱਕ ਕਰੋ** — ਉਹ ਜਿਸਦੀ ਲੇਅਰ ਤੁਸੀਂ ਕਾਪੀ ਕਰਨਾ ਚਾਹੁੰਦੇ ਹੋ।
4. ਸਾਰੀਆਂ ਚੁਣੀਆਂ ਗਈਆਂ ਇਕਾਈਆਂ ਤੁਰੰਤ ਸਰੋਤ ਇਕਾਈ ਦੀ ਲੇਅਰ 'ਤੇ ਚਲੀਆਂ ਜਾਂਦੀਆਂ ਹਨ।

**ਸਰਗਰਮ ਕਰੋ, ਫਿਰ ਚੁਣੋ**:

1. ਕੁਝ ਵੀ ਚੁਣੇ ਬਿਨਾਂ `LayerMatch` ਟਾਈਪ ਕਰੋ ਜਾਂ ਟੂਲਬਾਰ ਬਟਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ।
2. **ਟਾਰਗੇਟ ਇਕਾਈਆਂ ਚੁਣੋ** — ਵਿਅਕਤੀਗਤ ਇਕਾਈਆਂ ਟੌਗਲ ਕਰਨ ਲਈ ਕਲਿੱਕ ਕਰੋ ਜਾਂ ਖੇਤਰ ਦੁਆਰਾ ਚੁਣਨ ਲਈ ਖਿੱਚੋ।
3. ਚੋਣ ਦੀ ਪੁਸ਼ਟੀ ਕਰਨ ਲਈ **Enter** ਜਾਂ **Space** ਦਬਾਓ।
4. **ਸਰੋਤ ਇਕਾਈ 'ਤੇ ਕਲਿੱਕ ਕਰੋ** — ਇਸਦੀ ਲੇਅਰ ਸਾਰੇ ਟਾਰਗੇਟਾਂ 'ਤੇ ਲਾਗੂ ਹੁੰਦੀ ਹੈ।

## ਕੀਬੋਰਡ ਰੈਫਰੈਂਸ

| ਕੀ | ਕਾਰਵਾਈ |
|-----|--------|
| `Enter` / `Space` | ਟਾਰਗੇਟ ਚੋਣ ਦੀ ਪੁਸ਼ਟੀ ਕਰੋ ਅਤੇ ਸਰੋਤ-ਚੋਣ ਪੜਾਅ 'ਤੇ ਜਾਓ |
| `Escape` | ਰੀਸੈੱਟ ਕਰੋ — ਟਾਰਗੇਟ ਚੋਣ 'ਤੇ ਵਾਪਸ ਜਾਓ ਜਾਂ ਪੂਰੀ ਤਰ੍ਹਾਂ ਰੱਦ ਕਰੋ |

## ਵਿਵਹਾਰ ਵੇਰਵੇ

- ਸਿਰਫ਼ `layer` ਵਿਸ਼ੇਸ਼ਤਾ ਬਦਲਦੀ ਹੈ — ਰੰਗ, ਲਾਈਨਟਾਈਪ, ਲਾਈਨਵੇਟ, ਅਤੇ ਜਿਓਮੈਟਰੀ ਅਛੂਤੀ ਰਹਿੰਦੀ ਹੈ।
- ਸਰੋਤ ਇਕਾਈ ਖੁਦ ਸੋਧੀ ਨਹੀਂ ਜਾਂਦੀ।
- ਸਰੋਤ ਕਲਿੱਕ ਕਰਨ ਤੋਂ ਬਾਅਦ ਕਮਾਂਡ ਸਮਾਪਤ ਹੋ ਜਾਂਦੀ ਹੈ।
- ਸਰੋਤ-ਚੋਣ ਪੜਾਅ ਦੌਰਾਨ ਖਾਲੀ ਕੈਨਵਸ 'ਤੇ ਕਲਿੱਕ ਕਰਨ ਨਾਲ ਕੁਝ ਨਹੀਂ ਹੁੰਦਾ।
