# Rotate

`rotate` ਕਮਾਂਡ ਚੁਣੀਆਂ ਐਂਟਿਟੀਆਂ ਨੂੰ ਇੱਕ ਬੇਸ ਪੁਆਇੰਟ ਦੁਆਲੇ ਘੁੰਮਾਉਂਦੀ ਹੈ। ਤੁਸੀਂ ਘੁੰਮਾਓ ਕੋਣ ਜਾਂ ਤਾਂ ਡਿਗਰੀਆਂ ਵਿੱਚ ਇੱਕ ਸੰਖਿਆ ਟਾਈਪ ਕਰਕੇ ਜਾਂ ਕਲਿੱਕ ਕਰਕੇ ਦੱਸਦੇ ਹੋ — ਕੋਣ ਬੇਸ ਪੁਆਇੰਟ ਅਤੇ ਕਲਿੱਕ ਸਥਿਤੀ ਵਿਚਕਾਰਲੀ ਦਿਸ਼ਾ ਤੋਂ ਗਿਣਿਆ ਜਾਂਦਾ ਹੈ।

## ਸ਼ੁਰੂ ਕਰਨ ਦੇ ਦੋ ਤਰੀਕੇ

**ਪਹਿਲਾਂ ਚੁਣੋ, ਫਿਰ ਘੁੰਮਾਓ** — ਪਹਿਲਾਂ ਐਂਟਿਟੀਆਂ ਚੁਣੋ, ਫਿਰ ਸਰਗਰਮ ਕਰੋ:

1. ਕੈਨਵਸ 'ਤੇ ਇੱਕ ਜਾਂ ਵੱਧ ਐਂਟਿਟੀਆਂ ਚੁਣੋ।
2. ਟਰਮੀਨਲ ਵਿੱਚ `rotate` ਟਾਈਪ ਕਰੋ ਜਾਂ **Rotate** ਟੂਲਬਾਰ ਬਟਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ।
3. **ਬੇਸ ਪੁਆਇੰਟ 'ਤੇ ਕਲਿੱਕ ਕਰੋ** — ਘੁੰਮਾਓ ਦਾ ਕੇਂਦਰ। ਜਾਂ ਸਹੀ ਕੋਆਰਡੀਨੇਟ ਲਈ `X,Y` ਟਾਈਪ ਕਰੋ ਅਤੇ **Enter** ਦਬਾਓ।
4. **ਇੱਕ ਕੋਣ ਟਾਈਪ ਕਰੋ ਅਤੇ Enter ਦਬਾਓ**, ਜਾਂ ਕਰਸਰ ਦਿਸ਼ਾ ਤੋਂ ਕੋਣ ਸੈੱਟ ਕਰਨ ਲਈ **ਕਲਿੱਕ ਕਰੋ**।

**ਸਰਗਰਮ ਕਰੋ, ਫਿਰ ਚੁਣੋ** — ਕੁਝ ਵੀ ਚੁਣੇ ਬਿਨਾਂ ਕਮਾਂਡ ਸ਼ੁਰੂ ਕਰੋ:

1. `rotate` ਟਾਈਪ ਕਰੋ ਜਾਂ ਟੂਲਬਾਰ ਬਟਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ।
2. **ਆਬਜੈਕਟ ਚੁਣੋ** — ਟੌਗਲ ਕਰਨ ਲਈ ਕਲਿੱਕ ਕਰੋ, ਜਾਂ ਖੇਤਰ ਦੁਆਰਾ ਚੁਣਨ ਲਈ ਖਿੱਚੋ।
3. ਚੋਣ ਪੁਸ਼ਟੀ ਕਰਨ ਲਈ **Enter** ਜਾਂ **Space** ਦਬਾਓ।
4. **ਬੇਸ ਪੁਆਇੰਟ 'ਤੇ ਕਲਿੱਕ ਕਰੋ** (ਕੋਆਰਡੀਨੇਟ ਐਂਟਰੀ ਉਪਲਬਧ), ਫਿਰ ਕੋਣ ਸੈੱਟ ਕਰੋ।

```
  Before:            After (90° rotation around ●):
                        ╔══╗
  ●  [entity]    →   ● ║    ║
                        ╚══╝
```

ਬੇਸ ਪੁਆਇੰਟ ਸੈੱਟ ਹੋਣ ਤੋਂ ਬਾਅਦ ਘੁੰਮਾਈਆਂ ਐਂਟਿਟੀਆਂ ਦਾ ਇੱਕ ਲਾਈਵ ਭੂਤ ਪ੍ਰੀਵਿਊ ਕਰਸਰ ਕੋਣ ਦਾ ਪਿੱਛਾ ਕਰਦਾ ਹੈ।

## ਕੋਣ ਸੈੱਟ ਕਰਨਾ

**ਟਾਈਪ ਕੀਤਾ ਕੋਣ** — ਬੇਸ ਪੁਆਇੰਟ ਰੱਖੇ ਜਾਣ ਤੋਂ ਬਾਅਦ ਕਿਸੇ ਵੀ ਸਮੇਂ ਇੱਕ ਸੰਖਿਆ (ਡਿਗਰੀਆਂ ਵਿੱਚ) ਟਾਈਪ ਕਰੋ। Enter ਦਬਾਉਣ ਤੋਂ ਪਹਿਲਾਂ ਤੁਹਾਡੇ ਵੱਲੋਂ ਸਮਾਯੋਜਿਤ ਕਰਦੇ ਰਹਿਣ ਦੌਰਾਨ ਪ੍ਰੀਵਿਊ ਟਾਈਪ ਕੀਤੇ ਕੋਣ 'ਤੇ ਸਨੈਪ ਹੁੰਦਾ ਹੈ।

**ਕਲਿੱਕ ਕੋਣ** — ਜੇ ਕੋਈ ਟਾਈਪ ਕੀਤਾ ਮੁੱਲ ਮੌਜੂਦ ਨਹੀਂ, ਕਲਿੱਕ ਕਰਨ ਨਾਲ ਕੋਣ `atan2(cursorY − baseY, cursorX − baseX)` ਦੇ ਬਰਾਬਰ ਸੈੱਟ ਹੋ ਜਾਂਦਾ ਹੈ — ਬੇਸ ਪੁਆਇੰਟ ਤੋਂ ਕਲਿੱਕ ਤੱਕ ਦੀ ਦਿਸ਼ਾ, ਡਿਗਰੀਆਂ ਵਿੱਚ।

| ਕੀ | ਕਾਰਵਾਈ |
|-----|--------|
| `0`–`9`, `.` | ਕੋਣ ਮੁੱਲ ਵਿੱਚ ਅੰਕ ਜੋੜੋ |
| `-` | ਨਕਾਰਾਤਮਕ ਕੋਣ (ਸਿਰਫ਼ ਪਹਿਲਾ ਅੱਖਰ) |
| `Backspace` | ਆਖਰੀ ਟਾਈਪ ਕੀਤਾ ਅੱਖਰ ਮਿਟਾਓ |
| `Enter` | ਟਾਈਪ ਕੀਤੇ ਕੋਣ 'ਤੇ ਘੁੰਮਾਓ ਲਾਗੂ ਕਰੋ |

## ਕੋਣ ਦਿਸ਼ਾ

ਕੋਣ **DXF ਕਨਵੈਨਸ਼ਨ** ਦੀ ਪਾਲਣਾ ਕਰਦੇ ਹਨ:

- **ਸਕਾਰਾਤਮਕ** ਮੁੱਲ ਡਰਾਇੰਗ ਕੋਆਰਡੀਨੇਟਾਂ (Y-ਉੱਪਰ) ਵਿੱਚ **ਘੜੀ ਦੀ ਉਲਟ ਦਿਸ਼ਾ** ਵਿੱਚ ਘੁੰਮਦੇ ਹਨ।
- ਸਕ੍ਰੀਨ 'ਤੇ, ਜਿੱਥੇ Y-ਧੁਰਾ ਉਲਟਿਆ ਹੋਇਆ ਹੁੰਦਾ ਹੈ (Y-ਹੇਠਾਂ), ਸਕਾਰਾਤਮਕ ਕੋਣ **ਘੜੀ ਦੀ ਦਿਸ਼ਾ** ਵਿੱਚ ਦਿਖਾਈ ਦਿੰਦੇ ਹਨ।

ਆਮ ਮੁੱਲ: `90` = ਚੌਥਾਈ ਮੋੜ, `180` = ਅੱਧਾ ਮੋੜ, `-90` = ਉਲਟ ਚੌਥਾਈ ਮੋੜ।

| ਕੀ | ਕਾਰਵਾਈ |
|-----|--------|
| `Enter` / `Space` | ਚੋਣ ਪੁਸ਼ਟੀ ਕਰੋ |
| `0`–`9`, `.`, `-` | X ਕੋਆਰਡੀਨੇਟ ਐਂਟਰੀ ਸ਼ੁਰੂ ਕਰੋ (ਬੇਸ-ਪੁਆਇੰਟ ਪੜਾਅ), ਜਾਂ ਕੋਣ ਮੁੱਲ (ਕੋਣ ਪੜਾਅ) |
| `,` | X ਲਾਕ ਕਰੋ ਅਤੇ Y ਐਂਟਰੀ 'ਤੇ ਜਾਓ (ਬੇਸ-ਪੁਆਇੰਟ ਪੜਾਅ) |
| `Backspace` | ਆਖਰੀ ਟਾਈਪ ਕੀਤਾ ਅੱਖਰ ਮਿਟਾਓ |
| `Enter` | ਕੋਆਰਡੀਨੇਟ ਪੁਸ਼ਟੀ ਕਰੋ ਜਾਂ ਘੁੰਮਾਓ ਲਾਗੂ ਕਰੋ |
| `Escape` | ਰੱਦ ਕਰੋ ਅਤੇ ਰੀਸੈੱਟ ਕਰੋ |

## ਸਮਰਥਿਤ ਐਂਟਿਟੀਆਂ

Rotate ਹਰੇਕ ਐਂਟਿਟੀ ਕਿਸਮ 'ਤੇ ਕੰਮ ਕਰਦੀ ਹੈ। ਹਰੇਕ ਐਂਟਿਟੀ ਦੀ ਜਿਓਮੈਟਰੀ ਬੇਸ ਪੁਆਇੰਟ ਦੁਆਲੇ ਘੁੰਮਾਈ ਜਾਂਦੀ ਹੈ — ਉਦਾਹਰਨ ਵਜੋਂ, ਇੱਕ Circle ਆਪਣਾ ਕੇਂਦਰ ਹਿਲਾਉਂਦਾ ਹੈ ਜਦਕਿ ਰੇਡੀਅਸ ਉਹੀ ਰਹਿੰਦਾ ਹੈ; ਇੱਕ Arc ਆਪਣਾ ਕੇਂਦਰ ਹਿਲਾਉਂਦਾ ਹੈ ਅਤੇ ਆਪਣੇ ਸ਼ੁਰੂ ਅਤੇ ਅੰਤ ਕੋਣਾਂ ਨੂੰ ਘੁੰਮਾਓ ਦੀ ਮਾਤਰਾ ਨਾਲ ਸ਼ਿਫਟ ਕਰਦਾ ਹੈ; ਇੱਕ Text ਐਂਟਿਟੀ ਆਪਣਾ ਐਂਕਰ ਪੁਆਇੰਟ ਹਿਲਾਉਂਦੀ ਹੈ ਅਤੇ ਆਪਣੀ Rotation Degree ਵਿਸ਼ੇਸ਼ਤਾ ਵਿੱਚ ਕੋਣ ਜੋੜਦੀ ਹੈ।
