# Copy

`copy` ਕਮਾਂਡ ਚੁਣੀਆਂ ਐਂਟਿਟੀਆਂ ਦੇ ਸਥਾਨਾਂਤਰਿਤ ਡੁਪਲੀਕੇਟ ਬਣਾਉਂਦੀ ਹੈ ਅਤੇ ਉਹਨਾਂ ਨੂੰ ਇੱਕ ਬੇਸ ਪੁਆਇੰਟ ਤੋਂ ਇੱਕ ਮੰਜ਼ਿਲ ਤੱਕ ਦੇ ਆਫਸੈੱਟ 'ਤੇ ਰੱਖਦੀ ਹੈ — ਮੂਲ ਬਿਲਕੁਲ ਉੱਥੇ ਹੀ ਰਹਿੰਦੇ ਹਨ ਜਿੱਥੇ ਹਨ। ਇਹ [Move](./move) ਤੋਂ ਇੱਕ ਮੁੱਖ ਅੰਤਰ ਹੈ: Copy ਡਰਾਇੰਗ ਵਿੱਚ ਨਵੀਆਂ ਐਂਟਿਟੀਆਂ ਜੋੜਦੀ ਹੈ; Move ਮੌਜੂਦਾ ਐਂਟਿਟੀਆਂ ਨੂੰ ਮੁੜ-ਸਥਾਪਿਤ ਕਰਦੀ ਹੈ।

## ਸ਼ੁਰੂ ਕਰਨ ਦੇ ਦੋ ਤਰੀਕੇ

**ਪਹਿਲਾਂ ਚੁਣੋ, ਫਿਰ ਕਾਪੀ ਕਰੋ** — ਪਹਿਲਾਂ ਐਂਟਿਟੀਆਂ ਚੁਣੋ, ਫਿਰ ਸਰਗਰਮ ਕਰੋ:

1. ਕੈਨਵਸ 'ਤੇ ਇੱਕ ਜਾਂ ਵੱਧ ਐਂਟਿਟੀਆਂ ਚੁਣੋ।
2. ਟਰਮੀਨਲ ਵਿੱਚ `copy` ਟਾਈਪ ਕਰੋ ਜਾਂ **Copy** ਟੂਲਬਾਰ ਬਟਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ।
3. **ਬੇਸ ਪੁਆਇੰਟ 'ਤੇ ਕਲਿੱਕ ਕਰੋ**, ਜਾਂ ਸਹੀ ਕੋਆਰਡੀਨੇਟ ਲਈ `X,Y` ਟਾਈਪ ਕਰੋ ਅਤੇ **Enter** ਦਬਾਓ।
4. **ਮੰਜ਼ਿਲ 'ਤੇ ਕਲਿੱਕ ਕਰੋ** — ਬੇਸ→ਮੰਜ਼ਿਲ ਆਫਸੈੱਟ 'ਤੇ ਡੁਪਲੀਕੇਟ ਦਿਖਾਈ ਦਿੰਦੇ ਹਨ। ਕੋਆਰਡੀਨੇਟ ਐਂਟਰੀ ਇੱਥੇ ਵੀ ਕੰਮ ਕਰਦੀ ਹੈ।

**ਸਰਗਰਮ ਕਰੋ, ਫਿਰ ਚੁਣੋ** — ਕੁਝ ਵੀ ਚੁਣੇ ਬਿਨਾਂ ਕਮਾਂਡ ਸ਼ੁਰੂ ਕਰੋ:

1. `copy` ਟਾਈਪ ਕਰੋ ਜਾਂ ਟੂਲਬਾਰ ਬਟਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ।
2. **ਆਬਜੈਕਟ ਚੁਣੋ** — ਵਿਅਕਤੀਗਤ ਐਂਟਿਟੀਆਂ ਟੌਗਲ ਕਰਨ ਲਈ ਕਲਿੱਕ ਕਰੋ, ਜਾਂ ਖੇਤਰ ਦੁਆਰਾ ਚੁਣਨ ਲਈ ਖਿੱਚੋ।
3. ਚੋਣ ਪੁਸ਼ਟੀ ਕਰਨ ਲਈ **Enter** ਜਾਂ **Space** ਦਬਾਓ।
4. **ਬੇਸ ਪੁਆਇੰਟ 'ਤੇ ਕਲਿੱਕ ਕਰੋ**, ਫਿਰ **ਮੰਜ਼ਿਲ 'ਤੇ ਕਲਿੱਕ ਕਰੋ** (ਦੋਵੇਂ ਪੜਾਵਾਂ 'ਤੇ ਕੋਆਰਡੀਨੇਟ ਐਂਟਰੀ ਉਪਲਬਧ)।

```
  Before:               After:
  [entity A]            [entity A]  ← originals untouched
  [entity B]            [entity B]
                        [copy of A] ← new entities
                        [copy of B]
```

ਕਾਪੀਆਂ ਦਾ ਇੱਕ ਭੂਤ ਪ੍ਰੀਵਿਊ ਬੇਸ ਪੁਆਇੰਟ ਤੋਂ ਮੰਜ਼ਿਲ ਤੱਕ ਕਰਸਰ ਦਾ ਪਿੱਛਾ ਕਰਦਾ ਹੈ।

## ਕੋਆਰਡੀਨੇਟ ਐਂਟਰੀ

ਬੇਸ-ਪੁਆਇੰਟ ਜਾਂ ਮੰਜ਼ਿਲ ਪੜਾਅ 'ਤੇ, ਕਲਿੱਕ ਕਰਨ ਦੀ ਬਜਾਏ ਸਹੀ ਸਥਿਤੀ ਟਾਈਪ ਕਰੋ:

1. X ਮੁੱਲ ਟਾਈਪ ਕਰੋ।
2. `,` ਦਬਾਓ — ਟਰਮੀਨਲ `[X], [Y{cursor}]` ਦਿਖਾਉਂਦਾ ਹੈ।
3. Y ਮੁੱਲ ਟਾਈਪ ਕਰੋ।
4. ਪੁਸ਼ਟੀ ਕਰਨ ਲਈ **Enter** ਦਬਾਓ।

## ਐਂਗਲ ਲਾਕਿੰਗ ਅਤੇ ਸਹੀ ਦੂਰੀ

ਬੇਸ ਪੁਆਇੰਟ ਸੈੱਟ ਹੋਣ ਤੋਂ ਬਾਅਦ ਕਮਾਂਡ 45° ਧੁਰਿਆਂ (0°, 45°, 90°, 135°, …) 'ਤੇ ਸਨੈਪ ਕਰਦੀ ਹੈ ਜਦੋਂ ਕਰਸਰ ਕਾਫ਼ੀ ਦੂਰ ਅਤੇ ਧੁਰੇ ਦੇ ਨੇੜੇ ਹੋਵੇ। ਲਾਕ ਹੋਣ ਦੌਰਾਨ, ਕਾਪੀਆਂ ਨੂੰ ਠੀਕ ਉਸ ਆਫਸੈੱਟ 'ਤੇ ਰੱਖਣ ਲਈ ਇੱਕ ਦੂਰੀ ਟਾਈਪ ਕਰੋ ਅਤੇ **Enter** ਦਬਾਓ।

| ਕੀ | ਕਾਰਵਾਈ |
|-----|--------|
| `0`–`9`, `.` | ਦੂਰੀ ਮੁੱਲ ਵਿੱਚ ਜੋੜੋ |
| `-` | ਨਕਾਰਾਤਮਕ ਦੂਰੀ — ਧੁਰੇ ਦੇ ਨਾਲ ਦਿਸ਼ਾ ਉਲਟਾਉਂਦੀ ਹੈ (ਸਿਰਫ਼ ਪਹਿਲਾ ਅੱਖਰ) |
| `Backspace` | ਆਖਰੀ ਟਾਈਪ ਕੀਤਾ ਅੱਖਰ ਮਿਟਾਓ |
| `Enter` | ਟਾਈਪ ਕੀਤੀ ਦੂਰੀ 'ਤੇ ਕਾਪੀਆਂ ਰੱਖੋ |

## ਕੀਬੋਰਡ ਰੈਫਰੈਂਸ

| ਕੀ | ਕਾਰਵਾਈ |
|-----|--------|
| `Enter` / `Space` | ਚੋਣ ਪੁਸ਼ਟੀ ਕਰੋ ਅਤੇ ਬੇਸ-ਪੁਆਇੰਟ ਪੜਾਅ 'ਤੇ ਅੱਗੇ ਵਧੋ |
| `0`–`9`, `.`, `-` | X ਕੋਆਰਡੀਨੇਟ ਐਂਟਰੀ ਸ਼ੁਰੂ ਕਰੋ, ਜਾਂ ਐਂਗਲ-ਲਾਕ ਹੋਣ 'ਤੇ ਦੂਰੀ |
| `,` | X ਲਾਕ ਕਰੋ ਅਤੇ Y ਐਂਟਰੀ 'ਤੇ ਜਾਓ |
| `Backspace` | ਆਖਰੀ ਟਾਈਪ ਕੀਤਾ ਅੱਖਰ ਮਿਟਾਓ |
| `Enter` | ਕੋਆਰਡੀਨੇਟ ਪੁਸ਼ਟੀ ਕਰੋ ਜਾਂ ਟਾਈਪ ਕੀਤੀ ਦੂਰੀ 'ਤੇ ਕਾਪੀ ਲਾਗੂ ਕਰੋ |
| `Escape` | ਰੱਦ ਕਰੋ ਅਤੇ ਰੀਸੈੱਟ ਕਰੋ |

## ਕਮਾਂਡ ਦੌਰਾਨ ਚੋਣ

| ਢੰਗ | ਵਿਵਹਾਰ |
|--------|-----------|
| **ਕਲਿੱਕ** | ਕਰਸਰ ਹੇਠਾਂ ਐਂਟਿਟੀ ਨੂੰ ਚੋਣ ਵਿੱਚ/ਬਾਹਰ ਟੌਗਲ ਕਰਦਾ ਹੈ |
| **ਸੱਜੇ ਖਿੱਚੋ** (ਸਖ਼ਤ) | ਬਾਕਸ ਦੇ ਪੂਰੀ ਤਰ੍ਹਾਂ ਅੰਦਰ ਵਾਲੀਆਂ ਐਂਟਿਟੀਆਂ ਜੋੜਦਾ ਹੈ |
| **ਖੱਬੇ ਖਿੱਚੋ** (ਕ੍ਰਾਸਿੰਗ) | ਬਾਕਸ ਦੀ ਸੀਮਾ ਨੂੰ ਕੱਟਣ ਵਾਲੀਆਂ ਐਂਟਿਟੀਆਂ ਜੋੜਦਾ ਹੈ |
| **Enter** / **Space** | ਚੋਣ ਪੁਸ਼ਟੀ ਕਰਦਾ ਹੈ |

## ਕਾਪੀ ਤੋਂ ਬਾਅਦ

**ਮੂਲ ਚੁਣੇ ਰਹਿੰਦੇ ਹਨ** — ਨਵੀਆਂ ਕਾਪੀਆਂ ਡਰਾਇੰਗ ਵਿੱਚ ਜੋੜੀਆਂ ਜਾਂਦੀਆਂ ਹਨ ਪਰ ਚੋਣ ਸਾਫ਼ ਹੋ ਜਾਂਦੀ ਹੈ ਅਤੇ ਕਮਾਂਡ ਬੰਦ ਹੋ ਜਾਂਦੀ ਹੈ। ਕਾਪੀਆਂ ਨਾਲ ਤੁਰੰਤ ਕੰਮ ਕਰਨ ਲਈ, ਚੋਣ 'ਤੇ Copy ਦੁਬਾਰਾ ਚਲਾਓ, ਜਾਂ ਇੱਕ ਨਵੀਂ ਕਮਾਂਡ ਸ਼ੁਰੂ ਕਰੋ।

## ਸਮਰਥਿਤ ਐਂਟਿਟੀਆਂ

Copy ਹਰੇਕ ਐਂਟਿਟੀ ਕਿਸਮ 'ਤੇ ਕੰਮ ਕਰਦੀ ਹੈ। ਸਾਰੀਆਂ ਐਂਟਿਟੀਆਂ ਅੰਦਰੂਨੀ ਤੌਰ 'ਤੇ `translate(dx, dy)` ਲਾਗੂ ਕਰਦੀਆਂ ਹਨ ਇਸ ਲਈ ਕੋਈ ਵੀ ਬਾਹਰ ਨਹੀਂ ਹੈ।
