# Font Manager

`FontManager` ਕਮਾਂਡ ਫੌਂਟਾਂ ਨੂੰ ਬ੍ਰਾਊਜ਼ ਕਰਨ ਅਤੇ ਚੁਣਨ ਲਈ, ਅਤੇ [Text](../text/) ਅਤੇ [Multileader](../leader/) ਇਕਾਈਆਂ ਵਿੱਚ ਵਰਤਣ ਲਈ ਆਪਣੀਆਂ `.ttf` ਫਾਈਲਾਂ ਅੱਪਲੋਡ ਕਰਨ ਲਈ ਇੱਕ ਡਾਇਲਾਗ ਖੋਲ੍ਹਦੀ ਹੈ।

## Font Manager ਖੋਲ੍ਹਣਾ

- ਟਰਮੀਨਲ ਵਿੱਚ `FontManager` ਟਾਈਪ ਕਰੋ, **ਜਾਂ**
- [ਟੈਕਸਟ ਐਡੀਟਰ](../../interface/text-editor/) ਟੂਲਬਾਰ ਵਿੱਚ **Font Manager** ਬਟਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ।

## ਫੌਂਟ ਗਰੁੱਪ

| ਗਰੁੱਪ | ਸਮੱਗਰੀ |
|-------|---------|
| **Default** | ਬਿਲਟ-ਇਨ ਸੈਂਸ-ਸੈਰਿਫ ਫੌਂਟ — ਹਮੇਸ਼ਾ ਉਪਲਬਧ |
| **User** | ਤੁਹਾਡੇ ਆਪਣੇ ਅੱਪਲੋਡ ਕੀਤੇ `.ttf` ਫੌਂਟ (ਸਿਰਫ਼ ਇੱਕ ਜੋੜਨ ਤੋਂ ਬਾਅਦ ਦਿਖਾਈ ਦਿੰਦੇ ਹਨ) |
| **Free** | 15 ਬੰਡਲ ਕੀਤੇ Google Fonts (EB Garamond, Fira Code, Inter, Lato, Merriweather, Montserrat, Nunito, Open Sans, Oswald, Playfair Display, Poppins, Raleway, Roboto, Roboto Condensed, Source Code Pro) |
| **System** | ਆਮ OS ਫੌਂਟ (Courier New, Georgia, Helvetica, Impact, Lucida Console, Tahoma, Times New Roman, Trebuchet MS, Verdana) |

ਸੂਚੀ ਵਿੱਚ ਕਿਸੇ ਵੀ ਫੌਂਟ 'ਤੇ ਕਲਿੱਕ ਕਰਕੇ ਇਸਨੂੰ ਸੱਜੇ ਪਾਸੇ ਪ੍ਰੀਵਿਊ ਕਰੋ — ਨਾਮ, ਅਲਫਾਬੈੱਟ ਸੈਂਪਲ, ਇੱਕ pangram, ਅਤੇ ਅੰਕ।

## ਕਸਟਮ ਫੌਂਟ ਅੱਪਲੋਡ ਕਰਨਾ

1. ਡਾਇਲਾਗ ਦੇ ਫੁੱਟਰ ਵਿੱਚ **Add Font** 'ਤੇ ਕਲਿੱਕ ਕਰੋ (ਜਾਂ ਫਾਈਲ ਪਿਕਰ ਨੂੰ ਸਿੱਧਾ ਖੋਲ੍ਹਣ ਲਈ ਟਰਮੀਨਲ ਵਿੱਚ `Font+` ਟਾਈਪ ਕਰੋ)।
2. ਇੱਕ `.ttf` ਫਾਈਲ ਚੁਣੋ। ਸਿਰਫ਼ TrueType ਫੌਂਟ ਸਮਰਥਿਤ ਹਨ — `.otf` ਅਤੇ `.woff`/`.woff2` ਨਹੀਂ।
3. ਫਾਈਲ ਦਾ ਨਾਮ (ਐਕਸਟੈਂਸ਼ਨ ਤੋਂ ਬਿਨਾਂ) **User** ਗਰੁੱਪ ਵਿੱਚ ਫੌਂਟ ਦਾ ਨਾਮ ਬਣ ਜਾਂਦਾ ਹੈ। ਉਦਾਹਰਨ ਲਈ, `MyFont.ttf` ਅੱਪਲੋਡ ਕਰਨ ਨਾਲ `MyFont` ਨਾਮ ਦਾ ਇੱਕ ਫੌਂਟ ਜੁੜ ਜਾਂਦਾ ਹੈ।

ਅੱਪਲੋਡ ਕੀਤੇ ਫੌਂਟ ਬ੍ਰਾਊਜ਼ਰ ਵਿੱਚ (IndexedDB) ਸਥਾਈ ਤੌਰ ਤੇ ਸੇਵ ਹੁੰਦੇ ਹਨ ਅਤੇ ਅਗਲੀ ਵਾਰ ਜਦੋਂ ਤੁਸੀਂ KulmanLab CAD ਖੋਲ੍ਹਦੇ ਹੋ ਤਾਂ ਆਪਣੇ ਆਪ ਦੁਬਾਰਾ ਲੋਡ ਹੁੰਦੇ ਹਨ।

## ਕਸਟਮ ਫੌਂਟ ਹਟਾਉਣਾ

**User** ਗਰੁੱਪ ਵਿੱਚ ਕਿਸੇ ਫੌਂਟ 'ਤੇ ਹੋਵਰ ਕਰੋ ਅਤੇ ਇਸਦੇ ਨਾਲ ਵਾਲੇ **×** ਬਟਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ। ਬਿਲਟ-ਇਨ ਫੌਂਟ (Default, Free, System) ਹਟਾਏ ਨਹੀਂ ਜਾ ਸਕਦੇ।

## ਕੀਬੋਰਡ ਰੈਫਰੈਂਸ

| ਕੀ | ਕਾਰਵਾਈ |
|-----|--------|
| `↑` / `↓` | ਫੌਂਟ ਸੂਚੀ ਵਿੱਚ ਚੋਣ ਨੂੰ ਉੱਪਰ ਜਾਂ ਹੇਠਾਂ ਲਿਜਾਓ |
| `Escape` | Font Manager ਬੰਦ ਕਰੋ |

## DXF ਅਨੁਕੂਲਤਾ

ਫੌਂਟ ਦਾ ਨਾਮ ਐਕਸਪੋਰਟ ਕੀਤੀਆਂ **MTEXT** ਇਕਾਈਆਂ ਵਿੱਚ ਇੱਕ ਇਨਲਾਈਨ ਫਾਰਮੈਟਿੰਗ ਕੋਡ ਵਜੋਂ ਸ਼ਾਮਲ ਹੁੰਦਾ ਹੈ, ਇਸ ਲਈ KulmanLab CAD ਰਾਹੀਂ ਰਾਊਂਡ-ਟ੍ਰਿਪ ਕੀਤੀ ਇੱਕ DXF ਆਪਣੀ ਫੌਂਟ ਅਸਾਈਨਮੈਂਟ ਬਰਕਰਾਰ ਰੱਖਦੀ ਹੈ। ਕਸਟਮ ਫੌਂਟ *ਫਾਈਲਾਂ* DXF ਵਿੱਚ ਸ਼ਾਮਲ ਨਹੀਂ ਹੁੰਦੀਆਂ — ਸਿਰਫ਼ ਫੌਂਟ ਦਾ *ਨਾਮ* ਸ਼ਾਮਲ ਹੁੰਦਾ ਹੈ। ਜੇ ਤੁਸੀਂ ਅਜਿਹੀ ਡਰਾਇੰਗ ਦੁਬਾਰਾ ਇੰਪੋਰਟ ਕਰਦੇ ਹੋ ਜੋ ਇੱਕ ਕਸਟਮ ਫੌਂਟ ਦਾ ਹਵਾਲਾ ਦਿੰਦੀ ਹੈ ਜੋ ਤੁਸੀਂ ਇਸ ਡਿਵਾਈਸ 'ਤੇ ਅੱਪਲੋਡ ਨਹੀਂ ਕੀਤਾ, ਤਾਂ ਟੈਕਸਟ ਡਿਫਾਲਟ ਫੌਂਟ ਵਿੱਚ ਦਿਖਾਈ ਦਿੰਦਾ ਹੈ ਜਦੋਂ ਤੱਕ ਤੁਸੀਂ ਉਸੇ ਨਾਮ ਵਾਲਾ ਫੌਂਟ ਅੱਪਲੋਡ ਨਹੀਂ ਕਰਦੇ।
