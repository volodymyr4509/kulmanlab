# Dimension Radius

`dimradius` ਕਮਾਂਡ ਇੱਕ ਆਰਕ ਜਾਂ ਸਰਕਲ 'ਤੇ ਰੇਡੀਅਸ ਡਾਇਮੈਂਸ਼ਨ ਰੱਖਦੀ ਹੈ। ਡਾਇਮੈਂਸ਼ਨ ਲਾਈਨ ਕੇਂਦਰ ਤੋਂ ਕਰਸਰ ਦੀ ਦਿਸ਼ਾ ਵਿੱਚ ਪਰੀਧੀ ਦੇ ਇੱਕ ਬਿੰਦੂ ਤੱਕ ਚੱਲਦੀ ਹੈ, ਜਿਸਨੂੰ `R <value>` ਲੇਬਲ ਕੀਤਾ ਜਾਂਦਾ ਹੈ। ਇਸਦੀ ਬਜਾਏ ਪੂਰਾ ਡਾਇਆਮੀਟਰ ਦਰਸਾਉਣ ਲਈ [Dimension Diameter](../dim-diameter/) ਵਰਤੋ।

## ਇੱਕ ਰੇਡੀਅਸ ਡਾਇਮੈਂਸ਼ਨ ਦੀ ਸੰਰਚਨਾ

```
  ● (center)
   \
    \  R 5.00
     \
      ●────── text (cursor side)
   (arc point)
```

- **ਡਾਇਮੈਂਸ਼ਨ ਲਾਈਨ** — ਕੇਂਦਰ ਤੋਂ ਇੱਕ ਆਰਕ ਬਿੰਦੂ ਵਿੱਚੋਂ ਕਰਸਰ ਵੱਲ, ਆਰਕ 'ਤੇ ਇੱਕ ਤੀਰ ਦੇ ਨਾਲ।
- **ਲੇਬਲ** — `R` ਦੇ ਬਾਅਦ ਰੇਡੀਅਸ ਮੁੱਲ।

## ਇੱਕ ਰੇਡੀਅਸ ਡਾਇਮੈਂਸ਼ਨ ਰੱਖਣਾ

1. ਟਰਮੀਨਲ ਵਿੱਚ `dimradius` ਟਾਈਪ ਕਰੋ ਜਾਂ **Dimension Radius** ਟੂਲਬਾਰ ਬਟਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ।
2. ਇਸਨੂੰ ਚੁਣਨ ਲਈ **ਇੱਕ ਆਰਕ ਜਾਂ ਸਰਕਲ 'ਤੇ ਕਲਿੱਕ ਕਰੋ**।
3. ਡਾਇਮੈਂਸ਼ਨ ਲਾਈਨ ਨੂੰ ਸਥਿਤ ਕਰਨ ਲਈ **ਕਰਸਰ ਹਿਲਾਓ** — ਆਰਕ-ਬਿੰਦੂ ਕੇਂਦਰ ਤੋਂ ਕਰਸਰ ਦੀ ਦਿਸ਼ਾ ਦਾ ਪਿੱਛਾ ਕਰਦਾ ਹੈ।
4. ਡਾਇਮੈਂਸ਼ਨ ਰੱਖਣ ਲਈ **ਕਲਿੱਕ ਕਰੋ**।

ਸਿਰਫ਼ **Arc** ਅਤੇ **Circle** ਇਕਾਈਆਂ ਚੁਣੀਆਂ ਜਾ ਸਕਦੀਆਂ ਹਨ। ਕਿਸੇ ਹੋਰ ਇਕਾਈ ਕਿਸਮ 'ਤੇ ਕਲਿੱਕ ਕਰਨ ਨਾਲ ਕੁਝ ਨਹੀਂ ਹੁੰਦਾ।

| | Dimension Radius | Dimension Diameter |

| ਡਾਇਮੈਂਸ਼ਨ ਲਾਈਨ | ਕੇਂਦਰ → ਆਰਕ ਬਿੰਦੂ | ਆਰਕ ਬਿੰਦੂ → ਆਰਕ ਬਿੰਦੂ (ਕੇਂਦਰ ਵਿੱਚੋਂ) |
| ਲੇਬਲ ਪ੍ਰੀਫਿਕਸ | `R` | `⌀` |
| ਤੀਰ | ਇੱਕ (ਆਰਕ ਬਿੰਦੂ 'ਤੇ) | ਦੋ (ਦੋਵੇਂ ਆਰਕ ਬਿੰਦੂਆਂ 'ਤੇ) |
| ਇਸ ਲਈ ਵਧੀਆ | ਇੱਕ ਕਰਵਡ ਵਿਸ਼ੇਸ਼ਤਾ ਦੇ ਇੱਕ ਪਾਸੇ ਨੂੰ ਦਰਸਾਉਣਾ | ਪੂਰੇ ਸਰਕੂਲਰ ਡਾਇਮੈਂਸ਼ਨਾਂ ਨੂੰ ਦਰਸਾਉਣਾ |

## ਲੇਬਲ ਦਾ ਸੰਪਾਦਨ — ਸਿੰਪਲ ਮੋਡ

ਇੱਕ ਰੱਖੇ ਗਏ ਰੇਡੀਅਸ ਡਾਇਮੈਂਸ਼ਨ 'ਤੇ **ਡਬਲ-ਕਲਿੱਕ** ਕਰਨ ਨਾਲ ਟੈਕਸਟ ਐਡੀਟਰ **ਸਿੰਪਲ** ਮੋਡ ਵਿੱਚ ਖੁੱਲ੍ਹਦਾ ਹੈ। ਐਡੀਟਰ ਮੌਜੂਦਾ ਰੈਂਡਰ ਕੀਤੇ ਮੁੱਲ (ਜਿਵੇਂ `R 5.00`) ਨਾਲ ਪਹਿਲਾਂ ਤੋਂ ਭਰਿਆ ਹੁੰਦਾ ਹੈ ਤਾਂ ਜੋ ਤੁਸੀਂ ਕਰਸਰ ਨੂੰ ਸਥਿਤ ਕਰਕੇ ਸਿੱਧਾ ਸੰਪਾਦਿਤ ਕਰ ਸਕੋ।

| ਵਿਸ਼ੇਸ਼ਤਾ | ਵਿਵਹਾਰ |
|---------|-----------|
| Bold / Italic / Font / Height | **ਪੂਰੇ** ਲੇਬਲ 'ਤੇ ਇੱਕਸਾਰ ਲਾਗੂ ਹੁੰਦੇ ਹਨ |
| ਪ੍ਰਤੀ-ਅੱਖਰ ਫਾਰਮੈਟਿੰਗ | ਸਮਰਥਿਤ ਨਹੀਂ |
| `Enter` | ਮੁੱਲ ਕਮਿੱਟ ਕਰਦਾ ਹੈ ਅਤੇ ਐਡੀਟਰ ਬੰਦ ਕਰਦਾ ਹੈ |
| ਮਲਟੀ-ਲਾਈਨ | ਸਮਰਥਿਤ ਨਹੀਂ |

ਪੂਰੇ ਰੈਫਰੈਂਸ ਲਈ [Text Editor — simple mode](../interface/text-editor#simple-mode) ਵੇਖੋ।

## DXF — DIMENSION ਰੇਡੀਅਸ ਐਂਟਿਟੀ

ਰੇਡੀਅਸ ਡਾਇਮੈਂਸ਼ਨ `DIMENSION` ਐਂਟਿਟੀਆਂ ਦੇ ਰੂਪ ਵਿੱਚ ਸੰਭਾਲੇ ਜਾਂਦੇ ਹਨ ਜਿਹਨਾਂ ਵਿੱਚ ਰੇਡੀਅਸ-ਕਿਸਮ ਦੀ ਜਿਓਮੈਟਰੀ ਹੁੰਦੀ ਹੈ, ਜੋ ਕੇਂਦਰ ਦੇ ਨਿਰਦੇਸ਼ਾਂਕ, ਆਰਕ-ਬਿੰਦੂ ਸਥਿਤੀ, ਅਤੇ ਮਾਪਿਆ ਗਿਆ ਰੇਡੀਅਸ ਮੁੱਲ ਸੰਭਾਲਦੀ ਹੈ। ਸਾਰੀਆਂ ਵਿਸ਼ੇਸ਼ਤਾਵਾਂ ਬਿਨਾਂ ਕਿਸੇ ਨੁਕਸਾਨ ਦੇ ਰਾਊਂਡ-ਟ੍ਰਿਪ ਕਰਦੀਆਂ ਹਨ।
