# Move

`move` ਕਮਾਂਡ ਚੁਣੀਆਂ ਐਂਟਿਟੀਆਂ ਨੂੰ ਇੱਕ ਬੇਸ ਪੁਆਇੰਟ ਤੋਂ ਇੱਕ ਮੰਜ਼ਿਲ ਬਿੰਦੂ ਤੱਕ ਸਥਾਨਾਂਤਰਿਤ ਕਰਦੀ ਹੈ। ਹਰੇਕ ਚੁਣੀ ਐਂਟਿਟੀ 'ਤੇ ਲਾਗੂ ਹੋਣ ਵਾਲਾ ਵਿਸਥਾਪਨ ਬੇਸ ਤੋਂ ਮੰਜ਼ਿਲ ਤੱਕ ਦਾ ਵੈਕਟਰ ਹੁੰਦਾ ਹੈ। ਮੂਵ ਤੋਂ ਬਾਅਦ ਸਾਰੀਆਂ ਐਂਟਿਟੀਆਂ ਆਪਣੀ ਨਵੀਂ ਸਥਿਤੀ 'ਤੇ ਚੁਣੀਆਂ ਰਹਿੰਦੀਆਂ ਹਨ, ਹੋਰ ਸੰਪਾਦਨ ਲਈ ਤਿਆਰ।

## ਸ਼ੁਰੂ ਕਰਨ ਦੇ ਦੋ ਤਰੀਕੇ

**ਪਹਿਲਾਂ ਚੁਣੋ, ਫਿਰ ਮੂਵ ਕਰੋ** — ਪਹਿਲਾਂ ਐਂਟਿਟੀਆਂ ਚੁਣੋ, ਫਿਰ ਸਰਗਰਮ ਕਰੋ:

1. ਕੈਨਵਸ 'ਤੇ ਇੱਕ ਜਾਂ ਵੱਧ ਐਂਟਿਟੀਆਂ ਚੁਣੋ।
2. ਟਰਮੀਨਲ ਵਿੱਚ `move` ਟਾਈਪ ਕਰੋ ਜਾਂ **Move** ਟੂਲਬਾਰ ਬਟਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ।
3. **ਬੇਸ ਪੁਆਇੰਟ 'ਤੇ ਕਲਿੱਕ ਕਰੋ**, ਜਾਂ ਸਹੀ ਕੋਆਰਡੀਨੇਟ ਲਈ `X,Y` ਟਾਈਪ ਕਰੋ ਅਤੇ **Enter** ਦਬਾਓ।
4. **ਮੰਜ਼ਿਲ 'ਤੇ ਕਲਿੱਕ ਕਰੋ** — ਸਾਰੀਆਂ ਚੁਣੀਆਂ ਐਂਟਿਟੀਆਂ ਬੇਸ→ਮੰਜ਼ਿਲ ਵੈਕਟਰ ਨਾਲ ਸ਼ਿਫਟ ਹੁੰਦੀਆਂ ਹਨ। ਕੋਆਰਡੀਨੇਟ ਐਂਟਰੀ ਇੱਥੇ ਵੀ ਕੰਮ ਕਰਦੀ ਹੈ।

**ਸਰਗਰਮ ਕਰੋ, ਫਿਰ ਚੁਣੋ** — ਕੁਝ ਵੀ ਚੁਣੇ ਬਿਨਾਂ ਕਮਾਂਡ ਸ਼ੁਰੂ ਕਰੋ:

1. `move` ਟਾਈਪ ਕਰੋ ਜਾਂ ਟੂਲਬਾਰ ਬਟਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ।
2. **ਆਬਜੈਕਟ ਚੁਣੋ** — ਵਿਅਕਤੀਗਤ ਐਂਟਿਟੀਆਂ ਟੌਗਲ ਕਰਨ ਲਈ ਕਲਿੱਕ ਕਰੋ, ਜਾਂ ਖੇਤਰ ਦੁਆਰਾ ਚੁਣਨ ਲਈ ਖਿੱਚੋ।
3. ਚੋਣ ਪੁਸ਼ਟੀ ਕਰਨ ਲਈ **Enter** ਜਾਂ **Space** ਦਬਾਓ।
4. **ਬੇਸ ਪੁਆਇੰਟ 'ਤੇ ਕਲਿੱਕ ਕਰੋ**, ਫਿਰ **ਮੰਜ਼ਿਲ 'ਤੇ ਕਲਿੱਕ ਕਰੋ** (ਦੋਵੇਂ ਪੜਾਵਾਂ 'ਤੇ ਕੋਆਰਡੀਨੇਟ ਐਂਟਰੀ ਉਪਲਬਧ)।

```
  Before:                    After:
  ● base                       → ● destination
  [entity A]                      [entity A moved]
  [entity B]                      [entity B moved]
```

ਸਾਰੀਆਂ ਚੁਣੀਆਂ ਐਂਟਿਟੀਆਂ ਦਾ ਇੱਕ ਭੂਤ ਪ੍ਰੀਵਿਊ ਬੇਸ ਪੁਆਇੰਟ ਤੋਂ ਮੰਜ਼ਿਲ ਤੱਕ ਕਰਸਰ ਦਾ ਪਿੱਛਾ ਕਰਦਾ ਹੈ, ਕਲਿੱਕ ਕਰਨ ਤੋਂ ਪਹਿਲਾਂ ਨਤੀਜਾ ਦਿਖਾਉਂਦਾ ਹੈ।

## ਕੋਆਰਡੀਨੇਟ ਐਂਟਰੀ

ਬੇਸ-ਪੁਆਇੰਟ ਜਾਂ ਮੰਜ਼ਿਲ ਪੜਾਅ 'ਤੇ, ਕਲਿੱਕ ਕਰਨ ਦੀ ਬਜਾਏ ਸਹੀ ਸਥਿਤੀ ਟਾਈਪ ਕਰੋ:

1. X ਮੁੱਲ ਟਾਈਪ ਕਰੋ।
2. `,` ਦਬਾਓ — ਟਰਮੀਨਲ `[X], [Y{cursor}]` ਦਿਖਾਉਂਦਾ ਹੈ।
3. Y ਮੁੱਲ ਟਾਈਪ ਕਰੋ।
4. ਪੁਸ਼ਟੀ ਕਰਨ ਲਈ **Enter** ਦਬਾਓ।

## ਐਂਗਲ ਲਾਕਿੰਗ ਅਤੇ ਸਹੀ ਦੂਰੀ

ਬੇਸ ਪੁਆਇੰਟ ਸੈੱਟ ਹੋਣ ਤੋਂ ਬਾਅਦ ਕਮਾਂਡ 45° ਸਨੈਪ ਧੁਰੇ (0°, 45°, 90°, 135°, …) ਲਈ ਧਿਆਨ ਰੱਖਦੀ ਹੈ। ਜਦੋਂ ਕਰਸਰ ਬੇਸ ਤੋਂ ਕਾਫ਼ੀ ਦੂਰ ਹੋਵੇ ਅਤੇ ਧੁਰੇ ਦੀ ਇੱਕ ਗ੍ਰਿਪ ਚੌੜਾਈ ਦੇ ਅੰਦਰ ਹੋਵੇ ਤਾਂ ਦਿਸ਼ਾ **ਲਾਕ** ਹੋ ਜਾਂਦੀ ਹੈ। ਲਾਕ ਹੋਣ ਦੌਰਾਨ:

- ਭੂਤ ਪ੍ਰੀਵਿਊ ਧੁਰੇ 'ਤੇ ਸਨੈਪ ਹੁੰਦਾ ਹੈ।
- ਲਾਕ ਕੀਤੀ ਦਿਸ਼ਾ ਦੇ ਨਾਲ ਠੀਕ ਉਸ ਦੂਰੀ 'ਤੇ ਮੂਵ ਕਰਨ ਲਈ ਇੱਕ ਦੂਰੀ ਟਾਈਪ ਕਰੋ ਅਤੇ **Enter** ਦਬਾਓ।
- ਕਲਿੱਕ ਕਰਨ ਨਾਲ ਧੁਰੇ 'ਤੇ ਪ੍ਰੋਜੈਕਟ ਹੁੰਦਾ ਹੈ, ਇਸ ਲਈ ਮੰਜ਼ਿਲ ਹਮੇਸ਼ਾ ਠੀਕ ਉਸ 'ਤੇ ਪੈਂਦੀ ਹੈ।

| ਕੀ | ਕਾਰਵਾਈ |
|-----|--------|
| `0`–`9`, `.` | ਦੂਰੀ ਮੁੱਲ ਵਿੱਚ ਜੋੜੋ |
| `-` | ਨਕਾਰਾਤਮਕ ਦੂਰੀ — ਧੁਰੇ ਦੇ ਨਾਲ ਦਿਸ਼ਾ ਉਲਟਾਉਂਦੀ ਹੈ (ਸਿਰਫ਼ ਪਹਿਲਾ ਅੱਖਰ) |
| `Backspace` | ਆਖਰੀ ਟਾਈਪ ਕੀਤਾ ਅੱਖਰ ਮਿਟਾਓ |
| `Enter` | ਟਾਈਪ ਕੀਤੀ ਦੂਰੀ 'ਤੇ ਮੂਵ ਲਾਗੂ ਕਰੋ |

## ਕੀਬੋਰਡ ਰੈਫਰੈਂਸ

| ਕੀ | ਕਾਰਵਾਈ |
|-----|--------|
| `Enter` / `Space` | ਚੋਣ ਪੁਸ਼ਟੀ ਕਰੋ ਅਤੇ ਬੇਸ-ਪੁਆਇੰਟ ਪੜਾਅ 'ਤੇ ਅੱਗੇ ਵਧੋ |
| `0`–`9`, `.`, `-` | X ਕੋਆਰਡੀਨੇਟ ਐਂਟਰੀ ਸ਼ੁਰੂ ਕਰੋ, ਜਾਂ ਐਂਗਲ-ਲਾਕ ਹੋਣ 'ਤੇ ਦੂਰੀ |
| `,` | X ਲਾਕ ਕਰੋ ਅਤੇ Y ਐਂਟਰੀ 'ਤੇ ਜਾਓ |
| `Backspace` | ਆਖਰੀ ਟਾਈਪ ਕੀਤਾ ਅੱਖਰ ਮਿਟਾਓ |
| `Enter` | ਕੋਆਰਡੀਨੇਟ ਪੁਸ਼ਟੀ ਕਰੋ ਜਾਂ ਟਾਈਪ ਕੀਤੀ ਦੂਰੀ 'ਤੇ ਮੂਵ ਲਾਗੂ ਕਰੋ |
| `Escape` | ਰੱਦ ਕਰੋ ਅਤੇ ਰੀਸੈੱਟ ਕਰੋ |

## ਗ੍ਰਿਪ ਤੋਂ Move ਸਰਗਰਮ ਕਰਨਾ

ਇੱਕ ਚੁਣੀ ਹੋਈ [Line](./line) ਦੀ **ਮਿਡਪੁਆਇੰਟ ਗ੍ਰਿਪ** 'ਤੇ ਕਲਿੱਕ ਕਰਨ ਨਾਲ Move ਆਪਣੇ ਆਪ ਸ਼ੁਰੂ ਹੋ ਜਾਂਦੀ ਹੈ, ਮਿਡਪੁਆਇੰਟ ਪਹਿਲਾਂ ਹੀ ਬੇਸ ਪੁਆਇੰਟ ਵਜੋਂ ਸੈੱਟ ਅਤੇ ਮੂਵ ਪੜਾਅ ਸਰਗਰਮ ਹੁੰਦਾ ਹੈ। ਚੋਣ ਪੜਾਅ ਵਿੱਚੋਂ ਲੰਘੇ ਬਿਨਾਂ ਇੱਕ ਸਿੰਗਲ ਲਾਈਨ ਨੂੰ ਮੁੜ-ਸਥਾਪਿਤ ਕਰਨ ਦਾ ਇਹ ਸਭ ਤੋਂ ਤੇਜ਼ ਤਰੀਕਾ ਹੈ।

## ਕਮਾਂਡ ਦੌਰਾਨ ਚੋਣ

ਜਦੋਂ ਕਮਾਂਡ ਚੋਣ ਪੜਾਅ ਵਿੱਚ ਸ਼ੁਰੂ ਹੁੰਦੀ ਹੈ:

| ਢੰਗ | ਵਿਵਹਾਰ |
|--------|-----------|
| **ਕਲਿੱਕ** | ਕਰਸਰ ਹੇਠਾਂ ਐਂਟਿਟੀ ਨੂੰ ਚੋਣ ਵਿੱਚ/ਬਾਹਰ ਟੌਗਲ ਕਰਦਾ ਹੈ |
| **ਸੱਜੇ ਖਿੱਚੋ** (ਸਖ਼ਤ) | ਬਾਕਸ ਦੇ ਪੂਰੀ ਤਰ੍ਹਾਂ ਅੰਦਰ ਵਾਲੀਆਂ ਐਂਟਿਟੀਆਂ ਜੋੜਦਾ ਹੈ |
| **ਖੱਬੇ ਖਿੱਚੋ** (ਕ੍ਰਾਸਿੰਗ) | ਬਾਕਸ ਦੀ ਸੀਮਾ ਨੂੰ ਕੱਟਣ ਵਾਲੀਆਂ ਐਂਟਿਟੀਆਂ ਜੋੜਦਾ ਹੈ |
| **Enter** / **Space** | ਚੋਣ ਪੁਸ਼ਟੀ ਕਰਦਾ ਹੈ ਅਤੇ ਬੇਸ-ਪੁਆਇੰਟ ਪੜਾਅ 'ਤੇ ਜਾਂਦਾ ਹੈ |

## ਮੂਵ ਤੋਂ ਬਾਅਦ

ਮੂਵ ਕੀਤੀਆਂ ਐਂਟਿਟੀਆਂ ਆਪਣੀ ਨਵੀਂ ਸਥਿਤੀ 'ਤੇ ਚੁਣੀਆਂ ਰਹਿੰਦੀਆਂ ਹਨ। ਇਸਦਾ ਮਤਲਬ ਹੈ ਕਿ ਤੁਸੀਂ ਤੁਰੰਤ:
- ਉਹਨਾਂ ਨੂੰ ਹੋਰ ਖਿਸਕਾਉਣ ਲਈ **Move** ਦੁਬਾਰਾ ਚਲਾ ਸਕਦੇ ਹੋ।
- ਦੁਬਾਰਾ ਚੁਣੇ ਬਿਨਾਂ [Copy](./copy), [Rotate](./rotate), ਜਾਂ [Scale](./scale) ਚਲਾ ਸਕਦੇ ਹੋ।
- ਉਹਨਾਂ ਨੂੰ ਹਟਾਉਣ ਲਈ **Delete** ਦਬਾ ਸਕਦੇ ਹੋ।

## ਸਮਰਥਿਤ ਐਂਟਿਟੀਆਂ

Move ਹਰੇਕ ਐਂਟਿਟੀ ਕਿਸਮ 'ਤੇ ਕੰਮ ਕਰਦਾ ਹੈ: Line, Polyline, Rectangle, Circle, Arc, Ellipse, Text, Spline, Dimension, Leader, ਅਤੇ ਹੋਰ ਸਾਰੀਆਂ। ਸਾਰੀਆਂ ਐਂਟਿਟੀਆਂ `translate(dx, dy)` ਲਾਗੂ ਕਰਦੀਆਂ ਹਨ ਇਸ ਲਈ ਕੋਈ ਵੀ ਬਾਹਰ ਨਹੀਂ ਹੈ।
