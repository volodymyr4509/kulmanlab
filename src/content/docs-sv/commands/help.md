# Help

Öppnar hjälppanelen i appen med en sammanfattning av alla tangentbordsgenvägar, navigeringstips och allmän användning.

## Så här använder du det

Klicka på verktygsfältsknappen **Help** eller skriv `help` i terminalen. Hjälppanelen öppnas omedelbart och kommandot avslutas.

## Vad som finns i hjälppanelen

### Navigering
| Genväg | Åtgärd |
|----------|--------|
| `Scrollhjul` | Zooma in / ut |
| `Mittenklick + dra` | Panorera vyn |
| `Dubbelt mittenklick` | Anpassa alla entiteter till skärmen |

### Kommandon
| Genväg | Åtgärd |
|----------|--------|
| Skriv | Börja skriva ett kommandonamn — förslag visas på ritytan |
| `Delete` | Ta bort alla markerade entiteter |
| `Enter` / `Space` | Bekräfta kommando eller inmatning |
| `Enter` / `Space` (inaktiv) | Upprepa det senast använda kommandot |
| `Escape` | Avbryt det aktuella kommandot eller rensa markeringen |

### Markering
| Genväg | Åtgärd |
|----------|--------|
| `Vänsterklick` | Markera en entitet |
| `Dra →` | Markera endast entiteter som ligger **helt inom** dragområdet |
| `Dra ←` | Markera alla entiteter som **berör** dragområdet |
| `Vänsterklick på grepp` | Redigera en vertex eller ändpunkt direkt |

### Rittips
- **Fångpunkter** — markören fångas automatiskt till närliggande entiteters ändpunkter.
- **Klibbig vinkel** — flytta markören tillräckligt långt i en riktning för att låsa till steg om 45°.
- **Exakt längd** — medan vinkeln är låst, skriv ett tal och tryck på **Enter** för att ställa in ett exakt avstånd.
