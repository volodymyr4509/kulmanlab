# PageManager

`PageManager`-kommandot öppnar dialogrutan för sidinställningar för det aktiva layoutet, där du kan ändra pappersformatet, orienteringen och skalan mellan ritningsenheter och millimeter.

## Öppna Page Manager

- Skriv `PageManager` i terminalen, **eller**
- Högerklicka på en layoutflik längst ner och välj **Page Manager**.

## Inställningar

| Inställning | Beskrivning |
|---------|-------------|
| **Pappersstorlek** | Standardarkstorlek (A4, A3, A2, Letter, etc.) |
| **Orientering** | Stående eller Liggande |
| **Skala** | Ritningsenheter per mm — styr hur modellkoordinater mappas till papperet |

Att ändra pappersstorleken eller skalan skalar om alla befintliga viewportar i layoutet proportionellt så att deras positioner förblir korrekta.

## Anteckningar

- PageManager gäller endast för papperslayouter. Det är inte tillgängligt i modellutrymme.
- För att lägga till ett nytt layout, klicka på **+**-knappen i fliklisten längst ner.
- För att byta namn på eller ta bort ett layout, högerklicka på dess flik.
